# <a name="test-voice-routing-in-lync-server-2013"></a>Проверка маршрутизации голосовой связи в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-24_

Вы можете настроить сценарии тестовых примеров с помощью вкладки " **Тестовая Маршрутизация голосовой связи** " в панели управления Lync Server. Чтобы задать тестовый случай, следует указать абонентскую группу, политику голосовой связи, вариант использования PSTN и маршрут голосовой связи, применительно к которому следует выполнять тестирование указанного номера телефона.

Перед фактическим развертыванием конфигурации маршрутизации голосовых вызовов рекомендуется протестировать ее на различных телефонных номерах, чтобы убедиться в том, что результаты являются ожидаемыми.

<div>


> [!TIP]  
> Можно использовать команды <STRONG>Экспорт тестовых случаев</STRONG> и <STRONG>Импорт тестовых случаев</STRONG> для сохранения случаев тестирования маршрутизации голосовой связи и их импорта для использования на другом компьютере.



</div>

<div>


> [!WARNING]  
> Если вы удаляете какую-либо часть конфигурации маршрутизации голосовой связи, например абонентскую группу, политику голосовой связи, маршрут голосовой связи или вариант использования телефона, следует просмотреть и обновить случаи тестирования маршрутизации голосовой связи. Панель управления Lync Server не предупреждает о тестовых случаях, которые больше не являются допустимыми из-за изменения конфигурации.
