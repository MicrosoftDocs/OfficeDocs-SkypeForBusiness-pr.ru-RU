# Предоставление разрешений на установку в Lync Server 2013

 

_**Дата изменения раздела:** 2012-08-27_

Командлет **Grant-CsSetupPermission** можно использовать для добавления разрешений уровня Read, Write, ReadSPN и WriteSPN в группу RTCUniversalServerAdmins для указанного подразделения Active Directory. После этого члены группы RTCUniversalServerAdmins данного подразделения, которые не являются участниками группы «Администраторы домена» смогут устанавливать серверы под управлением Lync Server 2013 в указанном домене.

С помощью командлета **Test-CsSetupPermission** можно проверить разрешения, заданные с использованием командлета **Grant-CsSetupPermission**.

Вы можете использовать командлет **Revoke-CsSetupPermission** для удаления разрешений, предоставленных с помощью командлета **Grant-CsSetupPermission**.

## Порядок предоставления разрешений на установку

1.  Выполните вход в систему компьютера, на котором выполняется Lync Server 2013, в домене, в котором необходимо предоставить разрешения на установку. Используйте учетную запись, которая является членом группы администраторов домена или группы администраторов предприятия, если подразделение находится в другом дочернем домене.

2.  Запустите командную консоль Lync Server: нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы** и **Microsoft Lync Server 2013** и щелкните элемент **Командная консоль Lync Server**.

3.  Выполните следующую команду:
    
        Grant-CsSetupPermission -ComputerOu <DN of the OU or container where the computer objects that will run Lync Server reside > [-Domain <Domain FQDN>]
    
    Можно указать параметр ComputerOu в соответствии с контекстом именования по умолчанию указанного домена (например, CN=Computers). Кроме того, можно указать этот параметр как полное различающееся имя подразделения (например, CN=Computers, DC=Contoso, DC=com). В последнем случае необходимо указать полное различающееся имя подразделения, соответствующее указанному домену.
    
    Если параметр Domain не указан, значением по умолчанию является локальный домен.
