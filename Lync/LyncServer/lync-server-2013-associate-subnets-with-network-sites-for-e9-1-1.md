# Сопоставление подсетей с сетевыми сайтами для E9-1-1 в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-02_

Каждая подсеть в вашей сети, для которой вы хотите включить службу E9-1-1, должна быть связана с конкретным узлом сети. Это обусловлено тем, что сведения о подсети используются для определения сайта сети, на котором находится конечная точка.

E9-1-1 не предъявляет никаких особых требований относительно связи подсетей с сайтами сети. Для создания связи между подсетями и сайтами сети в топологии следуйте процедурам, описанным в [Связь подсети с сетевым сайтом в Lync Server 2013](lync-server-2013-associate-a-subnet-with-a-network-site.md).
