# Настройка узла-наблюдателя для запуска искусственных транзакций

 

_**Дата изменения раздела:** 2015-03-09_

После установки файлов агента System Center необходимо настроить узел-наблюдатель. Действия, выполняемые при настройке узла-наблюдателя, зависят от того, относится ли компьютер, являющийся узлом-наблюдателем, к сети периметра или находится за ее пределами.

При настройке узла-наблюдателя необходимо также выбрать тип метода проверки подлинности для развертывания этим узлом. Lync Server 2013 позволяет выбрать один из двух методов проверки подлинности: доверенный сервер или проверка подлинности учетных данных. Необходимо также проверить, заданы ли в брандмауэре правила входящего доступа для процессов MonitoringHost.exe и PowerShell.exe. Если эти процессы блокированы брандмауэром, то ваши искусственные транзакции будут выполнены неудачно с выводом сообщения об ошибке 504 (превышение времени ожидания сервера)..
