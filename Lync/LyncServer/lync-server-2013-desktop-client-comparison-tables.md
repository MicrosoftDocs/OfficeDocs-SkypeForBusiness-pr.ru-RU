# <a name="desktop-client-comparison-tables-for-lync-server-2013"></a>Таблицы сравнения клиентов для настольных ПК для Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2018-01-23_

В следующих таблицах сравниваются функции и возможности клиентов Lync 2013 в следующих категориях:

  - Поддержка расширенных возможностей присутствия

  - Поддержка контактов и групп контактов

  - Поддержка обмена мгновенными сообщениями

  - Поддержка конференц-связи

  - Поддержка телефонной связи

  - Поддержка внешних пользователей

  - Поддержка архивации, обеспечения соответствия требованиям и ведения журналов

  - Современная проверка подлинности

В этих таблицах указываются функции, доступные пользователям Lync в локальном развертывании Lync Server 2013. Те же функции доступны в Skype для бизнеса Online и Office 365 для пользователей, если не указано иное.

<div>


> [!IMPORTANT]  
> В <A href="lync-server-2013-mobile-client-comparison-tables.md">таблицах сравнения мобильных клиентов для Lync Server 2013</A> вы найдете подробные сведения о доступности функций в мобильных клиентах Lync 2013.

## <a name="client-limitations"></a>Ограничения для клиентов

<span id="Full_Basic"></span>

<div>

## <a name="basic-client-limitations"></a>Ограничения базового клиента

Приведенные ниже функции доступны при использовании полного клиента и недоступны в базовом клиенте.

  - Управление параметрами групповых звонков

  - Управление делегатами

  - Звонки от имени другого контакта (сценарий "руководитель/делегат")

  - Обработка звонков другого пользователя делегатом

  - Управление большим количеством звонков

  - Инициация звонка в группу ответа

  - Парковка вызовов

  - Ответ на вызов в группе

</div>

<span id="Online_Hybrid"></span>

<div>

## <a name="online-or-hybrid-user-account-limitations"></a>Ограничения учетных записей в сети или гибридной учетной записи пользователя

Учетные записи пользователей могут существовать в Интернете или локальной среде и влияют на возможности, доступные для этого пользователя. Пользователи, у которых есть учетные записи в Skype для бизнеса Online, не смогут получить доступ к следующим функциям, даже при использовании полного клиента:

  - Расширенные сведения о присутствии: использовать фотографию с любого общедоступного сайта для моего изображения

  - Контакты: Поиск групп ответа

  - Поддержка обмена мгновенными сообщениями: интеграция сохраняемого чата (группового чата)

  - Поддержка обмена мгновенными сообщениями: расширение комнаты сохраняемого чата на собрание одним щелчком мыши

  - Внешние пользователи: выполнение двусторонних или многосторонних вызовов с внешними пользователями

</div>
