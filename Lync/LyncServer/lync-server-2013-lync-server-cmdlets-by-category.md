# <a name="lync-server-2013-cmdlets-by-category"></a>Командлеты Lync Server 2013 по категориям

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2017-09-20_

Microsoft Lync Server 2013 поставляется с практическими командлетами 550, разработанными специально для разрешения администраторам управления сервером Lync Server из командной строки. Вы получите доступ к командлетам из командной консоли Lync Server Management Shell. Вы можете получить справку по командлету прямо из командной строки, введя команду, подобную следующей:

    Get-Help New-CsVoicePolicy -Full

Предыдущая команда извлечет все справочные материалы, доступные для командлета **New-ксвоицеполици** . Замените ссылку на **New-ксвоицеполици** именем командлета, для которого требуется получить справку.

Чтобы получить полный список командлетов, доступных для управления Microsoft Lync Server 2013, в командной строке оболочки Lync Server Management Shell введите следующую команду:

    Get-Command * -Module Lync -CommandType cmdlet

Если вы не знаете, какие командлеты вам нужны, мы также предоставили список командлетов и их справочные разделы. Вы обнаружите, что некоторые командлеты отображаются в нескольких категориях, которые были намеренно применены к нескольким областям продукта. Ниже приведен список категорий.
