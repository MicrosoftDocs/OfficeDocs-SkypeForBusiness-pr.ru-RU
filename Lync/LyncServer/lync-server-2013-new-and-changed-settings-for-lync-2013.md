# <a name="new-and-changed-settings-for-lync-2013"></a>Новые и измененные параметры для Lync 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-12-05_

В этом разделе обсуждаются изменения командлетов командной консоли Lync Server, которые относятся непосредственно к управлению клиентами. Lync Server 2013 содержит несколько новых параметров, а также устаревшие параметры для функций, которые можно настроить с помощью других средств.
