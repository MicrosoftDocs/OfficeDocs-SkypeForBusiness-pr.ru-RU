# <a name="client-policies-and-settings-in-lync-server-2013"></a>Политики и параметры клиента в Lync Server 2013

_**Последнее изменение темы:** 2012-06-18_

В этом разделе представлен обзор параметров и политик, связанных с клиентами, которые можно настроить в Lync Server 2013. Lync Server 2013 содержит следующие инструменты для управления и настройки клиентов:

  - Панель управления Lync **Server 2013**     Веб-интерфейсный пользовательский интерфейс для управления и настройки серверов, пользователей, клиентов и устройств.

  - **Командная консоль**     Lync Server Интерфейс управления с широким набором командлетов интерфейса командной строки Windows PowerShell и несколькими предварительно определенными скриптами.

  - **Групповая политика**     Lync 2013 Набор политик, которые можно настроить для клиентов с помощью административного шаблона групповой политики Office. Прежде чем развертывать клиенты Lync 2013, необходимо настроить некоторые политики начальной загрузки клиента. Другие необязательные параметры из Lync 2010 продолжают учитываться в Lync 2013.

В этом разделе описываются изменения параметров, связанных с клиентами, в Lync Server 2013.
