# <a name="disabling-uploading-and-downloading-files-in-chat-rooms-in-lync-server-2013"></a>Отключение отправки и скачивания файлов в комнатах чата в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-12_

По умолчанию пользователи могут отправлять и загружать файлы в сообщениях, которые они отправляют. Администратор может отключить эту функциональность и запретить пользователям отправку и загрузку файлов двумя способами.

  - Запретить всем пользователям отправку и скачивание файлов в определенной комнате сохраняемого чата или в категории комнат чата.

  - Запретить определенному пользователю отправку и загрузку файлов во всех комнатах чата. Этот пользователь не сможет отправлять и загружать файлы в любой комнате чата, включая комнаты чата, которые разрешают отправку и загрузку файлов.

Для получения дополнительных сведений о том, как включить и отключить отправку и скачивание файлов, ознакомьтесь со статьей [Настройка категорий в Lync Server 2013](lync-server-2013-configure-categories.md) и [Управление категориями](manage-categories.md) в документации по развертыванию.
