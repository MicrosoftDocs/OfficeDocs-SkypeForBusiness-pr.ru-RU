# Управление большими собраниями

 

_**Дата изменения раздела:** 2012-10-01_

После настройки выделенного пула для конференций с большим числом участников вы можете проверить, обеспечивают ли собрания с большим числом участников, размещенные в пуле, наилучшее взаимодействие с пользователями. В подразделах этого раздела описывается, как организовывать собрания с большим числом участников и управлять ими.
