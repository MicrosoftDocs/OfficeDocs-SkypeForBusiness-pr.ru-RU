# <a name="new-group-call-pickup-feature-in-lync-server-2013"></a>Новая функция ответа на звонки в группе Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-12_

Накопительные пакеты обновления для Lync Server 2013: в феврале 2013 полагаются на запрос групп в качестве новой функции корпоративной голосовой связи. С помощью группового ответа на звонки вы можете назначить пользователям группы, чтобы другие пользователи могли отвечать на входящие вызовы пользователей, находящихся в группе, с собственных телефонов.

Группа ответа на звонки основана на приложении парковки вызовов. Для ответа на групповые звонки, как в случае парковки вызовов, вы настраиваете диапазон виртуальных номеров для использования в качестве номеров групп. Пользователь набирает номер группы, чтобы получить вызов, который будет звонить другому пользователю, находящийся в группе.
