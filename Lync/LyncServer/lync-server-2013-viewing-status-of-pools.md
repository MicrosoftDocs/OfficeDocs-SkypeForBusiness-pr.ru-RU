# Viewing status of pools in Lync Server 2013

 

_**Дата изменения раздела:** 2014-05-01_

For each Enterprise Pool and the pool of each Standard Edition Server, you can view information about pool status and the status of other components used by the pool through the управления Lync Server 2013. We recommended implementing System Center Operations Manager 2012 with the управления Lync Server to help administrators in viewing the health of pools within their environment.

## View pools

1.  Using the управления Lync Server, open the управления Lync Server.

2.  Click Topology, and check status of all servers in the Topology.
