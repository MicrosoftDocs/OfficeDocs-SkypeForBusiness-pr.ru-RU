# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Требования к сертификатам для сервера сохраняемого чата в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-03_

Чтобы установить сервер сохраняемого чата, у вас должен быть сертификат, выданный тем же центром сертификации, что и у сервера Lync Server 2013, который используется на внутренних серверах, на котором запущены веб-службы сохраняемого чата для отправки и загрузки файлов. Убедитесь, что у вас есть необходимые сертификаты перед началом установки сохраняемого чата, особенно если вы используете внешний центр сертификации.
