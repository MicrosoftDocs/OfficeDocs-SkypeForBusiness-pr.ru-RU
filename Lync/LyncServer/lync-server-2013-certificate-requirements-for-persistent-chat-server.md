# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Требования к сертификатам для сервера сохраняемого чата в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-03_

Чтобы установить сервер сохраняемого чата, у вас должен быть сертификат, выданный тем же ЦС, что и сервер Lync Server 2013 Internal Servers для каждого сервера, на котором запущены веб-службы сохраняемого чата для отправки и загрузки файлов. Перед началом установки сохраняемого чата убедитесь, что у вас есть необходимые сертификаты, особенно при использовании внешнего центра сертификации.
