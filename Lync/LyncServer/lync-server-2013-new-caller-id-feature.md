# <a name="new-caller-id-feature-in-lync-server-2013"></a>Новая функция идентификации звонящего в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-05_

Lync Server 2013 предоставляет администратору гибкие возможности для изменения формата телефонного номера абонента. Данная функция представления кода вызывающего абонента дает администратору возможность изменять номер телефона вызывающей стороны в формат набора, который понятен для однорангового узла магистрали, если в этом возникает необходимость. Например, можно написать правило преобразования для удаления +44 из начала строки набора и замены на 0144.

Теперь, при использовании Lync Server 2013, телефонный номер абонента и номер телефона вызываемого абонента могут быть переведены в разные форматы по мере необходимости. Такая гибкость позволяет Lync Server 2013 выступать в качестве переводчика магистрали между различными системами телефонии.
