# Уменьшение количества нежелательных мгновенных сообщений для Lync Server 2013

 

_**Дата изменения раздела:** 2013-12-05_

Приложение Intelligent IM Filter помогает защитить развертывание Microsoft Lync Server 2013 от наиболее распространенных вирусов с минимальным уменьшением удобства работы пользователей. Фильтр Intelligent IM предоставляет следующее:

  - улучшенную фильтрацию URL-адресов;

  - улучшенную фильтрацию при передаче файлов.

Используйте Intelligent IM Filter, чтобы настроить фильтры для блокировки потенциально вредоносных мгновенных сообщений от неизвестных конечных точек за пределами корпоративного брандмауэра. Фильтры настраиваются путем задания критерия блокировки. Например, можно блокировать сообщения, содержащие гиперссылки и файлы с определенными расширениями.

Перед разворачиванием приложения Intelligent IM Message Filter следует понять, как параметры фильтрации применяются при маршрутизации сообщений от одного сервера Lync Server 2013 к другому. Параметры фильтрации применяются согласованно независимо от расположения серверов — как в пределах одной организации, так и при обмене данными между несколькими организациями. Это согласованное применение относится к способу вставки в сообщения настраиваемых уведомлений и предупреждений и к способу их отправки на другие серверы.

Рекомендованный вариант фильтрации — разрешить мгновенные сообщения с гиперссылками, но требовать, чтобы приложение Intelligent IM Filter блокировало ссылку, вставляя перед ней символ подчеркивания. В этом случае есть дополнительная возможность создания уведомления для пользователей, которое появляется в начале каждого мгновенного сообщения с гиперссылкой.

Второй вариант фильтрации — разрешить мгновенные сообщения с немодифицированными гиперссылками. В этом случае есть дополнительная возможность вставки в каждое сообщение предупреждения для пользователей (рекомендуется).

Третий вариант — блокировать все мгновенные сообщения с гиперссылками. В этом случае сервер отправляет пользователю предупреждение. Текст этого предупреждения вы должны написать.
