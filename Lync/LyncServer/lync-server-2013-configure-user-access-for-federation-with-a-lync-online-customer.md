# <a name="configure-user-access-for-federation-with-a-lync-online-customer-in-lync-server-2013"></a>Настройка доступа пользователей для Федерации с клиентом Lync Online в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-11-01_

Вам нужно настроить учетные записи всех пользователей организации, которым должно быть разрешено взаимодействие с федеративными партнерами. Эта конфигурация применяется для всех федеративных партнеров, включая все домены клиентов Microsoft Lync Online 2010, с которыми поддерживается Федерация. Дополнительные сведения о настройке поддержки федерации для учетных записей пользователей приведены в статье [Настройка политик для управления доступом федеративных пользователей в Lync server 2013](lync-server-2013-configure-policies-to-control-federated-user-access.md) и [Назначение политики доступа внешних пользователей пользователю, поддерживающему Lync](lync-server-2013-assign-an-external-user-access-policy-to-a-lync-enabled-user.md) , в документации по операциям в среде Lync Server 2013.
