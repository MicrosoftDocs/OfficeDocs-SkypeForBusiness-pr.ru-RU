# <a name="delete-dial-in-conferencing-pin-settings-for-a-site-or-group-of-users-in-lync-server-2013"></a>Удаление параметров контакта конференц-связи с телефонным подключением для сайта или группы пользователей в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-18_

Выполните эти действия, чтобы удалить политику ПИН уровня пользователя или сайта.

<div>


> [!NOTE]
> Глобальную политику ПИН-кодов удалить нельзя.



</div>

<div>

## <a name="to-delete-a-user-or-site-pin-policy"></a>Удаление политики ПИН для пользователя или сайта

1.  Войдите в учетную запись пользователя, которая является членом группы Рткуниверсалсерверадминс (или имеет эквивалентные права пользователей) или назначьте роль Кссерверадминистратор или Ксадминистратор, выполните вход на любой компьютер в сети, в которой вы развернули Lync Server 2013.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных способах, которые можно использовать для запуска панели управления Lync Server, приведены в разделе [Открытие меню администрирования Lync server 2013](lync-server-2013-open-lync-server-administrative-tools.md).

3.  В левой панели навигации щелкните элемент **Конференц-связь**, а затем щелкните элемент **Политика ПИН-кодов**.

4.  На странице " **политика ПИН-кода** " в поле "Поиск" введите имя или часть имени политики, которую вы хотите удалить.

5.  В списке политик выберите необходимую политику, щелкните **Правка**, затем выберите **Удалить**.

6.  Нажмите **ОК**.
