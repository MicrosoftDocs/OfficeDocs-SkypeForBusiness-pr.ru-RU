# Управление инфраструктурой сети Lync Server 2013

 

_**Дата изменения раздела:** 2014-02-11_

Microsoft Lync Server 2013 обеспечивает поддержку контроля допуска звонков (CAC) и обход сервера-посредника. Для реализации этих возможностей необходимо настроить сетевые области, узлы, подсети и т. д., которые позволят регулировать полосу пропускания в ситуациях, когда передачу аудио- и видеоданных нужно ограничить. Можно также использовать сетевую технологию качества обслуживания (QoS) для обеспечения оптимальной работы пользователя при организации аудио- и видеосвязи.

Панель управления Lync Server можно использовать для настройки и управления CAC, обхода сервера-посредника и QoS. В следующих разделах описаны действия, позволяющие это сделать
