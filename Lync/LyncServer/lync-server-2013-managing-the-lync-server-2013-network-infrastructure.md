# <a name="managing-the-lync-server-2013-network-infrastructure"></a>Управление инфраструктурой сети Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-02-11_

Microsoft Lync Server 2013 включает поддержку контроля допуска звонков (CAC) и обход сервера-посредника. Для реализации этих возможностей необходимо настроить сетевые области, узлы, подсети и т. д., которые позволят регулировать полосу пропускания в ситуациях, когда передачу аудио- и видеоданных нужно ограничить. Можно также использовать сетевую технологию качества обслуживания (QoS) для обеспечения оптимальной работы пользователя при организации аудио- и видеосвязи.

С помощью панели управления Lync Server вы можете настроить CAC и управлять ими, а также выполнять обход сервера мультимедиа и качество обслуживания. В следующих разделах описаны действия, позволяющие это сделать
