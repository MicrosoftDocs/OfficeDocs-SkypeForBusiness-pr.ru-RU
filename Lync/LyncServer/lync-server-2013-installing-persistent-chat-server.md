# <a name="installing-persistent-chat-server-in-lync-server-2013"></a>Установка сервера сохраняемого чата в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-01_

Установка сервера сохраняемого чата встроена в программу установки Lync Server 2013 и использует тот же загрузчик, который Lync Server 2013 использует для установки.
