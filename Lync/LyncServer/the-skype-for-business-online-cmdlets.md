# <a name="the-skype-for-business-online-cmdlets"></a>Командлеты Lync Online

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2013-07-05_

Когда вы подключаетесь к Skype для бизнеса Online с помощью Windows PowerShell, набор командлетов Skype для бизнеса Online копируется в память компьютера. Эти командлеты наряду с любыми другими командлетами, уже установленными на локальном компьютере (в том числе с помощью основных командлетов, установленных при установке Windows PowerShell), можно использовать для управления развертыванием Skype для бизнеса Online и Skype для Учетные записи пользователей из бизнес-сети. Командлеты Skype для бизнеса Online представлены в следующих разделах:

  - [Управление клиентами Lync Online](https://docs.microsoft.com/skypeforbusiness/set-up-your-computer-for-windows-powershell/manage-skype-for-business-online-organizations)

  - [Управление пользователями и свойствами учетной записи пользователя в Skype для бизнеса Online](https://docs.microsoft.com/skypeforbusiness/manage/user-accounts/user-accounts)

  - [Управление политиками в Skype для бизнеса Online](https://docs.microsoft.com/office365/enterprise/powershell/manage-skype-for-business-online-policies-with-office-365-powershell)

  - [Управление клиентом Skype для бизнеса в Skype для бизнеса Online](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/deploy-the-skype-for-business-client-in-office-365)

  - [Управление единой системой обмена сообщениями Exchange и размещенной голосовой почтой в Skype для бизнеса Online](https://docs.microsoft.com/skypeforbusiness/set-up-your-computer-for-windows-powershell/manage-exchange-unified-messaging-and-hosted-voicemail)

  - [Управление связью в Skype для бизнеса Online с внешними пользователями и организациями](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)

  - [Управление встречами и конференциями Lync Online](https://docs.microsoft.com/skypeforbusiness/manage/conferencing/conferencing-policies)

  - [Управление сотовыми телефонами и мобильными устройствами в Skype для бизнеса Online](https://docs.microsoft.com/skypeforbusiness/set-up-policies-in-your-organization/set-up-mobile-policies-for-your-organization)
