# <a name="setting-who-can-post-messages-in-an-auditorium-chat-room-in-lync-server-2013"></a>Настройка пользователей, которые могут отправлять сообщения в комнату чата аудитория в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-06_

В чате аудитории сообщения могут отправлять только те пользователи, которым назначена роль выступающего. Все остальные участники могут только читать сообщения. Выступающие в чате аудитории должны быть участниками чата.

Дополнительные сведения об использовании интерфейса командной строки Windows PowerShell для управления комнатами чата аудитория см в разделе [Manage комнаты](manage-rooms.md) в документации по развертыванию.

Несмотря на то, что администраторы комнаты сохраняемого чата и руководители комнаты чата могут управлять параметрами комнаты чата, они не могут отправлять сообщения в комнату аудитория, если они не являются **докладчиками**.
