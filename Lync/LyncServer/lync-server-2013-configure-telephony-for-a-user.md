# Настройка телефонии для пользователя

 

_**Дата изменения раздела:** 2012-11-01_

Параметры телефонии — это некоторые из тех индивидуальных параметров учетной записи пользователя, которые можно настроить в управления Lync Server для пользователя (в том случае, если для Lync Server 2013 включена поддержка индивидуальных пользователей и организация поддерживает телефонию).

Параметры телефонии пользователей Lync Server включают следующие:

  - **Аудио/видео отключено**   Пользователь не может выполнять вызовы с использованием функций аудио- и видеосвязи.

  - **Только с ПК на ПК**   Пользователь может выполнять голосовые вызовы и видеовызовы только с компьютера на компьютер.

  - **Корпоративная голосовая связи**   Пользователь может использовать инфраструктуру Lync Server 2013 для маршрутизации всех входящих и исходящих вызовов. Пользователь также может выполнять вызовы с компьютера на компьютер.

  - **Удаленное управление вызовами**   Пользователь может использовать Lync Server 2013 для управления настольным телефоном, а также выполнять вызовы с компьютера на компьютер.

Для получения более подробных сведений о настройке телефонии для организации см. раздел [Настройка телефонии для пользователя](lync-server-2013-configure-telephony-for-a-user.md) и [Развертывание корпоративной голосовой связи в Lync Server 2013](lync-server-2013-deploying-enterprise-voice.md) в документации по развертыванию.

## Настройка телефонии для конкретной учетной записи пользователя

1.  Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес для администрирования, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Lync Server см. в разделе [Открытие средств администрирования Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  На левой панели навигации щелкните **Пользователи**.

4.  В поле **Поиск пользователей** введите отображаемое имя (полностью или первую его часть), имя, фамилию, имя учетной записи SAM (Security Accounts Manager — диспетчер учетных записей безопасности), SIP-адрес или линейный идентификатор URI (Uniform Resource Identifier — универсальный код ресурса) учетной записи пользователя, которая требуется, а затем щелкните **Найти**.

5.  В таблице щелкните учетную запись пользователя, которую необходимо изменить.

6.  В меню **Правка** щелкните **Изменить**.

7.  В разделе **Телефония** выполните следующие действия:
    
      - Чтобы отключить аудио- и видеовызовы для пользователя, щелкните **Аудио/видео отключено**.
    
      - Чтобы включить аудиосвязь с компьютера на компьютер для пользователя, но при этом не включать удаленное управление вызовами или корпоративную голосовую связь, щелкните **Только с ПК на ПК**. Укажите значение для идентификатора **Линейный URI** для телефона, который пользователь использует для выполнения аудиовызовов с компьютера на компьютер.
    
      - Чтобы перенаправлять телефонные вызовы пользователя с использованием инфраструктуры Lync Server 2010 в соответствии с классом политики службы, включая аудиосвязь с компьютера на компьютер, щелкните **Корпоративная голосовая связь**. В поле **Линейный URI** укажите номер телефона для корпоративной голосовой связи. В полях **Политика абонентских групп** и **Политика голосовой связи** укажите соответствующие политики для пользователя. Чтобы указать правила нормализации для преобразования номеров телефона, набираемых пользователем, в формат E.164, выберите соответствующий профиль местоположения в разделе **Политика определения местоположения**.
    
      - Чтобы активировать функцию удаленного управления вызовами, которая позволяет пользователям управлять линией настольного телефона с помощью Lync Server 2013 для выполнения вызовов с компьютера на компьютер и с компьютера на телефон щелкните **Удаленное управление вызовами**. В поле **Линейный URI** укажите номер телефона для удаленного управления вызовами. Для маршрутизации вызовов пользователю требуется настольный телефон и подключение к системе PBX.
