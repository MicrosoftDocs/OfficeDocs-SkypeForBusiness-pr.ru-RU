# <a name="moving-users-from-lync-online-to-lync-on-premises-in-lync-server-2013"></a>Перемещение пользователей из Lync Online в локальную среду Lync в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-05-08_

Выполнила ли ваша организация развертывание Lync Online перед развертыванием локальной среды Lync Server? Теперь вы хотите переместить учетные записи пользователей, изначально включенные для Lync в Lync Online, в локальное развертывание Lync Server? Это можно сделать с помощью этого раздела.

<div>


> [!NOTE]  
> Этот раздел предназначен только для перемещения пользователей, изначально включенных для Lync в Lync Online. Возможно, у вас есть учетные записи пользователей, для которых предварительно включен локальный доступ к Lync, а затем они перемещены в Lync Online? Если да, и вы хотите переместить их обратно в локальную среду, процесс будет простым: <A href="lync-server-2013-administering-users-in-a-hybrid-deployment.md">Администрирование пользователей в гибридном развертывании Lync Server 2013</A>.
