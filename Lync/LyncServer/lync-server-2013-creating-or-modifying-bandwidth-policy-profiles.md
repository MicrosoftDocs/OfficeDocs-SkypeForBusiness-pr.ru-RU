# <a name="creating-or-modifying-bandwidth-policy-profiles-in-lync-server-2013"></a>Создание и изменение профилей политики пропускной способности в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-15_

В рамках управления допуском звонков (CAC) для определения ограничений пропускной способности для некоторых модальностей используется политика пропускной способности. В Microsoft Lync Server 2013 можно назначать ограничения пропускной способности только для модальностей аудио и видео. Вы можете настроить общие ограничения пропускной способности и ограничения сеанса. С помощью панели управления Lync Server вы можете создавать, изменять и удалять профили контейнеров для этих политик. Каждый профиль политики пропускной способности может быть связан с одним или несколькими сетевыми сайтами. Чтобы создать или изменить профиль политики пропускной способности, выполните указанные ниже действия. Чтобы удалить профиль политики пропускной способности, ознакомьтесь со сведениями [Удаление профилей политики пропускной способности сети в Lync Server 2013](lync-server-2013-deleting-network-bandwidth-policy-profiles.md)

<div>

## <a name="to-create-a-new-bandwidth-policy-profile"></a>Создание нового профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных способах, которые можно использовать для запуска панели управления Lync Server, приведены в разделе [Открытие меню администрирования Lync server 2013](lync-server-2013-open-lync-server-administrative-tools.md).

3.  На панели навигации слева выберите пункт **Настройка сети** , а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** нажмите кнопку **создать**.

5.  В окне **новый профиль политики пропускной способности**введите имя в поле **имя** . Это имя должно быть уникальным среди всех профилей политики пропускной способности.

6.  В поле **предельное число звуков** введите числовое значение. Это значение — это максимальный объем пропускной способности, выделяемый для всех звуковых подключений, выраженный в кбит/с.

7.  Введите числовое значение в поле " **Ограничение сеанса голосовой связи** ". Это значение — это максимальный объем пропускной способности, выделяемый для отдельного звукового подключения, выраженный в кбит/с. Это значение должно быть 40 или выше.

8.  Введите числовое значение в поле "предельный **Размер видео** ". Это значение — это максимальный объем пропускной способности, выделяемый для всех видеоподключений, выраженный в кбит/с.

9.  Введите числовое значение в поле " **Ограничение сеанса видео** ". Это значение — это максимальный объем пропускной способности, выделяемый для отдельного видеосоединения, выраженный в КБ/с. Это значение должно быть 100 или выше.

10. Необязательно Введите значение в поле **Описание** , чтобы получить дополнительные сведения об этом профиле политики пропускной способности, которое не может быть выражено только именем.

11. Нажмите **Исполнить**.
    
    <div>
    

    > [!NOTE]  
    > Создание профиля политики пропускной способности не приводит к автоматическому применению ограничений для пропускной способности. Сначала необходимо связать профиль политики с сайтом. Сведения о том, как связать профиль политики с сайтом, можно найти <A href="lync-server-2013-creating-or-modifying-network-sites.md">в разделе Создание или изменение сетевых сайтов в Lync Server 2013</A>.

    
    </div>

## <a name="to-modify-a-bandwidth-policy-profile"></a>Изменение профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных способах, которые можно использовать для запуска панели управления Lync Server, приведены в разделе [Открытие меню администрирования Lync server 2013](lync-server-2013-open-lync-server-administrative-tools.md).

3.  На панели навигации слева выберите пункт **Настройка сети** , а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** выберите профиль политики пропускной способности, который вы хотите изменить.

5.  В меню **Правка** щелкните **Подробнее**.

6.  На странице **профиля политики изменения пропускной способности** измените соответствующие поля (Дополнительные сведения можно найти в разделе "Создание профиля политики пропускной способности" в этой статье выше).

7.  Нажмите **Исполнить**.
    
    <div>
    

    > [!NOTE]  
    > При изменении профиля политики пропускной способности немедленно обновляются ограничения пропускной способности всех сайтов сети, связанных с этим профилем политики пропускной способности.
