# Настройка маршрутизации на основе расположения в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

В накопительном обновлении 1 (CU1) для сервера Lync Server 2013 маршрутизация на основе расположения является функцией корпоративной голосовой связи. Маршрутизация на основе расположения – функция управления вызовами, которая управляет маршрутизацией вызовов с помощью Lync Server 2013 CU1. Она ставит ограничения на маршрутизацию звонков в пункты назначения PBX или PSTN, в зависимости от расположения абонента Lync. Маршрутизация на основе расположения применяет правила авторизации звонка к звонкам PSTN с учетом сетевого расположения абонента. Расположение вызывающего абонента определяется с учетом сетевого сайта, связанного с подсетью, к которой подключен вызывающий абонент. Для настройки маршрутизации на основе расположения требуется развертывание корпоративной голосовой связи, а затем настройка сетевых регионов, сайтов и подсетей. Это создает основу для включения маршрутизации на основе расположения.

Перед развертыванием маршрутизации на основе расположения сначала нужно развернуть корпоративную голосовую связь, а затем настроить сетевые регионы, сайты и связать подсети с сетевыми сайтами. Затем можно настроить маршрутизацию на основе расположения. Подробные сведения о том, как настроить регионы сети, сайты и подсети см. в статье [Развертывание расширенных функций корпоративной голосовой связи в Lync Server 2013](lync-server-2013-deploying-advanced-enterprise-voice-features.md)
