# <a name="whats-new-for-devices-in-lync-server-2013"></a>Новые возможности для устройств в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

_**Последнее изменение темы:** 2012-06-22_

Lync Server 2013 включает Lync Phone Edition, программное обеспечение, которое работает на соответствующих устройствах и предоставляет традиционные и расширенные функции телефонии, встроенную безопасность, управляемость и многое другое. Lync Phone Edition работает так же, как и Lync Server 2013, как и в случае Lync Server 2010. Дополнительные сведения о новейших возможностях, связанных с устройствами, можно найти в статье [новые возможности для устройств](https://go.microsoft.com/fwlink/p/?linkid=256490) в библиотеке TechNet для Lync Server 2010.
