# <a name="whats-new-for-devices-in-lync-server-2013"></a>Новые возможности для устройств в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-06-22_

Lync Server 2013 включает Lync Phone Edition, программное обеспечение, которое работает на соответствующих устройствах и предоставляет традиционные и расширенные возможности телефонии, встроенную безопасность, управляемость и многое другое. Lync Phone Edition работает аналогично с Lync Server 2013, как это происходит в Lync Server 2010. Подробные сведения о новых возможностях, связанных с устройствами, вы можете найти в статьях [новые возможности для устройств](http://go.microsoft.com/fwlink/p/?linkid=256490) в библиотеке TechNet 2010 для Lync Server.
