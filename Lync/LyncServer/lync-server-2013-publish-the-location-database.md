# Публикация базы данных местоположений в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-30_

Новые расположения, добавленные вами в базу данных местоположений, останутся недоступны клиенту, пока не будут опубликованы.

Дополнительные сведения см. в описании следующего командлета в документации по командной консоли Командная консоль Lync Server

  - **Publish-CsLisConfiguration**

Если вы используете шлюзы Emergency Location Identification Number (ELIN), необходимо также загрузить номера ELIN в базу данных Automatic Location Identification (ALI) сети PSTN. Оператор PSTN может потребовать использования определенного формата для записей ELIN. Обратитесь к оператору PSTN для получения более подробных сведений. Можно экспортировать базу данных информирования о местонахождении и отформатировать ее необходимым образом.

## Публикация базы данных местоположений

  - Запустите командную консоль Lync Server: нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы** и **Microsoft Lync Server 2013** и щелкните элемент **Командная консоль Lync Server**.

  - Чтобы опубликовать базу данных местоположений, выполните следующий командлет.
    
        Publish-CsLisConfiguration
