# <a name="deploying-devices-in-lync-server-2013"></a>Развертывание устройств в Lync Server 2013

_**Последнее изменение темы:** 2013-09-10_

Lync Server 2013 включает Lync Phone Edition, программное обеспечение, которое работает на соответствующих устройствах и предоставляет традиционные и расширенные функции телефонии, встроенную безопасность, управляемость и многое другое. Lync Phone Edition работает так же, как и Lync Server 2013, как и в случае Lync Server 2010. Более подробную информацию о развертывании устройств можно узнать в статье [deploy Lync Phone Edition](https://go.microsoft.com/fwlink/p/?linkid=285880) в библиотеке TechNet для lync Server 2010.
