# Сводный отчет по качеству среды в Lync Server 2013

 

_**Дата изменения раздела:** 2016-06-29_

Сводный отчет по качеству мультимедиа, возможно, является лучшим предложением для анализа качества звонков в организации: он предоставляет подробные метрики качества взаимодействия при вызовах, распределенные по следующим категориям:

  - одноранговые вызовы в объединенных коммуникациях (например, вызов из Microsoft Lync 2013 в Microsoft Lync 2013);

  - Сеансы конференц-связи по объединенным коммуникациям

  - Сеансы конференц-связи по ТСОП

  - Звонки по ТСОП: обходной канал передачи медиаданных

  - Звонки по ТСОП (без обхода): ветвь объединенных коммуникаций

  - Звонки по ТСОП (без обхода): ветвь шлюза

  - Другие типы звонков

При первом открытии отчета отображается сводка по каждой из этих категорий. Не закрывая отчет, можно развернуть каждую категорию, чтобы просмотреть подкатегории, например, вызовы, сделанные из Office Communicator 2007 R2 в Lync 2013. Затем можно также детализировать эти подкатегории, чтобы просмотреть сведения о каждом вызове, сделанном в этой подкатегории.

В Microsoft Lync Server 2013 сводный отчет по качеству мультимедиа дополнительно разбивает данные на три типа вызовов: голосовые звонки, видеозвонки и звонки с общим доступом к приложениям. Для каждого типа вызова в отчете имеется свой собственный раздел и свой собственный ряд метрик вызова.

Кроме того, сводный отчет по качеству мультимедиа позволяет применять фильтры, чтобы сравнивать качество проводных и беспроводных вызовов, внутренних и внешних вызовов, а также вызовов с использованием VPN и вызовов без VPN.

## Доступ к сводному отчету по качеству мультимедиа

Сводный отчет по качеству мультимедиа можно вызвать на домашней странице отчетов наблюдения. Можно перейти в [Отчет по списку звонков в Lync Server 2013](lync-server-2013-call-list-report.md), щелкнув какую-либо из следующих метрик:

  - Громкость вызова

  - Процент звонков низкого качества

Кроме того, можно вызвать отчет по распределению метрик качества мультимедиа, щелкнув какую-либо из метрик звонка:

  - Время кругового пути (мс)

  - Деградация (MOS)

  - Потеря пакетов

  - Колебание (мс)

  - Healer concealed ratio (Коэффициент скрытых сэмплов)

  - Healer stretched ratio (Коэффициент растянутых сэмплов)

  - Healer compressed ratio (Коэффициент сжатых сэмплов)

## Фильтры

Фильтры позволяют получать более адресный набор данных или просматривать полученные данные разными способами. Например, в сводном отчете по качеству мультимедиа можно фильтровать данные по типу доступа (т.е. по внутреннему или внешнему доступу), а также по проводному или беспроводному типу подключения. Кроме того, можно выбрать способ группирования данных. В данном случае вызовы группируются по часу, дню, неделе или месяцу.
