# Настройка хранилища файлов для Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Lync Server 2013поддерживает использование общих папок в распределенной файловой системе (DFS). Подробные сведения о распределенной файловой системе для Windows Server 2008 см. в пошаговом руководстве по распределенным файловым системам в Windows Server 2008 по адресу [http://go.microsoft.com/fwlink/p/?linkId=202835](http://go.microsoft.com/fwlink/p/?linkid=202835). Для использования распределенной файловой системы в Lync Server 2013 должны выполняться следующие требования.

  - Пространства имен являются доменными.

  - Все серверы пространств имен работают под управлением Windows 2008 или более поздней версии.

Для установки Lync Server 2013 требуется, чтобы разрешения для общей папки обеспечивали полный доступ администратору. В этом случае Lync Server 2013 будет использовать разрешения файлов NTFS для добавления папок в список управления доступом. Унаследованные разрешения для общих папок DFS не будут использоваться для ограничения доступа.

Дополнительные сведения о требованиях для использования общих папок см. в разделе [Поддержка хранения файлов в Lync Server 2013](lync-server-2013-file-storage-support.md) документации по поддержке.

Ниже описывается, как правильно настроить разрешения для общих папок с помощью мастера пространств имен DFS (как описано в руководстве по настройке DFS).

## Настройка разрешений для общих папок

1.  В меню **Пуск** последовательно выберите пункты **Все программы** , **Администрирование** и **Управление DFS** .

2.  В дереве консоли оснастки "Управление DFS" щелкните правой кнопкой мыши сервер пространства имен (например, filesrv1.contoso.com) и выберите пункт **Изменить параметры** .

3.  Выберите пункт **Разрешения для общей папки** .

4.  Выберите пункт **Использовать пользовательские разрешения** .

5.  Для группы администраторов выберите в разделе **Разрешить** следующие пункты:
    
      - **Полный доступ**
    
      - **Изменение**
    
      - **Чтение**

6.  Нажмите кнопку **Применить** , а затем кнопку **ОК** .
