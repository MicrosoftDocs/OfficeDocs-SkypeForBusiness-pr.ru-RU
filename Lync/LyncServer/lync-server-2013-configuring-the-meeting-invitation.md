# Настройка приглашения на собрание в Lync Server 2013

 

_**Дата изменения раздела:** 2013-07-16_

Приглашения на собрания, отправляемые надстройкой надстройка собраний по сети для Lync 2013, можно настраивать, включая в текст приглашения следующие дополнительные элементы.

  - **Эмблема организации.** Эмблему организации можно включить в приглашение на собрание с помощью параметра "Logo URL" ("URL-адрес эмблемы"). если приглашения на собрания будут отправляться людям вне организации, то изображение эмблемы должно располагаться по общедоступному URL-адресу. Поддерживаются форматы изображений GIF и JPG. Хотя сервер Lync Server 2013 сохраняет URL-адрес без ограничений размера изображения, для лучшего результата рекомендуется, чтобы максимальный размер изображения был 30 пикселей по высоте и 188 пикселей по ширине.

  - **Ссылка на специальную справку или на службу поддержки.** Добавьте URL-адрес справки или веб-сайта службы поддержки вашей организации. Если приглашения на собрания будут отправляться людям вне организации, то этот URL-адрес должен быть общедоступным. Максимальная длина URL-адреса — 1 КБ.

  - **Текст заявления об отказе от ответственности.** Добавьте URL-адрес юридического заявления или заявления об отказе от ответственности, которое будет отображаться во всех приглашениях на собрания. Если приглашения на собрания будут отправляться людям вне организации, то этот URL-адрес должен быть общедоступным. Максимальная длина URL-адреса — 1 КБ.

  - **Текст нижнего колонтитула.** Добавьте текст, который будет отображаться в приглашении как нижний колонтитул. Максимальная длина этого текста — 2 КБ.

Эти параметры можно настраивать с помощью панели управления Lync Server или командной консоли Командная консоль Lync Server.


**Настройка приглашения на собрание с помощью панели управления Lync Server**

1.  Войдите на любой компьютер, подключенный к сети, где развернут Lync Server 2013, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsServerAdministrator или CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес для администрирования, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Lync Server см. в разделе [Открытие средств администрирования Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  В левой панели навигации последовательно нажмите пункты **Conferencing (Конференц-связь)** и **Meeting Configuration (Конфигурация собрания)**.

4.  На странице **конфигурации собрания** нажмите **New (Создать)**, а затем выполните одно из следующих действий.
    
      - Чтобы создать политику на уровне узла, выберите пункт **Site configuration (Конфигурация узла)**. В поле поиска **Select a Site (Выбор узла)** введите имя (или часть имени) узла, для которого требуется определить параметры присоединения к собранию. В появившемся списке узлов выберите нужный и нажмите кнопку **ОК**.
    
      - Чтобы создать политику на уровне пула, выберите пункт **Pool configuration (Конфигурация пула)**. В поле поиска **Select a Service (Выбор службы)** введите имя (или его часть) службы пула, для которой требуется определить параметры присоединения к собранию. В появившемся списке служб выберите нужный пул и нажмите кнопку **ОК**.

5.  Выполните любые из следующих действий.
    
      - В поле **Logo URL (URL-адрес эмблемы)** введите URL-адрес изображения эмблемы вашей организации.
    
      - В поле **Help URL (URL-адрес справки)** введите URL-адрес сайта справки или службы поддержки вашей организации.
    
      - В поле **Legal text (Юридическое заявление)** введите URL-адрес юридического заявления или заявления об отказе от ответственности, которое требуется включить в приглашения на собрания.
    
      - В поле **Custom footer text (Текст нижнего колонтитула)** введите текст нижнего колонтитула объемом до 2 КБ.

**Настройка приглашения на собрание с помощью командной консоли Командная консоль Lync Server**

1.  Запустите командную консоль Lync Server: нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы** и **Microsoft Lync Server 2013** и щелкните элемент **Командная консоль Lync Server**.

2.  Выполните командлет **New-CsMeetingConfiguration** или **Set-CsMeetingConfiguration**, чтобы создать или настроить параметры приглашения на собрание. Например, выполните следующую команду:
    
        New-CsMeetingConfiguration -Identity site:Redmond -EnableInviteCustomization $True -LogoURL "http://www.contoso.com/logo/contosobanner.gif" -HelpURL "http://www.contoso.com/support" -LegalURL "http://www.contoso.com/disclaimer" -CustomFooterText "Communications may be monitored or recorded."
