## <a name="interoperability-among-clients"></a>Взаимодействие между клиентами

Благодаря выпуску Lync Server 2013 различные клиентские версии могут беспрепятственно взаимодействовать в одноранговых сценариях и конференц-связи. Этот раздел содержит сведения о доступности функций при взаимодействии пользователей с другими пользователями, использующими разные версии клиентов и серверов.

<div>

## <a name="peer-to-peer-feature-support"></a>Поддержка одноранговых функций

Одноранговые функции поддерживаются для пользователей, которые находятся в разных версиях сервера и используют разные клиентские версии. Взаимодействие с конечным пользователем и доступные возможности соответствуют возможностям клиента пользователя и версии сервера, на котором пользователь вошел в систему. Другими словами:

  - Если пользователь вошел в Lync Server 2013 с более ранней версией клиента, он будет использовать тот же интерфейс, что и для этого пользователя. Новые функции, представленные в Lync Server 2013, будут недоступны до тех пор, пока клиент пользователя не будет обновлен. Примеры: Просмотр коллекции видео, видео в формате HD, обновленный общий доступ к PowerPoint, а также возможность отключения звука и видео всех участников во время ввода собрания. Новые функции, описанные в разделе [новые возможности конференции в Lync server 2013](lync-server-2013-new-conferencing-features.md) , и [новые возможности для клиентов в Lync Server 2013](lync-server-2013-what-s-new-for-clients.md).

  - Если пользователь вошел в Lync Server 2010 с помощью клиента Lync 2013, все новые возможности, не поддерживаемые Lync Server 2010, будут недоступны до тех пор, пока пользователь не будет перенесен на Lync Server 2013.

В следующей таблице сравниваются возможности, доступные в одноранговых сеансах, в которых клиент вошел на сервер Lync Server 2013 или Lync Server 2010.

## <a name="conferencing-feature-support-for-lync-2013-clients-in-lync-server-2010-meetings"></a>Поддержка функций конференций для клиентов Lync 2013 в Lync Server 2010 для собраний

  - Если щелкнуть пользователя правой кнопкой мыши в списке участников, вы не **** **сможете получить** доступ к параметрам управления участниками коллекции.

При переходе с Office Communications Server 2007 R2 подходящими для обеспечения взаимодействия и обмена мгновенными сообщениями является Office Communicator 2007 R2, но пользователи должны использовать Lync Web App 2013, чтобы присоединиться к собраниям Lync Server 2013.

## <a name="scheduling-add-in-support"></a>Поддержка планирования надстроек

Поддержка различных надстроек для планирования в соответствии с серверной и клиентской версиями обеспечивается совместимостью. Как правило, в Lync Server 2013 поддерживаются следующие надстройки для планирования. Несмотря на то, что предыдущие версии надстроек не предоставляют новые функции надстройки Lync 2013, такие как возможность отключить Микрофоны всех участников и звуков видео при вводе собрания.

  - **Надстройка "собрание по сети" для Lync 2013**   предоставляет те же возможности, что и надстройка "собрание по сети" для Lync 2010, при условии, что участники собрания организаторов запланировать конференции, в которых микрофон участника и видео отключены. по умолчанию. Администраторы также могут настроить приглашения на собрание Организации, добавив собственный логотип, URL-адрес службы поддержки, URL-адрес юридического лица или настраиваемый текст нижнего колонтитула.

  - **Надстройка "собрание по сети" для Lync 2010**   — Планирование собраний Lync и удаление возможности планирования конференций Office Live Meeting.

  - **Надстройка Office Communicator 2007 R2**   обеспечивает планирование как для конференций Office Live Meeting, так и для конференций Office Communicator 2007 R2.

## <a name="support-for-joining-meetings"></a>Поддержка присоединения к собраниям

Все клиенты, поддерживаемые Lync Server 2013, могут присоединиться к собраниям Lync 2013. Поскольку Lync Web App — это веб-компонент сервера, то в тех случаях, когда Lync Web App используется для присоединения к собранию Lync Server 2013, всегда используется новая версия Lync Web App.

Пользователи Lync 2013 могут присоединяться к собраниям, размещенным на Lync 2010 и Office Communications Server 2007 R2, с масштабированием функций. Возможности, доступные в собрании, ограничиваются версией сервера, на котором размещается собрание.
