# Регистрация вызовов (CDR) в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-22_

Записи регистрации вызовов (CDR) и диагностическая информация об одноранговой активности, включая мгновенные сообщения, VoIP-вызовы, общий доступ к приложениям, передачу файлов и собрания. Данные об использовании могут применяться для расчета окупаемости инвестиций, а данные диагностики — для устранения неполадок, связанных с одноранговой активностью и собраниями. При установке Lync Server 2013 также устанавливается предварительно определенная коллекция глобальных параметров конфигурации для регистрации вызовов. Используйте темы этого раздела для настройки регистрации вызовов.
