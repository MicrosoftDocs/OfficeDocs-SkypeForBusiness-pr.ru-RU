# <a name="call-detail-recording-cdr-in-lync-server-2013"></a>Запись сведений о звонке (CDR) в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-22_

Записи регистрации вызовов (CDR) и диагностическая информация об одноранговой активности, включая мгновенные сообщения, VoIP-вызовы, общий доступ к приложениям, передачу файлов и собрания. Данные об использовании могут применяться для расчета окупаемости инвестиций, а данные диагностики — для устранения неполадок, связанных с одноранговой активностью и собраниями. При установке Lync Server 2013 Вы также устанавливаете предопределенную коллекцию глобальных параметров конфигурации для CDR. Используйте темы этого раздела для настройки регистрации вызовов.
