# <a name="call-park-application-cmdlets-in-lync-server-2013"></a>Командлеты приложения парковки вызовов в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-03-21_

Приложение парковки вызовов позволяет пользователю помещать вызов на удержание, а затем получать этот вызов с другого телефона. Используйте эти командлеты для настройки параметров для орбиты парковки вызовов и приложения парковки вызовов.
