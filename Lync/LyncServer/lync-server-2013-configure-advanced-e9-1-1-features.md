# <a name="configure-advanced-e9-1-1-features-in-lync-server-2013"></a>Настройка расширенных возможностей E9-1-1 в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-06-06_

Lync Server 2013 включает следующие функции, которые можно использовать для настройки развертывания E9-1-1:

  - Интерфейс веб-службы для подключения сервера сведений о расположении к приложению SNMP.

  - Интерфейс веб-службы для подключения к источнику данных в дополнительном расположении.
