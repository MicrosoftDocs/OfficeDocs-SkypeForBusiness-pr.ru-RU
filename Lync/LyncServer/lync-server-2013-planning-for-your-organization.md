# <a name="organization-planning-for-lync-server-2013"></a>Планирование организации для Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-24_

В подразделах этого раздела приводятся сведения, которые помогут вам приступить к планированию развертывания Lync Server.

<div>

## <a name="in-this-section"></a>Содержание

  - [Решение о развертывании Lync server 2013](lync-server-2013-deciding-how-to-deploy-microsoft-lync.md) содержит рекомендации по выбору между этими основными сценариями развертывания lync Server 2013.

  - [Начало процесса планирования для Lync Server 2013](lync-server-2013-beginning-the-planning-process.md) поможет вам разобраться, как приступить к планированию локального развертывания и как документация по планированию работает с построителем топологий.

  - [Основы топологии, которые необходимо знать перед планированием для Lync server 2013](lync-server-2013-topology-basics-you-must-know-before-planning.md) описывает основные принципы топологий Lync Server, в том числе сайты, пулы серверов и топологии, поддерживающие высокую доступность и аварийное восстановление.

  - [Начальные решения по планированию для Lync Server 2013](lync-server-2013-initial-planning-decisions.md) просматривает вопросы, которые необходимо ответить, чтобы решить, какие рабочие нагрузки и функции Lync Server развертывают.

  - [Клиенты для Lync Server 2013](lync-server-2013-clients.md) описывают различные типы клиентского программного обеспечения, которые можно развернуть для пользователей организации, в том числе клиентское программное обеспечение, веб-клиенты и мобильные устройства.

  - [В эталонных топологиях в Lync Server 2013](lync-server-2013-reference-topologies.md) показаны три примера топологии, иллюстрирующие хорошую конструкцию топологии в трех типовых организациях, а также объясняется причина многих решений при проектировании этих топологий.
