# Компоненты VoIP для сети периметра для Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-21_

Внешние абоненты, которые используют унифицированные коммуникационные клиенты для выполнения индивидуальных вызовов или конференц-вызовов, могут применять сервер для голосовой связи с коллегами.

На сервер пограничная служба доступа предоставляет сигналы SIP для вызовов, выполняемых пользователями Lync, которые находятся за пределами брандмауэра организации. Служба A/V Edge обеспечивает обход NAT и брандмауэров. Абонент, использующий унифицированный коммуникационный клиент (UC) за пределами корпоративного брандмауэра, применяет службу для выполнения индивидуальных вызовов или конференц-вызовов.

Служба проверки подлинности аудио и видео связана с пограничной службой аудио- и видеоданных и предоставляет услуги проверки подлинности. Внешним пользователям, которые пытаются подключиться к пограничной службе аудио- и видеоданных, требуется маркер проверки подлинности, предоставляемый службой проверки подлинности аудио и видео, для прохождения вызовов.
