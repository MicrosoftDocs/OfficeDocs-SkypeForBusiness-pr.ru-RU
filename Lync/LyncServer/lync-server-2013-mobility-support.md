# <a name="mobility-support-in-lync-server-2013"></a>Поддержка мобильных устройств в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-06-29_

Благодаря Lync Server 2013 Mobility пользователи имеют доступ к функциям Lync 2013 на своих мобильных устройствах. К поддерживаемым мобильным устройствам относятся следующие:

  - Поддерживаемые устройства Apple iOS

  - Windows Phone

  - Android

  - Nokia

Для получения дополнительных сведений о функциях и требованиях для мобильных устройств, ознакомьтесь со статьей [планирование мобильных устройств в Lync Server 2013](lync-server-2013-planning-for-mobility.md) в документации по планированию.
