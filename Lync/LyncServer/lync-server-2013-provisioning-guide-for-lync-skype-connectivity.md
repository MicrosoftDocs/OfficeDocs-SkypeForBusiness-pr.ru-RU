# <a name="provisioning-guide-for-lync-skype-connectivity-in-lync-server-2013"></a>Руководство по подготовке для подключения Lync — Skype в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-11-26_

Lync Server 2013 поддерживает подключение к Skype. Благодаря этому пользователи Lync 2013 могут добавлять контакты Skype с помощью учетной записи Майкрософт пользователя Skype (MSA). Клиенты Skype также могут добавлять пользователей Lync в свои списки контактов. В соответствии с политиками, настроенными администратором в Lync Server, пользователи Lync и Skype могут обмениваться мгновенными сообщениями, просматривать сведения о присутствии друг друга и запускать голосовые и видеовызовы. Lync Online Connectivity — это функция Lync Online, которую можно включить для пользователей Lync Online в центре администрирования Lync в центре администрирования Microsoft 365.

<div>

> [!IMPORTANT]  
> Если Lync Server уже настроен для подключения к Windows Messenger с использованием общедоступной службы обмена мгновенными сообщениями (PIC), то развертывание уже настроено для подключения Lync-Skype. Единственное изменение, которое вы можете подумать — переименовать существующую запись Messenger PIC как Skype. Дополнительные сведения: Настройка параметров поставщика Skype PIC для Lync позже в этом руководстве.
