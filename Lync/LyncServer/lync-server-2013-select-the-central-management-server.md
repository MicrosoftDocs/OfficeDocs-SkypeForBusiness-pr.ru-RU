# <a name="select-the-central-management-server-in-lync-server-2013"></a>Выбор центрального сервера управления в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-01-02_

Прежде чем можно будет определить и настроить топологию, необходимо сначала определить расположение для установки центрального сервера управления.

<div>


> [!NOTE]  
> Это не вступит в силу, пока топология не будет опубликована в построителе топологии. Чтобы настроить Центральный сервер управления до создания и публикации топологии, выполните <STRONG>Set-ксконфигуратионсторелокатион</STRONG>.



</div>

<div>

## <a name="to-select-the-central-management-server"></a>Выбор центрального сервера управления

1.  Запустить построитель топологии: нажмите кнопку **Пуск**, выберите пункт **все программы**, а затем — **Microsoft Lync Server 2013**и нажмите кнопку **Построитель топологии Lync Server**.

2.  Щелкните правой кнопкой мыши узел Lync Server 2013 и выберите команду **изменить свойства**.

3.  На панели центрального сервера управления выберите сервер переднего плана, на котором нужно установить центральный сервер управления, и нажмите кнопку **ОК**.
