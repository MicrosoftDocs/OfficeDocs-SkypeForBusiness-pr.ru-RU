# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Удаление устаревшего авторизованного узла в Lync Server 2013 (необязательно)

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-05_

При переносе удаленного управления звонками в развертывание Lync Server необходимо удалить устаревшие авторизованные записи узла (называемые *записями доверенных приложений* в Lync Server) для всех шлюзов SIP/CSTA в устаревшем развертывании. Для удаления записей авторизованных узлов из этих развертываний необходимо использовать соответствующие средства администрирования.
