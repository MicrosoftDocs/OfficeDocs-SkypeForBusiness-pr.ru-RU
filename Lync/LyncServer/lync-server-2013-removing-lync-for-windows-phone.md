# <a name="removing-lync-for-windows-phone-in-lync-server-2013"></a>Удаление Lync для Windows Phone в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2013-02-19_

Чтобы удалить приложение Lync 2013 для Windows Phone с мобильного устройства, выполните указанные ниже действия.

1.  На мобильном устройстве на начальном экране проведите пальцем, чтобы увидеть список приложений.

2.  Коснитесь приложения Lync и удерживайте его, а затем выберите **Удалить**.
