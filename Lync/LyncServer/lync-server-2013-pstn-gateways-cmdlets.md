# <a name="pstn-gateways-cmdlets-in-lync-server-2013"></a>Командлеты PSTN Gateway в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-03-21_

Шлюзы PSTN позволяют вашим корпоративным пользователям звонить и принимать телефонные звонки от пользователей сети PSTN (то есть с помощью общедоступной телефонной сети с коммутируемым подключением). Эти шлюзы играют роль моста между сервером-посредником и сетью PSTN.

<div>

## <a name="pstn-gateways-cmdlets"></a>Командлеты PSTN Gateway

Командлеты [Test-кспстнаутбаундкалл](https://technet.microsoft.com/en-us/library/Gg398207(v=OCS.15)) и [Test-кспстнпиртопиркалл](https://technet.microsoft.com/en-us/library/Gg398662(v=OCS.15)) позволяют удостовериться, что пользователи смогут звонить по сети PSTN.

**Шлюзы ТСОП**

    [Set-CsPstnGateway](https://technet.microsoft.com/en-us/library/Gg398408(v=OCS.15))
