# <a name="installing-the-planning-tool-in-lync-server-2013"></a>Установка средства планирования в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-11-07_

Прежде чем приступить к разработке и планированию инфраструктуры Lync Server 2013 с помощью средства планирования Microsoft Lync Server 2013, Planning Tool, необходимо сначала установить средство планирования. Средство планирования не требуется развертывать на рабочей станции или сервере, входящем в домен или инфраструктуру, в которой планируется установить Lync Server 2013. В файле readme, прилагаемом к средству планирования, описывается важная информация об установке и использовании средства. Некоторые сведения в файле readme повторяются здесь для ясности.

<div>


> [!IMPORTANT]  
> Для работы средства планирования необходимо установить пользователь с правами и разрешениями администратора на компьютере, на котором будет установлен инструмент.



</div>

Для установки и работы средства планирования поддерживаются следующие операционные системы:

  - Windows 8

  - Windows 8.1

  - Windows Server 2012

  - Windows Server 2012 R2

  - Windows 7, 32 — разрядная версия

  - Windows 7, 64-разрядный выпуск с использованием Windows в Win32 (WOW)

  - Windows Server 2008 R2 с использованием WOW

Кроме того, для средства планирования требуется Microsoft .NET Framework 4,5.

После выполнения требований по предварительной установке вы можете установить средство планирования.

<div>

## <a name="to-install-the-planning-tool"></a>Установка средства планирования

1.  Войдите на локальный компьютер как член группы "Администраторы".

2.  С помощью проводника Windows или командного окна откройте каталог, в который вы скачали установочные файлы средства планирования.

3.  Откройте файл Линкпланнингтул. msi. В проводнике Windows дважды щелкните файл. В командной строке введите имя файла, а затем нажмите клавишу **Ввод** , чтобы запустить файл.

4.  На странице приветствия в **Microsoft Lync Server 2013, мастере установки средства планирования**нажмите кнопку **Далее**.

5.  Ознакомьтесь с лицензионным **соглашением**, установите флажок **я принимаю условия лицензионного соглашения** , если вы приняли условия использования в лицензионном соглашении, а затем нажмите кнопку **Далее**.

6.  Выберите место установки файлов средства планирования. Расположение по умолчанию: C\\: Program Files (\\x86) средство планирования\\Microsoft Lync Server 2013. Если вы хотите изменить расположение установки, нажмите кнопку **изменить**. В разделе **Изменение конечной папки**найдите или введите расположение для установки файлов, нажмите кнопку **ОК**, а затем нажмите кнопку **Далее**.

7.  Теперь Установщик готов к установке средства планирования. Нажмите кнопку **установить** , чтобы начать процесс установки.

8.  Начнется установка, и отобразится ход выполнения. После успешного завершения установки нажмите кнопку **Готово**.

9.  Средство планирования готово к использованию.
