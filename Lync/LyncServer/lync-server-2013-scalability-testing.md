# Тестирование масштабируемости

 

_**Дата изменения раздела:** 2012-10-01_

Lync Server 2013 предоставляет серверную инфраструктуру для всех взаимодействий Lync Server в режиме реального времени, включая обмен мгновенными сообщениями и сведениями о присутствии, конференц-связь и корпоративной голосовой связи. Сюда входят все функции, использующие аппаратные ресурсы пула Lync Server 2013 и таким образом влияющие на производительность и масштабирование. Все организации используют функции по-разному.

Например, некоторые организации активно используют видео в конференциях, а другие менее активно или вообще не используют его. Некоторые организации вместо общего доступа к приложениям предпочитают использовать общий доступ к слайдам PowerPoint, а другие — наоборот. Организации, в которых развернута корпоративной голосовой связи, могут активно использовать приложение "Группа ответа" или не использовать его вообще. Большинство организаций разворачивают серверы мониторинга, но не многие из них разворачивают серверы архивации. Кроме того, инфраструктура организаций, возможности оборудования, пропускная способность сети, а также число и размеры развернутых пулов могут отличаться. Разнообразие функций и инфраструктур затрудняет тестирование масштабируемости, поскольку невозможно смоделировать все возможные комбинации функций и инфраструктур.

Для определения уровня масштабируемости Lync Server Майкрософт выполняет тестирование на основе усредненной модели использования (пользовательская модель), в рамках которого все функции Lync Server используются одновременно. Для определения пользовательской модели рабочих нагрузок Lync Server Майкрософт анализирует различные данные, включая опросы клиентов, отзывы, полученные в рамках программы улучшения качества программного обеспечения, данные об использовании Lync Server, полученные от внутреннего ИТ-отдела Майкрософт, а также данные, полученные от службы Live Meeting Service. В большинстве случаев в пользовательской модели заложены максимальные нагрузки, гарантирующие бесперебойную работу в обычных условиях эксплуатации.

В тестах масштабируемости пулы Lync Server 2013 настраиваются в соответствии с рекомендуемыми спецификациями оборудования и программного обеспечения, включая компоненты инфраструктуры, например доменные службы Доменные службы Active Directory, аппаратные балансировщики нагрузки и брандмауэры. При настройке сред Lync Server используются параметры, максимально приближенные к реальным средам. Для имитации нагрузки на Lync Server 2013 (на основании нашей пользовательской модели) используется средство Lync Server 2013 нагрузочного тестирования и проверки производительности.

Для всех тестов масштабируемости выполняется три итерации (включая повторные запуски тестов с интервалом в 3 недели). Полученные результаты тестирования используются для настройки производительности и проверки поддержки масштабируемости в пользовательской модели.
