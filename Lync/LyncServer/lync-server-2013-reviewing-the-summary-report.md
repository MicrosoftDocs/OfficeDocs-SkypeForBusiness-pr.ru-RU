# <a name="reviewing-the-summary-report-in-lync-server-2013"></a>Обзор сводного отчета в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-21_

Отчет администратора Lync Server — это первый из четырех ценных отчетов, которые подробно задокументированы в вашем проекте. Сведения, приведенные в этом отчете, и другие три связанных отчета — отличная документация для ваших команд информационных технологий.

![Общий сводный отчет для администраторов](images/Gg558626.9c529ef7-cb1b-4ce1-a8bc-3ec79aba2377(OCS.15).jpg "Общий сводный отчет для администраторов")

  - Отчет о сертификатах

  - Отчет о брандмауэре

  - Отчет DNS

Сводный отчет содержит общие сведения о конфигурации, связанные с пограничной сетью. Документируется расположение, полное доменное имя и IP-адрес, тип сети и комментарии, относящиеся к данной роли.

Дизайнер и каждая из команд, которые будут развертывать, управлять и обслуживать инфраструктуру, должны просмотреть сводный отчет для точности и убедиться в том, что ошибки являются минимальными.
