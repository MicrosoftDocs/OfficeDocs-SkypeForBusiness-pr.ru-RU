## <a name="sign-in-to-lync-2013-for-the-first-time"></a>Первый вход в Lync 2013

Данные для входа в Lync обычно настраиваются автоматически при установке Lync 2013. Но в первый раз при использовании Lync вам может потребоваться запустить клиент вручную.

**Первый вход в Lync**

1.  Войдите в сеть организации.

2.  Нажмите кнопку **запустить** \> **все программы** \> **, \> Microsoft Lync Lync 2013**.
    
    Вы увидите экран входа в Lync.
    
      - Если поле адреса уже заполнено, проверьте правильность адреса.
    
      - Если это значение неверно или поле пусто, введите адрес для входа в Lync (обычно это то же самое, что и адрес электронной почты).
    
      - Если на экране отображается пустое поле пароля, введите свой пароль.

3.  Выберите **Вход**.

## <a name="sign-in-to-lync-with-a-smart-card"></a>Вход в Lync с помощью смарт-карты

В некоторых организациях теперь для пользователей Lync 2013 используется многоэтапный вход в систему, называемый двухфакторной проверкой подлинности. Если вы предполагаете использовать этот параметр, для входа в Lync вам понадобится "смарт-карта". Смарт-карты бывают двух видов, физических и виртуальных.

  - **Физическая**   информация о размере кредитной карты. При входе вставляется в устройство чтения смарт-карт.

  - **Виртуальная**   не физическая, но электронный идентификатор, который записывается в специальную микросхему вашего компьютера, что на самом сути создает смарт-карту на компьютере. Доступно только для компьютеров с Windows 8, которые содержат микросхему TPM (доверенный платформенный модуль).

<div>

## <a name="enroll-your-smart-card"></a>Регистрация смарт-карты

Перед первым входом по смарт-карте ее необходимо зарегистрировать, то есть сопоставить с ней учетные данные пользователя. Это относится как к физической, так и к виртуальной карте. Этот процесс может быть уже выполнен администратором Lync Server. В случае сомнений обратитесь к нему.

<div>


> [!NOTE]  
> Так как каждая виртуальная смарт-карта связана только с устройством, на котором она установлена, для каждого используемого компьютера с Windows 8 необходимо зарегистрировать отдельную карту.



</div>

**Регистрация смарт-карты вручную**

1.  Войдите в систему на компьютере, на котором будет запущено приложение Lync.

2.  В браузере Internet Explorer откройте страницу регистрации сертификатов через Интернет вашей организации.
    
    Попросите администратора Lync Server сделать веб-адрес этого ресурса, если у вас его еще нет. URL-адрес будет выглядеть примерно так: https://MyCA.\[йоуркомпанинаме\]. com/certsrv.
    
    <div>
    

    > [!NOTE]  
    > Если вы используете Internet Explorer 10, возможно, этот сайт потребуется открыть в режиме совместимости.

    
    </div>

3.  На странице сертификации войдите со своей учетной записью в домене (а не с учетной записью администратора своего компьютера).

4.  На странице приветствия на веб-сайте выберите **Запросить сертификат**.

5.  Выберите **Расширенный запрос**.

6.  Выберите **Создать и выдать запрос к этому ЦС**, затем нажмите кнопку **Далее**.

7.  На экране появится страница "Станция подачи заявок смарт-карт". Согласитесь на установку элемента ActiveX и заполните форму расширенного запроса сертификата, указав приведенные ниже данные.
    
    1.  В раскрывающемся списке **Шаблон сертификата** выберите **Пользователь смарт-карты**.
    
    2.  Выберите **Создать новый набор ключей**.
    
    3.  Найдите сведения об изготовителе на этикетке смарт-карты и выберите этого изготовителя в раскрывающемся списке **Поставщик служб шифрования**.
    
    4.  В качестве формата запроса выберите **Поставщик служб шифрования**, если этот вариант еще не выбран.
    
    5.  В раскрывающемся списке "алгоритм хэширования" выберите **sha1**, если этот вариант еще не выбран.
    
    6.  Присвойте сертификату имя, позволяющее распознавать его, и нажмите кнопку **Отправить**.

8.  Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.

9.  По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    <div>
    

    > [!NOTE]  
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678.

    
    </div>

10. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.

11. Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.

12. По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    <div>
    

    > [!NOTE]  
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678.

    
    </div>

13. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.

14. Нажмите кнопку **ОК** для подтверждения сведений, отображаемых в сертификате.

15. При появлении уведомления о выдаче сертификата нажмите кнопку **Установить этот сертификат** для завершения процедуры регистрации.

</div>

<div>

## <a name="sign-in-to-lync-with-your-smart-card-credentials"></a>Вход в Lync с помощью учетных данных смарт-карты

Прежде чем использовать смарт-карту в первый раз, рекомендуется нажать кнопку **удалить мои данные для входа** на странице входа в Lync. При этом удаляются хранящиеся на компьютере учетные данные для входа, которые могут служить причиной ошибок.

**Вход в Lync с помощью учетных данных смарт-карты**

1.  Запустите клиент Lync.

2.  На экране входа введите имя учетной записи пользователя для входа в поле **Адрес для входа**, затем нажмите кнопку **Вход**.

3.  Если вы используете виртуальную смарт-карту, пропустите этот этап.
    
    В случае физической смарт-карты вставьте ее по запросу в устройство чтения смарт-карт и после обнаружения карты нажмите кнопку **ОК**.

4.  Введите PIN‑код смарт-карты,затем нажмите кнопку **ОК**.
    
    <div>
    

    > [!NOTE]  
    > Если специалист службы поддержки не предоставил вам ПИН-код для смарт-карты, введите значение по умолчанию: 12345678.

    
    </div>
