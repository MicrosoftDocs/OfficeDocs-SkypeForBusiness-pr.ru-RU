# <a name="managing-response-group-queues-in-lync-server-2013"></a>Управление очередями групп ответа в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-02_

Очереди содержат направленные в группу ответа звонки, пока агент не ответит на звонок. При управлении очередью вы назначаете ей одну или нескольких групп агентов и указываете параметры очереди, такие как число звонков, которое может вместить в себя очередь перед выполнением действия при переполнении, и период времени, в течение которого звонок ожидает агента перед выполнением действия таймаута. Когда приложение группы ответа выполняет поиск доступного агента, он выполняет поиск групп агентов в том порядке, в котором они перечислены.
