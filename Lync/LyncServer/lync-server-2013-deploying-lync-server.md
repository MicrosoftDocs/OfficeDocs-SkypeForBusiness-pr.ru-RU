# Развертывание Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-18_

Процедура развертывания Lync Server 2013 зависит от топологии и устанавливаемых компонентов Lync Server, а также от того, планируете ли вы развертывать переднего плана и Сервер Standard Edition. Подразделы этого раздела помогут вам определить необходимые компоненты и выполнить процедуру развертывания.
