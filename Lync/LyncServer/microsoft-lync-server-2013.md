# Microsoft Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Microsoft Lync Server 2013 ПО предлагает решения для обмена мгновенными сообщениями, сбора сведений о присутствии, конференц-связи и телефонной связи, которые могут поддерживать требования совместной работы на уровне предприятия.

Необходимо загрузить копию документации Lync Server 2013? Скомпилированный файл справки (.chm) **LyncServer2013\_ITPro** можно загрузить на странице Microsoft Download Center по адресу [http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103).

В следующей таблице представлен список тем в библиотеке документов сервера Lync Server 2013.
