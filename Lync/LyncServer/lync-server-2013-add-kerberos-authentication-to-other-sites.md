# <a name="in-lync-server-2013-add-kerberos-authentication-to-other-sites"></a>В Lync Server 2013 добавить проверку подлинности Kerberos для других сайтов

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2010-11-03_

Если необходимо добавить сайты в существующее развертывание, которое было настроено для проверки подлинности Kerberos, можно использовать существующую учетную запись проверки подлинности Kerberos для нового сайта или создать новую учетную запись. Дополнительные сведения о создании новой учетной записи для сайта можно найти [в статье Создание учетной записи проверки подлинности Kerberos в Lync Server 2013](lync-server-2013-create-a-kerberos-authentication-account.md). Для получения дополнительных сведений об использовании одной учетной записи для нового сайта путем назначения существующей учетной записи на сайте, ознакомьтесь с [Разназначением учетной записи проверки подлинности Kerberos сайту в Lync Server 2013](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md).
