# Добавление в Lync Server 2013 проверки подлинности Kerberos для других сайтов

 

_**Дата изменения раздела:** 2010-11-03_

Если необходимо добавить сайты в существующее развертывание, которое было настроено для проверки подлинности Kerberos, можно использовать существующую учетную запись проверки подлинности Kerberos для нового сайта или создать новую учетную запись. Сведения о создании новой учетной записи для сайта см. в разделе [Создание учетной записи для проверки подлинности Kerberos в Lync Server 2013](lync-server-2013-create-a-kerberos-authentication-account.md). Сведения об использовании одной и той же записи для нового сайта путем назначения сайта существующей записи см. в разделе [Назначение учетной записи проверки подлинности Kerberos для сайта в Lync Server 2013](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md).
