# <a name="using-the-lync-room-system-administrative-web-portal-in-lync-server-2013"></a>Использование веб-портала администрирования системы комнат Lync в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2014-11-10_

После развертывания ЛРС на сервере вы можете проверить состояние всех комнат ЛРС, войдя на веб-портал ЛРС администрирования из браузера.

<div>

## <a name="sign-in"></a>Вход

1.  Перейдите по следующему URL-адресу:
    
    https://\<Fe-Server\>/ЛРС

2.  Введите учетные данные учетной записи Лрссуппорт или учетной записи, добавленной в группу безопасности Лрссуппортадминграуп.

![Экран входа в систему на портале администратора системы Lync в комнате] (images/Dn436326.050bcf70-2f3b-46b2-9b96-ebd12679b713(OCS.15).png "Экран входа в систему на портале администратора системы Lync в комнате")

</div>

<div>

## <a name="lrs-administrative-web-portal-summary-page"></a>Общая страница веб-портала ЛРС администрирования

На странице Сводка представлены следующие сведения о всех ЛРС комнатах, развернутых на сервере.

  - **Пометьте**   настраиваемое имя, которое администратор предоставляет в комнате. Тег можно задать на портале, щелкнув имя конференц-зала.

  - **** Состояние работоспособности комнаты, которая является производной от сводного состояния работоспособности комнаты, которая отображается в разделе работоспособность на странице Параметры комнаты.   

  - **Следующее собрание**   . Дата и время запланированного следующего собрания.

  - **Версия ЛРС, производитель, модель**   эти значения предварительно заданы в ЛРС. В зависимости от производителя данные поля могут оставаться пустыми.

  - **Последнее обновление**   показывает время последнего обновления веб-страницы.

![Представление сводки на портале администратора системы Lync в комнате] (images/Dn743660.f829ce90-dd95-4725-bd94-6870c5dcf046(OCS.15).png "Представление сводки на портале администратора системы Lync в комнате")

</div>

<div>

## <a name="lrs-room-information"></a>Информация о ЛРС помещении

Раздел "сведения о комнате" на портале позволяет просматривать и настраивать отдельные комнаты ЛРС. В нем содержатся четыре раздела: параметры, сведения, устранение неполадок и исправность.

<div>

## <a name="settings"></a>Параметры

В разделе "Параметры" для выбранного конференц-зала можно задать пароль, тег конференц-зала и уровни громкости по умолчанию. Если вы настроили эти параметры, изменения реплицируются только после перезапуска консоли ЛРС. Вы увидите только параметры обновлений системы для комнатных систем Lync, которые имеют версию 15,12 и более поздние версии.

![Параметры комнаты на портале администратора системы Lync в комнате] (images/Dn743660.ab162e19-41ac-4991-9b2a-92575aa53eda(OCS.15).png "Параметры комнаты на портале администратора системы Lync в комнате")

</div>

<div>

## <a name="details"></a>Подробности

В разделе "сведения" представлена сводка по параметрам ЛРС комнаты, которая доступна только для чтения, в том числе: время последнего обновления; следующее собрание; Последние обновления, Обслуживание и калибровка; Параметры динамика, микрофона и звонка по умолчанию; Версия УНИВЕРСАЛЬНЫЙ КОД РЕСУРСА (URI) SIP; Количество экранов и подробные сведения о каждом экране; состояние и активность.

![Представление сведений о портале администрирования системы Lync в комнате] (images/Dn743660.2958bbba-db74-4670-a920-87fdfb2fc22d(OCS.15).png "Представление сведений о портале администрирования системы Lync в комнате")

</div>

## <a name="troubleshooting"></a>Поиск и устранение неполадок

Раздел "Поиск и устранение неполадок" можно использовать для удаленного сбора журналов и их сохранения в указанном месте. Вы также можете перезапустить консоль ЛРС (пользовательский интерфейс ЛРС) или перезапустить всю систему. Чтобы собрать журналы, укажите путь к папке в указанном формате и убедитесь, что в папке есть разрешения на запись, предоставленные учетной записи компьютера ЛРС. Если журнал имеет слишком большой размер, процесс сбора журналов может занять до пяти минут. Для просмотра актуального состояния обновите страницу.

![Ведение журнала на комнате администратора системы Lync в комнате] (images/Dn743660.749aee71-deaa-4ace-a146-fe2b349f0f42(OCS.15).png "Ведение журнала на комнате администратора системы Lync в комнате")

</div>

## <a name="frequently-asked-questions"></a>Ответы на часто задаваемые вопросы

<div>

## <a name="why-cant-i-sign-in-to-the-administrative-web-portal"></a>Почему не удается войти на веб-портал администрирования?

  - После открытия https://localhost/lrsстраницы входа вы сможете увидеть ее, но при вводе учетных данных вы не сможете войти в нее. В этом случае вы должны открыть https://FQDNofFEserver/lrs для входа на веб-портал администрирования.

  - Если компьютер, с которого осуществляется доступ к веб-порталу администрирования, находится в Рабочей группе, "http://" работать не будет. Вместо этого используйте "HTTPS".

</div>

<div>

## <a name="why-cant-i-see-lrs-in-the-administrative-web-portal"></a>Почему я не вижу ЛРС на веб-портале администрирования?

  - Убедитесь в том, что у вас есть учетные записи ЛРС в развертывании и они созданы в соответствии с рекомендациями по развертыванию веб-портала ЛРС администратора. Убедитесь в том, что учетные записи ЛРС предоставлены с помощью команды Enable-Ксмитингрум, а не Enable-CsUser на сервере Lync.

  - Если вы создали учетные записи ЛРС и не видите учетные записи на веб-портале администрирования, соберите журналы сервера с помощью средства ведения журнала на Lync Server с выбранным компонентом **митингпортал** , а затем отправьте их в контакт службы поддержки ЛРС.

</div>

<div>

## <a name="why-cant-i-see-the-status-of-lrs-in-the-administrative-web-portal"></a>Почему я не вижу состояние ЛРС на веб-портале администрирования?

  - Убедитесь, что для учетной записи пользователя LRSApp включена поддержка SIP.

  - Если у вас по-прежнему возникают проблемы, выполните сбор файла **Trace. log** в системе ЛРС с D\\:\\Tracing\\лрсадминлогс, а затем отправьте его в контакт службы поддержки ЛРС.

</div>
