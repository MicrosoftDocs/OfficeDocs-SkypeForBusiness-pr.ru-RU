# Поддерживаемые пути миграции и сценарии сосуществования в Lync Server 2013

 

_**Дата изменения раздела:** 2012-04-30_

Lync Server 2013 поддерживает перенос с Microsoft Lync Server 2010 и Microsoft Office Communications Server 2007 R2, включая совместное использование с определенными клиентскими и серверными компонентами этих предыдущих развертываний.
