# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Командлеты веб-сервера и служб в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2013-02-18_

Многие компоненты Microsoft Lync Server 2013 — это веб-компоненты, которые используются для выполнения своих задач, а также веб-службами. С помощью командлетов веб-сервера и веб-служб можно настраивать параметры веб-сервера и управлять простыми URL-адресами. Простые URL-адреса упрощают присоединение к собраниям и конференциям и упрощение входа пользователей на панель управления Lync Server 2013.
