# <a name="active-directory-cmdlets-in-lync-server-2013"></a>Командлеты Active Directory в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-06-20_

Командлеты Active Directory обычно используются программой установки и редко вызываются администратором напрямую. Однако администраторы могут использовать эти командлеты для подготовки (или разподготовки) домена или леса для Microsoft Lync Server 2013, а также для установки необходимых файлов схемы Active Directory.

<div>
