# Поддерживаемое совместное размещение серверов в Lync Server 2013

 

_**Дата изменения раздела:** 2012-06-29_

Система Lync Server 2013 поддерживает выровненное размещение некоторых ролей серверов и компонентов. То, какие роли сервера и компоненты вы можете выровнять, частично зависит от того, выполняете ли вы развертывание интерфейсного пула или сервера Standard Edition.
