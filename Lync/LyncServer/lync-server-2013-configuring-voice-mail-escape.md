# Настройка отмены для голосовой почты в Lync Server 2013

 

_**Дата изменения раздела:** 2013-02-22_

Если пользователь настроит одновременные звонки на мобильный телефон, то в случае, когда его мобильный телефон окажется выключенным, разряженным или же будет находиться вне зоны действия, звонящий обычно будет маршрутизироваться на личную голосовую почту этого пользователя. При наличии Microsoft Lync Server 2013 пользователь может выбрать возможность, благодаря которой звонки, связанные с его работой, будут маршрутизироваться в корпоративную систему голосовой почты. Эта возможность реализуется посредством следующей настройки таймера: если на звонок, поступающий в пределах определенного интервала времени, будет отвечать голосовая почта оператора, то Lync Server 2013 отключится от системы голосовой почты оператора (и личной голосовой почты пользователя); при этом вызов будет продолжать поступать в оставшиеся конечные точки пользователя, расположенные в корпоративной системе. Таким образом звонящий будет автоматически маршрутизироваться в корпоративную систему голосовой почты пользователя.

Эта настройка выполняется с помощью командлета Командная консоль Lync Server, Set-CsVoicePolicy, на уровне голосовой политики с использованием следующих параметров.

## Настройка escape-последовательности голосовой почты

1.  Запустите командную консоль Lync Server: нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы** и **Microsoft Lync Server 2013** и щелкните элемент **Командная консоль Lync Server**.

2.  Задайте следующие параметры для **Set-CsVoicePolicy**:
    
      - **EnableVoicemailEscapeTimer** — включает или отключает таймер escape-последовательности.
    
      - **PSTNVoicemailEscapeTimer** — задает значение времени ожидания в миллисекундах. Значение по умолчанию составляет 1500 миллисекунд; допустимыми являются значения из диапазона 0 – 8000 миллисекунд.

## Пример

    Set-CsVoicePolicy UserVoicePolicy -EnableVoiceMailEscapeTimer $true - PSTNVoicemailEscapeTimer 2000
    
    Set-CsVoicePolicy -Identity site:SitePolicy -EnableVoiceMailEscapeTimer $true -PSTNVoicemailEscapeTimer 1500
