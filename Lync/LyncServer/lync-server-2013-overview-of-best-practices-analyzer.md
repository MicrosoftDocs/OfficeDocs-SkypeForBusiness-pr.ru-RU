# <a name="overview-of-best-practices-analyzer-in-lync-server-2013"></a>Обзор анализатора соответствия рекомендациям в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-09-19_

Вы можете использовать Lync Server 2013, анализатор соответствия рекомендациям для выявления и устранения проблем, возникающих при развертывании Lync Server 2013. Lync Server 2013, анализатор соответствия рекомендациям собирает сведения о конфигурации из компонентов Lync Server 2013.

При правильном доступе к сети анализатор соответствия рекомендациям может проверять серверы, на которых работают доменные службы Active Directory, единая система обмена сообщениями Exchange Server (UM) и Lync Server 2013. Анализатор соответствия рекомендациям можно использовать для выполнения указанных ниже действий.

  - Упреждающее выполнение проверок, проверка того, что конфигурация задана в соответствии с рекомендованными рекомендациями.

  - Автоматическое определение необходимых обновлений для Lync Server 2013.

  - Создание списка проблем, таких как Неоптимальные параметры конфигурации, неподдерживаемые параметры, отсутствующие обновления или рекомендации, которые мы не рекомендуем использовать.

  - Помощь в устранении неполадок и устранение определенных проблем.

Анализатор соответствия рекомендациям предоставляет следующие возможности:

  - Минимальные требования для установки.

  - Интерактивная документация по обнаруженным проблемам, в том числе советы по устранению неполадок.

  - Сведения о конфигурации, которые можно сохранить для последующего рецензирования.

  - Штатный анализ системы.

Анализатор соответствия рекомендациям использует набор XML-файлов конфигурации, чтобы определить, какие данные нужно собирать из среды Lync Server 2013. Помимо проверки доменных служб Active Directory, она проверяет источники, такие как реестр операционной системы Windows Server и параметры в инструментарии управления Windows (WMI).

Анализатор соответствия рекомендациям сравнивает собираемые данные с набором предопределенных правил для параметров и конфигураций развертываний Lync Server 2013.

После того как вы сравните собранные данные с предопределенными правилами, средство сообщит о проблемах. В анализаторе соответствия рекомендациям содержатся сведения о том, что было обнаружено в отсканированной среде Lync Server 2013 и рекомендуемая конфигурация для каждой ошибки. Анализатор соответствия рекомендациям также содержит ссылки на более подробные сведения о конкретных проблемах.

<div>


> [!NOTE]  
> Lync Server 2013, анализатор соответствия рекомендациям собирает сведения о конфигурации только из компонентов Lync Server 2013. Вы можете использовать предыдущие версии средства для поиска предыдущих сред. Дополнительные сведения можно найти <A href="lync-server-2013-requirements-for-running-best-practices-analyzer.md">в разделе Требования для анализатора соответствия рекомендациям в Lync Server 2013</A>.
