# <a name="planning-for-response-groups-in-lync-server-2013"></a>Планирование групп ответа в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-06-19_

Если в Организации есть группы людей, которые отвечают на определенные типы звонков, такие как обслуживание клиентов, внутреннюю службу поддержки или общая поддержка по телефону для отдела, можно развернуть приложение группы ответа Lync Server для управления этими типами вызовов. Приложение группы ответа маршрутизирует и ставит в очередь входящие вызовы назначенных лиц, которые называются агентами. С помощью групп ответа можно увеличить использование телефонной службы поддержки и сократить расходы на работу этих служб. В этом разделе описываются рекомендации по планированию для группы ответа.
