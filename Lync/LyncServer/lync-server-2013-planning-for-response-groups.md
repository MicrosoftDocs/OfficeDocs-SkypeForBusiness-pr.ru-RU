# <a name="planning-for-response-groups-in-lync-server-2013"></a>Планирование групп ответа в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-06-19_

Если в вашей организации есть группы пользователей, которые отвечают и управляют определенными типами звонков, например для обслуживания клиентов, внутренней службы поддержки и поддержки по телефону для отдела, вы можете развернуть приложение группы ответа Lync Server для управления этими типами. из звонков. Приложение "группа ответа" маршрутизирует и помещает входящие звонки определенным людям, которые называются агентами. С помощью групп ответа можно увеличить степень использования телефонной службы поддержки и сократить расходы на работу этих служб. В этом разделе описаны вопросы планирования для группы ответа.
