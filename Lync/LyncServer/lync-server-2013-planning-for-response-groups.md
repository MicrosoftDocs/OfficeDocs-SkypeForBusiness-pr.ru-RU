# Планирование групп ответа в Lync Server 2013

 

_**Дата изменения раздела:** 2012-06-19_

Если в организации имеются группы сотрудников, отвечающие на определенные типы звонков и управляющие ими, такие как отдел по обслуживанию клиентов, внутренняя служба технической поддержки или общая телефонная служба поддержки для отдела, то для управления этими типами вызовов можно развернуть приложение "Группа ответа"Lync Server. Приложение "Группа ответа" маршрутизирует и ставит в очередь входящие звонки для указанных лиц, называемых агентами. С помощью групп ответа можно увеличить использование телефонной службы поддержки и сократить расходы на работу этих служб. В этом разделе описываются рекомендации по планированию групп ответа.
