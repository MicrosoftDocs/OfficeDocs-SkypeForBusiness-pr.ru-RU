# Планирование сервера сохраняемого чата в Lync Server 2013

 

_**Дата изменения раздела:** 2014-11-04_

Сервер Lync Server 2013сохраняемого сеанса беседыможно использовать для разрешения нескольким пользователям принимать участие в беседах, в которых они отправляют и получают доступ к контенту по определенным темам, включая текст, ссылки и файлы. Хотя пользователи в течение сеанса могут взаимодействовать в режиме реального времени, контент каждого сеанса является постоянным, и доступен после окончания сеанса.

В этом разделе описываются рекомендации по планированию развертывания сервера сохраняемого сеанса беседы в сервере Lync Server 2013, включая определение требований, идентификацию компонентов и поддерживаемых топологий, а также рекомендации по развертыванию.
