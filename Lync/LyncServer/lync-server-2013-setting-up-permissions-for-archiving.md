# Настройка разрешений для архивации

 

_**Дата изменения раздела:** 2012-10-01_

В Lync Server 2013 для выполнения определенных задач по-прежнему требуется быть членом одной или нескольких групп. Однако вы также можете использовать управление доступом на основе ролей для предоставления прав путем назначения пользователям предварительно определенных административных ролей Lync Server. Перед развертыванием архивации следует убедиться в том, что пользователям предоставлены соответствующие права и разрешения, а также назначены роли управления доступом на основе ролей. Подробные сведения о правах, разрешениях и ролях пользователей, необходимых для развертывания архивации, см. в разделе [Контрольный список развертывания для архивации в Lync Server 2013](lync-server-2013-deployment-checklist-for-archiving.md) в документации по планированию и документации по развертыванию. Подробные сведения об управлении доступом на основе ролей см. в разделе [Планирование контроля доступа на основе ролей в Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md) в документации по планированию.
