# <a name="separate-large-meeting-calendar-in-lync-server-2013"></a>Отдельный календарь больших собраний в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-01_

Для каждого пула больших собраний необходимо хранить в отдельном календаре большое количество собраний, запланированных в этом пуле. Например, вы можете использовать единую учетную запись пользователя в пуле больших собраний и использовать надстройку Outlook с Exchange и "собрание по сети" для Lync 2013 для поддержки отдельного календаря. Если вы используете несколько учетных записей пользователей, чтобы предоставить сотрудникам службы поддержки возможность создания больших собраний, можно настроить отдельный календарь, объединяющий все крупные собрания, созданные участниками сотрудника службы поддержки.

Ведение отдельного календаря больших собраний помогает избежать конфликтов и убедиться, что в любой момент времени активно только одно большое собрание.
