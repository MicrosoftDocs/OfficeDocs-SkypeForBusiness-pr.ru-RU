# Настройка Lync Server 2013 для маршрутизации к шлюзу SIP/CSTA

 

_**Дата изменения раздела:** 2012-10-05_

Шлюз SIP/CSTA – это шлюз между SIP и приложением CSTA. Шлюз SIP/CSTA обеспечивает интерфейс между существующей УАТС и Lync Server для маршрутизации запросов на удаленное управление звонками к УАТС. После установки шлюза SIP/CSTA необходимо выполнить следующие процедуры для каждого пула Lync Server, для которого требуется настроить удаленное управление звонками:

  - [Настройка статического маршрута для дистанционного управления вызовами в Lync Server 2013](lync-server-2013-configure-a-static-route-for-remote-call-control.md)

  - [Настройка записи доверенного приложения для удаленного управления звонками в Lync Server 2013](lync-server-2013-configure-a-trusted-application-entry-for-remote-call-control.md)
