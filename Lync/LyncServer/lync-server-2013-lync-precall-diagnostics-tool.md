# <a name="lync-precall-diagnostics-tool-in-lync-server-2013"></a>Средство диагностики предзвонков для Lync в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Последнее изменение темы:** 2016-11-04_

Средство диагностики предварительных вызовов Lync (PCD) — это клиентское приложение, которое позволяет увидеть, как текущее состояние сети может повлиять на качество звука в предстоящем голосовом звонке предприятия.

PCD наиболее полезен в ситуациях, когда последний транзит сети скорее всего самый слабый (например, с ноутбуками в общедоступной сети Wi-Fi или для домашних пользователей). PCD создает небольшой пакетный поток, который выполняет обход этого последнего участка сети. Затем средство анализирует поток пакета для оценки того, как колебание и убытки по этому принципу могут повлиять на качество звонков, а затем предоставляет отчет. PCD можно непрерывно запускать на клиенте, даже при размещается вызовов. Поток пакета не оказывает существенного влияние на пропускную способность.

**Последний выпуск PCD версии 1,1 содержит следующие усовершенствованные возможности:**

  - Поддержка более длинных паролей, которые теперь могут содержать до 127 символов

  - Дополнительная диагностика проблем с входом в систему проверки подлинности

  - Улучшенная поддержка гибридных развертываний Lync

  - Обновление средства выбора учетных данных

  - Улучшения стабильности

## <a name="running-lync-pcd"></a>Запуск Lync PCD

<div>

## <a name="running-windows-desktop-app"></a>Запуск классического приложения Windows

1.  Чтобы запустить PCD в системе Windows 7, выберите команду " **Диагностика вызовов** " в меню " **Пуск** ".
    
    Чтобы запустить PCD в системе Windows 8, нажмите значок на начальном экране или выполните поиск по запросу "Диагностика до вызова".
    
    ![Значок средства диагностики перед вызовом](images/Dn451255.c9800fde-54f6-4efe-bb35-1a38064ec380(OCS.15).png "Значок средства диагностики перед вызовом")

2.  При запуске средства выберите предпочтительный способ предоставления учетных данных и выберите режим работы сети в диалоговом окне **Параметры средства диагностики перед вызовом** , а затем нажмите кнопку **ОК**:

3.  Нажмите кнопку **начать проверку** , чтобы начать выполнение диагностики.
    
    Если вы выбрали параметр **использовать сетевые учетные данные** , тест начнется немедленно.
    
    Если выбран параметр **Разрешить ввод учетных данных** , откроется диалоговое окно " **Безопасность Windows** ". После ввода учетных данных тест начнется.

## <a name="uninstalling-lync-pcd"></a>Удаление Lync PCD

Чтобы удалить Lync PCD, выполните процедуру, описанную в операционной системе:

  - В системе Windows 7 откройте **Панель управления**, выберите пункт **программы и компоненты**, а затем дважды щелкните элемент Диагностика предварительных **вызовов Lync 2013**.

  - В системе Windows 8 щелкните правой кнопкой мыши плитку PCD и выберите **Удалить** на панели приложения в нижней части начального экрана.
