Командлеты управления пользователями, включенные в Microsoft Lync Server 2013, позволяют включать, отключать и изменять учетные записи пользователей Lync Server.

<div>

## <a name="user-management-cmdlets"></a>Командлеты для управления пользователями

Большинство задач управления, которые применяются к пользователям и учетным записям пользователей, можно выполнить с помощью панели управления Lync Server. Основными исключениями являются командлеты, связанные с поставщиками аудиоконференций. Задачи управления пользователями можно выполнять с помощью командлетов в командной консоли Lync Server или из скрипта. Используя скрипты, можно автоматизировать определенные задачи.
