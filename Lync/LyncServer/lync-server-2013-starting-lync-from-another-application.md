# Запуск Lync из другого приложения

 

_**Дата изменения раздела:** 2015-03-09_

Вы можете использовать параметры командной строки для быстрого запуска Lync 2013. Например, если пользователь щелкает номер телефона в другом приложении, это приложение может запустить экземпляр Lync 2013 и инициировать звонок на данный номер.

Lync 2013 также может распознать список с именами контактов, разделенными точкой с запятой, для многосторонней конференц-связи.

Если Lync 2013 настроен на автоматически вход при запуске, то в случае запуска Lync 2013 с параметрами командной строки открывается главное окно Lync. Если продукт Lync не настроен на автоматический вход при запуске, открывается окно входа.

В следующей таблице приведены доступные параметры.

<tr class="odd">
<td><p>tel:</p></td>
<td><p>tel URI</p></td>

</tr>
<tr class="even">
<td><p>callto:</p></td>
<td><p>tel:, sip: или типируемый tel URI</p></td>
<td><p>Открывает окно беседы для звонка, но не набирает указанный номер.</p></td>
</tr>
<tr class="odd">
<td><p>sip:</p></td>
<td><p>SIP URI</p></td>
<td><p>Открывает окно беседы с указанным URI для SIP в списке участников.</p></td>
</tr>
<tr class="even">
<td><p>Sips:</p></td>
<td><p>SIP URI</p></td>
<td><p>Если Lync 2013 настроен на использование протокола TLS, работает аналогично sip:. Если TLS не используется, отображает диалоговое окно, уведомляющее пользователя о том, что требуется более высокий уровень безопасности.</p></td>
</tr>
<tr class="odd">
<td><p>conf:</p></td>
<td><p>SIP URI конференции, к которой требуется присоединиться</p></td>
<td><p>Если URI является самостоятельным, создает экземпляр центра конференций и открывает представление только с составом участников. В противном случае открывает представление состава участников, но не отправляет INVITE.</p>
<p></p></td>
</tr>
<tr class="even">
<td><p>im:</p></td>
<td><p>SIP URI</p></td>
<td><p>Отображает окно беседы только с обменом мгновенными сообщениями с URI для SIP. Принимает несколько URI для SIP, указанных внутри угловых скобках (&lt;&gt;) без разделителя.</p>
<pre><code>im:&lt;sip:user1@host&gt;&lt;sip:user2@host&gt;</code></pre></td>
</tr>
</tbody>
</table>

### Примеры параметров командной строки

<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />

<td><p>Tel:+14255550101</p></td>
<td><p>Открывает представление только телефонной связи для номера +14255550101.</p></td>

<tr class="odd">
<td><p>Callto:sip:kazuto@litwareinc.com</p></td>

<td><p>sip:kazuto@litwareinc.com</p></td>
<td><p>Открывает окно беседы для контакта kazuto@litwareinc.com.</p></td>
</tr>

<td><p>conf:sip:https://meet.contoso.com/kazuto/7322994</p></td>
