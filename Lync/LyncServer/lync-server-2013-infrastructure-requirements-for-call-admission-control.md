# <a name="infrastructure-requirements-for-call-admission-control-in-lync-server-2013"></a>Требования к инфраструктуре для контроля допуска звонков в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-08-21_

Отсутствуют дополнительные требования к инфраструктуре, такие как специальные сетевые маршруты. При развертывании Lync Server 2013 будет автоматически установлена служба политики пропускной способности, используемая для применения политик контроля допуска звонков (CAC). Контроль допуска звонков работает только с клиентами, работающими под управлением Lync.
