# Отчет об активности в конференции в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

Отчет о конференциях позволяет легко найти ответ на такие вопросы, как сколько конференций проводится каждый день и когда они проводятся. Такая информация полезна не только сама по себе, но и в целях устранения неполадок. Например, предположим, что пользователи жалуются на то, что работа сети особенно замедляется в середине дня. Быстро просмотрев отчет о конференциях, можно определить одну возможную причину: гораздо больше конференций планируются на время с 10:00 до 14:00, чем на любое другое время.

Если замедление работы сети вызывает проблемы, вы можете порекомендовать пользователям перенести некоторые конференции на менее загруженное время суток.

## Доступ к отчету о конференциях

Доступ к отчету о конференциях можно получить из [Сводный отчет по конференц-связи в Lync Server 2013](lync-server-2013-conference-summary-report.md), выбрав один из следующих показателей:

  - Всего конференций

  - Общее количество участников

## Рекомендации по использованию отчета о конференциях

По умолчанию в отчете о конференциях приводится общее количество конференций за указанный период времени (например, за день или определенный час). Однако вы также можете просмотреть общее число участников за этот период или общее число минут участия в конференциях. Для этого нажмите кнопку "Показать или скрыть параметры", чтобы отобразить параметры фильтрации, и затем выберите в раскрывающемся списке "Отчет по" один из следующих пунктов:

  - Число участников

  - Минут участия

  - Число конференций
