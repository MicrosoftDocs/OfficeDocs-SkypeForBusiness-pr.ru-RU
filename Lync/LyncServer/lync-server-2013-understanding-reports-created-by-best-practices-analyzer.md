<span data-ttu-id="4360b-123">В отчетах, создаваемых анализатором соответствия рекомендациям, указываются конкретные проблемы, которые идентифицированы во время сканирования рабочей среды, включая следующие типы проблем:</span><span class="sxs-lookup"><span data-stu-id="4360b-123">The reports generated by Best Practices Analyzer indicate specific issues that are identified during the scan of your environment, including following types of issues:</span></span>

  - <span data-ttu-id="4360b-124">**Ошибки**   , необходимые для внесения изменений в среду.</span><span class="sxs-lookup"><span data-stu-id="4360b-124">**Errors**   Critical issues that require you to make a change in your environment.</span></span> <span data-ttu-id="4360b-125">Например, если основные компоненты Lync Server 2013 не установлены, в журнал записывается сообщение об ошибке.</span><span class="sxs-lookup"><span data-stu-id="4360b-125">For example, if Lync Server 2013 Core Components are not installed, an error is logged.</span></span>

    <span data-ttu-id="4360b-p106">Проблемы, которые классифицируются как ошибки, идентифицируются в отчете по красному символу X.   - <span data-ttu-id="4360b-130">**Проблемы с предупреждениями**   , которые не соответствуют реализации рекомендаций.</span><span class="sxs-lookup"><span data-stu-id="4360b-130">**Warnings**   Issues that are not consistent with the implementation of a best practice.</span></span> <span data-ttu-id="4360b-131">Это может указывать или не указывать на необходимость изменений в рабочей среде.</span><span class="sxs-lookup"><span data-stu-id="4360b-131">This may or may not indicate the need for a change in your environment.</span></span> <span data-ttu-id="4360b-132">Проблема может быть известной проблемой с конкретным параметром, который не нуждается в изменении.</span><span class="sxs-lookup"><span data-stu-id="4360b-132">The issue could be a known issue with a specific setting that you do not need to change.</span></span> <span data-ttu-id="4360b-133">Например, службы, которые не запускаются на сервере, регистрируются как предупреждения.</span><span class="sxs-lookup"><span data-stu-id="4360b-133">For example, services that are not started on a server are logged as warnings.</span></span>

  - <span data-ttu-id="4360b-138">**Информация**   включает все проблемы, которые не классифицируются как ошибки или предупреждения.</span><span class="sxs-lookup"><span data-stu-id="4360b-138">**Information**   Includes all issues that are not classified as errors or warnings.</span></span> <span data-ttu-id="4360b-139">Например, количество объектов сервера Standard Edition Lync Server 2013 в доменных службах Active Directory классифицируется как информационный вопрос.</span><span class="sxs-lookup"><span data-stu-id="4360b-139">For example, the number of Lync Server 2013 Standard Edition server objects in Active Directory Domain Services is classified as an information issue.</span></span>
