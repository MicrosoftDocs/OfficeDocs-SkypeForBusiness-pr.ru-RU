# <a name="assigning-per-user-policies-in-lync-server-2013"></a>Назначение политик для отдельных пользователей в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-14_
