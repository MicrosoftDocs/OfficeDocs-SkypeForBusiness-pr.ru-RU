# <a name="qoe-reports-in-lync-server-2013"></a>Отчеты QoE в Lync Server 2013

## <a name="qoe-summarytrend-reports"></a>Отчеты о тенденциях и тенденциях QoE

Отчеты сводки и тенденций QoE полезны для поиска пикового времени использования и изучения качества мультимедиа в это время, чтобы обеспечить достаточные сетевые ресурсы Организации. В организации также можно использовать многие фильтры, доступные в отчете, чтобы изолировать номера производительности для определенных расположений, типов клиентов и устройств, а также серверов.

Сводный отчет о QoE и тенденциях состоят из следующих элементов:

  - Сводный отчет по объединению Объединенных коммуникаций и тенденций

  - Сводный отчет по КТСОП/тенденции

  - Сводный отчет по конференциям/тенденцию

</div>

<div>

## <a name="qoe-performance-reports"></a>Отчеты о производительности QoE

Отчеты о производительности QoE предоставляют подробные сведения о трех отчетах, которые сосредоточены на производительности серверов-посредников, серверов аудио-и видеоконференций и расположений конечных точек QoE.

</div>

<div>

## <a name="mediation-server-performance-report"></a>Отчет о производительности сервера-посредника

В отчете о производительности сервера-посредника перечислены показатели, достигнутые одним или несколькими исправлениями в течение заданного периода времени. Сведения о метриках сервера единой системы обмена сообщениями (UC) и шлюза посредника между шлюзами каждого вызова указаны отдельно. Используйте этот отчет для сравнения объема и производительности различных серверов-посредников вашей организации.

Для каждого сервера-посредника (и для каждой стороны звонка) в отчете отображается следующее:

  - Количество вызовов

  - Потеря пакетов

  - Время кругового пути

  - Колебани

  - Оценка мнения общения (MOS)

  - Отправка MOS

  - Прослушивание MOS

  - Сетевые MOS

  - Снижение производительности сети MOS

  - Возврат эха

  - Уровень сигнала

</div>

<div>

## <a name="av-conferencing-server-performance-report"></a>Отчет о производительности сервера аудио-и видеоконференций

В отчете о производительности сервера аудио-и видеоконференций представлены списки метрик, полученные одним или несколькими серверами аудио-и видеоконференций в течение заданного периода времени. Этот отчет можно использовать для сравнения объема и производительности различных серверов аудио-и видеоконференций Организации. В организации также можно изолировать отчет, чтобы отображался только интерфейс для определенных типов клиентов, например клиентов Lync или клиентов PSTN.

Для каждого сервера аудио-и видеоконференций, в отчете отображается следующее:

  - Количество конференций

  - Потеря пакетов

  - Время кругового пути

  - Колебани

  - Оценка мнения общения (MOS)

  - Отправка MOS

  - Прослушивание MOS

  - Сетевые MOS

  - Снижение производительности сети MOS

  - Возврат эха

  - Уровень сигнала

</div>

<div>

## <a name="location-based-performance-report"></a>Отчет о производительности на основе расположения

В отчете о производительности на основе расположения представлен список сетевых расположений и для каждого местоположения отображается количество вызовов в каждом предварительно определенном диапазоне качества. Цель этого отчета заключается в том, чтобы узнать, как получить сведения о качестве массовых телефонных звонков в Организации для различных расположений, чтобы можно было определить неплохо выполняемые расположения и посмотреть различные оценки качества мультимедиа в Организации разных расположениях.

При отображении отчета отображаются различные таблицы метрики — по одной таблице для каждой метрики, на которую Организация решает отчитываться. Для этого отчета можно выбрать один из следующих показателей:

  - Оценка мнения общения (MOS)

  - Сетевые MOS

  - Снижение производительности сети MOS

  - Отправка MOS

  - Прослушивание MOS

  - Потеря пакетов

  - Колебани

  - Задержка

</div>
