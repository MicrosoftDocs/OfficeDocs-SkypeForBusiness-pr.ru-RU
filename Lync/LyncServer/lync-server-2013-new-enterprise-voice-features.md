# <a name="new-enterprise-voice-features-in-lync-server-2013"></a>Новые функции корпоративной голосовой связи в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-05-01_

В Lync Server 2013 представлено несколько новых функций маршрутизации и управления звонками, которые улучшают корпоративную голосовую связь.

Lync Server 2013 поддерживает несколько магистральных каналов между серверами-посредниками и шлюзами. *Магистраль* — это логическая связь между номером порта и сервером-посредником с помощью номера порта и шлюза. Это означает, что сервер-посредник может иметь несколько магистральных линий на различные шлюзы, а шлюз может иметь несколько магистральных каналов для разных серверов-посредников. Маршрутизация по магистрали позволяет Lync Server 2013 подключаться к IP-УАТС к шлюзу телефонной сети общего пользования (PSTN) или к нескольким системам IP-УАТС. Lync Server 2013 выступает в качестве связывающего (то есть, подключения) между различными системами телефонии.

Microsoft Lync Server 2013 делает улучшения в областях переадресации вызовов, одновременных звонков, обработки голосовой почты и презентации идентификации звонящего. Эти функции улучшают вызов корпоративной голосовой связи.

  - [Новая функция голосовой почты в Lync Server 2013](lync-server-2013-new-voice-mail-feature.md)

  - [Новая функция магистрали в Lync Server 2013](lync-server-2013-new-trunk-feature.md)

  - [Новая функция межмагистрали в Lync Server 2013](lync-server-2013-new-intertrunk-feature.md)
