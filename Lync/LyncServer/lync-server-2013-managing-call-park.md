# <a name="managing-call-park-in-lync-server-2013"></a>Управление Парковким вызовов в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-10_

Приложение парковки вызовов позволяет пользователю корпоративной голосовой связи выполнять вызов на удержании с одного телефона, а затем получать его позже с любого телефона. Когда пользователь Паркс вызов, Lync Server передает вызов на временный номер, который называется *орбитой*, когда вызов удерживается, пока кто-то не загружает его или не истечет.

В подразделах этого раздела приводятся пошаговые процедуры для задач, которые можно выполнять для настройки и обслуживания приложения парковки вызовов в развертывании.
