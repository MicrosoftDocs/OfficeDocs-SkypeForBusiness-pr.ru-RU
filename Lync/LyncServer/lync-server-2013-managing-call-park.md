# Управление парковкой вызовов в Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-10_

Приложение приостановки вызовов позволяет пользователю корпоративной голосовой связи поставить звонок на удержание с одного телефона, а затем восстановить его на другом телефоне. Когда пользователь паркует вызов, Lync Server переводит этот вызов на временный номер, называемый *орбитой*, на котором вызов удерживается, пока не будет восстановлен или не превысит время ожидания.

В подразделах этого раздела представлены пошаговые процедуры для выполнения задач настройки и обслуживания приостановки вызовов в развертывании.
