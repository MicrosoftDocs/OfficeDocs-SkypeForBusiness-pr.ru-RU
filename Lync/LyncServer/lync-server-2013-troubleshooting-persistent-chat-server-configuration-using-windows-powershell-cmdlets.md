# Устранение неполадок конфигурации сервера сохраняемого чата с помощью командлетов Windows PowerShell в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-06_

Используйте следующие обходные пути, чтобы устранить проблемы конфигурации сервера сохраняемого сеанса беседы с помощью командлетов интерфейса командной строки Windows PowerShell.

  - Если отображается сообщение об ошибке «субъект не подготовлен», значит добавляемый субъект не существует.
    

    > [!TIP]
    > Обходной путь – отсутствует. Это сообщение об ошибке просто указывает на то, что данный субъект не существует.
