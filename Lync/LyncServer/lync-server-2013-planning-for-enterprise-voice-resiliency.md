# Планирование устойчивости корпоративной голосовой связи в Lync Server 2013

 

_**Дата изменения раздела:** 2014-10-17_

Устойчивость голосовой связи означает возможность пользователей продолжать совершать и принимать вызовы, если центральный узел, в котором размещается Lync Server 2013, становится недоступным из-за сбоя глобальной сети или по другой причине. При сбое центрального узла служба Enterprise Voice должна без прерываний перейти на резервный узел. При сбое глобальной сети вызовы узла филиала должны перенаправляться на локальный шлюз ТСОП. В данном разделе описывается планирование устойчивости голосовой связи при сбое центрального узла или глобальной сети.
