# Новые возможности мониторинга в Lync Server 2013

 

_**Дата изменения раздела:** 2012-11-08_

В сервере Lync Server 2013 отсутствует отдельная роль сервера мониторинга. Мониторинг является дополнительной функцией, доступной на всех серверах переднего плана в развертывании Enterprise Edition, а также на серверах Standard Edition, которые можно реализовать и настроить для пула или сайта. В Lync Server 2013 добавлена возможность включения зеркалирования базы данных SQL Server для базы данных мониторинга.
