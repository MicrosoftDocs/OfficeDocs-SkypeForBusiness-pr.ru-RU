# Настройка надстройки собраний по сети в Lync Server 2013

 

_**Дата изменения раздела:** 2012-06-28_

надстройка собраний по сети для Lync 2013 поддерживает управление собраниями из клиента обмена сообщениями и совместной работы с данными Outlook. Существует несколько способов настройки рабочих характеристик надстройки. Среди них — новые параметры для добавления эмблем и текста в основную часть приглашений на интерактивное собрание.
