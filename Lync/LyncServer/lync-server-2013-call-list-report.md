# Отчет по списку звонков в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

Отчет по списку обзвона предоставляет параметры качества взаимодействия (QoE) для отдельных вызовов, выполненных и принятых сотрудниками организации. Обратите внимание, что действительные параметры, содержащиеся в отчете, зависят от способа доступа к отчету по списку обзвона. Например, если открыть отчет из [Отчет об устройстве в Lync Server 2013](lync-server-2013-device-report.md), вы увидите следующие показатели (которые также включаются в отчет по устройствам):

  - Микрофон вызывающего абонента

  - Динамик вызывающего абонента

  - Микрофон вызываемого абонента

  - Динамик вызываемого абонента

  - Соотношение времени голосовой коммутации

Однако, если вы откроете отчет по списку обзвона из [Отчет о местоположении в Lync Server 2013](lync-server-2013-location-report.md), эти показатели не будут отображаться; вместо них вы увидите следующие показатели:

  - Время кругового пути (мс)

  - Деградация (MOS)

  - Потеря пакетов

  - Колебание (мс)

Это показатели, содержащиеся в отчете о местоположении. Однако вы всегда можете щелкнуть «Подробные показатели» в отчете по списку обзвона, чтобы получить полную информацию QoE для любого вызова.

## Доступ к отчету по списку обзвона

Доступ к отчету по списку обзвона можно получить из любого из следующих отчетов:

  - [Отчет о местоположении в Lync Server 2013](lync-server-2013-location-report.md) (щелкните показатель громкости вызова или процента звонков низкого качества)

  - [Отчет об устройстве в Lync Server 2013](lync-server-2013-device-report.md) (щелкните показатель громкости вызова или процента звонков низкого качества)

  - [Сводный отчет по качеству среды в Lync Server 2013](lync-server-2013-media-quality-summary-report.md) (щелкните показатель громкости вызова или процента звонков низкого качества)

  - [Отчет о производительности сервера в Lync Server 2013](lync-server-2013-server-performance-report.md) (щелкните показатель громкости вызова или процента звонков низкого качества)

В отчете по списку обзвона можно получить доступ к [Отчет о звонке в Lync Server 2013](lync-server-2013-call-detail-report.md), щелкнув «Подробные показатели».

## Эффективное использование отчета по списку обзвона

Если вы не помните, для чего именно используются некоторые показатели отчета по списку обзвона (например, «Соотношение времени голосовой коммутации»), наведите указатель мыши на метку показателя; отобразится всплывающая подсказка, содержащая краткое описание показателя.

## Фильтры

Нет. Фильтрация отчета по списку обзвона невозможна.
