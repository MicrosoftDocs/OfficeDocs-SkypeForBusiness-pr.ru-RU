# Гибридная среда Lync Server 2013

 

_**Дата изменения раздела:** 2016-05-25_

В гибридных развертываниях Lync одни пользователи могут находиться в локальном развертывании Lync, а другие – в Lync Online, совместно используя один и тот же домен. Это может облегчить предоставление услуг Lync пользователям организации, расположенной а различных географических местоположениях, или пользователям, которые подключаются удаленно. Преимущества гибридных конфигураций Lync также можно использовать в качестве пути миграции к Office 365.

В этом разделе описываются гибридные конфигурации Lync Server 2013.
