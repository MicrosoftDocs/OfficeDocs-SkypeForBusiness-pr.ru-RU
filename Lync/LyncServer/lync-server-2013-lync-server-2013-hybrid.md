# <a name="lync-server-2013-hybrid"></a>Lync Server 2013 для гибридной среды

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2016-05-25_

С помощью гибридных развертываний Lync вы можете иметь некоторые локальные пользователи Lync и другие пользователи в Lync Online, которые совместно имеют общий доступ к одному и тому же домену. Это упрощает предоставление служб Lync пользователям вашей организации в различных географических расположениях или при удаленном подключении. Вы также можете воспользоваться преимуществами гибридных конфигураций Lync в качестве пути миграции к Office 365.

В этом разделе описываются гибридные конфигурации для Lync Server 2013.
