# Единственный директор в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-22_

Директор можно развернуть в конфигурации с одним Директором или в виде пула Директоров. В данном разделе описываются топология и конфигурация для отдельного Директора. Если вы планируете развернуть пул Директоров для увеличения емкости и обеспечения высокой доступности, см. рекомендации по планированию для такой топологии в разделе [Масштабируемый пул директоров в Lync Server 2013](lync-server-2013-scaled-director-pool.md).
