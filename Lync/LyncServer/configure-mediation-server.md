# <a name="configure-mediation-server"></a>Настройка сервера-посредника

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение раздела:** 2012-09-28_

В этой процедуре приведены действия для настройки пула Lync Server 2013 для использования сервера посредника Lync Server 2013 вместо прежних версий Office Communications Server 2007 R2 сервер-посредник.

Для успешной публикации, включения или отключения топологии при добавлении или удалении роль сервера, вы должны войти в систему как пользователь, являющийся членом группы RTCUniversalServerAdmins и "Администраторы домена". Можно также делегировать соответствующие права и разрешения администратора для добавления роли сервера. Дополнительные сведения см Делегирование разрешений на установку на сервере Standard Edition или Enterprise Edition server документации по развертыванию. Другие изменения конфигурации необходим только входить в группу RTCUniversalServerAdmins.

## <a name="to-configure-mediation-server-using-topology-builder"></a>Настройка с помощью построителя топологии Mediation Server

1.  Откройте существующую топологию в построителе топологий.

2.  В левой области перейдите к **пункту шлюзы ТСОП**.

3.  Щелкните правой кнопкой мыши **шлюзы ТСОП**и выберите команду **Создать шлюз IP/ТСОП**.

4.  Заполните страницу **Определение нового шлюза IP/ТСОП** со следующими сведениями:
    
      - Введите полное доменное имя или IP-адрес шлюза. Полное доменное имя шлюза является обязательным, если шлюз использует протокол TLS.
    
      - Примите значение по умолчанию, **порт прослушивания для шлюза IP/ТСОП** или введите новый порт прослушивания, если она была изменена.
    
      - Задайте **транспортный протокол Sip**.

5.  В левой области перейдите к **пулу переднего плана Enterprise Edition** или **Сервера Standard Edition**.

6.  Щелкните пул правой кнопкой мыши и выберите команду **Изменить свойства**.

7.  В разделе **Сервер-посредник**задайте **порты прослушивания**.

8.  Затем сопоставьте новый созданный шлюз ТСОП, выбрав его и нажав кнопку **Добавить**.

9.  В **Построителе топологий**выберите узел верхнего уровня **Lync Server**.

10. В меню **Действие** выберите команду **Опубликовать топологию** и нажмите кнопку **Далее**.

11. После завершения работы **мастера публикации** , нажмите кнопку **Готово** , чтобы закрыть мастер.
