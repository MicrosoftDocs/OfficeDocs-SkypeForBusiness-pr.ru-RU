# <a name="tblprincipalmemberdifference-in-lync-server-2013"></a>tblPrincipalMemberDifference в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-12_
