# <a name="overview-of-persistent-chat-server-in-lync-server-2013"></a>Обзор сервера сохраняемого чата в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-29_

Lync Server 2013, сервер сохраняемого чата позволяет пользователям участвовать в многосторонних беседах на основе темы, которые сохраняются со временем. Сервер сохраняемого чата поможет вашей организации выполнить следующие действия:

  - Совершенствование обмена данными между участниками географически распределенных многофункциональных рабочих групп. С помощью сохраняемого чата Teams может эффективно обмениваться информацией, идеями и решениями друг с другом. Сообщения, опубликованные в комнатах чата (на форумах) могут сохраняться (то есть оставаться доступными по прошествии времени), что позволяет участвовать в беседах людям, работающим в разных отделах и находящимся в разных местах, даже если они не выходят в сеть одновременно. Когда пользователь подключается к комнате чата, автоматически загружается определенное количество сообщений из истории чата, чтобы помочь пользователю включиться в беседу.

  - Усовершенствованное информирование. С помощью клиентских фильтров пользователи могут определять условия, например ключевые слова в содержимом сообщения, или значение поля "от" в сообщении — для получения уведомлений при выполнении этих условий в мгновенных сообщениях или сообщениях комнаты чата. Таким образом пользователи могут своевременно получать актуальное интересующее их содержимое.

  - Совершенствование обмена данными в рамках расширенной организации. Благодаря упрощению совместной работы над длинными разделами в Организации и предоставлением надежного места для обмена данными, сохраняемый чат помогает увеличить связь.

  - Сокращение перегруженности информацией. Пользователи могут следить за интересующими их комнатами чата и сообщениями с помощью клиентских фильтров и добавлять в список контактов нужные комнаты чата.

  - Упрощает распространение важных знаний и информации. В беседы можно включать документы и ссылки, делая их доступными для всех членов рабочей группы. Публикуя вопросы, пользователи могут получить ответы от специалистов в соответствующей области. Интеграция с другими информационными системами позволяет легко доносить важные данные до больших групп пользователей.

Чтобы включить комнаты чата в Lync Server 2013, разверните сохраняемый чат Lync Server 2013. Сведения о том, как включать комнаты чата, можно найти в <https://go.microsoft.com/fwlink/p/?linkid=270945>справке по постоянному чат. Если пользователям разрешено использовать Lync Server, а поддержка Lync Server развернута, пользователи могут установить и использовать Lync 2013 сохраняемый чат, чтобы обеспечить поддержку комнаты чата.

Если в Организации требуется соблюдение нормативных требований, при необходимости можно развернуть службу соответствия сохраняемого чата.
