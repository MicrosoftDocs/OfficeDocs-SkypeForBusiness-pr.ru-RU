# Обзор сервера сохраняемого чата в Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Lync Server 2013, сохраняемого сеанса беседы позволяет пользователям участвовать в многосторонних тематических беседах, данные которых сохраняются в течение длительное время. сохраняемого сеанса беседыпредоставляет организациям следующие возможности:

  - Совершенствование обмена данными между участниками географически распределенных многофункциональных рабочих групп. С помощью сохраняемый сеанс беседы рабочие группы могут эффективно обмениваться друг с другом информацией, идеями и решениями. Сообщения, опубликованные в комнатах чата (на форумах) могут сохраняться (то есть оставаться доступными по прошествии времени), что позволяет участвовать в беседах людям, работающим в разных отделах и находящимся в разных местах, даже если они не выходят в сеть одновременно. Когда пользователь подключается к комнате чата, автоматически загружается определенное количество сообщений из истории чата, чтобы помочь пользователю включиться в беседу.

  - Усовершенствованное информирование. С помощью клиентских фильтров пользователи могут определять условия, например ключевые слова в содержимом сообщений или значение в поле «Отправитель», и получать уведомления, если эти условия выполняются в мгновенных сообщениях или сообщениях чата сохраняемый сеанс беседыmessages. Таким образом пользователи могут своевременно получать актуальное интересующее их содержимое.

  - Совершенствование обмена данными в рамках расширенной организации. Упрощая проведение длительных совместных обсуждений важных вопросов и предоставляя хранилище для обмена информацией, сохраняемый сеанс беседы позволяет повысить эффективность обмена данными.

  - Сокращение перегруженности информацией. Пользователи могут следить за интересующими их комнатами чата и сообщениями с помощью клиентских фильтров и добавлять в список контактов нужные комнаты чата.

  - Упрощает распространение важных знаний и информации. В беседы можно включать документы и ссылки, делая их доступными для всех членов рабочей группы. Публикуя вопросы, пользователи могут получить ответы от специалистов в соответствующей области. Интеграция с другими информационными системами позволяет легко доносить важные данные до больших групп пользователей.

Чтобы включить комнаты чата в Lync Server 2013 разверните Lync Server 2013сохраняемый сеанс беседы. Сведения о включении комнат чата см. в справке по сохраняемый сеанс беседы по адресу <http://go.microsoft.com/fwlink/p/?linkid=270945>. Если развернут сервер Lync Server и для пользователей включена поддержка Lync Server, пользователи могут устанавливать и использовать Lync 2013сохраняемый сеанс беседы для работы с комнатами чата.

Если в вашей организации должны выполняться нормативные правила, вы также можете развернуть службу обеспечения соответствия сохраняемый сеанс беседы.
