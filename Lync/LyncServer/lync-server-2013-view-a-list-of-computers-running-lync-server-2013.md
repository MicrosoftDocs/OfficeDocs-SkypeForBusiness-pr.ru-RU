# Просмотр списка компьютеров, на которых работает Lync Server 2013

 

_**Дата изменения раздела:** 2012-11-01_

Панель управления Lync Server 2013 можно использовать для просмотра списка всех компьютеров в топологии, на которых работает сервер Lync Server 2013, и проверки состояния службы на каждом. Этот список можно упорядочивать по компьютерам, пулам или сайтам.

## Просмотр списка компьютеров, на которых работает Lync Server

1.  Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи, назначенной любой предварительно заданной административной роли Lync Server 2013. Дополнительные сведения о предварительно заданных административных ролях, которые доступны в Lync Server 2013, см. в разделе [Планирование контроля доступа на основе ролей в Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md).

2.  Откройте окно браузера и введите URL-адрес для администрирования, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Lync Server см. в разделе [Открытие средств администрирования Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  В левой панели навигации щелкните элемент **Топология**, а затем элемент **Состояние**.

4.  На странице **Состояние** выполните все необходимые действия из приведенных ниже.
    
      - Упорядочите список, щелкнув заголовок столбца **Компьютер**, **Пул** или **Сайт**, а затем щелкнув стрелку вверх или вниз.
    
      - Нажмите кнопку **Обновить**, чтобы просмотреть наиболее актуальный список.
    
      - Найдите конкретный компьютер, введя его имя в поле поиска.
