# <a name="reviewing-the-dns-report-in-lync-server-2013"></a>Просмотр отчета DNS в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-21_

Отчет DNS, который входит в отчет администратора, подробно описывает все Рекомендуемые и известные записи для системы доменных имен (DNS) во внутренней, внешней и внешней сетях. Если разработчик выполнил изменения в сетевом графике, а все IP-адреса и полные доменные имена (FQDN) определены для их рабочих значений, то отчет DNS предоставляет отличный ресурс конфигурации. Этот отчет также может служить в качестве операционного документа для устранения неполадок.

![Отчет администратора DNS](images/Gg558641.9dd1e810-ddc7-4816-a806-4239baf9ec51(OCS.15).jpg "Отчет администратора DNS")

Вы должны убедиться, что Группа управления DNS тщательно просмотрела отчет DNS, чтобы убедиться в отсутствии ошибок, которые могут привести к возникновению трудностей при развертывании, или что может усложнить сеанс устранения неполадок.
