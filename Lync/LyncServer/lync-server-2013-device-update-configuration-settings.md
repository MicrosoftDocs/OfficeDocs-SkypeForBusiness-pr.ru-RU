#  Параметры конфигурации обновления устройств в Lync Server 2013

 

_**Дата изменения раздела:** 2013-02-20_

Управление обновления устройств осуществляется с помощью параметров конфигурации устройств. Эти параметры можно применять на глобальном уровне или на уровне сайта.
