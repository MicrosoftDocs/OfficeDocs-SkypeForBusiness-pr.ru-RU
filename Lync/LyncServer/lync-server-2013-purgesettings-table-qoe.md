# Таблица PurgeSettings (качество взаимодействия) в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

В таблице PurgeSettings содержатся сведения о том, должны ли устаревшие записи автоматически удаляться из базы данных качества обслуживания, и когда это должно происходить. Обратите внимание на то, сведения об очистке можно также получить из командная консоль Microsoft Lync Server 2013, выполнив следующую команду:

    Get-CsQoEConfiguration

Эта таблица была представлена в Microsoft Lync Server 2013.
