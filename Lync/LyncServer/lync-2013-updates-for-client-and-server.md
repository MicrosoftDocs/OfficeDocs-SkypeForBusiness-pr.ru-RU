## <a name="determining-your-lync-desktop-client-version"></a>Определение версии клиента Lync для настольных ПК

Чтобы узнать, какую версию клиента для настольного компьютера Lync вы используете, откройте диалоговое окно **About Lync** , щелкнув значок Параметры. В меню **Параметры** выберите пункт **Справка** , а затем — пункт **о программе Microsoft Lync**. Номер версии клиента обычно отображается в первой строке диалогового окна " **о Lync** ".

</div>

<div>

## <a name="lync-client-updates"></a>Обновления клиента Lync

Одна из наиболее распространенных проверок, выполняемых при устранении проблем с клиентом Lync, состоит в том, что вы используете последнюю версию клиента Lync и устанавливаете ли все исправления. Большинство проблем клиента Lync, о которых сообщалось в отчете, разрешаются путем обновления до последней версии или обновления продукта. Таким образом, перед обращением в службу поддержки продуктов Майкрософт настоятельно рекомендуется установить последнюю версию клиента Lync вместе с последними обновлениями. Эта [статья](https://docs.microsoft.com/SkypeForBusiness/sfb-client-updates#lync-2013-client-updates) поможет вам определить правильный номер версии выпуска.

У каждого обновления для клиента Lync есть свой номер версии. Номер версии, связанный с обновлением, можно определить, изучив соответствующую статью базы знаний, связанную со следующей таблицей.

</div>
