# Отчеты по использованию системы в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-21_

В отчетах об использовании системы предоставляется информация об использовании системы с учетом данных регистрации вызовов (CDR), собранных Lync Server.

## Содержание

  - [Отчет о регистрации пользователей в Lync Server 2013](lync-server-2013-user-registration-report.md)
    
    В отчете представлены сводные данные о подключении пользователей к развертыванию Lync Server 2013 по данным таких событий регистрации, как вход в систему. В этом отчете можно просмотреть сведения о внешних и внутренних входах в систему и сравнить число пользователей, выполнивших вход в Lync Server 2013, с числом пользователей, фактически работавших со службой после входа.

  - [Сводный отчет по одноранговой активности в Lync Server 2013](lync-server-2013-peer-to-peer-activity-summary-report.md)
    
    Содержит сводку по одноранговым мгновенным сообщениям, аудио- и видеовызовам, передаче файлов и сеансам совместного использования приложений. Одноранговые сеансы – это сеансы с участием всего двух пользователей.

  - [Сводный отчет по конференц-связи в Lync Server 2013](lync-server-2013-conference-summary-report.md)
    
    Сводка по всем действиям, связанным с конференциями. Конференции – это сеансы с участием трех или более пользователей.

  - [Сводный отчет по конференц-связи через ТСОП в Lync Server 2013](lync-server-2013-pstn-conference-summary-report.md)
    
    Сводка по всем конференциям ТСОП. Это конференции, к которым хотя бы один пользователь подключается по телефонной сети общего пользования (ТСОП) (т.н. *конференц-связь с телефонным подключением* .

  - [Отчет по использованию группы ответа в Lync Server 2013](lync-server-2013-response-group-usage-report.md)
    
    Сводка по использованию ответа. The "Группа ответа" позволяет автоматически выполнять маршрутизацию телефонных звонков таким объектам, как службы поддержки.

  - [Отчет по инвентарю IP-телефонов в Lync Server 2013](lync-server-2013-ip-phone-inventory-report.md)
    
    Предоставляет сведения об IP-телефонах, используемых в данный момент в организации. Следует учесть, что этот отчет основан на данных о регистрации телефонов и входах в систему, и его не следует считать полным отчетом об инвентаризации. Например, вы могли избавиться от телефонов, которые все еще указаны в отчете (так как они применялись для входа по крайней мере один раз), или приобрести новые телефоны, которые не будут отображаться в отчете, только потому, что они еще не использовались для входа в Lync Server.

  - [Отчет по контролю допуска звонков в Lync Server 2013](lync-server-2013-call-admission-control-report.md)
    
    Предоставляет список действий с одноранговыми сеансами и конференциями, которые используют контроль допуска звонков. Контроль допуска звонков (CAC) позволяет определить, следует ли разрешить сеансы общения в реальном времени, например голосовые или видеовызовы, на основе ограничения пропускной способности.
