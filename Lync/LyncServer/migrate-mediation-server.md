# Перенос сервера-посредника

 

_**Дата изменения раздела:** 2012-09-28_

Сервер-посредник объединяется с пилотной топологией Lync Server 2013 при запуске мастера объединения. Однако сервер-посредник Lync Server 2013 настраивается только после того, как будет завершена миграция всех пользователей, так как пул Office Communications Server 2007 R2 не может обмениваться данными с сервером-посредником Lync Server 2013. При выполнении параллельной миграции пул Lync Server 2013 обменивается данными с сервером-посредником Office Communications Server 2007 R2.

При настройке сервера-посредника Lync Server 2013 вам также необходимо обновить или заменить шлюзы Office Communications Server 2007 R2. Шлюзы Office Communications Server 2007 R2 не поддерживают сервер-посредник Lync Server 2013. Вам нужно развернуть шлюзы, сертифицированные для Lync Server 2013, и связать их с сервером-посредником Lync Server 2013. Это действие является обязательным перед окончательным выводом развертывания Office Communications Server 2007 R2 из эксплуатации.

В подразделах этого раздела описываются действия по настройке, которые необходимо выполнить после завершения миграции сервера-посредника Lync Server 2013. Переход с совместно размещенного сервера-посредника на отдельный сервер-посредник является необязательным.
