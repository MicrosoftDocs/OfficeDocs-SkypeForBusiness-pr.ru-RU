# Проверка среды Lync Server 2010

 

_**Дата изменения раздела:** 2012-10-19_

Перед развертыванием Lync Server 2013 в состоянии сосуществования с Lync Server 2010 необходимо убедиться, что службы Lync Server 2010 настроены и запущены. Важно определить основные службы и функции, существующие в устаревшей среде, перед развертыванием пилотного пула Lync Server 2013. Перед развертыванием XMPP Microsoft Lync Server 2013 в состоянии сосуществования с устаревшим развертыванием XMPP необходимо убедиться, что устаревшие службы XMPP настроены и запущены, и определить, какие федеративные партнеры поддерживаются устаревшей конфигурацией XMPP. Проверка устаревшего развертывания Lync Server 2010 включает следующие действия:

  - проверку, запущены ли службы Lync Server 2010;

  - просмотр топологии и пользователей в Lync Server 2010;

  - проверку параметров федерации и пограничных серверов;

  - проверку служб XMPP и федеративных партнеров.

**Проверка, запущены ли службы Lync Server 2010**

1.  С сервера переднего плана системы Lync Server 2010 перейдите к приложению "Средства администрирования\\Службы".

2.  Убедитесь, что на сервере переднего плана выполняются следующие службы:
    
    ![Список служб, выполняющихся на сервере переднего плана](images/JJ205231.639f2729-b759-4d8e-b4ad-59d7f68adcd2(OCS.15).jpg "Список служб, выполняющихся на сервере переднего плана")

**Просмотр топологии Lync Server 2010 в управления Lync Server**

1.  Войдите на сервер переднего плана с помощью учетной записи, являющейся членом группы RTCUniversalServerAdmins или членом административной роли CsAdministrator или CsUserAdministrator.

2.  Откройте панель управления Lync Server.

3.  Выберите элемент **Топология** . Убедитесь, что перечислены разные серверы вашего развертывания Lync Server 2010.
    
    ![Страница топологии в панели правления Lync Server 2010](images/JJ205231.338ce4fb-2162-4176-a249-ec4ae021fa6a(OCS.15).jpg "Страница топологии в панели правления Lync Server 2010")

**Просмотр пользователей Lync Server 2010 в управления Lync Server**

1.  Откройте панель управления Lync Server.

2.  Выберите элемент **Пользователи** и нажмите кнопку **Найти** (Поиск).

3.  Убедитесь, что в столбце **Registrar Pool** (Пул регистратора) для каждого перечисленного пользователя указывается пул Lync Server 2010.
    
    ![Список пользователей в панели правления Lync Server 2010](images/JJ205231.a9378c40-7a52-4c78-ad83-1463847c9edb(OCS.15).jpg "Список пользователей в панели правления Lync Server 2010")

**Проверка параметров пограничного сервера Lync Server 2010 и федерации**

1.  Запустите построитель топологий.

2.  Выберите параметр **Загрузить топологию из существующего развертывания**.

3.  Выберите имя файла и сохраните топологию с типом файла по умолчанию (TBXML).

4.  Разверните узел Lync Server 2010, чтобы открыть разные серверные роли в развертывании.

5.  Выберите узел сайта и проверьте, установлено ли значение **Назначение маршрута федерации для сайта** (Назначение федеративного маршрута к узлу).
    
    ![Построитель топологий, федеративный маршрут сайта](images/JJ205231.87de3735-af7e-4280-8d72-c42cb0ea1c05(OCS.15).jpg "Построитель топологий, федеративный маршрут сайта")

6.  Затем выберите сервер Standard Edition или интерфейсный пул Enterprise Edition. Определите, настроен ли пограничный пул для мультимедиа в разделе **Associations** (Связи).
    
    ![Построитель топологий, показывающий серверы и пулы](images/JJ205231.5ad5ea3b-b122-44dd-8968-f1147d6d45f1(OCS.15).jpg "Построитель топологий, показывающий серверы и пулы")

7.  Наконец, выберите пограничный пул и определите, настроен ли пул следующих прыжков в разделе **Выбор узла следующего перехода** (Выбор следующего прыжка).
    
    ![Построитель топологий, выбор узла следующего перехода](images/JJ205231.3121e723-fba7-498e-a786-bde7be1a55e2(OCS.15).jpg "Построитель топологий, выбор узла следующего перехода")

**Проверка устаревшей конфигурации федеративных партнеров XMPP**

1.  На устаревшем сервере XMPP перейдите к приложению "Администрирование\\Службы".

2.  Убедитесь, что служба шлюза XMPP Office Communications Server запущена.
    
    ![Служба шлюза XMPP Office Communications Server](images/JJ205231.23223724-3c4b-4cb9-ace2-1cab2c3c91c3(OCS.15).jpg "Служба шлюза XMPP Office Communications Server")
