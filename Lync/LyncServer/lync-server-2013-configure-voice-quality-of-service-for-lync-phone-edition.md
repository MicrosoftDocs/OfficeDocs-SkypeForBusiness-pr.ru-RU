# Настройка качества голосовой связи для Lync Phone Edition

 

_**Дата изменения раздела:** 2012-09-29_

Вы можете настроить требования качества обслуживания (QoS) голосовой связи для устройств Lync Phone Edition в пуле, установив уровень качества обслуживания для подключаемых к системе Lync Server 2013 IP-телефонов.

## Настройка качества обслуживания голосовой связи для Lync Phone Edition

1.  Откройте окно браузера и введите URL-адрес для администрирования, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Lync Server см. в разделе [Открытие средств администрирования Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

2.  В левой панели навигации щелкните элемент **Clients** (Клиенты), а затем **Device Configuration** (Конфигурация устройства).

3.  В списке конфигураций на странице **Device Configuration** (Конфигурация устройства) дважды щелкните конфигурацию, для которой хотите изменить параметры качества обслуживания.

4.  Укажите уровень качества обслуживания в области **Voice quality of service** (Качество обслуживания голосовой связи) окна **Edit Device Configuration** (Изменение конфигурации устройства). По умолчанию используется значение **40**.
