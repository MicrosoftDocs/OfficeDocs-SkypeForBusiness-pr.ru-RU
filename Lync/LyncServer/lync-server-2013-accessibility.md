Майкрософт стремится к тому, чтобы сделать использование своих продуктов и служб простым и удобным для каждого. В следующих разделах представлены сведения о возможностях, продуктах и службах, облегчающих доступ к Lync Server 2013 для людей с ограниченными возможностями.

## <a name="accessibility-features-of-lync-server-2013"></a>Специальные возможности Lync Server 2013

Следующие функции в Lync Server 2013 помогают сделать ее более доступной для людей с ограниченными возможностями.

  - Сочетания клавиш

  - Альтернативный текст для рисунков

Кроме того, некоторые специальные возможности и средства Windows могут быть полезны пользователям Lync Server с ограниченными возможностями. Изменения размера и цвета Windows PowerShell обеспечивают специальные возможности при использовании командной консоли Lync Server. Дополнительные сведения о специальных возможностях Windows PowerShell: "Специальные возможности в Windows PowerShell 2,0" в библиотеке TechNet по адресу [https://go.microsoft.com/fwlink/p/?linkId=98964](https://go.microsoft.com/fwlink/p/?linkid=98964) .

## <a name="alternate-text-for-figures"></a>Альтернативный текст для рисунков

На каждом рисунке в справке Lync Server 2013, включая снимки экрана, диаграммы, блоковые диаграммы и другие фигуры, связан альтернативный текст. Если пользователю сложно рассмотреть рисунок, он может навести на него указатель мыши и прочитать соответствующий альтернативный текст. Альтернативный текст представляет собой описание того, что изображено на рисунке.

## <a name="accessibility-features-of-windows"></a>Специальные возможности Windows

В операционной системе Windows имеется множество встроенных специальных возможностей, которые могут оказаться удобными для тех, у кого возникают трудности при вводе или использовании мыши, могут быть слабыми или недостаточными. Эти компоненты устанавливаются с помощью программы установки ОС. Подробные сведения об этих возможностях можно найти в справке по Windows или Microsoft Accessibility на сайте [https://go.microsoft.com/fwlink/p/?linkId=18139](https://go.microsoft.com/fwlink/p/?linkid=18139) .

  - **Бесплатные пошаговые учебники**     Корпорация Майкрософт предлагает ряд пошаговых руководств, в которых приведены подробные инструкции по настройке специальных возможностей и параметров компьютера. Действия с мышью и клавиатурой описаны параллельно, чтобы можно было научиться работать с помощью мыши, клавиатуры или того и другого.
    
    Чтобы найти пошаговые учебники по продуктам корпорации Майкрософт, обратитесь к разделу Специальные возможности корпорации Майкрософт по адресу [https://go.microsoft.com/fwlink/p/?linkId=18139](https://go.microsoft.com/fwlink/p/?linkid=18139) .

  - **Продукты с поддержкой специальных возможностей для Windows**     Для людей с ограниченными возможностями можно использовать широкий спектр вспомогательных продуктов. Вы можете выполнить поиск в каталоге продуктов со специальными возможностями, которые запускаются в Windows на веб-сайте специальных возможностей Майкрософт по адресу [https://go.microsoft.com/fwlink/p/?linkId=18139](https://go.microsoft.com/fwlink/p/?linkid=18139) .
    
    В случае использования специальных возможностей перед обновлением программного обеспечения или оборудования обязательно свяжитесь с поставщиком специальных возможностей, чтобы узнать о потенциальных проблемах совместимости.
