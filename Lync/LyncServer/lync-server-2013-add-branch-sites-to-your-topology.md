# Добавление сайтов филиалов в топологию в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-05_

Сайты филиалов представляют отделения организации, подключенные к центральным офисам с помощью канала глобальной сети. Чтобы добавить сайт филиала в топологию Lync, выполните на центральном сайте следующую процедуру.

## Добавление сайтов филиалов в топологию

1.  Нажмите кнопку **Пуск** и затем последовательно выберите **Все программы** , **Microsoft Lync Server** и **Lync Server Topology Builder** (Построитель топологий Lync Server).

2.  В дереве консоли разверните центральный сайт, щелкните правой кнопкой мыши элемент **Общие компоненты** и выберите команду **Создать сайт филиала** .

3.  В диалоговом окне **Определение нового сайта филиала для сайта** нажмите **Имя**, а затем введите имя сайта филиала.

4.  (Необязательно) Нажмите **Описание** и введите содержательное описание для сайта филиала.

5.  Нажмите кнопку **Далее** .

6.  (Необязательно) В следующем диалоговом окне **Определение нового сайта филиала для сайта** выполните любое из следующих действий:
    
      - Нажмите **Город**, а затем введите название города, в котором расположен сайт филиала.
    
      - Нажмите **Регион**, а затем введите название региона или области, где расположен сайт филиала.
    
      - Нажмите **Код страны**, а затем введите двузначный вызывной код для страны/региона, где расположен сайт филиала.

7.  Нажмите **Далее** и выполните одно из следующих действий:
    
      - Если на этом сайте вы используете сервер или для обеспечения связи в филиалах, то установите флажок **Open the New Survivable Wizard when this wizard closes** (Открыть новый мастер обеспечения связи после завершения работы этого мастера), нажмите кнопку **Готово** и затем следуйте указаниям открывшегося мастера. Дополнительные сведения об этом мастере см. в разделе [Определение устройства или сервера для обеспечения связи в филиалах в Lync Server 2013](lync-server-2013-define-a-survivable-branch-appliance-or-server.md).
    
      - Если на этом сайте не используется сервер или для обеспечения связи в филиалах, снимите флажок **Open the New Survivable Wizard when this wizard closes** (Открыть новый мастер обеспечения связи при закрытии этого мастера) и затем нажмите кнопку **Finish** (Готово).

8.  Повторите предыдущие действия для каждого сайта филиала, который необходимо добавить в топологию.
