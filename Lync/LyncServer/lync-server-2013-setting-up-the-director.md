# <a name="setting-up-the-director-in-lync-server-2013"></a>Настройка директора в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-05-05_

Если вы разворачиваете доступ для внешних пользователей, развертывая пограничные серверы, один из вариантов — развернуть директорию. Директор — это сервер, на котором работает Microsoft Lync Server 2013, который проверяет подлинность запросов пользователей, но не предоставляет никаких учетных записей пользователей. Теперь это не обязательно, но очень полезно, если вы беспокоитесь о производительности и хотите упростить запросы проверки подлинности. Если вы решили, что это хорошая идея для вашей организации, действия по настройке директора или пула директоров похожи на настройку пула переднего плана Enterprise Edition или сервера Standard Edition. После определения директоров в построителе топологий необходимо выполнить действия, описанные в этом разделе.
