# <a name="clients-supported-for-response-group-in-lync-server-2013"></a>Клиенты, поддерживаемые для группы ответа в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2014-03-28_

Приложение группы ответа поддерживает следующие клиенты:

  - Клиент Lync 2013 для настольных ПК

  - Клиент Lync 2010 для настольных ПК

  - Lync 2010 Attendant

  - Помощник Office Communications Server 2007 R2

  - Lync Phone Edition

<div>


> [!NOTE]  
> Приложение группы ответа не поддерживается в клиентах Lync Mobile.



</div>

Для получения дополнительных сведений о новых возможностях обратитесь к разделу [новые функции приложения группы ответа в Lync Server 2013](lync-server-2013-new-response-group-application-features.md) в документации "Приступая к работе".

Используемый клиент зависит от типа пользователя группы ответа, который вы используете:

  - **Вызывающие** могут вызывать группу ответа с помощью любого вышеуказанного клиента или обычного телефона ТСОП.

  - **Неформальные агенты** (агенты, которые не входят в свои группы для принятия вызовов) могут принимать звонки с помощью помощника, Lync или Lync Phone Edition. Неформальные агенты автоматически подписываются в свои группы при входе в Lync Server 2013 с помощью одного из этих клиентов.

  - **Формальные агенты** (агенты, которые должны входить в свои группы и принимать звонки) могут принимать звонки с помощью Lync 2013 и доступа к консоли агента из элемента меню, а также с помощью помощника и доступа к консоли агента непосредственно из Internet Explorer.
