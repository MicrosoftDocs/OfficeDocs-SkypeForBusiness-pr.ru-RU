# <a name="client-management-cmdlets-in-lync-server-2013"></a>Командлеты управления клиентом в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-09-27_

Управление клиентами состоит в основном за определение того, какие клиентские приложения (например, Microsoft Lync 2013) смогут входить на сервер Lync Server 2013 и определить возможности, доступные этим клиентским приложениям после того, как они вошли в систему.

<div>


> [!NOTE]
> Дополнительные сведения о командлетах можно найти в блоге Lync&nbsp;Server Windows PowerShell по <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>адресу. Содержимое всех блогов и их URL-адреса могут быть изменены без уведомления.



</div>

## <a name="client-management-cmdlets"></a>Командлеты управления клиентом

Большинство задач управления, применяемых к управлению клиентами, можно выполнить с помощью панели управления Lync Server 2013. Эти же задачи можно выполнить с помощью командлетов из командной консоли Lync Server Management Shell или из сценария. Используя сценарий, вы можете автоматизировать определенные задачи.
