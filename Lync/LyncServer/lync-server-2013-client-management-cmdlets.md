# <a name="client-management-cmdlets-in-lync-server-2013"></a>Командлеты управления клиентами в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-27_

Управление клиентами состоит в основном за определение того, какие клиентские приложения (например, Microsoft Lync 2013) могут выполнять вход в Lync Server 2013 и определять возможности, доступные этим клиентским приложениям после входа в систему.

<div>


> [!NOTE]
> Дополнительные сведения о командлетах можно найти в блоге Lync&nbsp;Server Windows PowerShell по <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>адресу. Содержимое и URL-адрес любого блога могут быть изменены без предварительного уведомления.

## <a name="client-management-cmdlets"></a>Командлеты для управления клиентами

Большинство задач управления, применяемых к управлению клиентами, можно выполнить с помощью панели управления Lync Server 2013. Эти же задачи можно выполнить с помощью командлетов из командной консоли Lync Server или из скрипта. Используя скрипты, можно автоматизировать определенные задачи.
