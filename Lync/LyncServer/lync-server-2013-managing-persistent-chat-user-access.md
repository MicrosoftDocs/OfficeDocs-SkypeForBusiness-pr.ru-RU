# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Управление доступом пользователей сохраняемого чата в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-06_

В следующих разделах описано, как использовать панель управления Lync Server 2013 и страницу **сохраняемого чата** для управления доступом пользователей в сохраняемом чате. Для управления сервером сохраняемого чата необходимо обладать правами и разрешениями администратора сохраняемого чата в системе. При установке сервера сохраняемого чата необходимо указать одного или нескольких пользователей, у которых есть необходимые права и разрешения администратора сохраняемого чата. Чтобы предоставить права администратора и разрешения для дополнительных учетных записей пользователей, используйте **Управление пользователями и группами пользователей**. Дополнительные сведения: [предоставление пользователю возможности управлять категориями, комнатами чата и правами и разрешениями пользователей в Lync Server 2013](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md).
