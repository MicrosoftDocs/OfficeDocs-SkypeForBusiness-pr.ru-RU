# <a name="running-voice-routing-tests-in-lync-server-2013"></a>Выполнение тестов маршрутизации голосовой связи в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-21_

Lync Server 2013 предоставляет два различных метода тестирования маршрутов голосовой связи: неформальное, нерегламентированное тестирование по любому номеру телефона и любому маршруту голоса; Вы также можете выполнить более формальное тестирование с помощью тестовых случаев маршрутизации голосовых вызовов. При формальном тестировании вы определяете, как набираемый номер, назначенная абонентская политика и политика голосовой связи, а Lync Server может проверить, что при наличии этих условий предоставленный номер может успешно маршрутизироваться в сеть PSTN. Оба этих метода описаны в последующих разделах настоящей документации.
