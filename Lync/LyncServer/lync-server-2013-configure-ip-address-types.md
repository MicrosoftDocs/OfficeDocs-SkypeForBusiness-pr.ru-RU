# Настройка типов IP-адресов в Lync Server 2013

 

_**Дата изменения раздела:** 2012-06-13_

Типы IP-адресов разворачиваются с помощью параметров топологии, настроенных в построителе топологий. В этой статье описывается развертывание типов IP-адресов в пулах серверов переднего плана, на серверах-посредниках и в пулах пограничных сервер.
