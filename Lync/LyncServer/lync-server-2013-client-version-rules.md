# Правила версий клиентов

 

_**Дата изменения раздела:** 2012-12-11_

Политика версий клиентов состоит из набора правил политики версий клиентов. Эти правила определяют действия, которые следует предпринять при попытке пользователей выполнить вход с использованием определенных клиентов и их версий. Воспользуйтесь следующими процедурами, чтобы настроить правила политики версий клиентов для Lync Server 2013
