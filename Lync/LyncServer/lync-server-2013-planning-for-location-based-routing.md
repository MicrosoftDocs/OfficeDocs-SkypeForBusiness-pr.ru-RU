# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>Планирование маршрутизации Location-Based в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Последнее изменение темы:** 2013-07-31_

Сведения, приведенные в этом разделе, относятся к накопительным обновлениям для Lync Server 2013:2013 февраля.

Маршрутизация Location-Based позволяет ограничить маршрутизацию вызовов между конечными точками VoIP и PSTN в зависимости от расположения сторон в вызове. Location-Based маршрутизация является частью инфраструктуры Lync Server 2013 Enterprise Voice. Location-Based маршрутизация — это функция управления звонками, которая управляет тем, как звонки направляются Lync Server 2013 CU1. Он применяет правила авторизации вызовов для маршрутизации звонков в конечные точки УАТС или PSTN на основе географического расположения абонента Lync.
