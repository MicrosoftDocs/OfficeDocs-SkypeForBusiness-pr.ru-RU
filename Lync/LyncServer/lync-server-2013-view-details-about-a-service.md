# Просмотр сведений о службе в Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-21_

Для просмотра сведений о каждой службе, запущенной на определенном компьютере в топологии, можно использовать панель управления Lync Server. Панель управления предоставляет сведения о состоянии каждой службы, а также дополнительные данные о службе, например сведения о подключенных базах данных, портах и зависимых службах.

## Просмотр сведений о службе

1.  Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи, назначенной любой предварительно заданной административной роли Lync Server 2013. Дополнительные сведения о предварительно заданных административных ролях, которые доступны в Lync Server 2013, см. в разделе [Планирование контроля доступа на основе ролей в Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md).

2.  Откройте окно браузера и введите URL-адрес для администрирования, чтобы открыть панель управления Lync Server. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Lync Server см. в разделе [Открытие средств администрирования Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  В левой панели навигации щелкните **Топология** и затем щелкните **Состояние**.

4.  На странице **Состояние** выполните сортировку списка или найдите требуемый компьютер.

5.  Щелкните **Свойства**.

6.  В окне **Просмотр сведений о компьютере** отсортируйте список служб при необходимости и выберите службу, которую необходимо просмотреть.

7.  Выполните следующие действия:
    
      - Чтобы просмотреть последнее состояние определенной службы, щелкните **Получить сведения о состоянии службы**.
    
      - Чтобы просмотреть сведения об определенной службе, щелкните **Свойства** и затем нажмите кнопку **Закрыть**.
    
      - Чтобы получить список компьютеров, используемых в топологии, нажмите кнопку **Закрыть**.
