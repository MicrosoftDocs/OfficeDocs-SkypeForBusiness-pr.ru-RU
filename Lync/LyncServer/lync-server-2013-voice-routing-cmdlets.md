# <a name="voice-routing-cmdlets-in-lync-server-2013"></a>Командлеты голосовой маршрутизации в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-06-26_

Голосовые маршруты содержат инструкции, которые сообщают Microsoft Lync Server 2013 о том, как перенаправлять звонки от пользователей голосовой почты на номера телефонов в коммутируемой телефонной сети (КТСОП) или в частной среде обмена филиалами (АТС).

<div>
