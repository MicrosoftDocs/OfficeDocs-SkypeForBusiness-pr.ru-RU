## <a name="public-instant-messaging-clients-and-lync-2013"></a>Общедоступные клиенты для обмена мгновенными сообщениями и Lync 2013

Если вы настроили сервер для общедоступной службы обмена мгновенными сообщениями, Lync поддерживает следующие возможности: общедоступные сети обмена мгновенными сообщениями. Состояние присутствия фильтруется для тех состояний присутствия, которые поддерживаются клиентом общедоступной службы обмена мгновенными сообщениями. Подробные сведения можно найти в разделе [Планирование подключений к службе обмена мгновенными сообщениями в Lync server 2013](lync-server-2013-planning-for-public-instant-messaging-connectivity.md) в документации по планированию и [Управление политикой внешнего доступа в Lync Server 2013](lync-server-2013-manage-external-access-policy-for-your-organization.md) в документации по эксплуатации.

Кроме того, функция интеграции КСМПП в Lync Server 2013 позволяет пользователям обмениваться мгновенными сообщениями и сведениями о присутствии с пользователями общедоступных служб обмена МГНОВЕНными сообщениями, в том числе с помощью Google разговора. Подробности можно найти [в разделе Планирование Федерации протоколов КСМПП в Lync Server 2013](lync-server-2013-planning-for-extensible-messaging-and-presence-protocol-xmpp-federation.md) в документации по планированию.
