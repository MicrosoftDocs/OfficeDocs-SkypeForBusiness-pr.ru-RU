# <a name="server-hardware-platforms-for-lync-server-2013"></a>Аппаратные серверные платформы для Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

</div>

_**Последнее изменение темы:** 2016-07-28_

Для серверных ролей и компьютеров Lync Server 2013, на которых запущены средства администрирования Lync Server, требуется 64 — разрядное оборудование.

Конкретное оборудование, используемое для развертывания Lync Server 2013, может различаться в зависимости от требований к размеру и использованию. В данном разделе описывается рекомендованное оборудование. Несмотря на то, что это рекомендации, а не требования, использование оборудования, не соответствующего рекомендациям, может привести к серьезным проблемам с производительностью и другим сложностям.

<div>

## <a name="recommended-hardware-platform"></a>Рекомендуемая аппаратная платформа

Для оптимальной производительности рекомендуется запускать Lync Server на серверах с оборудованием, удовлетворяющим требованиям, описанным в следующей таблице. Использование менее мощного оборудования может привести к проблемам функциональности или низкой производительности. Обратите внимание на то, что эти требования выше, чем в предыдущих версиях Lync Server, в основном потому что в Lync Server 2013 все серверы переднего плана работают под управлением SQL Server.

<div>


> [!NOTE]  
> Объединение сетевых карт поддерживается и должно быть прозрачным для Lync Server. Дополнительные сведения: <A href="https://go.microsoft.com/fwlink/p/?linkid=389910">Communications Server или Lync Server, а также объединение сетевых адаптеров</A>.



</div>
