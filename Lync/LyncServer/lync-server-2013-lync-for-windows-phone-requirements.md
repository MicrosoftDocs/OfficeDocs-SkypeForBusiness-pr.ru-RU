# <a name="lync-for-windows-phone-requirements-in-lync-server-2013"></a>Требования Lync для Windows Phone в Lync Server 2013

_**Последнее изменение темы:** 2013-02-20_

Microsoft Lync 2013 для Windows Phone обеспечивает обмен мгновенными сообщениями, расширенные сведения о присутствии и телефонию пользователей в вашей организации, которые подключаются к смартфону или мобильному устройству Windows профессиональный. Мобильные устройства позволяют пользователям расширять доступ к Lync 2013. В этом разделе описываются вопросы планирования для Lync 2013 для Windows Phone, которые включают определение необходимых условий и технических требований, обязательные компоненты и руководство по развертыванию.

<div>

## <a name="lync-for-windows-phone-prerequisites"></a>Предварительные требования Lync для Windows Phone

Ниже приведены необходимые условия для Lync 2013 для Windows Phone.

  - Windows Phone 8, коденамед "Аполло" или "Последняя версия".

  - На устройстве с Windows Phone должны быть доступны последние обновления от корпорации Майкрософт. Дополнительные сведения см. в статье журнал обновлений Windows Phone 8 по адресу [https://go.microsoft.com/fwlink/p/?LinkID=281961](https://go.microsoft.com/fwlink/p/?linkid=281961) .

  - На устройстве должно быть не более 12 МБ свободного места на диске.

  - Пользователь должен иметь план голосовой связи и данных от оператора.
