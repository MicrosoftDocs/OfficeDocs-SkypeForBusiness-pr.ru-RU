# Сводка по сертификатам — обратный прокси-сервер в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

Требования к сертификатам для обратного прокис-сервера намного проще, чем требования для сервер. В приведенной блок-схеме представлены обязательные требования. В сопутствующей таблице приводятся типичные имена субъектов сертификатов и альтернативные имена субъектов, связанные с сценариями, которые мы рассматривали в обсуждениях сервер. Для получения более подробных сведений о сценариях сервер см. в разделе [Сценарии доступа внешних пользователей в Lync Server 2013](lync-server-2013-scenarios-for-external-user-access.md).
