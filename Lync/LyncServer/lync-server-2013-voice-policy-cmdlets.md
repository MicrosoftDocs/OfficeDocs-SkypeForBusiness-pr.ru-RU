# <a name="voice-policy-cmdlets-in-lync-server-2013"></a>Командлеты голосовой политики в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2013-02-18_

Управление голосовой связью включает в себя настройку таких параметров, как политики голосовой связи и абонентские группы, а также Связывание политик голосовой связи с голосовыми маршрутами. Командлеты, связанные с управлением политиками голосовой связи, можно использовать для настройки таких функций, как одновременный звонок (возможность использовать второй телефонный звонок каждый раз, когда кто-то звонит на рабочий телефон), переадресация звонков и подключение к телефону.
