# <a name="managing-call-management-features-in-lync-server-2013"></a>Управление функциями управления звонками в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-12-18_

Функции управления вызовами в корпоративной голосовой связи управляют тем, каким образом происходит маршрутизация входящих вызовов и ответ на них. Lync Server 2013 предоставляет следующие функции управления звонками:

  - **Парковки вызовов:** Позволяет голосовым пользователям временно приостановить вызов, а затем выбрать его на том же телефоне или на другом телефоне.

  - **Подбор групп:** Позволяет пользователям получать вызовы, которые вызывают звонок для других пользователей, набираемый номер группы ответа на звонки.

  - **Группа ответа:** Направляет входящие вызовы в группы агентов с помощью сервисных групп или вопросов и ответов интерактивного речевого ответа (IVR).

  - **Извещение:** Воспроизводит сообщение для вызовов, выполненных с неназначенным номером, или направляет вызов в другое место или обоими способами.

В этом разделе описывается, как управлять этими функциями управления звонками в развертывании корпоративной голосовой связи.
