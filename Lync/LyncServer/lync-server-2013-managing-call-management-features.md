  - <span data-ttu-id="b5853-107">**Подбор групп:** Позволяет пользователям получать вызовы, которые вызывают звонок для других пользователей, набираемый номер группы ответа на звонки.</span><span class="sxs-lookup"><span data-stu-id="b5853-107">**Group Pickup:** Enables users to pick up calls that are ringing for other users by dialing a call pickup group number.</span></span>
