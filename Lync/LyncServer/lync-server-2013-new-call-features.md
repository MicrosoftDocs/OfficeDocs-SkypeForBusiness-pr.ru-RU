# <a name="new-call-features-in-lync-server-2013"></a>Новые функции звонков в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-10_

Lync Server 2013 предоставляет значительно более широкий спектр параметров конфигурации для переадресации вызовов и одновременных звонков. Например, если организации требуется запретить внешнее перенаправление входящих звонков в ТСОП, администратор может применить специальную политику голосовой связи для реализации такого ограничения.

Кроме того, делегаты теперь могут настроить одновременные звонки на их мобильные устройства для входящих вызовов для их руководителей. Это предоставляет делегатам высокую гибкость, что позволяет им отвечать на вызовы от лица руководителя без привязки к стационарному телефону.
