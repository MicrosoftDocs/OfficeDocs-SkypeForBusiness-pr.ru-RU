# Новые функции звонков в Lync Server 2013

 

_**Дата изменения раздела:** 2012-10-10_

Lync Server 2013 предоставляет значительно более широкий выбор вариантов конфигурации для переадресации звонков. Например, если организации требуется запретить внешнее перенаправление входящих звонков в ТСОП, администратор может применить специальную политику голосовой связи для реализации такого ограничения.

Кроме того, делегаты теперь могут настроить одновременные звонки на их мобильные устройства для входящих вызовов для их руководителей. Это предоставляет делегатам высокую гибкость, что позволяет им отвечать на вызовы от лица руководителя без привязки к стационарному телефону.
