# <a name="troubleshooting-enterprise-voice-cmdlets-in-lync-server-2013"></a>Устранение неполадок командлетов корпоративной голосовой связи в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-06-20_

Настройка корпоративной голосовой связи в рамках реализации Microsoft Lync Server 2013 включает в себя создание маршрутов, политик и правил, которые должны работать вместе, чтобы обеспечить выполнение входящих и исходящих вызовов должным образом. Командная консоль Lync Server включает командлеты, которые можно использовать для тестирования подключений и путей, а также для устранения неполадок, которые могут возникнуть во время реализации.
