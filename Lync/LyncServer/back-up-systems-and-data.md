# <a name="back-up-systems-and-data"></a>Резервное копирование систем и данных

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-09-18_

Прежде чем приступить к миграции на Lync Server 2013, настоятельно рекомендуется выполнить полную архивацию системы и документировать существующую систему, включая учетные записи пользователей, размещенные в каждом пуле, чтобы вернуться на Lync Server 2010, если он становится необходимой. Для резервного копирования и восстановления данных, параметров и систем доступны различные инструменты и программы.

Подробные сведения и инструкции можно найти в разделе [резервное копирование и восстановление сервера Lync Server 2010](http://go.microsoft.com/fwlink/p/?linkid=265417) .
