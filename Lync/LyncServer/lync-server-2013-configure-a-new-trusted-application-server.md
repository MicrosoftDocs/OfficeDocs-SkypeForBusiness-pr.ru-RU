# Настройка нового доверенного сервера приложений в Lync Server 2013

 

_**Дата изменения раздела:** 2016-12-08_

Доверенные приложения — это приложения, созданные на основе SDK для Microsoft Unified Communications Managed API (UCMA) 3.0, который является доверенным для Microsoft Lync Server 2013. Дополнительные сведения о приложениях UCMA см. в разделе "Документация по пакету SDK для Unified Communications Managed API 3.0 Core" на веб-странице [http://go.microsoft.com/fwlink/?linkid=210320\&clcid=0x419](http://go.microsoft.com/fwlink/?linkid=210320%26clcid=0x419).

Сведения о настройке Microsoft Outlook Web Access (OWA) и Lync Server 2013 см. в разделе "Настройка интеграции Outlook Web App и Lync Server 2010" документации Microsoft Exchange Server 2013.

Чтобы успешно опубликовать, включить или отключить топологию при добавлении или удалении роли сервера, необходимо войти в систему с учетной записью члена групп RTCUniversalServerAdmins или "Администраторы домена". Кроме того, вы можете делегировать соответствующие разрешения и права администратора для добавления ролей сервера. Дополнительные сведения см. в разделе [Делегирование разрешений на установку в Lync Server 2013](lync-server-2013-delegate-setup-permissions.md) документации по развертыванию. Для остальных изменений конфигурации требуется только членство в группе RTCUniversalServerAdmins.

## Настройка сервера доверенных приложений

1.  Войдите на компьютер, где установлен построитель топологий, с использованием учетной записи, входящей в группу администраторов домена и группу RTCUniversalServerAdmins.

2.  Запустите построитель топологий: нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы** и **Microsoft Lync Server 2013** и щелкните элемент **Построитель топологий Lync Server**.

3.  Выберите **Download topology from existing deployment** (Загрузить топологию из существующего развертывания) и затем нажмите кнопку **ОК**.

4.  В диалоговом окне **Save Topology As** (Сохранение топологии) выберите требуемый файл топологий и затем нажмите кнопку **Save** (Сохранить).

5.  В области слева щелкните правой кнопкой мыши пункт **Серверы доверенных приложений** и выберите команду **Создать пул доверенных приложений**.

6.  В поле **Pool FQDN** (Полное доменное имя пула) введите полное доменное имя пула доверенных приложений, укажите параметры пула (пул из одного компьютера или нескольких компьютеров) и затем нажмите кнопку **Next** (Далее).

7.  На странице **Select the next hop** (Выбор следующего прыжка) выберите Lync Server 2013 переднего плана.

8.  Нажмите кнопку **Finish** (Готово).

9.  Выберите верхний узел **Lync Server 2013** и затем в меню **Действия** выберите пункт **Опубликовать**.
    
    **Пул доверенных приложений** успешно создан и связан с требуемым переднего плана.
