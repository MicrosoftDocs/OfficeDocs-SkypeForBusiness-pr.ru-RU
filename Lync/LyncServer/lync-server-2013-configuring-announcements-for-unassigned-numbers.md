# Настройка объявлений для неназначенных номеров в Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-11_

"Объявление" – это функция системы корпоративной голосовой связи, которая позволяет настраивать процедуры обработки вызовов, поступающих на неприсвоенные расширения (расширения, которые принадлежат организации, но не назначены пользователю или телефону). Например, можно настроить эту функцию таким образом, чтобы при выполнении вызовов на неприсвоенные номера воспроизводилось сообщение и/или эти вызовы передавались на другое назначение.

"Объявление" устанавливается как функция "Группа ответа" сервере переднего плана или как Сервер Standard Edition при развертывании системы корпоративной голосовой связи. Для настройки оповещений следует загрузить аудиофайлы или настроить преобразование текста в речь и сконфигурировать таблицу неприсвоенных номеров.

В этом разделе приведена процедура настройки оповещений Lync Server. Предполагается, что вы уже изучили разделы по планированию, относящиеся к оповещениям, и развернули сервер Enterprise Edition или Standard Edition с системой корпоративной голосовой связи.
