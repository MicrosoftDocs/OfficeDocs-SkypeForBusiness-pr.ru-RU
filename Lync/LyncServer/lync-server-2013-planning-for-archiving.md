# Планирование архивации в Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-28_

К корпорациям и другим организациям применяется все больше отраслевых и государственных нормативов, требующих сохранения определенного типа коммуникаций. Если в вашей организации используются такие требования, вы можете применять компонент архивации в Lync Server 2013 для архивации коммуникаций мгновенных сообщений и конференций (собраний), отправленных через Lync Server, для поддержки некоторых требований соответствия.
