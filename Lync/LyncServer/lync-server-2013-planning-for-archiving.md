# <a name="planning-for-archiving-in-lync-server-2013"></a>Планирование архивации в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-28_

К корпорациям и другим организациям применяется все больше отраслевых и государственных нормативов, требующих сохранения определенного типа коммуникаций. Если в организации имеются такие требования, вы можете использовать архивацию в Lync Server 2013 для архивации обмена мгновенными сообщениями и обмена мгновенными сообщениями, отправляемых с помощью Lync Server, чтобы обеспечить поддержку некоторых требований соответствия требованиям.
