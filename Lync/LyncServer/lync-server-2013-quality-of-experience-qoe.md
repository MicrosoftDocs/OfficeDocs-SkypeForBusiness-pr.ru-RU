# <a name="quality-of-experience-qoe-in-lync-server-2013"></a>Качество работы (QoE) в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-11-01_

Служба отслеживания качества взаимодействия (QoE) записывает числовые данные, которые показывают качество мультимедиа-данных, и сведения об участниках, именах устройств, драйверах, IP-адресах и типах конечных точек, использованных во время звонка или сеанса. При установке Lync Server 2013 Вы также устанавливаете предопределенную коллекцию глобальных параметров конфигурации для QoE. Используйте разделы этой главы для настройки значений QoE.
