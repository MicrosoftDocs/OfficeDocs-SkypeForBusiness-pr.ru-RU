# <a name="port-requirements-for-lync-server-2013"></a>Требования к портам для Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-03-27_

Lync Server требует, чтобы определенные порты в брандмауэре были открыты. Кроме того, если в организации развернут протокол IPsec, то он должен быть отключен в диапазоне портов, используемых для доставки звука, видео и панорамного видео.
