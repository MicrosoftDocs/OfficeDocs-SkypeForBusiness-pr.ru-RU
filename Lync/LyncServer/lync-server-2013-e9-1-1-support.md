<span data-ttu-id="f3a37-104">Lync Server 2013 поддерживает улучшенные 9-1-1 (E9-1-1) в составе корпоративного развертывания.</span><span class="sxs-lookup"><span data-stu-id="f3a37-104">Lync Server 2013 supports Enhanced 9-1-1 (E9-1-1) as part of an enterprise deployment.</span></span> <span data-ttu-id="f3a37-105">E9-1-1 — это функция, которая связывает телефонный номер вызывающего абонента с гражданским (то есть) адресом получателя.</span><span class="sxs-lookup"><span data-stu-id="f3a37-105">E9-1-1 is an emergency notification feature that associates the calling party’s telephone number with a civic (that is, a street) address.</span></span> <span data-ttu-id="f3a37-106">Поддержка E9-1-1 доступна только в США.</span><span class="sxs-lookup"><span data-stu-id="f3a37-106">E9-1-1 support is available only in the United States.</span></span>
