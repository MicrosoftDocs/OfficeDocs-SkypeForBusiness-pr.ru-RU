# Поддержка E9-1-1 в Lync Server 2013

 

_**Дата изменения раздела:** 2012-09-21_

Lync Server 2013 поддерживает Enhanced 9-1-1 (E9-1-1) как часть корпоративного развертывания. E9-1-1 ? это компонент экстренного уведомления, который связывает номер телефона вызывающей стороны с городским адресом. Поддержка E9-1-1 доступна только в США.

Чтобы поддерживать E9-1-1 как часть развертывания Lync Server 2013, необходимо получить службу маршрутизации E9-1-1 от сертифицированного поставщика услуг экстренной связи или использовать шлюз ELIN. Поставщик услуг экстренной связи или оператор связи маршрутизируют экстренные вызовы от Lync Server 2013 в соответствующую точку PSAP на основе сведений о расположении в вызове. Дополнительные сведения о поддержке E9-1-1 см. в разделе [Планирование чрезвычайных служб (E9-1-1) в Lync Server 2013](lync-server-2013-planning-for-emergency-services-e9-1-1.md) документации по планированию.
