# <a name="navigating-the-planning-tool-in-lync-server-2013"></a>Навигация в средстве планирования в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2013-02-21_

Вы переходите к инструменту Microsoft Lync Server 2013, планирование с помощью сочетания панели инструментов, специальных кнопок и ссылок, а также контекстных областей. Контекстно-зависимые области предоставляют сведения о проектировании для планирования и мощности, относящиеся к параметрам выделения на определенной странице.

При запуске средства планирования дизайнер сначала видит страницу " **Добро пожаловать в средство планирования для Microsoft Lync Server 2013** ".

![Страница приветствия средства планирования](images/Gg558601.ff5b72e6-bcf0-49e9-8784-3636fe8187c5(OCS.15).jpg "Страница приветствия средства планирования")

На странице приветствия разработчик выбирает **Начало работы**, **проектирование сайтов**и **Отображение**. Для получения дополнительных сведений ознакомьтесь со статьей [Создание первоначальной структуры топологии для Lync Server 2013](lync-server-2013-create-the-initial-topology-design.md).

В верхней части средства планирования представлена панель инструментов, предоставляющая простой доступ к часто используемым функциям. Панель инструментов отображается здесь для справки, а каждая функция обсуждается в связанных разделах.

![Панель инструментов средства планирования](images/Gg558601.a008ddd1-b73d-4406-9d4b-df68bed9906e(OCS.15).jpg "Панель инструментов средства планирования")

В левой части средства планирования есть раздел внешние ссылки. Отсюда дизайнер имеет простой доступ к сведениям о планировании и развертывании, а также другие технические ресурсы, такие как обучающие, технические блоги, форумы и другие загружаемые ресурсы. Кроме того, в разделе Внешние ссылки можно отправить ссылку на команду средства планирования Lync Server 2013.

![Диалоговое окно "внешние ссылки" средства планирования](images/Gg558601.76959057-8eb2-4158-b1b3-585cca80be7e(OCS.15).jpg "Диалоговое окно "внешние ссылки" средства планирования")

Область действий, зависящая от контекста, отображается на многих страницах средства планирования. Панель действий предоставляет дизайнеру простой доступ к основным разделам топологии. Ссылки, доступные в области действий, изменяются в зависимости от уровня детализации в топологии. Область действий доступна после завершения вопросов и отображения топологии. В область действия включено раздел Обзор, в котором отображаются числа, введенные разработчиком в рамках процесса собеседования. Обзор контекстно связан с отображаемыми данными.

Кроме того, сведения о оборудовании отображаются в обзоре в области действия. В разделе Конфигурация оборудования отображается список требований к оборудованию, рекомендуемых текущей топологией.

![Область действий в средстве планирования](images/Gg558601.9679d8fd-4de8-4a5a-bfcf-699da9aa7283(OCS.15).jpg "Область действий в средстве планирования")
