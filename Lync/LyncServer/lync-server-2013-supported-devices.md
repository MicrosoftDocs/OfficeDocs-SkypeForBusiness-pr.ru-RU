# <a name="supported-devices-in-lync-server-2013"></a>Поддерживаемые устройства в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

_**Последнее изменение темы:** 2012-10-08_

Lync Server поддерживает все телефоны, перечисленные в следующей таблице. Обратите внимание, что не все поддерживаемые телефоны могут запускать Lync Phone Edition (некоторые из них могут работать только в более ранних версиях). Используйте таблицу, чтобы узнать, какие телефоны могут запускать Lync Phone Edition, и где получить подробные сведения обо всех поддерживаемых телефонах.

<div>


> [!NOTE]  
> Поддерживаются три типа телефонов: стационарные телефоны (IP-устройства сотовой связи или USB-устройства, предназначенные для использования сотрудниками в офисе на рабочем месте), устройства для конференц-связи (IP-телефоны или USB-телефоны, предназначенные для использования в комнатах для собраний) и телефонные телефоны (IP-телефоны, предназначенные для использования в общих областях, такие как залов, и общими рабочими областями, которые можно настроить для предоставления различных наборов компонентов Lync Phone Edition.



</div>
