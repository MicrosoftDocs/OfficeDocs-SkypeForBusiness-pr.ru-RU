# <a name="new-im-and-presence-features-in-lync-server-2013"></a>Новые функции обмена мгновенными сообщениями и присутствия в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-10-19_

Microsoft Lync Server 2013 добавляет следующие новые функции обмена мгновенными сообщениями и присутствия, чтобы расширить возможности Lync для пользователей.

  - Если в организации также работает Exchange 2013, пользователи могут использовать единое хранилище контактов. Пользователи могут управлять своими контактами в Outlook 2013, Outlook Web App, а также в Lync 2013.

  - Пользователи могут обмениваться мгновенными сообщениями и сведениями о присутствии с пользователями общедоступных служб обмена мгновенными сообщениями, которые используют расширенный протокол обмена сообщениями и присутствия, например Google говорите, из-за возможности **интеграции XMPP** в Lync Server 2013. Интеграция с XMPP встроена в серверы переднего плана и пограничные серверы, поэтому вы можете включить и настроить данную возможность.
