# <a name="new-im-and-presence-features-in-lync-server-2013"></a>Новые функции обмена мгновенными сообщениями и сведениями о присутствии в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-19_

Microsoft Lync Server 2013 добавляет следующие новые возможности обмена мгновенными сообщениями (IM) и присутствия, чтобы расширить возможности Lync для пользователей.

  - Если в вашей организации также работает Exchange 2013, пользователи смогут пользоваться единым хранилищем контактов. Пользователи могут управлять своими контактами в Outlook 2013, Outlook Web App, а также в Lync 2013.

  - Пользователи могут обмениваться мгновенными сообщениями и сведениями о присутствии с пользователями общедоступных служб обмена мгновенными сообщениями, в том числе с помощью протокола Google чата, благодаря возможности **интеграции КСМПП** в Lync Server 2013. Интеграция КСМПП, встроенная в сервер переднего плана и пограничные серверы, и ее можно включить и настроить для разрешения этой функции.
