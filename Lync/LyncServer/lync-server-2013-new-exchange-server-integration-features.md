# <a name="new-exchange-server-integration-features-in-lync-server-2013"></a>Новые функции интеграции с Exchange Server в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Последнее изменение темы:** 2012-09-24_

Lync Server 2013 поддерживает новые функции при развертывании вместе с Microsoft Exchange Server 2013, как описано в следующем списке. Для работы каждой из этих функций необходимо развернуть Lync Server 2013 и Exchange 2013.

  - Вы можете использовать **единое хранилище контактов**, в котором список контактов Lync хранится в Exchange 2013, и вы можете управлять хранилищем контактов в Lync 2013, Outlook 2013 и Outlook Web App.

  - Можно использовать **фотографии высокого разрешения** для контактов. Фотографии с 648x648 пикселами хранятся в Exchange 2013 и становятся доступными для клиентов, включая Lync 2013, Outlook 2013, Microsoft Lync Web App и Outlook Web App.

  - Вы можете включить **интеграцию с архивацией Lync**, которая интегрирует архивацию lync Server 2013 в функцию хранения на месте Exchange 2013, для пользователей, размещенных на сервере Exchange 2013, что позволяет использовать один общий интерфейс администраторам для обеспечения соответствия требованиям и обнаружения электронных данных.
