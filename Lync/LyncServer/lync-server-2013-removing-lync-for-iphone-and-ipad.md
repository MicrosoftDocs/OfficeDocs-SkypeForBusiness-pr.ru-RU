# Удаление Lync для iPhone и iPad

 

_**Дата изменения раздела:** 2013-02-19_

Чтобы удалить с устройства Lync 2013 для iPhone или Microsoft Lync 2010 для iPad, выполните следующие действия:

1.  Коснитесь и удерживайте значок Lync на начальном экране мобильного устройства.

2.  Когда иконки начнут дрожать и появится символ **X**, коснитесь **X** для удаления приложения.
