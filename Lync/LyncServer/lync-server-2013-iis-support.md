_**Последнее изменение темы:** 2012-06-29_

Для некоторых компонентов Lync Server 2013 требуются информационные службы Интернета (IIS). Поддерживаются следующие версии служб IIS:

  - IIS 8.0

  - IIS 7.5

Для получения дополнительных сведений о требованиях IIS для компонентов Lync Server 2013, обратитесь к разделу [требования к службам IIS в Lync server 2013](lync-server-2013-internet-information-services-iis-requirements.md) в документации по планированию.
