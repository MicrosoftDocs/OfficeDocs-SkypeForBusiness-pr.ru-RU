# <a name="registrar-and-director-cmdlets-in-lync-server-2013"></a>Командлеты регистраторов и режиссеры в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-10-09_

Регистраторы и режиссеры используются для проверки подлинности запросов на вход и для хранения сведений о состоянии пользователей и доступности. Командлеты регистраторов и режиссер позволяют управлять параметрами конфигурации для этих серверов.
