# Контрольный список развертывания для E9-1-1 в Lync Server 2013

 

_**Дата изменения раздела:** 2015-03-09_

Чтобы разработать эффективный план для Enhanced 9-1-1 (E9-1-1), обязательно включите в него следующие требования по развертыванию:

  - необходимые компоненты для развертывания E9-1-1;

  - шаги, необходимые для развертывания E9-1-1;

## Необходимые компоненты для развертывания E9-1-1

Перед развертыванием E9-1-1 необходимо развернуть внутренние серверы Lync Server, в том числе управления, переднего плана или сервер Standard Edition, один или несколько Серверы-посредники или серверов-посредников и клиенты Lync Server. Кроме того, для развертывания E9-1-1 требуется SIP-магистраль до квалифицированного поставщика услуг E9-1-1 или шлюз ELIN до телефонной сети общего пользования (ТСОП). Lync Server поддерживает использование поставщиков услуг E9-1-1 только в США.
