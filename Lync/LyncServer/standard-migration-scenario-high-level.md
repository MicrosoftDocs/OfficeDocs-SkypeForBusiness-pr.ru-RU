# Стандартный сценарий миграции — высокоуровневый

 

_**Дата изменения раздела:** 2013-01-30_

Используйте следующие элементы в качестве отправной точки при миграции групповой беседы Lync Server 2010 или групповой беседы Office Communications Server 2007 R2 на сервер сохраняемого сеанса беседы сервера Lync Server 2013, . Стандартный путь миграции для сервера Lync Server 2013 следующий.

  - В организации имеется предварительно развернутый сервер групповой беседы Lync Server 2010 или групповая беседа Office Communications Server 2007 R2, и требуется развернуть сервер сохраняемого сеанса беседы сервера Lync Server 2013.

  - Разверните сервер Lync Server 2013, а затем разверните пул (или пулы) серверов сохраняемого сеанса беседы.

  - Подготовьте и спланируйте миграцию комнат сохраняемый сеанс беседы, а также определите подходящее время остановки системы для выполнения миграции.

  - Запустите командлеты Windows PowerShell (**Export-CsPersistentChatData** и **Import-CsPersistentChatData**) для переноса содержимого на сервер сохраняемого сеанса беседы.

  - Убедитесь, что перенос прошел успешно.

  - Выведите из эксплуатации устаревшее развертывание.

  - Настройте сохраняемого сеанса беседы, чтобы устаревшие клиенты могли подключаться к Lync Server 2013 и сохраняемого сеанса беседы. Это необходимо, так как для развертывания новых клиентов нужно время, а вам нужно предоставить текущим пользователям с устаревшими клиентами доступ к комнатам чата как можно быстрее.

  - Разверните новых клиентов, продолжая обеспечивать доступ к комнатам чата с устаревшим клиентом групповых бесед.
