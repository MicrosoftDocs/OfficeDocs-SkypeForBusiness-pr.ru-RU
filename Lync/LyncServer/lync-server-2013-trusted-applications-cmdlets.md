_**Последнее изменение темы:** 2012-06-20_

Доверенное приложение — это приложение, разработанное сторонним производителем, которое получает состояние "доверенное" для работы в составе Microsoft Lync Server 2013, но не является встроенной частью продукта. Lync Server 2013 предоставляет командлеты, которые можно использовать для настройки доверенных приложений и управления ими.
