# <a name="mobile-client-comparison-tables-for-lync-server-2013"></a>Таблицы сравнения мобильных клиентов в Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2015-09-21_

В приведенных ниже таблицах сравниваются функции и возможности мобильных клиентов Lync 2013 и клиента Lync 2013 для настольных систем в следующих категориях:

  - вход, push-уведомления и общие функциональные возможности;

  - расширенные сведения о присутствии;

  - контакты и группы контактов;

  - обмен мгновенными сообщениями;

  - Аудио-и видеофайлы в Lync

  - конференц-связь

  - телефония;

  - внешние пользователи;

  - архивирование и соответствие требованиям.

В этих таблицах указываются возможности, доступные пользователям Lync в локальном развертывании Lync Server 2013. Те же возможности доступны в Skype для бизнеса Online и Microsoft Office 365, если только не указано иное в сносках таблицы.
