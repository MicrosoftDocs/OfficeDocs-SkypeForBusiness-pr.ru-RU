# Администрирование серверов после миграции

 

_**Дата изменения раздела:** 2012-09-29_

В общем случае для управления сервером необходимо использовать средство администрирования, соответствующее его версии. На одном компьютере нельзя установить Lync Server 2013 и средства администрирования Office Communications Server 2007 R2. Кроме того, панель управления Lync Server 2013 не устанавливается на каждый сервер автоматически. Чтобы установить панель управления Lync Server 2013, выполните процедуру, описанную в разделе [Установка средств администрирования Lync Server 2013](lync-server-2013-install-lync-server-administrative-tools.md) в документации по развертыванию.
