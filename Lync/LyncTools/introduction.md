Средство нагрузки и производительности Lync Server 2013 поддерживает создание и Федерацию нагрузки между пулами с помощью расширенной конфигурации.

Кроме того, средство не моделирует нагрузку для следующих клиентов:

  - Office Live Meeting 2007

  - Lync 2013 сохраняемый чат

В результате средство нагрузки и производительности Lync Server 2013 не будет поддерживать тестирование следующих компонентов:

  - Lync 2013 сохраняемый чат

  - Сценарии интеграции с Exchange
