# <a name="interpreting-the-results"></a>Интерпретация результатов

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2013-02-24_

В средстве нагрузки и производительности Lync Server 2013 (Линкперфтул. exe) есть множество счетчиков, которые можно использовать для понимания того, что делает клиент и при возникновении проблем.

<div>

## <a name="client-counters"></a>Счетчики клиента

У каждого запущенного экземпляра Линкперфтул. exe есть отдельный экземпляр счетчиков. Каждый экземпляр именуется ИДЕНТИФИКАТОРом процесса.

Если клиенты перегружены, могут возникать проблемы. Чтобы предотвратить эти проблемы, выполните указанные ниже действия.

1.  Отслеживайте ресурсы ЦП и использование памяти на клиентских компьютерах. Если ЦП постоянно превышает 90%, сократите количество пользователей.

2.  Если объем памяти велик, вы можете столкнуться с проблемами, если файл подкачки слишком мал. Убедитесь, что зарядка фиксации не достигнута на компьютере. Если вы используете ограничения памяти, рекомендуем увеличить размер файла подкачки или уменьшить количество пользователей.
