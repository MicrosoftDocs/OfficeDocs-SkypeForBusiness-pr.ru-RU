# <a name="getting-started-with-your-microsoft-teams-upgrade"></a>Начало перехода на Microsoft Teams

Поздравляем с переходом от Skype для бизнеса в Microsoft Teams. Если вы только начинаете работать с Teams, вы уже используете Teams вместе со Skype для бизнеса или готовы к обновлению, мы хотели бы убедиться, что у вас есть все, что нужно для навигации в Teams по успешному пути.

## <a name="why-upgrade-to-microsoft-teams"></a>Зачем нужно переходить на Microsoft Teams?

Microsoft Teams расширяет возможности Skype для бизнеса, объединяя чат, собрания, звонки, совместной работы, интеграцию приложений и хранение файлов в одном интерфейсе. Этот новый Центральный центр для совместной работы помогает оптимизировать работу пользователей, повышая степень удовлетворенности пользователей и ускоряя бизнес-результаты. Мы постоянно расширяем возможности Teams, позволяющие общаться и совместно работать с новыми способами, разбивать организационные и географические барьеры, а эффективность процесса и принятия решений. Узнайте больше о преимуществах обновления до Teams в Форрестер отчете: [Общее количество экономических эффектов в Microsoft Teams](https://www.microsoft.com/en-us/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams-Infographic.pdf).  

## <a name="when-should-my-organization-upgrade-to-teams"></a>Когда следует обновлять организацию до Teams?

Skype для бизнеса Online будет прекращен до 31 июля 2021 г., после чего он больше не будет доступен или поддерживаться. Несмотря на то, что Skype для бизнеса Online будет полностью работоспособен до 31 июля 2021 г., корпорация Майкрософт рекомендует клиентам начать свое путешествие по обновлению прямо сейчас, позволяя получить достаточно времени для завершения обновления до наступления даты выбытия.  Ознакомьтесь с [](https://aka.ms/sfboannounce) нашим извещением о выбытиях, чтобы узнать больше.

## <a name="can-we-continue-to-use-skype-for-business-online-as-we-prepare-for-and-execute-our-upgrade-to-teams"></a>Можно ли продолжать использовать Skype для бизнеса Online в качестве подготовки к обновлению и выполнению нашего обновления для Teams?

Да, в Microsoft есть несколько вариантов для совместного использования Skype для бизнеса Online и Teams. Эти режимы совместного существования позволяют пользователям ознакомиться с Teams сегодня и ускорить переход на новую версию только для Teams. Дополнительные сведения можно найти в [https://aka.ms/SkypeToTeams-Coexist](https://aka.ms/SkypeToTeams-Coexist)разделе.
 
## <a name="how-is-microsoft-helping-customers-with-their-upgrade-to-teams"></a>Как корпорация Майкрософт помогает клиентам с помощью обновления до Teams? 

Корпорация Майкрософт считает своим долгом вносить ваши изменения из Skype для бизнеса в Teams, на которые вы успешно являетесь. Корпорация Майкрософт предоставляет надежное руководство, созданное на основе проверенной, сквозной платформы успешности обновления, разработанной для планирования и выполнения обновления до Teams. Наше руководство предназначено для каждого этапа путешествия по обновлению, если вы только приступись к планированию, уже используете команды в Skype для бизнеса или готовы к переходу на Teams.

Для упрощения планирования и реализации обновлений корпорация Майкрософт размещает бесплатные интерактивные семинары по планированию обновления, предназначенные для ознакомления с нашей платформой обновления, предоставления сведений о подготовке пользователей для Обновите и определите правильный путь Организации к Teams. Присоединяйтесь к нам по обновлению сеанса обзора [https://aka.ms/SkypeToTeamsPlanning](https://aka.ms/SkypeToTeamsPlanning)для планирования:.
 
## <a name="why-do-i-need-upgrade-guidance-cant-i-just-deletedecommission-skype-for-business"></a>Зачем нужны инструкции по обновлению? Не можете ли я просто удалить или списать Skype для бизнеса? 

Переход с Skype для бизнеса на Teams является более чем техническим вариантом миграции. Она представляет собой преобразование, в котором пользователи обмениваются данными и совместной работы, а также изменения не всегда просто. Оптимальный подход к обновлению должен решить технические аспекты вашего обновления, а также способствовать принятии пользователей и принятием поддержки групп, что повлияет на работу пользователей и реализацию результатов в деловой деятельности. 

## <a name="where-do-i-start-planning-for-teamsmy-upgrade-to-teams"></a>Где можно начать планирование Teams/My Upgrade для Teams? 

Начните путешествие с помощью нашей инфраструктуры успешности и связанных ресурсов. Это руководство служит основой для навигации в командировках из Skype для бизнеса в Teams.

[Узнайте о платформе обновления](upgrade-framework.md) и присоединитесь к нам в сеансе [обзорного планирования для динамического обновления](https://aka.ms/SkypeToTeamsPlanning) .

> [!IMPORTANT]
> Помните о том, что успешное обновление обеспечивает как техническую, так и пользовательскую готовность, поэтому не забудьте использовать руководство для навигации в Microsoft Teams.
