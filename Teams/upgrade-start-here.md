# <a name="getting-started-with-your-microsoft-teams-upgrade"></a>Приступая к работе с Microsoft группам обновления

## <a name="why-upgrade-to-microsoft-teams"></a>Зачем выполнять обновление группами Майкрософт?

Группами Майкрософт предлагает центрального сервера-концентратора для работы. Расширение возможностей Скайп для бизнеса, группами объединяет чата, собраний, в вызове, совместной работы, интеграции приложений и хранение файлов в один интерфейс упрощение способ пользователям выполнять задачи, в свою очередь, может помочь удовлетворенности пользователей диска и ускорить деятельности. Как существующие Скайп для клиента Business мы отправить приглашение к обновлению до группами Майкрософт, чтобы испытать этот полный набор возможностей обмена данными и совместной работы в одного клиентского интерфейса. 

## <a name="when-should-my-organization-upgrade-to-teams"></a>Когда следует обновить Моя организация по группам

Для оптимизации реализации преимущество рабочих групп, мы рекомендуем клиентам для обновления до группы, как можно скорее. Мы понять, что изменение может занять некоторое время, и даже в том случае, если вы еще не готово к только что еще обновления, можно запустить сегодня осознавать преимущество группами. Развертывание групп наряду с Скайп для бизнеса позволяет пользователям ознакомиться с группами теперь которого могут помочь упростить к обновлению до версии группами по времени. На самом деле наших клиентов, отображается согласованным образом: shift их связи действий к группам пользователей.  
 
## <a name="how-is-microsoft-helping-customers-with-their-upgrade-to-teams"></a>Как Microsoft помогает пользователям с помощью их обновления групп? 

Microsoft стремится успеха и обеспечивает надежное рекомендациями, построенные вокруг framework проверенных, начала до конца успешности обновления помогут спланировать и выполнить обновление для групп. Указаниям по предназначена для всех этапов обновления реализация как являются только что Приступая к работе с планирования, уже выполняется группами наряду с Скайп для бизнеса, так и подготовиться к переходу на обновление до группами.  
 
## <a name="why-do-i-need-upgrade-guidance-cant-i-just-deletedecommission-skype-for-business"></a>Почему требуется обновление рекомендации? Не удается я только что delete/выводить из эксплуатации Скайп для бизнеса? 

Обновление с Скайп для бизнеса группам больше, чем технические миграции. Представляет преобразования в том, как пользователи общения и совместной работы и изменения не всегда легко. Идеальный способа обновления следует устранить технические аспекты вашего обновления, а также поощрение приемочного и внедрения командами, управляющий реализацию результат взаимодействия и бизнес-положительное пользователя.
