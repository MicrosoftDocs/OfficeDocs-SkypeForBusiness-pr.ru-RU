# <a name="plan-for-microsoft-365-groups-when-creating-teams-in-microsoft-teams"></a>Планирование групп Microsoft 365 групп при создании групп в Microsoft Teams

При рассмотрении использования Microsoft 365 групп или создании команд подумайте, для чего будет использоваться команда, для кого должен быть доступ и какой результат она ожидать. Особое внимание уделите числу создаваемых каналов, так как слишком сильное дробление контента (то есть разнесение его по большому числу каналов) быстро утомляет людей.

Существует два сценария, которые требуют обсуждения планирования Microsoft 365 групп и их влияния на группы (или по) Microsoft Teams.

-   Во-первых, поскольку у клиентов могут быть инвестиции в Группы, в настоящее время мы поддерживаем как общедоступные, так и закрытые [группы.](./limits-specifications-teams.md)Количество поддерживаемых в настоящее время участников см. в Microsoft Teams. Как упоминалось выше, вы хотите управлять членством людей в команде с помощью клиента Teams, а не Центр администрирования Microsoft 365. В этом случае, если люди привыкли к цепочкам бесед в Microsoft 365 Groups, имеет смысл отметить, что беседа Группы фактически является электронной почтой, а не сообщением чата в канале Teams. Сообщите людям об этом различии и порекомендуйте освоить более гибкий формат сообщений чата в Teams вместо электронной почты на базе Outlook или OWA.

-   Во-вторых, для клиентов, у которых в Microsoft 365 не определено существующее Microsoft 365, вы можете создавать их с помощью Центр администрирования Microsoft 365, веб-Teams или классических клиентов. Как упоминалось выше, управляйте всеми будущими членством в группе Microsoft 365 с помощью Teams клиента. Так как членство в группе также определяет членство в Microsoft 365, следует подготовить пользователей к этому изменению.

## <a name="teams-respects-microsoft-365-groups-naming-policy"></a>Teams с соблюдением Microsoft 365 именования групп

Любая Microsoft 365 именования групп, настроенная администратором, будет применяться в Teams при создании или редактировании имен групп пользователями. Сюда входят такие аспекты, как обязательные префиксы или суффиксы, а также исключение запрещенных слов.

Чтобы узнать больше, ознакомьтесь с [Microsoft 365 именования групп в Teams.](https://support.office.com/article/Office-365-Groups-Naming-Policy-6ceca4d3-cad1-4532-9f0f-d469dfbbb552)
