# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>Аудио- и видео устройства USB, сертифицированные для Microsoft Teams

Программа сертификации периферийных устройств гарантирует, что периферийные устройства USB, такие как гарнитуры, динамики, веб-камеры и мониторы, обеспечивают высокое качество звука и видео. Сертифицированные устройства (подключаемый модуль для & воспроизведения) работают без дополнительной настройки и предлагают функцию управления звонками в Microsoft Teams и Skype для бизнеса. Эти продукты предоставляют большое впечатление для конечных пользователей и администраторов.

Чтобы изучить устройства, протестированных и сертифицированные для Microsoft Teams, посетите [Магазин Teams.](https://products.office.com/microsoft-teams/across-devices/devices)

Дополнительные сведения о поддержке управления звонками в Teams можно прочитать с помощью гарнитуры [в Teams.](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8)

> [!TIP]
> Ищете устройства, сертифицированные для Skype для бизнеса? Перена веб-приложение USB, сертифицированное для аудио- и видео устройств [Skype для бизнеса.](https://docs.microsoft.com/skypeforbusiness/certification/devices-usb-devices)

## <a name="understanding-certification-categories"></a>Категории сертификации

Пользователям не нужно заменять ранее сертифицированные устройства при переходе с Skype для бизнеса на Teams.  Все сертифицированные устройства поддерживаются как для пользователей Microsoft Teams, так и для пользователей Skype для бизнеса с несколькими различиями в функциональности.  Кроме того, сертификация позволяет дополнительно обозначение некоторых категорий устройств.

- **Skype для бизнеса сертифицирован с обновлением Teams** Устройства, сертифицированные на этом уровне, изначально сертифицированы для Skype для бизнеса. Чтобы предоставить пользователям как можно больше времени для жизни, корпорация Майкрософт вместе с производителями устройств обеспечивает как можно больше интеграции Teams с этими существующими устройствами без каких-либо изменений исходного оборудования. Кнопка *"Подключиться" действует* как кнопка Teams, когда не выполняется звонок, и может включать дополнительные функции со временем.  Устройства в этой категории могут иметь эмблему сертификации Teams в маркетинговых материалах.
- **Сертифицировано для Microsoft Teams** Устройства, сертифицированные на этом уровне, указывают последние устройства, сертифицированные для компрометации Teams. На этих устройствах есть все преимущества предыдущей сертификации, а также выделенная кнопка Teams с индикатором LED. При большинстве форм кнопка Teams является легкой тактильной кнопкой с логотипом Teams. Корпорация Майкрософт со временем продолжает работать с производителями устройств, чтобы обеспечить новую ценность с помощью обновлений по настройке.

### <a name="special-usage-designations"></a>Специальные обозначений использования

- **Premium Microphone для открытия Office** Все гарнитуры проверены на четкость звука, но открытый офис и другие шумные среды выполняют большую задачу: четко возьмитесь за голос пользователя гарнитуры, но не передают звук его микрофона.
- **Размер комнаты для собрания** Общие устройства, такие как динамики и камеры, имеют поддерживаемое расстояние использования и рекомендуемые расстояния могут быть разными производителями. Чтобы предоставить клиентам четкие рекомендации, все общие устройства тестироваться на сертификации на четко определенных размерах помещений и шумовых средах.

Просмотр [устройств, сертифицированных для Microsoft Teams.](https://products.office.com/microsoft-teams/across-devices/devices)

## <a name="end-user-experience"></a>Пользовательский интерфейс

### <a name="new-teams-certification-benefits"></a>Новые преимущества сертификации Teams

- Выделенная кнопка Teams и индикатор LED.
- Уведомления пользователей о событиях и оповещениях от клиента Teams (например, о начале собрания, голосовой почте или пропущенных звонках).
- Возможность активации клиента Teams или ответа на уведомление с помощью нажатия кнопки (например, перейти к собранию).
- Сертификация, предназначенная для использования размежещенности (новые функции устройств и клиентов, которые можно получить через обновление по фирменому ПО).

### <a name="core-certification-benefits"></a>Основные преимущества сертификации

- Автоматический выбор стандартного звукового устройства и рас приоритетов, если есть несколько периферийных устройств звука.
- Устройства автоматически регистрируются в клиенте Teams и готовы к использованию.
- Звуковые устройства (например, гарнитуры и динамики) предлагают:
  - Основные функции управления звонками на компьютерах с Windows и Mac. К основным средствам управления относятся ответы и вешать трубки, включать и отключать звук, а также громкость.
  - Широкополосый звук без эха, искажения или лишних сбоев. Отмена эха также поддерживается на разных устройствах.
- Видео устройства соответствуют требованиям для шума, цвета, детализации изображения, дрожаия, задержки, частоты кадров и поля зрения.

## <a name="administrator-experience"></a>Опыт администратора

- Подготовка не требуется
- На всех устройствах поддерживается обновление подержка, что позволяет улучшить новые функции и производительность.

Если вы изготовитель и хотите присоединиться к [](https://docs.microsoft.com/skypeforbusiness/certification/how-to-join) программе сертификации, см. статью "Как присоединиться, чтобы ознакомиться с требованиями и доступными программами".
