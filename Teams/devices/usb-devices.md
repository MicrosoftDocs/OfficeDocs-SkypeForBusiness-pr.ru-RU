# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>USB-аудио- и видеоустройства, сертифицированные для Microsoft Teams

Программа сертификации периферийных устройств гарантирует, что usb-периферийные устройства (например, гарнитуры, динамики, веб-камеры и мониторы) обеспечивают высокое качество звука или видео. Сертифицированные устройства работают (подключаются & play) без дополнительной настройки и предлагают управление звонками с помощью Microsoft Teams и Skype для бизнеса. Продукты обеспечивают отличное взаимодействие с конечными пользователями и администраторами.

Чтобы изучить устройства, протестированные и сертифицированные для Microsoft Teams, перейдите в [магазин Teams](https://products.office.com/microsoft-teams/across-devices/devices).

Дополнительные сведения о поддержке управления звонками в Teams см. в разделе [звонков с помощью гарнитуры в Teams](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8).

> [!TIP]
> Ищете устройства, сертифицированные для Skype для бизнеса? Перейдите на [USB-аудио- и видеоустройства, сертифицированные для Skype для бизнеса](/skypeforbusiness/certification/devices-usb-devices).

## <a name="understanding-certification-categories"></a>Основные сведения о категориях сертификации

Пользователям не нужно заменять ранее сертифицированные устройства при обновлении до Teams с Skype для бизнеса.  Все сертифицированные устройства поддерживаются для Microsoft Teams Skype для бизнеса пользователей с несколькими отличиями.  Сертификация также предоставляет дополнительные сведения для определенных категорий устройств.

- **Skype для бизнеса, сертифицированные с помощью устройств обновления Teams**, сертифицированных на этом уровне, изначально сертифицированы для Skype для бизнеса. Корпорация Майкрософт работала с производителями устройств, чтобы обеспечить наилучший интерфейс интеграции Teams без каких-либо изменений в исходном оборудовании. *Кнопка hookswitch* действует как кнопка Teams, если она не вызывается, и может включать дополнительные функции с течением времени.  Устройства в этой категории могут содержать эмблему сертификации Teams в маркетинговых материалах.
- **Сертифицировано для Microsoft Teams** Устройства, сертифицированные на этом уровне, указывают на новые сертифицированные устройства для использования Teams без компрометации. Эти устройства предлагают все преимущества предыдущей сертификации, а также выделенную кнопку Teams с индикатором индикатора. В большинстве форм-факторов кнопка Teams будет удобной для поиска тактильной кнопкой с логотипом Teams. Корпорация Майкрософт продолжает работать с производителями устройств, чтобы обеспечить новые возможности благодаря обновлениям встроенного ПО с течением времени.

### <a name="special-usage-designations"></a>Специальные параметры использования

- **Встроенные сертифицированные аудио периферийные устройства Bluetooth**  Большинство гарнитур Bluetooth и динамиков, сертифицированных для Microsoft Teams, предназначены для использования с USB-прямоугольником для подключения к компьютерам и узлам Mac. Многие пользователи запрашивают возможность прямого связывания устройства с узлом, так как у них недостаточно USB-портов или им не нужно помнить, что usb dongle нужно использовать при работе в разных расположениях.

    Собственные устройства, сертифицированные по Bluetooth, включая гарнитуры и динамики, поддерживаются на компьютерах с Windows (macOS может поддерживаться в будущем). Устройства проверяются на производительность звука, управление звонками и интеграцию кнопок Teams как с помощью USB-ключа, так и без нее.

    На производительность в собственном режиме Bluetooth влияет радио Bluetooth на хост-компьютере, поэтому мы рекомендуем установить последние обновления встроенного ПО как для устройств, так и для bluetooth-радио. Небольшой части пользователей может потребоваться продолжать использовать USB-прямоугольник, например при работе в радио окружении с высокой плотностью, если их КОМПЬЮТЕР Bluetooth активно используется несколькими парными устройствами или с некоторыми компьютерами с плохо работающими радиочастотами Bluetooth.

- **Микрофон уровня "Премиум" для открытия Office** Все гарнитуры проверяются для предоставления четкого звука, но открытие офиса и других шумных сред является более сложной задачей. Гарнитуры с таким оформлением передают голос пользователя гарнитуры, но не передают звук соседей.
- **Размер комнаты для собраний** Общие устройства, такие как динамики и камеры, имеют поддерживаемое расстояние использования. Рекомендуемые расстояния могут отличаться в зависимости от производителя. Чтобы предоставить клиентам четкие рекомендации, все общие устройства проверяются при сертификации на соответствие четко определенным размерам помещений и средам шума.

Обзор [устройств, сертифицированных для Microsoft Teams](https://products.office.com/microsoft-teams/across-devices/devices).

## <a name="end-user-experience"></a>Характер работы пользователей

### <a name="new-teams-certification-benefits"></a>Новые преимущества сертификации Teams

- Выделенная кнопка Teams и индикатор индикатора.
- Уведомление пользователей о событиях и оповещениях от клиента Teams (например, начало собрания, голосовая почта или пропущенный звонок).
- Возможность активации клиента Teams или ответа на уведомление с помощью нажатия кнопки (например, для присоединения к собранию).
- Сертификация, предназначенная для расширения (новые функции устройства или клиента для доставки через обновление встроенного ПО).

### <a name="core-certification-benefits"></a>Основные преимущества сертификации

- Автоматический выбор звукового устройства по умолчанию и определение приоритетов при наличии нескольких звуковых периферийных устройств.
- Устройства автоматически регистрируются в клиенте Teams и готовы к использованию.
- Предлагаются звуковые устройства (например, гарнитуры и динамики):
  - Базовое управление звонками при использовании с компьютером или устройством Mac. Основные элементы управления включают функцию ответа или зависания, отключение и включение звука, а также управление громкости.
  - Широкоштатный звук без эха, искажения или чрезмерного сбоя. Эхо-отмена также поддерживается на разных устройствах.
- Видеоустройства соответствуют требованиям к шуму, цвету, сведениям об изображении, дрожаниям, задержке, частоте кадров и полю просмотра.

## <a name="administrator-experience"></a>Взаимодействие с администратором

- Подготовка не требуется
- Все устройства поддерживают обновления встроенного ПО, чтобы обеспечить новые функции и улучшения производительности.

Если вы являетесь производителем и хотите присоединиться к программе сертификации, см. [](/skypeforbusiness/certification/how-to-join) раздел "Присоединение к требованиям и доступным программам".
