# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>Аудио-и видеоустройства USB, сертифицированные для Microsoft Teams

Программа сертификации периферийных устройств обеспечивает высокое качество звука и видео с помощью периферийных устройств, таких как гарнитура, динамики, веб-камеры и мониторы. Сертифицированные устройства работают в окне (Plug & Play) без дополнительной настройки, а также предлагают Управление звонками в Microsoft Teams и Skype для бизнеса. Продукты предоставляют для конечных пользователей и администраторов удобную работу.

Для исследования устройств, протестированных и сертифицированных для Microsoft Teams, заголовков в [магазине Teams](https://products.office.com/microsoft-teams/across-devices/devices).

Более подробную информацию о поддержке звонков в Teams можно узнать в статье Управление звонками [с помощью гарнитуры в Teams](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8).

> [!TIP]
> Ищете устройства, сертифицированные для Skype для бизнеса? Подключение к [аудио-и видеоустройствам, сертифицированным для Skype для бизнеса](https://docs.microsoft.com/skypeforbusiness/certification/devices-usb-devices).

## <a name="understanding-certification-categories"></a>Общие сведения о категориях сертификации

Пользователи не нуждаются в замене ранее сертифицированных устройств при обновлении до Teams в Skype для бизнеса.  Все сертифицированные устройства поддерживаются как для пользователей Microsoft Teams, так и для пользователей Skype для бизнеса с учетом нескольких различий в функциональных возможностях.  Кроме того, сертификация обеспечивает дальнейшее проектирование некоторых категорий устройств.

- **Skype для бизнеса, сертифицированный с использованием команды обновления** Устройства, сертифицированные на этом уровне, изначально были сертифицированы для Skype для бизнеса. Для обеспечения максимально возможной продолжительности сеанса Корпорация Майкрософт работала с производителями устройств, чтобы обеспечить максимально возможную интеграцию с существующими устройствами без каких – либо изменений исходного оборудования. Кнопка *hookswitch* действует как кнопка команды, если она не находится в звонке, и может включать дополнительные функции за определенный период времени.  Устройства в этой категории могут содержать логотип Teams в маркетинговых материалах.
- **Сертифицировано для Microsoft Teams** Устройства, сертифицированные на этом уровне, указывают новейшие сертифицированные устройства, а не подвергать риску взаимодействие с другими участниками группы. Эти устройства предлагают все преимущества предыдущей сертификации, а также включают специальную кнопку группы с индикатором индикатора. В большинстве форм кнопка "команды" будет легко находить кнопочная кнопку с логотипом Teams. Корпорация Майкрософт продолжает работать с производителями устройств, чтобы получать новые значения с помощью обновлений микропрограмм с течением времени.

### <a name="special-usage-designations"></a>Специальные обозначения использования

- **Высококачественный микрофон для Open Office** Все гарнитуры проверяются на наличие четких звуковых файлов, но открытие Office и других бесшумной среды обеспечивает более высокую сложность: чтобы отдать пользователю возможность выбрать четкий голос, но не передавать звуковое сопровождение соседей.
- **Размер комнаты для собрания** Общие устройства, такие как динамики и камеры, имеют поддерживаемое расстояние по использованию, а Рекомендуемые расстояния могут регистрироваться разными производителями. Чтобы предоставить пользователям четкие рекомендации, все общие устройства тестируются по сертификации с использованием строго определенных размеров комнат и шумовых сред.

Поиск [устройств, сертифицированных для Microsoft Teams](https://products.office.com/microsoft-teams/across-devices/devices).

## <a name="end-user-experience"></a>Работа с конечными пользователями

### <a name="new-teams-certification-benefits"></a>Новые преимущества сертификации Teams

- Кнопки и индикатор индикаторов для выделенной команды.
- Уведомление пользователей о событиях и оповещениях от клиента Teams (например, начало собрания, голосовая почта или пропущенный звонок).
- Возможность активации клиента Teams или ответа на уведомление с помощью нажатия кнопки (например, для присоединения к собранию).
- Сертификация, разработанная для расширения (новые функции устройства и клиента, которые должны быть доставлены через обновление встроенного по).

### <a name="core-certification-benefits"></a>Основные преимущества сертификации

- Автоматический выбор звукового устройства по умолчанию и определение приоритетов при наличии нескольких звуковых периферийных устройств.
- Устройства автоматически регистрируются в клиенте Teams и готовы к использованию.
- Звуковые устройства (например, гарнитуры и микрофоны) предлагают:
  - Простое управление звонками при использовании на компьютере с Windows или Mac-устройствами. Основные элементы управления включают ответы/зависания, включение и отключение звука, а также регулятор громкости.
  - Высокострочный звуковой канал без эха, искажения или чрезмерных сбоев. Отмена эха также поддерживается на разных устройствах.
- Видеоустройства соответствуют требованиям для шума, цвета, деталей изображения, колебаний, задержки, частоты кадров и поля представления.

## <a name="administrator-experience"></a>Взаимодействие с администратором

- Подготовка не требуется
- Все устройства поддерживают обновления встроенного по для обеспечения новых функций и улучшения производительности.

Если вы являетесь производителем и хотите присоединиться к программе сертификации, Узнайте, [как присоединиться](https://docs.microsoft.com/skypeforbusiness/certification/how-to-join) к требованиям и доступным программам.
