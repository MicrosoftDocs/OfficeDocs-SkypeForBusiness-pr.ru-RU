# <a name="create-a-microsoft-365-resource-account-using-the-microsoft-365-admin-center"></a>Создание учетной записи ресурса Microsoft 365 с помощью Центра администрирования Microsoft 365

Учетные записи ресурсов Microsoft 365 — это почтовые ящики и учетные записи Teams, предназначенные для определенных ресурсов, таких как помещение, проектор и другие. Эти учетные записи ресурсов могут автоматически отвечать на приглашения на собрания с помощью правил, которые вы определяете при их создания. Например, если у вас есть общий ресурс, например конференц-зал, вы можете настроить для него учетную запись ресурса, которая будет автоматически принимать или отклонить приглашения на собрания в зависимости от доступности календаря.

## <a name="licensing"></a>Лицензирование

Прежде чем создавать учетную запись ресурса Microsoft 365, проверьте, какая у нее лицензия. Если для этого используется только учетная запись ресурса (приглашение на собрание с автоматическим принятием или отклонением приглашения), назначать лицензию учетной записи ресурса не нужно. Вам потребуется назначить лицензию учетной записи ресурса в следующих ситуациях:

- **Собрание Teams** Если вы хотите, чтобы ресурс (например, консоль комнат Microsoft Teams, панель совместной работы и так далее) присоединился к собранию Teams, чтобы участники могли использовать его для проведения видео- и аудиосвязи, вам потребуется лицензия на комнату для собрания. 
- **Вызовы через ННР** Если вы хотите, чтобы ресурс звонит или принимает звонки на внешние телефонные номера или с них (они называются звонками по телефонной сети общего звонков с телефонным переключением), требуется лицензия на телефонную систему Microsoft 365 или Microsoft 365 Business Voice.

Дополнительные сведения о лицензиях на надстройки Microsoft Teams для собраний, телефонной системы и [бизнес-голосовой связи](../teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

## <a name="create-a-resource-account-in-the-microsoft-365-admin-center"></a><a href="" id="create-device-acct-m365-admin-ctr"></a>Создание учетной записи ресурса в Центре администрирования Microsoft 365

1. Во sign in to Microsoft 365 by visiting https://admin.microsoft.com
2. Задайте учетные данные администратора для клиента Microsoft 365. После этого вы будете перенанастряться в Центр администрирования Microsoft 365.

:::image type="content" source="../media/collaboration-bar-m365-admin-center.png" alt-text="Центр администрирования Microsoft 365":::
3. В Центре администрирования  перейдите к области "Ресурсы"  на левой панели (может потребоваться сначала выбрать "Показать все") и выберите **"& оборудования".**

:::image type="content" source="../media/collaboration-bar-m365-resources-rooms.png" alt-text="Центр администрирования Microsoft 365 — ресурсы":::
4. Выберите **"Добавить почтовый ящик ресурса",** чтобы создать учетную запись помещения. Введите отображаемую имя и адрес электронной почты для учетной записи, выберите **"Добавить",** а затем выберите **"Закрыть".** Мы рекомендуем стандартизировать соглашение об именовке для всех учетных записей ресурсов.

> [!NOTE]
> По умолчанию для учетных записей ресурсов настроены следующие параметры: Если вы хотите изменить их, выберите **"Запланировать параметры планирования"** перед на планированием **"Закрыть".** Если вы хотите изменить их позже, перейдите в & ресурсов, выберите учетную запись ресурса и выберите "Изменить" в  >   **параметрах резервирования.** 
>
> - Разрешить повторение собраний
> - Автоматическое отклонение собраний вне следующих ограничений
>   - Окно резервирования (дней): 180
>   - Максимальная длительность (часы): 24
> - Автоматическое принятие запросов на собрания

:::image type="content" source="../media/collaboration-bars-admin-resources.png" alt-text="Центр администрирования Microsoft 365 — добавление ресурсов":::
5. Перейдите в **раздел "Пользователи"** в Центре администрирования и в списке **"Активные** пользователи" вы увидите только что созданную комнату.

:::image type="content" source="../media/collaboration-bars-M3565-admin-active-users.png" alt-text="Центр администрирования Microsoft 365 — см. активных пользователей":::
6. Выберите имя комнаты, и справа появится панель свойств учетной записи.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-settings.png" alt-text="Центр администрирования Microsoft 365 — свойства пользователя":::
7. Теперь вам нужно назначить пароль для учетной записи ресурса. На панели вы увидите свойства учетной записи и несколько необязательных действий. Чтобы **сменить пароль,** выберите значок сброса пароля под име пользователя. Если не сделать этого, пользователь должен изменить пароль при **первом входе.** Изменить пароль с помощью устройства нельзя. Выберите **"Сброс".**

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-password.png" alt-text="Центр администрирования Microsoft 365: сброс пароля":::
8. В разделе  **"Лицензии и приложения"** выберите страну или регион, в которых будет установлено устройство. Прокрутите страницу вниз и выберите поле рядом с лицензией, которая будет назначена (например, "Комната собрания"), и выберите **"Сохранить изменения".** Лицензия может отличаться в зависимости от организации.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-assign-license.png" alt-text="Центр администрирования Microsoft 365 — назначение лицензии":::
