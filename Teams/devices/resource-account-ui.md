# <a name="create-a-microsoft-365-resource-account-using-the-microsoft-365-admin-center"></a>Создание учетной Microsoft 365 ресурса с помощью Центр администрирования Microsoft 365

Microsoft 365 ресурсов — это почтовые ящики и Teams, предназначенные для определенных ресурсов, таких как помещение, проектор и так далее. Эти учетные записи ресурсов могут автоматически отвечать на приглашения на собрания с помощью правил, которые вы определяете при их создания. Например, если у вас есть общий ресурс, например конференц-зал, вы можете настроить для него учетную запись ресурса, которая будет автоматически принимать или отклонить приглашения на собрания в зависимости от доступности в календаре.

## <a name="licensing"></a>Лицензирование

Прежде чем создавать учетную запись Microsoft 365 ресурса, проверьте, какая у нее лицензия. Если вы используете только учетную запись ресурса для заказа ресурса (т. е. пригласить его на собрание и иметь возможность автоматически принять или отклонить приглашение), назначать лицензию учетной записи ресурса не нужно. Вам потребуется назначить лицензию учетной записи ресурса в следующих ситуациях:

- **Teams**. Если вы хотите, чтобы ресурс (например, консоль Комнаты Microsoft Teams, панель совместной работы и так далее) присоединился к собранию Teams, чтобы участники могли использовать его для проведения видео- и аудиосвязи, вам потребуется Конференц-зал лицензия. 
- **Вызовы через ОКП** Если вы хотите, чтобы ресурс звонит или принимает звонки на внешние телефонные номера (которые называются телефонной сетью общего звонков или звонит по ОКП), требуется лицензия на Microsoft 365 телефонная система или Голосовая связь Microsoft 365 бизнес телефонную Голосовая связь Microsoft 365 бизнес телефонную сеть.

Дополнительные сведения о лицензиях Конференц-зал, телефонная система и бизнес-голосовой голосовой Microsoft Teams лицензиях на [надстройки.](../teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

## <a name="create-a-resource-account-in-the-microsoft-365-admin-center"></a><a href="" id="create-device-acct-m365-admin-ctr"></a>Создание учетной записи ресурса в Центр администрирования Microsoft 365

1. Во sign in to Microsoft 365 by visitinghttps://admin.microsoft.com
2. Укайте учетные данные администратора для Microsoft 365 клиента. После этого вы будете перена Центр администрирования Microsoft 365.

:::image type="content" source="../media/collaboration-bar-m365-admin-center.png" alt-text="Центр администрирования Microsoft 365.":::
3. В Центре администрирования на панели слева  перейдите к ресурсу (может потребоваться сначала выбрать  показать все), а затем выберите Помещения **& оборудования**.

:::image type="content" source="../media/collaboration-bar-m365-resources-rooms.png" alt-text="Центр администрирования Microsoft 365 — Ресурсы.":::
4. Выберите **Добавить почтовый ящик ресурса,** чтобы создать новую учетную запись комнаты. Введите отображаемую имя и адрес электронной почты для учетной записи, выберите **Добавить**, а затем выберите **Закрыть**. Мы рекомендуем стандартизировать соглашение об именовке для всех учетных записей ресурсов.

> [!NOTE]
> По умолчанию для учетных записей ресурсов настроены следующие параметры: Если вы хотите изменить их, выберите **Настроить параметры планирования** , прежде чем на выбор **Закрыть**. Если вы хотите изменить их позже,  >  перейдите в **resourcesRooms & оборудование**, выберите учетную запись ресурса, а затем выберите **Изменить** в **параметрах резервирования**.
>
> - Разрешить повторение собраний
> - Автоматическое отклонение собраний вне следующих ограничений
>   - Окно резервирования (дней): 180
>   - Максимальная длительность (часы): 24
> - Автоматическое принятие запросов на собрания

:::image type="content" source="../media/collaboration-bars-admin-resources.png" alt-text="Центр администрирования Microsoft 365 — добавление ресурсов.":::
5. Перейдите в **раздел Пользователи** в Центре администрирования и в списке **Активные** пользователи увидите только что созданную комнату.

:::image type="content" source="../media/collaboration-bars-M3565-admin-active-users.png" alt-text="Центр администрирования Microsoft 365 . См. активных пользователей.":::
6. Выберите имя комнаты, и справа появится панель свойств учетной записи.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-settings.png" alt-text="Центр администрирования Microsoft 365 — свойства пользователя.":::
7. Теперь необходимо назначить пароль учетной записи ресурса. На панели вы увидите свойства учетной записи и несколько необязательных действий. Чтобы **сменить пароль,** выберите значок Сброс ключа пароля под име пользователем. Отменить. **Требовать, чтобы пользователь изменил пароль при первом входе**. Изменить пароль с помощью процесса регистрации устройства невозможно. **Выберите Сброс**.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-password.png" alt-text="Центр администрирования Microsoft 365 — сброс пароля.":::
8. В разделе **Лицензии и приложения** выберите расположение для  страны или региона, в которых будет установлено устройство. Прокрутите страницу вниз и выберите рядом с назначенной лицензией (например, Конференц-зал), а затем выберите **Сохранить изменения**. Лицензия может отличаться в зависимости от организации.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-assign-license.png" alt-text="Центр администрирования Microsoft 365 — назначение лицензии.":::
