# <a name="create-a-microsoft-365-resource-account-using-the-microsoft-365-admin-center"></a>Создание учетной записи ресурса Microsoft 365 с помощью центра администрирования Microsoft 365

Учетные записи ресурсов Microsoft 365 — это учетные записи почтового ящика и группы, выделенные для определенных ресурсов, таких как комната, проектор и т. д. Эти учетные записи ресурсов могут автоматически отвечать на приглашения на собрания с помощью правил, которые определяются при их создании. Например, если у вас есть общий ресурс (например, Конференц-зал), вы можете настроить учетную запись ресурса для нее, чтобы она автоматически принимала или отклоняет приглашения на собрания в зависимости от его доступности в календаре.

## <a name="licensing"></a>Лицензирование

Перед созданием учетной записи ресурса Microsoft 365 убедитесь, что у вас есть лицензия, необходимая ему. Если вы будете использовать учетную запись ресурса для учета ресурсов (то есть пригласить ресурса на собрание и получить его автоматически или отклонить), вам не нужно будет назначать лицензию для учетной записи ресурса. Вам потребуется назначить лицензию для учетной записи ресурса в следующих ситуациях:

- **Собрание Teams** Если вы хотите, чтобы этот ресурс (например, консоль комнаты для совместной работы и т. д.) присоединится к собранию Teams, чтобы участники могли использовать его для показа видео и голосовой связи, вам понадобится лицензия на Конференц-зал. 
- **Коммутируемые звонки** Если вы хотите, чтобы ресурс выработал и получал звонки на внешние телефонные номера (называемые коммутируемой телефонной сетью или КОММУТИРУЕМым звонком), вам нужна телефонная система Microsoft 365 или лицензия Microsoft 365 Business Voice.

Дополнительные сведения о помещениях для собраний, телефонной системы и голосовых лицензиях можно найти [в разделе лицензии на надстройки для Microsoft Teams](../teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

## <a name="create-a-resource-account-in-the-microsoft-365-admin-center"></a><a href="" id="create-device-acct-m365-admin-ctr"></a>Создание учетной записи ресурса в центре администрирования Microsoft 365

1. Вход в Microsoft 365 путем посещенияhttps://admin.microsoft.com
2. Укажите учетные данные администратора для вашего клиента Microsoft 365. Это позволит вам перейти в центр администрирования Microsoft 365.

:::image type="content" source="../media/collaboration-bar-m365-admin-center.png" alt-text="Центр администрирования Microsoft 365":::
3. В центре администрирования перейдите к разделу **ресурсы** на левой панели (возможно, потребуется сначала выбрать **Показать все** ), а затем выберите **помещения & оборудования**.

:::image type="content" source="../media/collaboration-bar-m365-resources-rooms.png" alt-text="Центр администрирования Microsoft 365 – ресурсы":::
4. Выберите **добавить почтовый ящик ресурсов** , чтобы создать новую учетную запись комнаты. Введите отображаемое имя и адрес электронной почты для учетной записи, нажмите кнопку **Добавить**, а затем — кнопку **Закрыть**. Рекомендуется стандартизировать соглашения об именах для всех учетных записей ресурсов.

> [!NOTE]
> По умолчанию для учетных записей ресурсов устанавливаются следующие параметры. Если вы хотите изменить их, выберите команду **задать параметры планирования** , прежде чем нажать кнопку **Закрыть**. Если вы хотите изменить их позже, перейдите в помещения **ресурсы**  >  **& оборудование**, выберите учетную запись ресурса, а затем щелкните **изменить** в разделе **Параметры резервирования**.
>
> - Разрешить повторение собраний
> - Автоматическое отклонение собраний за пределами указанных ниже ограничений
>   - Окно резервирования (дн.): 180
>   - Максимальная длительность (в часах): 24
> - Автоматическое одобрение приглашений на собрания

:::image type="content" source="../media/collaboration-bars-admin-resources.png" alt-text="Центр администрирования Microsoft 365: Добавление ресурсов":::
5. Перейдите в раздел **Пользователи** в центре администрирования и в списке **Активные пользователи** появится только что созданная комната.

:::image type="content" source="../media/collaboration-bars-M3565-admin-active-users.png" alt-text="Центр администрирования Microsoft 365 — Просмотр активных пользователей":::
6. Выберите имя комнаты, и в правой части экрана появится панель свойств учетной записи.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-settings.png" alt-text="Центр администрирования Microsoft 365 — свойства пользователя":::
7. Теперь вам нужно назначить пароль для учетной записи ресурса. На панели можно увидеть свойства учетной записи и несколько необязательных действий. Чтобы изменить пароль, щелкните значок **сброса ключа пароля** под именем пользователя. Снимите флажок **требовать от пользователя изменения пароля при первом входе**. Изменить пароль через процесс входа на устройство невозможно. Нажмите кнопку **Сброс**.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-password.png" alt-text="Центр администрирования Microsoft 365 — сброс пароля":::
8. В разделе " **лицензии и приложения** " установите **переключатель в положение выбрать расположение** для выбранной страны или региона, где будет установлено устройство. Прокрутите вниз и установите флажок рядом с лицензией, которую вы хотите назначить, а затем нажмите кнопку **сохранить изменения**. Лицензия может различаться в зависимости от вашей организации.

:::image type="content" source="../media/collaboration-bar-m365-admin-center-active-user-assign-license.png" alt-text="Центр администрирования Microsoft 365 — назначение лицензии":::
