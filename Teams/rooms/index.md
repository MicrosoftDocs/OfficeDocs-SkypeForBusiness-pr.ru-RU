# <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

:::row:::
    :::column span="2":::
       Преобразуйте пространства собраний в диапазоне от небольших областей huddle до больших конференц-залов с широкими возможностями совместной работы Teams, которые просты в использовании, развертывании и управлении.

       Начните собрания во время с помощью одно касания, а затем мгновенно проецируете экран в комнате и поделитесь ими с удаленными участниками.

       Выберите правильные периферийные устройства системы и аудиоконферентивов от одного из наших партнеров: Dell, Yealink, Logitech,Wnron, Polycom, Lenovo и HP.

       Чтобы быстро просмотреть Комнаты Teams, щелкните видео справа.

> [!NOTE]
> Комнаты Microsoft Teams предназначен для использования с Microsoft Teams, Skype для бизнеса Server 2019 или Skype для бизнеса Server 2015.
>
> Более ранние платформы, такие как Lync Server 2013, не поддерживаются Комнаты Microsoft Teams.

Эти статьи предназначены для пользователей, которым по задаче — планирование, развертывание и администрирование этих Комнаты Microsoft Teams. Конечные пользователи будут больше заинтересованы в том, Комнаты Microsoft Teams [в Интернете](https://support.office.com/article/Skype-Room-Systems-version-2-help-e667f40e-5aab-40c1-bd68-611fe0002ba2).
