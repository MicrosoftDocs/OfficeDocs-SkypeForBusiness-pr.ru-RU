# <a name="manage-a-microsoft-teams-rooms-console-settings-remotely-with-an-xml-configuration-file"></a>Удаленное Комнаты Microsoft Teams консоли с помощью XML-файла конфигурации

В этой статье рассмотрено удаленное управление настройками по умолчанию, используемыми устройством Комнаты Microsoft Teams, включая применение пользовательской темы. В нем обсуждаются вопросы создания файла параметров и ссылки на обсуждения того, как разместить их на устройствах с удаленным управлением.
  
Вы можете изменить параметры по умолчанию для удаленно управляемых устройств, обновив XML-файл и отправив копии на управляемые консоли. Вы также можете реализовать настраиваемые темы на консолях Комнаты Microsoft Teams с помощью XML-файлов конфигурации.
  
## <a name="create-an-xml-configuration-file"></a>Создание файла конфигурации XML

Для создания файла параметров можно использовать любой текстовый редактор. В **таблице XML Elements** (Элементы XML) объясняются элементы, показанные в этом примере SkypeSettings.xml файла (обязательное имя файла).
  
```XML
<SkypeSettings>
    <AutoScreenShare>true</AutoScreenShare>
    <HideMeetingName>true</HideMeetingName>
    <UserAccount>
        <SkypeSignInAddress>RanierConf@contoso.com</SkypeSignInAddress>
        <ExchangeAddress>RanierConf@contoso.com</ExchangeAddress>
        <ModernAuthEnabled>false</ModernAuthEnabled>
        <DomainUsername>Seattle\RanierConf</DomainUsername>
        <Password>password</Password>
        <ConfigureDomain>domain1, domain2</ConfigureDomain>
    </UserAccount>
    <IsTeamsDefaultClient>false</IsTeamsDefaultClient>
    <BluetoothAdvertisementEnabled>true</BluetoothAdvertisementEnabled>
    <AutoAcceptProximateMeetingInvitations>false</AutoAcceptProximateMeetingInvitations>
    <AutoExitMeetingEnabled>true</AutoExitMeetingEnabled>
    <SkypeMeetingsEnabled>false</SkypeMeetingsEnabled>
    <TeamsMeetingsEnabled>true</TeamsMeetingsEnabled>
    <WebExMeetingsEnabled>true</WebExMeetingsEnabled>
    <ZoomMeetingsEnabled>true</ZoomMeetingsEnabled>
    <UseCustomInfoForThirdPartyMeetings>true</UseCustomInfoForThirdPartyMeetings>
    <CustomDisplayNameForThirdPartyMeetings>Rainier Conference Room</CustomDisplayNameForThirdPartyMeetings>
    <CustomDisplayEmailForThirdPartyMeetings>RanierConf@contoso.com</CustomDisplayEmailForThirdPartyMeetings> 
    <DualScreenMode>true</DualScreenMode>
    <DuplicateIngestDefault>false</DuplicateIngestDefault>
    <DisableTeamsAudioSharing>false</DisableTeamsAudioSharing>
    <CortanaEnabled>false</CortanaEnabled>
    <CortanaWakewordEnabled>false</CortanaWakewordEnabled>
    <SendLogs>
        <EmailAddressForLogsAndFeedback>RanierConf@contoso.com</EmailAddressForLogsAndFeedback>
        <SendLogsAndFeedback>true</SendLogsAndFeedback>
    </SendLogs>
    <Devices>
        <MicrophoneForCommunication>Microsoft LifeChat LX-6000</MicrophoneForCommunication>
        <SpeakerForCommunication>Realtek High Definition Audio</SpeakerForCommunication>
        <DefaultSpeaker>Polycom CX5100</DefaultSpeaker>
        <ContentCameraId>USB\VID_046D&amp;PID_0843&amp;MI_00\7&amp;17446CF2&amp;0&amp;0000</ContentCameraId>
        <ContentCameraInverted>false</ContentCameraInverted>
        <ContentCameraEnhancement>true</ContentCameraEnhancement>
    </Devices>
    <Theming>
        <ThemeName>Custom</ThemeName>
        <CustomThemeImageUrl>file name</CustomThemeImageUrl>
        <CustomThemeColor>
            <RedComponent>100</RedComponent>
            <GreenComponent>100</GreenComponent>
            <BlueComponent>100</BlueComponent>
        </CustomThemeColor>
    </Theming>
    <CoordinatedMeetings enabled="true"> 
        <TrustedAccounts>room@contoso.com</TrustedAccounts>
        <Settings> 
            <Audio default="false" enabled="false" />
            <Video default="false" enabled="true" /> 
            <Whiteboard default="false" enabled="false"/>
        </Settings> 
    </CoordinatedMeetings>
</SkypeSettings>
```

Если переменное значение имеет неправильный тип, элементы не по порядку, элементы не скрыты или обнаружена другая ошибка, XML-файл *формируется неправильно.* При обработке XML-файла, сформированного с ошибкой, применяются параметры до того места, где возникает ошибка, а остальной файл игнорируется. Любые неизвестные элементы XML-файла также пропускаются. Соответствующие пропущенным элементам параметры устройств остаются без изменений. Если значение параметра является недопустимым, его предыдущее значение остается без изменений.

## <a name="manage-console-settings-with-an-xml-configuration-file"></a>Управление параметрами консоли с помощью XML-файла конфигурации

Если при запуске консоль Комнаты Microsoft Teams находит XML-файл с именем SkypeSettings.xml, он применяет параметры конфигурации, указанные XML-файлом, а затем удаляет `C:\Users\Skype\AppData\Local\Packages\Microsoft.SkypeRoomSystem_8wekyb3d8bbwe\LocalState` XML-файл.
  
В зависимости от Комнаты Microsoft Teams устройств и способа их настройки можно разместить XML-файл конфигурации несколькими способами. После принудительной передачи файла на консоль необходимо перезапустить ее для обработки изменений. После успешной обработки XML-файл конфигурации удаляется. Методы управления, предлагаемые для Комнаты Microsoft Teams, рассмотрены в
  
- [Настройка групповой политики для Комнаты Microsoft Teams](rooms-operations.md#GroupPolicy)
- [Удаленное управление с помощью PowerShell](rooms-operations.md#RemotePS) [и настройка элемента файла](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc772536(v=ws.11))

Вы можете использовать любой из этих способов при условии, что он позволяет передавать файлы и перезапускать устройство консоли. Файл должен быть удобным для чтения, записи и удаления в учетной записи локального пользователя устройства. Предпочтительно, чтобы оно принадлежит пользователю и имеет все права, предоставленные этому пользователю. Если разрешения для файла настроены неправильно, программное обеспечение может не применить параметры, не удалить файл после успешной обработки и даже привести к сбою.
  
## <a name="custom-theme-images"></a>Изображения настраиваемых тем

<a name="Themes"> </a>

Файл пользовательского изображения темы должен быть помещен в `C:\Users\Skype\AppData\Local\Packages\Microsoft.SkypeRoomSystem_8wekyb3d8bbwe\LocalState` папку. Введите имя файла и расширение в \<CustomThemeImageUrl\> переменной.
  
Файл изображения должен иметь размер 3840X1080 пикселей и иметь один из следующих форматов: JPG, JPEG, PNG и BMP. Если вашей организации нужно получить пользовательское изображение, графический дизайнер может использовать пользовательский шаблон [Photoshop темы](../downloads/ThemingTemplateMicrosoftTeamsRooms_v2.1.psd). Она содержит более подробные сведения о том, где различные элементы пользовательского интерфейса относительно остальной части изображения темы и какие области отображаются на консолях и дисплеях.
  
Изображение темы будет распознаваться только после того, как XML-файл конфигурации обновится при запуске устройства. После обработки и удаления XML-файла файл темы удаляется из каталога.
  
## <a name="locate-the-content-camera-usb-instance-path"></a>Поиск пути usb-экземпляра "Камера содержимого"

Чтобы найти путь экземпляра:

1. Перейдите Windows параметры на консоли Комнаты Microsoft Teams.
2. Введите пароль администратора.
3. В командной подсказке `devmgmt.msc` введите, чтобы вывести Диспетчер устройств.
4. В **диспетчере устройств** найдите узел **Устройства обработки** изображений и найдите камеру содержимого.
5. Щелкните камеру правой кнопкой мыши и откройте **свойства**.
6. Перейдите **на вкладку** Сведения и найдите свойство **Путь экземпляра устройства** в drop-down.
7. Показанное значение — это путь к экземпляру устройства, заданная в XML-файле конфигурации. При указании пути в XML замените амперсанд (&) на `&amp;` .
