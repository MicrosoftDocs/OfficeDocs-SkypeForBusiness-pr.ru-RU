# <a name="microsoft-teams-rooms-pro-management"></a>Комнаты Microsoft Teams Pro Management

Служба Комнаты Microsoft Teams Pro Management — это облачное решение для управления, которое заранее отслеживает и обновляет Комнаты Microsoft Teams устройств и их периферийных устройств. Решение для управления Pro предназначено для организаций, которым требуется оптимизировать взаимодействие с комнатами для собраний для конечных пользователей и быстро масштабировать занимаемое ими место, благодаря мониторингу и управлению устройствами Комнаты Microsoft Teams в режиме реального времени. 

Решение Комнаты Microsoft Teams мониторинга pro создано на основе знаний корпорации Майкрософт по управлению конечными точками с поддержкой программного обеспечения и интеллектуальной автоматизации. 


Ниже перечислены три ключевых аспекта службы.  

- Интеллектуальные операции  

   Программное обеспечение и машинное обучение, которое автоматизирует обновления, обнаружение проблем, разрешение проблем для Комнаты Microsoft Teams.  

- Управление обновлениями  

   Автоматическая оркестрация приложений для собраний и обновлений Windows на основе кругов развертывания, настраиваемых клиентом.

- Расширенные аналитические сведения  

   Полнофункциональный анализ, отчетность и проверенные учебные процессы в большом масштабе для многих клиентов.

Дополнительные сведения см. [в разделе "](rooms-prep.md)Подготовка среды" Комнаты Microsoft Teams документации.

## <a name="data-collection"></a>Сбор данных

Комнаты Microsoft Teams Pro отслеживает устройства, собирает данные об устройствах и позволяет корпорации Майкрософт удаленно получать доступ к устройствам комнат и управлять ими. У Комнаты Microsoft Teams управляемой службы нет доступа к интерактивному содержимому собрания.

Собранные данные относятся только к учетной записи комнаты, а не к отдельному пользователю. Случайные ссылки на отдельного пользователя могут присутствовать в журнале действий во время использования устройства. Собранные данные обрабатываются корпорацией Майкрософт в соответствии с заявлением [о конфиденциальности Майкрософт](https://aka.ms/privacy).
