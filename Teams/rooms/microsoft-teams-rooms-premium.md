# <a name="microsoft-teams-rooms-managed-service"></a>Служба, управляемой комнатами Microsoft Teams

Управленная служба Комнаты Microsoft Teams — это облачная служба ИТ-управления и мониторинга, которая позволяет поддерживать устройства с комнатами Microsoft Teams и их периферийные устройства в рамках поддержки среды, оптимизированной для удобного пользовательского интерфейса.

Управленная служба Комнаты Microsoft Teams — это служба для организаций, которым необходимо улучшить работу в комнате для собраний и быстро масштабировать свою среду, предлагая упреждающий подход к управлению устройствами Комнаты Microsoft Teams. Эта служба позволяет организациям экономить время, деньги и знания, которые в противном случае будут потрачены на управление комнатами самостоятельно.  

Управленная служба Комнаты Microsoft Teams построена на основе знаний корпорации Майкрософт в области управления конечными точками, машинного обучения и интеллектуальной автоматизации с поддержкой программного обеспечения. С помощью управляемой службы Комнаты Microsoft Teams организации могут обеспечить высокое качество работы на нескольких удаленных сайтах и обеспечить любой темп роста, который может потребоваться организации.

## <a name="service-prerequisites"></a>Необходимые условия для обслуживания

Чтобы управляемые службой Комнаты Microsoft Teams могли поддерживать качество обслуживания, зарегистрированные комнаты должны быть поддерживаемыми устройствами [Комнаты Microsoft Teams](requirements.md#hardware-requirements), и необходимо подготовить свою среду со всеми требуемыми URL-адресами [и диапазонами IP-адресов Office 365,](/office365/enterprise/urls-and-ip-address-ranges) которые полностью разрешены и недоступны с помощью устройств в комнате собраний. Подробнее см. в [документе Подготовка](rooms-prep.md) среды в документации к комнатам Microsoft Teams.

Как поставщик управляемой службы корпорация Майкрософт должна работать с ответственными заинтересованными лицами в помещениях в организации, чтобы поддерживать работу помещений и восстанавливать их. Вы несете ответственность за несдерживаемые устройства, используемые в комнатах для собраний, стороне или пользовательском программном обеспечении, выбранном для развертывания на устройствах, и за изменение методов управления, рекомендуемых корпорацией Майкрософт.

Ваша организация или партнер, договорившейся с вашей организацией, должны быть готовы устранять проблемы, выявленные службой на последнем расстоянии.

## <a name="data-collection"></a>Сбор данных

Служба Комнаты Microsoft Teams отслеживает устройства, собирает данные устройств и позволяет корпорации Майкрософт удаленно получать доступ к устройствам помещений и управлять ими. У Комнаты Microsoft Teams управляемых служб нет доступа к содержимому собрания в прямом эфире.

Данные собираются только для учетной записи комнаты, а не отдельного пользователя. При использовании устройства в журнале действий могут присутствовать случайные ссылки на отдельных пользователей. Собранные данные обрабатываются корпорацией Майкрософт в соответствии с Заявлением [о конфиденциальности Корпорации Майкрософт.](https://aka.ms/privacy)
