# <a name="microsoft-teams-rooms-managed-service"></a>Служба, управляемой комнатами Microsoft Teams

Управленная служба Комнаты Microsoft Teams — это облачная служба ИТ-управления и мониторинга, которая позволяет поддерживать устройства Microsoft Teams Rooms и их периферийные устройства в рамках поддержки среды, оптимизированной для удобного пользовательского интерфейса.

Управленная служба Комнаты Microsoft Teams — это служба, которая предлагает упреждающие средства управления для устройств с комнатами Microsoft Teams. Эта служба позволяет организациям экономить время, деньги и знания, которые в противном случае будут потрачены на управление комнатами самостоятельно.  

Управленная служба Microsoft Teams Rooms построена на основе знаний корпорации Майкрософт в области управления конечными точками, машинного обучения и интеллектуальной автоматизации с поддержкой программного обеспечения. С помощью управляемой службы Комнаты Microsoft Teams организации могут обеспечить высокое качество работы на нескольких удаленных сайтах и обеспечить любой темп роста, который может потребоваться организации.

## <a name="service-prerequisites"></a>Необходимые условия для обслуживания

Для обеспечения качества обслуживания управляемых служб Комнаты Microsoft Teams зарегистрированные комнаты должны быть устройствами, поддерживаемыми [комнатами Microsoft Teams,](requirements.md#hardware-requirements)и необходимо подготовить свою среду со всеми требуемыми URL-адресами [и диапазонами IP-адресов Office 365,](/office365/enterprise/urls-and-ip-address-ranges) которые полностью разрешены и недоступны с помощью устройств в комнате для собраний. Подробнее см. в [документе Подготовка](rooms-prep.md) среды в документации microsoft Teams Rooms.

Как поставщик управляемой службы корпорация Майкрософт должна работать с ответственными заинтересованными лицами в помещениях в организации, чтобы поддерживать работу помещений и восстанавливать их. Вы несете ответственность за несдержиживемые устройства, используемые в комнатах для собраний, стороне или пользовательском программном обеспечении, которое вы решили развернуть на устройствах, и за изменение методов управления, рекомендуемых корпорацией Майкрософт.

Ваша организация или партнер, договорившейся с вашей организацией, должны быть готовы устранять проблемы на последнем расстоянии, обнаруженные службой.

## <a name="data-collection"></a>Сбор данных

В управляемых помещениях Microsoft Teams служба отслеживает устройства, собирает данные об устройствах и позволяет корпорации Майкрософт удаленно получать доступ к устройствам помещений и управлять ими. Служба Microsoft Teams Rooms не имеет доступа к содержимому собрания в прямом эфире.

Данные собираются только для учетной записи комнаты, а не отдельного пользователя. При использовании устройства в журнале действий могут присутствовать случайные ссылки на отдельных пользователей. Собранные данные обрабатываются корпорацией Майкрософт в соответствии с [Заявлением о конфиденциальности Корпорации Майкрософт.](https://aka.ms/privacy)
