# <a name="microsoft-teams-rooms-app-version-support"></a>Комнаты Microsoft Teams версии приложения
 
Приложение Комнаты Microsoft Teams получает обновления через Windows хранилища. Microsoft Teams Room использует жизненный цикл продукта, и в любой момент времени поддерживаются только текущая и последняя версии приложения. Приложение Microsoft Teams Room объединяет определенную версию Teams, измененную для использования в помещении. Приложение Teams обновляется каждые две недели. Дополнительные сведения о Teams [обновления](../teams-client-update.md). Это означает Комнаты Teams что текущая версия приложения версии 1 может содержать до шести обновлений классических приложений Teams, поэтому рекомендуется постоянно обновлять приложение Teams Room до последней версии приложения Комнаты Teams. 

Структура поддержки для Комнаты Teams является динамической и зависит от доступности последней версии. При возникновении ошибки кода в версии приложения, которая не является последней, необходимо установить последнюю версию, чтобы получить исправление.

Все выпуски перечислены в Комнаты Microsoft Teams [выпусков](rooms-release-note.md).

> [!IMPORTANT]
> При установке нового устройства, которое поставляется с более старой версией приложения комнаты Teams, рекомендуется вручную обновить приложение после [](manual-update.md) настройки учетной записи перед скачиванием Windows обновлений. Это гарантирует правильную версию ОС Windows установки обновлений на устройстве.  

## <a name="windows-10-release-support"></a>Windows 10 выпусков

Комнаты Microsoft Teams требуются номера SKU Windows 10 IoT Корпоративная или Windows 10 Корпоративная в Semi-Annual канале. Другие выпуски Windows 10 не поддерживаются:

- Windows 10 Корпоративная long-term Servicing Branch (LTSB) / Long Term Servicing Channel (LTSC) editions
- Windows 10 Internet of Things (IoT) Enterprise LTSB или LTSC
- любой другой выпуск Windows, например Windows 10 Pro или Home Edition

Новые Windows 10 компонентов не предлагаются на Комнаты Microsoft Teams устройствах. После даты общедоступной версии, опубликованной на странице сведений о выпуске Windows 10, существует намеренная задержка до [шести месяцев](/windows/release-information/) или более. Это время используется для проверки совместимости Windows 10 выпусков для Комнаты Microsoft Teams, оборудования устройства и сертифицированных периферийных устройств для аудиоконферентов. Проверка начинается и продолжается во время активной разработки каждого основного выпуска Windows 10. Дополнительное время требуется для проверки того, что все производители устройств создали обновленные образы для своих устройств, а корпорация Майкрософт должна сертифицировать и протестировать эти образы. В течение периода проверки приложение Microsoft Teams Room использует групповые политики [клиентский компонент Центра обновления Windows для](/windows/deployment/update/waas-manage-updates-wufb) бизнеса для задержки Windows 10 обновлений компонентов. После того как все проблемы совместимости будут найдены и устранены, блокировка будет снята путем обновления групповых политик с помощью нового выпуска приложения в Windows Store. Устройства, на которых выполняется Комнаты Microsoft Teams приложения, автоматически обновляются до соответствующего Windows 10 во время перезагрузки ночного обслуживания. Доступна версия MSI для клиентов, которые должны вручную управлять обновлениями.  

> [!IMPORTANT]
> В течение периода проверки Комнаты Microsoft Teams устройства не должны обновляться до  следующего выпуска Windows 10 любыми способами. Это включает переопределение групповых политик или использование System Center или других сторонних служб управления устройствами. Любое из этих действий может привести к проблемам с приложением Microsoft Teams Room или привести к недоступности устройств.
