# <a name="microsoft-teams-rooms-app-version-support"></a>Поддержка версии приложения "Комнаты Microsoft Teams"
 
Приложение "Комнаты Microsoft Teams" обновляется несколько раз в год. Каждое обновление поддерживается в течение 12 (двенадцати) месяцев с даты выпуска общедоступной версии. Техническая поддержка предоставляется в течение всего 12 (12) месяцев. Однако структура поддержки динамическая с двумя отдельными этапами, которые зависят от доступности последней версии:

- **Этап обслуживания и критических обновлений** \- При запуске последней версии приложения "Комнаты Microsoft Teams" вы регулярно получаете обновления, содержащие обновления системы безопасности и *обслуживания.*

- **Этап "Только обновления для системы безопасности"** \- При выпуске новой версии приложения "Комнаты Microsoft Teams" для более старых  версий снижается уровень поддержки благодаря обновлениям системы безопасности только для остального 12-месячного жизненного цикла.

> [!NOTE]
> Последняя версия всегда находится на этапе обслуживания и критических обновлений. Если вы столкнулись с дефектом кода, который требует критического обновления, для получения исправления также должна быть установлена последняя версия. Все остальные поддерживаемые версии смогут получать только обновления для системы безопасности.

Поддержка заканчивается по истечении 12-месячного жизненного цикла одной версии или, если с этого момента было выпущено более двух обновлений. После этого пользователи должны обновиться до поддерживаемой версии.

Все выпуски перечислены в заметках [о выпуске комнат Microsoft Teams.](rooms-release-note.md)

## <a name="windows-10-release-support"></a>Поддержка выпусков Windows 10

Для комнат Microsoft Teams требуются номера СКОС Windows 10 IoT Enterprise или Windows 10 корпоративный в Semi-Annual каналов обслуживания. Другие выпуски Windows 10 не поддерживаются:

- Ветви долгосрочного обслуживания Windows 10 корпоративный (LTSB) и Long Term Servicing Channel (LTSC)
- Выпуски корпоративных LTSB и LTSC для Windows 10
- любой другой выпуск Windows, например Windows 10 Pro или Home Edition

Обновление функций Windows 10 не предоставляется и не обновляется на устройствах комнат Microsoft Teams сразу. Преднамеренный задержка, которая составляет до шести месяцев после даты общего доступа, опубликованной на странице сведений о выпуске [Windows 10.](https://docs.microsoft.com/windows/release-information/) Время задержки используется для проверки совместимости выпусков Windows 10 с приложением для комнат Microsoft Teams, оборудованием устройств и сертифицированными периферийными устройствами для звука. Проверка начинается и продолжается во время активной разработки каждого крупного выпуска Windows 10. Дополнительное время необходимо для проверки того, что все изготовители устройств создали обновленные изображения для своих устройств, а также для Сертификации и проверки этих изображений в Microsoft Teams. В период проверки приложение комнат Microsoft Teams использует групповые политики  [Windows Для](https://docs.microsoft.com/windows/deployment/update/waas-manage-updates-wufb) бизнеса, чтобы отложить обновление функций Windows 10. После того как все проблемы совместимости найдены и устранены, этот блок будет снят с помощью обновления групповых политик в новом выпуске приложений в Магазине Windows. Устройства, которые запускают приложение "Комнаты Microsoft Teams", автоматически обновляются до соответствующего выпуска Windows 10 во время ночного обслуживания перезагрузки. Версия MSI доступна для клиентов, которые хотят вручную управлять обновлениями.  

> [!IMPORTANT]
> В период проверки устройства комнат  Microsoft Teams не должны быть обновлены до следующего выпуска Windows 10. Это включает переопределение групповых политик на месте, а также использование System Center или других сторонних служб управления устройствами. Это может привести к проблемам в приложении комнат Microsoft Teams или оставить устройства неиспользоваемыми.
