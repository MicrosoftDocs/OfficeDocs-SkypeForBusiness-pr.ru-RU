# <a name="use-the-microsoft-teams-rooms-recovery-tool"></a>Используйте средство восстановления приложения "Комнаты Microsoft Teams"

В этой статье описывается, как использовать средство восстановления для Комнаты Microsoft Teams, которое будет использоваться для перевода системы из устарелого состояния в поддерживаемое состояние. Это средство следует применять, если в консоли Комнаты Microsoft Teams отображается ошибка "Устарела конфигурация системы" или перед выполнением восстановления фабрики сбросить [кнопку](./rooms-operations.md#microsoft-teams-rooms-reset-factory-restore).

## <a name="prerequisites"></a>Необходимые компоненты

Скачайте последнюю [версию Комнаты Microsoft Teams](https://go.microsoft.com/fwlink/?linkid=851168) установки и извлеките ее на USB-накопитель памяти или в сетевую папку, доступную для Комнаты Microsoft Teams.

> [!NOTE]
> Извлечение файлов из MSI можно выполнить различными способами. Любой механизм, который извлекает все файлы и сохраняет их структуру каталогов, приемлем. Один из `msiexec /a PathToMsi /qb TARGETDIR=PathToTarget` `PathToMsi` таких способов — использовать команду, в которой представлен полный путь к пакету установки Комнаты Microsoft Teams и полный путь к папке, `PathToTarget` в которую вы хотите извлечь файлы.

## <a name="running-the-tool"></a>Запуск средства

1) Войдите в учетную запись администратора на устройстве Комнаты Microsoft Teams и запустите командную строку с повышенными привилегиями.
2) Убедитесь, Комнаты Microsoft Teams `RecoveryTool.ps1` устройства есть доступ к файлу, который включен в файлы, извлеченные из Комнаты Microsoft Teams установки. Комплект можно найти в сетевой папке или USB-накопителе, используемом при подготовке необходимых компонентов.
3) Выполните .`powershell.exe -ExecutionPolicy Unrestricted -File "<path to RecoveryTool.ps1>"`
4) Чтобы выполнить восстановление фабрики, выполните следующие действия.
   1. При появлении запроса в сценарии выберите вариант 2: **Сброс**.
   2. Если BitLocker включен, выполните инструкции, приведенные в конце выходных данных скрипта, чтобы отключить его.
   3. Выполните восстановление фабрики.
      1. Откройте приложение **"Параметры**" и выберите "Обновить **& безопасности"**.
      2. Перейдите на **вкладку "Восстановление** ".
      3. Под **кнопкой "Сбросить этот компьютер**" выберите **"Начало работы"**
      4. Выберите " **Удалить все"**, затем **"Далее"** и " **Сбросить".**
        > [!WARNING]
        > Устройство Комнаты Microsoft Teams может стать непригодным для использования, если функция "Сохранить мои файлы" удаляет приложения и параметры **,** но в процессе сброса Windows выбран параметр сохранения личных файлов. Не используйте этот параметр.
      5. Система будет перезагружаться несколько раз. После завершения сброса система будет отображаться на экране "При первом включении при первом включении" (OOBE) Windows.
