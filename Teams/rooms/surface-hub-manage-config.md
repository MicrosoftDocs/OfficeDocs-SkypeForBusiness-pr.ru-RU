# <a name="manage-microsoft-teams-settings-on-surface-hub"></a>Управление настройками Microsoft Teams на Surface Hub

Вы можете управлять настройками Microsoft Teams на Surface Hub с помощью конструктора конфигураций Windows или Microsoft Intune в Microsoft Endpoint Manager. Для внесения изменений в параметры Teams необходимо знание конструктора конфигурации Windows или Microsoft Intune. Дополнительные сведения об этих параметрах см. в следующих статьях:

- [Создание пакета подготовка для Windows 10](https://docs.microsoft.com/windows/configuration/provisioning-packages/provisioning-create-package)
- [Что такое управление устройствами Microsoft Intune?](https://docs.microsoft.com/mem/intune/remote-actions/device-management)

Конструктор конфигураций Windows — это хороший вариант, если у вас всего несколько устройств Surface Hub и вы можете легко получить к ним доступ. Если у вас много surface Hub или они размещены в удаленных расположениях, используйте Microsoft Intune в Microsoft Endpoint Manager, если она развернута в организации. Независимо от способа необходимо создать XML-файл конфигурации для изменения параметров Teams на Surface Hub.

## <a name="teams-configuration-file-syntax"></a>Синтаксис файла конфигурации Teams

Конфигурация Teams на Surface Hub определена с помощью XML-файла. XML-файл содержит все параметры, которые можно использовать для управления работой Teams. Конструктор конфигураций Windows и Microsoft Intune используют один и тот же синтаксис XML. Вот пример XML-файла конфигурации Teams:

```xml
<SurfaceHubSettings>
    <BluetoothAdvertisementEnabled>true</BluetoothAdvertisementEnabled>
    <AutoAcceptProximateMeetingInvitations>true</AutoAcceptProximateMeetingInvitations>
    <CoordinatedMeetings enabled="true"> 
        <TrustedAccounts>room@contoso.com</TrustedAccounts>
        <Settings> 
            <Audio default="false" enabled="false" />
            <Video default="false" enabled="true" /> 
        </Settings> 
    </CoordinatedMeetings>
</SurfaceHubSettings>
```

## <a name="apply-teams-settings-to-surface-hub"></a>Применение параметров Teams к Surface Hub

Применение или обновление параметров конфигурации Teams на Surface Hub с помощью конструктора конфигураций Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="use-windows-configuration-designer"></a>Использование конструктора конфигурации Windows

С помощью конструктора конфигураций Windows можно создать пакет, который можно использовать для применения параметров Teams к surface Hub. XML-файл, созданный выше, будет вклеит в конструктор конфигураций Windows для создания пакета подготовка.

> [!IMPORTANT]
> Если вы уже применили конфигурацию Teams к Surface Hub с помощью пакета подготовка и хотите изменить ее, сначала необходимо удалить существующий пакет. Дополнительные сведения см. в [теме "Удаление пакета подготовка, созданного конструктором конфигураций Windows".](#remove-a-provisioning-package-created-by-windows-configuration-designer)

Чтобы создать пакет подготовка в конструкторе конфигураций Windows, сделайте следующее:

1. Установка конструктора конфигурации Windows из Магазина Windows на локальном компьютере и его открытие
2. Выберите **"Подготовка устройств Surface Hub"** и **переключение в расширенный редактор**
3. На следующем экране раздайте **WindowsTeamSettings** Teams и выберите  >   **"Конфигурации"**
4. В поле рядом с **полем Configurations** (Конфигурации) в средней области в поле XML,созданного выше
5. Выберите **пакет экспорта** для  >  **подготовка**
6. Укаймь имя пакета подготовка в **Name** и выберите **"Далее"**  >  
7. Укажите расположение для сохранения пакета подготовка и выберите **"Далее"**
8. Выберите **"Сборка",** чтобы создать пакет и **завершить**

Наконец, создав пакет, сделайте следующее, чтобы применить его к Surface Hub:

1. Сохранение пакета предварительной подготовка, созданного выше, на USB-накопителе
2. Вставка USB-накопителя в Surface Hub
3. На устройстве Surface Hub откройте меню "Пуск", выберите **"Все приложения",** а затем выберите **"Параметры"**
4. Укайте имя пользователя и пароль администратора, а затем выберите **"Да"**
5. Перейдите в **Surface Hub,** **"Управление устройствами",** **добавьте или** удалите пакет, а затем **добавьте пакет**
6. В **области "Выберите пакет"** выберите **"Добавить"** рядом с пакетом, а затем перезапустите Surface Hub.

### <a name="use-microsoft-intune"></a>Использование Microsoft Intune

Если вы управляете вашими Surface Hubs с помощью Microsoft Intune в microsoft Endpoint Management, вы можете использовать его для применения параметров Teams к вашим Surface Hubs. Вам нужно создать новый профиль конфигурации, а затем ввести в него XML-файл, созданный выше.

> [!IMPORTANT]
> Ваши устройства Surface Hub должны быть в группе устройств, чтобы Microsoft Intune определял, к каких устройствам нужно применить профиль конфигурации. Сведения о том, как создать группу устройств, см. в группе "Добавление групп для [организации пользователей и устройств".](https://docs.microsoft.com/mem/intune/fundamentals/groups-add)

Чтобы создать профиль конфигурации для применения параметров Teams к surface Hub, сделайте следующее:

1. Во sign in to Microsoft Endpoint Manager by visiting https://endpoint.microsoft.com/
2. Перейдите к **профилям**  >  **конфигурации устройств и** выберите **"Создать профиль"**
3. В **области "Платформа"** выберите **Windows 10 и более поздней версии**
4. В **области "Профиль"** выберите **"Настраиваемый"** и нажмите кнопку **"Создать"**
5. На **вкладке "Основы"** в **области "Имя"** в качестве описательного имени для профиля конфигурации выберите "Далее" 
6. На **вкладке "Параметры конфигурации"** выберите **"Добавить"**
7. В области **добавления строк** сделайте следующее:
    1. В качестве описательного имени и (при желании) описания добавляемого параметра Teams
    2. В **OMA-URI введите**`./Vendor/MSFT/SurfaceHub/InBoxApps/Teams/Configurations`
    3. В **типе данных** выберите **строку (XML-файл)**
    4. Откройте файл в браузере, выберите созданный выше XML-файл и **откройте**
8. Выберите **"Добавить"** и **"Далее"**
9. **Убедитесь, что** на вкладке "Задания" **выбраны** **выбранные группы.**
10. В **группе "Выбранные группы"** выберите "Выбрать группы",  а затем выберите группу, которая содержит ваши узлы Surface Hub, а затем **выберите**
11. Выберите **"Далее",** **"Далее"**
12. На **кнопке "Рецензить+ создать" выберите** **"Создать"**

## <a name="remove-teams-settings-from-a-surface-hub"></a>Удаление параметров Teams из Surface Hub

Удалите параметры конфигурации Teams на Surface Hub с помощью конструктора конфигурации Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="remove-a-provisioning-package-created-by-windows-configuration-designer"></a>Удаление пакета подготовка, созданного конструктором конфигураций Windows

Если вы применили параметры Teams к Surface Hub с помощью пакета подготовка, созданного конструктором конфигурации Windows, чтобы удалить пакет и его параметры, с помощью следующих действий:

1. На устройстве Surface Hub откройте меню "Пуск", выберите **"Все приложения",** а затем выберите **"Параметры"**
2. Укайте имя пользователя и пароль администратора, а затем выберите **"Да"**
3. Перейдите в **Surface Hub** и **перейдите в "Управление** устройствами", а затем **добавьте или удалите пакеты для подготовка**
4. Рядом с пакетом, который вы хотите удалить, выберите **"Удалить"**
5. Перейдите на **Surface Hub,** а затем **в приложение & функций**
6. Найдите **Microsoft Teams для Surface Hub** и выберите **"Дополнительные параметры"**
7. Выберите **"Сброс",** а затем **еще раз сброс**
8. Перезапустите Surface Hub

### <a name="remove-settings-applied-by-microsoft-intune"></a>Удаление параметров, примененных Microsoft Intune

Если вы применили параметры Teams к Surface Hub с помощью Microsoft Intune в microsoft Endpoint Management, удалите профиль конфигурации и его параметры с помощью следующих действий:

1. Во sign in to Microsoft Endpoint Manager by visiting https://endpoint.microsoft.com/
2. Перейти к  >  **профилям конфигурации устройств**
3. Выберите профиль конфигурации, содержащий параметры согласованных собраний, которые вы хотите удалить
4. На странице сведений профиля конфигурации выберите **"Удалить" и** "ОК" 

После удаления профиля конфигурации, который содержал параметры согласованных собраний для Surface Hub, сбросить приложение Teams на Surface Hub с помощью следующих действий:

1. На устройстве Surface Hub откройте меню "Пуск", выберите **"Все приложения",** а затем выберите **"Параметры"**
2. Укайте имя пользователя и пароль администратора, а затем выберите **"Да"**
3. Перейдите на **Surface Hub,** а затем **в приложение & функций**
4. Найдите **Microsoft Teams для Surface Hub** и выберите **"Дополнительные параметры"**
5. Выберите **"Сброс",** а затем **еще раз сброс**
6. Перезапустите Surface Hub
