# <a name="manage-microsoft-teams-settings-on-surface-hub"></a>Управление параметрами Microsoft Teams на Surface Hub

Вы можете управлять параметрами Microsoft Teams на Surface Hub с помощью конструктора конфигураций Windows или Microsoft Intune в Microsoft Endpoint Manager. Знание конструктора конфигураций Windows или Microsoft Intune требуется для внесения изменений в параметры Teams. Дополнительные сведения об этих параметрах см. в следующих статьях:

- [Создание пакета подготовки для Windows 10](/windows/configuration/provisioning-packages/provisioning-create-package)
- [Что такое Microsoft Intune устройств?](/mem/intune/remote-actions/device-management)

Конструктор конфигураций Windows — это хороший вариант, если у вас всего несколько устройств Surface Hub и вы можете легко получить к ним доступ. Если у вас много устройств Surface Hub или они находятся в удаленных расположениях, используйте Microsoft Intune microsoft Endpoint Manager, если они развернуты в вашей организации. Независимо от выбранного метода необходимо создать XML-файл конфигурации для внесения изменений в параметры Teams на Surface Hub.

## <a name="teams-configuration-file-syntax"></a>Синтаксис файла конфигурации Teams

Конфигурация Teams на Surface Hub определяется с помощью XML-файла. XML-файл содержит все параметры, которые можно использовать для управления принципом работы Teams. Конструктор конфигураций Windows Microsoft Intune использовать один и тот же синтаксис XML. Ниже приведен пример XML-файла конфигурации Teams:

```xml
<SurfaceHubSettings>
    <BluetoothAdvertisementEnabled>true</BluetoothAdvertisementEnabled>
    <AutoAcceptProximateMeetingInvitations>true</AutoAcceptProximateMeetingInvitations>
    <CoordinatedMeetings enabled="true"> 
        <TrustedAccounts>room@contoso.com</TrustedAccounts>
        <Settings> 
            <Audio default="false" enabled="false" />
            <Video default="false" enabled="true" /> 
        </Settings> 
    </CoordinatedMeetings>
</SurfaceHubSettings>
```

## <a name="apply-teams-settings-to-surface-hub"></a>Применение параметров Teams к Surface Hub

Примените или обновите параметры конфигурации Teams на Surface Hub с помощью конструктора конфигураций Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="use-windows-configuration-designer"></a>Использование конструктора конфигураций Windows

Конструктор конфигураций Windows можно использовать для создания пакета подготовки, который можно использовать для применения параметров Teams к устройствам Surface Hub. Чтобы создать пакет подготовки, вставьте созданный ранее XML-файл в конструктор конфигураций Windows.

> [!IMPORTANT]
> Если вы уже применили конфигурацию Teams к Surface Hub с помощью пакета подготовки и хотите изменить ее, сначала необходимо удалить существующий пакет подготовки. Дополнительные сведения см. в разделе ["Удаление пакета подготовки,созданного конструктором конфигураций Windows"](#remove-a-provisioning-package-created-by-windows-configuration-designer).

Выполните следующие действия, чтобы создать пакет подготовки в конструкторе конфигураций Windows:

1. Установка конструктора конфигураций Windows из Магазина Windows на локальном компьютере и его открытие
2. Выберите **"Подготовка устройств Surface Hub** " и **перейдите в расширенный редактор**
3. На следующем экране разверните **WindowsTeamSettings** > **Teams** и выберите **"Конфигурации"**.
4. В поле рядом с разделом **"** Конфигурации" в средней области вставьте одну строку XML, созданную ранее.
5. Выбор **пакета подготовки** > **экспорта**
6. Укажите имя пакета подготовки в **поле "Имя** " и нажмите кнопку **"Далее** > **"**
7. Укажите расположение для сохранения пакета подготовки и нажмите кнопку " **Далее".**
8. Выберите **"Сборка**", чтобы создать пакет подготовки, а затем — **"Готово"**.

Наконец, после создания пакета подготовки выполните следующие действия, чтобы применить пакет подготовки к Surface Hub:

1. Сохраните созданный ранее пакет подготовки на USB-накопитель.
2. Вставка USB-накопителя в Surface Hub
3. На Устройстве Surface Hub откройте меню "Пуск", выберите "**Все приложения**", а затем выберите "**Параметры"**.
4. Укажите имя пользователя и пароль администратора, а затем нажмите кнопку **"Да"**.
5. Перейдите **к Surface Hub**, **управлению устройствами**, добавьте или удалите пакет **подготовки**, а затем **добавьте пакет**
6. В **разделе "Выбор пакета" выберите** **"** Добавить рядом с пакетом подготовки", а затем перезапустите Surface Hub.

### <a name="use-microsoft-intune"></a>Использование Microsoft Intune

Если устройства Surface Hub управляются с Microsoft Intune в Microsoft Endpoint Management, его можно использовать для применения параметров Teams к устройствам Surface Hub. Вы создадите новый профиль конфигурации, а затем вставьте в него созданный ранее XML-файл.

> [!IMPORTANT]
> Устройства Surface Hub должны быть в группе устройств, чтобы Microsoft Intune могли определить, к каку устройствам следует применить профиль конфигурации. Сведения о создании группы устройств см. в разделе "Добавление [групп для организации пользователей и устройств"](/mem/intune/fundamentals/groups-add).

Выполните следующие действия, чтобы создать профиль конфигурации для применения параметров Teams к устройствам Surface Hub:

1. Войдите в Microsoft Endpoint Manager, посетивhttps://endpoint.microsoft.com/
2. Перейдите **к профилям** > **конфигурации устройств** и выберите " **Создать профиль"**
3. В **разделе "Платформа****" выберите Windows 10 и более поздних версий.**
4. В **разделе "Профиль**" выберите **"Пользовательский**" и нажмите кнопку " **Создать".**
5. На **вкладке "** Основные сведения" в **поле "** Имя" укажите описательное имя профиля конфигурации и нажмите кнопку " **Далее".**
6. На **вкладке "Параметры конфигурации**" выберите "**Добавить"**.
7. В области **добавления** строк выполните следующие действия.
    1. Укажите описательное имя и(или) описание добавляемого параметра Teams.
    2. В **OMA-URI введите**`./Vendor/MSFT/SurfaceHub/InBoxApps/Teams/Configurations`
    3. В **поле "Тип данных**" выберите **string (XML-файл)**
    4. Откройте браузер файлов, выберите созданный ранее XML-файл **и откройте**
8. Нажмите **кнопку "** Добавить", а затем **— "Далее".**
9. На **вкладке "** Назначения" убедитесь, что для **параметра "** Назначить" задано значение **"Выбранные группы"**.
10. В **разделе "Выбранные группы**" выберите "Выбрать **группы** " и выберите группу, содержащую ваши Устройства Surface Hub, а затем нажмите кнопку **"Выбрать".**
11. Нажмите **кнопку "Далее**", **"Далее"**
12. На странице **"Просмотр и создание" нажмите** кнопку **"Создать"**

## <a name="remove-teams-settings-from-a-surface-hub"></a>Удаление параметров Teams из Surface Hub

Удалите параметры конфигурации Teams на Surface Hub с помощью конструктора конфигураций Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="remove-a-provisioning-package-created-by-windows-configuration-designer"></a>Удаление пакета подготовки, созданного конструктором конфигураций Windows

Если вы применили параметры Teams к Surface Hub с помощью пакета подготовки, созданного конструктором конфигураций Windows, выполните следующие действия, чтобы удалить пакет и его параметры:

1. На Устройстве Surface Hub откройте меню "Пуск", выберите "**Все приложения**", а затем выберите "**Параметры"**.
2. Укажите имя пользователя и пароль администратора, а затем нажмите кнопку **"Да"**.
3. Перейдите **в Surface Hub**, **управление устройствами** , а **затем добавьте или удалите пакет подготовки.**
4. Рядом с пакетом подготовки, который требуется удалить, выберите " **Удалить".**
5. Перейдите к **Surface Hub** , а затем к **функциям & приложений**
6. **Найдите Microsoft Teams для Surface Hub** и выберите "**Дополнительные параметры"**
7. Нажмите **кнопку "Сбросить**", а затем **снова сбросьте**
8. Перезапуск Surface Hub

### <a name="remove-settings-applied-by-microsoft-intune"></a>Удаление параметров, примененных Microsoft Intune

Если вы применили параметры Teams к Surface Hub с помощью Microsoft Intune в Microsoft Endpoint Management, выполните следующие действия, чтобы удалить профиль конфигурации и его параметры:

1. Войдите в Microsoft Endpoint Manager, посетивhttps://endpoint.microsoft.com/
2. Переход к **профилям** > **конфигурации устройств**
3. Выберите профиль конфигурации, содержащий параметры координированного собрания, которые нужно удалить.
4. На странице сведений о профиле конфигурации нажмите **кнопку "Удалить**" и нажмите кнопку "**ОК"**.

После удаления профиля конфигурации, содержащего параметры координированного собрания для Surface Hub, выполните следующие действия, чтобы сбросить приложение Teams на Surface Hub:

1. На Устройстве Surface Hub откройте меню "Пуск", выберите "**Все приложения**", а затем выберите "**Параметры"**.
2. Укажите имя пользователя и пароль администратора, а затем нажмите кнопку **"Да"**.
3. Перейдите к **Surface Hub** , а затем к **функциям & приложений**
4. **Найдите Microsoft Teams для Surface Hub** и выберите "**Дополнительные параметры"**
5. Нажмите **кнопку "Сбросить**", а затем **снова сбросьте**
6. Перезапуск Surface Hub
