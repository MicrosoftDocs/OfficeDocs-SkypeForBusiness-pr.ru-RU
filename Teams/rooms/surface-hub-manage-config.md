# <a name="manage-microsoft-teams-settings-on-surface-hub"></a>Управление Microsoft Teams на Surface Hub

Вы можете управлять Microsoft Teams на Surface Hub с помощью Windows или Microsoft Intune в Microsoft Endpoint Manager. Для внесения Windows параметров Microsoft Intune настройки требуется знание Teams конфигурации. Дополнительные сведения об этих параметрах см. в следующих статьях:

- [Создание пакета подготовка для Windows 10](/windows/configuration/provisioning-packages/provisioning-create-package)
- [Что Microsoft Intune управления устройствами?](/mem/intune/remote-actions/device-management)

Windows Конструктор конфигураций — это хороший вариант, если у вас всего несколько Surface Hub и вы можете легко получить к ним доступ. Если у вас много surface Hub или они размещены в удаленных расположениях, используйте Microsoft Intune в Microsoft Endpoint Manager, если он развернут в вашей организации. Вне зависимости от того, какой метод вы выбрали, необходимо создать XML-файл конфигурации для внесения изменений Teams параметры на Surface Hub.

## <a name="teams-configuration-file-syntax"></a>Teams синтаксис файла конфигурации

Teams конфигурация Surface Hub определяется с помощью XML-файла. XML-файл содержит все параметры, которые можно использовать для управления Teams работой. Оба Windows и Microsoft Intune синтаксис XML. Вот пример XML-файла конфигурации Teams конфигурации:

```xml
<SurfaceHubSettings>
    <BluetoothAdvertisementEnabled>true</BluetoothAdvertisementEnabled>
    <AutoAcceptProximateMeetingInvitations>true</AutoAcceptProximateMeetingInvitations>
    <CoordinatedMeetings enabled="true"> 
        <TrustedAccounts>room@contoso.com</TrustedAccounts>
        <Settings> 
            <Audio default="false" enabled="false" />
            <Video default="false" enabled="true" /> 
        </Settings> 
    </CoordinatedMeetings>
</SurfaceHubSettings>
```

## <a name="apply-teams-settings-to-surface-hub"></a>Применение Teams к Surface Hub

Применив или Teams параметры конфигурации на Surface Hub с помощью Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="use-windows-configuration-designer"></a>Использование Windows конфигурации

С помощью Windows Configuration Designer можно создать пакет, который можно использовать для Teams параметров surface Hub. XML-файл, созданный выше, в Windows, чтобы создать пакет.

> [!IMPORTANT]
> Если вы уже применили Teams к своей Surface Hub и хотите изменить ее, сначала необходимо удалить существующий пакет. Дополнительные сведения см. в теме Удаление пакета [подготовка, созданного](#remove-a-provisioning-package-created-by-windows-configuration-designer)Windows configuration Designer.

Чтобы создать пакет подготовка в конструкторе конфигураций Windows, сделайте следующее:

1. Установите Windows configuration Designer из Windows Store на локальном компьютере и откройте его
2. Выберите **Подготовка Surface Hub и** **переключение на расширенный редактор**
3. На следующем экране раздайте **список WindowsTeamSettings**  >  **Teams** выберите **Конфигурации.**
4. В поле рядом с **полем Configurations** (Конфигурации) в средней области в поле XML- или XML-поля, созданного выше.
5. Выберите **экспорт**  >  **пакета подготовка**
6. Укай имя пакета подготовка в **name (Имя)** и выберите **Далее**  >  
7. Укажите расположение для сохранения пакета подготовка и выберите **Далее**
8. Выберите **"Сборка",** чтобы создать пакет для подготовка, а **затем** готово

Наконец, создав пакет, сделайте следующее, чтобы применить его к Surface Hub.

1. Сохранение пакета, созданного выше, на USB-накопителе
2. Вставка USB-накопителя в Surface Hub
3. На Surface Hub меню Пуск выберите Все **приложения**, а затем **Параметры**
4. Укайте имя пользователя и пароль администратора, а затем **выберите** Да
5. Перейдите **в Surface Hub**, Управление **устройствами**, **Добавьте или** удалите пакет, а затем Добавьте **пакет**
6. В **области Выберите пакет** выберите **Добавить** рядом с пакетом, а затем перезапустите Surface Hub

### <a name="use-microsoft-intune"></a>Использование Microsoft Intune

Если вашими Surface Hubs управляется с Microsoft Intune в управлении конечными точками Майкрософт, вы можете использовать его для применения Teams параметров surface Hub. Вам нужно создать новый профиль конфигурации, а затем в него ввести XML-файл, созданный выше.

> [!IMPORTANT]
> Ваши устройства Surface Hub должны быть в группе устройств, чтобы Microsoft Intune определять устройства, к которым нужно применить профиль конфигурации. Сведения о том, как создать группу устройств, см. в группе Добавление групп для [организации пользователей и устройств.](/mem/intune/fundamentals/groups-add)

Чтобы создать профиль конфигурации для применения параметров Teams устройств Surface Hub, сделайте следующее:

1. Во войти в Microsoft Endpoint Manager, посетивhttps://endpoint.microsoft.com/
2. Перейдите к  >  **профилям конфигурации устройств и** выберите Создать **профиль**
3. В **области Платформа** выберите Windows 10 и более **поздней**
4. В **области Профиль** выберите **Настраиваемые** и нажмите кнопку **Создать.**
5. На **вкладке Основы** в **области Имя** укайте описательное имя профиля конфигурации и выберите **Далее.**
6. На **вкладке "Параметры конфигурации"** выберите **"Добавить"**
7. В области **Добавить строку** сделайте следующее:
    1. В качестве описательного имени и (при желании) описания добавляемого параметра Teams имени
    2. В **OMA-URI введите**`./Vendor/MSFT/SurfaceHub/InBoxApps/Teams/Configurations`
    3. В **области Тип данных** выберите **Строка (XML-файл)**
    4. Откройте файл в браузере, выберите XML-файл, созданный выше, **и** откройте
8. Выберите **"Добавить"** и **"Далее"**
9. На **вкладке** "Задания"  убедитесь, что для задания выбрано **"Выбранные группы"**
10. В **группе Выбранные группы** выберите Выберите группы, которые нужно включить, и выберите группу, которая содержит ваши Surface Hubs, а затем выберите **Выбрать** 
11. Выберите **Далее**, **Далее**
12. На **рецензии + создать** выберите **Создать**

## <a name="remove-teams-settings-from-a-surface-hub"></a>Удаление Teams параметров из Surface Hub

Удалите Teams конфигурации Surface Hub с помощью Windows или Microsoft Intune в Microsoft Endpoint Manager.

### <a name="remove-a-provisioning-package-created-by-windows-configuration-designer"></a>Удаление пакета подстановки, созданного Windows Configuration Designer

Если вы применили Teams к Surface Hub с помощью пакета, созданного с помощью конструктора конфигураций Windows, удалите пакет и его параметры с помощью следующих действий:

1. На Surface Hub меню Пуск выберите Все **приложения**, а затем **Параметры**
2. Укайте имя пользователя и пароль администратора, а затем **выберите** Да
3. Перейдите **в Surface Hub**, управление **устройствами,** а затем Добавьте или **удалите пакет подготовка**
4. Рядом с пакетом, который вы хотите удалить, выберите **удалить**
5. Перейдите **в Surface Hub,** а затем **приложения & функции**
6. Найдите **Microsoft Teams для Surface Hub** и выберите **Дополнительные параметры**
7. Выберите Сброс , а затем **сброс еще** раз
8. Перезапустите Surface Hub

### <a name="remove-settings-applied-by-microsoft-intune"></a>Удаление параметров, примененных Microsoft Intune

Если вы применили Teams к Surface Hub с помощью Microsoft Intune в microsoft Endpoint Management, удалите профиль конфигурации и его параметры с помощью следующих действий:

1. Во войти в Microsoft Endpoint Manager, посетивhttps://endpoint.microsoft.com/
2. Перейти к   >  **профилям конфигурации устройств**
3. Выберите профиль конфигурации, содержащий параметры "Координированные собрания", которые вы хотите удалить.
4. На странице сведений профиля конфигурации выберите **Удалить, а** затем **ОК**

После удаления профиля конфигурации, который содержал параметры координированные собрания для Surface Hub, сбросить приложение Teams на странице Surface Hub:

1. На Surface Hub меню Пуск выберите Все **приложения**, а затем **Параметры**
2. Укайте имя пользователя и пароль администратора, а затем **выберите** Да
3. Перейдите **в Surface Hub,** а затем **приложения & функции**
4. Найдите **Microsoft Teams для Surface Hub** и выберите **Дополнительные параметры**
5. Выберите Сброс , а затем **сброс еще** раз
6. Перезапустите Surface Hub
