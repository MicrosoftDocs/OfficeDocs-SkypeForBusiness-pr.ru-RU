# <a name="managed-meeting-rooms-in-microsoft-teams"></a>Управляемые помещения для собраний в Microsoft Teams

Управляемые комнаты для собраний — это облачная служба ИТ-управления и мониторинга, позволяющая хранить устройства в Microsoft Teams, а также следить за тем, чтобы они были заблаговременно обновлены и поддерживали среду, оптимизированную для удобного взаимодействия с пользователем.

Управляемые комнаты для собраний — для организаций, которые хотят улучшить работу с Конференцией, и быстро масштабировать их занимаемое ими место с помощью упреждающего управления устройствами в Microsoft Teams. Служба позволяет организациям экономить время, деньги и квалификацию, которые в противном случае будут потрачены на управление комнатой.  

Служба управляемых комнат для собраний основана на надежном фундаменте Microsoft, которая включает в себя Управление конечными точками, машинное обучение и интеллектуальную автоматизацию. Управляемые комнаты для собраний помогают организациям обеспечивать высококачественное взаимодействие с несколькими удаленными сайтами, которые могут потребовать больших темпов развития Организации.

## <a name="service-prerequisites"></a>Необходимые условия для служб

Служба управляемой комнаты для собраний — это дополнение к лицензии на комнату. Комнатам Microsoft Teams требуется отдельный пользователь или лицензия Microsoft Teams для комнаты.

Для управляемых комнат для собраний для обеспечения качества обслуживания, зарегистрированные комнаты должны быть [поддерживаемыми устройствами Microsoft Teams](requirements.md#hardware-requirements), поэтому необходимо подготовить среду с использованием всех [необходимых URL-адресов Office 365 и диапазонов IP-адресов](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges?redirectSourcePath=%252fen-us%252farticle%252fOffice-365-URLs-and-IP-address-ranges-8548a211-3fe7-47cb-abb1-355ea5aa88a2) , которые будут полностью разрешены и доступны для устройств с комнатой для собраний. Дополнительные сведения можно найти в разделе [Подготовка среды](rooms-prep.md) в документации Microsoft Teams комнаты.

Как поставщик управляемой службы, корпорация Майкрософт должна работать с ответственными заинтересованными лицами в помещениях в Организации, чтобы сохранить помещения в рабочее состояние и устранить проблемы. Вы несете ответственность за несертифицированные устройства, используемые в помещениях для собраний, сторонние или пользовательские программы, которые вы выбрали для развертывания на устройствах для собраний, и придерживайтесь рекомендаций по управлению, рекомендованным корпорацией Майкрософт.

Ваша организация или партнер, подписанные вашей организацией, должны быть готовы к исправлению проблем с последними, обнаруженными службой.

Управляемые комнаты для собраний SOC предоставляют первый ответ и необходимые эскалация билетов, инициированных системой или вашей организацией.

|**Область**   |Проблемы, связанные с приложением "комната для собраний", устройствами для помещения и периферийными приложениями, входят в состав поддержки управляемых комнат для собраний.|
|**Посетител**  |Только операции         |
|**Поддерживаемые языки**    |Английский         |
|**Поддержка доступности на портале**   | Круглосуточная      |
|**Начальное время отклика для управляемых комнат для собраний SOC**   |  4 часа       |
|**Автоматическое замыкание билетов в случае отсутствия на работе пользователя**    |Билеты, инициированные клиентом: 3 рабочих дня<br>Билеты, инициированные системой: 15 рабочих дней</li>   </ul>      |
|**Перенаправление на более широкие возможности поддержки Майкрософт**   |Проблемы, связанные с другим продуктом Майкрософт за пределами области управляемых комнат для собраний, направляются в соответствующую группу поддержки.         |

## <a name="data-collection"></a>Сбор данных

Служба управляемой комнаты для собраний отслеживает устройства, собирает данные устройства, а также позволяет корпорации Майкрософт получать удаленный доступ к устройствам и управлять ими. Служба управляемой комнаты для собраний не получает доступ к содержимому собрания.

Собранные данные специфичны для учетной записи комнаты, а не отдельного пользователя. Случайные ссылки на отдельного пользователя могут присутствовать в журнале активности при использовании устройства. Собранные данные обрабатываются корпорацией Майкрософт в соответствии с заявлением [о конфиденциальности корпорации Майкрософт](https://aka.ms/privacy).
