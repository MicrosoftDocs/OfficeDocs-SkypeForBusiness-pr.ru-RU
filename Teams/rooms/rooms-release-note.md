# <a name="release-notes-for-microsoft-teams-rooms"></a>Заметки о выпуске для Комнаты Microsoft Teams

В этой статье рассматриваются совокупные улучшения в Комнаты Microsoft Teams.

Существует два типа обновлений для Комнаты Teams: обновления приложений Комнаты Teams и веб-клиент Teams.

Комнаты Teams обновления приложений выполняются либо через Microsoft Store, либо через [обновление вручную](manual-update.md). Обновления применяются к приложению универсальная платформа Windows (UWP), установленному локально на устройстве.

Обновления веб-клиента Teams выполняются через службы доставки веб-приложений Teams. Веб-клиент Teams — это облачная служба, которая не требует обновления локального приложения UWP, установленного на устройстве.

Дополнительные сведения о том, как выполняется обновление Teams, см. в разделе [Процесс обновления Teams](../teams-client-update.md).

Комнаты Teams регулируется современной политикой жизненного цикла. Дополнительные сведения см. в разделе [Процесс обновления Teams](../teams-client-update.md#servicing-agreement).

### <a name="415580-12152022"></a>4.15.58.0 (12/15/2022)

Представлено в этом обновлении:

- Улучшение уведомлений на собрании
- Чат собрания в коллекции, большой коллекции и режиме "Вместе <sup>" 1</sup>
- Запуск доски в собраниях Teams
- Управление списком видео в помещении подгонки к кадру
- Поддержка собраний переполнения
- Присоединяйтесь к собраниям Zoom по идентификатору с помощью прямого гостевого присоединения. Поддержка присоединения к собраниям Zoom по идентификатору была добавлена в этом выпуске и будет включена в ближайшие недели.

- Исправление сведений о лицензиях в параметрах, показывающих отсутствие лицензии, когда Teams не вошел в систему
- Новые обои по умолчанию (яркий флаг) и четыре других новых обои

<sup>1</sup> Конфигурацию администратора для скрытия чата собрания из всех макетов собраний с помощью XML можно найти в разделе [Удаленное управление параметрами Комнаты Microsoft Teams консоли с помощью XML-файла конфигурации](xml-config-file.md).

### <a name="release-to-market-1272016"></a>Выпуск на рынок (07.12.2016)

**Функции:**

 **Оптимизация для Skype для бизнеса**

- Присоединение к собранию в Skype одним нажатием кнопки
- Интерфейс собрания Skype, оптимизированный для помещений с заполнением экрана HD-видео и широкополосным звуком HD
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате

 **Преобразование любой комнаты для собраний**

- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Повторное использование существующих инвестиций в передней части помещения или проекторов
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенный проводной прием для проецирования общего доступа к рабочему столу в комнату и собрание Skype

 **Простота развертывания и управления**

- Устройство always-on, которое автоматически активирует дисплеи при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Отслеживаются и управляются как Windows 10 Корпоративная устройство через Intune и Configuration Manager (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
