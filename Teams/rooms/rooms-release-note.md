## <a name="microsoft-teams-rooms-feature-introduction-and-issue-resolution"></a>Функции Microsoft Teams комнаты введение и устранение проблем
### <a name="43420-03022020"></a>4.3.42.0 (03/02/2020)

Представлено в этом обновлении:

- Обновления политики для "обновления для Windows для бизнеса" 
- Исправление ошибок, возникающих при составлении отчетов о событиях устройств в мониторе Azure

### <a name="43330-1102020"></a>4.3.33.0 (1/10/2020)

Представлено в этом обновлении:

- Исправление проблемы с изменением размера и мерцания окна, которое отображается в некоторых конфигурациях
- Обработка календаря для собраний третьих лиц удалена
- Параметр состояния кортаны удален

### <a name="43230-12132019"></a>4.3.23.0 (12/13/2019)

Представлено в этом обновлении:

- Автоматический ответ на звонки и параметры администратора для управления этим устройством
- Параметры администратора устройства обновить пользовательский интерфейс с добавлением конфигурации устройства в разделе "о программе"
- Управление комнатой назад к главному экрану
- SKU комнаты для собраний, доступные в GCC
- Поддержка видеокамеры для системы Surface Pro (минимальная требуемая сборка приложения: 4.2.4.0)

### <a name="4240-10072019"></a>4.2.4.0 (10/07/2019)

Представлено в этом обновлении:

- Поддержка Windows 10 1903. Обновление для Windows 10 1903 предлагается через несколько дней после обновления приложения
- Исправления для экранной клавиатуры не отображаются надежно

### <a name="41220-08152019"></a>4.1.22.0 (08/15/2019)

Представлено в этом обновлении:

- Новая функция для работы с содержимым, которая позволяет пользователям интеллектуально включать традиционные доски в собрания Teams
- Дополнительные усовершенствования в пользовательском интерфейсе консоли для уменьшения порядка и перемещения параметров на новую боковую панель, доступ к которой осуществляется с помощью других возможностей на консоли
- Кнопка "отключить доступ к лотку", если кабель локального содержимого не подключен или не подключена камера содержимого
- Исправлена ошибка, связанная с сенсорной клавиатурой, в которой он не появился в первый раз после перезапуска системы МТР
- Исправления качества и надежности

### <a name="401050-07102019"></a>4.0.105.0 (07/10/2019)

Представлено в этом обновлении:

- Приложение Skype для помещения системы из магазина на "комнаты Microsoft Teams"
- Пользовательский интерфейс консоли Microsoft Teams был выровнен в Microsoft Teams
- Обновление темы: сохранение пользовательского фонового изображения только на лицевой стороне комнаты, в то время как создание фонового цвета на фоне, чтобы обеспечить соответствие цветовой контрастности элементам управления в пользовательском интерфейсе: требования к специальным возможностям
- Универсальная линейка для звонков в Skype для собраний по звонкам и собраниям в Teams для обеспечения соответствия требованиям в Microsoft Teams PC/Web/Mobile Clients<sup>1</sup>
- Оценка отзыва о звонке после звонков<sup>и собраний</sup> по качеству
- Прием/прорисовка доски Microsoft в помещениях Microsoft Teams на лицевую часть экрана, когда она предоставлена на компьютере с Windows/Web/Mobile Teams<sup>1</sup> <sup>2</sup>
- Поддержка обновлений для Windows 10 версии 1809 была удалена из-за проблем совместимости с клиентом комнат Microsoft Teams. Поддержка Windows 10 версии 19H1 будет добавлена в будущих выпусках

<sup>1</sup> выпуск службы Microsoft Teams с использованием колец групп. Эта функция может быть доступна более ранней или более поздней версии, чем обновление клиента 4.0.105.0

<sup>2</sup> для включения доски Microsoft требуется ИТ – администраторы. Кроме того, если вы включили экран с сенсорным экраном, вы должны откалибровать несколько сенсорных дисплеев с помощью параметров Windows с именем входа администратора устройства, чтобы начать использовать доску Microsoft для совместной работы с презентацией, доступ к которой открыт на собрании Teams.

### <a name="40850-0482019"></a>4.0.85.0 (04/8/2019)

Представлено в этом обновлении:

- Устранена проблема, связанная с функцией "Обратная связь" 
- Оптимизация для подготовки к предустановленному обновлению устройства в Microsoft Teams для Windows 10 версии 1809

### <a name="40780-03142019"></a>4.0.78.0 (03/14/2019)

Представлено в этом обновлении:

- Исправление ошибки "зависание при запуске приложения", на которой изменяются уязвимые устройства на устаревшем этапе сборки Windows 10 RS2. 


### <a name="40760-03042019"></a>4.0.76.0 (03/04/2019)

Представлено в этом обновлении:

- Клавиша DTMF для собраний по сети Microsoft Teams P2P и КОММУТИРУЕМых звонков. Чтобы сделать Microsoft Teams клиентом по умолчанию, администраторы должны установить для Истеамсдефаултклиент значение true.
- Закрепите входящее видео удаленного участника на весь экран перед отображением комнаты. Использование команды "закрепить" в списке участников на консоли
- Улучшенные уведомления о выдачах за добавление уведомления о помещении
- Значок "приведение" на лицевой стороне экрана удаляется, когда на устройстве Microsoft Teams не включена функция маяка Bluetooth
- Исправление проблемы с управлением громкостью в собраниях Teams

### <a name="release-to-market-1272016"></a>Выпуск от выпуска к рынку (12/7/2016)

**Функция (-ы):**

 **Оптимизация для Skype для бизнеса**

- Присоединение к собранию в Skype одним нажатием кнопки
- Возможности собраний Skype, оптимизированные для комнат с помощью высококачественного видео и высокостандартного звукового сопровождения HD
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате

 **Преобразование любой комнаты для собраний**
 
- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Повторное использование существующих инвестиций на экране внешнего вида или проектора комнаты
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенные проводные функции приема для демонстрации совместного доступа к рабочему столу в комнате и в собрании Skype

 **Простота развертывания и управления**
 
- Постоянное устройство, которое автоматически пробуждает дисплеи при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Мониторинг и управление как корпоративным устройством с Windows 10 с помощью Intune и Configuration Manager (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
