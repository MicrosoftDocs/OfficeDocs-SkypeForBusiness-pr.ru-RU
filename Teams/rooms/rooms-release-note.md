# <a name="release-notes-for-microsoft-teams-rooms"></a>Заметки о выпуске Комнаты Microsoft Teams

В этой статье рассматриваются накопительные улучшения в Комнаты Microsoft Teams.

Существует два типа обновлений для Комнаты Teams: Комнаты Teams обновления приложения и Teams веб-клиента. 

Комнаты Teams обновления приложения происходят либо через Microsoft Store, либо [вручную](manual-update.md). Это обновляет приложение универсальная платформа Windows (UWP), установленное локально на устройстве.

Teams обновления веб-клиента происходят через службы Teams веб-приложений. Это облачная служба, которая не требует обновления локального приложения UWP, установленного на устройстве.

Дополнительные сведения о том, как Teams обновления, см[. в Teams процесса обновления.](../teams-client-update.md)

Комнаты Teams управляется современной политикой жизненного цикла. [Дополнительные Teams процесса](../teams-client-update.md#servicing-agreement) обновления.

- Единый элемент управления списком с классическим клиентом Teams со структурированным группированием собраний, параметрами собраний и элементами управления для выступающих и участников, созданием порядка сортировки рук и возможностью приглашать пользователей из чата или приглашения на собрание непосредственно из списка

- Интересное или закрепление нескольких участников Teams собраниях
- Поддержка больших собраний с помощью элементов управления докладчика и участников, доступных путем касания участника из списка

- Поддержка реагирования на Teams собраниях

### <a name="release-to-market-1272016"></a>Выпуск на рынок (7.01.2016)

**Функции:**

 **Оптимизация для Skype для бизнеса**

- Присоединение к собранию в Skype одним нажатием кнопки
- Skype собраний, оптимизированных для помещений с видео hd-видео с заполнением экрана и широкоштатным звуком HD
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате

 **Преобразование любой комнаты для собраний**

- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Повторное использование существующих инвестиций в внешний интерфейс дисплея комнаты или проекторов
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенный проводной прием данных для совместного использования рабочего стола проекта в комнате и на Skype собрания

 **Простота развертывания и управления**

- Постоянное устройство, которое автоматически пробуждает дисплеи при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Отслеживается и управляется как Windows 10 Корпоративная через Intune и Configuration Manager (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
