## <a name="microsoft-teams-rooms-feature-introduction-and-issue-resolution"></a>Введение в комнаты Microsoft Teams и решение проблем

### <a name="48250-04222021"></a>4.8.25.0 (04/22/2021)

В этом обновлении представлены:
- Исправление проблемы, из-за которой сведения о помещении на консолях комнат Teams не выкаживались для учетных записей комнат, скрытых из глобального списка адресов (GAL).

> [!NOTE]
> Клиенты GCCH могут скачать пакет обновления с устройства Microsoft [Teams Rooms,](manual-update.md) обновляемого вручную


### <a name="48190-04062021"></a>4.8.19.0 (04/06/2021)

В этом обновлении представлены:
- Поддержка правительственных сообществ в облаке (GCCH) для комнат Teams. Клиенты GCCH с существующими устройствами Teams Rooms могут скачать версию 4.8.19.0 с ручного обновления [устройства Комнаты Microsoft Teams](manual-update.md)
- Присоединяйтесь к собраниям с повышением качества видео (поддержка 720p) и получите галерею видео участников
- Баннер сбоя при входе в Skype для бизнеса удален для режима по умолчанию Teams. Это изменение поддерживает удаление инфраструктуры Skype для бизнеса
- Средства обработки связей для собраний Teams теперь обрабатывают безопасные ссылки Advanced Thread Protection в Microsoft Defender, что обеспечивает беспрепятственное присоединение к внешним командам
- Исправление проблемы масштабирования общего содержимого в собраниях Skype для бизнеса, если на компьютере sharer установлен пользовательский DPI в Windows
- Исправления для качества и надежности

### <a name="47190-02032021"></a>4.7.19.0 (02/03/2021)

В этом обновлении представлены:
- Исправления для качества и надежности

### <a name="47150-12112020"></a>4.7.15.0 (12/11/2020)

В этом обновлении представлены:

- Поделиться звуком HDMI с участниками собрания в собрании Teams
- Голосовые навыки Кортаны (предварительная версия)
- Запретить звук на основе разрешений на звук, когда комната Teams присоединяется в качестве участника. Дополнительные сведения см. в теме Управление разрешениями на аудиосвязь участников [в собраниях Teams.](https://support.microsoft.com/office/manage-attendee-audio-permissions-in-teams-meetings-f9db15e1-f46f-46da-95c6-34f9f39e671a)
- Поохищайте видео другого человека с консоли комнаты Teams и поохищайте видео на дисплеях комнаты

> [!NOTE]
> Голосовые навыки Кортаны доступны для выбранных звуковых периферийных устройств для клиентов, расположенных в США. В будущем будут добавлены дополнительные страны и регионы. Дополнительные сведения см. в голосовой помощи [Кортаны в Teams.](../cortana-in-teams.md)

### <a name="release-to-market-1272016"></a>Выпуск на рынок (07.01.2016)

**Функции.**

 **Оптимизация для Skype для бизнеса**

- Присоединение к собранию в Skype одним нажатием кнопки
- Skype Meeting experience optimized for rooms with screen-filling HD video and HD wide-band audio
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате

 **Преобразование любой комнаты для собраний**

- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Повторное использование существующих инвестиций перед отображением помещений или проекторами
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенные проводные функции приема для демонстрации совместного доступа к рабочему столу в комнате и в собрании Skype

 **Простота развертывания и управления**

- Всегда на устройстве, которое автоматически пробуждает дисплеи при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Отслеживается и управляется как устройство с Windows 10 Enterprise с помощью Intune и Configuration Manager (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
