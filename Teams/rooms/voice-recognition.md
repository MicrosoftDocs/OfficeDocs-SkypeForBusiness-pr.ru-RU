# <a name="manage-voice-recognition-technology-controls-for-an-intelligent-speaker"></a>Управление средствами распознавания речи для интеллектуального динамика

Интеллектуальный динамик использует сведения голосового профиля, чтобы распознать, кто что сказал в прямом разговоре. Если в Комнаты Microsoft Teams для Windows собрания есть интеллектуальный динамик, во время собрания можно использовать запись разговоров в прямом эфире. В этой статье объясняется, как вы, администратор клиента, управляете профилированием голоса, которое используется для распознавания речи для создания транскрибации в прямом эфире. Вы можете управлять тем, в какой степени в организации используется распознавание речи, и следующие функции:

- Изменить имя выступающего в расшифровке.
- Измените выступающего в расшифровке или измените его во всех записях (но не в последующих расшифровких).
- Измените идентификацию выступающего для пользователей, указанных на собрании.
- Удалите идентификацию одной или более записей, которые определены как динамики, в каждой записи разговоров.

## <a name="review-intelligent-speaker-requirements"></a>Просмотр требований интеллектуальных динамиков

Интеллектуальный динамик содержит специальный массив с семью микрофонами. Система использует сведения голосового профиля для определения голосов до 10 человек в комнатах для собраний.

Требования интеллектуальных динамиков к следующим элементами:

- Клиент должен быть расположен в США (Северная Америка). <sup>1</sup>
- В комнате собрания должно присутствовать не более 10 человек.
- В комнате собрания есть ссылка для отправки не менее 7 Мбит/с.

 <sup>1</sup> Интеллектуальный динамик, связанный с голосовым профилем и использованием будут доступны только на языках EN-US и клиентах США (ЕНЕ.В.РЕГИОНЕ). Оба условия должны быть истинными для пользователя клиента, чтобы зарегистрироваться и использовать интеллектуальный динамик для записи разговоров.

## <a name="set-up-an-intelligent-speaker"></a>Настройка интеллектуального динамика

Интеллектуальный динамик подключается к консоли Комнаты Teams USB. Для лучших результатов фирменные динамики Yealink следует использовать с консолью фирменой консоли Yealink.

> [!NOTE]
> Интеллектуальный динамик Yealink **должен использоваться** с консолью Yealink.

> [!NOTE]
> Мы не поддерживаем интеллектуальный динамик, подключенный к logitech Surface Pro Комнаты Microsoft Teams. Известно, что Комнаты Teams не распознает интеллектуальный динамик через док-станцию.

Интеллектуальный динамик следует разместить не менее чем на 8 дюймов (20 см) от стен и больших объектов, таких как ноутбуки. Если USB-кабель интеллектуального динамика недостаточно длинный для настройки, используйте кабельные удлинители.

## <a name="frequently-asked-questions-faq"></a>Вопросы и ответы

**Где хранятся данные профиля голосовой почты?**

Данные профиля голосовой связи хранятся в Office 365 содержимым пользователя.

**Что такое временная шкала хранения и политика?**

Общая политика хранения представлена в обзоре [хранения данных.](/compliance/assurance/assurance-data-retention-deletion-and-destruction-overview) Кроме того, данные голосового профиля пользователя удаляются через 3 года, если пользователь не приглашен на собрания с интеллектуальным динамиком в течение этого 3-летнего периода. Данные не используются на собраниях для существующих сотрудников. Если сотрудник покинул компанию, данные голосового профиля считаются контентом пользователя и обрабатываются как таковые в Office 365 политики хранения данных, описанной в обзоре хранения [данных](/compliance/assurance/assurance-data-retention-deletion-and-destruction-overview).

**Используются ли данные голосового профиля во службы Майкрософт?**

Нет, данные голосового профиля используются только для того, чтобы пользователь предоставил согласие. Корпорация Майкрософт не будет использовать данные профиля голосовой почты, за исключением Teams сценариев распознавания речи.

Например, корпорация Майкрософт не будет использовать данные в следующих ситуациях:

**Используются ли мои данные голосового профиля, когда я присоединяюсь к собранию в другой организации?**

Не только в собраниях, организованных пользователем в вашей организации.

**Как экспортировать профиль голосовой почты?**

Ваш ИТ-администратор может экспортировать звуковые данные в любое время.
