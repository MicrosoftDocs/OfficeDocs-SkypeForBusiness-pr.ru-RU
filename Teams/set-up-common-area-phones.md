# <a name="set-up-the-common-area-phone-license-for-microsoft-teams"></a>Настройка лицензии для телефонов общего пользования для Microsoft Teams

Телефон общего пользования обычно размещается в область, например зал ожидания или другой области, которое входит многим пользователям звонок; Например, для приема области, зал ожидания или конференции телефон. Региональные телефоны настраиваются как устройства, а не пользователей и автоматически может войти в сети.

В приведенные ниже действия поможет учетной записи службы для телефонной системой развертывание телефонов общего пользования для вашей организации. Для получения более полного конференц-зала работы, включая аудиоконференций, приобретите выделенного лицензии комнату переговоров с собрания комнаты устройства. 

Первое, что необходимо выполнить являются приобретения лицензии телефона общие области (узел) и убедитесь в том, что вы используете сертифицированного телефон. Для поиска и Дополнительные сведения о сертифицированного телефоны, перейдите на [устройствах группами Майкрософт](https://products.office.com/microsoft-teams/across-devices?ms.url=officecomteamsdevices&rtc=1). 

## <a name="step-1---buy-the-licenses"></a>Шаг 1. Приобретение лицензий

1. В центре администрирования Office 365 перейдите **по выставлению счетов** > **приобретение служб** и затем разверните узел **другие планы**.

    ![Снимок экрана, показывающий плитку телефон общего пользования](media/set-up-common-area-phone-image1.png)

2. Выбор **телефона общего пользования** > **купить**.

3. На странице " **Извлечение** " нажмите кнопку **купить**.

4. Разверните **дополнительный компонент подписки** и нажмите кнопку купить вызов планирование. Выберите **Внутренний вызов план** или **Планирование внутренних и международных звонков**.

> [!NOTE]
> Вам не нужна лицензия на телефонную систему. Она включена в лицензию Телефона общего пользования.

Дополнительные сведения о лицензии см. [: Лицензирование дополнительный компонент группами Майкрософт](teams-add-on-licensing/microsoft-teams-add-on-licensing.md).

## <a name="step-2---create-a-new-user-account-for-the-phone-and-assign-the-licenses"></a>Шаг 2. Создание новой учетной записи пользователя для телефона и назначение лицензии

1. В центре администрирования Office 365 перейдите к **пользователям** > **активных пользователей** > **Добавить пользователя**.

2. Введите имя пользователя как «Главная» для имени и «Прием» для второго имени.

3. Введите отображаемое имя, если она не автоматически заполнять одно как «Main прием».

4. Введите имя пользователя, например «MainReception» или «Mainlobby».

5. Для телефонов общего пользования можно вручную задать пароль или использовать тот же пароль для всех вашей телефонов общего пользования. Кроме того можно подумать о снять флажок **пользователь изменить пароль при первом входе в** .

6. Назначение лицензий для пользователя. На этой же странице щелкните мышью и разверните **Лицензии продуктов**. Включите телефон общего пользования и ответить на **Планирование внутренних звонков** или через **Внутренний и планирование International вызов**. 

    ![Назначение лицензий показывает снимок экрана](media/set-up-common-area-phone-image2.png)

Дополнительные сведения можно [Добавить пользователя](https://docs.microsoft.com/office365/admin/add-users/add-users?redirectSourcePath=%252farticle%252f1970f7d6-03b5-442f-b385-5880b9c256ec&view=o365-worldwide).

## <a name="step-3---assign-a-phone-number-to-the-common-area-phone-user-account"></a>Шаг 3 - Назначьте номер телефона учетной записи пользователя телефона общего пользования

Используйте Скайп по центру администрирования Business для присвоения номера для пользователя.

1. В центре администрирования Microsoft 365 выберите **центры администрирования** > **команды & Скайп** > **портала прежних версий**.

2. В Скайп по центру администрирования бизнеса, выберите **голосовой связи** > **телефонных номеров**.

3.  Выберите номер телефона из списка и нажмите **Назначить**.

4. На странице " **Назначение** " в поле голосовой связи пользователя введите имя пользователя, который будет использовать телефон, а затем выберите пользователя в раскрывающемся списке **выберите пользователя голосовой связи** .

5. Пока вы находитесь на этой странице, необходимо добавить адрес для обращения в экстренных случаях. Выберите в раскрывающемся списке **поиска, Город**, **Поиск, описание**или **в одном месте** и введите в текстовом поле Город, описание или расположение. Когда выполняется поиск, найдите раздел **выберите экстренных адрес** для выберите нужную для вас.

6. Нажмите **Сохранить**. Ваш пользователь должен выглядеть следующим образом:

   ![Назначение лицензий показывает снимок экрана](media/set-up-common-area-phone-image3.png)

> [!NOTE]
> Пользователи будут отображаются только при наличии применена лицензия телефонной системой. Если вы только что выполнили эти действия, может потребоваться некоторое время, чтобы пользователь отобразился в списке.

Дополнительные сведения см в [начало номера телефонов для пользователей](https://docs.microsoft.com/skypeforbusiness/what-are-calling-plans-in-office-365/getting-phone-numbers-for-your-users).

Вы также можете номер телефона с другого поставщика и «порт» или передается по Office 365. В разделе [Передача телефонных номеров в Office 365](transfer-phone-numbers-to-office-365.md).
