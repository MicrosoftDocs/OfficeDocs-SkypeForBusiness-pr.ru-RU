# <a name="deploy-common-area-phones-for-microsoft-teams"></a>Развертывание общих телефонов с Microsoft Teams

Стандартный телефон с районами обычно находится в таких же зонах, как "lobby" или в другой области, в которую может позвонить большое количество людей. Например, это может быть приемная, зал "зал", конференц-телефон. Обычные телефоны с городами— это учетные записи, привязанные к общей Телефон лицензии.

В этой статье представлен обзор развертывания и настройки Teams телефонов в качестве общих телефонов для общих помещений. Для более полного опытом работы в комнате для собраний, включая аудиоконференцию, можно приобрести специальную лицензию Конференц-зал с устройством комнаты для собраний.

## <a name="overview"></a>Обзор

Общая лицензия Телефон области поддерживает: 


| &nbsp;  |  Телефон общего пользования  |
|---------|---------|
|Skype для бизнеса |   &#x2714; |
|Microsoft Teams |   &#x2714; |
|Телефонная система |    &#x2714; |
|Аудиоконференция |       &#x2718; &sup1;  |
|Microsoft Intune |    &#x2718; |
|Доступность по всему миру |       &#x2718; &sup2;  |
|Доступность канала |    EA, EAS, CSP, GCC, EES, Web Direct  |
|      |         |

&sup1; Common Area Phones can join audio conferences via dial-in number provided by the meeting organizer

&sup2; Недоступны в независимых облаках  

>[!NOTE]
> Учетные записи для общих объектов телефонов с Skype для бизнеса Server, созданных в Microsoft Teams. Выполните действия, которые вы выполните в этой статье, чтобы воссоздать эти учетные записи для Teams и при необходимости перенести подключение к ТСЗ.

## <a name="step-1---buy-the-licenses"></a>Шаг 1. Приобретение лицензий

Сначала необходимо приобрести общую лицензию Телефон (CAP) и убедиться, что у вас есть сертифицированный телефон. Чтобы найти сертифицированные телефоны и узнать больше о них, перейдите [на Microsoft Teams устройствах](https://products.office.com/microsoft-teams/across-devices?ms.url=officecomteamsdevices&rtc=1).

1. В Центр администрирования Microsoft 365 перейдите в **службы** **BillingPurchase** > . 

2. Если раздел **Просмотр по категориям** еще не отображается, перейдите в раздел Приобретение у **Майкрософт** и выберите **Просмотреть продукты**. Затем выберите **Совместная работа и общение**.  

3. В списке продуктов найдите **общие области** Телефон и выберите **Сведения**.

4. Введите нужное количество лицензий и выберите **Купить**.

>[!NOTE]
>Если вы используете Intune в своей среде и у вас есть правила условного доступа, которые требуют соответствия устройствам, необходимо назначить учетной записи устройства лицензии на Azure Active Directory Premium План 1 и Intune для учетной записи устройства для общего телефонного номера.
>
>Правила условного доступа и другие конфигурации удостоверений, такие как многофакторная проверка подлинности, могут влиять на распространенные телефоны с районами. Дополнительные действия см. в Teams [для устройств с Android](devices/authentication-best-practices-for-android-devices.md).

## <a name="step-2---create-a-new-user-account-and-assign-licenses"></a>Шаг 2. Создание учетной записи пользователя и назначение лицензий

### <a name="using-the-microsoft-365-admin-center"></a>Использование Центр администрирования Microsoft 365

Если вы развертываете сразу несколько общих телефонов в области, узнайте, как создавать учетные записи и назначать лицензии [с помощью PowerShell](#using-powershell).

Если вы развертываете одно устройство:

1. В Центр администрирования Microsoft 365 перейдите к **usersActive** >  **UsersAdd** >  **пользователя**.

2. Введите имя пользователя, например "Основной" для имени, и "Прием" для второго имени.

3. Введите отображаемую фамилию, если она не будет автоматически сгенерироваться, например "Главный прием".

4. Введите имя пользователя, например "MainReception" или "Mainlobby".

5. Установите пароль для телефона общей области вручную, чтобы предотвратить это. Для этого включив автоматический создание пароля, с помощью этого действия пользователь должен будет сменить **пароль при первом входе**.  

    >[!Important]
    > Чтобы предотвратить проблемы со входом для конечных пользователей, настоятельно рекомендуется установить пароль вручную для общих телефонов в области.

6. Выберите расположение использования устройства и назначьте учетной записи лицензию Телефон области. Если нужны какие-либо другие лицензии, например планы звонков, назначьте их.

>[!NOTE]
> Вам не нужно добавлять телефонная система лицензию. Она включена в лицензию Телефона общего пользования.
>
>Если вы не используете Телефон (Майкрософт) или оператора Подключение, вы можете добавить лицензии на планы звонков. Дополнительные сведения о лицензиях см. [в Microsoft Teams лицензировании надстройки](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md).

### <a name="using-powershell"></a>С помощью PowerShell

Используйте PowerShell, если вы хотите одновременно создать и назначить лицензии для более чем одной учетной записи пользователя. Дополнительные сведения см. в Microsoft 365 учетных записей пользователей с помощью [PowerShell](/microsoft-365/enterprise/create-user-accounts-with-microsoft-365-powershell?view=o365-worldwide) и Назначение лицензий Microsoft 365 пользователям с [помощью PowerShell](/microsoft-365/enterprise/assign-licenses-to-user-accounts-with-microsoft-365-powershell?view=o365-worldwide).

## <a name="step-3---set-policies-for-common-area-phones"></a>Шаг 3. Настройка политик для общих телефонов в области

Используйте политики для управления возможностями, доступными пользователям на общих телефонах с регионами.

>[!NOTE]
>После назначения политики вы можете выйти из телефона и снова войти в нее. На то, чтобы назначение политики вступает в силу, может потребоваться до часа.

### <a name="ip-phone-policies"></a>Политики IP-телефонов

На телефонах с учетными записями, которые имеют общую область Телефон, будет отображаться общий пользовательский интерфейс области.

Если вы хотите переопрепрестить интерфейс телефона по умолчанию, рассмотрите возможность создания [политики IP-телефонии](/powershell/module/skype/new-csteamsipphonepolicy?view=skype-ps). Например, если в общей области используется общая телефонная сеть, установите политику IP-телефонов, чтобы ограничить поиск в глобальной адресной книге организации и заблокировать службу hot-desking. Дополнительные [Teams см](devices/Teams-Android-devices-user-interface.md). в Teams пользовательского интерфейса устройств с Android.

### <a name="calling-policies"></a>Политики звонков

Используйте политики звонков, чтобы включить частные звонки, используя переадваровку звонков или одновременные звонки на обычные телефоны в области звонков. [Дополнительные Teams в](teams-calling-policy.md) этой области.

По умолчанию парк  вызовов не включен для общих телефонов с районами. Вам потребуется создать политику, чтобы включить ее. [Дополнительные данные см](call-park-and-retrieve.md). в Microsoft Teams в Microsoft Teams.

## <a name="step-5---sign-in"></a>Шаг 5. Вход

После создания и настройки учетной записи пользователя вы можете войти в телефон. В зависимости от того, сколько телефонов вы развертываете, у вас есть три варианта для входов:

- [Локальный вход](#local-sign-in)
- [Вход с другого устройства](#sign-in-from-another-device)
- [Вход с помощью центра Teams администрирования](#sign-in-using-the-teams-admin-center)

### <a name="local-sign-in"></a>Локальный вход

Чтобы войти локально с помощью имени пользователя и пароля: 

1. Включим общий телефон с областью

2. Выберите **"Войти на этом устройстве"**

3. Следуйте указаниям по входу на устройстве. После этого на телефоне будет отображаться общая телефонная интерфейс области.

### <a name="sign-in-from-another-device"></a>Вход с другого устройства

Вы также можете войти в общую телефонную зону с другого устройства, используя код. При этом вы вводите имя пользователя и пароль на другом устройстве, а не на самом телефоне.

1. Во-первых, на телефоне с общей областью наймите код, который отображается на экране для регистрации.

2. На другом устройстве перейдите в https://www.microsoft.com/devicelogin.

3. Введите код и следуйте инструкциям по входу.

### <a name="sign-in-using-the-teams-admin-center"></a>Вход с помощью центра Teams администрирования

Как администратор вы можете удаленно подрезать и войти в общую телефонную зону из Teams администрирования. Это наиболее эффективный способ регистрации при одновременном развертывании большого количества телефонов. [Дополнительные узнать об этом см](devices/remote-provision-remote-login.md). в Teams удаленной подготовка и вход на устройствах с Android.
