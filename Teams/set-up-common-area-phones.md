# <a name="set-up-the-common-area-phone-license-for-microsoft-teams"></a>Настройка лицензии для телефонов общего пользования для Microsoft Teams
> [!NOTE]
> Обычные телефоны с регионами не поддерживают голосовую почту.

Стандартный телефон с районами обычно находится в таких же зонах, как "lobby" или в другой области, в которую может позвонить большое количество людей. Например, это может быть приемная, зал "зал", конференц-телефон. Обычные телефоны с зонами зоны, в которые были вписаны учетные записи, привязанные к общей Телефон лицензии. Для общего пользовательского интерфейса телефона также должна быть настроена политика TeamsIPPhone.

Ниже мы поможем вам настроить учетную запись для телефонная система для развертывания общих телефонов в области для вашей организации. Для более полного проведения собраний, включая аудиоконференцию, приобретите специальную лицензию Конференц-зал с устройством комнаты для собраний. 

Сначала необходимо приобрести лицензию на Телефон (CAP) и убедиться, что у вас есть сертифицированный телефон. Чтобы найти сертифицированные телефоны и узнать больше о них, перейдите на Microsoft Teams [устройства.](https://products.office.com/microsoft-teams/across-devices?ms.url=officecomteamsdevices&rtc=1) 

## <a name="step-1---buy-the-licenses"></a>Шаг 1. Приобретение лицензий

1. В Центр администрирования Microsoft 365 перейдите к приобретаемой службе вы выставлению счета  >   и разйдите в список **Другие планы.**

    ![Снимок экрана: общая область Телефон плитка.](media/set-up-common-area-phone-image1.png)

2. Выберите **Common Area Телефон** Купить  >  **.**

3. На странице "Регистрация" нажмите **кнопку Купить**.

4. **Разойдитесь на подписки на** надстройки и щелкните, чтобы приобрести план звонков. Выберите план внутренних **звонков или** план внутренних и **международных звонков.**

> [!NOTE]
> Если вы используете Телефон (Майкрософт) прямой маршрутизов системы, лицензия на план звонков не требуется.

> [!NOTE]
> Вам не нужно добавлять телефонная система лицензию. Она включена в лицензию Телефона общего пользования.

Дополнительные сведения о лицензиях см. в Microsoft Teams [лицензировании надстройки.](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

Лицензия common Area Телефон поддерживает: 


| &nbsp;  |  Телефон общего пользования  |
|---------|---------|
|Skype для бизнеса |   &#x2714; |
|Microsoft Teams |   &#x2714; |
|Телефонная система |    &#x2714; |
|Аудиоконференция |       &#x2718; &sup1;  |
|Microsoft Intune |    &#x2718; |
|Доступность по всему миру |       &#x2718; &sup2;  |
|Доступность канала |    EA, EAS, CSP, GCC, EES, Web Direct  |
|      |         |

&sup1; Common Area Phones can join audio conferences via dial-in number provided by the meeting organizer

&sup2; Недоступны в независимых облаках  



## <a name="step-2---create-a-new-user-account-for-the-phone-and-assign-the-licenses"></a>Шаг 2. Создание новой учетной записи пользователя для телефона и назначение лицензии

1. В Центр администрирования Microsoft 365 перейдите к **пользователям, которые**  >  **добавляют**  >  **активного пользователя**.

2. Введите имя пользователя, например "Основной", для имени и "Прием" для второго.

3. Введите отображаемую фамилию, если она не будет автоматически сгенерироваться, например "Главный прием".

4. Введите имя пользователя, например "MainReception" или "Mainlobby".

5. Для общих телефонов в области зоны можно установить пароль вручную или использовать один и тот же пароль для всех общих телефонов. Кроме того, вы можете  сменить этот пароль при первом входе.

6. Назначьте лицензии пользователю. На этой же странице щелкните мышью и разверните **Лицензии продуктов**. Включив общую область Телефон выберите  план внутренних звонков или план внутренних и **международных звонков.** 

    ![Снимок экрана: назначение лицензии с планом внутренних звонков и вариантами внутренних и международных звонков.](media/set-up-common-area-phone-image2.png)

> [!NOTE]
> Если вы используете Телефон (Майкрософт), назначать лицензию на план звонков не нужно.

Дополнительные сведения см. в [том, как назначить лицензии пользователям.](/microsoft-365/admin/manage/assign-licenses-to-users)

## <a name="step-3---assign-a-phone-number-to-the-common-area-phone-user-account"></a>Шаг 3 - Назначьте номер телефона учетной записи пользователя телефона общего пользования

Чтобы назначить Teams пользователю, используйте центр администрирования Teams.

1. В Центре Teams выберите Номера **голосовых**  >  **Телефон голосовой Телефон**.

3. Выберите номер телефона из списка и нажмите **Назначить**.

4. На **странице Назначение** в поле Пользователь голосовой связи введите имя пользователя, который будет использовать  телефон, а затем выберите его в списке Выберите пользователя голосовой связи.

5. Затем нужно добавить адрес для экстренного устранения проблемы. В **списке выберите** Поиск по  городу, Поиск по описанию или Поиск по расположению, а затем введите в текстовое поле город, описание или расположение.  После поиска в области **Выберите адрес для экстренного** обращения выберите нужный адрес.

6. Нажмите **Сохранить**. Ваш пользователь должен выглядеть следующим образом:

   ![Снимок экрана: пример назначения лицензии пользователя.](media/set-up-common-area-phone-image3.png)

> [!NOTE]
> Пользователи будут показываться только в том случае, если телефонная система лицензии. Если вы только что выполнили эти действия, может потребоваться некоторое время, чтобы пользователь отобразился в списке.

Дополнительные сведения см. в [сведениях Получение номеров телефонов для пользователей.](getting-phone-numbers-for-your-users.md)

Вы также можете перенести номер телефона другого оператора и перенести его в Microsoft 365 или Office 365. См. [перенос номеров телефонов в Teams.](phone-number-calling-plans/transfer-phone-numbers-to-teams.md)
