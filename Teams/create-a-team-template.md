# <a name="create-a-custom-team-template-in-microsoft-teams"></a>Создание пользовательского шаблона команды в Майкрософт Teams

**Пользовательские шаблоны пока не поддерживаются для клиентов EDU.**

Пользовательский шаблон команды — это предопределенная структура команды с набором каналов, вкладок и приложений. Вы можете разработать шаблон, который поможет быстро создать нужное пространство для совместной работы. Шаблон настраиваемой команды использует предпочитаемые параметры.  

<br>

> [!VIDEO https://www.microsoft.com/en-us/videoplayer/embed/RE4P5rx]


Чтобы приступить к работе, выполните приведенные далее действия.

1. Войдите в Центр администрирования Teams.

2. В области навигации слева разверните узел **Шаблоны команд** **Teams** > .

3. Нажмите **Добавить**.

    ![Изображение диалогового окна "Шаблоны группы" с выделенным элементом "Добавить".](media/team-templates-new.png)

4. В разделе **Шаблоны команд** выберите **Создать новый шаблон**.

5. В разделе **Параметры шаблона** заполните следующие поля и нажмите кнопку **Далее**:
    - Имя шаблона
    - Краткие и длинные описания шаблонов
    - Видимость языкового стандарта  

    ![Изображение диалогового окна именования параметров шаблонов группы.](media/template-add-a-name.png)

6. В разделе **Каналы, вкладки и приложения** добавьте все каналы и приложения, необходимые вашей команде.

    1. В разделе **Каналы** выберите **Добавить**.
    2. В диалоговом окне **Добавление** назовите канал.
    3. Добавьте описание.
    4. Решите, должен ли канал отображаться по умолчанию.
    5. Найдите имя приложения, которое нужно добавить в канал.
    6. По завершении нажмите **кнопку Применить** .

    ![Изображение экрана каналов, вкладок и приложений шаблонов команд.](media/template-channels-tabs-apps.png)

8. По завершении нажмите **кнопку Отправить** .

Новый шаблон отобразится в списке **Шаблоны группы** . Шаблон можно использовать для создания команды в Teams.

> [!Note]
> Пользователям teams может потребоваться до 24 часов, чтобы увидеть изменения пользовательского шаблона в коллекции.

## <a name="customizing-website-tab-apps"></a>Настройка приложений вкладки веб-сайтов

> [!Note]
> Эта функция доступна в ранней предварительной версии

Может потребоваться указать URL-адреса для вкладок веб-сайтов для каналов в пользовательских шаблонах команд. Конечные пользователи, создающие команды с шаблонами, будут иметь вкладки веб-сайта, которые настроены на указанный URL-адрес сайта.

Чтобы приступить к работе, выполните приведенные далее действия.

1. Создайте шаблон команды или измените существующий шаблон команды.

2. В разделе Каналы добавьте новый канал или выберите существующий канал и нажмите **кнопку Изменить**.

3. В разделе **Добавление приложения для этого шаблона** добавьте приложение веб-сайта.

    ![добавьте приложение для этого параметра шаблона.](media/add-an-app-template.png)

4. Щелкните значок правки и введите URL-адрес по своему усмотрению.

    ![добавить URL-адрес приложения.](media/add-url-app-template.png)

5. Выберите **Сохранить** для изменений в приложении вкладки, а затем нажмите **кнопку Применить** , чтобы сохранить изменения.

## <a name="known-issues"></a>Известные проблемы

**Проблема**. Если вы создали команду на основе пользовательского шаблона, содержащего дополнительные настраиваемые вкладки, вместо пользовательских вкладок вы можете увидеть пустые вкладки. Вкладки по умолчанию (например, **"Записи"**, **"Файлы"** и **"Вики-сайт**") будут отображаться должным образом.

**Решение**. Чтобы устранить эту проблему, удалите настраиваемую вкладку и добавьте новую вкладку с тем же приложением. Если у вас нет разрешений на удаление настраиваемой вкладки и добавление новой вкладки, обратитесь за помощью к владельцу команды.

В настоящее время мы работаем над исправлением для будущих команд, созданных на основе пользовательских шаблонов.

**Проблема**. При использовании Teams в браузере некоторые веб-сайты не поддерживают отображение на вкладке Teams.

![сообщение об ошибке браузера.](media/browser-error-message.png)

**Решение**. Если у вас возникли проблемы с просмотром содержимого вкладки веб-сайта, вы будете перенаправлены на открытие вкладки на отдельной веб-странице или открыть Teams в классическом приложении, чтобы просмотреть приложение вкладки веб-сайта.
