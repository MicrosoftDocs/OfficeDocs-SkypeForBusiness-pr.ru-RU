# <a name="microsoft-teams-rooms-app-version-support"></a>Поддержка версии приложения в комнатах Microsoft Teams
 
Корпорация Майкрософт планирует выпускать обновления для комнат Microsoft Teams несколько раз в год, каждый из которых поддерживается в течение двенадцати (12) месяцев от даты выпуска общей доступности. Техническая поддержка предоставляется для всего двенадцати (12) месяцев. Тем не менее, структура поддержки теперь динамична, развивается на две различные фазы обслуживания, зависящие от доступности последней версии.

**Этап обслуживания обслуживания и критических обновлений** \- При запуске последней версии комнат Microsoft Teams вы получаете регулярные обновления, содержащие обновления для системы безопасности и обслуживания.

**Фаза обслуживания обновлений для системы безопасности (только для)** \- После выпуска новой версии поддержка устаревшей ветви сокращается до обновлений для системы безопасности только в оставшейся части двенадцати (12) месяцев, поддерживаемых обновлениями, которые поддерживают жизненный цикл.

> [!NOTE]
> Новейшая версия всегда находится на этапе обслуживания обслуживания и критических обновлений. Это означает, что в случае, если вы столкнулись с дефектом кода, который гарантирует критическое обновление, для получения исправления необходимо установить последнюю версию. Все другие поддерживаемые версии будут иметь право на получение обновлений для системы безопасности.

Срок действия всей поддержки заканчивается после истечения двенадцати (12) месяцев для версии или в случае выпуска более двух обновлений с момента их появления. Затем пользователи должны обновить версию до поддерживаемой версии.

Все выпуски перечислены в заметках о выпуске в [комнатах Microsoft Teams](srs2-release-note.md).

# <a name="os-version-support"></a>Поддержка версий ОС
Обновления компонентов для Windows 10 для устройств, на которых запущены комнаты Microsoft Teams, не предлагаются в течение шести месяцев с момента, когда Windows выдает обновление выпуска. Это достигается путем добавления специального блока для устройств Microsoft Teams в канале обновления Windows для бизнеса (то есть, одноранговый канал) и параметров приложения. В течение этого периода Корпорация Microsoft производит различные тесты как внутри компании, так и с помощью наших OEM-партнеров, чтобы убедиться в том, что новая функция Windows 10 работает в гармонии с приложением и периферийными устройствами Microsoft Teams, подключенными к нему. Это важно для обеспечения безопасности устройства, согласованного взаимодействия с пользователем и обеспечения качества взаимодействия, предоставляемых в приложении Microsoft Team Помещенийs. 

Из блока Time удаляется (это значит, что предлагается обновление для Windows 10, которое можно скачать на этих устройствах), комнаты Microsoft Teams поддерживают специальную выпускную функцию в течение 12 месяцев с помощью политики поддержки приложений. Так как обновления функций для Windows 10 предлагаются примерно раз в шесть месяцев, это означает, что в Microsoft Teams есть еще несколько выпусков для проверки на момент завершения поддержки текущей версии. Это также означает, что версия Windows 10 разблокируется каждые шесть месяцев до всех пользователей комнат Microsoft Teams. Приложение постоянно меняется и разрабатывается по сравнению с последней разблокированной выпуском Windows. Чтобы убедиться в том, что вы получаете исправление для проблемы, которая возникла на устройстве Microsoft Teams, мы настоятельно рекомендуем всем клиентам обновить эти устройства до последней версии, предлагаемой для Windows 10.

Таким образом, для устройств Microsoft Teams требуется Windows 10 версии 1709 в качестве минимальной поддерживаемой версии, которая может 2019. Новые выпуски приложений не предлагаются для систем в Windows 10 версии 1703 или ниже.

> [!NOTE]
> Если устройство Microsoft Teams для комнат совместимо со следующей версией Windows 10 OS, устройство автоматически обновляется до следующей версии с помощью центра обновления Windows. Устройства в Microsoft Teams не должны обновляться до следующего выпуска Windows 10 вручную или путем включения групповых политик центра обновления Windows (ВУФБ) с помощью параметра "выберите уровень готовности Windows для обновлений, которые вы хотите получить" и "выбрать, когда Предварительные сборки и обновления компонентов получаются с помощью объекта групповой политики. Это может привести к возникновению проблем между обновлением операционной системы Windows 10 и приложением "комнаты Microsoft Teams".
