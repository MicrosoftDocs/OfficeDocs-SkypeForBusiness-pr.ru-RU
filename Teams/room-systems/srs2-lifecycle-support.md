# <a name="microsoft-teams-room-app-version-support"></a>Поддержка версии приложения комнаты Microsoft Teams
 
Корпорация Майкрософт планирует выпускать обновления для комнат Microsoft Teams несколько раз в год, каждый из которых поддерживается в течение двенадцати (12) месяцев от даты выпуска общей доступности. Техническая поддержка будет предоставляться для всех двенадцати (12) месяцев. Тем не менее, наша структура поддержки теперь динамично развивается на две различные фазы обслуживания, зависящие от доступности последней версии.

**Этап обслуживания обслуживания и критических обновлений** \- При запуске последней версии комнат Microsoft Teams вы получаете регулярные обновления, содержащие обновления для системы безопасности и обслуживания.

**Фаза обслуживания обновлений для системы безопасности (только для)** \- После выпуска новой версии поддержка устаревшей ветви будет снижена до обновлений для системы безопасности только в течение 12 (двенадцати) месяцев, поддерживаемых обновлениями, которые поддерживают жизненный цикл.

> [!NOTE]
> Новейшая версия всегда находится на этапе обслуживания обслуживания и критических обновлений. Это означает, что в случае, если вы столкнулись с дефектом кода, который гарантирует критическое обновление, для получения исправления необходимо установить последнюю версию. Все другие поддерживаемые версии будут иметь право на получение обновлений для системы безопасности.

Срок действия всей поддержки заканчивается после истечения двенадцати (12) месяцев для версии или в случае выпуска более двух обновлений с момента их появления. В это время пользователи должны обновить версию до поддерживаемой версии.

Все выпуски перечислены в заметках о выпуске в [комнатах Microsoft Teams](srs2-release-note.md).

# <a name="os-version-support"></a>Поддержка версий ОС
Обновления компонентов Windows 10 для устройств, на которых запущены комнаты Microsoft Teams, не предлагаются в течение 6 месяцев с момента выпуска операционной системы Windows. Это достигается путем добавления специального блока для устройств Microsoft Teams в канале обновления Windows для бизнеса (например, с однорангового канала) и с помощью параметров приложения. В течение этого периода Корпорация Microsoft производит различные тесты как на дому, так и с помощью наших OEM-партнеров, чтобы убедиться, что новая функция Windows 10 работает в гармонии с приложением Microsoft Teams и периферийными устройствами, подключенными к нему. Это важно для обеспечения безопасности устройства, согласованного взаимодействия с пользователем и обеспечения качества взаимодействия, предоставляемых в приложении Microsoft Team Помещенийs.   

Из блока Time (т. е. предлагается обновление для Windows 10, которое можно скачать на эти устройства), комнаты Microsoft Teams поддерживаются в соответствии с политикой поддержки приложений в течение 12 месяцев. Так как обновления компонентов для Windows 10 предлагаются каждые 6 месяцев, это означает, что в Microsoft Teams имеется два дополнительных выпуска для проверки на время завершения поддержки текущей версии. Это также означает, что версия Windows 10 будет разблокирована каждые 6 месяцев для пользователей комнат Microsoft Teams. По мере того как мы постоянно вносите изменения в приложения и разрабатываете на основе последнего незаблокированного выпуска Windows, в некоторых случаях необходимо принять зависимость от функций Windows, доступных в более новых версиях Windows, и убедиться в том, что она усложняет внесение изменений в корпорацию Майкрософт. версия, в которой 12-18 месяцев устарела. Чтобы получить исправление для решения проблемы, возникшей на устройстве Microsoft Teams, мы настоятельно рекомендуем всем пользователям обновить эти устройства до новейшего обновления компонентов Windows 10, предлагаемого для поддержания поддержки по версиям Windows.

Таким образом, для устройств Microsoft Teams требуется Windows 10 версии 1709 в качестве минимальной поддерживаемой версии, которая может 2019. Новые выпуски приложений не предлагаются для систем в Windows 10 версии 1703 или ниже.

> [!NOTE]
> Если устройство Microsoft Teams для комнат совместимо со следующей версией Windows 10 OS, устройство автоматически обновляется до следующей версии с помощью центра обновления Windows. Устройства в Microsoft Teams не должны обновляться до следующего выпуска Windows 10 вручную или путем включения групповых политик центра обновления Windows (ВУФБ) с помощью параметра "выберите уровень готовности Windows для обновлений, которые вы хотите получить" и "выбрать, когда Предварительные сборки и обновления компонентов получаются с помощью объекта групповой политики. Это может привести к возникновению проблем между обновлением операционной системы Windows 10 и приложением "комнаты Microsoft Teams".
