## <a name="microsoft-teams-rooms-feature-introduction-and-issue-resolution"></a>Функции Microsoft Teams комнаты введение и устранение проблем

### <a name="40850-0482019"></a>4.0.85.0 (04/8/2019)

Представлено в этом обновлении:

- Устранена проблема, связанная с функцией "Обратная связь" 
- Оптимизация для подготовки к обновлению устройства МТР до Windows 10 версии 1809

### <a name="40780-03142019"></a>4.0.78.0 (03/14/2019)

Представлено в этом обновлении:

- Исправление ошибки "зависание при запуске приложения", которая влияет на работу устройств в устаревшей версии Windows 10 RS2 сборки.  


### <a name="40760-03042019"></a>4.0.76.0 (03/04/2019)

Представлено в этом обновлении:

- Клавиша DTMF для собраний по сети Microsoft Teams P2P и КОММУТИРУЕМых звонков. Чтобы сделать Microsoft Teams клиентом по умолчанию, администраторы должны установить для Истеамсдефаултклиент значение true.
- Закрепите входящее видео удаленного участника на весь экран перед отображением комнаты. Использование команды "закрепить" в списке участников на консоли
- Улучшенные уведомления о выдачах за добавление уведомления о помещении
- На передней панели показывается, что значок "приведение" удаляется, когда на системном устройстве комнат не включена функция маяка Bluetooth
- Исправление проблемы с управлением громкостью в собраниях Teams


### <a name="40640-12142018"></a>4.0.64.0 (12/14/2018)

Представлено в этом обновлении:

- Отображение содержимого на обеих сторонах комнаты (для) на компьютерах с двумя экранами
- Улучшенные возможности пользовательского интерфейса комнаты и их части
- Поддержка на стороне клиента TLS 1,2. Для локальных пользователей для включения коммунЦиатион через TLS 1,2 для комнат Microsoft Teams требуется Skype для бизнеса Server 2015 Куммулативе Update 9 (CU9) или Skype для бизнеса сервера 2019 Куммулативе обновление 1 (CU1).

### <a name="40510-11172018"></a>4.0.51.0 (11/17/2018)

Представлено в этом обновлении:

- Поддержка двух дисплеев (с передней стороны комнаты) для собраний Teams 

### <a name="40310-10162018"></a>4.0.31.0 (10/16/2018)

Представлено в этом обновлении:

- Исправления качества и надежности

### <a name="40270-1012018"></a>4.0.27.0 (10/1/2018)

Представлено в этом обновлении:
 
- Изменения кода, необходимые для подготовки приложения Microsoft Teams комнаты для более поздних версий Windows 10 версии 1803
- Устранение проблемы с форматированием в локализованных лицензионных соглашениях (в частности, на норвежском языке), который не позволяет установить окно настройки OOBE.
- Изменения кода, необходимые для того, чтобы приложение Microsoft Teams запускалось на старых системах Lync. Дополнительные сведения [](https://aka.ms/lrsupgrade)
 
### <a name="40190-8312018"></a>4.0.19.0 (8/31/2018)
Представлено в этом обновлении:

- Исправление для приложения Крестрон, которое не запускается, как правило, будет доступно при нажатии кнопки "приложение" на устройствах Крестрон SR. Приложение Microsoft Teams Restart требуется перезапуск после установки 4.0.19.0. 

### <a name="40180-08272018"></a>4.0.18.0 (08/27/2018)
Представлено в этом обновлении:

- Улучшена функция "сообщить о проблеме" в режиме Teams (эквивалент "Обратная связь" в режиме Skype для бизнеса)
- Возможность перехода из Teams в режим Skype для бизнеса для звонков по протоколу SIP
- Улучшения специальных возможностей (Экранный диктор, экранная лупа)
- Автоматически перезапускать приложение после применения изменений подготовки XML
- Различные исправления

### <a name="4080-07062018"></a>4.0.8.0 (07/06/2018)

Представлено в этом обновлении:
- Это обновление позволяет проводить поддержку собраний Skype для бизнеса *и* Teams на устройствах с системами комнат.  После применения обновления Teams по умолчанию отключен.  Администраторы могут включать команды локально в параметрах устройства или с помощью удаленной отправки XML.

### <a name="311150-06182018"></a>3.1.115.0 (06/18/2018)

Представлено в этом обновлении:

- Ошибка "исправление в адрес" в некоторых системах при запуске приложения.

### <a name="311130-06132018"></a>3.1.113.0 (06/13/2018)
Представлено в этом обновлении:

- Изменения, позволяющие Microsoft более гибко управлять обновлениями Windows.
- Никаких изменений в работе конечных пользователей.

### <a name="311120-06052018"></a>3.1.112.0 (06/05/2018)

Представлено в этом обновлении:

- Устранение проблем с откликом консоли, которые были замечены на устройствах Surface Pro 2017, подключенных к двум экранам передней и видеосвязи, и видеоконференции
- Автоматическая проверка, чтобы убедиться в том, что система работает в последнюю версию сценария подготовки.

### <a name="311040-04162018"></a>3.1.104.0 (04/16/2018)

Представлено в этом обновлении:

- Исправление для улучшения ЭКРАННОЙ (экранная клавиатура) в системах на базе Windows 10 версии 1709
- Улучшения для подготовки к будущим обновлениям операционной системы

### <a name="311000-03162018"></a>3.1.100.0 (03/16/2018)

### <a name="31980--382018"></a>3.1.98.0 (3/8/2018)

Представлено в этом обновлении:

### <a name="release-to-market--1272016"></a>Выпуск от выпуска к рынку (12/7/2016)

**Функция (-ы):**

 **Оптимизация для Skype для бизнеса**
  
- Присоединение к собранию в Skype одним нажатием кнопки
- Интерфейс приложения для собраний Skype оптимизирован для комнат с полноэкранным видео и широкополосным аудиоканалом высокой четкости
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате

  **Преобразование любой комнаты для собраний**
  
- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Использование существующих инвестиций во фронтальные экраны или проекторы
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенные проводные функции приема для демонстрации совместного доступа к рабочему столу в комнате и в собрании Skype

  **Простота развертывания и управления**
  
- Постоянно включенное устройство, которое автоматически переключает экраны из спящего режима при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Мониторинг и управление на уровне устройства для Windows 10 Корпоративная с использованием Intune и SCCM (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
