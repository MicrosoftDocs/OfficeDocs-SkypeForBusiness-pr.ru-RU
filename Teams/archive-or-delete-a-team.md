<a name="archive-or-delete-a-team-in-microsoft-teams"></a>Архивировать или удалять группы в группах Майкрософт
===========================================

Со временем группы, созданные в группами Майкрософт могут относиться неиспользуемые или может потребоваться архивировать или удалять группы в конце проекта. Если вы являетесь admin группами Майкрософт, выполните действия, описанные в этой статье архивировать или удалять группы, которая больше не требуется. При сохранении группы прекращает все действия, выполненные для этой группы, но можно по-прежнему добавлять или удалять участников и обновить роли и по-прежнему можно просмотреть все действия группы в каналы, файлы и обсуждения. При удалении группы, группы активности в связанной каналы, файлы и чаты также удаляется. 

> [!IMPORTANT]
> Может быть включена повторно архивных группы, но нельзя отменить удаление группы, который был удален. Рекомендуется сначала архивации группа и откладывать удаление, пока вы уверены, что группа больше не требуется.

## <a name="archive-a-team"></a>Архив группы

Выполните следующие действия в архив группы.

1. В центре администрирования группами Майкрософт выберите **команды**.
2. Выберите группу, щелкнув имя группы.
3. Выберите **архив**. Отобразится следующее сообщение.

    ![Снимок экрана с группами архива сообщений](media/teams-archive-message.png)

4. Если вы хотите сделать сайт SharePoint для группы только для чтения, установите флажок.
5. Выберите **архив** в архив группа. Группа состояние изменится на **"архивированы"**.

## <a name="make-an-archived-team-active"></a>Сделать активным архивных группы

Выполните следующие действия, чтобы снова активировать архивных группы.

1. В центре администрирования группами Майкрософт выберите **команды**.
2. Выберите группу, щелкнув имя группы.
3. Выберите **Unarchive**. Группа состояние изменится на **активной**.

## <a name="delete-a-team"></a>Удаление группы

Если команда не требуется указать в будущем, затем можно удалить его, а не его архивации. Выполните следующие действия для удаления группы.

1.  В центре администрирования группами Майкрософт выберите **команды**.
2.  Выберите группу, щелкнув имя группы.
3.  Выберите команду **Удалить**. Появится сообщение с подтверждением.
4.  Выберите **Удалить** , чтобы окончательно удалить группу.
