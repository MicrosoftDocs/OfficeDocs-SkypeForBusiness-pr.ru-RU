# <a name="manage-caller-id-policies-in-microsoft-teams"></a>Управление политиками ИД звоня в Microsoft Teams

>[!INCLUDE [new-feature-teams-admin-center](includes/new-feature-teams-admin-center.md)]

Администраторы могут использовать политики ИД звоня в Microsoft Teams, чтобы изменить или заблокировать его (также известный как "ИД строки звонков"). По умолчанию телефонный номер пользователей Teams можно увидеть при звонии на телефон по ПСОП, а номер телефона вызывающих пользователей ДНР можно увидеть, когда звонят пользователи Teams. С помощью политик ИД звоня можно отобразить альтернативный номер телефона для пользователей Teams в организации или заблокировать отображение входящих номеров.

Например, когда звонят пользователи, вы можете изменить их ИД так, чтобы вместо номеров пользователей отображался основной номер телефона вашей организации.

Вы управляете политиками по учетным данным вызываемого звоня, переходить к политикам ИД вызываемого звоня в Центре администрирования   >   Microsoft Teams. Вы можете использовать глобальную (по умолчанию в пределах организации) политику или создавать и присваивать настраиваемые политики. Пользователи вашей организации автоматически получают глобальную политику, если вы не создали и не назначили настраиваемую политику.

## <a name="create-a-custom-caller-id-policy"></a>Создание настраиваемой политики ИД вызываемого вызываемого вызова

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к политикам, задамые для службы голосовых  >  **вызовов.**
2. Нажмите **Добавить**. <br>
![Снимок экрана: страница политики для новых вызовов в Центре администрирования](media/caller-id-policies-add-policy.png)
3. Введите имя и описание для политики.
4. Выберите нужные параметры.

    - **Блокировать ИД входящих звонков:** включив этот параметр, вы не можете отобразить Входящие звонки, включив этот параметр.
    - **Переопределение** политики " Включив этот параметр, пользователи смогут переопределять параметры политики, касающиеся отображения номеров вызывающих вызовов. Это означает, что пользователи могут выбрать, следует ли отображать свой ИД звоня. Дополнительные сведения см. в [сведениях о том,](https://docs.microsoft.com/microsoftteams/how-can-caller-id-be-used-in-your-organization#end-user-control-of-outbound-caller-id)как управлять исходящие вызовы конечными пользователями.
    - **Замените ИД вызываемого** пользователя следующим образом: установите для пользователей этот ИД:

        - **Номер пользователя:** отображает номер пользователя. 
        - **Номер службы:** позволяет настроить номер телефона службы, который будет отображаться в качестве ИД вызываемого.
        - **Анонимный:** отображает анонимный ИД звоня.

    - **Замените ИД вызываемого** вызываемого пользователя на номер службы: выберите номер службы, который заменит ИД вызываемого пользователя. Этот параметр доступен, если вы выбрали **"Номер** службы" в оке **"Заменить ИД вызываемого звонка".**

5. Щелкните **Сохранить**.

## <a name="edit-a-caller-id-policy"></a>Изменение политики ИД вызываемого вызываемого вызова

Вы можете изменить глобальную политику или любые настраиваемые политики, которые вы создаете. 

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к политикам, задамые для службы голосовых  >  **вызовов.**
2. Выберите политику, щелкнув слева от ее имени, а затем нажмите **Изменить**.
3. Измените нужные параметры и нажмите кнопку **"Сохранить".**
