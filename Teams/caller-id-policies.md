# <a name="manage-caller-id-policies-in-microsoft-teams"></a>Управление политиками идентификации вызывающего абонента в Microsoft Teams

По умолчанию, когда пользователь Teams звонит на телефон ТСОП, отображается номер телефона пользователя Teams. Аналогичным образом, когда абонент по ТСОП звонит пользователю Teams, отображается номер телефона звонящего по ТСОП.

Администратор может использовать политики идентификатора вызывающего объекта для изменения или блокировки идентификатора вызывающего объекта (также известного как идентификатор вызывающей строки). Политики идентификаторов звонящего можно использовать для отображения альтернативного номера телефона для пользователей Teams в вашей организации, блокировки номера исходящего телефона, блокировки отображения входящего номера или установки имени звонящей стороны (CNAM). Например, когда пользователь совершает звонок, вы можете изменить идентификатор звонящего, чтобы отобразить основной номер телефона организации и название компании вместо номера телефона пользователя.

Вы можете управлять политиками идентификаторов вызывающего абонента, перейдя в **раздел Политики идентификаторов** **голосовых** >  абонентов в Центре администрирования Майкрософт Teams. Вы можете использовать глобальную (по умолчанию в пределах организации) политику или создавать и присваивать настраиваемые политики. Пользователи вашей организации автоматически получают глобальную политику, если вы не создали и не назначили настраиваемую политику.

## <a name="create-a-custom-caller-id-policy"></a>Создание настраиваемой политики идентификатора вызывающего объекта

1. В левой области навигации Центра администрирования Майкрософт Teams перейдите в раздел **Политики идентификаторов** **голосовых** >  абонентов.
2. Нажмите **Добавить**. <br>
![Снимок экрана: страница новой политики идентификатора вызывающего абонента в Центре администрирования.](media/caller-id-policies-add-policy.png)
3. Введите имя и описание для политики.
4. Здесь выберите нужные параметры:

    - **Блокировать идентификатор входящего абонента**. Включите этот параметр, чтобы запретить отображение идентификатора вызывающей стороны входящих вызовов.
    - **Переопределение политики идентификатора вызывающего абонента**. Включите этот параметр, чтобы позволить пользователям переопределить параметры в политике относительно отображения их номера вызывающим абонентам или нет. Это означает, что пользователи могут выбрать, следует ли отображать идентификатор вызывающего объекта. Дополнительные сведения см. в разделе [Управление конечным пользователем идентификатором исходящего вызывающего объекта](./how-can-caller-id-be-used-in-your-organization.md#end-user-control-of-outbound-caller-id).
    - **Замените идентификатор вызывающего на**: Задайте идентификатор вызывающего объекта, который будет отображаться для пользователей, выбрав один из следующих вариантов:

        - **Номер пользователя**: отображает номер пользователя. 
        - **Номер службы**. Позволяет задать номер служебного телефона для отображения в качестве идентификатора звонящего.
        - **Анонимная**. Отображает идентификатор вызывающего объекта как анонимный.

    - **Замените идентификатор вызывающего абонента этим номером службы**. Выберите номер службы, чтобы заменить идентификатор вызывающего абонента пользователей. Этот параметр доступен, если вы выбрали **Номер службы** в **разделе Заменить идентификатор звонящего** на .

5. Нажмите кнопку **Сохранить**.

## <a name="edit-a-caller-id-policy"></a>Изменение политики идентификатора вызывающего объекта

Вы можете изменить глобальную политику или любые пользовательские политики, которые вы создаете. 

1. В левой области навигации Центра администрирования Майкрософт Teams перейдите в раздел **Политики идентификаторов** **голосовых** >  абонентов.
2. Выберите политику, щелкнув слева от ее имени, а затем нажмите **Изменить**.
3. Измените нужные параметры и нажмите кнопку **Сохранить**.
