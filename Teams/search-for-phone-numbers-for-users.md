# <a name="search-for-phone-numbers-for-users"></a>Поиск номеров телефонов для пользователей

При настройке работы с телефонными звонками для пользователей в организации необходимо использовать Центр администрирования **Microsoft Teams,** чтобы получить номера телефонов, которые могут быть назначены пользователям. Номер телефона, который вы назначаете пользователю, будет номером, выбранным для вашей организации, и будет указан в списке, когда вы редактируете свойства пользователя и нажимаете кнопку "Назначить".
  
Прежде чем назначать номера телефонов пользователям,  для поиска номеров, доступных в вашем регионе, воспользуйтесь страницей "Получить новые номера". Вы можете искать данные по стране или региону, типу номера и местоположению, а затем вводить номера телефонов, необходимые для пользователей. 
  
Если вам нужна помощь с получением [](/microsoftteams/manage-phone-numbers-for-your-organization) номеров телефонов, см. статью "Управление номерами телефонов для организации" или "Обратиться в службу поддержки продуктов для [бизнеса" — справка для администраторов](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
  
## <a name="search-for-phone-numbers"></a>Поиск номеров телефонов

**Поиск номеров телефонов для пользователей**

![Значок логотипа Teams ](media/teams-logo-30x30.png) **с помощью Центра администрирования Microsoft Teams**
  
1. Перейдите в **Центр администрирования Microsoft Teams.**

2. На левой навигации выберите   >  **"Номера голосовых телефонов"**  >  **"Получить новые номера".**
  
    > [!IMPORTANT]
    > Чтобы в центре  администрирования Microsoft Teams в левой области навигации был параметр "Голосовая почта", необходимо сначала приобрести по крайней мере одну лицензию на корпоративный **E5,** одну лицензию на надстройку телефонной системы или одну лицензию на надстройку для аудиоконференции.    

3. На странице **"Выбор расположения и** количества"  выберите расположение в списке "Страна или регион".

4. Выберите тип числа в списке **"Тип** номера".

5. В поле **"Расположение"** введите название города, в котором находится пользователь, и выберите его из списка. Нажмите **кнопку "Добавить расположение",** если нужное расположение не появится в списке.

6. Выберите код города для местоположения.

7. В **области "Количество"** введите нужное число номеров телефонов для организации и нажмите кнопку "Далее".  На выбор номеров телефонов отводится 10 минут. По истечении 10 минут номера будут возвращены в пул номеров телефонов.

    > [!NOTE]
    > Число доступных вам номеров телефонов (с учетом количества лицензий) указано рядом с номером **"Количество".** 
  
8. На странице **"Получить номера"** выберите номера телефонов, нажмите кнопку "Получить **номера"** и нажмите кнопку "Далее". 

    > [!IMPORTANT]
    > Вы можете получить больше номеров телефонов, чем у вас есть лицензии. Чтобы определить, сколько телефонных номеров можно получить, возьмите количество лицензий, добавьте к нему 10 процентов от этого количества, а затем — прибавьте 10. Например, если у вас 100 лицензий  на план внутренних и /или международных звонков, вы можете зарезервировать 120 номеров телефонов при условии, что вы еще не приобрели некоторые номера телефонов для этих 100 пользователей.  Для получения дополнительных сведений см.  [Сколько телефонных номеров можно получить?](./how-many-phone-numbers-can-you-get.md)

9. На странице **подтверждения** проверьте варианты и нажмите кнопку **"Заказать".**

10. Вернувшись на страницу **"Номера** телефонов", выберите номера телефонов,  которые вы хотите назначить, и нажмите кнопку "Изменить", чтобы назначить их пользователю.
