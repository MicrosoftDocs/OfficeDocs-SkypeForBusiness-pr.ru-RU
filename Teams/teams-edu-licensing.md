# <a name="assign-microsoft-teams-licenses-for-education"></a>Назначение лицензий Microsoft Teams для образовательных учреждений

Эта статья предназначена для ИТ-администраторов в образовательных учреждениях, которым необходимо назначить лицензии команды преподавателям, сотрудникам и учащимся.

Чтобы пользователи были готовы к работе в Teams, вам потребуется:

1. [Включите Microsoft Teams для учебного заведения в Центр администрирования Microsoft 365](/microsoft-365/education/intune-edu-trial/enable-microsoft-teams).
2. Назначьте лицензии учетным записям пользователей для доступа к службам Microsoft 365, включая Teams.

## <a name="ways-to-assign-teams-licenses"></a>Способы назначения лицензий Teams

Вы можете назначить лицензии учетным записям пользователей:

- По отдельности или к небольшой группе пользователей в Центр администрирования Microsoft 365.
- Автоматически через членство в группе с помощью [PowerShell](/office365/enterprise/powershell/assign-licenses-to-user-accounts-with-office-365-powershell) или лицензирования на основе групп [Active Directory](/azure/active-directory/users-groups-roles/licensing-groups-assign).

В этой статье описано, как назначать лицензии в Центр администрирования Microsoft 365.

В Центр администрирования Microsoft 365 вы можете назначить лицензии пользователям в следующих приложениях:

- Страница **"Лицензии** " для назначения лицензий на продукт определенным пользователям.
- Страница **"Активные пользователи** " для назначения лицензий пользователям определенным продуктам.

> [!NOTE]
> Необходимо иметь права глобального администратора, администратора выставления счетов, администратора лицензий или администратора управления пользователями. Дополнительные сведения см. в статье [Роли администраторов в Office 365](/microsoft-365/admin/add-users/about-admin-roles).

## <a name="assign-licenses-to-users-on-the-licenses-page"></a>Назначение лицензий пользователям на странице "Лицензии"

На странице **"** Лицензии" вы увидите список всех продуктов, для которых у вас есть подписки, общее количество лицензий для каждого продукта, количество назначенных лицензий и количество доступных.

1. В Центре [администрирования](https://go.microsoft.com/fwlink/p/?linkid=2024339) перейдите на страницу **"Лицензии на** > [выставление счетов](https://go.microsoft.com/fwlink/p/?linkid=842264) ".

2. Выберите продукт, для которого нужно назначить лицензии. Microsoft Teams является частью бесплатной версии Microsoft 365 A1 SKU для учащихся.

3. Нажмите **Назначение лицензий**.

4. В панели **Назначение лицензий пользователям** начните вводить имя, благодаря чему отобразится список имен.

5. Выберите среди результатов имя, которое вы ищете, и добавьте его к списку. Вы можете добавить до 20 пользователей одновременно.

6. Выберите **Включение и отключение приложений и служб**, чтобы назначить или удалить доступ к определенным элементам, например, к Microsoft Teams. Убедитесь в том, что выбраны **Microsoft Teams** и **Office в Интернете (для образовательных учреждений)**.

7. После завершения нажмите **Назначить**, а затем — **Закрыть**.

### <a name="change-the-apps-and-services-a-user-has-access-to"></a>Изменение приложений и служб, к которые пользователь имеет доступ

1. Выберите строку с пользователем.

2. В области справа выберите приложения и службы, к которым нужно предоставить доступ, или отмените выбор, чтобы удалить соответствующий доступ.

3. После завершения нажмите **Сохранить**, а затем — **Закрыть**.

## <a name="assign-licenses-to-users-on-the-active-users-page"></a>Назначение лицензий пользователям на странице "Активные пользователи"

1. В Центре [администрирования перейдите](https://go.microsoft.com/fwlink/p/?linkid=2024339) на страницу **"Активные** > [пользователи](https://go.microsoft.com/fwlink/p/?linkid=834822) ".

2. Установите флажки рядом с именем (именами) пользователя (пользователей), которому (которым) вы хотите назначить лицензии.

3. Вверху выберите " **Управление лицензиями на продукты"**.

4. На панели **Управление лицензиями на продукты** выберите **Добавить к имеющимся назначениям лицензий на продукты** > **Далее**.

5. В области **Добавить к имеющимся продуктам** установите переключатель для лицензии, которую вы хотите назначить выбранным пользователям, в положение **Вкл.**. Убедитесь в том, что выбраны **Microsoft Teams** и **Office в Интернете (для образовательных учреждений)**.

   Пользователю (пользователям) по умолчанию автоматически назначаются все службы, связанные с этой лицензией (этими лицензиями). Вы можете ограничить доступные для пользователей службы. Для служб, которых не должно быть у пользователей, установите переключатель в положение **Выкл**.

6. В нижней части панели нажмите кнопку " **Добавить** > **закрыть"**.
