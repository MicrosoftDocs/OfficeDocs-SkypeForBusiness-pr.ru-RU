# <a name="get-started-with-team-templates-using-microsoft-graph"></a>Начало работы с шаблонами команд с помощью Microsoft Graph

> [!NOTE]
> Шаблоны команд в настоящее время не поддерживают создание частных каналов. Создание частного канала не включено в определения шаблонов.

Шаблон команды в Microsoft Teams — это определение структуры команды, разработанной на основе бизнес-потребностей или проекта. С помощью шаблонов команд можно быстро и легко создавать широкие пространства для совместной работы с предопределенными параметрами, каналами и приложениями. Шаблоны команд помогают развертывать согласованные команды в организации.

С помощью Microsoft Graph вы можете [](/graph/api/resources/teamtemplate?view=graph-rest-beta) создавать собственные шаблоны или использовать предварительно созданные шаблоны команд, включаемые в Teams, для создания команд. В этой статье вы узнаете о свойствах, которые можно определить в шаблонах, и предварительно созданных шаблонах, доступных только в Microsoft Graph.

Эта статья предназначена для вас, если вы:

- Отвечает за планирование, развертывание и управление несколькими командами в организации<br>
- Разработчик, желающий программно создать команду с предопределенными каналами и приложениями

## <a name="team-template-capabilities"></a>Возможности шаблона команды

Большинство свойств в команде включаются и поддерживаются шаблонами. Но есть несколько свойств и функций, которые в настоящее время не поддерживаются. Ниже приведена краткая сводка о том, что включено и что не включено в шаблоны команд.

| **Свойства команды, поддерживаемые шаблонами команд** | **Свойства команды еще не поддерживаются шаблонами команд** |
| ------------------------------------------------ | -------------------------------------------------------- |
| Тип шаблона | Участники команды |
| Название команды | Изображение команды |
| Описание команды | Параметры канала |
| Видимость команды (общедоступная или закрытая) | Соединители |
| Параметры команды (например, член, гость, @ упоминания) | Файлы и контент |
| Автоматически избранный канал | |
| Установлено приложение | |
| Закрепленные вкладки | |

> [!NOTE]
> Мы добавим дополнительные возможности шаблонов в будущих выпусках Microsoft Teams, поэтому ознакомьтесь с актуальными сведениями о поддерживаемых свойствах.

## <a name="pre-built-templates"></a>Предварительно созданные шаблоны

Предварительно созданные шаблоны команд — это шаблоны, созданные для определенных отраслей. Ниже приведены предварительно созданные шаблоны, доступные только в Microsoft Graph.

| Тип шаблона | TemplateId | Свойства этого шаблона |
| ------------------ | -------------- | ----------------------------------------------------- |
| Standard | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('standard')` | Нет дополнительных приложений и свойств |
| Образования-<br>Группа классов | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationClass')` | Приложения:<ul><li>Записная книжка oneNote для занятий (закреплена на **вкладке "Общие** ") </li><li>Приложение "Назначения" (закреплено на **вкладке "Общие** ")</li></ul> Свойства команды:<ul><li>Для видимости команды задано **значение HiddenMembership** (не может быть переопределено)</li></ul> |
| Образования-<br>Группа персонала | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationStaff')` | Приложения:<ul><li>Записная книжка OneNote для преподавателей (закреплена на **вкладке "Общие** ")</li></ul> |
|Образования-<br>Команда PLC |`https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationProfessionalLearningCommunity')` | Приложения:<ul><li>Записная книжка OneNote PLC (закреплена на **вкладке "Общие** ")</ul></li>|

> [!NOTE]
> Список предварительно созданных шаблонов, которые можно использовать в клиенте Teams и в Microsoft Graph, см. в статье "Начало работы с шаблонами групп" в [Центре администрирования Teams](get-started-with-teams-templates-in-the-admin-console.md).
