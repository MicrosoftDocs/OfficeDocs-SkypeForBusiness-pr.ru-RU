# <a name="get-started-with-team-templates-using-microsoft-graph"></a>Начало работы с шаблонами команд с помощью Microsoft Graph

> [!NOTE]
> Шаблоны команд в настоящее время не поддерживают создание частных каналов. Создание частного канала не включается в определения шаблонов.

Шаблон команды в Microsoft Teams — это определение структуры команды, разработанное на основе бизнес-потребности или проекта. С помощью шаблонов команд можно быстро и легко создавать многофункциональные пространства для совместной работы с предопределенными параметрами, каналами и приложениями. Шаблоны команд помогают развертывать согласованные команды в организации.

С помощью Microsoft Graph можно [создавать собственные шаблоны](/graph/api/resources/teamtemplate?view=graph-rest-beta) или использовать готовые шаблоны команд, которые входят в состав Teams, для создания команд. В этой статье вы узнаете о свойствах, которые можно определить в шаблонах, и о готовых шаблонах, доступных только в Microsoft Graph.

Эта статья для вас, если вы:

- Отвечает за планирование, развертывание и управление несколькими командами в организации<br>
- Разработчик, желающий программно создать команду с предопределенными каналами и приложениями

## <a name="team-template-capabilities"></a>Возможности шаблонов команды

Большинство свойств в команде включены и поддерживаются шаблонами. Но есть несколько свойств и функций, которые в настоящее время не поддерживаются. Ниже приведены краткие сведения о том, что входит в шаблоны команд и что не входит в них.

| **Свойства группы, поддерживаемые шаблонами команд** | **Свойства команды, которые еще не поддерживаются шаблонами команд** |
| ------------------------------------------------ | -------------------------------------------------------- |
| Тип шаблона | Участники команды |
| Название команды | Изображение команды |
| Описание команды | Параметры канала |
| Видимость команды (общедоступная или частная) | Соединители |
| Параметры команды (например, участник, гость, @упоминания) | Файлы и контент |
| Автоматически избранный канал | |
| Установлено приложение | |
| Закрепленные вкладки | |

> [!NOTE]
> Мы добавим дополнительные возможности шаблонов в будущих выпусках Microsoft Teams, поэтому ознакомьтесь с самой актуальной информацией о поддерживаемых свойствах.

## <a name="pre-built-templates"></a>Предварительно созданные шаблоны

Предварительно созданные шаблоны команд — это шаблоны, созданные для конкретных отраслей. Ниже приведены готовые шаблоны, доступные только в Microsoft Graph.

| Тип шаблона | TemplateId | Свойства этого шаблона |
| ------------------ | -------------- | ----------------------------------------------------- |
| Standard | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('standard')` | Нет дополнительных приложений и свойств |
| Образования-<br>Группа класса | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationClass')` | Приложения:<ul><li>Записная книжка OneNote для занятий (закреплена на вкладке **Общие** ) </li><li>Приложение "Назначения" (закреплено на вкладке **"Общие** ")</li></ul> Свойства команды:<ul><li>Для видимости команды задано значение **HiddenMembership** (невозможно переопределить)</li></ul> |
| Образования-<br>Команда сотрудников | `https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationStaff')` | Приложения:<ul><li>Записная книжка oneNote для сотрудников (закреплена на вкладке **Общие** )</li></ul> |
|Образования-<br>Команда PLC |`https://graph.microsoft.com/v1.0/`<br>`teamsTemplates('educationProfessionalLearningCommunity')` | Приложения:<ul><li>Записная книжка OneNote PLC (закреплена на вкладке **Общие** )</ul></li>|

> [!NOTE]
> Список готовых шаблонов, которые можно использовать в клиенте Teams и в Microsoft Graph, см. в статье [Начало работы с шаблонами команд в Центре администрирования Teams](get-started-with-teams-templates-in-the-admin-console.md).
