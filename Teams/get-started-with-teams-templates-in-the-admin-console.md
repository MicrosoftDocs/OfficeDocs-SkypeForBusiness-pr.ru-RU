# <a name="get-started-with-team-templates-in-the-teams-admin-center"></a>Начало работы с шаблонами команд в Центре администрирования Teams

**Возможность создания пользовательских шаблонов пока не поддерживается для клиентов EDU.**

> [!NOTE]
> - Частные и общие каналы в настоящее время не поддерживаются в шаблонах команд. Создание частного и общего каналов не включено в определения шаблонов.
>
> - Метки конфиденциальности не поддерживаются в шаблонах групп в GCC средах. Параметр метки конфиденциальности в команде "Создать команду из потока шаблонов" не будет применен к команде.

## <a name="overview"></a>Обзор

Шаблон команды в Microsoft Teams — это определение структуры команды, разработанной на основе бизнес-потребностей или проекта. Как администратор вы можете использовать шаблоны для простого развертывания согласованных команд в организации. С помощью шаблонов пользователи могут быстро создавать широкие пространства для совместной работы с предопределенными параметрами, каналами и приложениями.

Вы можете управлять шаблонами команд в Microsoft Teams центра администрирования или с помощью PowerShell. Вы можете использовать предварительно созданные шаблоны, которые мы предоставляем, а также создавать [собственные настраиваемые шаблоны](#create-your-own-team-templates). Вы также можете [применить политики шаблонов](#apply-team-template-policies) для управления тем, какие шаблоны доступны пользователям в Teams.

В этой статье приводятся общие сведения о работе с шаблонами команд в Teams администрирования. Вы узнаете о свойствах, которые поддерживаются в шаблонах, предварительно созданных шаблонах, которые мы предоставляем, ограничениях размера шаблонов, о создании шаблонов и управлении ими и т. д.

> [!NOTE]
> Пользователи могут [создавать команды на основе предварительно](https://support.microsoft.com/office/create-a-team-from-a-template-a90c30f3-9940-4897-ab5b-988e69e4cd9c) созданных или настраиваемых шаблонов команд в Teams приложения. Разработчики также могут программно создавать команды на основе предварительно созданных шаблонов команд с помощью Microsoft Graph. Дополнительные сведения см[. в начало работы шаблонов команд с помощью Microsoft Graph](get-started-with-teams-templates.md).

## <a name="team-template-capabilities"></a>Возможности шаблона команды

Большинство свойств в команде включаются и поддерживаются шаблонами команды. Но есть несколько свойств и функций, которые в настоящее время не поддерживаются. Ниже приведена сводка по включению и не включению в шаблоны команд.

| **Свойства команды, поддерживаемые шаблонами команд** | **Свойства команды еще не поддерживаются шаблонами команд** |
| ------------------------------------------------ | -------------------------------------------------------- |
| Тип шаблона | Участники команды |
| Название команды | Изображение команды |
| Описание команды | Параметры канала |
| Видимость команды (общедоступная или закрытая) | Соединители |
| Параметры команды (например, член, гость, @ упоминания) | Файлы и контент |
| Канал Autofavorite | |
| Установленное приложение | |
| Закрепленные вкладки | |

> [!NOTE]
> Мы добавим дополнительные возможности шаблонов в будущих выпусках Microsoft Teams, поэтому ознакомьтесь с актуальными сведениями о поддерживаемых свойствах.

Шаблоны ограничены определенным количеством каналов, вкладок и приложений.

 > [!Note]
 > После создания шаблона в команду можно добавить дополнительные каналы, вкладки и приложения.

|Функция | Предел|
|-|-|
|Каналы для каждого шаблона | 15 |
|Вкладки на канал в шаблоне | 20 |
|Приложения для каждого шаблона | 50|

## <a name="manage-team-templates"></a>Управление шаблонами команд

### <a name="manage-team-templates-in-the-teams-admin-center"></a>Управление шаблонами команд в Центре администрирования Teams

#### <a name="view-team-templates"></a>Просмотр шаблонов команд

Чтобы просмотреть шаблоны команд, в левой области навигации центра администрирования Teams перейдите  >  к Teams **Team**. Выберите шаблон, чтобы просмотреть дополнительные сведения, включая каналы и приложения, которые он содержит.

#### <a name="create-your-own-team-templates"></a>Создание собственных шаблонов команд

Вы можете создавать собственные настраиваемые шаблоны с нуля, из существующей команды и из существующего шаблона. Дополнительные сведения см. в статьях

- [Создание настраиваемого шаблона команды](create-a-team-template.md)
- [Создание шаблона из существующей команды](create-template-from-existing-team.md)
- [Создание шаблона команды на основе существующего шаблона команды](create-template-from-existing-template.md)

#### <a name="apply-team-template-policies"></a>Применение политик шаблонов команд

Чтобы управлять шаблонами, которые пользователи видят в Teams для создания [команд, можно](https://support.microsoft.com/office/create-a-team-with-team-templates-702a2977-e662-4038-bef5-bdf8ee47b17b) задать политики шаблонов и назначить их пользователям и группам в организации. Дополнительные сведения см. в [разделе "Управление шаблонами групп" Teams центре администрирования](templates-policies.md).

### <a name="manage-team-templates-using-powershell"></a>Управление шаблонами команд с помощью PowerShell

Используйте следующие командлеты для управления шаблонами в PowerShell.

- [Get-CsTeamTemplate](/powershell/module/teams/get-csteamtemplate)
- [Get-CsTeamTemplateList](/powershell/module/teams/get-csteamtemplatelist)
- [New-CsTeamTemplate](/powershell/module/teams/new-csteamtemplate)
- [Remove-CsTeamTemplate](/powershell/module/teams/remove-csteamtemplate)
- [Update-CsTeamTemplate](/powershell/module/teams/update-csteamtemplate)
