# <a name="get-started-with-team-templates-in-the-teams-admin-center"></a>Начало работы с шаблонами команд в Центре администрирования Teams

**Возможность создания пользовательских шаблонов пока не поддерживается для клиентов EDU.**

> [!NOTE]
> - Частные и общие каналы в настоящее время не поддерживаются в шаблонах команд. Создание частных и общих каналов не включается в определения шаблонов.
>
> - Метки конфиденциальности не поддерживаются в шаблонах команд в средах GCC. Параметр метки конфиденциальности в потоке Создания команды из шаблона не будет применяться к команде.

## <a name="overview"></a>Обзор

Шаблон команды в Microsoft Teams — это определение структуры команды, разработанное на основе бизнес-потребности или проекта. Как администратор, вы можете использовать шаблоны для быстрого развертывания согласованных команд в организации. С помощью шаблонов пользователи могут быстро создавать многофункциональные пространства для совместной работы с предопределенными параметрами, каналами и приложениями.

Вы можете управлять шаблонами команд в Центре администрирования Microsoft Teams или с помощью PowerShell. Вы можете использовать готовые шаблоны, которые мы предоставляем, а также [создавать собственные пользовательские шаблоны](#create-your-own-team-templates). Вы также можете [применить политики шаблонов](#apply-team-template-policies) для управления тем, какие шаблоны доступны пользователям в Teams.

В этой статье приведены общие сведения о работе с шаблонами команд в Центре администрирования Teams. Вы узнаете о свойствах, которые поддерживаются в шаблонах, готовых шаблонах, которые мы предоставляем, ограничениях на размер шаблонов, способах создания шаблонов и управления ими и т. д.

> [!NOTE]
> Пользователи могут [создавать команды из готовых или настраиваемых шаблонов команд](https://support.microsoft.com/office/create-a-team-from-a-template-a90c30f3-9940-4897-ab5b-988e69e4cd9c) в приложении Teams. Разработчики также могут программно создавать команды на основе готовых или пользовательских шаблонов команд с помощью Microsoft Graph. Дополнительные сведения см. в статье [Начало работы с шаблонами команд с помощью Microsoft Graph](get-started-with-teams-templates.md).

## <a name="team-template-capabilities"></a>Возможности шаблонов команды

Большинство свойств в команде включены и поддерживаются шаблонами команд. Но есть несколько свойств и функций, которые в настоящее время не поддерживаются. Ниже приведена сводка о том, что входит в шаблоны команд и что не входит в них.

| **Свойства группы, поддерживаемые шаблонами команд** | **Свойства команды, которые еще не поддерживаются шаблонами команд** |
| ------------------------------------------------ | -------------------------------------------------------- |
| Тип шаблона | Участники команды |
| Название команды | Изображение команды |
| Описание команды | Параметры канала |
| Видимость команды (общедоступная или частная) | Соединители |
| Параметры команды (например, участник, гость, @упоминания) | Файлы и контент |
| Канал autofavorite | |
| Установлено приложение | |
| Закрепленные вкладки | |

> [!NOTE]
> Мы добавим дополнительные возможности шаблонов в будущих выпусках Microsoft Teams, поэтому ознакомьтесь с самой актуальной информацией о поддерживаемых свойствах.

## <a name="pre-built-team-templates-in-the-teams-admin-center"></a>Предварительно созданные шаблоны команд в Центре администрирования Teams

Ниже приведены готовые шаблоны команд, доступные в Центре администрирования Teams. Предварительно созданные шаблоны — это шаблоны, созданные для конкретных отраслей. Чтобы просмотреть эти шаблоны, в левой области навигации Центра администрирования Teams перейдите в раздел **Шаблоны команд** **Teams** > .

Вы можете дублировать предварительно созданные шаблоны, но не можете изменить их. Если вы хотите изменить свойства в предварительно созданном шаблоне, можно создать новый шаблон на основе существующего, а затем добавить или удалить нужные свойства. Помните, что некоторые свойства в некоторых шаблонах нельзя изменить.

> [!NOTE]
> Звездочка (*) указывает, что шаблон является *подключенным шаблоном Microsoft 365*. Когда пользователи создают команду с помощью шаблона, подключенный шаблон SharePoint применяется к сайту и команде. Компоненты SharePoint, такие как страницы, списки и интеграции Power Platform, автоматически добавляются и закрепляются как вкладки в канале "Общие" в команде. Пользователи могут редактировать эти страницы и списки прямо из Teams.

## <a name="team-template-size-limits"></a>Ограничения на размер шаблона группы

Шаблоны ограничены определенным количеством каналов, вкладок и приложений.

 > [!Note]
 > Вы можете добавить в команду дополнительные каналы, вкладки и приложения после ее создания на основе шаблона.

|Функция | Предел|
|-|-|
|Каналы на шаблон | 15 |
|Вкладки на канал в шаблоне | 20 |
|Приложения на шаблон | 50|

Дополнительные сведения см. в разделе [Ограничения и спецификации Teams](limits-specifications-teams.md).

## <a name="manage-team-templates"></a>Управление шаблонами команд

### <a name="manage-team-templates-in-the-teams-admin-center"></a>Управление шаблонами команд в Центре администрирования Teams

#### <a name="view-team-templates"></a>Просмотр шаблонов команд

Чтобы просмотреть шаблоны команд, в левой области навигации Центра администрирования Teams перейдите в раздел **Шаблоны команд** **Teams** > . Выберите шаблон, чтобы просмотреть дополнительные сведения, включая содержащиеся в нем каналы и приложения.

#### <a name="create-your-own-team-templates"></a>Создание собственных шаблонов команд

Вы можете создавать собственные пользовательские шаблоны с нуля, из существующей команды и из существующего шаблона. Дополнительные сведения см. в статьях

- [Создание пользовательского шаблона группы](create-a-team-template.md)
- [Создание шаблона из существующей команды](create-template-from-existing-team.md)
- [Создание шаблона команды на основе существующего шаблона команды](create-template-from-existing-template.md)

#### <a name="apply-team-template-policies"></a>Применение политик шаблонов команды

Чтобы управлять шаблонами, которые пользователи видят в Teams для [создания команд](https://support.microsoft.com/office/create-a-team-with-team-templates-702a2977-e662-4038-bef5-bdf8ee47b17b), можно задать политики шаблонов и назначить их пользователям и группам в вашей организации. Дополнительные сведения см. [в статье Управление шаблонами команд в Центре администрирования Teams](templates-policies.md).

### <a name="manage-team-templates-using-powershell"></a>Управление шаблонами команд с помощью PowerShell

Используйте следующие командлеты для управления шаблонами в PowerShell.

- [Get-CsTeamTemplate](/powershell/module/teams/get-csteamtemplate)
- [Get-CsTeamTemplateList](/powershell/module/teams/get-csteamtemplatelist)
- [New-CsTeamTemplate](/powershell/module/teams/new-csteamtemplate)
- [Remove-CsTeamTemplate](/powershell/module/teams/remove-csteamtemplate)
