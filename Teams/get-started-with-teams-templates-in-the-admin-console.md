# <a name="get-started-with-teams-templates-in-the-teams-admin-console"></a>Начало работы с шаблонами Teams в консоли администрирования Teams

[!INCLUDE [template](includes/preview-feature.md)]

**Настраиваемые шаблоны пока не поддерживаются для образования.**

> [!NOTE]
> Шаблоны Teams в настоящее время не поддерживают создание частных каналов. Создание частного канала не включается в определения шаблонов.

Шаблоны команд — это готовые определения структуры команды, разработанных в бизнес-потребностях или проектах. Используйте готовые шаблоны или создайте собственный. Шаблоны Teams позволяют быстро создавать специально ее пространство для совместной работы с каналами для различных тем и предварительно установить приложения, которые можно извлечь в миссию и услуги. Шаблоны teams обеспечивают стандартную структуру команд, помогающую легко создавать согласованные команды в масштабах всей организации. В настоящее время вы можете создать команду на основе шаблона в Teams или [Microsoft Graph.](get-started-with-teams-templates.md)

В этой статье объясняются свойства, которые могут быть заданы в шаблонах, какие типы шаблонов являются базовыми типами, и как с помощью нескольких образцов запросов создать команду на основе шаблона.

Эта статья для вас, если вы:

- Ответство на планирование, развертывание и управление несколькими командами в организации<br>
- Разработчик, который требуется программно создать команду с предопределенными каналами и приложениями

## <a name="teams-template-capabilities"></a>Возможности шаблона Teams

Большинство свойств команды включаются и поддерживаются шаблонами. Но сейчас есть несколько свойств и функций, которые в настоящее время не поддерживаются. В таблице ниже кратко описаны элементы, которые входят в шаблоны Teams и что не включено.

| **Свойства команды, поддерживаемые шаблонами Teams** | **Свойства команды пока не поддерживаются шаблонами Teams** |
| ------------------------------------------------ | -------------------------------------------------------- |
| Тип шаблона базового шаблона | Участники команды |
| Название команды | Изображение группы |
| Описание команды | Параметры канала |
| Видимость команды (общедоступная или закрытая) | Соединители |
| Параметры команд (например, участник, гость, @упоминания) | Файлы и контент |
| Автоматическое избранное канал | |
| Установленное приложение | |
| Закрепленные вкладки | |

> [!NOTE]
> В будущих версиях Microsoft Teams мы добавим возможности шаблонов, поэтому проверьте наличие самых актуальных сведений о поддерживаемых свойствах.

## <a name="what-are-base-template-types"></a>Что представляют собой базовые типы шаблонов

Базовые шаблоны — это особые шаблоны, созданные корпорацией Майкрософт для определенных отраслей. Эти базовые шаблоны часто содержат приложения, доступные в магазине приложений.

После определения базового типа можно расширить или переопределить эти специальные шаблоны с дополнительными свойствами, которые вы хотите задать. Однако некоторые базовые шаблоны содержат свойства, которые невозможно переопределено.

## <a name="template-size-limits"></a>Ограничения размера шаблонов

Шаблоны ограничены определенным количеством каналов, вкладок и приложениями.

 > [!Note]
 > После создания в команду вы сможете добавить в нее дополнительные каналы, вкладки и приложения.

|Функция | Ограничение|
|-|-|
|Каналы на шаблон | 15 |
|Вкладки на канал в шаблоне | 20 |
|Приложения на основе шаблона | 50|
|||

Дополнительные сведения см. в разделе "Ограничения и [спецификации Teams".](limits-specifications-teams.md)
