# <a name="get-started-with-microsoft-teams-for-remote-learning"></a>Начало работы с Microsoft Teams для удаленного обучения

Microsoft Teams — это платформа, в одном приложении которой объединены беседы, контент, назначения и приложения. Создавайте классы для совместной работы, подключайтесь к профессиональным образовательным сообществам и общайтесь с коллегами — все это в одном опыте.

Используйте рекомендации, описанные в этой статье, чтобы приступить к работе с Microsoft Teams для образовательных задолженности. Microsoft Teams можно использовать, чтобы обеспечить совместную работу с классом для учащихся в беседах, обеспечивая виртуальную платформу для собраний и распространение назначений. Учебные администраторы и сотрудники могут оставаться на связи с помощью Teams для объявлений и тематических разговоров. Преподаватели могут делиться учебными материалами, используя профессиональные образовательные сообщества.

В Microsoft Teams доступны [Клиенты](get-clients.md) для настольных компьютеров (Windows, Mac и Linux), Интернет-и мобильных устройств (Android и iOS), чтобы убедиться в том, что все сотрудники и студенты смогут оставаться на связи.

Дополнительные сведения о сценариях использования Microsoft Teams в [серии семинаров для образовательных учреждений](https://aka.ms/TeamsEDUWebinars).

## <a name="user-accounts-licenses-and-identity-security"></a>Учетные записи пользователей, лицензии и безопасность удостоверений

Microsoft Teams использует возможности Microsoft 365 для проверки подлинности пользователей и предоставления услуг. Для упрощения совместной работы у сотрудников, лекторов и студентов должны быть установлены удостоверения. Если удостоверения еще не существуют, выполните эти действия, чтобы установить их.

[Лицензии Teams должны быть включены для пользователей,](user-access.md) прежде чем они смогут приступить к использованию возможностей Teams. Teams полагается на дополнительные возможности Microsoft 365, такие как [Office 365 Groups](Office-365-groups.md), [Exchange](Exchange-Teams-interact.md), [SharePoint и OneDrive](SharePoint-OneDrive-interact.md) , чтобы включить сценарии для совместной работы. Пользователи получают лучшие навыки работы с группами, если все эти службы также включены. [Teams поддерживается для пользователей с электронной почтой, размещенными в Google](https://docs.microsoft.com/microsoft-365/education/deploy/enabling-teams-for-education-for-google-users).

## <a name="easily-set-up-microsoft-teams"></a>Простота настройки Microsoft Teams

Ниже приведены два действия, которые необходимо выполнить, чтобы приступить к работе с Teams.

### <a name="1-allow-users-to-create-teams"></a>1. Разрешите пользователям создавать группы.

Учащиеся и преподаватели получат максимум пользы от Teams, когда они смогут использовать их с минимальными барьерами и настроить их для их нужд. Один из способов настроить взаимодействие с Teams — это возможность создавать команды, соответствующие их нуждам. По **умолчанию все пользователи могут создавать группы и команды Office 365**. Бывают ситуации, когда эта возможность может быть неприменима. Например, для некоторых пользователей может потребоваться запретить создание групп из-за дополнительных учащихся. При необходимости вы можете ограничить возможности создания групп и групп Office 365 [определенными группами безопасности](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) в среде.

Пользователи с более высокими учреждениями образования получают доступ к своим категориям, в том числе к студентам, а также создавать группы для классов, исследований, групповых проектов и учебных групп. Основным и дополнительным школы может потребоваться запретить студентам создавать группы, чтобы убедиться в том, что все учащиеся будут происходить на форуме, включающем взрослый. В этом случае группа и создание групп Office 365 могут быть ограничены всеми преподавателями и сотрудниками.

### <a name="2-configure-user-experiences-using-policies"></a>2. Настройка взаимодействия с пользователем с помощью политик

[Политики Teams](teams-policies.md) предоставляют возможность управлять параметрами, доступными для конкретных пользователей или групп пользователей. Политики можно применять для определения того, кому разрешено использовать частный чат, частный звонок, планирование собраний, типы контента, которые можно использовать совместно и многое другое.

**Более высокие образовательные учреждения, преподаватели и учащиеся** выигрывают от возможностей, включенных в стандартные (глобальные) политики. Некоторые дополнительные параметры политики можно включить, чтобы добавить дополнительные функции в Microsoft Teams, в том числе [включить возможности перевода в политику обмена сообщениями](https://docs.microsoft.com/microsoftteams/messaging-policies-in-teams#messaging-policy-settings) и разрешить автоматическое создание транскрипции собраний в политике собраний.

Для **студентов-получателей с дополнительными учебными заведениями** могут потребоваться ограниченные возможности, доступные для студентов. Политики устанавливают границы для учащихся. Так как заполнение учащихся часто является большим набором пользователей, и они часто получают наиболее строгие параметры, рекомендуется вносить изменения в политику для учащихся в политики "Глобальная (по умолчанию)".

Вот набор распространенных конфигураций политик, не используемых по умолчанию, которые будут назначены основным дополнительным студентам для ограничения неуправляемой связи между студентами.

#### <a name="messaging-policy"></a>Политика обмена сообщениями

- Изменить значение на "Выкл."
- Для оценки содержимого GIF установлено значение "Strict"
- Перевести сообщения на "вкл."
- Отправка срочных сообщений с помощью уведомлений о приоритетах со значением OFF
- Удаление пользователей из групповых чатов, для которых выбрано значение OFF

#### <a name="meeting-policy"></a>Политика собраний

- Разрешить собрание прямо сейчас в каналах со значением OFF
- Установка запрета на отключение надстройки Outlook
- Разрешить для расписания собраний на канале значение OFF
- Разрешить планирование личных собраний со значением "выключено"
- Разрешить собрание прямо сейчас в личных собраниях со значением "Выкл."

#### <a name="live-events-policy"></a>Политика живых событий

- Для расписания установлен запрет на отключение

#### <a name="calling-policy"></a>Политика звонков

- Установка значения "OFF" для частных вызовов

#### <a name="teams-policy"></a>Политика Teams

- Создание частных каналов со значением OFF

**Основным сотрудникам учебного заведения и преподавателям** должны быть назначены политики, которые предоставляют основные возможности, которые могут быть ограничены для учащихся. Создавайте новые политики, которые разрешают подписываться на личное общение и планирование собраний (параметры по умолчанию для новой политики). [Назначьте эти политики вашим сотрудникам и преподавателям с помощью членства в группе безопасности](https://docs.microsoft.com/microsoftteams/assign-policies#assign-a-policy-to-a-group).

## <a name="start-using-teams"></a>Начало работы с Teams

### <a name="create-class-teams-for-secure-classroom-use"></a>Создание групп классов для безопасного использования аудитории

Microsoft Teams для образовательных учреждений — это [конкретные типы групп](https://support.office.com/article/choose-a-team-type-to-collaborate-in-microsoft-teams-0a971053-d640-4555-9fd7-f785c2b99e67) для обучающего использования. [Тип группы класс](https://support.office.com/article/create-a-class-team-in-microsoft-teams-fae422eb-58b7-4431-9ff2-a4b9b6ae7c5b) разработан для учебных заведений, в том числе: назначение, записная книжка OneNote для занятий, [Папка "материалы](https://support.office.com/article/Use-folders-to-create-read-only-files-for-students-or-other-team-members-0e7791d7-8c9c-4749-9bca-984289477988) для занятий", позволяющая защитить содержимое только для учащихся только для чтения, а также возможность отключения от слушателей. Существует два способа развертывания команд класса.

1. Приложение [School Data Sync](https://sds.microsoft.com/) (SDS) можно **настроить**, допуская создание групп классов для всех классов на основе данных в информационной системе School. В ходе этого процесса будут подготавливаться команды для каждого раздела, и вы сможете выполнять синхронизацию с преподавателем и студентами учащихся. [Преподаватели получат возможность подготовить свою команду](https://support.office.com/article/activate-early-access-class-teams-created-with-school-data-sync-0d154696-66ab-4fcf-b22f-c3d9a82aaf78) до адмиттинг учащихся. Кроме того, если преподаватель не использует команду, учащиеся не будут допущены в группу, так как преподаватель не щелкает "активировать". SDS поддерживает более 80 разных информационных систем (SIS) для импорта данных, и [Группа поддержки SDS](https://aka.ms/SDSSupport) готова помочь вам в планировании и настройке.
1. **Преподаватели настроили** собственный тип класса и приглашат учащихся. Преподаватели могут сделать это [с помощью добавления учащихся в группу](https://support.office.com/article/add-a-student-to-a-class-team-b88263bb-ace1-4702-8a48-f8a2cf4af954), [совместного использования кода присоединения](https://support.office.com/article/Create-a-link-or-a-code-for-joining-a-team-11b0de3b-9288-4cb4-bc49-795e7028296f)или [предоставления ссылки на команду](https://support.office.com/article/Create-a-link-or-a-code-for-joining-a-team-11b0de3b-9288-4cb4-bc49-795e7028296f). По возможности вы должны добавить в группу учащихся, чтобы слушатели могли получать доступ к ним и получать уведомления о том, что они были добавлены в группу.

После настройки Teams владельцы могут [настраивать параметры своей группы](https://support.office.com/article/find-your-class-team-s-settings-in-microsoft-teams-2592d4de-581d-4952-9028-02317880c158) , включая добавление [изображения группы](https://support.office.com/article/change-your-team-picture-02ea2af6-b49d-4de8-9551-1a5e472993c0), [создание каналов](https://support.office.com/article/create-student-project-groups-channels-in-microsoft-teams-f85b3c07-fb87-4b94-883b-9be55f4b1e45?ui=en-US&rs=en-US&ad=US) для субъектов или групп для совместной работы, [Добавление приложения](https://support.office.com/article/add-an-app-to-teams-b2217706-f7ed-4e64-8e96-c413afd02f77) , такого как куизлет/Flipgrid/Kahoot, к существующему образовательному материалу, а также [упоминание команды для их первой записи](https://support.office.com/article/using-the-conversation-tab-in-microsoft-teams-53d1c530-3797-4a6f-9892-6760f8763df2) , чтобы уведомить всех и начать беседу.

### <a name="create-staff-teams-for-staff-communication-and-collaboration"></a>Создание отделов персонала для общения и совместной работы

[Группы сотрудников](https://support.office.com/article/create-a-staff-team-in-microsoft-teams-314ac9d5-36a9-408e-8ae4-7ef20e9f1ddf) предназначены для администраторов и сотрудников для учебного заведения, которые упрощают совместное использование информации и совместную работу с другими приложениями, включая объявления, параметры собраний, общий доступ к содержимому и внешние приложения, такие как [планировщик для слежения за](https://support.office.com/article/create-a-plan-with-planner-d000976a-7490-4ddf-b9af-09ee764891e2)выполнением задач. Администраторы учебного заведения могут добавлять в группу участников учебного заведения с помощью мастера создания групп, [добавлять участников после создания группы](https://support.office.com/article/add-members-to-a-team-in-teams-aff2249d-b456-4bc3-81e7-52327b6b38e9), а также [делиться кодом присоединения или ссылкой на группу](https://support.office.com/article/create-a-link-or-a-code-for-joining-a-team-11b0de3b-9288-4cb4-bc49-795e7028296f). [Создание каналов](https://support.office.com/article/create-a-channel-in-teams-fda0b75e-5b90-4fb8-8857-7e102b014525) — это отличный способ организовать беседу и файлы по рабочему потоку или теме. [Руководство для владельцев групп](https://support.office.com/article/go-to-guide-for-team-owners-75f9669b-bd8f-457d-b60b-ac2ac9c8ead4?ui=en-US&rs=en-US&ad=US) — это отличное место, в котором рассказывается о пошлинах и возможностях владельцев групп.

## <a name="teams-meeting-scenarios"></a>Сценарии для собраний Teams

### <a name="collaborative-meetings-for-virtual-classes"></a>Собрания по совместной работе с виртуальными классами

[Собрания Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/tutorial-meetings-in-teams) поддерживают до 250 одновременных участников, в том числе возможность включать звук, видео, [общий доступ к контенту](https://support.office.com/article/show-your-screen-during-a-meeting-90c84e5a-b6fe-4ed4-9687-5923d230d3a7), доски и общие заметки. Собрания можно запланировать в рамках клиента Microsoft Teams для [собрания в личном пространстве или в рамках канала группы](https://docs.microsoft.com/MicrosoftTeams/tutorial-meetings-in-teams), чтобы все участники группы знали о ней. Собрания могут быть записаны и сохранены для просмотра другими пользователями позже. Эти записи также могут быть [транскрибед для упрощения поиска](https://support.office.com/article/Microsoft-Stream-automatically-creates-closed-captions-for-videos-8d6ac353-9ff2-4e2b-bca1-329499455308) обсуждаемого содержимого. Для собраний можно использовать веб-камеру, микрофон и динамик мобильного телефона, а также качество звука и видео от [Microsoft Teams с оптимизированными устройствами](https://products.office.com/microsoft-teams/across-devices/devices).

### <a name="districtuniversity-events-or-updates"></a>Мероприятия и обновления для студентов и университетов

Для некоторых инструкций требуются большие аудитории и дополнительные возможности производства. Эти собрания часто задали выступающие, производители и модераторы Q&A. Microsoft Teams поддерживает эти сеансы с помощью [событий Microsoft Teams Live](teams-live-events/what-are-teams-live-events.md). События Live можно использовать в сценариях, например округах, обновлениях для учащихся, адресах лидерского заведений и инструкции для крупных классов или групп студентов, а также для продления подписки на сообщество пользователей. Дополнительные сведения о проведении Live Sessions можно найти в статье [планирование и планирование событий](https://support.office.com/article/video-plan-and-schedule-a-live-event-f92363a0-6d98-46d2-bdd9-f2248075e502)Live, [Создание событий](https://support.office.com/article/video-produce-a-live-event-34c89e79-ffd4-4a6a-baf6-77055e0709cb)в реальном времени, [участие](https://support.office.com/article/video-attend-a-live-event-d837ad8d-ce34-44d0-9744-9beb50e943ac)в мероприятиях и использование [Q&a](https://support.office.com/article/video-moderating-a-q-a-4984e582-8c66-4ea3-aaaf-d93cf62e1b76).
