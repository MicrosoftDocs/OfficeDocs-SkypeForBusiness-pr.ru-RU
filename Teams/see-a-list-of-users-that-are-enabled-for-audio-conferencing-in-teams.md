# <a name="see-a-list-of-users-that-are-enabled-for-audio-conferencing-in-microsoft-teams"></a>Просмотреть список пользователей, для которых включено для аудиоконференции в группах Майкрософт

После включения Microsoft группам пользователей в вашей организации для аудиоконференции можно просмотреть список тех пользователей, которые будут включены. При просмотре списка, также будут отображаться для каждого пользователя в списке тип поставщика услуг аудиоконференций, они используется номер телефона-связи по умолчанию для пользователя, и, если ваша организация не включен для идентификаторы динамических конференции, статические конференции идентификаторы для собраний аудиоконференций, которые они организации.

> [!NOTE]
> [!INCLUDE [updating-admin-interfaces](includes/updating-admin-interfaces.md)]
  
## <a name="viewing-a-list-of-users"></a>Просмотр списка пользователей

- В левой панели навигации щелкните **Пользователи**.


## <a name="what-else-should-i-know"></a>Дополнительные сведения

- При просмотре списка пользователей, для которых включено, можно выбрать пользователя из списка, чтобы изменить параметры аудиоконференций для этого пользователя.
    
- При выборе одного пользователя, который настроен для использования в качестве поставщика аудиоконференций Майкрософт, можно просмотреть номер телефона по умолчанию и включена ли вашей организации для динамического конференции идентификаторы и может сбросить идентификатор конференции для собраний, Упорядочивает пользователя.
    
- Когда вы выбираете один пользователь, который настроен на использование поставщика аудиоконференций сторонних производителей, можно просмотреть имя поставщика аудиоконференций, бесплатный номер телефона и номер бесплатный номер телефона (если они настроены).
    
   
- Кнопка поиска можно использовать для поиска для отдельного пользователя в списке.
