# <a name="plan-for-microsoft-365-government---gcc-high-deployments"></a>Планирование для государственных учреждений 365 Microsoft - высокая GCC развертываниях

— Это руководство для ИТ-специалистов, которые ведут развертываний Office 365 в "мне НРАВИТСЯ" Федеральное государственных сущностей или других лиц, которые обрабатывают данные, подлежащие правительственным нормативам и требования, где использовать государственных 365 Майкрософт — высокая GCC лучше для удовлетворения этих требований.

> [!NOTE]
> Если ваша организация уже выполнены государственных 365 Майкрософт — высокая GCC требованиям к и применяется для и принятия в программу, можно пропустить шаги 1 – 4 и перейти непосредственно на шаге 5, чтобы приступить к развертыванию.

## <a name="step-1-determine-whether-your-organization-needs-microsoft-365-government---gcc-high-and-meets-eligibility-requirements"></a>Шаг 1. Определяют, требуется Microsoft 365 государственных организаций - GCC высокой и удовлетворяет требованиям к вашей организации. 

Microsoft 365 государственных организаций - GCC высокой среды соблюдение США государственных требований для облачных служб. В дополнение к использовать функции и возможности Office 365, организации используют преимущества следующие функции, которые являются уникальными для государственных организаций Microsoft 365 — высокая GCC:

- Содержимое вашей организации клиента логически подразделить от клиента контента в службах профессиональной Office 365 корпорацией Майкрософт.
- В Соединенных Штатах Америки будет храниться содержимое вашей организации клиента.
- Доступ к содержимому вашей организации клиента ограничено промежуточная численности специалистов корпорации Майкрософт.
- Microsoft 365 государственных – GCC высокой соответствует сертификации и другие навыки, необходимые для клиентов в государственном секторе.

Можно найти дополнительные сведения о Microsoft 365 государственных – GCC высокой предложение для НАС правительственных клиентов в [Office 365 государственных планы](https://products.office.com/government/compare-office-365-government-plans), включая [требования](https://products.office.com/government/compare-office-365-government-plans#EligibilityRequirements).

[Описание службы Office 365 США государственных](https://docs.microsoft.com/en-us/office365/servicedescriptions/office-365-platform-service-description/office-365-us-government/office-365-us-government) : описание преимуществ платформы, которые центром на обеспечение соответствия требованиям в США.


> [!Tip]
> Может потребоваться перенос таблиц данных в описание службы в книгу Excel и добавьте два столбца: **Relevant для организации планировщика** и **потребностям организации планировщика**. Затем можно просмотреть этот список с коллегами, чтобы убедиться, что данная служба отвечает требованиям вашей организации.
