# <a name="edit-microsoft-teams-user-settings-in-bulk"></a>Изменение параметров пользователя группами Майкрософт в пакетном режиме

Как администратор управлять группами параметров пользователя в центре администрирования группами Майкрософт. На странице " **Пользователи** " можно просматривать сведения, такие как учетная запись и сведения о лицензиях и изменение политики и другие параметры. В то же время, можно изменить параметры для пользователей по отдельности или для нескольких пользователей.

## <a name="edit-user-settings-in-bulk"></a>Изменение параметров пользователя в пакетном режиме

Использование центра администрирования группами Майкрософт для изменения параметров для нескольких пользователей за раз. Мы рекомендуем изменения параметров для 20 пользователей за раз. Чтобы изменить параметры для большого числа пользователей, с помощью PowerShell. Для получения дополнительных сведений см [Команды PowerShell](teams-powershell-overview.md).

1. В левой области переходов центра администрирования группами Майкрософт выберите **пользователей**.
2. Поиск пользователей, которые необходимо изменить или фильтрации представления для отображения пользователей, вы хотите изменить.
3. В **& #x 2713;** столбец (флажок), выбора пользователей, выполните одно из следующих:
    - Выберите пользователей по одному за раз. **_AMP_ #x 2713;** отображается рядом с пользователями, которые вы выбрали. Если выбран параметр более 20 пользователей, не будут блокироваться, но следует помнить, что выбрано большее количество пользователей, операция будет занимать значительно больше времени выполнения.

        ![Снимок экрана страницы пользователей, отражающая Выбор пользователя](media/bulk-edit-user-settings-select-users.png)

    - Нажмите кнопку & #x 2713; (флажок) в верхней части таблицы для выбора всех пользователей (не более 20 пользователей) и нажмите кнопку **Продолжить выберите все** , чтобы завершить выбор в диалоговом окне **Выбор ограничение** .

        ![Снимок экрана страницы пользователей, отражающая предел выбора](media/bulk-edit-user-settings-select-all-limit.png) <br> **_AMP_ #x 2713;** отображается рядом с элементом выбранных пользователей.

        ![Снимок экрана страницы пользователей, отражающая 20 выбранных пользователей](media/bulk-edit-user-settings-select-all.png)
4. Нажмите кнопку **Изменить параметры**, внесите необходимые изменения и нажмите кнопку **Сохранить**.

    ![Снимок экрана изменить параметры области](media/bulk-edit-user-settings-edit-settings.png)
