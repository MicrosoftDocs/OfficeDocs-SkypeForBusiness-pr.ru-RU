# <a name="set-up-call-analytics"></a>Настройка аналитики звонков

Администратор Teams или Skype для бизнеса Online может использовать средство аналитики звонков для устранения проблем со связью и качеством связи между Skype для бизнеса и Microsoft Teams. Возможно, вам будет полезно настроить следующие возможности в средстве аналитики звонков.
  
- Настройте разрешения, которые позволяют другим сотрудникам, например агентам службы поддержки, использовать средство аналитики звонков, но запретить им доступ к остальной части центра администрирования Microsoft Teams. 
    
- Добавьте сведения о здании, сайте и клиенте для вызова аналитического файла, загрузяя файл данных в формате TSV или CSV.
    
**Средство аналитики звонков теперь доступно в центре администрирования Microsoft Teams**. Чтобы просмотреть все сведения о звонках и данные для пользователя, используйте вкладку **Журнал звонков** . Это можно сделать на странице профиля пользователя, выполнив одно из указанных ниже действий.

- Найдите пользователя на панели мониторинга.
  
   ![Снимок экрана: Поиск пользователей на панели мониторинга](media/set-up-call-analytics-image-1.png)

-  В области навигации слева выберите пункт **Пользователи** .

   ![Снимок экрана: левая область навигации](media/set-up-call-analytics-image-2.png)
  
## <a name="set-call-analytics-permissions"></a>Настройка разрешений на доступ к анализу звонков
<a name="BKMK_SetCAPerms"></a>

У администратора есть полный доступ ко всем функциям средства аналитики звонков. Кроме того, вы можете назначать роли Azure Active Directory сотрудникам службы поддержки. Назначьте роль специалиста по поддержке взаимодействия Teams для пользователей, которые должны иметь ограниченный вид средства аналитики звонков. Назначьте роль инженера поддержки взаимодействия Teams пользователям, которым необходим доступ ко всем функциональным возможностям аналитики звонков. Оба уровня разрешений запрещают доступ к оставшейся части центра администрирования Microsoft Teams.

> [!NOTE]
> Роль специалиста по поддержке связи эквивалентна поддержке уровня 1, и роль инженера по поддержке связи эквивалентна поддержке уровня 2.

Дополнительные сведения о ролях администраторов Teams можно найти [в разделе Использование ролей администратора Microsoft Teams для управления группами](using-admin-roles.md).

Вместо этого они собирают связанную информацию, а затем эскалировать специалистам службы поддержки по связи. Специалистам службы поддержки связи доступны сведения в подробных журналах звонков, которые скрыты специалистами службы поддержки связи. ## <a name="upload-a-tsv-or-csv-file-to-add-building-site-and-tenant-information"></a>Добавление файлов в формате TSV или CSV для добавления сведений о сборке, сайте и клиенте
<a name="BKMK_UploadFiles"> </a>

Вы можете добавить сведения о зданиях, сайтах и клиентах для вызова аналитического файла, загрузя CSV-или. tsv. С помощью этой информации средство аналитики звонков может сопоставлять IP-адреса с физическими адресами. Агенты службы поддержки могут найти эти данные, чтобы помочь вам выявить тенденции проблем с звонком. Например, почему количество пользователей в одной и той же сборке имеет аналогичные проблемы с качеством связи? 

Если вы являетесь администратором Teams и Skype для бизнеса, вы можете использовать существующий файл данных из группы & панель мониторинга качества звонков в Skype для бизнеса. Сначала файл загружается из панели мониторинга качества вызовов, а затем вы отправляете его для анализа звонков. 

- Чтобы загрузить существующий файл данных, перейдите на > ******панель мониторинга качества звонков**в >  **центре администрирования Microsoft Teams**. В списке **Мои отправки** выберите пункт **загрузить** рядом с нужным файлом.

- Чтобы добавить новый файл, перейдите в раздел > **Расположение** **центра администрирования Microsoft Teams**, а затем выберите команду **отправить данные о местоположении** или **заменить данные о местоположении**.
  
Если вы создаете файл в формате TSV или CSV с нуля, ознакомьтесь со статьей [Формат файла данных клиента и структура файла данных для построения](turning-on-and-using-call-quality-dashboard.md#BKMKTenantDataFile).
