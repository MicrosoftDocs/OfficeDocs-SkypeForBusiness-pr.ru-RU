# <a name="set-up-call-analytics"></a>Настройка аналитики звонков

Как группы или Скайп для бизнеса в Интернет администратора Analytics вызова можно использовать для устранения неполадок Скайп для бизнеса и группами Майкрософт вызвать проблемы качества и подключения. Может быть полезно для настройки следующих возможностей в аналитике звонков:
  
- Установка разрешений, которые позволяют другим персонала, таких как служба технической поддержки агенты, используйте вызова аналитики, но запретить доступ к остальной части центра администрирования группами Майкрософт. 
    
- Добавьте построение, сайта и сведения о клиента вызова Analytics путем загрузки .tsv или CSV-файла данных.
    
**Вызов Analytics теперь доступен в центре администрирования группами Майкрософт**. Чтобы просмотреть сведения о вызовах и данные для пользователя, используйте вкладку **Журнал звонков** . Это можно сделать, используя на странице профиля пользователя, выполните одно из следующих:

- Поиск пользователей из панели мониторинга.
  
   ![Снимок экрана пользовательского поиска на панели мониторинга](media/set-up-call-analytics-image-1.png)

-  Выберите **пользователей** в левой панели навигации.

   ![Снимок экрана с левой области переходов](media/set-up-call-analytics-image-2.png)

## <a name="upload-a-tsv-or-csv-file-to-add-building-site-and-tenant-information"></a>Отправка .tsv или CSV-файла для добавления построение сайта и клиентов сведения
<a name="BKMK_UploadFiles"> </a>

Можно добавить построение, сайта и сведения о клиента для вызова Analytics путем отправки файла CSV или .tsv. Эти сведения вызова Analytics можно сопоставить IP-адресов физических местоположениях. Вы или служба технической поддержки агенты полезные эти сведения помогут отслеживать тенденции задач на звонок. Например, почему похожи количество пользователей в одном здании необходимости вызвать проблемы качества? 

Если вы групп и Скайп для администрирования бизнес существующего файла данных из группы & Скайп можно использовать для мониторинга качества звонков Business. Во-первых Загрузите файл из вызывать мониторинга качества и затем вы отправляете для вызова аналитики. 

- Чтобы загрузить файл данных, перейдите в **Центр администрирования группами Майкрософт** > **Панели мониторинга качества звонков** > **Отправить сейчас**. В списке **Мои передача** рядом с пунктом нужный файл нажмите кнопку **загрузить** .

- Чтобы загрузить файл, перейдите в **Центр администрирования группами Майкрософт** > **расположения**и выберите команду **Отправить данные о размещении** или **заменить данные о размещении**.
  
Если вы создаете .tsv или CSV-файл с нуля, видеть [данные клиента файла формат и структура файла данных построения](turning-on-and-using-call-quality-dashboard.md#BKMKTenantDataFile).
