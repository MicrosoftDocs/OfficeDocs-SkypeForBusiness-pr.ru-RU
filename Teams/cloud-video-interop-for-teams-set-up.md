# <a name="set-up-cloud-video-interop-for-microsoft-teams"></a>Настройка взаимодействия Microsoft Teams с облачными видеослужбами

Выбрав партнеров по облачному видеосвязи [, необходимо](cloud-video-interop.md) спланировать развертывание, настроить сведения о подготовке и ключ клиента партнера, а также дать согласие на использование приложения для взаимодействия с видео в организации. Процесс описан на следующей схеме.

![Развертывание CVI в организации.](media/deploying-cvi.png)

## <a name="plan"></a>Планирование

Сведения [об идентификации партнеров или партнеров для использования в организации см](cloud-video-interop.md). в статье Microsoft Teams Cloud Video Interop for Microsoft Teams.

Чтобы спланировать включение на уровне пользователя, одновременный или на уровне сайта, спланируйте следующее:

- Выбор модели развертывания или размещенной модели для использования
- Выберите оптимальный для вашей организации план лицензирования.
- Запланируйте емкость виртуальных машин, так как вы размещаете видео инфраструктуру.

## <a name="configure"></a>Настройка

Чтобы настроить облачное видеосвязь, выполните следующие действия.

1. Получите сведения о конфигурации от выбранного партнера или партнера (ключ клиента, идентификаторы приложения...). В организации можно использовать одного или нескольких партнеров по видеосвязи.

2. Убедитесь, что сеть настроена правильно. Настройте брандмауэр видео на основе стандартов для поддержки обхода сети периметра. Например:
    - Cisco VCS-e
    - Polycom RPAD

3. Настройте интегрированные комнаты с exchange и OTD. В большинстве случаев в вашей среде необходимо настроить дополнительный ретранслятор.

## <a name="provision"></a>Предоставление

Ключом клиента будет подключение к службе партнеров. В следующем примере 813878896@t.plcm.vc ключ клиента.

![Пример ключа клиента.](media/tenant-key-example.png)

Для подготовки ключа клиента необходимо выполнить следующие командлеты, а также разрешить выбранным пользователям или всей организации создавать собрания с координатами видеосвязи.

- **[Get-CsTeamsVideoInteropServicepolicy](/powershell/module/skype/get-csteamsvideointeropservicepolicy):** Корпорация Майкрософт предоставляет предварительно созданные политики для каждого из наших поддерживаемых партнеров, которые позволяют назначить партнеров, которые будут использовать для взаимодействия с облачными видео.

    Этот командлет позволяет определить предварительно созданные политики, которые можно использовать в организации. Эту политику можно назначить одному или нескольким пользователям, которые используют Grant-CsTeamsVideoInteropServicePolicy командлета.

- **[Grant-CsTeamsVideoInteropServicePolicy](/powershell/module/skype/grant-csteamsvideointeropservicepolicy):** Командлет Grant-CsTeamsVideoInteropServicePolicy позволяет назначить предварительно созданную политику для использования в организации или назначить политику определенным пользователям.

- **[New-CsVideoInteropServiceProvider](/powershell/module/skype/new-csvideointeropserviceprovider):** Используйте этот New-CsVideoInteropServiceProvider, чтобы указать сведения о поддерживаемом партнере CVI, который будет использоваться вашей организацией.

- **[Set-CsVideoInteropServiceProvider](/powershell/module/skype/set-csvideointeropserviceprovider):** Используйте этот Set-CsVideoInteropServiceProvider для обновления сведений о поддерживаемом партнере CVI, используемом вашей организацией.

- **[Get-CsVideoInteropServiceProvider](/powershell/module/skype/get-csvideointeropserviceprovider):** Получение всех поставщиков, настроенных для использования в организации.

- **[Remove-CsVideoInteropServiceProvider](/powershell/module/skype/remove-csvideointeropserviceprovider):** Используйте Remove-CsVideoInteropServiceProvider, чтобы удалить все сведения о поставщике, который больше не используется в вашей организации.

## <a name="consent"></a>Согласия

Для присоединения устройств видеоконференций (VTC) к собраниям организаций через службу партнеров необходимо предоставить согласие на разрешение. Эта ссылка на согласие также будет предоставлена вашим партнером.

После выполнения этих действий пользователи, которые включены по отдельности с помощью командлета Grant выше, или все пользователи в организации, если клиент включен, будут иметь координаты VTC во всех Teams собраниях, которые они запланирует. Любые участники VTC могут присоединяться к этим собраниям по этим координатам.

|Имя|Краткое описание разрешения приложения| Описание|
|---|---|---|
|Calls.JoinGroupCall.All|Присоединение к групповым звонкам и собраниям в качестве приложения (предварительная версия)|Позволяет приложению присоединяться к групповым звонкам и запланированным собраниям в организации без необходимости входа пользователя.  Приложение будет присоединено с правами пользователя каталога к собраниям в клиенте.|
|Calls.JoinGroupCallasGuest.All|Присоединение к групповым звонкам и собраниям в качестве гостевого пользователя (предварительная версия)|Позволяет приложению анонимно присоединяться к групповым звонкам и запланированным собраниям в организации без необходимости входа пользователя.  Приложение будет присоединено в качестве гостя к собраниям в клиенте.|
|Calls.AccessMedia.All|Доступ к потокам мультимедиа в вызове в виде приложения (предварительная версия)|Позволяет приложению получать прямой доступ к потокам мультимедиа в вызове без необходимости входа пользователя.|
|OnlineMeetings.Read.All|Чтение сведений о собрании по сети (предварительная версия)|Позволяет приложению считывать сведения о собрании по сети в организации без необходимости входа пользователя.|

## <a name="join"></a>Join

Вы можете присоединяться Teams к собраниям с устройствами VTC следующими способами:

- IVR (интерактивный голосовой ответ)
  - Вы можете выполнить вход на IVR партнера с помощью tenantkey@domain.
  - После входа в партнер IVR вам будет предложено ввести VTC conferenceId, который затем подключит вас к Teams собрания.
- Прямой набор
  - Вы можете напрямую подключаться к собранию Teams без взаимодействия с IVR партнера, используя функцию прямого набора номера, используя полную строку tenantkey. VTC ConferenceId@domain.
- Одно касание набора
  - Если у вас есть встроенная Teams, вы можете использовать возможности одно касания, предлагаемые партнером (без необходимости ввести строку набора номера).

Наконец, взаимодействуйте с Teams пользователями на собраниях, используя аудио- и видеосвязь и общий доступ к содержимому.
