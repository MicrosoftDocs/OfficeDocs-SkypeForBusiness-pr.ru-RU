# <a name="broadcast-meeting-content"></a>Трансляция содержимого собрания 



Teams предоставляет два варианта для трансляции Teams собрания: интерфейс сетевого устройства (NewTek NDI®) и серийный цифровой интерфейс (SDI):

- Технология New® NDI — это современное решение для подключения устройств мультимедиа (например, камеры студии и миксера). Вместо физических подключений технология NDI® обеспечивает подключение к локальной интрасети, в том числе на локальном компьютере.

  Технология NDI® стала стандартным отраслевым решением для создания трансляции содержимого для потоков и стала заметной в мире профессионального широковещательного широковещательного канала.

- SDI используется в широковещательных версиях с 1989 г. и поддерживается на большинстве устаревших аппаратных устройств studio. Аппаратные устройства из AJA Video Systems и Blackmagic Design обеспечивают подключение к устаревшим широковещательным устройствам, которые используют SDI.

> [!NOTE]
> Функция выхода видеоматериалов, поддерживаемая SDI, в настоящее время находится в предварительной версии.

NDI® и SDI поддерживаются во всех региональных сообществах.

Доступ к использованию NDI и SDI определяется политикой собраний для пользователя, который пытается активировать эту функцию. Для наиболее безопасного решения не включай локальный параметр потоковой передачи в качестве глобального параметра.


## <a name="enable-broadcast-features"></a>Включить функции широковещательного вещания

Чтобы включить функции ® NDI и SDI для пользователя:

1. Администратор клиента должен включить локализованную потоковую передачу для политики собраний. 

2. Конечный пользователь должен включить локализованную потоковую передачу для своего клиента.


Чтобы включить этого пользователя, воспользуйтесь центром администрирования Teams или Teams PowerShell.

В Центре Teams выберите Политики собраний > **видео** & аудио и выберите **Разрешить потоковую передачу NDI**.

Чтобы использовать PowerShell, используйте Set-CsTeamsMeetingPolicy powerShell следующим образом:

```PowerShell
Set-CsTeamsMeetingPolicy -Identity MEETING_POLICY -AllowNDIStreaming $true
```

После заполнения этого изменения конечный пользователь должен включить локовую потоковую передачу для своего клиента из **Параметры**  >  **разрешений**. Дополнительные сведения [см.](https://support.microsoft.com/office/broadcasting-audio-and-video-from-teams-with-ndi-technology-e91a0adb-96b9-4dca-a2cd-07181276afa3)в этой Teams.
