# <a name="shared-line-appearance-in-microsoft-teams"></a>Функция Shared Line Appearance в Microsoft Teams

Внешний вид общей строки является частью компонента делегирования, который позволяет пользователю выбрать представителю ответ или обработки звонков от их имени. Эта возможность полезна, если пользователь имеет помощника, которые регулярно обрабатывает вызовы пользователя. В контексте внешний вид общей строки, руководитель — это кто-то кто авторизует делегата выполнять либо принимать звонки от их имени и делегат может совершать и получать звонки от имени другого пользователя.

> [!IMPORTANT]
> Эта функция доступна только в режиме только группы развертывания. Для получения дополнительных сведений о режимах развертывания групп видеть [понять группами Майкрософт и Скайп для бизнеса сосуществования и взаимодействия](teams-and-skypeforbusiness-coexistence-and-interoperability.md)

## <a name="license-required"></a>Требуется лицензия

Пользователь должен быть пользователем голосовой связи предприятия делегата или настроить делегирование и разрешить другим пользователям выполнять либо принимать звонки от его имени.

Менеджеры и делегаты должны быть включены корпоративной голосовой связи. На взаимодействие с общей строки является частью делегирование и требуется дополнительная лицензия. Дополнительные сведения о модели лицензирования в разделе [Лицензирование Office 365 для групп Майкрософт](office-365-licensing.md).

## <a name="configuring-delegation-and-shared-line-appearance"></a>Настройка делегирования и внешний вид общей строки

Делегирование и внешний вид общей строки — на основе функций: отсутствуют параметры администрирования для настройки. Сведения о том, как использовать функцию в разделе [общий доступ к телефонной линии с помощью делегата](https://support.office.com/article/share-a-phone-line-with-a-delegate-16307929-a51f-43fc-8323-3b1bf115e5a8)

Администратор клиента должен включить делегирование с помощью параметра **TeamsCallingPolicy AllowDelegation** для этой функции для работы.

## <a name="limitations"></a>Ограничения

Руководители могут добавлять до 25 делегатов и делегаты могут иметь до 25 менеджеры. Нет ограничений на номер отношения делегирования, которые могут быть созданы в клиент. 
 
Если представитель и делегат не в том же месте, — разрешить Звонящего отображались в разных географических расположениях для делегированных (от имени владельца) вызова поставщику PSTN.
