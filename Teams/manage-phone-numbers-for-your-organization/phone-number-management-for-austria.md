# <a name="phone-number-management-for-austria"></a>Управление номерами телефонов для Австралии

Эта таблица содержит сведения о получении номеров телефонов в Австралии для Microsoft Teams и управлении ими.
  
Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
|**Требуемое действие**|**Скачайте подходящие формы и отправьте их нам**|
|:-----|:-----|
|Получение новых номеров телефонов пользователей и служб, которые не указаны в Центре администрирования Microsoft Teams.   <br/> |Скачайте одну из этих форм. <br/>  [Новый запрос на номер телефона для Австралии (версия 5.0) (de-de)](https://download.microsoft.com/download/6/7/e/67e342b8-babc-4b85-bc0d-a7d9cdfa3d87/new-phone-number-request-for-austria-(v.5.0)-(de-de).pdf). <br/>  [Новый запрос на номер телефона для Австралии (версия 5.0) (en-us)](https://download.microsoft.com/download/c/0/e/c0e39cf0-6fa1-41c6-95fd-6f69ed030d15/new-phone-number-request-for-austria-(v.5.0)-(en-us).pdf). <br/>  <br/>Заполните его и [обратитесь к Service Desk TNS.](contact-tns-service-desk.md) <br/>  <br/>Необходимо также включить счет за служебную программу с бизнес-адреса, для которого вы хотите получить новые номера. <br/> 
|Получение номеров телефонов, переданных нам.  <br/> | Скачайте эту форму. <br/>[Буква авторизации (LOA) для Австралии (все цифры) (версия 5.0) (de-at)](https://download.microsoft.com/download/3/e/2/3e27f3ee-0f38-4d73-b8e8-b43b9e1d1381/letter-of-authorization-(loa)-for-austria-(all-numbers)-(v.5.0)-(de-at).pdf) <br/> <br/>Заполните его и [обратитесь к Service Desk TNS.](contact-tns-service-desk.md)  <br/> <br/>Необходимо также включить счет за служебную программу с бизнес-адреса, для которого требуется перенести номера.<br/> |
|Публикация и отмена публикации номеров телефонов в каталоге национальных телефонов.  <br/> |[Обратитесь к Service Desk TNS](contact-tns-service-desk.md), чтобы получить инструкции по отправке списка номеров телефонов, которые вы хотите опубликовать или отменить публикацию.      <br/> |
|Измените номер пользователя на номер службы.  <br/> |[Обратитесь к Service Desk TNS](contact-tns-service-desk.md), чтобы получить инструкции по отправке списка номеров пользователей, которые требуется преобразовать в номера служб. <br/> |
|Обновление службы каталогов для номеров очереди звонков или автосекретаря|Чтобы добавить или изменить номера очереди звонков или автосекретаря, обратитесь к [Service Desk TNS](contact-tns-service-desk.md). |
