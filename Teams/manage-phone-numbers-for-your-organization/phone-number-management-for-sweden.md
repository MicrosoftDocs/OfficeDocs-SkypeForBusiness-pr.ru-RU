# <a name="phone-number-management-for-sweden"></a>Телефон управления номерами для Швеции

В этой таблице вы найдете сведения о получении номеров телефонов в Швеции и управлении Microsoft Teams.
  
Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
|**Требуемое действие**|**Скачайте подходящие формы и отправьте их нам**|
|:-----|:-----|
|Получите новые номера телефонов пользователей или служб, которые не указаны в Microsoft Teams центре администрирования.   <br/> |Скачайте одну из этих форм. <br/>  [Новый запрос номера телефона для Швеции (v.5.0) (sv-se)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-sweden-(v.5.0)-(sv-se).pdf?raw=true) <br/>  [Новый номер телефона для Швеции (v.5.0) (en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-sweden-(v.5.0)-(en-us).pdf?raw=true)<br/>  <br/>Заполните ее и [отправьте его нам](mailto:ptneu@microsoft.com).|
Перенаправь нам номера телефонов.  <br/> |Скачайте эту форму. <br/>[Док-досье (LOA) для Швеции (все номера) (v.5.0) (sv-se)](https://download.microsoft.com/download/b/d/0/bd030c0c-67c5-40fb-a148-d12a4675aa79/letter-of-authorization-(loa)-for-sweden-(all-numbers)-(v.5.0)-(sv-se).pdf)<br/><br/> Заполните ее и [отправьте его нам](mailto:ptneu@microsoft.com). <br/> |
|Публикация и публикация номеров телефонов в национальном каталоге телефонов.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com) список номеров телефонов для публикации или отмены публикации.      <br/> |
|Измените номер пользователя на номер службы.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com) список номеров пользователей, которые необходимо преобразовать в номера служб. <br/> |
