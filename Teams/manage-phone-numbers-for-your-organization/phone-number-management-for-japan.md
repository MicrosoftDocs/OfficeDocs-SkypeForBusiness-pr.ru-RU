# <a name="phone-number-management-for-japan"></a>Управление номерами телефонов для Японии

> [!IMPORTANT]
> Использование SoftBank для звонков в Японии в настоящее время доступно только для самых ранних последователи (EA).

В Японии мы соработали с SoftBank для приобретения и подготовки планов звонков для клиентов с адресом выставления счетов в Японии. SoftBank продавал абонентские планы, предоставит номера пользователей для Японии, а также предоставляет поддержку для управления телефонными номерами, но и номера услуг и лицензии для голосовой конференции, автосекретарей и очередей звонков.  будет по-прежнему продаваться и управляться Office 365.

Дополнительные сведения о [вызове Softbank для Office 365](https://aka.ms/SoftBankVoicePlan).

Используйте эту таблицу для поиска сведений о том, как получить и управлять номерами телефонов в Японии для Microsoft Teams.

Дополнительные сведения можно найти [в разделе Управление номерами телефонов для Организации](manage-phone-numbers-for-your-organization.md).
  
|**Требуемое действие**|**Выполните следующие действия и загрузите подходящие формы**|
|:-----|:-----|
|Получайте новые телефонные номера для планов звонков в Японии.   <br/> |Чтобы получить новые номера телефонов для пользователей, обратитесь в [Softbank по телефону в Office 365](https://aka.ms/SoftBankVoicePlan) . Затем  эти номера будут добавлены вашей организации, вы можете назначить их пользователям. <br/>
|Получайте новые номера телефонов служб для голосовой конференции, автосекретарей и очередей звонков в Японии.  <br/> |[Свяжитесь с нами](mailto:ptnapac@microsoft.com).|
|Получение телефонных номеров, передаваемых в Office 365 для пользователей в Японии.  <br/> |Для того чтобы передать номера телефонов, обратитесь в [Softbank по телефону в Office 365](https://aka.ms/SoftBankVoicePlan) . Затем эти номера будут добавлены вашей организации.  <br/> |
|Получение телефонных номеров, передаваемых в Office 365 для голосовой конференции, автосекретарей и очередей звонков в Японии.  |[Свяжитесь с нами](mailto:ptnapac@microsoft.com) |
|Изменить назначение экстренной адресации пользователям, у которых есть телефонный номер тарифного плана. |Чтобы внести изменения в адрес для экстренного реагирования, обратитесь в [Softbank звоните по Office 365](https://aka.ms/SoftBankVoicePlan) . Они могут обновлять эти сведения для вас.|
|Измените номер пользователя на номер Услуги. |В настоящее время не поддерживается при использовании вызова SoftBank для Office 365.
