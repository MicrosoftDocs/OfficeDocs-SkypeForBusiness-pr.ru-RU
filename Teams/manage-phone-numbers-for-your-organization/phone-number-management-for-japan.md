# <a name="phone-number-management-for-japan"></a>Телефон управления номерами для Японии

В Японии мы сотрудничаем с SoftBank для приобретения и подготовка планов звонков для клиентов с адресом вы выставления счета в Японии. SoftBank будет продавать планы звонков, предоставлять номера пользователей для Японии и предоставлять поддержку управления номерами телефонов. Однако номера служб и лицензии на аудиоконференцию, автослужбу и очереди вызовов будут по-прежнему продаваться и управляться Microsoft 365 и Office 365.

Дополнительные сведения о службе [SoftBank Calling.](https://aka.ms/SoftBankVoicePlan)

Эта таблица используется для получения номеров телефонов в Японии и управления Microsoft Teams.

Дополнительные сведения см. в [управлении номерами телефонов для организации.](manage-phone-numbers-for-your-organization.md)
  
|**Требуемое действие**|**Выполните следующие действия и загрузите подходящие формы**|
|:-----|:-----|
|Получите новые номера телефонов пользователей для планов звонков в Японии.|За [новыми номерами телефонов](https://aka.ms/SoftBankVoicePlan) для пользователей обращайтесь в службу SoftBank Calling. Затем  эти номера будут добавлены вашей организации, вы можете назначить их пользователям. <br/>
|Получите новые номера телефонов служб для аудиоконференций, автослужб и очередей звонков в Японии.|[За инструкциями обратитесь в службу поддержки ТССК.](contact-pstn-service-desk.md)|
|Перевод номеров телефонов в Microsoft 365 или Office 365 **пользователям в** Японии.|Обратитесь к [службе SoftBank Calling,](https://aka.ms/SoftBankVoicePlan) чтобы перенести номера телефонов пользователей. Затем эти номера будут добавлены вашей организации.  <br/> |
|Перенос номеров телефонов в Microsoft 365 или Office 365 очередей **звонков** в Японии.|Обратитесь к [службе SoftBank Calling,](https://aka.ms/SoftBankVoicePlan) чтобы перенести номера телефонов службы. Затем эти номера будут добавлены вашей организации.|
|Измените адрес для экстренного вызова, который назначен пользователям с номером телефона плана звонков.|Обратитесь в [службу звонков SoftBank,](https://aka.ms/SoftBankVoicePlan) чтобы внести изменения в адрес для экстренного вызова. Они могут обновлять эти сведения для вас.|
|Измените номер пользователя на номер службы. |В настоящее время не поддерживается при использовании звонков SoftBank.|
|Обновление службы каталогов для номеров очереди вызовов или автозаетаря|Чтобы добавить или изменить номера очереди вызовов или номера автослужб, обратитесь за инструкциями в службу [поддержки ТСОП.](contact-pstn-service-desk.md) |
