# <a name="phone-number-management-for-japan"></a>Управление номерами телефонов для Япония

В Японии мы партнерсировали с родительской лицензией (SoftBank) для приобретения и подготовки планов звонков для клиентов с адресом выставления счетов в Японии. SoftBank продает планы звонков, предоставление номеров пользователей для японии и предоставьте поддержку для управления номерами телефонов. Тем не менее номера служб и лицензии для аудиоконференций, автосекретарей и очередей звонков будут проданы и управляются Microsoft 365 и Office 365.

Дополнительные сведения о [вызове softbank.](https://aka.ms/SoftBankVoicePlan)

Эта таблица поможет вам найти сведения о получении номеров телефонов и управлении ими в Японии для Microsoft Teams.

Дополнительные сведения см. в [разделе "Управление номерами телефонов для организации".](manage-phone-numbers-for-your-organization.md)
  
|**Требуемое действие**|**Выполните следующие действия и загрузите подходящие формы**|
|:-----|:-----|
|Получите новые номера телефонов для планов звонков в Японии.|Обратитесь [к службе SoftBank](https://aka.ms/SoftBankVoicePlan) для получения новых номеров телефонов ваших пользователей. Затем  эти номера будут добавлены вашей организации, вы можете назначить их пользователям. <br/>
|Получайте новые номера телефонов для аудиоконференций, автосекретарей и очередей звонков в Японии.|[Свяжитесь с нами](mailto:ptnapac@microsoft.com).|
|Перенесите телефонные номера в Microsoft 365 или Office 365 **для пользователей** в Японии.|Обратитесь [к службе SoftBank для](https://aka.ms/SoftBankVoicePlan) переноса номеров телефонов пользователей. Затем эти номера будут добавлены вашей организации.  <br/> |
|Перенесите номера телефонов в Microsoft 365 или Office 365 для **очередей** звонков в Японии.|Обратитесь к [службе Поддержка по переносу](https://aka.ms/SoftBankVoicePlan) номеров служб. Затем эти номера будут добавлены вашей организации.|
|Измените адрес для экстренного реагирования, назначенных пользователям, у которых есть номер телефона для плана звонков.|Чтобы [внести изменения адресов для экстренного](https://aka.ms/SoftBankVoicePlan) реагирования, обратитесь к попросите службы звонков по программе SoftBank. Они могут обновлять эти сведения для вас.|
|Изменить номер пользователя на случайный. |В настоящее время не поддерживается при использовании звонков SoftBank.
