# <a name="phone-number-management-for-japan"></a>Управление номерами телефонов для Японии

> [!IMPORTANT]
> Использование SoftBank для звонков в Японии в настоящее время доступно только для самых ранних последователи (EA).

В Японии мы соработали с SoftBank для приобретения и подготовки планов звонков для клиентов с адресом выставления счетов в Японии. SoftBank будет продавать планы звонков, предоставлять номера пользователей для Японии и поддерживать управление телефонными номерами. Однако номера служб и лицензии для голосовой конференции, автосекретарей и очередей звонков по-прежнему будут продаваться и управляться Microsoft 365 и Office 365.

Дополнительные сведения о [вызове Softbank](https://aka.ms/SoftBankVoicePlan).

Используйте эту таблицу для поиска сведений о том, как получить и управлять номерами телефонов в Японии для Microsoft Teams.

Дополнительные сведения можно найти [в разделе Управление номерами телефонов для Организации](manage-phone-numbers-for-your-organization.md).
  
|**Требуемое действие**|**Выполните следующие действия и загрузите подходящие формы**|
|:-----|:-----|
|Получайте новые телефонные номера для планов звонков в Японии.|Обратитесь в [Softbank](https://aka.ms/SoftBankVoicePlan) для получения новых номеров телефонов для пользователей. Затем  эти номера будут добавлены вашей организации, вы можете назначить их пользователям. <br/>
|Получайте новые номера телефонов служб для голосовой конференции, автосекретарей и очередей звонков в Японии.|[Свяжитесь с нами](mailto:ptnapac@microsoft.com).|
|Получайте доступ к номерам телефонов, передаваемым в Microsoft 365 или Office 365 для **пользователей** в Японии.|Обратитесь в [Softbank](https://aka.ms/SoftBankVoicePlan) , чтобы передать номера абонентов пользователей. Затем эти номера будут добавлены вашей организации.  <br/> |
|Получение телефонных номеров, передаваемых в Microsoft 365 или Office 365 для **очередей звонков** в Японии.|Пожалуйста, свяжитесь с [абонентом Softbank](https://aka.ms/SoftBankVoicePlan) , чтобы передать услуги на телефонные номера. Затем эти номера будут добавлены вашей организации.|
|Изменение адреса для экстренного реагирования, назначенного пользователям, у которых есть телефонный номер тарифного плана.|Чтобы изменить адрес для экстренного реагирования, обратитесь в [Softbank позвонить](https://aka.ms/SoftBankVoicePlan) . Они могут обновлять эти сведения для вас.|
|Измените номер пользователя на номер Услуги. |В настоящее время не поддерживается при использовании вызова SoftBank.
