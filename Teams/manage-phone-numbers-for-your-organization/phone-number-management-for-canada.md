# <a name="phone-number-management-for-canada"></a>Управление номерами телефонов в Канаде

Эта таблица используется для поиска сведений о получении номеров телефонов в Канаде для Microsoft Teams и управлении им.

Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).

| **Требуемое действие** | **Выполните следующие действия и загрузите подходящие формы** |
|:---   |:---    |
| Получите новые номера телефонов пользователей и служб, которых нет в центре администрирования Microsoft Teams.  <br/>   | Скачайте одну из следующих форм:. <br/> [Новый запрос на номер телефона для Канады (5.0) (en-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-canada-(v.5.0)-(en-ca).pdf?raw=true) <br/>[Новый запрос на номер телефона для Канады (5.0) (fr-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-canada-(v.5.0)-(fr-ca).pdf?raw=true)  <br/> <br>Заполните ее и [отправьте его нам](mailto:ptn@microsoft.com).      |
|Перенаправь нам номера телефонов.  <br/> | Скачайте одну из следующих форм:. <br/> [Досье для Канады (номера пользователей и служб) (v.3.2) (en-ca)](https://download.microsoft.com/download/7/5/2/7525e2cb-d496-4133-87a9-bac3ee4a3216/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.3.2)-(en-ca).pdf) <br/> [Досье для Канады (номера пользователей и служб) (v.1.1) (fr-ca)](https://download.microsoft.com/download/c/8/2/c82bbb76-5a2d-4b84-b4a8-f7e32dfd7b94/Letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.1.1)-(fr-ca).pdf)<br/> [Досье для Канады (бесплатные номера) (v.3.2) (en-ca)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf ) <br/>[Доверенность для Канады, бесплатные номера телефонов, версия 1.1 на французском (Канада)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf)<br/> <br>**Примечание.** Если вы пытаетесь получить более 999 номеров телефонов подписчиков или пользователей, заполните досье и отправьте его нам. Дополнительные [сведения см. в подмносях "Перенос](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) номеров телефонов в Teams". <br/> <br>Заполните ее и [отправьте его нам](mailto:ptn@microsoft.com).  <br/> |
|Изменение номера пользователя на номер службы.  <br/>                                 | [Отправьте нам](mailto:ptn@microsoft.com) свой ПИН-код и список номеров пользователей, которые нужно преобразовать в номера служб. Процедуру настройки ПИН-кода см. [в этой статье](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Перенос номеров из одного Microsoft 365 или Office 365 в другой.  <br/>               | [Отправьте нам](mailto:ptn@microsoft.com) список номеров телефонов пользователей (подписчиков), которые необходимо передать, вместе с ПИН-кодом исходной организации. Процедуру настройки ПИН-кода см. [в этой статье](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Обновление службы каталогов для номеров очереди вызовов или номеров автозавода|Чтобы добавить или изменить номера очереди вызовов или номера автоотетаря, обратитесь в службу [поддержки ННР.](contact-pstn-service-desk.md) |
