# <a name="phone-number-management-for-canada"></a>Управление номерами телефонов в Канаде

В этой таблице содержатся сведения о том, как получить и управлять номерами телефонов в Канаде для Microsoft Teams.

Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).


| **Требуемое действие** | **Выполните следующие действия и загрузите подходящие формы** |
|:---   |:---    |
| Получайте новые номера телефонов пользователей, которые не указаны в центре администрирования Microsoft Teams.  <br/>   | Скачайте одну из следующих форм:. <br/> [Запрос на получение новых номеров телефонов для Канады, пользовательские и служебные номера телефонов, версия 1.0 на английском (Канада)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-canada-(user-and-service-numbers)-(v.1.0)-(en-ca).pdf?raw=true) <br/>[Запрос на получение новых номеров телефонов для Канады, пользовательские и служебные номера телефонов, версия 1.0 на французском (Канада)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-canada-(user-and-service-numbers)-(v.1.0)-(fr-ca).pdf?raw=true)  <br/> <br>Заполните ее и [отправьте его нам](mailto:ptn@microsoft.com).  <br/>        
| Получайте новые номера телефонов служб, которые не указаны в центре администрирования Microsoft Teams. <br/> | Для служебных номеров за пределами США [Скачайте эту форму](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-numbers-outside-the-u.s.-(service-numbers)-(v.1.0)-(en-us).pdf?raw=true), заполните ее и [отправьте нам](mailto:ptneu@microsoft.com). |
|Получение телефонных номеров, перенесенных в США.  <br/> | Скачайте одну из следующих форм:. <br/> [Буква авторизации (LOA) для Канады (номера пользователя и службы) (v. 3.2) (en-CA)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.3.2)-(en-ca).pdf?raw=true) <br/> [Доверенность для Канады, пользовательские и служебные номера телефонов, версия 1.0 на французском (Канада)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.1.0)-(fr-ca).pdf?raw=true) <br/> [Буква авторизации (LOA) для Канады (бесплатные номера бесплатно) (v. 3.2) (en-CA)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf?raw=true) <br/>[Доверенность для Канады, бесплатные номера телефонов, версия 1.1 на французском (Канада)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/Letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.1.1)-(fr-ca).pdf?raw=true)<br/> <br>**Примечание.** Если вы хотите получить более 999 абонентов или телефонов пользователей, вам потребуется заполнить письмо для проверки подлинности (LOA) и отправить нам сообщение. Дополнительные сведения [можно найти в разделе Перенос номеров телефонов в Teams](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) . <br/> <br>Заполните ее и [отправьте его нам](mailto:ptn@microsoft.com).  <br/> |
|Изменение номера пользователя на номер Услуги.  <br/>                                 | [Отправьте нам](mailto:ptn@microsoft.com) свой ПИН-код и список номеров пользователей, которые нужно преобразовать в номера служб. Процедуру настройки ПИН-кода см. [в этой статье](/SkypeForBusiness/what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider).  <br/> |
|Перенесите номера из одной Microsoft 365 или Office 365 в другую.  <br/>               | [Отправьте нам](mailto:ptn@microsoft.com) список номеров телефонов пользователей (подписчиков), которые необходимо передать, вместе с ПИН-кодом исходной организации. Процедуру настройки ПИН-кода см. [в этой статье](/SkypeForBusiness/what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider).  <br/> |
