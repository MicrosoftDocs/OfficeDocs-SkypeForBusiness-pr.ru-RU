# <a name="phone-number-management-for-canada"></a>Управление номерами телефонов в Канаде

В этой таблице вы найдете сведения о получении номеров телефонов в Канаде и управлении Microsoft Teams.

Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).

| **Требуемое действие** | **Выполните следующие действия и загрузите подходящие формы** |
|:---   |:---    |
| Получите новые номера телефонов пользователей и служб, которые не указаны в Microsoft Teams центре администрирования.  <br/>   | Скачайте одну из следующих форм:. <br/> [Новый номер телефона для Канады (v.5.0) (en-ca)](https://download.microsoft.com/download/b/4/6/b460be9b-74c6-4ed7-8cdb-e1bebab5d667/new-phone-number-request-for-canada-(v.5.0)-(en-ca).pdf) <br/>[Новый номер телефона для Канады (v.5.0) (fr-ca)](https://download.microsoft.com/download/b/0/2/b020b607-a802-41da-bc6c-b307903c51f1/new-phone-number-request-for-canada-(v.5.0)-(fr-ca).pdf)  <br/> <br>Заполните ее и [отправьте его нам](mailto:ptn@microsoft.com).      |
|Перенаправь нам номера телефонов.  <br/> | Скачайте одну из следующих форм:. <br/> [Док-досье (LOA) для Канады (номера пользователей и служб) (v.3.2) (en-ca)](https://download.microsoft.com/download/7/5/2/7525e2cb-d496-4133-87a9-bac3ee4a3216/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.3.2)-(en-ca).pdf) <br/> [Доверенность для Канады, пользовательские и служебные номера телефонов, версия 1.0 на французском (Канада)](https://download.microsoft.com/download/b/1/a/b1ada279-0711-4144-bc6c-d0f5541c27ef/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.1.0)-(fr-ca).pdf)<br/> [Досье для Канады (бесплатные номера) (v.3.2) (en-ca)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf ) <br/>[Доверенность для Канады, бесплатные номера телефонов, версия 1.1 на французском (Канада)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf)<br/> <br>**Примечание.** Если вы пытаетесь получить более 999 номеров телефонов подписчиков или пользователей, заполните досье и отправьте его нам. Дополнительные [сведения см. в Teams](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) в этой области. <br/> <br>Заполните его и [обратитесь за инструкциями в службу TNS Service Desk.](contact-tns-service-desk.md)  <br/> |
|Измените номер пользователя на номер службы.  <br/>                                 | [Чтобы получить инструкции по](contact-tns-service-desk.md) ПИН-коду и списку пользовательских номеров, которые вы хотите преобразовать в номера служб, обратитесь в службу TNS Service Desk. Процедуру настройки ПИН-кода см. [в этой статье](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Перенос чисел из Microsoft 365 или Office 365 в другой.  <br/>               | [Обратитесь в службу TNS Service Desk,](contact-tns-service-desk.md) чтобы получить инструкции относительно списка номеров телефонов пользователей (абонентов), которые вы хотите перенести вместе с ПИН-кодом организации-источника, из которой вы хотите перенести номера. Процедуру настройки ПИН-кода см. [в этой статье](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Обновление службы каталогов для номеров очереди вызовов или автозаетаря|Чтобы добавить или изменить номера очереди вызовов или автозавода, обратитесь за инструкциями в [службу TNS Service Desk.](contact-tns-service-desk.md) |
