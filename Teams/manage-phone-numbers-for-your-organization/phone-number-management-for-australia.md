# <a name="phone-number-management-for-australia"></a>Управление номерами телефонов в Австралии
В Австралии мы сотрудничаем с telstra для приобретения и подготовка планов звонков для клиентов с адресом вы счета в Австралии. Telstra будет продавать свои планы звонков, предоставлять номера пользователей из Австралии, а также предоставлять поддержку управления номерами телефонов для номеров пользователей и номеров служб для автоответов и очередей звонков. Однако номера служб и лицензии для аудиоконференций будут по-прежнему продаваться и управляться Microsoft 365 и Office 365.

Дополнительные сведения о звонках [Telstra.](https://aka.ms/TelstraVoicePlan)

Эта таблица используется для поиска сведений о получении номеров телефонов для Microsoft Teams и управлении им в Австралии.

Дополнительные сведения см. в [подмногании "Управление номерами телефонов для организации".](manage-phone-numbers-for-your-organization.md)

|Получение телефонных номеров нового пользователя для тарифных планов в Австралии.   <br/> |Обратитесь в [telstra Calling](https://aka.ms/TelstraVoicePlan) для новых номеров телефонов пользователей. Затем  эти номера будут добавлены вашей организации, вы можете назначить их пользователям. <br/>
|При использовании звонков Telstra вы можете получить новые телефонные номера служб, которые можно использовать с автослужбуми облака и очередями звонков для Австралии из Telstra. <br/> |Обратитесь в [telstra Calling для](https://aka.ms/TelstraVoicePlan) нового номера телефона.
