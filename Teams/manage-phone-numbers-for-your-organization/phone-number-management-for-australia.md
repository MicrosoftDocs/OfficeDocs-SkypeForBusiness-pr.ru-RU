# <a name="phone-number-management-for-australia"></a>Управление номерами телефонов в Австралии

В Австралии мы сделали партнером telstra для приобретения и подготовка планов звонков для клиентов с адресом вы счета в Австралии. Telstra будет продавать свои планы звонков и предоставлять номера пользователей из Австралии. Telstra поддерживает управление номерами телефонов как для пользователей, так и для номеров служб для автоотетаря и очередей звонков. Однако номера служб и лицензии для аудиоконференций будут по-прежнему продаваться и управляться Microsoft 365 и Office 365.

Дополнительные сведения о звонках [Telstra.](https://aka.ms/TelstraVoicePlan)

Эта таблица используется для поиска сведений о получении номеров телефонов для Microsoft Teams и управлении им в Австралии.

Дополнительные сведения см. в [подмногании "Управление номерами телефонов для организации".](manage-phone-numbers-for-your-organization.md)
