# <a name="phone-number-management-for-the-united-states"></a>Телефон управления номерами для США

В этой таблице вы найдете сведения о получении номеров телефонов в США (включая Пуэрто-Рико) и управлении Microsoft Teams.

Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
| Требуемое действие | Выполните следующие действия и загрузите подходящие формы |
|:-----|:-----|
|Получите новые номера пользователей или номеров телефонов служб, которые не указаны в Microsoft Teams центре администрирования.  <br/> |Скачайте эту форму.<br/>  [Новый номер телефона для США (v.5.0) (en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-the-united-states-(v.5.0)-(en-us).pdf?raw=true) <br/><br/> Заполните его и обратитесь за инструкциями в [службу поддержки ТССК.](contact-pstn-service-desk.md) |
|Перенаправь нам номера телефонов.  <br/> | Скачайте одну из следующих форм:. <br/> [Досье (LOA) для США (номера пользователей и служб) (v.3.3)(en-us)](https://download.microsoft.com/download/7/3/8/73843692-632f-4078-874d-021f9680e12b/letter-of-authorization-(loa)-for-the-u.s.-(user-and-service-numbers)-(v.3.3)-(en-us).pdf)<br/>[Досье (LOA) для США (бесплатные номера) (v.3.3)(en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(toll-free-numbers)-(v.3.3)-(en-us).pdf?raw=true)<br/><br/>**Примечание** Если вы пытаетесь получить более 999 номеров телефонов подписчиков или пользователей, заполните досье (LOA) и отправьте его нам. Дополнительные [сведения см. в Teams](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) в этой области.          <br/>  <br/>   Заполните его и обратитесь за инструкциями в [службу поддержки ТССК.](contact-pstn-service-desk.md)  <br/> |
|Изменение номера службы или номера службы на номер пользователя  <br/> |[Отправьте нам](mailto:ptn@microsoft.com) ПИН-код и список чисел, которые вы хотите преобразовать. Чтобы настроить ПИН-код, см. [здесь](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Перенос номеров из Microsoft 365 или Office 365 в другой.  <br/> |[Чтобы получить инструкции](contact-pstn-service-desk.md) по списку номеров пользователей (абонентов), которые вы хотите перенести, а также ПИН-код организации-источника, из которой вы хотите перенести номера, обратитесь в службу поддержки ТССК. Чтобы настроить ПИН-код, см. [здесь](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
|Обновление службы каталогов для номеров очереди вызовов или автозаетаря|Чтобы добавить или изменить номера очереди вызовов или номера автослужб, обратитесь за инструкциями в службу [поддержки ТСОП.](contact-pstn-service-desk.md) |
