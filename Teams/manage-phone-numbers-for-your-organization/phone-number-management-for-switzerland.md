# <a name="phone-number-management-for-switzerland"></a>Управление номерами телефонов в Швейцарии

Эта таблица содержит сведения о получении номеров телефонов в Швейцарии для Microsoft Teams и управлении ими.
  
Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).

|:-----|:-----|
|Получение новых номеров телефонов пользователей или служб, которые не указаны в Центре администрирования Microsoft Teams.   <br/> |Скачайте одну из этих форм. <br/>  [Новый запрос номера телефона для Швейцарии (версия 5.0) (de-de)](https://download.microsoft.com/download/6/f/0/6f01dfb0-3e32-410f-8b54-aa1403c41959/new-phone-number-request-for-switzerland-(v.5.0)-(de-de).pdf) <br/>[Новый запрос на номер телефона для Швейцарии (версия 5.0) (fr-fr)](https://download.microsoft.com/download/a/b/5/ab5f89d5-7dd4-4d74-ac71-ee48815b9eb1/new-phone-number-request-for-switzerland-(v.5.0)-(fr-fr).pdf) <br/>[Новый запрос номера телефона для Швейцарии (версия 5.0) (it-it)](https://download.microsoft.com/download/f/0/6/f0602963-745e-4e1c-87fa-bbdb2dc6395b/new-phone-number-request-for-switzerland-(v.5.0)-(it-it).pdf)<br/>  [Новый запрос номера телефона для Швейцарии (версия 5.0) (en-us)](https://download.microsoft.com/download/2/1/d/21d9d44d-43dc-481c-aed8-c9a0a85b372e/new-phone-number-request-for-switzerland-(v.5.0)-(en-us).pdf)<br/><br/>Заполните его и [обратитесь к Service Desk TNS.](contact-tns-service-desk.md)|
|Получение номеров телефонов, переданных нам.  <br/> | Скачайте одну из этих форм. <br/>[Буква авторизации (LOA) для Швейцарии (все цифры) (версия 5.0) (de-ch)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-switzerland-(all-numbers)-(v.5.0)-(de-ch).pdf?raw=true)<br/>[Доверенность (LOA) для Швейцарии (все цифры) (версия 6.0) (fr-ch)](https://download.microsoft.com/download/c/c/c/cccf5c0b-a838-4d2a-a476-430e4ad79a17/letter-of-authorization-(loa)-for-switzerland-(all-numbers)-(v.6.0)-(fr-ch).pdf)<br/>[Буква авторизации (LOA) для Швейцарии (все цифры) (версия 6.0) (it-ch)](https://download.microsoft.com/download/c/a/0/ca098e50-9e77-4dd6-9a79-9074421c7652/letter-of-authorization-(loa)-for-switzerland-(all-numbers)-(v.6.0)-(it-ch).pdf) <br/> <br/>Заполните его и [обратитесь к Service Desk TNS.](contact-tns-service-desk.md) <br/> |

|Измените номер пользователя на номер службы.  <br/> |[Обратитесь к Service Desk TNS](contact-tns-service-desk.md), чтобы получить инструкции по списку номеров пользователей, которые требуется преобразовать в номера служб. <br/> |
