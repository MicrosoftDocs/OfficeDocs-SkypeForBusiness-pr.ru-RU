# <a name="using-powershell-to-manage-teams"></a>Управление группами с помощью PowerShell

Функции в группах можно управлять с помощью PowerShell или группами Майкрософт и Скайп по центру администрирования бизнеса. 

> ! [Примечание] Не все компоненты в группах можно управлять с помощью модуля соединителя группы. Может потребоваться использовать Скайп для Business connector. Просмотреть [загрузить и установить Скайп для бизнеса в Интернет соединителя](https://docs.microsoft.com/skypeforbusiness/set-up-your-computer-for-windows-powershell/download-and-install-the-skype-for-business-online-connector)

### <a name="step-1-prerequisites"></a>Шаг 1: предварительные требования

Удаленное управление группами Майкрософт с помощью PowerShell поддерживается только в 64-разрядных компьютеров под управлением одной из следующих операционных систем:
  
- Windows 10
- Windows 8.1
- Windows 8 
- Windows Server 2012 R2
- Windows Server 2012
- Windows Server 2008
- Windows 7
    
В дополнение к поддерживаемая операционная система на компьютере необходимо также работать следующие:
  
- PowerShell 3.0 или выше
    
- Модуль PowerShell соединителя группы

[Используйте этот раздел справки](https://docs.microsoft.com/skypeforbusiness/set-up-your-computer-for-windows-powershell/download-and-install-windows-powershell-3-0)
