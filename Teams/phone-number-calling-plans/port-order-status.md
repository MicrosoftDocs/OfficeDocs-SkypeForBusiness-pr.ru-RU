# <a name="whats-the-status-of-your-port-orders"></a>Каково состояние ваших заказов на перенос номеров?

Чтобы узнать состояние заказа на перенос, в левой области навигации Центра администрирования Microsoft Teams перейдите в > **голосового** порта и выберите История  >   **заказов**. Каждый статус заказа на перенос указан в **столбце Состояние.** Узнайте, [сколько времени займет перенос номеров,](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) чтобы узнать о процессе заказа. 

В таблице ниже перечислены состояния заказа на перенос, а также действия, которые можно при необходимости принять.

|**Статус**|**Можете ли вы просмотреть заказ?**|**Можете ли вы изменить заказ?**|**Можно ли отменить заказ?**|**Можно ли удалить заказ?**|**Описание**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Обработки** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ создан администратором и получен корпорацией Майкрософт.  <br/> |
|**Связь с оператором** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ получен и утвержден корпорацией Майкрософт, и мы работаем над его одобрением у потери оператора связи.  <br/> |
|**Перенос утвержден** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ принят у потери оператора связи, а также установлен срок выполнения заказа.  <br/> |
|**Ожидание передачи** <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Передача будет отознана менее чем через 24 часа, поэтому заказ больше нельзя будет изменить или отменить.  <br/> |
|**Ошибка** <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Да (в настоящее время вы не можете удалить заказ на перенос при ошибке. Заказ на перенос необходимо создать повторно или обратиться в службу поддержки [ННР.](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md)  <br/> |Проиграв оператор отклонил заказ.  <br/> |
|**Завершена** <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Номера успешно перенесены.  <br/> |
|**Отменен** <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Администратор отменил заказ.  <br/> |

Полные пошаговую инструкции см. в этой [Teams.](transfer-phone-numbers-to-teams.md)

Если вам нужна помощь или вам нужно получить дополнительные номера телефонов, обратитесь в службу [поддержки ННР.](../manage-phone-numbers-for-your-organization/contact-pstn-service-desk.md)
