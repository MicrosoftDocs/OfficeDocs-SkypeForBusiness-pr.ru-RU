# <a name="whats-the-status-of-your-port-orders"></a>Каково состояние ваших заказов на перенос номеров?

Чтобы просмотреть состояние заказа на перенос, в левой области навигации Центра администрирования Microsoft Teams перейдите к разделу > **голосовых** >  портов **и** щелкните "**Журнал заказов"**. Каждое состояние заказа порта отображается в **столбце "Состояние** ". Узнайте [, сколько времени занимает перенос номеров](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) , чтобы узнать о процессе заказа. 

В следующей таблице перечислены состояния порядка портов, а также действия, которые можно выполнить при необходимости.

|**Статус**|**Можно ли просмотреть заказ?**|**Можно ли изменить порядок?**|**Можно ли отменить заказ?**|**Можно ли удалить заказ?**|**Описание**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Обработки** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Администратор создал заказ, и он был получен корпорацией Майкрософт.  <br/> |
|**Связывание с оператором связи** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ был получен и утвержден корпорацией Майкрософт, и мы работаем с проигравшей службой, чтобы утвердить его.  <br/> |
|**Передача утверждена** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ был принят проигравшей службой, а дата обязательства по фирме заказа (FOC) задана.  <br/> |
|**Ожидание передачи** <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Время передачи меньше 24 часов, поэтому заказ больше не может быть изменен или отменен.  <br/> |
|**Ошибка** <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Да (в настоящее время вы не можете удалить порядок портов при возникновении ошибки. Необходимо повторно создать заказ на перенос или обратиться в [службу технической поддержки TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).  <br/> |Проиграв оператор отклоняет заказ.  <br/> |
|**Завершена** <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Номера успешно переданы.  <br/> |
|**Отменен** <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Администратор отменил заказ.  <br/> |

Полные пошаговые инструкции см. в разделе ["Передача номеров телефонов в Teams"](transfer-phone-numbers-to-teams.md).

Если вам нужна помощь или вам нужно получить дополнительные номера телефонов, обратитесь в [службу технической поддержки TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).
