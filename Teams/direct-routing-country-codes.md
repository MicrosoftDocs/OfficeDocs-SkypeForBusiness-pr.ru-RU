# <a name="direct-routing-media-path-country-codes"></a>Коды стран пути к носителю прямой маршрутизации

При выборе пути маршрутизации для мультимедиа прямая маршрутизация по умолчанию всегда назначает центр обработки данных на основе общедоступного IP-адреса пограничного контроллера сеансов (SBC) и всегда выбирает путь, ближайший к центру обработки данных SBC.

Однако в некоторых случаях путь к носителю по умолчанию может быть неоптимальным. Например, общедоступный IP-адрес из диапазона США может быть назначен SBC, расположенному в Европе. 

С помощью параметра -MediaRelayRoutingLocationOverride с командлетами New-CsOnlinePSTNGateway и Set-CsOnlinePSTNGateway можно указать предпочтительный регион для трафика мультимедиа. Например, следующая команда указывает, что предпочтительный регион — Германия:

Set-CSOnlinePSTNGateway -Identity sbc1.contoso.com –MediaRelayRoutingLocationOverride DE 

Обратите внимание, что корпорация Майкрософт рекомендует задать этот параметр только в том случае, если в журналах вызовов явно указано, что назначение центра обработки данных по умолчанию для пути мультимедиа не использует путь, ближайший к центру обработки данных SBC.
