# <a name="step-3-assign-business-voice-licenses"></a>Шаг 3. Назначение лицензий на бизнес-голосовые номера

Каждому пользователю, который должен использовать Голосовая связь Microsoft 365 бизнес голосовой почты, должны быть назначены лицензии на бизнес-голосовые данные.

1. В центре [Microsoft 365 администрирования](https://admin.microsoft.com)перейдите на **страницу** Лицензии  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">на выставление</a> счета.
2. Выберите **Голосовая связь Microsoft 365 бизнес**.
3. На странице сведений о продукте выберите **Назначение лицензий**.
4. В области **Назначение лицензий пользователям** начните вводить имя, а затем выберите его в результатах, чтобы добавить его в список. Вы можете добавить до 20 пользователей одновременно.
5. После завершения нажмите **Назначить**, а затем — **Закрыть**.
