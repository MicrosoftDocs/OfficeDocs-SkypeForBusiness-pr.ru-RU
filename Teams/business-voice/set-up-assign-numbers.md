<span data-ttu-id="e7117-104">Чтобы пользователи могли Teams или принимать телефонные звонки на обычные телефонные линии или из них, необходимо назначить им номера телефонов.</span><span class="sxs-lookup"><span data-stu-id="e7117-104">Before users can use Teams to make or receive phone calls to or from regular phone lines, you need to assign phone numbers to them.</span></span> <span data-ttu-id="e7117-105">В Microsoft Teams при нажатии кнопки Звонки будет указан номер телефона, который вы **назначаете.**</span><span class="sxs-lookup"><span data-stu-id="e7117-105">In Microsoft Teams, the phone number that you assign is listed when a user clicks **Calls**.</span></span> <span data-ttu-id="e7117-106">Для каждого пользователя, для которых требуется номер телефона, сделайте следующее:</span><span class="sxs-lookup"><span data-stu-id="e7117-106">Do the following for each user that needs a phone number.</span></span>

> [!NOTE]
> <span data-ttu-id="e7117-108">Если вы не видите телефонных номеров, подождите.</span><span class="sxs-lookup"><span data-stu-id="e7117-108">If you don't see any phone numbers, please wait.</span></span> <span data-ttu-id="e7117-109">Для того чтобы новые телефонные номера стали доступны в новой Teams, может потребоваться несколько часов.</span><span class="sxs-lookup"><span data-stu-id="e7117-109">It may take several hours for your new phone numbers to become available in Teams.</span></span>

1. <span data-ttu-id="e7117-110">В области навигации слева выберите **Голосовая**  >  **Телефон номера**.</span><span class="sxs-lookup"><span data-stu-id="e7117-110">In the left navigation, click **Voice** > **Phone numbers**.</span></span>
2. <span data-ttu-id="e7117-111">На странице **Телефон номеров** выберите в списке ненаученный номер и нажмите кнопку **Изменить**.</span><span class="sxs-lookup"><span data-stu-id="e7117-111">On the **Phone numbers** page, select an unassigned number in the list, and then click **Edit**.</span></span>  
3. <span data-ttu-id="e7117-112">В области **правки** в **области** Назначено найдите пользователя по отображаемом имени или имени пользователя, а затем нажмите кнопку **Назначить**.</span><span class="sxs-lookup"><span data-stu-id="e7117-112">In the **Edit** pane, under **Assigned to**, search for the user by display name or user name, and then click **Assign**.</span></span>

5. <span data-ttu-id="e7117-114">**Убедитесь, что выбран пользователь электронной** почты с информацией о телефонном номере, чтобы он получил приветствие по электронной почте с новым номером телефона.</span><span class="sxs-lookup"><span data-stu-id="e7117-114">Make sure **Email user with telephone number information** is selected so the user receives a welcome email with their new phone number.</span></span>
