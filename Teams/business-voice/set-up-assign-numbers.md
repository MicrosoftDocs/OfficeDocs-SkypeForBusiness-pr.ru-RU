# <a name="step-5-assign-teams-phone-system-phone-numbers-to-your-users"></a>Шаг 5. Назначение телефонных номеров телефонной системы Teams пользователям

Прежде чем пользователи смогут использовать Teams для телефонных звонков на обычные телефонные линии или из них, необходимо назначить им номера телефонов. В клиентах Microsoft Teams номер телефона, который вы назначаете пользователю, отображается на панели набора номера, когда пользователь нажимает кнопку **"Звонки"**. Выполните следующие действия для каждого пользователя, которому требуется номер телефона.

> [!NOTE]
> Если номера телефонов не отображаются, подождите. Чтобы новые номера телефонов стали доступны в Teams, может потребоваться несколько часов.
