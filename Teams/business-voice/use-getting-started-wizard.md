# <a name="use-the-getting-started-wizard-to-set-up-business-voice"></a>Настройка корпоративной голосовой связи с помощью мастера начальной настройки

Мастер начальной настройки для корпоративной голосовой связи Microsoft 365 позволяет легко и быстро получать и совершать телефонные звонки в Microsoft Teams. Небольшой начинающей компании мастер поможет быстро начать работу с номерами телефонов, меню звонков, приветствиями и т. д. Крупным организациям с внедренным решением для телефонной связи мастер поможет настроить пилотную корпоративную голосовую связь, чтобы попробовать ее для нескольких пользователей перед применением для всех сотрудников. В любом случае вы можете начать использовать корпоративную голосовую связь сразу после завершения работы мастера!

Перед запуском мастера рекомендуется ознакомиться с этой статьей. По готовности вы можете открыть мастер, выбрав **Начать работу** на странице [Начать работу с корпоративной голосовой связью Microsoft 365](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/featureexplorer/apps/SmbVoice). Выполните вход с помощью учетной записи, использованной для создания подписки, или учетной записи глобального администратора.

> [!IMPORTANT]
> В настоящее время корпоративная голосовая связь доступна в Канаде и Соединенном Королевстве. Другие страны и регионы станут доступны в 2020 г.
>
> Microsoft Teams и корпоративная голосовая связь работает, только если почтовые ящики пользователей находятся в Microsoft 365.  Почтовые ящики, размещенные на локальном сервере Exchange, не поддерживаются.

## <a name="company-phone-number"></a>Номер телефона компании

<table>
    <tr>
        <td>Помимо настройки нового локального телефонного номера вы можете приобрести бесплатный номер или перенести существующий номер в Microsoft 365. Если вы хотите настроить бесплатный номер, потребуется приобрести тарифный план. Если вы хотите перенести один или несколько номеров в Microsoft 365, вы можете сделать это в [Центре администрирования Teams](https://admin.teams.microsoft.com) после завершения работы мастера.
        </td>
        <td><img src="https://docs.microsoft.com/MicrosoftTeams/media/voice-wizard-choose-number.png" width="400">
        </td>
    </tr>
</table>

> [!IMPORTANT]
> Если вы решили перенести один или несколько существующих номеров телефона в Microsoft 365, то в мастере по-прежнему будет отображаться временный номер телефона. Это нормально. Когда вы завершите действия в мастере и закончите процесс переноса, временный номер телефона будет заменен вашим номером телефона.

Вы можете выбрать доступное число лицензий корпоративной голосовой связи. Если вам требуются дополнительные лицензии, вы можете приобрести их после завершения работы мастера.

## <a name="incoming-call-greeting"></a>Приветствие для входящих звонков

<table>
    <tr>
        <td>Вы можете добавить звуковой файл (MP3 или WAV) размером до 5 мегабайт (МБ), чтобы использовать его в качестве приветствия. Вы также можете ввести свое приветствие, и Microsoft 365 воспользуется преобразованием текста в речь, чтобы прочитать его для звонящего. Приветствие — это то, что услышат впервые позвонившие по номеру телефона вашей компании. Для правильного произношения может потребоваться неправильно написать слова или использовать фонетическое написание.

## <a name="call-menu-and-forwarding"></a>Меню звонка и перенаправление

<table>
    <tr>
        <td>Вы можете перенаправлять все звонки конкретному пользователю или настроить меню звонка, в котором можно выбирать варианты. При создании меню звонка можно указать варианты, доступные для выбора звонящим с помощью нажатия цифры на клавиатуре телефона или произнесения варианта голосовой команды. Каждый вариант меню можно перенаправлять одному пользователю. <br>
        Вы можете добавить звуковой файл (MP3 или WAV) размером до 5 МБ, содержащий инструкции для звонящего, или ввести эти инструкции. Microsoft 365 использует преобразование текста в речь, чтобы прочитать инструкции звонящему. Вам может потребоваться написать слова фонетически, чтобы обеспечить правильное произношение.

> Мастер «Начало работы» поможет настроить простое меню звонков, чтобы быстро приступить к работе.
