# <a name="manage-tags-in-microsoft-teams"></a>Управление тегами в Microsoft Teams

> [!NOTE]
> Одна из функций, описанных в этой статье, **маркируется с помощью клавиш Shift**, еще не была отпущена. Она была объявлена, и вскоре она скоро появится.Если вы являетесь администратором, вы можете узнать, когда эта функция будет выпущена в центре сообщений (в [центре администрирования Microsoft 365](https://portal.office.com/adminportal/home)). Чтобы всегда оставаться на связи с возможностями предстоящих групп, ознакомьтесь с [планом Microsoft 365](https://www.microsoft.com/microsoft-365/roadmap?filters=&searchterms=microsoft%2Cteams).

## <a name="overview"></a>Обзор

Теги в Microsoft Teams позволяют пользователям быстро и легко подключаться к подмножеству участников команды. Вы можете создавать и назначать пользовательские теги для классификации пользователей на основе их атрибутов, таких как роль, проект, квалификация или расположение. Кроме того, теги могут автоматически назначаться людям на основе их расписания и сведений о смене в [приложении "Смена](https://support.microsoft.com/office/apps-and-services-cc1fba57-9900-4634-8306-2360a40c665b?#PickTab=Shifts) " (ожидается в ближайшее время). После того как тег добавлен к одному или нескольким членам группы, он может быть использован в @mentions любым пользователем в рамках команды в канале или для начала беседы только с теми людьми, которым назначен этот тег.

Как упоминалось ранее, в Teams существует два типа тегов.

- **Пользовательские теги**: владельцы группы и участники группы (если она включена) может вручную создавать и назначать теги для пользователей. Например, тег "дизайнер" или "radiologist" будет достигать эти наборы людей в команде без ввода их имен.
- **Добавление тегов на смену** (вскоре): с помощью этой функции люди автоматически назначают теги, соответствующие своему расписанию и названию группы Shift в [приложении "Смена](https://support.microsoft.com/office/get-started-in-shifts-5f3e30d8-1821-4904-be26-c3cd25a497d6#bkmk_openshiftsappdesktop) " в Teams. Например, тег "EngineerOnCall" можно использовать для всех инженеров, которые запланированы в сменах, чтобы работать в момент, когда тег используется в чате или канале. Благодаря разметке с помощью тегов в команде guesswork знать название сотрудника, когда пользователи должны быстро пересылать информацию. Расстановка тегов по смене также может быть вызвана крупными системами управления ресурсами, такими как JDA, Kronos и AMiON, интегрируя их с помощью смен в Teams. Чтобы узнать больше о том, как настроить эту функцию, ознакомьтесь со статьей [Настройка тегирования с помощью клавиш Shift](#set-up-tagging-by-shift-coming-soon).

> [!NOTE]
> В закрытых каналах Теги пока не поддерживаются. Теги пока не доступны в облаке (GCC), высоком или подразделении для защиты от США.

## <a name="how-tags-work"></a>Принцип работы тегов

Тег может быть добавлен вручную или автоматически назначен пользователю в определенной команде. Затем его можно использовать в @mentions в строке " **Кому** " чата или в записи на любом стандартном канале команды. Вот несколько примеров использования тегов в teams:

- Менеджер магазина отправляет извещение в канал, чтобы уведомить всех кассиров.
- Администратор больницы отправляет сообщение всем radiologists в канале.
- Менеджер по маркетингу начинает групповой чат со всеми дизайнерами.
- Медсестры отправляет сообщение всем вызываемым cardiologists. (ожидается в ближайшее время)
- Системный инженер отправляет объявление в канал, чтобы уведомить всех инженеров по смене полей. (ожидается в ближайшее время)

Если во время беседы в канале @mentioned тег, участники группы, связанные с этим тегом, будут получать уведомления, как и любые другие @mention.

## <a name="manage-custom-tags-for-your-organization"></a>Управление настраиваемыми тегами в Организации

Администраторы могут управлять использованием тегов в рамках Организации в центре администрирования Microsoft Teams.

![Снимок экрана: параметры разметки в центре администрирования Microsoft Teams](media/manage-tags-admin-settings.png)

У группы может быть до 100 тегов, к которым можно назначить до 100 членов группы, а для одного пользователя можно назначить до 25 тегов. 

### <a name="set-who-can-add-custom-tags"></a>Назначение пользователей, которые могут добавлять пользовательские теги

По умолчанию владельцы групп могут добавлять пользовательские теги. Вы можете изменить этот параметр, чтобы разрешить владельцам групп и участникам групп создавать, редактировать и изменять теги, а также отключать теги для Организации.

1. В левой области навигации центра администрирования Microsoft Teams щелкните **Org-wide settings**  >  **Параметры Teams**для организационной настройки.
2. В разделе " **Разметка**" рядом с **тегами**выберите один из следующих вариантов:

    - **Владельцы и участники групп**: разрешение владельцам групп и членам управлять тегами.
    - **Владельцы групп**: разрешение владельцам групп управлять тегами.
    - **Disabled**(отключить Теги).

### <a name="configure-custom-tags-settings"></a>Настройка параметров настраиваемых тегов

Вы можете настроить следующие параметры тегов, чтобы управлять использованием настраиваемых тегов в Организации.

1. В левой области навигации центра администрирования Microsoft Teams щелкните **Org-wide settings**  >  **Параметры Teams**для организационной настройки.
2. В разделе **Разметка**выберите следующие значения в зависимости от потребностей Организации.

    - **Разрешить владельцам групп переопределять, кто может управлять тегами**: при включении этого параметра владельцы групп могут задавать, могут ли участники групп создавать и управлять тегами в группе, а значения **тегов** — по умолчанию для каждой команды. Если вы отключите этот параметр, **теги управляются с помощью** параметра "Изменить" для каждой команды.
    - **Предлагаемые Теги по умолчанию**: Используйте этот параметр, чтобы добавить набор тегов по умолчанию. Вы можете добавить до 25 тегов, и каждый тег может содержать не более 25 символов. Владельцы и участники групп (если эта функция включена) может использовать эти предложения, добавлять в них или создавать новые теги.
    - **Разрешить создание настраиваемых тегов**: Включите этот параметр, чтобы разрешить пользователям добавлять теги, отличные от предложенных тегов по умолчанию, которые вы хотите использовать. Если этот параметр отключен, пользователи могут использовать только предлагаемые теги по умолчанию. Если вы отключите этот параметр, убедитесь, что вы добавляете один или несколько тегов по умолчанию.

## <a name="manage-custom-tags-settings-for-a-team"></a>Управление настройками настраиваемых тегов для команды

Если вы включили параметр **разрешить владельцам групп настройку тегов** в центре администрирования Microsoft Teams, владельцы групп смогут определять, могут ли участники добавлять теги на уровне группы. Для этого на вкладке **Параметры** группы перейдите к разделу **теги**и выберите, кто может добавлять теги.

![Снимок экрана: Настройка тегов на уровне группы](media/manage-tags-team-settings.png)

## <a name="use-tags"></a>Использование тегов

Вот как можно добавить пользовательские теги и настроить теги с помощью Shift (если вы используете приложение "Смена" в Teams). Чтобы узнать больше, ознакомьтесь [с помощью тегов в Teams](https://support.office.com/article/using-tags-in-teams-667bd56f-32b8-4118-9a0b-56807c96d91e).

### <a name="create-and-assign-custom-tags"></a>Создание и назначение настраиваемых тегов

Для создания и назначения настраиваемых тегов выберите **команды** в левой части приложения, а затем найдите нужную команду в списке. Выберите **̇ ̇ ̇ Дополнительные параметры**, а затем выберите **Управление тегами**. Здесь вы можете создавать теги и назначать их участникам группы.

![Снимок экрана: применение тегов в клиенте Teams ](media/manage-tags-teams.png)

Чтобы удалить тег, выберите **̇ ̇ ̇ Дополнительные параметры** рядом с тегом и нажмите кнопку **Удалить тег**.

### <a name="set-up-tagging-by-shift-coming-soon"></a>Настройка тегов с помощью Shift (ожидается в ближайшее время)

1. В Teams перейдите в [приложение "Смена](https://support.microsoft.com/office/get-started-in-shifts-5f3e30d8-1821-4904-be26-c3cd25a497d6#bkmk_openshiftsappdesktop)".
2. Создавайте [группы Shift](https://support.microsoft.com/office/fill-out-a-schedule-in-shifts-2d58df9b-1c6c-4c84-b0c3-835de7ad13ea#bkmk_organizeshiftsbygroup) и назовите их после атрибута, например роли. Например, EngineerOnCall. Имя группы Shift будет именем тега.
3. Чтобы [заполнить расписание](https://support.microsoft.com/office/fill-out-a-schedule-in-shifts-2d58df9b-1c6c-4c84-b0c3-835de7ad13ea) , назначайте смену участникам Teams. Когда вы закончите работу, в правом верхнем углу приложения Сменаок выберите команду **поделиться с командой**.
4. Подождите 15 минут, пока запланированные смены заполнят службу тегов.
5. Используйте тег в любом месте, где вы используете Теги Teams.

Разметка с помощью клавиши Shift позволяет пользователям достигать пользователей в режиме реального времени. Уведомления отправляются только тем пользователям, которые находятся на смене в момент, когда тег начинается с начала чата или в канале сообщения.
