# <a name="microsoft-365-phone-system--virtual-user-license"></a>Телефонная система Microsoft 365 — лицензия виртуального пользователя

Организации с лицензированными пользователями телефонной системы могут назначить учетным записям ресурсов бесплатную телефонную систему Microsoft 365 — лицензию виртуального пользователя или платную телефонную систему. План звонков не требуется. Для всех автоотчетных услуг или очередей вызовов требуется связанная учетная запись ресурса. Для использования телефонного номера для учетных записей ресурсов необходима бесплатная телефонная система Microsoft 365 — лицензия виртуального пользователя или лицензия пользователя платной телефонной системы.

> [!TIP]
> Для учетных записей ресурсов, которые будут использоваться со вложенными автоотетарями или очередями звонков, не назначены номера телефонов, лицензия не требуется. Для справки см. следующую схему: 

![Лицензии виртуальных пользователей](../media/resource-account.png)

## <a name="virtual-user-license-allocation"></a>Выделение лицензий виртуальному пользователю

Вашей организации выделена телефонная система Microsoft 365 — лицензии виртуальных пользователей в зависимости от ее общего размера. У любой организации, у которых есть хотя бы одна лицензия, включая телефонную систему или телефонную систему, есть 25 лицензий виртуальных пользователей бесплатно. При добавлении в организации 10 лицензий на телефонную систему становится доступной еще одна телефонная система Microsoft 365 — виртуальная лицензия пользователя.

> [!NOTE]
> Телефонная система — это лицензия на надстройки для Microsoft 365 и Office 365 E1 и E3. Телефонная система также входит в состав лицензий на Microsoft 365 E5, Office 365 E5 и Microsoft 365 Business Voice.

Если ваша организация использует доступную бесплатную телефонную систему Microsoft 365 — лицензии виртуальных пользователей при создании автоотетаря или узлов очереди звонков, вы по-прежнему можете использовать лицензии платной телефонной системы с учетной записью ресурса. В большинстве организаций достаточно лицензий виртуальных пользователей на основе плана масштабирования. 

### <a name="license-allocation-example"></a>Пример выделения лицензии

Компания Contoso, Inc. приобрела 600 лицензий, включаемых в телефонную систему (по одной для каждого сотрудника). Contoso первоначально получает 25 плюс 60 телефонной системы Microsoft 365 — лицензии виртуальных пользователей (всего 85). В их организации 90 очередей звонков и автоотетарей с номерами телефонов. Им необходимо назначить все лицензии на телефонную систему Microsoft 365 — лицензии виртуальных пользователей и получить пять лицензий на телефонную систему с обычной стоимостью.

Contoso рассмотрит возможность переработания автозавода и системы очередей вызовов. Если используется меньше номеров телефонов и больше вложенных узлов, которые не требуются, внедрение упрощает внедрение и снижает затраты.

## <a name="how-to-buy-microsoft-365-phone-system--virtual-user-licenses"></a>Как приобрести телефонную систему Microsoft 365 — лицензии виртуальных пользователей

1. Войдите в Центр администрирования Microsoft 365.
2. Перейти к   >  **надстройкам "Приобретение служб**  >  **вы выставления счета"**
3. Прокрутите страницу вниз, чтобы найти **лицензию на телефонную систему Microsoft 365 — виртуального** пользователя. Выберите **"Купить сейчас".**

> [!NOTE]
> Помните, что вам необходимо  **приобрести** лицензию, даже если ее стоимость не равно нулю.

## <a name="change-an-existing-resource-account-to-use-a-microsoft-365-phone-system--virtual-user-license"></a>Изменение существующей учетной записи ресурса для использования телефонной системы Microsoft 365 — лицензия виртуального пользователя

Если вы решите перейти с лицензии на телефонную систему в учетной записи ресурса на лицензию на телефонную систему Microsoft 365 — лицензия виртуального пользователя:

1. Получите новую телефонную систему Microsoft 365 — лицензию виртуального пользователя.
2. Чтобы переместить пользователей на другую подписку, [](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users#move-users-to-a-different-subscription)следуйте связанным шагам в Центре администрирования Microsoft 365.

> [!WARNING]
> Всегда удалять полную лицензию на телефонную систему и назначать лицензию На Телефонную систему Microsoft 365 — виртуального пользователя в одном действии с лицензией. Если удалить старую лицензию, сохранить изменения учетной записи, добавить новую лицензию, а затем снова сохранить параметры учетной записи, учетная запись ресурса может перестать работать должным образом. В этом случае мы рекомендуем создать новую учетную запись ресурса для телефонной системы Microsoft 365 — лицензию виртуального пользователя и удалить неотвеяжаемую учетную запись ресурса.
