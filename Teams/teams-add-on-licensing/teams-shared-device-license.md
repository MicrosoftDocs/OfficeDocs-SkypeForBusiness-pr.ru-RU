# <a name="microsoft-teams-shared-devices-licensing"></a>Лицензирование общих устройств Microsoft Teams

Microsoft Teams поддерживает гибридные рабочие сценарии и позволяет пользователям работать в гибких пространствах. Лицензия на надстройку **"Общие устройства Microsoft Teams"** позволяет офисам назначать устройства в качестве общих устройств, включая телефоны общего пользования, дисплеи Teams для горячих столов и панели Teams для конференц-залов.

Сведения о других лицензиях на надстройки Teams см. в разделе [Лицензии на надстройки Microsoft Teams](/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing).

Лицензия **на общие устройства Microsoft Teams** поддерживает:

- Microsoft Teams
- Телефонная система Microsoft Teams &sup1;
- Аудиоконференции &sup2;
- Панели Teams
- Дисплеи Teams
- Microsoft Intune
- Azure Active Directory Premium план 1
- Exchange Online план 2 &sup3;

&sup1; Ранее называлась *телефонной системой*. </br>
&sup2; Телефоны общего пользования могут присоединяться к аудиоконферендам с помощью номера телефонного подключения, предоставленного организатором собрания. </br>
&sup3; Только облачные возможности голосовой почты.

## <a name="teams-shared-devices-license-availability"></a>Доступность лицензий на общие устройства Teams

Лицензия **на общие устройства Teams** доступна для приобретения по всему миру для коммерческих клиентов, клиентов государственного сектора, GCC, GCC High и DoD.

Чтобы приобрести лицензию **на общие устройства Teams** , вы можете выбрать предпочтительный канал приобретения: EA, EAS, EES, CSP, Web Direct, NCE — клиент и NCE — партнер.

## <a name="acquiring-teams-shared-devices-licenses"></a>Приобретение лицензий на общие устройства Teams

Лицензию **на общие устройства Teams** можно приобрести в [Центре администрирования Майкрософт](https://go.microsoft.com/fwlink/p/?linkid=2024339).

1. В [Центр администрирования Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=2024339) перейдите к **разделу Выставление счетов** > **.**

1. Найдите раздел **Приобретение от Майкрософт** и выберите **Просмотреть продукты**.

1. Выберите фильтр **Совместная работа и обмен данными** .

1. В списке продуктов найдите **общие устройства Microsoft Teams** и выберите **Сведения**.

1. Введите необходимое количество лицензий и частоту выставления счетов.

1. Выберите **Купить**.
