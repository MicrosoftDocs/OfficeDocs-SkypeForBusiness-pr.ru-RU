# <a name="messaging-policies-licensing"></a>Лицензирование политик обмена сообщениями

Клиенты по всему миру слышат, как с его стороны были созданы новые проблемы при работе с coVID-19 (Coronavirus) в ходе ежедневной работы. Возможность напрямую и быстро связываться с другими командами при быстро меняющихся условиях важна для обеспечения непрерывности работы во время реагирования в условиях реагирования в условиях, когда многие из нас возвращаются на рабочее место.

Уведомления о приоритетах для Microsoft Teams [](https://support.microsoft.com/article/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462) были объявлены в июне 2019 г., и пользователи могут постоянно отправлять уведомления, которые повторяются каждые две минуты в течение 20 минут для всех сообщений, помеченных как срочные. Уведомления о приоритетах были доступны в качестве рекламных акций для всех клиентов Teams до второй половины 2020 г.

Мы немедленно завершили эту рекламную кампанию и сделали уведомления о приоритетах доступными для Teams клиентов в качестве базовой возможности. Теперь пользователи смогут отправлять и получать неограниченное число срочных сообщений с уведомлением о приоритете. [Подробнее об управлении уведомлениями о приоритетах.](../messaging-policies-in-teams.md)
