# <a name="messaging-policies-licensing"></a>Лицензирование политик сообщений

Начиная с 1 января 2020, лицензированные пользователи смогут отправлять срочные сообщения с уведомлениями о приоритетах в соответствии с условиями их подписок. ИТ администраторам будет предоставлено сообщение об использовании приоритетных уведомлений для поддержки соответствующего лицензирования пользователей. Пользователи смогут получать неограниченные срочные сообщения с уведомлениями о приоритетах, независимо от лицензии.
