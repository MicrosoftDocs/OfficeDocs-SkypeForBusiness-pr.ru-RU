# <a name="messaging-policies-licensing"></a>Лицензирование политик обмена сообщениями

Клиенты по всему миру слышат, как совместим с COVID-19 (Coronavirus) проблемы, которые они создают, чтобы справиться с кардинальными сдвигами в ежедневной работе. Возможность напрямую и быстро связываться с другими командами при быстро меняющихся условиях важна для обеспечения непрерывности работы во время реагирования в условиях реагирования в условиях, когда многие из нас возвращаются на рабочее место.

Изначально было объявлено в июне 2019 г. уведомления о приоритете для Microsoft Teams дает пользователям возможность постоянно отправлять уведомления, которые повторяются каждые две минуты в течение 20 минут для всех сообщений, помеченных как срочные. [](https://support.microsoft.com/article/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462) Уведомления о приоритетах были доступны в качестве рекламных акций для Teams клиентов до второй половины 2020 г.

Мы немедленно завершили эту рекламную акцию и сделали уведомления о приоритетах доступными Teams клиентами в качестве базовой возможности. Теперь пользователи смогут отправлять и получать неограниченное число срочных сообщений с уведомлением о приоритете. [Подробнее об управлении уведомлениями о приоритетах.](../messaging-policies-in-teams.md)
