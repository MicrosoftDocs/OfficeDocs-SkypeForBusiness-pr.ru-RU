# <a name="what-are-cloud-auto-attendants"></a>Что представляют собой облачные автосекретари?

телефонная система предоставляет автоответчиков, с помощью которых внешние и внутренние звоняющие могут перемещаться по системе меню, чтобы находить, размещать и переводить звонки пользователям или отделам организации.
  
Автозаводом чаще всего является узел в системе, в результате на который вызывающего звоняющего вы услышите несколько голосовых подсказок или звуковых файлов, а не оператора. Когда пользователи звонят на номер, связанный с автоотправлением, они могут перенаправить звонок пользователю или найти пользователя в вашей организации, а затем подключиться к этому пользователю. Они могут выразить свой выбор и работать с системой меню с помощью клавиатурной панели телефона (DTMF) или распознавания речи. Они также могут перенаправлять звонок другому автоотправлению или в очередь зовов.
  
Чтобы настроить автозавод для телефонная система, перейдите в статью Настройка автозавода в [облаке.](create-a-phone-system-auto-attendant.md)
  
Облачный автозаводщик имеет следующие функции:
  
- Воспроизведение корпоративных или информационных приветствий.
- Поддержка настраиваемых корпоративных меню. Вы можете настроить эти меню, чтобы они имели несколько уровней.
- Оно обеспечивает поиск по каталогу, который позволяет вызываемой звонящем искать имя в каталоге организации.
- Это позволяет человеку, который звонит в вашу организацию, связаться с сообщением или оставить его.
- Она поддерживает несколько языков для подсказок, функций распознавания текста и речи.
- В нем можно указать праздники и часы работы.
- Она поддерживает перенастройку звонка оператору, другим пользователям, очередям вызовов и автоотещающим.
- Она поддерживает общую голосовую почту, чтобы вызыватели покидали сообщение для организации.

> [!NOTE]
> Эта статья относится как к Microsoft Teams, так и Skype для бизнеса Online.

Это позволяет получить доступ к автозапуску с нескольких номеров телефонов. Чаще всего для учетной записи ресурса используется бесплатная лицензия телефонная система виртуального пользователя. ## <a name="feature-overview"></a>Обзор функций

### <a name="searching-for-users"></a>Поиск пользователей

Набор номера по имени — это функция автозавода, которая также называется поиском по каталогу. Он позволяет вызывающим автозаправщикам использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести полное или частичное имя для поиска в каталоге компании, найти человека, а затем перенаправить звонок на него. Пользователям, с помощью которых вы хотите позвонить по имени, не требуется номер телефона или назначены планы звонков, но они должны иметь лицензию телефонная система, если они являются сетевыми пользователями, или Корпоративная голосовая связь включена для Skype для бизнеса Server **пользователей.** С помощью звонка по имени можно даже находить и переводить звонки Microsoft Teams пользователям, которые находятся в разных странах или регионах для международных организаций. С учетом необходимых условий вы явным образом включаете набор номера по имени в автоотехе.

Набор номера по расширению — это функция автозавода, которая также входит в поиск по каталогу. Он позволяет вызывающим автозаправщикам использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести расширение телефона пользователя, с которым он пытается связаться, а затем перенаправить звонок на него. Пользователям, с помощью которых вы хотите звонить по расширению, не требуется номер телефона или назначены планы звонков, но они должны иметь лицензию телефонная система, если они являются сетевыми пользователями, или Корпоративная голосовая связь для Skype для бизнеса Server **пользователей.** Кроме того, для пользователей должна быть настроена соответствующая телефонная плана. С помощью телефонного звонка по расширению вы даже сможете находить и переводить звонки Microsoft Teams пользователям, которые находятся в разных странах или регионах для международных организаций. С учетом необходимых условий вы явным образом включаете набор номеров по расширению в автоотехе.

#### <a name="maximum-directory-size"></a>Максимальный размер справочника

Количество пользователей Active Directory, набрав номера по имени и по расширению, не ограничивается. Вызываемая может вводить частичные или полные имена (имя + Фамилия, а также Фамилия + Имя), но требуется полный номер расширения. Максимальный размер списка имен, поддерживаемый одним автозаводом при использовании распознавания речи, составляет 80 000 пользователей.
  
|Тип ввода|Формат поиска|Максимальное число пользователей в организации|
|:-----|:-----|:-----|
|DTMF (ввод с клавиатурной панели) |Частичный  <br/> Имя + фамилия  <br/> Фамилия + Имя |Без ограничений  |
|Распознавание речи (голосовой ввод) |Firstname  <br/> Lastname  <br/> Имя + фамилия  <br/> Фамилия + Имя  | 80 000 пользователей |

> [!NOTE]
> Если вы используете функцию набора номера по имени с распознаванием речи, но в каталоге Active Directory организации больше 80 000 пользователей и вы еще не ограничили область набора по имени с помощью функции "Набор по имени", функция "Набрать по имени" по-прежнему будет работать для вызывающих пользователей с помощью клавиатурной панели телефона, а голосовой ввод будет доступен для всех других сценариев. С помощью функции "Область набора номера" можно сузить список имен, которые можно найти, изменив область набора номера по имени для конкретного автозавода.
  
### <a name="dial-by-name---keypad-dtmf-entry"></a>Звонок по имени  ввод с клавиатурной панели телефона (DTMF)

С помощью звонка по имени можно связаться с пользователями, указав полное или частичное имя человека, с которого он пытается связаться. Существует несколько форматов, которые можно использовать при в введенном имени.

При поиске в каталоге организации с помощью клавиши "0" (ноль) можно указать пробел между именем и фамилией и именем. При вводе имени будет предложено завершить ввод клавиатуры с помощью клавиши #. Например, "После ввода имени человека, с которого вы пытаетесь связаться, нажмите кнопку #". Если найдено несколько имен, вызываемого вызываемого человека выдается список имен для выбора.
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|Формат имени|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный  |Иван0Макаренко# |Иван Макаренко |
|Фамилия + Имя |Полный |Макаренко0Иван#  |Иван Макаренко |
|Firstname  |Полный   |Иван#   |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Lastname |Полный |Макаренко#  |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко" |
|Имя или фамилия |Частичный |Ма# |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко" |
|Имя + Фамилия |Частичный |Amos0Mar # |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия + Имя |Частичный |Ма0Ив# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |

Существует несколько специальных символов, которые используются при поиске людей, использующих клавиатурную панель телефона. Например, человеку будет предложено использовать клавишу с фунтом (#), а для пробела между именами используется ноль (0). При нажатии клавиши звездочка (*) будет повторяться список совпадающих имен с человеком.
  
|Специальный знак на клавиатуре телефона|Описание|
|:-----|:-----|
|#   |Завершение ввода имени. |
|0   |Пробел между именами. |
|*    |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска. |

#### <a name="dial-by-name---name-recognition-with-speech"></a>Звонок по имени  распознавание имен при голосовом вводе

Другие люди могут искать других людей в своей организации с помощью голосовой связи (распознавания речи). Они также могут связаться с любыми из пользователей Active Directory, назовя имя человека, которого он пытается найти. С помощью голосового ввода можно распознавать имена в различных форматах, включая Имя, Фамилия, Имя + Фамилия или Фамилия + Имя.
  
Вы можете включить распознавание речи для автозавода, но запись на клавиатурной панели телефона (DTMF) не отключена. Телефон клавиатуры можно использовать в любое время, даже если автозаводщик включил распознавание речи.
  
Как и при вводе на клавиатурной панели телефона, если найдено несколько имен, вызываемая звонок будет слышать список имен для выбора.
  
Вызыватели могут называть имена в следующих форматах:
  
|Имя с речью|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный |Иван Макаренко |Иван Макаренко |
|Фамилия + Имя |Полный  |Макаренко Иван |Иван Макаренко |
|Firstname |Полный |Иван |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров" |
|Lastname |Полный |Макаренко |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко" |

> [!NOTE]
> Имя нового пользователя может быть указано в каталоге для набора номера по имени с распознаванием речи из-за задержки репликации Active Directory.

### <a name="the-operator-option"></a>Параметр "Оператор"

При желании автозаводщика можно настроить, чтобы дать вызываемой функции возможность поговорить с оператором.
  
Клавиша 0 и голосовая команда "Оператор" по умолчанию перенаправят звонок на назначенный оператор. Это касается всех языков, поддерживаемых для распознавания речи. Вы также можете использовать **параметры меню Установить** для настройки пользовательского значения оператора.
  
Оператором может быть:
  
- Пользователь Microsoft Teams или Skype для бизнеса Server, который Корпоративная голосовая связь включен.
- Другой настроенный в организации автосекретарь.
- Любая существующая очередь вызовов, настроенная в вашей организации. Дополнительные информацию об очередях вызовов см. в [этой теме.](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue)

### <a name="business-hours-and-call-handling"></a>Рабочие часы и обработка звонков

Для каждого автозавода можно настроить часы работы. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. В течение дня можно настроить часы работы с перерывом, а все часы, которые не считаются неавтными, считаются неавтными. Вы можете настроить различные параметры обработки входящих параметров обработки параметров и приветствия (необязательно) для работы и после часа.
  
У каждого автозаполнения есть несколько параметров обработки параметров.
  
- Звонок может отключаться после того, как воспроизводит приветствие.
- Также вы можете:
  - Перенаправляйте звонок пользователю Microsoft Teams,  у которого есть телефонная система лицензия, Корпоративная голосовая связь или которому назначены планы звонков. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.

  - Перенаправить звонок в очередь зовов. Дополнительные информацию об очередях вызовов см. в [этой теме.](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue)

  - Перенаправлять звонок другому автоотправлению.

Эти параметры выражаются автозапускам во время вызова меню. Например: "Нажмите 1 для продажи, нажмите 2 для служб. Чтобы поговорить с оператором, нажмите 0 в любое время".

### <a name="set-menu-options"></a>Настройка меню "Параметры"

Автоответы в облаке позволяют создавать запросы меню ("Нажмите 1 для продажи, нажмите 2 для служб") и настроить параметры меню для маршрутизируются исходя из выбора пользователем. Параметры меню для автоответчика предоставляют организации ряд вариантов, которые ускоряют работу со звонителями по их назначениям без использования оператора человека для обработки входящих звонков. Для создания подсказок меню можно использовать функцию "Текст в речь" (системные запросы) или путем отправки записанного звукового файла. Распознавание речи принимает голосовые команды для свободной навигации с помощью голосовых команд, но для навигации по меню также можно использовать клавиатурную панель телефона.
  
Клавиши 0–9 могут быть назначены автоотводитесу в центре Skype для бизнеса администрирования. Можно создавать различные наборы параметров меню для работы и после, а также включить или отключить функцию набора номера по имени в **параметрах меню.** Для перенажатия звонков можно использовать:
  
- Оператор (по умолчанию назначается клавиша 0). Однако вы можете перена назначенную клавишу или удалить ее из меню.
- Очередь вызовов.
- Другой автосекретарь. Многоуровневые меню можно настроить, указав параметр меню в одном автозапуске другому автозаводщику с собственным набором параметров меню, который называется "вложенным" автозаводом. 
- Пользователь Microsoft Teams, у которого **есть** телефонная система лицензия, Корпоративная голосовая связь или которому назначены планы звонков. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.
  
Если включена распознавание речи, имя каждого параметра меню становится ключевым словом для распознавания речи. Например, можно сказать "Один", чтобы выбрать параметр меню, который назначен клавише 1, или "Продажи", чтобы выбрать тот же параметр меню с именем "Продажи".
  
Чтобы настроить автозапуск и параметры меню, перейдите в статью Настройка автозапуска [в облаке.](create-a-phone-system-auto-attendant.md)
  
### <a name="assigning-phone-numbers-for-an-auto-attendant"></a>Назначение номеров телефонов для автозавода

Вы можете назначить номер службы Майкрософт, прямой маршрутизводной или гибридный номер учетной записи связанного ресурса (или нескольким учетным записям ресурсов, если требуется несколько номеров). Дополнительные [сведения см. в](direct-routing-plan.md) этой ссылке.

Чтобы назначить номер службы, необходимо получить или порт существующих платных или бесплатных номеров служб. После получения платных или бесплатных номеров телефонов служб они будут Skype для бизнеса голосовой Телефон центре  >    >  **администрирования**. **Тип номера** указан как **Услуга — бесплатный**. Чтобы получить номера служб, см. получение номеров телефонов служб для Skype для бизнеса и [Microsoft Teams](./getting-service-phone-numbers.md) или, если вы хотите перенести и существующий номер службы, см. перенос номеров телефонов в [Teams.](phone-number-calling-plans/transfer-phone-numbers-to-teams.md)
  
> [!NOTE]
> Если вы находитесь за пределами США, вы не можете использовать центр администрирования Microsoft Teams для получения номеров служб. Чтобы [узнать, как это сделать,](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) перейдите в управление номерами телефонов для организации.
