# <a name="what-are-phone-system-auto-attendants"></a>Что такое автосекретари телефонной системы?

Телефонной системой в Office 365 предоставляет автосекретарей, которые могут быть использованы для внешнего и внутреннего вызывающих объектов перемещения по системе меню Поиск и поместить или переводить звонки компании пользователей или отделов организации.
  
Автосекретарь представляет собой ряд голосовых запросов или звуковые файлы, которые абоненты прослушивают вместо человеческого оператор, когда он совершает вызов организации. При людей вызвать номер, связанный с автосекретарем, их вариантов можно перенаправить звонок на пользователя или найдите кто-то в вашей организации и затем подключитесь к этому пользователю. Они могут express их варианты и взаимодействовать с системой меню с помощью клавиатуры телефона (DTMF) или распознавания речи.
  
Чтобы настроить автосекретарь для телефонной системой в Office 365, перейдите [настроить автосекретарь телефонной системой](create-a-phone-system-auto-attendant.md).
  
Автосекретарь телефонной системой имеет следующие особенности:
  
- Воспроизведение корпоративных или информационных приветствий.
- Поддержка настраиваемых корпоративных меню. Вы можете настроить эти меню, чтобы они имели несколько уровней.
- Она предоставляет поиск в каталоге, который позволяет пользователям вызывать в каталоге организации с именем.
- Включение человека, который вызывает достичь или оставлять сообщение для лица в вашей организации.
- Поддержка нескольких языков для приглашения, распознавания речи и преобразования текста в речь.
- Поддержка указание праздников и рабочие часы.
- Поддержка передача звонка на оператор, других пользователей, очереди вызовов и автосекретарей.

> [!NOTE]
> Эта статья относится к группам Майкрософт и Скайп для бизнеса в Интернет.

## <a name="getting-started"></a>Начало работы

Перед началом работы с функцией автосекретаря необходимо запомнить следующие моменты.
  
- Необходимо назначить учетную запись ресурса, который будет связан с вашей автосекретаря лицензию телефонной системой. Дополнительные сведения о лицензировании см. [: Лицензирование дополнительный компонент группами Майкрософт](teams-add-on-licensing/microsoft-teams-add-on-licensing.md).
    
    > [!TIP]
    > Можно настроить переадресацию звонков для оператора или пункт меню, который является Online пользователя с **Телефонной системой** лицензии, необходимо будет включить их для корпоративной голосовой связи или назначить вызов планы им. В разделе [Назначение групп Майкрософт лицензий](assign-teams-licenses.md). Кроме того, можно использовать Windows PowerShell. Например, выполните:`Set-CsUser -identity "Amos Marble" -EnterpriseVoiceEnabled $true`
  
- Для получения и использовать службы бесплатных номеров для вашей автосекретари, необходимо настроить связь кредитов. Чтобы сделать это, обратитесь к разделу [Каковы кредитов коммуникации?](what-are-communications-credits.md) и [настроить кредитов коммуникаций для вашей организации](set-up-communications-credits-for-your-organization.md).

    > [!IMPORTANT]
    > Номера телефонов пользователей (абонентов) нельзя назначить автосекретарям  можно использовать платные или бесплатные номера телефонов службы.
- Автосекретарь необходимо иметь учетную запись связанных ресурсов. Сведения о учетные записи ресурса в разделе [Управление учетными записями ресурсов в группах](manage-resource-accounts.md) .
- Полный auto attendant система обычно включает несколько автосекретарей и только может потребоваться один назначенный номер телефона для автоматической верхнего уровня или запись автосекретаря. Другие автосекретари или очереди вызовов из всей системы потребуется только номер телефона, если вы хотите предоставлять несколько точек входа в систему.
- Можно применить более одного номера телефона автосекретарю связывая более одной учетной записи ресурса автосекретарю.
  
## <a name="feature-overview"></a>Обзор возможностей

### <a name="dial-by-name"></a>Звонок по имени

Удаленный по имени — это функция автосекретарь, которая также называется поиск в каталоге. Включение пользователей, которые вызова автосекретаря введите полное или частичное имя для поиска каталога компании, выберите нужного получателя с помощью голосовой связи (распознавание речи) или их ответы телефона клавиатуре (DTMF), а затем перевести звонок на них. Пользователям, необходимо находится и связаться с вами с помощью вызова по имени **не требуется иметь номер телефона или вызов планы им назначены, но они должны иметь лицензию телефонной системой**. Удаленный по имени даже смогут найти и переводить звонки пользователям группами Майкрософт, которые размещены в разных странах или регионах для глобальной экономике организаций.

### <a name="maximum-directory-size"></a>Максимальный размер справочника

Нет ограничений на размер Active Directory, для которого поддерживается телефонным по имени при на клавиатуре телефона для поиска для ввода имен частичного или полного (FirstName LastName, а также LastName + FirstName). Тем не менее размер списка максимальное имя, что одного автосекретаря может поддерживать распознавание имя речи — 80 000 пользователей.
  
|Тип ввода|Формат поиска|Максимальное число пользователей в организации|
|:-----|:-----|:-----|
|DTMF (ввод с клавиатурной панели) |Частичный  <br/> Имя + фамилия  <br/> Фамилия + имя |Без ограничений  |
|Распознавание речи (голосовой ввод) |FirstName  <br/> Фамилия  <br/> Имя + фамилия  <br/> Фамилия + имя  | 80 000 пользователей |

> [!NOTE]
> При использовании удаленного по имени с речи распознавания, но Active Directory вашей организации размером более 80000 пользователей и еще не ограничивается область телефонным по имени с помощью функции область номера, телефонным по имени будет работать для вызывающих объектов с помощью клавиатуры телефона , и речевой ввод будут доступны для всех других сценариев. Можно использовать функцию телефонным области Чтобы сузить имена, которые доступны при изменении области телефонным по имени для отдельного автосекретаря.
  
### <a name="dial-by-name---keypad-dtmf-entry"></a>Звонок по имени  ввод с клавиатурной панели телефона (DTMF)

Вызов в людей можно использовать телефонным по имени для можно сделать доступными пользователям, указав либо полное или частичное имя контакта, которого удается связаться. Существуют различные форматы, которые можно использовать для ввода имени.

При поиске в каталоге организации, людей можно использовать "0" (ноль) ключ для указания пробела между имя первого и последнего или последнего имя и первый. Когда поступают имя они будет предложено завершать их запись клавиатуре клавиши #. Например «после ввода имени контакта, с которым вы пытаетесь достичь нажмите #.» Если существует несколько имен, которые находятся вызывающий абонент будет предоставлен список имен для выбора.
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|Формат имени|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный  |Иван0Макаренко# |Иван Макаренко |
|Фамилия + Имя |Полный |Макаренко0Иван#  |Иван Макаренко |
|FirstName  |Полный   |Иван#   |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия |Полный |Макаренко#  |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко" |
|Имя или фамилия |Частичный |Ма# |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко" |
|Имя + Фамилия |Частичный |Ма0Иван# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия + Имя |Частичный |Ма0Ив# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |

Существует несколько специальных символов, которые используются при поиске людей, с помощью клавиатуры телефона. Например пользователя будет предложено использовать ключ решетки (#), а нуль (0) ключ для пробела между имена. Нажатие клавиши звездочкой (*) будет повторите список совпадающих имен для пользователя.
  
|Специальный знак на клавиатуре телефона|Описание|
|:-----|:-----|
|#   |Завершение ввода имени. |
|0   |Пробел между именами. |
|*    |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска. |

### <a name="dial-by-name---name-recognition-with-speech"></a>Звонок по имени  распознавание имен при голосовом вводе

Для других пользователей в своей организации, с помощью своих голосовой связи (распознавание речи) можно поиска людей. Они также достигнут всем пользователям в организации в Active Directory с сообщением о том, имя контакта, с которым они пытаетесь найти. С помощью речевой ввод может распознать имена в различных форматах, включая FirstName, LastName, FirstName LastName, или LastName + FirstName.
  
При включении распознавания речи для автосекретаря запись клавиатуру телефона (DTMF) не будет отключаться, чтобы использовать обоих типов входные данные. Запись клавиатуру телефона не может быть отключена и может использоваться в любое время, даже если распознавания речи включен для автосекретаря.
  
Как и при вводе с клавиатурной панели телефона, если найдено несколько пользователей, выдается список имен для выбора.
  
Вызывающие абоненты могут произносить имена в следующих форматах.
  
|Назовите с использованием речи|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный |Иван Макаренко |Иван Макаренко |
|Фамилия + Имя |Полный  |Макаренко Иван |Иван Макаренко |
|FirstName |Полный |Иван |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров" |
|Фамилия |Полный |Макаренко |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко" |

> [!NOTE]
> Может потребоваться до 36 часов для нового пользователя к своим именем, из списка в каталоге абонентских групп с именем распознавания речи из-за задержки репликации Active Directory.

### <a name="using-the-operator-option"></a>Переключение на оператора

С помощью оператора для автосекретаря — это необязательный параметр, который предоставляет вызывающего абонента с параметром говорить человеческого оператор.
  
Клавиша 0 и голосовые команды «Оператор» прямой звонок на нужный оператор по умолчанию. Это обоснование для всех языков, поддерживаемых для распознавания речи. Можно также использовать **Параметры меню** для установки пользовательское значение для оператора.
  
Оператор можно установить.
  
- Пользователь группами Майкрософт или Скайп для бизнеса на пользователей локально, включена ли для корпоративной голосовой связи.
  
- Другой настроенный в организации автосекретарь.
- Любой существующий вызова очередь, настроенный в организации. Дополнительные сведения о очереди вызовов см [Создание очереди вызовов с телефонной системой](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue).

### <a name="business-hours-and-call-handling"></a>Рабочие часы и обработка звонков

Рабочие часы настраиваются отдельно для каждого автосекретаря. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. Можно задать рабочих часов с разрывов в момент в течение дня и все данные о количестве часов, которые не установлены, как рабочих часов, считаются нерабочее время. Можно задать различные параметры входящей обработки звонков и различных приветствие (которые являются необязательными), и как можно задать для рабочее и нерабочее время.
  
Каждый автосекретарь содержит параметры обработки звонков можно задать:
  
- Можно настроить отключение звонка сразу после воспроизведения приветствия.
- Также вы можете:
  - Перенаправить звонок на группами Майкрософт пользователь, имеющий лицензию **Телефонной системой** , включена ли для корпоративной голосовой связи или вызова планы назначаемых. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.

  
  - Перенаправление звонка в очередь вызова. Дополнительные сведения о очереди вызовов см [Создание очереди вызовов с телефонной системой](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue).

  - Перенаправить звонок на другого автосекретаря, были установлены.
- Создание параметров меню и воспроизведение подсказок для абонента. Пример: "Для связи с отделом продаж нажмите 1. Для связи с отделом обслуживания нажмите 2. Чтобы связаться с оператором, нажмите 0".

### <a name="menu-options"></a>Параметры меню

Автосекретари телефонной системы позволяют создать меню приглашения («Press 1 для продаж, нажмите клавишу 2 для служб») и настроить параметры меню для маршрутизации вызовов на основании пользователь выбирает. Настройка параметров меню для автосекретаря позволяет организации для обеспечения интерактивного руководства для получения человека по назначению быстрее, без использования человеческого оператор для обработки входящих звонков. Меню приглашения можно создавать с помощью преобразования текста в речь (сгенерированное системой запросы) или отправка звукового файла, который был записан. Распознавание речи использует голосовые команды для громкой навигации, но людей, пользующихся также можно использовать клавиатуру телефона для перехода меню.
  
Клавиши от 0 до 9 могут быть назначены **Параметры меню** в автосекретарь с помощью Скайп по центру администрирования бизнеса. Можно создать различные наборы параметров меню для рабочее и нерабочее и можно включать и отключать телефонным по имени в **Меню Параметры**. Ключи может быть сопоставлен переводить звонки:
  
- Оператор (по умолчанию назначается клавиша 0). Тем не менее он может быть повторно назначается другим ключом или удалена из меню.
- Очередь звонок.
- Другой автосекретарь. Многоуровневых меню можно настроить с помощью команды **Пункта меню** в один автосекретаря другого автосекретаря собственный набор параметров меню, который называется «вложенных» автосекретаря.
- Пользователь группами Майкрософт, имеющий с **Телефонной системой** лицензировать, включена ли для корпоративной голосовой связи или вызова планы назначаемых. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.
  
Имя каждого пункта меню становится ключевое слово распознавания речи включенный распознавания речи. Например абоненты могут сказать "Один", чтобы выбрать пункт меню, сопоставленное с ключом 1 или можно просто говорят «Продажи», чтобы выбрать тот же параметр меню под названием «Sales».
  
Чтобы настроить автосекретарь и параметры меню, перейдите [настроить автосекретарь телефонной системой](create-a-phone-system-auto-attendant.md).
  
### <a name="assigning-phone-numbers-for-an-auto-attendant"></a>Назначение телефонных номеров для автосекретаря

Можно назначить Microsoft вызова номер службы плана или прямой маршрутизации гибридного номер для вашей автосекретаря. Для получения дополнительных сведений см. [Планирование прямой маршрутизации](direct-routing-plan.md) .

Чтобы назначить номер службы, необходимо будет получить или перенос существующих международную или бесплатная служба номеров. После получения счета или бесплатная служба телефонных номеров, они будут отображаться в <!-- validate nav path --> **Скайп по центру администрирования Business** > **голосовой связи** > **номера телефонов**и **тип номера** в списке будет отображаться как **Служба — бесплатный номер**. Для получения номера службы видеть [Приступая к службе номера телефонов для Скайп для бизнеса и группами Майкрософт](/SkypeForBusiness/what-is-phone-system-in-office-365/getting-service-phone-numbers) или существующий номер службы и передача, см [передачи телефонных номеров в Office 365](transfer-phone-numbers-to-office-365.md).
  
> [!NOTE]
> Если вы находитесь за пределами США, не могут использовать Центр администрирования группами Майкрософт для получения номера службы. Вместо этого перейдите [Управление номера телефонов для вашей организации](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) для просмотра как это сделать.
