# <a name="what-are-cloud-auto-attendants"></a>Что представляют собой облачные автосекретари?

Телефонная система предоставляет автоответчиков, что позволяет внешним и внутренним звонятелям перемещаться по системе меню для поиска, перемещения и перевода звонков на пользователей или отделы в организации.
  
Чаще всего автострада — это узел в системе, который позволяет вызывающим звонить из голосовых подсказок или звуковых файлов, а не оператором. Когда кто-то звонит на номер, связанный с автозаправщиком, его варианты могут перенаправить звонок пользователю или найти пользователя в организации, а затем подключиться к этому пользователю. Они могут выразить свой выбор и взаимодействовать с системой меню с помощью клавиатурной панели телефона (DTMF) или распознавания речи. Кроме того, они могут перенаправить звонок на другого автозавода или в очередь зовов.
  
Чтобы настроить автозавод для телефонной системы, перейдите в статью "Настройка автозавода в [облаке".](create-a-phone-system-auto-attendant.md)
  
Автозаводщик в облаке имеет следующие возможности:
  
- Воспроизведение корпоративных или информационных приветствий.
- Поддержка настраиваемых корпоративных меню. Вы можете настроить эти меню, чтобы они имели несколько уровней.
- Оно обеспечивает поиск по каталогу, с помощью которого вызывавшие вызовы люди могут искать имя в каталоге организации.
- Это позволяет человеку, который звонит по телефону, связаться с сообщением или оставить сообщение для человека в вашей организации.
- Оно поддерживает запросы, функцию распознавания текста и речи на нескольких языках.
- В ней можно указать праздники и часы работы.
- Она поддерживает перенастройку звонка оператору, другим пользователям, очередям вызовов и автоотекатарям.
- Она поддерживает общую голосовую почту, чтобы вызыватели покидали сообщение для организации.

> [!NOTE]
> Эта статья относится как к Microsoft Teams, так и к Skype для бизнеса Online.

## <a name="getting-started"></a>Начало работы

Перед началом работы с функцией автосекретаря необходимо запомнить следующие моменты.

- Для использования учетной записи связанного ресурса требуется автоотчет. Подробные [сведения об учетных записях ресурсов](manage-resource-accounts.md) см. в сведениях об управлении учетными записями ресурсов в Teams. <!-- You can either use an existing resource account or create a new resource account as you set up your auto attendant. -->
- При назначении номера телефона автосектарю необходимо строго сказать, что вы назначаете его учетной записи ресурса, связанной с этим автосектарем. Это позволяет получить доступ к автозапуску с нескольких номеров телефонов. Чаще всего для учетной записи ресурса используется лицензия виртуального пользователя телефонной системы без затрат. Эта лицензия предоставляет возможности телефонной системы для телефонных номеров на уровне организации, а также позволяет создавать автоотправления и очереди звонков.

> [!NOTE]
> Прямой маршрутизинг номеров служб для автозагрузки и очередей вызовов поддерживается только для пользователей Microsoft Teams и агентов по звонкам.

   > [!TIP]
   > Чтобы перенаправить звонки оператору или пользователю Online  с лицензией на телефонную систему, необходимо включить для этого оператора Корпоративная голосовая связь назначить ему планы звонков. См. ["Назначение лицензий на надстройки Microsoft Teams".](teams-add-on-licensing/assign-teams-add-on-licenses.md) Вы также можете использовать Windows PowerShell. Например, запустите:  `Set-CsUser -identity "Amos Marble" -EnterpriseVoiceEnabled $true`
  
- Чтобы получить и использовать бесплатные номера служб для автослужбника, необходимо настроить кредиты на связь. Для этого см. ["Что такое кредиты](what-are-communications-credits.md) на связь?" и "Настройка кредитов на связь [для организации".](set-up-communications-credits-for-your-organization.md)

    > [!IMPORTANT]
    > Номера телефонов пользователей (абонентов) нельзя назначить автосекретарям  можно использовать платные или бесплатные номера телефонов службы.

- Полная система автозаполнеия обычно включает нескольких автозаполнеников.
- К автозаполневающим можно применить несколько номеров телефонов.
- Автоответы или очереди звонков уровня без входа в полную систему требуются только в том случае, если они будут звонить по ННОП.
  
## <a name="feature-overview"></a>Обзор функций

### <a name="searching-for-users"></a>Поиск пользователей

Функция "Набор по имени" — это функция автозавода, которая также называется поиском по каталогу. Она позволяет пользователям, которые звонят автоотправлению, использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести полное или частичное имя для поиска в каталоге компании, найти человека и перенаправить звонок. Пользователям, с которых вы хотите получить доступ с помощью звонка по имени, не требуется номер телефона или назначены планы звонков, но они должны иметь лицензию на телефонную систему, если пользователи в сети, или Корпоративная голосовая связь для пользователей Skype для бизнеса **Server.** Кроме того, с помощью звонка по имени пользователи Microsoft Teams, которые находятся в разных странах и регионах разных стран и регионов, смогут находить и переводить звонки. С учетом необходимых условий вы явным образом включаете в автозапросе набор по имени.

Набор по расширению — это функция автозавода, которая также входит в поиск по каталогу. Она позволяет пользователям, которые звонят автоотправлению, использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести расширение телефона пользователя, с которым он пытается связаться, а затем перенаправить звонок на него. Пользователям, с которых вы хотите получить доступ с помощью телефонного звонка по расширению, не требуется номер телефона или назначены планы звонков, но они должны иметь лицензию на телефонную систему, если они являются сетевыми пользователями или Корпоративная голосовая связь для пользователей Skype для бизнеса **Server.** Кроме того, для этого потребуется настроить для пользователей соответствующую телефонную плану. Кроме того, с помощью телефонного звонка по расширению можно находить и переводить звонки пользователям Microsoft Teams, которые находятся в разных странах или регионах для международных организаций. С учетом необходимых условий вы явным образом включаете "Звонить по расширению" для автозавода.

#### <a name="maximum-directory-size"></a>Максимальный размер справочника

Число пользователей Active Directory, которые могут звонить по имени и по расширению, не ограничивается. Вызываемая звонок может ввести полное или частичное имя (имя + фамилия, а также фамилия + имя), но нуждается в полном расширении. Максимальный размер списка имен, поддерживаемый одним автозаводом при использовании распознавания речи, составляет 80 000 пользователей.
  
|Тип ввода|Формат поиска|Максимальное число пользователей в организации|
|:-----|:-----|:-----|
|DTMF (ввод с клавиатурной панели) |Частичный  <br/> Имя + фамилия  <br/> Фамилия + Имя |Нет ограничений  |
|Распознавание речи (голосовой ввод) |FirstName  <br/> Фамилия  <br/> Имя + фамилия  <br/> Фамилия + Имя  | 80 000 пользователей |

> [!NOTE]
> Если вы используете функцию звонка по имени с распознаванием речи, но в Active Directory вашей организации больше 80 000 пользователей и вы еще не ограничили область действия функции звонка по имени с помощью функции "Набор по имени", функция "Звонок по имени" по-прежнему будет работать для вызывающих пользователей с помощью клавиатурной панели телефона, а голосовой ввод будет доступен во всех других сценариях. С помощью функции "Область набора" можно сузить список имен, которые можно получить, изменив область набора по имени для конкретного автозавода.
  
### <a name="dial-by-name---keypad-dtmf-entry"></a>Звонок по имени  ввод с клавиатурной панели телефона (DTMF)

С помощью звонка по имени пользователи могут связаться с пользователями по полному или частичному имени. Существуют различные форматы, которые можно использовать при в введенном имени.

При поиске в каталоге организации с помощью клавиши 0 (ноль) можно указать пробел между именем и фамилией и именем. При вводе имени будет предложено прервать ввод клавиатуры с помощью клавиши #. Например: "После ввода имени человека, с которого вы пытаетесь связаться, нажмите кнопку "#". Если найдено несколько имен, вызываемму человеку будет предоставлен список имен для выбора.
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|Формат имени|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный  |Иван0Макаренко# |Иван Макаренко |
|Фамилия + Имя |Полный |Макаренко0Иван#  |Иван Макаренко |
|FirstName  |Полный   |Иван#   |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия |Полный |Макаренко#  |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко" |
|Имя или фамилия |Частичный |Ма# |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко" |
|Имя + Фамилия |Частичный |Amos0Mar # |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия + Имя |Частичный |Ма0Ив# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |

Существует несколько специальных символов, которые используются при поиске людей, использующих клавиатурную панель телефона. Например, человеку будет предложено использовать клавишу с фунтом (#), а для пробела между именами будет использоваться ноль (0). При нажатии клавиши звездочка (*) список имен, совпадающих с человеком, будет повторяться.
  
|Специальный знак на клавиатуре телефона|Описание|
|:-----|:-----|
|#   |Завершение ввода имени. |
|0   |Пробел между именами. |
|*    |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска. |

#### <a name="dial-by-name---name-recognition-with-speech"></a>Звонок по имени  распознавание имен при голосовом вводе

Люди могут искать других людей в своей организации с помощью голосового голоса (распознавания речи). Кроме того, он может связаться с любым человеком в Active Directory, назовя имя нужного человека. С помощью голосового ввода можно распознавать имена в различных форматах, включая имя, фамилию, имя + фамилия или фамилию + имя.
  
Вы можете включить распознавание речи для автозавода, но запись на клавиатурной панели телефона (DTMF) не отключена. Ввод с клавиатурной панели телефона можно использовать в любое время, даже если автозавод включена распознавание речи.
  
Как и при вводе на клавиатурной панели телефона, если найдено несколько имен, вызываемая вами учетная запись будет слышать список имен для выбора.
  
Вызыватели могут называть имена в следующих форматах:
  
|Имя с речью|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный |Иван Макаренко |Иван Макаренко |
|Фамилия + Имя |Полный  |Макаренко Иван |Иван Макаренко |
|FirstName |Полный |Иван |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров" |
|Фамилия |Полный |Макаренко |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко" |

> [!NOTE]
> Имя нового пользователя может быть указано в каталоге для функции "Набрать по имени" с распознаванием речи из-за задержки репликации Active Directory.

### <a name="the-operator-option"></a>Параметр оператора

При желании можно настроить автоотвод, чтобы дать вызываемой функции возможность поговорить с оператором.
  
Клавиша 0 и голосовая команда "Оператор" перенаправят звонок на назначенный по умолчанию оператор. Это касается всех языков, поддерживаемых для распознавания речи. Вы также можете использовать **параметры меню "Настройка"** для настройки пользовательского значения оператора.
  
Оператором может быть:
  
- Пользователь Microsoft Teams или сервер Skype для бизнеса Server с Корпоративная голосовая связь поддержки.
- Другой настроенный в организации автосекретарь.
- Любая настроенная в организации очередь вызовов. Дополнительные информацию об очередях вызовов см. в [подмносях "Создание очереди звонка в облаке".](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue)

### <a name="business-hours-and-call-handling"></a>Рабочие часы и обработка звонков

Для каждого автозавода можно настроить часы работы. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. Для работы можно установить перерывы в течение дня, а все часы, которые не считаются часами, считаются неавтными. Вы можете настроить разные параметры обработки входящих параметров обработки параметров звонка и другие приветствия (необязательно) для работы и после работы.
  
У каждого автозаполнения есть несколько возможных параметров обработки параметров.
  
- Звонок может отключиться после воспроизведение приветствия.
- Также вы можете:
  - Переадресуйте звонок пользователю  Microsoft Teams, у которого есть лицензия на телефонную систему, которая Корпоративная голосовая связь включена или ему назначены планы звонков. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.

  - Перенаправление звонка в очередь. Дополнительные информацию об очередях вызовов см. в [подмносях "Создание очереди звонка в облаке".](/SkypeForBusiness/what-is-phone-system-in-office-365/create-a-phone-system-call-queue)

  - Перенаправить звонок другому автозаводу.

Эти параметры выражается вызываемой функцией автозапуска при его вызове в меню. Например: "Нажмите 1 для отдела продаж, нажмите 2 для служб. Чтобы поговорить с оператором, нажмите 0 в любое время".

### <a name="set-menu-options"></a>Настройка параметров меню

Автоответы в облаке позволяют создавать запросы меню ("Нажмите 1 для отдела продаж, нажмите 2 для служб") и настроить параметры меню для перенастройки звонков в зависимости от выбора пользователем. Параметры меню для автоответа предоставляют организации ряд вариантов, которые ускоряют перенаправление вызывающего вызова, не влияя на оператора человека для обработки входящих звонков. Для создания подсказок в меню можно использовать функцию "текст в речь" (системные запросы) или путем отправки записанного звукового файла. Распознавание речи принимает голосовые команды для беззвучной навигации, но звоня люди также могут перемещаться по меню с помощью клавиатурной панели телефона.
  
Клавиши с 0 по 9 можно на основе Центра администрирования Skype для бизнеса на основе набора клавиш автоотекатаря. Для работы и после работы можно создавать различные наборы параметров меню, а также включить или отключить функцию "Набор по имени" в параметрах **меню.** Для передачи звонков можно использовать клавиши:
  
- Оператор (по умолчанию назначается клавиша 0). Однако его можно перена назначено любой другой клавише или удалить из меню.
- Очередь вызовов.
- Другой автосекретарь. Чтобы настроить многоуровневые меню, наводя указатель на параметры меню одного автозавода на другого автозавода с собственным набором параметров меню (вложенный автозапуск). 
- Пользователь Microsoft Teams с  лицензией на телефонную систему, которая Корпоративная голосовая связь включена или ему назначены планы звонков. Ее можно настроить так, что звонки вызывающего абонента будут отправляться в голосовую почту. Для этого выберите **Сотрудник вашей компании** и установите для этого абонента режим переадресации звонков непосредственно на голосовую почту.
  
Если включена распознавание речи, название каждого параметра меню становится ключевым словом распознавания речи. Например, можно сказать "Один" для выбора параметра меню, назначенного клавише 1, или "Продажи", чтобы выбрать тот же пункт меню с именем "Продажи".
  
Чтобы настроить автозапуск и параметры меню, перейдите к пункту "Настройка автозапуска в [облаке".](create-a-phone-system-auto-attendant.md)
  
### <a name="assigning-phone-numbers-for-an-auto-attendant"></a>Назначение номеров телефонов для автозавода

Вы можете назначить номер службы Майкрософт, прямой маршрутизный номер или гибридный номер учетной записи связанного ресурса автозаправщика (или нескольким учетным записям ресурсов при желании несколько номеров телефонов). Дополнительные [сведения см. в перенаправке](direct-routing-plan.md) плана.

Чтобы назначить номер службы, необходимо получить или переназначить существующие платные или бесплатные номера служб. После получения платных или бесплатных номеров телефонов служб они будут показываться в голосовых номерах Центра администрирования **Skype**  >    >  **для бизнеса.** **Number type** is listed as **Service - Toll-Free.** Чтобы узнать, как получить номера служб, см. получение номеров телефонов служб для Skype для бизнеса и [Microsoft Teams](./getting-service-phone-numbers.md) или, если вы хотите перенести и использовать существующий номер службы, см. перенос номеров телефонов [в Teams.](phone-number-calling-plans/transfer-phone-numbers-to-teams.md)
  
> [!NOTE]
> Если вы находитесь за пределами США, вы не можете использовать Центр администрирования Microsoft Teams для получения номеров служб. Чтобы [узнать, как это сделать, перейдите](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) в "Управление номерами телефонов для организации".
