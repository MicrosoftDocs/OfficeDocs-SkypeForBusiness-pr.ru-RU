# <a name="set-up-holidays-in-microsoft-teams"></a>Настройка выходных в Microsoft Teams

Вы можете использовать функцию "праздники группы" для предоставления альтернативных сообщений и перенаправления тем, чтобы они выработали определенные даты и время, когда отделы, очереди звонков и пользователи в вашей организации будут находиться в разных рабочих часах или не будут доступны. Например, вы можете создать праздничный день для нового года, когда ваша организация может быть закрыта.

Праздничные дни, которые вы создаете, доступны при [настройке автосекретаря](create-a-phone-system-auto-attendant.md), каждый из которых имеет собственное приветствие и настройки маршрутизации звонков.

## <a name="create-a-holiday"></a>Создание праздника

Создание праздника

1. В центре администрирования Microsoft Teams перейдите в раздел "Праздничные параметры" в **Организации**  >  **Holidays**.

2. Выберите **новый праздник**.

3. Введите имя праздника.

4. Нажмите кнопку **Добавить новую дату**.

5. В разделе **время начала** щелкните значок календаря и выберите дату начала праздника.

6. С помощью раскрывающегося списка выберите время начала праздника.

7. В разделе **время окончания** щелкните значок календаря и выберите дату окончания праздника.

8. С помощью раскрывающегося списка выберите время окончания праздника. **Время окончания** должно быть позже **времени начала**.  

   > [!NOTE]
   > Если праздник за один целый день (например, в течение 24 часов), **время окончания** должно быть установлено на следующий день, а время — на 12:00. Например, если ваша организация закрылась 1 января для нового года, установите **время начала** в январе 1 12:00, а затем укажите **время окончания** на 2 января 12:00 г.

9. При необходимости добавьте дополнительные даты для повторяющихся праздников.

10. Нажмите кнопку **сохранить**.

    ![Снимок экрана: Пользовательский интерфейс праздников с датами, заданными в течение трех лет](media/holidays-set-up.png)

## <a name="change-a-holiday"></a>Изменение праздника

Изменение праздников

1. В центре администрирования Microsoft Teams перейдите в раздел "Праздничные параметры" в **Организации**  >  **Holidays**.

2. Выберите праздничный день в списке.

3. В разделе **время начала** щелкните значок календаря и выберите дату начала праздника.

4. С помощью раскрывающегося списка выберите время начала праздника.

5. В разделе **время окончания** щелкните значок календаря и выберите дату окончания праздника. 

6. С помощью раскрывающегося списка выберите время окончания праздника. **Время окончания** должно быть позже **времени начала**.  

7. Нажмите кнопку **сохранить**.
