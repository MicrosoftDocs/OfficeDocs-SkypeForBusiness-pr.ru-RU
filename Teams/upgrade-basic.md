# <a name="upgrade-basic"></a>Обновить базовую

<a name="about-upgrade-basic"></a>

Предназначен для небольших организаций или пользователей Skype для бизнеса Online только для мгновенных сообщения (чатов) и собраний, контрольный список Обновить базовый — это ускоренный план действий, включающий основные, рекомендуемые действия и связанные ресурсы для успешного перемещения с Skype для бизнеса на Teams.

Эти десять простых действий предоставляют все необходимое для успешного обновления. Срок их выполнения составляет от 30 до 45 дней, но даты завершения задач следует настраивать в зависимости от расписания обновления организации.

> [!IMPORTANT]
> Skype для бизнеса Будет отменено 31 июля 2021 г. После этого веб-служба Skype для бизнеса будет недоступна и не поддерживается. Чтобы максимально увеличить преимущества и убедиться, что у вашей организации есть надлежащее время на реализацию обновления, мы рекомендуем начать свой путь к Microsoft Teams сегодня.

Что произойдет Skype для бизнеса после обновления? После перевода пользователей на Teams (режим **Только Teams**):

- Клиент Skype для бизнеса отключен, а все чаты и звонки перена могут переходить в Teams. Обратите внимание, что при этом клиент не будет удалиться с компьютера.
- Все Skype для бизнеса, которые были запланированы до обновления, но все новые собрания планируются в Teams. Подключаемый Skype для бизнеса больше не будет доступен в Outlook. 
- Если пользователи попытаются войти в Skype для бизнеса, они получат от клиента уведомление о том, что они обновлены до Teams.
- Пользователям необходимо вручную удалить клиент Skype для бизнеса на своих мобильных устройствах.

Дополнительные [вопросы об обновлении](./faq-journey.yml) см. в вопросах и ответах.

Не знакомы с Teams? Узнайте, как [Teams](https://products.office.com/microsoft-teams/group-chat-software) объединяет беседы, собрания, файлы, приложения Office и интеграцию со сторонними Microsoft 365 и Office 365.

<a name="step-2"></a>

## <a name="step-2-prepare-your-organization-for-teams"></a>Шаг 2. Подготовка организации к использованию Teams

*(Около четырех–шести недель до обновления)*

Teams совместимы с Skype для бизнеса мгновенными сообщениями (чатом) и собраниями, но она также позволяет делать многое другое. В качестве истинного центра командной работы Teams позволяет группам управлять проектами, файлами, беседами и приложениями в одном месте. По умолчанию продукт Microsoft Teams включен для всех организаций. Решите, как ваша организация будет Teams и настройте среду для достижения успеха.

> Как существующий клиент Skype для бизнеса, ваша текущая инфраструктура сети, скорее всего, уже настроена для Teams. Чтобы подтвердить это, следуйте указаниям по полному техническому планированию, ссылки на которые приведены ниже (необязательно).
