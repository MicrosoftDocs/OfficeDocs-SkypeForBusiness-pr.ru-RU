# <a name="upgrade-basic"></a>Обновление уровня "Базовый"

<a name="about-upgrade-basic"></a>

Для небольших организаций или тех, кто использует Skype для бизнеса Online только для обмена мгновенными сообщениями (чат) и собраний, контрольный список обновления уровня "Базовый" — это план ускоренного действия, который включает основные, рекомендуемые действия и связанные ресурсы для реализации успешного перехода с Skype для бизнеса на Teams.

Эти десять простых шагов предоставляют все необходимое для успешного обновления. Они предназначены для выполнения в течение 30–45 дней, но вам следует настроить даты завершения задач в соответствии с расписанием обновления вашей организации.

> [!IMPORTANT]
> Skype для бизнеса Online прекращена 31 июля 2021 г. Чтобы максимально эффективно реализовать преимущества и убедиться, что у вашей организации есть соответствующее время для реализации обновления, мы рекомендуем начать работу с Microsoft Teams уже сегодня.

Что произойдет Skype для бизнеса после обновления? После перевода пользователей на Teams (режим **Только Teams**):

- Клиент Skype для бизнеса отключен, а все звонки и чаты перенаются в Teams. Обратите внимание, что это не приведет к удалению клиента на рабочих столах.
- Все Skype для бизнеса, запланированные до обновления, работают должным образом, но все новые собрания запланированы в Teams. Подключаемый модуль Skype для бизнеса больше не будет доступен в Outlook. 
- Если пользователи пытаются войти в Skype для бизнеса, они получают от своего клиента уведомление о том, что они были обновлены до Teams.
- Пользователям необходимо вручную удалить клиент Skype для бизнеса на своих мобильных устройствах.

Дополнительные [вопросы о](./faq-journey.yml) вашем обновлении см. в наших часто задаваемых вопросах.

Не знакомы с Teams? Узнайте, как [Teams](https://products.office.com/microsoft-teams/group-chat-software) объединяет беседы, собрания, файлы, приложения Office и сторонние интеграции, предоставляя единый центр для командной работы в Microsoft 365 и Office 365.

## <a name="step-1-notify-your-key-stakeholders"></a>Шаг 1. Уведомление ключевых заинтересованных лиц

*(Около четырех-шести недель до обновления)*

Руководители высшего звена должны быть перечислены за успешное завершение работы компании; не забудьте оставить их в знаниях об изменениях технологий. Так как возможно, что не все получили или прочли уведомление о соответствии обновлению, перед началом планирования обновления необходимо проинформировать заинтересованных лиц (например, генерального директора, ИТ-специалистов, специалистов по маркетингу и службу технической поддержки).

<a name="step-2"></a>

## <a name="step-2-prepare-your-organization-for-teams"></a>Шаг 2. Подготовка организации к использованию Teams

*(Около четырех-шести недель до обновления)*

Teams предлагает совместимые Skype для бизнеса, такие как обмен мгновенными сообщениями (чат) и собрания, но это также может сделать гораздо больше. В качестве истинного центра командной работы Teams позволяет рабочим группам управлять проектами, файлами, беседами и приложениями в одном расположении. По умолчанию продукт Microsoft Teams включен для всех организаций. Решите, как ваша организация будет использовать Teams, и настройте среду для успешного выполнения. 

> [!Note]
> Как существующий Skype для бизнеса, текущая сетевая инфраструктура, скорее всего, уже настроена для Teams. Чтобы подтвердить это, следуйте инструкциям по завершению технического планирования, приведенным ниже (необязательно).

## <a name="step-3-know-your-skype-for-business-users"></a>Шаг 3. Сведения о Skype для бизнеса пользователей

*(Около четырех недель до обновления)*

Пользователям, которые глубоко внедрены в Skype для бизнеса, может потребоваться немного больше времени или помощи для перехода на Teams. Проверьте текущее использование Skype для бизнеса, чтобы определить основных пользователей, которым требуется дополнительная поддержка, и определить базовые показатели использования, которые можно отслеживать с помощью номеров после обновления.

## <a name="step-7-upgrade-users-to-teams"></a>Шаг 7. Обновите пользователей до Teams!

*(День обновления)*

Сегодня день, когда ваша организация официально обновляется до Teams в качестве решения для взаимодействия и совместной работы. В Центре администрирования Microsoft Teams активируйте переключатель обновления, заключив режим сосуществования только **в Teams**. (В Центре администрирования перейдите в **Teams** >  **Параметры обновления Teams**.) Пользователи получат уведомление в клиенте Skype для бизнеса о том, что они были обновлены до Teams.

После обновления рекомендуется отправить в Teams сообщение электронной почты с приветствием.
