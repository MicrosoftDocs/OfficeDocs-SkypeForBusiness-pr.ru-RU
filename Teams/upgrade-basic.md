# <a name="upgrade-basic"></a>Обновить базовую

<a name="about-upgrade-basic"></a>

Он предназначен для небольших организаций или пользователей, использующих Skype для бизнеса Online только для мгновенных сообщения (чатов) и собраний, контрольный список "Обновление: базовое обновление" — это ускоренный план действий, который включает основные, рекомендуемые действия и ресурсы, необходимые для успешного перемещения со Skype для бизнеса на Teams.

Эти десять простых шагов обеспечивают все необходимое для успешного обновления. Их можно завершить в течение 30–45 дней, однако даты завершения задач следует настраивать в зависимости от расписания обновления организации.

> [!IMPORTANT]
> Skype для бизнеса Online будет отменен 31 июля 2021 г. После этого служба Skype для бизнеса Online больше не будет доступна и не поддерживается. Чтобы максимально расширить реализацию преимуществ и убедиться, что у вашей организации есть время на реализацию обновления, мы рекомендуем приступить к использованию Microsoft Teams уже сегодня.

Что происходит со Skype для бизнеса после обновления? После перевода пользователей на Teams (режим **Только Teams**):

- Клиент Skype для бизнеса отключен, а все чаты и звонки переначаты в Teams. Обратите внимание, что при этом клиент не будет удалиться с их компьютеров.
- Все собрания Skype для бизнеса, запланированные до обновления, работают так, как планировалось, но все новые собрания планируются в Teams. Подключаемый модуль Skype для бизнеса больше не будет доступен в Outlook. 
- Если пользователи попытаются войти в Skype для бизнеса, они получат от клиента уведомление о том, что они обновлены до Teams.
- Пользователям необходимо вручную удалить клиент Skype для бизнеса на мобильных устройствах.

Дополнительные [вопросы об](https://aka.ms/SkypeToTeams-FAQ) обновлении см. в вопросах и ответах.

Не знакомы с Teams? Узнайте, [как Teams](https://products.office.com/microsoft-teams/group-chat-software) объединяет беседы, собрания, файлы, приложения Office и интеграцию со сторонними службами— это единое центр для командной работы в Microsoft 365 и Office 365.

*(Примерно за четыре–шесть недель до обновления)*

<a name="step-2"></a>

## <a name="step-2-prepare-your-organization-for-teams"></a>Шаг 2. Подготовка организации к использованию Teams

*(Примерно за четыре–шесть недель до обновления)*

Teams поддерживает совместимые функции Skype для бизнеса, такие как мгновенные сообщения (чат) и собрания, но это также позволяет делать многое другое. В качестве истинного центра командной работы Teams позволяет группам управлять проектами, файлами, беседами и приложениями в одном месте. По умолчанию продукт Microsoft Teams включен для всех организаций. Определите, как ваша организация будет использовать Teams, и настройте среду для достижения успеха. 

> [!Note]
> Как клиент Skype для бизнеса ваша текущая инфраструктура сети, скорее всего, уже настроена для Teams. Чтобы подтвердить это, следуйте указаниям по полному техническому планированию, ссылки на которые приведены ниже (необязательно).

## <a name="step-3-know-your-skype-for-business-users"></a>Шаг 3. Know your Skype for Business users

*(Примерно за четыре недели до обновления)*

## <a name="step-7-upgrade-users-to-teams"></a>Шаг 7. Обновив пользователей до Teams!

*(День обновления)*
