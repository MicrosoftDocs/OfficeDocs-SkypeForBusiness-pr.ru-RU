# <a name="manage-skype-for-business-settings-in-the-microsoft-teams-admin-center"></a>Управление Skype для бизнеса параметров в Центре администрирования Microsoft Teams

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="sfb-settings"> </a>
<!-- Do not remove the bookmark link above. -->

Как администратор, Центр администрирования Microsoft Teams позволяет управлять функциями Skype для бизнеса для Skype для бизнеса пользователей в организации. Вы можете управлять параметрами [](#manage-skype-for-business-settings-for-your-organization) организации на странице Skype для бизнеса и  параметрами отдельных пользователей на вкладке **Skype для бизнеса** страниц с подробными сведениями о пользователе.[](#manage-skype-for-business-settings-for-individual-users)

Вы увидите страницу Skype для бизнеса только  в том случае, если для режима сосуществования вашей организации задано не **только Teams**. Аналогичным образом вы увидите вкладку Skype для бизнеса для пользователя, только если режим сосуществования пользователя не только **Teams**. Дополнительные сведения о режимах сосуществования см. в статье "Общие сведения о [teams и](teams-and-skypeforbusiness-coexistence-and-interoperability.md) Skype для бизнеса сосуществование и взаимодействие, а также настройка параметров сосуществования и [обновления"](setting-your-coexistence-and-upgrade-settings.md).

> [!NOTE]
> Skype для бизнеса параметры ранее находились на **портале прежних версий** в Центре администрирования Microsoft Teams. С прекращением использования устаревшего портала мы перенесите параметры в эти новые расположения в Центре администрирования Teams для Skype для бизнеса управления.

Вам должна быть назначена роль [Azure AD](/azure/active-directory/roles/permissions-reference) глобального администратора или администратора Skype для бизнеса для управления Skype для бизнеса в Центре администрирования Microsoft Teams.

## <a name="manage-skype-for-business-settings-for-your-organization"></a>Управление Skype для бизнеса параметров для организации

В левой области навигации Центра администрирования Microsoft Teams перейдите  >  к параметрам на уровне организации **Skype для бизнеса**. Здесь вы можете настроить трансляцию собраний Skype, конфиденциальность сведений о присутствии и уведомления мобильных устройств и управлять ими для всех Skype для бизнеса пользователей в организации.

### <a name="skype-meeting-broadcast"></a>Трансляция собрания Skype

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="sfb-org-wide-broadcast"> </a>
<!-- Do not remove the bookmark link above. -->

Используйте следующие параметры для управления трансляцией [собраний Skype](https://support.microsoft.com/office/what-is-a-skype-meeting-broadcast-c472c76b-21f1-4e4b-ab58-329a6c33757d) в организации.

:::image type="content" source="media/skype-for-business-settings-meeting-broadcast.png" alt-text="Снимок экрана: параметры трансляции собраний Skype в Центре администрирования.":::

- **Трансляции собраний Skype**: включите эту функцию, чтобы включить трансляцию собраний Skype для вашей организации. После включения этой функции необходимо настроить сеть для трансляции [собраний Skype](/skypeforbusiness/set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast).
- **Ознакомьтесь с предварительными версиями** функций. Включите эту функцию, чтобы получить ранний доступ к новым функциям.
- **Организаторы могут планировать** анонимные собрания. Включите эту функцию, если хотите разрешить организаторам создавать трансляции, позволяющие всем пользователям за пределами вашей организации присоединяться к собранию без необходимости входа в систему. 
- **Запись собраний трансляции** собраний Skype: включите эту функцию, чтобы организаторы и выступающие могут записывать собрания.  
- **URL-адрес поддержки службы** технической поддержки для участников: введите URL-адрес службы поддержки вашей организации, который участники собрания могут использовать, если им нужна помощь во время собрания.

### <a name="presence-and-mobile-notifications"></a>Уведомления о присутствии и мобильных устройствах

Используйте следующие параметры для управления конфиденциальностью Skype для бизнеса присутствия и мобильными уведомлениями в организации.

:::image type="content" source="media/skype-for-business-settings-presence-mobile.png" alt-text="Снимок экрана: параметры присутствия в Центре администрирования.":::

#### <a name="presence"></a>Присутствие

По умолчанию Skype для бизнеса пользователи в организации могут видеть состояние присутствия (например, "Доступно", "Занят" или "Нет на месте") других Skype для бизнеса пользователей. Выберите один из следующих параметров, чтобы указать, кто может видеть Skype для бизнеса пользователей.

- **Автоматически отображать сведения** о присутствии: любой Skype для бизнеса в вашей организации, который не был добавлен во внешний или заблокированный список пользователя, может  видеть его  присутствие.
- Отображение **сведений** о присутствии только для контактов пользователя. Любой пользователь Skype для бизнеса в списке контактов пользователя, который не добавлен в список внешних или заблокированных, может  видеть его присутствие. Пользователи могут переопределить этот параметр в Skype для бизнеса, перейдя в меню **"Параметры** > **средств** > **"**.

#### <a name="mobile-notifications"></a>Мобильные уведомления

Вы можете указать, будут ли Skype для бизнеса получать оповещения о входящих и пропущенных мгновенных сообщениях, сообщениях голосовой почты и пропущенных звонках через службу push-уведомлений. В зависимости от мобильных устройств, используемых в организации, вы можете использовать службу push-уведомлений **Майкрософт**, службу **push-уведомлений Apple** или и то, и другое.

Учитывайте следующее.

- Если отключить push-уведомления, пользователи будут получать все оповещения при следующем запуске Skype для бизнеса мобильном устройстве.
- По умолчанию push-уведомления включены. Отдельные пользователи могут отключить их в Skype для бизнеса мобильном устройстве.
- При отключении push-уведомлений пользователи не могут включить их снова. 

> [!IMPORTANT]
> Корпорация Майкрософт использует другие компании для предоставления уведомлений Skype для бизнеса мобильных устройств Windows Phone, iPhone и iPad. См. это [заявление о конфиденциальности](https://go.microsoft.com/fwlink/p/?linkid=247732).

## <a name="manage-skype-for-business-settings-for-individual-users"></a>Управление Skype для бизнеса параметров для отдельных пользователей

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="sfb-user-settings"> </a>
<!-- Do not remove the bookmark link above. -->

Чтобы управлять Skype для бизнеса для отдельных пользователей, в левой области навигации Центра администрирования Teams перейдите к разделу "Пользователи **",** щелкните отображаемое имя пользователя, чтобы открыть страницу сведений о пользователе, а затем выберите вкладку **Skype для бизнеса** параметров. Здесь можно настроить параметры внешнего доступа и собраний для пользователя.

:::image type="content" source="media/skype-for-business-settings-user.png" alt-text="Снимок экрана: Skype для бизнеса на странице сведений о пользователе.":::

### <a name="external-access-settings"></a>Параметры внешнего доступа

Вы можете выборочно разрешить или заблокировать возможность взаимодействия пользователя с людьми за пределами вашей организации.

- **Внешние Skype для бизнеса** пользователей: включите этот параметр, если вы хотите разрешить пользователю взаимодействовать с Skype для бизнеса пользователями в федеративных доменах.
- **Внешние пользователи Skype**: включите этот параметр, если хотите разрешить пользователю взаимодействовать с пользователями Skype. 

### <a name="meeting-settings"></a>Параметры собрания

Вы можете настроить следующие параметры собрания для пользователя.

- **Аудио & видео**: выберите один из следующих параметров звука и видео:

    - **Нет**: пользователь не может использовать звук или видео.
    - **Только звук**: пользователь может использовать звук, но не видео.
    - **Звук и видео**: пользователь может использовать звук и видео.
    - **Аудио и видео (HD)**: пользователь может использовать аудио и видео с высоким уровнем определения.
    
- **Запись бесед & собраниях**: включите эту функцию, чтобы разрешить пользователю записывать беседы и собрания.
- **Соответствие** требованиям. Включите этот параметр, если вам по юридическим требованиям необходимо хранить хранимую в электронном виде информацию.
