<a name="use-activity-reports-for-microsoft-teams"></a>Использование отчетов об активности в Microsoft Teams 
========================================

Чтобы увидеть, как пользователи в вашей организации используется группами Майкрософт, можно использовать отчеты об активности в центре администрирования Microsoft 365. Например если некоторые еще не используется группами Майкрософт, они могут не знать как приступить к работе или понять, как можно использовать команды для повысить эффективность и совместной работы. Вы можете использовать отчеты об активности в вашей организации, чтобы определить, где существует приоритетная потребность в обучении и улучшении коммуникации.

## <a name="how-to-view-the-teams-reports-in-the-reports-dashboard"></a>Просмотр групп отчетов в панели мониторинга "Отчеты"

1. В [центре администрирования Microsoft 365](https://portal.office.com/adminportal/home)выберите **отчеты о** > **об использовании**.
 
2. На странице **Использование** выберите **Выберите отчет**и затем в разделе **Группами Майкрософт** в список отчетов, выберите отчет, который нужно просмотреть.

## <a name="teams-activity-reports-that-are-available"></a>Отчеты об активности группами, доступные

В настоящее время существует два отчета активности, которые можно просмотреть.

- [Отчет о действиях пользователей Microsoft Teams](#microsoft-teams-user-activity-report) 
- [Отчет об использовании устройств с Microsoft Teams](#microsoft-teams-device-usage-report) 

### <a name="microsoft-teams-user-activity-report"></a>Отчет о действиях пользователей Microsoft Teams

Отчета об активности пользователей группам позволяет представление наиболее распространенных операций, которые пользователям выполнять в группах. Это включает в себя сколько людей вступать в чата в канале, сколько связь посредством частной беседы сообщения и сколько участвовать в звонков или собрания. Можно видеть эти сведения для всей организации, а также для каждого пользователя.

![Снимок экрана с группами пользователей отчете в центре администрирования Office 365.](media/teams-user-activity-report.png)

## <a name="who-can-access-the-teams-activity-reports"></a>Пользователей, которые могут получить доступ к отчеты об активности групп

Отчеты об активности пользователей, для которых может осуществляться:

- Роль глобального администратора Office 365
- Роль администратора конкретного продукта (Exchange, Скайп для бизнеса или SharePoint)
- Роль читателя отчетов

### <a name="reports-reader-role"></a>Роль читателя отчетов

Можно назначить роль *чтения отчетов* для ИТ-специалистов, которые будут иметь доступ к отчетам. Назначая эта роль руководителями обучения или заинтересованным, можно убедиться в том, что они имеют доступ к средствами, которые могут быть полезны для внедрения диска и отслеживания рабочих групп.

### <a name="at-a-glance-activity-widget"></a>Мини-приложения в быстрого активности

Панель мониторинга отчетов содержит данные об использовании из рабочих групп в мини-приложения в быстрого действия, который дает представление связанных продуктов как пользователей общаться и совместно работать с помощью различных служб в Office 365.

![Снимок экрана мини-приложения в быстрого действия группы в отчеты панели мониторинга.](media/at-a-glance-activity-widget.png)

### <a name="teams-activity-card"></a>Карточка активности групп

Карточки действия группы на панели мониторинга отчетов дается обзор действия в групп, включая число активных пользователей, чтобы позволяет быстро понять, сколько пользователей с помощью службы.
