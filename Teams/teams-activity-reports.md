<a name="use-activity-reports-for-microsoft-teams"></a>Использование отчетов об активности в Microsoft Teams 
========================================

С помощью отчетов об активности в центре администрирования Microsoft 365 можно увидеть, как пользователи в вашей организации работают в Microsoft Teams. Например, если некоторые не работают в Microsoft Teams, они могут не знать, как приступить к работе, и понять, как они могут использовать группы для повышения продуктивности и совместной работы. Вы можете использовать отчеты об активности в вашей организации, чтобы определить, где существует приоритетная потребность в обучении и улучшении коммуникации.

## <a name="how-to-view-the-teams-reports-in-the-reports-dashboard"></a>Просмотр отчетов о командах на панели мониторинга "отчеты"

1. В [центре администрирования Microsoft 365](https://portal.office.com/adminportal/home)выберите пункт Использование **отчетов** > ****.
 
2. На странице **Использование** нажмите **кнопку Выбрать отчет**, а затем в разделе **Microsoft Teams** (список отчетов) выберите отчет, который вы хотите просмотреть.

## <a name="teams-activity-reports-that-are-available"></a>Доступные отчеты об активности в Teams

В настоящее время доступны два отчета о действиях, которые можно просмотреть:

- [Отчет о действиях пользователей Microsoft Teams](#microsoft-teams-user-activity-report) 
- [Отчет об использовании устройств с Microsoft Teams](#microsoft-teams-device-usage-report) 

### <a name="microsoft-teams-user-activity-report"></a>Отчет о действиях пользователей Microsoft Teams

С помощью отчета о действиях пользователей в Teams вы получаете представление о наиболее распространенных действиях, выполняемых пользователями в Teams. Сюда входит количество людей, участвующих в чате в канале, количество контактов через личное сообщение чата, а также количество входящих звонков и собраний. Вы можете просматривать эти сведения для всей Организации, а также для каждого отдельного пользователя.

![Снимок экрана: отчет о действиях пользователей в центре администрирования.](media/teams-user-activity-report.png)

## <a name="who-can-access-the-teams-activity-reports"></a>Кто может получать доступ к отчетам об активности в Teams

Доступ к отчетам об активности может получать пользователи, которым назначены:

- Роль глобального администратора Office 365
- Роль администратора для определенного продукта (Exchange, Skype для бизнеса или SharePoint)
- Роль читателя отчетов

### <a name="reports-reader-role"></a>Роль читателя отчетов

Вы можете назначить роль *читателя отчетов* неит-специалистам, которым вы хотите получить доступ к этим отчетам. Назначая эту роль руководителям по обучению или заинтересованным лицам для бизнеса, вы можете убедиться, что у них есть доступ к аналитическим аналитикам, которые полезны для отслеживания принятия групп.

## <a name="other-information-on-the-reports-dashboard"></a>Другие сведения на панели мониторинга "отчеты"

### <a name="at-a-glance-activity-widget"></a>Наглядный графический элемент "действия"

Панель мониторинга "отчеты" содержит данные об использовании в мини-приложении "действия", которое позволяет получить представление о том, как пользователи обмениваются информацией и совместно работать с другими различными службами в Office 365.

![Снимок экрана: мини-приложение "действия" в команде "быстрый просмотр".](media/at-a-glance-activity-widget.png)

### <a name="teams-activity-card"></a>Карточка активности Teams

На панели мониторинга "отчеты" выводятся общие сведения о действиях в Teams, в том числе количество активных пользователей, с помощью которых можно быстро понять, сколько пользователей использует службу. Нажатие на карточке действий на панели мониторинга позволяет перейти к отчету об активности пользователей Teams. 

![Снимок экрана: карточка активности команды.](media/teams-activity-card.png)
