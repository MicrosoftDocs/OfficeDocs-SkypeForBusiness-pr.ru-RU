# <a name="information-barriers-and-shared-channels-preview"></a>Информационные барьеры и общие каналы (предварительная версия)

[Общие каналы в](shared-channels.md) Microsoft Teams создают пространства для совместной работы, где можно приглашать людей, которые не являются участниками команды. [Информационные барьеры Microsoft Purview](/microsoft-365/compliance/information-barriers) — это политики, которые можно реализовать, чтобы ограничить и запретить пользователям и группам взаимодействовать друг с другом внутри организации и за ее пределами.

Общие каналы включены по умолчанию в Teams. Создав политику каналов, можно определить: могут ли пользователи создавать общие каналы; могут ли пользователи предоставлять доступ к каналам для пользователей вне организации; могут ли пользователи участвовать во внешних общих каналах. Если в вашей организации настроены политики информационных барьеров, при настройке общих каналов выполняются проверки того, что ни один из существующих участников канала и новых пользователей, добавленных в общий канал, не нарушает условия политики информационных барьеров.

Используйте следующую таблицу, чтобы понять, как политики информационных барьеров могут повлиять на обмен данными и привести к определенному поведению при настройке общих каналов:

|**Сценарий**|**Поведение информационных барьеров**|
|:-----------|:--------------------------------|
| **Предоставление общего доступа к каналу пользователю в организации** | Если пользователю запрещено взаимодействовать с общими участниками канала в соответствии с политикой информационных барьеров, пользователь не отображается в поиске пользователя и канал не используется совместно с командой. <br><br> Если пользователь не может быть добавлен в соответствии с политикой информационных барьеров, вы увидите следующее сообщение: Мы не нашли *совпадений. Обратитесь к ИТ-администратору, чтобы расширить область поиска.* |
| **Совместное использование канала в организации с другой командой, владельцем которого вы владеете** | Если в другой команде есть пользователи, которым запрещено взаимодействовать с общими участниками канала в соответствии с политикой информационных барьеров, канал не будет предоставлен команде. <br><br> Если обмен данными запрещен в соответствии с политикой информационных барьеров, вы увидите следующее сообщение: канал не может быть предоставлен этой *команде. Выберите другую команду или обратитесь к администратору для получения дополнительных сведений.* |
| **Совместное использование канала в организации с другой командой, которая не принадлежит вам** | Если владельцу команды или другим пользователям другой команды запрещено взаимодействовать с общими участниками канала в соответствии с политикой информационных барьеров, доступ к каналу не может быть предоставлен команде. <br><br> Если обмен данными запрещен в соответствии с политикой информационных барьеров, вы увидите следующее сообщение: канал не может быть предоставлен этой *команде. Выберите другую команду или обратитесь к администратору для получения дополнительных сведений.* |
| **Добавление нового пользователя в команду, если у команды есть общие каналы с другими командами** | Если новому пользователю запрещено взаимодействовать с членами команды общего канала в соответствии с политикой информационных барьеров, его нельзя добавить в команду. При добавлении пользователя в команду с шестью или более общими каналами он сразу же добавляется в канал, и общий доступ поддерживается. Общий доступ для команды и ранее общих каналов может быть остановлен, если новый пользователь не соответствует политике информационных барьеров.<br><br> Если пользователь не может быть добавлен в соответствии с политикой информационных барьеров, вы увидите следующее сообщение: Не удается добавить пользователя из-за политики информационных *барьеров.* |
| **Предоставление доступа к каналу внешней команде** | Политики информационных барьеров для внутренних и внешних клиентов не ограничивают обмен данными между пользователями из разных клиентов. Общие каналы доступны для совместного использования с внешними пользователями. |
