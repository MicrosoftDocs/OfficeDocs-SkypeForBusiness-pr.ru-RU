# <a name="manage-emergency-call-routing-policies-for-direct-routing"></a>Управление политиками маршрутинга экстренных вызовов для прямой маршрутизы

Если вы развернули прямую [маршрутику телефонная система](direct-routing-landing-page.md) в организации, вы можете использовать политики маршрутизации экстренных вызовов в Microsoft Teams, чтобы настроить номера экстренных служб и настроить маршруты экстренных вызовов. Политика маршрутинга экстренных вызовов определяет, включены ли расширенные экстренные службы для пользователей, которым назначена политика, какие номера используются для звонков в экстренные службы (например, 911 в США) и как перена могут перенапределяться звонки в экстренные службы.

Вы управляете политиками маршрутизов экстренных вызовов, используя политики голосового экстренного вызова Microsoft Teams центре администрирования или с помощью   >   Windows PowerShell. Политики могут быть назначены пользователям и [сетевым сайтам.](cloud-voice-network-settings.md)

Для пользователей можно использовать глобальную политику (по умолчанию в организации) или создавать и назначать настраиваемые политики. Пользователи будут автоматически получать глобальную политику, если только вы не создайте и не назначите настраиваемую политику. Имейте в виду, что вы можете изменить параметры глобальной политики, но не можете переименовать или удалить ее. Для сетевых сайтов создаются и назначаются настраиваемые политики.

Если политика маршрутинга экстренных вызовов назначена сетевому сайту и пользователю, а пользователь находится на этом сайте, политика, назначенная сетевому сайту, переопределит политику, назначенную пользователю.

## <a name="create-a-custom-emergency-call-routing-policy"></a>Создание настраиваемой политики маршрутизов экстренных вызовов

### <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к центру Политики для экстренного вызова голосовой почты и перейдите на вкладку Политики  >   **маршрутации** вызовов.
2. Нажмите **Добавить**.
3. Введите имя и описание для политики.
4. Чтобы включить динамические экстренные вызовы, включите **динамические экстренные вызовы.** Если динамические экстренные вызовы включены, Teams извлекает сведения о политике и расположении из службы и включает их в экстренные вызовы.
5. Определите один или несколько номеров для экстренного ситуация. Для этого в области **Номера экстренных служб** нажмите **кнопку Добавить** и сделайте следующее:
    1. **Строка набора номера для экстренного помощи:** введите строку набора для экстренного помощи. Эта строка набора указывает на то, что звонок является экстренным.
        > [!NOTE]
        > При прямой маршрутике мы отправляем Teams, которые отправляют экстренные звонки со строкой "+" перед строкой набора экстренных служб. До завершения перехода шаблон голосового маршрута, который соответствует строке набора экстренных служб, должен обеспечить соответствие строкам, которые имеют и не имеют предыдущего "+", например 911 и +911. Например, ^ \\ +?911 или .*.
    2. **Маска для экстренного набора** номера: для каждого номера можно указать ноль или больше маски для экстренного набора номера. Маска набора номера — это номер, который вы хотите перевести в значение строки набора для экстренного срочная помощь. Это позволяет набирать альтернативные номера экстренных служб и по-прежнему звонить в экстренные службы. <br>Например, вы добавляете 112 в качестве маски набора номера для экстренного обслуживания, которая является номером экстренных служб для большинства стран Европы, а 911 — в качестве строки набора номера для экстренного обслуживания. Пользователь Teams из Европы может не знать, что 911 — это номер для экстренного вызова в США, а при наборе номера 112 звонят на номер 911. Чтобы определить несколько масок набора номера, разделите каждое значение за за semicolon. Например, 112;212.
    3. **Запись использования ДНР:** выберите запись использования телефонной сети общего пользования (ННР). Запись использования ОКП используется для определения маршрута, используемого для маршрутизации экстренных вызовов от пользователей, которым разрешено их использовать. Маршрут, связанный с этим использованием, должен указать на магистраль SIP, выделенную для экстренных вызовов, или на шлюз ELIN, который перенаправляет экстренные вызовы в ближайший общедоступный пункт ответа (PSAP).

    > [!NOTE]
    > Строки набора и маски набора должны быть уникальными в политике. Это означает, что для политики можно определить несколько номеров экстренных служб, а также настроить несколько масок набора для строки набора, но каждую из них необходимо использовать только один раз.

6. Нажмите кнопку **Сохранить**.

## <a name="edit-an-emergency-call-routing-policy"></a>Изменение политики маршрутизов экстренных вызовов

### <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

Вы можете изменить глобальную политику или любые настраиваемые политики, которые вы создаете.

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к центру Политики для экстренного вызова голосовой почты и перейдите на вкладку Политики  >   **маршрутации** вызовов.
2. Выберите политику, щелкнув слева от ее имени, а затем нажмите **Изменить**.
3. Внести нужные изменения и нажмите кнопку **Сохранить.**

### <a name="using-powershell"></a>С помощью PowerShell

См. [set-CsTeamsEmergencyCallRoutingPolicy.](/powershell/module/skype/set-csteamsemergencycallroutingpolicy)

## <a name="assign-a-custom-emergency-call-routing-policy-to-a-network-site"></a>Назначение настраиваемой политики маршрутизов экстренных вызовов на сетевой сайт

Используйте [командлет Set-CsTenantNetworkSite,](/powershell/module/skype/set-cstenantnetworksite) чтобы назначить политику маршрутизации экстренных вызовов на сетевой сайт.

В этом примере показано, как назначить сайту Site1 политику маршрутизов экстренных вызовов 1.

```PowerShell
Set-CsTenantNetworkSite -identity "site1" -EmergencyCallRoutingPolicy "Emergency Call Routing Policy 1"
```
