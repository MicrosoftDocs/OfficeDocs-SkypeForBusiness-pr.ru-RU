# <a name="how-to-look-up-your-support-pin"></a>Как найти ПИН-код поддержки

Чтобы найти ПИН-код поддержки, который позволяет службу поддержки Майкрософт по телефону, выполните следующие действия: 

1. Войдите в Центр [Office администрирования](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/homepage). 

2. Щелкните значок шестеренки.

3. Пин-код поддержки будет отображаться в правой нижней области. При необходимости вы также можете повторно создать ПИН-код.
