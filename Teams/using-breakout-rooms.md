# <a name="using-breakout-rooms-in-microsoft-teams"></a>Использование переговорных комнат в Microsoft Teams

Организатор собрания может создавать новые переговорные комнаты в начале запланированного собрания.

## <a name="create-breakout-rooms"></a>Создание переговорных комнат

Выполните следующие действия, чтобы создать переговорную комнату после начала собрания:

1. В Teams на панели управления щелкните значок переговорных комнат.

2. Используйте раскрывающееся меню, чтобы выбрать количество комнат, которые вы хотите создать. За одно собрание можно создать до 50 переговорных комнат.

3. Выберите нужный вариант переговорной комнаты:

    - **Автоматически:** Teams разделять участников поровна между комнатами.
    - **Вручную**: вы можете выбрать, в какую комнату помещается каждый участник.

4. Выберите **"Создать комнаты"**. Откроется панель переговорных комнат.

## <a name="start-breakout-rooms"></a>Запуск переговорных комнат

Когда вы будете готовы начать сеансы останова, выберите " **Пуск комнат"**.

Вы увидите участников, покидающих основное собрание и присоединяющихся к их переговорным комнатам.

Если участник покидает переговорную комнату, организатор может переместить участника обратно в переговорную, щелкнув многоточие **[...]** рядом с именем участника, а затем выбрав "Попросить присоединиться **".**

Если участник присоединяется к главному собранию позднее после открытия переговорных комнат, организатор может добавить участника в переговорную комнату. Если участник пропустил уведомление о присоединении к комнате, выберите "Попросить **присоединиться"**.

## <a name="send-announcements"></a>Отправка объявлений

Организатор собрания может отправлять объявления всем участникам, пока открыты переговорные комнаты. Щелкните многоточие **[...]** в заголовке комнат останова.

Например, вы можете напомнить @mention организатору собрания, когда им нужна помощь, или дать им пятиминутный обратный отсчет, прежде чем перетащить их обратно в главную комнату.
Участники получают объявление в чате переговорных комнат.

## <a name="close-rooms-and-pull-participants-back-to-the-main-meeting"></a>Закрытие комнат и извлечение участников обратно на основное собрание

Выберите **"Закрыть комнаты** ", чтобы вернуть всех участников в главную комнату.

Кроме того, щелкните многоточие **[...]** для определенной переговорной комнаты, а затем выберите **"Закрыть комнату"**.
После закрытия переговорной комнаты чат для этого сеанса закрывается. Участники не могут продолжать общаться в чате.
