# <a name="use-onedrive-for-business-and-sharepoint-or-stream-for-meeting-recordings"></a>Использование OneDrive для бизнеса SharePoint или Stream для записи собраний

В Microsoft Teams существует новый метод сохранения записей собраний. В качестве первого этапа перехода с классической Microsoft Stream на новый [Stream](/stream/streamnew/new-stream) этот метод хранит записи в Microsoft OneDrive для бизнеса и SharePoint в Microsoft 365 и предоставляет множество преимуществ.

> [!NOTE]
> Если запись собрания Teams не удается отправить в OneDrive или SharePoint, появится сообщение об ошибке "Запись неожиданно завершена", а запись будет временно сохранена в Службах мультимедиа Azure (AMS). После сохранения в AMS повторные попытки автоматической отправки записи в OneDrive, SharePoint или Stream не выполняются.

Записи собраний, сохраненные в AMS, доступны в течение 21 дня, а затем автоматически удаляются. При необходимости сохранить копию пользователи могут скачать видео из AMS.

Преимущества использования OneDrive для бизнеса SharePoint для хранения записей:

- Политики хранения для записи собраний Teams (TMR) (метки автоматического хранения S+C E5)
- Преимущества системы OneDrive для бизнеса и управления информацией SharePoint
- Простота настройки разрешений и общего доступа
- Общий доступ к записям с гостями только с явной общей папкой
- Поток запроса доступа
- Предоставление OneDrive для бизнеса и общих ссылок SharePoint
- Записи собраний доступны быстрее
- Поддержка **локального** клиента
- Поддержка нескольких регионов — записи хранятся в регионе, определенном для этого пользователя
- Поддержка создания собственных ключей (BYOK)

См. полный список [возможностей, доступных в настоящее время и ожидаемых в будущем](/stream/streamnew/features-new-version-stream).

Дополнительные сведения см. в статье "Новые возможности записей собраний в Microsoft Teams".

## <a name="set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint"></a>Настройка параметра записи собрания для OneDrive для бизнеса и SharePoint

Параметр записи собраний — это параметр на уровне политики Teams. В приведенном ниже примере показано, как задать глобальную политику. Убедитесь, что вы задаете параметр записи собраний для политики или политик, которые вы назначили пользователям.

> [!NOTE]
> Teams meeting policy changes take a while to propagate. Check back after a few hours of setting it, then sign out and sign in to the Teams Desktop app again or simply restart your computer.

1. Установите Teams PowerShell.

   > [!NOTE]
   > Соединитель Skype для бизнеса Online в настоящее время является частью последнего модуля Teams PowerShell. Если вы используете последний общедоступный выпуск Teams PowerShell, вам не нужно устанавливать соединитель Skype для бизнеса Online. См. статью [Управление Skype для бизнеса Online с помощью PowerShell](/microsoft-365/enterprise/manage-skype-for-business-online-with-microsoft-365-powershell?preserve-view=true&view=o365-worldwide)

2. Запустите PowerShell от имени администратора.

3. Установите модуль [Teams PowerShell](./teams-powershell-install.md).

4. Импортируйте модуль MicrosoftTeams и выполните вход в качестве администратора Teams.

   ```powershell
   # When using Teams PowerShell Module
   
   Import-Module MicrosoftTeams
   $credential = Get-Credential
   Connect-MicrosoftTeams -Credential $credential
   ```

5. Используйте [Set-CsTeamsMeetingPolicy](/powershell/module/skype/set-csteamsmeetingpolicy), чтобы настроить политику собраний Teams для перехода с хранилища Stream на OneDrive для бизнеса SharePoint.

   ```powershell
   Set-CsTeamsMeetingPolicy -Identity Global -RecordingStorageMode "OneDriveForBusiness"
   ```

> [!NOTE]
> Если некоторым пользователям назначена политика для каждого организатора или пользователя, необходимо задать этот параметр для этой политики, если вы хотите, чтобы они также могли хранить записи собраний в OneDrive для бизнеса и SharePoint. Дополнительные сведения см. в статье [Управление политиками собраний в Teams](meeting-policies-overview.md).

## <a name="frequently-asked-questions"></a>Вопросы и ответы

**Где будет храниться запись собрания?**

- Для собраний, отличных от каналов, запись хранится в папке с именем **"** Записи", которая находится на верхнем уровне OneDrive для бизнеса, которая принадлежит пользователю, который начал запись собрания. Пример.

  *Записи записи OneDrive для бизнеса*/ **Запные записи**

- Для собраний канала запись хранится в библиотеке документации сайта Teams в папке **"Записи"**. Пример.

  *Имя Teams — имя канала*/ **Документы**/ **Записи**

**Когда файлы Stream (например, записи) хранятся в SharePoint или OneDrive, как это сделать? Есть ли у администратора возможность изменить место, куда он переходит?**

По умолчанию все файлы записи будут перенаходить в учетную запись OneDrive пользователя, выбраввшего **запись**. Для собраний канала запись всегда будет перенаходить на сайт SharePoint канала. Администратор не может изменить место хранения записи.

**Разделы справки записи от прежних сотрудников?**

Так как видео аналогично любому другому файлу в OneDrive для бизнеса SharePoint, обработка владения и хранения после увольнения сотрудника будет соответствовать обычному процессу OneDrive для бизнеса [и SharePoint](/onedrive/retention-and-deletion).

**У кого есть разрешения на просмотр записи собрания?**

- Для собраний, не включаемых в канал, все приглашенные участники, за исключением внешних участников, автоматически получают личные ссылки. Организатор собрания или пользователь, который начал запись собрания, должен явно добавить внешних участников в общий список.

- Для собраний канала разрешения наследуются от владельцев и участников списка в канале.

> [!NOTE]
> После сохранения записи вы не получите сообщение электронной почты, но после завершения записи она появится в чате собрания. Это произойдет гораздо быстрее, чем в Stream ранее.

**Как управлять субтитрами?**

Субтитры для записей собраний Teams будут доступны во время воспроизведения, только если во время записи у пользователя было включено транскрибирование. Администраторы должны [включить транскрибирование](meetings-policies-recording-and-transcription.md#transcription) записи, чтобы пользователи могли записывать собрания с транскрибированием.

Captions help create inclusive content for viewers of all abilities. As an owner, you can hide captions on the meeting recording, although the meeting transcript will still be available on Teams unless you delete it there.

Субтитры поддерживаются для записей собраний Teams в течение 60 дней с даты записи собрания.

Скрытые субтитры не полностью поддерживаются, если запись собрания Teams перемещается или копируется из исходного расположения в OneDrive для бизнеса SharePoint.

> [!NOTE]
> Субтитры доступны только на английском языке (транскрибирование собраний пока недоступно в GCC).

**Как повлияет квота хранилища?**

Файлы записи собраний Teams OneDrive для бизнеса SharePoint и включаются в квоту для этих служб. См[. сведения о квотах](/sharepoint/sites/plan-site-maintenance-and-management#quotas) [и OneDrive для бизнеса SharePoint](/onedrive/set-default-storage-space).

Вы получаете больше хранилища с [OneDrive для бизнеса](/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits) по сравнению с Stream и более простое хранилище с SharePoint.

**Как воспроизвести запись собрания Teams?**

Ваше видео будет воспроизводится в видеопроигрывателе OneDrive для бизнеса SharePoint в зависимости от того, где вы будете получать доступ к файлу.

**Если вы планируете нерекомендуемо добавлять в Stream, будут ли существующие видео оставаться как есть и как долго?**

Stream как платформа не будет объявлен устаревшим в ближайшем будущем. Видео, которые в настоящее время находятся в Stream, останутся там, пока мы не начнем миграцию. После миграции эти видео также будут перенесены OneDrive для бизнеса или SharePoint. [Дополнительные сведения см](/stream/streamnew/migration-details). в сведениях о миграции.

**Разделы справки метку хранения к записям собраний Microsoft Teams?**

См [. раздел "Автоматическое применение метки хранения"](/microsoft-365/compliance/apply-retention-labels-automatically).

**Разделы справки назначать политики пользователям в Microsoft Teams и какие политики имеют приоритет?**

Узнайте [, какая политика имеет приоритет?](./policy-assignment-overview.md#which-policy-takes-precedence).

**Куда будет записываться запись, если у пользователя нет OneDrive для бизнеса SharePoint или квота хранилища заполнена?**

Запись будет расположена во временном хранилище, где она будет храниться в течение 21 дня. В течение этого времени организатор должен скачать запись. Если запись не загружена в течение 21 дня, запись удаляется.
