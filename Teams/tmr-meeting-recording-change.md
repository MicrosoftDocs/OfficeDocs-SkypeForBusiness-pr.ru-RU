В Microsoft Teams есть новый способ сохранения записей собраний. В качестве первого этапа перехода с классической программы Microsoft Stream на новый [Stream](/stream/streamnew/new-stream)этот метод сохраняет записи в Microsoft OneDrive для бизнеса и SharePoint в Microsoft 365 и предоставляет множество преимуществ.

Преимущества использования OneDrive для бизнеса и SharePoint для хранения записей:

- Политики хранения для записи собраний Teams (TMR) (метки автозавершения S+C5)
- Преимущества управления сведениями в OneDrive для бизнеса и SharePoint
- Простое настройка разрешений и общего доступа
- Делиться записями только с гостями (внешними пользователями)
- Запрос потока доступа
- Предоставление общих ссылок в OneDrive для бизнеса и SharePoint
- Записи собраний доступны быстрее
- **Поддержка локального** клиента
- Поддержка нескольких регионов— записи хранятся в регионе, конкретном пользователе
- Поддержка собственного ключа (BYOK)

Полный список [функций, доступных на сегодняшний день, и возможности, которые ожидаются в течение этого времени.](https://docs.microsoft.com/stream/streamnew/features-new-version-stream) 

Дополнительные сведения можно получить в видео "Новые возможности записей собраний Microsoft Teams".

## <a name="set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint"></a>Настройка записи собрания в OneDrive для бизнеса и SharePoint

Параметр записи собрания настраивается на уровне политики Teams. В следующем примере показано, как настроить глобальную политику. Убедитесь, что вы настроите параметр записи собрания для политик, которые вы настроите для пользователей.

> [!Note]
> Распространение изменений в политике собраний Teams займет некоторое время. Проверьте по и после нескольких часов настройки, а затем выходить и снова входить в нее.

1. Установите Teams PowerShell PowerShell.

   > [!NOTE]
   > Соединитель Skype для бизнеса Online сейчас входит в состав последнего модуля Teams PowerShell. Если вы используете последний общедоступный выпуск Teams PowerShell, вам не нужно устанавливать соединитель Skype для бизнеса Online. См. [управление Skype для бизнеса Online с помощью PowerShell.](/microsoft-365/enterprise/manage-skype-for-business-online-with-microsoft-365-powershell?preserve-view=true&view=o365-worldwide)

1. Запустите PowerShell в качестве администратора.

2. Установите [модуль Teams PowerShell.](./teams-powershell-install.md)

3. Импорт модуля MicrosoftTeams и вход в Teams в качестве администратора.


```powershell
  # When using Teams PowerShell Module

   Import-Module MicrosoftTeams
   $credential = Get-Credential
   Connect-MicrosoftTeams -Credential $credential
```

4. Используйте [Set-CsTeamsMeetingPolicy,](/powershell/module/skype/set-csteamsmeetingpolicy) чтобы настроить политику собраний Teams для перехода с хранилища Stream на OneDrive для бизнеса и SharePoint.

   ```powershell
   Set-CsTeamsMeetingPolicy -Identity Global -RecordingStorageMode "OneDriveForBusiness"
   ```

> [!Note]
> Если некоторым пользователям назначена политика "на пользователя" или "на пользователя", необходимо настроить эту политику, если вы хотите, чтобы они также хранили записи собраний в OneDrive для бизнеса и SharePoint. Дополнительные сведения см. в [сведениях об управлении политиками собраний в Teams.](meeting-policies-in-teams.md)

## <a name="opt-out-of-onedrive-for-business-and-sharepoint-to-continue-using-stream"></a>Отказ от использования OneDrive для бизнеса и SharePoint для использования Stream

Даже если политика настроена на **Stream,** возможно, она не настроена. Как правило, если политика не настроена, по умолчанию используется параметр **Stream.** Однако если после этого изменения вы захотите отказаться от использования SharePoint или OneDrive для бизнеса,  вам необходимо сбросить политику в **Stream,** чтобы использовать stream по умолчанию.

```PowerShell
Set-CsTeamsMeetingPolicy -Identity Global -RecordingStorageMode "Stream"
```

## <a name="frequently-asked-questions"></a>Вопросы и ответы

**Где будет храниться запись собрания?**

- Для собраний, не в каналах, запись хранится в папке "Записи" верхнего уровня oneDrive для бизнеса, принадлежащее человеку, который запустил запись собрания.  Пример.

  <i>OneDrive для бизнеса</i> / в записи **Записи**

- Для собраний канала запись хранится в библиотеке документации сайта Teams в папке **"Записи".** Пример.

  <i>Название группы — название канала</i> / **Документы** / **Записи**

**Каким образом файлы Stream (например, записи) хранятся в SharePoint/OneDrive? Есть ли у администратора возможность изменять место ее изменению?**

По умолчанию все файлы записи будут перенаходить в учетную запись OneDrive выбранного **пользователя.** Для собраний канала запись всегда идет на сайт SharePoint канала. Администратор не может изменить место хранения записи.

**Как обрабатывать записи бывших сотрудников?**

Так как видео похожи на любые другие файлы в OneDrive для бизнеса и SharePoint, обработка владения и хранения после того, как сотрудник уедет, будет выполняться обычный процесс OneDrive для [бизнеса и SharePoint.]( https://docs.microsoft.com/onedrive/retention-and-deletion#the-onedrive-deletion-process)

**У кого есть разрешения на просмотр записи собрания?**

- Для собраний, не включающим каналы, все приглашенные, за исключением внешних пользователей, будут автоматически получать личную общую ссылку. Внешние пользователи должны быть явно добавлены в общий список организатором собрания или человеком, который запустил запись собрания.

- Для собраний канала разрешения наследуются от списков владельцев и участников канала.

**Как управлять субтитрами?**

Субтитры для записей собраний Teams доступны во время воспроизведения только в том случае, если на момент записи у пользователя была включена транскрибация. Администраторы должны [включить запись]( https://docs.microsoft.com/microsoftteams/cloud-recording#turn-on-or-turn-off-recording-transcription) транскрибации с помощью политики, чтобы обеспечить возможность записи собраний с транскрибцией.

Субтитры помогают создавать инклюзивное содержимое для всех пользователей. Владелец может скрыть субтитры в записи собрания, но расшифровка собрания будет доступна в Teams до тех пор, пока вы не удалите ее. 

Субтитры поддерживаются для записи собраний Teams в течение 60 дней после записи собрания.

Субтитры не полностью поддерживаются при перемещении или копировании записи собрания Teams из исходного расположения в OneDrive для бизнеса или SharePoint.

**Как повлияет на квоту хранилища?**

Файлы записи собраний Teams находятся в OneDrive для бизнеса и SharePoint и включаются в квоту этих служб. См. [квоту SharePoint](/sharepoint/sites/plan-site-maintenance-and-management#quotas) и [квоту OneDrive для бизнеса.](/onedrive/set-default-storage-space)

OneDrive для бизнеса — это дополнительный объем [хранилища](/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits) по сравнению с Stream и более интересным хранилищем с SharePoint.

**Как записать собрание Teams?**

В зависимости от того, где вы имеете доступ к файлу, видео будет опубликовано в видеоиграммщике OneDrive для бизнеса или SharePoint.

**Если вы планируете откат к добавлению в Stream, будут ли существующие видео оставаться в том же и в течение долгого времени?**

Потоковая передача в качестве платформы в ближайшем будущем не будет завершена. Видео, которые в настоящее время находятся в Stream, будут находиться там до начала миграции. После миграции эти видео также будут перенесены в OneDrive для бизнеса или SharePoint. Дополнительные [сведения можно получить в классической миграции Stream.](/stream/streamnew/classic-migration)
