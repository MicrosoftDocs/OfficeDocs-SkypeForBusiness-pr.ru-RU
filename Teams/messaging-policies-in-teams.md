# <a name="manage-messaging-policies-in-teams"></a>Управление политиками обмена сообщениями в Teams

<!--- Add zone marker here--->

Политики сообщений используются для управления тем, какие функции чата и обмена сообщениями в каналах доступны пользователям в Microsoft Teams. Вы можете использовать политику по умолчанию, созданную автоматически, или создать одну или несколько настраиваемых политик сообщений для пользователей в вашей организации. После создания политики ее можно назначить пользователю или группе пользователей в Организации.

По умолчанию создается политика Global (значение по умолчанию для всей организации). Всем пользователям в Организации будет назначена эта политика сообщений по умолчанию. Вы можете внести изменения в эту политику или создать одну или несколько настраиваемых политик и назначить им пользователей. После создания настраиваемой политики можно разрешить или запретить пользователям предоставлять доступ к определенным функциям и назначить ее одному или нескольким пользователям, которым нужны параметры. 

## <a name="change-or-create-a-messaging-policy"></a>Изменение или создание политики обмена сообщениями

Вы можете легко управлять политиками сообщений в центре администрирования Microsoft Teamshttp://admin.teams.microsoft.com) (с помощью учетной записи администратора и выбрав пункт **политики обмена сообщениями** в левой области навигации). Чтобы изменить существующую политику сообщений по умолчанию для Организации, выберите глобальную строку **(по умолчанию** в масштабе организации) и внесите необходимые изменения. Чтобы создать новую политику электронной почты, выберите команду **создать политику**, присвойте новой политике имя и выберите нужные параметры. Когда все будет готово, нажмите кнопку **сохранить** .

Например, предположим, что вы хотите убедиться, что отправленные сообщения не удаляются или не изменяются. Вы можете создать новую настраиваемую политику с именем "сохранять отправленные сообщения" и отключить следующие параметры:

- Владельцы могут удалять отправленные сообщения
- Пользователи могут удалять отправленные сообщения
- Пользователи могут редактировать отправленные сообщения

Затем назначьте политику для пользователей.

> [!NOTE] 
> Пользователю может быть назначено только по одной политике обмена сообщениями за один раз.
 
## <a name="assign-a-messaging-policy-to-a-user"></a>Назначение пользователю политики обмена сообщениями

Если вы создаете собственную политику сообщений, она будет активна только в том случае, если она назначена пользователю. Чтобы назначить пользователю специальную политику, перейдите в центр администрирования Microsoft Teams, в области навигации слева выберите пункт **Пользователи** , а затем выберите пользователя, которому вы хотите назначить политику. На странице пользователя нажмите кнопку **изменить** рядом с пунктом **назначенные политики**. Затем в области **изменить политики пользователей** в разделе **Политика сообщений**выберите политику сообщений из раскрывающегося списка и нажмите кнопку **сохранить**. Вы также можете изменить параметры в списке пользователей. Для этого выберите пользователя, щелкнув слева от отображаемого имени пользователя. Нажмите кнопку **изменить параметры**. Затем в области **изменить параметры** в разделе **Политика сообщений**выберите политику из раскрывающегося списка и нажмите кнопку **сохранить**.

Если вы применяете политику к нескольким пользователям, выберите каждого из них, щелкнув слева от имени пользователя, а затем щелкните **изменить параметры**. На панели **изменить параметры** в разделе **Политика сообщений**выберите политику из раскрывающегося списка и нажмите кнопку **сохранить**.

Вы также можете назначить политику обмена сообщениями для одного или нескольких пользователей, выполнив указанные ниже действия.

1. Перейдите в раздел > **политики сообщений** **центра администрирования Microsoft Teams**.
2. Выберите политику, щелкнув слева от имени политики.
3. Выберите **Управление пользователями**.
4. В области **Управление пользователями** найдите пользователя по отображаемому имени или по имени пользователя, выберите имя и нажмите кнопку **Добавить**. Повторите этот шаг для каждого пользователя, которого вы хотите добавить.
5. Завершив добавление пользователей, нажмите кнопку **сохранить**.

> [!NOTE]
> Вы не можете удалить политику, если ей назначены пользователи. Вы должны сначала назначить другую политику всем пользователям, которым угрожает влияние, а затем удалить исходную политику.

## <a name="messaging-policy-settings"></a>Параметры политики сообщений

Используйте указанные ниже параметры для изменения политики глобальных сообщений или создания новой особой политики.

- **Владельцы могут удалять отправленные сообщения**  Используйте этот параметр, чтобы разрешить владельцам удалять сообщения, отправленные пользователями в чате.
- **Пользователи могут удалять отправленные сообщения** Используйте этот параметр, чтобы разрешить пользователям удалять сообщения, отправленные в чате.
- **Пользователи могут редактировать отправленные сообщения** Используйте этот параметр, чтобы разрешить пользователям редактировать сообщения, которые они отправили в чате.
- **Уведомления о** прочтении Используйте этот параметр, чтобы указать, что уведомления о прочтении управляются пользователями, для всех пользователей и выключены для всех. Параметр по умолчанию управляется пользователем. 
    - **Управляемый пользователем** Это означает, что пользователи смогут решить, должны ли они получать уведомления о прочтении. Параметр по умолчанию в приложении включен. Пользователи могут отключить этот параметр.
    - **Вкл. для всех** Это означает, что у всех пользователей в клиенте есть функция без параметров, чтобы ее отключить. 
    - **Выключено для всех** Это означает, что эта функция отключена, и никто из клиента не имеет уведомлений о прочтении и не может включить ее. 
<a name="bkchat"> </a>

- **Чат**  Включите этот параметр, если вы хотите, чтобы пользователи в организации могли использовать приложение Teams для общения с другими людьми.
- **Использование giphy в беседах**  Если включить этот параметр, пользователи смогут включать Giphy в беседах в чате с другими людьми. GIF — это Интернет-база данных и поисковая система, с помощью которой пользователи могут искать и совместно использовать анимированные GIF-файлы. Каждому Gifу назначается рейтинг контента.
- **Оценка содержимого GIF** 
    - **Без ограничений** Это означает, что пользователи смогут вставлять любые GIF в чате, независимо от их рейтинга.
    - **Средний** уровень  Это означает, что пользователи смогут вставлять Giphy в беседы, но они будут умеренно ограничены на основе содержимого для взрослых.
    - **Strict**  Это означает, что пользователи смогут вставлять Giphy в беседы, но они будут строго запрещены от содержимого для взрослых.
- **Использование мемов в беседах** Если включить этот параметр, пользователи смогут включать мемов в беседах в чате с другими людьми. 
- **Использование наклеек в беседах** Если включить этот параметр, пользователи смогут добавлять наклейки в беседы в чате с другими людьми.
- **Разрешить предварительный просмотр URL-адресов** Этот параметр используется для включения и отключения автоматического просмотра URL-адресов в сообщениях.
- **Разрешить пользователям переводить сообщения** Включите этот параметр, чтобы разрешить пользователям автоматически переводить сообщения Teams на язык, заданный их личными языковыми параметрами для Office 365.
- **Разрешить иммерсивное средство чтения для просмотра сообщений** Включите этот параметр, чтобы разрешить пользователям просматривать сообщения в Microsoft иммерсивное средство чтения. Иммерсивное средство чтения — это инструмент для обучения, который обеспечивает полноэкранный режим чтения для удобства чтения текста.
- **Пользователи могут отправлять <a name="urgent-message"></a> уведомления о приоритетах** Если включить этот параметр, пользователи смогут отправлять сообщения, в которых используются уведомления с приоритетами. Уведомления о приоритетах уведомляют пользователей каждые 2 минуты в течение 20 минут, а также до тех пор, пока сообщение не будет отправлено и прочитано получателем, что максимально вероятное время, по истечении которого оно поступило и обработано в течение срока действия.   [!INCLUDE [pri-message-offer](includes/pri-message-offer.md)]
- **Создание голосовых сообщений** 
    - **Разрешено в беседах и каналах** Это означает, что пользователи могут оставлять голосовые сообщения в обоих разговорах и каналах.
    - **Разрешено только в беседах** Это означает, что пользователи могут оставлять голосовые сообщения в чате, но не в каналах.
    - **Отключено** Это означает, что пользователи не могут создавать голосовые сообщения в чате и каналах.  
- **На мобильных устройствах отображение избранных каналов над последними чатами** Включите этот параметр, чтобы переместить избранные каналы в верхнюю часть экрана мобильного устройства, чтобы пользователь не проходил для их поиска. 
- **Разрешение пользователям удалять пользователей из группового чата** Включите этот параметр, чтобы разрешить пользователю удалять других пользователей из группового чата. Благодаря этой функции вы сможете продолжить чат с небольшим количеством пользователей, не теряя историю чата.
