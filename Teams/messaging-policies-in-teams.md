# <a name="manage-messaging-policies-in-teams"></a>Управление политиками обмена сообщениями в Teams

<!--- Add zone marker here--->

Политик обмена сообщениями используются для управления чата и канала функции обмена сообщениями, которые доступны для пользователей, групп Майкрософт. Можно использовать политику по умолчанию, который создается автоматически, или создание одного или нескольких настраиваемых политик обмена сообщениями для пользователей в вашей организации. После создания политики, можно назначить пользователю или группе пользователей в вашей организации.

По умолчанию создается политика, с именем Global (по умолчанию в масштабах организации). Все пользователи в вашей организации будет назначен этой политики обмена сообщениями по умолчанию. Можно внести изменения в этой политике или создание одного или нескольких настраиваемых политик и назначение пользователям. При создании пользовательскую политику, можно разрешить или запретить определенные функциональные возможности доступными для пользователей и назначьте его одному или нескольким пользователям, у которых потребуется параметров, примененных к ним. 

## <a name="change-or-create-a-messaging-policy"></a>Изменение или создание политики обмена сообщениями

Можно легко управлять политик обмена сообщениями в центре администрирования группами Майкрософт (http://admin.teams.microsoft.com) , вход с использованием учетных данных администратора и выбрав **политики сообщений** в левой области навигации. Чтобы изменить существующее значение по умолчанию, система обмена сообщениями политики для вашей организации, выберите строку **Global (по умолчанию в масштабах организации)** и внесите необходимые изменения. Для создания нового настраиваемого политики обмена сообщениями, выберите **Новая политика**, задайте имя новой политики и выберите параметры. Закончив, нажмите кнопку **Сохранить** .

Например предположим, что вы хотели бы убедиться, отправки сообщений не удалены или изменены. Вам нужно создать новый пользовательскую политику с именем «Сохранение отправленных сообщений» и отключить следующие параметры:

- Владельцы сайтов можно удалить отправленных сообщений
- Пользователи могут удалять отправленных сообщений
- Пользователи могут редактировать отправленные сообщения

Назначьте политику для пользователей.

> [!NOTE] 
> Пользователя можно назначить только один политики обмена сообщениями за раз.
 
## <a name="assign-a-messaging-policy-to-a-user"></a>Назначение политики обмена сообщениями для пользователя

При создании настраиваемой политики обмена сообщениями, он будет только active для пользователя, если пользователю назначена политика. Чтобы назначить пользовательскую политику для пользователя, перейдите в центр администрирования группами Майкрософт, выберите **пользователей** в левой панели переходов и выберите пользователя, необходимо назначить политику. На странице пользователя и выберите **Изменить** рядом с пунктом **назначения политик**. В области **Изменение политик пользователей** в разделе **политика системы обмена сообщениями**, выберите политика обмена сообщениями из раскрывающегося списка и выберите команду **Сохранить**. Кроме того, можно изменить параметры в списке пользователей. Для этого выберите пользователя, нажав кнопку слева от отображаемое имя пользователя. Выберите **Изменить параметры**. Затем в области **Изменение параметров** в разделе **политика системы обмена сообщениями**, выберите параметр из раскрывающегося списка и выберите команду **Сохранить**.

В случае применения политики более чем одному пользователю, выберите каждого из пользователей, щелкнув слева от имени пользователя и выберите **Изменить параметры**. В области **Изменить параметры** в разделе **политика системы обмена сообщениями**выберите параметр из раскрывающегося списка и выберите команду **Сохранить**.

Можно также назначить политики обмена сообщениями для одного или нескольких пользователей следующим образом:

1. Перейдите в **Центр администрирования группами Майкрософт** > **политик системы обмена сообщениями**.
2. Выберите политику, нажав кнопку слева от имени политики.
3. Выберите **Управление пользователями**.
4. В области **Управление пользователями** поиска для пользователя по отображаемому имени или по имени пользователя, выберите имя и выберите команду **Добавить**. Повторите этот шаг для каждого пользователя, который вы хотите добавить.
5. Завершив добавление пользователей, выберите команду **Сохранить**.

> [!NOTE]
> Не удается удалить политику, если пользователи назначаются. Сначала необходимо назначить разные политики для всех заинтересованных пользователей и удалите исходное политики.

## <a name="messaging-policy-settings"></a>Параметры политики обмена сообщениями

Изменение глобальной политики обмена сообщениями или создать новую политику настраиваемых используйте следующие параметры:

- **Владельцы сайтов можно удалить отправленных сообщений**  Используйте этот параметр для позволить владельцам удалять сообщения, которое пользователи, отправленных в чата.
- **Пользователи могут удалять отправленных сообщений** Используйте этот параметр, чтобы дать возможность пользователям удалять сообщения, которые они отправлены в чата.
- **Пользователи могут редактировать отправленные сообщения** Используйте этот параметр, чтобы дать возможность пользователям изменение сообщений, которые они отправлены в чата.
- **Уведомления о прочтении** Используйте этот параметр для указания ли уведомления о прочтении, пользователь управляет, включен для всех пользователей или отключен.
<a name="bkchat"> </a>

- **Чата**  Включите этот параметр если пользователи в вашей организации должны иметь возможность использовать приложения группы для разговора с другими пользователями.
- **Использование Giphys в беседах**  При включении этого, пользователи могут включать Giphys в беседах разговор с другими людьми. Giphy является online базы данных и модуль поиска, который позволяет пользователям выполнять поиск и совместное использование анимационной файлы в формате GIF. Каждый Giphy назначается оценки содержимого.
- **Оценки содержимого Giphy** 
    - **Без ограничений** Это означает, что пользователи смогут вставить любой Giphy в беседах вне зависимости от оценки содержимого.
    - **Контроль**  Это означает, что пользователи будут иметь возможность вставить Giphys в беседах, но будет средней полномочий на содержимое для взрослых.
    - **Строгое**  Это означает, что пользователи будут иметь возможность вставить Giphys в беседах, но будет строго полномочий на содержимое для взрослых.
- **Использование Memes в беседах** При включении этого, пользователи могут включать Memes в беседах разговор с другими людьми. 
- **Использование наклейки в беседах** При включении этого, пользователи могут включать наклейки в беседах разговор с другими людьми.
- **Предварительный просмотр разрешить URL-адрес** Используйте этот параметр, чтобы включить автоматическое URL-адрес Предварительный просмотр включено или отключено в сообщениях.
- **Разрешить пользователям включать поддержку для перевода сообщений** Включите этот параметр позволяет пользователям автоматически преобразовывать сообщения команды на язык, указанный с личных языковых параметров для Office 365.
- **Разрешить насыщенные чтения для просмотра сообщений** Включите этот параметр вход в систему позволяет пользователям просмотр сообщений в Microsoft насыщенные чтения. Насыщенные читатель — это средство обучения, который предоставляет полноэкранный режим возможности для улучшения читаемости текста для чтения.
- **Пользователи могут отправлять уведомления о приоритет** Если включить это, пользователи могут отправлять сообщения, использующего приоритет уведомлений. Приоритет уведомлений уведомить пользователей об несколько раз в течение 20 минут или пока не будет обработан сообщений и читать получателю, максимальной вероятность, что сообщение обработан и действие своевременно.
- **Создание сообщения голосовой связи** 
    - **Разрешенный в беседах и каналы** Это означает, что пользователи могут оставлять сообщения голосовой связи в беседах и каналы.
    - **Разрешенный в только обсуждений** Это означает, что пользователь оставить голосовых сообщений в беседах, но не в каналы.
    - **Отключено** Это означает, что пользователи не могут создать голосовых сообщений в чаты или каналы.  
- **На мобильных устройствах отображения избранных каналы над последних сеансов** Включите этот параметр переместить избранные каналы в верхней части экрана мобильного устройства, чтобы пользователь не требуется прокрутка их поиск. 
- **Разрешить пользователям удалить пользователей из группы чата** Включите этот параметр предоставить пользователям возможность удалить других пользователей из группы чата. Эта функция позволяет продолжить разговор с небольшим группой людей, не потерять журнал чата.
