### <a name="step-1-prepare-your-microsoft-365-tenant-for-career-coach"></a>Шаг 1. Подготовка клиента Майкрософт 365 для карьерного тренера

#### <a name="turn-on-linkedin-account-connections"></a>Включение подключений к учетным записям LinkedIn

Существует два способа использования подключений учетных записей LinkedIn карьерным тренером:

- Чтобы разрешить учащимся, преподавателям и сотрудникам подключать свою учетную запись Майкрософт 365 к учетной записи LinkedIn и [отправлять приглашения LinkedIn от Карьерного тренера](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_connect_your_linkedin_account).
- Включение функций ["Создание сети](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_build_your_network) " и ["Изучение карьеры" с помощью общедоступных данных выпускников](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_discover_your_career_path) из LinkedIn.

Чтобы включить подключения к учетной записи LinkedIn, выполните приведенные ниже действия.

1. Войдите в [Центр администрирования Azure AD](https://go.microsoft.com/fwlink/p/?linkid=2067268) с [учетной записью глобального администратора](#identify-roles-and-permissions) для Azure AD организации.

2. Выберите **Пользователи**.

3. На странице **Пользователи** выберите **Параметры пользователей**.

4. Для правильной настройки **подключений к учетной записи LinkedIn** необходимо задать **значение Да** или **Выбранная группа**.

   - Выберите **Да** , чтобы включить службу для всех пользователей в вашем учебном заведении.
   - Выберите **Выбранная группа** , чтобы включить службу только для группы выбранных пользователей в вашем учебном заведении.

Дополнительные сведения см. [в статье Подключения учетных записей LinkedIn в Azure Active Directory](/azure/active-directory/enterprise-users/linkedin-integration).

> [!NOTE]
> Данные не передаются без согласия пользователя.

#### <a name="allow-microsoft-apps-in-teams"></a>Разрешить приложения Майкрософт в Teams

Карьерный тренер — это приложение Майкрософт, поэтому оно должно быть разрешено в политиках разрешений приложений Teams.

1. Войдите в **[Центр администрирования Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851)**.

2. В области навигации слева выберите **Приложения** >  Teams **[Политики разрешений](https://admin.teams.microsoft.com/policies/manage-apps)**.

3. Выберите предпочитаемую политику.
    1. Если вы не знаете, какую политику следует использовать, ознакомьтесь с документацией  по  [управлению политикой Teams Майкрософт](policy-packages-edu.md)или используйте [мастер](easy-policy-setup-edu.md) политики для образования, чтобы настроить политику для Майкрософт Teams.

Дополнительные сведения о [настройке политик разрешений](teams-app-permission-policies.md).

#### <a name="add-career-coach-as-an-installed-app"></a>Добавление Карьерного тренера в качестве установленного приложения

Этот шаг гарантирует, что карьерный тренер правильно настроен для вашего учебного заведения, и учащиеся могут найти карьеру тренера.

1. Войдите в [Центр администрирования Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851).

2. Выберите Политики **установки**  **приложений** > Teams и выберите предпочитаемую политику.
    1. Если вы не знаете, какую политику следует использовать, ознакомьтесь с документацией  по  [управлению политикой Teams Майкрософт](policy-packages-edu.md)или используйте [мастер](easy-policy-setup-edu.md) политики для образования, чтобы настроить политику для Майкрософт Teams.

3. В разделе **Установленные приложения** выберите  **+ Добавить приложения**.

4. В области **Добавление установленных приложений** найдите приложения, которые вы хотите автоматически установить для пользователей при запуске Teams. Вы также можете фильтровать приложения с помощью политики разрешений приложений.

5. Выбрав список приложений, нажмите **кнопку Добавить**.

6. Выберите  **Save** Confirm (Сохранить  > **подтвердить**).

Изменение или назначение политики может занять несколько часов, чтобы изменения вступили в силу. Карьерный тренер не будет доступен в Майкрософт Teams до тех пор, пока изменения не будут завершены.

### <a name="step-2-purchase-career-coach-licenses"></a>Шаг 2. Приобретение лицензий Карьерного тренера

#### <a name="license-types"></a>Типы лицензий

Для доступа к приложению для карьерного тренера требуется лицензия.

Существует два типа лицензий.

- **Студенческая лицензия** предназначена для учащихся.
- **Лицензия на факультет** предназначена для преподавателей, сотрудников и ИТ-специалистов, участвующих в поддержке студентов с помощью карьерного тренера.
  - Ит-администратору, завершив настройку, необходимо назначить **лицензию на карьерный** тренер.

#### <a name="purchase-licenses"></a>Приобретение лицензий

Карьерный тренер доступен по всему миру (за исключением Китая и России) для образовательных учреждений в качестве дополнительной лицензии через регистрацию для образовательных решений (EES), поставщиков облачных услуг (CSP) и Центр администрирования Microsoft 365 (через Интернет).

Как приложение Майкрософт Teams, клиент должен иметь Microsoft 365 A3/A5 или Office 365 A1/A3/A5, чтобы приобрести лицензию на надстройку Career Coach. Для учащихся и преподавателей предлагаются отдельные лицензии.

#### <a name="sign-up-for-a-free-trial"></a>Зарегистрируйтесь для получения бесплатной пробной версии

Стандартная 90-дневная бесплатная пробная версия доступна для 25 студентов и 25 лицензий преподавателей. Для каждого соответствующего клиента доступна одна пробная версия. Пробные лицензии могут быть активированы с Центр администрирования Microsoft 365 арендаторами, имеющими право на приобретение лицензий Career Coach.

Для активации пробной версии войдите в [Центр администрирования Microsoft 365 > выставление счетов > покупки служб > поиск по запросу Карьерный тренер](https://go.microsoft.com/fwlink/p/?linkid=868433), чтобы найти [пробную версию лицензии учащегося](https://signup.microsoft.com/signup?OfferId=b3a40ff2-3d0d-481e-a0ed-f4de1069f201) и [пробную версию лицензии преподавателя](https://signup.microsoft.com/signup?OfferId=6f6e7db5-b9ab-4baa-86be-f13d0ae6a2c8).

Этот шаг можно выполнить, только если [подключения к учетной записи LinkedIn включены в Azure Active Directory](#turn-on-linkedin-account-connections). Параметр подключения LinkedIn включает функции [Создание сети](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_build_your_network) и [Изучение карьеры](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_discover_your_career_path) .

> [!IMPORTANT]
> Это обязательный раздел. Карьерный тренер не может быть включен без проверки подключения к странице LinkedIn School.
>
> Подключение LinkedIn поддерживает несколько клиентов.

##### <a name="add-the-linkedin-school-page-url"></a>Добавление URL-адреса страницы LinkedIn school

Процесс добавления URL-адреса учебной страницы LinkedIn обрабатывается администратором Teams. Последующий шаг проверки URL-адреса выполняется администратором LinkedIn School Page вашего учебного заведения.

1. Войдите в **[Центр администрирования Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851)**.

2. Выберите **Приложения** >  Teams **Управление приложениями** > **Параметры карьеры тренера** >  > **Подключение LinkedIn**.

3. Выберите **Подключиться к LinkedIn**.

4. Найдите страницу учебного заведения LinkedIn, выполнив поиск в LinkedIn и выбрав фильтр **Учебное заведение** . Или свяжитесь с сотрудником отдела маркетинга в вашем учреждении, чтобы определить правильную страницу LinkedIn School для использования. Дополнительные сведения см. в статье [Определение страниц LinkedIn](https://www.linkedin.com/help/linkedin/answer/40133/differences-between-a-linkedin-page-for-a-school-and-company?lang=en).

    ![Снимок экрана: поиск учебного заведения в LinkedIn.](media/career-coach-school-search-updated.png)

5. Добавьте **URL-адрес страницы LinkedIn school**. URL-адрес должен быть *учебной,* а не *корпоративной страницей* и обычно имеет `https://www.linkedin.com/school/willow-university/`формат .

   ![Снимок экрана: пример страницы LinkedIn School.](media/career-coach-linkedin-page-url-updated.png)

6. Выберите **Отправить**.

7. После успешной отправки **URL-адреса страницы LinkedIn School** страница параметров обновится, чтобы отобразить  **ссылку** проверка и  **срок действия ссылки проверки**. Срок действия ссылки для проверки истекает через 30 дней.

8. Скопируйте **ссылку Проверка** и поделитесь ею с суперадминистратором LinkedIn School Page вашего учебного заведения.

9. Администратор LinkedIn School Page Super будет использовать уникальную ссылку для проверки [страницы LinkedIn School](#verify-the-linkedin-school-page) и связать ее с Карьерным тренером.

10. Чтобы убедиться, что проверка и параметры успешно отправлены, убедитесь, что [состояние конфигурации Карьерного тренера](#configuration-status) помечено как **Завершено**.

##### <a name="verify-the-linkedin-school-page"></a>Проверка страницы LinkedIn School

Проверка страницы LinkedIn School должна быть выполнена [администратором LinkedIn School Page вашего учебного заведения суперадминистратором.](https://www.linkedin.com/help/linkedin/answer/a541981)

Вы можете проверить роли администратора учетной записи LinkedIn, войдя в LinkedIn и перейдя на страницу учебного заведения LinkedIn School. Если вашей учетной записи назначена роль суперадминистраторов, вы увидите **представление суперадминистратор** рядом с именем вашего учебного заведения на странице LinkedIn. Если вы не видите тег **представления суперадминистраторов** , вы не являетесь суперадминистратором для страницы вашего учебного заведения.

1. После отправки **URL-адреса страницы LinkedIn school** администратором Teams на странице будут отображаться **ссылка Проверка** и **Срок действия ссылки проверки**. Срок действия ссылки для проверки истекает через 30 дней.

   ![Снимок экрана: подключения LinkedIn для приложения карьерного тренера.](media/career-coach-linkedin-updated.png)  

2. Скопируйте ссылку для проверки и поделитесь ею с суперадминистратором LinkedIn School Page.

3. На странице LinkedIn Суперадминистратор откроет ссылку для проверки, чтобы связать карьерного тренера со страницей вашего учебного заведения. Дополнительные сведения см. в [статье Дополнительная документация по проверке страницы LinkedIn](https://www.linkedin.com/help/linkedin/answer/102672).

4. После завершения проверки администратор Teams может просмотреть, успешно ли отправлены параметры, проверив, имеет ли [состояние конфигурации Карьерного тренера](#configuration-status) значение **Завершено**.

   ![Снимок экрана: проверка страницы LinkedIn на портале разработчика LinkedIn.](media/career-coach-linkedin-verification-updated.png)

#### <a name="brand-and-preferences-optional"></a>Торговая марка и параметры (необязательно)

Настройка карьерного тренера в соответствии с брендом вашего образовательного учреждения. Вы несете ответственность за соблюдение прав других лиц, включая авторские права и права на товарные знаки.

1. Войдите в **[Центр администрирования Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851)**.

2. Выберите **Приложения** >  Teams **Управление приложениями** > **Параметры карьеры****тренера** >  > **Изменение фирменного стиля и настроек**.

3. В разделе **Фирменная символика** добавьте **название организации**.

4. Отправьте **значок "Организация**". Этот значок используется в карьерном тренере для идентификации содержимого, уникального для вашего учебного заведения, ресурсов каталога курсов в приложении и в разделе реальных возможностей на панели мониторинга.

    Значок лучше всего отформатировать следующим образом:

    - Прозрачный PNG
    - Пропорции 1:1
    - Максимальный размер 64 пикселей x 64 пикселей

5. Отправьте **изображение эскиза содержимого обучения**. Эскиз будет использоваться для учебных ресурсов каталога курсов в приложении, если определенное изображение не указано для курса, предлагаемого вашим учебным заведением.

    Эскиз лучше всего отформатировать следующим образом:

    - A PNG
    - Пропорции 16:9
    - Максимальный размер 360 пикселей x 200 пикселей

6. Это необязательный шаг. Добавьте **URL-адрес политики конфиденциальности организации**. Если это будет добавлено, политика конфиденциальности учебного заведения будет доступна для ознакомления учащихся в разделе Карьера тренера.

7. Выберите **Отправить**.

8. Чтобы убедиться, что параметры успешно отправлены, установите флажок [Состояние конфигурации Карьерного тренера](#configuration-status) для **параметра Завершено**.

#### <a name="course-catalog-optional"></a>Каталог курсов (необязательно)

В каталоге курсов записываются курсы и классы, предлагаемые вашим учебным заведением.

Карьерный тренер использует данные каталога курсов, чтобы определить навыки учащегося из стенограммы и предложить курсы для прохождения.

Эти курсы используются в карьерном тренере в двух областях:

- Курсы и метаданные курса, такие как названия и описания курсов, используются, чтобы помочь учащимся определить свои навыки при [отправке расшифровки](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708#bkmk_Set_up_your_profile).

Чтобы создать каталог курсов, составьте список всех курсов, преподаваемых в вашем учебном заведении, и отправьте его в виде CSV-файла, используя [формат документов и схему каталога курсов](#course-catalog-document-format-and-schema).

Начните с [примера документа каталога](https://aka.ms/career-coach/docs/it-admins/sample-catalog) курсов, чтобы обеспечить правильное форматирование.Дополнительные сведения об обязательных и рекомендуемых полях можно также получить в разделе [формата документа каталога курса и схеме](#course-catalog-document-format-and-schema) .

##### <a name="add-the-course-catalog"></a>Добавление каталога курсов

3. Выберите **Отправить каталог курсов** > Отправить курсы в формате CSV с обязательными столбцами: courseId, title и sourceLink.
    1. Каждая строка должна содержать данные для каждого из обязательных столбцов.
    1. *Включение рекомендуемых полей улучшает взаимодействие с учащимися, возвращая лучшие результаты поиска и идентификацию навыков.*

4. Для справки появится раздел загруженного каталога курсов.

5. Выберите **Отправить** , когда все будет готово.

6. [Состояние отправленного документа](#course-catalog-status) отображается на странице параметров.

7. Чтобы убедиться, что параметры успешно отправлены, убедитесь, что [состояние конфигурации Карьерного тренера](#configuration-status) помечено как **Завершено**.

Чтобы отправить новый файл, убедитесь, что последний файл очищен, выбрав (**X**), чтобы закрыть предварительный просмотр документа. Это действие позволит снова отобразить кнопку **Отправить** .

##### <a name="course-catalog-document-format-and-schema"></a>Формат документа и схема каталога курсов

Документ должен быть в формате CSV с максимальным размером 18 МБ. Большие файлы должны быть разделены на несколько небольших файлов, которые содержат не более 15 000 строк для успешной обработки.

Документ должен содержать необходимые метаданные курса: **название курса**,  **идентификатор курса** и  **URL-адрес курса**.

Начните с [примера документа каталога курсов]( https://aka.ms/career-coach/docs/it-admins/sample-catalog) , чтобы обеспечить правильное форматирование. *Включение рекомендуемых полей улучшает взаимодействие с учащимися, возвращая лучшие результаты поиска и идентификацию навыков.*

В следующей таблице показаны элементы, которые необходимо включить в каталог курсов.

| Имя             | Статус      | Тип   | Описание                                                                    |
|------------------|-------------|--------|--------------------------------------------------------------------------------|
| courseId         | Обязательно    | Строка | CourseId сопоставляется с тем, что создается в расшифровке учащегося.             |
| Название            | Обязательно    | Строка | Название курса.                                                              |
| sourceLink       | Обязательно    | URL    | Ссылка на веб-сайт на страницу курса, чтобы узнать больше о курсе и программе.   |
| Описание      | Рекомендуется | Строка | Вводный текст для курса, в котором описываются цели обучения.       |
| language         | Рекомендуется | Строка | Язык курса. Используйте стандартные коды языков.                           |
| Формат           | Рекомендуется | Строка | Режим обучения (онлайн, видео, лично).                                   |
| thumbnailLink    | Рекомендуется | URL    | Ссылка на эскиз на изображение курса.                                            |
| thumbnailAltText | Рекомендуется | Строка | Замещающий текст для изображения со специальными возможностями                                           |
| educationLevel   | Рекомендуется | Строка | Уровень обучения, например Бакалавр или выпускник.                                       |
| Темы           | Рекомендуется | Строка | Темы или теги, связанные с навыками, которые преподают курсы.          |

##### <a name="course-catalog-status"></a>Состояние каталога курсов

Состояние каталога курсов отображается на странице Параметры каталога курсов после отправки документа с подробными сведениями о состоянии отправки и обработки документа.

Во время обработки карьерный тренер проанализирует документ на наличие дубликатов, нормализует и обогащает каталог, извлекая навыки из заголовков и описаний, а также сохранит его для использования на странице Learn и во время отправки расшифровки учащихся для идентификации навыка.

![Снимок экрана: состояние отправки каталога курсов для приложения карьерного тренера.](media/career-coach-course-catalog-status-updated.png)

| Столбец           | Значение     | Описание                                                                                        |
| ---------------- | --------- | -------------------------------------------------------------------------------------------------- |
| Время отправки    | Timestamp | Дата и время отправки документа ИТ-администратором.                                                     |
| Время завершения   | Timestamp | Дата и время полной обработки документа.                                               |
| Отправленные курсы | Целое число   | Количество курсов, найденных в документе.                                                           |
| Состояние приема | Pending   | Документ в очереди для обработки.                                                                  |
| Состояние приема | Запущена   | Документ в настоящее время обрабатывается. Этот процесс может занять до 6 часов в зависимости от размера документа. |
| Состояние приема | Success (Успешно)   | Процесс приема завершен, и курсы будут доступны в карьерном тренере после настройки всех необходимых параметров. |
| Состояние приема | Сбой при    | Проверьте формат документа и перезагрузите его.                                                           |
| Дубликаты       | Целое число   | Количество повторяющихся курсов, найденных в документе.                                                 |

Если столбец в каталоге курса пуст, документ в настоящее время обрабатывается, а эти значения недоступны. Этот процесс может занять до 6 часов в зависимости от размера каталога. После обработки документа будут заполнены значения. Вы можете обновить страницу, чтобы проверить наличие обновлений.

#### <a name="fields-of-study-optional"></a>Области обучения (необязательно)

Области обучения являются синонимами основных областей интересов, академических специальностей и дипломных программ. На эти области обучения ссылаются учащиеся, когда они начинают использовать карьерный тренер и начинают настраивать свой персонализированный профиль.

Список полей позволяет учащимся находить области обучения, которые могут их заинтересовать, и добавить их запланированный академический фокус в свой профиль в карьере тренера.

> [!IMPORTANT]
> Если вы настраиваете поля обучения для карьерного тренера, добавьте все области обучения, доступные для учащихся, такие как инженерный, английский язык, бизнес и т. д.

3. Выберите **Отправить** , чтобы отправить поля учебного файла в формате CSV. Появится предварительный просмотр полей учебного документа.

4. Выберите **Отправить**.

5. Чтобы убедиться, что параметры успешно отправлены, убедитесь, что [состояние конфигурации Карьерного тренера](#configuration-status) помечено как **Завершено**.

##### <a name="fields-of-study-document-format-and-schema"></a>Поля формата и схемы документа исследования

## <a name="step-5-make-career-coach-available-to-your-institution"></a>Шаг 5. Предоставление вашему учебному заведению доступа к карьере тренера

На данный момент карьерный тренер настроен для вашего учреждения.

Затем выполните следующие действия, чтобы обеспечить доступность карьерного тренера в вашем учреждении в Майкрософт Teams.

### <a name="assign-career-coach-licenses-to-users"></a>Назначение пользователям лицензий Карьерного тренера

Пошаговые инструкции см. в статье [Назначение лицензий пользователям](/microsoft-365/admin/manage/assign-licenses-to-users).

### <a name="set-up-policies-and-pin-the-app"></a>Настройка политик и закрепление приложения

Закрепление карьерного тренера добавит его на левую границу окна Майкрософт Teams, чтобы сделать его более доступным и видимым для учащихся. Если вы предпочитаете закрепить Карьерный тренер для подмножества пользователей, вам потребуется реализовать [политику установки](teams-app-setup-policies.md) с включенной группой.

1. Войдите в **[Центр администрирования Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851)**.

2. Выберите **Приложения** >  Teams **. Политики установки** и выберите предпочитаемую политику.

    Если вы не знаете, какую политику следует использовать, ознакомьтесь с [документацией по управлению политикой Teams Майкрософт](policy-packages-edu.md) или с помощью [мастера политики образования](easy-policy-setup-edu.md) настройте политику для Майкрософт Teams.

3. В разделе **Закрепленные приложения** выберите **Добавить приложения**.

4. Выберите предпочитаемую политику в разделе **Поиск на основе этой политики разрешений приложения**.

5. Выполните поиск по **запросу Карьерный тренер** **в разделе Поиск по имени**, а затем выберите  **Добавить** > **добавить,** чтобы закрыть панель.

6. Выберите заказ, в который будет отображаться приложение, и нажмите **кнопку Подтвердить**.

Учащиеся будут уведомлены в Майкрософт Teams о том, что карьерный тренер был закреплен.
