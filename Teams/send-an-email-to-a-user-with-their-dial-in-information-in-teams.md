# <a name="send-an-email-to-a-user-with-their-audio-conferencing-information-in-microsoft-teams"></a>Отправка пользователю письма с данными для аудиоконференций в Microsoft Teams

Иногда пользователям Microsoft Teams может потребоваться отправить им свои сведения о аудиоконференции. Это можно сделать, нажав кнопку "Отправить сведения о конференции **" по электронной** почте в разделе свойств пользователя. При отправке этого сообщения электронной почты он будет содержать все сведения о аудиоконференции, включая:
  
- Телефонный номер конференции или номер телефонного подключения для пользователя.

- Идентификатор конференции пользователя.

Ниже приведен пример отправленного сообщения электронной почты.

## <a name="send-an-email-with-audio-conferencing-information-to-a-user"></a>Отправка пользователю сообщения электронной почты с информацией для участия в аудиоконференциях

### <a name="sending-an-email-using-the-microsoft-teams-admin-center"></a>Отправка сообщения электронной почты с помощью Центра администрирования Microsoft Teams

1. В области навигации слева **щелкните "** Пользователи", а затем выберите пользователя из списка доступных пользователей.

2. В верхней части страницы нажмите кнопку "Изменить **"**.

3. В разделе **Аудиоконференции** щелкните **Отправить информацию для конференций в электронном письме**.

## <a name="what-else-should-you-know-about-this-email"></a>Что еще необходимо знать о таких сообщениях?

- После включения аудиоконференций пользователям в организации отправляется несколько сообщений электронной почты:

  - При назначении пользователям лицензии **аудиоконференций**.

  - При сбросе ПИН-кода аудиоконференций пользователя вручную.

  - Выполнение сброса идентификатора конференции пользователя вручную.

  - При изъятии лицензии **аудиоконференций** у пользователя.

  - При изменении поставщика аудиоконференций для пользователя с Microsoft на другого поставщика или **none**.

  - При изменении поставщика аудиоконференций для пользователя на Microsoft.
