# <a name="reset-the-audio-conferencing-pin-in-microsoft-teams"></a>Сброс ПИН-кода аудиоконференций в Microsoft Teams

ПИН-код — это код, который состоит из чисел, созданных для каждого пользователя Microsoft Teams, которому разрешена аудиоконференции. ПИН-коды аудиоконференций используют организаторы собраний для подтверждения статуса организатора собрания и запуска собрания по телефону. Если они используют приложение Microsoft Teams для начала собрания, ПИН-код не требуется. Если пользователь забыл свой ПИН-код, и ему не удается найти сообщение электронной почты, отправленное ему при предоставлении возможности участия в аудиоконференциях, администратор может выполнить сброс его ПИН-кода, или же он может изменить свой ПИН-код самостоятельно.
  
Собрания можно начать, когда прошедший проверку подлинности пользователь присоединяется с помощью приложения Microsoft Teams или когда организатор присоединяется к ПИН-коду по телефону. Когда для начала собрания требуется ПИН-код, пользователи, которые присоединяются по телефону, помещаются в зал ожидания и будут прослушивать музыку на удержании, пока организатор не допустит их. Если организатор собрания не запрашивает ПИН-код при открытии собрания по телефону, то при подключении к собранию участникам не потребуется вводить ПИН-код.

## <a name="reset-a-users-pin"></a>Сброс ПИН-кода пользователя

С помощью Центра администрирования Microsoft Teams:

1. В области навигации слева **щелкните "** Пользователи", а затем выберите пользователя из списка доступных пользователей.

2. Нажмите **кнопку "Изменить"**.

3. В **разделе "Аудиоконференции" нажмите** кнопку **"Сбросить ПИН-код"**.

4. Нажмите **кнопку "Сбросить"**.

> [!INCLUDE [updating-admin-interfaces](includes/updating-admin-interfaces.md)]

## <a name="have-a-user-reset-their-own-pin"></a>Сброс собственного ПИН-кода пользователем

1. Пользователь должен перейти к [https://dialin.teams.microsoft.com/usp](https://dialin.teams.microsoft.com/usp).
2. Нажмите **кнопку "Сбросить ПИН-код"**.

> [!NOTE]
> Для GCCH перейдите к: [https://dialin.cpc.gov.teams.microsoft.us/usp](https://dialin.cpc.gov.teams.microsoft.us/usp).
> Для DoD перейдите к: [https://dialin.cpc.dod.teams.microsoft.us/usp](https://dialin.cpc.dod.teams.microsoft.us/usp).

> [!NOTE]
> Сообщение электронной почты для сброса ПИН-кода не будет отправлено пользователю при использовании этого метода.

## <a name="what-else-should-you-know-about-pins"></a>Что еще необходимо знать о ПИН-кодах?

- В целях безопасности ПИН-код отображается администратору только один раз при сбросе ПИН-кода. После сброса ПИН-кода администратором ПИН-код будет указан как **********.

- Автоматическая отправка сообщений электронной почты пользователям включена по умолчанию, и пользователи получат сообщение электронной почты со своим ПИН-кодом, если они включены для аудиоконференций или при сбросе ПИН-кода. Но если вы отключите автоматическую отправку сообщений электронной почты, пользователю не будет отправлено сообщение о сбросе ПИН-кода, и вам придется вручную отправить пользователю сведения о ПИН-коде.

- При запуске собрания организатор должен принять всех пользователей ТСОП в "зале ожидания", чтобы присоединиться к собранию. Например, если два участника ТСОП попытаются присоединиться к собранию перед его началом, они помещаются в "зал ожидания" и будут прослушивать музыку на удержании, а когда организатор собрания присоединяется с помощью ПИН-кода по телефону, собрание начнется, и организатор сможет использовать команду в собрании (нажмите клавишу *21), чтобы принять всех пользователей ТСОП в зале ожидания.

- По умолчанию анонимные абоненты не могут начать собрание.

- При включении аудиоконференций пользователю по умолчанию отправляются сообщения электронной почты с информацией о конференции и ПИН-кодом. У пользователя должен быть почтовый ящик Microsoft 365 или Office 365, так как при сбросе ПИН-кода новый ПИН-код будет отправлен пользователю по электронной почте на основной SMTP-адрес (псевдоним), задаваемый для пользователя.

- При настройке аудиоконференций вы задаете цифры, которые должны быть включены в ПИН-коды в вашей организации. Пин-коды могут быть от 4 до 12 цифр. Значение по умолчанию — 5. При изменении длины ПИН-кода новое значение применяется только к новым ПИН-кодам и не применяется к ПИН-кодам для существующих пользователей, которым разрешено участие в аудиоконференциях. См. [Настройка длины PIN-кода для собраний с аудиоконференцией](Set-the-PIN-length-for-Audio-Conferencing-meetings-in-teams.md).

- По умолчанию для сообщения электронной почты будет задано значение Microsoft 365 Office 365 основной SMTP-адрес пользователя. Вы можете отправить сообщение электронной почты на адрес электронной почты, отличный от Microsoft 365 или Office 365, например адрес электронной почты Hotmail или MSN. Используемый по умолчанию адрес электронной почты можно изменить с помощью Windows PowerShell. Это полезно, если у пользователей нет почтового ящика Exchange в Microsoft 365 или Office 365.
