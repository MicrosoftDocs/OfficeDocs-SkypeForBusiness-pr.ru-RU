# <a name="reset-the-audio-conferencing-pin-in-microsoft-teams"></a>Сброс ПИН-кода аудиоконференций в Microsoft Teams

ПИН-код — это код, который состоит из чисел, которые создаются для каждого пользователя Microsoft Teams, который может включить аудиоконференцию. ПИН-коды аудиоконференций используют организаторы собраний для подтверждения статуса организатора собрания и запуска собрания по телефону. Если собрание начинается Microsoft Teams приложением, ПИН-код не требуется. Если пользователь забыл свой ПИН-код, и ему не удается найти сообщение электронной почты, отправленное ему при предоставлении возможности участия в аудиоконференциях, администратор может выполнить сброс его ПИН-кода, или же он может изменить свой ПИН-код самостоятельно.
  
Собрания можно начать, когда пользователь, авторификация, присоединяется с помощью Microsoft Teams или когда организатор присоединяется с помощью ПИН-кода по телефону. Когда для начала собрания требуется ПИН-код, пользователи, присоединились к собранию по телефону, помещаются в "lobby" и будут прослушивать музыку на удержании, пока организатор не допустит их. Если организатор собрания не запрашивает ПИН-код при открытии собрания по телефону, то при подключении к собранию участникам не потребуется вводить ПИН-код.

## <a name="reset-a-users-pin"></a>Сброс ПИН-кода пользователя

![Значок с логотипом Microsoft Teams](media/teams-logo-30x30.png) **Использование центра администрирования Microsoft Teams**

1. В левой области навигации щелкните **Пользователи** и выберите пользователя из списка доступных пользователей.

2. Нажмите **кнопку Изменить**.

3. В **области Аудиоконференция** нажмите кнопку **Сбросить ПИН-код.**

4. Нажмите **кнопку Сброс**.

## <a name="what-else-should-you-know-about-pins"></a>Что еще необходимо знать о ПИН-кодах?

- В целях безопасности ПИН-код отображается только один раз администратору при сбросе ПИН-кода. После сброса ПИН-кода администратором ОН будет указан в списке *********.
    
- По умолчанию автоматическое отправка сообщений электронной почты пользователям включено, и пользователи получают электронное письмо со своим ПИН-кодом, когда им будет включена аудиоконференция или при сбросе ПИН-кода. Но если вы отключили автоматическое отправку сообщений электронной почты, пользователю не будет отправлено сообщение об сбросе ПИН-кода, и вам придется вручную отправлять сведения о ПИН-коде пользователю.
    
- Когда собрание начинается, организатор должен допустить всех пользователей STN в " "в"", чтобы присоединиться к собранию. Например, если два участника ДНР пытаются присоединиться к собранию до его начала, они будут помещаются в ""в" и будут прослушивать музыку на удержании, а когда организатор собрания присоединяется с помощью ПИН-кода по телефону, начинается собрание, и организатор может использовать команду "Присоединиться к собранию" (нажмите *21), чтобы допустить всех пользователей ДНР в ".
    
- По умолчанию анонимным звонителям не разрешается начать собрание.
    
- Когда пользователь включает аудиоконференцию, по умолчанию ему отправляются сообщения электронной почты, включающие сведения о аудиоконференции и ПИН-код. У пользователя должен быть почтовый ящик Microsoft 365 или Office 365, так как при сбросе ПИН-кода новый ПИН-код отправляется пользователю по электронной почте на его основной SMTP-адрес (псевдоним), установленный для пользователя.
    
- При настройке аудиоконференций вы задаете цифры, которые должны быть включены в ПИН-коды в вашей организации. ПИН-коды могут быть от 4 до 12 цифр ( значение по умолчанию — 5). При изменении длины ПИН-кода новое значение применяется только к новым ПИН-кодам и не применяется к ПИН-кодам для существующих пользователей, которым разрешено участие в аудиоконференциях. См. [Настройка длины PIN-кода для собраний с аудиоконференцией](Set-the-PIN-length-for-Audio-Conferencing-meetings-in-teams.md).
    
- По умолчанию для сообщения электронной почты Microsoft 365 или Office 365 основным SMTP-адресом пользователя. Вы можете отправлять сообщения электронной почты на Microsoft 365 адрес электронной почты Office 365 hotmail или MSN. Используемый по умолчанию адрес электронной почты можно изменить с помощью Windows PowerShell. Это полезно, если у пользователей нет почтового ящика Exchange в Microsoft 365 или Office 365.
