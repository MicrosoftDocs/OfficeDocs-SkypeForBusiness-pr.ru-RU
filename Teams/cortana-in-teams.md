# <a name="cortana-voice-assistance-in-teams"></a>Голосовая помощь Кортаны в Teams

> [!NOTE]
> Голосовая поддержка Кортаны поддерживается в мобильных приложениях Microsoft Teams для iOS и Android, а microsoft Teams отображается для пользователей в США, Соединенном Королевстве, Канаде, Индии и Австралии. Комнаты Microsoft Teams Windows поддерживается только для устройств с языковым стандартом en-us. В настоящее время голосовая помощь Кортаны недоступна для клиентов GCC, GCC-High, DoD и EDU, отличных от США. Голосовая помощь Кортаны в мобильном приложении Teams теперь доступна для клиентов EDU в сша. Расширение на дополнительные языки и регионы будет происходить в рамках будущих выпусков.

Голосовая помощь Кортаны в мобильном приложении Teams, на Комнаты Microsoft Teams в Windows и на устройствах с отображением Microsoft Teams позволяет Microsoft 365 корпоративный пользователям оптимизировать взаимодействие, совместную работу и задачи, связанные с собраниями, с помощью естественного языка речи. Пользователи могут говорить с Кортаной, нажатием кнопки микрофона, расположенной в правом верхнем углу мобильного приложения Teams, или фразой "Кортана" в комнате Microsoft Teams или при использовании дисплея Microsoft Teams. Чтобы быстро подключиться к своей команде, находясь в пути, пользователи могут выполнять такие запросы, как "позвонить Мегане" или "отправить сообщение на следующее собрание". Пользователи также могут присоединяться к собраниям, говоря "Присоединиться к следующему собранию", и использовать голосовую поддержку для совместного использования файлов, проверки календаря и т. д. Эти возможности голосовой помощи предоставляются с помощью служб Корпоративного уровня Кортаны, которые полностью соответствуют требованиям Office 365 конфиденциальности, безопасности и соответствию требованиям, как указано в условиях использования [веб-служб](/microsoft-365/admin/misc/cortana-integration) [(OST).](https://www.microsoft.com/licensing/product-licensing/products?rtc=1&preserve-view=true)

## <a name="admin-control-and-limitations"></a>Администратор и ограничения

Голосовая помощь Кортаны в Teams предоставляется с помощью служб, которые полностью соответствуют требованиям Office 365 конфиденциальности, безопасности и соответствия требованиям корпоративного уровня, как указано в условиях использования веб-служб (OST). Эта функция будет включена по умолчанию для клиентов.

Администраторы клиента могут контролировать, кто в своем клиенте может использовать голосовую поддержку Кортаны в Teams с помощью политики (TeamsCortanaPolicy). Эта политика задается на уровне учетной записи пользователя или клиента. Администраторы могут использовать поле CortanaVoiceInvocationMode в этом элементе управления политикой, чтобы определить, отключена ли Кортана, включена только с вызовом push-кнопки или с вызовом слова пробуждения (применимо к устройствам, которые ее поддерживают, например, при отображении Microsoft Teams).

Администраторы могут использовать следующие командлеты PowerShell для управления этой политикой (политика в настоящее время недоступна в Центре администрирования Microsoft Teams).

- [New-CsTeamsCortanaPolicy](/powershell/module/skype/New-CsTeamsCortanaPolicy)

- [Get-CsTeamsCortanaPolicy](/powershell/module/skype/Get-CsTeamsCortanaPolicy)

- [Grant-CsTeamsCortanaPolicy](/powershell/module/skype/Grant-CsTeamsCortanaPolicy)

- [Set-CsTeamsCortanaPolicy](/powershell/module/skype/Set-CsTeamsCortanaPolicy)

- [Remove-CsTeamsCortanaPolicy](/powershell/module/skype/Remove-CsTeamsCortanaPolicy)

Например, приведенная ниже команда создает политику с именем EmployeeCortanaPolicy, в которой голосовая помощь Кортаны в Microsoft Teams отключена.

```PowerShell
PS C:\> New-CsTeamsCortanaPolicy -Identity EmployeeCortanaPolicy -CortanaVoiceInvocationMode Disabled
```

В этом примере показано обновление существующей политики с именем EmployeeCortanaPolicy и включение голосовой помощи Кортаны в Microsoft Teams только с вызовом push-кнопки. Пользователи смогут вызывать Кортану, нажатием кнопки микрофона Кортаны в Teams. Слово пробуждения ("Привет, Кортана" или "Кортана") будет отключено.

```PowerShell
PS C:\> Set-CsTeamsCortanaPolicy -Identity EmployeeCortanaPolicy -CortanaVoiceInvocationMode PushToTalkUserOverride
```

В этом примере показано обновление политики и включение голосовой помощи Кортаны с помощью кнопки push-уведомлений и вызова слова пробуждения.

```PowerShell
PS C:\> Set-CsTeamsCortanaPolicy -Identity EmployeeCortanaPolicy -CortanaVoiceInvocationMode WakeWordPushToTalkUserOverride
```

На момент первоначального выпуска для Microsoft 365 корпоративный пользователей в США на английском языке доступны следующие функции:

- Мобильное приложение Teams не поддерживает активацию слова пробуждения, но оно будет поддерживаться в будущем.

- Комнаты Microsoft Teams устройства windows и Microsoft Teams поддерживают активацию слова пробуждения.

## <a name="user-control"></a>Пользовательский элемент управления

Отдельные пользователи могут использовать голосовую поддержку Кортаны на разных устройствах:

- Нажмите кнопку микрофона в мобильном приложении Teams.

- Нажмите кнопку микрофона или скажите "Кортана" в Комнаты Microsoft Teams.

- Скажите "Кортана" в Microsoft Teams отображает устройства.

Вы можете контролировать, включена ли Кортана в Teams для вашего устройства, с помощью параметра на устройстве.

### <a name="microsoft-teams-rooms-on-windows"></a>Комнаты Microsoft Teams Windows

Внесение изменений на уровне устройства доступно, только если Кортана включена на уровне клиента.

На уровне устройства можно настроить Использование Кортаны двумя разными способами. Вы можете включить один из вариантов или оба одновременно:

- Нажатие микрофона, который называется "Кортана _Push to talk" (Принудительная передача Кортаны для разговора)._
- Произнеся фразу "Привет, Кортана", которая называется _голосовой активацией Кортаны_

Кортана Push _to talk_ включена по умолчанию, если ваша комната настроена на любом из следующих языков: en-au (Австралия), en-ca (Канада), en-gb (Соединенное Королевство), en-in (Индия), en-us (США). [Подробнее.](/MicrosoftTeams/rooms/console#to-apply-your-desired-language) Значок Кортаны отобразит _кнопку "Показать_ " в меню "Дополнительно _"_ в консоли комнат Teams. Чтобы отключить кортану _push-уведомлений для беседы_ , используйте PowerShell. [Подробнее.](/powershell/module/skype/new-csteamscortanapolicy?view=skype-ps#example-1)

Чтобы включить голосовую _активацию_ Кортаны, необходимо выполнить следующие условия:

- Сертифицированное устройство Кортаны должно быть подключено к комнате Teams. Список сертифицированных устройств можно найти в конце этой статьи.
- Комната Teams должна быть настроена на любом из следующих языков: en-au (Австралия), en-ca (Канада), en-gb (Соединенное Королевство), en-in (Индия), en-us (США). Позже будут доступны дополнительные языки.
- Необходимо внести одно из следующих изменений конфигурации:
  - включите эту функцию в Центре администрирования Teams [.](/microsoftteams/rooms/rooms-manage)
  - добавьте следующий XML-атрибут в XML-файл SkypeSettings:

    ```xml
    <SkypeSettings>
        <CortanaWakewordEnabled>true</CortanaWakewordEnabled>
    </SkypeSettings>
    ```

На уровне собрания внесение изменений доступно только в том случае, если голосовая активация _Кортаны_ включена на уровне устройства.  Чтобы включить  голосовую _активацию_ Кортаны во время собрания, включите или выключите переключатель. После завершения собрания Кортана возвращается к набору параметров уровня устройства.

Внесение изменений на уровне собрания доступно, если Кортана включена на уровне устройства.

Чтобы включить голосовую _активацию_ Кортаны во время собрания, переместите переключатель **в положение "Вкл.** " или **"Выкл."**. После завершения собрания Кортана возвращается к набору параметров уровня устройства.
