# <a name="edit-guest-user-information"></a>Изменение сведений о гостевых пользователях

Вы можете изменить гостевую информацию в Центре администрирования Microsoft 365 или на портале Azure Active Directory.

## <a name="guest-users-in-the-microsoft-365-admin"></a>Гостевых пользователей в администраторе Microsoft 365

1. Чтобы изменить гостевую информацию в Центре администрирования [Microsoft 365,](https://admin.microsoft.com)выберите **"Пользователи-**  >  **гостевых пользователей".**

   ![Редактируемая информация гостевого пользователя](media/access-guest-user.png)

2. Выберите пользователя в **гостевом доступе.**

3. В сведениях о гостевых пользователях выберите **"Управление контактной информацией".**

   ![Управление контактной информацией ](media/guest-user-data1.png)

4. Отредактировать любые  выбранные поля, кроме отображаемой, в области "Управление контактной информацией" и выбрать **"Сохранить изменения".**

   ![Изменение контактных данных гостевого пользователя](media/manage-guest-contact.png)

Гостевой пользователь также можно изменить на [портале Azure Active Directory.](https://aad.portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/MsGraphUsers)
