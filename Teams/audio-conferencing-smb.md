# <a name="set-up-audio-conferencing-for-small-and-medium-businesses"></a>Настройка аудиоконференций для компаний малого и среднего бизнеса

С помощью аудиоконференций пользователи могут выполнять звонки на собрания Teams с помощью телефона вместо использования приложения Teams на мобильном устройстве или с компьютера.  

Если вы являетесь небольшим или средним бизнесом с 300 пользователями и у вас нет лицензий на аудиоконференции, вы можете бесплатно получить аудиоконференции в течение одного года. Это бесплатное предложение доступно с 1 октября 2020 г.

Лицензия на надстройку аудиоконференций может применяться к пользователям с лицензиями Microsoft 365 бизнес базовый, Business Standard, Business Premium, Enterprise E1 или Enterprise E3. Дополнительные сведения см. в разделе ["Лицензии на надстройки Teams"](teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

> [!NOTE]
> Если у вас есть Корпоративная E5 или Голосовая связь Microsoft 365 бизнес, вы не сможете использовать бесплатное предложение аудиоконференций, так как эти лицензии уже включают аудиоконференции.

В этой статье мы рассмотрим, как настроить аудиоконференции. Вам нужно только настроить аудиоконференции для пользователей, планирующих или проводящих собрания. Участникам собрания, которые звонит на собрания, не требуются лицензии или другая настройка. Дополнительные сведения см. в статье ["Аудиоконференции"](audio-conferencing-in-office-365.md).

## <a name="set-up-audio-conferencing"></a>Настройка аудиоконференций

При настройке аудиоконференций номер телефона автоматически назначается мосту конференц-связи, чтобы его можно было использовать в приглашениях на собрания. Номер телефона, назначенный в качестве номера по умолчанию для моста конференц-связи, будет 1 из страны или региона вашей организации. Этот номер телефона представляет собой платный номер, в котором может взиматься плата за междустройки.

> [!NOTE]
> Вы также можете использовать бесплатный номер, для которого требуется выполнить несколько дополнительных действий. Дополнительные сведения о номерах телефонов для моста конференц-связи см. далее в этой статье.[](#audio-conferencing-phone-numbers)

### <a name="step-1-get-audio-conferencing-licenses"></a>Шаг 1. Получение лицензий на аудиоконференции

Получите по одной лицензии на аудиоконференции для каждого человека, который будет вести собрания. Используйте Центр администрирования Microsoft 365 для этого.

1. В Центр администрирования Microsoft 365 перейдите к службам **покупки** >  выставления **счетов, а** затем в нижней части страницы выберите **"Надстройки"**.
2. Выберите **microsoft 365 Audio Conferencing Adoption Promo** > **Details** и нажмите кнопку **"Получить сейчас"**.
3. Введите количество лицензий, необходимых для организаторов собраний, а затем выполните заказ.

    :::image type="content" source="media/audio-conferencing-smb-add.png" alt-text="Снимок экрана: лицензия на аудиоконференции для внедрения promo.":::

    > [!NOTE]
    > Снимите или выберите автоматическое назначение всем пользователям без лицензий **в зависимости** от того, хотите ли вы автоматически назначить лицензию на аудиоконференции всем пользователям, у которых нет этой лицензии.

### <a name="step-2-assign-an-audio-conferencing-license-to-users-who-lead-meetings"></a>Шаг 2. Назначение лицензии на аудиоконференции пользователям, которые ведут собрания

Назначьте лицензию каждому пользователю, который будет вести собрания. Используйте Центр администрирования Microsoft 365 для этого.

#### <a name="assign-a-license-to-one-user"></a>Назначение лицензии одному пользователю

1. В Центр администрирования Microsoft 365 перейдите в раздел **"Активные пользователи** > **"**.  
2. Выберите строку пользователя, которому нужно назначить лицензию, а затем в области выберите **"Лицензии и приложения"**.
3. Установите **флажок "Аудиоконференции Microsoft 365" и** нажмите кнопку **"Сохранить изменения"**.

#### <a name="assign-a-license-to-multiple-users"></a>Назначение лицензии нескольким пользователям

1. В Центр администрирования Microsoft 365 перейдите в раздел **"Активные пользователи** > **"**.  
2. Выберите круги рядом с пользователями, которым вы хотите назначить лицензию, а затем выберите "Управление **лицензиями на продукты"**.
3. В области **"Управление лицензиями на продукты** " выберите " **Назначить больше"**.
4. Установите **флажок "Аудиоконференции Microsoft 365" и** нажмите кнопку **"Сохранить изменения"**.  

> [!IMPORTANT]
> Отображение назначенных номеров телефонов в приглашении на собрание может занять до 24 часов. Если обновленные номера не отображаются, подождите по крайней мере 24 часа, прежде чем обращаться в службу поддержки.

## <a name="schedule-teams-meetings-in-outlook"></a>Планирование собраний Teams в Outlook

Организаторы собраний теперь могут планировать собрания в Outlook. В Outlook перейдите в **календарь** и нажмите кнопку "Создать **собрание Teams** ". Номера телефонного подключения к собранию и идентификатор конференции автоматически добавляются в приглашение на собрание, которое отправляется участникам собрания. Дополнительные сведения см [. в статье "Планирование собрания Teams в Outlook"](https://support.microsoft.com/office/schedule-a-teams-meeting-from-outlook-883cc15c-580f-441a-92ea-0992c00a9b0f).

> [!NOTE]
> При необходимости вы можете настроить приглашения на собрания, чтобы добавить логотип компании, ссылки на веб-сайт службы поддержки и юридическое заявление об отказе, а также текстовый нижний колонтитул. Дополнительные сведения см. [в разделе "Настройка приглашений на собрания"](meeting-settings-in-teams.md#customize-meeting-invitations).

## <a name="audio-conferencing-phone-numbers"></a>Номера телефонов аудиоконференций

Существует два типа чисел, которые можно использовать для моста конференц-связи. Можно **использовать как общие** номера (как в разделе "[](#set-up-audio-conferencing)Настройка аудиоконференций" выше в этой статье), так и **выделенные номера**. Ниже приведены дополнительные сведения о каждом из них.

### <a name="shared-numbers"></a>Общие номера

Общий номер — это число, которое совместно используется во всех организациях. Общие номера — это платные номера, которые автоматически назначаются при настройке аудиоконференций.

Чтобы просмотреть номер по умолчанию, назначенный мосту конференц-связи, в левой области навигации Центра администрирования Microsoft Teams  >  перейдите к мостам конференций собраний **, а** затем найдите номер ближайшего к вам расположения.

### <a name="dedicated-numbers"></a>Выделенные номера

Выделенное число — это число, доступное только пользователям. Выделенный номер может быть платным или бесплатным. Чтобы использовать выделенный номер, необходимо сначала получить номер, назначить его мосту конференц-связи, а затем назначить номер каждому пользователю, который будет вести собрания.

Существует несколько способов получить выделенное число. Вы можете получить номер от корпорации Майкрософт или перенести (порт) существующий номер от текущего поставщика услуг в корпорацию Майкрософт. Дополнительные сведения о том, как это сделать, см. в статье ["Получение номеров служб"](getting-service-phone-numbers.md).

Помните, что при использовании бесплатного номера необходимо сначала назначить лицензию на кредиты на связь каждому пользователю, который будет вести собрания. Дополнительные сведения см. в статье ["Настройка кредитов на связь для вашей организации"](set-up-communications-credits-for-your-organization.md).

Получив свой номер, назначьте его мосту конференций. Для этого используйте Центр администрирования Microsoft Teams.

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к мостам **конференций собраний** > .
2. Нажмите **кнопку "** Добавить", а затем выберите **"Платный номер"** или **"Бесплатный номер"**.
3. В области **"Добавить номер телефона** " выберите номер и нажмите кнопку "Применить **"**.

#### <a name="assign-dial-in-phone-numbers-for-users-who-lead-meetings"></a>Назначение номеров телефонов с телефонным номером для пользователей, которые ведут собрания

Сведения о [настройке номеров телефонов, включенных в приглашения в Microsoft Teams, см. здесь](set-the-phone-numbers-included-on-invites-in-teams.md).

> [!NOTE]
> Вы также можете задать номера телефонов, добавив их в *политику TeamsAudioconferencingpolicy* и назначив политику пользователям. Платные и бесплатные телефонные номера, добавленные в политику, имеют приоритет над номерами телефонов, заданные отдельно для пользователей через панель параметров аудиоконференций. Если в *Teamsaudioconferencingpolicy* не добавляются номера телефонов, номер телефона, заданный отдельно для пользователей с помощью области параметров аудиоконференций, будет отображаться в приглашениях на собрания Microsoft Teams. [Дополнительные сведения см. в](audio-conferencing-toll-free-numbers-policy.md) параметрах политики аудиоконференций для платных и бесплатных номеров.
