# <a name="set-up-audio-conferencing-for-small-and-medium-businesses"></a>Настройка аудиоконференций для компаний малого и среднего бизнеса

Используя аудиоконференцию, люди могут звонить на собрания Teams с помощью телефона, а не с помощью приложения Teams на мобильном устройстве или с компьютера.  

Если вы малый или средний бизнес с 300 пользователями и у вас нет лицензий на аудиоконференцию, вы можете бесплатно получить аудиоконференцию в течение одного года. Это бесплатное предложение доступно с 1 октября 2020 г.

Лицензия на надстройку аудиоконференции может применяться к пользователям с лицензиями на Microsoft 365 бизнес базовый, бизнес-стандарт, бизнес-Premium, Enterprise E1 или Enterprise E3. Подробнее см. в Teams [надстройки](teams-add-on-licensing/microsoft-teams-add-on-licensing.md)

> [!NOTE]
> Если вы Enterprise E5 или Голосовая связь Microsoft 365 бизнес, вы не сможете использовать бесплатное предложение аудиоконференций, так как эти лицензии уже включают аудиоконференцию.

В этой статье мы построим аудиоконференцию. Вам нужно только настроить аудиоконференции для пользователей, планирующих или проводящих собрания. Участникам собрания, которые звонят на собрания, не требуются лицензии или другая настройка. Дополнительные информации см. в записи [аудиоконференции.](audio-conferencing-in-office-365.md)

## <a name="set-up-audio-conferencing"></a>Настройка аудиоконференций

При настройках аудиоконференции мосту аудиоконференций автоматически устанавливается номер телефона, который можно использовать в приглашениях на собрания. Номер телефона, который назначен мосту для связи по умолчанию, будет номером из страны или региона вашей организации. Этот номер телефона является платным номером, на который могут распространяться расходы на междугородние и междугородние связи.

> [!NOTE]
> Вы также можете использовать бесплатный номер, который потребует нескольких дополнительных действий. Дополнительные информацию о номерах телефонов для моста [](#audio-conferencing-phone-numbers) аудиоконференций см. в статье Номера телефонов для аудиоконференции далее в этой статье.

### <a name="step-1-get-audio-conferencing-licenses"></a>Шаг 1. Получить лицензии на аудиоконференцию

Получите одну лицензию на аудиоконференцию для каждого человека, который будет вести собрания. Для этого Центр администрирования Microsoft 365 с помощью этой Центр администрирования Microsoft 365.

1. В Центр администрирования Microsoft 365 перейдите в выставление счета За покупки служб, а затем в нижней части страницы выберите  >   **Надстройки**.
2. Выберите **Microsoft 365 аудиоконференцию с** подробными сведениями о внедрении аудиоконференции , а затем выберите Получить  >   **сейчас**.
3. Введите количество лицензий, необходимых организаторам собраний, а затем заполните свой заказ.

    :::image type="content" source="media/audio-conferencing-smb-add.png" alt-text="Снимок экрана: лицензия на аудиоконференцию с промолицами на внедрение аудиоконференций":::

    > [!NOTE]
    > В зависимости от того, хотите ли вы автоматически назначить лицензию на аудиоконференцию всем пользователям, у которых нет лицензий, или выберите автоматически назначать лицензию на аудиоконференцию всем пользователям, у которых нет этой лицензии.

### <a name="step-2-assign-an-audio-conferencing-license-to-users-who-lead-meetings"></a>Шаг 2. Назначение лицензии на аудиоконференцию пользователям, которые ведут собрания

Назначьте лицензию каждому человеку, который будет вести собрания. Для этого Центр администрирования Microsoft 365 с помощью этой Центр администрирования Microsoft 365.

#### <a name="assign-a-license-to-one-user"></a>Назначение лицензии одному пользователю

1. В Центр администрирования Microsoft 365 в области Пользователи   >  **Активные пользователи**.  
2. Выберите строку пользователя, лицензию на который вы хотите назначить, а затем в области выберите **Лицензии и приложения**.
3. Выберите  Microsoft 365 аудиоконференцию и выберите Сохранить **изменения.**

#### <a name="assign-a-license-to-multiple-users"></a>Назначение лицензии нескольким пользователям

1. В Центр администрирования Microsoft 365 в области Пользователи   >  **Активные пользователи**.  
2. Выберите круги рядом с пользователями, которые хотите назначить лицензию, а затем выберите Управление **лицензиями продуктов**.
3. В области **Управление лицензиями продуктов** выберите **Назначить больше**.
4. Выберите  Microsoft 365 аудиоконференцию и выберите Сохранить **изменения.**  

## <a name="schedule-teams-meetings-in-outlook"></a>Запланировать Teams собрания в Outlook

Организаторы собраний теперь могут планировать собрания в Outlook. В Outlook перейдите в **календарь** и выберите кнопку **Teams собрание.** Номера для телефонного дозвона и номер конференции автоматически добавляются в приглашение на собрание, которое отправляется участникам собрания. Дополнительные информации см. в [Teams собрания в Outlook.](https://support.microsoft.com/office/schedule-a-teams-meeting-from-outlook-883cc15c-580f-441a-92ea-0992c00a9b0f)

> [!NOTE]
> При этом вы можете настроить приглашения на собрания, чтобы добавить логотип компании, ссылки на веб-сайт поддержки и юридическое заявление об отказе, а также текстовый прикладный. Подробнее см. в [настройках приглашений на собрания.](meeting-settings-in-teams.md#customize-meeting-invitations)

## <a name="audio-conferencing-phone-numbers"></a>Номера телефонов для аудиоконференций

Существует два типа номеров, которые можно использовать для мостаконференций. Вы можете использовать **как** общие [](#set-up-audio-conferencing) номера (как в разделе Настройка аудиоконференции ранее в этой статье), так и **выделенные номера.** Дополнительные сведения о каждом из них.

### <a name="shared-numbers"></a>Общие номера

Общий номер — это число, которое совместно с другими организациями. Общие номера — это платные номера, которые автоматически устанавливаются при настроить аудиоконференцию.

Чтобы узнать номер по умолчанию, который назначен мосту конференц-связь, на левой навигации Центра администрирования Microsoft Teams перейдите к мосту Конференц-связь собраний и найдите номер ближайшего  >  расположения.

### <a name="dedicated-numbers"></a>Выделенные числа

Выделенное число — это число, доступное только пользователям. Выделенный номер может быть как платным, так и бесплатным. Чтобы использовать выделенный номер, необходимо сначала получить номер, назначить его мосту для проведения собраний, а затем назначить номер каждому человеку, который будет вести собрания.

Существует несколько способов получить выделенный номер. Вы можете получить номер от Майкрософт или перенести (перенос) существующего номера от текущего поставщика услуг в Корпорацию Майкрософт. Дополнительные информацию о том, как это сделать, см. в теме [Получение номеров служб.](getting-service-phone-numbers.md)

Помните, что если вы используете бесплатный номер, сначала необходимо назначить лицензию на кредиты на связь каждому человеку, который будет вести собрания. Дополнительные см. в настройках кредитов на [связь для организации.](set-up-communications-credits-for-your-organization.md)

После того как у вас есть номер, назначьте его мосту конференц-залов. Для этого Microsoft Teams центр администрирования.

1. В левой области навигации Центра администрирования Microsoft Teams перейдите к **мосту**  >  **Конференц-связь собраний**.
2. Выберите **Добавить**, а затем — **Платный или** **Бесплатный номер**.
3. В области **Добавление номера** телефона выберите номер и выберите **применить**.

Затем назначьте номер каждому человеку, который будет вести собрания. Для этого Microsoft Teams центр администрирования.

1. В левой области навигации Центра Microsoft Teams выберите **Пользователи**, щелкните отображаемого имени пользователя и выберите **Изменить**.
2. Выберите **Изменить** рядом с функцией Аудиоконференция, а затем в области Аудиоконференция выберите номер в списках Платный номер или Бесплатный номер и выберите **Применить**.
