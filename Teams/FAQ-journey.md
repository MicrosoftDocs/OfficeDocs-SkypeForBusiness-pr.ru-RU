# <a name="faq--upgrading-from-skype-for-business-to-microsoft-teams"></a>Вопросы и ответы. Переход со Skype для бизнеса на Microsoft Teams


## <a name="general-questions"></a>Общие вопросы

**Существует ли крайний срок, к которому клиентам требуется перейти со Skype для бизнеса Online на Teams?**<br>
Да. [Поддержка Skype для бизнеса Online будет прекращена](skype-for-business-online-retirement.md) 31 июля 2021 г., после чего эта служба больше не будет доступна. Рекомендуем клиентам Skype для бизнеса Online приступить к использованию Teams и планированию перехода уже сейчас, чтобы оставить достаточно времени для перехода до прекращения поддержки.


**Сколько времени займет переход моей организации на Teams?**<br>
Вы можете сами выбрать, как ваша организация будет переходить со Skype для бизнеса на Teams. Чтобы помочь вам с планированием и исполнением, мы составили исчерпывающее руководство по переходу с использованием проверенного плана, призванного помочь вам разобраться с техническими и организационными аспектами изменения. Для начала ознакомьтесь с [планом успешного перехода](upgrade-framework.md) и другими материалами по теме, которые станут вашими ориентирами при переходе со Skype для бизнеса на Teams.


**Существует ли рекомендуемый порядок перехода со Skype для бизнеса?**<br>
Мы рекомендуем запланировать переход со Skype для бизнеса на Teams с помощью руководства и других материалов, представленных по адресу [aka.ms/SkypetoTeams](upgrade-start-here.md) и посетить бесплатный [семинар по планированию обновления](upgrade-workshops-landing-page.yml), чтобы определить и выполнить процедуру перехода, наиболее соответствующую потребностям организации.

**Где можно найти дополнительные сведения о режимах сосуществования в Центре администрирования Microsoft Teams?**<br>
В Центре администрирования Microsoft Teams доступны различные режимы сосуществования, которые помогут оптимальным образом контролировать переход вашей организации со Skype для бизнеса на Teams. Дополнительные сведения см. в статье [Общие сведения о сосуществовании и взаимодействии Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

**Как подготовиться к переходу?**<br>
Успешный переход включает проверку технической готовности наряду с готовностью пользователей к принятию. Даже если вы определите, что ваша организация не готова к переходу на Teams, вы можете приступить к планированию уже сегодня. Кроме того, можно начать использование Teams вместе со Skype для бизнеса. Начните переход [со Skype на Teams[(https://aka.ms/skypetoteams-home) уже сегодня.

Кроме того, корпорация Майкрософт проводит интерактивные семинары, в ходе которых мы делимся рекомендациями, лучшими методиками и ресурсами для быстрого планирования и реализации перехода. [Подробнее о семинарах по планированию перехода](https://aka.ms/SkypeToTeamsPlanning).


**Моя организация уже использует Teams совместно со Skype для бизнеса. Можно ли просто отключить Skype для бизнеса?**<br>
Нет. Для полного перехода на Teams нужно перевести пользователей в режим "Только Teams". Если ваша организация готова к переходу на Teams, уделите некоторое время тому, чтобы проинформировать пользователей о переходе, и дайте им время, чтобы привыкнуть к Teams. Это поможет упростить переход и уменьшить количество обращений в вашу службу поддержки. Шаблоны для информирования входят в состав [комплекта инструментов для успешного перехода](https://aka.ms/UpgradeSuccessKit).


**К кому можно обратиться по вопросам, связанным с процессом перехода?**<br>
Если у вас есть вопросы, связанные с переходом, обратитесь к назначенным вам контактным лицам, например из службы технической поддержки учетных записей Майкрософт, у наших партнеров или в [FastTrack](https://www.microsoft.com/en-us/fasttrack?rtc=1). Вы также можете отправить запрос в службу поддержки в [Центре администрирования Microsoft Teams](https://admin.teams.microsoft.com/), нажав кнопку **Нужна помощь** .

![Кнопка "Нужна помощь"](media/need-help-button.png)


**Нужно ли переводить на Teams всех пользователей моего клиента одновременно, или можно переводить по несколько пользователей за раз?**<br>
Вы можете переводить пользователей на Teams так, как вам удобно: по одному, группами или всю организацию сразу. Чтобы подобрать оптимальный подход для вашей организации, рассмотрите доступные [режимы сосуществования и перехода](teams-and-skypeforbusiness-coexistence-and-interoperability.md).


**Что произойдет после перевода моих пользователей на Teams?**<br>

После перевода пользователей на Teams (режим **Только Teams** ):

- Их клиент Skype для бизнеса будет отключен, так как все чаты и звонки будут перенесены в Teams. Этот клиент будет по-прежнему использоваться для ранее запланированных собраний Skype для бизнеса. Если удалить этот настольный клиент, то ранее запланированные собрания будут доступны пользователям в веб-приложении Skype для бизнеса.

- Все собрания Skype для бизнеса, запланированные до перехода, будут работать в том виде, как они планировались, но все новые собрания будут назначаться в Teams. Если пользователь попробует войти в Skype для бизнеса, он получит из клиента оповещение о том, что он переведен на Teams. Пользователям потребуется вручную удалить клиент Skype для бизнеса на мобильном устройстве.



**Смогут ли пользователи по-прежнему работать в Skype для бизнеса после того, как я активирую уведомление о переходе в их клиентах?**<br>
Да. Уведомление просто информирует пользователей о предстоящем переходе со Skype для бизнеса на Teams и приглашает их начать работу с Teams, если они этого еще не сделали. Рекомендуем дополнить это уведомление информационной кампанией (по электронной почте, в виде вопросов и ответов, с привлечением службы поддержки, с помощью постеров и информационных материалов), чтобы пользователи смогли узнать больше о переходе конкретно в вашей организации: время обновления, призывы пользователей к действиям, доступ к обучающим материалам и т. д. Шаблоны для информирования входят в состав [комплекта инструментов для успешного перехода](https://aka.ms/UpgradeSuccessKit).


**Что это означает с точки зрения лицензирования? Как клиенты будут оплачивать службы интеллектуальных коммуникаций в Teams?**<br>
Продукт Teams предоставляется в составе наборов Microsoft 365 и Office 365. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Сделанные клиентами приобретения по лицензиям переносятся в Teams. Например, если клиент приобрел вместе со Skype для бизнеса компонент аудиоконференций (отдельный или в составе подписки E5), он сохранит доступ к аудиоконференциям и в Teams.


**Собирается ли Майкрософт проводить плановые обновления?**<br>
Мы пока не собираемся проводить подобные обновления для корпоративных клиентов. Клиенты могут перейти на Teams в удобное для организации время до 31 июля 2021 года, когда будет прекращена поддержка Skype для бизнеса Online. Мы поможем администраторам и пользователям перейти на Teams, предоставив им необходимые инструменты и руководства.

Чтобы помочь небольшим организациям, у которых может не быть выделенных ИТ-ресурсов, Майкрософт оказывает им поддержку с автоматическим переходом на Teams со Skype для бизнеса Online. Соответствующие клиенты получают уведомление об обновлении по электронной почте и в уведомлениях Центра сообщений. В этих сообщениях содержится более подробная информация. Дополнительные сведения см. в статье [Автоматический переход со Skype для бизнеса Online на Microsoft Teams](upgrade-automated.md).



## <a name="microsoft-teams-capabilities-and-roadmap"></a>Возможности и стратегия развития Microsoft Teams

**В чем заключаются преимущества серверной инфраструктуры Teams?**<br>
Служба Teams создана для облака и построена на архитектуре микрослужб с широкими возможностями масштабирования. Эта архитектура эффективно использует пропускную способность, обеспечивает более функциональную телеметрию и позволяет выполнять обслуживание и обновление, практически не затрагивая текущую работу. В результате пользователи могут быстрее присоединяться к собраниям и эффективно работать в браузере без необходимости в загрузке подключаемых модулей. Современная инфраструктура позволяет легко подключаться к службам Microsoft Cognitive Services, которые поддерживают транскрибирование, перевод, распознавание речи и машинное обучение — возможности, которые существенно упрощают общение и совместную работу и повышают их эффективность.

**Как клиентам узнать, когда возможности Skype для бизнеса будут доступны в Teams?**<br>
См. [стратегию развития Microsoft 365](https://aka.ms/O365Roadmap).

**Какие API-интерфейсы и пакеты SDK будут доступны для Teams?**<br>
Сведения о доступных API-интерфейсах и пакетах SDK доступны на [платформе разработчиков Microsoft Teams](https://docs.microsoft.com/microsoftteams/platform/).


**Будет ли Teams поддерживать возможности сторонней разработки?**<br>
Да, интеграция приложений — одно из ключевых преимуществ перехода на Teams. Сейчас Teams поддерживает сторонние боты, соединители и расширения. Кроме того, в магазине приложений Microsoft Teams доступен широкий набор надстроек.

**Доступен ли продукт Teams в планах Microsoft 365 для образования?**<br>
Teams доступен во всех лицензиях наборов Microsoft 365 для образования (Microsoft A1, A3 и A5).

**Доступна ли служба Teams в облаке сообщества для государственных организаций (GCC)?**<br>
Да. Служба Teams доступна в облаке сообщества для государственных организаций США (GCC). Дополнительные сведения см. в статье [Планирование развертываний Microsoft 365 GCC](https://docs.microsoft.com/microsoftteams/plan-for-government-gcc).



## <a name="calling-capabilities"></a>Возможности для звонков

**Какие планы корпорация Майкрософт имеет относительно голосовых звонков по сети?**<br>
В основе нашего решения для голосовой связи лежит телефонная система, доступная уже сегодня. Кроме того, клиенты могут добавить тарифный план Майкрософт, обеспечивающий полную поддержку звонков, включая получение и назначение номеров непосредственно в Microsoft 365. Клиенты, желающие сохранить свои телефонные линии, могут использовать функцию прямой маршрутизации, входящую в состав телефонной системы. Совмещайте эти функции согласно потребностям организации, чтобы составить оптимальное решение для голосовой связи.

**Что рекомендуется делать клиентам, уже развернувшим телефонную систему (облачную УАТС) на базе Skype для бизнеса Online?**<br>
Звонки в Microsoft Teams готовы к любому применению. Рекомендуем всем клиентам Microsoft 365 начать использовать Teams отдельно или одновременно со Skype для бизнеса.


**Что вы посоветуете клиентам, которые используют корпоративную голосовую связь и хотят перейти на использование звонков в Teams?**<br>
Клиенты, которые хотят перенести собственную телефонную службу в Teams, могут сделать это с помощью общедоступной функции [прямой маршрутизации](https://techcommunity.microsoft.com/t5/Microsoft-Teams-Blog/Direct-Routing-is-now-Generally-Available/ba-p/210359). Прямая маршрутизация и планы звонков — две функции для поддержки тонового набора в Microsoft Teams.

## <a name="meeting-capabilities"></a>Возможности проведения собраний


**Отличается ли охват аудиоконференций в Teams от охвата в Skype для бизнеса?**<br>
Внедрение аудиоконференций в Teams не повлияет на область охвата этой службы. В данный момент она поддерживается более чем в 90 странах и 400 городах, и это останется так в обоих продуктах. Текущий список городов и стран см. в статье [Доступность аудиоконференций и планов звонков в различных странах и регионах](https://docs.microsoft.com/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans).



**Поддерживает ли Teams сторонних поставщиков аудиоконференций?**<br>
Мы не планируем поддерживать в Teams сторонние поставщики аудиоконференций (ACP). Мы считаем, что лучшим вариантом реализации аудиоконференций для клиентов Teams и Skype для бизнеса будет работа с нашими службами аудиоконференций. Те, кому необходима поддержка сторонних поставщиков аудиоконференций в собраниях Skype для бизнеса, могут продолжить использовать свой клиент Skype для бизнеса для подключения к собраниям Skype для бизнеса. Для собраний, запланированных в Teams, потребуется использовать службы аудиоконференций из Microsoft 365.

Поддержка интеграции сторонних поставщиков аудиоконференций (ACP) со Skype для бизнеса Online была продлена до 31 июля 2021 года, а ограниченная поддержка оставшихся активных клиентов оставляет дополнительное время для перехода. Это обновление применяется к расписанию ACP, опубликованному в апреле 2018 г.



**Каковы ваши планы по поддержке совместимости видео для собраний Teams?**<br>
[Устройства для конференц-залов](https://products.office.com/microsoft-teams/across-devices) занимают ключевое место в нашей концепции современного рабочего пространства. [Облачные службы видеосвязи](https://docs.microsoft.com/microsoftteams/cloud-video-interop), поддерживающие собрания Teams и существующие системы видеоконференций, доступны у наших партнеров Pexip, Polycom и Blue Jeans.

**Будет ли новейшее поколение Систем комнат Skype (версии 2) поддерживать собрания в Teams?**<br>
Мы заменили приложение "Системы комнат Skype" на программу "Комнаты Microsoft Teams", которая полностью поддерживает собрания Microsoft Teams и дает возможность легко перейти со Skype для бизнеса на Teams, просто включив Teams на устройстве.

Пользователи могут не только обнаруживать находящиеся поблизости комнаты Microsoft Teams, но и объединять собрания Teams одним щелчком благодаря поддержке двух экранов и приложения "Доска (Майкрософт)". Мы продолжаем добавлять такие инновационные функции, как камера контента с интеллектуальным сканированием.


**Будет ли приложение "Системы комнат Skype" версии 1 обновлено для поддержки собраний Teams?**<br>
Поддержка устройств системы комнат Lync (LRS), на которых установлено приложение "Системы комнат Skype" версии 1 (SRS v1), была прекращена 9 октября 2018 г. Это означает, что для приложения "Системы комнат Skype" версии 1 больше не будут выпускаться обновления и исправления. Пользователям устройств системы комнат Lync, использующие приложение "Системы комнат Skype" версии 1, рекомендуем установить на своих устройствах приложение "Комнаты Microsoft Teams". Дополнительные сведения см. в статье [Миграция устройств системы комнат Lync (LRS) в комнаты Microsoft Teams](https://docs.microsoft.com/microsoftteams/rooms/lrs-migration).

**Прекратится ли поддержка трансляций собраний Skype одновременно со Skype для бизнеса Online?**<br>
Да. [Трансляции Teams](https://docs.microsoft.com/microsoftteams/teams-live-events/what-are-teams-live-events) — это решение, заменяющее трансляции собраний Skype.


## <a name="management-capabilities"></a>Возможности управления

**Какие возможности управления доступны для Teams?**<br>
Как и консоль администрирования Skype для бизнеса, [Центр администрирования Microsoft Teams](https://admin.teams.microsoft.com/) в Центре администрирования Microsoft 365 представляет собой единое расположение для управления новыми возможностями Teams. С помощью этого портала администраторы могут реализовать настраиваемые политики присутствия, чатов, приложений, собраний и голосовой связи, а также назначать эти политики пользователям Teams.

## <a name="device-compatibility"></a>Совместимость устройств

**Можно ли использовать Teams на Surface Hub?**<br>
Собрания Teams теперь доступны на Surface Hub в специальном интерфейсе звонков и собраний. Дополнительные сведения см. в статье [Развертывание Microsoft Teams для Surface Hub](https://docs.microsoft.com/microsoftteams/teams-surface-hub).

**Будут ли имеющиеся IP-телефоны (3PIP) сторонних производителей работать с Microsoft Teams? А что насчет сторонних телефонов SIP?**<br>
Использование существующего оборудования может помочь снизить затраты при переходе на новую телефонную систему. Корпорация Майкрософт продлила поддержку телефонов Skype для бизнеса (3PIP) и будет осуществлять ее и после 2023 г., поэтому вы можете продолжать использовать существующие телефоны Skype для бизнеса при переходе на Teams. Кроме того, начиная с первой половины 2021 г., в Teams будут поддерживаться основные функции вызовов на телефоны SIP от Cisco, Yealink, Polycom и т. д.

**Будут ли телефоны, сертифицированные для Skype для бизнеса Online, работать с Teams?**<br>
Ответы на вопросы, связанные с совместимостью телефонов, см. в статье блога [Certified Skype for Business Online Phones and what this means for Teams](https://techcommunity.microsoft.com/t5/Skype-for-Business-Blog/Certified-Skype-for-Business-Online-Phones-and-what-this-means/bc-p/125309) (Телефоны, сертифицированные для Skype для бизнеса Online, и их значение для Teams).
