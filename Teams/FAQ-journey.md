<a name="faq---journey-from-skype-for-business-to-microsoft-teams"></a>Вопросы и ответы. Переход со Skype для бизнеса на Microsoft Teams
==========================================================

<a name="intelligent-communications-vision"></a>Концепция интеллектуальных коммуникаций
----------------------------------

### <a name="what-is-microsofts-new-vision-for-intelligent-communications-aka-unified-communications-vision"></a>В чем заключается новая концепция интеллектуальных коммуникаций (или объединенных коммуникаций) корпорации Майкрософт? 

Мы добились большого успеха с Office 365 — число активных пользователей с ежемесячной коммерческой подпиской, которые полагаются на Office 365 в своей повседневной работе, превысило 100 миллионов. Теперь вы решили привнести облачные возможности Skype для бизнеса в систему Microsoft Teams, чтобы создать единый центр для командной работы со встроенной и полностью интегрированной голосовой и видеосвязью. Активное внедрение функций взаимодействия в приложения, которые команды используют для повседневной совместной работы, а также наличие Искусственного интеллекта, Microsoft Graph, LinkedIn, Cognitive Services и других служб для обработки данных позволили нам реализовать концепцию интеллектуальных коммуникаций, которая кардинально изменяет работу со звонками и собраниями.

### <a name="why-are-you-planning-on-bringing-skype-for-business-and-teams-together"></a>Почему вы планируете объединить Skype для бизнеса и Teams? 

По мере того, как пользователи участвуют в работе все большего числа команд, у нас появляется возможность легко интегрировать функции взаимодействия в Microsoft Teams, чтобы предложить пользователям единое место для бесед, контактов и контента. Мы считаем, что Microsoft Teams также предоставляет современную облачную инфраструктуру, позволяющую нам использовать имеющиеся активы, такие как Искусственный интеллект, Microsoft Graph и LinkedIn, для обеспечения интеллектуальных коммуникаций. Microsoft Teams позволяет нам создавать новые интерфейсы для собраний и звонков, включая этапы подготовки, проведения и последующего сопровождения.

### <a name="is-there-a-firm-deadline-by-which-customers-need-to-move-from-skype-for-business-online-to-teams"></a>Существует ли крайний срок, к которому клиентам требуется перейти со Skype для бизнеса Online на Teams? 

Мы опубликуем свою стратегию по Microsoft Teams, чтобы клиенты могли оценить требуемые им возможности Teams и спланировать переход. Клиенты могут определить сроки перехода на Teams, исходя из своих потребностей.

Чтобы оставаться в курсе новостей о переходе со Skype для бизнеса на Microsoft Teams, изучите загружаемый PDF-файл [Skype for Business to Microsoft Teams Capabilities Roadmap (Переход со Skype для бизнеса на Microsoft Teams: стратегия внедрения возможностей)](https://aka.ms/skype2teamsroadmap). Пока что он доступен только на английском языке. Этот PDF-файл рассказывает, чего сейчас ожидает корпорация Майкрософт в отношении переноса возможностей Skype для бизнеса в Microsoft Teams.

### <a name="what-is-the-future-of-skype-for-business-server"></a>Что ждет Skype для бизнеса Server? 

Мы понимаем, что по определенным причинам многие клиенты вынуждены использовать локальную версию Skype для бизнеса для некоторых пользователей или в некоторых географических регионах.  Мы планируем сориентировать следующую версию Skype для бизнеса Server на локальную среду, выпустив новый набор серверных обновлений к концу 2018 календарного года (конкретные даты могут изменяться).

### <a name="what-will-happen-to-the-skype-brand"></a>Что произойдет с брендом Skype? 

Мы сохраним верность общей концепции бренда Skype, который представляет семейство функций взаимодействия для потребительского и коммерческого сегментов. Skype останется названием продукта для нашего потребительского предложения, а Skype для бизнеса останется названием продукта для локального сервера. Бренд Skype для бизнеса будет также упоминаться внутри Microsoft Teams в связи с голосовой и видеосвязью.

### <a name="what-does-this-mean-from-a-licensing-perspective-how-will-customers-pay-for-uc-services-in-teams"></a>Что это означает с точки зрения лицензирования? Как клиенты будут оплачивать службы объединенных коммуникаций в Teams?

Продукт Microsoft Teams предоставляется в составе наборов Office 365. Возможности, являющиеся платными рабочими нагрузками в Skype для бизнеса в Office 365, останутся таковыми и в Microsoft Teams. Вложения клиентов в лицензии перейдут в Microsoft Teams. Например, если клиент приобрел отдельный компонент Аудиоконференций или подписку E5 со Skype для бизнеса, он сохранит доступ к Аудиоконференциям и в Microsoft Teams.

### <a name="is-microsoft-planning-scheduled-upgrades"></a>Собирается ли Майкрософт проводить плановые обновления? 

Мы пока не собираемся проводить подобные обновления для корпоративных клиентов. Клиенты могут самостоятельно принять решение о переходе на продукт Microsoft Teams, если его функции соответствуют их потребностям. Мы постараемся помочь администраторам и пользователям осуществить переход на Microsoft Teams, выпустив соответствующие средства и рекомендации.

### <a name="what-does-this-mean-for-existing-office-365-customers"></a>Как это отразится на существующих клиентах Office 365?

Со временем мы реализуем для Teams ключевой набор возможностей Skype для бизнеса в Office 365, а также новые возможности голосовой и видеосвязи, а также работы с собраниями. Клиенты смогут самостоятельно принять решение о миграции пользователей Skype для бизнеса, когда функциональность продукта будет соответствовать их требованиям. Мы рекомендуем всем клиентам Office 365 начать использовать Teams либо отдельно, либо параллельно со Skype для бизнеса.

### <a name="what-do-you-recommend-for-customers-who-are-currently-in-the-process-or-planning-to-deploy-skype-for-business"></a>Что вы порекомендуете клиентам, которые сейчас планируют или осуществляют развертывание Skype для бизнеса? 

Им следует оценить стратегию Teams, чтобы определить требуемые функции и соответствующие сроки. В этом случае им нужно скорректировать свои планы, добавив в них развертывание Teams. Оно включает в себя планирование и вложения в инфраструктуру, например оценки сети. В противном случае клиентам следует продолжить развертывание Skype для бизнеса. Мы рекомендуем всем клиентам Office 365 начать использовать Teams либо отдельно, либо параллельно со Skype для бизнеса.

### <a name="what-support-are-you-offering-to-customers-who-wish-to-move-to-teams-especially-those-who-have-made-significant-investments-in-skype-for-business-in-office-365"></a>Какую поддержку вы оказываете клиентам, желающим перейти на Teams, особенно тем из них, которые вложили значительные средства в Skype для бизнеса в Office 365? 

Клиенты продолжат использовать Skype для бизнеса в Office 365, пока Microsoft Teams не начнет отвечать их бизнес-потребностям.  Мы предоставим клиентам, желающим перейти на Teams, средства и указания для реализации данного процесса и управления им.  Указания и рекомендации будут опубликованы на сайте https://aka.ms/skypeandteams. Мы упростим переход для пользователей, не знакомых с Microsoft Teams, с помощью ознакомительного обзора при первом запуске и подсказок в самом продукте.  При подобном переходе клиенты смогут воспользоваться FastTrack, поддержкой клиентов и услугами партнеров. 

### <a name="how-can-customers-learn-when-skype-for-business-capabilities-will-be-available-in-microsoft-teams"></a>Как клиентам узнать, когда возможности Skype для бизнеса будут доступны в Microsoft Teams?

Мы будем публиковать сведения о запланированных к выпуску функциях Teams в [стратегии Office 365](https://go.microsoft.com/fwlink/?linkid=859057).

### <a name="what-is-the-roadmap-for-new-features-in-teams"></a>Каковая стратегия по выпуску новых функций в Teams? 

В рамках новой концепции интеллектуальных коммуникаций мы объявляем об инновациях в Teams, затрагивающих обмен сообщениями, работу с собраниями и звонками, включая:

-   улучшенное взаимодействие Teams и Skype для бизнеса с поддержкой пользователей, находящихся за пределами вашей организации, универсальное присутствие в Office 365, а также федеративные звонки и чат;

-   входящие и исходящие звонки на номера ТСОП, а также базовые функции звонков (удержание, передача, быстрый набор, голосовая почта);

-   усовершенствования корпоративных собраний с анонимным присоединением и "залом ожидания", элементами управления для выступающих, общим доступом к приложениям и PowerPoint, голосовой конференц-связью с телефонным подключением, а также записью и транскрибированием в облако.

В будущем мы опубликуем эту стратегию, чтобы клиенты могли оценить, насколько Microsoft Teams отвечает их потребностям в коммуникациях.

### <a name="will-you-continue-to-invest-in-improving-skype-for-business-in-office-365"></a>Продолжите ли вы улучшать Skype для бизнеса в Office 365? 

Наша концепция по объединению интеллектуальных коммуникаций и совместной работы нацелена на Microsoft Teams. Мы планируем продолжить поддержку Skype для бизнеса в Office 365.

### <a name="is-there-a-point-at-which-skype-for-business-will-no-longer-be-included-in-new-office-365-subscriptions"></a>Существует ли момент, с которого Skype для бизнеса перестанет входить в новые подписки Office 365? 

Мы планируем продолжить поддержку службы и клиента Skype для бизнеса в Office 365. Сейчас у нас нет планов по исключению Skype для бизнеса из подписок Office 365.

### <a name="when-will-the-skype-for-business-clients-be-retired-will-on-premises-customers-need-to-switch-to-the-teams-client"></a>Когда будет прекращено использование клиентов Skype для бизнеса? Потребуется ли локальным клиентам переходить на клиент Teams? 

Мы планируем продолжить поддержку клиентов Skype для бизнеса, предоставляющего доступ к локальному серверу и веб-службам.

### <a name="what-is-the-plan-for-microsofts-online-voice-capabilities"></a>Какие планы корпорация Майкрософт имеет относительно голосовых звонков по сети?

Мы реализуем в Microsoft Teams уже доступные возможности облачной УАТС и голосовой связи, а также дополнительные возможности, входившие в состав Skype для бизнеса, в рамках стратегии Office 365 и дальнейшего обсуждения.  Со временем это произойдет со всеми ключевыми функциями, запланированными к выпуску во втором полугодии 2018 календарного года (конкретные даты могут изменяться).

### <a name="what-is-the-guidance-for-customers-already-deployed-on-cloud-pbx-in-skype-for-business-online"></a>Что следует предпринять клиентам, уже развернувшим облачную УАТС в Skype для бизнеса Online?

Им следует оценить стратегию Teams, чтобы определить, когда функции продукта будут отвечать их требованиям. Мы рекомендуем всем клиентам Office 365 начать использовать Teams либо отдельно, либо параллельно со Skype для бизнеса.

### <a name="can-customers-use-existing-devices-such-as-ip-phones-that-currently-work-with-skype-for-business-in-office-365-with-teams"></a>Могут ли клиенты использовать в Teams имеющиеся устройства (например, IP-телефоны), которые сейчас работают со Skype для бизнеса в Office 365?

Да, мы дадим клиентам возможность использовать уже приобретенное USB- и IP-оборудование в Teams.

### <a name="when-will-hybrid-customers-be-able-to-move-to-teams"></a>Когда в Teams смогут перейти на гибридные клиенты?

Мы планируем упростить клиентам процедуру подключения магистральных каналов или <sup></sup>сторонних телефонных систем к Office 365 на основе полученных отзывов. Этот новый подход запланирован к выпуску во втором полугодии 2018 календарного года. Дата может изменяться, поэтому клиентам рекомендуется отслеживать актуальные сведения о стратегии для Teams. Клиенты, использующие гибридную голосовую связь или нуждающиеся в ней, могут и дальше работать со Skype для бизнеса Online.

### <a name="will-people-centric-conversations-be-available-in-the-new-client"></a>Будут ли ориентированные на пользователей беседы доступны в новом клиенте?

Одно из множества преимуществ Microsoft Teams заключается в том, что пользователи получают доступ к новым функциям взаимодействия. В Teams весь чат ориентирован на пользователей. Пользователи легко могут просмотреть всю историю чатов с другими людьми из клиента или браузера. Поиск стал еще проще.

### <a name="what-is-the-plan-for-video-interoperability-support-for-teams-meetings"></a>Каковы ваши планы относительно поддержки видеовзаимодействия для собраний Teams?

Устройства для конференц-залов занимают важное место в нашей концепции современного рабочего пространства.  25 сентября на Ignite мы объявили, что сотрудничаем с Blue Jeans и Pexip, чтобы реализовать в Microsoft Teams облачное видеовзаимодействие. Клиенты могут продолжить использование Polycom Real Connect for Office 365 вместе со Skype для бизнеса Online. Мы опубликуем свою стратегию, чтобы клиенты могли оценить, насколько Microsoft Teams отвечает их бизнес-потребностям.

### <a name="will-the-latest-generation-of-skype-room-systems-support-meetings-in-teams"></a>Будет ли последнее поколение Систем комнат Skype поддерживать собрания в Teams?

Устройства для конференц-залов занимают важное место в нашей концепции современного рабочего пространства, поэтому мы продолжим работу над Системами комнат Skype. Изначально Системы комнат Skype будут поддерживать присоединение к собраниям Teams одним щелчком, а также большинство функций Систем комнат Skype, за исключением режима двойного экрана и панорамных видеоустройств. Поддержка этих компонентов появится в последующих выпусках.

### <a name="will-skype-room-systems-v1-be-updated-to-support-microsoft-teams-meetings"></a>Будут ли Системы комнат Skype версии 1 обновлены, чтобы поддерживать собрания Microsoft Teams? 

Устройства для конференц-залов занимают важное место в нашей концепции современного рабочего пространства.  Системы комнат Skype версии 1 (или Системы комнат Lync) будут обновлены, что позволит подобным устройствам присоединяться к собраниям Teams. Кроме того, в целях повышения безопасности эти устройства также будут обновлены с Windows 7 до Windows 10. На устройствах с сенсорным экраном вам будут недоступны функции сенсорного управления, а также такие возможности, как рукописный ввод, аннотирование PowerPoint и использование доски, а также запуск собрания из зала по касанию.

### <a name="what-is-planned-for-new-management-experiences"></a>Каковы планы относительно новых интерфейсов управления?

Майкрософт планирует запустить в Центре администрирования Office 365 новый портал администрирования для ИТ-специалистов. Как и консоль администратора Skype для бизнеса, этот новый портал будет выступать в качестве единого расположения для администрирования новых функций. Первоначальный выпуск портала запланирован на начало календарного 2018 года. Они призван обеспечить поддержку администрирования посредством корпоративной модели политик для клиентов, использующих только Teams, а также заменить собой существующий интерфейс администрирования для Teams. С помощью этого портала мы сможем реализовать настраиваемые политики присутствия, чата, приложений, собраний и голосовой связи, а также назначить их пользователям Teams. 

### <a name="can-i-use-microsoft-teams-on-surface-hub"></a>Можно ли использовать Microsoft Teams на Surface Hub?

Сейчас многие процедуры Teams реализуются на Surface Hub через браузер Edge. Войдите в систему с приветственного экрана Surface Hub, откройте Edge и введите **teams.microsoft.com**, чтобы получить доступ к своим командам, каналам, документам, беседам и собраниям. Surface Hub поддерживает собрания Teams, и мы надеемся, что в будущем сможем подробнее рассказать о других интегрированных процедурах Teams на Surface Hub.
