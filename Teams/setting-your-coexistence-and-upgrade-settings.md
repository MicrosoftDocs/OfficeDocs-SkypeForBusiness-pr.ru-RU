# <a name="set-your-coexistence-and-upgrade-settings"></a>Настройка сосуществования и обновления


При обновлении Skype для бизнеса пользователей для использования Teams есть несколько вариантов, которые помогут вам сделать этот процесс простым для пользователей. Вы можете одновременно внести параметры сосуществования и обновления для всех пользователей в организации или внести изменения в параметры для одного или нескольких пользователей в организации. Обратите внимание, что более старые версии Skype для бизнеса могут не учитывать эти параметры. Дополнительные сведения о Skype для бизнеса версиях клиентов см. на Skype для бизнеса [загрузки и обновления](/skypeforbusiness/software-updates). 

Вы можете лучше понять доступные режимы, прочитав статью "Общие сведения Microsoft Teams и [Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md) сосуществование и взаимодействие или сосуществование с [Skype для бизнеса.](coexistence-chat-calls-presence.md)

## <a name="set-upgrade-options-for-all-users-in-your-organization"></a>Настройка параметров обновления для всех пользователей в организации

 **С помощью Центра администрирования Microsoft Teams**

1. В Microsoft Teams [центра администрирования](https://admin.teams.microsoft.com/) в области навигации слева перейдите **Teams** >  **Teams параметры обновления**. 

2. В верхней части страницы **Teams измените** следующие параметры по своему усмотрению.

    - **Задайте режим сосуществования**.
        - **Islands** — используйте этот параметр, если вы хотите, чтобы пользователи могли одновременно Skype для бизнеса и Teams.
        - **Skype для бизнеса только**. Используйте этот параметр, если вы хотите, чтобы пользователи могли использовать только Skype для бизнеса.
        - **Skype для бизнеса с Teams** совместной работы. Используйте этот параметр, если вы хотите, чтобы пользователи Skype для бизнеса в дополнение к использованию Teams для совместной работы групп (каналов).
        - **Skype для бизнеса совместной Teams** и собраний. Используйте этот параметр, если вы хотите, чтобы пользователи могли использовать Skype для бизнеса в дополнение к использованию Teams для групповой совместной работы (каналов) и Teams собраний.
        - **Teams только**. Используйте этот параметр, если вы хотите, чтобы пользователи могли использовать только Teams. Обратите внимание, что даже с помощью этого параметра пользователи по-прежнему могут присоединяться к собраниям, размещенным в Skype для бизнеса.

          > [!IMPORTANT]
          > Вы можете назначить режим TeamsOnly всему клиенту только после выполнения обоих следующих действий:
          >  - Все локальные пользователи были перемещены в Teams только с помощью Move-CsUser в Skype для бизнеса Server локальном наборе инструментов.
          >  - Вы обновили все Skype для бизнеса DNS, указав на Microsoft 365. 
          >
          > Дополнительные сведения см. в разделе ["Отключение гибридной конфигурации для завершения миграции Teams"](/skypeforbusiness/hybrid/cloud-consolidation-disabling-hybrid).
        
    - **Задайте для Skype для бизнеса уведомления пользователей о Teams, доступных для обновления**. Если вы включите эту функцию, пользователи Skype для бизнеса, что они скоро будут обновлены до Teams приложения.

    - **Задайте предпочитаемое приложение, чтобы пользователи присоединялись Skype для бизнеса собраниям**. Этот параметр определяет, какое приложение используется для присоединения Skype для бизнеса собраний, и учитывается независимо от значения режима сосуществования.
      - **Skype "Собрания"**
      - **Skype для бизнеса с ограниченными возможностями**

    - Укажите, **следует ли скачивать Teams в** фоновом режиме для Skype для бизнеса пользователей. Этот параметр автоматически скачивает приложение Teams для пользователей, Skype для бизнеса на Windows. Он учитывается только в том случае, если режим сосуществования для пользователя Teams только в том случае, если уведомления об ожидающих обновлениях включены в Skype для бизнеса.

3. После **внесения** изменений нажмите кнопку "Сохранить".

## <a name="set-upgrade-options-for-a-single-user-in-your-organization"></a>Настройка параметров обновления для одного пользователя в организации

 **С помощью Центра администрирования Microsoft Teams**

1. В области навигации слева перейдите к **пользователям** **UsersManage** >  и выберите пользователя из списка. 
2. На **вкладке "** Учетная запись" для пользователя **в Teams обновления** нажмите кнопку "Изменить **"**.
3. Можно задать режим **сосуществования**. Режимы, отличные от Teams, могут применяться только к пользователям, которые размещены в локальной среде Skype для бизнеса Server, и, наоборот, только пользователи, размещенные в облаке, могут иметь режим TeamsOnly.  Выберите один из следующих вариантов:
     - **Используйте параметры на уровне** организации. Используйте этот параметр, если вы хотите, чтобы пользователь использует параметры в **параметрах** всей организации. 
     - **Islands** — используйте этот параметр, если вы хотите, чтобы пользователь Skype для бизнеса и Teams. 
     - **Skype для бизнеса только**. Используйте этот параметр, если вы хотите, чтобы пользователь Skype для бизнеса.
     - **Skype для бизнеса с Teams** совместной работы. Используйте этот параметр, если вы хотите, чтобы пользователь Skype для бизнеса в дополнение к использованию Teams для совместной работы групп (каналов).
      - **Skype для бизнеса совместной Teams** и собраний. Используйте этот параметр, если вы хотите, чтобы пользователь Skype для бизнеса в дополнение к использованию Teams для групповой совместной работы (каналов) и Teams собраний.
     - **Teams только**. Используйте этот параметр, если вы хотите, чтобы пользователь Teams. Пользователь по-прежнему сможет присоединяться к Skype для бизнеса собраний.
4. Если вы выбрали любой  режим сосуществования, отличный от использования параметров на уровне **организации, вы** можете включить уведомления в приложении Skype для бизнеса пользователя, которое в ближайшее время Teams будет обновлено. Это уведомление можно включить для пользователя, включив параметр **"Уведомить Skype для бизнеса пользователя**".
5. После **внесения** изменений нажмите кнопку "Сохранить".
