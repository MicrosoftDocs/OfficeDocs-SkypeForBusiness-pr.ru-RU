# <a name="setting-your-coexistence-and-upgrade-settings"></a>Настройка сосуществования и обновления


Когда вы обновляете пользователей Skype для бизнеса, у вас есть несколько вариантов, с помощью которых вы можете сделать этот процесс бесперебойной работе пользователей. Вы можете настроить параметры сосуществования и обновления для всех пользователей в Организации одновременно или изменить параметры для одного или множества пользователей в Организации. Обратите внимание, что в более ранних версиях клиентов Skype для бизнеса эти параметры могут не поддерживаться. Дополнительные сведения о версиях клиента Skype для бизнеса можно найти на [странице загрузки и обновления в Skype для бизнеса](https://docs.microsoft.com/skypeforbusiness/software-updates). 

[С помощью Skype для бизнеса](coexistence-chat-calls-presence.md)вы можете лучше понять доступные для вас режимы [и ознакомиться с их сосуществованием в Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md) .

## <a name="set-upgrade-options-for-all-users-in-your-organization"></a>Настройка параметров обновления для всех пользователей в Организации

![Значок с логотипом Microsoft Teams](media/teams-logo-30x30.png) **Использование центра администрирования Microsoft Teams**

1. В [центре администрирования Microsoft Teams](https://admin.teams.microsoft.com/)на панели навигации слева перейдите к > **обновлению Teams**в **масштабе организации**. 

2. В верхней части страницы **Обновление Teams** измените указанные ниже параметры.
    - Установка режима **сосуществования** .
        - **О-ва** — используйте этот параметр, если вы хотите, чтобы пользователи могли одновременно использовать Skype для бизнеса и Teams.
        - **Только Skype для бизнеса** . Этот параметр используется только в том случае, если вы хотите, чтобы ваши пользователи использовали только Skype для бизнеса.
        - **Skype для бизнеса с совместной работой в Teams** — используйте этот параметр, если вы хотите, чтобы пользователи использовали Skype для бизнеса в дополнение к совместной работе с группами (каналы).
        - **Skype для бизнеса с совместной работой и собраниями в Teams** — используйте этот параметр, если вы хотите, чтобы пользователи использовали Skype для бизнеса в дополнение к использованию групп для совместной работы (каналов) и собраний Teams.
        - Если вы хотите, чтобы пользователи могли использовать только группы Teams, используйте этот параметр **только в Teams** . Обратите внимание, что даже с этим параметром пользователи по-прежнему могут присоединяться к собраниям в Skype для бизнеса.
        
    - Установите параметр **уведомлять пользователей Skype для бизнеса, что Teams доступен для обновления**. Если включить этот параметр, пользователи Skype для бизнеса смогут получать эти сведения о том, что они вскоре будут обновлены до приложения Teams.
    - Установка **предпочтительного приложения для присоединения к собраниям Skype для бизнеса**. Этот параметр определяет, какое приложение используется для присоединения к собраниям в Skype для бизнеса и учитывается независимо от значения режима сосуществования.
      - **Приложение "собрания Skype"**
      - **Skype для бизнеса с ограниченными возможностями**
    - Укажите, нужно ли **скачивать приложение Teams в фоновом режиме для пользователей Skype для бизнеса**.  Этот параметр автоматически загружает приложение Teams для пользователей, работающих под управлением Skype для бизнеса в Windows. Она обрабатывается только в том случае, если режим сосуществования для пользователя — только Teams или если в Skype для бизнеса включены уведомления о состоянии ожидания обновления.
3. После внесения изменений нажмите кнопку **сохранить** .

## <a name="set-upgrade-options-for-a-single-user-in-your-organization"></a>Настройка параметров обновления для отдельного пользователя в Организации

![Значок с логотипом Microsoft Teams](media/teams-logo-30x30.png) **Использование центра администрирования Microsoft Teams**

1. На панели навигации слева перейдите в раздел **Пользователи**и выберите пользователя из списка. 
2. На вкладке **учетная запись** пользователя в разделе **Обновление Teams**нажмите кнопку **изменить**.
3. Вы можете установить **режим сосуществования**. Выберите один из следующих вариантов:
     - **Используйте параметры в масштабах Организации** — используйте этот параметр, если вы хотите, чтобы пользователь использовал параметры, указанные в параметрах **уровня Организации** . 
     - **О-ва** – используйте этот параметр, если вы хотите, чтобы пользователь мог использовать Skype для бизнеса и Teams. 
     - **Только для Skype для бизнеса** – этот параметр используется, если вы хотите, чтобы пользователь использовал Skype для бизнеса.
     - **Skype для бизнеса с совместной работой в Teams** — используйте этот параметр, если вы хотите, чтобы пользователь использовал Skype для бизнеса в дополнение к использованию групп для совместной работы (каналов).
      - **Skype для бизнеса с совместным использованием команд и собраний** — используйте этот параметр, если вы хотите, чтобы пользователь использовал Skype для бизнеса в дополнение к использованию групп для совместной работы (каналов) и собраний Teams.
     - **Только для Teams** – этот параметр используется, если вы хотите, чтобы пользователь использовал только группы Teams. Пользователь по-прежнему сможет присоединиться к собраниям Skype для бизнеса.
4. Если вы выбрали **режим совместного существования** , отличный от **параметров в масштабе организации**, вы можете включить уведомления в приложении Skype для бизнеса, которое будет вскоре переходить на Teams. Вы можете включить это уведомление для пользователя, включив параметр **уведомлять пользователя Skype для бизнеса** .
5. После внесения изменений нажмите кнопку **сохранить** .
