# <a name="prepare-your-environment-for-upgrading-to-teams"></a>Подготовка среды для обновления до группы

Для успешного обновления группы в вашей организации, важно проверить вашей текущей Скайп для бизнес-среде и подготовка вашей сети. Подготовка текущей среды представлены сведения о убедитесь теперь высокое качество пользовательского интерфейса, в дополнение к улучшению качества пользователь опыт в группах. Планирование отдельные действия может помочь ускорение развертывания и убедитесь в том, что еще не пропущена любые элементы важные действия.

Выполните эти действия в рамках подготовки проверки готовности к вашей пользователя.

- [Подготовка ИТ-специалистов](upgrade-prepare-IT-pros.md) , чтобы гарантировать, что у них есть они необходимы для успешного обновления пути.
- Убедитесь, что среда соответствует всех [необходимых компонентов](upgrade-plan-journey-prerequisites.md)и понять зависимости между службами Office 365 и рабочих групп.
- [Оценка среды](upgrade-plan-journey-evaluate-environment.md) , выполняя окружающей среды обнаружения с помощью образца анкеты для подтверждения готовность своей организации предпринимать успешного обновления пути к группам.
- [Подготовка сети](upgrade-prepare-environment-prepare-network.md) через планирования, подготовки и получение действия все необходимые исправления для вашей сети для поддержки группам рабочих нагрузок.
- [Подготовка службы](upgrade-prepare-environment-prepare-service.md) для развертывания с помощью входящая контрольные списки для обеспечения Готово к поддерживает перенос пользователей из Скайп для бизнеса группам конфигурации группы.
