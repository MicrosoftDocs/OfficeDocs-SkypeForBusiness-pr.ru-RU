# <a name="plan-for-microsoft-365-government---dod-deployments"></a>Планирование для государственных учреждений 365 Microsoft - развертываний министерства обороны США

— Это руководство для ИТ-специалистов, которые ведут развертываний Office 365 в "мне НРАВИТСЯ" Федеральное государственных сущностей или других лиц, которые обрабатывают данные, подлежащие правительственным нормативам и требования, где подходят для использования государственных 365 Microsoft – министерства обороны США соответствовать следующим требованиям.

> [!NOTE]
> Если ваша организация уже выполнены государственных 365 Microsoft – Требования министерства обороны США и применяется для и были приняты в программу, можно пропустить шаги 1 и 2 и перейти к шагу 3.

## <a name="step-1-determine-whether-your-organization-needs-microsoft-365-government---dod-and-meets-eligibility-requirements"></a>Шаг 1. Определяют, требуется Microsoft 365 государственных организаций - министерства обороны США и соответствует требованиям к вашей организации. 

Правительство Microsoft 365 - среде министерства обороны США соблюдение США государственных требований для облачных служб. В дополнение к использовать функции и возможности Office 365, организации используют преимущества следующие функции, которые являются уникальными для государственных учреждений 365 Microsoft – министерства обороны США:

- Содержимое вашей организации клиента логически подразделить от клиента контента в службах профессиональной Office 365 корпорацией Майкрософт.
- В Соединенных Штатах Америки будет храниться содержимое вашей организации клиента.
- Доступ к содержимому вашей организации клиента ограничено промежуточная численности специалистов корпорации Майкрософт.
- Microsoft 365 государственных – министерства обороны США следует сертификаты и другие навыки, необходимые для клиентов в государственном секторе.

Можно найти дополнительные сведения о государственных 365 Microsoft – предложение для НАС правительственных клиентов в [Office 365 государственных планы](https://products.office.com/government/compare-office-365-government-plans), включая [требования](https://products.office.com/government/compare-office-365-government-plans#EligibilityRequirements)министерства обороны США.

[Описание службы Office 365 США государственных](https://docs.microsoft.com/en-us/office365/servicedescriptions/office-365-platform-service-description/office-365-us-government/office-365-us-government) : описание преимуществ платформы, которые центром на обеспечение соответствия требованиям в США.


> [!Tip]
> Может потребоваться перенос таблиц данных в описание службы в книгу Excel и добавьте два столбца: **Relevant для организации планировщика** и **потребностям организации планировщика**. Затем можно просмотреть этот список с коллегами, чтобы убедиться, что данная служба отвечает требованиям вашей организации.

## <a name="step-6-deploy-teams-for-collaboration"></a>Шаг 6. Развертывание групп для совместной работы

После использовали onboarded для государственных учреждений 365 Microsoft – министерства обороны США, можно следовать стандартное развертывание подход с использованием партнеру выбранного встроенного и [эту](https://www.microsoft.com/fasttrack) службу.

Когда вы будете готовы, развертывание групп для [включения совместной работы в рамках вашей организации с помощью групп и каналы](teams-overview.md). Убедитесь, что взаимодействовать с внедрение и управление изменение группы или группы чемпионатов.
