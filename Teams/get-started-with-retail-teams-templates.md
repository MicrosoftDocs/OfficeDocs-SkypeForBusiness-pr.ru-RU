# <a name="create-a-team-using-retail-team-templates"></a>Создание команды с использованием шаблонов розничных команд

Шаблоны команд Майкрософт позволяют быстро и легко создавать команды, предоставляя предопределяющий шаблон параметров, каналов и предварительно установленных приложений.

Шаблоны команд — это встроенные определения структур группы, разработанные для потребностей розничного продавца. Шаблоны групп можно использовать для быстрого создания типов команд, которые хорошо работают для розничных продавцов, и их развертывания в организации. Вы также можете расширить шаблоны команд для создания команд, специально разработанных для ваших организационных потребностей.

В этой статье мы познакомимся с каждым шаблоном команды и посоветуем использовать их.

Эта статья предназначена для вас, если вы отвечаете за планирование, развертывание и управление несколькими командами в своей организации розничной торговли. Вы уже развернули службу Teams в своей организации. Если вы еще не развернули Teams, см. статью [Как развернуть Microsoft Teams](./deploy-overview.md).

Дополнительные информацию о шаблонах команд в целом можно найти в этой ссылке: Начало [работы с шаблонами команд.](get-started-with-teams-templates.md)

| Кто | Используемый метод: |
| ---- | --------- |
| Администраторы и ИТ-специалисты | [Используйте центр Teams для](#use-the-team-templates-in-the-teams-admin-center) создания команд на основе шаблонов розничных групп.|
| Разработчики и системные интеграторы | [Используйте microsoft Graph](#use-the-team-templates-with-the-microsoft-graph) для создания команд на основе шаблонов розничных групп. |

### <a name="manager-collaboration"></a>Взаимодействие руководителей

## <a name="use-the-team-templates-with-the-microsoft-graph"></a>Использование шаблонов команд в microsoft Graph

### <a name="store-template"></a>Шаблон магазина

Шаблон магазина идеально подходит для создания команды, представляющей отдельно расположенный розничный магазин. С помощью шаблона магазина можно создать команду для каждого расположения розничного магазина в организации.

| Тип базового шаблона | baseTemplateId | Свойства базового шаблона |
| ------------------ | -------------- | ----------------------------------------------------- |
| Розничная торговля — <br>Магазин | `https://graph.microsoft.com/beta/`<br>`teamsTemplates('retailStore')`| Каналы <ul><li>Передача смен\*</li><li>Обучение\*</li></ul>\*Автоматически добавленные в избранное каналы<br><br>Свойства команды <ul><li>Установлен параметр видимости команды "Общедоступная"</li></ul> <br>Разрешения для участников <ul><li>Не может создавать, обновлять и удалять каналы </li><li>Не может добавлять и удалять приложения </li><li>Не может создавать, обновлять и удалять вкладки</li><li>Не может создавать, обновлять и удалять соединители</li><ul>|
||||

Рекомендуемые способы настройки шаблона магазина для организации:

- Если в вашей организации есть отделы в каждом магазине, добавьте канал для каждого отдела. Добавление канала упрощает взаимодействие и совместную работу в отделе.

- Если в вашей организации есть внутренние веб-сайты (например, сайт SharePoint), рассмотрите возможность закрепления их в качестве вкладок в соответствующем канале команды. Инструкции можно найти в этой ссылке: Начало работы с [шаблонами](get-started-with-teams-templates.md) команд.

### <a name="manager-collaboration-template"></a>Шаблон взаимодействия руководителей

Шаблон совместной работы руководителя — это еще один из шаблонов команд, разработанных для розничной торговли. Шаблон взаимодействия руководителей идеально подходит, чтобы создать команду для группы руководителей с целью совместной работы в магазинах, регионах и т. д. Например, если в вашей организации есть регионы, вы можете создать команду взаимодействия руководителей для Московской области и включить в нее всех руководителей магазинов в этом регионе, а также регионального руководителя.

||||

Рекомендуемые способы настройки шаблона взаимодействия руководителей для организации:

- Если в вашей организации есть внутренние веб-сайты (например, сайт SharePoint), важные для руководителей, рассмотрите возможность закрепления их в качестве вкладок в соответствующем канале команды. Инструкции можно найти [в](get-started-with-teams-templates.md) документации по шаблонам группы Майкрософт.

## <a name="how-to-use-first-party-templates"></a>Использование собственных шаблонов

Чтобы использовать эти шаблоны, измените свойство template@odata.bind в тексте запроса со standard на TemplateID, указанные выше.  Дополнительные сведения о развертывании шаблонов группы см. в статье microsoft Graph о том, как [создать команду.](/graph/api/team-post?view=graph-rest-beta)

> [!NOTE]
> Каналы в шаблоне будут автоматически созданы на вкладке "Общий".

### <a name="example-store-template-extension-script"></a>Пример: сценарий расширения шаблона магазина

``` PowerShell
{
  "template@odata.bind": "https://graph.microsoft.com/beta/teamsTemplates('retailStore')",
  "DisplayName": "Contoso Store",
  "Description": "Team for all staff in Contoso Store",
  "Channels": [
    {
      "displayName": "Additional store channel",
      "IsFavoriteByDefault": false
    }
  ]
}
```
