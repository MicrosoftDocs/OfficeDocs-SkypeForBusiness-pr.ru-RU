# <a name="get-started-with-teams-templates-in-retail"></a>Начало работы с шаблонами Teams для розничной торговли

Шаблоны Teams позволяют быстро и легко создавать команды, предоставляя предопределяющий шаблон настроек, каналов и предварительно установленных приложений.

Шаблоны Teams имеют встроенные определения структур групп, разработанных для потребностей розничного продавца. Вы можете использовать шаблоны Teams для быстрого создания типов команд, которые хорошо работают для розничных продавцов, и их развертывания в организации. Вы также можете расширить шаблоны Teams для создания команд, специально разработанных для ваших организационных потребностей.

В этой статье мы введем каждый из шаблонов Teams и посоветуем использовать их.

Эта статья для вас, если вы отвечаете за планирование, развертывание и управление несколькими командами в розничной сети. Вы уже развернули службу Teams в своей организации. Если вы еще не начали работу с Teams, начните с чтения этой [статьи.](How-to-roll-out-teams.md)

Подробнее о шаблонах команд читайте в теме "Начало [работы с шаблонами Teams".](get-started-with-teams-templates.md)

## <a name="store-template"></a>Шаблон Магазина

Шаблон Магазина идеально подходит для того, чтобы создать команду для представления отдельного расположения в розничном магазине. С помощью шаблона "Магазин" можно создать команду для каждого расположения в розничном магазине в организации.

| Тип базового шаблона | baseTemplateId | Свойства, которые доступны с этим базовым шаблоном |
| ------------------ | -------------- | ----------------------------------------------------- |
| Розничная торговля — <br>Хранилище | `https://graph.microsoft.com/beta/`<br>`teamsTemplates('retailStore')`| Каналы <ul><li>Смены\*</li><li>Обучение\*</li></ul>\*Автоматически избранные каналы<br><br>Свойства группы <ul><li>Видимость группы с установленным на "Общедоступный"</li></ul> <br>Разрешения для членов группы <ul><li>Не может создавать, обновлять или удалять каналы </li><li>Не может добавлять и удалять приложения </li><li>Не может создавать, обновлять или удалять вкладки</li><li>Не может создать, обновить или удалить соединители</li><ul>|
||||

Рекомендуемые способы настройки шаблона Магазина для организации:

- Если в вашей организации есть отделы в каждом магазине, добавьте канал для каждого отдела. Добавление канала упрощает взаимодействие и совместную работу внутри отдела.

- Если в вашей организации есть внутренние веб-сайты (например, сайт SharePoint), их можно закрепление в качестве вкладок в соответствующем канале группы. Инструкции [по началу работы с шаблонами Teams](get-started-with-teams-templates.md) можно найти в этой ссылке.

## <a name="manager-collaboration-template"></a>Шаблон совместной работы руководителя

Шаблон совместной работы руководителя — это еще один шаблон Teams, разработанный для розничного продавца. Шаблон совместной работы руководителя идеально подходит для создания команды для группы руководителей для совместной работы в магазинах и регионах и не только. Например, если в вашей организации есть регионы, вы можете создать команду совместной работы руководителя для Новской области и включить в нее всех менеджеров магазинов в этом регионе, а также регионального руководителя в этом регионе.

| Тип базового шаблона | baseTemplateId | Свойства, которые доступны с этим базовым шаблоном |
| ------------------ | -------------- | ----------------------------------------------------- |
| Розничная торговля — <br>Хранилище | `https://graph.microsoft.com/beta/`<br>`teamsTemplates('retailManagerCollaboration')`| Каналы <ul><li>Операции\*</li><li>Обучение\*</li></ul>\*Автоматически избранные каналы<br><br>Свойства группы <ul><li>Видимость группы : "Частная"</li></ul> <br>Разрешения для членов группы <ul><li>Создание, обновление и удаление каналов </li><li>Может добавлять и удалять приложения </li><li>Создание, обновление и удаление вкладок</li><li>Создание, обновление и удаление соединители</li><ul>|
||||

Рекомендуемые способы настройки шаблона совместной работы руководителя в организации:

- Если в вашей организации есть какие-либо внутренние веб-сайты, например сайт SharePoint, которые важны для руководителей, их можно закрепление в качестве вкладок в соответствующем канале группы. Инструкции вы можете найти в документации по шаблонам [Microsoft Teams.](get-started-with-teams-templates.md)

## <a name="how-to-use-first-party-templates"></a>Использование шаблонов от первой стороны

Чтобы использовать эти шаблоны, измените свойство template@odata.bind" в теле запроса со стандартного на шаблон TemplateIDs выше.  Дополнительные сведения о развертывании шаблонов Teams см. в статье Microsoft Graph о том, как [создать команду.](https://docs.microsoft.com/graph/api/team-post?view=graph-rest-beta)

> [!NOTE]
> Каналы в шаблоне будут автоматически созданы на вкладке "Общие".

### <a name="example-store-template-extension-script"></a>Пример: сценарий расширения шаблона Store

``` PowerShell
{
  "template@odata.bind": "https://graph.microsoft.com/beta/teamsTemplates('retailStore')",
  "DisplayName": "Contoso Store",
  "Description": "Team for all staff in Contoso Store",
  "Channels": [
    {
      "displayName": "Additional store channel",
      "IsFavoriteByDefault": false
    }
  ]
}
```
