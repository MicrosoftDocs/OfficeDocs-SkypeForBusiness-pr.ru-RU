# <a name="create-a-team-using-teams-retail-templates"></a>Создание команды с помощью розничных шаблонов Teams

С помощью шаблонов Microsoft Teams можно быстро и удобно создавать команды: готовые шаблоны содержат параметры, каналы и предустановленные приложения.

Шаблоны Teams содержат предварительно созданные определения структур команд, предназначенные для розничной торговли. Вы можете использовать шаблоны Teams, чтобы быстро создавать команды, подходящие для розничной торговли, и развертывать их в организации. Вы также можете расширить шаблоны Teams, чтобы создавать команды, адаптированные под конкретные потребности вашей организации.

В этой статье вы познакомитесь с каждым шаблоном Teams и рекомендуемыми способами их использования.

Эта статья предназначена для вас, если вы отвечаете за планирование, развертывание и управление несколькими командами в своей организации розничной торговли. Вы уже развернули службу Teams в своей организации. Если вы еще не развернули Teams, см. статью [Как развернуть Microsoft Teams](./deploy-overview.md).

Другие общие сведения о шаблонах команд см. в статье [Начало работы с шаблонами Teams](get-started-with-teams-templates.md).

| Кто | Используемый метод: |
| ---- | --------- |
| Администраторы и ИТ-специалисты | [Используйте Центр администрирования Teams для](#use-the-teams-templates-in-the-teams-admin-center) создания команд на основе шаблонов Teams в розничной сети.|
| Разработчики и системные интеграторы | [Используйте Microsoft Graph для](#use-the-teams-templates-with-the-microsoft-graph) создания команд на основе шаблонов Teams в розничной сети. |

Объедините своих сотрудников розничной торговли в едином интерфейсе для управления задачами, обмена документами и решения проблем клиентов. Интегрируйте дополнительные приложения для упрощения начала и завершения смен.

||||

### <a name="manager-collaboration"></a>Взаимодействие руководителей

Шаблон Совместной работы руководителя идеально подходит для создания команды для группы руководителей для совместной работы в магазинах или регионах и т. д. ## <a name="use-the-teams-templates-with-the-microsoft-graph"></a>Использование шаблонов Teams с Microsoft Graph

### <a name="store-template"></a>Шаблон магазина

Шаблон магазина идеально подходит для создания команды, представляющей отдельно расположенный розничный магазин. С помощью шаблона магазина можно создать команду для каждого расположения розничного магазина в организации.

| Тип базового шаблона | baseTemplateId | Свойства базового шаблона |
| ------------------ | -------------- | ----------------------------------------------------- |
| Розничная торговля — <br>Магазин | `https://graph.microsoft.com/beta/`<br>`teamsTemplates('retailStore')`| Каналы <ul><li>Передача смен\*</li><li>Обучение\*</li></ul>\*Автоматически добавленные в избранное каналы<br><br>Свойства команды <ul><li>Установлен параметр видимости команды "Общедоступная"</li></ul> <br>Разрешения для участников <ul><li>Не может создавать, обновлять и удалять каналы </li><li>Не может добавлять и удалять приложения </li><li>Не может создавать, обновлять и удалять вкладки</li><li>Не может создавать, обновлять и удалять соединители</li><ul>|
||||

Рекомендуемые способы настройки шаблона магазина для организации:

- Если в вашей организации есть отделы в каждом магазине, добавьте канал для каждого отдела. Добавление канала упрощает взаимодействие и совместную работу в отделе.

- Если в вашей организации есть внутренние веб-сайты (например, сайт SharePoint), рассмотрите возможность закрепления их в качестве вкладок в соответствующем канале команды. Инструкции см. в статье [Начало работы с шаблонами Teams](get-started-with-teams-templates.md).

### <a name="manager-collaboration-template"></a>Шаблон взаимодействия руководителей

Шаблон взаимодействия руководителей — это еще один из шаблонов Teams, предназначенных для розничной торговли. Шаблон взаимодействия руководителей идеально подходит, чтобы создать команду для группы руководителей с целью совместной работы в магазинах, регионах и т. д. Например, если в вашей организации есть регионы, вы можете создать команду взаимодействия руководителей для Московской области и включить в нее всех руководителей магазинов в этом регионе, а также регионального руководителя.

||||

Рекомендуемые способы настройки шаблона взаимодействия руководителей для организации:

- Если в вашей организации есть внутренние веб-сайты (например, сайт SharePoint), важные для руководителей, рассмотрите возможность закрепления их в качестве вкладок в соответствующем канале команды. Инструкции см. в [документации по Microsoft Teams](get-started-with-teams-templates.md).

## <a name="how-to-use-first-party-templates"></a>Использование собственных шаблонов

Чтобы использовать эти шаблоны, измените свойство template@odata.bind в тексте запроса со standard на TemplateID, указанные выше.  Дополнительные сведения о развертывании шаблонов Teams см. в статье Microsoft Graph о том, как [создать команду](/graph/api/team-post?view=graph-rest-beta).

> [!NOTE]
> Каналы в шаблоне будут автоматически созданы на вкладке "Общий".

### <a name="example-store-template-extension-script"></a>Пример: сценарий расширения шаблона магазина

``` PowerShell
{
  "template@odata.bind": "https://graph.microsoft.com/beta/teamsTemplates('retailStore')",
  "DisplayName": "Contoso Store",
  "Description": "Team for all staff in Contoso Store",
  "Channels": [
    {
      "displayName": "Additional store channel",
      "IsFavoriteByDefault": false
    }
  ]
}
```
