Профконсультант — это Microsoft Teams для образования, которое работает на сайте LinkedIn, которое предоставляет учащимся высшего образования персонализированные рекомендации по переходу к карьере. [Подробнее](../career-coach.md).

## <a name="march-31-2021"></a>31 марта 2021 г.

### <a name="share-to-teams-from-outlook"></a>Поделиться с Teams из Outlook

Найдите **параметр Поделиться Teams** на ленте Outlook или в меню действий для сообщения электронной почты. Share to Teams поддерживает Outlook в Интернете, Outlook для Windows и новые Outlook для Mac Preview. [Подробнее](../teams-outlook-share-teams.md).

## <a name="march-23-2021"></a>23 марта 2021 г.

### <a name="supervised-chats"></a>Контрольные чаты

Контрольный чат позволяет назначенным преподавателям начинать чаты с учащимися и не позволяет учащимся начинать новые чаты, если только не присутствует соответствующий преподаватель. Если включен контроль чата, руководители не могут выйти из чатов, а другие участники не могут удалять их, обеспечивая правильное наблюдение за чатами, в которых участвуют учащиеся.
[Подробнее](../supervise-chats-edu.md).

### <a name="masking-phone-numbers"></a>Маскировка номеров телефонов

Мы выпустили новый параметр администратора для маскровки номеров телефонов для участников телефонного звонка, присоединяющихся к собранию. Администраторы могут скрыть телефонные номера от всех на собрании (кроме организатора), только для внешних людей или отключить маскировку. Если администраторы отключили этот параметр, телефонные номера будут полностью отображаться на собрании. [Подробнее](../ptsn-mask-phone-numbers.md).

### <a name="create-a-phone-system-auto-attendant"></a>Создание автозавода телефонной системы

Автозаверяющие могут звонить в вашу организацию и переходить в систему меню, чтобы поговорить с нужным отделом, очередью вызовов, человеком или оператором. Вы можете создавать автоотводы для своей организации с помощью Microsoft Teams центра администрирования или PowerShell. [Подробнее](../create-a-phone-system-auto-attendant.md).
