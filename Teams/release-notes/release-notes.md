# <a name="release-notes-for-microsoft-teams-admin-features"></a>Заметки о выпуске для функций администратора Microsoft Teams

Эти заметки о выпуске содержат сведения о новых возможностях администратора для Microsoft Teams.

## <a name="june-11-2020"></a>11 июня 2020 г.

### <a name="call-queues---presence-based-routing"></a>Очереди звонков — маршрутизация на основе присутствия

Если этот параметр отключен, звонки будут перенаправляться в агенты, которые выбрал для приема звонков, независимо от состояния присутствия. При включении входные агенты будут получать вызовы только в том случае, если их состояние присутствия доступно. [Подробнее](../create-a-phone-system-call-queue.md).


## <a name="june-10-2020"></a>10 июня 2020 г.

### <a name="call-queues---longest-idle-routing"></a>Очереди вызовов — незанятая маршрутизация в течение длительного времени
Когда вы выберете этот параметр, входящие звонки будут перенаправляться в агенты звонков, которые еще не ответили на входящий звонок в течение более длительного периода времени. [Подробнее](../create-a-phone-system-call-queue.md).

## <a name="june-1-2020"></a>1 июня 2020 г.

### <a name="raise-hand-in-meetings"></a>Создание рук в собраниях

Теперь пользователи могут создавать виртуальные руки на собрании. Другие участники увидят свое руки рядом с именем на этапе собрания и рядом с вашим именем в списке.

## <a name="may-15-2020"></a>15 мая 2020 г.

### <a name="add-more-people-to-a-chat"></a>Добавление участников в чат

Теперь мы сделали так, чтобы в одном потоке чата можно было добавлять до 350 пользователей.

## <a name="may-5-2020"></a>5 мая 2020 г.

### <a name="easily-access-meeting-options-from-within-a-teams-meeting-in-progress"></a>Быстрый доступ к параметрам собрания в ходе собрания команд

Мы упростили организаторов собраний, чтобы быстро и легко изменить параметры докладчика и "зал ожидания" после начала собрания Teams, предоставив ссылку для быстрого доступа прямо в области "участники". Новые функциональные возможности будут представлены как для запланированных, так и для собраний "собрание сейчас".

### <a name="download-a-participant-report-in-a-teams-meeting"></a>Загрузка отчета участника на собрание Teams

Организаторов собраний, особенно преподавателей, часто требуется узнать, кто присоединился к собраниям Teams. Вскоре вы сможете скачать отчет участника, который находится в представлении списка, включающем время присоединения к собраниям и выходу за него. В представлении "список", организаторов собраний, можно скачать отчет, который включает в себя пользователей, которые присоединились к нему во время показа организатора. Эта функция доступна только в рамках собрания, пока собрание активно. Доступно на рабочем столе (Windows и Mac) и в Интернете. [Подробнее](../teams-analytics-and-reports/meeting-attendance-report.md).

## <a name="april-26-2020"></a>26 апреля 2020 г.

### <a name="include-system-audio-in-live-events"></a>Включение системного звука в Live Events

Выступающие и производители в Live Events теперь могут включать звук системы при совместном доступе к рабочему столу или экрану окна при событии Live. Это позволит вашим пользователям слышать любые звуковые данные, к которым вы хотите предоставить общий доступ.

## <a name="april-24-2020"></a>24 апреля 2020 г.

### <a name="team-and-channel-analytics"></a>Анализ группы и канала

В дополнение к Team Analytics вы также можете просматривать метрики и аналитики на уровне каналов. Мы также улучшили период времени в 90 дней, чтобы вы могли анализировать данные в течение более длительных периодов. Помимо этого, в этом выпуске также используются новые показатели и диаграммы, обрабатываются количество записей, ответов и собраний для группы или канала. [Подробнее](../teams-analytics-and-reports/view-analytics.md).

## <a name="april-20-2020"></a>20 апреля 2020 г.

### <a name="enable-organizers-to-change-lobby-settings-for-dial-in-participants"></a>Включение организаторов для изменения параметров "зал ожидания" для участников телефонного подключения

Этот параметр определяет, будут ли пользователи, находящиеся на телефоне, присоединиться к собранию напрямую или ждать в зале ожидания, независимо от того, кто **автоматически согласен** .

## <a name="april-8-2020"></a>8 апреля 2020 г.

### <a name="customize-meeting-video-backgrounds"></a>Настройка фона видео для собрания

Если вы собираетесь использовать видео, вы можете выбрать другие статические фоновые изображения. Это позволит вам отобразить это изображение, а не фактический фон места, где вы размещаются. [Подробнее](../meeting-policies-in-teams.md).

## <a name="april-1-2020"></a>1 апреля 2020 г.

### <a name="teams-for-realwear"></a>Teams для RealWear

Клиент Microsoft Teams для RealWear позволяет Firstline работников, использующих RealWear ruggedized, смонтированные носимых устройств, для работы с Teams 100 в процентах от экспертов, использующих видеосвязь в Teams. [Подробнее](../flw-realwear.md).

## <a name="march-31-2020"></a>31 марта 2020 г.

### <a name="tag-youre-it"></a>И все!

Создавайте Теги и назначайте им людей, чтобы можно было @mention группу, роль, подразделение и т. д. Владельцы группы, попробуйте выполнить ее для yourselves. Перейдите в группу, выберите **Дополнительные параметры**  >  **управления тегами**. [Подробнее](../manage-tags.md).

## <a name="march-30-2020"></a>30 марта 2020 г.

### <a name="policy-based-recording-for-teams-calls-and-meetings"></a>Запись на основе политики для звонков и собраний в Teams

Эта функция позволяет автоматически создавать записи на основе политик для звонков и собраний в Teams с помощью сертифицированных корпорацией Майкрософт поставщиков программного обеспечения.

## <a name="march-25-2020"></a>25 марта 2020 г.

### <a name="manage-your-collaboration-bar-devices-in-microsoft-teams"></a>Управление устройствами на панели совместной работы в Microsoft Teams

Администратор хочет управлять всеми устройствами на панели совместной работы в центре администрирования Microsoft Teams. Вы можете просматривать инвентаризацию устройств, выполнять определенные действия на устройствах и давать команды удаленно на все устройства.
