Прежде чем приступить к работе, убедитесь, что у вас есть следующие предварительные требования:

- Blue Yonder WFM версии 2020.3, 2021.1 или 2021.2.

    > [!NOTE]
    > Если у вас есть Blue Yonder WFM 2020.3 или 2021.1, примените исправление 2020.3.0.4 или 2021.1.0.3. Это исправление устраняет проблему, из-за которой пользователи получают постоянное сообщение об ошибке в shifts. Она также устраняет проблему, которая не позволяет пользователям обновлять свою доступность в shifts.

- Имя учетной записи службы WFM Blue Yonder, пароль и URL-адреса службы:

    - URL-адрес федеративной проверки подлинности
    - URL-адрес проверки подлинности файлов cookie
    - URL-адрес самообслуживания сотрудников
    - URL-адрес веб-API розничной торговли
    - URL-адрес API диспетчера сайтов
    - URL-адрес API администрирования

    Если у вас нет этой информации, обратитесь в службу поддержки Blue Yonder. Учетная запись создается на уровне корневого предприятия администратором предприятия Blue Yonder. Он должен иметь доступ к API, администратору клиента и диспетчеру магазинов. Учетная запись и пароль необходимы для создания подключения.
- Федеративная проверка подлинности единого входа включена в среде Blue Yonder WFM. Обратитесь в службу поддержки Blue Yonder, чтобы убедиться, что федеративный единый вход включен. Им потребуются следующие сведения:

    - federatedSSOValidationService: `https://wfmconnector.teams.microsoft.com/api/v1/fedauth/{tenantId}/6A51B888-FF44-4FEA-82E1-839401E9CD74/authorize` где {tenantId} — это идентификатор клиента
     - proxyHeader: X-MS-AuthToken

- По крайней мере одна команда настроена в Teams.
- Вы добавили Microsoft 365 системную учетную запись в качестве владельца команды для всех команд, которые вы хотите сопоставить.</br> [Создайте эту учетную запись в Microsoft 365](/microsoft-365/admin/add-users/add-users) и назначьте Microsoft 365 лицензию. Затем добавьте учетную запись в качестве владельца команды во все команды, которые вы хотите сопоставить. Соединитель Shifts использует эту учетную запись при синхронизации изменений shifts из Blue Yonder WFM.

    Мы рекомендуем создать учетную запись специально для этой цели и не использовать учетную запись пользователя.
