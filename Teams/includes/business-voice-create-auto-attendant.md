#### <a name="before-you-begin"></a>Подготовка к работе

Получите номера служб (номера служб — это особый тип номеров телефонов, которые используются автослужавами), необходимые для автозаполнения, которые должны быть доступны при прямом наборе номера из-за пределов организации. Это может [быть перенос номеров от другого поставщика](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) или запрос новых [номеров служб.](../getting-service-phone-numbers.md)

Каждому автозаводщику должна быть назначена лицензия телефонная система — виртуальный пользователь. При приобретении бизнес-голосовой почты вы также телефонная система лицензий виртуальных пользователей, поэтому вам, вероятно, не нужно запрашивать дополнительные данные. Тем не менее, если в будущем вам потребуется больше, вы можете получить их, следуя инструкциям в телефонная система [- Лицензия виртуального пользователя](../teams-add-on-licensing/virtual-user.md).

Если вы хотите, чтобы в праздники звонки [](../set-up-holidays-in-teams.md) автоответы перенаводились по-разному, создайте праздники, которые вы хотите использовать, прежде чем создавать автозавод.

Для каждого создаемго автозавода необходима учетная запись ресурса. Эта учетная запись аналогична учетной записи пользователя, за исключением того, что она связана с автосчетом или очередью вызовов, а не с пользователем. На этом шаге мы создадим учетную запись, назначим Microsoft 365 телефонная система *лицензию виртуального* пользователя, а затем назначим номер службы.

### <a name="create-a-resource-account"></a>Создание учетной записи ресурса

Вы можете создать учетную запись ресурса в Teams администрирования.

1. В центре Teams разметка **параметров** для всей организации и нажмите кнопку **Учетные записи ресурсов.**

2. Нажмите **Добавить**.

3. В области Добавление **учетной записи** ресурса введите отображаемого имени **,** **имя пользователя** и выберите автоотчет для типа **учетной записи ресурса.** 

    ![Снимок экрана: добавление пользовательского интерфейса учетной записи ресурса](../media/resource-account-add.png)

4. Нажмите кнопку **Сохранить**.

    Новая учетная запись появится в списке учетных записей.

    ![Снимок экрана: список учетных записей ресурсов](../media/resource-accounts-page.png)

### <a name="assign-a-license"></a>Назначение лицензии

Учетной записи *ресурса Microsoft 365 телефонная система лицензию виртуальный* пользователь.

1. В центре Microsoft 365 выберите учетную запись ресурса, для которой вы хотите назначить лицензию.

2. На **вкладке Лицензии и приложения** в области **Лицензии** выберите Microsoft 365 телефонная система **— виртуальный пользователь**.

3. Нажмите **кнопку Сохранить изменения.**

    ![Снимок экрана: пользовательский интерфейс назначения лицензий в центре Microsoft 365 администрирования](../media/resource-account-assign-virtual-user-license.png)

### <a name="assign-a-service-number"></a>Назначение номера службы

Если вы хотите, чтобы с этим автоотчетом можно было связаться по номеру телефона, назначьте этот номер учетной записи ресурса.

1. В Центре Teams администрирования на  странице Учетные записи ресурсов выберите учетную запись ресурса, для которой нужно назначить номер службы, и нажмите кнопку **Назначить/отозначить**.

2. В **Телефон числовом типе** выберите нужный тип.

3. В **поле Назначен номер телефона** найдите нужный номер и нажмите кнопку **Добавить.**

    ![Снимок экрана: пользовательский интерфейс назначения номера службы](../media/resource-account-assign-phone-number.png)

4. Нажмите кнопку **Сохранить**.

# <a name="step-2brattendant-general-info"></a>[Шаг 2 <br> Общие сведения о помощнике](#tab/general-info)

Настройка автозавода

1. В центре администрирования Teams **Разойдите в центр администрирования, разойдите на кнопку Голосовая** почта, выберите автоотехники **и** нажмите кнопку **Добавить**.

2. Введите имя автозавода в поле в верхней части окна.

3. Если вы хотите назначить оператора, укажите назначение для звонков на этот оператор. Это необязательно (но рекомендуется). Вы можете установить параметр **Оператор,** чтобы разрешить вызывателям выйти из меню и поговорить с назначенным человеком.

4. Укажите часовой пояс для этого автозавода. Часовой пояс используется для вычисления рабочих часов, если вы создаете отдельный поток зовов в не часах.

5. Укажите [поддерживаемый язык](../create-a-phone-system-auto-attendant-languages.md) для этого автозавода. Этот язык будет использоваться для системных голосовых подсказок. 

6. Выберите, нужно ли включить голосовой ввод. Если этот параметр включен, имя каждого параметра меню становится ключевым словом распознавания речи. Например, можно сказать "Один" для выбора параметра меню, назначенного клавише 1, или "Продажи", чтобы выбрать пункт меню "Продажи".

    ![Снимок экрана: параметры автозавода для ввода имени, оператора, часового пояса, языка и голосового ввода](../media/auto-attendant-general-info-page-new.png)

7. Нажмите кнопку **Далее**.

# <a name="step-3brcall-flow"></a>[Шаг <br> 3. Поток вызовов](#tab/call-flow)

Выбор параметров потока зова

1. Выберите, хотите ли вы воспроизведения приветствия, когда автоответ отвечает на звонок.

    Если вы выбрали **"Воспроизведения звукового** файла", с помощью Upload файла можно добавить записанное приветствие, сохраненное как звуковое.  WAV, .MP3 или . Формат WMA. Размер записи не должен быть больше 5 МБ.

    Если выбрать **Введите приветствие,** система прочитает текст, который вы введите (до 1000 символов), когда автозавет ответит на звонок.

    ![Снимок экрана: параметры приветствия сообщения](../media/auto-attendant-call-flow-greeting-message.png)

2. Выберите, как вы хотите перена маршрутизовыть звонок.

    Если вы **выберете Отключить,** автозаводщик повесит вызов.

    Если вы выбрали **Перенаправить** звонок , вы можете выбрать одно из назначений маршрутиации зовов.

    Если вы выбрали **Параметры меню**  Воспроизведения, вы  можете выбрать вариант Воспроизведения звукового файла или Ввести приветствие, а затем выбрать параметры меню и поиск по каталогу.

    ![Снимок экрана: параметры маршрутизов параметров перенаходящего звонка](../media/auto-attendant-call-flow-route-call-message.png)

3. Если вы хотите использовать для навигации клавиши набора номера, в области Настройка параметров меню выберите, что должно происходить при нажатии клавиши набора.  (Если вы создаете этого автозавода в качестве каталога организации, оставьте параметры набора параметров пустыми.)

    Вы можете настроить любую из этих клавиш для следующих назначений:

    - **Человек в организации —** это человек в вашей организации, который может принимать голосовые звонки.
    - **Голосовое приложение** — другой автозаводщик или очередь вызовов.
    - **Внешний номер телефона** — любой номер телефона. Используйте такой формат: +[код страны][код города][номер телефона]
    - **Голосовая почта** — почтовый ящик голосовой почты, связанный с Microsoft 365 группой, которую вы указали. Вы можете выбрать транскрибацию голосовой почты и сообщение "Пожалуйста, оставьте сообщение после сигнала". системный запрос.
    - **Оператор** — оператор, определенный для автозавода. Определение оператора является необязательным. Оператор можно определить как любой другой пункт назначения в этом списке.

    Мы рекомендуем установить для оператора клавишу 0.

    Для каждого параметра меню укажите следующее:

    - **Клавиша набора** номера — клавиша на телефонной клавиатуре для доступа к этому параметру.

    - **Голосовая** команда — определяет голосовую команду, которую может предоставить вызываемая команда для доступа к этому параметру, если включены голосовые данные. Оно может содержать несколько слов, например "Обслуживание клиентов" или "Операции и причины". 

    - **Перенаправить в** место, куда нужно перенаправить звонок при выборе этого параметра. Если вы перенаправляете учетную запись к автозаправщику или очереди вызовов, выберите связанную с ней учетную запись ресурса.

    ![Снимок экрана: параметры набора клавиш](../media/auto-attendant-call-flow-menu-options-complete.png)

4. Если вы хотите использовать этого автоотображающего в качестве каталога организации, в области Поиск по каталогу **выберите** Набрать **по имени**. Если вы вжмем этот параметр, звоня люди, которые смогут ввести имя пользователя или ввести его на телефонную клавиатуру. Любой пользователь в Интернете с телефонная система лицензией является подходящим пользователем, и его можно найти с помощью набора номера по имени. 

    (Вы можете выбрать **Dial by extension**(Набор номера по расширению), но расширение необходимо настроить в Azure Active Directory.)

5. Выбрав параметр поиска по **каталогу, нажмите** кнопку **Далее.**

# <a name="step-4brafter-hours"></a>[Шаг <br> 4. Не по часам](#tab/after-hours)

Для каждого автозавода можно настроить часы работы. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. В течение дня можно настроить часы работы с перерывом, а все часы, которые не считаются неавтными, считаются неавтными. Вы можете настроить различные параметры обработки входящих параметров обработки параметров звонка и приветствия в течение часа.

В зависимости от настройки автозаполнения и очередей звонков может потребоваться указать маршрутику звонков только для автозаполнения с прямыми номерами телефонов.

Если вам нужна отдельная маршрутия для вызывающих телефонных вызовов в нечасовом времени, укажите часы работы для каждого дня. Нажмите **кнопку Добавить новое** время, чтобы указать несколько наборов часов для заданного дня, например, чтобы указать перерыв на обед.

![Снимок экрана: параметры дня и времени после окончания дня](../media/auto-attendant-business-hours.png)

После того как вы указали часы работы, выберите параметры маршрутизов параметров для неавтных часов. Доступны те же параметры, что и для маршрутичности параметров в часы, указанные в шаге **3 —поток зовов.**

Когда **все будет готово,** нажмите кнопку Далее.

# <a name="step-5brholidays"></a>[Шаг <br> 5. Праздники](#tab/holidays)

В праздники звонки автоответам могут перенанастраиться не так, как в другие дни. (Если вы не хотите использовать другой поток звонка для праздников, вы можете пропустить этот шаг.)

Для каждого настроенного праздника ваш автозаводщик может использовать поток зовов. Вы можете добавить до 20 запланированных праздников для каждого автосекретаря.

1. На странице параметров звонка Праздник нажмите кнопку **Добавить**.

2. Введите название этого параметра праздников.

3. В **dropdown Holiday** (Праздники) выберите праздник, который вы хотите использовать.

4. Выберите нужный тип приветствия.

    ![Снимок экрана: параметры поздравительных и праздничных приветствий](../media/auto-attendant-holiday-greeting.png)

5. Выберите, хотите ли вы **отключить или** **перенаправить** звонок.

6. Если вы решили перенаправить звонок, выберите пункт назначения маршрутиации для звонка.

    ![Снимок экрана: параметры действий по праздничным звонкам](../media/auto-attendant-holiday-actions.png)

7. Нажмите кнопку **Сохранить**.

    Повторите эти процедуры для каждого дополнительного праздника.

    ![Снимок экрана: параметры праздников со списком праздников](../media/auto-attendant-holiday-call-settings.png)

    Когда вы добавите все праздники, нажмите кнопку **Далее.**

# <a name="step-6brdirectory-members"></a>[Участники каталога, шаг 6 <br>](#tab/dial-scope)

Область *набора номера* определяет пользователей, доступных в каталоге, если вызываемая звонок использует по имени или по телефону по расширению. По умолчанию **все пользователи в** Сети включают всех пользователей в организации, которые являются пользователями Online с телефонная система лицензией.

Вы можете включить или исключить определенных  пользователей,  выбрав настраиваемую группу пользователей в группе Включить или Исключить и выбрав одну или несколько групп Microsoft 365, списков рассылки или групп безопасности.  Например, может потребоваться исключить руководителей организации из каталога набора номера. (Если пользователь есть в обоих списках, он будет исключен из каталога.)

![Снимок экрана: область набора номера, включаемая и исключаемая](../media/auto-attendant-dial-scope.png)

> [!NOTE]
> Для того чтобы новый пользователь указал свое имя в каталоге, может потребоваться до 36 часов.

После настройки области набора номера нажмите кнопку **Далее.**

# <a name="step-7brresource-accounts"></a>[Шаг <br> 7. Учетные записи ресурсов](#tab/resource-accounts)

У всех автозаводников должна быть связанная учетная запись ресурса.  Автослужам первого уровня потребуется по крайней мере одна учетная запись ресурса с связанным номером службы. При желании вы можете назначить автослужбу несколько учетных записей ресурсов с отдельным номером службы.

Добавление учетной записи ресурса

1. Нажмите **кнопку** Добавить и найдите учетную запись, которую вы хотите добавить. Нажмите **кнопку** Добавить и выберите **добавить**.

    ![Снимок экрана: панель добавления учетных записей для учетных записей ресурса](../media/auto-attendant-add-resource-account.png)

2. Завершив добавление учетных записей службы, нажмите кнопку **Отправить**.

    ![Снимок экрана: список учетных записей ресурсов с учетной записью ресурса с номером назначенной услуги](../media/auto-attendant-resource-account-assigned.png)

    В этом случае конфигурация автозаполнеющего будет завершена.
