# <a name="simplify-business-processes-in-retail-environments"></a>Упрощение бизнес-процессов в средах розничной торговли

Вы можете использовать Teams, Power Apps и Power BI, чтобы упростить бизнес-процессы в своих командах розничной торговли.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWRzfc]

> [!NOTE]
> Дополнительные возможности для сред розничной торговли доступны в облаке Microsoft Cloud для розничной торговли, которое содержит возможности из Dynamics 365 и Microsoft Azure, а также функции Microsoft 365. Дополнительные сведения см. в статье [Microsoft Cloud для розничной торговли](/industry/retail/).

## <a name="manage-schedules-tasks-lists-and-approvals"></a>Управление расписаниями, задачами, списками и утверждениями

Teams содержит множество приложений, которые помогут вам и вашей команде розничной торговли управлять повседневной работой:

- С помощью приложения [Смены](shifts-for-teams-landing-page.md) вы можете создавать и обновлять расписания для команды своего магазина. Ваши работники могут регистрироваться в сменах, при необходимости обмениваться сменами и входить при начале работы.
- Публикуйте и отслеживайте [задачи](../manage-tasks-app.md) для сотрудников, чтобы они знали об обязательных действиях на каждый день.
- Используйте [Утверждения](../approval-admin.md) прямо в чате, чтобы быстро утверждать выполненные задачи или отправляемые запросы.
- Отслеживайте все остальное с помощью [Списков](../manage-lists-app.md).

## <a name="conduct-store-walks-and-inventories"></a>Посещение магазина и инвентаризация

Добавьте Power Apps для создания приложений организации с малым использованием кода или совсем без кода. Вы можете создавать собственные приложения для подключения бизнес-данных, чтобы управлять запасами или посещать магазин с проверкой.

Дополнительные сведения: [Power Apps](../manage-power-platform-apps.md) и [Интеграция Power Apps и Microsoft Teams](/powerapps/teams/overview).

## <a name="track-key-performance-indicators-kpis-with-power-bi-reports"></a>Отслеживание ключевых показателей эффективности (KPI) с помощью отчетов Power BI

Делитесь интерактивным контентом Power BI и совместно используйте его в каналах и чатах Microsoft Teams. Вы можете добавить вкладку [Power BI](/microsoftteams/platform/tabs/what-are-tabs) в Teams, чтобы внедрять интерактивные отчеты и общаться в чате Teams о своих отчетах, а также получать уведомления о важных событиях Power BI прямо в ленте новостей Teams.

Дополнительные сведения: [Совместная работа в Microsoft Teams с помощью Power BI](/power-bi/collaborate-share/service-collaborate-microsoft-teams).
