# <a name="assignments-in-teams-for-education"></a>Задания в Teams для образовательных учреждений

Функции "Задания" и "Оценки" в Teams для образовательных учреждений позволяют преподавателям назначать учащимся задачи, задания или тесты. Преподаватели могут управлять временными шкалами заданий, инструкциями, добавлять ресурсы для списания, оценки с помощью рубрик и т. д. Они также могут отслеживать ход занятий и отдельных учащихся на вкладке "Оценки".

[Узнайте больше о заданиях и оценках в Teams для образовательных сфере.](https://support.office.com/article/microsoft-teams-5aa4431a-8a3c-4aa5-87a6-b6401abea114?ui=en-US&rs=en-IE&ad=IE#ID0EAABAAA=Assignments)

> [!Note]
> Подробные сведения о заданиях Teams на различных платформах см. в [различных платформах.](https://support.microsoft.com/office/teams-features-by-platform-debe7ff4-7db4-4138-b7d0-fcc276f392d3)

## <a name="assignments-integrations-in-the-microsoft-teams-admin-center"></a>Интеграция заданий в Центре администрирования Microsoft Teams

С помощью параметров администрирования в Центре администрирования Microsoft Teams вы можете включить или отключить функции для преподавателей в организации и их учащихся. Ниже параметров, связанных с заданиями.

<a name="#bkemaildigest"> </a>
### <a name="weekly-guardian-email-digest"></a>Еженедельный электронный дайджест для опекунов


Сообщения электронной почты для опекунов отправляются родителям и опекунам по выходным. Сообщение электронной почты содержит сведения о заданиях за предыдущую и предстоящую неделю. Вы можете настроить синхронизацию с родителями и опекунами с помощью [School Data Sync.](https://docs.microsoft.com/schooldatasync/parent-contact-sync)

1. Импортируйте контактные данные родителей с помощью синхронизации для родителей и опекунов в SDS. Инструкции о том, как включить синхронизацию для родителей и опекунов, см. в инструкциях по включению синхронизации для родителей и [опекунов.](https://docs.microsoft.com/schooldatasync/parent-contact-sync#enabling-parent-and-guardian-sync)

2. Включите параметр "Опекун" в Центре администрирования Microsoft Teams, так как этот параметр по умолчанию отключен. Это позволит преподавателям отправлять еженедельный дайджест.

   > [!NOTE]
   > Преподаватели могут отказаться от дайджеста, отобрав параметры в личной группе класса (параметры заданий > сообщения электронной почты для родителей и **опекунов).**

Чтобы убедиться, что родители получат сообщение, должны быть истинными следующие три пункта:

 - Адрес электронной почты, прикрепленный к профилю учащегося в SDS и помеченный как _родитель_ или _опекун._ Подробные сведения см. в [формате файла синхронизации](https://docs.microsoft.com/schooldatasync/parent-contact-sync-file-format)для родителей и опекунов.

 - Учащиеся относятся по крайней мере к одному классу, в котором преподаватель не отключил электронную почту в [параметрах задания.](https://support.microsoft.com/office/adjust-assignment-settings-in-your-class-team-05bb3b89-1cdf-415a-b6c7-44add0376a77)

 - В сообщениях электронной почты содержатся сведения о заданиях, срок выполнения которых был за предыдущую или предстоящую неделю.

Значение по умолчанию для этой функции — **выключено.**


<a name="bkmakecode"> </a>
### <a name="makecode"></a>MakeCode
Microsoft MakeCode — это платформа для кодирования на основе блоков, которая позволяет информатиковать всех учащихся. 

MakeCode — это продукт корпорации [](https://go.microsoft.com/fwlink/?LinkID=206977) Майкрософт, на который налагаются условия использования и [политики](https://go.microsoft.com/fwlink/?LinkId=521839) конфиденциальности корпорации Майкрософт.

Значение по умолчанию для этой функции — **выключено.**

Чтобы включить задания MakeCode в Teams, перейдите в  Центр администрирования **Teams,** перейдите в раздел "Задания" и включите параметр MakeCode.  Щелкните **Сохранить**. Чтобы эти параметры вступили в силу, можно в течение нескольких часов.

Дополнительные сведения о том, как работает эта функция, см. в этом [видеоролике.](https://makecode.com/blog/teams/teams-assignments)

[Подробнее о MakeCode.](https://aka.ms/makecode)

<a name="#turnitin"> </a>
### <a name="turnitin"></a>Turnitin

[Turnitin](https://www.turnitin.com/) — это служба целостности образовательных образования. Это сторонний продукт или служба, на которые налагаются собственные условия и политика конфиденциальности. Вы несете ответственность за использование сторонних продуктов и служб.

Значение по умолчанию — **"Выключено.".**

Чтобы включить Turnitin для организации, вам потребуется подписка на Turnitin. Затем вы можете ввести следующую информацию, которую можно найти в консоли администратора Turnitin:

  * **TurnitinApiKey:** 32-символный GUID, который находится в консоли администрирования в области Интеграции.
  * **TurnitinApiUrl:** ЭТО URL-адрес HTTPS консоли администратора Turnitin.

Ниже указаны некоторые инструкции, которые помогут вам получить эти сведения.

Адрес **TurnitinApiUrl** является адресом сайта консоли администратора.
Пример: `https://your-tenant-name.turnitin.com`

Консоль администрирования — это место, где можно создать интеграцию и ключ API, связанный с этой интеграцией.

Выберите **пункт "Интеграция"** в боковом меню, а затем выберите **"Добавить интеграцию"** и придайте интеграции имя.

![Снимок экрана: добавление новой интеграции](./educationImages/Assignments_mopo_turnitin2.png)

**TurnitinApiKey** будет предоставлен вам после того, как вы следуйте запросам. Скопируйте ключ API и вкопируйте его в Центр администрирования Microsoft Teams.  Это единственный раз, когда вы можете просмотреть ключ.

![Снимок экрана: копирование ключа API](./educationImages/Assignments_mopo_turnitin3.png)

Нажимая **кнопку** "Сохранить" в Центре администрирования для этого параметра, в течение нескольких часов эти параметры вступили в силу.
