# <a name="teams-for-nonprofit-organizations"></a>Teams для некоммерческих организаций

У некоммерческих организаций есть множество уникальных бизнес-потребностей, с которыми могут помочь Microsoft Teams и Microsoft 365. В частности, некоммерческие организации часто используют большое количество волонтеров для выполнения своих задач. Поэтому координация и общение с сотрудниками организации и волонтерами имеет первостепенное значение.

Следующие шаблоны предназначены специально для некоммерческих организаций. Используйте эти шаблоны для расширения возможностей сотрудников и привлечения волонтеров.

- Используйте [шаблон "Управление командой волонтеров"](../team-templates-nonprofit.md) для быстрой и легкой настройки команд в Teams, чтобы ваши сотрудники могли общаться и совместно работать над задачами и действиями по управлению волонтерами. Этот шаблон включает в себя параметры, каналы и приложения, предназначенные для упорядочения действий по управлению волонтерами.

    Ваши сотрудники могут использовать разные каналы для организации и обмена материалами и часто используемыми документами, просмотра отчетов, публикации объявлений о важных командах и мероприятиях и т. д. Шаблон также интегрируется с приложением [Управление волонтерами](/dynamics365/industry/nonprofit/volunteer-management-use) в составе [Microsoft Cloud для некоммерческих организаций](/industry/nonprofit), позволяя сотрудникам управлять возможностями вовлечения волонтеров в Teams.

- С помощью [шаблона сайта SharePoint "Центр волонтеров"](https://support.microsoft.com/office/use-the-sharepoint-volunteer-center-template-b2ddd3b5-eb1a-425d-b059-a51ed9bff819) вы можете создать центр, в котором ваши волонтеры смогут получить доступ к основной информации. Приветствуйте новых волонтеров, публикуйте объявления, демонстрируйте предстоящие мероприятия и представляйте свою организацию и сотрудников. Обеспечьте свободный доступ к материалам обучения и адаптации, а также к ключевым сайтам.

Функции Teams для некоммерческих организаций также являются частью Microsoft Cloud для некоммерческих организаций. Узнайте больше о [Microsoft Cloud для некоммерческих организаций](/industry/nonprofit), которое объединяет возможности Microsoft Dynamics 365, Microsoft Power Platform, Microsoft 365, Microsoft Azure и LinkedIn.
