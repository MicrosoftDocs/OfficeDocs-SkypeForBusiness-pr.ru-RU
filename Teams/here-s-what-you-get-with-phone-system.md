# <a name="heres-what-you-get-with-phone-system"></a>Возможности телефонной системы

В этой статье описаны функции телефонной системы. Чтобы узнать больше о том, как использовать телефонную систему в качестве замены УАТС и параметры для подключения к телефонной сети с открытым коммутируемым подключением (КТСОП), посмотрите, [что такое телефонная система](what-is-phone-system-in-office-365.md).

Клиенты доступны для ПК, Mac и мобильных устройств, которые предоставляют возможности на устройствах с планшетов и мобильных телефонов для компьютеров и IP-телефонов для настольных систем. Дополнительные сведения можно найти в разделе [получение клиентов для Microsoft Teams](get-clients.md).

  
## <a name="phone-system-features"></a>Возможности телефонной системы

Телефонная система предоставляет следующие возможности. Если не указано иное, возможности доступны в группах Teams и Skype для бизнеса Online.

Меньшее количество нарушений для получателей по сравнению с другими формами общего доступа к вызовам (например, переадресация звонков или Одновременный звонок), так как пользователи могут настроить способ уведомления о входящем звонке. |

Пользователь, который приприпаркован звонок или другой пользователь может затем использовать этот код и поддерживаемое приложение или устройство для получения вызова. <br/> |

|Федеративный Звонок  <br/> |Позволяет пользователям безопасно подключаться, общаться и совместно работать с пользователями в федеративных клиентах.  <br/> |

Эта функция отключена по умолчанию, но может быть включена администратором клиента. |
