# <a name="heres-what-you-get-with-phone-system"></a>Возможности телефонной системы

В этой статье описаны функции телефонной системы. Дополнительные сведения об использовании телефонной системы в качестве замены частного филиала Exchange (PBX) и параметрах подключения к телефонной сети общего звонков (STN) см. в теме "Что такое телефонная [система".](what-is-phone-system-in-office-365.md)

Клиенты доступны для ПК, Mac и мобильных устройств, которые предоставляют функции на устройствах: от планшетов и мобильных телефонов до ПК и IP-телефонов. Дополнительные сведения см. в [подмносях "Получить клиенты для Microsoft Teams".](get-clients.md)

 > [!Note]
> Подробные сведения о телефонных системах Teams на различных платформах см. в [различных платформах Teams.](https://support.microsoft.com/office/teams-features-by-platform-debe7ff4-7db4-4138-b7d0-fcc276f392d3)
  
## <a name="phone-system-features"></a>Функции телефонной системы

Телефонная система предоставляет следующие возможности: Если не указано иное, функции доступны как в Teams, так и в Skype для бизнеса Online.
  
|||
|:-----|:-----|
|**Функция телефонной системы** <br/> |**Описание** <br/> |
|[Автозаводы в облаке](what-are-phone-system-auto-attendants.md) <br/> |Позволяет создать систему меню, которая позволяет внешним и внутренним звонятелям находить, размещать и переводить звонки пользователям или отделам организации.  <br/> |
|[Очереди облачных вызовов](create-a-phone-system-call-queue.md) <br/> |Позволяет настроить управление очередями вызовов для организации: настроить приветствия и музыку в удержании, найти следующего доступного агента для обработки звонка и так далее.  <br/> |
|Музыка при удержании | Воспроизведение музыки по умолчанию, определенной службой, при постановке на удержание внешнего звонка из телефонной сети общего перейти на телефонную сеть (STN). Эта функция работает не только для звонков в очереди звонков, но и для звонков между ДНР и Teams. Эта функция обеспечивает парность уведомлений на удержании с другими платформами. Эта функция настраивается администратором, но в настоящее время [только с помощью PowerShell.](https://docs.microsoft.com/powershell/module/skype/set-csteamscallingpolicy?view=skype-ps) Удержание музыки также не поддерживается при консультации при передаче звонка по ДНР.|
|Ответ на звонок/инициировать (по имени и номеру)  <br/> |Позволяет пользователям отвечать на входящие звонки одним касанием и делать исходящие звонки, набрав полный номер телефона или щелкнув имя в клиенте.  <br/> |
|[Параметры переад вызовов и одновременные вызовы](https://support.office.com/article/call-forwarding-call-groups-and-simultaneous-ring-in-teams-a88da9e8-1343-4d3c-9bda-4b9615e4183e) <br/> |Позволяет пользователям настроить правила переадстройки, чтобы звонки могли переходить к ним отовсюду или переадрены на коллегу или на голосовую почту.  <br/> |
|[Групповая звонок и переадметка в группу](call-sharing-and-group-call-pickup.md) <br/> | Позволяет пользователям обмениваться входящие звонки с коллегами, чтобы они могли отвечать на звонки, которые происходят, когда пользователь недоступен. Менее нарушения для получателей, чем другие формы общего доступа к звонкам (например, переадстройка звонка или одновременный звонок), так как пользователи могут настроить уведомления о входящих общих звонках. |
|[Передача звонка и консультации](https://support.office.com/article/Transfer-a-call-in-Teams-b7f40f14-e083-46b9-b739-68038c8f73a0) <br/> |Позволяет пользователям перена передаче звонков другому пользователю. Или, если им нужно покинуть офис, но они хотят продолжить беседу, они могут перенести звонки со своего ПК или IP-телефона на свой мобильный телефон.  <br/> |
|[Перена передачи на голосовую почту во время звонка](https://support.office.com/article/Transfer-a-call-in-Teams-b7f40f14-e083-46b9-b739-68038c8f73a0) <br/> | Позволяет пользователям переводить сообщения на голосовую почту во время звонка. |
|[Приостановка и восстановление звонка](call-park-and-retrieve.md)  <br/> | Позволяет пользователям удерживать звонок в службе Teams в облаке. При приостровке звонка служба создает уникальный код для искомого звонка. Пользователь, приодердеривший звонок, или кто-то другой сможет использовать этот код и поддерживаемые приложения или устройства для получения звонка. <br/> |
|Звонок на номер телефона из поисковой системы  <br/> | Позволяет пользователям делать звонок из поле поиска с помощью команды /call с указанием имени или номера. <br/> |
|[Идентификатор звонящего](how-can-caller-id-be-used-in-your-organization.md)  <br/> |При звонках внутри компании отображается подробный ИД звонящая, который извлекет информацию из корпоративного каталога, отображая номер телефона и должность вместо номера телефона. Для звонков с внешних номеров телефонов отображается ИД звонив, предоставленный поставщиком услуг телефонной связи. Если внешние номера телефонов являются дополнительными номерами в корпоративном каталоге, будут отображаться сведения из корпоративного каталога.  <br/> |
|Переключение устройств  <br/> |Позволяет пользователям играть в звонок или собрание на другом устройстве HID, подключенного к Teams; Например, можно переключиться с динамиков компьютера на гарнитуру.   <br/> |
|Маршрутия вызовов на основе присутствия <br/> |Контролирует входящие сообщения с присутствием, позволяя пользователю блокировать все входящие сообщения, кроме указанных.  <br/> |
|[Встроенная панель набора номера](https://support.office.com/article/use-the-dial-pad-in-teams-27bc60b5-74c0-4e9c-808b-da4db9514d89) <br/> | Позволяет пользователям звонить по имени или номеру в любом месте на панели поиска и на панели набора номера, что ускоряет обработку исходяющих звонков. <br/> |
|Федераированные вызовы  <br/> |Обеспечивает безопасное подключение, общение и совместное взаимодействие с пользователями в федераированных клиентах.  <br/> |
|[Звонить и принимать видеосвязь](https://support.office.com/article/abf62493-670f-4b0d-b2cf-fe03b49caf42) <br/> | Если в учетной записи пользователя включена возможность видеозвонков, пользователь может звонить своим контактам в видеосвязь. Для этого им понадобится только камера, динамики и микрофон компьютера. Пользователи также могут использовать гарнитуру, если на компьютере нет встроенного звукового устройства.<br/> |
|[Облачная голосовая почта](set-up-phone-system-voicemail.md) <br/> | Когда пользователь получает голосовое сообщение, оно доставляется в почтовый ящик Exchange в виде сообщения электронной почты с голосовым сообщением в виде вложения. Пользователи могут прослушивать сообщения на сертифицированном настольном телефоне и во всех приложениях Teams или Skype для бизнеса. Транскрибация голосовой почты поддерживается с марта 2017 г. и включена по умолчанию для всех организаций и пользователей.   <br/> |
|[Параметры пользователей голосовой почты в облаке](https://support.office.com/article/manage-your-call-settings-in-teams-456cb611-3477-496f-b31a-6ab752a7595f?ui=en-US&rs=en-US&ad=US) <br/> | Позволяет пользователям настраивать параметры клиента для приветствий голосовой почты, правил ответа на звонки и языка приветствий, включая приветствия об этом.   |
|[Дополнительный звонок](https://support.office.com/article/Manage-your-call-settings-in-Teams-456cb611-3477-496f-b31a-6ab752a7595f) <br/> | Пользователи с несколькими устройствами громкой связи, подключенными к компьютеру, могут настроить дополнительные устройства для звонка в дополнение к динамику по умолчанию. Например, пользователь с гарнитурой, подключенной к компьютеру и динамикам на рабочем месте, может во время звонка использовать как гарнитуру, так и динамики, чтобы они не пропустили звонок.  |
|[Оповещение о звонках](https://support.office.com/article/Manage-your-call-settings-in-Teams-456cb611-3477-496f-b31a-6ab752a7595f) (только в Teams)<br/> |Позволяет пользователям выбирать отдельные мелодии звонка для обычных, переадных и делегвшихся звонков, чтобы различать тип звонка.  <br/> |
|[Распределенная линия](shared-line-appearance.md) <br/> | Позволяет пользователям делиться телефонной линией, чтобы другой пользователь получал и получал звонки от их имени.|
|[Занят (только](teams-calling-policy.md) в Teams) <br/> | Политика звонков, которая позволяет настроить обработку входящих звонков, если пользователь уже находится в звонке или конференции или помещен на удержание. Когда звонок уже находится на телефоне, он будет слышать сигнал занятости. Звонящая получает уведомление о пропущенных звонках, но не может отвечать на входящие звонки. Эта функция по умолчанию отключена, но может быть включена администратором клиента. |
|[Блокировка вызовов](https://support.office.com/article/manage-your-call-settings-in-teams-456cb611-3477-496f-b31a-6ab752a7595f?ui=en-US&rs=en-US&ad=US) <br/> | Позволяет пользователям добавлять номера телефонов (ННР) в список заблокированных, чтобы последующие звонки с этого номера не звонили пользователю.|
|[Телефоны общей области](set-up-common-area-phones.md) <br/> | Общий телефон с общей площадью обычно размещен в таких помещениях, как зал ожидания или конференц-зал, что делает его доступным для нескольких людей. Обычные телефоны с зонами настроены как устройства, а не пользователи, и могут автоматически в сети.|
|[Обход мультимедиа (только](direct-routing-plan-media-bypass.md) для прямой маршрутки Teams) <br/> | Для улучшения производительности мультимедиа хранятся между контроллером границы сеанса (SBC) и клиентом, а не через телефонную систему Майкрософт. |

В облачных облаках GCC High и DoD пока недоступны следующие возможности: 
- [Параметры параметров дополнительного звонка, голосовой почты и расширенной делегирования](https://support.office.com/article/Manage-your-call-settings-in-Teams-456cb611-3477-496f-b31a-6ab752a7595f)

- Звонок на номер телефона из панели поиска
- Музыка при удержании
- Подглядывание обратных номеров в Azure AD
