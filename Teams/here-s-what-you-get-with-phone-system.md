# <a name="heres-what-you-get-with-phone-system"></a>Возможности телефонной системы

В этой статье описаны функции телефонной системы. Дополнительные сведения об использовании телефонной системы в качестве замены УАТС и о вариантах подключения к телефонной сети общего пользования (ТСОП) см. в статье [Что такое телефонная система](what-is-phone-system-in-office-365.md).

Клиенты доступны для ПК, Mac и мобильных устройств, которые предоставляют функции на устройствах с планшетов и мобильных телефонов на ПК и настольных IP-телефонах. Дополнительные сведения см. в разделе [Получение клиентов для Майкрософт Teams](get-clients.md).

 > [!Note]
> Дополнительные сведения о телефонных системах Teams на разных платформах см. в разделе [Возможности Teams по платформам](https://support.microsoft.com/office/teams-features-by-platform-debe7ff4-7db4-4138-b7d0-fcc276f392d3).

Чтобы использовать функции телефонной системы, ваша организация должна иметь лицензию на телефонную систему. Для получения дополнительной информации о лицензировании см. [Дополнительное лицензирование Microsoft Teams](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md).

Помните, что большинству функций требуется назначить лицензию телефонной системы и убедиться, что пользователи имеют "голосовую связь". Чтобы назначить лицензию, используйте [командлет Set-CsPhoneNumberAssignment](/powershell/module/teams/set-csphonenumberassignment?view=teams-ps) и задайте для параметра **EnterpriseVoiceEnabled** значение $true. Для некоторых функций, таких как облачный автосекретарь, не требуется, чтобы пользователь был включен голосовой связью. Исключения перечислены в приведенной ниже таблице.
