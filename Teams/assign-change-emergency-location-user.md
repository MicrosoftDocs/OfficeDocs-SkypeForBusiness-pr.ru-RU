# <a name="assign-or-change-an-emergency-location-for-a-user"></a>Назначение или изменение расположения для экстренного реагирования для пользователя

Каждому номеру телефона или пользователю должно быть назначено местоположение для экстренного ситуация. Дополнительные сведения об адресах и местоположениях для экстренного вызова см. в этой [теме.](./what-are-emergency-locations-addresses-and-call-routing.md)

Для планов звонков Майкрософт в европейских странах местоположение для экстренного вызова связано с номером при его Microsoft 365 или при переносе номера в Microsoft 365. В США местоположение для экстренного экстренного ситуация связано с номером телефона, который назначен пользователю. Адрес для экстренного устранения проблем можно изменить, если пользователь, назначенный ему, перемещается в новое расположение.
  
Вы можете назначить или изменить местоположение для экстренного ситуация для пользователя в центре Microsoft Teams или с помощью PowerShell.

## <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

1. В левой области навигации Центра администрирования Microsoft Teams щелкните **Номера голосовой**  >  **Телефон голосовой Телефон**.

2. На странице **Телефон номера** щелкните **вкладку** Числа, выберите в списке номер пользователя и нажмите кнопку **Изменить**.

3. В области **Правка** в области **Местоположение для экстренного ситуация** сделайте одно из следующих:

   - Чтобы назначить местоположение для экстренного ситуация, найди его и выберите его.

   - Чтобы изменить местоположение для экстренного устранения проблемы, которое уже назначено пользователю, щелкните **X,** чтобы удалить существующее расположение, а затем найдите и выберите нужное расположение.

4. В зависимости от того, хотите ли вы отправить сообщение электронной почты пользователю со сведениями о телефонном номере, отключите или включите отправку электронной почты пользователю со сведениями о **номере телефона.** По умолчанию этот режим в том же режиме.

5. Нажмите кнопку **Применить**.
