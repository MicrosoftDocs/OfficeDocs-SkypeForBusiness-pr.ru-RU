# <a name="create-your-first-teams-in-microsoft-teams"></a>Создание первых команд в Microsoft Teams

Лучший способ принятия новых Teams — использовать продукт. Мы рекомендуем управлять развертыванием Microsoft 365 или Office 365 с помощью Teams. Также следует создать сообщество сотрудников, которые нацелены на командную работу и производительность. (Со временем исходные группы могут измениться.) Предлагаются такие группы:

| Название команды | Предложенные названия каналов |
| --------- | ---------------------- |
| Узнать, как Teams | Общие</br> Поздороваться</br> Как это сделать?</br>Отзывы </br> Идеи для будущих команд </br> Поддержка |
| Microsoft Teams Реализации | Общие <br/> Информирование и внедрение <br/> Взаимодействие с бизнесом <br/> Программа ранних принятия <br/> Отзывы и Аналитика <br/> Безопасность и соответствие требованиям <br/> Стратегия и планирование <br/> Состояние службы и инциденты <br/> Обучение <br/> Чат watercooler |
| Teamwork Champions | Общие <br/> Лучшие методики, учебные курсы и действия <br/> Уголок 18 <br/> Отзывы и поддержка <br/> Программы: интересов <br/> Social Hub |

В Microsoft Teams вы можете [](/microsoftteams/platform/concepts/tabs/tabs-overview) получить доступ ко всем приложениям Microsoft 365 или Office 365 сторонним приложениям, которые были включены администратором. Чтобы получить максимальное значение из своего продукта, мы рекомендуем закрепление ключевых ресурсов в качестве вкладок в каждом из соответствующих каналов. См. пример ниже.

![Снимок экрана, на котором показаны стандартные и настраиваемые вкладки.](media/teams-adoption-tab-example.png)

## <a name="making-the-switch-to-microsoft-teams"></a>Переход на Microsoft Teams

По мере того как вы начнете использовать Teams, вам и остальным командам проекта важно будет взять на себя обязательства по использованию чата, Teams каналов и ресурсов для запуска проекта. Использование вашей Teams очень важно для качества проекта по внедрению. Мы рекомендуем не разорять проект, продолжая беседы с основной командой проекта по электронной почте. Перейдя на Teams вы сможете моделировать поведение, которое в конечном итоге должны принять ваши сотрудники. Установите оттенок, начните и наслаждайтесь сведениями о проекте в одном месте!
