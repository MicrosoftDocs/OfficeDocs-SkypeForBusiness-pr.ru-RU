# <a name="cloud-video-interop-for-microsoft-teams"></a>Взаимодействие Microsoft Teams с облачными видеослужбами

Cloud Video Interop (CVI) — это стороннее решение Майкрософт, которое позволяет сторонним комнатам для собраний (телепретирование) и личным видеоустройствам присоединяться к собраниям Microsoft Teams.
 
С помощью Microsoft Teams вы получаете широкие возможности совместной работы с содержимым в Интернете на собраниях, включающее аудио, видео и общий доступ к содержимому. Это можно сделать с помощью настольного и веб-клиента, а также многих партнерских устройств, которые изначально интегрируются с Microsoft Teams. Однако многие клиенты уже вложить средства в видеоконференции и личные устройства видеосвязи, что может оказаться дорогостоящим для обновления. Cloud Video Interop предоставляет простое решение, позволяя продолжать использовать существующие решения, пока не будете готовы к обновлению.

С помощью Cloud Video Interop Microsoft Teams предоставляет собственный интерфейс для собраний для всех участников — в комнатах для собраний или в клиентах Teams.

### <a name="is-cloud-video-interop-for-me"></a>Облачное видеосвязь мне подходит?

Cloud Video Interop предоставляет промежуточную службу при переходе на полное собственное решение Microsoft Teams с использованием конечных точек Teams. Предоставляемая служба должна быть частью пути миграции.

Взаимодействие с видео в облаке предназначено для клиентов, отвечающих следующим критериям:

- Развертывание больших устройств конференц-залов и личных видеоустройств (более 50 устройств), которые не предназначены для прямой интеграции с Microsoft Teams
- поддерживаются одним из наших партнеров по облачному видеосвязи;
- Хотите сохранить ценность своих инвестиций в текущие устройства конференц-зала и личные видеоустройства во время миграции в собственное решение Microsoft Teams.

Хотя облачное видеосвязь — это отличное промежуточное решение, мы рекомендуем нашим клиентам в долгосрочной перспективе ознакомиться с собственными решениями для собраний Teams, такими как системы комнат Teams. 

### <a name="office-365-us-government-and-third-party-services"></a>Office 365 для государственных организаций США и сторонних служб

Office 365 позволяет интегрировать сторонние приложения в сайты SharePoint Online, Skype для бизнеса, Teams, приложения Office, включенные в Приложения Microsoft 365 для предприятий (например, Word, Excel, PowerPoint и Outlook), а также Outlook Web App. Кроме того, Office 365 поддерживает интеграцию со сторонними поставщиками услуг. Эти сторонние приложения и службы могут включать хранение, передачу и обработку данных клиентов вашей организации в сторонних системах, которые не находятся за пределами инфраструктуры Office 365 и, следовательно, не распространяются на Office 365 соответствия требованиям и обязательствам по защите данных. **При оценке правильного использования этих служб для вашей организации рекомендуется ознакомиться с заявлениями о конфиденциальности и соответствии, предоставленными третьими сторонами.**

### <a name="cloud-video-interop-overview"></a>Общие сведения об облачном видеосвязи

Решения, предлагаемые нашими партнерами, состоят из компонентов, которые можно развернуть либо полностью в облаке, либо частично или полностью локально.

При развертывании решения cloud Video Interop важно понимать, что вы развертываете партнерское решение. Общие действия, которые необходимо выполнить для развертывания Cloud Video Interop, перечислены на следующей схеме.

![Схема, описывая развертывание CVI в организации.](media/deploying-cvi.png)

### <a name="plan"></a>Планирование

На этапе планирования необходимо определить устройства, которые не будут заменяться собственным устройством Teams, и найти партнера по облачному видеосвязи, который может поддерживать эти устройства.  

Также важно понимать, что вам потребуется лицензия для каждого пользователя, который будет планировать собрания, к которым должно присоединяться устройство с поддержкой облачного видеосвязи. Обратите внимание, что точные требования к лицензированию можно получить у партнера cloud Video Interop. Перед началом развертывания убедитесь, что это понятно.

### <a name="configure"></a>Настройка

Партнер, выбранный для развертывания CVI, предоставит вам полный документ о развертывании, состоящий из всех шагов, необходимых для успешного развертывания в организации. К ним относятся порты брандмауэра и диапазоны IP-адресов, изменения конфигурации для устройств и другие параметры, которые необходимо изменить.

### <a name="provision"></a>Предоставление  

На этапе подготовки вы назначите лицензии соответствующим пользователям в соответствии с руководством по настройке партнера. Вам также потребуется пройти процесс предоставления согласия Azure, чтобы предоставить партнеру доступ к среде Teams. [Дополнительные сведения о процессе](/azure/active-directory/develop/v2-permissions-and-consent) предоставления согласия Azure см. платформа удостоверений Майкрософт разрешений и согласия в конечной точке платформа удостоверений Майкрософт.

### <a name="schedule"></a>Расписание

После того как пользователь будет включен для облачного видеосвязи, любое собрание, запланированное с помощью надстройки для собраний Teams для Outlook или клиента Teams, автоматически добавит соответствующие дополнительные сведения в собрание Teams, чтобы устройства, совместимые с облачным видеосвязи, могли присоединяться к этим собраниям.

### <a name="join"></a>Join

В зависимости от партнерского решения существует несколько способов присоединения к собранию с поддержкой облачного видеосвязи. Точные сценарии присоединения к собранию будут предоставлены вашим партнером cloud Video Interop. Ниже перечислены некоторые примеры.

- IVR (интерактивный голосовой ответ) 
  - Вы можете выполнить вход на IVR партнера с помощью tenantkey@domain.
  - Во время работы с партнером IVR вам будет предложено ввести VTC conferenceId, который затем подключит вас к собранию Teams.
- Прямой набор 
  - Вы можете напрямую подключаться к собранию Teams без взаимодействия с IVR партнера с помощью функции прямого набора номера, используя полную строку tenantkey. VTC ConferenceId@domain.
- Одно касание набора 
  - Если у вас есть интегрированная комната Teams, вы можете использовать возможности одно касания, предлагаемые партнером (без необходимости ввести строку набора номера).

## <a name="manage-cloud-video-interop"></a>Управление взаимодействием с облачными видео

После развертывания Cloud Video Interop вы можете управлять устройствами с помощью решений, предоставляемых нашими партнерами. Каждый партнер предоставит вам административный интерфейс, который будет включать как управление лицензиями, так и устройствами. 

Отчеты также доступны непосредственно из административного интерфейса партнера. Чтобы получить дополнительные сведения о возможностях создания отчетов, обратитесь к выбранному партнеру. 

### <a name="troubleshooting-cloud-video-interop"></a>Устранение неполадок облачного видеосвязи

Cloud Video Interop — это предоставляемая партнером служба. Если у вас возникают проблемы, первым шагом является подключение устройства с установленным клиентом Teams и подключение его к сегменту устройства Облачного видеосвязи, которое вызывает проблемы. 

Если Teams правильно работает в этом сегменте и вы также следовали всем рекомендациям по сети и конфигурации, предоставленным партнером, необходимо обратиться к партнеру для дальнейшего устранения неполадок. 

## <a name="powershell-for-cloud-video-interop"></a>Взаимодействие с Видео в PowerShell для облака

Ниже приведены командлеты PowerShell, которые можно (частично) автоматизировать развертывание облачного видеосвязи.

- **Get-CsTeamsVideoInteropServicepolicy**: корпорация Майкрософт предоставляет предварительно созданные политики для каждого из наших поддерживаемых партнеров, которые позволяют назначить партнеров, которые будут использовать для взаимодействия с облачными видео.<br>Этот командлет позволяет определить предварительно созданные политики, которые можно использовать в организации. Эту политику можно назначить одному или нескольким пользователям, используя Grant-CsTeamsVideoInteropServicePolicy командлета.
- **Grant-CsTeamsVideoInteropServicePolicy**: этот командлет позволяет назначить предварительно созданную политику для использования в организации или назначить политику определенным пользователям.
- **New-CsVideoInteropServiceProvider**: используйте этот командлет, чтобы указать сведения о поддерживаемом партнере CVI, который ваша организация будет использовать.
- **Set-CsVideoInteropServiceProvider**: используйте этот командлет для обновления сведений о поддерживаемом партнере CVI, используемом вашей организацией.
- **Get-CsVideoInteropServiceProvider**: используйте этот командлет, чтобы получить все поставщики, настроенные для использования в организации.
- **Remove-CsVideoInteropServiceProvider**: используйте этот командлет, чтобы удалить все сведения о поставщике, который больше не использует ваша организация.
