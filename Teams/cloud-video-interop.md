# <a name="cloud-video-interop-for-microsoft-teams"></a>Взаимодействие Microsoft Teams с облачными видеослужбами

Облако видео взаимодействия (CVI) — это решение сторонних производителей Qualified Майкрософт, которая позволяет комнаты для собраний сторонних производителей (виртуального присутствия) и личные видеоустройства (VTCs) для присоединения к собраниям группами Майкрософт.
 
С группами Майкрософт вы получаете форматированного контента совместная работа в Интернете в собраниях, включая аудио, видео и содержимого общего доступа. Это может смогли через рабочего стола и веб-клиент, а также многих устройствах партнеров, которые изначально интегрируются с группами Майкрософт. Тем не менее многие пользователи уже были вложены в видеоконференций и устройствах личных видеосвязь, которых может быть дорогим для обновления. Взаимодействие видео облако обеспечивает простое решение, что позволяет продолжить использование существующих решений, пока не будете готовы к обновлению.

С помощью взаимодействия видео в облаке группами Майкрософт обеспечивает удобство работы собственного собрания для всех участников – в конференц-залы или внутри групп клиентов.

### <a name="is-cloud-video-interop-for-me"></a>— Это облачных видео взаимодействия для меня?

Облако видео взаимодействие обеспечивает промежуточной службы во время перехода к решению групп полное собственный Microsoft с помощью команды конечных точек. Услуг, предоставляемых должна быть частью пути миграции.

Облако видео взаимодействия предназначено для пользователей, которые соответствуют следующим критериям:

- Большое развертывание устройств комнаты приглашений на собрания и развертывания личных видеоустройства (более 50 устройства), которые не определены для прямую интеграцию с группами Майкрософт
- Поддерживаемые с помощью одного из партнеров облачных видео взаимодействия
- Чтобы сохранять значение вложения в текущие собрания комнаты и личные видеоустройств во время миграции на собственный решение группами Майкрософт

Облако видео взаимодействия обеспечивает отличное промежуточное решение, мы рекомендуем наших клиентов для поиска в нашей собственный решения собрания групп, например систем комнаты групп, долгосрочной перспективе.

## <a name="deploy-cloud-video-interop"></a>Развертывание облачных видео взаимодействия

При развертывании решения облачных видео взаимодействия важно понять, что вы развертываете решение партнера. Общие действия необходимо выполнить для развертывания облака видео взаимодействия, перечислены в следующей схеме.

![Развертывание CVI в организации](media/deploying-cvi.png)

### <a name="plan"></a>Планирование

Во время на этапе планирования необходимо определить устройств, что нужно не заменить на собственный групп устройств и поиск партнера облачных видео взаимодействия, которая может поддерживать этих устройств.  

Также важно понимать, что для каждого пользователя, который будет планировать собрания, на котором будут поддержкой взаимодействия видео облачных устройства для присоединения к потребуется лицензия. Обратите внимание, что точное требований лицензирования можно получить из облака видео взаимодействия для партнеров. Убедитесь, что это очистить перед началом развертывания.

### <a name="configure"></a>Настройка

Партнеров, выбранное для вашего развертывания CVI будет предоставлять полного развертывания документа, которая включает все действия, необходимые для успешного развертывания в вашей организации. Это будет включать порты брандмауэра и диапазоны IP-адресов, изменения конфигурации для устройств и другие параметры, которые необходимо изменить.

### <a name="provision"></a>Подготовка к работе  

На этапе подготовки будут использоваться для назначения лицензий для соответствующих пользователей в соответствии с партнера руководство по конфигурации. Также необходимо пройти через процесс соглашаетесь Azure для обеспечения доступа к партнера для вашей среды групп. Здесь можно найти дополнительные сведения о процессе соглашаетесь Azure:https://docs.microsoft.com/en-us/azure/active-directory/develop/v2-permissions-and-consent

После включения пользователя для облачных видео взаимодействия любого совещания, запланированного с помощью любого из собрания команды надстройки для Outlook или в клиенте групп будут иметь соответствующие Дополнительные сведения, автоматически добавляются в группы, собрания, облако видео Взаимодействие совместимых устройств можно объединить эти собрания.

### <a name="join"></a>Join

В зависимости от решения партнеров существует несколько способов присоединение к собранию поддержкой взаимодействия видео облака. Точное присоединения к собранию, сценарии будет предоставляться партнером взаимодействия видео облака. Ниже приведены некоторые примеры ниже:

- IVR (систем речевого взаимодействия) 
  - Можно удаленного IVR партнера с помощью tenantkey@domain.
  - Когда вы используете партнера IVR, будет предложено ввести conferenceId VTC, в которой будет затем установлено соединение собрания групп.
- Прямой 
  - Можно напрямую удаленного собрания групп без взаимодействия с IVR партнера с помощью функции прямой с помощью полной строки tenantkey. VTC ConferenceId@domain.
- Номера одним нажатием 
  - При наличии интегрированной комнаты групп, можно использовать одним нажатием телефонным возможности, предоставляемые партнеру (без необходимости введите любую строку звонков).

## <a name="manage-cloud-video-interop"></a>Управление видео взаимодействия облако

После развертывания облака видео взаимодействия можно управлять устройств, с помощью решений, предоставленными партнерами. Каждый участник будет предоставлять интерфейс администрирования, который будет включать управление лицензии и устройства. 

Отчеты можно также получить непосредственно из интерфейса администрирования партнера. Дополнительные сведения о возможности создания отчетов в контактов партнера по вашему выбору. 

### <a name="troubleshooting-cloud-video-interop"></a>Устранение неполадок в облаке видео взаимодействия

Облако видео взаимодействия — это служба предоставленного партнера. Если возникают проблемы, первым делом для подключения устройства с установлен клиент групп и подключиться к сегмента как устройство облачных видео взаимодействия, которая вызывает проблемы. 

Если группы правильно работает в сегмент, а также были выполнены все сети и конфигурации рекомендаций, предоставленное партнеров, необходимо обратиться к партнеру для дальнейшего устранения неполадок. 

## <a name="powershell-for-cloud-video-interop"></a>PowerShell для облачных видео взаимодействия

Доступны следующие командлеты PowerShell для для автоматизации развертывания облака видео взаимодействия (частично).

- **Get-CsTeamsVideoInteropServicepolicy**: Корпорация Майкрософт предоставляет предварительно построенного политик для каждого из наших поддерживаемые партнеров, которые позволяют назначить какие партнерами для облачных видео взаимодействия.<br>Этот командлет позволяет определить предварительно построенного политик, которые можно использовать в организации. Эту политику можно назначить один или несколько пользователей с помощью командлета Grant-CsTeamsVideoInteropServicePolicy.
- **Предоставление CsTeamsVideoInteropServicePolicy**: этот командлет позволяет назначить предварительно построенного политику для использования в вашей организации или назначьте политику к определенным пользователям.
- **New-CsVideoInteropServiceProvider**: используйте этот командлет для указания сведений о поддерживаемых партнер CVI, вашей организации требуется использовать.
- **Set-CsVideoInteropServiceProvider**: этот командлет обновляет сведения о поддерживаемых CVI партнера, который используется в организации.
- **Get-CsVideoInteropServiceProvider**: используйте этот командлет для получения всех поставщиков, которые были настроены для использования в организации.
- **Remove-CsVideoInteropServiceProvider**: этот командлет удаляет все сведения о поставщике о поставщике, который больше не используется в организации.
