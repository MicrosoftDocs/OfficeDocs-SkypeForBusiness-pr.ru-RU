# <a name="cloud-video-interop-for-microsoft-teams"></a>Взаимодействие Microsoft Teams с облачными видеослужбами

Cloud Video Interop (CVI) — это сторонное решение Microsoft Qualified, которое позволяет присоединяться к собраниям Microsoft Teams с помощью сторонних комнат собраний (телепреенции) и личных видео устройств (VTCs).
 
С помощью Microsoft Teams вы получаете богатые функции совместной работы по сети во время собраний, включая обмен аудио- и видеофайлами, а также общий доступ к содержимому. Ее можно использовать как в клиенте для настольных компьютеров, так и в веб-клиенте, а также на многих устройствах партнеров, которые интегрируются с Microsoft Teams. Однако многие клиенты уже сделали это для видеоконференции и личных видеосопровождений, что может стоить затрат на обновление. Cloud Video Interop — это простое решение, которое позволяет и дальше использовать существующие решения, пока вы не будете готовы к обновлению.

С помощью облачной видеосвязи Microsoft Teams обеспечивает родной опыт проведения собраний для всех участников — в помещениях для собраний или в клиентах Teams.

### <a name="is-cloud-video-interop-for-me"></a>Для меня облачное видео пересекается?

Cloud Video Interop предоставляет промежуточную службу при переходе к полному родному решению Microsoft Teams с использованием конечных точек Teams. Предоставляемая служба должна быть частью пути миграции.

Cloud Video Interop предназначен для клиентов, которые соответствуют следующим условиям:

- Крупное развертывание устройств комнат для собраний и личных видео устройств (более 50 устройств), не готовых к прямой интеграции с Microsoft Teams
- Поддержка одного из наших партнеров Cloud Video Interop
- Хотят сохранить стоимость своих инвестиций в их текущие устройства комнаты для собраний и личные видео устройства во время миграции на свое решение Microsoft Teams

Облачный видеосображений — это отличное промежуточное решение, но мы рекомендуем нашим клиентам в долгосрочной перспективе искать свои решения для собраний Teams, такие как системы комнат Teams. 

### <a name="office-365-us-government-and-third-party-services"></a>Office 365 для государственных и сторонних служб

Office 365 позволяет интегрировать сторонние приложения на сайты SharePoint Online, Skype для бизнеса, Teams, приложения Office, включенные в приложения Microsoft 365 для предприятий (такие как Word, Excel, PowerPoint и Outlook) и Outlook Web App. Кроме того, Office 365 поддерживает интеграцию со сторонними поставщиками услуг. Эти сторонние приложения и службы могут включать хранение, передачу и обработку данных клиентов организации в сторонних системах, которые не находятся за пределами инфраструктуры Office 365 и поэтому не защищены обязательствами Office 365 по обеспечению соответствия требованиям и защите данных. **При оценке надлежащего использования этих служб в организации рекомендуется и просмотреть заявления о конфиденциальности и соответствии требованиям, предоставленные сторонними лицами.**

### <a name="cloud-video-interop-overview"></a>Обзор cloud Video Interop

Cloud Video Interop — это стороннее решение, предоставляемая нашими партнерами для обеспечения взаимодействия между существующими решениями для видеоконференций и личными видеоконферентивами в локальной сети и Microsoft Teams.

Решения, предлагаемые нашими партнерами, состоят из компонентов, которые можно развернуть полностью в облаке или частично или полностью локально.

## <a name="deploy-cloud-video-interop"></a>Развертывание Cloud Video Interop

При развертывании решения Cloud Video Interop важно понимать, что развертывание решения партнера. На схеме ниже перечислены общие действия, которые необходимо предпринять для развертывания Cloud Video Interop.

![Схема, описывающие развертывание CVI в организации](media/deploying-cvi.png)

### <a name="plan"></a>Планирование

На этапе планирования определите устройства, которые не будут заменяться на устройства Teams, и найдите партнера Cloud Video Interop, который поддерживает эти устройства.  

Кроме того, важно понимать, что лицензия потребуется для каждого пользователя, который будет планировать собрания, к которым будет присоединяться устройство с поддержкой cloud Video Interop. Обратите внимание, что точные требования к лицензированию можно получить у партнера Cloud Video Interop. Прежде чем начать развертывание, убедитесь, что все понятно.

### <a name="configure"></a>Настройка

Партнер, который вы выбрали для развертывания CVI, предоставит вам полный документ о развертывании, который включает все этапы, необходимые для успешного развертывания в организации. К ним относятся порты брандмауэра, диапазоны IP-адресов, изменения конфигурации устройств и другие параметры, которые нужно изменить.

### <a name="provision"></a>Подготовка  

На этапе подготовка вы назначите лицензии соответствующим пользователям в соответствии с руководством по настройке партнера. Вам также потребуется пройти процесс получения согласия Azure, чтобы предоставить партнеру доступ к вашей среде Teams. Дополнительные сведения о процессе получения согласия Azure см. в сведениях о разрешениях и согласии в [конечной](/azure/active-directory/develop/v2-permissions-and-consent) точке платформы удостоверений Майкрософт.

### <a name="schedule"></a>Запланировать

После включения пользователем облачной видеосвязи все собрания, запланированные с помощью надстройки "Собрание Teams" для Outlook или клиента Teams, будут автоматически добавлены в собрание Teams соответствующие дополнительные сведения, чтобы устройства, совместимые с облачным видео, могли присоединиться к таким собраниям.

### <a name="join"></a>Join

В зависимости от решения партнера существует несколько способов присоединиться к собранию с поддержкой cloud Video Interop. Точные сценарии присоединиться к собранию предоставляет ваш партнер Cloud Video Interop. Ниже приведены некоторые примеры.

- IVR (интерактивный голосовой ответ) 
  - Вы можете позвонить в IVR партнера с помощью tenantkey@domain.
  - В IVR партнера вам будет предложено ввести VTC conferenceId, который подключит вас к собранию Teams.
- Прямой набор номера 
  - Вы можете напрямую звонить на собрание Teams, не взаимодействуя с IVR партнера, используя функцию прямого набора номера, используя полную строку клиента. VTC ConferenceId@domain.
- One-touch dial 
  - Если у вас встроенная комната Teams, вы можете использовать возможности набора номера одним нажатием, предлагаемые вашим партнером (без необходимости ввести строку набора).

## <a name="manage-cloud-video-interop"></a>Управление облачным видео с interop

После развертывания Cloud Video Interop вы сможете управлять устройствами с помощью решений, предоставленных нашими партнерами. Каждый партнер предоставит вам интерфейс администрирования, который включает как лицензии, так и управление устройствами. 

Отчеты также доступны непосредственно в интерфейсе администрирования партнера. За дополнительными сведениями о возможностях отчетности обратитесь к партнеру по вашему выбору. 

### <a name="troubleshooting-cloud-video-interop"></a>Устранение неполадок cloud Video Interop

Cloud Video Interop — это служба, предоставляемая партнером. При проблемах прежде всего необходимо подключить устройство, на котором установлен клиент Teams, и подключить его к сегменту устройства Cloud Video Interop, который вызывает проблемы. 

Если Teams работает правильно в этом сегменте, а вы также придерживались всех рекомендаций по сети и настройке, предоставленных партнером, вам потребуется обратиться к партнеру для дальнейшего устранения неполадок. 

## <a name="powershell-for-cloud-video-interop"></a>PowerShell для cloud Video Interop

Чтобы автоматизировать развертывание Cloud Video Interop, можно (частично) автоматизировать развертывание Cloud Video Interop, вы можете сделать следующее:

- **Get-CsTeamsVideoInteropServicepolicy**: Корпорация Майкрософт предоставляет предварительно проинструкцию политик для каждого из наших поддерживаемых партнеров, которые позволяют назначать партнеров, которые будут использовать для Cloud Video Interop.<br>С помощью этого cmdlet можно определить предварительно заранее задав политики, которые можно использовать в вашей организации. Эту политику можно назначить одному или несколько пользователям с помощью Grant-CsTeamsVideoInteropServicePolicy управления.
- **Grant-CsTeamsVideoInteropServicePolicy:** с помощью этого cmdlet можно назначить предварительно проданную политику для использования в организации или конкретным пользователям.
- **New-CsVideoInteropServiceProvider**: используйте этот cmdlet, чтобы указать сведения о поддерживаемом партнере CVI, который будет использовать ваша организация.
- **Set-CsVideoInteropServiceProvider**: используйте этот cmdlet для обновления сведений о поддерживаемом партнере CVI, которую использует ваша организация.
- **Get-CsVideoInteropServiceProvider**: используйте этот cmdlet, чтобы получить все поставщики, настроенные для использования в организации.
- **Remove-CsVideoInteropServiceProvider**: используйте этот cmdlet, чтобы удалить все сведения о поставщике, который больше не используется в вашей организации.
