# <a name="cloud-video-interop-for-microsoft-teams"></a>Взаимодействие Microsoft Teams с облачными видеослужбами

Облачное видеовзаимодействие (КВИ) — это решение стороннего поставщика, позволяющее использовать сторонние комнаты собраний (в том числе) и персональные видеоустройства (Вткс) для присоединения к собраниям Microsoft Teams.
 
С помощью Microsoft Teams вы получаете широкие возможности совместной работы с Интернет-контентом в собраниях, включая звук, видео и общий доступ к контенту. Это может быть полезным для настольных систем и веб-клиента, а также для многих устройств, которые интегрируются с Microsoft Teams с помощью встроенной программы. Тем не менее, многие пользователи уже приложились к видеоконференциям и устройствам для персональных видеосвязи, что может привести к дорогостоящему обновлению. Облачное видеовзаимодействие — это простое решение, позволяющее продолжать использование существующих решений, пока вы не будете готовы к обновлению.

Благодаря облачному взаимодействию Microsoft Teams предоставляет участникам собрания по собственному расдействию для всех участников — в помещениях для собраний или в клиентах Teams.

### <a name="is-cloud-video-interop-for-me"></a>Видеовзаимодействие с облачной видеоподсистемой?

Облачное видеовзаимодействие обеспечивает промежуточную службу при переходе в полноценное собственное решение Microsoft Teams с помощью конечных точек Teams. Указанная служба должна быть частью пути миграции.

Облачное видеовзаимодействие предназначено для пользователей, которые отвечают указанным ниже критериям.

- Большое развертывание устройств для помещения на собрание и персональных видеоустройств (50 + устройств), которые не предназначены для непосредственной интеграции с Microsoft Teams
- Поддерживаются одним из наших партнеров по облачному видеоразговору
- Вы хотите, чтобы стоимость инвестиций оставалась на текущих устройствах для собраний и персональных видеоустройствах во время миграции в собственное решение Microsoft Teams.

Несмотря на то, что облачное видеовзаимодействие предоставляет удобное промежуточное решение, мы рекомендуем нашим клиентам найти свои решения для собраний с собственными Teams, такие как системы комнат в Teams, для долгосрочных дел.

### <a name="cloud-video-interop-overview"></a>Общие сведения о облачном взаимодействии

Облачное видеовзаимодействие — это сторонняя служба, предлагаемая нашими партнерами для обеспечения взаимодействия между существующими видеоконференциями и персональными решениями для видеосвязи в локальной среде и Microsoft Teams.

Решения, предлагаемые нашими партнерами, состоят из компонентов, которые можно развертывать либо полностью, либо частично или полностью локально.

## <a name="deploy-cloud-video-interop"></a>Развертывание облачной видеобеседы

При развертывании решения для облачного видеовзаимодействия важно понимать, что вы развертываете решение для партнеров. Общие действия, которые необходимо выполнить для развертывания облачного видеосеанса, перечислены на рисунке ниже.

![Диаграмма, описывающая развертывание Кви в Организации](media/deploying-cvi.png)

### <a name="plan"></a>Планирование

На этапе планирования вы должны определить устройства, которые не заменяются на устройство с машинными группами, и найти партнера по облачному взаимодействию, который может поддерживать эти устройства.  

Важно понимать, что вам понадобится лицензия для каждого пользователя, который будет планировать собрания, в которых вы хотите присоединиться к облачному взаимодействию. Обратите внимание, что конкретные требования к лицензированию можно получить у партнера по облачной взаимодействию. Прежде чем приступить к развертыванию, убедитесь, что этот параметр не очищен.

### <a name="configure"></a>Настройка

Партнер, выбранный для развертывания Кви, предоставит вам полный документ, содержащий все этапы, необходимые для успешного развертывания в Организации. Сюда будут включены порты брандмауэра и диапазоны IP-адресов, изменения конфигурации для устройств и другие параметры, которые необходимо изменить.

### <a name="provision"></a>Обеспечения  

На этапе подготовки вы будете назначать лицензии соответствующим пользователям в соответствии с руководством по конфигурации партнера. Кроме того, для предоставления партнеру доступа к среде Teams вам потребуется пройти процесс согласия с помощью Azure. Дополнительные сведения о процедуре разрешения Azure можно найти здесь:https://docs.microsoft.com/en-us/azure/active-directory/develop/v2-permissions-and-consent 

### <a name="schedule"></a>Задание

После включения пользователя для облачного видеосвязи любое собрание, запланированное с помощью надстройки "собрание Teams" для Outlook или клиента Teams, будет автоматически добавлено в собрание Teams, чтобы облачное видео Эти собрания могут присоединиться к взаимодействию с совместимыми устройствами.

### <a name="join"></a>Join

В зависимости от решения партнера есть несколько способов присоединиться к собранию с помощью облачного видеосеанса. Конкретные сценарии для присоединения к собранию будут предоставлены вашим партнером по видеосвязи с облачными видеозвонками. Ниже приведены некоторые примеры.

- Интерактивный автоответчик (интерактивный голосовой отклик) 
  - Вы можете звонить на интерактивный автоответчик партнера через домен тенанткэй @.
  - После входа в интерактивный автоответчик партнера вам будет предложено ввести ВТК Конференцеид, после чего вы сможете подключиться к собранию Teams.
- Прямой набор номера 
  - Вы можете напрямую звонить на собрание Teams без взаимодействия с интерактивный автоответчик партнера с помощью функции прямого набора номера с помощью полной строки тенанткэй. ВТК Конференцеид @ Domain (домен).
- Один сенсорный Звонок 
  - Если вы используете объединенную комнату рабочих групп, вы можете использовать возможности набора номера, предложенные вашим партнером (без ввода строки набора номера).

## <a name="manage-cloud-video-interop"></a>Управление облачным взаимодействием

После развертывания облачного видеовзаимодействия вы можете управлять устройствами, используя решения, предоставленные нашими партнерами. Каждый партнер предоставит вам административный интерфейс, который будет включать в себя управление лицензиями и устройствами. 

Кроме того, вы можете использовать отчеты прямо из интерфейса администрирования партнера. Для получения дополнительных сведений о возможностях создания отчетов обратитесь к своему выбору партнера. 

### <a name="troubleshooting-cloud-video-interop"></a>Устранение неполадок с облачным взаимодействием

Облачное видеовзаимодействие — это служба, обеспечиваемая партнером. Если у вас возникли проблемы, сначала необходимо подключить устройство, на котором установлен клиент Teams, и подключить его к тому же сегменту, что и при работе с облачным видеосвязьм, вызывающим проблемы. 

Если в этом сегменте правильно работают команды и вы также следовали всем рекомендациям по настройке сети и конфигурации партнера, вам потребуется обратиться к партнеру для получения дальнейших неполадок. 

## <a name="powershell-for-cloud-video-interop"></a>PowerShell для облачного видеосеанса

Следующие командлеты PowerShell можно использовать (частично) для автоматизации развертывания облачного видеосеанса.

- **Get-кстеамсвидеоинтеропсервицеполици**: Корпорация Майкрософт предоставляет предварительно созданные политики для каждого из наших поддерживаемых партнеров, позволяющие назначать участников, которые используются для облачной видеобеседы.<br>Этот командлет позволяет определить готовые политики, которые можно использовать в Организации. Вы можете назначить эту политику для одного или нескольких пользователей, используя командлет Grant-Кстеамсвидеоинтеропсервицеполици.
- **Grant-кстеамсвидеоинтеропсервицеполици**: Этот командлет позволяет назначать уже созданную политику для использования в организации или назначать ее конкретным пользователям.
- **New-ксвидеоинтеропсервицепровидер**: Используйте этот командлет, чтобы указать сведения о поддерживаемом партнере Кви, который ваша организация хотели бы использовать.
- **Set-ксвидеоинтеропсервицепровидер**: Используйте этот командлет для обновления сведений о поддерживаемом партнере Кви, который использует ваша организация.
- **Get-ксвидеоинтеропсервицепровидер**: Используйте этот командлет, чтобы получить все поставщики, настроенные для использования в Организации.
- **Remove-ксвидеоинтеропсервицепровидер**: Используйте этот командлет, чтобы удалить все сведения о поставщике, которые больше не используются вашей организацией.
