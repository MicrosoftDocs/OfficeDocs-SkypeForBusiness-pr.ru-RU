# <a name="routing-calls-to-unassigned-numbers"></a>Маршрутизация вызовов на неназначенные номера

Администратор может направлять звонки на неназначенные номера в вашей организации. Например, вы можете перенаправить вызовы на неназначенные номера следующим образом: 

- Перенаправьте все вызовы на заданный неназначаемый номер в пользовательское объявление.

- Перенаправьте все вызовы на заданный неназначаемый номер в основной коммутатор.

Вы можете направлять вызовы на неназначенные номера пользователю, в учетную запись ресурса, связанную с автосекретарем или очередью вызовов, или в службу объявлений, которая будет воспроизводить пользовательский звуковой файл вызывающей.

Вы можете настроить маршрутизацию неназначенных номеров с помощью Центра администрирования Teams или с помощью PowerShell.

## <a name="use-teams-admin-center"></a>Использование Центра администрирования Teams

1. В левой области навигации Центра администрирования Microsoft Teams перейдите в раздел **Номера голосовых** > **телефонов**.

2. На вкладке **Правила маршрутизации** нажмите кнопку **Добавить**.

3. Присвойте правилу имя, описание и укажите порядок оценки для правила.

4. Определите, какое правило нужно добавить. Вы можете выбрать правила, в которых тип шаблона номера телефона предварительно настроен, и завершить шаблон и параметр маршрутизации. Вы также можете выбрать расширенную настройку, где вы непосредственно вводите регулярное выражение для шаблона номера телефона и параметр маршрутизации.

5. Нажмите **Сохранить**.

Вы также можете создать правило маршрутизации непосредственно для неназначенных номеров телефона.

1. В левой области навигации Центра администрирования Microsoft Teams перейдите в раздел **Номера голосовых** > **телефонов**.

2. На вкладке **Числа** выберите неназначаемый номер телефона и нажмите кнопку **Маршрут** в верхней части представления.

4. Присвойте правилу имя, описание и укажите порядок оценки для правила.

4. Выберите параметр маршрутизации.

5. Нажмите **Сохранить**.

Вы можете протестировать правила маршрутизации с помощью функции "Тестировать номер".

1. В левой области навигации Центра администрирования Microsoft Teams перейдите в раздел **Номера голосовых** > **телефонов**.

2. На вкладке **Правила маршрутизации** щелкните **Тестовый номер**.

3. Введите номер телефона напрямую или щелкните **Выбрать номер** и выберите один из неназначенных номеров телефонов в раскрывающемся списке.

4. Выберите **Тест**.


## <a name="use-powershell"></a>Использование PowerShell

Для маршрутизации вызовов на неназначенный номер используйте командлет New/Get/Set/Remove-CsTeamsUnassignedNumberTreatment, доступный в модуле Teams PowerShell 2.5.1 или более поздней версии.

Необходимо указать вызываемого номера или диапазон номеров и соответствующую маршрутизацию для звонков на эти номера. Например, следующая команда указывает, что все вызовы на номер +1 (555) 222-3333 будут перенаправлены в учетную запись ресурса aa@contoso.com:

``` PowerShell
$RAObjectId = (Get-CsOnlineApplicationInstance -Identity aa@contoso.com).ObjectId

New-CsTeamsUnassignedNumberTreatment -Identity MainAA -Pattern "^\+15552223333$" -TargetType ResourceAccount -Target $RAObjectId -TreatmentPriority 1
```

В следующем примере указывается, что все вызовы в диапазоне номеров +1 (555) 333-0000 до +1 (555) 333-9999 будут перенаправлены в службу объявления, которая будет воспроизводить звуковой файл MainAnnouncement.wav вызывающей.

```PowerShell
$Content = [System.IO.File]::ReadAllBytes('C:\Media\MainAnnouncement.wav')

$AudioFile = Import-CsOnlineAudioFile -FileName "MainAnnouncement.wav" -Content $Content

$fid = [System.Guid]::Parse($AudioFile.Id)

New-CsTeamsUnassignedNumberTreatment -Identity TR1 -Pattern "^\+1555333\d{4}$" -TargetType Announcement -Target $fid.Guid -TreatmentPriority 2
```

## <a name="notes"></a>Notes

- При маршрутизации к объявлению звуковой файл будет воспроизводиться один раз вызывающей.

- Чтобы перенаправить вызовы на неназначенные номера подписчиков Плана звонков Майкрософт, ваш клиент должен иметь [доступные кредиты на связь](what-are-communications-credits.md).

- Чтобы перенаправить вызовы на неназначенные номера служб плана звонков Майкрософт, у клиента должна быть по крайней мере одна лицензия **Телефонная система Microsoft Teams учетной записи ресурсов**.

- Поддерживаемые пользовательские форматы аудиофайлов: WAV (несжатый, линейный PCM с глубиной 8/16/32 в моно или стерео), WMA (только моно) и MP3. Размер содержимого аудиофайла не может превышать 5 МБ.

> [!NOTE]
> Вы несете ответственность за независимое удаление и защиту всех необходимых прав и разрешений на использование любой музыки или аудиофайла в службе Microsoft Teams. Это может включать интеллектуальную собственность и другие права на любую музыку, звуковые эффекты, аудио, торговые марки, имена и другое содержимое аудиофайла от всех соответствующих владельцев прав. К держателям могут относиться художники, актеры, исполнители, музыканты, авторы песен, композиторы, звукозаписыватели, издатели музыки, профсоюзы, гильдии, правозащитные общества, организации коллективного управления и любые другие стороны, которые владеют авторскими правами на музыку, звуковыми эффектами, звуковыми эффектами и другими правами интеллектуальной собственности.

- Как для входящих вызовов в Microsoft Teams, так и для исходящих вызовов из Microsoft Teams будет проверяться номер вызова в диапазоне неназначенных номеров.

- Если указанный шаблон или диапазон содержит номера телефонов, назначенные пользователю или учетной записи ресурса в клиенте, звонки на эти номера телефонов будут направляться в соответствующий целевой объект, а не направляться на указанную обработку неназначенных номеров. Другие проверки чисел в диапазоне отсутствуют. Если диапазон содержит допустимый внешний номер телефона, исходящие звонки из Microsoft Teams на этот номер телефона будут направляться в соответствии с процедурой.
