# <a name="list-of-session-border-controllers-certified-for-direct-routing"></a>Список пограничных контроллеров сеансов, сертифицированных для прямой маршрутизации

Корпорация Майкрософт сотрудничает с рядом поставщиков пограничных контроллеров сеансов (SBC) и сертифицирует их контроллеры для работы с прямой маршрутизацией 

Корпорация Майкрософт сотрудничает с каждым поставщиком в: 

- Совместная работа над протоколами межсоединения SIP.
- Проводите сильное тестирование с помощью стороннее лабораторных анализов. Сертифицированы только устройства, которые прошли проверку. 
- Ежедневно проводите тесты на всех сертифицированных устройствах в производственной и предварительной средах. проверка устройств в предварительных средах гарантирует, что новые версии кода прямой маршрутизации в облаке будут работать с сертифицированными контроллерами SBC; 
- Совместная поддержка с поставщиками SBC.


  > [!NOTE]
  > Корпорация Майкрософт поддерживает телефонную систему только в том случае, если сертифицированные устройства подключены с помощью прямой маршрутки. Корпорация Майкрософт оставляет за собой право отклонить случаи, когда устройство, не сертифицированное к телефонной системе, подключено к телефонной системе с помощью прямой маршрутинга. Если корпорация Майкрософт определяет, что проблема с прямой маршрутией клиента имеется в устройстве SBC поставщика, клиенту потребуется привлечь его за поддержкой.

В таблице ниже перечислены устройства, сертифицированные для прямой маршрутизации. (Сведения о том, какие поставщики SBC поддерживают локализованную оптимизацию мультимедиа, см. в сведениях о настройке локальной оптимизации мультимедиа для [прямой маршрутизации.)](direct-routing-media-optimization-configure.md)
