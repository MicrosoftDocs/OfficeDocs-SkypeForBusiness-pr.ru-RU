<span data-ttu-id="103bc-104">Корпорация Майкрософт сотрудничает с рядом поставщиков пограничных контроллеров сеансов (SBC) и гарантирует, что их контроллеры сертифицированы для работы с прямой маршрутизацией.</span><span class="sxs-lookup"><span data-stu-id="103bc-104">Microsoft partners with selected Session Border Controllers (SBC) vendors to certify that their SBCs for work with Direct Routing.</span></span> 

<span data-ttu-id="103bc-105">При этом корпорация Майкрософт:</span><span class="sxs-lookup"><span data-stu-id="103bc-105">With each vendor, Microsoft:</span></span> 

- <span data-ttu-id="103bc-106">совместно с поставщиками SBC разрабатывает протоколы взаимодействия SIP;</span><span class="sxs-lookup"><span data-stu-id="103bc-106">Jointly works with the SBC vendors on the SIP interconnection protocols.</span></span>
- <span data-ttu-id="103bc-107">проводит интенсивные тесты в сторонней лаборатории (сертифицируются только устройства, прошедшие проверку);</span><span class="sxs-lookup"><span data-stu-id="103bc-107">Performs intense tests via a third-party lab; only devices that passed the tests are certified.</span></span> 
- <span data-ttu-id="103bc-108">проводит ежедневные тесты на всех сертифицированных устройствах в рабочей и предварительной средах —</span><span class="sxs-lookup"><span data-stu-id="103bc-108">Runs daily tests with all certified devices in production and pre-production environments.</span></span> <span data-ttu-id="103bc-109">проверка устройств в предварительных средах гарантирует, что новые версии кода прямой маршрутизации в облаке будут работать с сертифицированными контроллерами SBC;</span><span class="sxs-lookup"><span data-stu-id="103bc-109">Validating the devices in pre-production environments guarantees that new versions of Direct Routing code in the cloud will work with certified SBCs.</span></span>
