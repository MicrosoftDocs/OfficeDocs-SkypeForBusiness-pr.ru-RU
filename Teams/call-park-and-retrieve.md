# <a name="call-park-and-retrieve-in-microsoft-teams"></a>Парковка и восстановление звонков в Microsoft Teams

Метод парковки и извлечения звонков — это функция, с помощью которой пользователь может помещать Звонок на удержание в службе Teams в облаке. При приостановке звонка служба генерирует уникальный код для получения вызова. Пользователь, который приприпаркован звонок или другой пользователь может затем использовать этот код и поддерживаемое приложение или устройство для получения вызова. 

Ниже приведены некоторые распространенные сценарии использования функции приостановки звонка. 

- Рецептионист парки вызов для того, чтобы кто-то работал в фабрике. Затем рецептионист объявляет звонок и номер кода в системе общедоступной адресной системы. Пользователь, для которого выполняется звонок, может затем выбрать телефон на телефоне в фабричном цехе и ввести код для получения звонка.
- Пользователь парки Звонок на мобильном устройстве из-за отсутствия питания на батарее устройства. Пользователь может ввести этот код, чтобы получить звонок с телефонного рабочего стола Teams.
- Представитель службы поддержки парки звонок абоненту и отправляет извещение в канале Teams для эксперта по получению звонка и помогать клиенту. Эксперт вводит код в клиентские группы, чтобы получить звонок.

> [!IMPORTANT]
> Эта функция доступна только в режиме развертывания Teams. Для получения дополнительных сведений о режимах развертывания Teams ознакомьтесь со статьей общие сведения о [сосуществовании и взаимодействии в Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md)

## <a name="license-required"></a>Требуется лицензия

Чтобы приостановить и получить звонки, пользователь должен быть вашим корпоративным голосом, а администратор должен предоставить пользователю политику парковки для приема звонков. Дополнительные сведения о модели лицензирования можно найти в разделе [Лицензирование Office 365 для Microsoft Teams](office-365-licensing.md).

## <a name="call-park-and-retrieve-feature-availability"></a>Приостановка звонков и получение сведений о доступности компонентов

Метод парковки и извлечения в настоящее время поддерживается следующими клиентами и устройствами. (Поддерживается только в режиме "только в Teams", с подключением и без поддержки PSTN.)

| Возможностями | Классическое приложение Teams | Приложение Teams для Mac | Teams Web App (EDGE) |Приложение Teams Mobile для iOS и Android | IP-телефон для Teams | IP-телефон Skype для бизнеса |
|------------|---------------|---------------|----------------------|-----------------------------|----------------|-----------------------------|
| Приостановка звонка | Да | Да | Да | Да | Ожидается в ближайшее время| Нет |
| Получение приостановленного звонка | Да | Да | Да | Да | Ожидается в ближайшее время| Нет |
| Неизвлеченный ответный звонок | Да | Да | Да | Да | Ожидается в ближайшее время| Нет |

## <a name="configuring-call-park-and-retrieve"></a>Настройка парковки и извлечение звонков

Вы должны быть администратором для настройки парковки и получения звонков, и эта функция отключена по умолчанию. Вы можете включить ее для пользователей и создавать группы пользователей с помощью политики парковки звонков. Если вы применяете одну и ту же политику к набору пользователей, они могут приостановить и получить звонки между собой. Чтобы настроить приостановку звонков для пользователей и создать группы пользователей для парковки на приостановке, следуйте инструкциям, приведенным в разделе [Назначение политики приостановки вызова](#assign-a-call-park-policy) .

Сведения о том, как использовать функцию парковки и получения звонков, можно найти [в разделе Присоединение к звонку в Teams](https://support.office.com/article/park-a-call-in-teams-8538c063-d676-4e9a-8045-fc3b7299bb2f).

### <a name="enable-a-call-park-policy"></a>Включение политики парковки звонков

Чтобы включить политику парковки звонков, выполните указанные ниже действия.

1. Перейдите в раздел**политики**в >  **центре администрирования Microsoft Teams**для**голосовой** > связи.
2. Выберите пункт **создать политику**.
3. Присвойте политике имя, а затем установите переключатель **Разрешить приостановку звонка** **на включен**.
4. Нажмите кнопку **сохранить**.

### <a name="assign-a-call-park-policy"></a>Назначение политики парковки звонков

Выполните указанные ниже действия, чтобы назначить политику приостановки вызова для одного или нескольких пользователей.

1. Перейдите в раздел**политики**в >  **центре администрирования Microsoft Teams**для**голосовой** > связи.
2. Выберите политику, щелкнув слева от имени политики.
3. Выберите **Управление пользователями**.
4. В области **Управление пользователями** найдите пользователя по отображаемому имени или по имени пользователя, выберите имя и нажмите кнопку **Добавить**. Повторите этот шаг для каждого пользователя, которого вы хотите добавить.
5. Завершив добавление пользователей, нажмите кнопку **сохранить**.
 
### <a name="configure-call-park-and-retrieve-with-powershell"></a>Настройка парковки звонков и получение с помощью PowerShell

Используйте командлет PowerShell [New-кстеамскаллпаркполици](https://docs.microsoft.com/powershell/module/skype/new-csteamscallparkpolicy?view=skype-ps) , чтобы создать политику парковки звонков.

С помощью командлета PowerShell [Grant-кстеамскаллпаркполици](https://docs.microsoft.com/powershell/module/skype/grant-csteamscallparkpolicy?view=skype-ps) можно предоставить политику парковки звонков.

Вы можете изменить параметр по умолчанию, используя [Set-кстеамскаллпаркполици](https://docs.microsoft.com/powershell/module/skype/set-csteamscallparkpolicy?view=skype-ps) , как описано ниже.

`Set-CsTeamsCallParkPolicy -Identity Global -AllowCallPark $true`


## <a name="troubleshooting"></a>Поиск и устранение неполадок

Если пользователи не видят кнопку "приостановить" или "получить": 

- Убедитесь в том, что у пользователя включена политика парковки на звонки. 

Если пользователь попытается получить вызов и не прошел его, проверьте следующее:

- Убедитесь в том, что пользователь использует клиент Teams или устройство или телефон с поддержкой Teams.
- Группировка — пользователь является членом группы "парковки звонков", на основе которой назначена та же политика приостановки вызова для групп. 
- Режим острова: метод парковки и извлечение недоступен в режиме островов Teams.
- Звонок уже получен или прерван.
