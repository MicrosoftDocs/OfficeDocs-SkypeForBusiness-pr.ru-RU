# <a name="call-park-and-retrieve-in-microsoft-teams"></a>Парковка и восстановление звонков в Microsoft Teams

Парковка вызовов и получение — это функциональная возможность, которая позволяет пользователю удержание звонка в службу группами в облаке. При приостановке звонка службу создает уникальный код для извлечения звонок. Выберите пользователя, который приостанавливать звонок или другому пользователю можно использовать кода и поддерживаемые приложение или устройство для возврата звонка. 

Ниже указаны некоторые общие сценарии для использования парковки вызовов. 

- Для кого-либо работа в фабрике секретарю parks звонка. Секретари нажмите сигнал вызова и номер кода через общедоступный адрес системы. Пользователь, для вызова можно трубку телефона группами в рабочей среде и введите код для возврата звонка.
- Пользователь parks звонка на мобильном устройстве, так как батарей устройства, недостаточно питания. Затем пользователь может ввести код, чтобы вернуть звонок с группами стационарный телефон.
- Пример парков поддержки клиента вызова и отправляет оповещения по каналу группами для expert возврата звонка и помощь в клиента. Эксперт вводит код в клиентах группами для возврата звонка

> [!IMPORTANT]
> Эта функция доступна только в режиме только группы развертывания. Дополнительные сведения о режимах развертывания группы в разделе [понять группами Майкрософт и Скайп для бизнеса сосуществования и взаимодействия](teams-and-skypeforbusiness-coexistence-and-interoperability.md)

## <a name="license-required"></a>Требуется лицензия

Чтобы приостановить и извлечение вызовов, пользователь должен быть пользователем корпоративной голосовой связи и администратор должен предоставить пользователю политику парковки вызовов. Дополнительные сведения о модели лицензирования см в [Office 365 лицензирования для групп Майкрософт](office-365-licensing.md).

Парковка вызовов и извлекать в настоящее время поддерживается следующие клиенты и устройства. (Поддерживается только группы режиме или без подключения к ТСОП.)

|------------|---------------|---------------|----------------------|-----------------------------|----------------|-----------------------------|
| Приостановка звонка | Да | Да | Да | Да | Ожидается в ближайшее время| Нет |
| Восстановление приостановленного звонка | Да | Да | Да | Да | Ожидается в ближайшее время| Нет |

## <a name="configuring-call-park-and-retrieve"></a>Настройка парковки вызовов и извлечения

Необходимо иметь права администратора для настройки парковки вызовов и получения, и эта функция отключена по умолчанию. Можно включить для пользователей и создание групп пользователей, с помощью политики парковки вызовов. В случае применения та же политика для группы пользователей, они могут приостанавливать и получить вызовы друг с другом. Для настройки парковки вызовов для пользователей и создания парковки вызовов групп пользователей, выполните следующие действия [Назначение политики парковки вызовов](#assign-a-call-park-policy) .

Сведения о том, как использовать парковки вызовов и получить возможность видеть [парковки вызовов в группах](https://support.office.com/article/park-a-call-in-teams-8538c063-d676-4e9a-8045-fc3b7299bb2f).

### <a name="enable-a-call-park-policy"></a>Включите политику парковки вызовов

Выполните следующие действия, чтобы включить политику парковки вызовов.

1. Перейдите в **Центр администрирования группами Майкрософт** > **голосовой связи** > **политик парковки вызовов**.
2. Выберите **новую политику**.
3. Задайте имя политики и перейдите в **на** **парковки разрешить звонок** .
4. Нажмите кнопку **Сохранить**.

### <a name="assign-a-call-park-policy"></a>Назначение политики парковки вызовов

Выполните следующие действия, чтобы назначить политику парковки вызовов для одного или нескольких пользователей.

1. Перейдите в **Центр администрирования группами Майкрософт** > **голосовой связи** > **политик парковки вызовов**.
2. Выберите политику, нажав кнопку слева от имени политики.
3. Выберите **Управление пользователями**.
4. В области **Управление пользователями** поиска для пользователя по отображаемому имени или по имени пользователя, выберите имя и выберите команду **Добавить**. Повторите этот шаг для каждого пользователя, который вы хотите добавить.
5. Завершив добавление пользователей, выберите команду **Сохранить**.
 
### <a name="configure-call-park-and-retrieve-with-powershell"></a>Настройка парковки вызовов и получить с помощью PowerShell

Командлет [New-CsTeamsCallParkPolicy](https://docs.microsoft.com/powershell/module/skype/new-csteamscallparkpolicy?view=skype-ps) PowerShell для создания политики парковки вызовов.

Командлет [Grant-CsTeamsCallParkPolicy](https://docs.microsoft.com/powershell/module/skype/grant-csteamscallparkpolicy?view=skype-ps) PowerShell для предоставления политику парковки вызовов.

По умолчанию можно изменить с помощью [Set-CsTeamsCallParkPolicy](https://docs.microsoft.com/powershell/module/skype/set-csteamscallparkpolicy?view=skype-ps) следующим образом:

`Set-CsTeamsCallParkPolicy -Identity Global -AllowCallPark $true`


## <a name="troubleshooting"></a>Поиск и устранение неполадок

Если пользователи не могут видеть парковки и получить кнопки: 

- Убедитесь, что пользователь имеет политикой парковки вызовов. 

Если пользователь пытается получить вызов, а не выполняется, проверьте следующее:

- Убедитесь, что пользователь использует клиент групп или с поддержкой групп устройств/телефона
- Группирование — — пользователь, должна быть членом группы парковки вызовов?
- Остров режим – парковки вызовов и извлекать недоступны в режиме остров группами.
- Вызов уже извлечен или завершен.
