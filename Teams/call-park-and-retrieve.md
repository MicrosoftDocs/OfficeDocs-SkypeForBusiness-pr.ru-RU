# <a name="call-park-and-retrieve-in-microsoft-teams"></a>Парковка и восстановление звонков в Microsoft Teams

Метод парковки и получения звонков — это функция, с помощью которой пользователь может поставить звонок на удержание. При приостановке звонка служба генерирует уникальный код для получения вызова. Пользователь, который приприпаркован звонок или другой пользователь может затем использовать этот код с поддерживаемым приложением или устройством для получения вызова. (Дополнительные сведения см. в разделе Приостановка [звонка в Teams](https://support.office.com/article/park-a-call-in-teams-8538c063-d676-4e9a-8045-fc3b7299bb2f) .)

Ниже приведены некоторые распространенные сценарии использования функции приостановки звонка.

- Receptionist парки вызов для того, чтобы кто-то работал в фабрике. Затем receptionist объявляет звонок и номер кода в системе общедоступной адресной системы. Пользователь, для которого выполняется звонок, может затем выбрать телефон на телефоне в фабричном цехе и ввести код для получения звонка.
- Пользователь парки Звонок на мобильном устройстве из-за отсутствия питания на батарее устройства. Пользователь может ввести этот код, чтобы получить звонок с телефонного рабочего стола Teams.
- Представитель службы поддержки парки звонок абоненту и отправляет извещение в канале Teams для эксперта по получению звонка и помогать клиенту. Эксперт вводит код в клиентские группы, чтобы получить звонок.

Чтобы приостановить и получить звонки, пользователь должен быть пользователем голосовой связи и должен быть включен в политику приостановки звонка.

> [!NOTE]
> Функция парковки и извлечение звонков доступна только в [режиме развертывания Teams](teams-and-skypeforbusiness-coexistence-and-interoperability.md) и не поддерживается на IP-телефонах Skype для бизнеса.

## <a name="configure-call-park-and-retrieve"></a>Настройка парковки и извлечение звонков

Вы должны быть администратором Teams, чтобы настроить приостановку и получение звонков. По умолчанию он отключен. Вы можете включить ее для пользователей и создавать группы пользователей с помощью политики парковки звонков. Если вы применяете одну и ту же политику к набору пользователей, они могут приостановить и получить звонки между собой.

Включение политики парковки звонков

1. В левой области навигации центра администрирования Microsoft Teams перейдите к **Voice**  >  **политикам парковки**для приема голосовой связи.
2. На вкладке **Управление политиками** нажмите кнопку **Добавить**.
3. Присвойте политике имя, а затем установите переключатель **Разрешить приостановку звонка** **на включен**.

    ![Снимок экрана: настройки политики парковки звонков](media/call-park-add-policy.png)

4. Нажмите кнопку **сохранить**.

Чтобы изменить политику, выберите ее в списке и нажмите кнопку **изменить**.

Чтобы политика работала, она должна быть назначена пользователям. Политику можно [назначить пользователям по отдельности](assign-policies.md) или назначить группе.

Назначение групповой политики вызова группе

1. На странице **политики парковки звонков** на вкладке **назначение групповой политики** нажмите кнопку **Добавить группу**.
2. Найдите группу, которую вы хотите использовать, и нажмите кнопку **Добавить**.
3. Выберите ранг по сравнению с другими назначениями групп.
4. В разделе **Выбор политики**выберите политику, которой вы хотите назначить эту группу.

    ![](media/call-park-assign-policy-to-group.png)

5. Нажмите кнопку **Применить**.
