# <a name="configure-call-park-and-retrieve"></a>Настройка парковки вызовов и извлечение

Приостановка и извлечение вызовов позволяет пользователю поместить звонок на удержание. При приостановки вызова служба создает уникальный код для извлечения вызова. Пользователь, приостановивший вызов, или кто-то другой может использовать этот код с поддерживаемым приложением или устройством для получения вызова. (Дополнительные сведения см [. в разделе "Приостановка звонка в Teams](https://support.office.com/article/park-a-call-in-teams-8538c063-d676-4e9a-8045-fc3b7299bb2f)".)

Ниже приведены некоторые распространенные сценарии использования парковки вызовов.

- Администратор вызывает человека, работающего в фабрике. Затем регистратор объявляет звонок и номер кода через общедоступную адресную систему. Затем пользователь, для которого выполняется звонок, может получить телефон Teams на производственном цехе и ввести код для получения звонка.

- Пользователь выполняет звонок на мобильном устройстве из-за того, что аккумулятор устройства не работает. Затем пользователь может ввести код для получения звонка с рабочего телефона Teams.

- Представитель службы поддержки отвечает на звонок клиента и отправляет в канал Teams объявление, чтобы эксперт извлекал звонок и помогал клиенту. Эксперт вводит код в клиентах Teams для получения вызова.

Для приостановки и извлечения вызовов пользователь должен быть Корпоративная голосовая связь пользователем и должен быть включен в политику парковки вызовов.

По умолчанию диапазон номеров для ответа на звонок — от 10 до 99. Вы также можете создать собственный пользовательский диапазон в диапазоне от 10 до 9999. Первый припаркованный вызов будет отображать код ответа на начало диапазона (например, 10). Следующий припаркованный вызов будет отображать код ответа, который увеличивается на 1; то есть 11 и т. д., пока конец диапазона не будет преобразован в код ответа. После этого отрисованные коды возврата снова начинаются с начала диапазона. 

Вы можете указать время ожидания в секундах, прежде чем звонить обратно, когда приостановленный вызов не был выполнен. Допустимый диапазон — 120–1800 секунд, а значение по умолчанию — 300 секунд.

Чтобы настроить парковку вызовов и получить ее, необходимо быть администратором Teams. По умолчанию он отключен. Вы можете включить его для пользователей и создать группы пользователей с помощью политики парковки вызовов. При применении одной и той же политики к набору пользователей они могут припарковать и извлечь вызовы между собой.

> [!NOTE]
> Парковка вызовов и извлечение доступны только в [режиме развертывания Teams Only](teams-and-skypeforbusiness-coexistence-and-interoperability.md). Он не поддерживается на Skype для бизнеса IP-телефонов.

Вы можете настроить парковку вызовов и получить ее с помощью Центра администрирования Teams или PowerShell.

## <a name="use-teams-admin-center"></a>Использование Центра администрирования Teams

Чтобы создать новый экземпляр политики парковки вызовов:

1. В левой области навигации Центра администрирования Microsoft Teams перейдите **к политикам** >  парковки **голосовых вызовов**.

2. На **вкладке "Управление политиками** " нажмите кнопку " **Добавить"**.

3. Присвойте политике имя, а затем переключите **параметр "Разрешить парковку вызовов** " на **"Включено"**.

4. Измените диапазон и время ожидания приостановки при необходимости.

5. Нажмите **Сохранить**.

Вы можете изменить политику, выбрав ее в списке и нажав кнопку **"Изменить"**.

Для работы политики ее необходимо назначить пользователям. Политику можно [назначить пользователям по отдельности](assign-policies-users-and-groups.md) или группе.

Чтобы назначить группу политику парковки вызовов:

1. На странице **"Политики парковки вызовов**" на  вкладке назначения групповой политики нажмите кнопку **"Добавить группу"**.

2. Найдите группу, которую вы хотите использовать, и нажмите кнопку **"Добавить"**.

3. Выберите ранг по сравнению с другими назначениями групп.

4. В **разделе "Выбор политики**" выберите политику, которой вы хотите назначить эту группу.

5. Нажмите **Применить**.

## <a name="use-powershell"></a>Использование PowerShell

Чтобы управлять парковкой вызовов и получать политики с помощью PowerShell, используйте следующие командлеты модуля Teams PowerShell:

- [New-CsTeamsCallParkPolicy](/powershell/module/skype/new-csteamscallparkpolicy)

- [Set-CsTeamsCallParkPolicy](/powershell/module/skype/set-csteamscallparkpolicy)

- [Get-CsTeamsCallParkPolicy](/powershell/module/skype/get-csteamscallparkpolicy)

- [Remove-CsTeamsCallParkPolicy](/powershell/module/skype/remove-csteamscallparkpolicy)

- [Grant-CsTeamsCallParkPolicy](/powershell/module/skype/grant-csteamscallparkpolicy)

### <a name="examples"></a>Примеры

#### <a name="new-custom-call-park-policy"></a>Новая настраиваемая политика парковки вызовов

В следующем примере создается настраиваемая политика парковки вызовов, которая создает номера для получения от 500 до 1500 и будет звонить обратно в парковку через 600 секунд, если припаркованный вызов не будет отвечен.

```powershell
PS C:\> New-CsTeamsCallParkPolicy -Identity "SalesPolicy" -AllowCallPark $true -PickupRangeStart 500 -PickupRangeEnd 1500 -ParkTimeoutSeconds 600
```

#### <a name="change-a-call-park-policy"></a>Изменение политики парковки вызовов

Следующий пример отключает политику парковки вызовов:

```powershell
PS C:\> Set-CsTeamsCallParkPolicy -Identity "SalesPolicy" -AllowCallPark $false
```

#### <a name="grant-a-call-park-policy-to-a-user"></a>Предоставление пользователю политики парковки вызовов

В следующем примере политика парковки вызовов предоставляется пользователю:

```powershell
PS C:\> Grant-CsTeamsCallParkPolicy -PolicyName "SalesPolicy" -Identity Ken.Myer@contoso.com
```

#### <a name="remove-a-call-park-policy"></a>Удаление политики парковки вызовов

В следующем примере удаляется политика парковки вызовов:

```powershell
PS C:\> Remove-CsTeamsCallParkPolicy -Identity "SalesPolicy"
```
