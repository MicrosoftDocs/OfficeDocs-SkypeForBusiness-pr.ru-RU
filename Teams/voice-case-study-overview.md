# <a name="contoso-case-study-teams-voice-migration-overview"></a>Исследование успеха Contoso: Общие сведения о переносе голосовых команд

## <a name="contoso-business-goals-for-teams"></a>Бизнес-цели Contoso для Teams

Для миграции локальных пользователей в Teams для единой системы обмена сообщениями, совместной работы и голосовой связи компания Contoso решила следующие бизнес-цели:

- Включение в Teams 

  Группа единой системы обмена сообщениями и совместной работы в Contoso включает в себя необходимые политики для управления и обеспечения безопасной внутренней и внешней совместной работы. 

- Переход со Skype для бизнеса на Teams 

  Skype для бизнеса был широко развернут в компании Contoso. После того как вам нужно будет продвигать старые системы, компания Contoso решила обновить пользователей Skype для бизнеса до Teams. Дополнительные сведения можно найти в статье [исследование успеха Contoso: план обновления Teams](voice-case-study-migration-plan.md).

- Телефонная система  

  Skype для бизнеса с корпоративной голосовой связью был широко развернут в компании Contoso. При необходимости отключить старые системы, которые были следующим прыжком для серверов-исправлений, компания Contoso перенеса пользователей голосовой связи Skype для бизнеса в телефонную систему. Сайты Contoso использовали план звонков Microsoft, прямую маршрутизацию телефонной системы или их сочетание. Дополнительные сведения можно найти в статье [исследование успеха Contoso: телефонная система](voice-case-study-phone-system.md).

- Маршрутизация на основе расположения 

  С помощью расположений Office в странах, использующих телефонную связь, компания Contoso требовала повторного создания маршрутизации на основе местоположения, которая была представлена в Skype для бизнеса в своем развертывании телефонной системы. Дополнительные сведения можно найти в статье [исследование успеха Contoso: Маршрутизация на основе местоположения](voice-case-study-location-based-routing.md).

- Экстренный вызов 

  Если вы реализовали прямую переадресацию, компания Contoso настроила вызов экстренной помощи с утвержденными третьими лицами. Дополнительные сведения можно найти в статье [исследование успеха Contoso: вызов экстренной](voice-case-study-emergency-calling.md)помощи.

- Аудиоконференция 

  Компания Contoso настроила номера служб голосовой связи, которые были размещены на их магистралье SIP с поставщиком PSTN. Дополнительные сведения можно найти в статье [исследование успеха Contoso: Голосовая конференция](voice-case-study-audio-conferencing.md). 

- Оптимизация локальных файлов мультимедиа 

  Компания Contoso проделала преимущества локальной оптимизации мультимедиа в расположениях, в которых у них есть прямая маршрутизация на телефонную систему Microsoft, которая использовалась удаленными сайтами. Дополнительные сведения можно найти в разделе [планирование оптимизации локальных файлов](direct-routing-media-optimization.md) и [Настройка оптимизации локальных файлов мультимедиа](direct-routing-media-optimization-configure.md).

- Автоматические ассистенты и очереди звонков

  В результате Covid-19 компания Contoso хотела бы предоставить поддержку receptionist, когда их сотрудники работали в удаленном режиме. Компания Contoso использовала автоматические ассистенты и очереди звонков для управления входящими звонками на номер телефона receptionist. Дополнительные сведения можно найти в статье [исследование успеха Contoso: автосекретарей и очереди звонков](voice-case-study-call-queues.md).
