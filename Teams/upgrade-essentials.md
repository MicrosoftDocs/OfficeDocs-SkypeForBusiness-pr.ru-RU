# <a name="about-the-upgrade-essentials-checklist"></a>О контрольный список обновления Essentials 

Предназначена для небольших организаций или тех, кто использует Скайп для бизнеса в Интернет для обмена мгновенными Сообщениями (разговор) и собрания только контрольный список обновления Essentials — план ускорение действий, который включает в себя процессора с тактовой частотой рекомендуемые действия и связанные ресурсы, необходимые для реализации успешное перемещение из Скайп для бизнеса группам.

Десять следующие действия укажите все необходимые для обновления успеха. Обратите внимание на то, что временной шкалы является приблизительным; необходимо изменить его на основе графика обновления вашей организации.

Не знакомы с группами? [Ознакомьтесь со сведениями о том, как группы](https://products.office.com/en-us/microsoft-teams/group-chat-software) объединяет бесед, собраний, файлов, приложения Office и сторонних производителей интеграции — предоставление одного сервера-концентратора для работы в Office 365.

Есть вопросы о обновления? Вы можете просмотрите вопросы и ответы.

<a name="step-1"></a>

## <a name="step-1-notify-your-key-stakeholders"></a>Шаг 1. Уведомить ключевых заинтересованных лиц

*(Около четырех до шести недель до обновления)*


Старший руководители несут ответственность за успех компании; обязательно сохраните их и тенденциями об изменениях технологии. Так как не всем полученным и прочитать уведомление о возможности обновления, необходимо проинформировать другие заинтересованные стороны (например, CEO, ИТ преимуществ, отдела маркетинга и ведущие сотрудники службы поддержки) перед началом планирования обновления.

<a name="step-2"></a>

## <a name="step-2-envision-your-organization-on-teams"></a>Шаг 2. Представления вашей организации в группах

*(Около четырех до шести недель до обновления)*

Команды предлагает совместимого Скайп для бизнес-функции — Мгновенных сообщений (чата) и собраний —, но его можно также выполнить многие другие. Как значение true, сервер-концентратор для командной работы группами позволяет рабочих групп для управления проектами, файлы, беседы и приложений в одном месте. Решите, как будут использование групп и включение среды для успеха вашей организации.

> Как существующие Скайп для клиента Business текущей сетевой инфраструктуры скорее всего, уже настроен для групп. Чтобы проверить это, можно следовать указаниям «Выполнения технические планирования», связанный с ниже (необязательно).
