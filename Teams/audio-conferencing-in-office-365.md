# <a name="audio-conferencing-in-office-365"></a>Аудиоконференции в Office 365
Голосовые конференции в Office 365 дают пользователям возможность звонить на собрания с телефонов. Голосовые конференции допускают до 250 мобильных участников.

## <a name="what-is-audio-conferencing"></a>Что такое Голосовая Конференция?
Звонки в Skype для собраний очень полезны для пользователей, находящихся в пути, и не могут принять участие в собрании с помощью приложения Microsoft Teams, в том числе на ноутбуках или на мобильных устройствах. Но существуют и другие сценарии, в которых для участия в собрании Skype для бизнеса или Microsoft Teams может быть удобнее использовать приложение на компьютере.
  
- Подключение к Интернету ограничено.
- Собрание — это только звук.
- Кто-то попытался присоединиться к собранию Skype для бизнеса, и это не удалось.
- Качество связи лучше при наборе номера.
- Пользователи могут присоединиться к собранию "руки бесплатно" с помощью устройств Bluetooth.
- Поиск людей проще и удобнее в своей ситуации.

Вам нужно только настроить аудиоконференции для пользователей, планирующих или проводящих собрания. Присоединяющимся к собранию по телефону участникам не требуются какие-либо назначенные им лицензии или другие настройки.

После того как участники будут присоединены к собранию, они также смогут звонить и приглашать других абонентов на собрание Skype для бизнеса или Microsoft Teams. Ознакомьтесь с [разбираемым звонком из собрания Teams, чтобы другие люди могли присоединиться](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md) к [собранию и звонить из собрания Skype для бизнеса, чтобы другие люди могли присоединиться к ней](/SkypeForBusiness/audio-conferencing-in-office-365/dialing-out-from-a-meeting-so-other-people-can-join-it).

## <a name="what-does-it-cost"></a>Каковы затраты на это?
Сведения о ценах можно найти в статье [цены для голосовой конференции](https://products.office.com/skype-for-business/audio-conferencing#Requirements).

## <a name="where-is-it-available"></a>Где она доступна?
С помощью голосовой конференции пользователи могут использовать платные и бесплатные телефонные номера для подключения к собраниям. Номера платных платных услуг автоматически назначаются в качестве номеров общих звуковых конференций для организаций, когда они включены для голосовой конференции. Выделенные платные и бесплатные номера можно назначить вашей организации из других городов.

Бесплатные телефонные номера (номера услуг) доступны, но только в некоторых странах и регионах. Чтобы узнать, какие возможности доступны в вашей стране или регионе, ознакомьтесь со сведениями о [доступности страны и региона для голосовой конференции и планов звонков](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).

После того как вы решите, какие видеоконференции вы хотите использовать для своей организации, вам нужно приобрести одну лицензию на **проведение** голосовой связи для каждого сотрудника в Организации, который планирует или размещает звуковое собрание.

## <a name="how-do-conferencing-bridges-work"></a>Как работают мосты конференц-связи?
При настройке голосовой конференции в Skype для бизнеса или Microsoft Teams вы получите мост для голосовой связи. Мост конференц-связи может содержать один или несколько номеров телефонов. Номер телефона, который вы настроили, будет включен в приглашения на собрания для приложений Skype для бизнеса и Microsoft Teams. Вы можете [изменить номера телефонов для моста Конференц](change-the-phone-numbers-on-your-audio-conferencing-bridge.md)-связи, а также [изменить другие параметры моста голосовой конференции](change-the-settings-for-an-audio-conferencing-bridge.md). 
  
С помощью телефонного моста вы можете ответить на звонок для пользователей, которые подключаются к собранию по телефону. Она отвечает вызывающему абоненту с голосовыми запросами из автосекретаря, а затем, в зависимости от ваших настроек, может воспроизводить уведомления и попросить абонентов записать свое имя. С помощью **параметров моста Microsoft** вы можете изменить параметры уведомлений о собраниях и присоединения к собраниям, а также указать продолжительность контактов, которые используются в организаторов собраний [в Microsoft Teams](set-the-pin-length-for-audio-conferencing-meetings-in-teams.md) или в [Skype для бизнеса Online](/SkypeForBusiness/audio-conferencing-in-office-365/set-the-pin-length-for-audio-conferencing-meetings). Организаторов собрания используют контакты для начала собраний, если они не могут присоединиться к собранию с помощью приложения Skype для бизнеса или Microsoft Teams.

## <a name="dial-in-phone-numbers-set-on-an-audio-conferencing-bridge"></a>Телефонные номера для телефонного подключения, заданные в Мосте голосовой конференц-связи
Существует два типа номеров телефонов для голосовой связи, которые можно назначить своему мосту конференц-связи: **Общие** и **выделенные**. Оба типа номеров могут использоваться любым вызывающим абонентом для присоединения к звуковым собраниям, удерживаемым в вашей организации.
  
 **Выделенные телефонные номера**  это номера, доступные только для пользователей внутри организации. Вы можете изменить языки, которые используются при звонках на эти номера. Для этого вам потребуется получить номер телефона службы.
  
 **Общедоступные телефонные номера**  это номера, доступ к которым можно предоставлять другим организациям Office 365. Изменить языки, которые используются при звонках на эти номера, нельзя.
  
Несмотря на то, что номер голосовой конференции по умолчанию, назначенный организатору, включен только в приглашение на собрание, абонент может использовать любые телефонные номера, назначенные вашему мосту конференц-связи, чтобы присоединиться к собранию. Список номеров, которые можно использовать для подключения к собранию, открывается при переходе по ссылке **Найти местный номер**, которая включена во все приглашения.

Дополнительные сведения можно найти в разделе [номера телефонов для конференц-связи в Microsoft Teams](phone-numbers-for-audio-conferencing-in-teams.md) или [телефонные номера для голосовой связи в Skype для бизнеса Online](/SkypeForBusiness/audio-conferencing-in-office-365/phone-numbers-for-audio-conferencing).
  
## <a name="automatically-assigned-audio-conferencing-phone-numbers"></a>Автоматически назначаемые номера телефонов для голосовой конференции
Номера телефонов для конференц-связи с телефонным сопровождением автоматически назначаются организациям, в которых они включены для голосовой связи. Полученные номера телефонов назначаются мосту аудиоконференции в качестве номеров по умолчанию. Номер телефона, назначенный по умолчанию мосту, должен соответствовать стране или региону нахождения организации.
  
> [!NOTE]
> Страну или регион Организации можно найти, войдя в **центр администрирования Microsoft 365** и просматривая в разделе " **профиль организации**". 
  
> [!CAUTION]
> Из-за ограниченной доступности платных телефонных номеров в Венесуэла, Индонезия и Объединенных Арабских ОАЭ (ОАЭ) в организации из этих стран или регионов не будет автоматически назначаться номер для голосовой конференции. Бесплатные номера доступны для этих стран независимо от существующих ограничений. 
  
Список этих стран или регионов, имеющих номера телефонов, автоматически назначаемых для организаций, приведен в разделе [Страны и регионы, для которых доступны аудиоконференции и тарифные планы](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
  
## <a name="how-do-you-get-dedicated-phone-numbers"></a>Как получить выделенные телефонные номера?
Специальные номера телефонов — это номера служб, которые можно получить и назначить вашей организации. Вы можете получить доступ к мостам для моста конференц-связи с помощью специальных платных и бесплатных телефонных номеров одним из трех способов:

- **Используйте центр администрирования Skype для бизнеса.** В некоторых странах и регионах вы можете получать номера для мостов конференций с помощью центра администрирования Skype для бизнеса. Смотрите раздел [Знакомство с телефонными номерами служб](/microsoftteams/getting-service-phone-numbers).
    
- **Портируйте номера, которые у вас уже есть.** Вы можете перенести существующие номера из текущего поставщика услуг или оператора телефона в Office 365. Для получения дополнительных полезных сведений о том, как это сделать, см. статью [Перемещение телефонных номеров в Office 365](transfer-phone-numbers-to-office-365.md) или [Управление телефонными номерами вашей организации](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).  
  
- **Используйте форму запроса для новых номеров.** Иногда (в зависимости от страны или региона) вы не сможете получить новые телефонные номера с помощью центра администрирования Skype для бизнеса, или вам понадобятся конкретные номера телефонов или коды города. В таких случаях необходимо скачать соответствующую форму, заполнить ее и отправить в нашу службу поддержки. Для получения дополнительных сведений см. статью [Управление номерами телефонов организации](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).
