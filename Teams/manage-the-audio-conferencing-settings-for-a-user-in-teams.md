# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

В учетной записи администратора Office 365 можно по отдельному пользователю организации вносить изменения в параметры аудиоконференций, такие как провайдер, номера платной или бесплатной линии по умолчанию, идентификатор конференции или ПИН-код. Если вы хотите изменить параметры организации, ознакомьтесь со сведениями о том, как [управлять параметрами голосовой](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md)связи в Организации.

## <a name="an-icon-showing-the-microsoft-teams-logo-using-the-microsoft-teams-admin-center"></a>![Значок с логотипом Microsoft Teams](media/teams-logo-30x30.png) С помощью Центра администрирования Microsoft Teams

1. На панели навигации слева выберите пункт **Пользователи**, а затем выберите пользователя из списка доступных пользователей.

2. Нажмите кнопку **изменить**.

3. В разделе **Голосовая конференция**измените любое из указанных ниже действий.

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить или отключить звуковое сопровождение для пользователя, нажмите кнопку **изменить** рядом с кнопкой **звуковые конференции**, а затем в области **звуковые** конференции включите или отключите **голосовую конференцию** .|
|**Отправка сведений о конференции по электронной почте**  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Нажмите кнопку **Сброс идентификатора конференции** , если необходимо сбросить идентификатор конференции для пользователя. Для дополнительных сведений см. [Сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**КРЕПЛЯТЬ** |Чтобы сбросить PIN-код для пользователя, нажмите кнопку **сбросить PIN-код** . Для дополнительных сведений см. [Сбросить PIN-код для аудиоконференций](reset-the-audio-conferencing-pin-in-teams.md). |
|**Номер телефона конференц-связи по умолчанию** (обязательно) |Это номера, заданные в Мосте голосовой связи. Отформатируйте номера в том виде, в котором они должны отображаться в приглашениях на собрания в Skype для бизнеса и Microsoft Teams. Чтобы изменить номер, установленный по умолчанию, нажмите кнопку **изменить** рядом с надписью **звуковая конференция** и выберите номер в разделе **Audio Conferencing** **платный номер**. |
|**Приглашения от этого пользователя могут включать бесплатный номер**|Чтобы изменить этот параметр, нажмите кнопку **изменить** рядом с кнопкой " **звуковые конференции** " и в области " **телеконференции** ", чтобы включить или отключить **Бесплатные номера в приглашениях на собрания от этого пользователя** . |
|**Пользователи без проверки подлинности могут быть первым участником собрания**|Чтобы изменить этот параметр, переключить **пользователей, не прошедших проверку подлинности, может быть первым участником собрания** .
|**Разрешения на исходящие звонки**|Чтобы изменить этот параметр, нажмите кнопку **изменить** рядом с кнопкой **звуковые конференции** и в области **звуковые конференции** выберите один из вариантов в разделе Исходящие **звонки с собраний**.|
