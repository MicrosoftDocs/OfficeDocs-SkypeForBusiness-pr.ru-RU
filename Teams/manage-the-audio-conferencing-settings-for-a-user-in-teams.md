# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление параметрами звука конференц-связи для пользователя в группах Майкрософт

Как администратор Office 365 можно изменить параметры звука конференц-связи — например, поставщика, счета по умолчанию или бесплатный номер, идентификатор конференции или ПИН-код — для отдельного пользователя в организации. Если вы хотите изменить параметры для вашей организации, ознакомьтесь со [управлять параметрами звука конференц-связи для организации](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md).


1. На панели навигации слева щелкните **Пользователи**и затем выберите пользователя в списке Доступные пользователи.

2. В верхней части страницы нажмите кнопку **Изменить**.

3. В разделе **Audio конференц-связи**измените любые из следующих:

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить аудиоконференций или отключить для пользователя, нажмите кнопку **Изменить** рядом с пунктом **Звукового конференц-связи**, а затем в области **Аудиоконференций** , включения и отключения **звука конференц-связи** включено или отключено.|
|**Отправить по электронной почте сведения о конференции**  |Перейдите по этой ссылке только в том случае, если вы хотите сразу же отправить сообщение электронной почты для пользователя с его конференции идентификатор и номер телефона. (Это сообщение не включает ПИН-кода.) В разделе [Отправить сообщение электронной почты для пользователя с помощью их аудиоконференции сведения](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Нажмите кнопку **сбросить идентификатор конференции** , если необходимо выполнить сброс идентификатор конференции для пользователя. Для получения дополнительных сведений см [сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**ПИН-КОД** |Нажмите кнопку **Сбросить ПИН-код** , если необходимо выполнить сброс ПИН-КОДОВ для пользователя. Для получения дополнительных сведений см [Сброс ПИН-кода конференц-связи звука](reset-the-audio-conferencing-pin-in-teams.md). |
|**По умолчанию номер телефона международную конференц-связи** (обязательно) |Это будут номера, заданные для звукового конференц-канала. Форматирование чисел, как они отображаются в Скайп для бизнеса и группами Майкрософт приглашения на собрания. Чтобы изменить номер счета по умолчанию, нажмите кнопку **Изменить** Далее **аудиоконференции** и в области **Звука конференц-связи** , выберите номер в поле **номер счета**. |
|**Приглашения от этого пользователя может включать телефоны**|Чтобы изменить этот параметр, нажмите кнопку **Изменить** рядом с пунктом **Аудиоконференции** и в области **Звука конференц-связи** , включения и отключения **включают бесплатные номера в приглашении на собрание из этого пользователя** включено или отключено. |
|**Разрешения по телефону**|Чтобы изменить этот параметр, нажмите кнопку **Изменить** рядом с пунктом **Аудиоконференции** и в области **Звука конференц-связи** , выберите параметр в группе **Подключение по телефонной линии разрешений из собрания**.|
