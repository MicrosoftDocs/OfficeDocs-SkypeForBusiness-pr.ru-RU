# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

Администраторы Microsoft 365 или Office 365 могут изменять параметры аудиоконференций, такие как поставщик, платный или бесплатный номер по умолчанию, идентификатор конференции или ПИН-код для отдельного пользователя в организации. Если вы хотите изменить параметры для организации, см. раздел "Управление параметрами аудиоконференций [для вашей организации"](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md).

## <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

1. В области навигации слева **щелкните "** Пользователи", а затем выберите пользователя из списка доступных пользователей.

2. Нажмите **кнопку "Изменить"**.

3. В **разделе "Аудиоконференции**" измените любое из следующих значений:

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить или отключить аудиоконференции для пользователя, нажмите кнопку "Изменить" рядом с аудиоконференции **, а** затем в области  аудиоконференций включите или выключите  аудиоконференции.|
|**Отправка сведений о конференции по электронной почте**  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Если **необходимо** сбросить идентификатор конференции для пользователя, нажмите кнопку "Сбросить идентификатор конференции". Для дополнительных сведений см. [Сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**Контактный** |Если **необходимо** сбросить ПИН-код для пользователя, нажмите кнопку "Сбросить ПИН-код". Для дополнительных сведений см. [Сбросить PIN-код для аудиоконференций](reset-the-audio-conferencing-pin-in-teams.md). |
|**Платный номер конференц-связи по** умолчанию (обязательно) |Это будут номера, заданные на мосте аудиоконференций. Форматирование чисел в том виде, в котором они должны Skype для бизнеса приглашениях на собрания Microsoft Teams. Чтобы изменить платный номер по умолчанию,  нажмите кнопку  "Изменить" рядом с аудиоконференции, а затем в области аудиоконференций выберите номер в разделе "Платный **номер"**. Вы также можете задать номера телефонов, добавив их в TeamsAudioConferencingPolicy и назначив политику пользователям. Номера телефонов, добавленные в политику, имеют приоритет над номерами телефонов, заданными с помощью платного номера конференц-связи по **умолчанию**. Если в TeamsAudioConferencingPolicy не добавлены номера телефонов, набор номеров телефонов, использующий платный номер конференц-связи по умолчанию, будет отображаться в приглашениях на собрания Microsoft Teams. |
|**Приглашения от этого пользователя могут включать бесплатный номер**|Этот параметр можно изменить только с помощью TeamsAudioconferecningPolicy. |
|**Не прошедшие проверку подлинности пользователи могут быть первым пользователем на собрании**|Чтобы изменить этот параметр, переключить пользователей, не прошедших проверку подлинности, может быть первым пользователем в собрании **" или** "Выкл.".
|**Разрешения на исходящий доступ**|Чтобы изменить этот параметр, нажмите кнопку "Изменить" рядом с аудиоконференции, а затем в области аудиоконференций выберите параметр в разделе "Исходящий доступ из **собраний"**.|
