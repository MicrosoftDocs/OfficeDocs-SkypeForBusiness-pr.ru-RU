# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

В учетной записи администратора Office 365 можно по отдельному пользователю организации вносить изменения в параметры аудиоконференций, такие как провайдер, номера платной или бесплатной линии по умолчанию, идентификатор конференции или ПИН-код. Если вы хотите изменить настройки для своей организации, см. статью [Управление настройками аудиоконференций для своей организации](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md).


1. В левой области навигации щелкните на элементе **Пользователи** и выберите пользователя в списке доступных пользователей.

2. В верхней части страницы щелкните на элементе **Изменить**.

3. В области **Аудиоконференции** измените любую из следующих настроек:

|**Настройка**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить или отключить аудиоконференции для пользователя, щелкните **Изменить** рядом с элементом **Аудиоконференции**, а затем в области **Аудиоконференции** переключите параметр **Аудиоконференции** на значение «Вкл.» или «Выкл.».|
|В разделе **Аудиоконференции щелкните Отправить информацию для конференций в электронном письме**.  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Щелкните на элементе **Сбросить идентификатор конференции**, если хотите сбросить идентификатор конференции для этого пользователя. Дополнительные сведения см. в статье [Сброс идентификатора конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**ПИН-код** |Щелкните на элементе **Сбросить ПИН-код**, если хотите сбросить ПИН-код для этого пользователя. Дополнительные сведения см. в статье [Сброс ПИН-кода для аудиоконференции](reset-the-audio-conferencing-pin-in-teams.md). |
|**Платный телефонный номер по умолчанию для конференц-связи** (обязательно) |Это будут номера, заданные для моста аудиоконференций. Задайте для номеров формат, в котором они должны отображаться в приглашениях на собрания Skype для бизнеса и Microsoft Teams. Чтобы изменить платный номер по умолчанию, щелкните **Изменить** рядом с элементом **Аудиоконференции**, а затем в области **Аудиоконференции** выберите номер в поле **Платный номер**. |
|**Приглашения от этого пользователя могут включать в себя бесплатный номер**|Чтобы изменить эту настройку, щелкните **Изменить** рядом с элементом **Аудиоконференции**, а затем в области **Аудиоконференции** установите для параметра **Включать бесплатные номера в приглашения на собрание от этого пользователя** значение «Вкл.» или «Выкл.». |
|**Разрешения на присоединение обратным звонком**|Чтобы изменить эту настройку, щелкните **Изменить** рядом с элементом **Аудиоконференции**, а затем в области **Аудиоконференции** выберите параметр в разделе **Разрешения на присоединение обратным звонком из собраний**.|
