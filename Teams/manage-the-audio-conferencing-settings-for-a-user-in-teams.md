# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

Как администратор Microsoft 365 или Office 365 вы можете изменять параметры аудиоконференции, такие как поставщик услуг, платный или бесплатный номер по умолчанию, код конференции или ПИН-код для отдельного пользователя в организации. Если вы хотите изменить параметры для своей организации, см. статью "Управление настройками аудиоконференции [для организации".](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md)

## <a name="an-icon-showing-the-microsoft-teams-logo-using-the-microsoft-teams-admin-center"></a>![Значок с логотипом Microsoft Teams](media/teams-logo-30x30.png) С помощью Центра администрирования Microsoft Teams

1. В области навигации слева **щелкните**"Пользователи" и выберите пользователя в списке доступных пользователей.

2. Нажмите **кнопку "Изменить".**

3. В **параметрах аудиоконференции** измените следующие параметры:

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить или отключить аудиоконференцию для  пользователя, щелкните "Изменить" рядом с  кнопкой "Аудиоконференция", а затем в области "Аудиоконференция" включите или выключите функцию аудиоконференции.  |
|**Отправка информации о конференции по электронной почте**  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Нажмите **кнопку "Сбросить ИД** конференции", если нужно сбросить его для пользователя. Для дополнительных сведений см. [Сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**ПИН-код** |Нажмите **кнопку "Сбросить** ПИН-код", если вам нужно сбросить ПИН-код пользователя. Для дополнительных сведений см. [Сбросить PIN-код для аудиоконференций](reset-the-audio-conferencing-pin-in-teams.md). |
|**Платный номер телефона** для телефонной связи по умолчанию (обязательно) |Это номера, которые настроены на мосте аудиоконференции. Отформатирование номеров в том виде, в каком они должны отображаться в запросах на собрания Skype для бизнеса и Microsoft Teams. Чтобы изменить платный номер по  умолчанию, щелкните "Изменить"  рядом с аудиоконференцией, а затем в области "Аудиоконференция" выберите номер в области "Платный **номер".**  |
|**Приглашения от этого пользователя могут включать бесплатный номер**|Чтобы изменить этот параметр, щелкните "Изменить" рядом  с аудиоконференцией и в  области "Аудиоконференция" включите или выключите в запросах на собрания бесплатные номера.   |
|**Неохваченные пользователи могут быть первыми пользователями на собрании**|Чтобы изменить этот параметр,  включите или выключите параметр "Неподтверченные пользователи" в числе первых участников собрания.
|**Разрешения для телефонного и телефонного ветвей**|Чтобы изменить этот  параметр,  щелкните "Изменить" рядом  с аудиоконференцией, а затем в области "Аудиоконференция" выберите параметр в диалоговом окте "Из **собрания".**|
