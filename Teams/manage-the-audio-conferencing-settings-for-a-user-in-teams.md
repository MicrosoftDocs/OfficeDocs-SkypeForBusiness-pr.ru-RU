# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

Администраторы Microsoft 365 или Office 365 могут изменять параметры аудиоконференции, такие как поставщик услуг аудиоконференции, платный или бесплатный номер по умолчанию, код конференции или ПИН-код для отдельного пользователя в организации. Если вы хотите изменить параметры для организации, см. статью Управление настройками аудиоконференции [для организации.](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md)

## <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

1. В левой области навигации щелкните **Пользователи** и выберите пользователя из списка доступных пользователей.

2. Нажмите **кнопку Изменить**.

3. В **области Аудиоконференция** измените следующие параметры:

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить или отключить аудиоконференцию для  пользователя, щелкните Изменить рядом с кнопкой Аудиоконференция, а  затем в области Аудиоконференция включите или выключите режим Аудиоконференция.  |
|**Отправка информации о конференции по электронной почте**  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Нажмите **кнопку Сбросить ИД** конференции, если нужно сбросить его для пользователя. Для дополнительных сведений см. [Сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**КОНТАКТНЫЙ** |Нажмите **кнопку Сбросить** ПИН-код, если вам нужно сбросить ПИН-код для пользователя. Для дополнительных сведений см. [Сбросить PIN-код для аудиоконференций](reset-the-audio-conferencing-pin-in-teams.md). |
|**Платный номер телефона для conferencing** по умолчанию (обязательно) |Это номера, которые установлены на мосте аудиоконференции. Отформатирование чисел в том виде, в каком они должны отображаться Skype для бизнеса Microsoft Teams собраниях. Чтобы изменить платный номер  по умолчанию, щелкните Изменить рядом  с кнопкой Аудиоконференция и в области Аудиоконференция выберите номер в области **Платный номер**.  |
|**Приглашения от этого пользователя могут включать бесплатный номер**|Чтобы изменить этот параметр,  щелкните Изменить рядом с параметром Аудиоконференция и  в области Аудиоконференция включите или выключите параметр Включить бесплатные номера в запросы на собрания от этого пользователя.   |
|**Неавтериентированные пользователи могут быть первым пользователем на собрании**|Чтобы изменить этот параметр, включите параметр **Неавтексированные** пользователи могут быть первым пользователем в собрании Вкл. или Выкл.
|**Разрешения для выбираемого номера**|Чтобы изменить этот  параметр, нажмите кнопку Изменить рядом  с параметром Аудиоконференция и в области Аудиоконференция выберите параметр в области Изменяемая телефонная установка из **собраний**. |
