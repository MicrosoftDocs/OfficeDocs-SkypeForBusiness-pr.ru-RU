# <a name="manage-the-audio-conferencing-settings-for-a-user-in-microsoft-teams"></a>Управление настройками аудиоконференций для пользователя в Microsoft Teams

В учетной записи администратора Office 365 можно по отдельному пользователю организации вносить изменения в параметры аудиоконференций, такие как провайдер, номера платной или бесплатной линии по умолчанию, идентификатор конференции или ПИН-код. Если вы хотите изменить параметры для вашей организации, ознакомьтесь со [управлять параметрами звука конференц-связи для вашей организации](manage-the-audio-conferencing-settings-for-my-organization-in-teams.md).

## <a name="teams-logo-30x30pngmediateams-logo-30x30png-using-the-microsoft-teams-admin-center"></a>![команды логотип 30x30.png](media/teams-logo-30x30.png) С помощью центра администрирования группами Майкрософт

1. На панели навигации слева щелкните **Пользователи**и затем выберите пользователя в списке Доступные пользователи.

2. Нажмите кнопку **Изменить**.

3. В разделе **Audio конференц-связи**измените любые из следующих:

|**Параметр**|**Описание**|
|:-----|:-----|
|**Аудиоконференции**|Чтобы включить аудиоконференций или отключить для пользователя, нажмите кнопку **Изменить** рядом с пунктом **Звукового конференц-связи**, а затем в области **Аудиоконференций** , включения и отключения **звука конференц-связи** включено или отключено.|
|**Отправить по электронной почте сведения о конференции**  |Перейдите по этой ссылке только в том случае, если необходимо пользователю с собственным идентификатором конференции и номером телефона сразу же отправить сообщение по электронной почте. (Это сообщение электронной почты не содержит PIN-код.) См. раздел [Отправить пользователю сообщение по электронной почте, используя его сведения для аудиоконференции](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md).  |
|**Идентификатор конференции**  |Нажмите кнопку **сбросить идентификатор конференции** , если необходимо выполнить сброс идентификатор конференции для пользователя. Для дополнительных сведений см. [Сбросить идентификатор конференции для пользователя](reset-a-conference-id-for-a-user-in-teams.md).  |
|**ПИН-КОД** |Нажмите кнопку **Сбросить ПИН-код** , если необходимо выполнить сброс ПИН-КОДОВ для пользователя. Для дополнительных сведений см. [Сбросить PIN-код для аудиоконференций](reset-the-audio-conferencing-pin-in-teams.md). |
|**По умолчанию номер телефона международную конференц-связи** (обязательно) |Это будут номера, заданные для звукового конференц-канала. Форматирование чисел, как они отображаются в Скайп для бизнеса и группами Майкрософт приглашения на собрания. Чтобы изменить номер счета по умолчанию, нажмите кнопку **Изменить** рядом с пунктом **Аудиоконференции** и в области **Звука конференц-связи** , выберите номер в поле **номер счета**. |
|**Приглашения от этого пользователя может включать телефоны**|Чтобы изменить этот параметр, нажмите кнопку **Изменить** рядом с пунктом **Аудиоконференции** и в области **Звука конференц-связи** , включения и отключения **включают бесплатные номера в приглашении на собрание из этого пользователя** включено или отключено. |
|**Не прошедшим проверку подлинности пользователи могут быть первого участника собрания**|Чтобы изменить этот параметр, включения и отключения **не прошедшим проверку подлинности пользователи могут быть первого участника собрания** включено или отключено.
|**Разрешения по телефону**|Чтобы изменить этот параметр, нажмите кнопку **Изменить** рядом с пунктом **Аудиоконференции** и в области **Звука конференц-связи** , выберите параметр в группе **Подключение по телефонной линии из собрания**.|
