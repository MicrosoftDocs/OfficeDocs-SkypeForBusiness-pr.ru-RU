# <a name="whats-the-status-of-my-port-orders-in-teams"></a>Что такое состояние Мои заказы порта в группах

Можно просмотреть состояние порта заказа, перейдя на **Центр администрирования группами Майкрософт** > **портала прежних версий** > **голосовой связи** > **заказов на порт**. В столбце **состояние** отображается состояние каждого порта порядке. Если вам требуется помощь, [обращение в службу поддержки продуктов бизнес - Admin справки](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).

В следующей таблице приведены состояния порта порядке, а также действия, которые нужно выполнить при необходимости.

|:-----|:-----|:-----|:-----|:-----|:-----|
|**Обработка** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Администратор создает порядок и принимается корпорацией Майкрософт.  <br/> |
|**Общаться с поставщика** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Порядок получения и утвержденных корпорацией Майкрософт и мы работают с ним проигравшей его утвержденных.  <br/> |
|**Передача утверждено** <br/> |Да  <br/> |Утверждение заказа Commitment(FOC)  <br/> |Да  <br/> |Да  <br/> |Порядок принято проигравшей передающую и Дата FOC имеет значение.  <br/> |
|**Ожидание передачи** <br/> |Да  <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Перенос является менее 24 часов, больше не порядок можно изменить или отменена.  <br/> |
