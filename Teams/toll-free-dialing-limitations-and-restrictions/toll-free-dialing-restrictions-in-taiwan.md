# <a name="toll-free-dialing-restrictions-in-taiwan"></a>Ограничения набора бесплатных номеров на Тайване

[Найдите другую страну или регион](../toll-free-dialing-limitations-and-restrictions.md)или узнайте больше об ограничениях и ограничениях для бесплатного набора номера.


|**Сведения**|**Описание**|
|:-----|:-----|
|Доступны ли бесплатные вызовы?  <br/> |Да  <br/> |
|Формат набора  <br/> |00 801 XXX XXX  <br/> |
|Код страны или региона по ISO  <br/> |TW  <br/> |
|Охватываемая территория  <br/> |Неприменимо  <br/> |
|Неохватываемая территория  <br/> |Неприменимо  <br/> |
|Доступность проводных и наземных линий связи  <br/> | ITFS: KG Telecom = НЕТ <br/> |
|Доступность мобильных сетей  <br/> |Частично. См. примечания.  <br/> |
|Примечания, ограничения и проблемы с доступом  <br/> |Проводной доступ недоступен для абонентов KG Telecom. Мобильный доступ: без повременной оплаты, для абонентов всех мобильных сетей кроме KG Telecom.  <br/> |
