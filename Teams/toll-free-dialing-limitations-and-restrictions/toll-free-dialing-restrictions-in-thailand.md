|**Details**|**Описание**|
|:-----|:-----|
|Доступны ли бесплатные вызовы?  <br/> |Да  <br/> |
|Формат набора  <br/> | 001 800 XXX XXX XXXX <br/>  1-800-XXX-XXX <br/> |
|Код страны или региона по ISO  <br/> |TH  <br/> |

|Доступность мобильных сетей  <br/> |Да. См. примечания.  <br/> |
