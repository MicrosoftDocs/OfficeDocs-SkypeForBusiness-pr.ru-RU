|**Сведения**|**Описание**|
|:-----|:-----|
|Доступен ли бесплатный звонок?  <br/> |Да  <br/> |
|Формат набора  <br/> |0004 019 XXXX  <br/> |
|Код страны или региона по ISO  <br/> |UY  <br/> |
|Охватываемая территория  <br/> |Неприменимо  <br/> |
|Неохватываемая территория  <br/> |Неприменимо  <br/> |

|Доступность мобильных сетей  <br/> |Частично. См. примечания.  <br/> |
|Примечания, ограничения и проблемы с доступом  <br/> |Мобильный доступ: Повременная оплата. Мобильный доступ: Доступно для абонентов Ancel и Movistar.  <br/> |
