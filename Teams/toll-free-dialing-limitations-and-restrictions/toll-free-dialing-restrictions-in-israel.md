|**Сведения**|**Описание**|
|:-----|:-----|
|Доступны ли бесплатные вызовы?  <br/> |Да  <br/> |
|Формат набора  <br/> | 1-80-94X-XXXX <br/>  1-80-92X-XXXX <br/> |
|Код страны или региона по ISO  <br/> |IL  <br/> |

|Доступность мобильных сетей  <br/> | Частично. См. примечания. <br/> |
