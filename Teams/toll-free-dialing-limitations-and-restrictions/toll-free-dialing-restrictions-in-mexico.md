# <a name="toll-free-dialing-restrictions-in-mexico"></a>Ограничения набора бесплатных номеров в Мексике

[Найдите другую страну или регион](../toll-free-dialing-limitations-and-restrictions.md)или узнайте больше об ограничениях и ограничениях для бесплатного набора номера.

|**Сведения**|**Описание**|
|:-----|:-----|
|Доступны ли бесплатные вызовы?  <br/> |Да  <br/> |
|Формат набора  <br/> | 844<br/>833<br/>855<br/>800<br/>866<br/>877<br/>888<br/>|
|Код страны или региона по ISO  <br/> |MX  <br/> |
|Охватываемая территория  <br/> |"Американский Самоа", "Ангиль", "Антигуа" и "Барбуда", Багамы, Фармадо, Квебек, Виргинские о-ва, Канада, Каймановы о-ва, Гала, Объединенная Республика, Гренада, Гуам, Квебек, Монсеррат, Северные Марианские острова, Пуэрто-Рико, Сент-Китс и Невис, Сент-Рика, Сент-о-ва и Гренадские острова, Сен-Мартен, Тринидад и Тобаго, О-ва Теркс и Кайкос, США, Виргинские острова <br/> |
|Доступность проводных и наземных линий связи  <br/> |Да  <br/> |
|Доступность мобильных сетей  <br/> |Да  <br/> |
|Примечания, ограничения и проблемы с доступом  <br/> |Нет  <br/> |

> [!NOTE]
> **Форматы набора:**
>
> - 800–389 являются специальными и для багам
> - 800–534 являются специальными и для Барбадо
> - 800-623 являются специальными и для нее
> - 800-415, 907 или 751 для Dom Republic
> - 800-271 являются специальными и для Тринидада
> - 800–555 — это диапазон строк с нарушением слуха

> [!IMPORTANT]
> **Бесплатные номера по** плану в Северной Америке: тарифы за минуту для бесплатных номеров по плану в Северной Америке определяются в зависимости от страны, в какой стране вы находитесь. Бесплатный поминутный тариф для звонков из США определяется как "Северная Америка". Однако для звонков из других стран Северной Америки, таких как Канада, Пуэрто-Рико и т. д., ставку платы за бесплатные звонки звонков 3655 звонков 2016 года.
