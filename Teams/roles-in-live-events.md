<a name="presenter-producer-and-attendee-capabilities-in-a-teams-live-event"></a>Возможности выступающего, организатора и участника в трансляции Teams
======================================================

Microsoft Teams поддерживает потоковые трансляции для больших аудиторий в Интернете. У участника пассивная роль, а у выступающего и организатора трансляции есть доступ к некоторым функциям собрания, к которым относятся:  

- Чат (включая фотографии и наклейки)
- Запись
- Файлы
- Вопросы и ответы

В этой статье описаны возможности, доступные в трансляции Teams.

## <a name="presenter-and-producer"></a>Выступающий и организатор

К выступающему и организатору относятся:

- Выступающие из моей организации
- Выступающие из других организаций. Выступающие назначаются организатором, и для них требуется личное приглашение от организатора.

У выступающих и организаторов есть доступ ко всем функциям трансляции.

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **Функция**       | До трансляции | Во время трансляции | После трансляции |
| Чат | Да | Да| Да |
| Запись | Недоступно |Недоступно | Да |
| Файлы | Да |Да | Да |
| Вопросы и ответы, если включено организатором| Да | Да | Да |
|||||||

## <a name="live-event-attendee"></a>Участник трансляции

Участник трансляции — это человек, у которого есть ссылка на трансляцию или на ее запись. Для участия в трансляции или ее просмотра не требуются специальные разрешения. Участник, просматривающий трансляцию в реальном времени, может участвовать в вопросах и ответах, если это разрешено выступающим и/или организатором. 

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **Функция**       | До трансляции | Во время трансляции | После трансляции |
| Вопросы и ответы, если включено организатором | Да | Да | Да |
| Запись | Недоступно | Недоступно| Да |
| Элементы управления воспроизведением видео, если включены организатором | Да | Да | Да |
|||||||
