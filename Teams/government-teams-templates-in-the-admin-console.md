# <a name="use-government-team-templates-in-the-admin-center"></a>Использование шаблонов команд для государственных организаций в Центре администрирования

Шаблоны команд позволяют быстро и легко создавать команды, предоставляя стандартный шаблон параметров, каналов и предварительно установленных приложений.

Шаблоны команд имеют предварительно созданные определения структур групп, предназначенных для государственных организаций. Вы также можете расширить шаблоны команд, чтобы создавать команды, адаптированные для конкретных потребностей организации.

В этой статье мы рассмотрим каждый из шаблонов команды и рекомендуем использовать их.

Эта статья предназначена для вас, если вы отвечаете за планирование, развертывание и управление несколькими командами в правительственных организациях. Вы уже развернули службу Teams в своей организации. Если вы еще не развернули Teams, см. статью [Как развернуть Microsoft Teams](./deploy-overview.md).

Дополнительные сведения о шаблонах команд см. в статье ["Начало работы с шаблонами групп"](get-started-with-teams-templates-in-the-admin-console.md).

## <a name="coordinate-incident-response"></a>Координация реагирования на инциденты

Централизация обмена данными и критически важных ресурсов для команды по управлению критическими инцидентами или реагированию на инциденты. В этой команде можно включить множество различных типов файлов, чтобы создать центральное место для всех документов. Используйте собрания по сети для улучшения потока информации и информирования о ситуации.

| Тип базового шаблона |baseTemplateId | Свойства базового шаблона |
|-------------------|-------|---------------------------------------------------------------------------|
|Координация реагирования на инциденты|`com.microsoft.teams.template.CoordinateIncidentResponse`|Каналы: <ul><li>Общий<li>Объявления</li><li>Логистика</li><li>Планирование</li><li>Восстановления</li><li>Срочно</li></ul> Приложения: <ul><li>Вики</li><li>Excel</li><li>OneNote</li><li>SharePoint</li><li>Планировщик</li></ul>|
