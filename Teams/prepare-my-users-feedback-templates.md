# <a name="user-feedback-surveys"></a>Опросы отзывов пользователей 

Используйте опросы свои отзывы и предложения для записи взаимодействия и обмена мнениями пользователей после развертывания. Вопросы должны выявление свой отзыв о receptiveness пользователей для изменения, эффективность учебные ресурсы готовности был плана поддержки и общую работу пользователей. Оценка ответов на опрос для измерения с смарт-целями (определенных, измерить, недостижима, соответствующих, привязанных к времени), заданных во время планирования проекта и определить, какие-либо действия требуется.

Настройка ниже вопросов и перенести их средство например SharePoint для удобства распространения и анализа.
