# <a name="upgrade-journey-and-coexistence-of-skype-for-business-and-microsoft-teams"></a>Обновление пути и сосуществование Скайп для бизнеса и группами Майкрософт

Как существующие Скайп для клиента Business завершения перехода к группам может занять некоторое время. Тем не менее можно начать в настоящее время осознавать значение группам, что позволяет пользователям использовать команды наряду с Скайп для бизнеса. Что есть некоторые функциональные возможности перекрывающиеся между двумя приложениями, рекомендуем просмотреть доступные режимы обновления можно определить, какой путь для своей организации. К примеру может выбрать для включения всех рабочих нагрузок на обоих решения без взаимодействия. Или можно управлять удобство работы пользователей с постепенно краткие сведения о возможности групп или ориентация групп пользователей для выбора возможностей, пока не будет готово к обновлению все команды вашей организации.

Как и в любом развертывании мы настоятельно рекомендуем вам для [Пилотный проект плана предполагаемая](pilot-essentials.md) с выбранной группы пользователей перед внедрением в работе групп для более широкой организации. Помните, что краткие сведения о новых технологий может быть аварийные для пользователей. Потратьте время оценить готовность пользователей и реализовывать обмена данными и план обучения для хранения и тенденциями пользователей и ускорить их принять группами. 

> [!IMPORTANT]
> Скайп для бизнеса группам обновления реализация быстрого развертывания. Эта статья поможет вам понять предстоящие команды обновления, но оно не содержит инструкции по обновлению начала до конца. Успешное обновление с Скайп для бизнеса будет включать действия зависящие от проверки готовности к техническим и пользователя. Мы работаем на данное руководство теперь и мы будем его публикации ближайшее время.
 
## <a name="upgrade-and-coexistence-modes-defined"></a>Обновление и сосуществование режимов определенных
Для помощи в процесс принятия решений, ознакомьтесь с различные режимы, принципы и терминологию, предназначенных для обновления Скайп для бизнеса до группами.

### <a name="skype-for-business-with-teams-collaborationndashonly-mode"></a>Скайп для бизнеса с совместной работы групп&ndash;только в режиме

В этом режиме группами настроена поддержка рабочих групп и на основе каналы бесед, только с помощью закрытого бесед, звонков и собраний этот параметр отключен.

Этот режим — это отличный способ, чтобы познакомить группами в вашей среде продолжить использование существующих инвестиций в Скайп для бизнеса. 

### <a name="skype-for-business-with-teams-collaboration-and-meetings-mode"></a>Скайп для бизнеса с режимом собраний и совместной работы групп
Наряду с помощью команды для групп и на основе каналы бесед в этом режиме, пользователям начать работу с группами для планирования и проводить их. Закрытый чаты и голосовую и видеосвязь, остаются на Скайп для бизнеса.

Возможности, предоставляемые группами для собраний необходимых для организации, но business должно необходимо хранить возможности голосовой связи предприятия в Скайп для бизнеса, можно настроить группы для предоставления Скайп для бизнеса с собраний и совместной работы групп возможности по всей организации (или некоторые части).

### <a name="islands-mode"></a>Режим острова
Это режим по умолчанию. Использовать его для развертывания группами, независимо от Скайп для бизнеса. 

В этом режиме групп пользователи могут сразу же использовать команды с его возможности и других пользователей, групп, то время как другие пользователи могут продолжить использовать Скайп для бизнеса и работать без воздействия на как они используют Скайп для бизнеса сегодня.

> [!NOTE]
> Режим острова поставляется с односторонней временные взаимодействия, обеспечивающий взаимодействие чата из групп Скайп для бизнеса, при выполнении следующих условий:
> - Группы пользователей, размещенных на и включен для Скайп для бизнеса в Интернет.
> - Скайп для корпоративных пользователей имеет никогда не использовали группы (или не будет с корпоративным лицензированием для групп).
>
> В этом режиме функция полезна, если требуется, пользователи, которые в первую очередь с помощью команды другим пользователям в организации, которые будут по-прежнему только с помощью Скайп для бизнеса.
>
> После пользователей, которые ранее используется только Скайп для бизнеса начало работы с группами, их чаты от других групп пользователей будут поступать в группах. Эти пользователи переключает в режим острова. С этого момента они должны работу группы, чтобы убедиться, что они оставаться на связи с другими группами пользователей в организации.

### <a name="teams-only-mode"></a>Режим только для групп
В этом режиме обновленных Скайп для клиента Business больше не предоставляет базовой обмена мгновенными Сообщениями, сведения о присутствии индикаторы, чата, вызова или возможности проведения собраний. Пользователи, работающие в режиме обновленных необходимо использовать как их основным средством коммуникаций и совместной работы групп.

Пользователи, все готово для использования в качестве их основным средством коммуникаций и совместной работы групп может быть обновлена только для группы пользователей.

Чтобы присоединиться к существующей Скайп для деловых встреч и собраний на Скайп для бизнеса, упорядоченные по необновленной пользователями или внешними участниками обновленных пользовательских можно использовать только Скайп для клиента Business. Можно продолжить обновленных пользовательских общаться с другими пользователями в организации, которые по-прежнему используется Скайп для бизнеса с помощью возможности взаимодействия между рабочими группами и Скайп для бизнеса.

### <a name="skype-for-businessndashonly-mode"></a>Скайп для бизнеса&ndash;только в режиме
Пользователей, которые должны быть в курсе в Скайп для бизнеса может быть настроен как Скайп для бизнеса&ndash;только пользователей. Пользователей, заданных в этом режиме не будут с помощью команды и будет взаимодействовать с только для группы пользователей с помощью возможностей взаимодействия между рабочими группами и Скайп для бизнеса.

## <a name="upgrade-and-coexistence-building-blocks"></a>Обновление и сосуществование стандартные блоки
Чтобы официально подготовить пути к группам в вашей организации, необходимо начать планирование сценарии обновления, которые со временем позволит вашей организации полностью принять группами как единственным коммуникаций и совместной работы решения.

Если некоторые пользователи все готово для использования только группы их повседневного общения и совместной работы должен, можно начать обновление этих пользователей по группам, включив режим только для групп для них.

Если это не возможно в организации для перемещения по группам, можно начать можно полностью принять группами как решение совместной работы группы сначала сохраняя Скайп для бизнеса в качестве решения объединенных коммуникаций вашей организации. Другая возможность — это переместить собрания групп, в дополнение к краткие сведения о возможности совместной работы группы команд, сохраняя остальные возможности объединенных коммуникаций в Скайп для бизнеса.

## <a name="upgrade-journeys"></a>Обновление поездок
Вы можете воспользоваться различные подходы к обновлению Скайп для бизнеса, либо через Интернет или локальной, к группам:
- В простой пути обновления, сначала развернуть групп в Скайп для бизнеса с совместной работы групп&ndash;только в процессе оценки и раннее внедрение и режим затем реализации только для групп с целью со временем закончится Скайп для бизнеса из среда для всех пользователей в организации.
- Постепенного обновления реализация обеспечивает определенный режим обновления к определенной группе пользователей (также называемая *cohort*), в зависимости от их коммуникаций и требованиям к совместной работе. Со временем во всей организации могут для достижения в только с помощью команды и в конечном счете замените Скайп для бизнеса. Тем не менее если в вашей организации есть убедительных причин business хранения Скайп для бизнеса, такие как зависимость от решения на основе Unified Communications Managed API UCMA, интегрированного с бизнес приложения или решения на базе корпоративными стандартами в настоящее время недоступен для Скайп для бизнеса только — большинство пользователей можно выполнить обновление до режима только для групп сохраняя Скайп для бизнес-пользователей в одном из Скайп для режимов Business части остальных пользователей.

> [!IMPORTANT]
> Для обоих типов обновления реализация вашей организации в настоящий момент Скайп Business локальной только для развертывания, требуется ли начала планирования для реализации Скайп для бизнеса гибридной среды перед обновлением до режима только для группы пользователей. Это также поможет улучшения взаимодействия с группами.
> Используйте [MyAdvisor](https://myadvisor.fasttrack.microsoft.com/) для вашей Скайп для реализации гибридных бизнеса.

> [!NOTE]
> Требуется только для групп, что пользователи, которые входят в состав последователей быть размещен в Скайп для бизнеса в Интернет и гибридного отношение между вашей Скайп для локального развертывания Business и вашей Скайп для бизнеса интерактивного клиента является обязательным для упрощения обновления и взаимодействие между Скайп для бизнеса и рабочих групп. Должен быть выполнен переход на Скайп для бизнеса в Интернет для пользователей, которые входят в состав последователей до обновления в режиме только для групп. Скайп для Business Server 2019, а также будущих накопительный пакет обновлений за Скайп для Business Server 2015, планирование для упрощения механизм обновление локальных пользователей по группам, управление миграции для Скайп для бизнеса в Интернет и обновление до только для групп пользователей режим, в один шаг.

Независимо от выбранного реализация выбран, корпорация Майкрософт считает здесь поддержку процесса обновления. Из технические требования для ресурсы по подготовке пользователей мы будем совместно использовать инструкции и рекомендации для обеспечения успешного перехода от Скайп для бизнеса в группы. Подробное руководство по обновлению будет вскоре будут готовы к работе, поэтому на наличие Дополнительные сведения.

### <a name="simple-upgrade-journey"></a>Простая реализация обновления
На следующей схеме показано простая реализация обновления.

![Снимок экрана простого пути обновления. Все пользователи изначально используется группами в Скайп для бизнеса с совместной работы групп&ndash;только режим, а затем переход в режим только для групп, с состоянием конечных во всей организации, обновленных до группами.](media/upgrade_and_coexistence_simple_upgrade_journey.png)

Группы развернуты для всех пользователей в организации и настроен в Скайп для бизнеса с совместной работы групп&ndash;только в режиме. Когда ваша организация определяет, что команды готова для выполнения всех коммуникаций и потребностями в совместной работе, все пользователи обновляются в режим только для групп. На этом этапе можно не поддерживается Скайп для бизнеса из среды.

### <a name="gradual-upgrade-journey"></a>Реализация постепенного обновления
На следующей схеме показан пример постепенного обновления пути.

![Постепенное обновление результат последователей пользователей изначально используется группами в различных обновления режима, рядом с Скайп для бизнеса. Некоторые последователей переход в режим только для групп, а по одной группе пользователей остается Скайп для бизнеса с помощью режима собраний и совместной работы групп.](media/upgrade_and_coexistence_gradual_upgrade_journey.png)

Команды будет развернут в организации с помощью различных режимах обновления для разных групп пользователей или последователей. Например, группы пользователей, которые могут быть включены для режима острова, другой включена поддержка Скайп для бизнеса совместной работы групп и режиме собраний, а третья группа пользователей изначально может быть включена поддержка Скайп для бизнеса с совместной работы групп&ndash;только режим.

Со временем в режим только для групп, а затем остальной части организации может быть обновлена групп пользователей. В конечном итоге во всей организации будет готов к удаление Скайп для бизнеса и использовать только группы коммуникаций и совместной работы, или, если бизнес-требований диктуют необходимость сохранения Скайп для бизнеса определенной группе — большинство пользователей в Организация может использовать только группы. <br><br>
