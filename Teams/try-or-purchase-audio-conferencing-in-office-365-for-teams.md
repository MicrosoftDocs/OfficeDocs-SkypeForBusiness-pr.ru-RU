# <a name="try-or-purchase-audio-conferencing-in-microsoft-365-for-microsoft-teams"></a>Попробуйте или приобретите аудиоконференции в Microsoft 365 для Microsoft Teams

В некоторых случаях пользователям в организации требуется телефон, чтобы присоединиться к собранию. Microsoft Teams включает функцию аудиоконференций только для этой ситуации! Пользователи могут выполнять звонки на собрания Microsoft Teams с помощью телефона, а не с помощью приложения Microsoft Teams на мобильном устройстве или компьютере.

Вам нужно настроить аудиоконференции только для тех, кто планирует планировать или вести собрания. Участникам, которые подключаются к собранию, не нужны ни специально назначенные им лицензии, ни другие настройки.

Информация о ценах приведена в разделе [Цены на аудиоконференции](https://www.microsoft.com/microsoft-teams/audio-conferencing?rtc=3).

## <a name="step-1-buy-and-assign-audio-conferencing-licenses"></a>Шаг 1. Приобретение и назначение лицензий на аудиоконференции

Для выполнения [этих действий](https://support.office.com/article/da585eea-f576-4f55-a1e0-87090b6aaa9d) необходимо быть глобальным администратором или администратором выставления счетов.

### <a name="to-buy-and-assign-user-audio-conferencing-licenses"></a>Приобретение и назначение пользовательских лицензий на аудиоконференции

1. Узнайте, **доступна ли аудиоконференции** в вашей стране или регионе. [Доступность стран и регионов для аудиоконференций и планов звонков](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).

2. Получите свою лицензию на **Аудиконференции**. Если Вы хотите:

   - **Попробуйте** сделать это перед покупкой: вы можете зарегистрироваться для использования бесплатной пробной Office 365 корпоративный E5, которая включает аудиоконференции. См. раздел [Пробная версия Office 365 корпоративный E5](https://portal.office.com/Signup?OfferId=101bde18-5ffb-4d79-a47b-f5b2c62525b3).

   - **Купить его**: см. [раздел "Лицензирование надстроек Microsoft Teams"](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md).

3. [Назначьте лицензии пользователям](/microsoft-365/admin/manage/assign-licenses-to-users) в организации, которые будут планировать или вести собрания.

4. Дополнительно назначить приобретенные Вами лицензии на надстройку аудиоконференции и кредиты на связь. Инструкции см. в [разделе "Назначение лицензий на надстройки Microsoft Teams"](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md).

### <a name="to-buy-and-assign-pay-per-minute-audio-conferencing-licenses"></a>Приобретение и назначение лицензий на аудиоконференции с оплатой поминутной оплаты

Клиенты Volume and Licensing могут получить лицензии на аудиоконференции с поминутной оплатой. Дополнительные сведения о лицензии на аудиоконференции с поминутной оплатой доступны в разделе [Аудиоконференции с поминутной оплатой](audio-conferencing-pay-per-minute.md).
  
1. Узнайте, **доступна ли аудиоконференции** в вашей стране или регионе. [Доступность аудиоконференций](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) и планов звонков по странам и регионам.

2. Получите свои лицензии на **Аудиоконференции**. Для приобретения лицензий с поминутной оплатой свяжитесь с консультантом по работе с клиентами.

3. [Настройка кредитов на связь для вашей организации](set-up-communications-credits-for-your-organization.md) . Для настройки функции предоставления кредитов на связь см. [Что такое кредиты на связь?](what-are-communications-credits.md)

    > [!IMPORTANT]
    > Если кредиты на связь не настроены, для любых пользователей с лицензиями с поминутной оплатой функция аудиоконференции работать не будет.

4. [Назначьте лицензии пользователям](/microsoft-365/admin/manage/assign-licenses-to-users) в организации, которые будут планировать или вести собрания.

    > [!NOTE]
    > Если у вас есть лицензии на аудиоконференции с оплатой поминутной оплаты, необходимо назначить лицензию на кредиты на связь отдельно для каждого пользователя.

## <a name="step-2-set-the-audio-conferencing-provider-for-people-who-lead-or-schedule-meetings"></a>Шаг 2: задать поставщика аудиоконференций для людей, проводящих или планирующих собрания

Когда вы назначаете  лицензию на аудиоконференции пользователям в организации, которые не интегрируются Skype для бизнеса со сторонним поставщиком аудиоконференций, все они настроены и готовы к работе! (Нет необходимости в настройке их поставщика аудиоконференций.)

Если у вас есть пользователи, включенные с помощью стороннего поставщика аудиоконференций, необходимо изменить поставщика этих пользователей на Microsoft. Сведения о том, как изменить поставщика для пользователя, см. в статье "Назначение Майкрософт в [качестве поставщика аудиоконференций"](/skypeforbusiness/audio-conferencing-in-office-365/assign-microsoft-as-the-audio-conferencing-provider).

## <a name="step-3-other-admin-tasks"></a>Шаг 3: другие задачи администратора

Следующие шаги являются **необязательными**, однако многие администраторы считают их необходимыми:

1. [Настройка приглашений на собрания](/skypeforbusiness/set-up-skype-for-business-online/customize-meeting-invitations). Номера телефонного подключения, задаваемые для пользователя, будут автоматически добавлены в приглашения на собрание, отправляемые участникам. Однако вы можете добавить собственные справочные и юридические ссылки, текстовое сообщение и небольшую графику компании.

2. [Задайте номера телефонов, включенные в приглашения](set-the-phone-numbers-included-on-invites-in-teams.md). Это —номер телефона, который будет отображаться в собрании, которое запланировал пользователь.

3. [Задайте](set-auto-attendant-languages-for-audio-conferencing-in-teams.md) языки автосекретаря для аудиоконференций, которые автосекретарь аудиоконференций использует для приветствие вызывающего абонента при подключении к номеру телефона аудиоконференции. Этот шаг необходим только в том случае, если Microsoft назначен в качестве поставщика аудиоконференции.

4. [Задайте длину ПИН-кода для собраний аудиоконференций](set-the-pin-length-for-audio-conferencing-meetings-in-teams.md).

> [!NOTE]
> Эта функция еще не доступна пользователям Office 365 под управлением компании 21Vianet в Китае. Дополнительные сведения см. в разделе [Узнать больше о версии Office 365 под управлением компании 21Vianett](https://support.office.com/article/A8AB5061-3346-4DA0-BB7C-5260822B53AE).
