# <a name="use-power-bi-to-analyze-cqd-data-for-microsoft-teams"></a>Используйте Power BI для анализа данных CQD для Microsoft Teams

Новые возможности в январе 2020 г.: [скачивание Power BI шаблонов запросов для CQD](https://www.microsoft.com/download/details.aspx?id=102291). Настраиваемые Power BI шаблоны, которые можно использовать для анализа данных CQD и создания отчетов о них.

Для отчетов панели мониторинга качества звонков (CQD) в Teams, если вы хотите использовать Power BI для запроса и отправки данных, скачайте шаблоны CQD Power BI. При открытии шаблонов в Power BI вам будет предложено войти с помощью учетных данных администратора CQD. Вы можете настроить эти шаблоны запросов и распространить их всем пользователям в организации, у которых есть Power BI лицензии и разрешения администратора CQD.

Прежде чем использовать эти PBIT-файлы, необходимо установить [соединитель Power BI для Microsoft CQD](CQD-Power-BI-connector.md) с помощью файла *MicrosoftCallQuality.pqx*, включенного в скачивание [.](https://www.microsoft.com/download/details.aspx?id=102291) 

Убедитесь, что у вас есть права [на доступ к CQD](turning-on-and-using-call-quality-dashboard.md#assign-admin-roles-for-access-to-cqd) для доступа к Power BI отчетов.
