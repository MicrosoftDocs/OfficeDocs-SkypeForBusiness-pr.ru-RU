# <a name="use-power-bi-to-analyze-cqd-data-for-microsoft-teams"></a>Анализ данных CQD для Microsoft Teams с помощью Power BI

Новые возможности в июле 2022 г. Новый шаблон качества взаимодействия (QER) добавлен в шаблоны запросов Power BI для загрузки [CQD](https://www.microsoft.com/download/details.aspx?id=102291). QER — это эффективный шаблон отчетов, который заменяет исходные шаблоны запросов Power BI CQD, выпущенные в 2020 году. Хотя исходные шаблоны останутся доступными для демонстрационных целей, они больше не поддерживаются, и мы рекомендуем клиентам переключиться на шаблон QER, который будет по-прежнему получать обновления. Обратите внимание, что это не применимо к отчету об истории автосекретаря CQD Teams & очереди вызовов.

Для отчетов панели мониторинга качества звонков (CQD) в Teams, если вы хотите использовать Power BI для запроса и отправки данных, скачайте шаблоны Power BI CQD. При открытии шаблонов в Power BI вам будет предложено войти с помощью учетных данных администратора CQD. Вы можете настроить эти шаблоны запросов и распространить их всем пользователям в организации, у которых есть лицензия Power BI и разрешения администратора CQD.

Прежде чем использовать эти PBIT-файлы, необходимо установить [соединитель Power BI для Microsoft CQD](CQD-Power-BI-connector.md) с помощью файла *MicrosoftCallQuality.pqx*, включенного в скачивание [.](https://www.microsoft.com/download/details.aspx?id=102291) 

Убедитесь, что у вас есть права [доступа к CQD](turning-on-and-using-call-quality-dashboard.md#assign-admin-roles-for-access-to-cqd) для доступа к отчетам Power BI.
