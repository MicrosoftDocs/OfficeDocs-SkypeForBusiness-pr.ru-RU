<a name="add-bots-for-personal-chats-group-chats-and-channels-in-microsoft-teams"></a>Добавление ботов для личных чатов, групповых чатов и каналов в Microsoft Teams
==========================================================
> [!IMPORTANT]
> [!INCLUDE [new-teams-sfb-admin-center-notice](includes/new-teams-sfb-admin-center-notice.md)]

Боты — это автоматические программы, которые отвечают на запросы или предоставляют интересующие пользователя уведомления и данные. Они позволяют пользователю взаимодействовать с облачными службами, предназначенными, например, для управления задачами, планирования и опросов, через беседы чата в Microsoft Teams. Боты для Teams создаются на платформе [Microsoft Bot Framework](https://go.microsoft.com/fwlink/?linkid=854370). Поэтому для Teams можно легко адаптировать любые боты, разработанные на этой платформе. Дополнительные сведения см. в статье [Управление параметрами Microsoft Teams в организации](enable-features-office-365.md).

В настоящее время в Teams осуществляется поддержка ботов в личных чатах, групповых чатах и каналах в группе. Администраторы могут разрешить или запретить боты в клиенте Office 365.<span id="_T-Bot" class="anchor"></span>

Teams позволяет использовать боты, разработанные сообществом. Для обеспечения работы настраиваемых ботов необходимо включить функциональность ботов и возможность отправки пользовательских приложений (также называемая "загрузка неопубликованных приложений") на уровне клиента. Боты могут использоваться для личных чатов, групповых чатов и каналов. В последних боты могут добавлять участники или владельцы команды.

Дополнительные сведения см. в статье [Приложения и службы](https://support.office.com/article/Apps-and-services-cc1fba57-9900-4634-8306-2360a40c665b).

> [!IMPORTANT]
> Добавление бота по GUID для каких-либо иных целей, кроме целей тестирования, не рекомендуется. Это значительно ограничит функциональность бота. Боты в рабочей среде следует добавлять в Teams в рамках приложения. См. статьи [Создание бота](https://docs.microsoft.com/microsoftteams/platform/concepts/bots/bots-create) и [Тестирование и отладка бота Microsoft Teams](https://docs.microsoft.com/microsoftteams/platform/concepts/bots/bots-test)

<a name="create-custom-bots-for-microsoft-teams"></a>Создание пользовательских ботов для Microsoft Teams
--------------------------------------

С помощью Microsoft Bot Framework вы можете легко создать бот, интегрирующийся с вашими бизнес-приложениями. Сведения о разработке и публикации своих ботов см. в руководстве [Создание и тестирование бота для Microsoft Teams](https://go.microsoft.com/fwlink/?linkid=854371).

Создавая и регистрируя бот в Bot Framework, вы также можете опубликовать его. Без публикации бот остается частным. Вы также можете потребовать, чтобы перед работой с ботом пользователи входили в систему. Такой обязательный вход позволяет предоставить доступ к боту только сотрудникам вашей организации, даже если станет известен его идентификатор приложения. В описании [*AuthBot*](https://go.microsoft.com/fwlink/?linkid=854372) на сайте GitHub приведен пример кода для проверки подлинности пользователей в Active Directory с помощью ботов.

Перед развертыванием в Teams боты можно тестировать с помощью [Bot Framework Emulator](https://go.microsoft.com/fwlink/?linkid=854373).

<a name="upload-your-bot-for-personal-chat"></a>Отправка бота для личного чата
---------------------------------------

1. После создания вашего бота, перейдите в раздел **Параметры приложения** для разработанного вами бота, затем в разделе **Параметры приложения** скопируйте значение параметра **MicrosoftAppId**.![Снимок экрана страницы Параметры приложения для бота](media/Add_bots_for_private_chats_and_channels_in_Microsoft_Teams_image5.png)

2.  В области **Чат** в Teams выберите **значок добавления чата**. В поле **Кому** вставьте **идентификатор приложения Майкрософт** своего бота. ![Снимок экрана области Чат с выделенным идентификатором приложения Майкрософт](media/Add_bots_for_private_chats_and_channels_in_Microsoft_Teams_image6.png)

3. Идентификатор разрешается в **имя бота**, после чего вы можете запустить беседу с этим ботом.

<a name="upload-your-bot-for-group-chats-or-channels"></a>Добавление вашего бота для групповых чатов или каналов
-----------------------------------

Если вы хотите поделиться вашим ботом с коллегами, ниже описано, как добавить бот в групповые чаты или каналы различных групп:

1. После [создания пакета приложения для вашего бота](https://docs.microsoft.com/microsoftteams/platform/concepts/apps/apps-upload), откройте Teams и перейдите в груупу, в которую вы хотите отправить бот.
2. Добавьте приложение **[App Studio](https://docs.microsoft.com/microsoftteams/platform/get-started/get-started-app-studio)** в Teams.
3. В App Studio откройте вкладку **Редактор манифеста** ![Снимок экрана вкладки Редактор манифеста.](media/Adding_Bot_To_Teams.png)
4. Чтобы добавить ваш бот, в разделе Возможности выберите бот и щелкните, чтобы добавить существующий бот. Затем выберите существующий бот или введите идентификатор существующего бота.
![Отображает выбор уже созданного вами бота.](media/Select_Existing_Bot.png)
5. Перейдите к расположению вашего пакета приложения, выберите его и нажмите **Открыть**.
6. Выберите имя вашего бота. (Не забудьте установить флажок **Групповой чат** или **Группа** в разделе Область).
7. Нажмите кнопку **Тестирование и распространение**.
8. Выберите групповой чат или группу, к которой вы хотите подключить ваш бот.

    Ваш бот будет доступен в вашем групповом чате или группу в Teams.
