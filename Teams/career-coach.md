# <a name="get-started-with-career-coach-for-microsoft-teams"></a>Начало работы с карьерным тренером для Майкрософт Teams

Карьерный тренер — это приложение Microsoft Teams для образования на платформе LinkedIn, которое предоставляет персонализированное руководство для учащихся по переходу по карьерной лестнице. С помощью карьерного тренера образовательные учреждения могут помочь учащимся открыть свой карьерный путь, развивать реальные навыки и строить свою сеть в одном месте.

Эта статья посвящена ИТ-администраторам образования, чтобы приступить к настройке карьерного тренера для своего учебного заведения.

Ниже приведены ресурсы карьерного тренера для руководителей образования, преподавателей и учащихся:

- [Обзор возможностей карьерного тренера](https://aka.ms/career-coach).
- [Рекомендации и полезные советы для студентов, преподавателей и сотрудников](https://support.microsoft.com/office/c5d0b934-bfcf-4fe7-8a85-ba7bbb1b6ad4).
- [Краткое руководство по началу работы с тренером по карьере для учащихся](https://support.microsoft.com/topic/career-coach-quick-start-guide-for-students-c419db47-9290-4961-9684-c3f86a9b3708).
- [Привлекайте учащихся с помощью планов уроков карьерного тренера](https://support.microsoft.com/topic/engage-students-with-career-coach-lesson-plans-086ce412-05de-4259-a9fd-c96471cef1b0).

## <a name="career-coach-supported-languages"></a>Языки, поддерживаемые карьерным тренером

Карьерный тренер локализован на следующих языках:

- Китайский (упрощенное письмо, материковый Китай)
- Китайский (традиционное письмо, Тайвань)
- Английский (США)
- Английский (Соединенное Королевство)
- Французский (Канада)
- Французский (Франция)
- Немецкий (Deutschland)
- Японский (Япония)
- Португальский (Бразилия)
- Испанский (Испания)
- Испанский (Мексика)

## <a name="career-coach-requirements"></a>Требования к карьере тренера

Ознакомьтесь с приведенными ниже требованиями, чтобы настроить карьерный тренер для вашего учреждения.

### <a name="technical-requirements"></a>Технические требования

- Office 365 клиента с Azure Active Directory с разрешенными подключениями к учетной записи LinkedIn.
- Майкрософт Teams.
- Страница LinkedIn School.

### <a name="data-and-files-from-your-educational-institution"></a>Данные и файлы из учебного заведения

- [URL-адрес учебной страницы LinkedIn учебного](https://www.linkedin.com/help/linkedin/answer/40133/differences-between-a-linkedin-page-for-a-school-and-company?lang=en) заведения (обязательно).
- Логотип учебного заведения и графические ресурсы [в требуемом формате](career-coach-set-up-steps.md#brand-and-preferences-optional) (необязательно).
- Данные каталога курсов. [Просмотр примера файла](https://aka.ms/career-coach/docs/it-admins/sample-catalog) и [формата документа](career-coach-set-up-steps.md#course-catalog-document-format-and-schema) (необязательно).
- Список предлагаемых областей обучения. [Просмотр примера файла](https://aka.ms/career-coach/docs/it-admins/sample-fieldsofstudy) (необязательно).
- URL-адрес политики конфиденциальности учебного заведения (необязательно).
- Образовательные учреждения ссылаться на ресурсы, связанные с карьерой, такие как веб-сайт карьерных услуг и публикации о вакансиях учащихся (необязательно).

## <a name="accessibility-in-career-coach"></a>Специальные возможности в карьерном тренере

Карьерный тренер несет ответственность за создание интерфейса, доступного для всех пользователей. Карьерный тренер соответствует WCAG 2.1 A, AA и AAA и принимает все глобальные стандарты доступности.

Специальные возможности Карьерного тренера включают в себя:

- Логическая структура заголовка и четкие индикаторы фокуса для поддержки навигации с помощью мыши, клавиатуры или голоса.
- Пропустить ссылки, чтобы пользователи могли быстрее получить доступ к важным функциям.
- Четкий и краткий текст, чтобы сделать чтение и понимание карьерного тренера простым для всех пользователей.
- Цветовая контрастность с режимами темной и высокой контрастности для различных потребностей и предпочтений.
- Пользовательский интерфейс правильного размера, проверенный во всех форматах (PC, Mac, Android и iPhone) с помощью [раздел 508 Доверенные тестировщики](https://www.dhs.gov/trusted-tester).
