# <a name="availability-in-australia"></a>Доступность в Австралии

[Найдите другую страну или регион](country-and-region-availability-for-audio-conferencing-and-calling-plans.md) или узнайте больше о доступности стран и регионов для аудио конференций и тарифных планов.

Ниже приведены службы Microsoft 365 и Office 365 для Skype для бизнеса и Microsoft Teams, доступных для Австралии.

|**Функция**|**Details**|
|:-----|:-----|
|Доступно ли приобретение аудиоконференций?  <br/> |Да  <br/> |
|Назначаются ли номера телефонов для аудиоконференций автоматически?  <br/> |Нет. [Свяжитесь с нами](mailto:ptnapac@microsoft.com). <br/> |
|Возможность получить обратный звонок с функцией "Звонок на мой номер"  <br/> |Доступно  <br/> |

> Дополнительные сведения можно найти в разделе [вызов Telstra](https://aka.ms/TelstraVoicePlan).

### <a name="if-your-tenant-is-located-outside-australia"></a>Если ваш клиент находится за пределами Австралии

Начиная с 1 сентября 2019, если ваш клиент находится за пределами Австралии, вы можете приобрести планы звонков для пользователей, которые находятся в Австралии. Эта возможность доступна для следующих стран и регионов.

|**Функция**|**Details**|
|---------|---------|
|Северная и Южная Америка    |США, внешние острова США      |
|Азиатско-Тихоокеанский регион     |Бангладеш, Бутан, Бруней, Камбоджа, Гонконг, Индонезия, Лаос, Макао, Малайзия, Мальдивские, Мьянма,, от|
|Европа, Ближний Восток и Африка    |Аландские острова, Андорра, Австрия, Бельгия, Буве острова, Болгария, Хорватия, Кипр, Чешская Республика, Дания, Эстония Фарерские острова, Финляндия, Франция, Германия, Гибралтар, Греция, Гренландия, Гернси, Венгерская, Исландия, Ирландия, остров, г., Нью-Йорк, г., Шпицберген, Лихтенштейн, Литва, нидерландский, Словакия, Монако, Нидерланды, Норвегия, Швеция, Португалия, Польша, Великобритания, Латвия  |

Обратите внимание, что вызов Telstra будет отображаться в виде надстройки только для тех пользователей, которые находятся в Австралии, а также в указанных выше странах и регионах, после того как они будут предоставлены Telstra. Для пользователей из других географических регионов они будут по мере необходимости подключены к планам звонков Microsoft и прямой маршрутизации. Дополнительные сведения можно найти в разделе [вызов Telstra](https://aka.ms/TelstraVoicePlan).
