# <a name="availability-in-australia"></a>Доступность в Австралии

[Укажите другую страну или регион](country-and-region-availability-for-audio-conferencing-and-calling-plans.md) либо получите дополнительную информацию о странах и регионах, где доступны аудиоконференции и планы звонков.

Ниже приведены службы Office 365 для Skype для бизнеса и Microsoft Teams, доступные для Австралии.

> Для получения дополнительных сведений см. [вызов Telstra для Office 365](https://aka.ms/TelstraVoicePlan).

### <a name="if-your-tenant-is-located-outside-australia"></a>Если ваш клиент находится за пределами Австралии

Начиная с 1 сентября 2019, если ваш клиент находится за пределами Австралии, вы можете приобрести планы звонков для пользователей, которые находятся в Австралии. Эта возможность доступна для следующих стран и регионов.

|||
|---------|---------|
|Америки    |США, внешние острова США      |
|Азиатско-Тихоокеанский регион     |Бангладеш, Бутан, Бруней, Камбоджа, Гонконг, Индонезия, Лаос, Макао, Малайзия, Мальдивские, Мьянма,, от|
|Европа, Ближний Восток и Африка    |Аландские острова, Андорра, Австрия, Бельгия, Буве островов, Болгария, Хорватия, Кипр, Чешская Республика, Дания, Эстония, чешский острова, Финляндии, Германия, Австрийская, Гернси, Венгерская, Гренландия, Ирландия, Греция, Исландия и т. д., остров Джерси, Латвия, Лихтенштейн, Литва, Люксембург, Мальта, Монако, нидерландский, Польша, Великобритания, Нидерланды, Норвегия, Польша, Португалия, Швеция, Шпицберген, Словакия, Словения, Великобритания, Шпицберген  |

Обратите внимание, что вызов Телстра для Office 365 будет показываться в качестве надстройки только для пользователей, которые находятся в Австралии, и указанных выше стран и регионов, после того как они будут предоставлены Телстра. Для пользователей из других географических регионов они будут по мере необходимости подключены к планам звонков Microsoft и прямой маршрутизации. Для получения дополнительных сведений см. [вызов Telstra для Office 365](https://aka.ms/TelstraVoicePlan).
