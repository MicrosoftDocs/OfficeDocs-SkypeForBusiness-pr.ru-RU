# <a name="plan-your-teams-voice-solution"></a>Планирование голосового Teams решения

В этой статье вы можете определить, какое голосовое решение Майкрософт лучше всего выбрать для вашей организации. После этого в этой статье содержится план для содержимого, которое позволит вам реализовать выбранное решение.

Вам может потребоваться простейшее решение телефонная система&mdash; с планом звонков. Этот вариант является полностью облачным решением Майкрософт, которое предоставляет функции частных филиалов Exchange (УАЦ) и звонки на телефонную сеть общего звонков (STN), как показано на приведенной ниже схеме. С помощью этого решения корпорация Майкрософт является вашим оператором ПСПС.

![На схеме 1 показаны телефонная система с планом звонков.](media/voice-solutions-simple.png)

Если вы отвечаете да, то телефонная система с помощью плана звонков — это правильное решение.

- План звонков доступен в вашем регионе.
- Не нужно хранить текущего оператора связи через ДНР.
- Вы хотите использовать доступ к STN, управляемый Майкрософт.

Однако ваша ситуация может быть более сложной. Например, у вас могут быть офисы в расположениях, где план звонков не доступен. Или вам может потребоваться комбинированное решение, которое поддерживает сложное развертывание на нескольких уровнях с различными требованиями для разных географических местоположений. Корпорация Майкрософт поддерживает сочетание решений:

- телефонная система с планом звонков
- телефонная система с собственным оператором ПСОП с оператором Подключение
- телефонная система с собственным оператором ПСПС с прямой маршрутией
- Комбинированное решение, использующее телефонная система с планом звонков, телефонная система с операторами Подключение и /или телефонная система с прямой маршрутией

>[!NOTE]
>Если вы малый и средний бизнес (не более 300 человек), корпорация Майкрософт теперь телефонная система с планом внутренних звонков. Дополнительные сведения см. [в](/microsoftteams/business-voice/whats-business-voice) телефонная система руководства для малого и среднего бизнеса по планированию, настройкам и управлению голосовым решением.

## <a name="phone-system"></a>Телефонная система

телефонная система технологией Майкрософт для включения управления звонками и личных ветвей Exchange (УАПС) в облаке Microsoft 365 с Microsoft Teams.

телефонная система работает с клиентами Teams и сертифицированными устройствами. телефонная система вы можете заменить существующую систему УАКС набором функций, непосредственно поставляемой из Microsoft 365. 

Звонки между пользователями в организации независимо от географического региона обрабатываются внутри телефонная система. Эти внутренние звонки никогда не перенапорядуются на телефонную сеть общего звонков (STN), поэтому ваша компания не взимает междугородние платежи.

### <a name="cloud-voicemail"></a>Облачная голосовая почта

облачная голосовая почта на платформе служб голосовой почты Azure, поддерживает депозиты голосовой почты только Exchange почтовым ящикам. Она не поддерживает сторонние почтовые системы. 

Голосовая облачная почта включает функцию расшифровки голосовых сообщений, которая по умолчанию включена для всех пользователей в организации. Для бизнес-потребностей может потребоваться отключить транскрибцию голосовой почты для определенных пользователей или всех пользователей в организации.

## <a name="public-switched-telephone-network-connectivity-options"></a>Параметры подключения к телефонной сети общего перейти

телефонная система предоставляет полный доступ к УАС для вашей организации. Однако для того чтобы пользователи могли звонить за пределы организации, необходимо подключить телефонная система к телефонной сети общего звонков (STN). Чтобы подключить телефонная система к STN, можно выбрать один из следующих вариантов:

- [**телефонная система с планом звонков**](pstn-connectivity.md#phone-system-with-calling-plan). Облачное решение, в качестве оператора связи по STN — Майкрософт.

- [**телефонная система с собственным оператором ПСОП с помощью оператора Подключение**](operator-connect-plan.md). Если оператор Подключение, если ваш оператор участвует в программе Подключение Майкрософт, он может управлять службой для перенаправки звонков по ПСОП Teams. Сведения о преимуществах и требованиях оператора Подключение см. в [Подключение.](operator-connect-plan.md)

- [**телефонная система с собственным оператором**](pstn-connectivity.md#phone-system-with-direct-routing) ННР, используя прямую маршрутику для подключения локальной среды к Teams.

Вы можете выбрать сочетание параметров, которое позволяет создать решение для сложной среды или управлять многошаговой миграцией. Подробнее о миграции вы сможете узнать позже.

Большинство телефонная система функций одинаковы независимо от того, какой способ подключения вы выбрали. Однако существуют некоторые отличия в функциональных возможностях, которые влияют на настройку некоторых функций телефонная система, таких как маршруты звонков и экстренные вызовы. Дополнительные сведения о параметрах подключения к ОКП и конфигурации см. в этой [теме](pstn-connectivity.md).
