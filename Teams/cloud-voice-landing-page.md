# <a name="cloud-voice-in-microsoft-teams"></a>Облако голосовой связи в группах Майкрософт

Вы завершили [приступить к работе](get-started-with-teams-quick-start.md). Вы для развертывания групп с [чата, групп, каналы, & приложения](deploy-chat-teams-channels-microsoft-teams-landing-page.md) внутри организации. Может быть развертывания [конференц-связи & собрания](deploy-meetings-microsoft-teams-landing-page.md). Теперь вы готовы к добавлять облачных возможности голосовой связи для пользователей. 

Облако голосовой предоставляет возможности Exchange закрытый филиала (УАТС) и параметры для подключения к общедоступной переключения телефонной сети общего пользования (PSTN).

Эта статья поможет вам решить, нужно ли изменять любые параметры по умолчанию облачных голосовой связи, на основе профилей вашей организации и бизнес-требования, а затем его — описание каждого изменения. Мы разделение параметры на две группы, начиная с базовый набор [вы, скорее всего, внесите изменения](#core-deployment-decisions). Второй группы включает в себя [Дополнительные параметры](#additional-deployment-decisions) , которые может потребоваться настроить, исходя из потребностей вашей организации.

Рекомендуется всей организации работает через основные решения и, если в вашей организации есть дополнительные требования, просмотрите следующие материалы.

### <a name="voicemail"></a>Голосовая почта

Система голосовой почты телефона, на базе служб Azure голосовой почты, поддерживает вклады голосовой почты в почтовые ящики Exchange только и не поддерживает систем электронной почты сторонних производителей. Голосовая почта Телефонной системы включает функцию расшифровки голосовых сообщений, которая по умолчанию включена для всех пользователей в организации. Бизнес-требованиям может потребоваться отключить транскрибирования голосовой почты для определенных пользователей или всем пользователям в организации.

|Задайте себе|Действие |
|:------------|:-------|
| Требуется ли Включение телефонной системы голосовой почты? | Процедуры настройки голосовой почты см [телефонной системы голосовой почты](set-up-phone-system-voicemail.md).
| Требуется ли включение транскрибирования голосовой почты для некоторых или всех Мои пользователи? | Чтобы отключить транскрибирования голосовой почты, обратитесь к [настройке политик голосовой почты в вашей организации](set-up-phone-system-voicemail.md#setting-voicemail-policies-in-your-organization).</li></ul>|
