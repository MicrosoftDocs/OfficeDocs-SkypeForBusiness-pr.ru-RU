# <a name="getting-phone-numbers-for-your-users"></a>Получение телефонных номеров для пользователей

[] Прежде чем настраивать возможности совершения и получения телефонных звонков для пользователей в организации, необходимо получить для них номера телефонов.
  
Получить номера пользователей можно тремя способами:

- **Используйте Центр администрирования Microsoft Teams.** Для некоторых стран и регионов можно получить номера пользователей с помощью Центра администрирования Microsoft Teams. См [. раздел Получение новых номеров телефонов для пользователей](#get-new-phone-numbers-for-your-users).

- **Портируйте номера, которые у вас уже есть.** Вы можете переносить или передавать существующие номера от текущего поставщика услуг или оператора связи. Дополнительные сведения см. в [разделах Перенос номеров телефонов в Teams](./phone-number-calling-plans/transfer-phone-numbers-to-teams.md) или [Управление номерами телефонов для вашей организации](/microsoftteams/manage-phone-numbers-for-your-organization).  
  
- **Используйте форму запроса для новых номеров.** Иногда (в зависимости от страны или региона) вы не сможете получить новые номера телефонов с помощью Центра администрирования Microsoft Teams или вам потребуются определенные номера телефонов или коды регионов. Дополнительные сведения см. в статье [Управление номерами телефонов для организации](/microsoftteams/manage-phone-numbers-for-your-organization).
  
> [!NOTE]
> Если вам нужна помощь по настройке номеров телефонов для вашей организации, [обратитесь в службу поддержки по бизнес-продуктам Администратор справка](/microsoft-365/admin/contact-support-for-business-products).

## <a name="get-new-phone-numbers-for-your-users"></a>Получение новых номеров телефонов для пользователей

**С помощью Центра администрирования Microsoft Teams**

Вы должны быть администратором службы Teams, чтобы вносить эти изменения. Сведения о получении ролей и разрешений администратора см. в статье [Управление Teams с помощью ролей администратора Teams](./using-admin-roles.md).

1. Войдите в [Центр администрирования Microsoft Teams](https://go.microsoft.com/fwlink/p/?linkid=2066851).

2. В области навигации слева перейдите в раздел **Номера голосовых** > **телефонов** и нажмите кнопку **Добавить**.

3. Введите имя заказа и добавьте описание.

4. На странице Расположение и количество выполните следующие действия.
    1. В разделе **Страна или регион** выберите страну или регион.
    2. В разделе **Тип номера** выберите **Пользователь (подписчик).**
    3. В разделе **Расположение** выберите расположение. Если необходимо создать новое расположение, выберите **Добавить расположение**.
    4. В разделе **Код города** выберите код области.
    5. В разделе **Количество** введите нужное количество номеров для вашей организации, а затем нажмите **кнопку Далее** , чтобы выбрать номера.

5. Выберите нужные числа. У вас есть 10 минут, чтобы выбрать номера телефонов и разместить заказ. Если вы занимаете более 10 минут, номера телефонов будут возвращены в пул номеров.

6. Когда вы будете готовы разместить заказ, выберите **Разместить заказ**.

    > [!IMPORTANT]
    > Количество номеров телефонов для пользователей (подписчиков) равно общему количеству назначенных лицензий на **план внутренних звонков** и **международных звонков** , умноженных на 1,1, плюс 10 дополнительных номеров телефонов. Например, если у вас есть в общей сложности 50 пользователей с планом внутренних звонков и (или) международным тарифным планом, вы можете приобрести **65** телефонных номеров **(50 x 1.1 + 10)**. Обратите внимание, что если у вас есть тарифный план с оплатой по мере использования, вы можете получить только один номер телефона для каждой назначенной лицензии.
    >
    > Дополнительные сведения см. в разделе [Сколько номеров телефонов можно получить?](./how-many-phone-numbers-can-you-get.md). Если вам нужно получить больше номеров телефонов, [обратитесь в службу поддержки по бизнес-продуктам — Администратор справке](/microsoft-365/admin/contact-support-for-business-products).
  
## <a name="port-or-transfer-phone-numbers-from-your-service-provider-or-phone-carrier"></a>Перенос номеров от вашего поставщика услуг или оператора телефонной связи.
  
- Если вам нужно 999 или меньше номеров телефонов для пользователей, используйте мастер переноса в Центре администрирования Microsoft Teams. Выполните действия, описанные в разделе [Перенос номеров телефонов в Teams](./phone-number-calling-plans/transfer-phone-numbers-to-teams.md). Если вам нужна помощь в этом процессе, вы можете [вручную отправить заказ на перенос](phone-number-calling-plans/manually-submit-port-order.md) или см. статью [Управление номерами телефонов для вашей организации,](/microsoftteams/manage-phone-numbers-for-your-organization) чтобы скачать правильное письмо о авторизации (LOA).

- Если вам нужно перенести более 999 телефонных номеров, вы можете [вручную отправить заказ на перенос](phone-number-calling-plans/manually-submit-port-order.md) или см. статью [Управление номерами телефонов для вашей организации,](/microsoftteams/manage-phone-numbers-for-your-organization) чтобы скачать правильное письмо о авторизации (LOA). Заполните и подпишите документы LOA, а затем обратитесь [в службу технической поддержки TNS](manage-phone-numbers-for-your-organization/contact-tns-service-desk.md) для вашего региона.

## <a name="view-the-phone-numbers-for-your-organization"></a>Просмотр номеров телефонов для вашей организации

**С помощью Центра администрирования Microsoft Teams**

В левой области навигации Центра администрирования перейдите в раздел **Номера голосовых** > **телефонов** , чтобы просмотреть номера для организации, включая расположение, тип номера и сведения о состоянии.
  
## <a name="assign-phone-numbers-to-users"></a>Назначение номеров телефонов пользователям

После получения номеров телефонов необходимо назначить номер каждому из пользователей. Дополнительные сведения см. в статье [Назначение, изменение или удаление номера телефона для пользователя](./assign-change-or-remove-a-phone-number-for-a-user.md).
