# <a name="getting-phone-numbers-for-your-users"></a>Получение телефонных номеров для пользователей

[] Прежде чем настраивать возможности совершения и получения телефонных звонков для пользователей в организации, необходимо получить для них номера телефонов.
  
Существует три способа получения номеров пользователей:

- **Используйте Центр администрирования Microsoft Teams.** В некоторых странах и регионах вы можете получить номера для пользователей с помощью Центра администрирования Microsoft Teams. См [. раздел "Получение новых номеров телефонов для пользователей"](#get-new-phone-numbers-for-your-users).

- **Портируйте номера, которые у вас уже есть.** Вы можете перенести или перенести существующие номера у текущего поставщика услуг или оператора связи. См. [Перевод телефонных номеров в Teams](./phone-number-calling-plans/transfer-phone-numbers-to-teams.md) или [Управление телефонными номерами вашей организации](/microsoftteams/manage-phone-numbers-for-your-organization) для получения дополнительной информации, чтобы помочь вам сделать это.  
  
- **Используйте форму запроса для новых номеров.** Иногда (в зависимости от страны или региона) вы не сможете получить новые номера телефонов с помощью Microsoft Teams Администратор Center или вам потребуются определенные номера телефонов или коды областей. Для получения дополнительных сведений см. статью [Управление номерами телефонов организации](/microsoftteams/manage-phone-numbers-for-your-organization).
  
> [!NOTE]
> Если вам нужна помощь по настройке номеров телефонов для организации, обратитесь в службу поддержки по вопросам продуктов для бизнеса [Администратор справку](/microsoft-365/admin/contact-support-for-business-products).

## <a name="get-new-phone-numbers-for-your-users"></a>Получение новых номеров телефонов для пользователей

**С помощью Центра администрирования Microsoft Teams**

Вы должны быть администратором службы Teams, чтобы вносить эти изменения. Сведения о получении ролей и разрешений администратора см. в статье [Управление Teams с помощью ролей администратора Teams](./using-admin-roles.md).

1. Перейдите в Центр Администратор Microsoft Teams.

2. В области навигации слева перейдите к **номерам голосовых** > **телефонов** и нажмите кнопку **"Добавить"**.

3. Введите имя заказа и добавьте описание.

4. На странице "Расположение и количество" сделайте следующее:
    1. В **разделе "Страна или регион**" выберите страну или регион.
    2. В **разделе "Тип номера**" **выберите "Пользователь (подписчик)"**.
    3. В **разделе "** Расположение" выберите расположение. Если необходимо создать новое расположение, выберите " **Добавить расположение"**.
    4. В **разделе "Код области**" выберите код области.
    5. В **поле "Количество**" введите нужное количество номеров для организации и нажмите кнопку **"Далее** ", чтобы выбрать нужные номера.

5. Выберите нужные числа. У вас есть 10 минут для выбора номеров телефонов и размещения заказа. Если потребуется более 10 минут, номера телефонов будут возвращены в пул номеров.

6. Когда вы будете готовы разместить заказ, выберите " **Разместить заказ"**.

    > [!IMPORTANT]
    > Количество номеров телефонов для пользователей (подписчиков) равно общему количеству назначенных вами  лицензий на план  внутренних звонков и план международных звонков, умноженное на 1,1 плюс 10 дополнительных номеров телефонов. Например, если у вас всего 50 пользователей с планом внутренних звонков и (или) планом международных звонков, вы можете приобрести **65** номеров телефонов **(50 x 1,1 + 10)**. Обратите внимание, что если у вас есть тарифный план с оплатой по мере использования, вы можете получить только один номер телефона на назначенную лицензию.
    >
    > Дополнительные сведения см. в статье [о том, сколько телефонных номеров можно получить?](./how-many-phone-numbers-can-you-get.md) Если вам нужно получить больше номеров телефонов, обратитесь в службу поддержки по вопросам продуктов для бизнеса [Администратор справку](/microsoft-365/admin/contact-support-for-business-products).
  
## <a name="port-or-transfer-phone-numbers-from-your-service-provider-or-phone-carrier"></a>Перенос номеров от вашего поставщика услуг или оператора телефонной связи.
  
- Если для пользователей требуется не более 999 телефонных номеров, воспользуйтесь мастером переноса в Центре Администратор Microsoft Teams. Выполните действия, описанные в [разделе "Передача номеров телефонов в Teams"](./phone-number-calling-plans/transfer-phone-numbers-to-teams.md). Если ваша страна или регион не указаны в мастере переноса, вы можете вручную отправить заказ на перенос или просмотреть [](/microsoftteams/manage-phone-numbers-for-your-organization) сведения об управлении номерами телефонов для вашей организации, чтобы скачать правильную букву авторизации (LOA).[](phone-number-calling-plans/manually-submit-port-order.md)

- Если вам нужно перенести более 999 номеров телефонов, можно вручную [](phone-number-calling-plans/manually-submit-port-order.md) отправить заказ на перенос или просмотреть раздел [](/microsoftteams/manage-phone-numbers-for-your-organization) "Управление номерами телефонов", чтобы скачать правильную букву авторизации (LOA), а затем отправить его в службу технической поддержки [TNS](manage-phone-numbers-for-your-organization/contact-tns-service-desk.md), чтобы перенести все номера.

## <a name="view-the-phone-numbers-for-your-organization"></a>Просмотр номеров телефонов для организации

**С помощью Центра администрирования Microsoft Teams**

В левой области навигации  >  Центра администрирования перейдите к номерам голосовых телефонов, чтобы просмотреть номера для организации, включая сведения о расположении, типе номера и состоянии.
  
## <a name="assign-phone-numbers-to-users"></a>Назначение номеров телефонов пользователям
