# <a name="manage-microsoft-teams-features-in-your-office-365-organization"></a>Управление группами Майкрософт возможностями в организации Office 365

Все параметры команды скоро перемещаются в новый группами Майкрософт & Скайп по центру администрирования бизнес. Только группы компонент, который управляется в центре администрирования Office 365 — приложения. 

Если не указано иное, по умолчанию параметр имеет значение **на**.

## <a name="office-365-tenant-wide-settings"></a>Параметры Office 365 на уровне клиента 

В **Параметры на уровне клиента**можно включить или отключить приложений.

Чтобы изменить **Параметры на уровне клиента** для групп, перейдите к группам Microsoft & Скайп по центру администрирования бизнес и выберите **портала прежних версий**. Выберите **Settings (Параметры)** > **Services & add-ins (Службы и надстройки)** > **Microsoft Teams**. Если вы вошли как администратор Office 365, воспользуйтесь ссылкой 
>  
> https://portal.office.com/adminportal/home#/Settings/ServicesAndAddIns  

### <a name="apps"></a>Приложения

Приложения — это предоставляемые сторонними службами вкладки, соединители, боты или любое их сочетание. Центр администрирования Office 365 позволяет настроить политики администрирования Microsoft Teams, которые определяют разрешенные внешние сторонние приложения. Эти политики позволяют указать, какие приложения могут и не разрешены, новое поведение внешнего приложения и разрешено ли загрузка со стороны приложений. 

В разделе **Приложения** вы можете настроить для своей организации следующие параметры: 

![Снимок экрана с разделом приложений.](media/Enable_Microsoft_Teams_features_in_your_Office_365_organization_image6.png)

- **Allow external apps in Microsoft Teams** (Разрешить внешние приложения в Microsoft Teams). Когда этот параметр включен, пользователи могут добавлять вкладки и боты, доступные для клиента Office 365. 
 
    ![Снимок экрана с параметром "Allow external apps control" (Разрешить управление внешними приложениями) в разделе приложений.](media/Enable_Microsoft_Teams_features_in_your_Office_365_organization_image6.2.png)

- **Enable new external apps by default** (Включить новые внешние приложения по умолчанию). Если этот параметр включен, пользователи могут активировать новые приложения сразу после их добавления в каталог приложений Microsoft Teams. Если вы хотите контролировать новые приложения, отключите этот параметр. В этом случае не забывайте периодически просматривать новые приложения, чтобы ваша организация не пропустила появление новых интересных программ. 

- **Разрешить sideloading внешних приложений**: при включении этого параметра пользователи могут устанавливать и включить настраиваемые программы-роботы и вкладок. 

Подробнее см. в статье [Параметры администратора для приложений в Microsoft Teams](admin-settings.md). 

## <a name="teams-org-wide-settings"></a>Параметры команды масштабе организации

Могут управлять параметрами пользователей организации в группами Майкрософт & Скайп по центру администрирования бизнеса. Чтобы изменить параметры на уровне организации, перейдите к Скайп Майкрософт по центру администрирования бизнеса и выберите пункт **Параметры масштабе организации**. Можно настроить следующие параметры.

### <a name="external-access"></a>Внешний доступ

**Внешний доступ** позволяет группам и Скайп для бизнес-пользователи взаимодействовать с пользователями, которые не являются сотрудниками вашей организации. Настройка внешнего доступа, перейдите к [Let разговор группам пользователей и взаимодействовать с пользователями в другой организации групп](let-your-teams-users-communicate-with-other-people.md).

### <a name="guest-access"></a>Гостевой доступ

**Доступ в качестве гостя** в группах Майкрософт позволяет группам в организации совместной работы с людьми за пределами вашей организации, предоставив им доступ к группам и каналы. Лица, имеющие business потребитель электронной почты учетной записи или, например, Outlook, Gmail или другим пользователям, могут участвовать в качестве гостя в группах с полным доступом для групп обсуждения, собраний и файлы. Для получения дополнительных сведений см. [гостевой доступ в группах Майкрософт](guest-access.md).

### <a name="teams-settings"></a>Параметры команды

**Параметры команды**можно выполнить настройку интеграции электронной почты, параметры хранения в облаке, Скайп взаимодействия бизнеса и устройств.

#### <a name="email-integration"></a>Интеграция электронной почты

Включите этот параметр, чтобы пользователи могли отправлять сообщения электронной почты на адрес канала в Teams. Подобную операцию пользователи могут выполнить для любого канала, относящегося к принадлежащей им команде. Пользователи также могут отправлять по электронной почте для любого канала в группы, которая имеет Добавление соединителей включена для участников группы. Чтобы включить интеграции электронной почты, убедитесь, что **Разрешить пользователям отправлять по электронной почте на адрес электронной почты канала** **на**. 

#### <a name="files"></a>Файлы

Здесь можно включить или отключить файл общего доступа и облачных файл варианты хранения. 

Пользователи могут загружать и отправлять друг другу файлы из облачных служб хранения в каналах и чатах Teams. Облако в группах в настоящее время доступны следующие варианты хранения ShareFile, общего банка данных, поля и Google диска. Выберите переключатель рядом с поставщиками услуг облачного хранения, которые нужно использовать в вашей организации.

#### <a name="organization"></a>Организация

Здесь можно включить на вкладке **организация** показаны подробные организационной диаграммы для организации пользователя. Дополнительные сведения можно [использовать вкладку организация в группах](https://support.office.com/article/use-the-organization-tab-in-teams-ff02568b-290a-46d6-ae7a-cda22f723894).

#### <a name="skype-for-business-interop"></a>Скайп для взаимодействия бизнеса

Используйте этот параметр, чтобы позволить группам пользователей разговаривать с Скайп для бизнес-пользователи. Получить подробные сведения о взаимодействии между рабочими группами и Скайп для бизнеса перейдите в [понять группами Майкрософт и Скайп для взаимодействие и сосуществование бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

#### <a name="devices"></a>Устройства

Эти параметры управления поведением учетная запись ресурса для сервера-концентратора поверхность устройств посещение совещаний групп Майкрософт. Используйте эти параметры для настройки требования к проверке подлинности, требуют контента ПИН-код и включите учетные записи ресурса сервера-концентратора поверхность для отправки сообщений.

- ПРИКРЕПЛЕНИЕ **требуется дополнительная форма проверки подлинности для доступа к контенту собрания** — выберите уровень доступа пользователей при вводе контента.
- **Задать содержимое ПИН-код** – требуются для пользователей вводить этот ПИН-кода для предотвращения несанкционированного доступа к документам. Это позволяет предотвратить неавторизованных пользователей из присоединения к собраниям предстоящие и просмотра вложений.
- **Учетные записи ресурсов могут отправлять сообщения** – включить этот параметр **на** передачу сообщений для отправки из учетной записи ресурса сервера-концентратора поверхности.

#### <a name="search"></a>Поиск

Поиск в каталоге с областью группами Майкрософт использует политики адресной книги Exchange (aPb в качестве), позволяющие организациям для создания виртуального границ, определяющие, как пользователи могут находить и общаться с другими пользователями в своей организации. Можно использовать поиск в каталоге с областью в подобных ситуациях:

- Вашей организации есть несколько компаний в его клиентов, вам необходимо хранить отдельно. 
- Школа требуется ограничение чаты между преподавателей и учащихся. 

Переключение этот параметр **на** Включение поиска в каталоге с заданной областью.

### <a name="teams-upgrade"></a>Обновление группы

Эти параметры можно использовать для настройки, как пользователи будут обновлены из Скайп для бизнеса для групп Майкрософт. 

#### <a name="coexistence-mode"></a>Режим совместимости
Можно указать режим сосуществования: **только команды**, **о-ва** (групп и Скайп для бизнеса будет сосуществовать) или **Скайп для бизнеса только**. Выбранным режимом сосуществования определяет маршрутизации входящих звонков и чатов и приложения, используемого пользователем, для инициации чаты и звонков или планировать собрания. Дополнительные сведения о режимах сосуществования перейдите в [понять группами Майкрософт и Скайп для взаимодействие и сосуществование бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

#### <a name="app-preferences"></a>Параметры приложения

Здесь можно выбрать приложение, которое пользователи будут использовать для присоединения к Скайп для бизнеса собрания (Скайп для бизнеса или [Приложение Скайп собрания](https://support.office.com/en-us/article/What-is-Skype-Meetings-App-Skype-for-Business-Web-App-1FF3D412-718A-4982-8FF2-A4992608CDB5)). Этот параметр не зависит от параметр режима совместной работы.
