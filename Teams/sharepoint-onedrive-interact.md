# <a name="how-sharepoint-online-and-onedrive-for-business-interact-with-microsoft-teams"></a>Взаимодействие SharePoint Online и OneDrive для бизнеса с Microsoft Teams

> [!Tip]
> Посмотрите следующий эпизод, чтобы узнать, как Teams взаимодействует с Azure Active Directory (AAD), группами Microsoft 365, Exchange, SharePoint и OneDrive для бизнеса: [Основы Microsoft Teams](https://aka.ms/teams-foundations)

Каждая команда в Microsoft Teams имеет сайт группы в SharePoint Online, а каждый стандартный канал команды получает папку в библиотеке документов сайта группы по умолчанию. Файлы, совместно используемые во время беседы, автоматически добавляются в библиотеку документов, а заданные в SharePoint разрешения и параметры безопасности файлов автоматически переносятся в Teams. Чтобы увидеть последствия изменения адреса сайта в SharePoint, см. статью ["Изменение адреса сайта".](/sharepoint/change-site-address)

> [!NOTE]
> Эта статья относится только к стандартным каналам. Архитектура частных каналов отличается от стандартных каналов. Каждый частный канал имеет собственное собрание веб-сайтов SharePoint, отдельное от родительского сайта группы. Подробнее см. в [закрытых каналах Microsoft Teams.](private-channels.md)

Личные файлы чата хранятся в папке OneDrive для бизнеса отправщика, и разрешения автоматически делиться файлами делиться с ними делиться данными делиться с участниками.

Если пользователям не назначены активные лицензии SharePoint Online, у них нет хранилища OneDrive для бизнеса в Microsoft 365 или Office 365. Общий доступ к файлам будет работать в стандартных каналах, но пользователи не смогут делиться файлами в чатах без хранилища OneDrive для бизнеса в Microsoft 365 или Office 365.

При сохранении файлов в библиотеке документов SharePoint Online и OneDrive для бизнеса выполняются все требования по обеспечению соответствия, заданные на уровне клиента. 

> [!NOTE]
> Интеграция с локальной службой SharePoint в настоящее время не поддерживается для Microsoft Teams.

Ниже приводится пример связей между командой, стандартным каналом и библиотекой документов.

Для каждой команды создается сайт SharePoint, а также папка по умолчанию **Общие документы**. У каждого стандартного канала, включая канал **"Общий"** (канал по умолчанию для каждой команды), есть папка в папке **"Общие документы".**

![Схема папок "Общие документы" в SharePoint Online.](media/Understand_how_SharePoint_Online_and_OneDrive_for_Business_interact_with_Microsoft_Teams_image1.png)

> [!NOTE]
> В настоящее время вы не можете заменить библиотеку документов и сайт SharePoint по умолчанию. Нас часто спрашивают об этой возможности, и мы рассматриваем пути ее внедрения. Обо всех готовящихся изменениях вы можете узнать в [Стратегии развития Teams](https://aka.ms/teamsroadmap) и в [посвященном Teams разделе UserVoice](https://aka.ms/TeamsUserVoice).

> [!TIP]
> Чтобы добавить в команду вкладку, которая связывает существующую страницу сайта SharePoint или существующую библиотеку документов SharePoint:
> 1. Выберите знак "плюс" рядом с вкладками.
> 2. Выберите **SharePoint для** существующей страницы  сайта SharePoint или библиотеку документов для существующей библиотеки документов.
> 3. Выберите соответствующую страницу или библиотеку документов.

У каждого пользователя имеется папка OneDrive **Файлы чатов Microsoft Teams** для хранения всех файлов, предоставленных для общего доступа в приватных чатах с другими пользователями (в формате один к одному или один ко многим). При этом разрешения для ограничения доступа настраиваются автоматически.

![Схема папки OneDrive с именем "Файлы чата Microsoft Teams"](media/Understand_how_SharePoint_Online_and_OneDrive_for_Business_interact_with_Microsoft_Teams_image2.png)

Обратите внимание, что для общедоступных команд сайт группы SharePoint имеет доступ "Все, кроме внешних пользователей". Эта команда не отображается в Teams для людей, которые не являются ее членами. Однако они могут получать доступ к контенту на сайте группы SharePoint, используя URL-адрес сайта группы SharePoint. 

## <a name="channel-files-tab"></a>Вкладка "Файлы канала"

> [!INCLUDE [new feature coming soon](includes/new-feature-coming-soon-section.md)]

Вкладка **"Файлы"** в Teams напоминает представление документов SharePoint. На **вкладке "Файлы"** пользователи могут:

- Дополнительные параметры в меню **"Новый** файл".
- Синхронизируйте файлы с локальным диском.
- В меню **"Все документы" переключение** **из** представления "Список" в представление **"Сжатое"** в представление **"Плитки".**
- Определите файлы, которые требуют внимания или имеют вредоносные программы.
- Сразу же проверяет, является ли файл просмотром только для чтения или иным.
- И узнайте, как это можно проверить.
- Прикрепить, открепить и изменить порядок сортировки файлов.
- Определение метаданных для каких файлов
- Вы можете выбрать один из множества других параметров фильтрации.
- Группировка файлов на основе заголовков столбцов.
- Изменение параметров столбца (перемещение влево или вправо, скрытие) и ширины столбца.

## <a name="default-link-type-setting"></a>Тип ссылки по умолчанию

В SharePoint и OneDrive есть параметр для администратора, который указывает тип ссылок по умолчанию для ссылок, созданных для файла. В Teams используется тот же подход, благодаря повторному использованию параметров, которые администратор устанавливает для SharePoint и OneDrive. Дополнительные сведения об этом способе описаны в разделе "Изменение типа ссылки по [умолчанию", когда пользователи получают ссылки для общего доступа.](/sharepoint/change-default-sharing-link)
