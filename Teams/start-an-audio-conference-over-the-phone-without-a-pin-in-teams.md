# <a name="start-an-audio-conference-over-the-phone-without-a-pin-in-microsoft-teams"></a>Начало аудиоконференции по телефону без ПИН-кода в Microsoft Teams

Пользователи, которые могут подключиться к собранию в зале ожидания, ожидают передачи музыки, так как организатор собрания Microsoft Teams еще не начал собрание.
  
Если организатор собрания вызывает собрание, по умолчанию для начала собрания требуется ПИН-код. Вы можете настроить его таким образом, чтобы любой пользователь может выполнить вход на собрание и не запрашивать ПИН-код для начала собрания. Центр администрирования можно использовать для включения или отключения этого параметра для одного пользователя.
  
ПИН-код не требуется для организатора собрания, если кто-то начал собрание из приложения Microsoft Teams. ПИН-код требуется только в том случае, если организатор собрания присоединяется к собранию по телефону. ПИН-код для собраний отправляется звуковому пользователю, когда ему назначена  лицензия на аудиоконференции и включена аудиоконференции. См [.](send-an-email-to-a-user-with-their-dial-in-information-in-teams.md) статью "Отправка сообщения электронной почты пользователю со сведениями о аудиоконференции и электронными письмами, которые автоматически отправляются пользователям при изменении параметров аудиоконференций ["](emails-sent-to-users-when-their-settings-change-in-teams.md).

## <a name="enable-or-disable-anonymous-callers-from-joining-a-meeting"></a>Разрешение и запрет присоединения анонимных абонентов к собранию

### <a name="using-the-microsoft-teams-admin-center"></a>С помощью Центра администрирования Microsoft Teams

1. В области навигации слева щелкните " **Пользователи"**.

2. Выберите пользователя в списке и нажмите кнопку **"Изменить** " в верхней части страницы.

3. Рядом с пунктом **Аудиоконференции** нажмите **Изменить**.

4. В области **аудиоконференций** включение или отключение абонентов с телефонным подключением может быть первым пользователем **на собрании**.

5. Нажмите кнопку **Применить**.

### <a name="using-windows-powershell"></a>Использование Windows PowerShell

Дополнительные сведения см. в справочнике [по Microsoft Teams PowerShell](/powershell/module/teams/?view=teams-ps) .

## <a name="what-else-should-you-know"></a>Дополнительные сведения

- Если вы хотите сбросить ПИН-код, см. раздел ["Сброс ПИН-кода аудиоконференций"](reset-the-audio-conferencing-pin-in-teams.md).

- Если анонимный доступ или пин-код не требуется для начала собрания, он отключен:

  - Если собрание еще не началось (в собрании еще нет ни одного пользователя): абоненту будет предложено, если он является организатором; Если он говорит "да", ему будет предложено ввести ПИН-код, а после ввода ПИН-кода начнется собрание, и пользователь присоединится к собранию.

  - Если собрание уже началось (кто-либо присоединился): абонент не получит сообщения с вопросом, является ли он организатором. Кроме того, PIN-код запрашиваться не будет. Собрание уже началось, и пользователь присоединится к нему.

- Если анонимный доступ или пин-код не требуется для начала собрания, включено:

  - Если собрание еще не началось (никто не присоединился): абонент не получит сообщения с вопросом, является ли он организатором. Кроме того, PIN-код запрашиваться не будет. Так как параметр организатора отключен, собрание начнется, а анонимные вызывающие абоненты присоединятся к собранию.

  - Если собрание уже запущено (кто-то уже находится на собрании): абонент не будет запрашиваться, если он является организатором, и ему никогда не будет предложено ввести ПИН-код. собрание уже запущено, и вызывающий объект присоединится к собранию.
