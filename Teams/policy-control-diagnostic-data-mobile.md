# <a name="required-mobile-diagnostic-data-for-microsoft-teams"></a>Обязательные мобильные диагностические данные для Microsoft Teams

Следующая статья содержит список мобильных событий в Microsoft Teams, а также список свойств, собираемых каждым событием.

Дополнительные сведения о диагностических данных, в том числе об управлении диагностическими данными, отправляемыми в корпорацию Майкрософт, см. в разделе [Диагностические данные, отправляемые из приложения Teams в корпорацию Майкрософт](policy-control-overview.md#diagnostic-data-sent-from-the-teams-app-to-microsoft). Чтобы просмотреть диагностические данные, отправляемые в корпорацию Майкрософт, можно использовать [Просмотр диагностических данных](https://support.microsoft.com/topic/cf761ce9-d805-4c60-a339-4e07f3182855).
