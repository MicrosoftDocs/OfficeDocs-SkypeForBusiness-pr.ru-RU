# <a name="microsoft-teams-pstn-blocked-users-report"></a>Отчет о заблокированных пользователях ТСОП в Microsoft Teams

Отчет о заблокированных пользователях ТСОП в Центре администрирования Microsoft Teams показывает пользователей в вашей организации, которым заблокировано выполнение звонков по ТСОП в Teams. Вы можете просмотреть дополнительные сведения о каждом заблокированном пользователе, включая назначенный им номер телефона и причину блокировки звонков.

## <a name="view-the-pstn-blocked-users-report"></a>Просмотр отчета о заблокированных пользователях ТСОП

В левой области навигации Центра администрирования Microsoft Teams щелкните **"Аналитика & отчетов** >  об **использовании"**. На **вкладке "Просмотр отчетов** " в разделе **"** Отчет" выберите заблокированных пользователей ТСОП **и нажмите** кнопку **"Запустить отчет"**.

![Снимок экрана: отчет о заблокированных пользователях ТСОП в Центре администрирования.](../media/teams-reports-pstn-blocked-users-with-callouts.png "Снимок экрана: отчет о заблокированных пользователях ТСОП в Центре администрирования Microsoft Teams с нумерованными выносами")

## <a name="interpret-the-report"></a>Толкование отчета

|Выноска |Описание  |
|--------|-------------|
|**1**   |Каждый отчет имеет дату создания. Действия обычно отражаются в отчетах с задержкой в 24-48 ч. |
|**2**   |Ось X — это дата. Ось Y — это количество пользователей. <br>Наведите указатель мыши на точку на заданную дату, чтобы просмотреть количество заблокированных пользователей в эту дату. |
|**3**   |В таблице приводится разбивка всех пользователей, которым заблокировано выполнение вызовов по ТСОП.  В нем отображаются все пользователи, которым назначена телефонная система или аудиоконференции, и приводятся дополнительные сведения о каждом пользователе. <ul><li>**Отображаемое** имя — это отображаемое имя пользователя. Вы можете щелкнуть отображаемое имя, чтобы перейти на страницу параметров пользователя в Центре администрирования Microsoft Teams. </li> <li>**Телефон** — это номер, назначенный пользователю.</li> <li>**Причина блокировки** — это причина, по которой пользователь не может совершать звонки.</li><li>**Заблокированное действие**  указывает, заблокирован или заблокирован ли пользователь при выполнении звонков по ТСОП в Teams.</li> <li>**Заблокированное время** — это дата и время (UTC), когда пользователю было заблокировано выполнение вызовов.</li></li> </ul>Чтобы нужные сведения отображались в таблице, добавьте в нее соответствующие столбцы. |
|**4**   |Нажмите **Изменение столбцов**, чтобы добавить или удалить столбцы в таблице.|
|**5**   |Выберите **"Полноэкранный** режим", чтобы просмотреть отчет в полноэкранном режиме.|
