# <a name="microsoft-teams-device-usage-report"></a>Отчет об использовании устройства Microsoft групп

Отчет об использовании групп устройств в группами Майкрософт & Скайп по центру администрирования Business предоставляет сведения о как пользователи подключаются к группам. Отчет можно использовать для просмотра устройств, которые используются в организации, включая сколько используйте команды из своих мобильных устройств, когда перейти на.  

![Снимок экрана отчета об использовании групп устройств в группами Майкрософт & Скайп по центру администрирования Business] (../media/teams-reports-device-usage.png "Снимок экрана отчета об использовании групп устройств в группами Майкрософт & Скайп по центру администрирования Business")

## <a name="view-the-report"></a>Просмотр отчета

1. Переход с группами Майкрософт & Скайп по центру администрирования бизнеса, на панели навигации слева щелкните **аналитики и отчетов**, а затем в разделе **отчета**, выберите **Использование групп устройств**. 
2. В разделе **диапазон дат**выберите диапазон и нажмите кнопку **выполнить отчет**.
