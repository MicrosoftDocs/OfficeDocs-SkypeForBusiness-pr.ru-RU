# <a name="upgrade-from-skype-for-business-online-to-teams"></a>Обновление от Скайп для бизнеса в Интернете по группам

Следуйте указаниям в этой статье, если полностью развернут Скайп для бизнеса в Интернет и требуется обновление пользователей с Скайп для бизнеса в группы. Можно обновить пользователей выборочно или файловый, основанным на обновление journey, ваша организация решила, назначив соответствующие сосуществования и обновления режима для пользователей.

> [!IMPORTANT]
> [!INCLUDE [upgrade-disclaimer](includes/upgrade-disclaimer.md)]

## <a name="assign-the-coexistence-and-upgrade-mode"></a>Назначение режима совместимости и обновления

Можно обновить пользователей, групп, назначение режим TeamsOnly TeamsUpgradePolicy, в которой может выполняться с помощью центра администрирования группами Майкрософт или Скайп для бизнеса удаленного сеанса Windows Powershell.

Для получения дополнительных сведений см [Задание вашей сосуществования и параметры обновления](https://aka.ms/SkypeToTeams-SetCoexistence) и [TeamsUpgradePolicy: управление миграция и сосуществование](migration-interop-guidance-for-teams-with-skype.md#teamsupgradepolicy-managing-migration-and-co-existence).

## <a name="upgrade-all-users-to-teams-at-one-time"></a>Обновление всех пользователей к группам за один раз

Выполните следующие действия для обновления всех пользователей к группам за один раз.

### <a name="step-1-notify-the-users-of-the-change"></a>Шаг 1: Уведомить пользователей об изменениях

1. В центре администрирования группами Майкрософт, выберите пункт **Параметры масштабе организации** > **Обновление группы**.
2. В разделе **режим совместимости**измените переключатель **Уведомить Скайп для бизнес-пользователей, что обновление группам доступен** **на**.

### <a name="step-2-set-the-coexistence-mode-for-the-users"></a>Шаг 2: Задайте режим совместной работы для пользователей

1. В центре администрирования группами Майкрософт выберите **Параметры на уровне организации**.
2. Выберите **Команды только** в режиме **сосуществования режим** раскрывающегося списка.

## <a name="upgrade-users-in-stages"></a>Обновление пользователей на стадии

Для постепенного обновления пользователей по группам, выполните приведенные далее действия.

### <a name="step-1-create-your-user-cohorts-for-the-upgrade"></a>Шаг 1: Создание вашего последователей пользователя для обновления

Последователей пользователя — это группы пользователей, которые будут перемещены в режим только для групп в то же время.

Для создания вашего пользовательского последователей (Добавить ссылку на страницу Выбор пользователя)

### <a name="step-2-set-the-user-mode-to-islands"></a>Шаг 2: Задайте режим пользователя о-ва

1. В центре администрирования группами Майкрософт Выбор **пользователей**и выберите cohort пользователя.
2. **Обновление группы**выберите **Изменить**.
3. На левой панели **Команд обновления** в **режиме сосуществования**выберите **острова** из раскрывающегося списка.

### <a name="step-3-set-notification-for-the-user-optional"></a>Шаг 3: Задайте уведомления для пользователя (необязательно)

1. В центре администрирования группами Майкрософт выберите **пользователей**и выберите cohort пользователя.
2. **Обновление группы**выберите **Изменить**.
3. На левой панели **Команд обновления** в разделе **режим совместимости**измените переключатель **Уведомлять Скайп для корпоративных пользователей** **на**.

### <a name="step-4-set-the-user-mode-to-teams-only"></a>Шаг 4: Задайте режим пользователей группам только

Режим сосуществования пользователя для групп только, когда все будет готово к обновлению пользователям использовать команды в качестве их только приложения.

1. В центре администрирования группами Майкрософт Выбор **пользователей**и выберите cohort пользователя.
2. **Обновление группы**выберите **Изменить**.
3. На левой панели **Команд обновления** в **режиме сосуществования**выберите **Группы только** из раскрывающегося списка.
