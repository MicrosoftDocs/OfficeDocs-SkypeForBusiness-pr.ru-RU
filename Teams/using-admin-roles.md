# <a name="use-microsoft-teams-administrator-roles-to-manage-teams"></a>Управление Microsoft Teams с помощью ролей администраторов

С помощью Azure Active Directory (Azure AD) можно назначить администраторов, которым требуются различные уровни доступа для управления Microsoft Teams. Администраторы могут управлять всей рабочей нагрузкой Teams или иметь делегированные разрешения для устранения неполадок с качеством звонков или управления потребностями телефонии вашей организации.

## <a name="teams-roles-and-capabilities"></a>Роли и возможности Teams

Доступно несколько ролей администратора Teams: администратор Teams, администратор связи Teams, специалист службы поддержки связи Teams, инженер службы поддержки связи Teams и администратор устройств Teams. Ознакомьтесь со следующей таблицей, чтобы понять, что может делать каждая роль и какие средства администратор может использовать в Центре администрирования Microsoft Teams и PowerShell.

> [!NOTE]
> Администраторы Skype для бизнеса Online могут управлять политиками приложений **Teams** и **Skype для бизнеса Online** с помощью PowerShell.

Чтобы следовать инструкциям, необходимо быть администратором. Инструкции по получении разрешений приведены в этой статье.

## <a name="assign-users-to-each-role"></a>Назначение пользователей каждой роли

Вы можете назначать пользователям эти роли в Azure AD. Сведения о назначении административных ролей пользователю в Azure AD см. в статье "Назначение роли администратора" [в Azure Active Directory](/azure/active-directory/fundamentals/active-directory-users-assign-role-azure-portal).

## <a name="cmdlets-available-for-each-role"></a>Командлеты, доступные для каждой роли

Большинство средств PowerShell для этих ролей администраторов находятся в модуле Teams PowerShell, и важно отметить, что некоторые командлеты, которые эти роли администратора имеют доступ к управлению общими параметрами, которые также используются для Skype для бизнеса Online. 

Чтобы просмотреть полный список командлетов, выполните следующие действия.

```powershell
Get-Command -Module MicrosoftTeams
 ```
