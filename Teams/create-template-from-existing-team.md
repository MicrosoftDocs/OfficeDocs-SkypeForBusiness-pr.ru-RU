<span data-ttu-id="99aa3-105">Microsoft Teams позволяет создать шаблон из существующей команды, который можно сохранить и изменить с учетом ваших организационных потребностей.</span><span class="sxs-lookup"><span data-stu-id="99aa3-105">Microsoft Teams lets you create a template from an existing team that can be saved and modified to meet your particular organizational needs.</span></span>

1. <span data-ttu-id="99aa3-106">Во sign in to the Teams admin center.</span><span class="sxs-lookup"><span data-stu-id="99aa3-106">Sign in to the Teams admin center.</span></span>

4. <span data-ttu-id="99aa3-109">Выберите **"Использовать существующую команду в качестве шаблона"** и выберите **"Далее".**</span><span class="sxs-lookup"><span data-stu-id="99aa3-109">Select **Use an existing team as a template**, and select **Next**.</span></span>

5. <span data-ttu-id="99aa3-111">Выберите команду, которая будет использовать в качестве отправной точки, выбрав рядом с ее названием и на этапе **"Далее".**</span><span class="sxs-lookup"><span data-stu-id="99aa3-111">Select a team to use as a starting point by selecting next to the team name, and then select **Next**.</span></span>

    - <span data-ttu-id="99aa3-115">Краткое и длинное описание шаблона</span><span class="sxs-lookup"><span data-stu-id="99aa3-115">Template short and long descriptions</span></span>
    - <span data-ttu-id="99aa3-116">Видимость локали</span><span class="sxs-lookup"><span data-stu-id="99aa3-116">Locale visibility</span></span>  
  
7. <span data-ttu-id="99aa3-117">В разделе **каналов, вкладок** и приложений добавьте все каналы и приложения, необходимые вашей команде.</span><span class="sxs-lookup"><span data-stu-id="99aa3-117">In the **channels, tabs, and apps** section, add any channels and apps that your team needs.</span></span>

    1. <span data-ttu-id="99aa3-118">В разделе **"Каналы"** выберите **"Добавить".**</span><span class="sxs-lookup"><span data-stu-id="99aa3-118">In the **Channels** section, select **Add**.</span></span>
    2. <span data-ttu-id="99aa3-119">В **диалоговом** окте "Добавление" привязайте название канала.</span><span class="sxs-lookup"><span data-stu-id="99aa3-119">In the **Add** dialog, name the channel.</span></span>
    3. <span data-ttu-id="99aa3-120">Добавьте описание.</span><span class="sxs-lookup"><span data-stu-id="99aa3-120">Add a description.</span></span>
    4. <span data-ttu-id="99aa3-121">Решите, должен ли канал показываться по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="99aa3-121">Decide if the channel should be shown by default.</span></span>
    5. <span data-ttu-id="99aa3-122">Найди имя приложения, которое вы хотите добавить на канал.</span><span class="sxs-lookup"><span data-stu-id="99aa3-122">Search for an app name that you want to add to the channel.</span></span>
    6. <span data-ttu-id="99aa3-123">Выберите **"Применить** по завершению".</span><span class="sxs-lookup"><span data-stu-id="99aa3-123">Select **Apply** when finished.</span></span>

8. <span data-ttu-id="99aa3-124">Чтобы **сохранить изменения,** выберите "Отправить".</span><span class="sxs-lookup"><span data-stu-id="99aa3-124">Select **Submit** to save your changes.</span></span>

<span data-ttu-id="99aa3-125">Новый шаблон отобразится в **списке шаблонов** группы.</span><span class="sxs-lookup"><span data-stu-id="99aa3-125">Your new template is displayed in the **Team templates** list.</span></span> <span data-ttu-id="99aa3-126">Шаблон можно использовать для создания команды в Teams.</span><span class="sxs-lookup"><span data-stu-id="99aa3-126">The template can be used to create a team in Teams.</span></span>
