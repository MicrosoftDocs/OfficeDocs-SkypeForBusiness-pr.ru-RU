# <a name="emails-sent-to-users-when-their-settings-change-in-microsoft-teams"></a>Сообщения электронной почты, отправляемые пользователям при изменении их настроек в Microsoft Teams

Сообщения электронной почты автоматически отправляются пользователям, которые [активированы для участия в аудиоконференциях](/SkypeForBusiness/audio-conferencing-in-office-365/set-up-audio-conferencing) с использованием Майкрософт в качестве поставщика услуг аудиоконференций.
  
По умолчанию существует четыре типа сообщений электронной почты, отправляемое для пользователей, которым разрешена для аудиоконференции. Тем не менее если необходимо ограничить число сообщений, отправляемых в пользователей, можно отключить его. Аудиоконференций в Office 365 отправит электронной почты для пользователей электронной почты, когда:
  
- **Им назначена лицензия аудиоконференций, или поставщик услуг аудиоконференций изменен на Майкрософт.**
    
     Это сообщение включает в себя идентификатор конференции, номер телефона конференции по умолчанию для собраний, аудиоконференций ПИН-код для пользователя и инструкции по и ссылки, будут использоваться для Business Online собрания обновить средство, которое используется для обновления существующих собраний для Скайп пользователь. В разделе [Назначение Скайп для бизнеса и группами Майкрософт лицензий](/SkypeForBusiness/skype-for-business-and-microsoft-teams-add-on-licensing/assign-skype-for-business-and-microsoft-teams-licenses) или [Назначить Microsoft в качестве поставщика аудиоконференций](/SkypeForBusiness/audio-conferencing-in-office-365/assign-microsoft-as-the-audio-conferencing-provider).
    
    > [!NOTE]
    > Если ваша организация была включена для динамического конференции идентификаторы, все собрания пользователя, которые они запланировать будут иметь конференции уникальные идентификаторы. Можно настроить [идентификаторы динамических звук конференц-связи в вашей организации](/skypeforbusiness/audio-conferencing-in-office-365/reset-a-conference-id-for-a-user). 
  
    Далее приведен пример этого сообщения электронной почты:
    
     ![Проверка лицензии Skype для бизнеса](media/audio-conferencing-user-enabled.png)
  
    Вы можете подробнее узнать о лицензировании в статье [Дополнительные лицензии для Skype для бизнеса и Microsoft Teams](/skypeforbusiness/skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing).
    
- **Изменяется идентификатор конференции и телефонный номер конференции по умолчанию для пользователя.**
    
    Это сообщение содержит идентификатор конференции, номер телефона конференции по умолчанию и инструкции по и ссылки для использования Скайп для бизнеса Online собрания обновить средство, которое используется для обновления существующих собраний для пользователя. Однако это сообщение не содержит аудиоконференций ПИН-код пользователя. Сброс ПИН-кода организатора конференции
  
    Далее приведен пример этого сообщения электронной почты:
    
     ![Изменились сведения конференц-связи с телефонным подключением.](media/audio-conferencing-info-change.png)
  
- **Выполнен сброс ПИН-кода аудиоконференций для пользователя.**
    
    Это сообщение содержит организатора аудиоконференций ПИН-код, существующий идентификатор конференции и номер телефона для конференции по умолчанию для пользователя. В разделе [Сброс аудиоконференций ПИН-кода](reset-the-audio-conferencing-pin-in-teams.md).
    
  
    Далее приведен пример этого сообщения электронной почты:

- **Удалена лицензия пользователя, или поставщик услуг аудиоконференций изменен с Майкрософт на другого поставщика или на значение «Нет».**
    
    Это происходит, когда лицензии **Аудиоконференций** удаляется от пользователя или при изменении поставщика аудиоконференций пользователя от корпорации Майкрософт для поставщика аудиоконференций сторонних производителей или при установке поставщика значение **None**. Это сообщение содержит инструкции и сведения для пользователя использовать Скайп для бизнеса Online средство обновления собрания для удаления аудиоконференций детальную информацию, например по умолчанию идентификатор конференции телефона номер или конференции.
    
    См. статью [Назначение и удаление лицензий для Office 365 для бизнеса](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc).

## <a name="make-changes-to-the-email-messages-that-are-sent-to-them"></a>Изменение отправляемых сообщений электронной почты

Можно внести изменения в сообщение электронной почты, которое отправляется пользователям, включая адрес электронной почты и отображаемое имя, включенную в контактных сведений *из* автоматически. По умолчанию отправителя сообщения электронной почты будут из Office 365, но можно изменить адрес электронной почты и отображаемое имя, с помощью Windows PowerShell. В разделе [Справочник по Microsoft команды PowerShell](https://docs.microsoft.com/powershell/module/teams/?view=teams-ps) для получения дополнительных сведений.
  
## <a name="what-if-you-dont-want-email-to-be-sent-to-them"></a>Что делать, если отправлять сообщение не требуется?

При отключении отправки сообщения электронной почты для пользователей электронной почты не будут отправляться даже в том случае, если пользователь получает назначена лицензия. В данном случае идентификатор конференции по умолчанию номер телефона конференц-связи, а важнее, их аудиоконференций ПИН-код не будут отправляться на пользователя. В этом случае пользователю необходимо предоставить, отправив им отдельные электронной почты или путем вызова их.
  
По умолчанию сообщения отправляются пользователям, но вы можете отключить сообщения по аудиоконференциям с помощью Microsoft Teams или Windows PowerShell. 

![teams-logo-30x30.png](media/teams-logo-30x30.png) **С помощью Microsoft Teams и Центра администрирования Skype для бизнеса**

1. На панели навигации слева перейдите в раздел **Собрания** > **Мосты конференц-связи**. 

2. В верхней части страницы **Мосты конференц-связи** щелкните **Настройки моста**. 

3. В области **Настройки моста** включите или отключите настройку **Автоматически отправлять пользователям электронные письма при изменении их настроек телефонного подключения**.

4. Нажмите кнопку **Сохранить**.
