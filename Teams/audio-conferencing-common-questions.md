# <a name="audio-conferencing-common-questions"></a>Общие вопросы об аудиоконференциях

Ниже перечислены некоторые популярные вопросы, которые мы получаем от наших пользователей, которые хотят использовать голосовую конференцию. 
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>Каковы преимущества голосовой конференции?

Звонить в собраниях очень удобно, если люди находятся в пути, а не могут принять участие в собрании с помощью приложения Skype для бизнеса или Microsoft Teams на своем ноутбуке или на мобильных устройствах. Но существуют и другие сценарии, в которых для участия в собрании Skype для бизнеса или Microsoft Teams может быть удобнее использовать приложение на компьютере.
  
- Подключение к Интернету ограничено.
    
- Собрание — это только звук.
    
- Кто-то попытался присоединиться к собранию Skype для бизнеса, и это не удалось.
    
- Качество связи лучше при наборе номера.
    
- Пользователи могут присоединиться к собранию "руки бесплатно" с помощью устройств Bluetooth.
    
- Поиск людей проще и удобнее в своей ситуации.
    
## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>Кто может принять участие в собрании по голосовой конференции? И кто может слышать?

Любой пользователь с номером телефона и ИДЕНТИФИКАТОРом Конференции может присоединиться к собранию Skype для бизнеса или Microsoft Teams, если собрание не будет заблокировано организатором собрания.
  
Независимо от того, используете ли вы телефон или приложение Skype для бизнеса или Microsoft Teams, вы сможете прослушать всех других участников звонка, и они смогут слышать вас. Организатор собрания имеет возможность отключить Микрофоны участников собрания, если они не будут слышать. 
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>Можно ли добавить бесплатный номер для пользователей конференц-связи?

Да, бесплатные телефонные номера (номера услуг) доступны, но только в некоторых странах и регионах. Список доступных номеров можно найти в разделе [доступность страны и региона для голосовой конференции и планов звонков](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>Сколько локальных номеров телефонного подключения в настоящее время поддерживается?

Вы можете присвоить вам местные номера для телефонного подключения, которые назначены вам при приобретении лицензий для голосовой конференции. Номера для телефонного подключения будут включены в приглашение на собрание. Эти местные номера будут доступны только вашей организации. Телефон, назначенный вашей организации, и этот номер предоставляется пользователям в пределах организации, которым разрешено выполнять голосовую конференцию. Таким образом, собрания Skype для бизнеса или Microsoft Teams, запланированные пользователем A, и другим пользователем B будут иметь одинаковый номер телефонного подключения.
  
Локальные номера для телефонного подключения, а также в некоторых случаях международные номера для телефонного подключения из страны, где находится ваша организация, будут включены в приглашение на собрание. Если участник собрания использует другой номер, включенный в приглашение, это будет общий номер телефона.

## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>Можно ли настроить местные номера для голосовой конференции из других городов в стране?

Если номера телефонов для голосовой связи не будут доступны в вашем регионе или вы не отвечаете требованиям вашей организации, пожалуйста, отправьте нам отзыв на [форумах форумах](http://www.skypefeedback.com/forums/299910--preview/category/119971-pstn-conferencing).
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>Какова максимальная длина собраний по голосовой конференции?

Максимальная продолжительность времени зависит от того, кто находится в собрании, и от типа проверки подлинности, с помощью которого вы присоединяется к собранию.
  
|**Участники собрания**|**Время окончания собрания**|
|:-----|:-----|
|Есть пользователи, которые присоединились к собранию с помощью приложения Skype для бизнеса или Microsoft Teams.  <br/> |Собрание завершается, если вы не изменяете список участников в течение 24 часов.  <br/> |
|Все пользователи будут набираться в собрании, но для ввода собрания кто-то использовал PIN.  <br/> |Собрание закончится через 24 часа.  <br/> |
|Все пользователи будут набираться в собрании, но никто, кто использовал ПИН-код для входа в собрание, не потратили.  <br/> |Собрание закончится через 4 часа.  <br/> |
   
## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>Сколько всего участников телефонной связи можно использовать в собраниях?

Голосовые конференции допускают до 250 мобильных участников.
  
Чтобы узнать о предельных значениях для собраний, ознакомьтесь с [ограничениями Skype для бизнеса Online](https://technet.microsoft.com/library/skype-for-business-online-limits.aspx#bkmk_Meeting_LyncOnlineLimits).

Дополнительные сведения о том, как отключить его, можно найти в статье [Включение и отключение отправки сообщений электронной почты при изменении параметров голосовой конференции в Microsoft Teams](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md) , [Включение и отключение отправки сообщений при изменении параметров голосовой конференции в Skype для бизнеса Online](/SkypeForBusiness/audio-conferencing-in-office-365/enable-or-disable-sending-emails-when-their-settings-change).
  
## <a name="can-audio-conferencing-be-used-by-the-users-who-are-part-of-an-on-premises-deployment-of-skype-for-business-server"></a>Можно ли использовать голосовые конференции для пользователей, которые являются частью локального развертывания Skype для бизнеса Server?

С [собраниями в первую очередь](meetings-first.md)пользователи Skype для бизнеса Server могут получить доступ к собраниям Teams и голосовой конференции Teams. 
  
## <a name="can-a-user-get-a-personal-conference-id"></a>Может ли пользователь получить личный идентификатор конференции?

Пользователи Skype для бизнеса и Microsoft Teams будут случайным образом назначили идентификаторы конференц-связи, и вы не можете зарезервировать или задать статический идентификатор конференции, который они могут использовать. 
  
## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>Можно ли использовать голосовые конференции в трансляции собраний Skype?

В настоящее время пользователи, которым разрешено проводить голосовую конференцию, не поддерживают присоединение к трансляции собраний Skype.
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>Может ли пользователь получить помощь оператора во время собрания?

Нет, пользователь не может получить помощь оператора или поддержку, нажав * 0 во время собрания. Если у вас возникли проблемы с голосовой Конференцией, администратор организации может обратиться в [службу поддержки Майкрософт для Office 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>Как пользователь может получить доступ к Конференции или изменить ее идентификатор?

Пользователи Skype для бизнеса или Microsoft Teams могут найти назначенный им идентификатор конференции, запланируя собрание в Outlook и Outlook в Интернете. Кроме того, пользователи могут найти идентификатор конференции в сообщении электронной почты, которое будет отправляться после настройки.
  
> [!NOTE]
> Пользователи не смогут сбросить идентификатор конференции. Идентификатор конференции может быть сброшен только вами, [администратором](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)для Организации. 
  
Мы работаем над решением, которое позволит пользователям получать доступ и сбрасывать идентификатор конференции без помощи администратора организации.
  
## <a name="how-does-a-user-access-or-change-hisher-pin"></a>Как пользователь может получить доступ к своему PIN-коду или изменить его?

Пользователи Skype для бизнеса или Microsoft Teams могут найти PIN-код в сообщении электронной почты, которое будет отправлено после их настройки.
  
> [!NOTE]
> Пользователи Skype для бизнеса или Microsoft Teams не смогут сбросить свой ПИН-код. ПИН-код можно сбросить только с помощью администратора. При сбросе PIN-кода пользователю отправляется сообщение электронной почты. 
  
Мы работаем над решением, которое позволит пользователям получать доступ и сбрасывать ПИН-код без помощи администратора организации.

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-skype-for-business-or-microsoft-teams-meeting"></a>Может ли участник звонить по международным телефонным номерам, когда они находятся в собрании Skype для бизнеса или Microsoft Teams?

Да, участники могут звонить на международную линию и пригласить других абонентов на собрание Skype для бизнеса или Microsoft Teams. Ознакомьтесь с [разбираемым набором из собрания Microsoft Teams, чтобы другие люди могли присоединиться к ней](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md) и [звонить из собрания Skype для бизнеса Online, чтобы другие люди могли присоединиться к ней](/SkypeForBusiness/audio-conferencing-in-office-365/dialing-out-from-a-meeting-so-other-people-can-join-it).
  
## <a name="how-does-a-skype-for-business-or-microsoft-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Как в Skype для бизнеса или Microsoft Teams вы планируете собрание с помощью сведений о собраниях голосовой конференции?

Если пользователю назначена лицензия на **голосовую конференцию** , а пользователь создает новое собрание Skype для бизнеса или Microsoft Teams в Outlook или Outlook в Интернете, телефонные номера для телефонного подключения и идентификаторы конференций будут добавлены в приглашение на собрание автоматически.
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>Как пользователь запланирует и запустит собрание, когда все участники будут использовать телефон для коммутируемого подключения?

Планирование собраний, которые будут присоединены ко всем участникам, использующих телефон для коммутируемого сетевого собрания, не отличается от их расписания. Тем не менее, для начала собрания, на котором все участники используют телефон для подключения, существует два способа.

- **Параметр #1**: по умолчанию, если организатор собрания и все участники присоединяется к собранию с помощью телефона, организатору собрания нужно ввести ПИН-код голосовой связи, чтобы запустить его. Вызывающие абоненты получают запрос на подтверждение подлинности организатора собрания, когда они набирает номер телефона для собрания по сети. Все участники, присоединяющиеся к собранию с помощью телефонного подключения до начала организатора, будут помещены в зал ожидания и прослушивать музыку на удержании. Для собраний Skype для бизнеса после того как организатор запустит его с помощью PIN-кода, вы сможете автоматически присоединиться к собранию. Для собраний Microsoft Teams участники будут присоединиться к собранию в соответствии со значением параметра автоматически допустить людей в политике собрания организатора.

- **Параметр #2**: Если для данного организатора выбран параметр "разрешить получателям, не прошедшим проверку подлинности", в качестве первых пользователей собрания (отключено по умолчанию), то все собрания, запланированные этим пользователем, смогут запуститься без ввода PIN-кода для голосовой связи. Если этот параметр включен, собрание начнется, как только первый участник присоединится к абоненту через телефонный номер, и он не будет помещен в зал ожидания. Дополнительные сведения можно найти в разделе [Управление параметрами голосовой конференции для пользователя в Microsoft Teams](manage-the-audio-conferencing-settings-for-a-user-in-teams.md) или [управлять параметрами голосовой конференции для пользователя в Skype для бизнеса Online](/SkypeForBusiness/audio-conferencing-in-office-365/manage-the-audio-conferencing-settings-for-a-user).
