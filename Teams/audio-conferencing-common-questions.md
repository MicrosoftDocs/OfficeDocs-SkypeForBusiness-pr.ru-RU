# <a name="audio-conferencing-common-questions"></a>Общие вопросы об аудиоконференциях

Ниже приведены некоторые из основных вопросов, которые мы получаем от наших клиентов, которые хотят использовать аудиоконференции.
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>Каковы преимущества аудиоконференций?

Приглашение на собрания полезно, например, когда люди находятся в дороге и не могут присутствовать на собрании с помощью приложения Майкрософт Teams на ноутбуке или мобильных устройствах. Но есть и другие сценарии, в которых использование телефона для участия в собрании Teams может быть лучшим вариантом, чем использование приложения на компьютере:
  
- Интернет-соединение ограничено.

- Собрание, только звук.

- Пользователь попытался присоединиться к собранию Teams, и оно не удалось.

- Качество звонка лучше, если они набирают.

- Люди могут присоединиться к собранию "без рук", используя устройства Bluetooth.

- Люди находят это проще и удобнее для своей ситуации.

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>Кто может присутствовать на собрании аудиоконференций? И кого я слышу?

Любой пользователь, имеющий номер телефона и идентификатор конференции, может присоединиться к собранию Teams, если организатор собрания не заблокировал собрание.
  
Независимо от того, звоните ли вы с помощью телефона или в приложениях Teams, вы сможете услышать всех остальных участников звонка, и они смогут услышать вас. Организатор собрания может отключить звук участников собрания, если они не хотят слышать их.
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>Можно ли добавить бесплатный номер для пользователей аудиоконференций?

Да, бесплатные номера телефонов (служебные номера) доступны, но только в некоторых странах и регионах. Список доступных номеров см. в разделе [Доступность по странам и регионам для аудиоконференций и планов звонков](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>Сколько локальных телефонных номеров поддерживается в настоящее время?

При покупке лицензий на аудиоконференции вам назначаются местные номера телефонного подключения. Номера телефонных номеров будут включены в приглашение на собрание. Эти местные номера будут доступны только вашей организации. Телефон, назначенный вашей организации, и этот номер предоставляется пользователям в этой организации, для которых включена аудиоконференция. Это означает, что собрания Teams, запланированные пользователем A и другим пользователем B, будут иметь одинаковый телефонный номер.
  
Местные номера телефонного номера, а также в некоторых случаях международные номера телефонного номера из страны, в которой находится ваша организация, будут включены в приглашение на собрание. Если участник собрания использует другой номер, включенный в приглашение, это будет общий номер телефона.
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Сколько международных телефонных номеров поддерживает аудиоконференция в Майкрософт 365 или Office 365?

Текущий список стран и регионов см. в разделе [Телефонные номера для аудиоконференций в Майкрософт Teams](phone-numbers-for-audio-conferencing-in-teams.md).
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>Какова максимальная продолжительность собраний аудиоконференций?

Максимальный период времени зависит от того, кто находится в собрании, и от типа проверки подлинности, используемой для присоединения к собранию.
  
|**Участники собрания**|**Время окончания собрания**|
|:-----|:-----|
|Пользователи присоединились к собранию с помощью приложения Teams или подключились к собранию.  <br/> |Собрание завершается, если по истечении 24 часов в списке участников нет изменений.  <br/> |
|Все пользователи получают доступ к собранию, но кто-то использует ПИН-код для входа в собрание.  <br/> |Собрание завершается через 24 часа.  <br/> |
|Все пользователи получают доступ к собранию, но никто не использовал ПИН-код для входа в собрание.  <br/> | Если анонимным пользователям разрешено начинать собрания, собрание завершается через четыре часа. Если анонимным пользователям не разрешено начинать собрания и все участники, прошедшие проверку подлинности, покинули собрание, собрание заканчивается через 90 минут после ухода последнего участника, прошедшего проверку подлинности. <br/> |

## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>Почему пользователи начали получать сообщения электронной почты с информацией о аудиоконференции?

Мы добавили новую функцию, которая позволяет [администратору](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) отправлять и обновлять сведения о аудиоконференциях и ПИН-коды в электронной почте. Дополнительные сведения об этом, в том числе об отключении, см. ## <a name="can-a-user-get-a-personal-conference-id"></a>Может ли пользователь получить личный идентификатор конференции?

Пользователям Teams будут случайным образом назначаться идентификаторы конференц-связи. Пользователи не могут зарезервировать или задать статический идентификатор конференции, который могут использовать только они.
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>Может ли пользователь получить помощь оператора во время собрания?

Нет, пользователь не может получить помощь или поддержку оператора, нажав клавишу *0 во время собрания. Если возникли проблемы с аудиоконференциями, администратор организации может обратиться [в службу поддержки Майкрософт для Office 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>Как пользователь получает доступ к идентификатору конференции или изменяет его?

Пользователи Teams могут найти назначенный им идентификатор конференции, запланируя собрание в Outlook и Outlook в Интернете. Кроме того, пользователи могут найти идентификатор конференции в сообщении электронной почты, которое будет отправлено им после настройки.
  
> [!NOTE]
> Пользователи не смогут сбросить свой идентификатор конференции. Идентификатор конференции может быть сброшен только [администратором](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504) организации.
  
Мы работаем над решением, которое позволит пользователю получить доступ к идентификатору конференции и сбросить его без помощи администратора организации.

## <a name="how-do-users-access-or-change-their-pin"></a>Как пользователи получают доступ к ПИН-коду или изменяют их?

Пользователи Teams могут найти ПИН-код в сообщении электронной почты, которое будет отправлено им после настройки.

> [!NOTE]
> Пользователи Teams могут изменять свои пин-коды. В нижней части приглашения на собрание есть ссылка для сброса пин-кода.
  
Мы работаем над решением, которое позволит пользователю получить доступ к ПИН-коду и сбросить его без помощи администратора организации.

Да, участники могут позвонить по всему миру и пригласить других абонентов в собрание Teams. См. [статью Подключение к собранию Майкрософт Teams, чтобы другие пользователи могли присоединиться к собранию](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md).
  
## <a name="how-does-a-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Как пользователь Teams планирует собрание с помощью сведений о собрании аудиоконференций?

Если пользователю назначена лицензия **на аудиоконференции** и пользователь создает новое собрание Teams в Outlook или Outlook в Интернете, номера телефонов с телефонным подключением и идентификаторы конференц-связи добавляются автоматически в приглашение на собрание.
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>Как пользователь запланирует и начинает собрание, когда все участники будут использовать телефон для набора номера?

Планирование собрания, к которому присоединяются все участники с помощью телефона для набора номера, ничем не отличается от планирования регулярного собрания по сети. Тем не менее, существует два способа начать собрание, на котором все участники используют телефон для набора:

- **Вариант 1**. По умолчанию, если организатор собрания и все участники присоединяются к собранию с помощью телефона, организатор собрания должен ввести ПИН-код аудиоконференции, чтобы начать его. При наборе номера телефона онлайн-собрания абонентам будет предложено пройти проверку подлинности в качестве организатора данного собрания. Все участники, которые присоединяются к собранию по телефону до начала организатора, будут размещены в вестибюле и будут слушать музыку при удержании. Для собраний Teams участники присоединяются к собранию в соответствии со значением параметра автоматического допуска пользователей в политике собрания организатора.

- **Вариант 2**. Параметр "Разрешить абонентам без проверки подлинности быть первыми людьми в собрании" по умолчанию отключен. Если этот параметр включен для определенного организатора, все собрания, запланированные этим организатором, можно начать до того, как организатор влоет ПИН-код аудиоконференции. Если этот параметр включен, собрание начнется, как только первый участник присоединится к нему по телефону с телефонным подключением, и первый участник не будет помещен в зал ожидания. Дополнительные сведения см. в статье [Управление параметрами аудиоконференций для пользователя в Майкрософт Teams](manage-the-audio-conferencing-settings-for-a-user-in-teams.md).
