# <a name="audio-conferencing-common-questions"></a>Общие вопросы об аудиоконференциях

Ниже верхней темой являются вопросы, которые мы получаем от наших клиентов, которые хотят использовать аудиоконференцию. 
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>Каковы преимущества аудиоконференции?

Участвовать в собраниях полезно, если люди находятся в пути и не могут участвовать в собрании с помощью приложения Skype для бизнеса или Microsoft Teams на ноутбуке или мобильном устройстве. Но есть и другие сценарии, в которых использование телефона для участия в собрании Skype для бизнеса или Microsoft Teams может быть лучшим вариантом, чем использование приложения на компьютере:
  
- Интернет-соединение ограничено.
    
- Собрание, только звук.
    
- Человек попытался присоединиться к встрече Skype для бизнеса, но это не удалось.
    
- Качество звонка будет лучше, если он позвонит по телефону.
    
- Люди могут присоединиться к собранию "без рук", используя устройства Bluetooth.
    
- Люди находят это проще и удобнее для своей ситуации.
    
## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>Кто может участвовать в собрании с аудиоконференцией? И кто может слышать?

Любой, у кого есть номер для телефонного номера и ид конференции, может присоединиться к собранию Skype для бизнеса или Microsoft Teams, если организатор собрания не заблокил собрание.
  
Позвоните ли вы с телефона или через приложения Skype для бизнеса или Microsoft Teams, вы сможете слышать других во время звонка и слышать вас. Организатор собрания может отключить звук для участников собрания, если они не хотят их слышать. 
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>Можно ли добавить бесплатный номер для пользователей аудиоконференции?

Да, бесплатные номера телефонов (номера служб) доступны, но только в некоторых странах и регионах. Список доступных номеров см. в списке стран и регионов, доступных для аудиоконференций [и планов звонков.](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>Сколько локальных номеров для телефонного дозвона в настоящее время поддерживается?

При приобретении лицензий на аудиоконференцию вам будут назначены местные номера для телефонного дозвона. Номера для телефонного дозвона будут включены в приглашение на собрание. Эти местные номера будут доступны только вашей организации. Телефон, который назначен вашей организации, и этот номер делиться пользователями в этой организации, которые могут включить аудиоконференцию. Таким образом, собрания Skype для бизнеса или Microsoft Teams, запланированные пользователем A и другим пользователем B, будут иметь одинаковый номер телефонного номера.
  
Местные номера для телефонного дозвона и (в некоторых случаях) международные номера из страны, в которой находится ваша организация, включаются в приглашение на собрание. Если участник собрания использует другой номер, включенный в приглашение, это будет общий номер телефона.
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Сколько международных номеров для телефонного дозвона поддерживает аудиоконференция в Microsoft 365 или Office 365?

Текущий список стран и регионов см. в номерах телефонов для аудиоконференций [в Microsoft Teams,](phone-numbers-for-audio-conferencing-in-teams.md) а номера телефонов для аудиоконференцов в Skype для [бизнеса Online.](/SkypeForBusiness/audio-conferencing-in-office-365/phone-numbers-for-audio-conferencing)
  
## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>Можно ли настроить местные номера для аудиоконференций из других городов в стране?

Если номера телефонов для аудиоконференций недоступны в вашем регионе или не соответствуют потребностям вашей организации, отправьте нам отзыв на [форумах SkypeFeedback.](http://www.skypefeedback.com/forums/299910--preview/category/119971-pstn-conferencing)
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>Какова максимальная продолжительность собраний с аудиоконференцией?

Максимальная продолжительность зависит от того, кто сейчас в собрании и какой тип проверки подлинности он использовал для присоединиться к собранию.
  
|**Участники собрания**|**Время окончания собрания**|
|:-----|:-----|
|Пользователи присоединились с помощью приложения Skype для бизнеса или Microsoft Teams или подключились к собранию по телефонной сети.  <br/> |Собрание закончится, если через 24 часа в списке участников не будет изменений.  <br/> |
|Все пользователи могут войти на собрание по телефонной сети, но кто-то использовал ПИН-код для входа в собрание.  <br/> |Собрание закончится через 24 часа.  <br/> |
|Все пользователи могут войти на собрание по телефонной сети, но никто не использовал ПИН-код для входа в собрание.  <br/> | Если анонимным пользователям разрешено начинать собрания, собрание закончится через четыре часа. Если анонимным пользователям запрещено начинать собрания и все участники, у которых есть проверка подлинности, покинули собрание, собрание закончится через 90 минут после того, как последний аутентификация завершился. <br/> |

## <a name="can-a-user-get-a-personal-conference-id"></a>Может ли пользователь получить личный ИД конференции?

Пользователям Skype для бизнеса и Microsoft Teams будут случайным образом назначены ИД для conferencing. Пользователи не могут резервировать или устанавливать статический ИД конференции, который могут использовать только они. 
  
## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>Можно ли использовать аудиоконференцию с трансляцией собрания Skype?

В настоящее время пользователям аудиоконференции не поддерживается возможность перейти к трансляции собрания Skype.
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>Может ли пользователь получить помощь от оператора во время собрания?

Нет, пользователь не может получить помощь или поддержку от оператора, нажав *0 во время собрания. При проблемах с аудиоконференцией администратор организации может обратиться в службу поддержки [Майкрософт по Office 365.](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>Как пользователь может получить доступ к конференции или изменить ее ID?

Пользователи Skype для бизнеса или Microsoft Teams могут найти назначенную им конференцию, запланируя собрание в Outlook и Outlook в Интернете. Кроме того, пользователь может найти его в сообщении электронной почты, которое будет отправлено им после его настроки.
  
> [!NOTE]
> Пользователи не смогут сбросить свой ИД конференции. Сбросить его может только администратор организации. [](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504) 
  
Мы работаем над решением, которое позволит пользователю получить доступ к и сбросить ИД конференции без помощи администратора организации.
  
## <a name="how-do-users-access-or-change-their-pin"></a>Как пользователи могут получить доступ к ПИН-коду или изменить его?

Пользователи Skype для бизнеса и Microsoft Teams могут найти ПИН-код в сообщении электронной почты, которое будет отправлено им после их настроки.
  
> [!NOTE]
> Пользователь Skype для бизнеса или Microsoft Teams не сможет сбросить СВОЙ ПИН-код. Сбросить ПИН-код может только администратор. При сбросе ПИН-кода пользователю отправляется сообщение электронной почты. 
  
Мы работаем над решением, которое позволит пользователю получить доступ к ПИН-коду и сбросить его без помощи администратора организации.

## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>Как пользователь может запланировать и начать собрание, когда все участники будут использовать телефон для телефонного звонка?

Планирование собрания, к которое присоединяется все участники с помощью телефона для телефонного звонка, не отличается от планирования обычного собрания по сети. Однако есть два способа начать собрание, на котором все участники используют телефон для телефонного звонка:

- **Вариант #1.** По умолчанию, если организатор и все участники присоединяются к собранию по телефону, организатору собрания необходимо ввести ПИН-код аудиоконференции, чтобы запустить его. При наборе телефонного номера онлайн-собрания звоня им будет предложено подтвердить подлинность в качестве организатора данного собрания. Все участники, которые присоединяются к собранию с помощью телефонного дозвона до начала собрания, помещаются в "вестибюль" и будут прослушивать музыку в удержании. После того как организатор начнет собрание Skype для бизнеса с помощью ПИН-кода для аудиоконференции, все участники в "вестибюле" автоматически присоединятся к собранию. Для собраний Microsoft Teams участники присоединяются к собранию в соответствии с значением параметра автоматического допуска участников в политике собрания организатора.

- **Параметр #2:** параметр "Разрешить неосторожным звонителям быть первыми пользователями на собрании" по умолчанию отключен. Если этот параметр включен для данного организатора, все собрания, запланированные этим организатором, можно начать, прежде чем организатор включит ПИН-код аудиоконференции. Если этот параметр включен, собрание начнется, как только первый участник присоединится к нему по телефонного номера для телефонного звонка, и первый участник не будет помещиться в "вестибюль". Дополнительные сведения см. в дополнительных сведениях: "Управление настройками аудиоконференций для пользователя в [Microsoft Teams"](manage-the-audio-conferencing-settings-for-a-user-in-teams.md) или "Управление настройками аудиоконференции для пользователя в [Skype для бизнеса Online".](/SkypeForBusiness/audio-conferencing-in-office-365/manage-the-audio-conferencing-settings-for-a-user)
