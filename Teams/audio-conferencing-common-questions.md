# <a name="audio-conferencing-common-questions"></a>Общие вопросы об аудиоконференциях

Ниже писуются некоторые наиболее часто задамые вопросы от наших клиентов, которые хотят использовать аудиоконференцию.
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>Каковы преимущества аудиоконференций?

Возможность звонков на собрания полезна, если люди находятся в пути и не могут участвовать в собрании с помощью приложения Skype для бизнеса или Microsoft Teams на ноутбуке или мобильном устройстве. Но есть и другие сценарии, в которых использование телефона для участия в собрании Skype для бизнеса или Microsoft Teams может быть лучшим вариантом, чем использование приложения на компьютере:
  
- Интернет-соединение ограничено.

- Собрание, только звук.

- Человек попытался присоединиться к встрече Skype для бизнеса, но это не удалось.

- Качество звонка будет лучше, если он позвонит по телефону.

- Люди могут присоединиться к собранию "без рук", используя устройства Bluetooth.

- Люди находят это проще и удобнее для своей ситуации.

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>Кто можете посетить собрание по аудиоконференции? А кто может слышать?

Любой, у кого есть номер для телефонного набора и ид конференции, может присоединиться к собранию Skype для бизнеса или Microsoft Teams, если организатор собрания не заблокил собрание.
  
Независимо от того, звоните ли вы с телефона, с помощью приложения Skype для бизнеса или Microsoft Teams, вы сможете слышать других во время звонка, и они могут слышать вас. Организатор собрания может "отключить" участников собрания, если они не хотят их слышать.
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>Можно ли добавить бесплатный номер для пользователей аудиоконференций?

Да, бесплатные номера телефонов (номера служб) доступны, но только в некоторых странах и регионах. Список доступных номеров см. в списке Доступность страны и региона для аудиоконференций и планов [звонков.](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>Сколько локальных номеров для телефонного набора в настоящее время поддерживается?

При приобретении лицензий на аудиоконференцию вам будут назначены локальные телефонные номера. Номера для телефонного набора будут включены в приглашение на собрание. Эти местные номера будут доступны только вашей организации. Телефон, который назначен вашей организации, и этот номер делиться пользователями в этой организации, которые могут включить аудиоконференцию. Таким образом, Skype для бизнеса или Microsoft Teams, запланированные пользователем A и другим пользователем B, будут иметь один и тот же номер телефонного номера.
  
Местные телефонные номера, а также в некоторых случаях международные телефонные номера из страны, в которой находится ваша организация, включаются в приглашение на собрание. Если участник собрания использует другой номер, включенный в приглашение, это будет общий номер телефона.
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Сколько международных телефонных номеров поддерживает аудиоконференция в Microsoft 365 или Office 365 аудиоконференцию?

Текущий список стран и регионов см. в Телефон номера для аудиоконференции в [Microsoft Teams](phone-numbers-for-audio-conferencing-in-teams.md) или Телефон для аудиоконференции в [Skype для бизнеса Online.](/SkypeForBusiness/audio-conferencing-in-office-365/phone-numbers-for-audio-conferencing)
  
## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>Можно ли настроить местные номера для аудиоконференций из других городов страны?

Если номера телефонов для аудиоконференций недоступны в вашем регионе или не соответствуют потребностям вашей организации, отправьте нам отзыв на [форумах SkypeFeedback.](http://www.skypefeedback.com/forums/299910--preview/category/119971-pstn-conferencing)
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>Каков максимальный размер собраний для аудиоконференций?

Максимальное время зависит от того, кто находится на собрании, и от типа проверки подлинности, который он использовал для присоединиться к собранию.
  
|**Участники собрания**|**Время окончания собрания**|
|:-----|:-----|
|Пользователи присоединились с помощью Skype для бизнеса или Microsoft Teams или присоединились к собранию по телефонной сети.  <br/> |Собрание завершается, если через 24 часа в списке участников не будет изменений.  <br/> |
|Все пользователи могут войти в собрание по номеру телефонного номера, но кто-то использовал ПИН-код для входа в собрание.  <br/> |Собрание закончится через 24 часа.  <br/> |
|Все пользователи могут принять участие в собрании по номеру телефонного номера, но никто не использовал ПИН-код для входа в собрание.  <br/> | Если анонимным пользователям разрешено начинать собрания, собрание закончится через четыре часа. Если анонимным пользователям запрещено начинать собрания, а все участники, авторификация которых прошла проверку подлинности, покинули собрание, собрание закончится через 90 минут после того, как покинул последний аутентификацию участника. <br/> |

## <a name="can-a-user-get-a-personal-conference-id"></a>Может ли пользователь получить личный ИД конференции?

Skype для бизнеса и Microsoft Teams пользователям будут случайным образом назначены ИД для conferencing. Пользователи не могут зарезервировать или установить статический ИД конференции, который могут использовать только они.
  
## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>Можно ли использовать аудиоконференцию с Skype трансляции собрания?

В настоящее время пользователям, включенным для аудиоконференции, не поддерживается Skype трансляции собрания.
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>Может ли пользователь получить помощь по операторам во время собрания?

Нет, пользователь не может получить помощь или поддержку от оператора, нажав *0 во время собрания. При проблемах с аудиоконференцией администратор организации может обратиться в службу поддержки [Майкрософт Office 365.](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>Как пользователь может получить доступ к своему ИД конференции или изменить его?

Skype для бизнеса или Microsoft Teams могут найти назначенную им ИД конференции, запланируя собрание в Outlook и Outlook в Интернете. Кроме того, пользователи могут найти его в сообщении электронной почты, которое будет отправлено им после их настроки.
  
> [!NOTE]
> Пользователи не смогут сбросить свой ИД конференции. Сбросить ИД конференции может только [](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)администратор организации.
  
Мы работаем над решением, которое позволит пользователю получить доступ к ИД конференции и сбросить его без помощи администратора организации.
  
## <a name="how-do-users-access-or-change-their-pin"></a>Как пользователи могут получить доступ к ПИН-коду или изменить его?

Skype для бизнеса или Microsoft Teams могут найти ПИН-код в сообщении электронной почты, которое будет отправлено им после их настроки.
  
> [!NOTE]
> Teams можете изменить пин-код. В нижней части приглашения на собрание есть ссылка для сброса пин-кода.
  
Мы работаем над решением, которое позволит пользователю получить доступ к ПИН-коду и сбросить его без помощи администратора организации.

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-skype-for-business-or-microsoft-teams-meeting"></a>Могут ли участники звонить на международные телефонные номера во время Skype для бизнеса или Microsoft Teams собрания?

Когда пользователю назначена  лицензия на аудиоконференцию и пользователь создает новое собрание Skype для бизнеса или Microsoft Teams в Outlook или Outlook в Интернете, номера телефонов для телефонного звонка и их удостоверения автоматически добавляются в приглашение на собрание.
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>Как пользователь запланировать и начать собрание, когда все участники будут использовать телефон для телефонного звонка?

Планирование собрания, к которое будут присоединяться все участники с помощью телефона для телефонного звонка, не отличается от планирования обычного собрания по сети. Однако существует два способа начать собрание, на котором все участники используют телефон для телефонного звонка:

- **Вариант 1.** Если организатор собрания и все участники присоединяются к собранию по телефону по умолчанию, для его запуска организатору необходимо ввести ПИН-код аудиоконференции. При наборе номера телефона онлайн-собрания вызывателям будет предложено подтвердить подлинность в качестве организатора заданного собрания. Все участники, которые присоединяются к собранию с помощью телефонного дозвона до начала собрания, помещаются в "lobby" и будут прослушивать музыку на удержании. Если Skype для бизнеса собрание начинается с ввода ПИН-кода для аудиоконференции, все участники в "вестибюле" автоматически присоединяются к собранию. Для Microsoft Teams собрания участники присоединяются к собранию в соответствии со значением параметра автоматического допуска людей в политике собрания организатора.

- **Вариант 2.** По умолчанию отключен параметр "Разрешить неавтоматическим звонителям быть первыми на собрании". Если этот параметр включен для заданного организатора, все собрания, запланированные этим организатором, можно начать до того, как организатор включит ПИН-код аудиоконференции. Если этот параметр включен, собрание начинается, как только первый участник присоединяется к нему по телефонным номерам для телефонного звонка, и первый участник не будет помещаться в "lobby". Дополнительные сведения см. в настройках аудиоконференции для пользователя в [Microsoft Teams](manage-the-audio-conferencing-settings-for-a-user-in-teams.md) Или Управление настройками аудиоконференции для пользователя в [Skype для бизнеса Online.](/SkypeForBusiness/audio-conferencing-in-office-365/manage-the-audio-conferencing-settings-for-a-user)
