# <a name="configure-operator-connect"></a>Настройка подключения к оператору

>[!NOTE]
>Оператор Connect в настоящее время доступен только в **режиме предварительного просмотра.** Общедоступный предварительный просмотр позволяет протестировать предстоящие возможности и предоставить отзывы. Функции, включенные в общедоступный предварительный просмотр, могут быть не завершены, могут быть внесены изменения и не поддерживаются в облачных службах Office 365 для государственных органов.

В этой статье описано, как настроить оператор Connect. Перед настройкой оператора Connect ознакомьтесь со сведениями о предварительных условиях и лицензировании в окне Планирование подключения к оператору. [](operator-connect-plan.md)

## <a name="enable-an-operator"></a>Включить оператор

Вы можете включить, изменить и удалить операторы в Центре администрирования Teams. В области навигации слева перейдите к voice **> Operators**.

Чтобы включить оператор:

1. **Выберите оператор.** На **вкладке Все операторы** отфильтруйте доступные операторы по регионам или службам, чтобы найти нужный оператор для ваших голосовых потребностей. Затем выберите оператор, который вы хотите включить.  

2. **Выберите страны.** В **области Параметры оператора** выберите страны, в которые вы хотите включить их.

3. **У вас есть контактные данные (необязательно).** Если вы хотите, чтобы операторы связывались с вами с дополнительными сведениями об операторе Connect, подумайте об этом и укажете свои контактные данные.  

4. **Примите уведомление о передаче данных.**

5. **Добавьте оператор.** Выберите **Добавить как оператор для** сохранения.

## <a name="set-up-phone-numbers"></a>Настройка номеров телефонов

То, как вы настроили номера телефонов, зависит от того, настраивали ли вы номера для новых пользователей или перемещали существующие номера из планов звонков Майкрософт или прямой маршрут.

- Если вам нужно получить номера телефонов для новых пользователей, см. приобретение номеров [для новых пользователей Teams.](#acquire-numbers-for-new-teams-users)

- Если вы хотите переместить существующие номера из планов звонков в оператор Connect, см. перемещение номеров из планов [звонков в оператор Connect.](#move-numbers-from-calling-plans-to-operator-connect)

- Если вы хотите переместить существующие числа из прямой маршрутии в оператор Connect, см. перемещение чисел из прямой [маршрутии](#move-numbers-from-direct-routing-to-operator-connect)в оператор Connect.

>[!IMPORTANT]
>**Адреса для экстренного устранения проблем:** Номера телефонов, связанные с адресами для экстренного обращения, управляются оператором. После создания адресов для обращения в экстренных службах в Центре администрирования Teams оператор назначит им номера телефонов. Чтобы внести изменения в адреса для экстренного звонков и их номера телефонов, обратитесь к своему оператору.

### <a name="acquire-numbers-for-new-teams-users"></a>Получение номеров для новых пользователей Teams

Чтобы получить номера для новых пользователей Teams, выполните следующие действия:

1. **Назначьте лицензию на телефонную систему.** Лицензию на телефонную систему можно назначить пользователям в Центре администрирования Microsoft 365 или с помощью PowerShell. Дополнительные сведения см. в том, как назначить пользователям лицензии на [надстройки Teams.](teams-add-on-licensing/assign-teams-add-on-licenses.md)

2. **Убедитесь, что вы в режиме только Teams.** Чтобы проверить, находится ли ваша организация в режиме только Teams, в Центре администрирования Teams перейдите к настройкам для всей организации **> обновления Teams.** Чтобы проверить, находится ли пользователь только в режиме Teams, перейдите в группу **Пользователи** и выберите учетную запись пользователя. На **вкладке Учетная** запись в области **Обновление Teams убедитесь,** что в режиме сосуществования установлено "Только Teams".

3. **Создание и проверка адресов для экстренного обращения.** В Центре администрирования Teams перейдите в > **адреса** для экстренного устранения экстренных ситуаций. Дополнительные действия см. в статьи Добавление, изменение и удаление местоположения для экстренного [устранения чрезвычайной ситуации для организации.](add-change-remove-emergency-location-organization.md)

4. **Приобрести номера.** Перейдите на веб-сайт оператора, чтобы заказать и приобрести номера телефонов. Список веб-сайтов-операторов см. в [списке Операторы.](#operators) Вам потребуется предоставить свой ИД клиента. Если вы не знаете свой ИД [](/onedrive/find-your-office-365-tenant-id) клиента, см. дополнительные сведения в этой области.

5. **Назначение чисел.** После завершения заказа оператор будет загружать номера в ваш клиент. Чтобы просмотреть номера и поставщика услуг в Центре администрирования Teams, переходить к голосовой > **номера телефонов**. Назначать номера пользователям можно в Центре администрирования Teams или с помощью PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)
 

### <a name="move-numbers-from-calling-plans-to-operator-connect"></a>Перемещение номеров из планов звонков в оператор Connect

1. Обратитесь к оператору для переноса номеров в оператор Connect. Чтобы [найти веб-сайт](#operators) оператора, см. операторы.

2. После завершения заказа на перенос номеров оператор может отозвать у пользователей телефонные номера планов звонков и удалить лицензию на план звонков. Затем оператор может отправить номера в ваш клиент.

3. Назначение операторов. Номера для подключения к пользователям можно использовать в Центре администрирования Teams или с помощью PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)

 
### <a name="move-numbers-from-direct-routing-to-operator-connect"></a>Перемещение чисел из прямой маршрутии в оператор Connect

1. Удалите у пользователя существующий номер телефона следующим образом:  

   Получите существующий URI on-prem Line путем запуска следующей команды PowerShell:

   ```
   Get-CsOnlineUser -Identity <user> | select OnPremLineURI 
   ```

   Удалите URI строки, выдворачив следующую команду PowerShell:  

   ```
   Set-CsUser -identity <user> - OnPremLineURI $null 
   ```

2. Удалите все ПСЗ, связанные с вашими пользователями, в противном случае звонки будут перенаправляемы на шлюз, указанный в окне "Использование ЗВОНКОВ по ННР". Чтобы узнать, как удалить использование ПСОП, см. [set-CsOnlinePstnUsage.](/powershell/module/skype/set-csonlinepstnusage?view=skype-ps)

3. Перейдите на веб-сайт оператора, чтобы заказать и приобрести номера телефонов. Список веб-сайтов-операторов см. в [списке Операторы.](#operators) Вам потребуется предоставить свой ИД клиента. Если вы не знаете свой ИД [](/onedrive/find-your-office-365-tenant-id) клиента, см. дополнительные сведения в этой области.

4. После завершения заказа оператор будет загружать номера в ваш клиент. Чтобы просмотреть номера и поставщика услуг в Центре администрирования Teams, переходить к голосовой > **номера телефонов**. Назначьте оператору Подключение номеров для пользователей через Центр администрирования Teams или с помощью PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)

   

### <a name="assign-numbers"></a>Назначение чисел

Независимо от того, добавляете ли вы новых пользователей Teams или перемещаете существующих пользователей в Operator Connect, для назначения чисел выполните следующие действия:

Чтобы назначить номера с помощью Центра администрирования Teams, перейдите к **номеру телефона**. Действия такие же, как назначение номеров для планов звонков. Дополнительные сведения см. в том, как назначить номер [телефона пользователю.](assign-change-or-remove-a-phone-number-for-a-user.md)

Чтобы назначить числа с помощью PowerShell, используйте Set-CsOnlineVoiceUser следующим образом:

```
Set-CsOnlineVoiceUser -Identity <user>  -TelephoneNumber <phone number> 
```

Например:

```
Set-CsOnlineVoiceUser -Identity john@contoso.com -TelephoneNumber +14255550101
```

Дополнительные сведения о назначении номеров телефонов пользователям см. в этой [записи.](assign-change-or-remove-a-phone-number-for-a-user.md)



## <a name="manage-your-operators"></a>Управление операторами

На вкладке Мои операторы можно просмотреть операторы и их состояние, а также внести следующие изменения в выбор:  

- Управление службами-операторами по стране
- Приостановка оператора
- Удаление оператора

> [!NOTE]
> Перед удалением оператора из организации или страны необходимо удалить все телефонные номера, которые назначены пользователям в организации или стране, и связаться с оператором, чтобы освободить эти номера.
