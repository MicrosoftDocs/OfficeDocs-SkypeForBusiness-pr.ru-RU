# <a name="configure-operator-connect"></a>Настройка Подключение

>[!NOTE]
>Оператор Подключение в настоящее время доступен только в **режиме предварительной версии.** Общедоступный предварительный просмотр позволяет протестировать предстоящие возможности и предоставить отзывы. Функции, включенные в общедоступный предварительный просмотр, могут быть не завершены, могут быть внесены изменения и не поддерживаются Office 365 для государственных организаций облаках.

В этой статье описано, как настроить Подключение. Перед настройкой Подключение операторов ознакомьтесь [](operator-connect-plan.md) со Подключение о предварительных условиях и лицензировании.

## <a name="enable-an-operator"></a>Включить оператор

Вы можете включить, изменить и удалить операторы в центре Teams администрирования. В области навигации слева перейдите к voice **> Operators**.

Чтобы включить оператор:

1. **Выберите оператор.** На **вкладке Все операторы** отфильтруйте доступные операторы по регионам или службам, чтобы найти нужный оператор для ваших голосовых потребностей. Затем выберите оператор, который вы хотите включить.  

2. **Выберите страны.** В **области Параметры оператора** выберите страны, в которые вы хотите включить их.

3. **У вас есть контактные данные (необязательно).** Если вы хотите, чтобы операторы связывались с вами с дополнительными сведениями об операторе Подключение, убедитесь в этом и укайте свои контактные данные.  

4. **Примите уведомление о передаче данных.**

5. **Добавьте оператор.** Выберите **Добавить как оператор для** сохранения.

## <a name="set-up-phone-numbers"></a>Настройка номеров телефонов

То, как вы настроили номера телефонов, зависит от того, настраивали ли вы номера для новых пользователей или перемещали существующие номера из планов звонков Майкрософт или прямой маршрут.

- Если вам нужно получить номера телефонов для новых пользователей, см. приобретение номеров для новых Teams [пользователей.](#acquire-numbers-for-new-teams-users)

- Если вы хотите переместить существующие номера из планов звонков в Подключение, см. перемещение номеров из планов звонков в [Подключение.](#move-numbers-from-calling-plans-to-operator-connect)

- Если вы хотите переместить существующие числа из прямой маршрутии в оператор Подключение, см. перемещение чисел из прямой маршрутии в [Подключение.](#move-numbers-from-direct-routing-to-operator-connect)

>[!IMPORTANT]
>**Адреса для экстренного обращения:** Адресами для экстренного обращения, связанными с номером, полученным от оператора, управляется непосредственно с помощью оператора. Чтобы внести изменения, обратитесь к оператору.

### <a name="acquire-numbers-for-new-teams-users"></a>Получение номеров для новых Teams пользователей

Чтобы получить номера для новых Teams пользователей, выполните следующие действия:

1. **Назначьте телефонная система лицензию.** Вы можете назначить пользователям телефонная система лицензию в Центре администрирования Microsoft 365 или с помощью PowerShell. Дополнительные сведения см. в Teams лицензий на [надстройки для пользователей.](teams-add-on-licensing/assign-teams-add-on-licenses.md)

2. **Убедитесь, что вы в режиме TeamsOnly.** Чтобы проверить это, Teams центре администрирования перейдите к настройкам для **всей организации > Teams обновления**. В режиме сосуществования должно быть за установлено Teams.

3. **Создание и проверка адресов для экстренного обращения.** В центре Teams перейдите в  > адреса для экстренного обращения, чтобы настроить адреса для экстренного устранения. Дополнительные действия см. в статьи Добавление, изменение и удаление местоположения для экстренного [устранения чрезвычайной ситуации для организации.](add-change-remove-emergency-location-organization.md)

4. **Приобрести номера.** Перейдите на веб-сайт оператора, чтобы заказать и приобрести номера телефонов. Список веб-сайтов-операторов см. в [списке Операторы.](#operators) Вам потребуется предоставить свой ИД клиента. Если вы не знаете свой ИД [](/onedrive/find-your-office-365-tenant-id) клиента, дополнительные сведения см. в Microsoft 365 ИД клиента.

5. **Назначение чисел.** После завершения заказа оператор будет загружать номера в ваш клиент. Вы можете просмотреть номера и поставщика услуг в Центре администрирования Teams, переходить к голосовым **> Телефон номерам**. Назначать номера пользователям можно в центре Teams или с помощью PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)
 

### <a name="move-numbers-from-calling-plans-to-operator-connect"></a>Перемещение номеров из планов звонков в Подключение

1. Обратитесь к оператору для переноса номеров в Подключение. Чтобы [найти веб-сайт](#operators) оператора, см. операторы.

2. После завершения заказа на перенос номеров оператор может отозвать у пользователей телефонные номера планов звонков и удалить лицензию на план звонков. Затем оператор может отправить номера в ваш клиент.

3. Назначьте Подключение с помощью центра администрирования Teams или PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)

 
### <a name="move-numbers-from-direct-routing-to-operator-connect"></a>Перемещение чисел из прямой маршрутии в Подключение

1. Удалите у пользователя существующий номер телефона следующим образом:  

   Получите существующий URI строки, вы можете использовать следующую команду PowerShell:

   ```
   Get-CsOnlineUser -Identity <user> | select OnPremLineURI 
   ```

   Удалите URI on-prem Line путем запуска следующей команды PowerShell:  

   ```
   Set-CsUser -identity <user> - OnPremLineURI $null 
   ```

2. Удалите все ПСЗ, связанные с вашими пользователями, в противном случае звонки будут перенаправляемы на шлюз, указанный в окне "Использование ЗВОНКОВ по ННР". Чтобы узнать, как удалить использование ПСОП, см. [set-CsOnlinePstnUsage.](/powershell/module/skype/set-csonlinepstnusage?view=skype-ps)

3. Перейдите на веб-сайт оператора, чтобы заказать и приобрести номера телефонов. Список веб-сайтов-операторов см. в [списке Операторы.](#operators) Вам потребуется предоставить свой ИД клиента. Если вы не знаете свой ИД [](/onedrive/find-your-office-365-tenant-id) клиента, дополнительные сведения см. в Microsoft 365 ИД клиента.

4. После завершения заказа оператор будет загружать номера в ваш клиент. Вы можете просмотреть номера и поставщика услуг в Центре администрирования Teams, переходить к голосовым **> Телефон номерам**. Назначьте Подключение с помощью центра администрирования Teams или PowerShell. Дополнительные сведения см. в [том, как назначить номера.](#assign-numbers)

   

### <a name="assign-numbers"></a>Назначение чисел

Независимо от того, добавляете ли вы новых Teams пользователей или перемещаете существующих пользователей в Подключение, для назначения чисел выполните следующие действия:

Чтобы назначить номера в Центре Teams администрирования, перейдите к Телефон **номеров**. Действия такие же, как назначение номеров для планов звонков. Дополнительные сведения см. в записи Назначение номера телефона [пользователю.](assign-change-or-remove-a-phone-number-for-a-user.md)

Чтобы назначить числа с помощью PowerShell, используйте Set-CsOnlineVoiceUser следующим образом:

```
Set-CsOnlineVoiceUser -Identity <user>  -TelephoneNumber <phone number> 
```

Например:

```
Set-CsOnlineVoiceUser -Identity john@contoso.com -TelephoneNumber +14255550101
```

Дополнительные сведения о назначении номеров телефонов пользователям см. в этой [записи.](assign-change-or-remove-a-phone-number-for-a-user.md)



## <a name="manage-your-operators"></a>Управление операторами

На вкладке Мои операторы можно просмотреть операторы и их состояние, а также внести следующие изменения в выбор:  

- Управление службами-операторами по стране
- Приостановка оператора
- Удаление оператора

> [!NOTE]
> Перед удалением оператора из организации или страны необходимо удалить все телефонные номера, которые назначены пользователям в организации или стране, и связаться с оператором, чтобы освободить эти номера.
