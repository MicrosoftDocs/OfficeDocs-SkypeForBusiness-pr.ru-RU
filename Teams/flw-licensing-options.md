# <a name="understand-frontline-worker-user-types-and-licensing"></a>Общие сведения о типах пользователей и лицензировании сотрудников без компьютеров

Организации во многих отраслях, таких как розничная торговля, производство, гостиничный бизнес и т. д., используют разные типы пользователей:

- Информационные работники, у которых, скорее всего, есть настольный компьютер или ноутбук. Они чаще работают над документами, а не непосредственно с клиентами, например сотрудники главного офиса.
- Сотрудники без компьютеров, которые часто работают на планшетах или телефонах и взаимодействуют непосредственно с клиентами и населением. Они предоставляют услуги, поддержку и продают продукты либо являются сотрудниками, непосредственно участвующими в производстве и распространении продуктов и услуг. Например, сотрудники розничной торговли, медицинские работники и медсестры, работники фабрик и т. д.

В Microsoft 365 и Microsoft Teams доступны функции и модель лицензирования для поддержки обоих типов пользователей в организации. [Microsoft 365 для сотрудников без компьютеров](https://www.microsoft.com/microsoft-365/enterprise/frontline) оптимизирован для мобильных рабочих ресурсов, которые в основном взаимодействуют с клиентами, но также должны оставаться на связи с остальными сотрудниками вашей организации.

## <a name="assign-appropriate-licenses-to-support-frontline-worker-and-information-worker-user-types"></a>Назначение соответствующих лицензий для поддержки пользовательских типов сотрудников без компьютеров и информационных работников

Если это применимо к вашей организации, вы можете назначать разным типам пользователей разные лицензии. На следующем рисунке показан один из способов применения лицензий и политик для сотрудников розничной торговли, например:

:::image type="content" source="./media/flw-license-options.png" alt-text="Сочетание типов лицензий для поддержки различных ролей в организации. Используйте корпоративные лицензии для информационных сотрудников, например сотрудников главного офиса, и лицензии сотрудников без компьютеров для мобильных рабочих ресурсов, таких как менеджеры магазинов, продавцы, сотрудники доставки, работники склада и сборщики.":::

В этом примере можно использовать разные типы лицензий для двух типов сотрудников организации, как в следующей таблице.

| Сотрудникам без компьютеров требуется быстрый доступ к инструментам, помогающим взаимодействовать с клиентами: | Информационным работникам необходимы инструменты для поддержки их задач, например: |
| ----- | ----- |
| Сотрудникам без компьютеров требуется доступ к информации на ходу. У них есть приложения Office на телефонах или планшетах. | Информационные работники создают, редактируют и рассылают документы. У них есть приложения Office на настольных компьютерах и ноутбуках, а также мобильные приложения на телефонах и планшетах. |
| Сотрудники без компьютеров взаимодействуют на ходу с помощью чата Teams, планируют работу с помощью приложения "Смены", вычеркивают выполненные элементы задач с помощью приложения Tasks и отслеживают их в Списках, а также применяют другие функции Teams.  | Информационные работники общаются в Teams с помощью чата, собраний и звонков, а также используют другие приложения Teams (в зависимости от конкретной лицензии). |
| Сотрудники без компьютеров могут находить ресурсы, средства коммуникации и учебные курсы для всей организации с помощью SharePoint, OneDrive, Yammer и Stream. | Информационные работники создают и используют ресурсы и средства коммуникации команды или всей организации с включенными службами Microsoft 365 (в зависимости от конкретной лицензии). |

Подробное сравнение содержимого различных лицензий Microsoft 365 см. в [таблице сравнения](https://go.microsoft.com/fwlink/?linkid=2139145).
