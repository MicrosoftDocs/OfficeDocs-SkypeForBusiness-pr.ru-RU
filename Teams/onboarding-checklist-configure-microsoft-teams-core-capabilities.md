# <a name="configure-microsoft-teams-core-capabilities"></a>Настройка групп Майкрософт основными возможностями

|----|-----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1  | Проверить, что среды включает в себя все необходимые компоненты групп | Команды зависит от других платформы для построения решения для совместной работы с начала до конца приложения. Работа с ИТ-группы чтобы гарантировать правильность развертывания и правильность настройки Exchange, SharePoint Online и OneDrive для бизнеса. | | [Взаимодействие SharePoint Online и OneDrive для бизнеса с Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/sharepoint-onedrive-interact) <br/><br/>[Взаимодействие Exchange и Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/exchange-teams-interact) |
| 2  | Проверьте, что команды был включен для клиента | Группы включена по умолчанию для всей организации. Проверьте странице **Services и надстроек** в центре администрирования Office 365, чтобы убедиться, что группы включена для вашего клиента и включить его, если это необходимо. | | [Настройка Microsoft Teams в вашей организации Office 365](https://docs.microsoft.com/MicrosoftTeams/office-365-set-up) |
| 3  | Настройка ролей и разрешений | Команды поддерживает два типа ролей: участников и владельцев. <br/><br/>После добавления члена группы, владельца можно также повысить уровень член роли владельца. Рекомендуется мы рекомендуем, что у вас есть по крайней мере два владельцев, назначенный для каждой группы. <br/><br/>По умолчанию всем пользователям в организации, который имеет почтового ящика, размещенного на Exchange Online можно создать группу. Пользователь, который создает новые группы автоматически предоставляются роли владельца для этой группы. <br/><br/>Если требуется, можно настроить параметры группы Office 365 позволяет только определенным пользователям создавать новые группы. | | [Назначение ролей и разрешений в Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/assign-roles-permissions) <br/><br/>[Office 365 групп и группами Майкрософт](https://docs.microsoft.com/MicrosoftTeams/office-365-groups) <br/><br/>[Управление пользователей, которые можно создавать группы Office 365](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618) |
| 4  | Настройте параметры группы на уровне клиента | Можно настроить некоторые параметры группы на уровне клиента. Пользователей, которым разрешена для групп наследуют эти параметры из конфигурации клиента:<ul><li>Общие</li><li>Интеграция электронной почты</li><li>Приложения</li><li>Настраиваемое облачное хранилище</li><li>Звонки и собрания</li><li>Обмен сообщениями</li></ul>| | [Включение функций Microsoft Teams в организации Office 365](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365) |
| 5  | Необязательно: Настройка доступ в качестве гостя | Использование гостевого доступа в группах для совместной работы пользователей за пределами организации, предоставив им доступ к группам и каналы. Доступ в качестве гостя устанавливается уровне клиента в группах. По умолчанию он отключен. <br/>Включить доступ в качестве гостя и настроить параметры гостевой всей клиента, если ваша организация планирует применять этот компонент. | | [Гостевой доступ в Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/guest-access) |
| 6  | Необязательно: Настройка политики именования групп | Команды использует политики назначения имен для Office 365 групп при создании или изменении названия групп. <br/><br/>По умолчанию не ограничения именования, применяются при создании пользователем команды. <br/><br/>Если требуется добавить правила для имен групп, настройте группы Office 365, именование политик, которые применяются к вашей организации. Можно задать обязательные префиксы и суффиксы и указать заблокированные слова. | | [Планирование использования групп Office 365 при создании команд в Microsoft Teams](https://docs.microsoft.com/microsoftteams/plan-office-365-groups) <br/><br/>[Политику именования групп с Office 365](https://support.office.com/article/Office-365-Groups-naming-policy-6ceca4d3-cad1-4532-9f0f-d469dfbbb552) |
| 7  | Настройка Exchange для домена SMTP группы | Команды используют Exchange Online для отправки уведомления участников группы с помощью домена SMTP — email.teams.microsoft.com, когда они были добавлены или удалены. <br/><br/>Не забудьте добавить этот домен SMTP в список обслуживаемых доменов в инфраструктуре Exchange. | | [Добавление домена SMTP Microsoft Teams в качестве обслуживаемого в Exchange Online](https://docs.microsoft.com/MicrosoftTeams/smtp-accepted-domain) |
| 8  | Настройка и управление доступом пользователей к группам | Хотя мы настоятельно рекомендуем разрешить всем пользователям для групп, можно разрешить или запретить доступ к группам на уровне пользователя, назначение или удаление группы лицензии. | | [Управление доступом пользователей к Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/user-access) |
| 9  | Назначение лицензий пользователям | Назначение лицензий пользователям для таких функций, как аудиоконференции, телефонной системой и вызов планы | | [Назначение лицензий Skype для бизнеса и Microsoft Teams](https://docs.microsoft.com/SkypeForBusiness/skype-for-business-and-microsoft-teams-add-on-licensing/assign-skype-for-business-and-microsoft-teams-licenses) <br/><br/>[MyAdvisor – сценариев Включение пользователя](https://myadvisor.fasttrack.microsoft.com/CloudVoice/Downloads?SelectedIDs=5_2_0_6,5_2_0_3) |
