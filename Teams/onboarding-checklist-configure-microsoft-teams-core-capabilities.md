Она по умолчанию отключена. <br/>В этом случае вы можете включить гостевой доступ и настроить параметры гостя для всего клиента, если в вашей организации планируется использовать эту функцию. | | [Гостевой доступ в Microsoft Teams](guest-access.md) |

| 7  | Настройка Exchange для домена TEAMS SMTP | Teams использует Exchange Online для отправки уведомлений участникам группы с помощью домена SMTP email.teams.microsoft.com , когда они были добавлены или удалены. <br/><br/>Не забудьте добавить этот SMTP-домен в список принятых доменов в Exchange инфраструктуре. | | [Создание списков надежных отправников в Exchange](/microsoft-365/security/office-365-security/create-safe-sender-lists-in-office-365?view=o365-worldwide) |
