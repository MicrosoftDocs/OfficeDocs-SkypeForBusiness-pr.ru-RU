# <a name="configure-the-skype-meetings-app-to-work-with-teams"></a>Настройка приложения "Собрания Skype" для работы с Teams

После обновления пользователя до Microsoft Teams администраторы могут использовать центр администрирования Microsoft Teams, чтобы указать предпочтительное приложение, которое будет применяться для Skype для бизнеса собраний.

Чтобы указать Skype приложения для собраний в качестве предпочтительного приложения:

1. Войдите в Центр администрирования Microsoft Teams.
2. В левой области в области **Параметры** для всей организации выберите Teams **обновить**.
3. На странице Teams обновления в списке Параметры приложения **выберите** приложение Skype  **"Собрания"** в списке Предпочитаемый, чтобы пользователи Skype для бизнеса собрания.

    ![Выберите Предпочитаемый приложение для пользователей, чтобы присоединиться Skype для бизнеса собрания.](media/teams-configure-skype-meetings-app-to-work-with-teams-image1.png)

## <a name="known-limitations"></a>Известные ограничения

На пользователей, которые используют приложение Skype "Собрания" Teams, налагаются следующие ограничения:

- У пользователей нет возможности изменить свое видео устройство.
- После обновления пользователя до Teams, если он находится на собрании с помощью приложения "Собрания Skype", а затем принимает звонок Teams, собрание в приложении Skype Meetings не помещается на удержание. Вместо этого пользователь подключен к обоим звонкам.
