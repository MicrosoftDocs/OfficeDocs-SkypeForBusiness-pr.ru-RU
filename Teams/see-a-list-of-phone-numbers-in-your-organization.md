# <a name="see-a-list-of-phone-numbers-in-your-organization"></a>Просмотр списка телефонных номеров в организации

Существует несколько типов номеров телефонов, которые можно назначить пользователям или другим службам (номерам служб), например для аудиоконференций в Microsoft 365 или Office 365.
  
## <a name="to-see-a-list-of-all-phone-numbers-that-you-have-for-your-organization"></a>Просмотр списка всех телефонных номеров, имеющихся в организации

![Значок с логотипом Teams.](media/teams-logo-30x30.png) **С помощью Центра администрирования Microsoft Teams**

1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в левой области навигации Центра администрирования Skype для бизнеса вы видите параметр Голосовая почта, сначала необходимо  приобрести по крайней мере одну  лицензию на **Enterprise E5,** одну лицензию на надстройку телефонная система или одну лицензию на надстройку аудиоконференции. 

3. Чтобы просмотреть номера телефонов, которые назначены, см. **столбец Состояние.**

4. Чтобы отфильтровать представление, щелкните значок фильтра. В области **Фильтр** можно отфильтровать представление по:

   - **Застроный** диапазон номеров. Можно искать по наименьшему или максимальному числу.

   - Числа, которые начинаются с числа, которое вы указали.

   - Состояние **активации номера.**

   - Тип **номера**.

   - Телефон состоянии **номера**.

## <a name="to-see-all-of-the-phone-numbers-that-are-assigned-to-users"></a>Просмотр всех телефонных номеров, назначенных пользователям

При настройке пользователей может потребоваться просто посмотреть список номеров телефонов, которые уже назначены пользователям, и доступные для них номера телефонов.
  
![Значок с логотипом Teams.](media/teams-logo-30x30.png) **С помощью Центра администрирования Microsoft Teams**

1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в левой области навигации в Центре администрирования Microsoft Teams вы видите параметр Голосовая почта, сначала необходимо  приобрести по крайней мере одну  лицензию на **Enterprise E5,** одну лицензию на надстройку телефонная система или одну лицензию на надстройку аудиоконференции. 

3. Чтобы быстро отсортировать числа, чтобы увидеть, какие из них назначены, щелкните **заголовок** столбца Состояние. Можно также щелкнуть значок фильтра, а затем отфильтровать представление, чтобы просмотреть номера телефонов, которые уже назначены пользователям, или неназначные номера, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:

   - **Назначено пользователю**

   - **Мост конференций назначен** 

   - **Неназначенные**

## <a name="to-see-the-phone-numbers-that-are-assigned-to-voice-users"></a>Просмотр телефонных номеров, назначенных пользователям голосовой связи

При настройке параметров пользователей в организации для совершения и приема телефонных звонков необходимо сначала получить номера телефонов, а затем назначить их пользователям. После того как вы получили номера телефонов, может потребоваться просто узнать состояние активации назначений номеров.

![Значок с логотипом Teams.](media/teams-logo-30x30.png) **С помощью Microsoft Teams администрирования** !
  
1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в левой области навигации в Центре администрирования Microsoft Teams вы видите параметр Голосовая почта, сначала необходимо  приобрести по крайней мере одну  лицензию на **Enterprise E5,** одну лицензию на надстройку телефонная система или одну лицензию на надстройку аудиоконференции. 

3. Щелкните значок фильтра, чтобы отфильтровать свое представление по состояниям **активации.** Вы можете отфильтровать его по:

   - **Активирован**

   - **Ожидается назначение**

   - **Сбой задания**

   - **Ожидается обновление**

   - **Не удалось обновить**

## <a name="using-the-teams-powershell-module"></a>Использование модуля Teams PowerShell

С помощью модуля Teams PowerShell можно получить ту же самую информацию из предыдущих разделов, но требуется версия 1.1.6 или более поздней, которая включает в себя интеграцию соединителю Skype для бизнеса Online. Дополнительные сведения о модуле см. в Microsoft Teams [Обзор PowerShell.](teams-powershell-overview.md)

Список всех номеров телефонов, которые вы используете для организации, можно увидеть с помощью cmdlet [Get-CsOnlineTelephoneNumber.](/powershell/module/skype/get-csonlinetelephonenumber) Например, чтобы увидеть каждый номер телефона и его состояние, можно выполнить следующую команду:

```PowerShell
Get-CsOnlineTelephoneNumber | ft Id,ActivationState
```

Вы можете увидеть все телефонные номера, которые назначены пользователям, с помощью [cmdlet Get-CsOnlineUser.](/powershell/module/skype/get-csonlineuser) Например, чтобы увидеть всех пользователей с номером телефона, можно выполнить следующую команду:

```PowerShell
Get-CsOnlineUser | Where-Object  { $_.LineURI -notlike $null } | ft DisplayName,UserPrincipalName,LineURI
```
