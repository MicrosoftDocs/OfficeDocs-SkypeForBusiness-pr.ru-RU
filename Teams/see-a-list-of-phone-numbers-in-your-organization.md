# <a name="see-a-list-of-phone-numbers-in-your-organization"></a>Просмотр списка телефонных номеров в организации

Есть различные типы телефонных номеров, которые в Office 365 можно назначить пользователям или другим службам (служебные номера), таким как Аудиоконференции.
  
## <a name="to-see-a-list-of-all-phone-numbers-that-you-have-for-your-organization"></a>Просмотр списка всех телефонных номеров, имеющихся в организации

![Значок с логотипом](media/teams-logo-30x30.png) Teams в **центре администрирования Microsoft Teams**

1. Перейдите в **центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .

3. Чтобы просмотреть назначенные номера телефонов, просмотрите столбец **состояние** .

4. Чтобы отфильтровать представление, щелкните значок фильтра. В области **фильтров** с помощью раскрывающегося списка можно отфильтровать представление, выполнив указанные ниже действия.

   - Заданный **диапазон номеров** . Вы можете выполнять поиск по наименьшему числу или самому высокому номеру.

   - Номера, которые начинаются с указанного числа.

   - **Состояние активации**номера.

   - **Тип**номера.

   - **Состояние**номера телефона.

## <a name="to-see-all-of-the-phone-numbers-that-are-assigned-to-users"></a>Просмотр всех телефонных номеров, назначенных пользователям

При настройке пользователей может потребоваться просмотр списка телефонных номеров, которые уже назначены пользователям, и номера телефонов, которые могут быть назначены им.
  
![Значок с логотипом](media/teams-logo-30x30.png) Teams в **центре администрирования Microsoft Teams**

1. Перейдите в **центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Microsoft Teams необходимо сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** , а также одну лицензию на использование **голосовой конференции** .

3. Чтобы быстро отсортировать числа, чтобы их можно было видеть, щелкните заголовок столбца **состояние** . Вы также можете щелкнуть значок фильтра, а затем отфильтровать свое представление, чтобы увидеть номера телефонов, которые уже назначены пользователям или неназначенные номера, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:

   - **Назначено пользователю**

   - **Назначено мосту конференц-связи** 

   - **Неназначенные**

## <a name="to-see-the-phone-numbers-that-are-assigned-to-voice-users"></a>Просмотр телефонных номеров, назначенных пользователям голосовой связи

При настройке параметров пользователей в организации для совершения и приема телефонных звонков необходимо сначала получить номера телефонов, а затем назначить их пользователям. После того как вы настроили номера телефонов, вам может быть нужно просто посмотреть состояние активации для присвоения номера.

![Значок с логотипом](media/teams-logo-30x30.png) Teams в **центре администрирования Microsoft Teams** !
  
1. Перейдите в **центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Microsoft Teams необходимо сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** , а также одну лицензию на использование **голосовой конференции** .

3. Щелкните значок "фильтр", чтобы отфильтровать представление по **состоянию активации** , по которому можно выполнить фильтрацию.

   - **Включена**

   - **Ожидание задания**

   - **Не удалось выполнить задание**

   - **Ожидается обновление**

   - **Обновление завершилось сбоем**
