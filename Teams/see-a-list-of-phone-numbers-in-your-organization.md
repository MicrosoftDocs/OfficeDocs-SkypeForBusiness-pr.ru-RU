# <a name="see-a-list-of-telephone-numbers"></a>См. список телефонных номеров 

Существует несколько типов телефонных номеров, которые можно назначить пользователям или приложениям голосовой связи, таким как аудиоконференция [или](deploy-audio-conferencing-teams-landing-page.md) очереди [вызовов.](plan-auto-attendant-call-queue.md) Дополнительные сведения см. в [управлении телефонными номерами для организации.](/microsoftteams/manage-phone-numbers-landing-page)

Эта статья относится к планам звонков и операторам Подключение. Сведения о прямой маршрутизаке см. в настройках телефонного номера и в том, как включить корпоративную [голосовую и голосовую почту.](direct-routing-enable-users.md#configure-the-phone-number-and-enable-enterprise-voice-and-voicemail-online)
  
## <a name="to-see-all-telephone-numbers-in-your-organization"></a>Чтобы увидеть все телефонные номера в организации

Чтобы посмотреть список всех телефонных номеров в организации:

1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

3. Чтобы просмотреть номера телефонов, которые назначены, см. столбец **Состояние** назначения, в котором также показано, какой тип службы назначен номеру.

4. Чтобы отфильтровать представление, щелкните значок фильтра. В области **Фильтр** можно отфильтровать представление по:

   - **Застроный** диапазон номеров. Можно искать по наименьшему или максимальному числу.

   - Числа, которые начинаются с числа, которое вы указали.

   - Состояние **активации номера.**

   - Тип **номера**.

   - Телефон состоянии **номера**.

## <a name="to-see-all-telephone-numbers-that-are-assigned-to-users"></a>Чтобы увидеть все телефонные номера, которые назначены пользователям

При настройке пользователей может потребоваться просто посмотреть список телефонных номеров, которые уже назначены пользователям, и номера телефонов, которые им доступны.

1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в левой области навигации Центра администрирования Microsoft Teams вы могли увидеть параметр Голосовая почта, необходимо сначала  приобрести по крайней мере одну  лицензию на **Enterprise E5,** одну лицензию на надстройку телефонная система или одну лицензию на надстройку аудиоконференции. 

3. Чтобы быстро отсортировать числа, чтобы увидеть, какие из них назначены, щелкните заголовок столбца **Состояние** задания. Можно также щелкнуть значок фильтра, а затем отфильтровать представление, чтобы просмотреть телефонные номера, которые уже назначены пользователям, или неназначные номера, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:

   - **Назначено пользователю**
   - **Мост конференций назначен** 
   - **Назначено приложению Голосовая почта (автоотекатарь или очередь зовов)**
   - **Неназначенные**

## <a name="to-see-all-telephone-numbers-that-are-assigned-to-voice-users"></a>Чтобы увидеть все телефонные номера, которые назначены пользователям голосовой связи

При настройке телефонных звонков для пользователей в организации необходимо сначала получить телефонные номера, а затем назначить их пользователям. После того как вы получили номера телефонов, может потребоваться узнать состояние активации их назначений.
  
1. Перейдите в **центр Microsoft Teams администрирования**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

3. Щелкните значок фильтра, чтобы отфильтровать свое представление по **состояниям активации**. Вы можете фильтровать по следующим параметрам:

   - **Активирован**
   - **Ожидается назначение**
   - **Сбой задания**
   - **Ожидается обновление**
   - **Не удалось обновить**

## <a name="using-the-teams-powershell-module"></a>Использование модуля Teams PowerShell

С помощью модуля Teams PowerShell можно получить ту же самую информацию из предыдущих разделов, но требуется версия 1.1.6 или более поздней, которая включает интеграцию соединителю Skype для бизнеса Online. Дополнительные сведения о модуле см. в Microsoft Teams [Обзор PowerShell.](teams-powershell-overview.md)

Чтобы увидеть список всех номеров телефонов, которые вы используете для своей организации, используйте для этого cmdlet [Get-CsOnlineTelephoneNumber.](/powershell/module/skype/get-csonlinetelephonenumber) Например, чтобы увидеть каждый номер телефона и его состояние, запустите следующую команду:

```PowerShell
Get-CsOnlineTelephoneNumber | ft Id,ActivationState
```

Чтобы увидеть все телефонные номера, которые назначены пользователям, используйте для этого cmdlet [Get-CsOnlineUser.](/powershell/module/skype/get-csonlineuser) Например, чтобы увидеть всех пользователей с назначенным номером телефона, запустите следующую команду:

```PowerShell
Get-CsOnlineUser | Where-Object  { $_.LineURI -notlike $null } | ft DisplayName,UserPrincipalName,LineURI
```
