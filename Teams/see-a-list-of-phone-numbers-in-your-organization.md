# <a name="see-a-list-of-phone-numbers-in-your-organization"></a>Просмотр списка телефонных номеров в организации

Существует несколько типов номеров телефонов, которые можно назначить пользователям или другим службам (номера служб), например для аудиоконференций в Microsoft 365 или Office 365.
  
## <a name="to-see-a-list-of-all-phone-numbers-that-you-have-for-your-organization"></a>Просмотр списка всех телефонных номеров, имеющихся в организации

![Значок логотипа Teams ](media/teams-logo-30x30.png) **с помощью Центра администрирования Microsoft Teams**

1. Перейдите в **Центр администрирования Microsoft Teams.**

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Для того чтобы  в левой области навигации Центра администрирования Skype для бизнеса была отписна  возможность голосовой связи, необходимо  сначала приобрести по крайней мере одну лицензию на корпоративный **E5,** одну лицензию на надстройку телефонной системы или одну лицензию на надстройку для аудиоконференции.

3. Чтобы просмотреть номера телефонов, которые назначены, см. **столбец "Состояние".**

4. Чтобы отфильтровать представление, щелкните значок фильтра. В области **"Фильтр"** с помощью списка можно отфильтровать представление по:

   - **Застроный** диапазон номеров. Можно искать по наименьшему или максимальному числу.

   - Числа, которые начинаются с за указанного числа.

   - Состояние **активации номера.**

   - Тип **номера.**

   - Состояние номера **телефона.**

## <a name="to-see-all-of-the-phone-numbers-that-are-assigned-to-users"></a>Просмотр всех телефонных номеров, назначенных пользователям

При настройке пользователей может потребоваться просто посмотреть список номеров телефонов, которые уже назначены пользователям, и доступные для них номера.
  
![Значок логотипа Teams ](media/teams-logo-30x30.png) **с помощью Центра администрирования Microsoft Teams**

1. Перейдите в **Центр администрирования Microsoft Teams.**

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в центре  администрирования Microsoft Teams в левой области навигации был параметр "Голосовая почта", необходимо сначала приобрести по крайней мере одну лицензию на корпоративный **E5,** одну лицензию на надстройку телефонной системы или одну лицензию на надстройку для аудиоконференции.  

3. Чтобы быстро отсортировать числа, чтобы увидеть,  какие из них назначены, щелкните заголовок столбца "Состояние". Можно также щелкнуть значок фильтра, а затем отфильтровать представление, чтобы просмотреть телефонные номера, которые уже назначены пользователям, или неназначные, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:

   - **Назначено пользователю**

   - **Мост конференций, который назначен мосту конференц-зала** 

   - **Ненаписано**

## <a name="to-see-the-phone-numbers-that-are-assigned-to-voice-users"></a>Просмотр телефонных номеров, назначенных пользователям голосовой связи

При настройке параметров пользователей в организации для совершения и приема телефонных звонков необходимо сначала получить номера телефонов, а затем назначить их пользователям. После того как вы получили номера телефонов, может потребоваться узнать состояние активации назначений номеров.

![Значок логотипа Teams ](media/teams-logo-30x30.png) **с помощью Центра администрирования Microsoft Teams!**
  
1. Перейдите в **Центр администрирования Microsoft Teams.**

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы в центре  администрирования Microsoft Teams в левой области навигации был параметр "Голосовая почта", необходимо сначала приобрести по крайней мере одну лицензию на корпоративный **E5,** одну лицензию на надстройку телефонной системы или одну лицензию на надстройку для аудиоконференции.  

3. Щелкните значок фильтра, чтобы отфильтровать свое представление **по** состояниям активации:

   - **Активировано**

   - **Ожидается выполнение задания**

   - **Сбой задания**

   - **Ожидается обновление**

   - **Сбой обновления**

## <a name="using-the-teams-powershell-module"></a>Использование модуля Teams PowerShell

С помощью модуля Teams PowerShell можно получить те же сведения из предыдущих разделов, но для этого требуется версия 1.1.6 или более поздней версии, которая включает в себя соединитель Skype для бизнеса Online. Дополнительные сведения о модуле см. в обзоре [Microsoft Teams PowerShell.](teams-powershell-overview.md)

Список всех номеров телефонов для организации можно увидеть с помощью [cmdlet Get-CsOnlineTelephoneNumber.](/powershell/module/skype/get-csonlinetelephonenumber) Например, чтобы увидеть номер телефона и его состояние, можно выполнить следующую команду:

```PowerShell
Get-CsOnlineTelephoneNumber | ft Id,ActivationState
```

Вы можете увидеть все телефонные номера, которые назначены пользователям, с помощью cmdlet [Get-CsOnlineUser.](/powershell/module/skype/get-csonlineuser) Например, можно выполнить следующую команду, чтобы увидеть всех пользователей с номером телефона:

```PowerShell
Get-CsOnlineUser | Where-Object  { $_.LineURI -notlike $null } | ft DisplayName,UserPrincipalName,LineURI
```
