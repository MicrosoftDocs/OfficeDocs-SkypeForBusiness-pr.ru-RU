# <a name="see-a-list-of-telephone-numbers"></a>Просмотр списка телефонных номеров 

Существуют различные типы телефонных номеров, которые можно назначить пользователям или голосовым приложениям, таким как аудиоконференции [или](deploy-audio-conferencing-teams-landing-page.md) [очереди звонков](plan-auto-attendant-call-queue.md). Дополнительные сведения см. в разделе ["Управление телефонными номерами для вашей организации"](/microsoftteams/manage-phone-numbers-landing-page).

Эта статья относится к тарифным планам, Operator Connect и Teams Phone Mobile. Сведения о прямой маршрутизации см. в разделе ["Настройка номера телефона и включение корпоративной голосовой связи"](direct-routing-enable-users.md#configure-the-phone-number-and-enable-enterprise-voice).
  
## <a name="to-see-all-telephone-numbers-in-your-organization"></a>Просмотр всех телефонных номеров в организации

Чтобы просмотреть список всех телефонных номеров в организации, с помощью следующих параметров:

1. Перейдите в **Центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

3. Чтобы просмотреть назначенные номера телефонов, см. столбец **"** Состояние назначения", в котором также показано, какой тип службы назначен номеру.

4. Чтобы отфильтровать представление, щелкните значок фильтра. В области **фильтра** можно использовать раскрывающийся список, чтобы отфильтровать представление по:

   - **Задаваемый** диапазон номеров. Поиск можно выполнять по наименьшему или самому высокому числу.

   - Числа, начинающееся с указанного числа.

   - Состояние **активации номера**.

   - Тип **числа**.

   - Состояние номера **телефона**.

## <a name="to-see-all-telephone-numbers-that-are-assigned-to-users"></a>Просмотр всех номеров телефонов, назначенных пользователям

При настройке пользователей может потребоваться просто просмотреть список телефонных номеров, которые уже назначены пользователям и какие телефонные номера им доступны.

1. Перейдите в **Центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

    > [!IMPORTANT]
    > Чтобы увидеть параметр "Голосовая связь" в левой области навигации в Центре администрирования Microsoft Teams, сначала необходимо приобрести по крайней мере одну корпоративную лицензию **E5**, одну лицензию на надстройку телефонной системы или одну лицензию на надстройку аудиоконференций.  

3. Чтобы быстро отсортировать числа, чтобы увидеть назначенные значения, щелкните заголовок столбца **"** Состояние назначения". Можно также щелкнуть значок фильтра, а затем отфильтровать представление, чтобы просмотреть номера телефонов, которые уже назначены пользователям, или неназначенных номеров, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:

   - **Назначено пользователю**
   - **Назначено мосту конференций** 
   - **Назначено приложению "Голосовая связь" (автосекретарь или очередь вызовов)**
   - **Неназначенные**

## <a name="to-see-all-telephone-numbers-that-are-assigned-to-voice-users"></a>Просмотр всех номеров телефонов, назначенных пользователям голосовой связи

При настройке пользователей в организации для выполнения и получения телефонных звонков необходимо сначала получить телефонные номера, а затем назначить их пользователям. После получения телефонных номеров может потребоваться просмотреть состояние активации назначений номеров.
  
1. Перейдите в **Центр администрирования Microsoft Teams**.

2. На панели навигации слева перейдите в раздел **Голосовая связь** > **Номера телефонов**.

3. Щелкните значок фильтра, чтобы отфильтровать представление по **состоянию активации**. Вы можете фильтровать по следующим параметрам:

   - **Активирован**
   - **Ожидается назначение**
   - **Сбой назначения**
   - **Ожидание обновления**
   - **Сбой обновления**

## <a name="using-the-teams-powershell-module"></a>Использование модуля Teams PowerShell

Вы можете использовать модуль Teams PowerShell, чтобы получить те же сведения из предыдущих разделов, но требуется версия 1.1.6 или более поздняя, которая включает интеграцию соединителя Skype для бизнеса Online. Дополнительные сведения о модуле см. в [обзоре Microsoft Teams PowerShell](teams-powershell-overview.md).

Чтобы просмотреть список всех номеров телефонов вашей организации, используйте командлет [Get-CsPhoneNumberAssignment](/powershell/module/teams/get-csphonenumberassignment) . Например, чтобы просмотреть каждый номер телефона, его тип и состояние, выполните следующую команду:

```PowerShell
Get-CsPhoneNumberAssignment | ft TelephoneNumber,ActivationState,NumberType
```

Чтобы просмотреть все телефонные номера, назначенные пользователям, используйте командлет [Get-CsOnlineUser](/powershell/module/skype/get-csonlineuser) . Например, чтобы просмотреть всех пользователей с назначенным номером телефона, выполните следующую команду:

```PowerShell
Get-CsOnlineUser | Where-Object  { $_.LineURI -notlike $null } | ft DisplayName,UserPrincipalName,LineURI
```
