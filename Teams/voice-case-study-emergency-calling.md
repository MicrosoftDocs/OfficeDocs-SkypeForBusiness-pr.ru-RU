# <a name="contoso-case-study-emergency-calling"></a>Исследование успеха Contoso: вызов экстренной помощи

Сведения о доступности и терминологии, связанные с экстренным звонком &mdash; , местом, местом хранения для экстренного реагирования и зарегистрированным адресом &mdash; компании Contoso проверено [Управление вызовом экстренной](what-are-emergency-locations-addresses-and-call-routing.md) помощи, [планирование и Настройка динамического вызова экстренной](configure-dynamic-emergency-calling.md)помощи.

В Office 365 пользователь плана звонков автоматически включается для вызова экстренной помощи. Но только пользователи плана звонков в США могут использовать динамические места для маршрутизации экстренных вызовов. 

Для прямой маршрутизации компания Contoso узнали, что требуется дополнительная настройка маршрутизации для экстренных вызовов и, возможно, для подключения партнеров. Администратор должен настроить подключение к поставщику услуг маршрутизации экстренной помощи (ERSP) (США) или настроить контроллер границ сеанса (SBC) для приложения идентификационного номера расположения экстренной помощи (ELIN).

Компания Contoso имеет офисы в США и за пределами США.

- В США в плане абонентов Contoso пользователи могут использовать динамические места для маршрутизации экстренных вызовов. 

- За пределами Соединенных Штатов компания Contoso имеет некоторые сайты, которые используют планы звонков и некоторые сайты, подключенные к телефонной системе с помощью прямой маршрутизации.

## <a name="emergency-calling-use-cases"></a>Варианты использования для экстренных вызовов

После того как вы решите, как Contoso подключится к телефонной системе, компания Contoso определила следующие варианты использования для экстренных вызовов: 

- [Абонентский план в США](#calling-plan-user-in-the-united-states) 

- [Пользователь планов звонков за пределами США](#calling-plan-user-outside-of-the-united-states)

- [Пользователь, который подключается к телефонной системе через прямую маршрутизацию](#user-who-connects-to-phone-system-through-direct-routing )


### <a name="calling-plan-user-in-the-united-states"></a>Абонентский план в США  

Существуют требования, которые необходимо выполнить, когда нужно связать номер телефона с местом для экстренного реагирования. Сведения о требованиях, предъявляемых компанией Contoso, рассмотрены в разделе [планы звонков](what-are-emergency-locations-addresses-and-call-routing.md#considerations-for-calling-plans). 

Согласно этим требованиям компания Contoso решила сопоставить место с номером телефона, когда номер назначен пользователю в США.

### <a name="calling-plan-user-outside-of-the-united-states"></a>Пользователь планов звонков за пределами США 

Чтобы узнать, когда нужно связать номер телефона с местом для экстренного реагирования, ознакомьтесь с [Размышленными планами](what-are-emergency-locations-addresses-and-call-routing.md#considerations-for-calling-plans), рассмотренными компанией Contoso. Согласно требованиям компания Contoso решила следующее:  

-  Компания Contoso свяжет место с номером телефона, когда номер назначен пользователю в Канаде. 

- Компания Contoso назначает место для экстренного реагирования при получении номера телефона из Office 365 или при передаче номера от другого поставщика услуг или оператора. 

### <a name="user-who-connects-to-phone-system-through-direct-routing"></a>Пользователь, который подключается к телефонной системе через прямую маршрутизацию 

Для планирования аварийной маршрутизации для этого случая использования компания Contoso проверила [рекомендации по прямой маршрутизации](what-are-emergency-locations-addresses-and-call-routing.md#considerations-for-direct-routing). Поскольку прямые пользователи маршрутизации не получают возможность вызова экстренной помощи таким же образом, как и пользователи плана звонков, компания Contoso приводила к тому, как обеспечить возможность вызова экстренной помощи. Прямая маршрутизация может быть подключена к поставщику услуг маршрутизации экстренной помощи (ERSP). Кроме того, прямая маршрутизация может иметь SBC, включающий идентификационный номер расположения для экстренного реагирования (ELIN).   

#### <a name="emergency-routing-service-provider-ersp-considerations"></a>Вопросы о поставщике услуг маршрутизации экстренной помощи (ERSP)

Поставщики услуг маршрутизации экстренной помощи (ERSPs) могут автоматически перенаправлять экстренные вызовы в зависимости от расположения вызывающего абонента.  

- Если поставщик услуг маршрутизации экстренной помощи интегрирован в прямое развертывание по маршрутизации, экстренные вызовы с динамически приобретенным расположением будут автоматически перенаправлены в точку ответа для общедоступной защиты (PSAP), обслуживающей это расположение. 

- Вызов экстренной помощи без динамически приобретенного местоположения — первый экран, чтобы определить текущее расположение пользователя, прежде чем присоединиться к соответствующему центру отправки, основываясь на обновленном расположении. 


#### <a name="elin-considerations"></a>Вопросы ELIN

Если приложение SBC ELIN интегрировано в прямое развертывание, необходимо выполнить дополнительные действия по настройке, чтобы связать адреса экстренного реагирования с телефонными номерами.  

Компания Contoso решила использовать контроллеры границ сеансов, в том числе приложения с идентификационным номером расположения для экстренного реагирования (ELIN).  

## <a name="security-desk-notification"></a>Уведомление на рабочем месте

Возможность уведомления службы безопасности о размещении экстренного звонка доступна как для планов звонков, так и для прямой маршрутизации телефонной системы. Компания Contoso проверила сведения в уведомлении о безопасности, чтобы определить, следует ли настроить его на доступ к своим офисам  

Компания Contoso решила использовать уведомление на службу безопасности.

- Назначение адресов для экстренного реагирования 

- Настройка параметров сети 

- Настройка службы сведений о расположении 

- Настройка политик экстренной помощи 

- Включение пользователей и сайтов 

- Проверка вызова экстренной помощи

- Чтобы создать места для зданий, этажей и офисов, компания Contoso предместит инструкции в статье [Добавление, изменение и удаление Места для экстренного](add-change-remove-emergency-place-organization.md) реагирования. 

- Чтобы назначить место для экстренного реагирования, компания Contoso проследует шаги в разделе [назначение или изменение расположения для экстренного реагирования для пользователя](assign-change-emergency-location-user.md).
