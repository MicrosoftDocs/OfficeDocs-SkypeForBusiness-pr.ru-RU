# <a name="best-practices-for-user-feedback-methods-in-microsoft-teams"></a>Рекомендации по механизмам обратной связи с пользователями в Microsoft Teams

Внедрение новой процедуры взаимодействия заметно повлияет на работу пользователей. Для изменения прав требуется обучение, поддержка и положительные примеры. Кроме того, очень важно, чтобы люди были слышны во время перехода. Если ранее вы реализовали в организации успешные механизмы обратной связи, рекомендуется распространить их и на внедрение Teams. Если вы еще не сделали этого, рассмотрите один из следующих подходов:

- Создайте в своей Microsoft Teams группу, в которую любой пользователь может присоединиться, чтобы оставить отзыв об их работе (помните, что общедоступные группы могут быть ограничены 5000 пользователями).

- Используйте Yammer, чтобы предоставить открытому сообществу лучшие методики и поддержку.

Чтобы реализовать обратную связь на базе сообщества, нужно привлечь лидеров и владельцев служб. Мы рекомендуем составить для этих участников план наблюдения. Один из подсказок — создать отдельную группу в Microsoft Teams в качестве места для совместной работы, обмена ресурсами и лучших методик, прежде чем делиться ими с другими коллегами. Они выполняют роль консультанта для пользователей, принявшим новую службу, обеспечивают более удобное и тесное взаимодействие с другими участниками группы развертывания, чтобы повысить и устранить проблемы.

## <a name="showcases"></a>Конкретные примеры

По мере продолжения этого пути вы найдете демонстрации в своей среде сотрудников, чьи творческие способности Microsoft Teams вас порадуют и удивят. Широко делитесь этими демонстрациями. Это будет способствовать внедрению новых технологий, обучению пользователей, которые, возможно, были раньше всех, и вознаградить тех, кто уехает время на то, чтобы подумать о том, как лучше Microsoft Teams в своей среде. Для поддержания этой инициативы можно задать измеримую цель по количеству примеров на квартал.
