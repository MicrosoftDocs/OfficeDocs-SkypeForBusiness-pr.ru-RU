## <a name="prioritize-important-network-traffic-using-qos"></a><span data-ttu-id="9baa7-123">Расставить приоритеты для важного сетевого трафика с помощью QoS</span><span class="sxs-lookup"><span data-stu-id="9baa7-123">Prioritize important network traffic using QoS</span></span>
