# <a name="live-event-recording-policies-in-microsoft-teams"></a>Политики записи трансляций в Microsoft Teams

У вас есть несколько вариантов записи трансляции Microsoft Teams. Параметры записи задается с помощью политик записи. В этой статье описываются различные параметры.

Параметры записи задается с помощью команды [PowerShell Set-CsTeamsMeetingBroadcastPolicy](/powershell/module/skype/set-csteamsmeetingbroadcastpolicy).

## <a name="scheduling-and-option-behaviors"></a>Планирование и поведение параметров

Существует два варианта организатора при планировании записи трансляции:

- Запись доступна для организаторов и выступающих

  - Файл записи: предоставляет файл записи, который производители и выступающие могут скачать после завершения события.

- Запись доступна для участников

  - DVR: цифровой видеозапокатор (DVR) позволяет участникам перемотать и приостанавливать события

  - VOD: видео по запросу (VOD) позволяет участникам наблюдать после завершения события

## <a name="broadcast-recording-policy-setting"></a>Параметр политики широковещательной записи

В рамках политики трансляции есть параметр, который можно включить или отключить для трансляции.

| &nbsp;| Запись доступна для организаторов и выступающих | Запись доступна для участников |
| ------------------------------- | ---------------------------------------------------- | ------------------------------------- |
| Всегда записывать               | Отключено и выбрано                                | Включено и выбрано         |
| Организатор может записывать или нет | Включено и выбрано по умолчанию                  | Включено и выбрано по умолчанию   |
| Никогда не записывать               | Отключено и не выбрано                            | Отключено и не выбрано      |
