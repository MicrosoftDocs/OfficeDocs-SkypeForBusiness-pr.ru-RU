# <a name="auto-attendant-and-call-queue-dialing-and-voice-recognition-reference"></a>Справочник по автоотзыву и очереди звонка и распознаванию речи

Функция "Набор по имени" — это функция автозавода, которая также называется поиском по каталогу. Она позволяет пользователям, которые звонят автоотправлению, использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести полное или частичное имя для поиска в каталоге компании, найти человека, а затем перенаправить ему звонок. Вы настраивали звонок по имени при настройке параметров потока [вызовов в автофигуре.](create-a-phone-system-auto-attendant.md#call-flow)

## <a name="searching-for-users"></a>Поиск пользователей

Пользователям, с которых вы хотите получить доступ с помощью звонка по имени, не требуется номер телефона или назначены планы звонков, но они должны быть Корпоративная голосовая связь включены для пользователей Skype для бизнеса **Server.** Кроме того, с помощью звонка по имени пользователи Microsoft Teams, которые находятся в разных странах и регионах разных стран и регионов, смогут находить и переводить звонки. С учетом необходимых условий вы явным образом включаете в автозапросе набор по имени.

Набор по расширениям — это функция автозавода, которая также является частью поиска по каталогу. Она позволяет пользователям, которые звонят автоотправлению, использовать голосовые (распознавание речи) или ответы на клавиатурную панель телефона (DTMF), чтобы ввести расширение телефона пользователя, с которым он пытается связаться, а затем перенаправить звонок на него. Пользователям, с которых вы хотите получить доступ с помощью звонка по расширению, не требуется номер телефона или назначены планы звонков, но они должны быть Корпоративная голосовая связь включены для пользователей Skype для бизнеса **Server.** Кроме того, для пользователей должна быть настроена соответствующая телефонная плана. Кроме того, с помощью телефонного звонка по расширению можно находить и переводить звонки пользователям Microsoft Teams, которые находятся в разных странах или регионах для международных организаций. С учетом необходимых условий вы явным образом включаете "Набор по расширению" для автозавода.

### <a name="maximum-directory-size"></a>Максимальный размер справочника

Число пользователей Active Directory, которые могут звонить по имени и по расширению, не ограничивается. Вызываемая звонок может ввести полное или частичное имя (имя + фамилия, а также фамилия + имя), но нуждается в полном расширении. Максимальный размер списка имен, поддерживаемый одним автозаводом при использовании распознавания речи, составляет 80 000 пользователей.
  
|Тип ввода|Формат поиска|Максимальное число пользователей в организации|
|:-----|:-----|:-----|
|DTMF (ввод с клавиатурной панели) |Частичный  <br/> Имя + фамилия  <br/> Фамилия + Имя |Нет ограничений  |
|Распознавание речи (голосовой ввод) |FirstName  <br/> Фамилия  <br/> Имя + фамилия  <br/> Фамилия + Имя  | 80 000 пользователей |

> [!NOTE]
> Если вы используете функцию звонка по имени с распознаванием речи, но в каталоге Active Directory вашей организации больше 80 000 пользователей и вы еще не ограничили область действия функции "Набор по имени" с помощью функции "Набор по имени", функция "Звонок по имени" по-прежнему будет работать для вызывающих пользователей с помощью клавиатурной панели телефона, а голосовой ввод будет доступен во всех других сценариях. С помощью функции "Область набора" можно сузить список имен, которые можно получить, изменив область набора по имени для конкретного автозавода.
  
## <a name="dial-by-name---keypad-dtmf-entry"></a>Звонок по имени  ввод с клавиатурной панели телефона (DTMF)
С помощью звонка по имени пользователи могут связаться с пользователями по полному или частичному имени. Существуют различные форматы, которые можно использовать при в введенном имени.

При поиске в каталоге организации с помощью клавиши 0 (ноль) можно указать пробел между именем и фамилией и именем. При вводе имени будет предложено прервать ввод клавиатуры с помощью клавиши #. Например: "После ввода имени человека, с которого вы пытаетесь связаться, нажмите кнопку "#". Если найдено несколько имен, вызываемму человеку будет предоставлен список имен для выбора.
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|Формат имени|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный  |Иван0Макаренко# |Иван Макаренко |
|Фамилия + Имя |Полный |Макаренко0Иван#  |Иван Макаренко |
|FirstName  |Полный   |Иван#   |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия |Полный |Макаренко#  |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко" |
|Имя или фамилия |Частичный |Ма# |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко" |
|Имя + Фамилия |Частичный |Amos0Mar # |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия + Имя |Частичный |Ма0Ив# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |

Существует несколько специальных символов, которые используются при поиске людей, использующих клавиатурную панель телефона. Например, человеку будет предложено использовать клавишу с фунтом (#), а для пробела между именами используется ноль (0). При нажатии клавиши звездочка (*) список имен, совпадающих с человеком, будет повторяться.
  
|Специальный знак на клавиатуре телефона|Описание|
|:-----|:-----|
|#   |Завершение ввода имени. |
|0   |Пробел между именами. |
|*    |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска. |

### <a name="dial-by-name---name-recognition-with-speech"></a>Звонок по имени  распознавание имен при голосовом вводе

Люди могут искать других людей в своей организации с помощью голосового голоса (распознавания речи). Кроме того, он может связаться с любым человеком в Active Directory, назовя полное или частичное имя нужного человека. С помощью голосового ввода можно распознавать имена в различных форматах, включая имя, фамилию, имя + фамилия или фамилию + имя.
  
Вы можете включить распознавание речи для автозавода, но ввод с клавиатурной панели телефона (DTMF) не отключен. Запись с клавиатурной панели телефона можно использовать в любое время, даже если автозавод включена распознавание речи.
  
Как и при вводе на клавиатурной панели телефона, если найдено несколько имен, вызываемая вами учетная запись будет слышать список имен для выбора.
  
Вызыватели могут называть имена в следующих форматах:
  
|Имя с речью|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный |Иван Макаренко |Иван Макаренко |
|Фамилия + Имя |Полный  |Макаренко Иван |Иван Макаренко |
|FirstName |Полный |Иван |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров" |
|Фамилия |Полный |Макаренко |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко" |
|Имя или фамилия |Частичный |Март |Нажмите или скажите 1 для mary Marble  <br/> Нажмите или скажите 2 для mary Jones  <br/> Нажмите или скажите 3 для Амоса Маркета |
|Имя + Фамилия |Частичный |Амос Мар |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или скажите 2 для "Амос Марк" |


> [!NOTE]
> Имя нового пользователя может быть указано в каталоге для функции "Набрать по имени" с распознаванием речи из-за задержки репликации Active Directory.
