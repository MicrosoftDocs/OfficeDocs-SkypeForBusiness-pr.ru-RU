# <a name="auto-attendant-and-call-queue-dialing-and-voice-recognition-reference"></a>Справочник по автоотводу и очереди вызовов и ссылке на распознавание речи

Набор номера по имени или расширению — это функция автоотвода, которая позволяет вызывающим Teams пользователям в вашей организации. С помощью голосовой или телефонной клавиатуры звоня люди могут говорить или вводить полное или частичное имя, а также расширение имени человека, с которого хотите связаться. Автозаводщик начнет поиск в каталоге организации, найдите нужного человека и переназовет вызывающего.  Набор номера по имени или По расширению — это параметры, настраиваемые при настройке параметров потока зова [в автозаводе.](create-a-phone-system-auto-attendant.md#call-flow)


## <a name="searching-for-users"></a>Поиск пользователей

Teams пользователям, с которыми можно звонить по имени, не требуется иметь номер телефона или иметь планы звонков, но они должны быть Корпоративная голосовая связь включены для Skype для бизнеса Server **пользователей.** В разных странах и регионах вызываемая телефонная звонок будет находить и Microsoft Teams пользователям, которые находятся в разных странах или регионах.

Teams пользователям, с которыми можно звонить по расширению, не требуется иметь номер телефона или иметь планы звонков, но они должны быть Корпоративная голосовая связь включены для Skype для бизнеса Server **пользователей.** Кроме того, для пользователей должна быть настроена соответствующая телефонная плана. Для международных организаций поиск и передача вызываемого звонка пользователям, которые находятся в разных странах и регионах Microsoft Teams в разных странах и регионах. 

С учетом необходимых условий набор номера по имени или расширению должен быть явно включен при настройке автозавода.

### <a name="maximum-directory-size"></a>Максимальный размер справочника

Количество пользователей Active Directory, набрав номера по имени и набрав по расширению, не ограничивается. Вызываемая может вводить частичные или полные имена (имя + Фамилия, а также Фамилия + Имя), но требуется полный номер расширения. Максимальный размер списка имен, поддерживаемый одним автозаводом при использовании распознавания речи, составляет 80 000 пользователей.
  
|Тип ввода|Формат поиска|Максимальное число пользователей в организации|
|:-----|:-----|:-----|
|DTMF (ввод с клавиатурной панели) |Частичный  <br/> Имя + фамилия  <br/> Фамилия + Имя |Без ограничений  |
|Распознавание речи (голосовой ввод) |Firstname  <br/> Lastname  <br/> Имя + фамилия  <br/> Фамилия + Имя  | 80 000 пользователей |

> [!NOTE]
> Если вы используете функцию набора номера по имени с распознаванием речи, но в active Directory организации больше 80 000 пользователей и вы еще не ограничили область набора по имени с помощью функции "Набор по имени", функция "Набрать по имени" по-прежнему будет работать для вызывающих пользователей с помощью клавиатурной панели телефона, а голосовой ввод будет доступен для всех других сценариев. [](create-a-phone-system-auto-attendant.md#dial-scope) С помощью функции "Область набора номера" можно сузить список имен, которые можно найти, изменив область набора номера по имени для конкретного автозавода.
 
### <a name="search-considerations"></a>Вопросы поиска 
Набор по имени выполняет поиск в каталоге организации, а затем фильтрует результаты по любым настроенным спискам Включить или Исключить для набора номера. Если исходный поиск возвращает более 100 пользователей, списки "Область набора" не будут применены, поиск будет сбойным, а звонящем будет сказано, что найдено слишком много имен.
 
 
## <a name="dial-by-name---keypad-dtmf-entry"></a>Звонок по имени  ввод с клавиатурной панели телефона (DTMF)
С помощью звонка по имени можно связаться с пользователями, указав полное или частичное имя человека, с которого они пытаются связаться. Существует несколько форматов, которые можно использовать при в введенном имени.

При поиске в каталоге организации с помощью клавиши "0" (ноль) можно указать пробел между именем и фамилией и именем. При вводе имени будет предложено завершить ввод клавиатуры с помощью клавиши #. Например, "После ввода имени человека, с которого вы пытаетесь связаться, нажмите кнопку #". Если найдено несколько имен, вызываемого вызываемого человека выдается список имен для выбора.

> [!NOTE]
> Если после применения каких-либо списков Включить в область набора или Исключить останется более 5 имен, поиск не удастся найти, а звонящем будет сказано, что найдено слишком много имен. 
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|Формат имени|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный  |Иван0Макаренко# |Иван Макаренко |
|Фамилия + Имя |Полный |Макаренко0Иван#  |Иван Макаренко |
|Firstname  |Полный   |Иван#   |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Lastname |Полный |Макаренко#  |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко" |
|Имя или фамилия |Частичный |Ма# |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко" |
|Имя + Фамилия |Частичный |Amos0Mar # |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |
|Фамилия + Имя |Частичный |Ма0Ив# |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко" |

Существует несколько специальных символов, которые используются при поиске людей, использующих клавиатурную панель телефона. Например, человеку будет предложено использовать клавишу с фунтом (#), а для пробела между именами используется ноль (0). При нажатии клавиши звездочка (*) будет повторяться список совпадающих имен с человеком.
  
|Специальный знак на клавиатуре телефона|Описание|
|:-----|:-----|
|#   |Завершение ввода имени. |
|0   |Пробел между именами. |
|*    |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска. |

### <a name="dial-by-name---name-recognition-with-speech"></a>Звонок по имени  распознавание имен при голосовом вводе

Другие люди могут искать других людей в своей организации с помощью голосовой связи (распознавания речи). Они также могут связаться с любыми из пользователей Active Directory, назовя полное или частичное имя человека, которого он пытается найти. С помощью голосового ввода можно распознавать имена в различных форматах, включая Имя, Фамилия, Имя + Фамилия или Фамилия + Имя.
  
Вы можете включить распознавание речи для автозавода, но запись на клавиатурной панели телефона (DTMF) не отключена. Телефон клавиатуры можно использовать в любое время, даже если автозаводщик включил распознавание речи.
  
Как и при вводе на клавиатурной панели телефона, если найдено несколько имен, вызываемая звонок будет слышать список имен для выбора.

> [!NOTE]
> Если после применения каких-либо списков Включить в область набора или Исключить останется более 5 имен, поиск не удастся найти, а звонящем будет сказано, что найдено слишком много имен. 
  
Вызыватели могут называть имена в следующих форматах:
  
|Имя с речью|Тип поиска|Пример|Результат поиска|
|:-----|:-----|:-----|:-----|
|Имя + фамилия |Полный |Иван Макаренко |Иван Макаренко |
|Фамилия + Имя |Полный  |Макаренко Иван |Иван Макаренко |
|Firstname |Полный |Иван |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров" |
|Lastname |Полный |Макаренко |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко" |
|Имя или фамилия |Частичный |Мар |Нажмите или скажите 1 для mary Marble  <br/> Нажмите или скажите 2 для Mary Jones  <br/> Нажмите или скажите 3 для Амоса |
|Имя + Фамилия |Частичный |Amos Mar |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или скажите 2 для Амоса |


> [!NOTE]
> Имя нового пользователя может быть указано в каталоге для набора номера по имени с распознаванием речи из-за задержки репликации Active Directory.
