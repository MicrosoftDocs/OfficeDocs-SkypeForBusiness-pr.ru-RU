# <a name="skype-for-business-online-retirement"></a>Прекращение поддержки Skype для бизнеса Online

В июле 2019 мы объявили о том, что **Skype для бизнеса Online будет прекращен до 31 июля 2021 г.,** предоставив нашим клиентам два года на планирование переходов в Microsoft Teams, а также в центре для совместной работы в Microsoft 365. Если осталось меньше года, мы хотели бы убедиться, что у пользователей есть сведения и ресурсы, необходимые для планирования и выполнения успешных переходов на Teams.
 
## <a name="why-is-skype-for-business-online-retiring"></a>Почему Skype для бизнеса Online устарел?

Благодаря своему введению Skype для бизнеса Online был ценным инструментом для миллионов людей по всему миру. Объединяя мгновенные сообщения, звонки и видео в одном приложении, Skype для бизнеса Online создал новые возможности для деловой связи. Microsoft Teams — это следующая глава этого видения.

Возможности Microsoft Teams выходят за пределы Skype для бизнеса Online. Объединяя, видеосвязь, звонки, совместная работа с документами и интеграцию приложений в единую среду, Teams позволяет полностью получать новости о работе. И текущие инновационные и разрабатываемые платформы означают, что пользователи Teams пользуются более высокой производительностью, функциональностью, гибкостью и безопасностью.

Teams – это не просто обновление Skype для бизнеса Online, но и мощное средство, которое позволяет компаниям, школах и организациям стать более гибким и повышать эффективность основных рабочих процессов. Узнайте больше о возможных преимуществах Teams для вашей организации в Forrester техническом документе, [а также о том, какие экономические последствия™ Microsoft Teams](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf?rtc=1).


## <a name="what-does-this-mean-for-skype-for-business-customers"></a>Что это означает для пользователей Skype для бизнеса?

После того как Skype для бизнеса Online прекратится 31 июля 2021 г., служба станет недоступна. До этого у пользователей Skype для бизнеса Online не будет никаких изменений и они смогут добавлять новых пользователей по мере необходимости. По состоянию на 1 сентября 2019 г. все новые клиенты Microsoft 365 были добавлены непосредственно в Teams для общения, собраний и звонков.

Время, необходимое для полного перехода в Teams, будет отличаться в зависимости от уникальных характеристик организации, в том числе количества пользователей, технических сложностей и внутренней адаптируемости для изменения. Таким образом, мы рекомендуем вам планировать переход на Teams сегодня, а не только для того, чтобы обеспечить адекватное выполнение успешных переходов с Skype для бизнеса, но для более быстрой реализации преимуществ групп.

Группы доступны в большинстве планов Microsoft 365 для бизнеса и предприятий, а существующие инвестиции в лицензирование переносятся в Teams. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Например, если вы приобрели голосовую конференцию или как часть "" в списке контактов "в Skype для бизнеса, голосовые конференции будут включены в Teams.

На ваш счет в Skype для бизнеса Online не влияет служба поддержки пользователей Skype и сервер Skype для бизнеса.

[Читайте вопросы и ответы](faq-journey.md) по обновлению до Teams и плана возможностей Teams.

## <a name="what-is-microsoft-doing-to-help-customers-transition-to-teams"></a>Что делает Microsoft, чтобы помочь пользователям перейти в Teams?

Воспользуйтесь преимуществами широкого спектра ресурсов, с помощью которых можно планировать развертывание групп и переход с Skype для бизнеса.

- [Документация по развертыванию и обновлению Teams](upgrade-start-here.md) – бесплатные технические рекомендации для ИТ-администраторов.

- [FastTrack для Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) — Справка по развертыванию Teams, которая доступна для подходящих планов.

- [Обучение по Teams Live](https://docs.microsoft.com/microsoftteams/instructor-led-training-teams-landing-page) – бесплатные учебные курсы по Интернету, разработанные для работы с Teams и конечными пользователями.

- [Teams Chalk](https://docs.microsoft.com/MicrosoftTeams/chalk-talks-landing-page) — бесплатные онлайн-семинары для ИТ-специалистов и сотрудников, принимающих решения, которые представляют собой рекомендации по некоторым наиболее популярным и привлекательным сценариям в Teams.

- [Семинары по планированию обновления Teams](https://docs.microsoft.com/MicrosoftTeams/upgrade-workshops-landing-page) – бесплатные интерактивные семинары по планированию обновления, где вы получите рекомендации, рекомендации и ресурсы, которые помогут вам спланировать и внедрить переход на Teams.

- [Партнеры Майкрософт](https://www.microsoft.com/solution-providers/home) — поставщики решений Майкрософт могут помочь вам в полной мере использовать все возможности Teams.

- [Блог Microsoft Teams](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/bg-p/MicrosoftTeamsBlog) – получайте последние новости группы, посвященные новым функциям, ресурсам по внедрению и использованию, устройствам Teams и интеграции с другими бизнес-приложениями.

Если вы являетесь текущим клиентом Skype для бизнеса Online, начните планировать переход в Teams прямо сейчас. Мы рады, что у вас есть широкие возможности общения и совместной работы, и мы стремимся помогать вам на своем пути.

## <a name="skype-for-business-online-retirement-faqs"></a>Вопросы и ответы по выбытием Skype для бизнеса Online

**Смогут ли текущие пользователи Skype для бизнеса Online добавлять новые рабочие места в свой клиент?**<br>
Да. Текущие пользователи Skype для бизнеса Online могут добавлять новые рабочие места в свой клиент, но им следует понимать, что со временем придется перевести этих пользователей на Microsoft Teams.

**Что произойдет со Skype для бизнеса Server?**<br>
Мы понимаем, что пользователи работают в Skype для бизнеса Server, и многие из них должны продолжать использовать Skype для бизнеса Server для некоторых пользователей и географических отсчетов в соответствии с требованиями. В таких случаях мы рекомендуем организациям установить Teams в дополнение к Skype для бизнеса Server, чтобы получить новые возможности командной работы. Новейшая версия Skype для бизнеса Server 2019 была выпущена в октябре 2018 с помощью основной поддержки, запланированной на 9 января 2024 г.

**Что это означает для службы потребителей Skype?**<br>
Выбытие Skype для бизнеса Online не повлияет на службу пользователей Skype.

**Что это означает для существующих пользователей Microsoft 365?**<br>
Мы реализовали в Teams основные возможности Skype для бизнеса Online, новые функции голосовой и видеосвязи, а также новые возможности для работы с собраниями. Мы рекомендуем всем пользователям Microsoft 365 приступить к работе с Teams прямо в своей организации, независимо друг от друга или параллельно с помощью Skype для бизнеса Online и/или Skype для бизнеса Server.

**Продолжите ли вы улучшать Skype для бизнеса Online?**<br>
Мы планируем поддерживать Skype для бизнеса Online до 31 июля 2021 даты ухода. Тем не менее, в Microsoft Teams замена Skype для бизнеса Online в качестве базового коммуникационного решения для Microsoft 365, она будет по-прежнему находиться на стадии разработки и инновациях. Пользователям, которым требуется получить последнюю функциональность и производительность, рекомендуется перейти на Microsoft Teams.

**Что рекомендуется для пользователей, которые в настоящее время находятся в процессе развертывания, или планирования для развертывания — Skype для бизнеса в Microsoft 365?**<br>
Клиенты, в настоящее время развертывающие Skype для бизнеса Online, могут продолжить этот процесс. Мы рекомендуем всем новым пользователям Microsoft 365 пользоваться группами, независимо от Skype для бизнеса и определять путь (сосуществование и только команды) и время, соответствующее их деловым потребностям. Всем клиентам, независимо от срока использования Skype для бизнеса Online, необходимо перейти на Teams до прекращения поддержки Skype для бизнеса Online.

**Что это означает с точки зрения лицензирования? Как клиенты будут оплачивать службы интеллектуальных коммуникаций в Teams?**<br>
Teams поддерживается многими из наборов Microsoft 365. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Сделанные клиентами приобретения по лицензиям переносятся в Teams. Например, если клиент приобрел вместе со Skype для бизнеса компонент аудиоконференций (отдельный или в составе подписки E5), он сохранит доступ к аудиоконференциям и в Teams.

**Собирается ли Майкрософт проводить плановые обновления?**<br>
Мы пока не собираемся проводить подобные обновления для корпоративных клиентов. Клиенты могут перейти на Teams в удобное для организации время до 31 июля 2021 года, когда будет прекращена поддержка Skype для бизнеса Online. Мы поможем администраторам и пользователям перейти на Teams, предоставив им необходимые инструменты и руководства.

Для поддержки меньших пользователей, которые не могут использовать выделенные ИТ-ресурсы, корпорация Майкрософт помогает с автоматическим обновлением из Skype для бизнеса Online в Teams. Соответствующие клиенты получают уведомление об обновлении по электронной почте и в уведомлениях Центра сообщений. В этих сообщениях содержится более подробная информация. Дополнительные сведения см. в статье [Автоматический переход со Skype для бизнеса Online на Microsoft Teams](https://docs.microsoft.com/microsoftteams/upgrade-automated).

**Как прекращение поддержки Skype для бизнеса Online повлияет на организации с гибридными конфигурациями (Skype для бизнеса Online и Skype для бизнеса Server)?**<br>
Клиенты с гибридными конфигурациями, включающими локальную и веб-версию, должны будут перевести пользователей со Skype для бизнеса Online на Microsoft Teams до 31 июля 2021 года. Способы обновления гибридных конфигураций, созданных на принципах сосуществования и взаимодействия, не изменятся после прекращения поддержки Skype для бизнеса Online. Дополнительные сведения см. в статьях [Сосуществование со Skype для бизнеса](coexistence-chat-calls-presence.md) и [Общие сведения о сосуществовании и взаимодействии Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

**Как выбытие Skype для бизнеса Online влияет на использование гибридных конфигураций Skype для бизнеса для локальных подключений по коммутируемой телефонной сети (PSTN) и возможностей телефонной системы в Интернете?**<br>
В некоторых организациях есть разделение пользователей между локальным подключением для подключений по протоколу PSTN и в Интернете для возможностей телефонной системы. После того как вы закончите доступ к Skype для бизнеса Online, организациям потребуется перевести пользователей из Интернета в Microsoft Teams и подключить локальную телефонную сеть к Teams с помощью [прямой маршрутизации](https://docs.microsoft.com/MicrosoftTeams/direct-routing-landing-page).

Клиентам с соответствующими подписками рекомендуется использовать [FastTrack](https://www.microsoft.com/fasttrack?rtc=1) для получения помощи с адаптацией к Teams.
