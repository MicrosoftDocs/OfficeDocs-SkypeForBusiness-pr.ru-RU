# <a name="skype-for-business-online-retirement"></a>Прекращение поддержки Skype для бизнеса Online

В июле 2019 г. мы объявили, что Skype для бизнеса Online будет отменен **31 июля 2021 г.,** и клиентам будет предоставлено два года на планирование перехода на Microsoft Teams, центр командной работы в Microsoft 365. Осталось менее года, поэтому мы хотим, чтобы у клиентов были необходимые сведения и ресурсы для планирования и успешного обновления До Teams.
 
## <a name="why-is-skype-for-business-online-retiring"></a>Почему skype для бизнеса Online больше не существует?

С момента его появления Skype для бизнеса Online стал ценным средством для миллионов людей по всему миру. Благодаря объединению мгновенных сообщений, звонков и видео в одном приложении Skype для бизнеса Online создал новые возможности для делового общения. Microsoft Teams — это следующая глава в этом виде.

Возможности Microsoft Teams выходят за рамки Skype для бизнеса Online. Благодаря объединению чата, видео, звонков, совместной работы с документами и интеграции приложений в единое взаимодействие Teams позволяет работать с новостями. А постоянное инновации и развитие платформы означает, что пользователи Teams получают выгоду от более функциональных возможностей, гибкости и безопасности.

Teams — это не просто обновление Skype для бизнеса Online, это эффективное средство, которое позволяет компаниям, учебным заведениям и организациям повысить гибкость и повысить эффективность ключевых процессов. Узнайте больше о возможных преимуществах Teams для вашей организации в документе Forrester Total [Economic Impact™ Microsoft Teams.](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf?rtc=1)


## <a name="what-does-this-mean-for-skype-for-business-customers"></a>Что это означает для клиентов Skype для бизнеса?

После того как Skype для бизнеса Online завершит работать 31 июля 2021 г., служба станет недоступна. До этого момента текущие клиенты Skype для бизнеса Online не будут иметь изменений в службе и смогут при необходимости добавлять новых пользователей. С 1 сентября 2019 г. все новые клиенты Microsoft 365 были непосредственно переначаты, собрания и вызовы в Teams.

Время, необходимое для полного перехода на Teams, зависит от уникальных характеристик организации, включая количество пользователей, техническую сложность и внутреннюю адаптивность для изменения. Поэтому мы рекомендуем запланируя обновление до Teams сегодня, не только чтобы обеспечить достаточно времени для успешного перехода со Skype для бизнеса, но и для более быстрого реализации преимуществ Teams.

Teams доступен в большинстве планов Microsoft 365 бизнес и корпоративный, и ваши существующие инвестиции в лицензирование переносят в Teams. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Например, если вы приобрели аудиоконференцию как автономный или в составе E5 в Skype для бизнеса, аудиоконференция будет включена в Teams.

Отставку Skype для бизнеса Online не влияет на службу skype для бизнеса и Skype для бизнеса Server.

[Ознакомьтесь с задаваемой](FAQ-journey.yml) вопросом о процессе перехода на Teams и план развития возможностей Teams.

## <a name="what-is-microsoft-doing-to-help-customers-transition-to-teams"></a>Что корпорация Майкрософт делает, чтобы помочь клиентам перейти на Teams?

При планировании развертывания и обновления Skype для бизнеса в Teams вам будут доступны разнообразные ресурсы.

- [Документация по развертыванию и обновлению Teams](upgrade-start-here.md) — бесплатное техническое руководство для ИТ-администраторов.

- [FastTrack для Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) — помощь по развертыванию Teams, доступная для участия в планах.

- [Интерактивные учебные](./instructor-led-training-teams-landing-page.yml) курсы Teams — бесплатные онлайн-учебные курсы, предназначенные для вас и ваших конечных пользователей с помощью Teams.

- [Teams Chalk Talks](./chalk-talks-landing-page.yml) — бесплатные веб-семинары для ИТ-специалистов и лиц, принимающих решения, которые делятся лучшими методиками для некоторых наиболее популярных и привлекательных сценариев в Teams.

- [Семинары по](./upgrade-workshops-landing-page.yml) планированию обновления Teams — бесплатные интерактивные семинары по планированию обновления, на которых вы получите рекомендации, рекомендации и ресурсы, которые помогут вам спланировать и внедрить обновление до Teams.

- [Партнеры Майкрософт](https://www.microsoft.com/solution-providers/home) — поставщики решений Майкрософт могут помочь вам использовать все преимущества Teams.

- [Блог Microsoft Teams](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/bg-p/MicrosoftTeamsBlog) — получение последних новостей Teams, в которые будут представлены новые функции, ресурсы по внедрению и использованию, устройства Teams и интеграция с другими бизнес-приложениями для повышения производительности.

Если вы сейчас клиент Skype для бизнеса Online, начните планировать переход на Teams уже сегодня. Мы рады, что вы одумаетесь в его мощных возможностях для общения и совместной работы, и мы стремимся помочь вам в этом.

## <a name="skype-for-business-online-retirement-faqs"></a>Skype для бизнеса Online: истицы для выхода на пенсию

**Смогут ли текущие пользователи Skype для бизнеса Online добавлять новые рабочие места в свой клиент?**<br>
Да. Текущие пользователи Skype для бизнеса Online могут добавлять новые рабочие места в свой клиент, но им следует понимать, что со временем придется перевести этих пользователей на Microsoft Teams.

**Что произойдет со Skype для бизнеса Server?**<br>
Мы знаем, что клиенты используют Skype для бизнеса Server, и многим из них необходимо продолжать использовать Skype для бизнеса Server для некоторых пользователей или региона в связи с их требованиями. В таких случаях мы рекомендуем организациям установить Teams в дополнение к Skype для бизнеса Server, чтобы получить новые возможности командной работы. Последняя версия Skype для бизнеса Server 2019 выпущена в октябре 2018 г. с основной фазой поддержки, запланированной на 9 января 2024 г.

**Что это означает для службы потребителей Skype?**<br>
Отставку Skype для бизнеса Online не повлияет на службу skype для потребителей.

**Что это означает для существующих клиентов Microsoft 365?**<br>
Мы реализовали в Teams основные возможности Skype для бизнеса Online, новые функции голосовой и видеосвязи, а также новые возможности для работы с собраниями. Мы рекомендуем всем клиентам Microsoft 365 приступить к использованию Teams в организации независимо или рядом со Skype для бизнеса Online или Skype для бизнеса Server.

**Продолжите ли вы улучшать Skype для бизнеса Online?**<br>
Мы планируем поддерживать Skype для бизнеса Online до 31 июля 2021 г. для выхода на пенсию. Однако microsoft Teams заменяет Skype для бизнеса Online в качестве основного коммуникационного решения для Microsoft 365 и будет по-прежнему оставаться в фокусе на разработке и инновации. Клиентам, которые ищут новейшие функции и производительность, рекомендуется перейти на Microsoft Teams.

**Что вы порекомендует клиентам, которые сейчас находятся в процессе развертывания Skype для бизнеса в Microsoft 365 или планируют развертывание?**<br>
Клиенты, в настоящее время развертывающие Skype для бизнеса Online, могут продолжить этот процесс. Мы рекомендуем всем новым клиентам Microsoft 365 приступить к работе с Teams независимо от Skype для бизнеса или параллельно, а также определить путь (сосуществование или только Teams) и время, которое отвечает их бизнес-потребностям. Всем клиентам, независимо от срока использования Skype для бизнеса Online, необходимо перейти на Teams до прекращения поддержки Skype для бизнеса Online.

**Что это означает с точки зрения лицензирования? Как клиенты будут оплачивать службы интеллектуальных коммуникаций в Teams?**<br>
Teams доступен во многих наборах Microsoft 365. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Сделанные клиентами приобретения по лицензиям переносятся в Teams. Например, если клиент приобрел вместе со Skype для бизнеса компонент аудиоконференций (отдельный или в составе подписки E5), он сохранит доступ к аудиоконференциям и в Teams.

**Собирается ли Майкрософт проводить плановые обновления?**<br>
Мы пока не собираемся проводить подобные обновления для корпоративных клиентов. Клиенты могут перейти на Teams в удобное для организации время до 31 июля 2021 года, когда будет прекращена поддержка Skype для бизнеса Online. Мы поможем администраторам и пользователям перейти на Teams, предоставив им необходимые инструменты и руководства.

Для поддержки небольших клиентов, у которые, возможно, нет выделенных ИТ-ресурсов, Корпорация Майкрософт оказывает помощь в автоматическом обновлении Skype для бизнеса Online до Teams. Соответствующие клиенты получают уведомление об обновлении по электронной почте и в уведомлениях Центра сообщений. В этих сообщениях содержится более подробная информация. Дополнительные сведения см. в статье [Автоматический переход со Skype для бизнеса Online на Microsoft Teams](./upgrade-assisted.md).

**Как прекращение поддержки Skype для бизнеса Online повлияет на организации с гибридными конфигурациями (Skype для бизнеса Online и Skype для бизнеса Server)?**<br>
Клиенты с гибридными конфигурациями, включающими локальную и веб-версию, должны будут перевести пользователей со Skype для бизнеса Online на Microsoft Teams до 31 июля 2021 года. Способы обновления гибридных конфигураций, созданных на принципах сосуществования и взаимодействия, не изменятся после прекращения поддержки Skype для бизнеса Online. Дополнительные сведения см. в статьях [Сосуществование со Skype для бизнеса](coexistence-chat-calls-presence.md) и [Общие сведения о сосуществовании и взаимодействии Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

**Как отключение Skype для бизнеса Online повлияет на организации с гибридной конфигурацией Skype для бизнеса для подключения к локальной телефонной сети общего звонков (STN) и сетевых телефонных систем?**<br>
В некоторых организациях пользователи разделены между локальной сетью для подключения к STN и домашней сетью с возможностями телефонной системы. После окончания доступа к Skype для бизнеса Online организациям потребуется перенаправить сетевых пользователей в Microsoft Teams и подключить их к локальной телефонной сети Teams с помощью прямой [маршрутии.](./direct-routing-landing-page.md)

Клиентам с соответствующими подписками рекомендуется использовать [FastTrack](https://www.microsoft.com/fasttrack?rtc=1) для получения помощи с адаптацией к Teams.
