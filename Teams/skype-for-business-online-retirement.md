# <a name="skype-for-business-online-retirement"></a>Прекращение поддержки Skype для бизнеса Online

В июле 2019 г. мы объявили о том, что Skype для бизнеса Online будет отменен **31 июля 2021 г.,** и клиентам будет предоставлено два года для планирования перехода на Microsoft Teams, центр командной работы в Microsoft 365. Осталось менее года, и мы хотим гарантировать, что у клиентов есть информация и ресурсы, необходимые для планирования и успешного обновления Teams.
 
## <a name="why-is-skype-for-business-online-retiring"></a>Почему skype для бизнеса Online больше не будет использовать?

С момента его появления Skype для бизнеса Online был ценным инструментом для миллионов людей по всему миру. Благодаря объединению служб обмена мгновенными сообщениями, звонков и видеосвязи в одном приложении Skype для бизнеса Online создал новые возможности для делового общения. Microsoft Teams — это следующая глава в этой концепции.

Возможности Microsoft Teams выходят за рамки Skype для бизнеса Online. Благодаря объединению чата, видео, звонков, совместной работы с документами и интеграции приложений Teams позволяет работать с абсолютно новостями. А постоянное инновации и развитие платформы означает, что пользователи Teams получают выгоду от более функциональных возможностей, гибкости и безопасности.

Teams — это не просто обновление Skype для бизнеса Online, это мощное средство, которое позволяет компаниям, учебным заведениям и организациям повышать гибкую методику и повысить эффективность ключевых процессов. Узнайте больше о потенциальных преимуществах Teams для вашей организации в документе Forrester ™ Итоговом влиянии [на экономические последствия Microsoft Teams.](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf?rtc=1)


## <a name="what-does-this-mean-for-skype-for-business-customers"></a>Что это означает для клиентов Skype для бизнеса?

С 31 июля 2021 г. служба будет недоступна. Пока же текущие клиенты Skype для бизнеса Online не изменят службу и смогут при необходимости добавлять новых пользователей. С 1 сентября 2019 г. все новые клиенты Microsoft 365 были непосредственно вовсюю в Teams для чата, собраний и звонков.

Время, необходимое для полного перехода на Teams, зависит от уникальных характеристик организации, включая количество пользователей, технические сложности и внутреннюю адаптацию для изменения. Поэтому мы рекомендуем запланируя обновление до Teams уже сегодня, чтобы не только обеспечить достаточно времени для успешного перехода со Skype для бизнеса, но и для более быстрого реализации преимуществ Teams.

Teams доступна в большинстве планов Microsoft 365 бизнес и корпоративный, и ваши существующие инвестиции в лицензирование переносят в Teams. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Например, если вы приобрели аудиоконференцию в рамках E5 в рамках E5, аудиоконференция будет включена в Teams.

Это не влияет на службу Skype для бизнеса и Skype для бизнеса Server.

[Ознакомьтесь с faqs](FAQ-journey.yml) о процессе обновления до Teams и плана развития возможностей Teams.

## <a name="what-is-microsoft-doing-to-help-customers-transition-to-teams"></a>Что корпорация Майкрософт делает для того, чтобы помочь клиентам перейти на Teams?

Для планирования развертывания и обновления Teams с помощью Skype для бизнеса доступны разнообразные ресурсы.

- [Документация по развертыванию и обновлению Teams](upgrade-start-here.md) — бесплатное техническое руководство для ИТ-администраторов.

- [FastTrack для Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) — помощь по развертыванию Teams, доступная для планов, которые квалифицировать можно.

- [Учебные курсы в](./instructor-led-training-teams-landing-page.yml) режиме онлайн-трансляции Teams, разработанные для вас и ваших конечных пользователей.

- [Teams Chalk Talks](./chalk-talks-landing-page.yml) — бесплатные интерактивные семинары для ИТ-специалистов и лиц, принимающих решения, которые делятся лучшими методиками для наиболее распространенных и привлекательных сценариев в Teams.

- [Семинары по](./upgrade-workshops-landing-page.yml) планированию обновления Teams — бесплатные интерактивные семинары по планированию обновления, на которых вы будете получать руководства, рекомендации и ресурсы, которые помогут вам спланировать и внедрить обновление до Teams.

- [Партнеры Майкрософт](https://www.microsoft.com/solution-providers/home) — поставщики решений Майкрософт могут помочь вам использовать все преимущества Teams.

- [Блог Microsoft Teams](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/bg-p/MicrosoftTeamsBlog) — получение последних новостей Teams, в которые будут вовсюю представлены новые функции, ресурсы по внедрению и использованию, устройства Teams и интеграция с другими бизнес-приложениями.

Если вы клиент Skype для бизнеса Online, начните планировать переход на Teams уже сегодня. Мы рады получить от вас мощные возможности для общения и совместной работы, и мы стремимся помочь вам на пути.

## <a name="skype-for-business-online-retirement-faqs"></a>Skype для бизнеса Online: как ото для выхода на пенсию: faq

**Смогут ли текущие пользователи Skype для бизнеса Online добавлять новые рабочие места в свой клиент?**<br>
Да. Текущие пользователи Skype для бизнеса Online могут добавлять новые рабочие места в свой клиент, но им следует понимать, что со временем придется перевести этих пользователей на Microsoft Teams.

**Что произойдет со Skype для бизнеса Server?**<br>
Мы понимаем, что клиенты используют Skype для бизнеса Server, и многим из них необходимо продолжать использовать Skype для бизнеса Server для некоторых пользователей или региона в связи с их требованиями. В таких случаях мы рекомендуем организациям установить Teams в дополнение к Skype для бизнеса Server, чтобы получить новые возможности командной работы. Последняя версия Skype для бизнеса Server 2019 выпущена в октябре 2018 г. с основной фазой поддержки, запланированной на 9 января 2024 г.

**Что это означает для потребительской службы Skype?**<br>
Это не повлияет на службу Skype для бизнеса Online.

**Что это означает для существующих клиентов Microsoft 365?**<br>
Мы реализовали в Teams основные возможности Skype для бизнеса Online, новые функции голосовой и видеосвязи, а также новые возможности для работы с собраниями. Мы рекомендуем всем клиентам Microsoft 365 приступить к использованию Teams в организации независимо или наряду со Skype для бизнеса Online и (или) Skype для бизнеса Server.

**Продолжите ли вы улучшать Skype для бизнеса Online?**<br>
Мы планируем поддерживать Skype для бизнеса Online до 31 июля 2021 г. Однако благодаря Microsoft Teams, заменив Skype для бизнеса Online в качестве основного решения для связи для Microsoft 365, эта служба будет продолжать оставаться в центре разработки и инноваций. Клиентам, которые ищут новейшие функции и производительность, рекомендуется перейти на Microsoft Teams.

**Что вы порекомендует пользователям, которые сейчас находятся в процессе развертывания (или планировании развертывания) Skype для бизнеса в Microsoft 365?**<br>
Клиенты, в настоящее время развертывающие Skype для бизнеса Online, могут продолжить этот процесс. Мы рекомендуем всем новым клиентам Microsoft 365 приступить к работе с Teams независимо от Skype для бизнеса или параллельно, а также определить путь (сосуществование или только Teams) и сроки, отвечающие требованиям их бизнеса. Всем клиентам, независимо от срока использования Skype для бизнеса Online, необходимо перейти на Teams до прекращения поддержки Skype для бизнеса Online.

**Что это означает с точки зрения лицензирования? Как клиенты будут оплачивать службы интеллектуальных коммуникаций в Teams?**<br>
Teams доступна во многих наборах Microsoft 365. Возможности, являющиеся платными рабочими нагрузками Skype для бизнеса Online, останутся таковыми и в Teams. Сделанные клиентами приобретения по лицензиям переносятся в Teams. Например, если клиент приобрел вместе со Skype для бизнеса компонент аудиоконференций (отдельный или в составе подписки E5), он сохранит доступ к аудиоконференциям и в Teams.

**Собирается ли Майкрософт проводить плановые обновления?**<br>
Мы пока не собираемся проводить подобные обновления для корпоративных клиентов. Клиенты могут перейти на Teams в удобное для организации время до 31 июля 2021 года, когда будет прекращена поддержка Skype для бизнеса Online. Мы поможем администраторам и пользователям перейти на Teams, предоставив им необходимые инструменты и руководства.

Для поддержки небольших клиентов, которые, возможно, не выделены ИТ-ресурсы, корпорация Майкрософт помогает автоматизировать обновление Skype для бизнеса Online до Teams. Соответствующие клиенты получают уведомление об обновлении по электронной почте и в уведомлениях Центра сообщений. В этих сообщениях содержится более подробная информация. Дополнительные сведения см. в статье [Автоматический переход со Skype для бизнеса Online на Microsoft Teams](./upgrade-automated.md).

**Как прекращение поддержки Skype для бизнеса Online повлияет на организации с гибридными конфигурациями (Skype для бизнеса Online и Skype для бизнеса Server)?**<br>
Клиенты с гибридными конфигурациями, включающими локальную и веб-версию, должны будут перевести пользователей со Skype для бизнеса Online на Microsoft Teams до 31 июля 2021 года. Способы обновления гибридных конфигураций, созданных на принципах сосуществования и взаимодействия, не изменятся после прекращения поддержки Skype для бизнеса Online. Дополнительные сведения см. в статьях [Сосуществование со Skype для бизнеса](coexistence-chat-calls-presence.md) и [Общие сведения о сосуществовании и взаимодействии Microsoft Teams и Skype для бизнеса](teams-and-skypeforbusiness-coexistence-and-interoperability.md).

**Как отключение Skype для бизнеса Online повлияет на организации с гибридными конфигурациями Skype для бизнеса для подключения к локальной телефонной сети общего звонков (ННР) и телефонной системы?**<br>
В некоторых организациях пользователи разделены между локальной службой связи через STN и веб-службами телефонной системы. После окончания доступа к Skype для бизнеса Online организации должны будут перейти на Microsoft Teams пользователей, которые подключаются к локальной телефонной сети, с помощью прямой [маршрутки.](./direct-routing-landing-page.md)

Клиентам с соответствующими подписками рекомендуется использовать [FastTrack](https://www.microsoft.com/fasttrack?rtc=1) для получения помощи с адаптацией к Teams.
