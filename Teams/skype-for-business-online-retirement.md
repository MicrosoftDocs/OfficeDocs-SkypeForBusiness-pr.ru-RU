# <a name="skype-for-business-online-retirement"></a>Прекращение поддержки Skype для бизнеса Online

31 июля 2021 г. корпорация Майкрософт больше не Skype для бизнеса Online. В июле 2019 г. было объявлено об этом, чтобы клиенты за два года заблаговременно спланировали обновление до Microsoft Teams. Teams является основным приложением для общения и совместной работы в Microsoft 365. После Skype для бизнеса Online корпорация Майкрософт хочет, чтобы у клиентов были необходимые сведения и ресурсы для планирования и успешного обновления Teams.  Это Skype не влияет на службу потребителей. Чтобы узнать, почему Skype для бизнеса Online была отменена, см. часто задаваемую Skype для бизнеса до [Microsoft Teams](FAQ-journey.yml).

Корпорация Майкрософт начнет списание инфраструктуры Skype для бизнеса Online не более 30 июня 2022 г. Эта статья содержит рекомендации для организаций с пользователями TeamsOnly, которые были обновлены с любой версии Skype для бизнеса.


## <a name="organizations-with-on-premises-deployments-of-skype-for-business-server"></a>Организации с развертыванием Skype для бизнеса Server

При этом Skype для бизнеса Online не влияет на поддержку локального развертывания Skype для бизнеса Server и Lync Server 2013. Тем не менее гибридные клиенты, которые могут работать как в Интернете, так и локально, должны обновить всех *пользователей в* Интернете. Все пользователи в сети должны быть назначены в режиме TeamsOnly с помощью TeamsUpgradePolicy. Корпорация Майкрософт предоставляет помощь в обновлении, чтобы автоматизировать обновление оставшихся пользователей Skype для бизнеса Online до режима TeamsOnly. В результате этого решения гибридные организации не *должны перемещать Skype для бизнеса* пользователей в облако. Корпорация Майкрософт полностью поддерживает гибридные организации с сочетанием пользователей TeamsOnly и Skype для бизнеса пользователей. Клиентам с гибридными развертываниями Skype для бизнеса Server или Lync Server 2013 следует просмотреть статью Skype для бизнеса [Online](/skypeforbusiness/hybrid/plan-hybrid-connectivity#implications-of-the-upcoming-retirement-of-skype-for-business-online).

## <a name="what-to-expect-post-retirement"></a>Что ожидается после выхода на пенсию

Пользователи, которые работают в облаке, больше не могут получить доступ к режиму, кроме TeamsOnly. Это означает:

 - При лицензировании новых пользователей, которые не находятся в локальной Skype для бизнеса Server, пользователям автоматически назначен режим TeamsOnly независимо от глобальной политики клиента TeamsUpgradePolicy.
 - В гибридных организациях при перемещении пользователей, которые находятся локально в облаке, пользователям автоматически назначен режим TeamsOnly `MoveToTeams` независимо от того, был ли в нем указан переключатель `Move-CsUser`.
 - Пользователям, которые работают в облаке, не может быть назначен режим, кроме TeamsOnly. Пользователи, которые являются пользователями в Интернете *,* не Skype для бизнеса локальном сервере.

Клиенты могут остаться пользователями, которые находятся в Skype для бизнеса Online и еще не назначены в режиме TeamsOnly. Клиенты должны как можно скорее назначить этим пользователям режим TeamsOnly. Корпорация Майкрософт будет предоставлять поддержку обновления для пользователей Skype для бизнеса Online, не в режиме TeamsOnly. Возможность обновления зависит от того, является ли ваша организация локальной или локальной Skype для бизнеса пользователями. Дополнительные сведения см[. в этой Skype для бизнеса Online Microsoft Teams](upgrade-assisted.md).

После завершения обновления все пользователи будут работать в режиме TeamsOnly. Пользователи в режиме TeamsOnly получают входящие чаты и звонки Teams, а также планировать собрания в Teams. Они не могут начинать чаты, звонки и планировать собрания в Skype для бизнеса Online.  Однако пользователи TeamsOnly могут присоединяться Skype для бизнеса собрания, которые у них уже есть или которые будут принимать в будущем. Наконец, *все пользователи, которые работают* локально, остаются локально и не будут использовать TeamsOnly.

## <a name="actions-to-take-before-june-30-2022"></a>Действия, которые необходимо принять до 30 июня 2022 г.
После Skype для бизнеса Online корпорация Майкрософт начнет списание вспомогательной инфраструктуры не ранее 30 июня 2022 г.  Для любой организации с пользователями TeamsOnly, которые были обновлены с любой версии Skype для бизнеса, необходимо принять меры, если это относится к одной из этих ситуаций:

- Если у вас есть пользователи TeamsOnly, у которых в предварительном обновлении были контакты Skype для бизнеса и которые еще не  вошли в Teams после обновления.
- Если у вас есть пользователи TeamsOnly, у Skype для бизнеса собрания по сети, которые они организовывали до обновления до TeamsOnly.

Если одна из этих ситуаций относится к вашей организации, необходимо принять меры до 30 июня 2022 г., как описано ниже.

 - **Skype для бизнеса** контактов в Сети. После обновления пользователя до режима TeamsOnly при первом входе пользователя в Teams все существующие контакты из учетной записи Skype для бизнеса Online будут перенесены в Teams. После удаления инфраструктуры Skype для бизнеса Online вы больше не сможете перенести контакты пользователей, которые еще не вошли *в Teams.* Чтобы перенести контакты из Skype для бизнеса в Teams, корпорация Майкрософт рекомендует пользователям, которые ранее Skype для бизнеса вход Teams хотя бы один раз до 30 июня 2022 г.

 - **Skype для бизнеса собраний** по сети: после обновления организации до TeamsOnly пользователи создают все новые собрания в Teams собраний. В некоторых случаях пользователи TeamsOnly могут по-прежнему Skype для бизнеса собрания по сети, которые они организовывали ранее. В настоящее время обновленные пользователи TeamsOnly и все приглашенные участники могут присоединиться к Skype для бизнеса собраниям по сети с помощью Skype для бизнеса клиента. Однако после удаления инфраструктуры Skype для бизнеса Online для определенного пользователя TeamsOnly все оставшиеся собрания Skype для бизнеса, организованные этим пользователем, больше не будут существовать. Организатор и все участники не смогут присоединиться к этим собраниям. Если у пользователей в организациях TeamsOnly есть оставшиеся Skype для бизнеса собрания по сети, которые они организовывали, корпорация Майкрософт рекомендует повторно захещать эти собрания Teams собраниями. Кроме того, администраторы могут использовать службу [переноса](/skypeforbusiness/audio-conferencing-in-office-365/setting-up-the-meeting-migration-service-mms#trigger-meeting-migration-manually-via-powershell-cmdlet) собраний, чтобы преобразовать эти собрания в Teams собрания. В обоих случаях выполните эти действия до 30 июня 2022 г.

- [Вспомогательные обновления с Skype для бизнеса Online до Microsoft Teams](upgrade-assisted.md) — автоматизированная программа, которая помогает обновить Skype для бизнеса Online до Teams.
