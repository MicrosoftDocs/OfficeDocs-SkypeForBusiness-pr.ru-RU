<a name="install-microsoft-teams-using-msi-with-sccm"></a>Установка группами Майкрософт, с помощью MSI (с SCCM)
===========================================

Корпорация Майкрософт условии, что файл MSI для "Администраторы" для массового развертывания групп для выбора пользователей или компьютеров. Администраторы могут использовать этот файл, чтобы удаленно развертывание групп, чтобы пользователи не имеют для ручной загрузки приложения группы. System Center Configuration Manager (SCCM) — это основное средство, используемое с помощью MSI-файлов для управляемых развертываний.
