<a name="manage-the-microsoft-teams-commercial-cloud-trial-offer"></a>Пробное предложение Microsoft Teams Commercial Cloud Trial
=======================================================

Группами Майкрософт — это превосходное средство совместной работы для вашей организации. Она дает людей и групп обсуждения, инновации и обмен информацией идеи, с помощью Office 365. Microsoft группами коммерческие облачных пробной версии предлагает существующих пользователей Office 365 в вашей организации, у которых нет лицензии для группами Майкрософт начать пробную версию 1 год продукта. «Администраторы» можно переключить этот компонент включено или отключено для пользователей в своей организации.

## <a name="whats-in-the-offer"></a>Что такое предложения

Планы обслуживания, включенных в это предложение являются:

- Exchange Foundation
- Поток для Office 365 (план 1)
- Планировщик работы Microsoft
- Группами Майкрософт (Teams1, группами IW)
- Office Online
- PowerApps для Office 365 разработать план 1
- SharePoint Online — базовые подписки
- Sway
- Сеть Yammer Enterprise

Пробная версия предоставляет пробной подписки один год во всей организации. Пробная версия делает доступными для назначения лицензий на 500 000. Для каждой лицензии, назначенной ознакомительной выделяет 2 ГБ SharePoint Online. 

## <a name="who-is-eligible"></a>Кто имеет право

Пользователям должна быть включена для регистрации приложений и пробные версии (в центре администрирования Office 365). Для получения дополнительных сведений см [ознакомительной](#manage-the-trial) далее в этой статье. 

Пользователи, у которых нет лицензии Office 365, которая включает команды можно инициировать предложение коммерческие облачных пробная версия Microsoft группами. Например если пользователь имеет Office 365 для бизнеса (который сюда не включены группы), они могут быть использованы для пробной версии.

## <a name="who-is-not-eligible"></a>Кто является недопустимой

Ваша организация не подходит для пробного if: 

- Вы являетесь клиентом синдикации партнера
- Вы являетесь клиентом партнера торговых посредников
- Вы являетесь клиентом государственных организаций или EDU

Если организация не имеет права Microsoft группами коммерческие облачных предоставление пробной версии, не появится параметр **Пользователи могут установить пробной версии приложений и служб** .

## <a name="how-users-sign-up-for-the-trial"></a>Как пользователи зарегистрироваться для использования пробной версии

Пользователей можно Подпишитесь предоставление пробной версии, ведение журнала в группы ([teams.microsoft.com](https://teams.microsoft.com)). Он будет видеть следующие экрана, чтобы начать пробную версию. 

![Снимок экрана с начальной страницы пробную версию информационный Работник группами.](media/iw-trial-start-screen.png)

Все пробные версии вашей организации совместно использовать же даты начала и окончания, которые — это дата, первый пользователь подписались на пробную версию. К примеру Если пользователь начинает первый пробная версия на 25 января 2019 и пользователь Б запускает пробная версия на 3 июня 2019, пробная версия оба пользователя истекает 25 января 2020 г.

## <a name="manage-the-trial"></a>Управление пробной версии

Администраторы могут управлять лицензиями для пользователей, которые уже зарегистрировались. 

Кроме того Администраторы можно отключить возможность конечным пользователям утверждения пробной версии приложений и служб в пределах организации. На данный момент пробная версия, описанного в данной статье является только пробной версии в этой категории, но его применения другие аналогичные программы в будущем. 

### <a name="prevent-users-from-installing-trial-apps-and-services"></a>Запретить пользователям устанавливать пробной версии приложений и служб

Можно отключить возможность установки пробной версии приложений и служб.

1. [Центр администрирования Microsoft 365](https://portal.office.com/adminportal/home), перейдите в раздел **Параметры** > **службы & надстроек** > **пользователя владельцем приложений и служб**.

    ![Снимок экрана страницы служб & надстроек в центре администрирования Office 365.](media/iw-trial-enable-1.png)

2. Отключите **Пользователи могут установить пробной версии приложений и служб**.

    ![Снимок экрана пользователя, владельцем которого страницы приложений и служб в центре администрирования Office 365.](media/iw-trial-enable-2.png)


### <a name="manage-trial-availability-for-a-user-with-a-license-that-includes-teams"></a>Управление пробного доступность для пользователей с лицензией, которая включает в себя группы

Пользователей, которым назначена лицензия, которая включает команды не имеет права использования пробной версии. При включении план обслуживания группы пользователя можно вход и используйте команды. Если план обслуживания отключен, пользователь не может войти в и не представлены с параметром пробной версии либо.

Чтобы отключить доступ к группам:

1. В центре администрирования Microsoft 365 выберите **пользователей** > **Активные пользователи**.

2. Установите флажок рядом с именем пользователя.

3. В правой части окна в строке **лицензий на продукт** , выберите **Изменить**.

4. На левой панели **лицензий на продукт** переключитесь в переключателя **отключено**.

    ![Снимок экрана страницы лицензий продукта в центре администрирования Office 365.](media/iw-trial-enable-3.png)

### <a name="manage-teams-availability-for-users-who-already-claimed-the-trial"></a>Управление группами доступности для пользователей, которые уже запросил пробной версии

Если пользователь затребованы Пробная лицензия групп, его можно удалить путем удаления план лицензии или службы.

Чтобы отключить лицензии для пробного использования:

1. В центре администрирования Microsoft 365 выберите **пользователей** > **Активные пользователи**.

2. Установите флажок рядом с именем пользователя.

3. В правой части окна в строке **лицензий на продукт** , выберите **Изменить**.

4. На левой панели **лицензий на продукт** переключитесь в переключателя **отключено**.

    ![Снимок экрана с параметр лицензии для пробного использования групп в области лицензий на продукт](media/iW-trial-enable-4.png)

### <a name="manage-teams-for-users-who-have-the-trial-license"></a>Управление группами для пользователей, имеющих лицензии для пробного использования

Может управлять пользователей, имеющих Пробная лицензия так же, как управлять пользователей, имеющих регулярных платной лицензии. Для получения дополнительных сведений см [Управление группами Майкрософт в организации Office 365](enable-features-office-365.md).

### <a name="upgrade-users-from-the-trial-license"></a>Обновление пользователей с лицензии для пробного использования

Для обновления пользователей от лицензии для пробного использования, выполните следующие действия.

1. Приобретение подписки, которая включает в себя группы.

2. Удаление группы пробной подписки от пользователя.

3. Назначьте вновь приобретенные лицензии.

Дополнительные сведения: [Лицензирование Office 365 для Microsoft Teams](Office-365-licensing.md).
