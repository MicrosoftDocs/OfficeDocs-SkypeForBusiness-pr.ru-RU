# <a name="data-transfers-between-carriers-and-microsoft-for-operator-connect"></a>Передача данных между операторами и корпорацией Майкрософт для Подключение

Корпорация Майкрософт — это контроллер личных данных, который обрабатывается в связи с оператором Подключение. Клиент должен уведомить конечных пользователей о следующих данных:

- С помощью функций Подключение операторов майкрософт для Microsoft Teams конечные пользователи могут общаться с Teams посредством звонков в Microsoft Teams.
- Если эта функция включена в вашей организации, Корпорация Майкрософт передает информацию операторам связи вашей организации (каждому оператору), чтобы обеспечить надлежащее функционирование службы и в целях устранения неполадок.

Сведения, которые Корпорация Майкрософт передает оператору при использовании оператора Подключение включает:

- ваш номер телефона и ,если применимо, назначенное Microsoft SIP URI (УНИФИЦИРОВАННЫЕ идентификаторы ресурсов протокола сеанса), назначенное вашему номеру телефона;
- сведения о звонке и данные о качестве звонка, например номера телефонов получателя; время и время окончания звонка; определенные идентификаторы вызываемой записи; длительность вызова; и сведения о подключениях к службам (например, IP-адреса и операционные системы вызываемого и вызываемого получателя); И
- сведения о местоположении экстренных вызовов.

Операторы смогут экспортировать и скачивать указанные выше сведения.

Оператор будет обрабатывать эти сведения в соответствии с правилами конфиденциальности вашего оператора. Дополнительные сведения можно найти в политике конфиденциальности вашего оператора и условиях использования. Сведения о политиках конфиденциальности корпорации Майкрософт можно найти в уведомлении о конфиденциальности корпорации Майкрософт, применимом Teams [по этой ссылке.](https://go.microsoft.com/fwlink/?LinkId=521839)
