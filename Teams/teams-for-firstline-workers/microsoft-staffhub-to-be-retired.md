# <a name="microsoft-staffhub-to-be-retired"></a>Прекращение поддержки Microsoft StaffHub

Начиная с 1 октября 2019, Microsoft StaffHub будет из эксплуатации. Мы создаем StaffHub возможностями, включая расписание и задачи управления в группами Майкрософт. Мобильное приложение группами теперь включает управление shift и качества начального экрана. Дополнительные возможности для работников firstline будут развернуты в группы по времени. 

Эти изменения являются частью нашей продолжением работы предоставить всех сотрудников с Microsoft 365. С возможностями для работников firstline теперь в группах каждого сотрудника в вашей организации могут использовать команды для упрощения их рабочего дня, совместно работать с коллегами и сведения о доступе и опыт, помогая им эффективную работу.

## <a name="frequently-asked-questions"></a>Вопросы и ответы

- **Когда будет Microsoft StaffHub из эксплуатации?**<br> Запуск 1 апреля 2019, Microsoft StaffHub больше не будут доступны для новых клиентов и все остальные точки доступа. Текущий пользователи по-прежнему смогут использовать службу до 1 октября 2019, однако не дополнительные функции будут доступны.

- **Что произойдет, если не поддерживается StaffHub?**<br>Microsoft StaffHub лицензий для всех пользователей на 1 октября 2019. Кто-то пытается открыть приложение будет выведено сообщение, предлагая, чтобы загрузить группами Майкрософт. Условии, что у пользователя есть активная лицензия, которая включает команды, данные и основные функциональные возможности из StaffHub будут доступны им при их переходе на группы.
Между апреля 2019 и 2019 октября StaffHub пользователи будут получать уведомления в приложении стимулирования их использование групп Майкрософт.

- **Что нужно сделать, чтобы подготовиться к этому изменению?**<br>ИТ-администраторов клиенты с пользователями active Microsoft StaffHub будут получать уведомления в центре сообщений Office 365 на действия, которые необходимо предпринять для перехода на приложение смен в группах. Сведения о способах миграции с StaffHub на смены в группах, включая средства, помогающие с миграцией, скоро будет доступен.

    Пользователи будут получать сообщения о о пенсионного обеспечения приложения через уведомлений из приложения, но может потребоваться передачи изменения для них также и обновление любой внутренний документации, который относится к Microsoft StaffHub.

    Перейдите на эту страницу для получения дополнительных сведений о том, как успешно переходе от StaffHub для смены в группах, включая перенос данных, управление новые функциональные возможности StaffHub, которые мы создаем в группы (в & группами Майкрософт Скайп для администратора предприятия Центр) и другие сведения о действия, необходимые для администраторов и конечных пользователей.

- **Группами Майкрософт предлагает все возможности, предоставляемые в настоящее время в Microsoft StaffHub?**<br>Да, начиная с ноября 2018, группами Майкрософт будет предлагать возможность смены, которая позволяет менеджерам создание, обновление и Управление расписаниями и позволяет публиковать их доступность, смены файл подкачки и время запроса сотрудников. Кроме того Домашняя страница в мобильное приложение команды будет отображаться расписание и объявления важные компании сотрудника. 

    Для получения дополнительных сведений, [Управление приложение смены для вашей организации в группах](manage-the-shifts-app-for-your-organization-in-teams.md)перейдите "Администраторы". Пользовательская справка по использованию смен в группах, извлекать [Смены Справка по firstline работников и менеджеров](https://support.office.com/article/apps-and-services-cc1fba57-9900-4634-8306-2360a40c665b). 

    [Схема](https://www.microsoft.com/microsoft-365/roadmap?filters=) полные сведения можно найти на дополнительные функции освободить в группах.

- **Почему недоступны Моя рабочая группа StaffHub из приложения смен в группах**<br>Команды, которые были созданы с помощью Microsoft StaffHub только можно просматривать и осуществляется в StaffHub. Поддержка для переноса ваша группа StaffHub смены приложения в группах выпуске, можно будет просматривать расписания StaffHub в смены.
