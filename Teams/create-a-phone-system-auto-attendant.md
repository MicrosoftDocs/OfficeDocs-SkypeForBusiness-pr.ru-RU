# <a name="set-up-an-auto-attendant"></a>Настройка автозавода

Автозаверяющие могут звонить в вашу организацию и переходить в систему меню, чтобы поговорить с нужным отделом, очередью вызовов, человеком или оператором. Вы можете создавать автоотводы для своей организации с помощью Microsoft Teams центра администрирования или PowerShell.

> [!TIP]
> Эта статья посвящена крупным организациям. Если ваша организация — малый бизнес, ознакомьтесь с учебным учебником Настройка автозавода [для малого](/microsoftteams/business-voice/create-a-phone-system-auto-attendant-smb) бизнеса.

Прежде чем выполнять действия, которые вы выполните в этой [](plan-auto-attendant-call-queue.md#getting-started) [статье, ознакомьтесь](plan-auto-attendant-call-queue.md) со статьей Планирование работы с Teams и очередей вызовов и следуйте этим шагам.

Автоответчики могут перенаправить звонки в одно из следующих расположений: <a name="call-routing-options" ></a>

- **Оператор** — оператор, определенный для автозавода. Определение оператора необязательно. Оператор можно определить как любой другой пункт назначения в этом списке.
- **Человек в организации —** это человек в вашей организации, который может принимать голосовые звонки. Этот пользователь может быть пользователем в Интернете или локально, используя Skype для бизнеса Server.
- **Голосовое приложение** — еще один автозаводщик или очередь вызовов. (Выберите учетную запись ресурса, связанную с автоотправлением или очередью вызовов при выборе этого назначения.)
- **Голосовая почта** — почтовый ящик голосовой почты, связанный с Microsoft 365 группой, которую вы указали. Вы можете выбрать транскрибацию голосовой почты и сообщение "Пожалуйста, оставьте сообщение после сигнала". системный запрос.
- **Внешний номер телефона** — любой номер телефона. (См. [технические сведения о внешнем переносе).](create-a-phone-system-auto-attendant.md#external-phone-number-transfers---technical-details)
- **Объявление (звуковой файл)** — воспроизведения звукового файла. Опубликованное сообщение с объявлением, сохраненное как звуковое. WAV, .MP3 или . Формат WMA. Размер записи не должен быть больше 5 МБ. Система воспроизводит объявление, а затем возвращается в меню автозапуска.
- **Объявление (тип)** — введите сообщение. Текст, который должна читать система. Можно ввести до 1000 символов. Система воспроизводит объявление, а затем возвращается в меню автозапуска.

При этом вам будет предложено выбрать один из этих параметров на разных этапах настройки автозаказа.

Чтобы настроить автоотображающий, в Центре администрирования Teams **Разорите голосовую** обработку , выберите Автоотображающие **,** а затем выберите **Добавить**.

## <a name="general-info"></a>Общие сведения

![Снимок экрана: параметры автозавода для имени, оператора, часового пояса, языка и голосового ввода](media/auto-attendant-general-info-page-new.png)

1. Введите имя автозавода в поле в верхней части окна.

2. Чтобы назначить оператора, укажите место назначения для звонков оператору. Это необязательный (но рекомендуемый вариант). Установите параметр **Оператор,** чтобы разрешить вызывателям выйти из меню и поговорить с назначенным человеком.

3. Укажите часовой пояс для этого автозавода. Часовой пояс используется для вычисления рабочих часов, если вы создаете отдельный поток [зовов в не часах.](#call-flow-for-after-hours)

4. Укажите [поддерживаемый язык](create-a-phone-system-auto-attendant-languages.md) для этого автозавода. Этот язык будет использоваться для системных голосовых подсказок.

5. Выберите, нужно ли включить голосовой ввод. Если этот параметр включен, имя каждого параметра меню становится ключевым словом распознавания речи. Например, можно сказать "Один", чтобы выбрать параметр меню, который назначен клавише 1, или слово "Продажи", чтобы выбрать пункт меню "Продажи".

   > [!NOTE]
   > Если на шаге 4 выбрать язык, который не поддерживает голосовой ввод, этот параметр будет отключен.

6. Выберите **Далее**.

## <a name="call-flow"></a>Поток вызовов

![Снимок экрана: параметры приветствия сообщения](media/auto-attendant-call-flow-greeting-message.png)

Выберите, хотите ли вы воспроизведения приветствия, когда автоответ отвечает на звонок.

Если вы **выберете "Звукозапись",** с помощью Upload файла можно добавить записанное приветствие, сохраненное в качестве звукового файла.  WAV, .MP3 или . Формат WMA. Размер записи не должен быть больше 5 МБ.

Если выбрать **Введите приветствие,** система будет читать текст, который вы введите (до 1000 символов), когда автозавет ответит на звонок.

![Снимок экрана: параметры маршрутизов параметров перенаходящего звонка](media/auto-attendant-call-flow-route-call-message.png)

Выберите, как вы хотите перена маршрутизовыть звонок.

Если вы **выберете Отключить**, автозаводщик повесит вызов.

Если вы выбрали **Перенаправить** звонок , вы можете выбрать одно из назначений маршрутизов.

Если вы выбрали **Параметры меню**  Воспроизведения, вы  можете выбрать вариант Воспроизведения звукового файла или Ввести приветствие, а затем выбрать параметры меню и поиск по каталогу.

### <a name="menu-options"></a>Параметры меню

![Снимок экрана: параметры набора номера](media/auto-attendant-call-flow-menu-options-complete.png)

Чтобы задать параметры набора, назначьте 0–9 клавиш на телефонной клавиатуре одному из назначений маршрутизов. (Клавиши \* (Повтор) и (Назад) зарезервированы системой и не \# могут быть перена назначены.)

Сопоставления ключей не должны быть непрерывными. Можно создать меню с клавишами 0, 1 и 3, относясь к параметрам, при этом клавиша 2 не используется.

Если вы настроили этот оператор, рекомендуем соендировать с оператором нулевые клавиши. Если оператор не назначен ни одной клавише, голосовая команда "Оператор" также отключена.

Для каждого параметра меню укажите следующие параметры:

- **Клавиша набора** номера — клавиша на телефонной клавиатуре для доступа к этому параметру. Если доступны голосовые данные, вызыватели также могут сказать этот номер, чтобы получить доступ к параметру.

- **Голосовая** команда — определяет голосовую команду, которую может предоставить вызываемая команда для доступа к этому параметру, если включены голосовые данные. Оно может содержать несколько слов, например "Обслуживание клиентов" или "Операции и причины". Например, можно нажать клавишу 2, сказать "два" или "Продажи", чтобы выбрать параметр, выбранный с двумя ключами. Этот текст также отрисовывания текста в речь при запросе подтверждения службы, который может выглядеть так: "Перена передаче звонка в отдел продаж".

- **Redirect to** (Перенаправление на) — пункт назначения маршрутиации пунктов, используемых при выборе этого параметра вызывателями. Если вы перенаправляете учетную запись к автоотправлению или очереди вызовов, выберите связанную с ней учетную запись ресурса.

### <a name="directory-search"></a>Поиск по каталогу

При назначении пунктов назначениям набираемой клавиши рекомендуется выбрать **вариант Нет** для **поиска по каталогу.** Если звоняющий пытается набрать имя или расширение с помощью клавиш, которые назначены определенным пунктам назначения, он может быть неожиданно перенаправлен в определенное место, прежде чем вводить имя или расширение. Мы рекомендуем создать отдельный автозаводщик для поиска по каталогу и связаться с ним с помощью клавиши набора номера.

Если вы не назначили клавиши набора номера, выберите параметр поиска по **каталогу.**

**Набрать номер по** имени. Если этот параметр включить, вызыватели смогут ввести имя пользователя или ввести его на телефонной клавиатуре. Любой пользователь в Интернете или любой пользователь, использующий локальное Skype для бизнеса Server, является подходящим пользователем, и его можно найти с помощью набора по имени. (На странице "Область набора номера" можно у установить, кто является и не включен в [каталог.)](#dial-scope)

**Набрать номер по расширению.** Если вы встроите этот параметр, вызывающие пользователи смогут связываться с пользователями в организации, набрав их расширения для телефона. Любой пользователь в Интернете или любой пользователь, использующий локальное Skype для бизнеса Server, является подходящим пользователем, и его можно найти с расширением **Dial by**.. (На странице "Область набора номера" можно у установить, кто является и не включен в [каталог.)](#dial-scope)

Для пользователей, которые вы хотите сделать доступными для расширения "Набор номера", необходимо добавить расширение, как часть одного [](/microsoft-365/admin/add-users/add-users) из следующих атрибутов телефонов, определенных в Active Directory или Azure Active Directory (дополнительные сведения см. в документе Добавление пользователей по одному или массово).)

- OfficePhone
- HomePhone
- Mobile/MobilePhone
- TelephoneNumber/PhoneNumber
- OtherTelephone

Формат, требуемого для ввода расширения в поле номера телефона пользователя, может иметь один из следующих форматов:

- *+\<phone number>;ext=\<extension>*
- *+\<phone number>X\<extension>*
- *X\<extension>*

- Пример 1. Set-MsolUser -UserPrincipalName usern@domain.com -Phonenumber "+1555555678;ext=5678"
- Пример 2. Set-MsolUser -UserPrincipalName usern@domain.com -Phonenumber "+1555555678x5678"
- Пример 3. Set-MsolUser -UserPrincipalName usern@domain.com -Phonenumber "x5678"

Вы можете установить расширение в центре администрирования [Microsoft 365](https://admin.microsoft.com/) или Azure Active Directory [центре администрирования](https://aad.portal.azure.com). Изменения, внесенные автоследникам и очередям вызовов, могут быть доступны в течение 12 часов.

> [!NOTE]
> Если вы хотите использовать  функции  "Набрать номер по имени" и "Набрать по расширению", назначьте клавишу набора главному автоотеке, чтобы связаться с автозаводом, включенным для функции "Набрать по **имени".** В пределах этого автозавода вы можете назначить 1-й  ключ (без букв), чтобы связаться с автозаводом для набора номера по расширению.

Выбрав параметр поиска по **каталогу, выберите** **Далее.**

## <a name="call-flow-for-after-hours"></a>Поток вызовов в нечасовом времени

![Снимок экрана: параметры дня и времени после окончания дня](media/auto-attendant-business-hours.png)

Для каждого автозавода можно настроить часы работы. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. В течение дня можно настроить часы работы с перерывом, а все часы, которые не считаются неавтными, считаются неавтными. Вы можете настроить различные параметры обработки входящих параметров обработки звонка и приветствия в течение часа.

В зависимости от настройки автозаполнения и очередей звонков может потребоваться указать маршрутику звонков только для автозаполнения с прямыми номерами телефонов.

Если вам нужна отдельная маршрутия для вызывающих телефонных вызовов в нечасовом времени, укажите часы работы для каждого дня. Выберите **Добавить новое время,** чтобы указать несколько наборов часов для заданного дня, например, чтобы указать перерыв на обед.

После того как вы указали часы работы, выберите параметры маршрутизов параметров для неавтных часов. Доступны те же параметры, что и для маршрутизов в часы работы, которые вы указали выше.

Когда **все будет готово,** выберите Далее.

## <a name="call-flows-during-holidays"></a>Потоки вызовов во время праздников

![Снимок экрана: параметры поздравительных и праздничных приветствий](media/auto-attendant-holiday-greeting.png)

Ваш автозаводщик может использовать поток зовов для каждого настроенного вами [праздника.](set-up-holidays-in-teams.md) Вы можете добавить до 20 запланированных праздников для каждого автосекретаря.

1. На странице параметров звонка праздников выберите **Добавить**.

2. Введите название этого параметра праздников.

3. В **dropdown Holiday** (Праздники) выберите праздник, который вы хотите использовать.

4. Выберите нужный тип приветствия.

    ![Снимок экрана: параметры действий по праздничным звонкам](media/auto-attendant-holiday-actions.png)

5. Выберите, хотите ли вы **отключить или** **перенаправить** звонок.

6. Если вы выбрали перенаправление, выберите пункт назначения маршрутиации для звонка.

7. Нажмите **Сохранить**.

![Снимок экрана: параметры праздников со списком праздников](media/auto-attendant-holiday-call-settings.png)

Повторите эти процедуры для каждого дополнительного праздника.

Когда вы добавите все праздники, выберите **Далее**.

## <a name="dial-scope"></a>Область набора номера

![Снимок экрана: область набора номера, включаемая и исключаемая](media/auto-attendant-dial-scope.png)

Область *набора определяет,* какие пользователи доступны в каталоге, если вызываемая звонок использует по имени или по телефону по расширению. По умолчанию **все пользователи в** Сети включают всех пользователей в организации, которые являются пользователями Online или которые находятся локально с помощью Skype для бизнеса Server.

Вы можете включить или исключить определенных  пользователей,  выбрав настраиваемую группу пользователей в группе Включить или Исключить и выбрав одну или несколько групп Microsoft 365, списков рассылки или групп безопасности.  Например, может потребоваться исключить руководителей организации из каталога набора номера. (Если пользователь есть в обоих списках, он будет исключен из каталога.)

> [!NOTE]
> Для того чтобы имя нового пользователя было указано в каталоге, может потребоваться до 36 часов.

После настройки области набора номера выберите **Далее.**

## <a name="resource-accounts"></a>Учетные записи ресурсов

У всех автозаводников должна быть связанная учетная запись ресурса.  Автослужам первого уровня потребуется по крайней мере одна учетная запись ресурса с связанным номером службы. При желании вы можете назначить автослужбу несколько учетных записей ресурсов с отдельным номером службы.

![Снимок экрана: панель добавления учетных записей для учетных записей ресурсов](media/auto-attendant-add-resource-account.png)

Чтобы добавить учетную запись ресурса, выберите **Добавить** учетную запись и найщите учетную запись, которую вы хотите добавить. Выберите **Добавить**, а затем **добавить**.

![Снимок экрана: список учетных записей ресурсов с учетной записью ресурса с номером назначенной услуги](media/auto-attendant-resource-account-assigned.png)

Завершив добавление учетных записей службы, выберите **Отправить,** чтобы завершить настройку автозаполнеющего.

## <a name="external-phone-number-transfers---technical-details"></a>Передача внешних номеров телефонов — технические сведения

Чтобы разрешить [автоответам](plan-auto-attendant-call-queue.md#prerequisites) переводить звонки за границу, обратитесь к условиям.  В дополнение:

- Для учетной записи ресурса с лицензией на план звонков [номер](calling-plans-for-office-365.md)внешнего переносного телефона должен быть введен в формате E.164 (+[код страны][код города][номер телефона]).

- Для учетной записи ресурса с лицензией телефонная система и политикой прямой маршрутистики в Интернете формат номера внешнего переносимого телефона зависит от параметров [SBC.](direct-routing-connect-the-sbc.md)

Отображаемая исходящие телефонные номера определяется следующим образом:

  - Для номеров планов звонков отображается исходный номер телефона звоня.
  - Для номеров прямой маршрутификации число, отправленное, основано на параметре P-Изучаемого удостоверения (PAI) в SBC следующим образом:
    - Если за установлено отключение, отображается исходный номер телефона вызываемого звоня. Этот параметр рекомендуется использовать по умолчанию.
    - Если для этого включено, отображается номер телефона учетной записи ресурса.

Для Skype для бизнеса гибридной среды для переадстройки вызова автоотехнщика в ПСОП создайте нового локального пользователя с номером для переадстройки вызовов через ПСОП. Пользователь должен быть включен для Корпоративная голосовая связь и иметь назначенную голосовую политику. Дополнительные см. в теме Перенастройка [автоотвода для перенастройки звонка на ПСОП.](/SkypeForBusiness/plan/exchange-unified-messaging-online-migration-support#auto-attendant-call-transfer-to-pstn)
