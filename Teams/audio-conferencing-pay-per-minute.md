# <a name="audio-conferencing-pay-per-minute"></a>Поминутная оплата аудиоконференций

В дополнение к Стандартная лицензия пользователя аудиоконференции Лицензирование и многопользовательская организаций можно также предложение минуту для включения возможности конференц-связи аудио для своей организации.
  

Можно найти ли вашей [страны или региона поддерживается](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) для аудиоконференции.

  
> [!NOTE]
> Так как это предложение доступен только для клиентов Лицензирование и многопользовательская, необходимо будет работать с представителем вашей учетной записи службы для получения лицензий. 
  
## <a name="need-some-details"></a>Требуются некоторые сведения?

В то время как предложение лицензии пользователя аудиоконференции включает использования-связь и некоторые подключение по телефонной линии, аудиоконференции оплаты в минуту **позволяет оплачиваться все использования конференц-связи звук на за минуту без ежемесячно оплаты для отдельных пользователей Лицензия**. С этого все-связь и исходящие вызовы будут оплачиваться на основе в минуту, включая:
  
- Входящие вызовы на номер телефона номеров аудиоконференции вашей организации.
    
- Входящие вызовы бесплатных номеров аудиоконференции вашей организации.
    
- Исходящие вызовы, помещенные на внешние телефонные номера из в рамках собрания из вашей организации.
    
> [!NOTE]
> Можно найти нормы-связь и подключение по телефонной линии, связанный со эти типы звонков, описанные в разделе **увидеть курсы, для которых нужно позвонить раздел** в [Аудиоконференции](https://products.office.com/microsoft-teams/online-meeting-solutions#Rates).
  
Минуту требуется вашей организации для [Коммуникаций кредитов](what-are-communications-credits.md) лицензию, назначенную для каждого пользователя в порядке для аудиоконференции для работы с включенными. Для получения дополнительных сведений в статье [Set up кредитов коммуникаций для вашей организации](set-up-communications-credits-for-your-organization.md) и/или [группами Майкрософт дополнительный компонент лицензирования](teams-add-on-licensing/microsoft-teams-add-on-licensing.md).
  
Чтобы включить аудиоконференции оплаты в минуту для пользователей в вашей организации, видеть [Попробуйте или Покупка Audio конференция в Office 365](try-or-purchase-audio-conferencing-in-office-365-for-teams.md).

## <a name="why-is-it-best-for-you"></a>Почему это лучший для вас?

- Оплата за минуту будут оплачиваться только на основе минуту для каждого вызова входящих или исходящих сделанным каждого участника во время запланированного собрания (ставки различаются для счета или бесплатных звонка и конечной) вместо использования Скайп для бизнеса или группами приложения в мобильных устройств или Мобильное устройство.

- Возможность управлять затраты, поскольку администраторы могут управлять для ограничения типов телефонные контакты (международные и Россия), которые можно выполнить из собрания инициатора. Просмотреть [исходящих вызовов ограничения политики для аудиоконференции и вызовы ТСОП пользователя](https://docs.microsoft.com/skypeforbusiness/audio-conferencing-in-office-365/outbound-calling-restriction-policies)

- В случае наличия лицензий на аудиоконференции с поминутной оплатой Вам необязательно отдельно для каждого пользователя специально назначать лицензии на кредиты на связь, чтобы использовать функцию аудиоконференций (однако, может потребоваться назначение лицензий на другие сервисы).

- Элемент управления и монитор оплаты за minute расходы с использованием коммуникаций кредитов.

- Большая гибкость при ценообразования для клиентов, которым не требуется всех пользователей на основе подписки. 

- Включение аудиоконференции оплаты в минуту вместе с месячная подписка E5 или автономного аудиоконференции подписки, обе службы будут работать так же, как. Изменения не повлияет на операции аудиоконференции или кредитов коммуникаций.
