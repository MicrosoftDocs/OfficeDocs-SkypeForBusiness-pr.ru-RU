# <a name="audio-conferencing-pay-per-minute"></a>Поминутная оплата аудиоконференций

Помимо стандартной лицензии на аудиоконференцию для каждого пользователя организации с корпоративной лицензией и лицензированием могут также выбрать предложение с поминутной оплатой, чтобы включить функции аудиоконференций для своих организаций.
  

Вы можете узнать, [](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) поддерживается ли аудиоконференция в вашей стране или регионе.

  
> [!NOTE]
> Поскольку это предложение доступно только клиентам с корпоративной лицензией и лицензированием, для получения этих лицензий вам потребуется поработать со своим представителем по работе с клиентами. 
  
## <a name="need-some-details"></a>Нужны некоторые сведения?

В то время как лицензия на аудиоконференцию для отдельных пользователей включает в себя использование телефонного дозвона и некоторые функции исходят из телефонного набора, аудиоконференция с поминутной оплатой позволяет оплачивать все данные об использовании аудиоконференций на поминутной основе без оплаты ежемесячной лицензии для отдельных **пользователей.** При этом все звонки с телефонным вызовом будут взиматься с каждой минуты, включая следующие:
  
- Входящие звонки на платные номера аудиоконференций в вашей организации.
    
- Входящие звонки на бесплатные номера аудиоконференций в вашей организации.
    
- Исходящие звонки на внешние телефонные номера из собрания вашей организации.
    
> [!NOTE]
> Чтобы узнать, какие тарифы звонков связаны с этими типами  звонков, просмотрите раздел "Тарифы для звонков в аудиоконференцию" в разделе "Тарифы для [звонков".](https://products.office.com/microsoft-teams/online-meeting-solutions#Rates)
  
  
Для оплаты поминутной оплаты организации необходимо настроить кредиты на связь. [](what-are-communications-credits.md) Если вам нужны дополнительные сведения, см. сведения о том, как настроить кредиты на связь для вашей организации и лицензирование надстройки [Microsoft Teams.](https://docs.microsoft.com/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing) [](set-up-communications-credits-for-your-organization.md)

  
Чтобы включить аудиоконференцию с поминутной оплатой для пользователей в организации, см. видео "Попробуйте или приобретите аудиоконференцию в [Microsoft Office 365 или Office 365".](try-or-purchase-audio-conferencing-in-office-365-for-teams.md)

## <a name="why-is-it-best-for-you"></a>Почему оно лучше для вас?

- Плата за минуту будет взиматься поминутно только за каждый входящий или исходящие звонки, которые каждый участник должен сделать во время запланированного собрания (тарифы зависят от платных или бесплатных звонков и по назначениям), а не использовать приложение Skype для бизнеса или Teams на мобильном устройстве или компьютере.

- Возможность управления затратами, так как администраторы могут контролировать, какие виды еконферентов (международные и внутренние) можно использовать на собраниях организатора. См. [политики ограничений исходящие звонков для аудиоконференций и звонков по ННР пользователей](./outbound-calling-restriction-policies.md)

- В случае наличия лицензий на аудиоконференции с поминутной оплатой Вам необязательно отдельно для каждого пользователя специально назначать лицензии на кредиты на связь, чтобы использовать функцию аудиоконференций (однако, может потребоваться назначение лицензий на другие сервисы).

- Контролируйте и отслеживайте оплату поминутной оплаты с помощью кредитов на связь.

- Более гибкое ценами для клиентов, которым не требуются все пользователи на основе подписки. 

- В течение этого действия обе службы будут работать одинаково с ежемесячной подпиской на план E5 или подпиской на автономные аудиоконференцию с поминутной оплатой аудиоконференций. Изменения не влияют на работу аудиоконференции или кредитов на связь.
