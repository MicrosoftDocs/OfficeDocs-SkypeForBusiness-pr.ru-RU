<a name="manage-the-office-365-g1-trial-for-us-government"></a>Управление пробной версией Office 365 G1 для государственных учреждений США 
==============================

С марта 2020 г. доступна 6-месячная бесплатная пробная версия Office 365 G1, включающая Microsoft Teams. Корпорация Майкрософт предоставляет государственным учреждениям США эту специальную лицензию на пробную версию G1 в связи с возросшиеми потребностями обеспечении работы сотрудников из дома в связи со вспышкой эпидемии COVID-19 (коронавируса).

Чтобы узнать, какие возможности доступны в рамках лицензии G1, ознакомьтесь с разделом [Планы Microsoft 365 для государственных учреждений](https://www.microsoft.com/microsoft-365/government/compare-office-365-government-plans) и [Возможности Microsoft Teams, доступные в облаке GCC](plan-for-government-gcc.md).

Не пропустите наши рекомендации по  [поддержке удаленной работы в Teams](support-remote-work-with-teams.md).

## <a name="how-to-get-an-g1-trial-license"></a>Как получить лицензию на пробную версию G1

Свяжитесь с представителем Майкрософт по работе с клиентами. Он предоставит вам промокод, с помощью которого можно зарегистрироваться для получения пробной версии G1.

## <a name="how-to-sign-up-for-the-g1-trial"></a>Как зарегистрироваться для получения пробной версии G1

Следуйте инструкциям в сообщении электронной почты, которое отправит вам представитель по работе с клиентами.

## <a name="whos-eligible"></a>Кто имеет право на использование?

Это предложение предназначено для клиентов из числа государственных учреждений США, работающих в облаке GCC под управлением представителей Майкрософт по работе с клиентами. Им могут воспользоваться организации, которые не активировали другие пробные версии Office 365 G1 в прошлом.

После применения промокода пробной версии G1 вы не сможете использовать его повторно, а также не сможете активировать другой код пробной версии G1.

## <a name="who-isnt-eligible"></a>Кто не имеет права на использование?

  - Это предложение недоступно для коммерческих клиентов, клиентов GCC High, DoD и образовательных учреждений.

  - Коммерческим клиентам следует использовать пробную версию Office 365 E1.

  - Клиентам из числа образовательных учреждений следует использовать бесплатную лицензию Office 365 A1.

## <a name="manage-the-g1-trial"></a>Управление пробной версией G1

После активации пробной версии Office 365 G1 включите лицензию для всех пользователей, которым она требуется. Сведения о том, как это сделать, см. в статье  [Управление доступом пользователей к Teams](user-access.md).

После включения пробной версии G1 для пользователей, которым она требуется, вы сможете управлять ими так же, как управляете пользователями с платной лицензией. Дополнительные сведения см. в статье  [Управление параметрами Teams в организации](enable-features-office-365.md).

### <a name="upgrade-users-from-the-office-365-g1-trial-license"></a>Переход пользователей с пробной лицензии Office 365 G1

Чтобы перевести пользователей пробной версии G1 на платную подписку:

1.  Приобретите подписку, в которую входит Teams.

2.  Удалите пробную подписку пользователя на Office 365 G1.

3.  Назначьте приобретенную лицензию.

Дополнительные сведения см. в статье [Teams для государственных учреждений](expand-teams-across-your-org/teams-for-government-landing-page.md).

> [!NOTE]
> Если срок лицензии на пробную версию G1 истек, а пользователь сразу не обновился до подписки, включающей Teams, пользовательские данные не удаляются. Пользователь по-прежнему существует в Azure Active Directory, и все данные в Teams сохраняются. После назначения новой лицензии пользователю, чтобы повторно включить функции Teams, содержимое продолжит существовать.
> 
### <a name="remove-an-office-365-g1-trial-license"></a>Удаление лицензии на пробную версию Office 365 G1

  - Если вы хотите удалить эту лицензию с помощью PowerShell, см. статью  [Использование PowerShell в Office 365 для удаления лицензий из учетных записей пользователей](https://docs.microsoft.com/office365/enterprise/powershell/remove-licenses-from-user-accounts-with-office-365-powershell)

  - Если вы хотите удалить эту лицензию на портале администрирования, см. статью  [Удаление лицензий пользователей в Office 365](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/remove-licenses-from-users?view=o365-worldwide)
