# <a name="deploy-edge-server-in-skype-for-business-server"></a>Развертывание edge Server в Skype для бизнеса Server
 
**Сводка:** Узнайте, как развернуть edge Server или edge pool в Skype для бизнеса Server среде.
  
Зачем развертывать edge Server или edge pool в Skype для бизнеса Server среде? Это необходимо, если вам нужны внешние пользователи, которые не вошли во внутреннюю сеть организации, чтобы иметь возможность взаимодействовать с внутренними пользователями. Эти внешние пользователи могут быть аутентификацией и анонимными удаленными пользователями, федерадными партнерами или другими мобильными клиентами.
