# <a name="integrate-skype-for-business-server-with-exchange-server"></a>Интеграция Skype для бизнеса Server с Exchange Server

**Сводка:** Просмотрите этапы интеграции Exchange Server 2013 или более поздней и Skype для бизнеса Server.

Exchange Server 2013 или Skype для бизнеса Server совместимы и хорошо интегрируются. Например, Skype для бизнеса сведения о присутствии пользователей можно отчитаться в Microsoft Outlook; Кроме того, Skype для бизнеса можно получить доступ к календарю Outlook пользователя, заметить, что у пользователя запланирована встреча, и показать его присутствие в качестве занятого во время собрания. Хотя вам не нужно запускать Exchange Server для Skype для бизнеса Server (или наоборот), эти два продукта вместе улучшают пользовательский интерфейс друг друга.

Эта документация содержит сведения об интеграции Skype для бизнеса Server и Exchange Server 2016 или Exchange Server 2013 г., но предполагает, что первоначальная настройка и конфигурация этих двух продуктов уже произошли. Дополнительные сведения о развертывании Skype для бизнеса Server см. в [Skype для бизнеса Server Tech Center.](../../../Hub/index.yml) Дополнительные сведения о развертывании Exchange Server см. в документации по развертыванию для вашей версии Exchange.

Если вы интегрировали локальное установку Skype для бизнеса Server с Microsoft Exchange Online, см. в рублях Настройка интеграции между локальной Skype для бизнеса Server [и Outlook Web App](outlook-web-app.md).

Если вы интегрировали Skype для бизнеса Online с Exchange Server на локальной основе, см. в рубке [Настройка OAuth](oauth-with-online-and-on-premises.md)между Skype для бизнеса Online и Exchange в помещениях .
