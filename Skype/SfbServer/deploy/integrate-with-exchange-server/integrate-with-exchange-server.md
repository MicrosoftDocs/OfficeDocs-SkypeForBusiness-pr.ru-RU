# <a name="integrate-skype-for-business-server-with-exchange-server"></a>Интеграция Skype для бизнеса Server с Exchange Server

**Сводка:** Просмотрите этапы интеграции Exchange Server 2013 или более позднего года и Skype для бизнеса Server.

Exchange Server 2013 или более поздней и Skype для бизнеса Server совместимы и хорошо интегрируются. Например, сведения о присутствии пользователей Skype для бизнеса можно отчитаться в Microsoft Outlook; Кроме того, Skype для бизнеса может получить доступ к календарю Outlook пользователя, заметить, что у пользователя запланирована встреча, и показать его присутствие как Занято во время собрания. Хотя вам не нужно запускать Exchange Server для запуска Skype для бизнес-сервера (или наоборот), эти два продукта вместе улучшают пользовательский интерфейс друг друга.

В этой документации содержится информация об интеграции Skype для бизнеса Server и Exchange Server 2016 или Exchange Server 2013 г., но предполагается, что первоначальная настройка и конфигурация этих двух продуктов уже произошли. Дополнительные сведения о развертывании Skype для бизнеса Server см. в техническом центре [Skype для бизнеса Server.](../../../Hub/index.yml) Дополнительные сведения о развертывании Exchange Server см. в документации по развертыванию для вашей версии Exchange.

Если вы интегрировали локальное устройство Skype для бизнеса Server с Microsoft Exchange Online, см. в примере Настройка интеграции между локальной установкой [Skype для](outlook-web-app.md)бизнеса Server и Outlook Web App .

Если вы интегрировали Skype для бизнеса Online с Exchange Server на месте, см. в рублях Настройка OAuth между Skype для бизнеса Online и [Exchange в помещениях.](oauth-with-online-and-on-premises.md)
