# <a name="create-a-persistent-chat-administrator-in-skype-for-business-server-2015"></a>Создание администратора сохраняемой беседы в Skype для бизнеса Server 2015 г.
 
**Сводка:** Ознакомьтесь с этой темой, чтобы узнать, как создать роль администратора сохраняемого сервера чата для обеспечения начальной конфигурации и управления службами сохраняемого чата в Skype для бизнеса Server 2015 г.
  
В Skype для бизнеса Server пользователи, которые выполняют определенные задачи, должны быть назначены членами одной или более определенных групп. Role-Based управления доступом (RBAC) используется для предоставления привилегий, назначая пользователям заданные Skype для бизнеса Server административные роли. Эти роли соответствуют универсальным группам безопасности в службах домена Active Directory. Членам группы безопасности администратора сохраняемого чата CsPersistentChatAdministrator предоставляется доступ к командым Persistent Chat Server, которые можно выполнять с помощью панели управления Skype для бизнеса Server или панели управления Skype для бизнеса Server.
  
Перед настройкой и администрированием Persistent Chat Server убедитесь в том, что соответствующие права и разрешения пользователей имеются и что все пользователи, которые будут выступать в качестве администраторов сохраняемых чатов, будут добавлены в группу безопасности администратора сохраняемой системы чата.
  
> [!NOTE] 
> Постоянный чат доступен в Skype для бизнеса Server 2015 г., но больше не поддерживается Skype для бизнеса Server 2019 г. Такая же функциональность доступна в Teams. Дополнительные сведения см. в [ссылке Начало работы с Microsoft Teams обновления](/microsoftteams/upgrade-start-here). Если вам нужно использовать постоянный чат, вы можете либо перенести пользователей, требующих Teams, либо продолжить использование Skype для бизнеса Server 2015 г.

## <a name="create-a-persistent-chat-administrator"></a>Создание администратора сохраняемого чата

Чтобы добавить пользователя в группу безопасности сохраняемой администратора чата, CsPersistentChatAdministrator выполните следующие действия:
  
1. Используя учетную запись, которая имеет разрешение на изменение членства в группе Active Directory, войдите на компьютер, на котором установлены пользователи и компьютеры Active Directory.
    
2. Нажмите кнопку Пуск, выберите Все программы, Администрирование и затем Пользователи и компьютеры Active Directory.
    
3. В окне «Пользователи и компьютеры Active Directory» разверните имя своего домена и щелкните контейнер Users.
    
4. Щелкните правой кнопкой мыши группу безопасности CsPersistentChatAdministrator и нажмите кнопку Свойства.
    
5. В диалоговом окне Свойства на вкладке Члены нажмите кнопку Добавить.
    
6. В диалоговом окне Выбор пользователей, компьютеров, контактов или групп введите имя пользователя или отображайте имя пользователя, которое будет добавлено в группу в введите имена объектов, чтобы выбрать поле, а затем нажмите кнопку ОК.
    
7. В диалоговом окне Свойства нажмите кнопку ОК.
