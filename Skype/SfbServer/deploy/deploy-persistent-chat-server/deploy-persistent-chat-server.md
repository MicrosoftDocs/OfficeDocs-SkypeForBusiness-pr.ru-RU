# <a name="deploy-persistent-chat-server-in-skype-for-business-server-2015"></a>Развертывание постоянного сервера чата в Skype для бизнеса Server 2015 г.
 
**Сводка:** Ознакомьтесь с этой темой, чтобы узнать, как развернуть Skype для бизнеса Server 2015 persistent Chat Server.
  
Чтобы развернуть стойкий сервер чата, необходимо: 
  
- Используйте Topology Builder для определения или импорта и последующей публикации топологии и компонентов, которые необходимо развернуть
    
- Подготовка среды для развертывания компонентов сохраняемой системы чат-сервера
    
- Установка и настройка компонентов сохраняемой системы чат-сервера для развертывания
    
Перед развертыванием сервера сохраняемой чат-сервера необходимо прочитать план для постоянного сервера чата в [Skype для бизнеса Server 2015 г](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md). В темах планирования описываются требования к оборудованию и программному обеспечению, возможные топологии и сценарии коллокации. 
  
> [!NOTE] 
> Постоянный чат доступен в Skype для бизнеса Server 2015 г., но больше не поддерживается Skype для бизнеса Server 2019 г. Такая же функциональность доступна в Teams. Дополнительные сведения см. в [ссылке Начало работы с Microsoft Teams обновления](/microsoftteams/upgrade-start-here). Если вам нужно использовать постоянный чат, вы можете либо перенести пользователей, требующих Teams, либо продолжить использование Skype для бизнеса Server 2015 г. 

## <a name="deployment-checklist"></a>Контрольный список развертывания
