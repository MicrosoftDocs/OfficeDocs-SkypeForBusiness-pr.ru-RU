# <a name="deploy-persistent-chat-server-in-skype-for-business-server-2015"></a>Развертывание сервера сохраняемой беседы в Skype для бизнеса Server 2015
 
**Сводка:** В этом разделе вы узнаете, как развернуть сервер сохраняемой беседы Skype для бизнеса Server 2015.
  
Для развертывания сервера сохраняемой беседы необходимо: 
  
- Использование построитель топологий для определения или импорта и последующей публикации топологии и компонентов, которые нужно развернуть
    
- Подготовка среды для развертывания компонентов сервера сохраняемой беседы
    
- Установка и настройка компонентов сервера сохраняемой беседы для развертывания
    
Перед развертыванием сервера сохраняемой беседы ознакомьтесь с документом [Plan for Persistent Chat Server in Skype for Business Server 2015.](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md) В темах планирования описываются требования к оборудованию и программному обеспечению, возможные topologies и сценарии выкладывания. 
  
> [!NOTE] 
> Сохраняемая беседа доступна в Skype для бизнеса Server 2015, но больше не поддерживается в Skype для бизнеса Server 2019. Такие же функции доступны в Teams. Дополнительные сведения [см. в сведениях о том,](/microsoftteams/upgrade-start-here)как начать обновление Microsoft Teams. Если необходимо использовать сохраняемого чата, вы можете либо перенести пользователей, которым требуются эти функции, в Teams, либо продолжить использование Skype для бизнеса Server 2015. 

## <a name="deployment-checklist"></a>Контрольный список развертывания
