# <a name="deploy-call-management-features-in-skype-for-business"></a>Развертывание функций управления звонками в Skype для бизнеса
 
Развертывание функций управления звонками в корпоративной голосовой связи Skype для бизнеса Server.
  
Возможности управления голосом на корпоративном звонке определяют маршрутизацию и ответ на входящие звонки. Skype для бизнеса Server предоставляет следующие возможности управления звонками: 
  
- **Припаркованный Звонок**: позволяет пользователям голосовой связи временно приостановить звонок, а затем выбрать его на другом телефоне.
    
- **Отправка группового звонка**: позволяет пользователям отвечать на звонки, выполненные другим пользователям, которые назначены в группу раскладки с помощью набора номера группы для отправки звонков.
    
- **Группа ответа**: Маршрутизация входящих звонков на группы агентов с помощью групп слежения и ответов на вопросы и ответы по интерактивный автоответчик.
    
- **Информационное сообщение**: воспроизводит сообщение для вызовов, выполненных на не назначенный номер, перенаправляет вызов куда-нибудь еще или выполняет оба действия.
    
В этом разделе рассказывается о том, как настроить эти функции управления звонками при развертывании корпоративной голосовой связи.
