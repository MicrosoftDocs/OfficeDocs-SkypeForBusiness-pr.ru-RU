# <a name="deploy-call-management-features-in-skype-for-business"></a>Развертывание функций управления звонками в Skype для бизнеса
 
Развертывание функций управления звонками в Skype для бизнеса Server Корпоративная голосовая связь.
  
Функции управления вызовами в корпоративной голосовой связи управляют тем, каким образом происходит маршрутизация входящих вызовов и ответ на них. Skype для бизнеса Server предоставляет следующие функции управления звонками: 
  
- **Парковка вызовов**: позволяет пользователям голосовой связи временно приостанавлить вызов, а затем получить его с того же телефона или другого телефона.
    
- **Групповой ответ** на звонки: позволяет пользователям отвечать на вызовы, сделанные другому пользователю, который назначен группе ответа, набирая номер группы ответа на звонки.
    
- **Группа ответа:** маршруты входящих звонков группам агентов с помощью сервисных групп или вопросов и ответов интерактивного голосового ответа (IVR).
    
- **Оповещение** — проигрывает сообщение о вызовах на неназначенный номер, выполняет маршрутизацию вызова в любое другое расположение или выполняет и то, и другое.
    
В этом разделе описывается настройка этих функций управления вызовами во время Корпоративная голосовая связь развертывания.
