# <a name="deploy-call-management-features-in-skype-for-business"></a>Развертывание функций управления вызовами в Skype для бизнеса
 
Развертывание функций управления вызовами в Skype для бизнеса Server Корпоративная голосовая связь.
  
Функции управления вызовами в корпоративной голосовой связи управляют тем, каким образом происходит маршрутизация входящих вызовов и ответ на них. Skype для бизнеса Server предоставляет следующие функции управления вызовами: 
  
- **Call Park**: позволяет пользователям голосовой связи временно припарковать вызов, а затем забрать его с того же телефона или другого телефона.
    
- **Групповой вызов**: позволяет пользователям отвечать на вызовы, сделанные другому пользователю, назначенного группе пикапа, набрав номер группы вызова.
    
- **Группа ответа**. Маршруты входящих звонков группам агентов с помощью групп охоты или интерактивных голосовых ответов (IVR) вопросов и ответов.
    
- **Оповещение** — проигрывает сообщение о вызовах на неназначенный номер, выполняет маршрутизацию вызова в любое другое расположение или выполняет и то, и другое.
    
В этом разделе описывается настройка этих функций управления вызовами во время Корпоративная голосовая связь развертывания.
