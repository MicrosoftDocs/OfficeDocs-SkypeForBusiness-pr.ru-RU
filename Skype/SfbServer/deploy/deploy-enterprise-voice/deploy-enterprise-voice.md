**Сводка:** Узнайте, как развернуть Корпоративная голосовая связь Skype для бизнеса Server на центральном сайте.

Вы можете пропустить следующий контент, если вы использовали Topology Builder для определения и публикации топологии, которая коллокирует сервер-посредник на каждом сервере переднего или стандартного выпуска, так как мастер развертывания уже автоматически установил файлы для сервера-посредника при установке файлов для пула переднего конечного сервера или сервера Standard Edition:
