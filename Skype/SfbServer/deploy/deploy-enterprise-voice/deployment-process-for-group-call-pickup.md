# <a name="deployment-process-for-group-call-pickup-in-skype-for-business-2015"></a>Deployment process for Group Call Pickup in Skype for Business 2015
 
Процесс развертывания и действия для отправки вызовов группы в Скайп Business Server корпоративной голосовой связи.
  
Раскладки вызывать группу Пользователи могут отвечать на входящие звонки для своих коллег из собственные телефоны. 
  
 Компоненты, используемые группы вызова раскладки автоматически установлены и включены на сервере переднего плана или сервере Standard Edition, при развертывании корпоративной голосовой связи. Тем не менее необходимо использовать следующие действия для настройки группы вызова раскладки перед его доступным для пользователей.
  
**Процесс раскладки развертывания групповой звонок**

|:-----|:-----|:-----|:-----|

Убедитесь, что SEFAUtil выполняется правильно, воспользовавшись им для отображения параметров переадресации звонков какого-либо пользователя в развертывании. |RTCUniversalServerAdmins  <br/> |[Развертывание средства SEFAUtil в Скайп для бизнеса 2015](deploy-the-sefautil-tool.md) <br/> [Новый CsTrustedApplicationPool](https://docs.microsoft.com/powershell/module/skype/new-cstrustedapplicationpool?view=skype-ps) [Новый CsTrustedApplication](https://docs.microsoft.com/powershell/module/skype/new-cstrustedapplication?view=skype-ps) [Командлет Enable-CsTopology](https://docs.microsoft.com/powershell/module/skype/enable-cstopology?view=skype-ps) <br/> [Документация по средства набора ресурсов Lync Server 2013](https://technet.microsoft.com/en-us/library/jj945604%28v=ocs.15%29.aspx).
