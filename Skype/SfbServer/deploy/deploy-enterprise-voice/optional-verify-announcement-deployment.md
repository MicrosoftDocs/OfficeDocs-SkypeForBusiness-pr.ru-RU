# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>(Необязательный) Проверка развертывания объявления в Skype для бизнеса
 
Проверка развертывания приложения Объявления в Skype для бизнеса Server Корпоративная голосовая связь. Это влияет на обработку вызовов на ненанаписаные номера.
  
После установки и настройки приложения "Оповещение" необходимо проверить конфигурацию, чтобы убедиться, что вызовы на неназначенные номера работают соответствующим образом. Как минимум, необходимо проверить следующее:
  
- Вызов номера, допустимого для организации, но не назначенного номера.
    
- вызвать неназначенный номер и проверить, воспроизводится ли правильное оповещение.
