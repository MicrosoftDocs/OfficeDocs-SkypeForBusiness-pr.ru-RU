# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>(Необязательно) Проверка развертывания объявлений в Skype для бизнеса
 
Проверка развертывания приложения объявлений в Skype для бизнеса Server Корпоративная голосовая связь. Это влияет на то, как обрабатываются вызовы на ненаписаные номера.
  
После установки и настройки приложения "Оповещение" необходимо проверить конфигурацию, чтобы убедиться, что вызовы на неназначенные номера работают соответствующим образом. Как минимум, необходимо проверить следующее:
  
- Позвоните по номеру, действительному для вашей организации, но не назначенном номеру.
    
- вызвать неназначенный номер и проверить, воспроизводится ли правильное оповещение.
