# <a name="export-or-import-a-voice-route-configuration-file-in-skype-for-business"></a>Экспорт или импорт файла конфигурации голосового маршрута в Skype для бизнеса
 
**Сводка:** Узнайте, как экспортировать или импортировать файл конфигурации голосовой маршрутики в Skype для бизнеса Server с помощью панели управления Skype для бизнеса Server.
  
Если вы хотите сохранить конфигурацию маршрутиации голосовой маршрутики без ее публикации, выполните следующие действия, чтобы сохранить и получить снимок конфигурации маршрутиации голосовой маршрутики. 
  
При импорте файла конфигурации маршрутификации голосовой маршрутики (.vcfg), но изменения были внесены в конфигурацию маршрутов голосовой маршрутики на сервере в это  время, страницы в группе голосового маршрутинга в панели управления Skype для бизнеса Server будут указывать на то, что в маршрутику голосовой маршрутки не были внесены какие-либо изменения. Эти несохраненные изменения представляют собой разницу между двумя конфигурациями, которые нуждаются в сверке.
  
Если вы сделали какие-либо несогласованые изменения параметров на любой странице в группе, изменения сохраняются в экспортируемом файле конфигурации голосовой связи (.vcfg). Это позволяет вносить изменения в конфигурацию маршрутивки голосовой связи во время нескольких сеансов перед публикацией изменений. 
  
### <a name="to-export-a-voice-routing-configuration"></a>Экспорт конфигурации маршрутизации голосовых данных

1. Войдите на компьютер в качестве члена группы RTCUniversalServerAdmins или роли администратора **CsVoiceAdministrator**, **CsServerAdministrator** или **CsAdministrator**.
    
2. Откройте панель Skype для бизнеса Server управления.
    
3. В левой панели навигации щелкните элемент **Маршрутизация голосовых данных**.
    
4. В меню **Действия** щелкните пункт **Экспорт конфигурации**.
    
5. Укажите расположение и имя файла, а затем нажмите кнопку **Сохранить**.
    
### <a name="to-import-a-voice-routing-configuration"></a>Импорт конфигурации маршрутов голосовой маршрутики

1. Войдите на компьютер в качестве члена группы RTCUniversalServerAdmins или роли администратора **CsVoiceAdministrator**, **CsServerAdministrator** или **CsAdministrator**.
    
2. Откройте панель Skype для бизнеса Server управления.
    
3. В левой области навигации щелкните элемент **Маршрутизация голосовых вызовов**.
    
4. В меню **Действия** щелкните **Конфигурация импорта**.
    
5. Найдите файл конфигурации, который необходимо импортировать, а затем нажмите кнопку **Открыть**.
    
6. Нажмите кнопку **Зафиксировать**, а затем нажмите кнопку **Зафиксировать все**.
    
    > [!NOTE]
    > Всякий раз, когда вы импортируете файл конфигурации голосовых данных, для публикации изменения конфигурации необходимо выполнить команду **Commit** all. Подробные сведения см. в [публикации "Публикация в](voice-route-config-changes.md) ожидании изменений конфигурации маршрутирования голосовой почты в Skype для бизнеса документации по операциям".
