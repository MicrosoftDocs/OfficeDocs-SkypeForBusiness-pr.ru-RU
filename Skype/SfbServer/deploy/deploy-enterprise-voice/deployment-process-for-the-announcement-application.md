# <a name="deployment-process-for-the-announcement-application-in-skype-for-business-server"></a>Процесс развертывания приложения объявлений в Skype для бизнеса Server
 
Процесс развертывания приложения объявлений в Skype для бизнеса Server Корпоративная голосовая связь.
  
Приложение "Объявление" — это функция Корпоративная голосовая связь, которая позволяет настроить, что происходит с вызовами ненаписаных расширений (расширения, допустимые для вашей организации, но не назначены человеку или телефону). Например, можно настроить эту функцию таким образом, чтобы при выполнении вызовов на неприсвоенные номера воспроизводилось сообщение и/или эти вызовы передавались на другое назначение.
  
Приложение "Объявление" устанавливается как функция приложения группы ответа на сервере переднего Корпоративная голосовая связь. Для настройки оповещений следует загрузить аудиофайлы или настроить преобразование текста в речь и сконфигурировать таблицу неприсвоенных номеров.
  
В этом разделе представлен обзор действий, необходимых для развертывания приложения "Объявление". Необходимо развернуть Корпоративная голосовая связь перед настройкой объявлений. Компоненты, необходимые приложению "Объявление", устанавливаются и включаются при развертывании Корпоративная голосовая связь.
