# <a name="deployment-process-for-the-announcement-application-in-skype-for-business-server"></a>Процесс развертывания приложения Объявления в Skype для бизнеса Server
 
Процесс развертывания и действия для приложения Объявления в Skype для бизнеса Server Корпоративная голосовая связь.
  
Приложение Announcement — это Корпоративная голосовая связь, которая позволяет настраивать то, что происходит с вызовами на неназваные расширения (расширения, которые действительны для вашей организации, но не назначены человеку или телефону). Например, можно настроить эту функцию таким образом, чтобы при выполнении вызовов на неприсвоенные номера воспроизводилось сообщение и/или эти вызовы передавались на другое назначение.
  
Приложение Announcement устанавливается в качестве функции приложения Группы отклика на переднем сервере или выпуск Standard при развертывании Корпоративная голосовая связь. Для настройки оповещений следует загрузить аудиофайлы или настроить преобразование текста в речь и сконфигурировать таблицу неприсвоенных номеров.
  
В этом разделе представлен обзор действий, участвующих в развертывании приложения Объявления. Необходимо развернуть Корпоративная голосовая связь перед настройкой объявлений. Компоненты, необходимые приложению Announcement, устанавливаются и включены при развертывании Корпоративная голосовая связь.
