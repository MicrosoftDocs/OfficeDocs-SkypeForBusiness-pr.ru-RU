# <a name="deployment-process-for-call-park-in-skype-for-business-2015"></a>Процесс развертывания для парковки вызовов в Skype для бизнеса 2015
 
Процесс развертывания и действия для парковки вызовов в Скайп Business Server корпоративной голосовой связи.
  
Парковка вызовов пользователь корпоративной голосовой связи в удержание звонка с одного телефона и затем извлекать вызов более поздней версии, набрав внутренний номер (который называется орбиты парковки вызовов) с любого телефона.
  
Компоненты, используемые парковки вызовов автоматически установлены и включены на сервере переднего плана или сервере Standard Edition, при развертывании корпоративной голосовой связи. Тем не менее необходимо использовать следующие действия для настройки парковки вызовов, перед его доступным для пользователей. 
  
**Процесс развертывания парковки вызовов**

|**Этап**|**Действия**|**Необходимые группы и роли**|**Документация по развертыванию**|

|Настройка диапазонов орбит парковки вызовов в таблице орбит  <br/> |Используйте Скайп для панели управления Business Server или командлета **New-CSCallParkOrbit** для создания диапазонов орбит в таблице орбит парковки вызовов и свяжите их с помощью службы приложения, в котором размещается приложение парковки вызовов. <br/> **Примечание:** Для облегчения интеграции с существующей абонентских групп диапазонов орбит обычно настраиваются как блок виртуального расширения. Назначение номеров DID как номеров орбит в таблице орбит парковки вызовов не поддерживается. <br/> |RTCUniversalServerAdmins  <br/> CsVoiceAdministrator  <br/> CsServerAdministrator  <br/> CsAdministrator  <br/> |[Создание или изменение диапазона орбит парковки вызовов в Скайп для бизнеса 2015](create-or-modify-a-call-park-orbit-range.md) <br/> |
