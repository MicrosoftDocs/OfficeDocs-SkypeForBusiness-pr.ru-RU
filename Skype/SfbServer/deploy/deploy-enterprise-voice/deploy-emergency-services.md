# <a name="deploy-emergency-services-in-skype-for-business-server"></a>Развертывание экстренных служб в Skype для бизнеса Server
 
Развертывание E9-1-1 в Skype для бизнеса Server Корпоративная голосовая связь. Включает предварительные условия и контрольный список процесса развертывания.
  
Enhanced 9-1-1 (E9-1-1) — это функция уведомлений об экстренных вызовах, которая связывает номер телефона вызываемой стороны с городом или адресом улицы. Используя эти сведения, PSAP (Public Safety Answering Point) может незамедлительно направить службы экстренной помощи абоненту, находящемуся в бедственном положении.
  
Для поддержки E9-1-1 Skype для бизнеса Server должен иметь возможность правильно связать расположение с клиентом и убедиться, что эти сведения используются для маршрутации экстренного вызова на ближайший psAP.
  
## <a name="deployment-prerequisites-for-e9-1-1"></a>Необходимые компоненты для развертывания E9-1-1

Перед развертыванием E9-1-1 необходимо развернуть внутренние серверы Skype для бизнеса Server, включая центральное хранилище управления, пул переднего сервера или сервер Standard Edition. Кроме того, необходимо развернуть один или несколько серверов-посредников, как автономных, так и вместе с серверами переднего сервера. Кроме того, для развертывания E9-1-1 требуется SIP-магистраль до квалифицированного поставщика услуг E9-1-1 или шлюз ELIN до телефонной сети общего пользования (ТСОП).
