# <a name="deploy-emergency-services-in-skype-for-business-server"></a>Развертывание экстренных служб в Skype для бизнеса Server
 
Разверните E9 1 – 1 в Skype для бизнеса Server Enterprise. Includes prerequisites and deployment process checklist.
  
Улучшенная 9-1-1 (E9-1-1) — это функция, которая связывает телефонный номер вызывающего абонента с гражданским или почтовым адресом. Используя эти сведения, PSAP (Public Safety Answering Point) может незамедлительно направить службы экстренной помощи абоненту, находящемуся в бедственном положении.
  
Для поддержки E9-1-1 необходимо, чтобы приложение Skype для бизнеса Server могло правильно связывать расположение с клиентом и для того, чтобы эта информация использовалась для направления вызова экстренной помощи на ближайшее ПСАП.
  
## <a name="deployment-prerequisites-for-e9-1-1"></a>Необходимые компоненты для развертывания E9-1-1

Прежде чем развертывать E9-1-1, вам необходимо предварительно развернуть внутренние серверы Skype для бизнеса Server, в том числе централизованное хранилище управления, пул переднего плана или сервер стандартных выпусков. Кроме того, необходимо развернуть один или несколько серверов исправлений, изолированных или размещенных на серверах переднего плана. In addition, an E9-1-1 deployment requires a SIP trunk to a qualified E9-1-1 service provider or an Emergency Location Identification Number (ELIN) gateway to your public switched telephone network (PSTN).
