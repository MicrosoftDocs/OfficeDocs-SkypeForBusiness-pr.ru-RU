# <a name="install-prerequisites-for-skype-for-business-server"></a>Установка необходимых условий для Skype для бизнеса Server
 
**Сводка:** Узнайте о серверах и ролях серверов, которые необходимо настроить перед установкой Skype для бизнеса Server. Скачайте бесплатную пробную версия Skype для бизнеса Server из [Центра оценки Майкрософт.](https://www.microsoft.com/evalcenter/evaluate-skype-for-business-server)
  
Установка необходимых компонентов состоит из настройки Windows Server путем установки необходимых ролей и компонентов на каждом сервере в топологии. Требования основаны на роли, которую сервер будет выполнять в топологии. Шаги 1-5 можно делать в любом порядке. Однако шаги 6, 7 и 8 необходимо делать по порядку и после шагов с 1 по 5, как описано на схеме. Установка необходимых условий — шаг 1 из 8.
  
![Обзорная схема — установка необходимых условий.](../../media/0a85349b-b398-4e04-8901-8f4bd25d8afe.png)
  
## <a name="setup-windows-server"></a>Настройка Windows Server

Для установки Skype для бизнеса Server требуется операционная система Windows Server и ряд необходимых условий. Подробные сведения о планировании необходимых условий см. в требованиях к [серверу для Skype для бизнеса Server.](../../../SfBServer2019/plan/system-requirements.md) 
  
> [!TIP]
> В этой процедуре используется Windows Server 2012 R2. Если вы используете другую версию Windows Server, процедура может немного отличаться. 
  
> [!IMPORTANT]
> Перед началом убедитесь, что Windows Server обновлен с помощью Обновления Windows. 
  
![Windows Server в состоянии обновления.](../../media/a8d57a97-a55e-443b-b304-c534ae9a71b2.png)

### <a name="install-required-roles-and-features-for-front-end-servers"></a>Установка необходимых ролей и компонентов для серверов переднего плана

С помощью диспетчера серверов можно установить необходимые роли и функции. 
    
1. Установите необходимые функции программного обеспечения, перечисленные в требованиях к [серверу для Skype для бизнеса Server.](../../../SfBServer2019/plan/system-requirements.md) Необходимое программное обеспечение должно быть на сервере, где будет работать Skype для бизнеса Server.
    
    > [!CAUTION]
    > Windows Server 2012 R2 не устанавливает все исходные файлы для необходимых компонентов по умолчанию. Если сервер не подключен к Интернету, необходимо вставить носители Windows Server 2012 R2 и выбрать "Указать альтернативный исходный путь" для установки необходимых компонентов.  Исходные файлы находятся в каталоге sources\sxs. Например, если носители Windows Server 2012 R2 на диске D, необходимо установить путь `d:\sources\sxs` к . Важно, чтобы у вас были последние обновления из Обновления Windows. Если вы не подключены к Интернету, вам потребуется вручную установить все необходимые обновления, а также все необходимые условия для необходимых обновлений. 
  
1. Когда в диалоговом окне будет указано, что установка завершена, для завершения процесса потребуется перезагрузка сервера.
    
1. Снова **запустите Windows Update,** чтобы проверить, есть ли обновления для установленных ролей и служб.
    
1. Если вы будете использовать панель управления Skype для бизнеса Server на этом сервере, необходимо также установить Silverlight. Чтобы установить Silverlight, [см. microsoft Silverlight.](https://www.microsoft.com/silverlight/)


> [!IMPORTANT]
> Необходимые условия для серверов, которые выполняют роли, кроме сервера переднего плана, такие как роль Директора, сохраняемого чата или Edge, имеют собственные необходимые условия. Подробные сведения о предварительных требованиях, необходимых для каждого типа сервера, см. в требованиях к серверу [для Skype для бизнеса Server.](../../../SfBServer2019/plan/system-requirements.md)
