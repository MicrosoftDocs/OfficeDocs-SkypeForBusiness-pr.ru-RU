# <a name="install-prerequisites-for-skype-for-business-server"></a>Установка необходимых компонентов для Skype для бизнеса Server
 
**Сводка:** Сведения о серверах и ролях сервера, которые необходимо настроить перед установкой Skype для бизнеса Server.
  
Установка необходимых компонентов состоит из настройки Windows Server путем установки необходимых ролей и компонентов на каждом сервере в топологии. Требования основаны на роли, которая будет выполняться сервером в топологии. Вы можете выполнить шаги с 1 по 5 в любом порядке. Однако необходимо выполнить шаги 6, 7 и 8 по порядку и после шагов 1–5, как показано на схеме. Установка необходимых компонентов — шаг 1 из 8.
  
![Обзорная схема: предварительные требования для установки.](../../media/0a85349b-b398-4e04-8901-8f4bd25d8afe.png)
  
## <a name="setup-windows-server"></a>Настройка Windows Server

Skype для бизнеса Server требуется операционная система Windows Server и ряд предварительных условий, прежде чем ее можно будет установить. Дополнительные сведения о планировании предварительных требований см. в разделе "[Требования к серверу для Skype для бизнеса Server](../../../SfBServer2019/plan/system-requirements.md)". 
  
> [!TIP]
> Эта процедура использует Windows Server 2012 R2. Если вы используете другую версию Windows Server, процедура может немного отличаться. 
  
> [!IMPORTANT]
> Перед началом работы убедитесь, что Windows Server находится в актуальном состоянии с помощью клиентский компонент Центра обновления Windows. 
  
![Windows Server в актуальном состоянии.](../../media/a8d57a97-a55e-443b-b304-c534ae9a71b2.png)

### <a name="install-required-roles-and-features-for-front-end-servers"></a>Установка необходимых ролей и компонентов для интерфейсных серверов

Необходимые роли и компоненты можно установить с помощью диспетчер сервера. 
    
1. Установите необходимые функции программного обеспечения, перечисленные [в требованиях к серверу для Skype для бизнеса Server](../../../SfBServer2019/plan/system-requirements.md). Необходимое программное обеспечение должно находиться на сервере, который будет работать Skype для бизнеса Server.
    
    > [!CAUTION]
    > Windows Server 2012 R2 не устанавливает все исходные файлы для необходимых компонентов по умолчанию. Если сервер не подключен к Интернету, необходимо вставить носитель Windows Server 2012 R2 и выбрать "Указать альтернативный путь к  источнику", чтобы установить необходимые компоненты. Исходные файлы находятся в каталоге sources\sxs. Например, если носитель Windows Server 2012 R2 находится на диске D, необходимо задать путь `d:\sources\sxs`к . Важно, чтобы у вас были последние обновления из клиентский компонент Центра обновления Windows. Если вы не подключены к Интернету, необходимо вручную установить все соответствующие обновления, а также все необходимые компоненты для необходимых обновлений. 
  
1. Когда диалоговое окно указывает, что установка завершена, для завершения процесса необходимо перезагрузить сервер.
    
1. **Запустите клиентский компонент Центра обновления Windows** еще раз, чтобы проверить наличие обновлений для установленных ролей и служб.
    
1. Если вы будете использовать Skype для бизнеса Server панель управления на этом сервере, необходимо также установить Silverlight. Сведения об установке Silverlight см. [в разделе Microsoft Silverlight](https://www.microsoft.com/silverlight/).


> [!IMPORTANT]
> Предварительные требования для серверов, которые выполняют роли, отличные от серверов переднего плана, такие как роль директора, сохраняемого чата или Пограничного сервера, имеют свои собственные предварительные требования. Дополнительные сведения о конкретных предварительных требованиях, необходимых для каждого типа сервера, см. в разделе "Требования к Skype для бизнеса Server["](../../../SfBServer2019/plan/system-requirements.md).
