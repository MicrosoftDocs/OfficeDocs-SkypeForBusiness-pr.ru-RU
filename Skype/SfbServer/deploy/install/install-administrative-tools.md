# <a name="install-administrative-tools-in-skype-for-business-server"></a>Установка средств администрирования в Skype для бизнеса Server
 
**Сводка:** Узнайте, как установить средства администрирования, необходимые для установки Skype для бизнеса Server.
  
Средства администрирования включают построитель топологий и панель управления. Средства администрирования должны быть установлены по крайней мере на одном сервере в топологии или на 64-разрядной рабочей станции управления с версией Windows ОС, которая поддерживается Skype для бизнеса Server. Вы можете выполнить шаги с 1 по 5 в любом порядке. Однако необходимо выполнить шаги 6, 7 и 8 по порядку и после шагов 1–5, как показано на схеме. Установка средств администрирования — шаг 3 из 8.
  
![Обзорная схема.](../../media/d856afe8-4758-432f-bc45-e1956016419a.png)
  
## <a name="install-skype-for-business-server-administrative-tools"></a>Установка Skype для бизнеса Server администрирования

Установный носитель для Skype для бизнеса Server предоставляет гибкий интерфейс. При первом запуске Setup.exe установлены только мастер развертывания Skype для бизнеса Server и Skype для бизнеса Server Management Shell. С помощью этих двух средств, называемых основными компонентами, можно продолжить процесс установки, но они не предоставляют основные функции для всей Skype для бизнеса Server среды. Мастер развертывания запускается автоматически после установки основных компонентов. В разделе мастера развертывания "Установка средств администрирования" Skype для бизнеса Server построитель топологий и Skype для бизнеса Server панель управления.
  
> [!IMPORTANT]
> В Skype для бизнеса Server среды должен быть хотя бы один сервер с установленными средствами администрирования. 
  
Просмотрите видео с инструкциями по **установке средств администрирования**:

### <a name="install-skype-for-business-server-administrative-tools-from-the-deployment-wizard"></a>Установка Skype для бизнеса Server администрирования из мастера развертывания

1. Вставьте Skype для бизнеса Server установки. Если установка не начинается автоматически, дважды щелкните " **Настройка"**.
    
2. Установимый носитель Microsoft Visual C++ запуска. Отобразит диалоговое окно с вопросом, хотите ли вы установить его. Нажмите кнопку **Да**.
    
3. С помощью интеллектуальной установки, новой функции в Skype для бизнеса Server, можно подключиться к Интернету, чтобы проверить наличие обновлений во время установки. Это обеспечивает более эффективное взаимодействие, благодаря установке последних обновлений продукта. Нажмите **кнопку "** Установить", чтобы начать установку.
    
4. Внимательно изучите лицензионное соглашение и, если вы согласны, выберите "Я принимаю условия лицензионного соглашения" и нажмите кнопку "**ОК"**.
    
5. На Skype для бизнеса Server будут установлены основные компоненты. 
    
    Основные компоненты состоят из следующих компонентов, как показано на рисунке.
    
    ![Основные компоненты на экране "Приложения".](../../media/0da1d983-4c4b-4b23-a196-c3bdba4857c6.png)
  
   - **Skype для бизнеса Server развертывания** — программа развертывания, которая предоставляет панель запуска для установки различных компонентов Skype для бизнеса Server.
    
   - **Skype для бизнеса Server командной консоли** — предварительно настроенная программа PowerShell, которая позволяет Skype для бизнеса Server.
    
     После завершения установки основных компонентов автоматически запустится Skype для бизнеса Server развертывания, как показано на рисунке. 
    
     ![Skype для бизнеса Server развертывания.](../../media/310c3437-83f9-48fa-a1e1-9fd09009fe31.png)
  
6. Помимо основных компонентов, вам также потребуется установить Skype для бизнеса Server построитель топологий и Skype для бизнеса Server панель управления по крайней мере на одном сервере в среде. **Щелкните "Установить средства администрирования**" в мастере развертывания.
    
7. Чтобы начать установку, нажмите кнопку **Далее**.
    
8. После завершения установки нажмите кнопку **"Готово"**. Средства администрирования теперь добавляются на сервер, как показано на рисунке.
    
    ![Skype для бизнеса Server администрирования.](../../media/760873dd-9c87-4efb-bf98-7162d876fd18.png)
  
   - **Skype для бизнеса Server построитель топологий** программа, используемая для создания, развертывания топологий и управления ими.
    
   - **Skype для бизнеса Server панель управления** программа, используемая для администрирования установки.
