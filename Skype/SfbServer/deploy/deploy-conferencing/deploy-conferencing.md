# <a name="deploy-conferencing-in-skype-for-business-server"></a>Развертывание конференций в Skype для бизнеса Server

**Сводка:** Ознакомьтесь с этой темой, чтобы узнать, как развернуть конференцию в Skype для бизнеса Server.

В Skype для бизнеса Server доступны четыре типа телефонных разговоров: веб-конференция, аудио- и видеоконференция (A/V), диалоговые и мгновенные сообщения(IM). Вы можете включить все типы конференций или использовать только один тип в зависимости от ваших потребностей.

При развертывании Skype для бизнес-сервера автоматически развертываются возможности конференций для im. При создании и публикации новой топологии с помощью Topology Builder вы указываете, следует ли развертывать веб-, A/V и диалоговое конференцию, как описано в следующих контрольных списках:

- [Контрольный список развертывания для веб-и аудио- и видеоконференций](deploy-conferencing.md#BKMK_ChecklistWebConferencing)

- [Поток развертывания и контрольный список для телефонных конференциалов](deploy-conferencing.md#BKMK_DialinConferencing)

## <a name="deployment-flowchart-and-checklist-for-dial-in-conferencing"></a>Поток развертывания и контрольный список для телефонных конференциалов
<a name="BKMK_DialinConferencing"> </a>

 Конференц-связь с диалогом позволяет пользователям набирать номера из сети общедоступных переключеных телефонов (PSTN), чтобы присоединиться к аудио- и видеоконференции.

Некоторые компоненты, необходимые для телефонных конференций, также используются для Корпоративная голосовая связь. Например, при развертывании Корпоративная голосовая связь необходимо также развернуть сервер-посредник и шлюз PSTN — компоненты, которые также необходимы для телефонных конференций. Поэтому развертывание телефонных конференций зависит от того, развертывается ли Корпоративная голосовая связь решение.

На потоке телефонных телефонных диалогов показаны действия, которые необходимо выполнять в зависимости от того, развертывается ли Корпоративная голосовая связь решение. В таблице, следующей за потоком, представлен обзор действий, необходимых и рекомендуемых для развертывания телефонных переговоров. Также включены ссылки на связанную документацию по планированию и процедуре. Дополнительные сведения о планировании полного решения Корпоративная голосовая связь см. в Корпоративная голосовая связь [в Skype для бизнеса Server.](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice-solution.md)
