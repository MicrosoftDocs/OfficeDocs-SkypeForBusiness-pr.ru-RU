# <a name="deploy-conferencing-in-skype-for-business-server"></a>Развертывание conferencing в Skype для бизнеса Server

**Сводка:** В этом разделе вы узнаете, как развернуть в Skype для бизнеса Server возможность работы с conferencing.

В Skype для бизнеса Server доступно четыре типа аудио- и видеоконференций, а также аудио- и видеоконференций. Вы можете включить все типы или использовать только один тип в зависимости от ваших потребностей.

При развертывании Skype для бизнеса Server функции im conferencing развертываются автоматически. При создании и публикации новой топологии с помощью построщика топологий вы указываете, следует ли развертывать веб-, видео- и видеоконференцию, а также видеоконференцию, как описано в следующих контрольных списках:

- [Flowchart and checklist for dial-in conferencing](deploy-conferencing.md#BKMK_DialinConferencing)

## <a name="deployment-flowchart-and-checklist-for-dial-in-conferencing"></a>Flowchart and checklist for dial-in conferencing

Некоторые компоненты, необходимые для телефонной видеоконференции, также используются для Корпоративная голосовая связь. Например, при развертывании Корпоративная голосовая связь необходимо также развернуть сервер-посредник и шлюз STN — компоненты, которые также необходимы для телефонной комконференации. Следовательно, способ развертывания телефонной связи зависит от того, развертывается ли Корпоративная голосовая связь решения.

На flowchart conferencing conferencing dial-in shows the steps you must follow depending on whether you are also deploying an Корпоративная голосовая связь solution. В таблице, следующей за потоком, представлен обзор действий, необходимых и рекомендуемых для развертывания телефонной системы. Также включены ссылки на связанную документацию по планированию и процедуре.
