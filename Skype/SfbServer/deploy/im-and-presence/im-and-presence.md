# <a name="deploy-instant-messaging-and-presence-in-skype-for-business-server-2015"></a>Развертывание функций обмена мгновенными сообщениями и сведениями о присутствии в Skype для бизнеса Server 2015
 
**Краткое содержание**. В этой статье содержатся инструкции по развертыванию обмена мгновенными сообщениями и сведениями о присутствии в Skype для бизнеса Server 2015.
  
## <a name="deployment-steps-for-instant-messaging-and-presence"></a>Развертывание обмена мгновенными сообщениями и оповещения о присутствии

В статье, посвященной планированию обмена мгновенными сообщениями и сведениями о присутствии, приводятся пошаговые инструкции по планированию развертывания. Обмен мгновенными сообщениями и присутствие включено при установке Скайп для Business Server 2015. Дополнительные сведения о планировании обмен мгновенными сообщениями и присутствия, видеть [Планирование обмена мгновенными сообщениями и присутствия в Скайп для Business Server 2015](../../plan-your-deployment/instant-messaging-and-presence.md).
