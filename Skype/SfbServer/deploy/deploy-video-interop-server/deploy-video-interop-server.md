# <a name="deploy-video-interop-server-in-skype-for-business-server-2015"></a>Развертывание сервера видеовзаимодействия в Skype для бизнеса Server 2015
 
**Сводка:** Развертывание VIS серверной роли в Скайп для Business Server 2015.
  
Скайп для Business Server теперь можно интегрировать непосредственно с системами телеконференций Cisco (VTCs), например Cisco C60 или Cisco MX300. Для этого необходимо введение новую роль сервера видео взаимодействия сервера по ДИАГОНАЛИ и правильную настройку VIS и оборудование будет будет взаимодействовать с. VTC проходит регистрацию в существующей инфраструктуре Cisco, например Cisco Unified Communication Manager (CUCM), а между CUCM и пулом VIS используется видеомагистраль SIP.

- [Создание пула VIS в Скайп для Business Server 2015](create-a-vis-pool.md)
    
- [Развертывание VIS серверной роли в Скайп for Business Server 2015](deploy-the-vis-server-role.md)
    
- [Настройка видео взаимодействия сервера в Скайп для Business Server 2015](configure-the-vis.md)
    
- [Настройка CUCM для взаимодействия с Скайп для Business Server 2015](configure-cucm-for-interoperation.md)
