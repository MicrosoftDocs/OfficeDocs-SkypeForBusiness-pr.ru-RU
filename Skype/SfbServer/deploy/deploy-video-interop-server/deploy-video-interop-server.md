# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Развертывание сервера видеосвязи в Skype для бизнеса Server
 
**Сводка:** Развертывание роли сервера VIS в Skype для бизнеса Server.
  
Skype для бизнеса Server теперь можно интегрировать напрямую с системами телеконференции Cisco (VTCs), такими как Cisco C60 или Cisco MX300. Для этого необходимо ввести новую роль сервера, называемую сервером video interop Server (VIS), и правильную конфигурацию сервера vis и оборудования, с помощью котором оно будет работать. VTC регистрируется в существующей инфраструктуре Cisco, такой как Cisco Unified Communication Manager (CUCM), а между CUCM и пулом VIS используется видеоМАПК.

Для настройки взаимодействия между сервером vis или пулом и системами VTC необходимо выполнить следующие пять процедур: 
  
- [Создание пула VIS в Skype для бизнеса Server](create-a-vis-pool.md)
    
- [Развертывание роли сервера VIS в Skype для бизнеса Server](deploy-the-vis-server-role.md)
