# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Развертывание видео взаимодействия сервера в Скайп для Business Server
 
**Сводка:** Развертывание VIS серверной роли в Скайп для Business Server.
  
Скайп для Business Server теперь можно интегрировать непосредственно с системами телеконференций Cisco (VTCs), например Cisco C60 или Cisco MX300. Для этого необходимо введение новую роль сервера видео взаимодействия сервера по ДИАГОНАЛИ и правильную настройку VIS и оборудование будет будет взаимодействовать с. VTC проходит регистрацию в существующей инфраструктуре Cisco, например Cisco Unified Communication Manager (CUCM), а между CUCM и пулом VIS используется видеомагистраль SIP.
  
## <a name="in-this-section"></a>Содержание

Настройка взаимодействия между серверов видеовзаимодействия или пулом и системами VTC требуется выполнения следующих пяти процедур: 
  
- [Создание пула VIS в Скайп для Business Server](create-a-vis-pool.md)
    
- [Развертывание VIS серверной роли в Скайп for Business Server](deploy-the-vis-server-role.md)
    
- [Настройка видео взаимодействия сервера в Скайп для Business Server](configure-the-vis.md)
    
- [Настройка CUCM для взаимодействия с Скайп для Business Server](configure-cucm-for-interoperation.md)
    
- [Настройка VTC для взаимодействия с Скайп для Business Server](configure-a-vtc-for-interoperation.md)
