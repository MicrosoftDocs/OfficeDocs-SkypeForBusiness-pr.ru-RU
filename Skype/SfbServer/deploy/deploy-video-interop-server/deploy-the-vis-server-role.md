# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Развертывание роли сервера VIS в Skype для бизнеса Server
 
**Сводка:** Развертывание роли video Interop Server (VIS) в Skype для бизнеса Server.
  
Чтобы настроить службу VIS на сервере, только что созданном в Topology Builder, запустите мастер развертывания Skype для бизнеса Server, нажмите кнопку Установить или обновить **Skype для бизнеса Server System** и выполните следующие действия в мастере:
  
1.  Выберите **Установите локальный магазин конфигурации.**
    
2. Выберите **установку или удаление Skype для бизнеса Server компонентов.**
    
3. Выберите **запрос, установка или назначение сертификатов.**
    
4. Выберите **Начните службы**.
    
Программное обеспечение для этой службы теперь установлено и запущено. Вы можете открыть средство MMC Services, чтобы узнать, работает ли Skype для бизнеса Server **video Interop Server** вместе с другими Skype для бизнеса Server службами. Далее необходимо настроить viS-сервер или пул.
