# <a name="skype-room-system-manageability-and-tools"></a>Управление системой комнат Skype и инструменты
 
В этом разделе вы узнаете о средствах управления для системы комнат Skype.
  
## <a name="administrative-portal"></a>Портал администрирования

Для локального развертывания Skype для бизнеса Server можно использовать портал администрирования системы комнат Skype для активного управления развертываниями системы комнат Skype в организации и отслеживания их развертывания.
  
Дополнительные сведения см. в следующей статье:
  
- [Развертывание веб-портала администрирования SRS 1 в Skype для бизнеса Server](../deploy-conferencing/room-system-v1-administrative-web-portal.md)
    
  
## <a name="exchange-checklist"></a>Контрольный список Exchange

- Подтвердим, что автооружие настроено, и для autodiscover.domain.com доступна внутренняя запись DNS A/CNAME.
    
- Автооружение Ping (например, Ping Autodiscover.contoso.com).
    
- Протестировать службу автооружия с помощью анализатора подключений Майкрософт. Выберите первый тест "Не могу войти в Office Outlook".
    
- If the meeting room already has a resource mailbox, extend this account for the Skype Room System (example script at the bottom of the page).
    
## <a name="skype-for-business-checklist"></a>Контрольный список Skype для бизнеса

- Запустите следующие средства:
    
  - Анализатор лучших методик Skype для бизнеса     
  - Средство анализа состояния Skype для бизнеса (Excel)    
  - 32-битный или 64-битный анализатор подключений Skype для бизнеса
    
- Просмотрите [полезные новые средства устранения неполадок и анализа для Office 365.](https://blogs.technet.microsoft.com/educloud/2013/08/13/useful-new-troubleshooting-and-analysis-tools-for-office-365/) Подтвердим, что у вас есть пул Skype для бизнеса и сервер Office Web Apps, и вы можете совместно использовать колоду PowerPoint с помощью клиента Skype для бизнеса.
    
- Если в комнате для собраний уже есть почтовый ящик ресурса, в качестве его можно использовать Skype для бизнеса.
    
- При необходимости запросим did (номер телефона) для системы комнат переговоров и обновим поле "Общий телефон" в средстве Active Directory.
    
## <a name="network"></a>Сеть

- Убедитесь, что у вас есть проводное сетевое подключение для системы комнат Skype.
    
- Ознакомьтесь с руководством по планированию сети для Skype для бизнеса.
    
- Запросить оценку сети Skype для бизнеса у партнера Skype для бизнеса.
    
- Прочитайте данные о производительности, полученные в средстве анализа работоспособности Skype для бизнеса (Excel).
    
- Запустите средство анализа сети.
    
- Запустите средство диагностики перед вызовом.
    
## <a name="skype-room-system-security"></a>Безопасность системы комнат Skype

Система комнат Skype — это встроенная система, которая может быть полностью интегрирована в развертывание Windows с помощью модели безопасности Skype для бизнеса, управления правами и средств управления, таких как SCOM. Поддерживаются перечисленные ниже возможности. 
  
- Фильтр записи для предотвращения записи диска в пользовательском режиме 
    
- Блокировка приложений для предотвращения запуска несанкционированных приложений. Все USB-порты отключены в пользовательском режиме.
    
  - На оборудовании системы комнат Skype нет стандартных приложений или средств просмотра. Весь контент отрисовка осуществляется по протоколам HTTP или RDP.
    
  - На компьютере устройства работает операционная система Windows Embedded Standard 7. Все оборудование, включая устройства, предоставляется партнерами по OEM.
    
  - Необязательное присоединить домен к доменным службам Active Directory (AD DS), включив локальное управление учетными записями безопасности.
    
- Вы также можете управлять учетной записью локального администратора с помощью Центра администрирования Skype для бизнеса.
    
- Система комнат Skype обновляется с помощью стандартных процессов Обновления Майкрософт.
    
- Система комнат Skype подключается к Skype для бизнеса.
    
  - Skype для бизнеса использует все виды шифрования и авторизации для всех режимов связи
    
  - Система комнат Skype поддерживает стандарты безопасности и соответствия требованиям Skype для бизнеса. Дополнительные [сведения см.](../../plan-your-deployment/security/security.md) в сведениях о планировании безопасности в Skype для бизнеса Server.
    
## <a name="license"></a>Лицензия

Убедитесь, что для активации программного обеспечения используется KMS. В этом случае вам может потребоваться проверить или добавить в него ключ KMS клиента Skype для бизнеса. Если вы не используете KMS, запросить ключ корпоративного лицензирования для ключа MAK клиента Skype для бизнеса.
  
## <a name="license-keys"></a>Ключи лицензий

Система комнат Skype запускает настольный клиент Skype для бизнеса в фоновом режиме. Если система комнат Skype является членом домена, она обнаружит ваш KMS. (и если он имеет ключ KMS корпоративного лицензирования, он будет активироваться автоматически). Корпоративное лицензирование также предоставляет mak, который вы вводите при отобраобраи xxxxx-xxxxx-xxxxx-xxxxx. (Для активации с помощью MAK необходим доступ к Интернету, но не KMS). Дополнительные сведения см. в теме "Активация office 2013 с многояговиной активацией".
  
- Чтобы ввести ключ MAK, перейдите в средство лицензирования SRS "Параметры \> OEM". Щелкните Проверить состояние . Если в состоянии "продукт не активирован", введите ключ.
    
- Если во время активации вы получите сообщение об ошибке "'Служба лицензирования программного обеспечения сообщила, что ключ продукта недействителен", убедитесь, что:
    
  - Ключ введен правильно.
    
  - Вы ввели ключ MAK Skype для бизнеса, а не другой ключ.
    
  - Система имеет доступ к Интернету.
    
- Вы можете активироваться по телефону, но сначала должны были попытаться активироваться с помощью средства лицензирования SRS. Чтобы активироваться по телефону, начните тестовую встречу (а не тестовый звонок, так как он слишком короткий). В мастере активации Office выберите "Активация по телефону", позвоните в корпорацию Майкрософт, введите длинный номер и введите ответ.
    
## <a name="certificate-authority"></a>Certificate Authority

Подтвердим, что в сертификате Office Web Apps Server 2013, используемом для выдачи сертификата Office Web Apps Server 2013, есть HTTP-путь, включенный в свойство списка отзыва сертификатов.
  
Импорт файла сертификата (CRT) в систему комнат Skype при использовании Skype для бизнеса Server. Его легко получить из папки CertEnroll сервера ЦС или в папке "Доверенный корневой" любого компьютера, который присоединяется к домену.
  
## <a name="certificates"></a>Сертификаты

Убедитесь, что в вашем сертификате есть http-путь для списка отзыва сертификатов. Если нет, обновите свой ЦС, чтобы включить его.
  
Установите сертификаты в конфигурации администратора системы комнат Skype в диспетчере сертификатов \> параметров системы. Для внутреннего сертификата необходим корпоративный корневой ЦС.
  
Один из способов получить необходимые сертификаты — обнаружить ЦС, выдав сертификаты. For Skype for Business Server, on a PC in Skype for Business, click Settings \> Tools \> Dial-in Conference Settings. Откроется веб-страница, защищенная ЦС, выдав внутренние сертификаты. Щелкните значок "Блокировка" в адресной панели браузера, чтобы отобразить отчет о безопасности. Щелкните "Просмотреть сертификаты" и проверьте свойство точки распространения CRL. Второй параметр CN должен быть именем сервера ЦС. Теперь откройте проводник Windows для этого \\ \< CA Server Name \> адреса \CertEnroll. Скопируйте два CRL-файла и CRT-файл на флэш-накопитель и поместите его в левую часть СМАРТ-доски.
  
Импортировать CRT-файл в систему комнат Skype в папке "Доверенный центр сертификации помещений".
  
Импортировать CRL-файлы в систему комнат Skype в папку промежуточных сертификатов. (Чтобы увидеть файлы, необходимо изменить фильтр расширения файлов в диспетчере сертификатов на CRL).
  
Примечание. Сервер Office Web Apps 2013 может совместно использовать тот же ЦС, что и Skype для бизнеса. Если это не так, вы не сможете поделиться PowerPoint на собрании. Ознакомьтесь с ИТ-ит-частью и получите CRT- и CRL-файлы из сетевой папки ЦС CertEnroll, как поясняется выше. 
  
Членство в домене может упростить некоторые аспекты, так как вы можете рассматривать систему комнат Skype как систему Windows и использовать Active Directory для некоторых аспектов сертификата. Тем не менее, лучше управлять этим вручную.
