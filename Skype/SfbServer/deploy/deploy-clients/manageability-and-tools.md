# <a name="skype-room-system-manageability-and-tools"></a>Возможности управления и инструменты для системы комнат Skype
 
В этом разделе описываются возможности инструментов управления для Системы комнат Skype.
  
## <a name="administrative-portal"></a>Портал администрирования

Для Скайп для локального развертывания Business Server портал администрирования системы комнаты Скайп можно использовать для активно управления и мониторинга системы комнаты Скайп развертываний вашей организации.
  
В следующей статье для получения дополнительных сведений:
  
- [Развертывание SRS v1 административного веб-портала в Скайп for Business Server](../deploy-conferencing/room-system-v1-administrative-web-portal.md)
    
## <a name="system-center-operations-manager"></a>System Center Operations Manager

Система Скайп комнаты можно отслеживать через System Center Operations Manager (SCOM), [Пакет управления System комнаты Скайп](https://www.microsoft.com/download/details.aspx?id=42320) загрузив и установив его на сервер SCOM. SCOM можно использовать для параметров оповещений, за работоспособностью системы Скайп помещения, создавать отчеты о работоспособности и многое другое. Система комнаты Скайп поставляется с предварительно установленные мониторинга агента, который можно настроить для указания на сервер SCOM. Обратитесь к руководство по установке предоставлено производителем системы комнаты Скайп необходимо знать о настройке мониторинга агента Скайп комнаты системе.
  
## <a name="exchange-checklist"></a>Контрольный список Exchange

- Убедитесь, что автообнаружение настроено и внутренняя запись DNS A/CNAME RECORD доступна для домена autodiscover.domain.com.
    
- Проверьте связь автообнаружения (например, проверьте связь с доменом Autodiscover.contoso.com).
    
- Протестируйте службу автообнаружения с помощью анализатора подключений корпорации Microsoft. Выбор первого теста, «Не удается войти с помощью Microsoft Office Outlook.»
    
- Если зала уже почтового ящика ресурса, расширьте эту учетную запись для системы комнаты Скайп (пример сценария, в нижней части страницы).
    
## <a name="skype-for-business-checklist"></a>Скайп для бизнеса контрольного списка

- Запустите следующие инструменты:
    
  - Скайп для бизнеса анализатора соответствия рекомендациям     
  - Скайп для средство анализа бизнеса (Excel)    
  - Скайп для Business Connectivity Analyzer 32-разрядная или 64-разрядная версия
    
- Ознакомьтесь с разделом [Новые полезные инструменты для устранения неисправностей и анализа для Office 365](https://blogs.technet.microsoft.com/educloud/2013/08/13/useful-new-troubleshooting-and-analysis-tools-for-office-365/). Подтверждение у пользователя имеется Скайп для бизнеса пула и сервера Office Web Apps и могут совместно использовать набор PowerPoint с помощью Скайп для клиента Business.
    
- Если зала уже почтового ящика ресурса, включить эту функцию для Скайп для бизнеса.
    
- При необходимости запросите DID (номер телефона) для системы комнат переговоров и обновите поле "Общий телефон" в инструменте Active Directory.

## <a name="skype-room-system-security"></a>Скайп места системы безопасности

Скайп помещения – внедренных система, полностью интегрирован в развертывании Windows, с помощью Скайп для бизнес-модели безопасности, управления правами и средства управления, такие как SCOM. Доступны следующие функции.
  
- Фильтр записи для предоставления записи на диск в пользовательском режиме 
    
- Блокировщик приложений для предотвращения запуска несанкционированных приложений. В пользовательском режиме все порты USB отключены.
    
  - Нет стандартных приложений или средства просмотра хранятся на устройстве Скайп комнаты оборудовании. All content is rendered via HTTP or RDP protocols.
    
  - Бытовой компьютер работает под управлением операционной системы Windows Embedded Standard 7. Все оборудование, включая устройства, предоставляются партнерами изготовителя оборудования.
    
  - Присоединение необязательного домена к службе Active Directory Domain Services (AD DS) обеспечивает управление и контроль локальных учетных записей безопасности.
    
- Также можно управлять с помощью Скайп по центру администрирования Business учетная запись локального администратора.
    
- Система комнаты Скайп обновляется посредством стандартных процессов Центр обновления Майкрософт.
    
- Система комнаты Скайп подключается с Скайп для бизнеса.
    
  - Скайп для бизнеса с помощью шифрования начала до конца и авторизации для всех режимы связи
    
  - Система комнаты Скайп поддерживает Скайп для бизнеса безопасность и соответствие требованиям стандартов. [Безопасность в Скайп Business Server](../../plan-your-deployment/security/security.md) для получения дополнительных сведений см.
    
## <a name="license"></a>Лицензия

Убедитесь, что используете KMS для активации программного обеспечения. Если это так, может потребоваться проверить или добавить Скайп для бизнеса клиентский ключ KMS. Если вы не используете KMS, запросите ключа корпоративной лицензии для Скайп для клиента Business MAK.
  
## <a name="license-keys"></a>Ключи лицензии

Система комнаты Скайп запускает Скайп для настольных компьютеров клиента Business в фоновом режиме. Если система комнаты Скайп является членом домена, ему будет обнаружить вашей KMS. (и, если он имеет Volume Licensing ключ KMS автоматическая активация). Volume Licensing также предоставляет ключ MAK, который вводится как отображает xxxxx-xxxxx-xxxxx-xxxxx. (Требуется доступ к Интернету для активации с помощью ключа MAK, но не KMS). Дополнительные сведения см в многопользовательской активации Office 2013.
  
- Чтобы ввести ключ многократной Активации, перейдите в раздел ПВТ параметры \> средство лицензирования SRS. Выберите "Проверка состояния". Когда состояние с текстом «продукт не активирован», введите ключ.
    
- Если во время активации появится сообщение о том, «"Служба лицензирования программного обеспечения сообщает, что ключ продукта недействителен,» затем убедитесь, что:
    
  - Ключ введен правильно.
    
  - Скайп ввести ключ многократной Активации бизнеса и не другой ключ.
    
  - У системы есть доступ к Интернету.
    
- Вы можете выполнить активацию с помощью телефона, но сначала потребуется выполнить попытку активации с помощью средства лицензирования SRS. Для активации через телефон начните тестовое собрание (не тестовый звонок, так как он слишком короткий). В окне мастера активации Office выберите активация по телефону, звонок Microsoft, ввести длинный номер и введите ответ.
    
## <a name="certificate-authority"></a>Центр сертификации

Убедитесь, что центр сертификации, выдавший сертификат Office Web Apps Server 2013, включил путь HTTP в свойство списка отзыва сертификатов (CRL).
  
Импортируйте файл сертификата (.crt) системы комнаты Скайп при использовании Скайп для Business Server. It's easily obtained from the CertEnroll share of the CA server, or in the Trusted Root folder of any domain joined PC.
  
## <a name="certificates"></a>Сертификаты

Убедитесь, что у Центра сертификации есть путь http для списка отзыва сертификатов (CRL). Если нет, обновите Центр сертификации, чтобы добавить путь.
  
Установка сертификатов в программе установки администрирования системы Скайп помещений в разделе Параметры системы \> диспетчера сертификатов. You need the Enterprise Root CA for your internal certificate.
  
Единственный способ получить необходимые сертификаты — обнаружить центр сертификации, выдавший сертификаты. Для Скайп сервера Business на ПК в Скайп для бизнеса, нажмите кнопку Параметры \> средств \> параметры конференции. Откроется веб-страница с центром сертификации, выпустивший внутренние сертификаты. Нажмите на значок замка в адресной строке браузера для отображения отчета по безопасности. Нажмите "Просмотр сертификатов" и проверьте свойство "Точка распространения списка отзыва сертификатов". Второй параметр CN должен быть именем сервера центра сертификации. Теперь откройте проводник Windows для этого адреса \\ \< имя сервера центра сертификации \>\CertEnroll. Скопируйте два файла .crl и файл .crt на устройство флэш-памяти и поместите его с левой части доски SMART.
  
Импортируйте файл .crt системы комнаты Скайп в папке доверенного центра сертификации комнаты.
  
Импорт файлов .crl в системе комнаты Скайп в папке промежуточные центры сертификации. (You need to change the file extension filter in Certificate Manager to .crl to see the files).
  
Примечание: Сервер Office Web Apps 2013 могут совместно использовать же сертификации как Скайп для бизнеса. Если это не так вы не сможете совместное использование PowerPoint на собрании. Обратитесь к ИТ и получить CRT и файлы CRL подключен к сети ЦС совместно использовать CertEnroll, как описано выше. 
  
Членство в домене позволяет упростить некоторые факторы, поскольку системы Скайп комнаты можно рассматривать как системы Windows и его можно использовать Active Directory для некоторых аспектов сертификата. However, it's best to manually manage this.
