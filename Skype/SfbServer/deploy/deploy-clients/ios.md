# <a name="install-and-test-skype-for-business-for-ios"></a>Установка и тестирование Skype для бизнеса для iOS
 
**Сводка:** Установка и тестирование приложения Skype для бизнеса для iOS.
  
Приложение Skype для бизнеса для iOS обеспечивает присутствие в Skype для бизнеса, обмен мгновенными сообщениями, голосовые и видеозвозы на устройства с iOS. Пользователи Lync 2013 получат обновленное приложение автоматически или получат запрос на обновление вручную в зависимости от параметров пользователя. Новые пользователи могут скачать его из Apple App Store. Приложение Skype для бизнеса для iOS доступно только в iOS версий 8.0 и более поздних версий.
  
## <a name="installing-skype-for-business-for-ios"></a>Установка Skype для бизнеса для iOS

1. На устройстве с iOS **коснитесь App Store** и найди **Skype для бизнеса для iOS.**
    
2. Tap **Download**  >  **Open**. 
    
3. Во sign in to the app.
    
## <a name="verifying-mobile-client-installation"></a>Проверка установки мобильного клиента

После успешного входе с помощью следующих тестов проверьте, правильно ли работает ваша установка Skype для бизнеса на устройстве с iOS. 
  
### <a name="search-for-a-contact-in-the-corporate-directory"></a>Поиск контакта в корпоративном каталоге

1. В списке контактов коснитесь верхней части панели поиска и начните вводить имя контакта, которое существует только в глобальном списке адресов (GAL). 
    
2. Убедитесь, что имя контакта отображается в результатах поиска. 
    
### <a name="test-instant-messaging-and-presence"></a>Проверка обмена мгновенными сообщениями и присутствия

1. В списке контактов коснитесь контакта. 
    
2. Убедитесь, что появится окно обмена мгновенными сообщениями и вы можете ввести и отправить мгновенные сообщения. 
    
### <a name="test-dial-out-conferencing"></a>Тестирование dial-out conferencing

1. В Outlook запланировать собрание Skype для бизнеса. 
    
2. На мобильном устройстве откройте приглашение на собрание. 
    
3. Нажмите **кнопку "Присоединиться к собранию".**
    
4. Ответьте на звонок из службы конференц-связи и убедитесь, что вы подключены к аудиосвязи собрания. 
    
### <a name="test-push-notifications"></a>Тестирование push-уведомлений

1. On user A's mobile device, sign in to Skype for Business with user A's account. 
    
2. Откройте другое приложение на мобильном устройстве. 
    
3. На другом клиенте во sign in to Skype for Business with user B's account. 
    
4. Отправьте мгновенные сообщение от пользователя Б пользователю А. 
    
5. Убедитесь, что уведомление об мгновенных сообщениях отображается на мобильном устройстве пользователя А.
