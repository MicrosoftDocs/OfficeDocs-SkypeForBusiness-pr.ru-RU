# <a name="skype-room-system-and-skype-for-business-federated-partners"></a>Skype и Skype для бизнеса федерадных партнеров
 
Ознакомьтесь с этой темой, чтобы узнать, как Skype систему номеров для Skype для бизнеса федераированных партнеров.
  
## <a name="skype-room-system-and-skype-for-business-federated-partners"></a>Skype и Skype для бизнеса федерадных партнеров

Skype в запросе собрания календаря Skype для бизнеса для системы номеров. Ссылка на соединение обычно находится в теле запроса на собрание. Однако Skype система номеров зависит от этой ссылки, которая будет присутствовать в свойствах MAPI сообщения. Когда этот запрос на собрание отправляется удаленным организациям (Skype для бизнеса федератный партнер), по умолчанию система Skype номеров удаленной организации не будет показывать ссылку на соединение собрания в календаре. Фактически все пользователи Outlook удаленной организации не смогут присоединиться к собранию Skype для бизнеса одним щелчком мыши элемента календаря или из напоминания собрания. Они должны открыть приглашение на собрание и щелкнуть Skype для бизнеса join Skype для бизнеса в тексте сообщения. 
  
Причина этого ограничения в том, что Outlook microsoft Exchange не используют специальный метод для упаковки сведений для отправки сообщений через Интернет. Этот метод, именуемый форматом транспортной нейтральной инкапсуляции (TNEF), отключен по умолчанию для сообщений, отправленных извне из Exchange организации. Чтобы присоединиться к собранию в удаленной системе Skype номеров, организация отправки должна включить TNEF с помощью следующей команды:
  
```powershell
New-RemoteDomain -DomainName Contoso.com -Name Contoso
Set-RemoteDomain -Identity Contoso -TNEFEnabled $true
```

После включения TNEF для удаленной организации любое сообщение, отправленное через Интернет в организацию, отправляется в виде вложения в формате TNEF. С включенной поддержкой TNEF, когда Skype для бизнеса собрания отправляется федератированному партнеру Skype для бизнеса, Skype Система номеров сможет отрисовки собрания Skype для бизнеса, а удаленные пользователи смогут присоединиться к Skype для бизнеса собрания.
