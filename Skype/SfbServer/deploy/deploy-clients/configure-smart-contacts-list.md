# <a name="configure-smart-contacts-list-in-skype-for-business-clients"></a>Настройка списка смарт-контактов в клиентах Skype для бизнеса

**Сводка:** Сведения о том, как включить функцию списка смарт-контактов в клиенте Skype для бизнеса.

Функция "список смарт-контактов" позволяет выполнять автоматическое заполнение списков контактов для конечных пользователей. При первом использовании Skype для бизнеса пользователи будут автоматически видеть своего руководителя и других участников группы. Эта функция включена по умолчанию для пользователей Microsoft 365 и Office 365, но необходимо явно включить эту функцию для локальных пользователей, настроив параметр политики клиента.

При настройке этой функции учитывайте следующее:

- Пользователи, до 13, автоматически добавляются в список смарт-контактов в следующем порядке:

  1. Директор

  2. Направление в алфавитном порядке

  3. Одноранговые узлы в алфавитном порядке

- При первом входе пользователя создается новая группа с именем My Group. В группу автоматически заполняются пользователи из отношения групп Active Directory пользователя, основанного на псевдониме пользователя, заполненном в поле "руководитель". Обратите внимание, что изменения, внесенные в членство в группе Active Directory, не приводят к обновлению группы после ее первоначального заполнения. Если пользователь удаляет контакт или группу, ни сам контакт, ни группа не будут созданы повторно. 

- Если включена автоматическая разметка, контакты в списке будут отмечены для изменения сведений о присутствии. Автоматическая маркировка включена по умолчанию, но ее можно отключить. 

- Все новые пользователи в группе получат уведомление о том, что они добавлены в список контактов. Пользователи могут вручную добавлять новых участников в свою группу "Мои группы" или другие группы по своему выбору.

- Эта функция работает только для пользователей, выполнивших вход в первый раз. Если пользователь ранее выполнил вход с любого устройства, включая, например, любое мобильное устройство или Mac--, эта функция не включена для этого пользователя.

## <a name="configure-smart-contacts-list"></a>Настройка списка смарт-контактов

Чтобы включить функцию списка смарт-контактов для пользователей, необходимо выполнить следующие действия: 

- Создайте новую запись CsClientPolicy и добавьте ее в глобальную политику клиента. 

- Убедитесь, что поиск в адресной книге настроен только для поиска в Интернете.

### <a name="create-a-policy-entry-to-enable-smart-contacts-list"></a>Создание записи политики для включения интеллектуального списка контактов

Чтобы создать запись политики для включения функции списка смарт-контактов, используйте командлет [New-CsClientPolicyEntry](https://docs.microsoft.com/powershell/module/skype/new-csclientpolicyentry?view=skype-ps) с параметром EnableClientAutoPopulateWithTeam, как показано ниже.

```powershell
$x=New-CsClientPolicyEntry -Name EnableClientAutoPopulateWithTeam -Value $True
```

Затем используйте командлет [Set – CsClientPolicy](https://docs.microsoft.com/powershell/module/skype/set-csclientpolicy?view=skype-ps) для записи изменений в глобальную политику следующим образом:

```powershell
Set-CsClientPolicy -Identity Global -PolicyEntry @{Add=$x}
```

При необходимости вы можете создать политику, чтобы отключить автоматическую разметку следующим образом:

```powershell
$x=New-CsClientPolicyEntry -Name TagContactsInClientAutoPopulatedGroup -Value $False
Set-CsClientPolicy -Identity Global -PolicyEntry @{Add=$x}
```

Кроме того, необходимо задать для соответствующей политики параметр Аддрессбукаваилабилити в Вебсеарчонли. Дополнительные сведения см. в статье [Set – CsClientPolicy](https://docs.microsoft.com/powershell/module/skype/set-csclientpolicy?view=skype-ps). 

### <a name="troubleshoot"></a>Устранение неполадок

Если список смарт-контактов работает не так, как ожидалось, проверьте следующее:

- Проверка конфигурации. 

- Убедитесь, что заполнены сведения об организации AD.

- Сбор журналов клиентов Skype для бизнеса на новом пользователе для дальнейшего анализа.

- Убедитесь, что в пользовательском интерфейсе клиента Skype для бизнеса не отображается сообщение о том, что оно не может подключиться к адресной книге. Чтобы подтвердить подключение к адресной книге, выполните поиск пользователя на панели поиска клиента Skype для бизнеса.

- Проблемы с репликацией AD DS могут привести к неразрешимым контактам при первом входе пользователя в Skype для бизнеса.
