# <a name="install-and-test-skype-for-business-for-windows-phone"></a>Установка и тестирование Skype для бизнеса для Windows Phone
 
**Сводка:** Узнайте, как установить и протестировать Skype для бизнеса на windows Phone.
  
Приложение Skype для бизнеса для Windows Phone обеспечивает наличие Skype для бизнеса, обмен мгновенными сообщениями, а также голосовые и видеозв вызовы на мобильные устройства с Windows. Пользователи Lync 2013 получат обновленное приложение автоматически или получат запрос на обновление вручную в зависимости от параметров пользователя. Новые пользователи могут скачать его из [Магазина Windows Phone Marketplace.](https://go.microsoft.com/fwlink/p/?linkid=231901) Приложение Skype для бизнеса для Windows Phone доступно только в Windows Phone 8.1 и более поздних версиях.
  
Прежде чем направлять пользователей в организации на скачивание приложения, необходимо выполнить следующие тесты, чтобы убедиться, что оно правильно интегрировано в среду. 
  
## <a name="install-skype-for-business-windows-phone-81"></a>Установка Skype для бизнеса Windows Phone 8.1

1. Перейдите в [центр обновления Windows Phone 8,](https://www.windowsphone.com/en-us/how-to/wp8/update-central) чтобы обновить телефон до Windows Phone 8.1.
    
2. На телефоне перейдите в **Магазин и** найдите Skype **для бизнеса.**
    
3. **Коснитесь "Установить".** 
    
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Первый вход в Skype для бизнеса

1. На экране **"Начните"** проведите пальцем влево, чтобы просмотреть установленные приложения, найди Skype для бизнеса для Windows Phone, а затем коснитесь значка, чтобы открыть приложение.
    
2. Введите свой адрес для входа (например, user@domain.com) и пароль, а затем нажмите кнопку **"Готово".**
    
     Вам может потребоваться указать как имя пользователя, так и адрес для входа. Имя пользователя используется для входов в сеть организации, user@domain.com домен\имя пользователя.
    
3. На экране **"Программа** улучшения качества по улучшению качества по работе с  клиентами" нажмите кнопку **"Присоединиться",** чтобы отправить анонимные данные о проблемах приложений и использовании в корпорацию Майкрософт, или нет, если вы не хотите участвовать в программе.
    
4. On the **Never Miss Your Work Calls** screen, enter your mobile number with country and region codes. Если Skype для бизнеса для Windows Phone не может использовать сеть передачи Wi-Fi передачи данных для аудио- или видеозвонков, вы будете автоматически вызваны по этому номеру и подключены к звуковой части вызова.
    
5. Нажмите **кнопку** "Далее" и просмотрите параметры доступа к уведомлению и телефонной книге:
    
   - **Push-уведомления** Получение оповещения при приеме нового im или вызова. Обычно **ветвь** (рекомендуется).
    
     > [!IMPORTANT]
     > Если вы отключите этот параметр, вы не будете уведомлены об IMs, звонках или других оповещениях Skype для бизнеса для Windows Phone, если приложение не является активным. 
  
   - **Разрешить доступ к телефонной книге** Поиск контактов на мобильном телефоне при поиске контактов в Skype для бизнеса для Windows Phone.
    
6. Нажмите **кнопку** "Далее", чтобы начать использовать Skype для бизнеса для Windows Phone.
    
    [Помощь по входу](https://support.office.com/article/6b827683-ad55-471a-bd4b-3d4ec098bf75)
    
## <a name="verify-mobile-client-installation"></a>Проверка установки мобильного клиента

После настройки клиента и успешного входе с помощью следующих тестов убедитесь, что установка Skype для бизнеса для Windows Phone работает правильно на мобильном устройстве.
  
### <a name="search-for-a-contact-in-the-corporate-directory"></a>Поиск контакта в корпоративном каталоге

1. В списке контактов коснитесь **"Поиск".**
    
2. Поиск контакта, который существует только в глобальном списке адресов.
    
3. Убедитесь, что имя контакта отображается в результатах поиска.
    
### <a name="test-instant-messaging-and-presence"></a>Тестирование обмена мгновенными сообщениями и присутствия

1. В списке контактов коснитесь контакта.
    
2. В карточке контакта коснитесь мгновенных сообщений ![Значок для обмена мгновенными сообщениями в Skype для бизнеса](../../media/90f8d5fa-7968-4ef7-bf5b-dddf9b893905.png).
    
3. Убедитесь, что появится окно для мгновенных мгновенных сообщенией и вы можете ввести и отправить сообщение.
    
### <a name="test-dial-out-conferencing"></a>Тестирование dial-out conferencing

1. В Outlook запланировать собрание Skype для бизнеса.
    
2. В Windows Phone откройте приглашение на собрание.
    
3. Щелкните ссылку на собрании, чтобы присоединиться.
    
4. Ответьте на звонок из службы конференций и убедитесь, что вы подключены к аудиосвязи собрания.
    
### <a name="test-push-notifications"></a>Тестирование push-уведомлений

1. Выберите две разные учетные записи пользователей для этого теста. 
    
2. На телефоне пользователя А во sign in to Skype for Business for Windows Phone with user A's account.
    
3. Откройте другое приложение на устройстве.
    
4. На другом клиенте, например на настольном компьютере, во sign in to Skype for Business with user B's account.
    
5. Отправьте мгновенные сообщение от пользователя Б пользователю А.
    
6. Убедитесь, что уведомление об мгновенных сообщениях отображается на мобильном устройстве пользователя А.
    
## <a name="remove-skype-for-business-from-your-windows-phone"></a>Удаление Skype для бизнеса из Windows Phone

Чтобы удалить приложение Skype для бизнеса для Windows Phone с мобильного устройства: 
  
1. С экрана начала проведите пальцем, чтобы увидеть список приложений. 
    
2. Коснитесь и удерживайте приложение Skype для бизнеса для Windows Phone, а затем выберите **"Удалить".**
