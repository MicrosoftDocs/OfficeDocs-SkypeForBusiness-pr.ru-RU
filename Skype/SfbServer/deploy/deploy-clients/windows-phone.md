# <a name="install-and-test-skype-for-business-for-windows-phone"></a>Install and test Skype for Business for Windows Phone
 
**Сводка:** Сведения о том, как установить и проверить Skype для бизнеса на устройстве с Windows Phone.
  
Приложение Skype для бизнеса для Windows Phone обеспечивает присутствие Skype для бизнеса, Обмен мгновенными сообщениями, голосовые и видеозвонки на устройства с Windows Mobile. Пользователи Lync 2013 в зависимости от назначенных им параметров получают обновленное приложение автоматически или обновляют его вручную при появлении соответствующего запроса. Новые пользователи могут загрузить ее с сайта [Windows Phone Marketplace](https://go.microsoft.com/fwlink/p/?linkid=231901). Приложение Skype для бизнеса для Windows Phone доступно только на Windows Phone 8,1 и более поздних версиях.
  
Прежде чем перенаправлять пользователей в вашей организации для загрузки приложения, необходимо выполнить следующие тесты, чтобы убедиться, что он правильно интегрирован в вашу среду. 
  
## <a name="install-skype-for-business-windows-phone-81"></a>Установка Skype для бизнеса на Windows Phone 8,1

1. Перейдите на [Центр обновления Windows Phone 8](https://www.windowsphone.com/en-us/how-to/wp8/update-central) , чтобы обновить свой телефон на windows Phone 8,1.
    
2. На телефоне перейдите в **магазин**и выполните поиск по запросу **Skype для бизнеса**.
    
3. Коснитесь кнопки **Установить**. 
    
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Первый вход в Skype для бизнеса

1. На **начальном** экране проведите пальцем влево, чтобы просмотреть установленные приложения, найдите Skype для бизнеса для Windows Phone, а затем коснитесь значка, чтобы открыть приложение.
    
2. Введите адрес для входа (например, user@domain.com) и пароль, а затем нажмите **Готово**.
    
     Может потребоваться ввести как имя пользователя, так и адрес для входа. Имя пользователя — это то, что вы используете для входа в сеть вашей организации: user@domain.com или домен \ имя_пользователя.
    
3. На экране **Программа улучшения качества программного обеспечения** коснитесь кнопки **Присоединиться** для передачи анонимных данных о неполадках приложений и их использовании в компанию Майкрософт или **Нет, спасибо** для отказа от участия в этой программе.
    
4. На экране **предотвращения пропуска рабочих вызовов** введите номер мобильного телефона с кодами страны и региона. Если Skype для бизнеса для Windows Phone не может использовать Wi-Fi или сеть сотовой связи для совершения голосовой и видеосвязи, вы будете автоматически звонить по этому номеру и подключаться к звуковой части звонка.
    
5. Коснитесь **значка Далее** и ознакомьтесь с настройками доступа к уведомлениям и телефонной книге:
    
   - **Push-уведомления** Получать оповещение при получении нового сообщения или звонка. Рекомендуется оставить стандартное значение **Вкл.**
    
     > [!IMPORTANT]
     > Если вы отключите этот параметр, вы не будете получать уведомления о мгновенных сообщениях, звонках и других возможностях Skype для бизнеса для Windows Phone, пока приложение не будет активно. 
  
   - **Разрешить доступ к телефонной книге** Поиск контактов на мобильном телефоне при поиске контактов в Skype для бизнеса для Windows Phone.
    
6. Нажмите кнопку **Далее** , чтобы начать пользоваться Skype для бизнеса для Windows Phone.
    
    [Нужна помощь по входу?](https://support.office.com/article/6b827683-ad55-471a-bd4b-3d4ec098bf75)
    
## <a name="verify-mobile-client-installation"></a>Проверка установки мобильного клиента

После того как вы настроили клиент и войдете в свою учетную запись, выполните следующие проверки, чтобы убедиться, что установка Skype для бизнеса для Windows Phone работает правильно на вашем мобильном устройстве.
  
### <a name="search-for-a-contact-in-the-corporate-directory"></a>Поиск контакта в корпоративном каталоге

1. В списке контактов коснитесь кнопки **поиска**.
    
2. Выполните поиск контакта, который существует только в глобальном списке адресов.
    
3. Убедитесь, что имя контакта отображается в результатах поиска.
    
### <a name="test-instant-messaging-and-presence"></a>Проверка функций обмена мгновенными сообщениями и сведений о присутствии

1. В списке контактов выберите контакт.
    
2. В карточке контакта выберите обмен мгновенными сообщениями (IM) ![Значок мгновенных сообщений в Skype для бизнеса](../../media/90f8d5fa-7968-4ef7-bf5b-dddf9b893905.png)ярлык.
    
3. Убедитесь в том, что открывается окно обмена мгновенными сообщениями, где можно ввести и отправить сообщение.
    
### <a name="test-dial-out-conferencing"></a>Проверка конференц-связи с телефонным подключением

1. В Outlook Запланируйте собрание Skype для бизнеса.
    
2. На устройстве Windows Phone откройте приглашение на собрание
    
3. Щелкните ссылку в собрании, чтобы присоединиться к нему.
    
4. Ответьте на звонок службы конференций и убедитесь, что вы можете прослушивать звук.
    
### <a name="test-push-notifications"></a>Проверка push-уведомлений

1. Выберите для этого испытания учетные записи двух разных пользователей. 
    
2. На устройстве A Windows Phone Войдите в Skype для бизнеса для Windows Phone с учетной записью пользователя A.
    
3. Откройте на устройстве другое приложение.
    
4. На другом клиенте, например в классическом клиенте, войдите в Skype для бизнеса с учетной записью пользователя B.
    
5. Отправьте мгновенное сообщение от пользователя Б пользователю А.
    
6. Убедитесь, что уведомление о мгновенных сообщениях появляется на мобильном устройстве "на пользователя".
    
## <a name="remove-skype-for-business-from-your-windows-phone"></a>Удаление Skype для бизнеса с телефона Windows Phone

Чтобы удалить приложение Skype для бизнеса для Windows Phone с мобильного устройства, выполните указанные ниже действия. 
  
1. Проведите пальцем по начальному экрану для просмотра списка приложений. 
    
2. Коснитесь приложения Skype для бизнеса для Windows Phone и удерживайте его, а затем выберите **Удалить**.
