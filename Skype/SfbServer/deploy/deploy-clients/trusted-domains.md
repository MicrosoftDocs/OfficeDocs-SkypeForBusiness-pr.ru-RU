# <a name="skype-room-system-trusted-domains"></a>Доверенные домены системы комнат Skype
 
Этот раздел содержит инструкции по настройке доверенных доменов для системы комнат Skype и Skype для бизнеса.
  
## <a name="trusted-domains"></a>Доверенные домены

Скайп для клиента Business отображает диалоговое окно для принятия сертификата из Скайп для Business Server, если домен SIP учетной записи пользователя для подписи отличается от имени, представленных в теме или Alt имя субъекта сертификата. Если сертификат, настроенный для Скайп для Business Server в организации отсутствует имя домена SIP Скайп комнаты системной учетной записи в теме или Alt имя субъекта, необходимо настроить эти домены, представленные на сертификат в разделе доверенные домены раздел реестра на компьютере консоли Скайп комнаты системы. Руководство по системе комнаты Скайп предоставленного производителя Скайп комнаты системного администратора объясняется, как и где Добавление доверенных доменов в Скайп для клиента Business. 
  
Например предположим, что сертификаты, настроенные на Скайп для Business Server Name задано значение темы или темы Alt «CONTOSO. ЛОКАЛЬНЫЙ» и один из доменов SIP, назначенных пользователю для системы комнаты Скайп адрес для входа является «confrm1@contoso.net». Так как contoso.net не в сертификат на компьютере Скайп комнаты системы, необходимо будет Настройка «contoso.local» в качестве доверенного домена в реестр, как описано в руководстве по системе комнаты Скайп предоставленного производителя Скайп комнаты системного администратора.
