# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>Установка SQL Server Reporting Services в Скайп для Business Server 
 
**Сводка:** Узнайте, где можно найти сведения о службах отчетов SQL Server используется Скайп для Business Server проверить.
  
Скайп для Business Server можно использовать службы SQL Server Reporting Services (SSRS) для просмотра и отчетов мониторинга. Чтобы использовать эту функциональную возможность, необходимо установить и настроить службы отчетов.
  
## <a name="install-sql-server-reporting-services"></a>Установка служб отчетов SQL Server

Если вы намерены использовать Скайп для отчетов мониторинга Business Server (в статье далее в этой документации для получения дополнительных сведений) необходимо сначала установить службы отчетов SQL Server; Службы отчетов можно установить в то же время, установка Microsoft SQL Server или в любое время после установки SQL Server. Если вы не установили SQL Server, затем следуйте инструкциям ранее в этой документации. При установке SQL Server, убедитесь в том, что на странице "Выбор компонентов" установите служб Reporting Services. Предназначенным для установки SQL Server Reporting Services.
  
Чтобы узнать, как для установки служб отчетов SQL Server, обратитесь к разделу [Службы SQL Server Reporting Services (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx).
  
Если вы уже установили SQL Server, но не установили службы отчетов SQL Server, этот компонент можно добавить, используя при необходимости соответствующий набор инструкций для служб отчетов SQL Server.
