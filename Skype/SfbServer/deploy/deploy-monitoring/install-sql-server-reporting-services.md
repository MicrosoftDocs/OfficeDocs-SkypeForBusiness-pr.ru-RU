# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>Установка служб SQL Server Reporting Services в Skype для бизнеса Server 
 
**Сводка:** Сведения о том, где найти сведения о службах SQL Server Reporting Services, используемых в Skype для бизнеса Server.
  
Skype для бизнеса Server может использовать службы SQL Server Reporting Services (SSRS) для просмотра и наблюдения за отчетами. Чтобы использовать эту функциональную возможность, необходимо установить и настроить службы отчетов.
  
## <a name="install-sql-server-reporting-services"></a>Установка служб отчетов SQL Server

Если вы планируете использовать отчеты наблюдения в Skype для бизнеса Server (для получения дополнительных сведений ознакомьтесь со следующей статьей в этой документации), необходимо сначала установить службы SQL Server Reporting Services; Службы Reporting Services могут быть установлены одновременно с установкой Microsoft SQL Server или в любое время после установки SQL Server. Если вы не установили SQL Server, следуйте инструкциям, приведенным выше в этой документации. При установке SQL Server убедитесь, что на странице выбора функций вы выбрали пункт службы отчетов. Будут установлены службы SQL Server Reporting Services.
  
Чтобы узнать, как установить службы отчетов SQL Server, ознакомьтесь со сведениями о [службах SQL Server Reporting Services (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx).
  
Если вы уже установили SQL Server, но не установили службы отчетов SQL Server, этот компонент можно добавить, используя при необходимости соответствующий набор инструкций для служб отчетов SQL Server.
