# <a name="configure-archiving-policies-for-skype-for-business-server"></a>Настройка политик архива для Skype для бизнеса Server
 
**Сводка:** Ознакомьтесь с этой темой, чтобы узнать, как настроить начальные политики архива для Skype для бизнеса Server пользователей.
  
В Skype для бизнеса Server вы используете политики для обеспечения и отключения архива внутренних коммуникаций и внешних коммуникаций для пользователей, которые находятся в Skype для бизнеса Server. В том числе:
  
- Глобальная политика, созданная по умолчанию при развертывании Skype для бизнеса Server
    
- Необязательные политики уровня сайта, которые указывают, как осуществляется архивативная реализация для определенного сайта
    
- Необязательные политики на уровне пользователей, определяющие, как осуществляется архивативная политика для определенных пользователей
    
Сначала вы настроить политики архивации при развертывании архивации, но после развертывания можно изменить, добавить и удалить политики. В Skype для бизнеса Server панели управления вы можете  использовать страницу политики  архивания группы архива и мониторинга для управления политиками на глобальном уровне, на сайте и уровне пользователей.
  
> [!NOTE]
> Чтобы контролировать реализацию архивации, необходимо указать параметры, например, архивировать im или conferencing, использовать критический режим и параметры чистки. По умолчанию параметры не включены в глобальной конфигурации архивации или конфигурации архивации сайта или пула. Перед включением архива внутренних или внешних коммуникаций необходимо указать все соответствующие параметры. Подробнее см. в [материале Configure archiving options for Skype для бизнеса Server.](configure-archiving-options.md) 
  
> [!NOTE]
> Если вы включаете интеграцию microsoft Exchange для развертывания, Exchange In-Place политики hold контролируют, включена ли архивация для пользователей, которые размещены в Exchange и наложены ли их почтовые ящики In-Place Hold. 
  
Сведения о работе политик архива, включая иерархию глобальных, веб-сайтов и политик пользователей, см. в материале [Plan for archiving in Skype для бизнеса Server.](../../plan-your-deployment/archiving/archiving.md) Сведения о том, как управлять политиками после развертывания, см. в материале Управление политиками архивации [в Skype для бизнеса Server.](../../manage/archiving/policies.md)
  
## <a name="global-policy"></a>Глобальная политика

При развертывании серверов переднего Skype для бизнеса Server создает глобальную политику архива. По умолчанию архивировка отключена в глобальной политике. Глобальная политика контролирует, включена ли архивация для внутренних и внешних коммуникаций для всего развертывания, если не настроены политики сайта или пользователей, которые переопределяют глобальную политику, или если вы используете интеграцию Microsoft Exchange для некоторых или всех пользователей. Если вы используете интеграцию microsoft Exchange, глобальная политика не применяется к пользователям, которые находятся в Exchange и имеют почтовые ящики, In-Place Hold.
  
### <a name="configure-the-global-policy-for-archiving-for-skype-for-business-server-archiving-databases"></a>Настройка глобальной политики архива для Skype для бизнеса Server баз данных архива

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
    
3. В левой панели навигации щелкните **Мониторинг** и архивации, а затем нажмите кнопку **Политика архивации**.
    
4. На странице **Archiving Policy** (Политика архивации) последовательно щелкните **Global** (Глобальная), **Edit** (Изменить) и **Show details** (Показать сведения).
    
5. На странице **Edit Archiving Policy - Global** (Изменение политики архивации — глобальный уровень) выполните следующие действия.
    
   - Если вы не хотите использовать имя Global по умолчанию, укажите в поле **Имя** новое имя глобальной политики. 
    
   - В **описании** уделяем сведения о том, что такое политика (например, глобальная политика *для divisionName.*
    
   - Для управления архивацией внутренних коммуникаций для всех сайтов и пользователей, которые не управляются отдельными политиками на уровне сайта или пользователя, снимите или установите флажок **Archive internal communications** (Архивировать внутренние коммуникации).
    
   - Для управления архивацией внешних коммуникаций для всех сайтов и пользователей, которые не управляются отдельными политиками на уровне сайта или пользователя, снимите или установите флажок **Archive external communications** (Архивировать внешние коммуникации).
    
6. Нажмите кнопку **Зафиксировать**.
    
## <a name="site-policies"></a>Политики сайта

Вы можете включить или отключить архивировать для определенных сайтов, создав политику архива для каждого из этих сайтов. Политика сайта переопределяет глобальную политику, а политики пользователей — политики сайта. Политики архивации применяются только в том случае, если вы не используете интеграцию Microsoft Exchange или, если вы используете интеграцию Microsoft Exchange, но у вас есть пользователи, которые не находятся в Exchange и их почтовые ящики In-Place Hold.
  
### <a name="create-an-archiving-policy-for-a-site"></a>Создание политики архива для сайта

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
    
3. В левой панели навигации щелкните **Мониторинг** и архивации, а затем нажмите кнопку **Политика архивации**.
    
    Сведения о работе политик архива, включая иерархию глобальных, веб-сайтов и политик пользователей, см. в материале [Plan for archiving in Skype для бизнеса Server.](../../plan-your-deployment/archiving/archiving.md)
    
4. Нажмите кнопку **Создать** и выберите **Политика сайта**.
    
5. В поле **Выбор сайта** щелкните сайт, к которому требуется применить политику.
    
6. В области **Создать политику архивации** выполните следующие действия.
    
   - В поле **Имя** укажите имя для политики сайта. 
    
   - В поле **Описание** укажите сведения о назначении политики сайта (например, "политика сайта для Москвы").
    
   - Чтобы выбрать управление архивацией внутренних коммуникаций для указанного сайта, установите или снимите флажок **Архивация внутренних коммуникаций**.
    
   - Чтобы выбрать управление архивацией внешних коммуникаций для указанного сайта, установите или снимите флажок **Архивация внешних коммуникаций**.
    
7. Нажмите кнопку **Зафиксировать**.
    
## <a name="user-policies"></a>Политики пользователей

Вы можете включить или отключить архивировать для определенных пользователей, создав и настроив политику архива для пользователей, а затем применяя ее к определенным пользователям или группам пользователей. Политики пользователей переопределяют любую глобальную политику и политики сайтов. Политики архивации применяются только в том случае, если вы не используете интеграцию Microsoft Exchange или, если вы используете интеграцию Microsoft Exchange, но у вас есть пользователи, которые не находятся в Exchange и их почтовые ящики In-Place Hold.
  
### <a name="configure-an-archiving-policy-for-users-homed-on-skype-for-business-server"></a>Настройка политики архива для пользователей, которые будут Skype для бизнеса Server

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
    
3. В левой панели навигации щелкните **Мониторинг** и архивации, а затем нажмите кнопку **Политика архивации**.
    
4. Щелкните элемент **New** (Создать) и **User policy** (Политика пользователей).
    
5. В окне **New Archiving Policy** (Новая политика архивации) выполните следующие действия:
    
   - В поле **Name** (Имя) укажите имя политики пользователей. 
    
   - В поле **Description** (Описание) укажите сведения о том, что из себя представляет данная политика пользователей (например, политика пользователей для юридического отдела).
    
   - Чтобы контролировать архивацию внутренних коммуникаций в рамках политики пользователей, установите или снимите флажок **Archive internal communications** (Архивировать внутренние коммуникации).
    
   - Чтобы контролировать архивацию внешних коммуникаций в рамках политики пользователей, установите или снимите флажок **Archive external communications** (Архивировать внешние коммуникации).
    
6. Щелкните элемент **Commit** (Исполнить).
    
Политика пользователей применяется только к тем пользователям, которым вы ее назначили.
### <a name="apply-a-skype-for-business-server-archiving-policy-to-a-user"></a>Применение политики Skype для бизнеса Server архива к пользователю

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
    
3. В левой панели навигации щелкните **Пользователи**, а затем выполните поиск учетной записи пользователя, которую требуется настроить.
    
4. В таблице со списком результатов поиска щелкните учетную запись пользователя, нажмите кнопку **Изменить**, а затем щелкните **Показать подробности**.
    
5. В **статье Изменение Skype для бизнеса Server пользователя** в соответствии с политикой архивного архива выберите политику архива, которую необходимо применить. 
    
    > [!NOTE]
    > Параметры применяют параметры установки **\<Automatic\>** сервера по умолчанию. Данный параметр автоматически применяется сервером.
  
6. Нажмите кнопку **Сохранить**.
