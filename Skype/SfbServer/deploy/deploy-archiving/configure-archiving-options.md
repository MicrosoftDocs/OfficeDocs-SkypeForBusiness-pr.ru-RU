# <a name="configure-archiving-options-for-skype-for-business-server-2015"></a>Настройка параметров архивации в Skype для бизнеса Server 2015
 
**Сводка:** Прочтите этот раздел, чтобы узнать, как Настройка начальной параметров архивации для Скайп для Business Server 2015. Изначально Настройка конфигураций архивации при развертывании архивации, но можно изменять, добавлять и удалять конфигурации после развертывания.
  
Настройка начальной архивации конфигурации, используйте Скайп для панели управления Business Server для укажите следующие параметры:
  
- Настройки глобального уровня, которая создается по умолчанию при развертывании Скайп для Business Server
    
- необязательные конфигурации на уровне сайтов, которые указывают, как архивация реализована для конкретных сайтов;
    
- Дополнительные конфигурации на уровне пула, которые определяют, как архивация реализуется для конкретного пула
    
При настройке конфигурации необходимо ответить на следующие вопросы:
  
- включить или отключить архивацию;
    
- архивировать ли сеансы обмена мгновенными сообщениями;
    
- архивировать ли сеансы веб-конференций;
    
- блокировать ли действия при невозможности архивации;
    
- пользоваться ли взаимодействием с Exchange;
    
- как настроить удаление и экспорт данных.
    
> [!NOTE]
> Все необходимые параметры следует задать до включения архивации. 
  
Для получения дополнительных сведений о том, как реализован конфигураций архивации, в том числе которых можно указать параметры и иерархии архивации конфигурации, ознакомьтесь со статьей [Plan для архивации в Скайп для Business Server 2015](../../plan-your-deployment/archiving/archiving.md). Для получения дополнительных сведений об управлении конфигураций после развертывания с помощью панели управления или с помощью Windows PowerShell обнаружить [Управление архивации в Скайп для Business Server 2015](../../manage/archiving/options.md).
  
## <a name="configure-global-level-archiving-options"></a>Настройка параметров архивации на глобальном уровне

При добавлении архивации в топологию и опубликовать эту топологию, Скайп для Business Server создается Глобальная конфигурация архивации. По умолчанию в глобальной конфигурации все параметры архивации отключены. Глобальная конфигурация определяет параметры, включенные для развертывания в целом, если не заданы конфигурации на уровне сайта или пула, переопределяющие глобальную конфигурацию.
  
Для настройки параметров архивации на глобальном уровне выполните следующие действия.
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
    
3. На левой панели навигации щелкните **Мониторинг и архивация**, а затем щелкните **Конфигурация архивации**.
    
4. На странице **Конфигурация архивации** щелкните **Глобальная** и **Изменить**, затем **Показать сведения**.
    
5. В разделе **Изменение настройки архивации — глобальная** выберите в раскрывающемся списке **Настройки архивации** один из следующих параметров архивации.
    
   - **Отключить архивацию**
    
   - **Архивировать сеансы мгновенных сообщений**
    
   - **Архивировать сеансы обмена мгновенными сообщениями и веб-конференций**
    
6. На странице **Изменение параметров архивации — глобальный** выполните следующее:
    
   - Чтобы заблокировать действия, когда архивирование невозможно, установите флажок **При сбое архивации заблокировать обмен мгновенными сообщениями или сеансы веб-конференции**.
    
   - Чтобы использовать Microsoft Exchange Server для хранения архивных данных, установите флажок **Интеграция с Microsoft Exchange** .
    
   - Чтобы включить удаление данных, установите флажок **Разрешить удаление данных архивации**, а затем выполните одно из следующих действий:
    
   - Чтобы задать удаление после определенного числа дней, нажмите **Очищать экспортированные данные архивации, а также данные архивации по окончании максимального срока хранения (дн.)**, а затем задайте число дней.
    
   - Чтобы ограничить удаление только теми данными, которые были экспортированы, нажмите **Удалять только экспортированные данные архивации**.
    
7. Щелкните **Исполнить**.
    
## <a name="configure-site-level-archiving-options"></a>Настройка параметров архивации на уровне сайта

Параметры архивации можно задать для отдельного сайта. Конфигурация сайта переопределяет глобальную конфигурацию, но только для сайта, указанного в конфигурации сайта. 
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
    
3. На левой панели навигации щелкните **Мониторинг и архивация**, а затем щелкните **Конфигурация архивации**.
    
4. На странице **Конфигурация архивации** нажмите кнопку **Создать**, затем выберите **Конфигурация сайта**.
    
5. В разделе **Выбор службы** выберите сайт для настройки архивации.
    
6. В разделе **Создание новой настройки архивации** выберите в раскрывающемся списке **Настройки архивации** один из следующих вариантов.
    
   - Если требуется разрешить архивацию только для сеансов обмена мгновенными сообщениями, щелкните **Архивировать сеансы мгновенных сообщений**.
    
   - Если требуется разрешить архивацию как для сеансов обмена мгновенными сообщениями, как и для конференц-связи, щелкните **Архивировать сеансы мгновенных сообщений и веб-конференций**.
    
   - Если требуется запретить архивацию для данной политики, щелкните **Отключить архивацию**.
    
7. На странице **Создание новой настройки архивации** можно также выполнить следующие действия.
    
   - Чтобы заблокировать действия, когда архивирование невозможно, установите флажок **При сбое архивации заблокировать обмен мгновенными сообщениями или сеансы веб-конференции**.
    
   - Чтобы использовать Microsoft Exchange Server для хранения архивных данных, установите флажок **Интеграция с Microsoft Exchange** .
    
   - Чтобы включить удаление данных, установите флажок **Разрешить удаление данных архивации**, а затем выполните одно из следующих действий:
    
   - Чтобы задать удаление после определенного числа дней, нажмите **Очищать экспортированные данные архивации, а также данные архивации по окончании максимального срока хранения (дн.)**, а затем задайте число дней.
    
   - Чтобы ограничить удаление только теми данными, которые были экспортированы, нажмите **Удалять только экспортированные данные архивации**.
    
8. Щелкните **Исполнить**.
