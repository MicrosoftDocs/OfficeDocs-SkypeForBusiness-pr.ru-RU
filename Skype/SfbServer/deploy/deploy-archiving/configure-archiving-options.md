# <a name="configure-archiving-options-for-skype-for-business-server"></a>Настройка параметров архивации в Skype для бизнеса Server
 
**Сводка:** В этой статье рассказывается о том, как настроить начальные параметры архивации в Skype для бизнеса Server. Первоначально конфигурации архивации задаются при развертывании архивации, но вы можете изменить, добавить и удалить настройки после развертывания.
  
Чтобы настроить начальные конфигурации архивации, используйте панель управления Skype для бизнеса Server, чтобы указать указанные ниже параметры.
  
- Конфигурация глобального уровня, созданная по умолчанию при развертывании Skype для бизнеса Server
    
- необязательные конфигурации уровня сайта, определяющие реализацию архивации на отдельных сайтах;
    
- Необязательные конфигурации уровня пула, определяющие способ реализации архивации для определенного пула
    
При настройке конфигурации необходимо ответить на следующие вопросы:
  
- включить или отключить архивацию;
    
- архивировать ли сеансы обмена мгновенными сообщениями;
    
- архивировать ли сеансы веб-конференций;
    
- блокировать ли действия при невозможности архивации;
    
- пользоваться ли взаимодействием с Exchange;
    
- как настроить удаление и экспорт данных.
    
> [!NOTE]
> Все необходимые параметры следует задать до включения архивации. 
  
Сведения о том, как реализуются конфигурации архивации, в том числе о параметрах, которые можно указать и в иерархии конфигураций архивации, приведены [в разделе Планирование архивации в Skype для бизнеса Server](../../plan-your-deployment/archiving/archiving.md). Дополнительные сведения об управлении конфигурациями после развертывания с помощью панели управления или с помощью Windows PowerShell можно найти в разделе [Управление параметрами архивации в Skype для бизнеса Server](../../manage/archiving/options.md).
  
## <a name="configure-global-level-archiving-options"></a>Настройка параметров архивации на глобальном уровне

При добавлении архивации в топологию и публикации топологии в Skype для бизнеса Server создается Глобальная конфигурация для архивирования. По умолчанию в глобальной конфигурации все параметры архивации отключены. Глобальная конфигурация определяет параметры, включенные для развертывания в целом, если не заданы конфигурации на уровне сайта или пула, переопределяющие глобальную конфигурацию.
  
Для настройки параметров архивации на глобальном уровне выполните следующие действия.
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
    
3. На левой панели навигации щелкните **Мониторинг и архивация**, а затем щелкните **Конфигурация архивации**.
    
4. На странице **Конфигурация архивации** щелкните **Глобальная** и **Изменить**, затем **Показать сведения**.
    
5. В разделе **Изменение настройки архивации — глобальная** выберите в раскрывающемся списке **Настройки архивации** один из следующих параметров архивации.
    
   - **Отключить архивацию**
    
   - **Архивировать сеансы мгновенных сообщений**
    
   - **Архивировать сеансы мгновенных сообщений и веб-конференций**
    
6. Кроме того, на странице **изменение параметров архивирования — Глобальная** страница выполните указанные ниже действия.
    
   - Для запрета действия при невозможности архивации установите флажок **При сбое архивации заблокировать обмен мгновенными сообщениями или сеансы веб-конференции**.
    
   - Чтобы использовать сервер Microsoft Exchange для хранения данных для архивации, установите флажок **интеграция Microsoft Exchange** .
    
   - Для включения функции удаления данных установите флажок **Разрешить удаление данных архивации**, затем выполните одно из следующих действий.
    
   - Для настройки удаления записей по истечении заданного срока (в днях) щелкните **Очищать экспортированные данные архивации, а также данные архивации по окончании максимального срока хранения (дн.)**, затем укажите количество дней.
    
   - Для удаления только экспортированных данных выберите **Удалять только экспортированные данные архивации**.
    
7. Нажмите **Исполнить**.
    
## <a name="configure-site-level-archiving-options"></a>Настройка параметров архивации на уровне сайта

Параметры архивации можно задать для отдельного сайта. Конфигурация сайта переопределяет глобальную конфигурацию, но только для сайта, указанного в конфигурации сайта. 
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator.
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
    
3. На левой панели навигации щелкните **Мониторинг и архивация**, а затем щелкните **Конфигурация архивации**.
    
4. На странице **Конфигурация архивации** нажмите кнопку **Создать**, затем выберите **Конфигурация сайта**.
    
5. В разделе **Выбор службы** выберите сайт для настройки архивации.
    
6. В разделе **Создание новой настройки архивации** выберите в раскрывающемся списке **Настройки архивации** один из следующих вариантов.
    
   - Если требуется разрешить архивацию только для сеансов обмена мгновенными сообщениями, щелкните **Архивировать сеансы мгновенных сообщений**.
    
   - Если требуется разрешить архивацию как для сеансов обмена мгновенными сообщениями, как и для конференц-связи, щелкните **Архивировать сеансы мгновенных сообщений и веб-конференций**.
    
   - Если требуется запретить архивацию для данной политики, щелкните **Отключить архивацию**.
    
7. На странице **Создание новой настройки архивации** можно также выполнить следующие действия.
    
   - Чтобы заблокировать действия, когда архивирование невозможно, установите флажок **При сбое архивации заблокировать обмен мгновенными сообщениями или сеансы веб-конференции**.
    
   - Чтобы использовать сервер Microsoft Exchange для хранения данных для архивации, установите флажок **интеграция Microsoft Exchange** .
    
   - Для включения функции удаления данных установите флажок **Разрешить удаление данных архивации**, затем выполните одно из следующих действий.
    
   - Для настройки удаления записей по истечении заданного срока (в днях) щелкните **Очищать экспортированные данные архивации, а также данные архивации по окончании максимального срока хранения (дн.)**, затем укажите количество дней.
    
   - Для удаления только экспортированных данных выберите **Удалять только экспортированные данные архивации**.
    
8. Щелкните **Исполнить**.
