# <a name="skype-for-business-server-capacity-planning-calculator"></a>Калькулятор планирования мощности Skype для бизнеса Server
 
**Сводка:** Использование калькулятора емкости.

> [!NOTE]
> Эта статья ссылается на загрузки Skype для бизнеса Server 2015, но применима к:
> - Skype для бизнеса Server 2019.
> - Skype для бизнеса Server 2015.
  
The [Skype for Business Server 2015 Capacity Calculator](https://www.microsoft.com/download/details.aspx?id=51196) and Skype for Business Server [2019 Capacity Calculator](https://www.microsoft.com/download/details.aspx?id=57509) augment the Skype for Business Planning [Tool](https://www.microsoft.com/download/details.aspx?id=50357) and your deployment documentation ( Plan for your Skype for Business[Server 2015 deployment](../plan-your-deployment/plan-your-deployment.md) and Plan for your Skype for Business Server [2019 deployment](../../SfBServer2019/plan/plan-your-deployment-2019.md) respectively). Используйте калькулятор после просмотра руководства и создания рекомендуемой топологии с помощью средства планирования.
  
Калькулятор емкости Skype для бизнеса Server помогает определить требования к серверу на основе количества пользователей и средств связи, которые использует ваша организация. После определения профиля пользователя и функций, которые необходимо включить для пользователей, используйте калькулятор, чтобы определить необходимое количество серверов, памяти и пропускной способности. В этой версии калькулятора не указаны требования к диску для работы с дисками.
  
Вы можете воспользоваться преимуществами калькулятора, если у вас есть точные и подробные сведения о вашем профиле пользователя. Например, процент пользователей с включенной поддержкой голосовой почты, среднее число звонков на пользователя в час, продолжительность вызова и процент одновременно включенных пользователей в конференциях могут внести огромное различие в требованиях к серверу. Точность рекомендаций, созданных калькулятором, зависит от точности предоставляемой вами информации.
  
После использования средства планирования и калькулятора планирования мощности следует смоделировать предложенную и запланированную нагрузку, чтобы обеспечить надлежащую подготовку Skype для бизнеса Server. Для выполнения нагрузок при имитированной нагрузке используйте средство Stress and Performance Skype для бизнеса [Server,](https://www.microsoft.com/download/details.aspx?id=50367) задокументированное в средстве "Нагрузка и производительность Skype для бизнеса [Server".](https://technet.microsoft.com/library/mt631400.aspx)
  
## <a name="using-the-capacity-calculator"></a>Использование калькулятора емкости

Калькулятор — это таблица Microsoft Excel. Входные ячейки окрашены оранжевым цветом. Значения по умолчанию ввели в ячейках (для Skype для бизнеса Server 2015 80 000 пользователей в одном пуле с двенадцатью серверами переднего сервера, а для Skype для бизнеса Server 2019 — 106 000 пользователей в одном пуле с шестнадцатью серверами переднего сервера), но эти значения следует изменить в соответствие с потребностями организации.
  
Модель использования содержит следующие разделы. Чтобы рассчитать требования к емкости, введите данные, как описано, начиная с верхней части листа и работая по строкам вниз: 
  
 **Обмен мгновенными сообщениями и сведениями о присутствии**
  
- В **области "Число пользователей"** введите количество пользователей, которые будут одновременно вписались в нее. Как правило, это число составляет 80 % от общего числа пользователей, для работы с которые была предусмотрена подготовка. В большинстве случаев для 100% пользователей с одновременной поддержкой мгновенных данных и присутствия будет включена возможность мгновенных услуг. Значение по умолчанию — 80 000 для Skype для бизнеса Server 2015 и 106 000 пользователей для Skype для бизнеса Server 2019.
    
- **Среднее количество контактов в списке** контактов указывает количество контактов, которые мы используем для проверки требований к системе. Это число является фиксированным и не должно изменяться.
    
  **Корпоративная голосовая связь**
  
- In **Users enabled for Корпоративная голосовая связь**, type the percentage of your users enabled for Корпоративная голосовая связь. Значение по умолчанию — 60%. 
    
- В среднем число звонков на пользователя в час **(пиковая нагрузка)** введите число вызовов в час, в течение часа, в которые будет участвовать средний пользователь во время пиковой нагрузки. По умолчанию используется значение 4. 
    
- В **процентах вызовов,** которые используют обход сервера-посредника, введите процент вызовов, которые будут переходить от пользователей к серверу-посреднику. Значение по умолчанию составляет 65 %, но может быть меньше, если вы располнились по географическим регионам или имеете большой процент пользователей, которые работают из дома.
    
- В процентах голосовых пользователей, участвующих в вызовах **UC-PSTN,** введите процент звонков организации, которые являются телефонными звонками UC-PSTN. Значение по умолчанию — 60%.
    
- Процент пользователей голосовой почты, участвующих в вызовах **UC-UC,** показывает процент пользователей, для которых включена Корпоративная голосовая связь, которые будут включены только для вызовов UC-UC. Это число рассчитывается на основе вводимых данных для процента голосовых пользователей, включенных для **вызовов UC-PSTN.** 
    
  **Конференции**
  
- В **процентах пользователей, одновременно** участвующих в конференциях, введите процент пользователей, которые будут одновременно участвовать в конференциях. Значение по умолчанию — 5%. 
    
- В **процентах конференций** только с групповым обменом мгновенными сообщениями (без голосовой связи) введите процент конференций, включающих только обмен мгновенными сообщениями и не включающих звук. Значение по умолчанию — 10 %.
    
- В процентах пользователей, использующих **конференц-конференц-конференцию** с использованием телефонного номера, введите процент участников конференций, которые будут одновременно использовать конференц-конференцию с использованием телефонного номера. Значение по умолчанию — 15%.
    
- В **процентах конференций** с использованием голоса введите процент конференций, которые будут включать звук. 
    
  - Если в 20% голосовых конференций также будет включено обычное видео, выберите "Включить видео **(без** нескольких представлений").
    
  - Если в 20% конференций также будет включено  видео с несколькими представлениями, выберите "Включить несколько представлений".
    
  - Если в 50% голосовых конференций также будет включен общий доступ к приложениям, выберите **"Включить** общий доступ к приложениям".
    
  - Если 20 % голосовых конференций включают отправку данных, например  презентаций PowerPoint, выберите этот флаг.
    
  **Мобильность**
  
- In **Percentage of users enabled for Mobility**, type the percentage of your users who will be enabled to connect to Skype for Business Server using mobile devices. Значение по умолчанию — 40%. 
    
Когда вы ввели все необходимые сведения, калькулятор мощности оценивает ваши требования. Желтые ячейки показывают вычислимые значения для требований к ЦП, памяти и пропускной способности на основе тестов, проведенных в лабораториях производительности Skype для бизнеса Server. Номера предоставляются в качестве рекомендации, не каждый вариант проверяется и проверяется. Вычисляются следующие значения: 
  
- ЦП переднего сервера **:** процент использования ЦП, если вся нагрузка обрабатывалась одним сервером переднего сервера с характеристиками, аналогичными серверу, который использовался при тестировании (см. описание в конце этой статьи).
    
- **Сеть в Мбит/с:** требования к пропускной способности в мегабитах в секунду (Мбит/с) для соответствующей рабочей нагрузки.
    
- **Память в ГБ:** память, необходимая для соответствующей рабочей нагрузки в гигабайтах (ГБ).
    
В зеленой ячейке кажутся рекомендации для введенной модели использования. 
  
- Всего серверов переднего сервера **:** необходимое количество физических серверов основано на выделенных серверах skype для бизнеса Server 2015 с двухъядерным шестиядерным процессором с 2260 мегациклами или Skype для бизнеса Server 2019 с Intel Xeon E5-2673 v3, двухъядерным шестиядерным процессором.
    
    Обратите внимание, что рекомендуется использовать hyperthreading, что повышает производительность серверов, которые поддерживают аудио- и видеосвязь.
    
- **Edge Servers**: необходимое количество серверов, основанное на 30 % всех пользователей, одновременно общающихся через эти серверы. Этот процент нельзя изменить в калькуляторе. 
    
- **Хранилище служб архивации,** регистрации вызовов и качества работы: количество хранилищ, необходимых для функций архивации или мониторинга, если они включены в организации.
    
- Требуется сервер базы данных на **тыловом сервере (необходимые пулы).** Количество серверов баз данных, необходимых для поддержки выбранной рабочей нагрузки.
    
Кроме того, в строке рядом с "Всего серверов переднего плана" предоставляются дополнительные сведения о нагрузке на серверы и сеть для всех запланированных рабочих нагрузок вместе.
  
- **Средняя загрузка ЦП**: среднее использование ЦП на сервер переднего сервера.
    
- **Сеть в Мбит/с:** необходимое выделение пропускной способности для поддержки введенной модели использования.
    
- **Память в ГБ:** память в гигабайтах, необходимая для каждого сервера переднего сервера.
    
### <a name="adjusting-for-your-processors"></a>Настройка для процессоров

Все показатели использования ЦП в электронной таблице предполагают, что каждый сервер Skype для бизнеса Server 2015 имеет двухъядерный шестиядерный процессор с тактами 2,26 ГГц, не менее 32 ГБ памяти и 8 или более жестких дисков со 10 000 об/мин и не менее 72 ГБ свободного места на диске. For each Skype for Business Server 2019 server, all the CPU usage figures in the spreadsheet assume that each server has a dual processor, hex-core with Intel Xeon E5-2673 v3, at least 64 GB of memory, and 8 or more 10,000-RPM hard disk drives with at least 72 GB free disk space.
  
Если ваши серверы имеют разные процессоры, вы можете изменить показатели в соответствие с оборудованием.
