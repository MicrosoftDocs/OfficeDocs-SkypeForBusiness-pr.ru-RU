# <a name="install-and-open-administrative-tools"></a>Установка и открытие средств администрирования

В этом разделе описывается установка средств администрирования, необходимых для развертывания и управления Skype для бизнеса Server. Средства администрирования устанавливаются по умолчанию на каждом сервере, на котором работает Skype для бизнеса Server. Кроме того, вы можете установить административные средства на другие компьютеры, например на выделенные административные консоли. Настоятельно рекомендуется установить средства администрирования на компьютер, который находится в том же домене или лесу, что и развертывание Skype для бизнеса Server, чтобы убедиться, что этапы подготовки доменных служб Active Directory уже выполнены. что позволяет позднее использовать средства администрирования на этом компьютере для публикации топологии. Кроме того, перед установкой или использованием средств администрирования Skype для бизнеса Server обязательно Проанализируйте необходимые требования. Ознакомьтесь с документацией по требованиям в [Skype для бизнеса server 2019](../../SfBServer2019/plan/system-requirements.md) или [Skype для бизнеса Server 2015](../plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md).
 
> [!Important]
> Если для организации требуется, чтобы вы могли размещать службы IIS и все веб-службы на диске, отличном от системного диска, можно изменить путь к папке установки для файлов Skype для бизнеса Server в диалоговом окне программы установки. Если вы устанавливаете файлы установки по этому пути, включая OCSCore. msi, остальные файлы сервера Skype для бизнеса будут развернуты и на этом диске. 

## <a name="to-install-the-administrative-tools"></a>Установка средств администрирования

1. Войдите в качестве локального администратора (минимальное требование) на компьютер, на котором требуется установить средства администрирования. Если вы вошли в систему под учетной записью обычного пользователя в Windows, а Управление учетными записями пользователей (UAC) включено, вам будет запрошен локальный администратор или домен, эквивалентный имени пользователя и паролю.
2. Найдите установочный носитель на компьютере, а затем дважды щелкните \Setup\amd64\setup.exe.
3. Если вам будет предложено установить распространяемый пакет Microsoft Visual C++, нажмите кнопку **Да**.
4. На странице Папка установки нажмите кнопку **ОК**. Измените этот путь на другую папку или диск, если вы хотите установить файлы в другом местоположении.

    > [!Important]
    > Если для организации требуется, чтобы вы могли размещать службы IIS и все веб-службы на диске, отличном от системного диска, можно изменить путь к папке установки для файлов Skype для бизнеса Server в диалоговом окне программы установки. Если вы устанавливаете файлы установки по указанному пути, включая OCSCore. msi, остальные файлы сервера Skype для бизнеса будут развернуты на этом диске. 

5. На странице Лицензионное соглашение просмотрите условия лицензионного соглашения, нажмите кнопку **Принимаю**, а затем нажмите **ОК**. Этот шаг обязателен для продолжения.
6. На странице "Мастер развертывания" щелкните **установить средства администрирования**. 
7. После успешного завершения установки нажмите кнопку **Выход**.

Используйте следующие процедуры для открытия средств администрирования для развертывания, настройки или устранения неполадок в топологии Skype для бизнеса Server.

- [Мастер развертывания](#deployment-wizard)
- [Построитель топологий](#topology-builder) 
- [Панель управления Skype для бизнеса Server](#skype-for-business-server-control-panel)
- [Командная консоль Skype для бизнеса Server](#skype-for-business-server-management-shell)

## <a name="deployment-wizard"></a>Мастер развертывания

Используйте следующую процедуру, чтобы запустить мастер развертывания локально, чтобы добавить или удалить файлы компонентов.

**Запуск мастера развертывания Skype для бизнеса Server**

1. Войдите на компьютер, на котором установлен мастер развертывания Skype для бизнеса Server, как член группы администраторов домена и группы RTCUniversalServerAdmins.
2. Нажмите кнопку **Пуск**, последовательно выберите пункты **все программы**, **Skype для бизнеса Server**, а затем выберите **Мастер развертывания Skype для бизнеса Server**.


## <a name="topology-builder"></a>Построитель топологий 

Используйте следующую процедуру, чтобы открыть построитель топологий, чтобы определить серверы, которые необходимо развернуть в топологии Skype для бизнеса Server.

**Открытие построителя топологий Skype для бизнеса Server для создания топологии**

1. Войдите в систему компьютера, на котором построитель топологий установлен как член группы администраторов доменов и группы RTCUniversalServerAdmins.
    > [!NOTE]
    > Вы можете определить топологию с помощью учетной записи, которая является членом локальной группы "Пользователи", но для чтения, публикации или включения топологии, необходимой для установки Skype для бизнеса Server на сервере, необходимо использовать учетную запись, входящую в группу "Администраторы домена" и e RTCUniversalServerAdmins Group и имеет разрешения на полный доступ (то есть чтение, запись и изменение) к общему файловому ресурсу, который планируется использовать для архивного хранилища файлов, чтобы построитель топологий мог настроить необходимый список управления доступом на уровне пользователей ( DACL) или учетная запись с эквивалентными правами пользователя.
 
2. Запустите построитель топологий: нажмите кнопку **Пуск**, последовательно выберите пункты **все программы**, **Skype для бизнеса Server**и **Построитель топологий Skype для бизнеса Server**.

## <a name="skype-for-business-server-control-panel"></a>Панель управления Skype для бизнеса Server 

Используйте одну из следующих процедур, чтобы открыть панель управления Skype для бизнеса Server для управления конфигурацией серверов, пользователей, клиентов и устройств в вашей среде.

> [!NOTE]
> С помощью учетной записи пользователя, назначенной роли CsAdministrator, можно выполнить какие-либо задачи на панели управления Skype для бизнеса Server. С помощью других ролей можно войти на панель управления Skype для бизнеса Server, чтобы выполнять определенные задачи администрирования, зависящие от задачи, которую необходимо выполнить. Например, вы можете использовать роли csarchivingadministrator для администрирования архивации на панели управления Skype для бизнеса Server. Сведения о ролях можно найти в статье [Планирование управления доступом на основе ролей](https://technet.microsoft.com/library/gg425917(v=ocs.15).aspx). Подробные сведения о ролях, которые можно использовать для выполнения особых задач, см. в документации по этим задачам. 

**Открытие панели управления Skype для бизнеса Server с любого компьютера в брандмауэре организации**

1. С помощью учетной записи, которой назначена роль CsAdministrator или другая роль с соответствующими правами и разрешениями для выполняемой задачи, войдите в любой компьютер во внутреннем развертывании, разрешение экрана которого не ниже 1024 x 768.

    > [!IMPORTANT]
    > Если вы настроили простой URL-адрес администрирования, вы можете получить доступ к панели управления Skype для бизнеса Server из Интернет-браузера, запущенного на любом компьютере в брандмауэре организации. Подробнее о настройке простого URL-адреса администрирования можно узнать в статье [планирование простых URL-адресов](https://technet.microsoft.com/library/gg398287(v=ocs.15).aspx) , а [также изменение или Настройка простых URL-адресов](https://technet.microsoft.com/library/gg398063(v=ocs.15).aspx). 

2. Откройте окно браузера и введите URL-адрес администрирования, настроенный для вашей организации.

**Открытие панели управления Skype для бизнеса Server на компьютере, на котором работает Skype для бизнеса Server**

1. Войдите на компьютер, на котором установлен Skype для бизнеса Server, из учетной записи пользователя, которая является участником роли CsAdministrator или другой роли с соответствующими правами и разрешениями для выполняемой задачи, и войдите на компьютер, на котором установлен Skype для бизнеса Server, или как минимум , средства администрирования Skype для бизнеса Server. Для настройки параметров разрешение экрана компьютера должно быть не ниже 1024 x 768.
2. Запустите Панель управления Skype для бизнеса Server: нажмите кнопку **Пуск**, последовательно выберите пункты **все программы**, **Администрирование**, **Skype для бизнеса Server**, а затем щелкните **элемент Панель управления Skype для бизнеса Server**.

## <a name="skype-for-business-server-management-shell"></a>Командная консоль Skype для бизнеса Server 

Используйте следующую процедуру, чтобы открыть консоль управления Skype для бизнеса Server для администрирования серверов, пользователей, клиентов и устройств в вашей среде с помощью командной строки.

> [!NOTE]
> Вы можете использовать учетную запись пользователя, назначенную роли CsAdministrator, для выполнения любой задачи в командной консоли Skype для бизнеса Server. Для выполнения особых административных задач можно использовать для входа другие роли, в зависимости от конкретной задачи. Например, роль CSArchivingAdministrator можно использовать для запуска командлетов, имеющих отношение к администрированию архивации. Сведения о ролях можно найти в статье [Планирование управления доступом на основе ролей](https://technet.microsoft.com/library/gg425917(v=ocs.15).aspx). Подробные сведения о ролях, которые можно использовать для выполнения конкретных командлетов, см. в документации по этим командлетам.<br/><br/>Определенные командлеты можно также выполнять с помощью пользовательской учетной записи, входящей в группу RTCUniversalServerAdmins, RTCUniversalUserAdmins или RTCUniversalReadOnlyAdmins, в зависимости от командлета. 

**Чтобы открыть консоль управления Skype для бизнеса Server**

Если открыть окно Windows PowerShell вместо командной консоли Skype для бизнеса Server, по умолчанию командлеты Skype для бизнеса Server не могут быть запущены. Чтобы запустить командлеты Skype для бизнеса Server в Windows PowerShell, введите в командной строку Windows PowerShell следующую команду:

`Import-Module Lync`

Запустите консоль управления Skype для бизнеса Server: нажмите кнопку **Пуск**, последовательно выберите пункты **все программы**, **Skype для бизнеса Server**, а затем выберите пункт **Командная консоль Skype для бизнеса Server**.
