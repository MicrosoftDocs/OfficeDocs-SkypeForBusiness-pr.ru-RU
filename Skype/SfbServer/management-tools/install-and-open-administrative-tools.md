# <a name="install-and-open-administrative-tools"></a>Установка и открытие средств администрирования

В этом разделе описано, как установить административные средства, необходимые для развертывания и управления Skype для бизнеса Server. Административные средства устанавливаются по умолчанию на каждом сервере, на который работает Skype для бизнеса Server. Кроме того, вы можете установить административные средства на другие компьютеры, например на выделенные административные консоли. Настоятельно рекомендуем установить административные средства на компьютере, который находится в том же домене или лесу, что и развертывание Skype для бизнес-сервера, которое вы создаете, чтобы убедиться, что этапы подготовки Active Directory Domain Services уже завершены, что позволяет использовать административные средства на этом компьютере позже для публикации топологии. Также убедитесь, что перед установкой или использованием средств администрирования Skype для бизнеса Server необходимо просмотреть необходимые требования. См. документацию по требованиям [в Skype для бизнеса Server 2019](../../SfBServer2019/plan/system-requirements.md) или Skype для бизнеса Server [2015](../plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md).
 
> [!Important]
> Если организации требуется найти службы интернет-информации (IIS) и все веб-службы на диске, помимо системного диска, можно изменить путь расположения установки для файлов Skype для бизнеса Server в диалоговом окне Установки. При установке файлов установки на этот путь, в том числе OCSCore.msi, остальные файлы Skype для бизнеса Server также будут развернуты на этом диске. 

## <a name="to-install-the-administrative-tools"></a>Установка административных средств

1. Войдите в качестве локального администратора (минимальное требование) на компьютер, на котором требуется установить средства администрирования. Если вы вошли в систему в качестве стандартного пользователя в Windows и управление учетной записью пользователя (UAC), вам будет предложено имя и пароль, эквивалентный домену.
2. Найдите установочный носитель на компьютере, а затем дважды щелкните \Setup\amd64\setup.exe.
3. Если вам предложено установить распространяемый microsoft Visual C++ , нажмите **кнопку Да**.
4. На странице Папка установки нажмите кнопку **ОК**. Измените этот путь на другую папку или диск, если вы хотите установить файлы в другом местоположении.

    > [!Important]
    > Если организации требуется найти службы интернет-информации (IIS) и все веб-службы на диске, помимо системного диска, можно изменить путь расположения установки для файлов Skype для бизнеса Server в диалоговом окне Установки. Если установить файлы установки на этот путь, включая OCSCore.msi, остальные файлы Skype для бизнеса Server будут развернуты и на этом диске. 

5. На странице Лицензионное соглашение просмотрите условия лицензионного соглашения, нажмите кнопку **Принимаю**, а затем нажмите **ОК**. Этот шаг обязателен для продолжения.
6. На странице Мастер развертывания нажмите **кнопку Установите средства администратора**. 
7. После успешного завершения установки нажмите кнопку **Выход**.

Используйте следующие процедуры, чтобы открыть средства администрирования для развертывания, настройки или устранения неполадок топологии Skype для бизнеса Server.

- [Мастер развертывания](#deployment-wizard)
- [Построитель топологий](#topology-builder) 
- [Панель управления Skype для бизнеса Server](#skype-for-business-server-control-panel)
- [Командная консоль Skype для бизнеса Server](#skype-for-business-server-management-shell)

## <a name="deployment-wizard"></a>Мастер развертывания

Используйте следующую процедуру, чтобы запустить мастер развертывания локально, чтобы добавить или удалить файлы компонентов.

**Запуск мастера развертывания Skype для бизнес-серверов**

1. Войдите на компьютер, на котором мастер развертывания Skype для бизнеса серверов установлен в качестве члена группы администраторов домена и группы RTCUniversalServerAdmins.
2. Нажмите **кнопку Начните,** нажмите кнопку **Все программы,** **щелкните Skype для** бизнеса Server, а затем нажмите Skype для мастер развертывания **бизнес-сервера**.


## <a name="topology-builder"></a>Построитель топологий 

Чтобы определить серверы, которые необходимо развернуть в топологии Skype для бизнеса Server, используйте следующую процедуру.

**Открытие топологии Skype для бизнес-серверов для разработки топологии**

1. Войдите в систему компьютера, на котором построитель топологий установлен как член группы администраторов доменов и группы RTCUniversalServerAdmins.
    > [!NOTE]
    > Топологию можно определить с помощью учетной записи, которая является членом локальной группы пользователей, но для чтения, публикации или включить топологию, которая необходима для установки Skype для бизнес-сервера на сервере, необходимо использовать учетную запись, которая является членом группы администраторов домена и группы RTCUniversalServerAdmins, и которая имеет полные разрешения управления (т. е. , чтение, запись и изменение) на файле, который вы собираетесь использовать для архива файловый магазин, чтобы Топология Builder можно настроить необходимый список управления дискреционным доступом (DACLs), или учетную запись с эквивалентными правами пользователя.
 
2. Начните строитель топологии: нажмите кнопку **Начните,** нажмите кнопку Все **программы,** **щелкните Skype для** бизнес-сервера, а затем нажмите Skype для **бизнес-сервера Топология Строитель**.

## <a name="skype-for-business-server-control-panel"></a>Панель управления Skype для бизнеса Server 

Используйте одну из следующих процедур, чтобы открыть панель управления Skype для бизнес-серверов для управления конфигурацией серверов, пользователей, клиентов и устройств в вашей среде.

> [!NOTE]
> Вы можете использовать учетную запись пользователя, назначенную роли CsAdministrator, для выполнения любой задачи в панели управления Skype для бизнес-серверов. Вы можете использовать другие роли для входа в панель управления Skype для бизнес-серверов для выполнения определенных задач администрирования в зависимости от задачи, необходимой для выполнения. Например, можно использовать CSArchivingAdministrator для администрирования архива в панели управления Skype для бизнес-серверов. Подробные сведения о ролях см. в [материале Планирование управления доступом на основе ролей.](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control) Подробные сведения о ролях, которые можно использовать для выполнения особых задач, см. в документации по этим задачам. 

**Открытие панели управления Skype для бизнес-серверов с любого компьютера на брандмауэре организации**

1. С помощью учетной записи, которой назначена роль CsAdministrator или другая роль с соответствующими правами и разрешениями для выполняемой задачи, войдите в любой компьютер во внутреннем развертывании, разрешение экрана которого не ниже 1024 x 768.

    > [!IMPORTANT]
    > Если вы настроили простой единый локатор ресурсов администрирования (URL-адрес), вы можете получить доступ к панели управления Skype для бизнес-серверов из интернет-браузера, который работает на любом компьютере в брандмауэре организации. Сведения о настройке простого URL-адреса администрирования см. в материале [Планирование](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-simple-urls) простых URL-адресов и изменение или [настройка простых URL-адресов.](/previous-versions/office/lync-server-2013/lync-server-2013-edit-or-configure-simple-urls) 

2. Откройте окно браузера и введите URL-адрес администрирования, настроенный для вашей организации.

**Открытие панели управления Skype для бизнес-серверов на компьютере под управлением Skype для бизнеса Server**

1. С учетной записи пользователя, которая является членом роли CsAdministrator или другой роли, которая имеет соответствующие права и разрешения для выполнения задачи, войдите на компьютер, на котором установлен Skype для бизнеса Server или, как минимум, административные средства Skype для бизнеса Server. Для настройки параметров разрешение экрана компьютера должно быть не ниже 1024 x 768.
2. Начните панель управления Skype для бизнес-серверов: нажмите кнопку Начните, нажмите кнопку Все **программы,** указать на административные **средства,** указать Skype для бизнеса **Server,** а затем нажмите панель управления Skype для **бизнес-серверов**.

## <a name="skype-for-business-server-management-shell"></a>Командная консоль Skype для бизнеса Server 

Используйте следующую процедуру, чтобы открыть командную оболочку Skype для бизнес-серверов для администрирования серверов, пользователей, клиентов и устройств в вашей среде с помощью командной строки.

> [!NOTE]
> Вы можете использовать учетную запись пользователя, назначенную роли CsAdministrator, для выполнения любой задачи в оболочке управления Skype для бизнес-серверов. Для выполнения особых административных задач можно использовать для входа другие роли, в зависимости от конкретной задачи. Например, роль CSArchivingAdministrator можно использовать для запуска командлетов, имеющих отношение к администрированию архивации. Подробные сведения о ролях см. в [материале Планирование управления доступом на основе ролей.](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control) Подробные сведения о ролях, которые можно использовать для выполнения конкретных командлетов, см. в документации по этим командлетам.<br/><br/>Определенные командлеты можно также выполнять с помощью пользовательской учетной записи, входящей в группу RTCUniversalServerAdmins, RTCUniversalUserAdmins или RTCUniversalReadOnlyAdmins, в зависимости от командлета. 

**Открытие оболочки управления Skype для бизнес-серверов**

Если вы откроете Windows PowerShell, а не оболочку управления Skype для бизнес-серверов, по умолчанию вы не сможете запускать команды Skype для бизнеса Server. Чтобы запустить командлеты Skype для бизнеса Server из Windows PowerShell, введите следующую команду в Windows PowerShell командной подсказке:

`Import-Module Lync`

Запустите оболочку управления Skype для бизнес-серверов: нажмите кнопку Начните, щелкните Все **программы,** щелкните **Skype для** бизнес-сервера, а затем нажмите кнопку Skype для управления **бизнес-сервером.**
