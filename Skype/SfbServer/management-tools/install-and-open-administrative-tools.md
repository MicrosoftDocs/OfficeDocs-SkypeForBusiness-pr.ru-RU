# <a name="install-and-open-administrative-tools"></a>Установка и открытие средств администрирования

В этой статье описано, как установить средства администрирования, необходимые для развертывания Skype для бизнеса Server и управления ими. Средства администрирования устанавливаются по умолчанию на каждом сервере, работающем под управлением Skype для бизнеса Server. Кроме того, вы можете установить средства администрирования на других компьютерах, таких как специальные консоли администрирования. Настоятельно рекомендуем установить средства администрирования на компьютере, который находится в том же домене или лесе, что и развертывание Skype для бизнеса Server, чтобы убедиться, что подготовительные действия для доменных служб Active Directory уже завершены. Это позволит вам позднее использовать средства администрирования на этом компьютере для публикации топологии. Кроме того, прежде чем устанавливать или использовать средства администрирования Skype для бизнеса Server, обязательно ознакомьтесь с необходимыми требованиями. Ознакомьтесь с документацией по требованиям в [Skype для бизнеса server 2019](../../SfBServer2019/plan/system-requirements.md) или [Skype для бизнеса Server 2015](../plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md).
 
> [!Important]
> Если в вашей организации требуется найти службы IIS и все веб-службы на диске, отличном от системного, вы можете изменить путь к папке установки сервера Skype для бизнеса в диалоговом окне Настройка. Если вы установили файлы установки по этому пути, включая Окскоре. msi, остальные файлы сервера Skype для бизнеса будут развернуты и на этом диске. 

## <a name="to-install-the-administrative-tools"></a>Установка средств администрирования

1. Войдите в систему под учетной записью локального администратора (минимально требование) на компьютер, на котором вы хотите установить средства администрирования. Если вы вошли в систему как обычный пользователь в Windows, а Управление учетными записями пользователей (UAC) включено, вам будет предложено указать локального администратора или доменного имени пользователя и пароля.
2. Найдите установочный носитель на компьютере и дважды щелкните \Setup\amd64\Setup.exe.
3. Если вам будет предложено установить распространяемый компонент Microsoft Visual C++, нажмите кнопку **Да**.
4. На странице Папка установки нажмите **ОК**. Если вы хотите, чтобы файлы были установлены в другое место, измените этот путь на другое место или на диск.

    > [!Important]
    > Если в вашей организации требуется найти службы IIS и все веб-службы на диске, отличном от системного, вы можете изменить путь к папке установки сервера Skype для бизнеса в диалоговом окне Настройка. Если вы установили файлы установки по этому пути, включая Окскоре. msi, остальные файлы сервера Skype для бизнеса будут развернуты на этом диске. 

5. На странице Лицензионное соглашение конечного пользователя ознакомьтесь с условиями лицензионного соглашения, нажмите кнопку **я принимаю**, а затем нажмите кнопку **ОК**. Чтобы продолжить, необходимо выполнить это действие.
6. На странице мастера развертывания щелкните ссылку **Установка средств администрирования**. 
7. Когда установка завершится успешно, нажмите кнопку **выход**.

Чтобы развернуть, настроить или устранить неполадки, связанные с топологией сервера Skype для бизнеса, выполните указанные ниже действия, чтобы открыть средства администрирования.

- [Мастер развертывания](#deployment-wizard)
- [топологий](#topology-builder) 
- [Панель управления Skype для бизнеса Server](#skype-for-business-server-control-panel)
- [Командная консоль Skype для бизнеса Server](#skype-for-business-server-management-shell)

## <a name="deployment-wizard"></a>Мастер развертывания

Используйте описанную ниже процедуру, чтобы запустить мастер развертывания локально, чтобы добавить или удалить файлы компонентов.

**Запуск мастера развертывания Skype для бизнеса Server**

1. Войдите в систему на компьютере, на котором установлен мастер развертывания Skype для бизнеса Server, в группе Администраторы домена и в группу Рткуниверсалсерверадминс.
2. Нажмите кнопку **Пуск**, выберите пункт **все программы**, а затем — **Skype для бизнеса Server**, а затем — **Мастер развертывания Skype для бизнеса Server**.


## <a name="topology-builder"></a>топологий 

Используйте описанную ниже процедуру, чтобы открыть построитель топологии, чтобы определить серверы, которые вы хотите развернуть, в топологии сервера Skype для бизнеса.

**Открытие построителя топологии Skype для бизнеса Server для создания топологии**

1. Войдите на компьютер, где установлен построитель топологий, с использованием учетной записи, входящей в группу администраторов домена и группу RTCUniversalServerAdmins.
    > [!NOTE]
    > Вы можете определить топологию с помощью учетной записи, которая является членом локальной группы "Пользователи", но для чтения, публикации или включения топологии, необходимой для установки сервера Skype для бизнеса на сервере, необходимо использовать учетную запись, которая входит в группу "Администраторы домена" и т. д. e Рткуниверсалсерверадминс Group и имеет разрешения на полный доступ (то есть чтение, запись и изменение) в общей папке, которую вы собираетесь использовать в хранилище файлов для архивации, чтобы построитель топологии мог настроить требуемый список управления доступом на уровне пользователей ( DACL) или учетной записи с эквивалентными правами пользователя.
 
2. Запустить построитель топологии: нажмите кнопку **Пуск**, выберите пункт **все программы**, а затем — **** построитель **топологии Skype для бизнеса Server**.

## <a name="skype-for-business-server-control-panel"></a>Панель управления Skype для бизнеса Server 

Воспользуйтесь одной из описанных ниже процедур, чтобы открыть панель управления Skype для бизнеса Server для управления конфигурацией серверов, пользователей, клиентов и устройств в среде.

> [!NOTE]
> Вы можете использовать учетную запись пользователя, назначенную роли Ксадминистратор, для выполнения любой задачи на панели управления Skype для бизнеса Server. С помощью других ролей можно войти на серверную панель управления Skype для бизнеса, чтобы выполнить определенные задачи администрирования, зависящие от задачи, которую необходимо выполнить. Например, вы можете использовать Ксарчивингадминистратор для управления архивацией на панели управления Skype для бизнеса Server. Сведения о ролях можно найти в разделе [Планирование управления доступом на основе ролей](https://technet.microsoft.com/en-us/library/gg425917(v=ocs.15).aspx). Подробные сведения о ролях, которые можно использовать для выполнения определенной задачи, описаны в документации по задаче. 

**Открытие панели управления Skype для бизнеса Server с любого компьютера в брандмауэре организации**

1. Войдите в учетную запись пользователя, которой назначена роль Ксадминистратор или другая роль, которая имеет соответствующие права и разрешения для выполнения задачи, выполните вход на любом компьютере во внутренней среде с минимальным разрешением экрана 1024 x 768.

    > [!IMPORTANT]
    > Если вы настроили URL-адрес для администрирования, вы можете получить доступ к панели управления Skype для бизнеса Server из Интернет-браузера, который запущен на любом компьютере в брандмауэре организации. Дополнительные сведения о настройке простого URL-адреса администрирования можно найти в разделе [планирование простых URL-адресов](https://technet.microsoft.com/en-us/library/gg398287(v=ocs.15).aspx) и [изменение или Настройка простых URL-адресов](https://technet.microsoft.com/en-us/library/gg398063(v=ocs.15).aspx). 

2. Откройте окно браузера и введите URL-адрес администратора, настроенный для Организации.

**Открытие панели управления Skype для бизнеса Server на компьютере с установленным Skype для бизнеса Server**

1. Войдите в учетную запись пользователя, которая является членом роли Ксадминистратор или другой роли, обладающей необходимыми правами и разрешениями для выполнения задачи, выполните вход на компьютер, на котором установлен Skype для бизнеса Server, или, по крайней мере, по меньшей мере , средства администрирования сервера Skype для бизнеса Server. Для настройки параметров на компьютере должно быть установлено минимальное разрешение экрана 1024 x 768.
2. Откройте панель управления Skype для бизнеса Server: нажмите **кнопку Пуск**, выберите **все программы**, наведите указатель на пункт **Администрирование**, выберите пункт **Skype для бизнеса Server**, а затем выберите пункт **Панель управления Skype для бизнеса Server**.

## <a name="skype-for-business-server-management-shell"></a>Командная консоль Skype для бизнеса Server 

Используйте описанную ниже процедуру, чтобы открыть консоль управления Skype для бизнеса Server для администрирования серверов, пользователей, клиентов и устройств в среде с помощью командной строки.

> [!NOTE]
> Вы можете использовать учетную запись пользователя, назначенную роли Ксадминистратор, для выполнения какой-либо задачи в командной консоли Skype для бизнеса Server. Вы можете войти в систему, используя другие роли для выполнения конкретных задач администрирования, в зависимости от задачи, которую необходимо выполнить. Например, для запуска командлетов, связанных с администрированием архивации, можно использовать Ксарчивингадминистратор. Сведения о ролях можно найти в разделе [Планирование управления доступом на основе ролей](https://technet.microsoft.com/en-us/library/gg425917(v=ocs.15).aspx). Подробные сведения о ролях, которые можно использовать для выполнения определенного командлета, описаны в документации для командлета.<br/><br/>Вы также можете запускать определенные командлеты с помощью учетной записи пользователя в группах Рткуниверсалсерверадминс, Рткуниверсалусерадминс или Рткуниверсалреадонлядминс, в зависимости от командлета. 

**Открытие командной консоли Skype для бизнеса Server**

Если вы открыли окно Windows PowerShell вместо командной консоли Skype для бизнеса Server, по умолчанию не удается запустить командлеты Skype для бизнеса Server. Чтобы запустить командлеты Skype для бизнеса Server в Windows PowerShell, введите следующую команду в командной строке Windows PowerShell:

`Import-Module Lync`

Запустите консоль управления в Skype для бизнеса Server: нажмите кнопку **Пуск**, выберите **все программы**, а затем — **Skype**для бизнеса Server, а затем — **Командная консоль управления Skype для бизнеса Server**.
