# <a name="skype-for-business-server-2015-stress-and-performance-tool"></a>Скайп для Business Server 2015 Stress and Performance Tool
 
Скайп для Business Server 2015 Stress and Performance Tool используется при планировании мощности и производительности, помощник по настройке в тестовой или тестовой среды.
  
Скайп для Business Server 2015 Stress and Performance Tool включает средства, упрощающие вашей планирование мощности для Скайп для Business Server 2015. Скайп для Business Server 2015 Stress and Performance Tool приведены инструкции по:
  
- Упростить планирование для Business Server Скайп оборудования
    
- Присвойте увеличено знания и практические советы по оптимизации производительности
    
- Измерение производительности вашей Скайп для развертываний Business Server
    
Проектирование топологии и оптимизация топологии с [Скайп для бизнеса 2015 калькулятор емкости Server планирования](../../management-tools/capacity-planning-calculator.md)с помощью [Скайп для средство планирования Business Server 2015](../../management-tools/planning-tool/planning-tool.md) обычно используется это средство. 
  
## <a name="tests"></a>Тесты

Stress and Performance Tool можно смоделировать эти типы пользовательской нагрузки:
  
|||
|:-----|:-----|
|Мгновенного обмена Мгновенными сообщениями и присутствия  <br/> |Аудиоконференции  <br/> |
|Общий доступ к приложениям  <br/> |Голосовой связи по протоколу IP (VoIP), включая телефонной сети общего пользования моделирования (PTSN)  <br/> |
|Веб-конференции клиентского доступа  <br/> |Автосекретарь конференции  <br/> |
|Группы ответа  <br/> |Раскрытие списка рассылки  <br/> |
|Загрузка адресной книги и запросов адресной книги  <br/> |Enhanced экстренные вызовы (службы E911) и профиля расположения (абонентской группы)  <br/> |
|MultiView  <br/> |Совместная работа с данными  <br/> |
|Мобильность  <br/> ||
   
## <a name="applications-and-files-included-with-the-skype-for-business-server-2015-stress-and-performance-tool"></a>Приложения и файлы, включенные в состав Скайп для Business Server 2015 Stress and Performance Tool

Эти приложения являются частью Скайп для Business Server Stress and Performance Tool:
  
|**Средство**|**Описание**|
|:-----|:-----|
|UserProvisioningTool.exe  <br/> |Это средство используется для создания пользователей и контактов.  <br/> |
|UserProfileGenerator.exe  <br/> |Используется для настройки характеристики нагрузки, в случае моделирования.  <br/> |
|LyncPerfTool.exe  <br/> |Средство, которая моделирует пользовательской нагрузки.  <br/> |
|Default.tmx  <br/> |Для использования Скайп средство ведения журнала 2015 Business Server требуется.  <br/> |
|Подготовка примеры сценариев  <br/> |Используется для настройки топологии для запуска нагрузочных тестов, в зависимости от конкретных сценариев. Скорее всего, вам потребуется изменить их, чтобы они соответствовали для используемой среды.  <br/> |
