# <a name="skype-for-business-server-2015-management-tools"></a>Средства управления Skype для бизнеса Server 2015
 
**Сводка:** Сведения о средствах управления службами в Skype для бизнеса Server 2015.
  
Приложение для обмена мгновенными сообщениями Skype для бизнеса Server 2015 (ранее Lync Server) предоставляет решения для обмена мгновенными сообщениями, присутствие, Конференц-связь и телефонные решения, которые поддерживают требования к совместной работе на уровне предприятия. Инструменты для управления этими службами являются гибкими и мощными.
