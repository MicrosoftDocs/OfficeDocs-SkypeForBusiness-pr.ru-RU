# <a name="skype-for-business-server-2015-management-tools"></a>Инструменты управления Skype для бизнеса Server 2015
 
**Сводка:** Узнайте о средствах управления услугами в Skype для бизнеса Server 2015.
  
Программное обеспечение Skype для Business Server 2015 (прежнее название — Lync Server) обеспечивает обмен мгновенными сообщениями, сведения о присутствии, Конференции и телефонные решения, которые поддерживают требования для совместной работы в корпоративном масштабе. Инструменты для управления этими службами отличаются гибкостью и мощностью.
