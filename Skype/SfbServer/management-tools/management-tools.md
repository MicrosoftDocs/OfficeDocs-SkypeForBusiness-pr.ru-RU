# <a name="skype-for-business-server-2015-management-tools"></a>Средства управления Skype для бизнеса Server 2015
 
**Сводка:** Узнайте о средствах управления службами в Skype для бизнеса Server 2015.
  
Skype для бизнеса Server 2015 communications software (ранее Lync Server) предлагает решения для обмена мгновенными сообщениями, присутствия, телефонии и телефонии, которые поддерживают требования совместной работы на уровне предприятия. Средства для управления этими службами являются гибкими и мощными.
