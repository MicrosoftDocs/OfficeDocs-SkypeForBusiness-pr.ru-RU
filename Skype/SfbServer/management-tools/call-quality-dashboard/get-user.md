# <a name="get-user"></a>Получение пользователя
 
**Сводка:** Узнайте об операции "Получить пользователя", которая входит в состав службы пользователей. Служба пользователей является частью API репозитория для панели мониторинга качества вызовов. Панель мониторинга качества звонков — это инструмент для Skype для бизнеса Server.
  
Операция "Получить пользователей" является частью службы пользователей в API репозитория для панели мониторинга качества вызовов.
  
## <a name="get-user"></a>Получение пользователя

Get User returns a user record from the repository.
  
|**Способ**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|GET  <br/> |https:// \<portal\> /QoERepositoryService/repository/user/{userId}  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Request Headers** - No additional headers.
  
 **Тело запроса** — нет.
  
 **Ответ.** Ответ включает код состояния HTTP и набор загодеров отклика.
  
 **Код состояния** : успешная операция возвращает код состояния 200 (OK). Если указанный код пользователя не найден, возвращается код состояния 404 (не найден).
  
 **Response Headers** - No additional headers.
  
 **Тело ответа.** Ниже приведен пример полезной нагрузки ответа в JSON.
  
```json
{
"userId": 0,
"loginName": "system",
"defaultItemId": 1655
}
```

 *userId*  — ИД пользователя.
  
 *loginName*  — идентификация внешних пользователей для обычных пользователей. Если для проверки подлинности пользователей используется проверка подлинности Windows, это может быть FQDN пользователя.
  
 *defaultItemId*  — ИД элемента по умолчанию для этого пользователя. Элемент по умолчанию — это самый верхний элемент, связанный с пользователем. Все остальные элементы, которые принадлежат этому пользователю, можно перемещать из элемента по умолчанию.
  
> [!NOTE]
> `defaultItemId`Укавите значение операции "Получить элемент", чтобы получить сведения об элементе по умолчанию.
