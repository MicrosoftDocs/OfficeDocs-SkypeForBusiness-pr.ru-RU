# <a name="get-user"></a>Получение пользователя
 
**Сводка:** Узнайте об операции Get User, которая является частью службы пользователей. Служба пользователей является частью API репозиториев для панели мониторинга качества вызовов. Панель мониторинга качества вызовов — это средство для Skype для бизнеса Server.
  
Операция Get Users является частью службы пользователей в API репозиториев для панели мониторинга качества вызовов.
  
## <a name="get-user"></a>Получение пользователя

Чтобы пользователь возвращал запись пользователя из репозитория.
  
|**Способ**|**Запрос URI**|**ВЕРСИЯ HTTP**|
|:-----|:-----|:-----|
|GET  <br/> |https:// \<portal\> /QoERepositoryService/repository/user/{userId}  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** . Нет.
  
 **Заглавные заглавные** запросы — дополнительных загонах не будет.
  
 **Тело запроса** . Нет.
  
 **Ответ.** Ответ включает код состояния HTTP и набор заглавных заглавных ответов.
  
 **Код состояния** — успешная операция возвращает код состояния 200 (OK). Если указанный код пользователя не найден, он возвращает код состояния 404 (Не найден).
  
 **Заготчики** ответа . Дополнительные заготчики не имеются.
  
 **Тело ответа** . Ниже приведен пример полезной нагрузки ответа в JSON.
  
```json
{
"userId": 0,
"loginName": "system",
"defaultItemId": 1655
}
```

 *userId*  — ID пользователя.
  
 *loginName*  — внешняя идентификация пользователей для обычных пользователей. Если Windows проверки подлинности используется для проверки подлинности пользователей, это может быть FQDN пользователя.
  
 *defaultItemId*  — ID элемента по умолчанию для этого пользователя. Элемент по умолчанию — это самый верхний элемент, связанный с пользователем. Все остальные элементы, которые принадлежат этому пользователю, могут перемещаться по умолчанию.
  
> [!NOTE]
> Поставляем  `defaultItemId` значение для операции Get Item для получения сведений о элементе по умолчанию.
