# <a name="get-user"></a>Получение пользователя
 
**Сводка:** Сведения об операции получения пользователя, входящего в состав службы пользователей. Служба пользователей является частью API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server.
  
Операция получения пользователей является частью службы пользователей в API репозитория для панели мониторинга качества звонков.
  
## <a name="get-user"></a>Получение пользователя

Получите пользователя возвращает запись пользователя из репозитория.
  
|**Метод**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |https://\<портала\>/QoERepositoryService/репозитория/пользователей / {userId}  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК). Если указанный пользователь с Идентификатором не найден, он возвращает код состояния 404 (не найден).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** - ниже приведен пример полезные данные ответа в формате JSON в.
  
```
{
"userId": 0,
"loginName": "system",
"defaultItemId": 1655
}
```

 *идентификатор пользователя* — идентификатор пользователя.
  
 *loginName* - идентификации внешних пользователей для обычных пользователей. Если используется проверка подлинности Windows для проверки подлинности пользователей, это может быть полное доменное имя пользователя.
  
 *defaultItemId* - идентификатор элемента по умолчанию для этого пользователя. Значение по умолчанию элемента — элемент верхнего уровня, назначенного пользователю. Все другие элементы, которому принадлежит этот пользователь может пройти от элемента по умолчанию.
  
> [!NOTE]
> Предоставить `defaultItemId` значение для операции получения элемента для получения сведений о элемента по умолчанию.
