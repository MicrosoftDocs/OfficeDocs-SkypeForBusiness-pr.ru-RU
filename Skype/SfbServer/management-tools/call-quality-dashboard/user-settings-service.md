# <a name="user-settings-service-for-call-quality-dashboard-cqd"></a>Служба параметров пользователя для панели мониторинга качества звонков (CQD)
 
**Сводка:** Сведения о службе параметров пользователя, которая является частью API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server.
  
Служба параметров пользователя входит в API репозитория для панели мониторинга качества звонков.
  
## <a name="user-settings-service"></a>Служба параметров пользователя

Параметры пользователя, пары "ключ значение", приложения могут использовать для хранения метаданных для различных целей, включая настройки отдельных на пользователя модели поведения приложений. Каждый пользователь получает хранилища для параметров пользователя. Только владелец можно добавлять, изменять и удалять пользовательские параметры.
  
 **Глобальные параметры**
  
Глобальные параметры являются параметры пользователя, принадлежащие пользователю системы и все пользователи имеют доступ только для чтения к ним. Глобальные параметры, являются константами: созданные ими во время создания репозитория, и они никогда не изменяется.
  
Каждый пользователь может переопределять значение глобальные параметры путем создания пользовательских параметров с ключами. Когда приложение запрашивает параметры действующего пользователя, API возвращает набор глобальных параметров объединяются с параметрами пользователей с заменой соответствующих глобальный параметр, если ключи совпадают с каждого из параметров пользователя. API-Интерфейс может также возвращать только что параметров пользователя, чтобы приложения узнать, какие параметры переопределяются.
