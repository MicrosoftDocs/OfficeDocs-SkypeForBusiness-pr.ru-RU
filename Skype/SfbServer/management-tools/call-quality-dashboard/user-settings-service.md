# <a name="user-settings-service-for-call-quality-dashboard-cqd"></a>Служба параметров пользователя для панели мониторинга качества вызовов (CQD)
 
**Сводка:** Узнайте о службе параметров пользователя, которая является частью API репозитория для панели мониторинга качества вызовов. Панель мониторинга качества звонков — это инструмент для Skype для бизнеса Server.
  
Служба параметров пользователя является частью API репозитория для панели мониторинга качества вызовов.
  
## <a name="user-settings-service"></a>Служба параметров пользователя

Пользовательские параметры — это пары "ключ-значение", которые приложения могут использовать для хранения метаданных для различных целей, включая настройку поведения приложений на уровне пользователя. Каждый пользователь получает хранилище для пользовательских параметров. Только владельцы могут добавлять, изменять и удалять параметры пользователя.
  
 **Глобальные параметры**
  
Глобальные параметры — это параметры пользователя, владельцем которых является системный пользователь, и все пользователи имеют к ним доступ только для чтения. Глобальные параметры являются константами: они создаются при создании репозитория и никогда не изменяются.
  
Каждый пользователь может переопределять глобальные параметры, создавая пользовательские параметры с одинаковыми ключами. Когда приложение запрашивает эффективные пользовательские параметры, API возвращает набор глобальных параметров, объединенных с пользовательскими настройками, при этом каждый параметр пользователя перемежает соответствующий глобальный параметр, если ключи одинаковы. API также может возвращать только параметры пользователя, чтобы приложения могли узнать, какие параметры переопределяются.
