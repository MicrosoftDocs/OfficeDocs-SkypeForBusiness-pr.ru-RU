# <a name="user-settings-service-for-call-quality-dashboard-cqd"></a>Служба Параметры для панели мониторинга качества вызовов (CQD)
 
**Сводка:** Узнайте о службе Параметры пользователя, которая входит в API репозиториев для панели мониторинга качества вызовов. Панель мониторинга качества вызовов — это средство для Skype для бизнеса Server.
  
Служба Параметры является частью API репозитория для панели мониторинга качества вызовов.
  
## <a name="user-settings-service"></a>Служба параметров пользователя

Параметры пользователя — это пары с ключевым значением, которые приложения могут использовать для хранения метаданных для различных целей, включая настройку поведения приложений на основе каждого пользователя. Каждый пользователь получает хранилище для параметров пользователя. Только владельцы могут добавлять, изменять и удалять параметры пользователей.
  
 **Глобальный Параметры**
  
Глобальные параметры — это пользовательские параметры, которые принадлежат пользователю системы, и все пользователи имеют к ним доступ только для чтения. Глобальные параметры являются константами: они создаются во время создания репозиториев и никогда не изменяются.
  
Каждый пользователь может переопределять глобальные параметры, создав параметры пользователей с одинаковыми ключами. Когда приложение запрашивает эффективные параметры пользователя, API возвращает набор глобальных параметров, объединенных с пользовательскими настройками, при этом каждый параметр пользователя, если клавиши одинаковы, замешает соответствующий глобальный параметр. API также может возвращать только параметры пользователя, чтобы приложения могли узнать, какие параметры переопределены.
