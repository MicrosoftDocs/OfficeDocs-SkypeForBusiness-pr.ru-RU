# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Панель мониторинга качества звонков в Skype для бизнеса Server
 
**Сводка:** Узнайте о панели мониторинга качества вызовов, которая является средством отчетности для Skype для бизнеса Server.
 
Панель мониторинга Skype для бизнеса Server вызовов (CQD) — это уровень отчетов в верхней части базы данных о качестве работы на сервере мониторинга в Skype для бизнеса Server и Lync Server 2013. CQD использует Microsoft SQL Server службы анализа для предоставления сведений о качестве использования и вызовов, а также фильтрации и поворота в наборе данных.
