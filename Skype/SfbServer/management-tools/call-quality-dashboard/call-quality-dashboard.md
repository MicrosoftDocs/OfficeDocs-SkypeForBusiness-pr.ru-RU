# <a name="call-quality-dashboard-for-skype-for-business-server-2015"></a>Панель мониторинга качества звонков для Скайп для Business Server 2015
 
**Сводка:** Узнайте о вызов качества панели мониторинга, которая является средством создания отчетов для Скайп для Business Server 2015.
 
Скайп для Business Server 2015 вызов качества панели мониторинга (CQD) — это отчетов уровня качества из базы данных взаимодействия на сервере мониторинга в Скайп для Business Server 2015 и Lync Server 2013. CQD использует Microsoft SQL Server Analysis Services для предоставления сбору статистических данных об использовании и вызова качества сведения, а также фильтрации и сведение набора данных.
