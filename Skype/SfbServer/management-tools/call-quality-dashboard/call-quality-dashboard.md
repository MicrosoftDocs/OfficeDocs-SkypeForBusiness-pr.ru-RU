# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Панель мониторинга качества звонков в Skype для бизнеса Server
 
**Сводка:** Learn about the Call Quality Dashboard, which is a reporting tool for Skype for Business Server.
 
Панель мониторинга качества звонков Skype для бизнеса Server ( CQD) — это уровень отчетов на основе базы данных качества работы на сервере мониторинга в Skype для бизнеса Server и Lync Server 2013. CQD использует Microsoft SQL Server analysis Services для предоставления сводных сведений об использовании и качестве вызовов, а также фильтрации и сводки по набору данных.
