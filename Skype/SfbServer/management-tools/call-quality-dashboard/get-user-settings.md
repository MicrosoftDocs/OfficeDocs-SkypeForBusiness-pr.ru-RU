# <a name="get-user-settings"></a>Получение параметров пользователя
 
**Сводка:** Узнайте о операцию получить параметры пользователя, который является частью службы параметров пользователя. Служба параметров пользователя входит в API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server 2015.
  
Операция получить параметры пользователя является частью служба параметров пользователя в API репозитория для панели мониторинга качества звонков.
  
## <a name="get-user-settings"></a>Получение параметров пользователя

Параметры пользователя Get возвращает список параметров для указанного пользователя.
  

|**Метод**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |https://\<портала\>/QoERepositoryService/репозитория/пользователей / {userId} аудио- и настройка  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI**
  
-  *эффективное* - необязательно. Этот параметр применяется только при использовании по умолчанию идентификатор определенного пользователя. В других случаях он будет игнорироваться. `True`Возвращает параметры действующих пользователей и `false` возвращает только что пользовательских параметров (по умолчанию).
    
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** - ниже приведен пример полезные данные ответа в формате JSON в.
  
```
[{
"userId": 6,
"key": "ShowDescriptions",
"value": "true"
},
{
"userId": 6,
"key": "ShowTimeStamps",
"value": "true"
}]

```
