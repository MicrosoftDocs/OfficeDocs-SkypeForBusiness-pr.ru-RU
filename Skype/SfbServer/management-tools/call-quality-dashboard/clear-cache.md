# <a name="clear-cache"></a>Очистка кэша
 
**Сводка:** Сведения о операции очистки кэша, которая входит в API данных для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.
  
Операция очистки кэша — это часть API данных для панели мониторинга качества звонков.
  
## <a name="clear-cache"></a>Очистка кэша

Операция очистки кэша удаляет кэш на сервере для запросов и данных. Это приведет к сбросу кэша, и мы постараемся получать новые данные из куба QoE для новых запросов.
  

|**Способов**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Поместить  <br/> |/Коедатасервице/клеаркаче\<портала\>HTTPS://  <br/> |HTTP/1.1  <br/> |
   
 **Параметры универсального кода ресурса** (None).
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Запросить текст** -None.
  
 **Response (ответ** ) — ответ включает код состояния HTTP и набор заголовков ответа.
  
 **Код состояния** — успешная операция возвращает код состояния 200 (ОК).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Текст ответа** — None (нет).
