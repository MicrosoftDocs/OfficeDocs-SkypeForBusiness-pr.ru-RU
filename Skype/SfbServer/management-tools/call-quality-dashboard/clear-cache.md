# <a name="clear-cache"></a>Очистить кэш
 
**Сводка:** Сведения об операции очистить кэш, который является частью API данных для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server 2015.
  
Очистить кэш операция является частью API данных для панели мониторинга качества звонков.
  
## <a name="clear-cache"></a>Очистить кэш

Операция очистки кэша удаляет кэш на сервере для запросов и данных. Эта команда сбросит кэша и мы получите новые данные из куба качества взаимодействия впоследствии для новых запросов.
  

|**Метод**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Поместить  <br/> |https://\<портала\>/QoEDataService/ClearCache  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** — нет.
