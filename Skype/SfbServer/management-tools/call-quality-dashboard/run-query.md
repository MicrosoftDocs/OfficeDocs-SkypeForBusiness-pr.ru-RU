# <a name="run-query"></a>Выполнение запроса

**Сводка:** Сведения о выполнении операции запроса, которая входит в API данных для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.

Операция выполнения запроса входит в API данных для панели мониторинга качества звонков.

## <a name="run-query"></a>Выполнение запроса

Операция выполнения запроса предоставляет возможность выполнения запроса для куба на основе указанных размеров, измерений и фильтров, а также возврата данных.


|**Способов**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Поместить  <br/> |/Коедатасервице/рункуери\<портала\>HTTPS://  <br/> |HTTP/1.1  <br/> |

 **Параметры универсального кода ресурса** (None).

 **Заголовки запроса** — без дополнительных заголовков.

 **Тело запроса** — образец полезных данных запроса в JSON. Она включает измерения, фильтры и измерения, необходимые для запроса.

```
{
"Filters": [{
"DataModelName": "[StartDate].[Month]",
"Caption": "July 2013",
"Value": "[2015-03-01T00:00:00]",
"Operand": 0,
"UnionGroup": ""
}],
"Dimensions": [{
"DataModelName": "[StartDate].[Month]"
}],
"Measurements": [{
"DataModelName": "[Measures].[Audio Good Streams Count]"
},
{
"DataModelName": "[Measures].[Audio UnClassified Streams Count]"
},
{
"DataModelName": "[Measures].[Audio Poor Streams Count]"
},
{
"DataModelName": "[Measures].[AudioPoorPercentage]"
}],
"Trend": {
"EnableTrend": true,
"SpanCount": 7,
"TrendDate": "2015-3",
"Type": 0
}
}
```

 *Filters* — список применяемых фильтров, в результате которого результирующий набор данных будет показывать только подмножество данных, которые будут представлять собой.

 *Измерения* — список измерений, которые будут использоваться для статистической обработки данных. Требуется хотя бы одно измерение, но для получения дополнительных уровней вложенных агрегатов может быть указано несколько измерений.

 *Измерения* — список измерений, также называемых фактами, которые представляют собой необходимые метрики, которые должны быть агрегированы на основе указанных вами измерений.

 *Тенденция* — дополнительные инструкции по настройке результирующих данных.

 **Response (ответ** ) — ответ включает код состояния HTTP и набор заголовков ответа.

 **Код состояния** — успешная операция возвращает код состояния 200 (ОК).

 **Заголовки ответа** — без дополнительных заголовков.

 **Тело ответа** — ниже показан пример полезных данных ответа в JSON. Она содержит таблицу данных, содержащую данные, также содержит метаданные, которые показывают время выполнения запроса и сведения о том, находятся ли данные из кэша.

        4.171],
        ["October 2014",
        37017,

        3305,
        8.197],
        ["November 2014",

        6.557],

 *Время выполнения* — общее время, затраченное сервером на возврат данных. Это может вызвать или не включать кэширование.

 *Результат данных* — результат запроса. Это двухмерный массив, который содержит все перестановки членов измерений и каждый элемент, содержащий имена членов измерения, а также агрегированные значения указанных измерений.

 *Результаты из кэша* — для диагностики. Указывает, получен ли результат из кэша или из куба QoE.
