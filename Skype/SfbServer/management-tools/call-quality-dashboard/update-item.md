# <a name="update-item"></a>Обновление элемента
 
**Сводка:** Сведения об операции обновления элемента, которая входит в состав службы номенклатур. Служба item входит в API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.
  
Операция обновления элемента является частью службы элементов в API репозитория для панели мониторинга качества вызова.
  
## <a name="update-item"></a>Обновление элемента

Обновление элемента обновляет определенный элемент в репозитории.
  

|**Способов**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Установка  <br/> |/Коерепоситорисервице/репоситори/итем/{итемид}\<портала\>HTTPS://  <br/> |HTTP/1.1  <br/> |
   
 **Параметры универсального кода ресурса** (None).
  
 **Заголовки запроса** — тип содержимого: Application/JSON.
  
 **Текст запроса** — JSON.
  
Пример полезных данных запроса:
  
```json
{
  content : "{ 'Product' : 'New Product Name'",
  type: "application/json"
}
```

 *Content (содержимое* )  Данные в формате JSON, сохраняемые в виде нового содержимого существующего подэлемента. Технически, в репозитории может храниться любое содержимое любой схемы, но при использовании для панели мониторинга качества звонков требуется либо отчет, либо запрос. *Type (тип* )  Всегда указывайте "Application/JSON" для панели мониторинга качества звонков.
  
 **Response (ответ** ) — ответ включает код состояния HTTP и набор заголовков ответа.
  
 **Код состояния** — успешная операция возвращает код состояния 204 (нет контента). Если указанный идентификатор элемента не найден, возвращается код состояния 404 (не найден).
  
> [!IMPORTANT]
> "Отсутствие содержимого" не является состоянием ошибки. Это означает, что ответ не возвращал ничего в тексте (в отличие от 200 ОК возвращает содержимое в теле). Оно указывает на то, что элемент успешно обновлен. 
  
 **Заголовки ответа** — нет.
  
 **Текст ответа** — None (нет).
