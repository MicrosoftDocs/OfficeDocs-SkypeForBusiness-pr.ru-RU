Data API предоставляет программный доступ к панели мониторинга качества звонков для Скайп для Business Server.

 **Общий доступ к поддержке (CORS) для данных API междоменных ресурсов**
  
Данные API поддерживает общий доступ к междоменных ресурсов (CORS). CORS — это функция HTTP, который позволяет в рамках одного домена под управлением веб-приложения для доступа к ресурсам в другом домене. Веб-браузеры реализации ограничения безопасности, известных как политики единого происхождения [Политики единого происхождения](https://www.w3.org/Security/wiki/Same_Origin_Policy) , не позволяющей веб-страницы из вызова API в другом домене. CORS предоставляет безопасный способ разрешить один домене (источник) вызов API-интерфейсы в другой домен. [Спецификация CORS](https://www.w3.org/TR/cors/) для получения дополнительных сведений см на CORS.
  
 **Включение CORS для данных API**
  
 Ниже приведен фрагмент файла Web.config Data API, отражающая двух доменов, перечисленных в параметрах приложения corsTrustedOrigin. Все запросы, выполненные с скрипты, загруженными из этих серверов являются доверенными Data API.
  
Не забудьте включить точное протокол, имя узла и порта (если они имеются). Не следует поместить какие-либо прямая косая черта (/) знаков в конце. Несколько записей можно указать, разделив их запятыми.
  
```
<configuration>
  <appSettings>
    <add key="corsTrustedOrigin" value="https://<trusted-server>,http://<another-trusted-domain>:8080" /> <!-- Domains which are trusted to get the data -->
    <add key="QoEDataLib.DebugMode" value="True" /> <!-- Setting this to True, allows seeing of the detail logs in status page -->
…  </appSettings>
</configuration>
```
