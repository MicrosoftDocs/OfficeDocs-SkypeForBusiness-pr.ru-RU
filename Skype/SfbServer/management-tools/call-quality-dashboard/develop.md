# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества вызовов для Skype для бизнеса Server
 
**Сводка:** Узнайте, как разрабатывать решения для панели мониторинга качества вызовов. Панель мониторинга качества вызовов — это средство для Skype для бизнеса Server.
  
В этом разделе содержится документация по разработке панели мониторинга качества вызовов (CQD) для Skype для бизнеса Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества вызовов для Skype для бизнеса Server

CQD можно расширить и интегрировать в среду отчетности с помощью настраиваемой разработки. 
  
Доступны два набора веб-API, один для доступа к данным в кубе (с помощью API данных) и один для доступа к метаданным отчета (через API репозиториев). 
  
- [API данных для панели мониторинга качества вызовов (CQD) в Skype для бизнеса Server](data-api.md)
    
- [API репозиториев для панели мониторинга качества вызовов (CQD) в Skype для бизнеса Server](repository-api.md)
    
- [Примеры разработки CQD](cqd-development-samples.md)
