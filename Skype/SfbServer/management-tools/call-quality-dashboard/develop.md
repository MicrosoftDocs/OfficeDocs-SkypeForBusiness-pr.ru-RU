# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества звонков для Skype для бизнеса Server
 
**Сводка:** Научитесь разрабатывать решения для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.
  
Этот раздел содержит документацию по разработке для панели мониторинга качества звонков (CQD) для Skype для бизнеса Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества звонков для Skype для бизнеса Server

CQD можно расширить и интегрировать с вашей средой создания отчетов с помощью настраиваемой разработки. 
  
Доступны два набора API, для доступа к данным в Кубе (посредством API данных) и для доступа к метаданным отчета (через API репозитория). 
  
- [API данных для панели мониторинга качества звонков (CQD) в Skype для бизнеса Server](data-api.md)
    
- [API репозитория для панели мониторинга качества звонков (CQD) в Skype для бизнеса Server](repository-api.md)
    
- [Примеры разработки CQD](cqd-development-samples.md)
