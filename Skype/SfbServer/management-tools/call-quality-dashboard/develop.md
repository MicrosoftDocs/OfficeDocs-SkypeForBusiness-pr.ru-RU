# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Создавать панели мониторинга качества звонков для Скайп для Business Server 2015
 
**Сводка:** Сведения о разработке решений для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server 2015.
  
В этом разделе представлены разработки документации для вызова панели мониторинга качества (CQD) для Скайп Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Создавать панели мониторинга качества звонков для Скайп для Business Server 2015

CQD можно расширять и интегрированной среде отчетов с помощью разработки пользовательских решений. 
  
Доступны два набора веб-интерфейсы API, другая — для доступа к данным в кубе (с помощью API-интерфейсы данных), а другая — для доступа к метаданные отчетов (с помощью API репозитория).
