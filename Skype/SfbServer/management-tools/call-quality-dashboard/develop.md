# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Создавать панели мониторинга качества звонков для Скайп для Business Server
 
**Сводка:** Сведения о разработке решений для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server.
  
В этом разделе представлены документации по разработке решений для вызова панели мониторинга качества (CQD) для Скайп для Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Создавать панели мониторинга качества звонков для Скайп для Business Server

CQD можно расширять и интегрированной среде отчетов с помощью разработки пользовательских решений. 
  
Доступны два набора веб-интерфейсы API, другая — для доступа к данным в кубе (с помощью API-интерфейсы данных), а другая — для доступа к метаданные отчетов (с помощью API репозитория).

- [Репозиторий API для мониторинга качества звонков (CQD) в Скайп для Business Server](repository-api.md)
