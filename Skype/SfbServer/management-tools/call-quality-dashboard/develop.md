# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества звонков для Skype для бизнеса Server
 
**Сводка:** Узнайте, как разрабатывать решения для панели мониторинга качества вызовов. Панель мониторинга качества звонков — это инструмент для Skype для бизнеса Server.
  
В этом разделе содержится документация по разработке панели мониторинга качества звонков (CQD) для Skype для бизнеса Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Разработка панели мониторинга качества звонков для Skype для бизнеса Server

CQD можно расширить и интегрировать в среду отчетов с помощью пользовательской разработки. 
  
Доступны два набора веб-API: один для доступа к данным в кубе (через API данных) и один для доступа к метаданным отчета (через API репозитория). 
  
- [API данных для панели мониторинга качества звонков (CQD) в Skype для бизнеса Server](data-api.md)
    
- [API репозитория для панели мониторинга качества звонков (CQD) в Skype для бизнеса Server](repository-api.md)
    
- [Примеры разработки CQD](cqd-development-samples.md)
