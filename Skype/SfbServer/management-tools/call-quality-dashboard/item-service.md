# <a name="item-service-for-call-quality-dashboard-cqd"></a>Служба элементов для панели мониторинга качества вызовов (CQD)
 
**Сводка:** Узнайте о службе Item Service, которая входит в API репозиториев для панели мониторинга качества вызовов. Панель мониторинга качества вызовов — это средство для Skype для бизнеса Server.
  
Служба элементов является частью API репозиториев для панели мониторинга качества вызовов.
  
## <a name="item-service"></a>Служба элемента

API репозитория предлагает простую службу управления контентом, известная как служба элементов, которая может использоваться для хранения любого контента, определяемого приложениями для пользователей. 
  
Содержимое системы принадлежит пользователю системы и передается всем пользователям с доступом только для чтения. Выделенные пользовательские содержимого принадлежат обычным пользователям, и только владельцы могут изменять или удалять их, но все пользователи по-прежнему имеют доступ только для чтения к ним.
  
> [!NOTE]
> Эта документация API охватывает операции API репозитория только для чтения. 
  
Панель мониторинга качества вызовов сохраняет отчеты и запросы в качестве элементов в базе данных репозиториев. Элемент может иметь необязательные элементы, а панель мониторинга качества вызовов организует отчеты и запросы в иерархической структуре с помощью функции sub-Items.
  
Служба элементов включает в себя следующие понятия:
  
- **Item** — основной элемент репозитория. Каждый элемент принадлежит точно одному пользователю.
    
- **Sub-Item** — основная организационная механика репозитория. Элемент может иметь ноль, один или несколько подчиненных элементов.
    
- **Предки элементов** — список элементов, начиная с самого верхнего элемента, который является элементом пользователя по умолчанию, что приводит к заданного элемента.
    
- **Содержимое элемента** — содержимое, определенное для приложения, храняное в Items. Панель мониторинга качества вызовов сохраняет представления отчетов и запросов JSON в контенте.
