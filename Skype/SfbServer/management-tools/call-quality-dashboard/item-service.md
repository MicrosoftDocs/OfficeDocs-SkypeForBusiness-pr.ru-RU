# <a name="item-service-for-call-quality-dashboard-cqd"></a>Служба элементов для панели мониторинга качества вызовов (CQD)
 
**Сводка:** Узнайте о службе item, которая является частью API репозитория для панели мониторинга качества вызовов. Панель мониторинга качества звонков — это инструмент для Skype для бизнеса Server.
  
Служба элементов является частью API репозитория для панели мониторинга качества вызовов.
  
## <a name="item-service"></a>Служба элемента

API репозитория предлагает простую службу управления контентом, которая называется службой элементов, которая может использоваться для хранения любого контента, определенного приложением, для пользователей. 
  
Содержимое системы принадлежит системным пользователям и совместное для всех пользователей с доступом только для чтения. Содержимое выделенного пользователя принадлежит обычным пользователям, и только владельцы могут изменять или удалять их, но все пользователи по-прежнему имеют доступ только для чтения.
  
> [!NOTE]
> Эта документация по API охватывает операции API репозитория только для чтения. 
  
Панель мониторинга качества вызовов сохраняет отчеты и запросы в качестве элементов в базе данных репозитория. Элемент может иметь необязательные подустанавли, а панель мониторинга качества вызовов упорядожит отчеты и запросы в иерархической структуре с помощью функции в субстантов.
  
Служба элементов включает следующие понятия:
  
- **Item** — базовый элемент репозитория. Каждый элемент принадлежит только одному пользователю.
    
- **Sub-Item** — базовая организационная механика репозитория. Элемент может иметь ноль, один или несколько подчиненных элементов.
    
- **"Предки элементов"** — список элементов, начиная с самого верхнего элемента, который является элементом пользователя по умолчанию, что приводит к заданным элементам.
    
- **Контент элементов** — контент, хранимый в элементах для конкретного приложения. Панель мониторинга качества вызовов сохраняет представления отчетов и запросов в содержимом в JSON.
