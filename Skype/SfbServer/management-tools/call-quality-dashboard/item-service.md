# <a name="item-service-for-call-quality-dashboard-cqd"></a>Служба элемента для панели мониторинга качества звонков (CQD)
 
**Сводка:** Сведения о службе элемента, который является частью API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server 2015.
  
Служба элемент является частью API репозитория для панели мониторинга качества звонков.
  
## <a name="item-service"></a>Служба элемента

API репозитория предлагает службы простое управление содержимым, известных как службы элемента, который может использоваться для хранения содержимого определенного приложения для пользователей. 
  
Компоненты системы владеет пользователь системы и общие для всех пользователей с доступом только для чтения. Содержимое выделенного пользователя принадлежат обычных пользователей только владельцы сайтов могут изменять или удалять их, а все пользователи по-прежнему имеют доступ только для чтения к ним.
  
> [!NOTE]
> В этой документации API охватывает только для чтения operations API репозитория. 
  
Панель мониторинга качества звонков сохраняет отчеты и запросы как элементы в базе данных репозитория. Элемент может содержать необязательные вложенные элементы и панель мониторинга качества звонков собирает отчеты и запросы в иерархической структуры, с помощью функции дочерних элементов.
  
Элемент Служба включает в себя следующие понятия:
  
- **Элемент** - основных элементов репозитория. Каждый элемент принадлежит только один пользователь.
    
- **Дочерний элемент** — основной организационной механизм репозитория. Элемент может иметь нулевую, одну или несколько подчиненных элементов.
    
- **Элемент предков** - списка элементов "," Приступая к работе с элемент верхнего уровня, который используется по умолчанию элемента пользователя, приводя к данного элемента.
    
- **Содержимое элемента** - содержимого конкретного приложения, хранимых в элементах. Панель мониторинга качества звонков сохраняет JSON представлений отчетов и запросов в содержимое.
