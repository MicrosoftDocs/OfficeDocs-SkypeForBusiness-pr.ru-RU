# <a name="item-service-for-call-quality-dashboard-cqd"></a>Служба элементов для панели мониторинга качества звонков (CQD)
 
**Сводка:** Сведения о службе элементов, которая входит в API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.
  
Служба item входит в API репозитория для панели мониторинга качества звонков.
  
## <a name="item-service"></a>Служба элемента

API репозитория — это простая служба управления содержимым, называемая "служба элемента", которая может использоваться для хранения любого содержимого, определенного приложением, для пользователей. 
  
Содержимое системы принадлежит пользователю системы и предоставляется всем пользователям, у которых есть доступ только для чтения. Выделенные пользователи принадлежат обычным пользователям, а только владельцы могут изменять или удалять их, но все пользователи по-прежнему имеют доступ к ним только для чтения.
  
> [!NOTE]
> Эта документация по API охватывает операции, предназначенные только для чтения, для API репозитория. 
  
Панель мониторинга качества звонка сохраняет отчеты и запросы как элементы в базе данных репозитория. Элемент может иметь необязательные дочерние элементы, а панель мониторинга качества вызова упорядочивает отчеты и запросы в иерархической структуре с помощью функции вложенных элементов.
  
Служба номенклатуры включает следующие основные понятия:
  
- **Item** — базовый элемент репозитория. Каждый элемент принадлежит только одному пользователю.
    
- **Дочерний элемент** — базовый стратегический механизм репозитория. Элемент может иметь ноль, один или несколько подчиненных элементов.
    
- **Родительский** элемент — список элементов, начиная с самого верхнего элемента, который является элементом по умолчанию для пользователя, что приводит к заданному элементу.
    
- **Содержимое элемента** — содержимое, зависящее от приложения, которое хранится в элементах. Информационная панель "качество звонка" позволяет экономить представления отчетов и запросов в формате JSON.
