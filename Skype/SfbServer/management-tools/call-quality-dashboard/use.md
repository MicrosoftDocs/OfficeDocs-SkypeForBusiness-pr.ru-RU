## <a name="call-quality-dashboard-user-guide"></a><span data-ttu-id="d37e7-110">Руководство пользователя по панели мониторинга качества звонков</span><span class="sxs-lookup"><span data-stu-id="d37e7-110">Call Quality Dashboard User Guide</span></span>

<span data-ttu-id="d37e7-p104">Панель мониторинга качества звонков (CQD) представляет собой веб-портал для быстрого формирования и систематизации отчетов на основе данных о качестве взаимодействия (QoE). В панели мониторинга качества звонков развернут куб SSAS, обеспечивающий сводное представление информации из базы данных показателей качества взаимодействия. Предусмотрена возможность прямого подключения Excel к кубу; при этом портал оптимизирован под различные рабочие процессы, предусматривающие обработку данных о качестве взаимодействия. К ним относится кэширование данных отчетов для быстрого доступа, прямые ссылки на страницы отчетов для обмена информацией и ее публикации, упрощенное редактирование и создание отчетов, а также редактируемые метаданные для описания отчетов. Панель мониторинга качества звонков также предоставляет доступ к веб-интерфейсам API, обеспечивающий возможность программного обращения к данным куба для их применения в пользовательских панелях мониторинга. </span><span class="sxs-lookup"><span data-stu-id="d37e7-p104">The Call Quality Dashboard (CQD) is a web portal for quickly creating and organizing reports based on Quality of Experience (QoE) data. CQD deploys a SSAS cube to aggregate the data in the QoE Metrics database. This enables users to create and modify reports and do investigations in real-time. While it is possible to use Excel to connect directly to the cube, the portal is optimized for several workflows involving QoE data. This data includes caching of report data for fast access, deep links to report pages for information sharing and publishing, streamlined report editing and creation, and editable metadata for report descriptions. Additionally, CQD exposes web APIs that give users programmatic access to the cube data for use in custom dashboards.</span></span>
