# <a name="get-item"></a>Получение элемента
 
**Сводка:** Узнайте о операцию получения элемента, который является частью службы элемента. Служба элемент является частью API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server.
  
Операции получения элемента является частью службы элемента в API репозитория для панели мониторинга качества звонков.
  
## <a name="get-item"></a>Получение элемента

Получение элемента возвращает конкретный элемент в репозитории.
  
|**Метод**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |https://\<портала\>/QoERepositoryService/репозитория/элемент / {itemId}  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК). Если элемент с указанным Идентификатором не найден, возвращает код состояния 404 (не найден).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** - ниже приведен пример полезные данные ответа в формате JSON в.
  
```
{
"itemId": 1652,
"userId": 0,
"content": "{\"Title\":\"All Audio Streams\",...}",
"type": "application/json",
"subItemIds": [1653, 1710]
}
```

 *идентификатор элемента* - идентификатор элемента.
  
 *идентификатор пользователя* — идентификатор пользователя, которому принадлежит этот элемент.
  
 *контент* - содержимого конкретного приложения.
  
 *Тип* — тип контента. В этом поле задается с помощью приложений.
  
 *subItemIds* - идентификаторы вложенные элементы, при их наличии. Это short-circuit операции получение дочерних элементов для сохранения звонка. Приложения также могут получать те же данные, с помощью операции получение дочерних элементов.
