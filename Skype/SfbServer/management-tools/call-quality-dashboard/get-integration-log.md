# <a name="get-integration-log"></a>Получение журнала интеграции
 
**Сводка:** Узнайте о операции получения интеграции журнала, которая является частью API данных для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server.
  
Операция получения журнала интеграции является частью API данных для панели мониторинга качества звонков
  
## <a name="get-integration-log"></a>Получение журнала интеграции

Получение журнала интеграции операция возвращает список записей журнала, описывающий действия в кубе качества взаимодействия обработки.
  
Эта операция отключена по умолчанию в целях безопасности. При отключении возвращает пустую строку. Чтобы включить эту операцию, администраторы должны настроить файл web.config для данных API размещения веб-приложения.
  

|Метод|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |https://\<портала\>/QoEDataService/IntegrationLog  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** - ниже приведен образец структуры записей журнала.
  
```
[
{"LogCategory":"<category>","LogTime":"2015-03-18T10:28:29.10","LogDescription":"<log description>"}
]
```
