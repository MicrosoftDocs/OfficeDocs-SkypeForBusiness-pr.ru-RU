# <a name="get-sub-items"></a>Получение дочерних элементов
 
**Сводка:** Узнайте о операцию получение дочерних элементов, который является частью службы элемента. Служба элемент является частью API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Скайп для Business Server 2015.
  
Получение дочерних элементов операции является частью службы элемента в API репозитория для панели мониторинга качества звонков.
  
## <a name="get-sub-items"></a>Получение дочерних элементов

Получите вложенные элементы возвращает конкретный элемент вложенные элементы.
  

|**Метод**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |https://\<портала\>/QoERepositoryService/репозитория/элемент / {itemId} / subitem  <br/> |HTTP/1.1  <br/> |
   
 **Параметры URI** — нет.
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Текст запроса** — нет.
  
 **Ответа** - ответ включает код состояния HTTP и набор заголовков ответов.
  
 **Код состояния** - успешные операции возвращает код состояния 200 (ОК). Если указанный пользователь с Идентификатором не найден, он возвращает код состояния 404 (не найден).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** - ниже приведен пример полезные данные ответа в формате JSON в.
  
> [!NOTE]
> Возвращает массив объектов элемента. 
  
```
[{
"itemId": 1653,
"userId": 0,
"type": "application/json"
},
{
"itemId": 1710,
"userId": 0,
"type": "json"
}]

```

Объект Item, возвращенные операцией вложенные элементы содержит только следующие три поля. 
  
 *идентификатор элемента* - идентификатор элемента.
  
 *идентификатор пользователя* — идентификатор пользователя, которому принадлежит этот элемент.
  
 *Тип* — тип контента. В этом поле задается с помощью приложений.
  
> [!NOTE]
>  `Content`и `subItems` не включены в ответ для сокращения объема данных, передаваемых по сети.
