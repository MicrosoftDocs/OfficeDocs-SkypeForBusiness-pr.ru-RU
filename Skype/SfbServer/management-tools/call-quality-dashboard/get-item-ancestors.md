# <a name="get-item-ancestors"></a>Получить родительских элементов
 
**Сводка:** Изучите операцию Get для предков элемента, которая является частью службы элемента. Служба item входит в API репозитория для панели мониторинга качества звонков. Панель мониторинга качества звонков — это средство для Skype для бизнеса Server.
  
Операция Get-предков элемента является частью службы элементов в API репозитория для панели мониторинга качества вызова.
  
## <a name="get-item-ancestors"></a>Получить родительских элементов

Получение предков элемента возвращает определенные предки элементов из репозитория.
  

|**Способов**|**URI запроса**|**Версия HTTP**|
|:-----|:-----|:-----|
|Получить  <br/> |/Коерепоситорисервице/репоситори/итеманцесторс/{итемид}\<портала\>HTTPS://  <br/> |HTTP/1.1  <br/> |
   
 **Параметры универсального кода ресурса** (None).
  
 **Заголовки запроса** — без дополнительных заголовков.
  
 **Запросить текст** -None.
  
 **Response (ответ** ) — ответ включает код состояния HTTP и набор заголовков ответа.
  
 **Код состояния** — успешная операция возвращает код состояния 200 (ОК). Если указанный идентификатор пользователя не найден, он возвращает код состояния 404 (не найден).
  
 **Заголовки ответа** — без дополнительных заголовков.
  
 **Тело ответа** — ниже показан пример полезных данных ответа в JSON.
  
```
[{
"item1": 1653,
"item2": 0,
"item3": "Audio Streams Monthly Trend"
},
{
"item1": 1652,
"item2": 1,
"item3": "All Audio Streams"
}]
```

 *Item1* — идентификатор элемента.
  
 *Элем2* — глубина — расстояние от элемента. 0 — непосредственный родительский элемент.
  
 *Item3* — название элемента.
