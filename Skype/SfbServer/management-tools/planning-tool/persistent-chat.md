# <a name="persistent-chat-planning-tool"></a>Persistent Chat (Planning Tool)
 
Сервер сохраняемого чата позволяет принимать участие в многокомпонентных беседах, которые сохраняются с течением времени.
  
Подробнее о том, как подготовить среду для работы с сохраняемым сервером чата, можно найти [в разделе Планирование сохраняемого сервера чата в Skype для бизнеса Server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
