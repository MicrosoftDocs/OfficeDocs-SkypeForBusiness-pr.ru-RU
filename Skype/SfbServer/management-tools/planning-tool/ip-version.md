# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
У вас есть три варианта настройки IP-адресов в Skype для бизнеса Server 2015. Вы можете настроить Skype для бизнеса Server 2015, чтобы он поддерживал только IP версии 4 (IPv4), только IP версии 6 (IPv6) или сочетание обеих (называемых двойных стеков).
  
Дополнительные сведения о выборе инфраструктуры IP можно найти в разделе [планирование IPv6 в Skype для бизнеса](../../plan-your-deployment/network-requirements/ipv6.md).
