# <a name="federation-planning-tool"></a>Федерация (средство планирования)
 
Пограничные серверы можно настроить так, чтобы разрешить внутренним и внешним пользователям получать доступ к контактам в партнерских организациях или службах.
  
 Федерация работает так же, как и для Microsoft Lync 2013. Дополнительные сведения о подготовке среды для Федерации можно найти в следующих разделах:
  
- [Планирование интеграции Lync Server 2013 и Office Communications Server](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [Планирование подключений к общедоступным службам обмена мгновенными сообщениями в Lync Server 2013](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [Планирование Федерации XMPP (Extensible Messaging and Presence Protocol) в Lync Server 2013](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
