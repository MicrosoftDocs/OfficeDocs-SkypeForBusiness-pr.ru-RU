# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Функция мобильности в Skype для бизнеса Server 2015 поддерживает функции Skype для бизнеса на мобильных устройствах.
  
Функциональность мобильности та же, что и в Lync Server 2013. Дополнительные сведения о подготовке среды для мобильной работы см. в [подплане мобильности Skype для бизнеса Server.](../../plan-your-deployment/mobility.md)
