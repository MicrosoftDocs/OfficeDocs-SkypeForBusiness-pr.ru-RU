# <a name="mobility-planning-tool"></a>Средство планирования мобильности
 
Функция мобильности в Skype для бизнеса Server 2015 г. поддерживает Skype для бизнеса на мобильных устройствах.
  
Функциональность мобильности такая же, как и в Lync Server 2013. Дополнительные сведения о подготовке среды для мобильности см. в этой [Skype для бизнеса Server.](../../plan-your-deployment/mobility.md)
