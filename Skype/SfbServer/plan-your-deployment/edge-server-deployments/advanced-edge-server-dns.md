## <a name="split-brain-dns"></a><span data-ttu-id="52291-142">Разделенная DNS</span><span class="sxs-lookup"><span data-stu-id="52291-142">Split-brain DNS</span></span>

<span data-ttu-id="52291-p108">Разделенная DNS — конфигурация DNS, в которой две зоны DNS имеют одно пространство имен.
