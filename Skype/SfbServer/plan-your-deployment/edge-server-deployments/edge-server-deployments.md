# <a name="plan-for-edge-server-deployments-in-skype-for-business-server-2015"></a>Планирование развертываний пограничного сервера в Skype для бизнеса Server 2015
 
**Краткое содержание.** Планирование среды пограничного сервера Skype для бизнеса Server 2015. Этот раздел содержит начальные сведения о принципах работы пограничных серверов, а также систематизированные ссылки на разделы с более подробной информацией.
  
При наличии Скайп для среды Business Server 2015, работа также во внутренней сети, следующим шагом для вас может быть представлена на пограничный сервер или пограничный пул в среде. Эта роль может быть важной, если требуется, чтобы службы, предоставляемые Скайп для 2015 Business Server для использования с людьми за пределами внутренней сети. Эти потенциально могут включать следующее:
  
- Удаленные пользователи: сотрудники, временно отсутствующие на рабочем месте, в том числе находящиеся в пути.
    
- Федеративные пользователи: Партнерские организации сотрудников.
    
- Пользователи мобильных устройств.
    
- Потенциальные заказчики, партнеры и анонимные пользователи, которых требуется приглашать на собрания и презентации.
    
Доступ внешних пользователей, которая предоставляют пограничных серверов, разрешить все это происходит. Внутренние пользователи смогут Оцените следующие службы, которые размещаются в вашей Скайп Business Server 2015 развертывания:
  
- Обмен мгновенными Сообщениями и сведениями о присутствии для обмена данными: полномочные внешние пользователи могут присоединиться к в мгновенные сообщения и конференциях. Их можно получить сведения о присутствии для других пользователей (получение информации о них сведений о присутствии слишком). Не сможет выполнить многосторонних конференциях, если вы используете общедоступный поставщик обмена мгновенными Сообщениями, который является строго peer-to-peer связи. Но протокола SIP и XMPP, поддерживаются.
    
- Аудио/видео (A / V) конференц-связи: полномочные внешние пользователи могут принимать участие в вашей Скайп Business Server 2015 аудио и видео в конференциях.
    
- Веб-конференции: вашей полномочные внешние пользователи могут принимать участие в вашей Скайп для конференций бизнеса. Если вы хотите, можно также включить участия для удаленных пользователей, федеративных пользователей и анонимных пользователей. Открытый обмен мгновенными Сообщениями пользователи не могут участвовать в конференциях. Также вы можете эти пользователи могут участвовать в приложения и рабочему столу и даже выступать в качестве организаторам собрания или выступающие.
    
Поддерживается доступа с мобильных устройств, как и корпоративной голосовой связи. Можно приглашать внешних пользователей на собрания, где требуется их присутствие; при необходимости можно предоставить разрешение участвовать в собрании даже анонимным пользователям.
  
Если в организации требуются такие службы, для их развертывания рекомендуется спланировать среду пограничных серверов. Ниже приведен список разделов, содержащих дополнительные сведения.
