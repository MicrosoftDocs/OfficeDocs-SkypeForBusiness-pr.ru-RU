# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Требования к среде Skype для бизнеса 2015 г.
 
**Сводка:** Наброски тем для системных требований и сетевых или инфраструктурных требований для Skype для бизнеса Server 2015 г. Если вам необходимо установить или настроить что-либо в качестве обязательного условия для установки и развертывания Skype для бизнеса Server 2015 г., он будет здесь.
  
Это посадочная страница для статей о требованиях к развертыванию Skype для бизнеса Server 2015 г. Это включает как требования к серверу, так и более общие условия для подготовки, включая доменную и сетевую структуру, а также сертификаты.
  
## <a name="whats-in-this-topic"></a>Что в этом разделе

В двух статьях в разделе требования к среде описано, что необходимо иметь и/или настроить, прежде чем вы сможете успешно развернуть или обновить Skype для бизнеса Server 2015 г. в серверной среде организации.
  
### <a name="server-requirements"></a>Требования к серверу

[Требования к серверу на Skype для бизнеса Server 2015](server-requirements.md) г. — это то, что необходимо получить на отдельных серверах, которые вы собираетесь использовать в развертывании.
  
### <a name="environmental-requirements"></a>Требования к среде

Требования к окружающей среде [Skype для бизнеса Server 2015](environmental-requirements.md) г. — это широкий спектр вещей, которые необходимо быть готовыми в инфраструктуре перед развертыванием Skype для бизнеса Server 2015 г.
