# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server"></a>Определение требований к экстренным вызовам в Skype для бизнеса Server
 
Суммирует действия, необходимые для включения E9-1-1 в Skype для бизнеса Server Корпоративная голосовая связь в зависимости от того, есть ли у вас поставщик услуг E9-1-1 магистрали SIP или шлюз ELIN.
  
Прежде чем приступить к развертыванию Skype для бизнеса Server E9-1-1, необходимо сначала ответить на вопросы, которые подробно описаны в следующих разделах. Необходимое планирование зависит от типа решения E9-1-1, которое планируется разворачивать — канал SIP для подключения к поставщику услуг E9-1-1  или шлюз ELIN.
