# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server-2015"></a>Определение требований для экстренных вызовов в Skype для бизнеса Server 2015

Прежде чем начать Скайп для развертывания сервера Business E9-1-1, сначала можно ответить на вопросы, описанные в следующих разделах. Необходимое планирование зависит от типа решения E9-1-1, которое планируется разворачивать — канал SIP для подключения к поставщику услуг E9-1-1 или шлюз ELIN.
