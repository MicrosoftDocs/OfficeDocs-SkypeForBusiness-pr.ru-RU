# <a name="plan-for-the-announcement-application-in-skype-for-business"></a>Планирование приложения "Объявления" в Skype для бизнеса

Планирование приложения объявлений в Skype для бизнеса Server Корпоративная голосовая связь, которое настраивает, что делать с телефонными звонками на ненаписаные телефонные номера в организациях. Включает требования к аудиофайлам.

Приложение объявлений Skype для бизнеса Server позволяет настроить обработку входящих телефонных звонков, если набираемый номер действителен для вашей организации, но не назначен пользователю или телефону. Такие вызовы можно передавать в предварительно заданное назначение (на телефонный номер, SIP URI или на голосовую почту), или воспроизводить вызывающему абоненту звуковое объявление, или выполнять оба действия. Приложение "Оповещение" помогает исключить ситуацию, в которой вызывающий абонент неправильно набирает номер и слышит сигнал "занято", или клиент SIP получает сообщение об ошибке. В этом разделе содержится информация о планировании, относяная к приложению "Извеение".

При развертывании приложения "Объявление" необходимо настроить таблицу ненаписаных номеров, которая определяет действие, которое будет происходить при наборе нена назначенного номера. Таблица ненаписаных номеров содержит диапазоны телефонных номеров, допустимые для организации, и указывает, какое приложение объявлений обрабатывает каждый диапазон. Когда звоняющий набирает номер телефона, допустимый для вашей организации, но не назначен кому-либо, Skype для бизнеса Server ищет номер в таблице маршрутизации ненаписаных номеров, определяет диапазон, в который попадает номер, и перенаносит вызов в приложение "Извеение", указанное для этого диапазона. Приложение "Извещение" отвечает на вызов и воспроизводит звуковое сообщение (если вы настроили его для этого), а затем либо отключает вызов, либо передает его в заранее задаованный пункт назначения, например оператору. Для настройки нескольких звуковых сообщений или передачи назначений можно использовать команды Skype для бизнеса Server Management Shell.

Способ настройки таблица неназначенных номеров зависит от ее последующего использования. Если у вас имеются определенные номера, которые больше не используются, и вы хотите воспроизводить для каждого номера индивидуальные сообщения, то можете ввести эти номера в таблицу неназначенных номеров. Например, если вы изменили номер отдела обслуживания клиентов, вы можете ввести старый номер обслуживания клиентов и сопоставить его с оповещением, в котором указывается новый номер. Если вы хотите воспроизвести сообщение общего характера любому человеку, позвонившему по неназначенному номеру, например номеру уволившегося сотрудника, то можете ввести диапазоны для всех допустимых добавочных номеров в организации. Таблица неназначенных номеров вызывается при наборе звонящим номера, который в данный момент не назначен.

## <a name="deployment-and-requirements"></a>Развертывание и требования

Приложение Tthe Announcement автоматически устанавливается вместе с приложением группы ответа. Приложения "Объявления" и "Группа ответа" являются стандартными компонентами развертывания Корпоративная голосовая связь: при развертывании Корпоративная голосовая связь развертывание обоих этих приложений развертывается автоматически.

### <a name="software-requirements"></a>Требования к программному обеспечению

На всех серверах переднего уровня или серверах Standard Edition, на которые запущено приложение "Объявление", должна быть установлена windows Media Format Runtime для серверов с Windows Server 2008 R2 или Microsoft Media Foundation для серверов под управлением Windows Server 2012 или Windows Server 2012 R2. Для Windows Server 2008 R2 windows Media Format Runtime устанавливается как часть рабочего стола Windows. Windows Media Format Runtime или Microsoft Media Foundation необходимы для файлов Windows Media Audio (WMA), которые приложение announcement воспроизводит для объявлений и музыки.

### <a name="port-requirements"></a>Требования к портам

Приложение "Объявление" **использует порт 5071 для** запросов прослушивания SIP.

> [!NOTE]
> Этот порт указан в параметрах по умолчанию, которые можно изменить с помощью командлета **Set-CsApplicationServer**. Подробные сведения об этом cmdlet см. в документации по skype для бизнеса Server Management Shell.

### <a name="audio-file-requirements"></a>Требования к аудиофайлам

Приложение announcement поддерживает формат файлов Wave (WAV) и формат звуковых файлов Windows Media (WMA) для музыки и объявлений. Требования к аудиофайлам для приложения "Объявление" такие же, как и для приложения группы ответа. Дополнительные сведения см. в статье [Technical Requirements for Response Groups](https://technet.microsoft.com/library/477488bd-124f-437b-9327-732a0d7271ca.aspx).
