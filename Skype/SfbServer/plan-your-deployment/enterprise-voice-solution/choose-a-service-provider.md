# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Выбор поставщика услуг E9-1-1 для Skype для бизнеса Server
 
Выбор поставщика услуг для развертывания E9-1-1 в Skype для бизнеса Server Enterprise Voice.
  
Поставщик услуг E9-1-1 отправляет экстренные вызовы из Skype для бизнеса Server в правильную общедоступную точку безопасного ответа (PSAP) на основе сведений о расположении, содержащихся в вызове. 
  
Для поддержки E9-1-1 в развертывании Skype для бизнеса Server необходимо получить службу маршрутизации E9-1-1 с помощью приложения Lync Open-1-1 поставщика услуг. Выберите поставщика, наиболее точно отвечающего требованиям вашей организации.

Чтобы просмотреть список служб маршрутизации E9 – 1 – 1 для Lync, ознакомьтесь со статьей [службы E – 911, квалифицированные для Lync](https://docs.microsoft.com/skypeforbusiness/lync-cert/e-911-service-providers).
