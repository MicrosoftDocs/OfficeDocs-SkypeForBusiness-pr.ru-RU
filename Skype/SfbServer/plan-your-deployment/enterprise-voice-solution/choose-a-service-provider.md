# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Выберите поставщика услуг E9-1-1 для Skype для бизнеса Server
 
Выбор поставщика услуг для развертывания E9-1-1 в Skype для бизнеса Server Корпоративная голосовая связь.
  
Поставщик услуг E9-1-1 маршруты экстренных вызовов, исходя из Skype для бизнеса Server, в правильный пункт ответа на вопросы общественной безопасности (PSAP) на основе сведений о расположении, содержащихся в вызове. 
  
Чтобы поддерживать E9-1-1 в рамках развертывания Skype для бизнеса Server, необходимо получить службу маршрутификации E9-1-1 от квалифицированного поставщика услуг E9-1-1 Lync Open Interoperability Program. Выберите поставщика, наиболее точно отвечающего требованиям вашей организации.

Список служб маршрутификации E9-1-1 для Lync см. в списке [служб E-911, квалифицированных для Lync](../../../SfbPartnerCertification/lync-cert/e-911-service-providers.md).
