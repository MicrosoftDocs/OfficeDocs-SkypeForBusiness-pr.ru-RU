# <a name="plan-for-shared-line-appearance-in-skype-for-business-server-2015"></a>Планирование внешнего вида общих строк в Skype для бизнеса Server 2015
 
В этом разделе вы узнаете, как спланировать общий вид линии (SLA) в накопительном обновлении Skype для бизнеса Server 2015 за ноябрь 2015 г. 
  
Внешний вид общей линии — это функция в Skype для бизнеса для обработки нескольких вызовов на определенный номер, называемый общим номером. SLA может настроить любого пользователя Skype для бизнеса с поддержкой корпоративной голосовой связи в качестве общего номера с несколькими строками для ответа на несколько вызовов. Вызовы фактически не получаются по общему номеру, а перенаададовыются пользователям, которые выступают в качестве делегатов общего номера. Любой из делегатов может получить вызов, а остальные делегаты получают на своем телефоне уведомление о том, кто и какая строка была занята в результате. Количество строк и делегатов можно настроить для общего номера в SLA. Кроме того, дополнительные параметры, такие как BusyOption (что происходит в ситуации, когда все линии заняты) и MissedCallOption (в случае, когда ни один из делегатов не берет вызов), также можно настроить для общего номера.
  
SLA поддерживается только на следующих телефонных устройствах (оно не поддерживается для клиентов Skype для бизнеса на компьютерах, мобильных телефонах и других устройствах): 
  
- Polycom VVX300 с обновлением микропрограмм 5.4.1
    
- Polycom VVX400 с обновлением микропрограммы 5.4.1
    
- Polycom VVX500 с обновлением микропрограммы 5.4.1
    
- Polycom VVX600 с обновлением микропрограммы 5.4.1
    
SLA — это новая функция в Skype для бизнеса Server, накопительное обновление за ноябрь 2015 г. 
  
For information about deploying SLA, see [Deploy Shared Line Appearance in Skype for Business Server 2015.](../../deploy/deploy-enterprise-voice/deploy-shared-line-appearance.md)
  
## <a name="feature-list"></a>Список функций

Настройка группы SLA включает следующие возможности:
  
- Все делегаты в группе могут отвечать на входящие вызовы на один общий номер. Вызовы могут быть основаны на STN или SIP.
    
- Делегаты могут удерживать и перебирать вызовы.
    
- Делегаты могут переводить звонки на номер за пределами группы SLA.
    
- Делегаты могут просмотреть, сколько вызовов в настоящее время находится на общем номере, и просмотреть состояние каждого из этих вызовов.
    
- Можно настроить максимальное количество одновременно звонков для общего номера. Вы также можете настроить, как будут обрабатываться дополнительные вызовы после того, как будет достигнуто это максимальное значение. Лишние вызовы можно отклонить с сигналом "занято", переадлить на другой номер или переадпоновать на голосовую почту.
    
- Вы можете настроить обработку пропущенных звонков (вызовы, которые не были обработаны через определенное время). Если включить голосовую почту для удостоверения группы, пропущенные звонки автоматически будут переходить на голосовую почту. Если голосовая почта для удостоверения группы (общий номер) не включена, можно отклонить пропущенные вызовы с сигналом "занято", переадлиться на другой номер или отключиться.
