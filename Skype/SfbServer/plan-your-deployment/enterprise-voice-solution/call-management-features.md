# <a name="plan-for-call-management-features-in-skype-for-business"></a>Планирование функций управления звонками в Skype для бизнеса

Обзор поддерживаемых функций управления звонками в Skype для бизнеса Server Корпоративная голосовая связь.

Функции управления вызовами в корпоративной голосовой связи управляют тем, каким образом происходит маршрутизация входящих вызовов и ответ на них. Skype для бизнеса Server предоставляет следующие функции управления звонками:

- **Парковка вызовов:** позволяет пользователям голосовой связи временно приостанавлить вызов, а затем выбрать его с того же или другого телефона.

- **Групповой выбор:** позволяет пользователям голосовой почты выбирать звонки, которые звонят другим пользователям голосовой почты, которые назначены группам.

- **Группа ответа:** маршруты входящих звонков группам агентов с помощью сервисных групп или вопросов и ответов интерактивного голосового ответа (IVR).

- **Оповещение** — проигрывает сообщение о вызовах на неназначенный номер, выполняет маршрутизацию вызова в любое другое расположение или выполняет и то, и другое.

Если планируется развертывание корпоративной голосовой связи, то можно указать, какие из этих функций управления вызовами следует реализовать.
