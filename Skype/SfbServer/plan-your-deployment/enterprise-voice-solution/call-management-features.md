# <a name="plan-for-call-management-features-in-skype-for-business"></a>Планирование функций управления вызовами в Skype для бизнеса

Обзор поддерживаемых функций управления вызовами в Skype для бизнеса Server Корпоративная голосовая связь.

Функции управления вызовами в корпоративной голосовой связи управляют тем, каким образом происходит маршрутизация входящих вызовов и ответ на них. Skype для бизнеса Server предоставляет следующие функции управления вызовами:

- **Call Park.** Позволяет пользователям голосовой связи временно припарковать вызов, а затем забрать его с того же или другого телефона.

- **Групповой выбор:** позволяет пользователям голосовой звонков, которые звонят для других пользователей голосовой почты, которым назначены группы вызова.

- **Группа ответа.** Маршруты входящих звонков группам агентов с помощью групп охоты или интерактивных голосовых ответов (IVR) вопросов и ответов.

- **Оповещение** — проигрывает сообщение о вызовах на неназначенный номер, выполняет маршрутизацию вызова в любое другое расположение или выполняет и то, и другое.

Если планируется развертывание корпоративной голосовой связи, то можно указать, какие из этих функций управления вызовами следует реализовать.
