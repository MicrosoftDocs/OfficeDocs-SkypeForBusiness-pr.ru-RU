# <a name="plan-for-call-management-features-in-skype-for-business"></a>Планирование функций управления звонками в Skype для бизнеса

Обзор поддерживаемых функций управления звонками в корпоративной голосовой связи Skype для бизнеса Server.

Возможности управления голосом на корпоративном звонке определяют маршрутизацию и ответ на входящие звонки. Skype для бизнеса Server предоставляет следующие возможности управления звонками:

- **Припаркованный Звонок**: позволяет пользователям голосового ввода временно приостановить звонок, а затем выбрать его на том же или другом телефоне.

- **Отправка групп**: позволяет пользователям голосовой связи получать звонки, которые будут звонить другим пользователям голосовой связи, которые назначены группам отправки звонков.

- **Группа ответа**: Маршрутизация входящих звонков на группы агентов с помощью групп слежения и ответов на вопросы и ответы по интерактивный автоответчик.

- **Информационное сообщение**: воспроизводит сообщение для вызовов, выполненных на не назначенный номер, перенаправляет вызов куда-нибудь еще или выполняет оба действия.

Если планируется развертывание корпоративной голосовой связи, то можно указать, какие из этих функций управления вызовами следует реализовать.
