# <a name="plan-for-group-call-pickup-in-skype-for-business"></a>Планирование для группы отправки вызовов в Скайп для бизнеса
 
Планирование для отправки вызовов группы в Скайп Business Server корпоративной голосовой связи, который позволяет пользователям отвечать на звонки, изначально предназначены для других пользователей.
  
Раскладки вызывать группу Пользователи могут отвечать на входящие звонки для своих коллег из собственные телефоны. Это повышает уровень доступности линии пользователя, позволяя другим пользователям ответить на входящий звонок, набрав номер группы ответа на звонки. При развертывании группы вызова раскладки количество входящих вызовов, которые направляются на голосовую почту значительно уменьшается, который особенно полезен для вызовы от клиентов, внешними по отношению к вашей организации.
  
Функция раскладки вызывать группу предназначена в частности для бизнес-единицы в средах open office. Входящие звонки не отвлекают внимания, так как сигнал подается только по планируемому назначению. При этом другие пользователи, которые слышат сигнал, могут ответить на звонок, просто набрав номер группы. 
  
В средах, где размещение всех пользователей в одном офисном помещении не применяется либо где пользователи, обладающие схожими обязанностями, находятся в разных географических местоположениях, оптимальным решением будет групповой звонок. Основное различие между группы вызова раскладки и групповой звонок является, с помощью группы вызова раскладки входящего звонка звонок выполняется только в место назначения, что всем пользователям по-прежнему можете ответить, набирая номер группы. При групповом звонке сигнал звучит на телефонах всех членов группы и любой пользователь из этой группы может поднять трубку и ответить на звонок. Дополнительное различие между группы вызова раскладки и групповой звонок —, что группа вызова раскладки управлять администратор, через Скайп для Business Server. Групповой звонок конечным пользователям управлять компонента с помощью Скайп для клиента Business. Таким образом, с помощью группы вызова раскладки этот аспект управления вызовами может быть централизованно.
  
Групповой звонок раскладки построена на приложение парковки вызовов. При развертывании группы вызова раскладки Настройка таблицы орбиты парковки вызовов с отдельные диапазоны добавочных номеров, обозначены как номера раскладки групповой звонок. Как и номера орбиты парковки вызовов, номера группы ответа на звонки должны быть виртуальными расширениями, которым не назначен пользователь или телефон. Каждый пул переднего плана, где развертывание группы вызова раскладки может иметь один или несколько диапазонов номеров раскладки групповой звонок. Диапазоны номеров группы должно отличаться глобально через Скайп для развертывания Business Server. 
  
> [!NOTE]
> Диапазоны номеров, которые используются для группы вызова раскладки номеров в таблице орбит парковки вызовов не может быть управляемой или просмотреть с помощью Скайп для панели управления Business Server. Для использования Скайп для консоли Business Server является единственным способом, чтобы увидеть все диапазоны номеров в таблице орбит парковки вызовов. Аналогичным образом единственный способ добавлять, изменять или удаление группы вызова раскладки номеров является использование Скайп для консоли Business Server. 
  
После настройки номеров группы ответа на звонки вы назначаете пользователей группе ответа на звонки. На звонки, которые адресованы любому из пользователей, назначенных группе ответа на звонки, могут ответить другие пользователи. При поступлении звонка для пользователя, назначенного группе ответа на звонки, любой заметивший это пользователь может ответить на данный звонок, вручную набрав номер группы ответа на звонки. Отвечающий на звонок пользователь может и не быть членом данной группы. Когда на звонок отвечает другой пользователь, на исходный вызываемый номер отправляется соответствующее уведомление.
  
> [!NOTE]
> Пользователь может быть членом только одной группы ответа на звонки. 
  
> [!NOTE]
> Несмотря на то, что любой пользователь Скайп для развертывания Business Server можно ответ на звонок участнику раскладки групповой звонок, ответ на звонок пользователя необходимо знать номер раскладки группы правильный вызов для присоединения. 
  
Если пользователь набирает номер группы ответа на звонки, чтобы ответить на звонок, когда в группе звонят несколько телефонов, он переключается на тот звонок, который дольше всего находится в режиме ожидания.
  
Параметры одновременных звонков будут активны для пользователей, имеющих групповой ответ на звонки. То есть вызов, выполненный пользователь, имеющий группы вызова раскладки звонков для всех настроенных назначения, а другой пользователь может ответить на звонок. Исключением из данного правила является ситуация, когда пользователь настраивает одновременные звонки для вызова всех членов группы.
  
Групповой звонок раскладки не может использоваться для ответов на следующие вызовов:
  
- Звонки по частной линии
    
- Звонки от контакта, которому назначен уровень конфиденциальности "Друзья и семья"
    
    > [!TIP]
    > Пользователь, который является участником группы вызова раскладки можно запретить определенные вызовы от полученной через группы вызова раскладки помечая контакт как личный контакт в Скайп для клиента Business. Чтобы сделать это, установите для контакта уровень конфиденциальности "Друзья и семья". Любой входящий звонок от контактов с уровень конфиденциальности равным друзей и семейства нельзя отменить с помощью группы вызова раскладки. 
  
- Видеосоставляющая голосовых звонков и видеозвонков 
    
    > [!NOTE]
    > Если пользователь отвечает на голосовой звонок или видеозвонок, то он получает только звук. Любой из собеседников может расширить звонок, чтобы добавить видеосоставляющую. 
  
- Одновременные звонки, направляемые членам группы групповых звонков
    
- Звонки, направляемые делегату
    
- Звонки, направляемые в группу ответа
    
Следующие типы пользователи не могут участвовать в группы вызова раскладки. То есть они не должны включаться в группе Группа вызова раскладки и нельзя трубку вызовов для пользователей, имеющих раскладки вызывать группу включен.
  
- Пользователи, размещаемые в сети в гибридном развертывании
    
- Пользователи, которые не размещаются на либо Скайп для пула Business Server 2015 или пула Lync Server 2013 с накопительные пакеты обновления для Lync Server 2013: февраля 2013 г. в локальном развертывании
    
Если никто не отвечает на звонок, адресованный члену группы ответа на звонки, выполняется перенаправление этого звонка в соответствии с параметрами клиента. То есть он попадает в голосовую почту или направляется в другое место, указанное в параметрах клиента.
  
## <a name="deployment-and-requirements"></a>Развертывание и требования

Групповой звонок раскладки развертывается автоматически при развертывании корпоративной голосовой связи и приложение парковки вызовов. Включение группы вызова раскладки, настроив таблицу орбиты парковки вызовов с отдельные диапазоны номеров обозначены как числа звонков раскладки группы, а затем, назначая пользователям вызывать раскладки групп и включение пользователей для отправки вызовов группы.
  
## <a name="clients-supported-for-group-call-pickup"></a>Клиенты, поддерживаемые для группы вызова раскладки

Любой из следующих клиентов можно использовать для отвечать на звонки членам группы вызова раскладки:
  
- Skype для бизнеса
    
- Lync 2013
    
- Lync 2010
    
- Lync Phone Edition
    
> [!NOTE]
> Пользователи могут использовать любой из этих клиентов будут отвечать на звонки членам группы вызова раскладки, но пользователи должны быть размещены на Скайп для пула сервера или пула Lync Server 2013 с накопительные пакеты обновления для Lync Server 2013: февраля 2013 г. 
  
Следующие клиенты и устройства не поддерживается для комплектации копирование вызовы раскладки вызывать группу элементов:
  
- Lync Mobile
    
- Приложения Lync для Windows 8 и Windows RT
    
- Lync для iPad
    
- Аналоговые телефоны
    
- Телефоны с номерами телефонной сети общего пользования (ТСОП).
