# <a name="components-required-for-enterprise-voice-in-skype-for-business-server"></a>Компоненты, необходимые для Корпоративная голосовая связь в Skype для бизнеса Server
 
Сводка компонентов Корпоративная голосовая связь в Skype для бизнеса Server.
  
Чтобы развернуть Корпоративная голосовая связь, в топологии требуются следующие компоненты. 
  
- Один или несколько серверов-посредников, которые переводят сигналы и, в некоторых конфигурациях, средства массовой информации между внутренней Skype для бизнеса Server, Корпоративная голосовая связь инфраструктурой и шлюзом телефонной сети общего присоединения (PSTN) или магистралью протокола инициации сеансов (SIP). Серверы-посредники являются наиболее важным компонентом Корпоративная голосовая связь развертывания. Дополнительные сведения см. в [компоненте Mediation Server в Skype для бизнеса Server.](mediation-server.md)
    
    Серверы-посредники могут быть расположены с помощью серверов переднего конца или установлены в качестве автономных серверов.
    
- Компоненты подключения к PSTN, которые могут включать магистрали SIP или шлюзы PSTN. Дополнительные сведения см. в [Skype для бизнеса Server.](pstn-connectivity.md)
    
- Edge Servers, которая позволяет использовать Корпоративная голосовая связь пользователями, когда они находятся вне брандмауэра организации. 
    
    Служба Access Edge предоставляет SIP-сигналы для вызовов Skype для бизнеса пользователей, которые находятся вне брандмауэра организации. Служба A/V Edge обеспечивает обход NAT и брандмауэров. Абонент, использующий унифицированный коммуникационный клиент (UC) за пределами корпоративного брандмауэра, применяет службу для выполнения индивидуальных вызовов или конференц-вызовов.
    
    Служба проверки подлинности аудио и видео связана с пограничной службой аудио- и видеоданных и предоставляет услуги проверки подлинности. Внешним пользователям, которые пытаются подключиться к пограничной службе аудио- и видеоданных, требуется маркер проверки подлинности, предоставляемый службой проверки подлинности аудио и видео, для прохождения вызовов.
    
- Кроме того, некоторые Корпоративная голосовая связь работают на серверах переднего входа. Сведения об этих компонентах см. в материале [Front End Server VoIP компоненты](front-end-server-voip.md) для Skype для бизнеса Server
