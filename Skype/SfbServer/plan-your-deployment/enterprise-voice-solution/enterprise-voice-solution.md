# <a name="plan-your-enterprise-voice-solution-in-skype-for-business-server"></a>Планирование решения корпоративной голосовой связи в Skype для бизнеса Server
 
**Сводка:** Сведения о том, как планировать единое голосовое решение и телекоммуникации в Skype для бизнеса Server.
  
В Skype для бизнеса Server предусмотрено два локальных варианта корпоративной голосовой связи: Корпоративная голосовая связь и Звонок через работу — для интеграции телефонной системы Организации с развертыванием Skype для бизнеса Server и создания решения для единой системы обмена сообщениями. Оба варианта позволяют пользователям использовать свой клиент Skype для бизнеса для инициирования голосовых вызовов с другими пользователями как в Организации, так и за ее пределами.
  
Вы также можете использовать как корпоративную голосовую связь, так и вызовы с помощью совместной работы.
  
- Корпоративная голосовая связь является самым полным предложением голосовых вызовов. Это решение для обеспечения голосовой связи по протоколу VoIP, обеспечивающее обширный набор функций, в том числе многофункциональную интеграцию с Outlook и Exchange, а также множество мощных функций, таких как группы ответа, парковки вызовов, звонки по группам, группового ответа на звонки и расширенная поддержка экстренных E9-1-1. При использовании корпоративной голосовой связи пользователи используют звуковое устройство, например гарнитуру с компьютером, или телефон с поддержкой VoIP, а не обычный телефон PSTN или УАТС.
    
- С помощью функции "позвонить с рабочего" вы можете интегрировать решение Skype для бизнеса с существующими телефонными системами УАТС. Пользователь, для которого настроена функция "позвонить с рабочего" может щелкнуть в Skype для бизнеса, чтобы позвонить другому пользователю в вашем развертывании или на внешнем пользователе. Вызов завершается с помощью телефона УАТС пользователя. Кроме того, эти пользователи могут щелкнуть Skype для бизнеса, чтобы присоединиться к конференциям.
    
    Развертывание вызовов с помощью Work также является преимуществом для пользователей, которые по-прежнему имеют телефоны УАТС, но переходят на корпоративную голосовую связь, так как она позволяет этим пользователям использовать клиент Skype для бизнеса для управления своими телефонными звонками.
    
     Call Via work предоставляет более ограниченный набор голосовых функций Санентерприсе Voice. Например, он не поддерживает делегирование, командный звонок, группу ответа или расширенный E9 – 1 – 1.
    
Конечно, некоторые пользователи могут использовать корпоративную голосовую связь, а другие пользователи — телефоны УАТС. Кроме того, при переходе на полноценное решение корпоративной голосовой связи вы можете разрешить пользователям, у которых есть телефоны УАТС для корпоративной голосовой связи. Эти пользователи могут использовать телефон УАТС на рабочем месте, а также использовать корпоративную голосовую связь для размещения или получения звонков с помощью VoIP, пока в других местах. Если эти пользователи работают с рабочими звонками, они могут использовать свой клиент Skype для бизнеса, чтобы управлять телефонным каналом УАТС на рабочем месте.
  
> [!NOTE]
> Удаленное управление звонками это функция, предлагаемая в предыдущих версиях Lync Server, позволяющая пользователям использовать свой клиент Skype для бизнеса для совершения и приема телефонных звонков на телефонах УАТС. Удаленное управление звонками не поддерживается для пользователей, размещенных на серверах Skype для бизнеса Server, но поддерживается для пользователей с клиентами Skype для бизнеса, которые по-прежнему размещены на серверах Lync Server 2013.
