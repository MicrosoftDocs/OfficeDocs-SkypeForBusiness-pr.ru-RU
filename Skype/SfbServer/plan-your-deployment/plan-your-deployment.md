# <a name="plan-for-your-skype-for-business-server-2015-deployment"></a>Планирование развертывания Skype для бизнеса Server 2015
 
**Сводка:** Изучите эти ссылки, чтобы узнать, как разрабатывать концептуальные, логические и физические макеты для настройки функций, серверов и конфигураций Skype для бизнеса Server 2015.
  
Skype для бизнеса Server предлагает решения для обмена мгновенными сообщениями, присутствия, видеоконференций и телефонии, которые могут поддерживать требования совместной работы на уровне предприятия. В этом разделе данная статья содержит ссылки на сведения о планировании успешного развертывания Skype для бизнеса Server. 
  
Содержимое этих статей постоянно обновляется с учетом новых знаний и отзывов пользователей. Эта страница будет часто обновляться, поэтому часто проверяйте ее.
