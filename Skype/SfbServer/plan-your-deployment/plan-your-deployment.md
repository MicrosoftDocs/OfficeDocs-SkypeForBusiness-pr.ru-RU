# <a name="plan-for-your-skype-for-business-server-2015-deployment"></a>Планирование развертывания Skype для бизнеса Server 2015
 
**Сводка:** В этой статье рассказывается о том, как разрабатывать концептуальные, логические и физические структуры для настройки функций, серверов и топологий Skype для бизнеса Server 2015.
  
Skype для бизнеса Server предлагает решения для обмена мгновенными сообщениями, предоставления информации о присутствии, конференций, видео и телефонии, которые могут поддерживать требования по совместной работе корпоративного уровня. В этой статье приведены ссылки на сведения о том, как планировать успешное развертывание Skype для бизнеса Server. 
  
Эти статьи регулярно обновляются с учетом появления новых сведений и отзывов пользователей. Данная страница будет постоянно обновляться, поэтому заходите почаще.
