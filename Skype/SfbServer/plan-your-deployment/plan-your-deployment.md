# <a name="plan-for-your-skype-for-business-server-2015-deployment"></a>Планирование развертывания Skype для бизнеса Server 2015
 
**Сводка:** В этих ссылках вы узнаете, как разрабатывать концептуальные, логические и физические структуры для настройки функций, серверов и топологий Skype для бизнеса Server 2015.
  
Skype для бизнеса Server обеспечивает обмен мгновенными сообщениями, сведения о присутствии, конференц-связи, видео и телефонные решения, которые могут поддерживать требования к совместной работе на уровне предприятия. В этом разделе приведены ссылки на сведения о планировании успешного развертывания Skype для бизнеса Server. 
  
Содержимое этих статей постоянно обновляется с учетом новых знаний и отзывов пользователей. Эта страница будет часто обновляться, поэтому регулярно проверяйте ее.
