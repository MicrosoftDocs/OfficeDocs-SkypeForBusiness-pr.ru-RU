# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств

**Сводка.** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса

Современные условия работы предполагают высокую мобильность сотрудников. Им приходится взаимодействовать и работать совместно в корпоративных офисах, в региональных филиалах, дома и в пути. Skype для бизнеса Server полностью отвечает таким потребностям благодаря коллекции клиентских интерфейсов, которые можно развернуть для пользователей организации. Грамотное планирование позволит предоставить всем сотрудникам все, что им нужно, и сделать Skype для бизнеса доступным для них вне зависимости от того, где они находятся.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса поддерживает клиенты нескольких типов, включая устанавливаемое на компьютере клиентское программное обеспечение, веб-клиенты и клиенты для мобильных устройств. Основные клиенты представлены в этом разделе, а подробное описание всех поддерживаемых клиентов см. в статье [Сравнение функций классических клиентов для Skype для бизнеса Server 2015](desktop-feature-comparison.md) или [Сравнение функций классических клиентов для Skype для бизнеса Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Если прежде вы использовали сочетание разных клиентов Lync, обратите внимание, что существуют неподдерживаемые [устаревшие клиенты](clients-and-devices.md#Legacy), которые несовместимы со Skype для бизнеса Server 2019. Клиенты регулярно обновляются, поэтому время от времени проверяйте информацию о последних версиях клиентов в данном разделе.

### <a name="skype-for-business-2019"></a>Skype для бизнеса 2019

Skype для бизнеса (2019) — это рекомендуемый полнофункциональный клиент для Skype для бизнеса Server 2015 и 2019. Подробнее о новых возможностях [в Skype для бизнеса: Подпишитесь на последние обновления](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Поддержка функций клиентов подробно описана в статье [Сравнение функций классических клиентов для Skype для бизнеса](desktop-feature-comparison.md), а пользовательская документация приведена в статье [Справка по Skype для бизнеса](https://support.office.com/en-US/Skype-for-business). Этот клиент устанавливается, когда пользователь устанавливает Office 365.

Доступен также бесплатный базовый клиент, поддерживающий меньшее количество функций. Обе версии можно скачать на странице [Загрузка Skype для бизнеса для всех устройств](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Отличия полного и базового клиентов описаны в разделе [Ограничения базового клиента](desktop-feature-comparison.md#Full-Basic).

### <a name="skype-for-business-2016"></a>Skype для бизнеса 2016

Skype для бизнеса 2016 — это полнофункциональный клиент для Skype для бизнеса Server 2015 или 2019. Описание новых возможностей см. в статье [Новые возможности Skype для бизнеса 2016](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781). Поддержка функций клиентов подробно описана в статье [Сравнение функций классических клиентов для Skype для бизнеса](desktop-feature-comparison.md), а пользовательская документация приведена в статье [Справка по Skype для бизнеса](https://support.office.com/en-US/Skype-for-business). Этот клиент устанавливается, когда пользователь устанавливает Office 365.

Доступен также бесплатный базовый клиент, поддерживающий меньшее количество функций. Обе версии можно скачать на странице [Загрузка Skype для бизнеса для всех устройств](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Отличия полного и базового клиентов описаны в разделе [Ограничения базового клиента](desktop-feature-comparison.md#Full-Basic).

### <a name="skype-for-business-2015"></a>Skype для бизнеса 2015

Skype для бизнеса 2015 является полнофункциональным клиентом для Skype для бизнеса Server версии 2015 или 2019. Пользовательский интерфейс Skype для бизнеса был полностью переработан и теперь содержит новые встроенные функции, такие как монитор звонков, интеграция справочника Skype, смайлики и многое другое. Краткое описание изменений см. в статье [Lync — это Skype для бизнеса. Новые возможности](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). Поддержка функций клиентов подробно описана в статье [Сравнение функций классических клиентов для Skype для бизнеса](desktop-feature-comparison.md), а пользовательская документация приведена в статье [Справка по Skype для бизнеса](https://support.office.com/en-US/Skype-for-business). Этот клиент устанавливается, когда пользователь устанавливает Office 365.

### <a name="skype-for-business-on-mac"></a>Skype для бизнеса на Mac

Клиент [Skype для бизнеса для Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) доступен для скачивания. Ознакомьтесь с [требованиями клиента Skype для бизнеса для Mac](mac-requirements.md) , чтобы ознакомиться с необходимыми условиями.

### <a name="skype-for-business-for-mobile-devices"></a>Skype для бизнеса на мобильных устройствах

Клиенты доступны для Windows Phone, iPhone, iPad и Android. Пользователи могут [загрузить Skype для бизнеса на всех своих устройствах](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Поддержка функций для этих клиентов подробно описана в таблице [Сравнение возможностей мобильных клиентов для Skype для бизнеса](mobile-feature-comparison.md).

> [!NOTE]
> Поддержка МККС (служба Mobility Service) для устаревших мобильных клиентов больше не доступна в Skype для бизнеса Server 2019. На всех текущих мобильных клиентах Skype для бизнеса уже используется веб-API единой системы обмена сообщениями (УКВА) для поддержки мгновенных сообщений, присутствия и контактов. Пользователи с устаревшими клиентами, использующими МККС, должны обновиться до текущего клиента.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Надстройка "собрание по сети" для Skype для бизнеса

Надстройка "собрание по сети" для Skype для бизнеса поддерживает управление собранием из клиента сообщений и совместной работы в Microsoft Outlook в Windows. Надстройка "собрание по сети" для программного обеспечения Skype для бизнеса устанавливается автоматически вместе со Skype для бизнеса.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Приложение Skype для бизнеса Web App и собрания Skype

Если приложение "Skype для бизнеса" не установлено на компьютере пользователя, а пользователь щелкает ссылку на собрание в приглашении на собрание на компьютере с Windows, будет установлена и открыта программа "собрания Skype" или Skype для бизнеса Web App.  Приложение "собрания Skype" — это клиент выбора для участников за пределами вашей организации. (Обратите внимание на то, что на компьютере Mac будут установлены и открыты Skype для бизнеса на Mac). Требования для использования этих клиентов можно найти в разделе [Планирование клиентов собраний (приложение для веб-приложений и собраний)](meetings-clients.md) .


### <a name="skype-for-business-web-scheduler"></a>Веб-планировщик Skype для бизнеса

[Интернет-планировщик Skype для бизнеса](https://sched.lync.com) — это веб-средство планирования и управления собранием для пользователей Skype для бизнеса Online, у которых нет доступа к Microsoft Outlook или в операционной системе, не основанной на Windows. С помощью веб-планировщика Skype для бизнеса пользователи могут создавать новые собрания, изменять существующие собрания и отправлять приглашения с помощью предпочтительной почтовой программы. [Документация по](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) веб-планировщику Skype для бизнеса содержит дополнительные сведения.

### <a name="vdi-plugins"></a>Подключаемые модули VDI

Среда инфраструктуры удаленных рабочих столов (VDI) используется в некоторых организациях, имеющих особые требования к безопасности и соответствию. Использование Skype для бизнеса с полным аудио-и видеоподключением, таким как, требует высокой нагрузки аудио-и видеообработки на клиентском компьютере, подключенном к виртуальному рабочему столу. Доступно дополнительное программное обеспечение для подключения к VDI, которое обрабатывается на локальном компьютере конечного пользователя и уменьшает нагрузку на виртуальный компьютер. Дополнительные сведения об использовании этих подключаемых модулей см. в статье [Plan for Skype for Business in VDI environments](vdi-environments.md).

### <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

Комнаты Microsoft Teams — это новейшее решение для конференций Microsoft, использующее знакомый интерфейс, которое легко развертывается и управляется, благодаря использованию существующего оборудования, например ЖК-панелей, для простоты установки. Комнаты Microsoft Teams используют приложение UWP, работающее под управлением Surface Pro 4 или Surface Pro, в режиме консоли (после развернутого приложения UWP это единственное приложение, которое будет работать на устройстве), и для его реализации требуется собственная учетная запись устройства. Программное обеспечение обновляется как посредством Магазина Windows, так и посредством Центра обновления Windows. Подробнее https://aka.ms/MTRDocs об использовании консолей комнат в развертывании смотрите в этой таблице.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Для MSI и установщиков типа "нажми и работай" доступны установщики клиентов. Стратегия клиента Skype для бизнеса — это может повлиять на ваши варианты, поэтому следует понимать следующие моменты:

- В общем Skype для бизнеса не добавляются новые возможности для ранее выпущенных клиентов

- В общем случае в Skype для бизнеса не планируется отгрузка новых функций в Skype для бизнеса MSI после первоначального выпуска. Усовершенствования MSI между выпусками главным образом будут касаться качества и безопасности.

- Последние и наиболее подходящие клиенты Skype для бизнеса можно найти в установщике "нажми и работай" в Skype для бизнеса 2019.

Вы можете выполнить настроенное развертывание клиента, как описано в разделе [Настройка установки клиента Windows в Skype для бизнеса Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Более подробные сведения о методах установки можно [получить в разделе Развертывание клиентов для Skype для бизнеса Server](../../deploy/deploy-clients/deploy-clients.md) .

### <a name="click-to-run"></a>Технология "нажми и работай"

"Нажми и работай" — это технология Microsoft Streaming и виртуализации, которую можно использовать для установки и обновления продуктов Office, включая Skype для бизнеса. Функции потоковой передачи и виртуализации основаны на технологиях в составе Microsoft Application Virtualization (App-V). Технология "нажми и работай" имеет следующие преимущества:

- потоковая установка набора Office, способствующая сокращению времени установки;

- интегрированные обновления и исправления;

- использование меньшего дискового пространства;

- лицензирование для пользователя: 5 установок на пользователя;

- возможность настройки с помощью редактора XML для установки независимых программ.

Для этого типа установщика может потребоваться использовать [средство развертывания Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) .

Как базовые, так и полные клиентские версии (с выбранным вариантом 32-и 64-разрядных версий) доступны при помощи установщика "нажми и работай", пользователи смогут скачать [Skype для бизнеса на всех устройствах](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI — это более традиционный способ установки, который используется в клиентах Skype для бизнеса 2015 и 2016. Она позволяет вручную устанавливать обновления и пакеты исправлений, использовать корпоративную лицензию и активацию, а также настраивать ее с помощью [средства настройки Office](https://www.microsoft.com/en-us/download/details.aspx?id=49030). Вы можете распространять клиентов, применяя групповые политики, с помощью System Center Configuration Manager или с помощью стороннего средства.
