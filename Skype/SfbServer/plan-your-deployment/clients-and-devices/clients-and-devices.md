# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств

**Сводка:** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса.

Современные сотрудники постоянно находятся в движении. Сотрудники должны общаться и сотрудничать, работая из корпоративного офиса, в региональных расположениях, в домашних офисах или в дороге. Skype для бизнеса Server поддерживает эти потребности с помощью коллекции клиентских интерфейсов, которые можно развернуть для пользователей организации. Вдумчивое планирование гарантирует, что сотрудники получают то, что им нужно, и что Skype для бизнеса доступен им, где бы они ни были.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса Server поддерживает несколько типов клиентов, включая клиентские программы с установленными на компьютере, веб-клиентов и клиентов для мобильных устройств. Основные клиенты представлены в этом разделе, подробный список всех поддерживаемых клиентов см. в статье Сравнение функций клиента desktop для [Skype для бизнеса Server 2015](desktop-feature-comparison.md) или сравнение функций для клиентов Desktop для Skype для бизнеса Server [2019.](../../../SfBServer2019/plan/feature-comparison.md) Если вы ранее использовали сочетание клиентов Lync, обратите внимание, что существуют неподтверченные клиенты [Legacy,](clients-and-devices.md#Legacy) несовместимые со Skype для бизнеса Server 2019. Обновления регулярно происходят, поэтому периодически проверяйте эту тему на предмет получения последних сведений о клиентах.

### <a name="skype-for-business-2019"></a>Skype для бизнеса (2019 г.)

Skype for Business (2019) — рекомендуемый полнокомендный клиент для Skype для бизнеса Server 2015 и 2019. Подробные [описания новых функций см.](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) в описании последних обновлений в Skype для бизнеса. Поддержка клиентских функций подробно описана в сопоставлении функций клиента desktop для [Skype для](desktop-feature-comparison.md)бизнеса, а документация пользователя находится в [справке Skype для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается при установке пользователем Microsoft 365 или Office 365.

Кроме того, доступен бесплатный базовый клиент, поддерживающей меньшее количество функций. Обе версии доступны для скачивания в [Download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Различия между клиентами Full и Basic описаны в разделе [Основные ограничения клиентов.](desktop-feature-comparison.md#Full-Basic)

### <a name="skype-for-business-2016"></a>Skype для бизнеса 2016

Skype for Business 2016 — полно функции клиента Skype для бизнеса Server 2015 или 2019. Описание новых функций см. в описании новых функций в Skype для бизнеса [2016.](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) Поддержка клиентских функций подробно описана в сопоставлении функций клиента desktop для [Skype для](desktop-feature-comparison.md)бизнеса, а документация пользователя находится в [справке Skype для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается при установке Office 365 пользователем.

Кроме того, доступен бесплатный базовый клиент, поддерживающей меньшее количество функций. Обе версии доступны для скачивания в [Download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Различия между клиентами Full и Basic описаны в разделе [Основные ограничения клиентов.](desktop-feature-comparison.md#Full-Basic)

### <a name="skype-for-business-2015"></a>Skype для бизнеса 2015

Skype for Business 2015 — полноразливный клиент Skype для бизнеса Server 2015 или 2019. Пользовательский интерфейс Skype для бизнеса полностью переработан и включает новые интегрированные функции, такие как Call Monitor, интеграция каталогов Skype, смайлики и другие. Сводка изменений см. в обзоре [Lync теперь Skype для бизнеса — см.](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)новые сведения. Поддержка клиентских функций подробно описана в сопоставлении функций клиента desktop для [Skype для](desktop-feature-comparison.md)бизнеса, а документация пользователя находится в [справке Skype для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается при установке Office 365 пользователем.

### <a name="skype-for-business-on-mac"></a>Skype для бизнеса на Mac

Клиент [Skype для бизнеса на Mac](https://www.microsoft.com/download/details.aspx?id=54108) доступен для скачивания. Чтобы [просмотреть необходимые](mac-requirements.md) условия, см. в обзоре требований клиентов Skype для бизнеса на Mac.

### <a name="skype-for-business-for-mobile-devices"></a>Skype для бизнеса для мобильных устройств

Клиенты доступны для Windows Phone, iPhone/iPad и Android. Пользователи могут получать их в [Download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Поддержка функций для этих клиентов подробно описана в сопоставлении функций [мобильных клиентов для Skype для бизнеса.](mobile-feature-comparison.md)

> [!NOTE]
> Поддержка mcX (Mobility Service) для устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019. Все текущие мобильные клиенты Skype для бизнеса уже используют веб-API единой связи (UCWA) для поддержки обмена мгновенными сообщениями, присутствия и контактов. Пользователям с устаревшими клиентами, использующими MCX, потребуется обновиться до текущего клиента.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Надстройка для skype для бизнеса

Надстройка для собраний в Интернете для Skype для бизнеса поддерживает управление собраниями из клиента microsoft Outlook для обмена сообщениями и совместной работы в Windows. Надстройка online meeting для программного обеспечения Skype для бизнеса устанавливается автоматически с помощью Skype для бизнеса.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype для бизнеса Веб-приложение и Skype Meetings App

Если Skype для бизнеса не установлен на компьютере пользователя и пользователь щелкает ссылку на собрание в запросе собрания на компьютере Windows, будет установлено и открыто приложение Skype Meetings App или Skype for Business Web App.  Приложение Skype Meetings — это клиент, который может выбирать участников за пределами организации. (Обратите внимание, что на Mac skype для бизнеса на Mac будет установлен и открыт.) См. [в приложении Plan for Meetings customers (Web App and Meetings App)](meetings-clients.md) для требований к использованию этих клиентов.


### <a name="skype-for-business-web-scheduler"></a>Skype для бизнеса веб-планировщик

[Skype for Business Web Scheduler](https://sched.lync.com) — это веб-средство планирования собраний и управления для пользователей Skype для бизнеса Online, которые не имеют доступа к Microsoft Outlook или находятся в операционной системе, не основанной на Windows. С помощью веб-расписания Skype для бизнеса пользователи могут создавать новые собрания, изменять существующие собрания и отправлять приглашения с помощью предпочитаемой программы электронной почты. Дополнительные сведения приводит [](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) документация Skype для веб-планиров для бизнеса.

### <a name="vdi-plugins"></a>Плагины VDI

Среда виртуальной инфраструктуры настольных компьютеров (VDI) используется в некоторых организациях, где вопросы безопасности и соответствия требованиям являются особенно чувствительными. Использование Skype для бизнеса с полным звуком и видео на подключении, которое требует тяжелых нагрузок обработки звука и видео на клиенте, размещенного на виртуальном рабочем столе. Доступно дополнительное программное обеспечение подключаемого программного обеспечения VDI, которое разгружает обработку локальной машины пользователя и снижает нагрузку на виртуальный рабочий стол. Сведения об использовании этих плагинов см. в материале [Plan for Skype for Business в средах VDI.](vdi-environments.md)

### <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

Microsoft Teams Rooms — это последнее решение microsoft conferencing, которое использует знакомый интерфейс и легко развертывается и управляется, используя существующее оборудование, например ЖК-панели для простоты установки. Microsoft Teams Rooms использует специально построенное приложение UWP, которое работает на Surface Pro 4 или Surface Pro в консольном режиме (после развертывания приложение UWP является единственным приложением, которое будет работать на устройстве), и для его реализации требуется собственная учетная запись устройства. Программное обеспечение обновляется с помощью магазина Windows и обновления Windows. Подробные https://aka.ms/MTRDocs сведения об использовании этих консолей комнаты в развертывании. 

### <a name="skype-for-business-on-surface-hub"></a>Skype для бизнеса на Surface Hub

Microsoft Surface Hub — это устройство, которое предназначено для мозгового штурма, совместной работы и презентаций. Она имеет собственную итерацию клиента Skype для бизнеса, задокументированную в [руководстве администрирования Microsoft Surface Hub.](/surface-hub/)

## <a name="choosing-your-organizations-preferred-client"></a>Выбор предпочтительного клиента организации
<a name="BK_client_choose"> </a>

Если организация приобрела соответствующие лицензии, выберите полный клиент, в противном случае выберите клиент Basic.

Пользователи могут самостоятельно устанавливать клиента из [скачать Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Клиент также устанавливается, когда пользователи устанавливают Microsoft 365 или Office 365 в Windows. Если у некоторых пользователей есть компьютеры Mac, у этих пользователей будет другой набор функций, как описано в предыдущих разделах.

Некоторые функции, доступные в Skype для бизнеса Server 2015, недоступны в Skype для бизнеса Online или Skype для бизнеса Server 2019, см. ограничения учетной записи пользователей Online или Hybrid за [2015](desktop-feature-comparison.md#Online-Hybrid) г., а также ограничения учетной записи пользователей Online или Hybrid на [2019](desktop-feature-comparison.md#Online-Hybrid) г. для конкретных данных. Администраторы Skype для бизнеса в Интернете могут обратиться к [описанию службы Skype для](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) бизнеса online для получения сведений о различных планах, доступных для них.

 Перед развертыванием или обновлением до Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте сравнение функций клиента desktop для [Skype для бизнеса,](desktop-feature-comparison.md) чтобы понять влияние поддержки функций на этих клиентов. Это поможет вам донести изменения до пользователей, побок процесса выкатки и полностью понять преимущества обновления до последнего клиента.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Клиентские установщики доступны как для MSI, так и для установщиков типа click-to-run. Стратегия клиентской поддержки Skype для бизнеса может повлиять на ваш выбор, поэтому следует понимать следующие моменты:

- В целом Skype для бизнеса не добавляет новые функции для ранее выпущенных клиентов

- Как правило, Skype для бизнеса не планирует отправку новых функций в Skype для бизнеса MSI после его первоначального выпуска. Улучшение MSI между выпусками будет в первую очередь качеством и безопасностью по своему характеру.

- Последняя и самая большая возможность работы с клиентом Skype для бизнеса будет найдена в установщике skype для бизнеса 2019.

Можно настроить развертывание клиента, как описано в настройках клиентской установки [Windows в Skype для бизнеса Server.](../../deploy/deploy-clients/customize-windows-client-installation.md) Методы установки более подробно описаны в развертывании [клиентов для Skype для бизнеса Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Щелкните кнопку на запуск

Click-to-run — это технология потоковой передачи и виртуализации Майкрософт, которую можно использовать для установки и обновления продуктов Office, включая Skype для бизнеса. Эти возможности потоковой передачи и виртуализации основаны на технологиях в microsoft Application Virtualization (App-V). Click-to-run имеет следующие преимущества:

- Потоковая установка пакета Office, которая приводит к короткому времени установки

- Slipstreamed Updates and patches

- Занимает меньше места на диске

- Лицензирование на базе пользователей: 5 установок на одного пользователя

- Настраиваемый с помощью редактора XML для установки независимых программ

Для этого типа установщика может потребоваться использовать [средство](https://www.microsoft.com/download/details.aspx?id=49117) развертывания Office.

Базовые и полные клиентские версии (с выбором 32- и 64-битных версий) доступны с помощью установщика мыши и запуска, пользователи могут скачать в [Download Skype для](https://products.office.com/skype-for-business/download-app?tab=tabs-3)бизнеса на всех устройствах.

### <a name="msi"></a>MSI

MSI — это более традиционный метод установки, используемый в клиентах Skype для бизнеса 2015 и 2016 г. Он позволяет вручную устанавливать обновления и исправления, использовать лицензирование и активацию тома, а также настраивается с помощью [средства настройки Office.](https://www.microsoft.com/download/details.aspx?id=49030) Вы можете распространять клиентов, применяя групповые политики, используя Microsoft Endpoint Configuration Manager или с помощью средства сторонних пользователей.
