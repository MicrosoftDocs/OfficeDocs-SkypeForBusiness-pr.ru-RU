# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств

**Сводка.** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса

Современные условия работы предполагают высокую мобильность сотрудников. Им приходится взаимодействовать и работать совместно в корпоративных офисах, в региональных филиалах, дома и в пути. Skype для бизнеса Server полностью отвечает таким потребностям благодаря коллекции клиентских интерфейсов, которые можно развернуть для пользователей организации. Грамотное планирование позволит предоставить всем сотрудникам все, что им нужно, и сделать Skype для бизнеса доступным для них вне зависимости от того, где они находятся.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса поддерживает клиенты нескольких типов, включая устанавливаемое на компьютере клиентское программное обеспечение, веб-клиенты и клиенты для мобильных устройств. Основные клиенты представлены в этом разделе, а подробное описание всех поддерживаемых клиентов см. в статье [Сравнение функций классических клиентов для Skype для бизнеса Server 2015](desktop-feature-comparison.md) или [Сравнение функций классических клиентов для Skype для бизнеса Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Если прежде вы использовали сочетание разных клиентов Lync, обратите внимание, что существуют неподдерживаемые [устаревшие клиенты](clients-and-devices.md#Legacy), которые несовместимы со Skype для бизнеса Server 2019. Клиенты регулярно обновляются, поэтому время от времени проверяйте информацию о последних версиях клиентов в данном разделе.

### <a name="vdi-plugins"></a>Подключаемые модули VDI

Среда инфраструктуры удаленных рабочих столов (VDI) используется в некоторых организациях, имеющих особые требования к безопасности и соответствию. С помощью Скайп для бизнеса с полной аудио- и видеоконференций на подключение так, требует больших нагрузках аудио и видео, обработки на стороне клиента, размещенных на виртуальных рабочих столов. Дополнительное программное обеспечение инфраструктуры виртуальных рабочих СТОЛОВ для подключаемого модуля доступна разгружает обработки на компьютер конечного пользователя, а также снизить нагрузку на виртуальном рабочем столе. Дополнительные сведения об использовании этих подключаемых модулей см. в статье [Plan for Skype for Business in VDI environments](vdi-environments.md).

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Для MSI и установщиков типа "нажми и работай" доступны установщики клиентов. Скайп для бизнес-стратегии sustainment клиент может повлиять на выбором, поэтому необходимо знать следующее:

- В общем Скайп для бизнеса не добавляет новые функции предыдущие клиентов

- В общем Скайп для бизнеса не планируется доставки новых функций в Скайп для бизнеса MSI после первого выпуска. Усовершенствования MSI между выпусками главным образом будут касаться качества и безопасности.

- Новейшие и интересных Скайп для взаимодействия с пользователем предприятия будут находиться в Скайп для бизнеса 2019 click-to-run установщика.

Это можно сделать настраиваемого развертывания клиента, как описано в [Настройка Windows Установка клиента в Скайп для Business Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Способы установки описаны более подробно описываются в [Развертывание клиентов для Скайп для Business Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Технология "нажми и работай"

Click-to-run — это Microsoft потоковой передачи и виртуализации, которые можно использовать для установки и обновления продуктов Office, включая Скайп для бизнеса. Функции потоковой передачи и виртуализации основаны на технологиях в составе Microsoft Application Virtualization (App-V). Технология "нажми и работай" имеет следующие преимущества:

- потоковая установка набора Office, способствующая сокращению времени установки;

- интегрированные обновления и исправления;

- использование меньшего дискового пространства;

- лицензирование для пользователя: 5 установок на пользователя;

- возможность настройки с помощью редактора XML для установки независимых программ.

Можно использовать [Средство развертывания Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) для этого типа установщика.

Как обычный и полный клиентские версии (с выбором 32 - и 64-разрядных версий) доступны с помощью установщика click-to-run, пользователей можно загрузить по [Загрузить Скайп для бизнеса на всех устройствах](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI является более традиционный способ установки, используется на Скайп для бизнеса 2015 и 2016 клиентов. Можно вручную установить обновления и исправления, использования корпоративного лицензирования и активации и можно настраивать с помощью [Центра развертывания Office](https://www.microsoft.com/en-us/download/details.aspx?id=49030). Можно распределить клиентов с помощью групповой политики, с помощью System Center Configuration Manager, или с помощью средства сторонних производителей.
