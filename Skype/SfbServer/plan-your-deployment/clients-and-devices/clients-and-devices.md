# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств

**Сводка:** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса.

Современные сотрудники постоянно находятся в движении. Сотрудники должны общаться и сотрудничать, работая из корпоративного офиса, в региональных расположениях, в домашних офисах или в дороге. Skype для бизнеса Server поддерживает эти потребности с помощью коллекции клиентских интерфейсов, которые можно развернуть для пользователей организации. Вдумчивое планирование гарантирует, что сотрудники получают то, что Skype для бизнеса им доступно, где бы они ни были.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса Server поддерживает несколько типов клиентов, в том числе установленное на компьютере клиентские программы, веб-клиенты и клиенты для мобильных устройств. Основные клиенты представлены в этом разделе, для подробного перечисления всех поддерживаемых клиентов см. сравнение функций клиента desktop за [Skype для бизнеса Server 2015](desktop-feature-comparison.md) г. или сравнение функций клиентов desktop для [Skype для бизнеса Server 2019](../../../SfBServer2019/plan/feature-comparison.md)г. . Если вы ранее использовали сочетание клиентов Lync, обратите внимание, что существуют неподтверченные клиенты [Legacy,](clients-and-devices.md#Legacy) несовместимые с Skype для бизнеса Server 2019 г. Обновления регулярно происходят, поэтому периодически проверяйте эту тему на предмет получения последних сведений о клиентах.

### <a name="skype-for-business-2019"></a>Skype для бизнеса (2019 г.)

Skype для бизнеса (2019) — рекомендуемый полнокомендный клиент для Skype для бизнеса Server 2015 и 2019 годов. Подробные описания новых функций [см.](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) в Skype для бизнеса обновлениях в этой области. Поддержка функций клиента подробно описана в сопоставлении функций клиента [desktop](desktop-feature-comparison.md)для Skype для бизнеса, а документация пользователя находится [Skype для бизнеса справке](https://support.office.com/Skype-for-business). Этот клиент включается, когда пользователь Microsoft 365 или Office 365.

Кроме того, доступен бесплатный базовый клиент, поддерживающей меньшее количество функций. Обе версии доступны для скачивания в [download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Различия между клиентами Full и Basic описаны в разделе [Основные ограничения клиентов.](desktop-feature-comparison.md#Full-Basic)

Описание новых функций см. в [Skype для бизнеса 2016](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) г. Этот клиент включается при установке Office 365.

### <a name="skype-for-business-2015"></a>Skype для бизнеса 2015

Skype для бизнеса 2015 г. является полноправным клиентом на Skype для бизнеса Server 2015 или 2019 гг. Пользовательский Skype для бизнеса полностью переработан и включает новые интегрированные функции, такие как Call Monitor, Skype интеграция каталогов, смайлики и другие. Сводка изменений см. в Skype для бизнеса [Lync.](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612) Поддержка функций клиента подробно описана в сопоставлении функций клиента [desktop](desktop-feature-comparison.md)для Skype для бизнеса, а документация пользователя находится [Skype для бизнеса справке](https://support.office.com/Skype-for-business). Этот клиент включается при установке Office 365.

Клиент [Skype для бизнеса Mac](https://www.microsoft.com/download/details.aspx?id=54108) доступен для скачивания. См. [Skype для бизнеса требования к клиентам Mac для](mac-requirements.md) проверки необходимых условий.

### <a name="skype-for-business-for-mobile-devices"></a>Skype для бизнеса для мобильных устройств

Клиенты доступны для Windows Phone, iPhone/iPad и Android. Пользователи могут получить их в [download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Поддержка функций для этих клиентов подробно описана в сопоставлении функций мобильных клиентов [для Skype для бизнеса.](mobile-feature-comparison.md)

> [!NOTE]
> Поддержка mcX (Mobility Service) для устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019 г. Все современные Skype для бизнеса мобильные клиенты уже используют веб-API единой связи (UCWA) для поддержки обмена мгновенными сообщениями,присутствия и контактов. Пользователям с устаревшими клиентами, использующими MCX, потребуется обновиться до текущего клиента.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Надстройка для собраний в интернете для Skype для бизнеса

Надстройка online meeting for Skype для бизнеса поддерживает управление собраниями из microsoft Outlook обмена сообщениями и совместной работы на Windows. Надстройка online Meeting для Skype для бизнеса автоматически устанавливается с помощью Skype для бизнеса.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype для бизнеса Web App и Skype собраний

Если Skype для бизнеса не установлен на компьютере пользователя и пользователь щелкает ссылку на собрание в запросе собрания на компьютере Windows, приложение Skype meetings или Skype для бизнеса Web App будет установлено и открыто.  Skype Приложение Meetings — это клиент, который может выбирать участников за пределами организации. (Обратите внимание, что на Mac Skype для бизнеса на Mac будут установлены и открыты.) См. [в приложении Plan for Meetings customers (Web App and Meetings App)](meetings-clients.md) для требований к использованию этих клиентов.


### <a name="skype-for-business-web-scheduler"></a>веб-планировщик Skype для бизнеса

[веб-планировщик Skype для бизнеса](https://sched.lync.com) является веб-средством планирования собраний и управления для пользователей Skype для бизнеса Online, которые не имеют доступа к Microsoft Outlook или которые находятся в операционной системе, не основанной на Windows. С веб-планировщик Skype для бизнеса пользователи могут создавать новые собрания, изменять существующие собрания и отправлять приглашения с помощью предпочитаемой программы электронной почты. веб-планировщик Skype для бизнеса [документация](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) содержит дополнительные сведения.

### <a name="vdi-plugins"></a>Плагины VDI

Среда инфраструктура виртуальных рабочих столов (VDI) используется в некоторых организациях, где вопросы безопасности и соответствия требованиям являются особенно чувствительными. Использование Skype для бизнеса с полным звуком и видео на подключении, которое требует тяжелых нагрузок аудио- и видеообработки на клиенте, размещенного на виртуальном рабочем столе. Доступно дополнительное программное обеспечение подключаемого программного обеспечения VDI, которое разгружает обработку локальной машины пользователя и снижает нагрузку на виртуальный рабочий стол. Подробные сведения об использовании этих плагинов см. в Skype для бизнеса в [средах VDI.](vdi-environments.md)

### <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

Комнаты Microsoft Teams является последним решением microsoft conferencing, которое использует знакомый интерфейс и легко развертывается и управляется, используя существующее оборудование, например ЖК-панели для простоты установки. Комнаты Microsoft Teams использует специально построенное приложение UWP, которое работает на Surface Pro 4 или Surface Pro в консольном режиме (после развертывания приложение UWP является единственным приложением, которое будет работать на устройстве), и для его реализации требуется собственная учетная запись устройства. Программное обеспечение обновляется с помощью Windows и Windows обновления. Подробные https://aka.ms/MTRDocs сведения об использовании этих консолей комнаты в развертывании. 

### <a name="skype-for-business-on-surface-hub"></a>Skype для бизнеса на Surface Hub

Microsoft Surface Hub — это устройство производительности "все в одном", предназначенное для мозгового штурма, совместной работы и презентаций. Она имеет собственную итерацию клиента Skype для бизнеса, задокументированную в [руководстве Microsoft Surface Hub администрирования.](/surface-hub/)

Если организация приобрела соответствующие лицензии, выберите полный клиент, в противном случае выберите клиент Basic.

Пользователи могут установить клиента для себя из [download Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Клиент также устанавливается при установке Microsoft 365 или Office 365 на Windows. Если у некоторых пользователей есть компьютеры Mac, у этих пользователей будет другой набор функций, как описано в предыдущих разделах.

Некоторые функции, доступные Skype для бизнеса Server 2015 г., недоступны в Skype для бизнеса Online или Skype для бизнеса Server 2019 г., см. ограничения учетной записи пользователей Online или Hybrid за [2015](desktop-feature-comparison.md#Online-Hybrid) г., а также ограничения учетных записей пользователей Online или Hybrid на [2019](desktop-feature-comparison.md#Online-Hybrid) г. для конкретных данных. Skype для бизнеса Администраторы сети могут обратиться к Skype для бизнеса [online Service Description](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) для получения сведений о различных планах, доступных для них.

 Перед развертыванием или обновлением Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте сравнение функций клиента desktop для [Skype для бизнеса,](desktop-feature-comparison.md) чтобы понять влияние поддержки функций на этих клиентов. Это поможет вам донести изменения до пользователей, побок процесса выкатки и полностью понять преимущества обновления до последнего клиента.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Клиентские установщики доступны как для MSI, так и для установщиков типа click-to-run. Стратегия Skype для бизнеса клиента может повлиять на ваш выбор, поэтому следует понимать следующие моменты:

- В общем Skype для бизнеса не добавляет новые функции к ранее выпущенным клиентам

- В общем Skype для бизнеса не планирует отправку новых функций в Skype для бизнеса MSI после его первоначального выпуска. Улучшение MSI между выпусками будет в первую очередь качеством и безопасностью по своему характеру.

- Последняя и самая Skype для бизнеса клиентская возможность будет найдена в Skype для бизнеса 2019 года.

Вы можете сделать настраиваемую развертывание клиента, как описано в настройках Windows в [Skype для бизнеса Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Методы установки более подробно описаны в [deploy customers for Skype для бизнеса Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Щелкните кнопку на запуск

Click-to-run — это технология потоковой передачи и виртуализации Майкрософт, которую можно использовать для установки и обновления Office, включая Skype для бизнеса. Эти возможности потоковой передачи и виртуализации основаны на технологиях в microsoft Application Virtualization (App-V). Click-to-run имеет следующие преимущества:

- Потоковая установка пакета Office, что приводит к короткому времени установки

- Slipstreamed Updates and patches

- Занимает меньше места на диске

- Лицензирование на базе пользователей: 5 установок на одного пользователя

- Настраиваемый с помощью редактора XML для установки независимых программ

Может потребоваться использовать средство [Office развертывания](https://www.microsoft.com/download/details.aspx?id=49117) для этого типа установщика.

Базовые и полные клиентские версии (с выбором 32- и 64-битных версий) доступны с помощью установщика мыши для запуска, пользователи могут скачать в [Download Skype для бизнеса](https://products.office.com/skype-for-business/download-app?tab=tabs-3)на всех устройствах.

### <a name="msi"></a>MSI

MSI — это более традиционный метод установки, используемый в Skype для бизнеса 2015 и 2016 годах. Она позволяет вручную устанавливать обновления и исправления, использовать лицензирование и активацию тома и настраивается с помощью средства Office [настройки.](https://www.microsoft.com/download/details.aspx?id=49030) Вы можете распространять клиентов, применяя групповые политики, используя Microsoft Endpoint Configuration Manager или с помощью средства сторонних пользователей.
