# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств

**Сводка:** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса.

Современные сотрудники непрерывно переходят. Сотрудники должны взаимодействовать с корпоративным офисом, а также совместно работать с ними, в регионах, в домашних офисах или в дороге. Skype для бизнеса Server поддерживает эти требования через набор клиентских интерфейсов, которые можно развернуть для пользователей организации. Продуманное планирование гарантирует, что сотрудники получат необходимые им действия и что Skype для бизнеса доступен им там, где бы они ни находились.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса Server поддерживает несколько типов клиентов, включая клиентское программное обеспечение, веб-клиенты и клиенты для мобильных устройств. В этом разделе представлены основные клиенты, в которых приведен подробный список всех поддерживаемых клиентов, а также сравнение [компонентов клиента для настольных ПК для Skype для бизнеса server 2015](desktop-feature-comparison.md) или [функции клиента для настольных ПК для Skype для бизнеса Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Если ранее использовалось сочетание клиентов Lync, обратите внимание на то, что существуют неподдерживаемые [устаревшие клиенты](clients-and-devices.md#Legacy) , несовместимые с Skype для бизнеса Server 2019. Обновления происходят регулярно, поэтому периодически проверяйте этот раздел на наличие последних сведений о клиенте.

Skype для бизнеса (2019) — это рекомендуемый полнофункциональный клиент для Skype для бизнеса Server 2015 и 2019. В статье [последние обновления в Skype для бизнеса](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) вы найдете описание новых функций. Поддержка функций клиентов описана в статье [Сравнение функций клиентского клиента для Skype для бизнеса](desktop-feature-comparison.md), а Документация пользователя — в [справке Skype для бизнеса](https://support.office.com/Skype-for-business). Этот клиент включается, когда пользователь устанавливает Microsoft 365 или Office 365.

Кроме того, доступен бесплатный базовый клиент, поддерживающий меньше компонентов. Обе версии доступны для скачивания при [загрузке Skype для бизнеса на всех устройствах](https://products.office.com/skype-for-business/download-app?tab=tabs-3). Различия между полным и базовым клиентами описаны в разделе [Основные ограничения клиента](desktop-feature-comparison.md#Full-Basic) .

Поддержка функций для этих клиентов описана в этой таблице: [Сравнение функций мобильных клиентов для Skype для бизнеса](mobile-feature-comparison.md).

> MCX (служба Mobility Service) поддержка устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019. Все текущие мобильные клиенты Skype для бизнеса уже используют веб-API объединенных коммуникаций (UCWA), чтобы поддерживать обмен мгновенными сообщениями, сведения о присутствии и контакты. Пользователям прежних версий клиентов, использующих MCX, необходимо выполнить обновление до текущего клиента.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Надстройка "собрание по сети" для Skype для бизнеса

Надстройка "собрание по сети" для Skype для бизнеса поддерживает управление собраниями в клиенте обмена сообщениями и совместной работы Microsoft Outlook в Windows. Надстройка "собрание по сети" для приложения Skype для бизнеса устанавливается автоматически вместе с Skype для бизнеса.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Приложение Skype для бизнеса Web App и собрания Skype

Приложение для собраний Skype — это клиент, который подходит для участников за пререшениями вашей организации. (Обратите внимание на то, что на компьютере Mac будет установлен и открыт Skype для бизнеса на Mac). ### <a name="skype-for-business-web-scheduler"></a>Веб-планировщик Skype для бизнеса

[Веб-планировщик Skype для бизнеса](https://sched.lync.com) — это веб-средство планирования собраний и управления для пользователей Skype для бизнеса Online, у которых нет доступа к Microsoft Outlook или в операционной системе, не основанной на Windows. С помощью веб-планировщика Skype для бизнеса пользователи могут создавать новые собрания, изменять существующие собрания и отправлять приглашения с помощью предпочтительной программы электронной почты. [Документация по](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) веб-планировщику Skype для бизнеса содержит дополнительные сведения.

### <a name="vdi-plugins"></a>Подключаемые модули VDI

Среда инфраструктуры виртуальных рабочих столов (VDI) используется в некоторых организациях, где вопросы безопасности и соответствия особенно конфиденциальны. Использование Skype для бизнеса с полным звуком и видео для подключения, например, требует высокой загрузки аудио-и видеообработки на клиенте, размещенном на виртуальном рабочем столе. Доступно дополнительное программное обеспечение для подключаемого модуля VDI, которое размещает обработку на локальном компьютере конечного пользователя и сокращает нагрузку на виртуальный рабочий стол. Сведения об использовании этих подключаемых модулей вы найдете [в статье Plan for Skype for Business в средах VDI](vdi-environments.md) .

### <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

Комнаты Microsoft Teams — это новейшее решение для конференц-связи Майкрософт, использующее привычный интерфейс, которое легко развертывается и управляется, используя имеющееся оборудование, например ЖК-панели, для простоты установки. Комнаты Microsoft Teams используют приложение UWP, работающее под управлением Surface Pro 4 или Surface Pro, в режиме консоли (после развертывания приложения UWP это единственное приложение, которое будет выполняться на устройстве), и для его реализации требуется собственная учетная запись устройства. Программное обеспечение обновляется с помощью Магазина Windows и центра обновления Windows. https://aka.ms/MTRDocsСведения об использовании консолей комнаты в развертывании приведены в статье.

Некоторые функции, доступные в Skype для бизнеса Server 2015, недоступны в Skype для бизнеса Online или Skype для бизнеса Server 2019, приведены в статье [ограничения учетной записи в Интернете или гибридной учетной записи пользователя для 2015](desktop-feature-comparison.md#Online-Hybrid) или [Online или гибридных ограничений учетных записей пользователей в целях 2019](desktop-feature-comparison.md#Online-Hybrid) . Администраторам Skype для бизнеса Online можно обратиться к [описанию службы Skype для бизнеса Online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) для получения сведений о различных планах, доступных для них.

 Перед развертыванием или обновлением до Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте [Сравнение функций клиентского клиента для Skype для бизнеса](desktop-feature-comparison.md) , чтобы узнать, какая функция поддерживает влияние на эти клиенты. Это поможет вам сообщить об изменениях пользователям, оценить процесс развертывания и полностью оценить преимущества обновления до последней версии клиента.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Установщики клиентов доступны для программ установки MSI и "нажми и работай". Стратегия обеспечения поддержки клиентов в Skype для бизнеса может повлиять на выбранные варианты, поэтому следует ознакомиться со следующими аспектами:

- В общем случае Skype для бизнеса не добавляет новые функции к ранее выпущенным клиентам

- В общем случае Skype для бизнеса не планирует отгрузить новые функции в пакет MSI Skype для бизнеса после первоначального выпуска. Усовершенствования MSI между выпусками в первую очередь будут иметь качество и безопасность.

- Новейший и наиболее производительный клиентский интерфейс Skype для бизнеса можно найти в установщике Skype для бизнеса 2019 нажми и работай.

Вы можете выполнить настраиваемое развертывание клиента, как описано в разделе [Настройка установки клиента Windows в Skype для бизнеса Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Более подробное описание способов установки: [развертывание клиентов для Skype для бизнеса Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>"Нажми и работай"

Технология "нажми и работай" — это технология потоковой передачи и виртуализации Майкрософт, которую можно использовать для установки и обновления продуктов Office, в том числе Skype для бизнеса. Эти возможности для потоковой передачи и виртуализации основаны на технологиях в Microsoft Application Virtualization (App-V). "Нажми и работай" имеет следующие преимущества:

- Потоковая Установка пакета Office, которая приводит к сокращению времени установки

- Интегрированные обновления и исправления

- Занимает меньше места на диске

- Базовая лицензия пользователя: 5 установок на пользователя

- Возможность настройки с помощью редактора XML для установки независимых программ

Для этого типа установщика может потребоваться использовать [средство развертывания Office](https://www.microsoft.com/download/details.aspx?id=49117) .

Как базовые, так и полные клиентские версии (с выбором 32 и 64-разрядной версии) доступны с помощью установщика "нажми и работай", пользователи могут скачать их при [загрузке Skype для бизнеса на всех устройствах](https://products.office.com/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI — это более традиционный метод установки, используемый на клиентах Skype для бизнеса 2015 и 2016. Она позволяет вручную устанавливать обновления и исправления, использовать корпоративную лицензию и активацию, а также настраивать с помощью центра [развертывания Office](https://www.microsoft.com/download/details.aspx?id=49030). Для распространения клиентов можно применить групповые политики, с помощью диспетчера конфигураций конечных точек Майкрософт или с помощью стороннего средства.
