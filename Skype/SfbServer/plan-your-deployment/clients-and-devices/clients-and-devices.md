# <a name="plan-for-clients-and-devices"></a>Планирование для клиентов и устройств
 
**Сводка:** Обзор поддерживаемых клиентов и приложений для Скайп для бизнеса.
  
Современные условия работы предполагают высокую мобильность сотрудников. Им приходится взаимодействовать и работать совместно в корпоративных офисах, в региональных филиалах, дома и в пути. Скайп для Business Server 2015 поддерживает следующие потребности в коллекции клиентских интерфейсов, которые можно развернуть для пользователей организации. Планирование продуманной гарантирует, что сотрудники получают необходимые условия и Скайп для бизнеса доступен им там, где они ни находились. 
  
## <a name="available-clients"></a>Доступные клиенты

Скайп для Business Server 2015 поддерживает несколько типов клиентов, включая компьютере установлены клиентского программного обеспечения, веб-клиенты и клиенты для мобильных устройств. Каждый из них кратко описан в этом разделе. Если вы ранее использовали сочетание клиентов Lync, обратите внимание на то, что существует неподдерживаемыми [прежних версий клиентов](clients-and-devices.md#Legacy) , несовместимы с Скайп для бизнеса. Клиенты регулярно обновляются, поэтому время от времени проверяйте информацию о последних версиях клиентов в данном разделе.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Скайп для бизнеса веб-приложения и приложения Скайп собраний

Скайп для бизнеса Web App — это веб-конференции браузера подключаемого модуля клиент для Скайп для собраний Business Server 2015. Участники собраний имеют доступ ко всем функциональным возможностям совместной работы и общего доступа, а выступающие — к элементам управления собранием. Этот клиент позволяет Гости из других организаций, у которых нет регулярного клиента собрания, созданный с вашей Скайп для коммерческих пользователей. 
  
Если на компьютере пользователя не установлен Скайп для бизнеса и пользователь щелкает ссылку собрания в приглашении на собрание, Скайп для бизнеса Web App будет установлен и откройте. Вы также можете настроить страницу присоединения к собранию, чтобы позволить пользователям присоединяться к собраниям с помощью предыдущих версий клиентов. Скайп для бизнеса Web App — это клиент выбора для участников за пределами вашей организации. С Скайп для бизнеса Web App не локального клиентской установки не требуется, несмотря на то, что аудио, видео и общего доступа к функции требуют установки подключаемого модуля при первом использовании. В разделе [Планирование для клиентов собрания (веб-приложения и приложения собраний)](meetings-clients.md) требования для использования этого клиента.
  
Приложение собраний Скайп является прямое приложением Windows 10, только с помощью браузера во время установки и работает как встроенное приложение Windows 10. Требуется настройка конкретного сервера для включения рассылки с сервера переднего плана. Как Скайп Business Web App этот клиент также позволяет Гости из других организаций, у которых нет регулярного клиента собрания, созданный с вашей Скайп для коммерческих пользователей. 
  
### <a name="skype-for-business-web-scheduler"></a>Скайп для бизнеса веб-планировщика

Скайп для бизнеса веб-планировщик — это веб-средство планирования и управления для пользователей, не имеющих доступ к Microsoft Outlook, а также в операционной системе, не входящий в систему Windows на собрания. С Скайп для бизнеса веб-планировщик пользователей можно создавать новые собрания, изменение существующих собраний и отправлять приглашения, с помощью программы их предпочитаемый электронной почты.
  
Документация [Скайп для бизнеса веб-планировщик](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) Дополнительные сведения.
  
### <a name="vdi-plugins"></a>Подключаемые модули VDI

Среда инфраструктуры удаленных рабочих столов (VDI) используется в некоторых организациях, имеющих особые требования к безопасности и соответствию. С помощью Скайп для бизнеса с полной аудио- и видеоконференций на подключение так, требует больших нагрузках аудио и видео, обработки на стороне клиента, размещенных на виртуальных рабочих столов. Дополнительное программное обеспечение инфраструктуры виртуальных рабочих СТОЛОВ для подключаемого модуля доступна разгружает обработки на компьютер конечного пользователя, а также снизить нагрузку на виртуальном рабочем столе. В разделе [Планирование Скайп для бизнеса в среде VDI](vdi-environments.md) подробные сведения об использовании этих подключаемые модули.
  
### <a name="skype-room-systems-v2"></a>Системы комнат Skype версии 2

V2 Скайп комнаты систем — корпорации Майкрософт последние конференц-связи решения, которое использует знакомых Скайп для интерфейса бизнеса и легко развернуть и управляемые и использование существующего оборудования, например Жидкокристаллических панелей для упрощения установки. Скайп помещения систем v2 использует специальные UWP приложения, работающего на 4 Surface Pro или Surface Pro в режиме консоли (после развертывания приложения UWP является единственным приложения, которое выполняется на устройстве) и требует собственную учетную запись устройства на вашей Скайп для реализации бизнеса. Программное обеспечение обновляется как посредством Магазина Windows, так и посредством Центра обновления Windows. В разделе [Планирование v2 систем комнаты Скайп](skype-room-systems-v2-0.md) подробные сведения об использовании этих помещений консоли в вашем развертывании.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Для MSI и установщиков типа "нажми и работай" доступны установщики клиентов. Скайп для бизнес-стратегии sustainment клиент может повлиять на выбором, поэтому необходимо знать следующее: 
  
- В общем Скайп для бизнеса не добавляет новые функции предыдущие клиентов
    
- В общем Скайп для бизнеса не планируется доставки новых функций в Скайп для бизнеса MSI после первого выпуска. Усовершенствования MSI между выпусками главным образом будут касаться качества и безопасности. 
    
- Новейшие и интересных Скайп для взаимодействия с пользователем предприятия будут находиться в Скайп для бизнеса 2016 click-to-run установщика. 
    
Это можно сделать настраиваемого развертывания клиента, как описано в [Настройка Windows Установка клиента в Скайп для Business Server 2015](../../deploy/deploy-clients/customize-windows-client-installation.md). Способы установки описаны более подробно описываются в [Развертывание клиентов для Скайп для Business Server 2015](../../deploy/deploy-clients/deploy-clients.md)
  
### <a name="click-to-run"></a>Технология "нажми и работай"

Click-to-run — это Microsoft потоковой передачи и виртуализации, которые можно использовать для установки и обновления продуктов Office, включая Скайп для бизнеса. Функции потоковой передачи и виртуализации основаны на технологиях в составе Microsoft Application Virtualization (App-V). Технология "нажми и работай" имеет следующие преимущества: 
  
- потоковая установка набора Office, способствующая сокращению времени установки;
    
- интегрированные обновления и исправления;
    
- использование меньшего дискового пространства;
    
- лицензирование для пользователя: 5 установок на пользователя;
    
- возможность настройки с помощью редактора XML для установки независимых программ.
    
Можно использовать [Средство развертывания Office 2016](https://www.microsoft.com/en-us/download/details.aspx?id=49117) для этого типа установщика.
  
Как обычный и полный клиентские версии (с выбором 32 - и 64-разрядных версий) доступны с помощью установщика click-to-run, пользователей можно загрузить по [Загрузить Скайп для бизнеса на всех устройствах](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). 
  
### <a name="msi"></a>MSI

MSI является более традиционный способ установки. Можно вручную установить обновления и исправления, использования корпоративного лицензирования и активации и можно настраивать с помощью [Центра развертывания Office](http://www.microsoft.com/en-us/download/details.aspx?id=49030). Можно распределить клиентов с помощью групповой политики, с помощью System Center Configuration Manager, или с помощью средства сторонних производителей.
