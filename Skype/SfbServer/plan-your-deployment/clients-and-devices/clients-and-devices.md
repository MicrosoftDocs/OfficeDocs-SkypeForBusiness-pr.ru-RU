# <a name="plan-for-clients-and-devices"></a>Планирование клиентов и устройств

**Сводка:** Обзор поддерживаемых клиентов и приложений для Skype для бизнеса.

Современные сотрудники постоянно находятся в движении. Сотрудникам необходимо взаимодействовать и совместно работать в корпоративном офисе, в региональных расположениях, дома или в пути. Skype для бизнеса Server поддерживает эти потребности с помощью коллекции клиентских интерфейсов, которые можно развернуть для пользователей вашей организации. Продуманное планирование гарантирует, что сотрудники смогут получить то, что им нужно, и что Skype для бизнеса доступен им, где бы они ни были.

## <a name="available-clients"></a>Доступные клиенты

Skype для бизнеса Server поддерживает несколько типов клиентов, включая установленное на компьютере клиентские программы, веб-клиенты и клиенты для мобильных устройств. В этом разделе представлены основные клиенты. Подробное описание всех поддерживаемых клиентов см. в сравнении функций классических клиентов [для Skype для бизнеса Server 2015](desktop-feature-comparison.md) или skype для бизнеса Server [2019.](../../../SfBServer2019/plan/feature-comparison.md) Если вы ранее использовали сочетание клиентов Lync, обратите внимание, что [](clients-and-devices.md#Legacy) существуют неподтверченные устаревшие клиенты, несовместимые со Skype для бизнеса Server 2019. Обновления происходят регулярно, поэтому периодически проверяйте в этом разделе последние сведения о клиенте.

### <a name="skype-for-business-2019"></a>Skype для бизнеса (2019)

Skype для бизнеса (2019) — это рекомендуемый полнокомандный клиент для Skype для бизнеса Server 2015 и 2019. Описание [новых функций см.](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) в описании последних обновлений в Skype для бизнеса. Поддержка функций клиентов подробно описана в сравнении функций классических клиентов [для Skype](desktop-feature-comparison.md)для бизнеса, а документация по пользователям находится в справке Skype [для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается, когда пользователь устанавливает Microsoft 365 или Office 365.

Кроме того, доступен бесплатный базовый клиент, поддерживающие меньше возможностей. Обе версии доступны для скачивания на [скачив Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Различия между клиентами Full и Basic описаны в разделе "Основные ограничения [клиента".](desktop-feature-comparison.md#Full-Basic)

### <a name="skype-for-business-2016"></a>Skype for Business 2016

Skype для бизнеса 2016 — это полно функции клиента Skype для бизнеса Server 2015 или 2019. Описание новых функций см. в описании новых возможностей Skype для бизнеса [2016.](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) Поддержка функций клиентов подробно описана в сравнении функций классических клиентов [для Skype](desktop-feature-comparison.md)для бизнеса, а документация по пользователям находится в справке Skype [для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается при установке Office 365 пользователем.

Кроме того, доступен бесплатный базовый клиент, поддерживающие меньше возможностей. Обе версии доступны для скачивания на [скачив Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Различия между клиентами Full и Basic описаны в разделе "Основные ограничения [клиента".](desktop-feature-comparison.md#Full-Basic)

### <a name="skype-for-business-2015"></a>Skype для бизнеса 2015

Skype для бизнеса 2015 — это полно функции клиента Skype для бизнеса Server 2015 или 2019. Пользовательский интерфейс Skype для бизнеса был полностью переработан и включает новые интегрированные функции, такие как монитор звонков, интеграция каталогов Skype, смайлики и другие. Сводку изменений см. в подстановке ["Lync теперь Skype для бизнеса" и "Новые возможности".](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612) Поддержка функций клиентов подробно описана в сравнении функций классических клиентов [для Skype](desktop-feature-comparison.md)для бизнеса, а документация по пользователям находится в справке Skype [для бизнеса.](https://support.office.com/Skype-for-business) Этот клиент включается при установке Office 365 пользователем.

### <a name="skype-for-business-on-mac"></a>Skype для бизнеса на Mac

Клиент [Skype для бизнеса на Mac](https://www.microsoft.com/download/details.aspx?id=54108) доступен для скачивания. См. [требования к клиенту Skype для бизнеса](mac-requirements.md) на Mac для проверки необходимых условий.

### <a name="skype-for-business-for-mobile-devices"></a>Skype для бизнеса для мобильных устройств

Клиенты доступны для Windows Phone, iPhone, iPad и Android. Пользователи могут получить их на [скачив Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Поддержка функций для этих клиентов подробно описана в сравнении функций [мобильных клиентов для Skype для бизнеса.](mobile-feature-comparison.md)

> [!NOTE]
> Поддержка MCX (Mobility Service) для устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019. Все текущие мобильные клиенты Skype для бизнеса уже используют веб-API объединенных коммуникаций (UCWA) для поддержки обмена мгновенными сообщениями, присутствия и контактов. Пользователям с устаревшими клиентами, использующими MCX, потребуется обновиться до текущего клиента.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Надстройка "Собрание по сети" для Skype для бизнеса

Надстройка "Собрание по сети" для Skype для бизнеса поддерживает управление собраниями из клиента Microsoft Outlook для обмена сообщениями и совместной работы в Windows. Надстройка "Собрание по сети" для Skype для бизнеса автоматически устанавливается вместе со Skype для бизнеса.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype для бизнеса Web App и приложение "Собрания Skype"

Если Skype для бизнеса не установлен на компьютере пользователя и пользователь щелкает ссылку на собрание в запросе на собрание на компьютере с Windows, будет установлено и открыто приложение "Собрания Skype" или Skype для бизнеса Web App.  Приложение "Собрания Skype" — это клиент для участников за пределами организации. (Обратите внимание, что на компьютере Mac будет установлен и открыт Skype для бизнеса на Mac.) Требования к использованию этих клиентов см. в приложении [Plan for Meetings (Web App и Meetings](meetings-clients.md) App).


### <a name="skype-for-business-web-scheduler"></a>Веб-планировщик Skype для бизнеса

Веб-планировщик Skype для бизнеса — это веб-средство планирования собраний и управления ими для пользователей [Skype](https://sched.lync.com) для бизнеса Online, которые не имеют доступа к Microsoft Outlook или работают в операционной системе, не основанной на Windows. С помощью веб-планировщика Skype для бизнеса пользователи могут создавать новые собрания, изменять существующие собрания и отправлять приглашения с помощью предпочитаемой почтовой программы. Документация по веб-плану Skype для [бизнеса предоставляет](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)  дополнительные сведения.

### <a name="vdi-plugins"></a>Подключаемый модуль VDI

Среда инфраструктуры виртуальных рабочих стола (VDI) используется в некоторых организациях, где вопросы безопасности и соответствия требованиям являются особенно конфиденциальными. Использование Skype для бизнеса с полным аудио- и видеосвязью для подключения, которое требует большой нагрузки обработки звука и видео на клиенте, размещенного на виртуальном рабочем столе. Доступно дополнительное подключаемое программное обеспечение VDI, которое разгружает обработку на локальный компьютер пользователя и снижает нагрузку на виртуальный рабочий стол. Подробные сведения об использовании этих подключаемых модульов см. в подключении "Планирование Skype для бизнеса в средах [VDI".](vdi-environments.md)

### <a name="microsoft-teams-rooms"></a>Комнаты Microsoft Teams

Комнаты Microsoft Teams — это новейшее решение корпорации Майкрософт для аудиоконференций, которое использует знакомый интерфейс и легко развертывается и управляется, используя существующее оборудование, такое как панели СДВВ для упростит установку. Комнаты Microsoft Teams используют специальное приложение UWP, запущенное на Surface Pro 4 или Surface Pro в режиме консоли (после развертывания приложение UWP будет единственным приложением, которое будет работать на устройстве), и для его реализации требуется собственная учетная запись устройства. Программное обеспечение обновляется как через Магазин Windows, так и через Обновление Windows. См. https://aka.ms/MTRDocs подробные сведения об использовании этих консолей комнат в развертывании. 

### <a name="skype-for-business-on-surface-hub"></a>Skype для бизнеса на Surface Hub

Microsoft Surface Hub — это одно и то же устройство для продуктивной работы, предназначенное для совместных и совместной работы, а также для презентаций. Он имеет собственную итерацию клиента Skype для бизнеса, задокументированную в руководстве администратора [Microsoft Surface Hub.](https://docs.microsoft.com/surface-hub/)

## <a name="choosing-your-organizations-preferred-client"></a>Выбор предпочитаемого клиента организации
<a name="BK_client_choose"> </a>

Если ваша организация приобрела соответствующие лицензии, выберите полный клиент, в противном случае выберите базовый клиент.

Пользователи могут самостоятельно установить клиент из [Скачивания Skype для бизнеса на всех устройствах.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Клиент также устанавливается, когда пользователи устанавливают Microsoft 365 или Office 365 в Windows. Если у некоторых пользователей есть компьютеры Mac, у них будет другой набор функций, как описано в предыдущих разделах.

Некоторые функции, доступные в Skype для бизнеса Server 2015, недоступны в Skype для бизнеса Online или Skype для бизнеса Server 2019. Конкретные особенности см. в ограничениях учетных записей online или гибридной учетной записи пользователя в [2015](desktop-feature-comparison.md#Online-Hybrid) г., а также в гибридных учетных записях пользователей за [2019](desktop-feature-comparison.md#Online-Hybrid) г. Администраторам Skype для бизнеса Online может потребоваться обратиться к описанию [службы Skype](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) для бизнеса Online для получения сведений о различных доступных для них планах.

 Перед развертыванием или обновлением до Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте сравнение [функций классических](desktop-feature-comparison.md) клиентов Skype для бизнеса, чтобы понять влияние поддержки функций на эти клиенты. Это поможет вам сообщить об изменениях пользователям, поработать с процессом его создания и полностью понять преимущества обновления до последней версии клиента.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Способы развертывания клиента для пользователей
<a name="BK_User_Deploy"> </a>

Клиентские установщики доступны как для установщиков MSI, так и для установщиков типа "нажми и нажми". Стратегия поддержки клиентов Skype для бизнеса может повлиять на ваш выбор, поэтому следует понимать следующие моменты:

- Как правило, Skype для бизнеса НЕ добавляет новые функции в ранее выпущенные клиенты

- Как правило, Skype для бизнеса НЕ планирует поставку новых функций в MSI Skype для бизнеса после первоначального выпуска. Улучшения MSI между выпусками в основном связаны с качеством и безопасностью.

- В установщике skype для бизнеса 2019 с технологией "нажми и пойми".

Вы можете настроить развертывание клиента, как описано в описании настройки установки клиента [Windows в Skype для бизнеса Server.](../../deploy/deploy-clients/customize-windows-client-installation.md) Методы установки более подробно описаны в описании развертывания [клиентов для Skype для бизнеса Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>"Нажми и запускай"

Технология "нажми и запуск" — это технология потоковой передачи и виртуализации Майкрософт, которую можно использовать для установки и обновления продуктов Office, включая Skype для бизнеса. Эти возможности потоковой передачи и виртуализации основаны на технологиях в microsoft Application Virtualization (App-V). "Нажми и запускай" имеет следующие преимущества:

- Потоковая установка набора Office, которая приводит к короткой установке

- Исправленные обновления и исправления

- Занимает меньше места на диске

- Лицензирование на основе пользователей: 5 установок на пользователя

- Настраиваемая с помощью редактора XML для установки независимых программ

Для этого типа установщика можно использовать средство развертывания [Office.](https://www.microsoft.com/download/details.aspx?id=49117)

Как базовая, так и полная клиентские версии (с выбором 32- и 64-битных версий) доступны [](https://products.office.com/skype-for-business/download-app?tab=tabs-3)с помощью установщика "нажми и работай". Пользователи могут скачать их на всех устройствах.

### <a name="msi"></a>MSI

MSI — это более традиционный метод установки, используемый на клиентах Skype для бизнеса 2015 и 2016. Это позволяет вручную устанавливать обновления и исправления, использовать корпоративное лицензирование и активацию, а также настраивать их с помощью средства [настройки Office.](https://www.microsoft.com/download/details.aspx?id=49030) Вы можете распространять клиенты, применяя групповые политики, используя Microsoft Endpoint Configuration Manager или сторонний инструмент.



## <a name="legacy-clients"></a>Устаревшие клиенты

Skype для бизнеса Server 2015 поддерживает следующие ранее выпущенные клиенты: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition и Lync 2010 Attendant. Сведения об этих клиентах при их сравнении с другими серверами см. в таблицах сравнения клиентов [для Lync Server 2013](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx) и [Lync Server 2010.](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx)
