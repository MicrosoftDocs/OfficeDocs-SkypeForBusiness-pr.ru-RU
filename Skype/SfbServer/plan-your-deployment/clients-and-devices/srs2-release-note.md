## <a name="skype-room-systems-v2-feature-introduction-and-issue-resolution"></a>Скайп помещения систем v2 компонента введение и проблема решение


### <a name="40780-03142018"></a>4.0.78.0 (14/03/2018)
Представлено в это обновление:
- Исправление для ошибки «зависают во время запуска приложения», на которые оказывает устройств на прежних версий Windows 10 RS2 построения.  


### <a name="40760-03042019"></a>4.0.76.0 (04/03/2019)
Представлено в это обновление:
- Клавиатура DTMF для собраний P2P группами Майкрософт и вызовы в ТСОП. Чтобы принять группами Майкрософт вызывающего клиента по умолчанию, администраторы необходимо установить IsTeamsDefaultClient имеет значение true
- Прикрепление удаленный участник входящего видео на весь экран на передней части экрана комнаты. Используйте команду «ПИН-кода» в списке участников на консоли
- Усовершенствования в зале ожидания уведомления с дополнительной службой уведомлений переднего комнаты
- Передней части экрана комнаты удаляет значок приведения при маяков Bluetooth не включено на устройстве системы комнаты
- Исправление для корпоративного управления проблемы в собраниях по группам


### <a name="40640-12142018"></a>4.0.64.0 (14/12/2018)
Представлено в это обновление:
- Отображение контента на обоих переднего из комнаты (для) отображается в системах двойной экрана комнаты
- Улучшения пользовательского интерфейса темы и переднего комнаты
- Поддержка со стороны клиента с тактовой частотой 1,2 TLS. Для клиентов локально Включение communciation через TLS 1.2 для версии 2 Скайп комнаты системы требует Скайп для Business Server 2015 Cummulative обновления 9 (CU9) или Скайп для рабочих серверов 2019 Cummulative обновления 1 (CU1).

### <a name="40510-11172018"></a>4.0.51.0 (11/17/2018)
Представлено в это обновление:
- Поддержка двойной отображения (в передней части комнаты) для собраний группы 

### <a name="40310-10162018"></a>4.0.31.0 (10/16/2018)
Представлено в это обновление:
- Качество и надежность исправлений 

### <a name="40270-1012018"></a>4.0.27.0 (10/1/2018)
Представлено в это обновление: 
- Изменения кода, необходимые для подготовки приложения SRSv2 более поздние обновления 1803 10 версии Windows
- Устранить проблему форматирования локализованных использованию - специально норвежский, которые предотвращает переход за пределы окна программы установки OOBE лицензионного соглашения
- Изменения кода, необходимо, чтобы сделать приложения v2 Скайп комнаты систем на устаревших систем комнаты Lync. Увеличить [здесь](https://aka.ms/lrsupgrade).
 

### <a name="40190-8312018"></a>4.0.19.0 (8/31/2018)
Представлено в это обновление: 
- Исправление для Crestron приложения, не запуская которого обычно бы доступно, нажав кнопку приложения на устройствах Crestron SR. Требуется после установки 4.0.19.0 перезагрузка SRSv2 приложения 

### <a name="40180-08272018"></a>4.0.18.0 (08/27/2018)
Представлено в это обновление: 
- Улучшения возможностей отчетов «проблемы» в режиме групп (эквивалент «Предоставить свои отзывы и предложения» в Скайп для режима бизнеса)
- Включите возможность резервный из группам Скайп для режима Business для вызовов SIP
- Улучшенные специальные возможности ("Экранный диктор", Экранная лупа)
- Автоматически перезапустите приложения при необходимости после внесения изменений подготовки XML
- Прочие исправления

### <a name="release-to-market--1272016"></a>Выпуск на рынок (7/12/2016)
**Компоненты:** 

 **Оптимизация для Skype для бизнеса**
  
- Присоединение к собранию в Skype одним нажатием кнопки    
- Интерфейс приложения для собраний Skype оптимизирован для комнат с полноэкранным видео и широкополосным аудиоканалом высокой четкости
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате
    
  **Преобразование любой комнаты для собраний**
  
- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
- Использование существующих инвестиций во фронтальные экраны или проекторы
- Работа со всеми типами конференц-залов любого размера
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
- Встроенные проводные функции приема для демонстрации совместного доступа к рабочему столу в комнате и в собрании Skype
    
  **Простота развертывания и управления**
  
- Постоянно включенное устройство, которое автоматически переключает экраны из спящего режима при обнаружении людей в комнате
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
- Мониторинг и управление на уровне устройства для Windows 10 Корпоративная с использованием Intune и SCCM (MDM)
- Надежность корпоративного уровня
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
- Выполняется на планшете Surface Pro 4
