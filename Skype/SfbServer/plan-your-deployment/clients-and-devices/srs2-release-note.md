### <a name="311040-04162018"></a>3.1.104.0 (04/16/2018)
**Новые функции** , введенные в это обновление: 
- Исправление для улучшения OSK (экранная клавиатура) поведение 1709 версии 10 окна с ОС
- Улучшения, чтобы подготовиться к обновлению будущих операционной системы


### <a name="311000-03162018"></a>3.1.100.0 (03/16/2018)
**Новые функции** , введенные в это обновление: 
- Обновлено с целью улучшения телеметрии приложения.

### <a name="31990-03142018"></a>3.1.99.0 (14/03/2018)
**Новые функции** , введенные в это обновление: 
- Исправления на проблему, где присоединение к собранию временные проблемы могут возникнуть.
- Исправления неполадок знать для привести устройство «зависают» качества.

### <a name="31980--382018"></a>3.1.98.0 (8/3/2018)
**Новые функции** , введенные в это обновление: 
- Для повышения стабильности устраняет ошибки или сбоя
- Поддержка размера переменной консоли
- Периферийный обработки звука разгрузки (разрабатывает дополнительные носители)
- Также включает оптимизацию, позволяющие ИТ-специалистов для построения самостоятельной изображений с пакетом обновления за январь 1709 10 версии Windows и более поздних версий.

### <a name="30150-1032017"></a>3.0.15.0 (10/3/2017)
**Новые функции** , введенные в это обновление: 
- Поддержка [Polycom MSR серии](http://www.polycom.com/hd-video-conferencing/microsoft-video/msr-series.mdl) закрепить оборудования
- Поддержка [Logitech Brio](https://www.logitech.com/en-us/product/brio)

### <a name="30120-912017"></a>3.0.12.0 (9/1/2017)
**Новые функции** , введенные в это обновление:  
- Запускает на планшетном ПК Surface Pro (2017)  
- Поддерживает обновление 10 Enterprise создателя (английский язык, построения 1703)    
- Поддержка [Crestron SR](http://www.crestron.com/products/line/sr-for-skype-for-business-room-system) закрепить оборудования    
- Поддержка ПВТ для элементов управления среды (Crestron)
    
64-разрядная версия Windows 10 Enterprise Годовщина выпуска (английский язык, версия 1607) больше не поддерживается по состоянию на версии 2 систем комнаты Скайп 3.0.12.0 (обновление 3). 

### <a name="3080-842017"></a>3.0.8.0 (8/4/2017) 
**Новые функции** , введенные в это обновление: 
- Устраняет проблемы, наблюдаемая при поиске федеративных пользователей с помощью поля поиска участников. Предшествующих это исправление результатов поиска для внешнего федеративных пользователей может не обработаны правильно и вместо этого возвращаются неверные результаты. 

### <a name="3060-772017"></a>3.0.6.0 (7/7/2017) 
**Новые функции** , введенные в это обновление:  
- Поддержка двойного экрана (для возможности устаревших систем)   
- Themability (встроенных темы и возможность установки пользовательской темы) 
- Возможность отправить отзыв для общедоступных построения     
- Улучшенная телеметрии вокруг надежности присоединиться к собранию     
- Дополнительные OMS отчетов     
- Возможности для ИТ-администратором для удаленной настройки устройств

### <a name="2020-03152017"></a>2.0.2.0 (03/15/2017)
**Новые функции** , введенные в это обновление: 
- Выбор пользователей в приложении приглашений на собрания комнаты аудио- и видеоконференций USB-устройства
- Интегрированное состояния консоли комнаты отчетов для клиентов, использующих пакет управления Microsoft Operations (см. [Управление планирование систем комнаты Скайп версии 2 с помощью OMS](oms-management.md)) 

### <a name="release-to-market--1272016"></a>Выпуск на рынок (7/12/2016)
**Компоненты:** 

 **Оптимизация для Skype для бизнеса**
  
- Присоединение к собранию в Skype одним нажатием кнопки
    
- Интерфейс приложения для собраний Skype оптимизирован для комнат с полноэкранным видео и широкополосным аудиоканалом высокой четкости
    
- Все участники могут присоединяться к собранию Skype с помощью любых устройств, где бы они ни находились
    
- Возможность приглашать людей из справочника, где можно мгновенно увидеть сведения об их доступности, или по телефону
    
- Поддержка функций конференц-связи и звонков по ТСОП в Skype для бизнеса, позволяющих заменить отдельные конференц-телефоны в комнате
    
 **Преобразование любой комнаты для собраний**
  
- Специализированное приложение для собраний Skype, оптимизированное для работы с настольным сенсорным контроллером и большим фронтальным экраном
    
- Использование существующих инвестиций во фронтальные экраны или проекторы
    
- Работа со всеми типами конференц-залов любого размера
    
- Поддержка сертифицированных звуковых и видеоустройств Skype для бизнеса для комнат любого размера
    
- Встроенные проводные функции приема для демонстрации совместного доступа к рабочему столу в комнате и в собрании Skype
    
    
 **Простота развертывания и управления**
  
- Постоянно включенное устройство, которое автоматически переключает экраны из спящего режима при обнаружении людей в комнате
    
- Простота развертывания и обновления приложения для собраний Skype на универсальной платформе Windows
    
- Блокировка устройств в приложении для собраний Skype с использованием технологии Windows AppLocker
    
- Мониторинг и управление на уровне устройства для Windows 10 Корпоративная с использованием Intune и SCCM (MDM)
    
- Надежность корпоративного уровня
    
- Минимальные затраты времени и ресурсов на обучение благодаря хорошо знакомому пользовательскому интерфейсу Skype
    
- Выполняется на планшете Surface Pro 4
