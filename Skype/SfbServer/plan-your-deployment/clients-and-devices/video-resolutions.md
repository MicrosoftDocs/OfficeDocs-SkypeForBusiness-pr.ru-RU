# <a name="skype-for-business-client-video-resolutions"></a>Разрешение видеороликов для клиентов Skype для бизнеса
 
**Сводка:** Изучите требования к видеоподсистеме при планировании Skype для бизнеса Server.
  
В этой статье описывается поддержка видеоаппарата для вызовов видеоконференций Skype для бизнеса и приводятся инструкции по определению ожидаемого качества видео для различных конфигураций компьютера, планшета и мобильных устройств. 
  
ИТ-специалисты будут использовать эту информацию для оценки пригодности ноутбуков, уже используемых в Организации, или в соответствии с рекомендациями по использованию. Кроме того, они могут выполнять поиск сведений о конкретных устройствах в [каталоге решений](https://partnersolutions.skypeforbusiness.com/solutionscatalog) .
  
## <a name="windows-desktop-mac-and-tablet-video-requirements-and-capabilities"></a>Требования и возможности для рабочего стола Windows, Mac и планшетного видео

Skype для бизнеса использует аппаратное ускорение для кодирования и декодирования видео в зависимости от стандарта H. 264/MPEG-4 в части 10 Advanced Video кодирования. Это позволяет компьютерам с более низкой тактовой частотой процессора кодировать и декодировать видео с более высоким разрешением. Требования к видеоустройствам зависят от конфигурации компьютера и необходимого разрешения видео.
  
Также приведены [требования к оборудованию для Windows и Mac](https://products.office.com/office-system-requirements).
  
### <a name="video-hardware-requirements"></a>Требования к оборудованию

|:-----|:-----|

Необходимо установить драйвер 15.28.9.2884 для графической подсистемы Intel HD или последнюю версию драйвера, включающую следующее:  <br/> • Драйвер дисплея 9.17.10.2884 или последний драйвер  <br/> • Аппаратное преобразование Media Foundation (ХМФТ) версии 3.12.10.31 или новейшее ХМФТ  <br/> Поддерживаются следующие решения для кодирования видео с аппаратным ускорением AMD:  <br/> • Модуль видеокодека AMD, который доступен на нескольких отдельных графических платах и в интегрированных процессорах с ускорением процессора серии AMD серии с ускорением процессора серии AMD. |Аппаратное ускорение кодирования H.264: требования к камере  <br/> |Видеокамеры USB со встроенным аппаратным кодировщиком H.264, поддерживающие спецификацию USB Video Class (UVC) версии 1.5.  <br/> **Примечание:** Skype для бизнеса поддерживает камеры UVC 1,5 с Windows 8 или Windows 8,1, которые включают поддержку UVC 1,5. Так как Windows 7 не включает поддержку UVC 1,5, Skype для бизнеса рассматривает камеры UVC 1,5 как обычные камеры без поддержки аппаратного кодирования. <br/> |
   
### <a name="determining-h264-video-encoding-and-decoding-capabilities"></a>Определение возможностей кодирования и декодирования видео H. 264

В общем случае максимальная производительность кодирования и декодирования для конкретной конфигурации компьютера определяется четырьмя основными факторами:
  
- поддержка аппаратного ускорения декодирования с помощью DXVA;
    
- поддержка аппаратного ускорения кодирования;
    
- количество физических ядер;
    
- индекс производительности Windows (WEI).
    
Средство оценки производительности WinSAT определяет индекс WEI. Средство WinSAT создает XML-документ Formal.Assessment в каталоге %windir%\Performance\WinSAT\DataStore. В этом XML-файле представлены два показателя, на основании которых можно определить возможности кодирования и декодирования:
  
- значение VideoEncodeScore показывает производительность компьютера по программному кодированию видео;
    
- значение GraphicsScore показывает производительность компьютера по кодированию видео с аппаратным ускорением.
