# <a name="skype-for-business-client-video-resolutions"></a>Скайп для клиента Business разрешений
 
**Сводка:** Просмотрите видео требования клиента при планировании для Скайп Business Server 2015.
  
В этой статье описывается поддержка видеоустройств для Скайп для бизнеса видеовызовы и описывается способ определения ожидаемого качества видео для различных компьютере, планшетном ПК и конфигурации мобильного устройства. 
  
ИТ-специалистов полезные эти сведения при оценке соответствия ноутбуки уже в среды в своей организации или с учетом для использования. Они также могут выполнять поиск в [Каталог решений](https://partnersolutions.skypeforbusiness.com/solutionscatalog) сведения на определенных устройствах.
  
## <a name="windows-desktop-mac-and-tablet-video-requirements-and-capabilities"></a>Рабочий стол Windows, Mac и планшетных видео требования и возможности

Скайп для бизнеса с помощью аппаратное ускорение видео кодирования и декодирования на основе стандарта H.264/MPEG-4 часть 10 расширенного видео написания кода. Это позволяет компьютерам с ЦП скорости работы для кодирования и декодирования выше разрешение видео. Требования к видеоустройствам зависят от конфигурации компьютера и необходимого разрешения видео.
  
Также обратиться к разделу [Windows и Mac требования к аппаратному обеспечению](https://products.office.com/en-us/office-system-requirements).

### <a name="determining-h264-video-encoding-and-decoding-capabilities"></a>Определение возможностей кодирования и декодирования видео H.264

В общем случае максимальная производительность кодирования и декодирования для конкретной конфигурации компьютера определяется четырьмя основными факторами:
  
- поддержка аппаратного ускорения декодирования с помощью DXVA;
    
- поддержка аппаратного ускорения кодирования;
    
- количество физических ядер;
    
- индекс производительности Windows (WEI).
    
Средство оценки производительности WinSAT определяет индекс WEI. Средство WinSAT создает XML-документ Formal.Assessment в каталоге %windir%\Performance\WinSAT\DataStore. В этом XML-файле представлены два показателя, на основании которых можно определить возможности кодирования и декодирования:
  
- значение VideoEncodeScore показывает производительность компьютера по программному кодированию видео;
    
- значение GraphicsScore показывает производительность компьютера по кодированию видео с аппаратным ускорением.
