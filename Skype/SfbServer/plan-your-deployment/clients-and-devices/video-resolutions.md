# <a name="skype-for-business-client-video-resolutions"></a>Разрешения видео для клиентов Skype для бизнеса
 
**Сводка:** Просмотрите требования к видео для клиента при планировании Skype для бизнеса Server.
  
В этой статье описывается поддержка видеосвязи для видеозвонков Skype для бизнеса и определение ожидаемого качества видео для различных конфигураций компьютеров, планшетов и мобильных устройств. 
  
ИТ-специалисты найдут эту информацию полезной при оценке пригодности ноутбуков, уже которые используются в организации, или при рассмотрении их использования. Кроме того, они могут искать [информацию на](https://partnersolutions.skypeforbusiness.com/solutionscatalog) определенных устройствах в каталоге решений.
  
## <a name="windows-desktop-mac-and-tablet-video-requirements-and-capabilities"></a>Требования и возможности видео для настольных компьютеров с Windows, Mac и планшетов

Skype для бизнеса использует аппаратное ускорение для кодирования и декодирования видео на основе стандарта H.264/MPEG-4 части 10 Advanced Video Coding. Это позволяет компьютерам с более низкой тактовой скоростью ЦП кодировать и декодировать видео с более высоким разрешением. Требования к видеоустройствам зависят от конфигурации компьютера и необходимого разрешения видео.
  
См. также [требования к оборудованию для Windows и Mac.](https://products.office.com/office-system-requirements)
  
### <a name="video-hardware-requirements"></a>Требования к оборудованию для видео

|**Функция**|**Требование**|
|:-----|:-----|
|Аппаратное ускорение декодирования H.264 с помощью DirectX Video Acceleration (DXVA)  <br/> |• Графическая карта должна поддерживать DirectX 9.0 и должна DXVA2_ModeH264_VLD_NoFGT режим декодирования и API DirectX 9.  <br/> • Должен быть установлен последний драйвер графической карты.  <br/> |
|Аппаратное ускорение кодирования H.264: требования к набору микросхем  <br/> |Поддерживаются следующие решения для кодивки видео с аппаратным ускорением Intel:  <br/> • Intel HD Graphics 2000, 2500, 3000 и 4000 микросхем второго и третьего поколения (или более поздних версий) со встроенными аппаратными кодировки видео. Требуется установка драйвера Intel HD Graphics 15.28.9.2884 или последнего драйвера, содержащего следующее:  <br/> • Отображение драйвера 9.17.10.2884 или последнего драйвера  <br/> • Аппаратное преобразование основы мультимедиа (HMFT) версии 3.12.10.31 или последней версии HMFT  <br/> Поддерживаются следующие решения для аппаратного ускорения видеокодинга AMD:  <br/> • Обработчик видео кодеков AMD, доступный в нескольких дискретных графических картах и интегрированных блоках обработки ускоренными процессорами AMD A-Series. Необходимо установить драйвер amD Video Codec Engine 9.12.0.0 или более высокого уровня.  <br/> |
|Аппаратное ускорение кодирования H.264: требования к камере  <br/> |Видеокамеры USB со встроенным аппаратным кодировщиком H.264, поддерживающие спецификацию USB Video Class (UVC) версии 1.5.  <br/> **Примечание.** Skype для бизнеса поддерживает камеры UVC 1.5 с Windows 8 или Windows 8.1, который включает поддержку UVC 1.5. Поскольку Windows 7 не поддерживает UVC 1.5, Skype для бизнеса рассматривает камеры UVC 1.5 как обычные камеры без поддержки аппаратного кодивинга. <br/> |
   
### <a name="determining-h264-video-encoding-and-decoding-capabilities"></a>Определение возможностей кодировки и декодирования видео H.264

В общем случае максимальная производительность кодирования и декодирования для конкретной конфигурации компьютера определяется четырьмя основными факторами:
  
- поддержка аппаратного ускорения декодирования с помощью DXVA;
    
- поддержка аппаратного ускорения кодирования;
    
- количество физических ядер;
    
- индекс производительности Windows (WEI).
    
Средство оценки производительности WinSAT определяет индекс WEI. Средство WinSAT создает XML-документ Formal.Assessment в каталоге %windir%\Performance\WinSAT\DataStore. В этом XML-файле представлены два показателя, на основании которых можно определить возможности кодирования и декодирования:
  
- значение VideoEncodeScore показывает производительность компьютера по программному кодированию видео;
    
- значение GraphicsScore показывает производительность компьютера по кодированию видео с аппаратным ускорением.
