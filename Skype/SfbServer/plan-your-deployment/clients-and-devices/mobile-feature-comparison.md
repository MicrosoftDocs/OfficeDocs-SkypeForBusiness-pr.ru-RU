# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Сравнение функций мобильных клиентов для Skype для бизнеса
 
**Сводка:** Ознакомьтесь со сведениями о поддержке функций для мобильных клиентов при планировании Skype для бизнеса Server.
  
В этой статье сравниваются функции и возможности мобильных клиентов Skype для бизнеса и клиента Skype для бизнеса в следующих категориях:
  
- Вход, Push-уведомления и общие функции
    
- Расширенное присутствие
    
- Контакты и группы контактов
    
- Обмен мгновенными сообщениями (IM)
    
- Поддержка звука и видео Skype для бизнеса в Skype для бизнеса
    
- Конференции
    
- Телефонии
    
- Внешние пользователи
    
- Архивация и соответствие требованиям
    
-  Современная проверка подлинности
    
В следующих таблицах перечислены возможности, доступные пользователям Skype для бизнеса в локальном развертывании Skype для бизнеса Server. Те же функции доступны в Skype для бизнеса Online и Microsoft 365 или Office 365 для пользователей, если не указано иное в сносках таблицы.
  
> [!NOTE]
> Для получения справки и ресурсов в Интернете для конечных пользователей, ознакомьтесь со статьей [Обнаружение Skype для бизнеса](https://go.microsoft.com/fwlink/p/?LinkId=528686). 
  
> [!NOTE]
> Чтобы сравнить функции, доступные в других клиентах Skype для бизнеса, ознакомьтесь со статьей [Сравнение функций клиента для настольных ПК для Skype для бизнеса](desktop-feature-comparison.md). 

> [!NOTE]
> MCX (служба Mobility Service) поддержка устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019. Все текущие мобильные клиенты Skype для бизнеса уже используют веб-API объединенных коммуникаций (UCWA), чтобы поддерживать обмен мгновенными сообщениями, сведения о присутствии и контакты. Пользователям прежних версий клиентов, использующих MCX, необходимо выполнить обновление до текущего клиента.
