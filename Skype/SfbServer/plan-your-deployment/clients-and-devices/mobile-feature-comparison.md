# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Сравнение возможностей мобильного клиента для Скайп для бизнеса
 
**Сводка:** Просмотрите поддержка функции для мобильного клиента при планировании для Скайп Business Server 2015.
  
В этой статье представлено сравнение функций и возможностей между Скайп для мобильных клиентов бизнеса и Скайп для настольных компьютеров клиента Business в следующих категориях:
  
- вход, push-уведомления и общие функциональные возможности;
    
- расширенные сведения о присутствии;
    
- контакты и группы контактов;
    
- обмен мгновенными сообщениями;
    
- Скайп для бизнеса на Скайп для бизнеса звука и видео
    
- конференц-связь
    
- телефония;
    
- внешние пользователи;
    
- архивирование и соответствие требованиям.
    
-  Современная проверка подлинности
    
В следующих таблицах перечислены функциональные возможности, доступные для Скайп для бизнес-пользователей в локальном развертывании Скайп для Business Server 2015. Те же возможности также доступны для Скайп для бизнеса в Интернет и Microsoft Office 365 пользователей, если не указано иное в таблице сносок.
  
> [!NOTE]
> Справка в Интернете и ресурсы для конечных пользователей в разделе [Обнаружение Скайп для бизнеса](https://go.microsoft.com/fwlink/p/?LinkId=528686). 
  
> [!NOTE]
> Чтобы сравнить функции, доступные в других Скайп пользователей, обратитесь к разделу [Сравнение функций для настольных компьютеров для Скайп для бизнеса](desktop-feature-comparison.md).
