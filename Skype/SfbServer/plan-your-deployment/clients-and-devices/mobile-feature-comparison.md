# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Сравнение функций мобильных клиентов для Skype для бизнеса
 
**Сводка:** Просмотрите поддержку функций мобильного клиента при планировании Skype для бизнеса Server.
  
В этой статье сравнивались функции и возможности мобильных клиентов Skype для бизнеса и настольного клиента Skype для бизнеса в следующих категориях:
  
- Вход, push-уведомления и общие функции
    
- Расширенные возможности присутствия
    
- Контакты и группы контактов
    
- Обмен мгновенными сообщениями
    
- Аудио- и видеосвязь между Skype для бизнеса и Skype для бизнеса
    
- Конференц-связь
    
- Телефония
    
- Внешние пользователи
    
- Архивная архивативная и соответствие требованиям
    
-  Современная проверка подлинности
    
В следующих таблицах представлены функции, доступные пользователям Skype для бизнеса в локальном развертывании Skype для бизнеса Server. Эти же функции доступны пользователям Skype для бизнеса Online и Microsoft 365 или Office 365, если в сносках таблицы не указано иное.
  
> [!NOTE]
> Справку и ресурсы в Интернете для конечных пользователей см. в справке [по Skype для бизнеса.](https://go.microsoft.com/fwlink/p/?LinkId=528686) 
  
> [!NOTE]
> Сравнение функций, доступных в других клиентах Skype для бизнеса, см. в сравнении возможностей классических клиентов [Skype для бизнеса.](desktop-feature-comparison.md) 

> [!NOTE]
> Поддержка СЛУЖБЫ MCX (Mobility Service) для устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019. Все текущие мобильные клиенты Skype для бизнеса уже используют веб-API объединенных коммуникаций (UCWA) для поддержки обмена мгновенными сообщениями, присутствия и контактов. Пользователям с устаревшими клиентами, использующими MCX, потребуется обновиться до текущего клиента.
