# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Сравнение функций мобильных клиентов для Skype для бизнеса
 
**Сводка:** Просмотрите поддержку функций для мобильного клиента при планировании Skype для бизнеса Server.
  
В этой статье сравнивают функции и возможности Skype для бизнеса мобильных клиентов и Skype для бизнеса настольного клиента в следующих категориях:
  
- Вход, push-уведомления и общие функции
    
- Расширенное присутствие
    
- Контакты и группы контактов
    
- Мгновенные сообщения (IM)
    
- Skype для бизнеса Skype для бизнеса аудио и видео
    
- Конференц-связь
    
- Телефония
    
- Внешние пользователи
    
- Архивавка и соответствие требованиям
    
-  Современная проверка подлинности
    
В следующих таблицах представлены функции, доступные Skype для бизнеса пользователям в локальном развертывании Skype для бизнеса Server. Эти же функции также доступны для Skype для бизнеса и Microsoft 365 или Office 365 пользователей, если иное не указано в сноски таблицы.
  
> [!NOTE]
> Справки и ресурсы для конечных пользователей см. в [Skype для бизнеса.](https://go.microsoft.com/fwlink/p/?LinkId=528686) 
  
> [!NOTE]
> Чтобы сравнить возможности, доступные в других Skype для бизнеса, см. в [Skype для бизнеса.](desktop-feature-comparison.md) 

> [!NOTE]
> Поддержка mcX (Mobility Service) для устаревших мобильных клиентов больше недоступна в Skype для бизнеса Server 2019 г. Все современные Skype для бизнеса мобильные клиенты уже используют веб-API единой связи (UCWA) для поддержки обмена мгновенными сообщениями,присутствия и контактов. Пользователям с устаревшими клиентами, использующими MCX, потребуется обновиться до текущего клиента.
