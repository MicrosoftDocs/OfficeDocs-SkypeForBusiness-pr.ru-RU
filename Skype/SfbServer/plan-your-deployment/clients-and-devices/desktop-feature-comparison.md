# <a name="desktop-client-feature-comparison-for-skype-for-business-server-2015"></a>Сравнение функций клиента настольных компьютеров Skype для бизнеса Server 2015 г.

**Сводка.** Skype для бизнеса Server 2015 или Skype для бизнеса администраторы Online могут использовать эти таблицы, чтобы понять, какие функции поддерживаются клиентами.
  
 Перед развертыванием или обновлением Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте таблицы ниже, чтобы понять влияние поддержки функций на этих клиентов. Это поможет вам донести изменения до пользователей, побок процесса выкатки и полностью понять преимущества обновления до последнего клиента.
  
Некоторые функции, доступные Skype для бизнеса Server 2015 г., недоступны в Skype для бизнеса Online, см. ограничения учетной записи пользователей Online или [Hybrid](desktop-feature-comparison.md#Online-Hybrid) для определенных компонентов. Skype для бизнеса Администраторы сети могут обратиться к Skype для бизнеса [online Service Description](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) для получения сведений о различных планах, доступных для них.

Сравнение клиентских функций настольных [компьютеров Skype для бизнеса 2019](../../../SfBServer2019/plan/feature-comparison.md) г. для поддержки Skype для бизнеса Server 2019 г.
  
В следующих таблицах представлены функции, доступные каждому клиенту, который работает с Skype для бизнеса Server 2015 или Skype для бизнеса Online. Вы также можете сослаться на сравнение функций [мобильных](mobile-feature-comparison.md) клиентов для Skype для бизнеса для сравнения функций смартфона и планшета. Лицензия на доступ к клиентам или лицензия на подписку на пользователя, приобретаемая организацией, также повлияет на то, какие функции доступны пользователям. Развертывание клиента Full или Basic для пользователей зависит от лицензии или плана, который выбирает ваша организация. Дополнительные [сведения см. в руководстве](https://products.office.com/skype-for-business/it-pros) по лицензированию.
  
> [!IMPORTANT]
> Skype для бизнеса Server 2015 и Skype для бизнеса Online поддерживают следующие ранее выпущенные клиенты: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Телефон Edition и Lync 2010 Attendant. Сведения об этих клиентах, используемых на других серверах, см. в таблицах сравнения клиента [для Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-desktop-client-comparison-tables) и таблицы сравнения клиентов [для Lync Server 2010.](/previous-versions/office/skype-server-2010/gg425836(v=ocs.14))

> [!NOTE]
> Клиент **Lync 2010 Attendant** не поддерживается в Skype для бизнеса Online.

> [!NOTE]
> Клиент Skype для бизнеса Web App браузера и приложение Skype meetings App Windows 10 предоставляют только [поддержку Meetings.](desktop-feature-comparison.md#BKMK_Conferencing) Дополнительные данные см. в приложении Plan [for Meetings (Web App and Meetings App).](meetings-clients.md)

## <a name="client-limitations"></a>Ограничения клиента

<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Ограничения базового клиента

<a name="Full-Basic"> </a>

Ниже приведены функции, доступные с помощью полного клиента и недоступные для базового клиента:

- Управление параметрами групповых звонков
- Управление делегатами
- Обработка звонков другого пользователя делегатом
- Управление большим количеством звонков
- Инициация звонка в группу ответа
- Парковка вызовов
- Изменение приветствия
- Ответ на вызов в группе
- Пропущенные сообщения уведомлений о вызове не создаются, когда состояние пользователя отключено и они используют устаревший Outlook клиент (2013 или более ранний)

### <a name="online-or-hybrid-user-account-limitations"></a>Ограничения учетной записи пользователей в Интернете или гибридной учетной записи

<a name="Online-Hybrid"> </a>

Учетные записи пользователей могут существовать как в Интернете, так и в локальном режиме, что повлияет на функции, доступные этому пользователю. Пользователи с учетными записями Skype для бизнеса Online не будут иметь доступ к следующим функциям, даже с полным клиентом:
  
- Расширенное присутствие: используйте фотографию с любого публичного сайта для my Picture
- Контакты: Поиск групп реагирования
- Поддержка чата: интеграция с постоянным чатом (групповой чат)
- Поддержка чата: одним щелчком мыши перейдите в Skype для бизнеса комнату чата
- Внешние пользователи: проведение двухпартийных или многопартийных звонков с внешними пользователями
