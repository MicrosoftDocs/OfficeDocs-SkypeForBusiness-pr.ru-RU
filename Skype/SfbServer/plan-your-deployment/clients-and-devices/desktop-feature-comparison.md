# <a name="desktop-client-feature-comparison-for-skype-for-business-server-2015"></a>Сравнение функций клиента для настольных ПК в Skype для бизнеса Server 2015

**Сводка:** Администраторы Skype для бизнеса Server 2015 или Skype для бизнеса Online могут использовать эти таблицы, чтобы узнать, какие функции поддерживаются в клиентах.
  
 Перед развертыванием или обновлением до Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте приведенные ниже таблицы, чтобы определить, какая функция поддерживает влияние на эти клиенты. Это поможет вам сообщить об изменениях пользователям, оценить процесс развертывания и полностью оценить преимущества обновления до последней версии клиента.
  
Некоторые функции, доступные в Skype для бизнеса Server 2015, недоступны в Skype для бизнеса Online, можно узнать об [ограничениях в Интернете или гибридной учетной записи пользователя](desktop-feature-comparison.md#Online-Hybrid) . Администраторам Skype для бизнеса Online можно обратиться к [описанию службы Skype для бизнеса Online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) для получения сведений о различных планах, доступных для них.

Ознакомьтесь со статьей [Сравнение функций клиента для Skype для бизнеса 2019](../../../SfBServer2019/plan/feature-comparison.md) для поддержки клиентов в Skype для бизнеса Server 2019.
  
В следующих таблицах показаны функции, доступные для каждого клиента, работающего с Skype для бизнеса Server 2015 или Skype для бизнеса Online. Вы также можете обратиться к статье [Сравнение функций мобильных клиентов для Skype для бизнеса](mobile-feature-comparison.md) при сравнении функций мобильных телефонов и клиентов планшета. Лицензия на клиентский доступ или лицензия на подписку пользователя также влияет на возможности, доступные для пользователей. Если вы развертываете весь клиент или базовый клиент для пользователей, зависит от лицензии или плана, в котором ваша организация выбирает купить. Более подробную информацию можно найти в [руководстве по лицензированию](https://products.office.com/skype-for-business/it-pros) .
  
> [!IMPORTANT]
> Skype для бизнеса Server 2015 и Skype для бизнеса Online поддерживают следующие ранее выпущенные клиенты: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition и Lync 2010 Attendant. Сведения об этих клиентах, используемых с другими серверами, можно найти в [таблицах сравнения клиентов для Lync server 2013](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx) и [таблицах сравнения клиентов для Lync Server 2010](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx).

> [!NOTE]
> Клиент **Lync 2010 Attendant** не поддерживается в Skype для бизнеса Online.

> [!NOTE]
> Приложение браузера веб-приложения Skype для бизнеса и приложения для собраний Skype для Windows 10 предоставляют [поддержку собраний](desktop-feature-comparison.md#BKMK_Conferencing). Ознакомьтесь со статьей [Plan for Meeting Clients (приложение для веб-приложений и собраний)](meetings-clients.md) для получения дополнительных сведений об этих клиентах.

## <a name="client-limitations"></a>Ограничения для клиентов

<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Ограничения базового клиента

<a name="Full-Basic"> </a>

Приведенные ниже функции доступны при использовании полного клиента и недоступны в базовом клиенте.

- Управление параметрами групповых звонков
- Управление делегатами
- Обработка звонков другого пользователя делегатом
- Управление большим количеством звонков
- Инициация звонка в группу ответа
- Парковка вызовов
- Изменение приветствия
- Ответ на вызов в группе
- Уведомления о пропущенных вызовах не создаются, когда состояние пользователя отключено и используется устаревший клиент Outlook (2013 или более ранняя)

### <a name="online-or-hybrid-user-account-limitations"></a>Ограничения учетных записей в сети или гибридной учетной записи пользователя

<a name="Online-Hybrid"> </a>

Учетные записи пользователей могут существовать в Интернете или локальной среде и влияют на возможности, доступные для этого пользователя. Пользователи, у которых есть учетные записи в Skype для бизнеса Online, не смогут получить доступ к следующим функциям, даже при использовании полного клиента:
  
- Расширенные сведения о присутствии: использовать фотографию с любого общедоступного сайта для моего изображения
- Контакты: Поиск групп ответа
- Поддержка обмена мгновенными сообщениями: интеграция сохраняемого чата (группового чата)
- Поддержка обмена мгновенными сообщениями: расширение комнаты сохраняемого чата на собрание Skype для бизнеса одним щелчком мыши
- Внешние пользователи: выполнение двусторонних или многосторонних вызовов с внешними пользователями
