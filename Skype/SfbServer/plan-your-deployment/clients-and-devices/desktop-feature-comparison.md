# <a name="desktop-client-feature-comparison-for-skype-for-business-server-2015"></a>Сравнение функций классических клиентов для Skype для бизнеса Server 2015

**Сводка:** Администраторы Skype для бизнеса Server 2015 или Skype для бизнеса Online могут использовать эти таблицы, чтобы понять, какие функции поддерживаются для каких клиентов.
  
 Перед развертыванием или обновлением до Skype для бизнеса проверьте, какие клиенты уже используются в вашей организации. Используйте таблицы ниже, чтобы понять влияние поддержки функций на эти клиенты. Это поможет вам сообщить об изменениях пользователям, поработать с процессом его создания и полностью понять преимущества обновления до последней версии клиента.
  
Некоторые функции, доступные в Skype для бизнеса Server 2015, недоступны в Skype для бизнеса Online. Конкретные особенности см. в ограничениях учетной записи пользователя Online или гибридной учетной записи. [](desktop-feature-comparison.md#Online-Hybrid) Администраторам Skype для бизнеса Online может потребоваться обратиться к описанию [службы Skype](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) для бизнеса Online для получения сведений о различных доступных для них планах.

Сравнение [функций классических клиентов в Skype для бизнеса 2019 для](../../../SfBServer2019/plan/feature-comparison.md) поддержки клиентов в Skype для бизнеса Server 2019.
  
В следующих таблицах представлены функции, доступные каждому клиенту, который работает со Skype для бизнеса Server 2015 или Skype для бизнеса Online. You may also want to refer to [Mobile client feature comparison for Skype for Business](mobile-feature-comparison.md) for smart phone and tablet client feature comparisons. Лицензия клиентского доступа или лицензия на подписку пользователя, приобретаемая организацией, также повлияет на доступность функций для пользователей. Выбор клиента "Полный" или "Базовый" для пользователей зависит от лицензии или плана покупки. Дополнительные [сведения см. в](https://products.office.com/skype-for-business/it-pros) руководстве по лицензированию.
  
> [!IMPORTANT]
> Skype для бизнеса Server 2015 и Skype для бизнеса Online поддерживают следующие ранее выпущенные клиенты: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition и Lync 2010 Attendant. Сведения об этих клиентах при их сравнении с другими серверами см. в таблицах сравнения клиентов [для Lync Server 2013](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx) и таблицах сравнения клиентов [для Lync Server 2010.](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx)

> [!NOTE]
> Клиент **Lync 2010 Attendant** не поддерживается в Skype для бизнеса Online.

> [!NOTE]
> Клиент браузера Skype для бизнеса Web App и приложение "Собрания Skype" для Windows 10 поддерживают [только собрания.](desktop-feature-comparison.md#BKMK_Conferencing) Дополнительные вопросы об этих клиентах см. в приложении [Plan for Meetings (Web App и Meetings App).](meetings-clients.md)

## <a name="client-limitations"></a>Ограничения клиентов

<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Ограничения базового клиента

<a name="Full-Basic"> </a>

Функции, доступные ниже, доступны с помощью полного клиента и недоступны в базовом клиенте:

- Управление параметрами групповых звонков
- Управление делегатами
- Обработка звонков другого пользователя делегатом
- Управление большим количеством звонков
- Инициация звонка в группу ответа
- Парковка вызовов
- Изменение приветствия
- Ответ на вызов в группе
- Уведомления о пропущенных вызовах не создаются, если отключена омма и используется устаревший клиент Outlook (2013 или более ранний)

### <a name="online-or-hybrid-user-account-limitations"></a>Ограничения учетной записи пользователя в Сети или гибридной сети

<a name="Online-Hybrid"> </a>

Учетные записи пользователей могут существовать как в сети, так и в локальной сети, что повлияет на функции, доступные этому пользователю. Пользователи с учетными записями в Skype для бизнеса Online не будут иметь доступа к следующим функциям, даже при работе с полным клиентом:
  
- Расширенные возможности присутствия: использование фотографии с любого открытого сайта для работы с изображением
- Контакты: поиск групп ответа
- Поддержка мгновенных сообщения: интеграция сохраняемой беседы (группового чата)
- Поддержка мгновенных сообщения: одним щелчком мыши довести комнату сохраняемой беседы до собрания Skype для бизнеса
- Внешние пользователи: проведение двухстолбных или многостолбных звонков с внешними пользователями
