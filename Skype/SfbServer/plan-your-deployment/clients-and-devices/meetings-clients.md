# <a name="plan-for-meetings-clients-web-app-and-meetings-app"></a>Планирование клиентов собраний (Веб-приложение и приложение для собраний)
 
**Сводка:** ИТ-специалистам следует рассмотреть требования к поддержке для Skype для бизнеса Web App и приложения собраний Skype при планировании Skype для бизнеса Server. Эта статья не предназначена для пользователей этих приложений.
  
После внедрения Skype для бизнеса Server предполагается, что в процессе развертывания у пользователей вашей организации будет установлен клиент Skype для бизнеса. 
  
Позднее эти пользователи могут создавать собрания и приглашать пользователей из-за пределов организации, и у этих приглашенных может не быть версии клиента Skype для бизнеса. Когда эти пользователи щелкают URL-адрес приглашения на собрание, будет обнаружено отсутствие клиента, и приглашенный без клиента Skype для бизнеса получит запрос на скачивание и установку облегченного клиента, доступного только для собраний, чтобы присоединиться к собранию.
  
> [!NOTE]
> Skype для бизнеса Web App и приложение "Собрания Skype" доступны только при попытке входа в собрание без Skype для бизнеса. Справка для пользователей по этим приложениям находится по указанным [https://aka.ms/smahelp](https://aka.ms/smahelp) здесь. 
  
> [!NOTE]
> Вы не можете предварительно установить Skype для бизнеса Web App или [](https://products.office.com/skype-for-business/download-app?tab=tabs-1) приложение [](https://products.office.com/skype-for-business/download-app?tab=tabs-2) "Собрания Skype", но пользователи смартфонов и планшетов могут установить дорогостоящие мобильные клиенты, которые они могут использовать для участия в собраниях.
  
По умолчанию сервер, на который размещено собрание, будет направлять пользователя скачать и установить Skype для бизнеса Web App, чтобы присоединиться к собранию. Skype для бизнеса Web App хранится на сервере переднего сервера и отправляется участнику собрания. 
  
Для Skype для бизнеса Server приложение "Собрания Skype" (в Windows) и Skype для бизнеса для Mac (на Mac) доступно в качестве замены Skype для бизнеса Web App, начиная с cu5, но для предоставления заменяющих приложений требуется дополнительная конфигурация, описанная в описании приложения "Включить собрания Skype" для замены [Skype для бизнеса Web App (необязательно).](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable) Если приложение "Собрания Skype" и Skype для бизнеса для Mac включены, пользователи скачав последнюю версию приложений из сети доставки содержимого (CDN) Microsoft 365 или Office 365, а не с сервера Skype для бизнеса. Для Skype для бизнеса Server 2019 единственным вариантом является использование приложения "Собрания Skype" и Skype для бизнеса для Mac.
  
Приложение "Собрания Skype" предлагает упрощенный браузер для скачивания и установки приложения и присоединения к собраниям, включая присоединение одним щелчком мыши для пользователей Internet Explorer. Приложение "Собрания Skype" также имеет множество улучшений по сравнению с Skype для бизнеса Web App для обеспечения надежности и работы с собраниями. 
  
> [!NOTE]
> В Skype для бизнеса Server 2015 CU5 или более поздней версии собрания, которые проводятся с помощью Skype для бизнеса Online, больше не будут отправлять пользователю без клиента Skype для бизнеса Web App; вместо этого ему будет отправлено приложение "Собрания Skype" (для Windows) или Skype для бизнеса для Mac (на Mac). В Skype для бизнеса Server 2015 CU5 или более поздней, если вы включаете приложение "Собрания Skype" для замены Skype для бизнеса [Web App (необязательно),](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable)пользователям без клиента будет отправлено приложение "Собрания Skype" или Skype для бизнеса для Mac вместо Skype для бизнеса Web App.

<a name="Network"> </a>

Если у пользователя Skype для бизнеса Web App или приложения "Собрания Skype" проблемы с подключением к собранию, скорее всего, в сетевой инфраструктуре организации не настроена поддержка Office 365, как описано в URL-адресах и диапазонах IP-адресов [Office 365.](https://support.office.com/article/Office-365-URLs-and-IP-address-ranges-8548a211-3fe7-47cb-abb1-355ea5aa88a2?ui=en-US&amp;rs=en-US&amp;ad=US) В этом случае собрание было создано пользователем Skype для бизнеса Online или Skype для бизнеса Server.

## <a name="known-issues-and-troubleshooting"></a>Известные проблемы и устранение неполадок
<a name="BKMK_Conferencing"> </a>

Для конечных пользователей доступна [веб-справка](https://aka.ms/smahelp) по этим приложениям. ИТ-специалистам следует знать о следующих проблемах:
  
- Если пользователь находится в сети, не [](meetings-clients.md#Network)настроенной для удовлетворения сетевых требований, многие функции приложения могут работать или не работать, и они могут не иметь возможности подключиться к собранию вообще.
    
- У некоторых пользователей могут быть компьютеры с корпоративным администрированием с отключенным разрешением на установку приложений. Для этих пользователей ни одно из [](https://products.office.com/skype-for-business/download-app?tab=tabs-1) приложений не [](https://products.office.com/skype-for-business/download-app?tab=tabs-2) является вариантом, но пользователи смартфонов и планшетов могут устанавливать дорогостоящие мобильные клиенты, которые они могут использовать для участия в собраниях.
    
    Другие проблемы с установкой также освещаются в [справке.](https://support.office.com/article/Trouble-installing-the-Skype-for-Business-Web-App-plug-in-958fc5f1-2d6f-42e3-815d-a9516c591274?ui=en-US&amp;rs=en-US&amp;ad=US) 
    
- Пользователи могут видеть предупреждение брандмауэра при первом запуске приложения для собраний. Им может быть предложено открыть порты для оптимизации работы, и для этого могут потребоваться права администратора на компьютере, на который у них нет. Приложение по-прежнему должно функционировать, и пользователь может безопасно отклодить открытие запрашиваемой портов. 
    
- Необходимо включить ActiveX [без](https://support.office.com/article/Turn-off-ActiveX-filtering-for-Skype-for-Business-Web-App-b6de8ff6-ac7e-4e2f-b18c-2f13db643c41?ui=en-US&amp;rs=en-US&amp;ad=US) фильтрации в Internet Explorer, даже если IE не является браузером по умолчанию. В Skype для бизнеса Web App для общего доступа к звукам, видео и экрану необходим ActiveX - небольшой модуль, который добавляет дополнительные функции в веб-приложение или другую программу.
    
- Чтобы некоторые функции Skype для бизнеса Web App работали правильно, необходимо разрешить браузеру сохранять файлы [cookie](https://support.office.com/article/Allow-cookies-for-Skype-Meetings-App-Skype-for-Business-Web-App-2108276b-b5c3-484b-bf2b-dac6eeba4c93) на компьютере или устройстве.
    
- Вам может потребоваться включить поддержку [JavaScript](https://support.office.com/article/Turn-on-JavaScript-for-Skype-Meetings-App-Skype-for-Business-Web-App-3d997bf9-637c-4fe6-8ee3-9e62bfda52cd) в браузере, чтобы некоторые функции Skype для бизнеса Web App работали как ожидалось.
    
### <a name="aes-support"></a>Поддержка AES 

В Skype для бизнеса Server 2015 CU5 AES не поддерживается для ASP.NET 4.6, и это может привести к сбойу запуска приложения "Собрания Skype". Дополнительные сведения см. в требованиях к шифрованию, предъявляемым [ASP .NET 4.5.](../security/user-and-client-authentication.md#cryptographic-requirements-due-to-asp-net-45)
