# <a name="plan-for-meetings-clients-web-app-and-meetings-app"></a>Планирование для клиентов собрания (веб-приложения и приложения собрания)
 
**Сводка:** ИТ-специалистов необходимо проверить требования к поддержка Скайп для бизнеса веб-приложения и приложения собраний Скайп при планировании для Скайп Business Server. В этой статье не предназначен для пользователей из этих приложений.
  
После реализации Скайп для Business Server пользователи вашей организации предположительно будут иметь Скайп для клиента Business установлен как часть процесса развертывания. 
  
Более поздней версии для этих пользователей может создавать собрания и приглашать пользователей за пределами организации, и эти приглашенные участники собрания не может иметь любой версии Скайп для клиента Business. При этих пользователей щелкните URL-адрес для приглашении на собрание, нехватка клиента будут определяться и приглашаемому пользователю без Скайп для клиента Business будет предложено загрузить и установить клиент упрощенный, только для собраний, поэтому они могут присоединиться к собранию.
  
> [!NOTE]
> Скайп для бизнеса Web App и Скайп собраний приложения доступны только при попытке входа собрание без необходимости Скайп для бизнеса. Справка пользователя для этих приложений был создан [https://aka.ms/smahelp](https://aka.ms/smahelp). 
  
> [!NOTE]
> Не удается предварительно установите либо Скайп для бизнеса веб-приложения или приложения Скайп собраний, но [для смартфонов](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-1) и [планшетных](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-2) пользователи могут получить доступ для установки недорогим мобильных клиентов, которые они могут использовать собрания.
  
По умолчанию на сервере, содержащем собрания будет направлять пользователя, чтобы загрузить и установить Скайп для бизнеса Web App для присоединения к собранию. Скайп для бизнеса Web App хранится на сервере переднего плана и сообщение отправляется участников собрания. 
  
Для Скайп для Business Server Скайп собраний приложения (в Windows) и Скайп для бизнеса для Mac (в Mac) доступны в качестве замены для Скайп для бизнеса Web App, начиная с накопительным пакетом обновления 5, но предоставление приложения замены требуется дополнительная настройка описываются в [Включить приложение Скайп собраний для замены Скайп для бизнеса веб-приложения (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable).  Если включены приложения Скайп собраний и Скайп для бизнеса для Mac, пользователи будут загружать последней версии сервера приложений из Office 365 доставки содержимого сети (CDN), а не из вашей Скайп для Business server. Для Скайп Business Server 2019 с помощью приложения Скайп собраний и Скайп для бизнеса для Mac является единственным вариантом.
  
Приложение собраний Скайп предлагает упрощенный опыт для загрузки и установки приложения и присоединение к собраниям, включая присоединиться к одним щелчком мыши для пользователей Internet Explorer. Скайп собраний приложения также имеет много улучшений по Скайп для Business Web App для обеспечения надежности и качества собрания. 
  
> [!NOTE]
> По состоянию на Скайп Business Server 2015 накопительным пакетом обновления 5 или более поздней версии, собрания, которые проводятся с помощью Скайп для бизнеса в Интернет будет больше не отправлять clientless пользователя Скайп для бизнеса Web App, они будут вместо отправляться Скайп собраний приложения (в Windows) или Скайп для бизнеса для Mac (в Mac). По состоянию на Скайп для Business Server 2015 накопительным пакетом обновления 5 или более поздней версии, если вы [Включить приложение Скайп собраний для замены Скайп для бизнеса веб-приложения (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable), clientless пользователей будут отправляться приложения Скайп собраний или Скайп для бизнеса для Mac вместо Скайп для бизнеса Web App.

## <a name="known-issues-and-troubleshooting"></a>Известные проблемы и устранение неполадок
<a name="BKMK_Conferencing"> </a>

Для конечных пользователей будет доступна [Справка в Интернете](https://aka.ms/smahelp) для этих приложений. ИТ-специалистам следует иметь в виду следующие моменты:
  
- Если пользователь находится в сети не настроены в соответствии с [требованиями сети](meetings-clients.md#Network), многие функции приложения может или работает не так, и они не смогут подключаться к собранию все.
    
- Некоторые пользователи могут иметь корпоративной администрировать компьютеры с отключенного разрешения для установки приложения. для этих пользователей не заданы параметры приложения — это параметр, но [для смартфонов](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-1) и [планшетных](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-2) пользователи могут получить доступ для установки недорогим мобильных клиентов, которые они могут использовать собрания.
    
    Другие проблемы установки также рассматривается в [разделах справки](https://support.office.com/en-us/article/Trouble-installing-the-Skype-for-Business-Web-App-plug-in-958fc5f1-2d6f-42e3-815d-a9516c591274?ui=en-US&amp;rs=en-US&amp;ad=US). 
    
- Пользователи могут видеть брандмауэр предупреждение при первом запуске приложения собрания. Может появиться запрос для открытия портов для Оптимизация работы и этого может потребоваться прав администратора на компьютере, на котором они не могут иметь. Приложение должно работать и пользователь может безопасно отклонить открыть запрошенный порты. 
    
- Необходимо [Включить ActiveX без фильтрации](https://support.office.com/en-us/article/Turn-off-ActiveX-filtering-for-Skype-for-Business-Web-App-b6de8ff6-ac7e-4e2f-b18c-2f13db643c41?ui=en-US&amp;rs=en-US&amp;ad=US) в Internet Explorer, даже если IE не браузере по умолчанию. В Скайп для бизнеса Web App, элемент управления ActiveX — небольшой модуль, который добавляет дополнительные функции для веб-приложения или другую программу — необходим для аудио-, видео и совместное использование рабочего стола.
    
- Для некоторых функций Скайп для бизнеса Web App для правильной работы необходимо разрешить браузера для [сохранения файлов cookie](https://support.office.com/en-us/article/Allow-cookies-for-Skype-Meetings-App-Skype-for-Business-Web-App-2108276b-b5c3-484b-bf2b-dac6eeba4c93) на компьютере или мобильном устройстве.
    
- Вы может необходимо включить [JavaScript](https://support.office.com/en-us/article/Turn-on-JavaScript-for-Skype-Meetings-App-Skype-for-Business-Web-App-3d997bf9-637c-4fe6-8ee3-9e62bfda52cd) поддерживает в браузере для некоторых Скайп для компонентов Business Web App для работает должным образом.
