# <a name="plan-for-meetings-clients-web-app-and-meetings-app"></a>Планирование для клиентов собраний (Skype для бизнеса Web App и приложение "Собрания Skype")
 
**Сводка:** ИТ-специалистам следует ознакомиться с требованиями к системе Skype для бизнеса Web App и приложения "Собрания Skype" при планировании использования Skype для бизнеса Server. Эта статья не предназначена для пользователей этих приложений.
  
После внедрения Skype для бизнеса Server, вероятно, в процессе развертывания на устройствах пользователей в вашей организации будет установлен клиент Skype для бизнеса. 
  
После этого пользователи смогут создавать собрания и приглашать на них пользователей извне организации, а у этих приглашенных пользователей может не быть никаких версий клиента Skype для бизнеса. Если такие пользователи перейдут по URL-адресу в приглашении на собрания, система определит, что у этих пользователей отсутствует клиент. Приглашенным пользователям, на устройствах которых нет клиента Skype для бизнеса, будет предложено загрузить и установить компактный клиент, предназначенный только для собраний, чтобы эти пользователи смогли присоединиться к собранию.
  
> [!NOTE]
> Skype для бизнеса Web App и приложение "Собрания Skype" доступны в случае, если пользователь, на устройстве которого нет клиента Skype для бизнеса, пытается войти на собрание. Справка для пользователей этих приложений доступна по адресу [https://aka.ms/smahelp](https://aka.ms/smahelp). 
  
> [!NOTE]
> Невозможно предустановить Skype для бизнеса Web App и приложение "Собрания Skype", но пользователи [смартфонов](https://products.office.com/ru-RU/skype-for-business/download-app?tab=tabs-1) и [планшетов](https://products.office.com/ru-RU/skype-for-business/download-app?tab=tabs-2) могут установить недорогие мобильные клиенты, с помощью которых можно участвовать в собраниях.
  
По умолчанию сервер, на котором размещено собрание, предложит пользователю загрузить и установить Skype для бизнеса Web App, чтобы присоединиться к собранию. Skype для бизнеса Web App хранится на интерфейсном сервере и отправляется участнику собрания. 
  
Для Skype для бизнеса Server приложения "Собрания Skype" (на устройствах с Windows) и Skype для бизнеса на Mac (на устройствах Mac) доступны в качестве замены Skype для бизнеса Web App, начиная с версии CU5. Тем не менее, для предоставления этих заменяющих приложений требуется дополнительная настройка, описанная в статье [Включение приложения "Собрания Skype" для замены Skype для бизнеса Web App (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable).  Если приложение "Собрания Skype" и Skype для бизнеса на Mac включены, пользователи будут загружать последние версии этих приложений из сети доставки содержимого (CDN) Office 365, а не с вашего сервера Skype для бизнеса. При работе со Skype для бизнеса Server 2019 можно использовать только приложение "Собрания Skype" и Skype для бизнеса на Mac.
  
В приложении "Собрания Skype" предоставляются упрощенные возможности веб-браузера для загрузки и установки клиентского приложения и присоединения к собраниям. Пользователи Internet Explorer смогут присоединяться к собраниям одним щелчком. В приложении "Собрания Skype" также реализован ряд усовершенствований по сравнению с клиентом Skype для бизнеса Web App: повышена надежность и расширена функциональность для собраний. 
  
> [!NOTE]
> В Skype для бизнеса Server 2015 CU5 и в более поздних версиях собрания, проводящиеся с помощью Skype для бизнеса Online, больше не будут предлагать пользователям, у которых нет клиента, использовать Skype для бизнеса Web App. Вместо этого таким пользователям будет предложено использовать приложение "Собрания Skype" (на устройствах с Windows) или Skype для бизнеса на Mac (на устройствах Mac). В Skype для бизнеса Server 2015 CU5 и в более поздних версиях, если [включить приложение "Собрания Skype" для замены Skype для бизнеса Web App (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable), пользователям без клиента будет предложено использовать приложение "Собрания Skype" или Skype для бизнеса на Mac вместо Skype для бизнеса Web App.

### <a name="skype-meetings-app"></a>Приложение "Собрания Skype"

Приложение "Собрания Skype" запускается в качестве приложения на компьютерах под управлением Windows 10, Windows 8.1, Windows 8 или Windows 7 с установленной 32- или 64-разрядной версией Internet Explorer 11. 
  
Сведения о других зависимостях см. в статье [Поддерживаемые платформы приложения "Собрания Skype"](https://support.office.com/ru-RU/client/results?Shownav=true&amp;lcid=1033&amp;ns=SKFBWA&amp;version=15&amp;omkt=en-US&amp;ver=15&amp;HelpID=SfBWebApp4001).
  
### <a name="skype-for-business-for-mac"></a>Skype для бизнеса на Mac

Skype для бизнеса на Mac работает на компьютерах под управлением macOS 10.8 или более поздней версии. 

## <a name="hardware-requirements"></a>Требования к оборудованию
<a name="OS-Browser"> </a>

Требования к компьютерному оборудованию определяются операционной системой и браузером. Для использования голосовой связи и телефонии необходимы микрофон и динамики, гарнитура с микрофоном или аналогичное устройство, совместимое с компьютером. Для использования видео необходимо совместимое с компьютером видеоустройство. Подробные сведения о поддержке видеооборудования и ожидаемом качестве видео см. в статье [Требования к клиентскому видеотрафику Skype для бизнеса](video-resolutions.md).
  
## <a name="network-requirements"></a>Требования к сети
<a name="Network"> </a>

Если у пользователя Skype для бизнеса Web App или приложения "Собрания Skype" возникают трудности с подключением к собранию, возможная причина состоит в том, что сетевая инфраструктура организации не настроена для поддержки Office 365, как описано в статье [URL-адреса и диапазоны IP-адресов Office 365](https://support.office.com/ru-RU/article/Office-365-URLs-and-IP-address-ranges-8548a211-3fe7-47cb-abb1-355ea5aa88a2?ui=en-US&amp;rs=en-US&amp;ad=US). Такая ситуация может возникнуть, если собрание было создано пользователем Skype для бизнеса Online или Skype для бизнеса Server. 
  
Если пользователь находится в сети, которая не была настроена описанным образом, многие функции приложения могут не работать, а также может полностью отсутствовать возможность подключиться к собранию.

## <a name="known-issues-and-troubleshooting"></a>Известные проблемы и устранение неполадок
<a name="BKMK_Conferencing"> </a>

Для конечных пользователей доступна [справка](https://aka.ms/smahelp) по этим приложениям. ИТ-специалистам необходимо учитывать следующее:
  
- Если пользователь находится в сети, которая не была настроена в соответствии с требованиями, перечисленными в разделе [Требования к сети](meetings-clients.md#Network), многие функции приложения могут не работать, а также может полностью отсутствовать возможность подключиться к собранию.
    
- У некоторых пользователей могут быть компьютеры с корпоративным администрированием, при этом разрешение на установку приложений может быть отключено. Такие пользователи не смогут использовать ни одно из перечисленных приложений. Тем не менее, пользователи [смартфонов](https://products.office.com/ru-RU/skype-for-business/download-app?tab=tabs-1) и [планшетов](https://products.office.com/ru-RU/skype-for-business/download-app?tab=tabs-2) могут установить недорогие мобильные клиенты для участия в собраниях.
    
    Другие проблемы, связанные с установкой, также описаны в [разделах справки](https://support.office.com/ru-RU/article/Trouble-installing-the-Skype-for-Business-Web-App-plug-in-958fc5f1-2d6f-42e3-815d-a9516c591274?ui=en-US&amp;rs=en-US&amp;ad=US). 
    
- При первом запуске приложения для собраний на экране может появляться предупреждение брандмауэра. Пользователям может быть предложено открыть определенные порты для оптимизации работы системы. Для этого также могут потребоваться права администратора на локальном компьютере (таких прав у пользователя может не быть). В таких случаях приложение все равно будет работать, а пользователи могут безопасно отказаться открывать запрошенные порты. 
    
- В Internet Explorer необходимо включить [ActiveX без фильтрации](https://support.office.com/ru-RU/article/Turn-off-ActiveX-filtering-for-Skype-for-Business-Web-App-b6de8ff6-ac7e-4e2f-b18c-2f13db643c41?ui=en-US&amp;rs=en-US&amp;ad=US), даже если IE не является браузером по умолчанию. В Skype для бизнеса Web App элемент ActiveX (это небольшой модуль, реализующий дополнительные функции для веб-приложений и других программ) требуется для использования функций звука и видео, а также для демонстрации экрана.
    
- Для правильной работы некоторых функций Skype для бизнеса Web App необходимо разрешить в браузере [сохранение файлов cookie](https://support.office.com/ru-RU/article/Allow-cookies-for-Skype-Meetings-App-Skype-for-Business-Web-App-2108276b-b5c3-484b-bf2b-dac6eeba4c93) на компьютере или устройстве.
    
- Для правильной работы некоторых функций Skype для бизнеса Web App может потребоваться [включить поддержку JavaScript](https://support.office.com/ru-RU/article/Turn-on-JavaScript-for-Skype-Meetings-App-Skype-for-Business-Web-App-3d997bf9-637c-4fe6-8ee3-9e62bfda52cd) в браузере.

Дополнительные сведения см. в статье [Требования к шифрованию в связи с использованием ASP .NET 4.5](../security/user-and-client-authentication.md#cryptographic-requirements-due-to-asp-net-45).
