# <a name="plan-for-meetings-clients-web-app-and-meetings-app"></a>Планирование для клиентов собраний (приложение для веб-приложений и собраний)
 
**Сводка:** ИТ-специалисты должны ознакомиться с требованиями к поддержке приложения Skype для бизнеса Web App и собрания Skype при планировании Skype для бизнеса Server. Эта статья не предназначена для пользователей этих приложений.
  
После внедрения Skype для бизнеса Server пользователи вашей организации могут предположить, что клиент Skype для бизнеса установлен в рамках процесса развертывания. 
  
Позже эти пользователи могут создавать собрания и приглашать пользователей за пределами Организации, и эти приглашенные на собрание могут не иметь никакой версии клиента Skype для бизнеса. Когда эти пользователи щелкают URL-адрес для приглашения на собрание, будет обнаружено отсутствие клиента, и приглашению без клиента Skype для бизнеса будет предложено скачать и установить простой клиент только для собраний, чтобы присоединиться к собранию.
  
> [!NOTE]
> Приложение Skype для бизнеса Web App и собрания Skype доступно только при попытке войти на собрание, не используя Skype для бизнеса. Справка для пользователей для этих приложений находится на сайте [https://aka.ms/smahelp](https://aka.ms/smahelp) . 
  
> [!NOTE]
> Вы не можете предварительно установить приложение Skype для бизнеса Web App или собрания Skype, но [смарт-телефоны](https://products.office.com/skype-for-business/download-app?tab=tabs-1) и [Планшетные](https://products.office.com/skype-for-business/download-app?tab=tabs-2) пользователи могут устанавливать недорогие мобильные клиенты, которые они могут использовать для участия в собрании.
  
По умолчанию сервер, на котором размещено собрание, направляет пользователю запрос на загрузку и установку Skype для бизнеса Web App, чтобы присоединиться к собранию. Веб-приложение Skype для бизнеса хранится на сервере переднего плана и отправляется участнику собрания. 
  
Приложение для собраний Skype для бизнеса Server, приложение для собраний Skype (в Windows) и Skype для бизнеса для Mac (на Mac) доступны как замены для Skype для бизнеса Web App, начиная с CU5, но предоставление приложений для замены требует дополнительной настройки, описанной в статье [Включение приложения для собраний Skype для замены Skype для бизнеса Web App (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable). Если приложение для собраний Skype и Skype для бизнеса для Mac включены, пользователи будут скачивать последнюю версию приложений из Microsoft 365 или Office 365 для сети доставки содержимого (CDN), а не из Skype для бизнеса Server. Для Skype для бизнеса Server 2019 использование приложения собраний Skype и Skype для бизнеса для Mac является единственным вариантом.
  
Приложение для собраний Skype предоставляет упрощенный интерфейс браузера для загрузки и установки приложения и присоединения к собраниям, в том числе один щелчок для пользователей Internet Explorer. Кроме того, приложение для собраний Skype значительно улучшилось по сравнению с веб-приложением Skype для бизнеса для обеспечения надежности и взаимодействия с собраниями. 
  
> [!NOTE]
> В Skype для бизнеса Server 2015 CU5 или более поздней версии собрания, хранящиеся в Skype для бизнеса Online, больше не будут отправлять пользователям, не подключенным к Skype для бизнеса, веб-приложение Skype для бизнеса, вместо этого они будут отправлены приложением для собраний Skype (в Windows) или Skype для бизнеса для Mac (на Mac). На Skype для бизнеса Server 2015 CU5 или более поздней версии, если вы [включите приложение для собраний Skype для замены Skype для бизнеса Web App (необязательно)](../../deploy/deploy-clients/deploy-web-downloadable-clients.md#SMA_Enable), пользователи, не являющиеся клиентами, будут отправлять приложение для собраний Skype или Skype для бизнеса для Mac, а не Skype для бизнеса Web App.

Мы рекомендуем использовать последнюю версию Skype для бизнеса для Mac, которая поддерживает сценарии анонимного подключения, перемещая их вперед.

<a name="OS-Browser"> </a>

Требования к оборудованию компьютера определяются операционной системой и браузером. Для функций голосовой связи и телефонии требуются микрофон и динамики, гарнитура с микрофоном или аналогичное устройство, совместимое с компьютером. Для функций видео требуется видеоустройство, совместимое с компьютером. Для получения подробных сведений о поддержке видеоаппарата и ожидаемом качестве видео обратитесь к разделу [разрешение для клиентского видео в Skype для бизнеса](video-resolutions.md).

## <a name="known-issues-and-troubleshooting"></a>Известные проблемы и устранение неполадок
<a name="BKMK_Conferencing"> </a>

Для конечных пользователей [интерактивную справку](https://aka.ms/smahelp) по этим приложениям легко получить. ИТ-специалисты должны учитывать следующие моменты.
  
- Если пользователь входит в сеть, не настроенную в соответствии с [требованиями к сети](meetings-clients.md#Network), многие функции приложений могут или не работать, и они не смогут подключаться к собранию вообще.
    
- Некоторые пользователи могут иметь корпоративные административные компьютеры с отключенными разрешениями на установку приложений. для этих пользователей ни одно приложение не является параметром, но [смарт-телефоны](https://products.office.com/skype-for-business/download-app?tab=tabs-1) и [Планшетные](https://products.office.com/skype-for-business/download-app?tab=tabs-2) пользователи могут устанавливать недорогие мобильные клиенты, которые могут использоваться для участия в собрании.
    
    Кроме того, в [справочных материалах](https://support.office.com/article/Trouble-installing-the-Skype-for-Business-Web-App-plug-in-958fc5f1-2d6f-42e3-815d-a9516c591274?ui=en-US&amp;rs=en-US&amp;ad=US)рассматриваются другие проблемы, связанные с установкой. 
    
- При первом запуске приложения собраний пользователи могут видеть предупреждение о брандмауэре. Возможно, вам будет предложено открыть порты для оптимизации взаимодействия, и для этого может потребоваться привилегия администратора на компьютере, у которого они могут отсутствовать. Приложение по-прежнему должно работать, и пользователь может безопасно отклонить, чтобы открыть запрошенные порты. 
    
- [Элемент ActiveX должен быть включен без фильтрации](https://support.office.com/article/Turn-off-ActiveX-filtering-for-Skype-for-Business-Web-App-b6de8ff6-ac7e-4e2f-b18c-2f13db643c41?ui=en-US&amp;rs=en-US&amp;ad=US) в Internet Explorer, даже если IE не является браузером по умолчанию. В Skype для бизнеса Web App элемент управления ActiveX — это небольшой модуль, который добавляет дополнительные функции для веб-приложения или другой программы — необходим для использования звука, видео и экрана.
    
- Чтобы некоторые функции Skype для бизнеса Web App работали правильно, необходимо разрешить браузеру [сохранять файлы cookie](https://support.office.com/article/Allow-cookies-for-Skype-Meetings-App-Skype-for-Business-Web-App-2108276b-b5c3-484b-bf2b-dac6eeba4c93) на компьютере или устройстве.
    
- Чтобы некоторые функции Skype для бизнеса Web App работали должным образом, вам может потребоваться [включить поддержку JavaScript](https://support.office.com/article/Turn-on-JavaScript-for-Skype-Meetings-App-Skype-for-Business-Web-App-3d997bf9-637c-4fe6-8ee3-9e62bfda52cd) в браузере.
    
### <a name="aes-support"></a>Поддержка AES 

В Skype для бизнеса Server 2015 CU5 алгоритм AES не поддерживается для ASP.NET 4,6, и это может привести к сбою запуска приложения для собраний Skype. [Требования к криптографии из-за наличия ASP .net 4,5](../security/user-and-client-authentication.md#cryptographic-requirements-due-to-asp-net-45) более подробные сведения.
