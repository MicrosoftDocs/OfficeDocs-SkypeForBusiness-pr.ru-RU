# <a name="plan-for-security-in-skype-for-business-server-2015"></a>Планирование системы безопасности в Skype для бизнеса Server 2015
 
Скайп содержимого безопасности Business Server 2015. 
  
Это содержимое относится к Скайп безопасности Business Server 2015. Безопасность — чрезвычайно обширная тема. Безопасность достигает через каждый компонент Скайп Business Server 2015, а также баз данных, служб и оборудования, входящих в состав экосистеме. Этот контент является дополнением для безопасности разделов, посвященных Скайп для Business Server 2015.
