# <a name="plan-for-security-in-skype-for-business-server"></a>Планирование безопасности в Skype для бизнеса Server 
 
Skype для бизнеса Server безопасности. 
  
Это содержимое относится к Skype для бизнеса Server безопасности. Безопасность — это очень широкая тема. Безопасность достигает всех функций Skype для бизнеса Server, а также баз данных, служб и оборудования, которые составляют экосистему. Это содержимое является дополнением для определенных тем безопасности в отношении Skype для бизнеса Server.
