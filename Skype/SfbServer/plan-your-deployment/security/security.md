# <a name="plan-for-security-in-skype-for-business-server"></a>Планирование безопасности в Skype для бизнеса Server 
 
Содержимое для обеспечения безопасности в Skype для бизнеса Server. 
  
Это содержимое относится к безопасности Skype для бизнеса Server. Безопасность — чрезвычайно обширная тема. Безопасность достигает всех функций Skype для бизнеса Server, а также баз данных, служб и оборудования, образующих экосистему. Этот материал является дополнением для конкретных вопросов безопасности, касающихся Skype для бизнеса Server.
