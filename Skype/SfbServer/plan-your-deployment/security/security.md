# <a name="plan-for-security-in-skype-for-business-server"></a>Планирование безопасности в Skype для бизнеса Server 
 
Содержимое системы безопасности Skype для бизнеса Server. 
  
Это содержимое относится к безопасности Skype для бизнеса Server. Безопасность — это очень обширная тема. Безопасность достигается во всех функциях Skype для бизнеса Server, а также в базах данных, службах и оборудовании, которые составляют экосистему. Это содержимое является дополнением к определенным темам безопасности, касающымся Skype для бизнеса Server.
