# <a name="windows-powershell-and-skype-for-business-server-management-tools"></a>Windows PowerShell Skype для управления бизнес-сервером
 
В Skype для бизнеса Server средства управления реализуются с Windows PowerShell. Windows PowerShell включает среду командной строки, команды, определенные продуктом, и полный язык скриптов. Инструменты Skype для бизнеса Server, реализуемые с Windows PowerShell, включают следующие: 
  
- **Строитель топологии**. Вы используете Topology Builder для создания, настройки и публикации запланированной топологии, и она проверяет топологию перед началом серверных установок. При установке Skype для бизнеса Server на отдельных серверах серверы читают опубликованную топологию в рамках процесса установки, а программа установки развертывает сервер, как указано в топологии. После настройки сведения о конфигурации автоматически реплицируются на все серверы. Компоненты можно добавлять в развертывание только с помощью Topology Builder.
    
- **Оболочка управления Серверами Skype** для бизнеса. Для полного управления развертыванием можно использовать командную строку управления Skype для бизнес-серверов.
    
- **Панель управления серверами Skype** для бизнеса . Вы можете использовать пользовательский интерфейс панели управления Skype для бизнес-серверов для управления наиболее распространенными задачами в развертывании.
    
Эти средства используют Windows PowerShell для управления развертыванием, включая около 550 определенных продуктов. В основном для управления проверкой подлинности, а также прав и разрешений пользователей используются в Skype для бизнеса Server. Широкий спектр командлетов доступен для управления проверкой подлинности, включая командлеты для проверки подлинности с помощью сертификатов и персональных идентификационных номеров (ПИН-кодов). Кроме того, ряд cmdlets позволяют использовать новую функцию управления доступом Role-Based доступа (RBAC) для делегирования административного управления Skype для бизнеса Server. Подробные сведения о коммлетах Skype для бизнеса Server см. в [материале Skype for Business Server Management Shell.](../../manage/management-shell.md)
  
Функции безопасности скриптов для Windows PowerShell специально разработаны для предотвращения некоторых проблем безопасности, связанных со сценарием, старых технологий, включая Microsoft Visual Basic Scripting Edition (VBScript). Функции Windows PowerShell безопасности предназначены для создания среды, в которой пользователи не могут легко или не знаком запускать сценарии. По умолчанию Windows PowerShell функции безопасности. Вы можете изменить состояние этих функций для удовлетворения ваших потребностей в скриптах и различных целей безопасности. Это не говорит о том, что оболочка не позволяет пользователям запускать сценарии. Скорее, оболочка затрудняет по умолчанию запуск скриптов пользователями, не осознавая, что они это делают. Подробные сведения [см. в Windows PowerShell Script Security.](/previous-versions/msdn10/gg261722(v=msdn.10))
