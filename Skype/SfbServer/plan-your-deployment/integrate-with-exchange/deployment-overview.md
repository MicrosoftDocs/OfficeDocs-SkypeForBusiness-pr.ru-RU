# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Обзор процесса развертывания для интеграции локальной единой системы обмена сообщениями и Skype для бизнеса
 
**Сводка:** Просмотрите эту тему при планировании интеграции Skype для бизнеса Server с Exchange 2013 или 2016 г.
  
 Если вы хотите интегрировать Exchange единой системы обмена сообщениями (единой системы обмена сообщениями) с Skype для бизнеса Server, необходимо выполнить задачи, описанные в этом разделе. Также убедитесь, что вы просмотрите лучшие практики планирования и [развертывания,](unified-messaging.md)описанные в плане интеграции Exchange единой системы обмена сообщениями в Skype для бизнеса . В этом разделе предполагается Skype для бизнеса Server что вы развернули Skype для бизнеса Server с сервером-посредником и включили пользователей для Skype для бизнеса Server, но вы, возможно, не выполнили все действия по развертыванию и настройке, чтобы включить Корпоративная голосовая связь, как описано в deploy [Корпоративная голосовая связь в Skype для бизнеса Server в](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) документации по развертыванию.
 
> [!NOTE]
> Exchange В Skype для бизнеса Server 2019 г. единая система обмена сообщениями больше недоступна, телефонная система используется для записи сообщений голосовой почты, а затем оставляет запись в Exchange почтовом ящике пользователя. Дополнительные [сведения см. в облачная голосовая почта службе](../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) Plan облачная голосовая почта.
