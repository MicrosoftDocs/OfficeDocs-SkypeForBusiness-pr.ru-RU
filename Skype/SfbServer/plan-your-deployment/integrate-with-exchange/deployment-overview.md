# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Обзор процесса развертывания для интеграции локальной единой системы обмена сообщениями и Skype для бизнеса
 
**Сводка:** Просмотрите этот раздел при планировании интеграции Skype для бизнеса Server с Exchange 2013 или 2016.
  
 Чтобы интегрировать единую систему обмена сообщениями Exchange со Skype для бизнеса Server, необходимо выполнить задачи, описанные в этом разделе. Кроме того, обязательно просмотрите лучшие методики планирования и развертывания, описанные в плане интеграции единой системы обмена сообщениями [Exchange в Skype для бизнеса.](unified-messaging.md) В этом разделе предполагается, что вы развернули Skype для бизнеса Server с совместимым сервером-посредником и включили пользователей skype для бизнеса Server, но, возможно, вы не выполнили все действия по развертыванию и настройке для включения Корпоративная голосовая связь, как описано в разделе ["Развертывание Корпоративная голосовая связь в Skype для](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) бизнеса Server" документации по развертыванию.
 
> [!NOTE]
> Как уже было известно, единая система обмена сообщениями Exchange больше недоступна в Skype для бизнеса Server 2019, который использует телефонную систему для записи сообщений голосовой почты, а затем оставляет запись в почтовом ящике Exchange пользователя. Дополнительные [сведения см. в](../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) записи "Планирование облачной голосовой почты".
