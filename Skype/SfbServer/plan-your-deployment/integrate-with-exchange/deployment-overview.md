# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Обзор процесса развертывания для интеграции локальной единой системы обмена сообщениями и Skype для бизнеса
 
**Сводка:** Просмотрите эту тему при планировании интеграции Skype для бизнеса Server с Exchange 2013 или 2016.
  
 Если вы хотите интегрировать Exchange Unified Messaging (ЕДИНОЙ Системы обмена сообщениями) с Skype для бизнеса Server, необходимо выполнить задачи, описанные в этом разделе. Также убедитесь, что вы просмотрите лучшие практики планирования и развертывания, описанные в плане интеграции единой системы обмена сообщениями [Exchange в Skype для бизнеса.](unified-messaging.md) В этом разделе предполагается, что вы развернули Skype для бизнеса Server с сервером-посредником и включили пользователей для Skype для бизнес-сервера, но вы, возможно, не выполнили все действия по развертыванию и настройке, чтобы включить Корпоративная голосовая связь, как описано в разделе Развертывание [Корпоративная голосовая связь в Skype для](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) бизнеса Server в документации по развертыванию.
 
> [!NOTE]
> Exchange Unified Messaging, как уже было известно, больше недоступна в Skype для бизнеса Server 2019, который использует телефонную систему для записи сообщений голосовой почты, а затем оставить запись в почтовом ящике exchange пользователя. Дополнительные сведения см. в службе голосовой почты Plan [Cloud.](../../../sfbhybrid/hybrid/plan-cloud-voicemail.md)
