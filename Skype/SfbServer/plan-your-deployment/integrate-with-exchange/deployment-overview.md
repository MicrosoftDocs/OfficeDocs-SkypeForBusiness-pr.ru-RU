# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Обзор процесса развертывания для интеграции локальной единой системы обмена сообщениями и Skype для бизнеса
 
**Сводка:** Ознакомьтесь с этим разделом в процессе планирования интеграции Skype для бизнеса Server с Exchange 2013 или 2016.
  
 Если вы хотите интегрировать единую систему обмена сообщениями Exchange с приложением Skype для бизнеса Server, необходимо выполнить задачи, описанные в этом разделе. Также убедитесь в том, что вы просматриваете рекомендации по планированию и развертыванию, описанные в разделе [Планирование интеграции Exchange единой системы обмена сообщениями в Skype для бизнеса](unified-messaging.md). В этой статье предполагается, что вы развернули сервер Skype для бизнеса на размещенном сервере, а вы включили пользователей для Skype для бизнеса Server, но вы не выполнили все действия по развертыванию и настройке для включения корпоративной голосовой связи. описано в разделе [Развертывание корпоративной голосовой связи в Skype для бизнеса Server](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) в документации по развертыванию.
 
> [!NOTE]
> Единая система обмена сообщениями Exchange больше не доступна в Skype для бизнеса Server 2019, который использует телефонную систему для записи голосовых сообщений, а затем оставляет запись в почтовом ящике Exchange пользователя. Дополнительные сведения приведены в разделе [планирование облачной службы голосовой почты](../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) .
