# <a name="plan-for-unified-contact-store-in-skype-for-business-server-2015"></a>План для единого хранилища контактов в Skype для бизнеса Server 2015
 
**Сводка:** Ознакомьтесь с этим разделом в процессе планирования интеграции Skype для бизнеса Server с Exchange 2013 или 2016.
  
Единое хранилище контактов обеспечивает согласованное взаимодействие между продуктами Microsoft Office и позволяет пользователям хранить все контактные данные в Exchange 2013, но обеспечивает возможность их глобального доступа в Skype для бизнеса, Exchange и Outlook. и Outlook Web Access.
  
## <a name="requirements-for-unified-contact-store"></a>Требования к единому хранилищу контактов

Чтобы внедрить единое хранилище контактов в Skype для бизнеса Server, выполните указанные ниже действия.
  
- На вашем компьютере должны быть установлены Skype для бизнеса Server и Exchange 2013 или 2016.
    
- Пользователи должны использовать Skype для бизнеса для переноса контактов из Skype для бизнеса Server в Exchange 2013 и 2016.
    
- Почтовые ящики пользователей должны быть перенесены в Exchange 2013.
    
- Требуется проверка подлинности серверов в Skype для бизнеса Server и Exchange 2013 или 2016.
    
    > [!NOTE]
    > Подробные сведения о настройке проверки подлинности между Skype для бизнеса Server и Exchange 2013 и 2016: [Управление приложениями для проверки подлинности серверов (OAuth) и партнерских программ в Skype для бизнеса Server](../../manage/authentication/server-to-server-and-partner-applications.md) в документации по эксплуатации.
