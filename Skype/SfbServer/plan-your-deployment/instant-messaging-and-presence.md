## <a name="plan-for-instant-messaging-and-presence-in-skype-for-business-server"></a>Планирование обмена мгновенными сообщениями и присутствия в Skype для бизнеса Server

Серверы переднего сервера предоставляют основные функции Skype для бизнеса Server, такие как обмен мгновенными сообщениями и присутствие, и включаются в каждое развертывание Skype для бизнеса Server. Доступно два выпуска: Skype для бизнеса Server Enterprise Edition, предназначенный в основном для крупных организаций, и Skype для бизнеса Server Standard Edition, предназначенный в первую очередь для небольших организаций, которым требуется меньшее количество инвестиций в оборудование и не требуются полные варианты обеспечения высокой доступности. Оба выпуска поддерживают все рабочие нагрузки Skype для бизнеса Server, включая мгновенные звонки, присутствие, Корпоративная голосовая связь.
  
Обмен мгновенными сообщениями дает пользователям возможность общаться друг с другом в режиме реального времени со своих компьютеров, используя текстовые сообщения. Поддерживаются как двухсторонние, так и многосторонние сеансы обмена мгновенными сообщениями. Участник двухсторонней текстовой беседы может в любое время пригласить третьего участника. Когда это происходит, окно беседы изменяется, чтобы поддерживать функции конференции.
  
Сведения о присутствии предоставляют пользователям сведения о состоянии других пользователей в сети. Сведения о состоянии присутствия пользователя помогают другим пользователям решить, следует ли связаться с пользователем и использовать ли обмен мгновенными сообщениями, телефон или электронную почту. Функция присутствия предлагает обмен мгновенными сообщениями всегда, когда это возможно, но также сообщает, когда пользователь находится на собрании или вне офиса, указывая, что обмен мгновенными сообщениями невозможен. Это состояние присутствия отображается как значок присутствия в Skype для бизнеса и других приложениях с учетом присутствия, включая клиент обмена сообщениями и совместной работы Microsoft Outlook, технологии Microsoft SharePoint и Microsoft Office. Значок присутствия представляет текущую доступность и готовность пользователя к общению. 
  
### <a name="technical-requirements"></a>Технические требования

Обмен мгновенными сообщениями и присутствие всегда работают в пулах переднего сервера Enterprise Edition и на серверах Standard Edition. For information on supported hardware, operating systems, and database software, see  [Certified Gateways](../../SfbPartnerCertification/certification/infra-gateways.md),  [Requirements for your Skype for Business 2015 environment,](requirements-for-your-environment/requirements-for-your-environment.md)and [Infrastructure requirements for Skype for Business Server 2019](../../SfBServer2019/plan/system-requirements.md).
  
### <a name="enabling-communication-with-external-users"></a>Обеспечение связи с внешними пользователями

Вы можете значительно увеличить преимущества своих инвестиций в Skype для бизнеса Server, позволяя пользователям общаться с внешними пользователями. К внешним пользователям относятся:
  
- Удаленные пользователи: пользователи вашей организации, работающие за пределами брандмауэра и использующие ноутбуки или другие устройства Skype для бизнеса Server.
    
- Федераированные пользователи: пользователи из компаний, с которыми вы работаете, которые также работают со Skype для бизнеса Server. Чтобы ваши пользователи могли легко общаться с такими пользователями, с этими компаниями создаются отношения федерации. 
    
- Пользователи Skype: пользователи Skype для бизнеса могут связаться с сотнями миллионов пользователей Skype с помощью мгновенных, голосовых и видеосвязи.
    
> [!NOTE]
> AOL, Yahoo и Google Talk больше не поддерживаются. 
  
> [!NOTE]
> Чтобы включить любой из этих сценариев или все эти сценарии, необходимо развернуть edge Server для обеспечения безопасного взаимодействия между развертыванием Skype для бизнеса Server и внешними пользователями. Удаленные пользователи и пользователи в федера организациях смогут видеть присутствие друг друга и общаться с помощью мгновенных данных. 
  
> [!NOTE]
> Протокол XMPP поддерживается только для сценариев сертификации совместной работы с единой платформой взаимодействия (UCCP) (JITC). 
  
### <a name="archiving-im-content"></a>Архивное содержимое мгновенных мгновенных мгновенных содержимого

Skype для бизнеса Server предоставляет функции, которые можно использовать, если ваша организация должна соблюдать нормативные требования. С помощью архивации можно архивировать контент мгновенных сообщений для всех пользователей в организации или только для некоторых указанных пользователей. For details, see [Plan for archiving in Skype for Business Server](archiving/archiving.md). 
  
Если вы также развернули Microsoft Exchange Server 2013, вы можете интегрировать архивировать данные Exchange с архивативами данных Skype для бизнеса Server и использовать одно средство для поиска архивных данных обоих типов. Дополнительные сведения см. в настройках Skype для бизнеса [Server для Exchange Server архива.](../deploy/integrate-with-exchange-server/use-exchange-archiving.md)
  
### <a name="topologies-and-components"></a>Topologies and components

Для обмена мгновенными сообщениями и сведениями о присутствии требуются только следующие компоненты:
  
- Серверы переднего сервера организации (известные как пул) или сервер Standard Edition. Функции обмена мгновенными сообщениями и сведениями о присутствии всегда включены на этих серверах. Дополнительные сведения о высокой доступности пула переднего уровня и управлении ими см. в этой [теме.](high-availability-and-disaster-recovery/high-availability.md)
    
- Балансировка нагрузки, если у вас есть пул переднего входа Enterprise Edition.
