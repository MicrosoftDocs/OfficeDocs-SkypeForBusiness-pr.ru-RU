# <a name="hardware-and-software-requirements-for-persistent-chat-server-in-skype-for-business-server-2015"></a>Требования к оборудованию и программному обеспечению для сохраняемой сервера чата в Skype для бизнеса Server 2015
 
**Сводка:** Ознакомьтесь с этой темой, чтобы узнать о требованиях к оборудованию и программному обеспечению для постоянного сервера чата в Skype для бизнеса Server 2015.
  
Стойкий сервер чата можно установить с помощью Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition. Требования зависят от того, какой выпуск Skype для бизнеса Server 2015 вы установили, и от требований к производительности вашего бизнеса. Enterprise Edition может поддерживать до 80 000 одновременно пользователей; Standard Edition может поддерживать до 20 000 одновременно пользователей. Постоянный чат состоит из переднего компонента, а также SQL базы данных.
  
Перед развертыванием постоянного сервера чата необходимо убедиться в том, что следующие требования к оборудованию и программному обеспечению будут выполнены:
  
- Оборудование, которое отвечает минимальным требованиям для поддержки Skype для бизнеса Server 2015, Persistent Chat Server, серверов баз данных и файлов. Дополнительные сведения см. в дополнительных сведениях о требованиях [к Серверу для Skype для бизнеса Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
    
- Поддерживаемая операционная система и программное обеспечение базы данных.
    
    Сведения о поддерживаемых операционных системах и программном обеспечении баз данных и требованиях к обновлению Windows см. в материале [Server requirements for Skype for Business Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
    
- Фронт-сервер Skype для бизнеса Server 2015. Передний конечный сервер является основой маршрутиации протокола инициирования сеанса (SIP), что позволяет общаться между компьютерами, работающими на стойком сервере чата, и функциями сохраняемого чата. 
    
- Программное обеспечение очереди сообщений. Используется службой сохраняемой системы чата и службой обеспечения соответствия требованиям при развертывании.
    
В следующих разделах описываются конкретные требования к серверу сохраняемой системы чата и базе данных, в которой хранится информация о сохраняемом чате.

> [!NOTE] 
> Постоянный чат доступен в Skype для бизнеса Server 2015, но больше не поддерживается в Skype для бизнеса Server 2019. Такие же функции доступны в Teams. Дополнительные сведения см. в [дополнительных сведениях о том, как начать работу с обновлением Microsoft Teams.](/microsoftteams/upgrade-start-here) Если вам нужно использовать постоянный чат, вы можете либо перенести пользователей, требующих этих функций, в Teams, либо продолжить использование Skype для бизнеса Server 2015. 
  
## <a name="front-end-server-requirements"></a>Требования к переднему серверу

Требования к переднему серверу зависят от того, развертывается ли стойкий чат-сервер в Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition.
  
- Если вы развертываете стойкий сервер чата в корпоративном выпуске Skype для бизнеса Server 2015, вы можете развернуть стойкий сервер чат-сервера на одном или более автономных компьютерах в пуле корпоративных выпусков. Вы не можете коллокировать серверы сохраняемой передней части чата на переднем сервере Skype для бизнеса Server 2015. 
    
    Один постоянный сервер чат-сервера может поддерживать 20 000 активных пользователей. Вы можете иметь пул Persistent Chat Server с до 4 активными передними концами, что поддерживает в общей сложности 80 000 одновременно пользователей. 
    
- Если вы развертываете стойкий чат-сервер в Стандартном выпуске Skype для бизнеса Server 2015, вы можете примитивировать постоянный чат с передним конечным сервером. Это развертывание одного сервера может поддерживать до 20 000 пользователей. 
    
## <a name="persistent-chat-server-database-requirements"></a>Постоянные требования к базе данных chat Server

Для сохраняемого сервера чата SQL Server базы данных для хранения истории и содержимого комнат чата, данных конфигурации, данных об обеспечении пользователей и других соответствующих метаданных. Необязательный набор данных для хранения данных соответствия требованиям. Постоянные базы данных чата могут быть расположены в одном SQL Server или даже SQL экземпляре, что и базы данных заднего баз данных. 
  
- Если вы устанавливаете стойкий сервер чата в Корпоративной версии Skype для бизнеса Server 2015, чтобы обеспечить оптимальную производительность, рекомендуется установить хранилище файлов сохраняемой чата.
    
- Если вы устанавливаете стойкий сервер чата в стандартном выпуске Skype для бизнеса Server 2015, вы можете развернуть сервер сохраняемой системы хранения чатов на локальном экземпляре SQL Server Express.
    
- База данных сохраняемого чата (mgc) и база данных соответствия требованиям (mgccomp) могут располагаться в одном экземпляре SQL Server или на SQL серверах.
    
Чтобы подготовить платформу сервера базы данных, убедитесь, что каждый компьютер соответствует требованиям к оборудованию, а затем установите все необходимые программные компоненты. Серверная платформа для серверов базы данных сохраняемого чата требует того же оборудования, что и сервер базы данных Skype для бизнеса Server 2015. Подробные сведения см. в [материале Server requirements for Skype for Business Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
  
На сервере базы данных убедитесь, что установлено одно из следующих программных приложений:

- Microsoft SQL Server 2017 с последним пакетом услуг.

- Microsoft SQL Server 2016 с Пакет обновления 1, и вы должны запустить с Skype для бизнес-сервера накопительного обновления 7 или более поздних выпусков. Мы рекомендовали SQL Server 2016 с последним пакетом услуг. Дополнительные сведения об установке Microsoft SQL Server 2016 г. см. в SQL Server [2016](/sql/database-engine/install-windows/install-sql-server?view=sql-server-2016)г.

- Microsoft SQL Server 2014 г., и вы должны запустить с помощью Накопительного обновления Skype для бизнеса Server 6 или более поздних выпусков. Мы рекомендовали SQL Server 2014 с последним пакетом служб. Дополнительные сведения об установке Microsoft SQL Server 2014 г. см. в SQL Server [2014](/sql/database-engine/install-windows/install-sql-server?view=sql-server-2014)г.

- Microsoft SQL Server 2012 (64-битное издание), и мы рекомендовали работать с последним пакетом служб. Дополнительные сведения об установке Microsoft SQL Server 2012 г. см. в SQL Server [2012](/previous-versions/sql/sql-server-2012/bb500395(v=sql.110))г.
