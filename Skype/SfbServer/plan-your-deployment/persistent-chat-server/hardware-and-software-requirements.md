# <a name="hardware-and-software-requirements-for-persistent-chat-server-in-skype-for-business-server-2015"></a>Требования к оборудованию и программному обеспечению для сервера сохраняемого чата в Skype для бизнеса Server 2015
 
**Сводка:** В этой статье приведены сведения о требованиях к оборудованию и программному обеспечению для постоянного сервера чата в Skype для бизнеса Server 2015.
  
Вы можете установить сохраняемый сервер чата в Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition. Требования зависят от того, какая версия Skype для бизнеса Server 2015 установлена, и требования к производительности вашей компании. Enterprise Edition поддерживает до 80 000 одновременно работающих пользователей, а Standard Edition — до 20 000 одновременно работающих пользователей. Программное обеспечение сохраняемого чата состоит из интерфейсного компонента и базы данных SQL, которая служит внутренним компонентом.
  
Прежде чем развертывать сохраняемый сервер чата, необходимо убедиться, что выполнены следующие требования к оборудованию и программному обеспечению:
  
- Оборудование, которое отвечает минимальным требованиям для поддержки Skype для бизнеса Server 2015, сервера сохраняемого чата, серверов баз данных и файловых серверов. Дополнительные сведения можно найти в разделе [требования к серверу для Skype для бизнеса Server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
    
- Поддерживаемые операционная система и программное обеспечение базы данных.
    
    Подробнее о поддерживаемых операционных системах и программном обеспечении баз данных и требованиях к обновлению для Windows можно найти в статьях [требования к серверу для Skype для бизнеса Server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
    
- Сервер клиентского доступа Skype для бизнеса Server 2015. Сервер переднего плана — это основа для маршрутизации протокола SIP, которая делает связь между компьютерами, на которых запущены сохраняемый сервер чата, и функциональность сохраняемого чата. 
    
- Программное обеспечение очередей сообщений. Используется при развертывании на сервере сохраняемого чата и в постоянной службе соответствия чатов.
    
В следующих разделах описаны особые требования к сохраняемым серверам чата и базе данных, хранящей сохраняемые данные чата.

> [!NOTE] 
> Сохраняемый чат доступен в Skype для бизнеса Server 2015, но больше не поддерживается в Skype для бизнеса Server 2019. Эта функция доступна в Teams. Дополнительные сведения можно найти в разделе [путешествие из Skype для бизнеса в Microsoft Teams](/microsoftteams/journey-skypeforbusiness-teams). Если вы хотите использовать сохраняемый чат, вы можете либо перенести пользователей, которым требуются эти функции, в Teams, либо продолжить работу с Skype для бизнеса Server 2015. 
  
## <a name="front-end-server-requirements"></a>Требования к внешнему серверу

Требования к серверу переднего плана зависят от того, как вы развертываете сохраняемый сервер чата в Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition.
  
- Если вы развертываете сохраняемый сервер чата в Skype для Business Server 2015 Enterprise Edition, вы можете развернуть сервер клиентского чата на одном или нескольких отдельных компьютерах в пуле Enterprise Edition. Вы не можете разгруппировать постоянные серверы для чата на интерфейсе сервера Skype для бизнеса Server 2015. 
    
    Один сервер-сервер для входа в чате может поддерживать 20 000 активных пользователей. Вы можете использовать в качестве пула серверов сохраняемого чата до 4 активных интерфейсов, благодаря чему поддерживается общее количество одновременных пользователей 80 000. 
    
- Если вы развертываете сохраняемый сервер чата в Skype для Business Server 2015 Standard Edition, вы можете разгруппировать сохраняемый чат с помощью клиентского сервера. Такое развертывание одного сервера поддерживает до 20 000 пользователей. 
    
## <a name="persistent-chat-server-database-requirements"></a>Требования к базам данных сервера в сохраняемом чате

Для сервера сохраняемого чата требуется, чтобы программа базы данных SQL Server содержала историю и содержимое комнаты чата, данные конфигурации, данные пользователя и другие необходимые метаданные. При необходимости для хранения данных о соответствии требованиям используется база данных соответствия требованиям сохраняемого чата. Допускается совмещенное расположение баз данных сохраняемого чата и внутренних баз данных на одном и том же сервере SQL Server или даже на одном и том же экземпляре SQL. 
  
- При установке сервера сохраняемого чата в Skype для бизнеса Server 2015 Enterprise Edition для обеспечения оптимальной производительности рекомендуется установить хранилище файлов сохраняемого чата.
    
- При установке сервера сохраняемого чата в Skype для Business Server 2015 Standard Edition вы можете развернуть сервер "сохраняемый чат" на обратном стороне локального экземпляра SQL Server Express.
    
- База данных сохраняемого чата (MGC) и база данных соответствия (мгккомп) могут находиться в одном и том же экземпляре SQL Server или на разных серверах SQL Server.
    
Для подготовки платформы базы данных необходимо обеспечить соответствие каждого компьютера требованиям к оборудованию и затем установить обязательное программное обеспечение. Серверная платформа для серверов базы данных сохраняемого чата требует такого же оборудования, как сервер базы данных Skype для бизнеса Server 2015. Подробности можно найти в разделе [требования к серверу для Skype для бизнеса Server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
  
На сервере базы данных убедитесь, что установлено одно из следующих приложений:

- Microsoft SQL Server 2017 с последним пакетом обновления.

- Microsoft SQL Server 2016 с пакетом обновления 1 (SP1) и вам необходимо использовать накопительную версию 7 или более поздних версий Skype для Business Server. Рекомендуем запустить SQL Server 2016 с последним пакетом обновления. Подробнее о том, как установить Microsoft SQL Server 2016, можно найти в статьях [Установка SQL server 2016](https://docs.microsoft.com/pt-pt/sql/database-engine/install-windows/install-sql-server?view=sql-server-2016).

- Microsoft SQL Server 2014, и вы должны работать со Skype для Business Server с пакетом обновления 6 или более поздними версиями. Рекомендуем запустить SQL Server 2014 с последним пакетом обновления. Подробнее о том, как установить Microsoft SQL Server 2014, можно найти в статьях [Установка SQL server 2014](https://docs.microsoft.com/pt-pt/sql/database-engine/install-windows/install-sql-server?view=sql-server-2014).

- Microsoft SQL Server 2012 (версия 64-bit Edition), и мы рекомендуем работать с последним пакетом обновления. Подробнее о том, как установить Microsoft SQL Server 2012, можно найти в статьях [Установка SQL server 2012](https://go.microsoft.com/fwlink/p/?LinkID=248559).
