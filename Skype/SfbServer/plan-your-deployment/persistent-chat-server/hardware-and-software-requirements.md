# <a name="hardware-and-software-requirements-for-persistent-chat-server-in-skype-for-business-server-2015"></a>Требования к оборудованию и программному обеспечению для сервера сохраняемого чата в Skype для бизнеса Server 2015
 
**Сводка:** В этом разделе рассказывается о требованиях к оборудованию и программному обеспечению для сервера сохраняемого чата в Skype для бизнеса Server 2015.
  
Сервер сохраняемого чата можно установить с помощью Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition. Требования зависят от установленного выпуска Skype для бизнеса Server 2015 и требований к производительности вашего бизнеса. Enterprise Edition поддерживает до 80 000 одновременно работающих пользователей; Выпуск Standard Edition может поддерживать до 20 000 одновременно работающих пользователей. Сохраняемый чат состоит из интерфейсного компонента, а также внутреннего компонента базы данных SQL.
  
Перед развертыванием сервера сохраняемого чата необходимо убедиться, что выполнены следующие требования к оборудованию и программному обеспечению:
  
- Оборудование, которое соответствует минимальным требованиям для поддержки Skype для бизнеса Server 2015, сервера сохраняемого чата, серверов баз данных и файловых серверов. Для получения дополнительных сведений см [серверные требования для Skype для бизнеса server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
    
- Поддерживаемая операционная система и программное обеспечение баз данных.
    
    Сведения о поддерживаемых операционных системах и программном обеспечении для баз данных и требованиях к обновлению Windows приведены в статье [требования к серверу для Skype для бизнеса server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
    
- Сервер переднего плана Skype для бизнеса Server 2015. Сервер переднего плана является основой для маршрутизации SIP, обеспечивающей связь между компьютерами с сервером сохраняемого чата и функциями сохраняемого чата. 
    
- Программное обеспечение очередей сообщений. Используется сервером сохраняемого чата и службой соответствия сохраняемого чата, если она развернута.
    
В следующих разделах описываются конкретные требования для сервера сохраняемого чата и базы данных, в которых хранятся данные сохраняемого чата.

> [!NOTE] 
> Сохраняемый чат доступен в Skype для бизнеса Server 2015, но больше не поддерживается в Skype для бизнеса Server 2019. В Teams доступны те же функции. Для получения дополнительных сведений обратитесь [к разделу Начало обновления Microsoft Teams](/microsoftteams/upgrade-start-here). Если вам нужно использовать сохраняемый чат, вы можете либо перенести пользователей, которым требуется эта функция, в Teams, либо продолжить работу с Skype для бизнеса Server 2015. 
  
## <a name="front-end-server-requirements"></a>Требования сервера переднего плана

Требования сервера переднего плана зависят от того, выполняется ли развертывание сервера сохраняемого чата с помощью Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition.
  
- При развертывании сервера сохраняемого чата с помощью Skype для бизнеса Server 2015 Enterprise Edition можно развернуть сервер переднего плана сервера сохраняемого чата на одном или нескольких автономных компьютерах в пуле Enterprise Edition. Невозможно совместное размещение серверов переднего плана сохраняемого чата на сервере переднего плана Skype для бизнеса Server 2015. 
    
    Один сервер переднего плана сервера сохраняемого чата может поддерживать 20 000 активных пользователей. Пул серверов сохраняемого чата может иметь до 4 активных интерфейсов, поэтому он поддерживает всего 80 000 пользователей одновременно. 
    
- При развертывании сервера сохраняемого чата с помощью Skype для бизнеса Server 2015 Standard Edition можно совместно разместить сохраняемый чат с сервером переднего плана. Это развертывание с одним сервером может поддерживать до 20 000 пользователей. 
    
## <a name="persistent-chat-server-database-requirements"></a>Требования к базе данных сервера сохраняемого чата

Для сервера сохраняемого чата требуется программное обеспечение базы данных SQL Server для хранения журнала и контента комнаты чата, данных конфигурации, данных подготовки пользователей и других соответствующих метаданных. При необходимости для хранения данных соответствия используется база данных соответствия сохраняемого чата. Базы данных сохраняемого чата можно размещать на одном сервере SQL Server или даже в том же экземпляре SQL, что и внутренние базы данных. 
  
- При установке сервера сохраняемого чата с помощью Skype для бизнеса Server 2015 Enterprise Edition, чтобы обеспечить оптимальную производительность, рекомендуется установить хранилище файлов сохраняемого чата.
    
- При установке сервера сохраняемого чата с помощью Skype для бизнеса Server 2015 Standard Edition можно развернуть внутренний сервер хранилища для сохраняемого чата на локальном экземпляре SQL Server Express.
    
- База данных сохраняемого чата (MGC) и база данных соответствия (mgccomp) могут находиться в том же экземпляре SQL Server или на разных серверах SQL Server.
    
Чтобы подготовить платформу сервера базы данных, убедитесь, что каждый компьютер соответствует требованиям к оборудованию, а затем установите все необходимые программные компоненты. Серверная платформа для серверов базы данных сохраняемого чата требует того же оборудования, что и сервер базы данных Skype для бизнеса Server 2015. Дополнительные сведения: [требования к серверу для Skype для бизнеса server 2015](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md).
  
Убедитесь, что на сервере базы данных установлено одно из следующих программных приложений:

- Microsoft SQL Server 2017 с последним пакетом обновления.

- Microsoft SQL Server 2016 с пакетом обновления 1 (SP1) и необходимо запускать с накопительным пакетом обновления 7 или более поздних версий Skype для бизнеса Server. Мы рекомендуем запустить SQL Server 2016 с последним пакетом обновления. Сведения о том, как установить Microsoft SQL Server 2016, приведены в разделе [Установка SQL server 2016](https://docs.microsoft.com/sql/database-engine/install-windows/install-sql-server?view=sql-server-2016).

- Microsoft SQL Server 2014, и его необходимо запускать с накопительным пакетом обновления 6 или более поздних версий с помощью Skype для бизнеса Server. Мы рекомендуем запустить SQL Server 2014 с последним пакетом обновления. Сведения о том, как установить Microsoft SQL Server 2014, приведены в разделе [Установка SQL server 2014](https://docs.microsoft.com/sql/database-engine/install-windows/install-sql-server?view=sql-server-2014).

- Microsoft SQL Server 2012 (64-разрядный выпуск), и мы рекомендуем работать с последним пакетом обновления. Сведения о том, как установить Microsoft SQL Server 2012, приведены в разделе [Установка SQL server 2012](https://go.microsoft.com/fwlink/p/?LinkID=248559).
