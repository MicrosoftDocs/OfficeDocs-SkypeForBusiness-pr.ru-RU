# <a name="hardware-and-software-requirements-for-persistent-chat-server-in-skype-for-business-server-2015"></a>Требования к оборудованию и программному обеспечению для сервера сохраняемой беседы в Skype для бизнеса Server 2015
 
**Сводка:** В этом разделе вы узнаете о требованиях к оборудованию и программному обеспечению для сервера сохраняемой беседы в Skype для бизнеса Server 2015.
  
Сервер сохраняемой беседы можно установить вместе со Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition. Требования зависят от установленного выпуска Skype для бизнеса Server 2015 и требований к производительности вашей компании. Enterprise Edition может поддерживать до 80 000 одновременно пользователей; Standard Edition может поддерживать до 20 000 одновременно пользователей. Сохраняемая беседа состоит из компонента переднего и SQL базы данных.
  
Перед развертыванием сервера сохраняемой беседы необходимо убедиться, что выполнены следующие требования к оборудованию и программному обеспечению:
  
- Оборудование, которое соответствует минимальным требованиям для поддержки Skype для бизнеса Server 2015, сервера сохраняемой беседы, серверов баз данных и файловой серверов. Дополнительные сведения см. в требованиях к [серверу для Skype для бизнеса Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
    
- Поддерживаемые программное обеспечение операционной системы и базы данных.
    
    Подробные сведения о поддерживаемых операционных системах и программном обеспечении баз данных, а также требованиях к обновлению Windows см. в требованиях к серверу для Skype для [бизнеса Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
    
- Сервер переднего сервера Skype для бизнеса Server 2015. Сервер переднего сервера является основой для маршрутации по протоколу SIP, что делает возможным взаимодействие между компьютерами, на которых работает сервер сохраняемого чата, и функциями сохраняемого чата. 
    
- Программное обеспечение очереди сообщений. Используется сервером сохраняемой беседы и службой соответствия сохраняемой беседы, если она развернута.
    
В следующих разделах описываются особые требования к серверу сохраняемой беседы и базе данных, в которой хранится информация о сохраняемом чате.

> [!NOTE] 
> Сохраняемая беседа доступна в Skype для бизнеса Server 2015, но больше не поддерживается в Skype для бизнеса Server 2019. Такие же функции доступны в Teams. Дополнительные сведения [см. в сведениях о том,](/microsoftteams/upgrade-start-here)как начать обновление Microsoft Teams. Если необходимо использовать сохраняемого чата, вы можете либо перенести пользователей, которым требуются эти функции, в Teams, либо продолжить использование Skype для бизнеса Server 2015. 
  
## <a name="front-end-server-requirements"></a>Требования к серверу переднего сервера

Требования к серверу переднего сервера зависят от того, развертывается ли сервер сохраняемой беседы со Skype для бизнеса Server 2015 Enterprise Edition или Standard Edition.
  
- При развертывании сервера сохраняемой беседы со Skype для бизнеса Server 2015 Enterprise Edition можно развернуть сервер переднего сервера сохраняемой беседы на одном или более автономных компьютерах в пуле Enterprise Edition. Вы не можете выполнить выполнив сохраняемую беседу, серверы переднего сервера переднего сервера Skype для бизнеса Server 2015. 
    
    Один сервер переднего сервера сохраняемой беседы может поддерживать 20 000 активных пользователей. Можно использовать пул серверов сохраняемого чата с 4 активными серверами переднего плана, что обеспечивает поддержку 80 000 одновременно действующих пользователей. 
    
- При развертывании сервера сохраняемой беседы со Skype для бизнеса Server 2015 Standard Edition можно размещать сохраняемую беседу вместе с сервером переднего сервера. Это развертывание с одним сервером может поддерживать до 20 000 пользователей. 
    
## <a name="persistent-chat-server-database-requirements"></a>Требования к базе данных сервера сохраняемой беседы

Серверу сохраняемого чата требуется SQL Server базы данных для хранения истории и контента комнат чата, данных конфигурации, данных пользовательской подготовка и других соответствующих метаданных. При желании для хранения данных соответствия требованиям используется база данных соответствия сохраняемой беседы. Базы данных сохраняемого чата могут быть размещены в одном SQL Server или даже в том же экземпляре SQL, что и базы данных на тыловом уровне. 
  
- При установке сервера сохраняемой беседы со Skype для бизнеса Server 2015 Enterprise Edition для обеспечения оптимальной производительности рекомендуется установить хранилище файлов сохраняемой беседы.
    
- Если вы устанавливаете сервер сохраняемой беседы со Skype для бизнеса Server 2015 Standard, вы можете развернуть сервер сохраняемой беседы на локальном экземпляре SQL Server Express.
    
- База данных сохраняемого чата (mgc) и база данных соответствия (mgccomp) могут располагаться в одном экземпляре SQL Server или на разных SQL серверах.
    
Чтобы подготовить платформу сервера базы данных, убедитесь, что каждый компьютер соответствует требованиям к оборудованию, а затем установите все необходимые программные компоненты. Для серверной платформы для серверов баз данных сохраняемого чата требуется то же оборудование, что и для серверной базы данных Skype для бизнеса Server 2015. Подробные сведения см. в требованиях к [серверу для Skype для бизнеса Server 2015.](../../plan-your-deployment/requirements-for-your-environment/server-requirements.md)
  
Убедитесь, что на сервере баз данных установлено одно из следующих программных приложений:

- Microsoft SQL Server 2017 с последним пакетом обновления.

- Microsoft SQL Server 2016 с Пакет обновления 1, и вы должны работать с накопительным обновлением Skype для бизнеса Server 7 или более поздних версий. Мы рекомендуем использовать SQL Server 2016 с последним пакетом обновления. Дополнительные сведения об установке Microsoft SQL Server 2016 см. в SQL Server [2016.](https://docs.microsoft.com/sql/database-engine/install-windows/install-sql-server?view=sql-server-2016)

- Microsoft SQL Server 2014, и вы должны работать с накопительным обновлением Skype для бизнеса Server 6 или более поздних версий. Мы рекомендуем использовать SQL Server 2014 с последним пакетом обновления. Дополнительные сведения об установке Microsoft SQL Server 2014 см. в SQL Server [2014.](https://docs.microsoft.com/sql/database-engine/install-windows/install-sql-server?view=sql-server-2014)

- Microsoft SQL Server 2012 (64-битная версия), и мы рекомендуем использовать последний пакет обновления. Дополнительные сведения об установке Microsoft SQL Server 2012 см. в SQL Server [2012](https://go.microsoft.com/fwlink/p/?LinkID=248559).
