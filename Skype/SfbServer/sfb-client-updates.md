# <a name="latest-updates-for-versions-of-skype-for-business-that-use-windows-installer-msi"></a>Последние обновления для версий Skype для бизнеса, Windows установки (MSI)

Используйте ссылки на этой странице, чтобы получить дополнительные сведения о последних обновлениях для постоянных версий Skype для бизнеса 2016, Skype для бизнеса 2015 и Lync 2013.

> [!NOTE]
> - Сведения в этой статье применимы только к вечным версиям Skype для бизнеса, которые используют технологию установки Windows установщика (MSI). Например, если установлена лицензированная версия Skype для бизнеса, например Skype для бизнеса с Office профессиональный плюс 2016.
> - Сведения в этой статье не применяются к Microsoft 365 или Office 365 версиям Skype для бизнеса, например к Skype для бизнеса, включенным в Приложения Microsoft 365 для предприятий.
> - Для последних обновлений мобильных клиентов перейдите в магазин приложений для вашего мобильного клиента (iTunes, Google Play или Microsoft Store), чтобы просмотреть сведения и установить обновления.
> - Чтобы узнать, какую версию Skype для бизнеса вы используете, см. в Skype для бизнеса [версии?](https://support.microsoft.com/help/3195481/how-to-check-the-version-of-skype-for-business-2016)
> - Дополнительные сведения об установке обновлений Office см. в статье [Установка обновлений Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5).

Skype для бизнеса совместимость с Microsoft 365 или Office 365 и локальной средой зависит от требований системы к развертыванию:

- Microsoft 365 и Office 365 с любой версией Skype для бизнеса, которая находится в основной поддержке, которая включает последнюю версию Skype для бизнеса 2016. Для предыдущих версий Skype для бизнеса только те, которые имеют расширенную поддержку, могут продолжать работать с Microsoft 365 или Office 365, хотя и с уменьшенными функциональными возможностями. Дополнительные сведения см. [в веб-сайте Microsoft Lifecycle Policy](https://support.microsoft.com/lifecycle).

- Для Skype для бизнеса Server локального развертывания выполните требования системы к вашей версии Skype для бизнеса:

  - [Требования к вашей Skype для бизнеса среде](./plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md)

  - [Требования к системе Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-system-requirements)
