# <a name="install-or-update-skype-for-business-server-system"></a>Установка или обновление системы Skype для бизнеса Server

Чтобы начать установку или обновить существующее развертывание Skype для бизнеса Server 2015, нажмите кнопку "Установить" или "Обновить **Skype для бизнеса Server".** В этом разделе будут устанавливаться компоненты для определенной роли сервера на основе документа построитель топологий. Прежде чем приступить к этой серии действий, необходимо определить развертывание в построитель топологий. Вы должны войти в развертываемый компьютер как локальный администратор и участник группы администраторов домена в домене, в котором находится сервер. Компьютер необходимо присоединить к домену до выполнения этого шага.

При обновлении развернутых серверов необходимо внести изменения в построитель топологий.
