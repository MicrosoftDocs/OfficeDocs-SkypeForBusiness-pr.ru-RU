# <a name="startservices-intro"></a>Запуск служб (введение)
 
Чтобы запустить службы на сервере роли Skype для бизнеса Server 2015, который вы развертываете, ознакомьтесь с начальным экраном. Обратите внимание на то, что команды, выданные для запуска Skype для бизнеса Server, не ожидают запуска службы. Это наилучший из возможных способов запуска служб; наблюдать за текущим состоянием служб следует с консоли управления (MMC) для служб Windows. Нажмите **Далее** для перехода к задаче **Запустить службы**.
