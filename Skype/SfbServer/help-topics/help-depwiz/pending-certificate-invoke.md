# <a name="pending-certificate-invoke"></a>Отложенный сертификат (вызов)
 
На странице **выполнение команд** отображается состояние команд, которые извлекают ожидающий сертификат. После завершения задачи будет доступен файл журнала " **Получение ожидающих сертификатов** ". Щелкните **Просмотреть журнал** , чтобы проверить состояние обработки и завершения задачи. Для продолжения нажмите кнопку **Далее**.
  
> [!IMPORTANT]
> Если вы еще не утвердили сертификат администратором центра сертификации, в файле журнала запросов на получение ожидающих запросов появится сообщение о том, что ожидаются запросы. Если администратор центра сертификации отклонил запрос, файл журнала сообщит об ошибке и сообщит о том, что запрос отвергнут, и имя пользователя, который отклонил запрос.
