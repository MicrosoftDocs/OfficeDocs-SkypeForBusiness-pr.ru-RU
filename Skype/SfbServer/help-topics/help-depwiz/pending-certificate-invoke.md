# <a name="pending-certificate-invoke"></a>Отложенный сертификат (вызов)
 
На странице **Выполнение команд** отображается состояние команд, получающих отложенный сертификат. По завершении задачи становится доступен файл журнала **Получение отложенных сертификатов**. Для просмотра состояния процесса и завершения задачи нажмите кнопку **Просмотреть журнал**. Нажмите кнопку **Далее**, чтобы продолжить.
  
> [!IMPORTANT]
> Если сертификат еще не утвержден администратором центра сертификации, в файле журнала получения отложенных запросов будет указано, что имеются отложенные запросы. Если администратор центра сертификации отклонил запрос, в файле журнала указывается ошибка и имя пользователя, отклонившего запрос.
