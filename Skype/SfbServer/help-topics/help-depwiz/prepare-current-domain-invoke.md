# <a name="prepare-current-domain-invoke"></a>Подготовка текущего домена (вызов)
 
На странице **выполнение команд** отображается сводка задач по мере их выполнения. Посмотрите на экран Сводка и убедитесь, что задачи выполнены успешно и ошибки не обнаружены. Ознакомьтесь с предупреждениями, чтобы убедиться, что они ожидаются. Под областью сводки вы должны увидеть **состояние задачи: завершено**. В раскрывающемся списке должен быть показан **домен подготовки** файла журнала. Если это не так, выберите файл журнала и щелкните **Просмотреть журнал** для просмотра подробных действий и результатов задач. Нажмите кнопку **Готово** , чтобы завершить подготовку домена.
