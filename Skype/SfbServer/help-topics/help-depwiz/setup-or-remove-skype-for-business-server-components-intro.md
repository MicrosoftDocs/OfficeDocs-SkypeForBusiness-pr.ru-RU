# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Установка или удаление компонентов Skype для бизнеса Server (введение)
 
Вы подготавливается к установке, активации или отключению и удалить компоненты Skype для бизнеса Server 2015. Процессы, которые будут выполняться, зависят от текущей конфигурации определения, опубликованного в центральном хранилище управления. Для успешного выполнения этих задач необходимо войти в систему как участник локальной группы администраторов и, как минимум, иметь возможность чтения доменных служб Active Directory.
  
Нажмите кнопку **Далее** для перехода на экран **Выполнение команд**.
