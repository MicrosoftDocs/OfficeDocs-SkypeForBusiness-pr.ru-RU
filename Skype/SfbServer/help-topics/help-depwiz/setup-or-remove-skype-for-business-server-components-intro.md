# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Установка или удаление компонентов Skype для бизнеса Server (введение)
 
Вы готовите к установке, активации или отключению и Skype для бизнеса Server 2015. Процессы, которые будут выполняться, зависят от текущей конфигурации определения, опубликованного в центральном хранилище управления. Для успешного выполнения задач необходимо войти в группу локальных администраторов и, как минимум, прочитать службы домена Active Directory.
  
Нажмите кнопку **Далее** для перехода на экран **Выполнение команд**.
