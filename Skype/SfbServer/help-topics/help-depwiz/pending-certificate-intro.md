# <a name="pending-certificate-intro"></a>Отложенный сертификат (введение)
 
Чтобы обработать отложенный запрос на онлайновый сертификат, то есть запрос, выданный вашим центром сертификации в Интернете, но для настройки которого требуется ручной процесс утверждения, нажмите кнопку **Далее**.
