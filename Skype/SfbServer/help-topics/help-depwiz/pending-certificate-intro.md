# <a name="pending-certificate-intro"></a>Отложенный сертификат (введение)
 
Для обработки отложенного запроса сертификата в Интернете, то есть запроса, выданного вашему веб-органу сертификации, но в административном отношении, необходимого для процесса утверждения вручную, нажмите **кнопку Далее**.
