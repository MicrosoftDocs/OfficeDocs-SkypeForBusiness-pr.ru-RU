# <a name="pending-certificate-intro"></a>Отложенный сертификат (введение)
 
Чтобы обработать ожидающих онлайн-запрос сертификатов, то есть запрос, который был выдан вашему веб-сайту сертификации, но административно настроен на требование ручного утверждения, нажмите кнопку **"Далее".**
