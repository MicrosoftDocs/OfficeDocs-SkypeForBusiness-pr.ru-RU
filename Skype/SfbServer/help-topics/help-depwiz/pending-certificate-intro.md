# <a name="pending-certificate-intro"></a>Отложенный сертификат (введение)
 
Для обработки запроса на сертификат ожидающие online, то есть, запрос, который выполнялась в локальную службу сертификации, но административно задано значение требуется процесс утверждения вручную, нажмите кнопку **Далее**.
