# <a name="prepare-current-forest-universal-group-location"></a>Подготовка текущего леса (местонахождение универсальной группы)
 
Необходимо указать, где необходимо создать универсальные группы в лесу. На странице **универсального расположения групп** используется домен, в котором компьютер, на который вы вошли, является членом по умолчанию. Если вы хотите создать универсальные группы в домене, отличном от **локального домена**по умолчанию, выберите полное доменное имя **домена** и введите в текстовом поле полные имена доменов (FQDN). Предоставление альтернативного домена удобно при запуске задачи подготовки леса из домена, отличного от того, где вы хотите создать универсальные группы. Для продолжения нажмите кнопку **Далее**.
