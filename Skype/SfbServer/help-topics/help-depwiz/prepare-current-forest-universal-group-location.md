# <a name="prepare-current-forest-universal-group-location"></a>Подготовка текущего леса (местонахождение универсальной группы)
 
Необходимо указать, где необходимо создать универсальные группы в лесу. **Универсальная группа расположение** страницы использует домен, в котором компьютере, на котором вы вошли является членом по умолчанию. Если вы хотите создать универсальных групп в домене, отличном от **домена**по умолчанию, выберите **Полное доменное имя домена** и введите полное доменное имя (FQDN) в текстовом поле. Создать при условии, что альтернативном домене полезна при выполнении задачи подготовки леса в домене, отличном от место универсальных групп. Для продолжения нажмите кнопку **Далее**.
