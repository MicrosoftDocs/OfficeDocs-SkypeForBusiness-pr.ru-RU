# <a name="certificate-request-online-offline"></a>Запрос сертификата (сетевой, автономный)
 
Чтобы создать отложенный или автономный запрос, выберите **Подготовить запрос сейчас, но отправить позже (автономный запрос сертификата)**. Чтобы создать непосредственный запрос в орган сертификации в Интернете (как правило, внутренняя система в инфраструктуре общедоступных ключей организации), выберите Отправить запрос немедленно в орган сертификации **в Интернете**.
  
Выбрав нужный параметр, нажмите кнопку **Далее**. Чтобы отменить действие без выполнения изменений нажмите кнопку **Отмена**. Чтобы вернуться на предыдущий шаг, нажмите кнопку **Назад**.
