# <a name="certificate-request-online-offline"></a>Запрос сертификата (сетевой, автономный)
 
Чтобы создать отложенный или автономный запрос, выберите **Подготовить запрос сейчас, но отправить позже (автономный запрос сертификата)**. Чтобы создать немедленный запрос к веб-сайту сертификации (обычно это внутренняя система в инфраструктуре открытых ключей организации), выберите "Отправить запрос немедленно в **интернет-службу сертификации".**
  
Выбрав нужный параметр, нажмите кнопку **Далее**. Чтобы отменить действие без выполнения изменений нажмите кнопку **Отмена**. Чтобы вернуться на предыдущий шаг, нажмите кнопку **Назад**.
