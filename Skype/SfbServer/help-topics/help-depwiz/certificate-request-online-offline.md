# <a name="certificate-request-online-offline"></a>Запрос сертификата (сетевой, автономный)
 
Чтобы создать отложенный или автономный запрос, выберите **Подготовить запрос сейчас, но отправить позже (автономный запрос сертификата)**. Для создания немедленного запроса в веб-орган сертификации (как правило, внутренняя система в инфраструктуре общедоступных ключевых элементов организации) выберите Отправить запрос немедленно в **веб-орган сертификации.**
  
Выбрав нужный параметр, нажмите кнопку **Далее**. Чтобы отменить действие без выполнения изменений нажмите кнопку **Отмена**. Чтобы вернуться на предыдущий шаг, нажмите кнопку **Назад**.
