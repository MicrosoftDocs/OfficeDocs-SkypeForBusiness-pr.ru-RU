# <a name="start-services"></a>Запуск служб
 
Чтобы запустить службы для развернутой роли сервера, нажмите кнопку **Выполнить** в окне **Шаг 4. Запустить службы**. Вы должны войти как участник локальной группы администраторов или пользователь с эквивалентными правами. 
  
> [!NOTE]
> Если задача **Запустить службы** уже была запущена, вы можете перезапустить ее, нажав кнопку **Перезапуск**.
