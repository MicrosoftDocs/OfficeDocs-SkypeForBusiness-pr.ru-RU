# <a name="certificate-assignment-summary"></a>Назначение сертификата (сводка)
 
Прежде чем завершить назначение сертификата, открывается страница **Сводка по назначениям сертификатов**, которая содержит сведения о назначаемом сертификате. Просмотрите эти сведения и убедитесь, что имя субъекта и альтернативные имена субъектов указаны правильно. Завершив просмотр сведений, нажмите кнопку **Далее**.
