# <a name="sql-server-reporting-services-intro"></a>Службы SQL Server Reporting Services (введение)
 
С каждым пулом переднего плана и комплексом для обеспечения связи в филиалах может быть связан только один сервер мониторинга. Если мониторинг включен для сайта, сервер мониторинга обеспечивает сбор данных и создание отчетов для регистрации вызовов (CDR) и качества взаимодействия (QoE).
  
Все пулы сайта и пулы нескольких центральных сайтов могут использовать один и тот же сервер мониторинга, если использование не превышает возможностей последнего. For details about designing a topology to support monitoring, see [Associate a monitoring store with a Front End pool in Skype for Business Server 2015](../../deploy/deploy-monitoring/associate-a-monitoring-store.md) in the Deployment documentation.
