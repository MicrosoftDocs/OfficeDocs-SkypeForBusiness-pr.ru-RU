# <a name="sql-server-reporting-services-groups"></a>Службы SQL Server Reporting Services (группы)

В окне **Определение группы с доступом только для чтения** укажите группу, которой хотите предоставить доступ только для чтения к отчетам по мониторингу. Используйте группу &lt;domainaccess&lt;&gt;\\ формата.&gt; Например: contoso\RTCUniversalReadOnlyAdmins.
  
Предоставление группе доступа только для чтения не является обязательным. Если вы не хотите делать этого сейчас, вы можете настроить доступ позже непосредственно в службах SQL Server Reporting Services.
