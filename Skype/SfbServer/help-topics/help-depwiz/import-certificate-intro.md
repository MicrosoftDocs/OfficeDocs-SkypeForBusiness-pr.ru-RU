# <a name="import-certificate-intro"></a>Импорт сертификата (введение)
 
Для импорта сертификата необходимо указать путь к файлу сертификата. В поле **Выбор файла сертификата** введите полный путь и имя файла или нажмите кнопку **Обзор** и перейдите к расположению, в котором указан путь и имя файла (обычно. p7b, PFX или CER-файл).
  
Если сертификат имеет закрытый ключ, установите флажок файл сертификата с флажком, который **включает закрытый ключ сертификата**. Если установлен этот флажок, ввод текста **пароля** включен. Если у вас есть сертификат, связанный с закрытым ключом, пароль обычно размещается на закрытом ключе при создании сертификата. Вы можете ввести пароль для закрытого ключа, чтобы разрешить импорт сертификата и закрытого ключа в хранилище сертификатов. После того как вы предоставили сведения о пути к файлу сертификата и, при необходимости, пароль закрытого ключа, нажмите кнопку **Далее**.
  
> [!IMPORTANT]
> Если вы не знаете пароль для закрытого ключа, импорт завершится сбоем.
