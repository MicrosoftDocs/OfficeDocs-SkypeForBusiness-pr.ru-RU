# <a name="view-certificate"></a>Просмотр сертификата
 
На странице " **Просмотр сертификата** " — область, в которой можно выбрать любой допустимый сертификат в локальном хранилище сертификатов. Выберите сертификат и нажмите кнопку **Просмотр сведений о сертификате**. Сертификат будет отображаться в новой области. На левой панели **сертификата** можно просмотреть сведения о сертификате Общие. Перейдите на вкладку **сведения** , чтобы просмотреть подробные сведения о сертификате, таких как имя субъекта и альтернативное имя субъекта. Путь сертификации будет отображаться путь «цепи доверия» для сертификата. По окончании просмотра сведения о сертификате, нажмите **кнопку ОК**.
