# <a name="certificate-assignment-from-request"></a>Назначение сертификата (из запроса)
 
На **странице Назначение сертификата** начинается процесс назначения сертификата запрашиваемого использования на сервере ролей. Для просмотра сертификата щелкните **Просмотреть сведения о сертификате**. Вы можете выбрать **вкладку General,** **вкладку Details** или вкладку **Путь сертификации.** На **вкладке General** будут отображаться основные сведения о сертификате, такие как полномочия по выдаче сертификата и состояние закрытого ключа. На вкладке "Сведения" отображается такая информация о сертификате, как имя субъекта, альтернативные имена субъектов и прочие сведения. Вкладка **Путь** сертификации будет отображать сертификат "цепочка доверия" из сертификата через любые органы сертификации или промежуточные (если они присутствуют) в корневой орган сертификации.
  
Нажмите кнопку **Далее**, чтобы продолжить.
