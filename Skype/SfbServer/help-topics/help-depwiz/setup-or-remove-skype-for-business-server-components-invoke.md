# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>Установка или удаление компонентов Skype для бизнеса Server (вызов)
 
На странице **Выполнение команд** отображается сводка выполнения задач по развертыванию роли сервера. После выполнения задач развертывание и активация, деактивации и деинсталлации отображается сводка. Кроме того, в раскрывающемся списке можно выбрать любые файлы журнала, которые создаются при выполнении команды. Выберите в раскрывающемся списке файл журнала, который хотите просмотреть, и нажмите кнопку **Просмотреть журнал**.
  
После просмотра сводки и файлов журнала нажмите кнопку **Готово**.
