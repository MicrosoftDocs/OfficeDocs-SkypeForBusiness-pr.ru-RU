# <a name="certificate-request-certificate-signing-request-generated"></a>Запрос сертификата (запрос подписи сертификата создан)
 
Страница " **файл запроса сертификата** " отображается после завершения запроса на сертификат в автономном режиме. Чтобы отобразить запрос сертификата и подготовить его к отправке в автономный центр сертификации (ЦС) или в общедоступный ЦС, нажмите кнопку **Просмотр** . Запрос на сертификат отображается в блокноте Windows (по умолчанию). Вы можете скопировать и вставить запрос на сертификат из этого отображения или завершить обработку запроса сертификата, следуя указаниям из выбранного автономного ЦС или общедоступного центра сертификации. Нажмите кнопку **Готово** , чтобы продолжить.
