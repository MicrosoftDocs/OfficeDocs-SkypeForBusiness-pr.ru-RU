# <a name="certificate-request-subject-alternate-names"></a>Запрос сертификата (альтернативные имена субъектов)
 
На странице " **альтернативное имя субъекта/тема** " представлена сводка по имени субъекта и по всем дополнительным именам тем, которые будут заполнять сертификат. Имена, представленные для имени субъекта и альтернативных имен субъектов, изменяются из документа определения в том виде, в котором они были созданы при определении развертывания в построителе топологии.
  
Если необходимо внести изменения в имя субъекта или в любое из альтернативных имен субъектов, нажмите кнопку **Отмена** , чтобы завершить работу мастера запроса сертификата и запустить построитель топологии. Затем повторно опубликуйте новую конфигурацию с помощью правильно определенных темы и альтернативных имен тем.
