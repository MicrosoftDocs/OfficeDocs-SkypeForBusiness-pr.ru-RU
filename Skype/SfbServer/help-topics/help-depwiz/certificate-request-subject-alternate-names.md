# <a name="certificate-request-subject-alternate-names"></a>Запрос сертификата (альтернативные имена субъектов)
 
На странице **Имя субъекта / альтернативные имена субъектов** представлена сводная информация по имени субъекта и всех альтернативных именах субъектов, которые будут указаны в сертификате. Имена, представленные для имени субъекта и альтернативных имен субъектов, собираются из документа определения, так как он был создан при определении развертывания в построитель топологий.
  
Если необходимо внести изменения в имя субъекта или  любое из альтернативных имен субъектов, нажмите кнопку "Отмена", чтобы выйти из мастера запроса сертификатов и повторно перезавести построитель топологий. Затем опубликуйте новую конфигурацию с правильно определенными именами.
