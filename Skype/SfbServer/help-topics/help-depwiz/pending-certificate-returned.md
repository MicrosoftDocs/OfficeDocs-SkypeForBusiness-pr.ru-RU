# <a name="pending-certificate-returned"></a><span data-ttu-id="5e59d-104">Отложенный сертификат (возврат)</span><span class="sxs-lookup"><span data-stu-id="5e59d-104">Pending Certificate (Returned)</span></span>
 
<span data-ttu-id="5e59d-p102">На странице **Состояние сетевого запроса сертификата** отображается текущее состояние сертификата. Если сертификат все еще ожидает утверждения администратором центра сертификации (то есть если он еще не утвержден), состояние показывает, что сертификат все еще обрабатывается.</span><span class="sxs-lookup"><span data-stu-id="5e59d-p102">The **Online Certificate Request Status** page returns the current status of the certificate. If the certificate is still in a pending approval state with the certification authority administrator—that is, if it has not yet been approved—the status will indicate that the certificate is still being processed.</span></span>
  
<span data-ttu-id="5e59d-107">Если отложенный запрос был утвержден и сертификат успешно получен, состояние показывает, что сертификат назначен локальному хранилищу сертификатов, и на панели состояния отображается отпечаток сертификата.</span><span class="sxs-lookup"><span data-stu-id="5e59d-107">If the pending request has been approved and the retrieval of the certificate is successful, the status will indicate that a certificate has been assigned to the local certificate store, and the status pane will also display the certificate thumbprint.</span></span>
  
<span data-ttu-id="5e59d-108">Если администратор центра сертификации отклоняет запрос сертификата, на панели состояния появляется сообщение о том, что отложенный сертификат с указанным идентификатором (который предоставляется центром сертификации) отклонен.</span><span class="sxs-lookup"><span data-stu-id="5e59d-108">If the certification authority administrator denies the certificate request, the status pane will report that the request of the pending certificate with a given ID (maintained by the certification authority) is denied.</span></span>
  
<span data-ttu-id="5e59d-109">Чтобы завершить обработку отложенного сертификата, нажмите кнопку **Готово**.</span><span class="sxs-lookup"><span data-stu-id="5e59d-109">Click **Finish** to complete the pending certificate processing.</span></span>
