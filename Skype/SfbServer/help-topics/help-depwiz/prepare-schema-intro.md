# <a name="prepare-schema-intro"></a>Подготовка схемы (введение)
 
Чтобы начать задачу расширения схемы на Skype для бизнеса Server 2015 г., вы начинаете на странице **Подготовка схемы**. Нажмите кнопку **Далее**, чтобы продолжить подготовку схемы.
