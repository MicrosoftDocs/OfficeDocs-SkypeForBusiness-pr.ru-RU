# <a name="certificate-request-organization-information"></a>Запрос сертификата (сведения об организации)
 
В области **Сведения об организации** позволяет указать имя вашей организации и подразделения (OU) (обычно отдела или подразделения), для которого запрашивается сертификат. Данные, введите в текстовых полях **Подразделение** и **организация** отображается в сертификате и используется для идентификации, запросившего сертификат, при просмотре сертификат с другими пользователями.
  
> [!NOTE]
> Имя Подразделения в сертификате не обязательно то же, что Подразделения в доменных службах Active Directory. В зависимости от того, как вы настроили Active Directory имя, которое вы используете может или не может быть одинаковым.
