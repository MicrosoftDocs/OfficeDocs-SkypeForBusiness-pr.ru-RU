# <a name="certificate-request-certificate-signing-request"></a>Запрос сертификата (запрос подписи сертификата)
 
Чтобы создать автономный запрос, необходимо сначала создать запрос подписи сертификата. В поле **Имя файла** введите путь и имя файла для запроса подписи сертификата. Можно нажать кнопку **Обзор** и выбрать место для создания запроса подписи сертификата. Нажмите кнопку **Далее**, чтобы продолжить.
  
Чтобы вернуться на предыдущий экран, нажмите кнопку **Назад**. Чтобы выйти без внесения изменений в систему, нажмите кнопку **Отмена**.
  
> [!IMPORTANT]
> Файл, созданный на этом шаге, отправляется в ваш общий центр сертификации (ЦС), в ваш внутренний автономный ЦС или администратору вашего ЦС для обработки. Запомните местонахождение и имя файла, чтобы можно было его извлечь и передать запрос подписи сертификата в ЦС. По умолчанию файл помещается в каталог "Документы" лица, запускающего мастер сертификатов.
