# <a name="certificate-assignment-invoke"></a>Назначение сертификата (вызов)
 
На странице **Выполнение команд** отображаются действия, выполняемые задачами для назначения сертификата серверу. По завершении задач можно просмотреть журнал, содержащий подробные сведения о процессе назначения сертификата. Для просмотра файла журнала нажмите кнопку **Просмотреть журнал**.
