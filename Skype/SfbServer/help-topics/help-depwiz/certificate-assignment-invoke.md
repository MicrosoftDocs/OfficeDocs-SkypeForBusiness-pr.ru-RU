# <a name="certificate-assignment-invoke"></a>Назначение сертификата (вызов)
 
На странице **Выполнение команд** отображается действиях, выполненных задач назначение сертификата на сервер. После выполнения задач, можно просмотреть журнал, подробно описывает процесс назначения сертификата. Для просмотра файла журнала нажмите кнопку **Просмотреть журнал**.
