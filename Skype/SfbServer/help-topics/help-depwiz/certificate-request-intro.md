# <a name="certificate-request-intro"></a>Запрос сертификата (введение)
 
Чтобы продолжить процесс запроса сертификата, нажмите кнопку **Далее**. Чтобы завершить процесс без внесения изменений в систему, нажмите кнопку **Отмена**.
