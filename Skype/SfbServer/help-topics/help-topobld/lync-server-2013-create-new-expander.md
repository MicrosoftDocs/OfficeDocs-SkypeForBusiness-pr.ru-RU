# <a name="lync-server-2013-create-new-expander"></a>Создание нового расширителя Lync Server 2013
 
Можно создавать новые роли сервера в узле Lync Server 2013 Topology Builder, щелкните правой кнопкой мыши на **Lync Server 2013**, выберите команду **Создать**, а затем выберите Создать роль сервера.
