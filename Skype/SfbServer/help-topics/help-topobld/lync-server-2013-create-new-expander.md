# <a name="lync-server-2013-create-new-expander"></a>Создание нового расширителя Lync Server 2013
 
Можно создать новые роли сервера в узле Lync Server 2013 построитель топологий, щелкнув правой кнопкой мыши  **Lync Server 2013,** выберите "Создать" и выберите роль сервера, которую необходимо создать.
