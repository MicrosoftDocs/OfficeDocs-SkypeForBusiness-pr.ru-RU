# <a name="lync-server-2013-create-new-expander"></a>Создание нового расширителя Lync Server 2013
 
Вы можете создать новые роли сервера в узле Topology Builder Lync Server 2013 правой кнопкой мыши на **Lync Server 2013,** выберите **New**, а затем выберите роль сервера для создания.
