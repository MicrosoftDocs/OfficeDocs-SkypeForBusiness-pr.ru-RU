# <a name="lync-server-2013-create-new-expander"></a>Создание нового расширителя Lync Server 2013
 
Вы можете создать новые роли сервера на узле Lync Server 2013 в построителе топологии, щелкнув правой кнопкой мыши **Lync server 2013**, выбрать команду **создать**, а затем выбрать роль сервера для создания.
