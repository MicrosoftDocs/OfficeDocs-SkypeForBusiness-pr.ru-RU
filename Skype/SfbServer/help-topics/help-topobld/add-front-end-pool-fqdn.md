# <a name="add-front-end-pool-fqdn"></a>Добавление полного доменного имени интерфейсного пула
 
Укажите полное доменное имя создаваемого интерфейсного пула. Невозможно изменить полное доменное имя пула после публикации топологии, содержащей интерфейсный пул. Если необходимо переименовать пул, следует удалить пул, а затем добавить новый пул с новым полным доменным именем.
  
> [!TIP]
> Если в дальнейшем планируется внедрить интерфейсный пул, выберите **Пул с несколькими компьютерами**. Несмотря на то, что пул является группой компьютеров с балансировкой нагрузки, можно создать пул, состоящий из одного компьютера, а затем присвоить полное доменное имя этому компьютеру. Когда вы будете готовы к добавлению дополнительных компьютеров в пул, необходимо повторно запустить построитель топологий, чтобы определить новый член пула, опубликовать новую топологию, а затем настроить новый член пула переднего плана с помощью мастера развертывания Skype для бизнеса Server. Необходимо также добавить этого нового участника пула в соответствующие балансировщики нагрузки пула: балансировка нагрузки DNS или аппаратные средства балансировки нагрузки. Во многих случаях необходимо задействовать обе системы балансировки нагрузки. Обязательно добавьте новый сервер участника в оба средства балансировки.
