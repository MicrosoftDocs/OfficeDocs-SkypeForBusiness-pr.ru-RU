# <a name="add-front-end-pool-fqdn"></a>Добавление полного доменного имени пула переднего плана
 
Укажите полное доменное имя (FQDN) для пула переднего плана, который вы хотите создать. После публикации топологии с интерфейсом переднего плана вы не можете изменить полное доменное имя пула в пуле. Если вам нужно переименовать пул, необходимо удалить пул и добавить новый пул с новым полным доменным именем.
  
> [!TIP]
> Если вы планируете реализовать интерфейсную группу переднего плана в будущем, выберите **несколько пулов компьютеров**. Хотя пул определяется как несколько компьютеров с балансировкой нагрузки, можно создать пул из одного компьютера и присвоить этому компьютеру полное доменное имя пула. Когда вы будете готовы добавить другие компьютеры в пул позже, необходимо запустить построитель топологии еще раз, чтобы определить нового участника пула, опубликовать новую топологию и настроить новый участник пула с помощью мастера развертывания Skype для бизнеса Server. Кроме того, необходимо добавить нового участника пула в соответствующие подсистемы балансировки нагрузки для пула, балансировки нагрузки службы доменных имен (DNS) или подсистем балансировки нагрузки для оборудования. Во многих случаях вы должны иметь обе системы балансировки нагрузки. Убедитесь, что вы добавляете нового сервера участника в оба.
