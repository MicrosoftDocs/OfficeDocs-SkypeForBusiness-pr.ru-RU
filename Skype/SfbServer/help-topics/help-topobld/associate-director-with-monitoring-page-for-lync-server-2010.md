# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Сопоставления директора с мониторингом страницы для Lync Server 2010
 
Чтобы настроить сервер мониторинга и связать его с директора или директора пула:
  
В разделе **Сервер мониторинга** выберите в списке сервер мониторинга.
  
Если вы хотите создать сервер мониторинга теперь нет сервера мониторинга, нажмите кнопку **Создать**.
  
 **Назад**: возврат в предыдущее диалоговое окно
  
 **Готово**: Фиксация изменений и завершения работы мастера
  
 **Отмена**: Отмена изменений и очистка конфигурации
  
 **Справка**: Отображение справочной информации, такой как данный раздел
