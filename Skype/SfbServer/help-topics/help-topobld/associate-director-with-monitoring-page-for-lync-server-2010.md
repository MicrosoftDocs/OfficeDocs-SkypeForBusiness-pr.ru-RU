# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Страница сопоставления директора с мониторингом для Lync Server 2010
 
Чтобы настроить сервер мониторинга и сопоставить его с директором или пулом:
  
В разделе **сервер мониторинга** выберите сервер мониторинга в списке.
  
Если сервер мониторинга отсутствует и вы хотите создать сервер мониторинга сейчас, нажмите кнопку **создать**.
  
 **Назад**. Возврат в предыдущее диалоговое окно
  
 **Готово**. Сохранение изменений и завершение работы мастера
  
 **Отмена**. Отмена изменений и очистка конфигурации
  
 **Справка**: отображение справочной информации, такой как данный раздел
