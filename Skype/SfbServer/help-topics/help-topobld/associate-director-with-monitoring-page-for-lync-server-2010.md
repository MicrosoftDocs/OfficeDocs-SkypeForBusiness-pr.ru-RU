# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Страница сопоставления директора с мониторингом для Lync Server 2010
 
Чтобы настроить сервер мониторинга и связать его с режиссером или режиссером пула, выполните указанные ниже действия.
  
В разделе **сервер мониторинга** выберите элемент из списка сервер мониторинга.
  
Если сервер мониторинга отсутствует и вы хотите создать сервер мониторинга сейчас, нажмите кнопку **создать**.
  
 **Назад**: переход к предыдущему диалоговому окну
  
 **Готово**: фиксирует изменения и завершает работу мастера.
  
 **Отмена**: удаляет изменения и отменяет конфигурацию.
  
 **Справка**: Отображает справочные сведения, такие как в этом разделе.
