# <a name="add-av-mcu-machine"></a>Добавление машины A/V MCU
 
Укажите полное доменное имя (FQDN) каждого компьютера, который вы хотите добавить в качестве сервера конференц-связи (A/v) в этом пуле конференц-связи. До публикации топологии можно в любой момент обновить полное доменное имя добавленного к списку компьютера или удалить его из пула. После публикации топологии изменение полного доменного имени требует удаления сервера в построителе топологии и добавления нового сервера в пул с новым полным доменным именем.
