# <a name="add-edge-server-support"></a>Добавление поддержки для пограничного сервера

Вы можете добавить поддержку доступа внешних пользователей или при развертывании первоначальной топологии, или позднее. Дополнительные сведения о добавлении пограничных серверов к существующей топологии см. в разделе [Define Your Edge Topology](/previous-versions/office/lync-server-2013/lync-server-2013-define-your-edge-topology) руководства по развертыванию пограничного сервера.
