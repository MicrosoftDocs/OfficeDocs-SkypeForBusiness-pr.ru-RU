# <a name="add-edge-server-support"></a>Добавление поддержки для пограничного сервера
 
Можно добавить поддержку для доступа внешних пользователей при развертывании первоначальной топологией или позже. Для получения дополнительных сведений о добавлении пограничных серверов в существующей топологии просмотрите [Определение топология с пограничным](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) в документации по развертыванию сервера пограничного сервера.
