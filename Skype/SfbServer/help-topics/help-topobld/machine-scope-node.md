# <a name="machine-scope-node"></a>Узел области компьютера
 
Этот узел отображает полное доменное имя и IP-адрес этого сервера. Можно выполнить следующие действия.
  
- **Изменение свойств**. Открывает страницу **Изменение свойств** для этого сервера, на которой можно изменить полное доменное имя сервера (если топология еще не опубликована), а также конфигурацию IP-адреса этого сервера.
    
- **Удалить**. Удаляет сервер из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
