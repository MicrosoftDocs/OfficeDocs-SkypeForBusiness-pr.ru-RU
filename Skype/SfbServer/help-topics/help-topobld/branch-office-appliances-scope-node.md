# <a name="branch-office-appliances-scope-node"></a>Узел области применения устройств для обеспечения связи в филиалах
 
В работающем узле " **устройства филиала** " перечисляются устройства, которые находятся на этом сайте филиала. Доступно следующее действие.
  
- **Создать устройство для обеспечения связи в филиалах**. Запускает мастер **определения нового** работающего устройства филиала, который используется для добавления нового работающего устройства филиала на этот сайт.
    
В дополнение к параметру бесперебойно работающего устройства филиала вы можете выполнять следующие общие задачи:
  
- **Топология**. Эта команда позволяет задать новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, выполнить объединение с компонентами прежних версий или удалить развертывание.
    
- **Вид**. Эта команда позволяет настроить представление в построителе топологий путем выбора элементов для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
