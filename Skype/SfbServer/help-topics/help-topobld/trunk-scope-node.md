# <a name="trunk-scope-node"></a>Узел области применения линии связи
 
Вы можете щелкнуть правой кнопкой мыши на существующей магистральной внешней сети SIP в построителе топологии, чтобы выполнить указанные ниже действия.
  
- **Изменение свойств**. Запуск мастера **Изменение свойств** для редактирования и или повторного определения конфигурации существующей магистральной линии SIP
    
Помимо изменения свойств, можно выполнить следующие задачи общего характера.
  
- **Топология**. Эта команда позволяет задать новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, выполнить объединение с компонентами прежних версий Lync Server или удалить развертывание.
    
- **Удалить**. Эта команда служит для удаления выбранного на данный момент элемента. Другие зависимости могут препятствовать удалению. В диалоговом окне указаны службы и зависимости, которые необходимо предварительно разрешить для удаления элемента.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
