# <a name="legacy-merge-add-machine"></a>Добавление компьютера при объединении с устаревшей версией
 
Для задания компьютеров, составляющих пул пограничных серверов, введите полное доменное имя хотя бы одного компьютера, входящего в состав консолидированного пограничного пула прежней версии с настроенной балансировкой нагрузки.
