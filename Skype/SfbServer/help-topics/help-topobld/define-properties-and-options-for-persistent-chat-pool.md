# <a name="define-properties-and-options-for-persistent-chat-pool"></a>Определение свойств и параметров для пула сохраняемого чата
 
Настраивайте параметры для вашего пула стойких чат-серверов или настойчивых серверов чата, определяя следующие свойства:
  
 **Отображение имени пула сохраняемого** чата: обязательное свойство, определя которое определяет удобное для пользователя имя, которое будет отображаться для этого сохраняемого сервера чата или пула сохраняемого сервера чата.
  
 **Стойкий порт** чата. Необходимое свойство, которое определит номер порта, который будет прослушивать этот постоянный сервер чата или пул сохраняемого сервера чата.
  
 **Включить соответствие** требованиям. Выберите контрольный ящик, если планируется развертывание и реализация необязательных функций соответствия требованиям и базы данных сохраняемой системы чата.
  
 Используйте **резервные SQL Server,** чтобы включить аварийное восстановление: Выберите этот контрольный ящик, если вы планируете развернуть и реализовать аварийное восстановление хранилища SQL Server сохраняемого чата из настроенного резервного набора магазинов на другом SQL Server. Подробные сведения см. в [материале Configure high availability and disaster recovery for Persistent Chat Server в Skype для бизнеса Server 2015 г.](../../deploy/deploy-persistent-chat-server/configure-hadr-for-persistent-chat.md)
  
> [!NOTE]
> Этот параметр доступен только для пулов из нескольких серверов. 
  
 **Используйте этот пул в \<site that this server or pool is being configured in\>** качестве по умолчанию для сайта: Выберите этот контрольный ящик, если это будет постоянный сервер чата по умолчанию или пул стойких чат-серверов для сайта. Вы должны иметь один постоянный сервер чата по умолчанию или pol на сайте.
  
> [!NOTE]
> Если топология включает несколько сайтов, также отображается флажок **Использовать пул по умолчанию для всех сайтов**.
  
Нажмите кнопку **Назад**, чтобы вернуться в предыдущее диалоговое окно определения пула.
  
Нажмите **кнопку Далее** после завершения ввода параметров для этого пула, чтобы приступить к определению пула постоянных чат-серверов.
  
Нажмите кнопку **Отмена**, чтобы отменить все изменения и завершить работу с мастером **Определение нового пула сохраняемого чата**.
  
Нажмите кнопку **Справка**, чтобы открыть контекстно-зависимую справку, аналогичную данной странице.
