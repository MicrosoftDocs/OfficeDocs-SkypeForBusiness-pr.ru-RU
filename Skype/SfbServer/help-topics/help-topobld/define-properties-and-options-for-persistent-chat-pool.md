# <a name="define-properties-and-options-for-persistent-chat-pool"></a>Определение свойств и параметров для пула сохраняемого чата
 
Вы можете настроить параметры сервера сохраняемого чата или пула серверов сохраняемого чата, определив следующие свойства:
  
 **Отображаемое имя пула сохраняемого чата**: обязательное свойство, которое определяет понятное имя пользователя, которое будет отображаться для этого сервера сохраняемого чата или пула серверов сохраняемого чата.
  
 **Порт сохраняемого чата**: обязательное свойство, которое определяет номер порта, который будет прослушивать этот сервер сохраняемого чата или пул серверов сохраняемого чата.
  
 **Включить соответствие**: Установите этот флажок, если планируется развертывание и реализация дополнительной функции обеспечения соответствия сохраняемого чата и базы данных.
  
 **Используйте резервное копирование хранилищ SQL Server для включения аварийного восстановления**: Установите этот флажок, если планируется развертывание и реализация аварийного восстановления хранилищ SQL Server сохраняемого чата из настроенного резервного набора хранилищ на другом сервере SQL Server. Дополнительные сведения: [Настройка высокой доступности и аварийного восстановления для сервера сохраняемого чата в Skype для бизнеса server 2015](../../deploy/deploy-persistent-chat-server/configure-hadr-for-persistent-chat.md).
  
> [!NOTE]
> Этот параметр доступен только для пулов из нескольких серверов. 
  
 **Использовать этот пул по умолчанию для сайта \<site that this server or pool is being configured in\> **: Установите этот флажок, если это будет сервер сохраняемого чата по умолчанию или пул серверов сохраняемого чата для сайта. Для каждого сайта должен быть установлен один сервер сохраняемого чата по умолчанию или POL.
  
> [!NOTE]
> Если топология включает несколько сайтов, также отображается флажок **Использовать пул по умолчанию для всех сайтов**.
  
Нажмите кнопку **Назад**, чтобы вернуться в предыдущее диалоговое окно определения пула.
  
После завершения ввода параметров для этого пула нажмите кнопку **Далее** , чтобы продолжить определение пула серверов сохраняемого чата.
  
Нажмите кнопку **Отмена**, чтобы отменить все изменения и завершить работу с мастером **Определение нового пула сохраняемого чата**.
  
Нажмите кнопку **Справка**, чтобы открыть контекстно-зависимую справку, аналогичную данной странице.
