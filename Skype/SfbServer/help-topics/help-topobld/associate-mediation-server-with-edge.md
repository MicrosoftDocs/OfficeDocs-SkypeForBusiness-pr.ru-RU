# <a name="associate-mediation-server-with-edge"></a>Связывание сервера-посредника с пограничным сервером
 
Чтобы определить сервер-посредник, укажите параметры, которые служат для сопоставления данного сервера-посредника с интерфейсным сервером и с пограничным сервером или пулом. Чтобы сопоставить сервер-посредник или пул с пограничным сервером или пулом, выполните следующие действия.
  
1. В раскрывающемся списке выберите пограничный пул или сервер, который следует сопоставить с сервером-посредником.
    
2. Нажмите кнопку **Далее**, чтобы продолжить.
    
> [!NOTE]
> Если в развертывании еще не определен пограничный сервер или пул, в раскрывающемся списке не будет каких-либо значений. Можно нажать кнопку **Далее**, чтобы вернуться к редактированию параметров сервера-посредника позже, после добавления в развертывание пограничного сервера или пула.
  
Дополнительные сведения можно найти в статье [Deploy a Server-посредника в построителе топологий в Skype для бизнеса Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
