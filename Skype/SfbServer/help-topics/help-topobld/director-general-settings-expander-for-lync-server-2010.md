# <a name="director-general-settings-expander-for-lync-server-2010"></a>Расширитель общих параметров директора для Lync Server 2010
 
Чтобы настроить **Общие** свойства директора или Director, сделайте следующее:
  
 **FQDN**: введите полное доменное имя для режиссера или пула каталогов.
  
 **Связи**
  
 **Хранилище файлов**: выберите в списке хранилище файлов для режиссера или для пула режиссеров.
  
Нажмите кнопку " **создать** ", чтобы создать новое хранилище файлов
  
Установите флажок " **сопоставить сервер мониторинга** " и выберите сервер мониторинга из списка.
  
Нажмите кнопку " **создать** ", чтобы определить новый сервер мониторинга
  
 **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
 **Отмена**. Отмена изменений и закрытие диалогового окна.
  
 **Справка**. Отображение этого экрана справки.
