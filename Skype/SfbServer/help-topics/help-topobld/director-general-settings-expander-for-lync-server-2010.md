# <a name="director-general-settings-expander-for-lync-server-2010"></a>Расширитель общих параметров директора для Lync Server 2010
 
Настройка **общих** свойств пула Директора или директора:
  
 **FQDN:** введите полное доменное имя директора или пула директоров
  
 **Ассоциации**
  
 **Хранилище файлов:** Выберите из списка хранилище файлов для пула директора или директора
  
Щелкните **Создать** для определения нового хранилища файлов
  
Выберите **контрольный ящик Associate Monitoring Server** и выберите сервер мониторинга из списка
  
Щелкните **Новое,** чтобы определить новый сервер мониторинга
  
 При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
 При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
 При нажатии кнопки **Справка** отображается данный экран справки.
