# <a name="director-general-settings-expander-for-lync-server-2010"></a>Расширитель общих параметров директора для Lync Server 2010
 
Чтобы настроить **Общие** свойства директора или пула директоров.
  
 **Полное доменное имя**: Введите полное доменное имя директора или пула директоров
  
 **Связи**
  
 **Хранение файлов**: выберите в списке хранилище файлов для директора или пула директоров
  
Нажмите кнопку **Создать** , чтобы определить новое хранилище файлов
  
Установите флажок **Сопоставить Север мониторинга** и выберите в списке сервер мониторинга
  
Нажмите кнопку **Создать** , чтобы определить новый сервер мониторинга
  
 **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
 **Отмена**. Отмена изменений и закрытие диалогового окна.
  
 **Справка**. Отображение этого экрана справки.
