# <a name="add-edge-server-external-ipv6"></a>Добавление внешнего адреса IPv6 пограничного сервера
 
В поле **внешний IPv6-адрес**введите внешний IP-адрес, указывающий на службу пограничного сервера доступа, пограничной службы веб-конференций и службу пограничного сервера аудио-и видеоконференций.
