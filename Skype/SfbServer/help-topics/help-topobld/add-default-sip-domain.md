# <a name="add-default-sip-domain"></a>Добавление домена SIP по умолчанию
 
Укажите основной домен Session Initiation Protocol (SIP), который будет развертывания для поддержки. Основной домен SIP может быть таким же, как, или отличается от домена доменных служб Active Directory. К примеру вашего домена Active Directory может быть fabrikam.net и можно указать основного SIP-домена быть contoso.net.
