# <a name="add-default-sip-domain"></a>Добавление домена SIP по умолчанию
 
Укажите основной домен SIP, который должно поддерживать ваше развертывание. Основной домен SIP может совпадать с доменом доменных служб Active Directory (или отличаться от него). Например, вы можете использовать домен Active Directory fabrikam.net и указать основной домен SIP contoso.net.
