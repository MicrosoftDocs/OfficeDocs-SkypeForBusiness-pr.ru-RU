# <a name="remove-deployment"></a>Удаление развертывания
 
Нажмите **кнопку** "Готово", чтобы создать пустую топологию в качестве первого шага при выводе из эксплуатации всего развертывания.
  
> [!CAUTION]
> Это действие невозможно отменить.
