# <a name="edge-server-fqdn-settings-expander-for-lync-server-2010"></a>Расширитель параметров полного доменного имени пограничного сервера для Lync Server 2010
 
Чтобы определить свойства в разделе **внешние параметры**, выполните указанные ниже действия.
  
Установите флажок **включить отдельное полное доменное имя и IP-адрес для конференций и/V** , если вы хотите задать индивидуальные доменные имена и IP-адреса для разных пулов и звуковых и видеоконференций.
  
> [!NOTE]
> Если вы решили не установить флажок для отдельных полных доменных имен и IP-адресов, необходимо предоставить отдельные порты для каждой из трех служб, предоставляемых пограничным сервером. Единственным полным доменным именем, которое нужно настроить, является полное доменное имя, связанное со службой Edge Access. 
  
Установите флажок **Служба пограничного устройства NAT включена** , если вы хотите, чтобы служба EDGE (a/v) ИСПОЛЬЗОВАЛА IP-адрес и конфигурацию преобразования сетевых адресов (NAT).
  
Для включенных служб Edge вы вводите **полное доменное имя пула** и порт в разделе **ports (порты** ).
  
- Определите полное доменное имя пула **службы Edge Access** и порт, который однозначно определяет службу.
    
- Определите полное доменное имя пула **служб для веб-конференций** (если не выбрано отдельное полное доменное имя и IP-адрес) и порт, который однозначно определяет службу.
    
- Определите полное доменное имя пула **служб EDGE (a/v** ) (если включено отдельное полное доменное имя и IP-адрес, а не выбрано значение A/V) и порт, который однозначно определяет службу.
