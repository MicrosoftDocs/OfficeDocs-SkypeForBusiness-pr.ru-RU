# <a name="publish-topology-select-cms-page"></a>Страница публикации CMS выбора топологии

Топология, настроенная с помощью построитель топологий, публикуется. Вам будет предложено выбрать из списка, какой сервер переднего плана или пул переднего плана будут выполнять роль хранения центрального банка управления. В любой момент времени эту роль может использовать только один сервер переднего плана или пул переднего плана.

Дополнительные сведения о том, что такое центральное хранилище управления, см. в перенаправлении центрального сервера управления [Lync Server 2010 в Lync Server 2013.](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx)
