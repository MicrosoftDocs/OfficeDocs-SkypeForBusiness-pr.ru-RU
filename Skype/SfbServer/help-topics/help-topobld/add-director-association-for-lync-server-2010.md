# <a name="add-director-association-for-lync-server-2010"></a>Добавление ассоциации директоров для Lync Server 2010
 
Вы определяете, будет ли пул Director или Director иметь связанный с ним сервер мониторинга. Выберите контрольный ящик, чтобы связать сервер мониторинга с пулом Director или Director.
  
> [!TIP]
> Если вы еще не создали базу данных Monitoring Server (и необязательный экземпляр), вам будет предоставлена возможность создать ее позже, если вы выберете этот контрольный ящик, чтобы связать сервер мониторинга с пулом Директора или Директора.
