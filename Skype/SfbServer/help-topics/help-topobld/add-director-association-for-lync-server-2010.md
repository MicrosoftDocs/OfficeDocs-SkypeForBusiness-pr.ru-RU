# <a name="add-director-association-for-lync-server-2010"></a>Добавление связи директора для Lync Server 2010
 
Определяется, будет ли с директором или пулом директоров связан сервер мониторинга. Чтобы связать сервер мониторинга с директором или пулом директоров, выберите его.
  
> [!TIP]
> Если вы еще не создали базу данных сервера мониторинга (и необязательный экземпляр), вам будет предоставлена возможность создать ее позже, если этот установлен, чтобы связать сервер мониторинга с директором или пулом директоров.
