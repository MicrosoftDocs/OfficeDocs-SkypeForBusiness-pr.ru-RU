# <a name="add-director-association-for-lync-server-2010"></a>Добавление ассоциации режиссера для Lync Server 2010
 
Вы определяете, какой из пулов и каталогов будет иметь сервер мониторинга, связанный с ним. Установите флажок, чтобы связать сервер мониторинга с режиссером или режиссером пула.
  
> [!TIP]
> Если вы еще не создали базу данных сервера мониторинга (и Необязательный экземпляр), вам будет предоставлена возможность ее создания позже, если установить этот флажок, чтобы связать сервер мониторинга с режиссером или режиссером пула.
