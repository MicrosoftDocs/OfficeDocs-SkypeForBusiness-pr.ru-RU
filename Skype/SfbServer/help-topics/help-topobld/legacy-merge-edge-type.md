# <a name="legacy-merge-edge-type"></a>Тип пограничного сервера при объединении с устаревшей версией
 
Для **отдельного пограничного сервера** или **пограничного объединенного сервера с балансировкой нагрузки** нажмите кнопку **Далее**, чтобы указать сведения о конфигурации. Для **расширенного края** конфигурацию каждого сервера необходимо изменить на консолидированную конфигурацию Edge, прежде чем она может быть объединена с Lync Server 2013.
