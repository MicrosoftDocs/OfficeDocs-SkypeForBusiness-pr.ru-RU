# <a name="legacy-merge-edge-type"></a>Тип пограничного сервера при объединении с устаревшей версией
 
Для **отдельного пограничного сервера** или **пограничного объединенного сервера с балансировкой нагрузки** нажмите кнопку **Далее**, чтобы указать сведения о конфигурации. Для **расширенного пограничного**сервера перед его слиянием с Lync Server 2013 необходимо изменить конфигурацию каждого сервера на объединенную конфигурацию пограничного сервера.
