# <a name="simple-url-settings-expander"></a>Расширитель параметров простого URL-адреса

Чтобы указать новые, удалить существующие или изменить активные простые URL-адреса, выполните указанные ниже действия.

## <a name="simple-urls"></a>Простые URL-адреса

Чтобы изменить или определить простые URL-адреса, сначала решите, какие из трех простых URL-адресов нужно изменить или изменить. Можно выбрать URL-адрес для доступа с телефона, URL-адрес собрания и URL-адрес административного доступа.

Если требуется изменить URL-адрес для доступа с телефона или собрания, выберите этот URL-адрес. Щелкните **Изменить URL-адрес**. Измените URL-адрес и нажмите кнопку **ОК** , чтобы сохранить URL-адрес. Для отмены изменений нажмите кнопку **Отмена**.

Для добавления нового URL-адреса нажмите кнопку **Добавить**. В диалоговом окне **Добавление простого URL-адреса** укажите URL-адрес и нажмите кнопку **ОК** , чтобы сохранить URL-адрес. Если этот URL-адрес должен быть активным, выберите **Задать этот адрес в качестве активного URL-адреса для выбранного домена**. Для отмены изменений нажмите кнопку **Отмена**.

Чтобы сделать URL-адрес активным (как отмечен зеленой галочкой рядом с URL-адресом), выберите URL-адрес и нажмите кнопку **сделать активным**.

> [!NOTE]
> Для каждого домена SIP может быть только один активный URL-адрес.

Чтобы удалить URL-адрес, выделите его и нажмите кнопку **Удалить**.

> [!CAUTION]
> Не забудьте прочитать сведения на странице "параметры простых URL-адресов". Удаление URL-адреса собрания может привести к отображению существующих собраний, запланированных пользователями, которые больше не доступны. После того как вы сделаете URL-адрес собрания активным, можно покинуть существующий URL-адрес. Если вы уверены, что пользователи больше не используют старый URL-адрес собрания, вы можете удалить его.

Чтобы изменить URL-адрес административного доступа, отредактируйте запись.

После внесения всех изменений на странице изменить свойства нажмите кнопку **ОК** , чтобы сохранить изменения. Для отмены изменений нажмите кнопку **Отмена**.
