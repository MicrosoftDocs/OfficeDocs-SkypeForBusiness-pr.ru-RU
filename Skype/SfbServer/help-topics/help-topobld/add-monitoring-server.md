# <a name="add-monitoring-server"></a>Добавление сервера мониторинга

Чтобы включить сервер мониторинга в схему топологии, необходимо указать полное доменное имя сервера, на котором требуется развернуть сервер мониторинга. Перед публикацией топологии, включающей сервер мониторинга и установку Skype для бизнеса Server на сервере, необходимо выполнить все необходимые условия для развертывания мониторинга. Сведения об этих предварительных требованиях см. в разделе [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) в документации по развертыванию.
