# <a name="lync-server-2010-create-new-expander"></a>Создание нового расширителя Lync Server 2010
 
Вы можете создать новые роли сервера на узле Lync Server 2010 в построителе топологии, щелкнув правой кнопкой мыши **Lync server 2010**, выбрать команду **создать**, а затем выбрать роль сервера для создания.
