# <a name="lync-server-2010-create-new-expander"></a>Создание нового расширителя Lync Server 2010
 
Чтобы создать новые роли сервера в узле Lync Server 2010 в построителе топологий, щелкните правой кнопкой мыши **Lync server 2010**, выберите пункт **создать**, а затем выберите роль сервера для создания.
