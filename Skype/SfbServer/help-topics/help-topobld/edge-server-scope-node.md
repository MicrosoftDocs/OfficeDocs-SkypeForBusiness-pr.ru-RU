# <a name="edge-server-scope-node"></a>Узел области пограничного сервера
 
Этот узел отображает полное доменное имя пула и другие базовые параметры пограничного пула. Доступны следующие действия.
  
- **Создать сервер**. Запускает мастер **Определение нового сервера**, с помощью которого можно добавить в этот пул другой пограничный сервер.
    
- **Изменение свойств**. Открывает страницу **Изменение свойств** для этого сервера, на которой можно определить внутреннее и внешнее полное доменное имя и порт пула, включить или отключить федерацию и преобразование адресов NAT, а также указать следующий элемент.
    
- **Удалить**. Удаляет пул из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
