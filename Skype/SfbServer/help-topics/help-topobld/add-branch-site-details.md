# <a name="add-branch-site-details"></a>Добавление сведений о среде филиала
 
Для задания сведений о сайте выполните следующие действия.
  
Укажите название города для нового сайта филиала. Значение города не является обязательной записью.
  
Укажите название области или региона для нового сайта филиала. Значение областью или региона не является обязательной записью.
  
Укажите название страны или региона для нового сайта филиала. Значение страны или региона не является обязательной записью.
  
[Для получения дополнительных сведений см.](http://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx)
