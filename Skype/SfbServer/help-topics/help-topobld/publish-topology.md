# <a name="publish-topology"></a>Публикация топологии
 
Страница **публикации топологии** отображается при запуске процесса публикации. Прочтите и убедитесь, что приведенные задачи завершены, прежде чем продолжить. Когда вы будете готовы продолжить, нажмите кнопку **Далее**. Если необходимо закрыть процесс публикации, нажмите кнопку **Отмена**.
