# <a name="publish-topology"></a>Публикация топологии
 
Страница **Публикация топологии** отображается при запуске публикации. Прежде чем продолжить, убедитесь, что указанные задачи завершены. Когда вы будете готовы продолжить, нажмите кнопку **Далее**. Если необходимо прервать процесс публикации, нажмите кнопку **Отмена**.
