# <a name="associate-branch-office-appliance-with-edge"></a>Связывание устройства для обеспечения связи в филиалах с пограничным сервером
 
Если развернуты edge Servers и вам необходимо предоставить службы мультимедиа для внешнего использования, выберите edge Server или пул, который будет связан с устройством для ветвей или уцелевшим сервером, выбрав его в выпадаемом списке. Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> При отсутствии определенного в топологии пограничного сервера или пула, можно пропустить это действие. При определении нового пула Edge Server или Edge Server, а также при редактировании страниц свойств аппарата для выживших филиалов или сервера-филиала после определения пула edge Server или Edge Server вы сможете связать сохранившийся ветвейный аппарат или выживший сервер филиала после определения в топологии пула edge Server или Edge Server.
