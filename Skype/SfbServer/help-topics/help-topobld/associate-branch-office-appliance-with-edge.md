# <a name="associate-branch-office-appliance-with-edge"></a>Связывание устройства для обеспечения связи в филиалах с пограничным сервером
 
Если развернуты edge Servers и необходимо предоставить службы мультимедиа для внешнего использования, выберите этот сервер или пул, который будет связан с устройством для обеспечения связи в филиалах или сервером для обеспечения связи в филиалах, выбрав его в выпадаемом списке. Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> При отсутствии определенного в топологии пограничного сервера или пула, можно пропустить это действие. Вы сможете связать устройство или сервер для survivable Branch Server с edge server или пулом edge server при определении нового пула или пула серверов или при редактировании страниц свойств устройства или сервера для survivable Branch Server после определения в топологии пула.
