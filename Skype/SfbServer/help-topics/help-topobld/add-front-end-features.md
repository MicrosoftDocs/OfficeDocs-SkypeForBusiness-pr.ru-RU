# <a name="add-front-end-features"></a>Добавление компонентов переднего плана

Выберите все возможности, которые хотите включить для интерфейсного пула. Следует развернуть по крайней мере систему обмена мгновенными сообщениями, так как эта возможность всегда выбрана и не может быть отключена. Вы также можете развернуть другие функции, такие как конференц-связь, Корпоративная голосовая связь и контроль допуска звонков (CAC). Некоторые компоненты являются зависимыми от топологии.

- Конференции

- Корпоративная голосовая связь

- Контроль допуска звонков

- Архивация

- Мониторинг

> [!NOTE]
> Если вы хотите включить в развертывании компонент контроля допуска звонков, следует включить его только в одном пуле на центральный сайт. Мы рекомендуем использовать контроль допуска звонков, если вы развертываете компоненты голосовой связи или компонент аудио- и видеоконференций.

Дополнительные сведения об указании возможностей для пула Enterprise Edition см. в разделе [Define and Configure a Front End Pool](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) документации по развертыванию.
