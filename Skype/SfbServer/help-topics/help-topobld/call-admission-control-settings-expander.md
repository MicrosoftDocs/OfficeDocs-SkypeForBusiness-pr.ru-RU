# <a name="call-admission-control-settings-expander"></a>Расширитель параметров контроля допуска звонков
 
Контроль доступа звонков — это сеть регионов, сайтов и подсетей, которые позволяют вам налагать ограничения на передачу звука и видео в зависимости от доступной пропускной способности. После настройки сети на использование контроля доступа звонков, следует включить контроль доступа звонков для применения заданных ограничений. 
  
> [!NOTE]
> Вы также можете использовать панели управления или команды оболочки управления, чтобы включить CAC. 
  
В разделе **Параметры контроля допуска звонков** диалогового окна **Изменение свойств** для вашего сайта можно изменить следующие параметры:
  
- **Включить управление приемом вызовов** Выберите этот параметр, чтобы включить CAC. Очистить этот параметр, чтобы отключить CAC для всей сети. Чтобы включить CAC, необходимо настроить сеть для CAC. Дополнительные сведения см. в документе Развертывание управления [приемом вызовов Skype для бизнеса Server 2015](../../deploy/deploy-enterprise-voice/deploy-call-admission-control.md) г. в документации по развертыванию.
    
- **Пул переднего конца для запуска управления приемом вызовов** Если вы включили CAC, вы можете изменить пул, который его запускает. Выберите пул из выпадаемого списка.
