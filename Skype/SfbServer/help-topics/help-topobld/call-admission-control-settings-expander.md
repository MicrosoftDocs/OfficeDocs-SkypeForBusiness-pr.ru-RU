# <a name="call-admission-control-settings-expander"></a>Расширитель параметров контроля допуска звонков
 
Контроль доступа звонков — это сеть регионов, сайтов и подсетей, которые позволяют вам налагать ограничения на передачу звука и видео в зависимости от доступной пропускной способности. После настройки сети на использование контроля доступа звонков, следует включить контроль доступа звонков для применения заданных ограничений. 
  
> [!NOTE]
> Контроль доступа звонков можно также включить на панели управления или с помощью командлетов командной консоли. 
  
В разделе **Параметры контроля допуска звонков** диалогового окна **Изменение свойств** для данного сайта можно изменить следующие параметры.
  
- **Включить управление допуском звонков** Выберите этот параметр, чтобы включить CAC. При его снятии контроль допуска звонков отключается для всей сети. Для включения контроля допуска звонков необходимо соответствующим образом настроить сеть. Подробнее смотрите в разделе [Развертывание элемента управления допуском звонков в Skype для бизнеса Server 2015](../../deploy/deploy-enterprise-voice/deploy-call-admission-control.md) в документации по развертыванию.
    
- **Пул переднего плана, на котором будет выполняться управление допуском звонков** Если включена служба CAC, вы можете изменить пул, на котором он выполняется. Выберите пул в раскрывающемся списке.
