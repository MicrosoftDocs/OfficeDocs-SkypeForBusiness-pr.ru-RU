# <a name="call-admission-control-settings-expander"></a>Расширитель параметров контроля допуска звонков
 
Контроль доступа звонков — это сеть регионов, сайтов и подсетей, которые позволяют вам налагать ограничения на передачу звука и видео в зависимости от доступной пропускной способности. После настройки сети на использование контроля доступа звонков, следует включить контроль доступа звонков для применения заданных ограничений. 
  
> [!NOTE]
> Для включения контроля допуска звонков также можно использовать панель управления или командлеты командной консоли. 
  
В разделе **Параметры контроля допуска звонков** диалогового окна **Изменение свойств** для вашего сайта можно изменить следующие параметры:
  
- **Включение контроля допуска звонков** Выберите этот параметр, чтобы включить службу контроля допуска звонков. Снимите этот флажок, чтобы отключить CAC для всей сети. Чтобы включить службу контроля допуска звонков, необходимо настроить сеть для CAC. Дополнительную информацию можно узнать в статье Deployment [контроля допуска звонков в Skype для бизнеса Server 2015](../../deploy/deploy-enterprise-voice/deploy-call-admission-control.md) в документации по развертыванию.
    
- **Интерфейсный пул для запуска контроля допуска звонков** Если включена служба контроля допуска звонков, вы можете изменить пул, который его выполняет. Выберите пул из раскрывающегося списка.
