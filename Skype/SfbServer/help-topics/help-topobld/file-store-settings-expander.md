# <a name="file-store-settings-expander"></a>Расширитель настроек хранилища файлов
 
Чтобы изменить свойства определенных файловых хранилищ:
  
Для имени файловый ресурс можно определить имя новой папки. Затем необходимо создать общую папку на сервере с именем в свойстве файлового сервера полное доменное имя (FQDN).
  
> [!NOTE]
> Полное доменное имя сервера, который указан в свойстве полное доменное имя сервера файл изменить нельзя. Если требуется поместить хранилища файлов общей папки на другом сервере, создать новую запись файла хранилища.
