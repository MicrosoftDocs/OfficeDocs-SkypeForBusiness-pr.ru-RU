# <a name="file-store-settings-expander"></a>Расширитель настроек хранилища файлов
 
Чтобы изменить свойства определенных хранилищ файлов, выполните указанные ниже действия.
  
Для имени общего файлового файла можно задать имя нового общего доступа. Затем необходимо создать общий доступ на сервере с именем, указанным в свойстве полное доменное имя (FQDN) файлового сервера.
  
> [!NOTE]
> Вы не можете изменить полное доменное имя сервера, указанное в свойстве FQDN файлового сервера. Если вам нужно поместить общую файловую систему хранилища файлов на другой сервер, создайте новую запись в хранилище файлов.
