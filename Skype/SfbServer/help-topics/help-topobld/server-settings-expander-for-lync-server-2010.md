# <a name="server-settings-expander-for-lync-server-2010"></a>Расширитель параметров сервера для Lync Server 2010
 
Чтобы изменить свойства компьютера, выполните указанные ниже действия.
  
- Измените полное **доменное имя (FQDN)** для этого компьютера. Этот параметр должен совпадать с именем компьютера, указанным в службе доменных имен (DNS), а также в списке альтернативных имен субъектов (SAN) или имени субъекта (SN) сертификата, связанного с этим компьютером.
    
- Вы выбираете один из указанных ниже вариантов.
    
    **Использовать все настроенные IP-адреса**: Выберите этот параметр, чтобы использовать все НАСТРОЕННЫЕ IP-адреса на компьютере.
    
    > [!IMPORTANT]
    > Если на компьютере установлено несколько IP-адресов, необходимо помнить, что службы, связанные с этим компьютером, будут использовать все IP-адреса для всех служб. Если сервер или служба ожидает связь с определенным IP-адресом и портом, служба может не выбрать оптимальный IP-адрес для прослушивания. 
  
    **Ограничение использования службы до выбранных IP-адресов**: Выберите этот параметр, если вы хотите определить конкретные IP-адреса для **основного IP-адреса** , который компьютер будет прослушивать для обмена данными с другими компьютерами и пулами в развертывании. Определите **IP-адрес PSTN** для определенного IP-адреса, который компьютер и служба будет прослушивать для обмена информацией и отправлять сообщения на определенный шлюз PSTN или IP-УАТС.
