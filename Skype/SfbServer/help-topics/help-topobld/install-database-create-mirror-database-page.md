Это Предварительная документация, которая может быть изменена. Пустые разделы включены в качестве заполнителей.
  
Если вы отказались от установки зеркальных баз данных SQL Server во время процесса публикации, установите их **с помощью правой** кнопки мыши на любом узле построителя топологий. Выберите зеркальное хранилище для установки. Щелкните пункт **Параметры** для определения расширенных параметров.
