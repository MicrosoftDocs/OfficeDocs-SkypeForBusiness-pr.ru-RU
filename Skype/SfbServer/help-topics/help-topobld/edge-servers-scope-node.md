# <a name="edge-servers-scope-node"></a>Узел области пограничных серверов
 
Узел **Пограничные пулы** содержит список пограничных пулов в развернутой топологии. Можно выполнить следующее действие.
  
- **Создать пограничный пул**. Служит для запуска мастера **Определение нового пограничного пула**, который используется для добавления в топологию нового пограничного пула.
    
Кроме параметров пограничного пула, можно выполнять следующие задачи общего характера:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
