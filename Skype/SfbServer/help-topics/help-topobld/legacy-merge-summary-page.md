# <a name="legacy-merge-summary-page"></a>Страница сводки по объединению с устаревшей версией

Миграция Microsoft Office Communications Server 2007 R2 на Lync Server 2013 использует функции Topology Builder для объединения существующих топологий. На сводном экране указаны все действия и выборы, которые были сделаны при конфигурации слияния Office Communications Server 2007 R2 в конфигурацию строителя топологии Lync Server 2013.

Просмотрите эти сводные сведения и оцените приемлемость собственных настроек. Продолжите объединение, если выбранные параметры, приведенные на этом экране, не вызывают сомнений.
