# <a name="install-mirror-database-option-page"></a>Страница параметров установки зеркальной базы данных
 
Настройте **Параметры зеркальной базы данных** , определив следующие значения:
  
- Введите **путь к общей папке** , чтобы задать расположение для резервных файлов SQL Server для зеркальной базы данных.
    
    > [!NOTE]
    > Основной экземпляр SQL Server (как именованный экземпляр, так и экземпляр по умолчанию) должен иметь разрешения на запись в указанную здесь файловую общую. Зеркальному экземпляру SQL Server (именованному экземпляру или экземпляру по умолчанию) должны быть предоставлены разрешения на чтение для того же общего файлового файла. 
  
  **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
  **Отмена**. Отмена изменений и закрытие диалогового окна.
  
  **Справка**. Отображение этого экрана справки.
