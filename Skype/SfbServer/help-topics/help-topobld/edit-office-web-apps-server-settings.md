# <a name="edit-office-web-apps-server-settings"></a>Изменение параметров сервера Office Web Apps

Вы изменяете свойства настроенного сервера Office Web Apps. Для редактирования доступны следующие свойства.

 ПОЛНОЕ доменное имя **сервера Office Web Apps**: это свойство определяет полное доменное имя сервера Office Web Apps и должно быть соответствующим DNS-узлу a или AAAA (если используется IPv6).

 **URL-адрес обнаружения сервера Office Web Apps**: URL-адрес для клиентского доступа к серверу Office Web Apps, возможно, потребуется изменить этот адрес по умолчанию, если сервер размещается в другой зоне сети, отличной от внутренней сети для вашего развертывания.

Установите флажок **Сервер Office Web Apps развернут во внешней сети**, если этот сервер развернут в сети периметра или в другой сетевой зоне за пределами внутреннего брандмауэра, разделяющего сеть периметра, сети без доверия и Интернет от внутреннего развертывания.
