# <a name="sql-instance-scope-node"></a>Узел области применения экземпляра SQL
 
В этом узле отображается полное доменное имя (FQDN) этого Microsoft SQL Server и экземпляр SQL Server на этом сервере, а также пулы или серверы, использующие этот SQL Server. Можно выполнить следующие действия.
  
- **Изменить свойства**. Открывает страницу **изменение свойств** для этого сервера, где можно изменить полное доменное имя сервера (Если топология еще не опубликована) и экземпляра SQL Server, который будет использоваться.
    
- **Удалить**. Удаляет сервер из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
