# <a name="install-and-create-databases"></a>Установка и создание баз данных

Вы выбираете базы данных, которые вы хотите создать для развертывания. По умолчанию база данных будет создана на сервере SQL Server, определенном в определенном сайте, и будет автоматически развертываться и настраиваться для файлов базы данных на основе сервера SQL Server, на котором вы размещаете базы данных.

 **Выберите базы данных, которые вы хотите создать**: установите флажки для всех баз данных, которые вы хотите развернуть и настроить. Установите флажки для всех баз данных, которые вы развернете.

> [!CAUTION]
> Сервер SQL Server должен быть уже настроен для экземпляра (если есть), а порты брандмауэра должны быть открыты для размещения экземпляра, на который развертываются базы данных. Дополнительные сведения можно найти в разделе [Настройка SQL Server для Lync server 2013 Preview](https://technet.microsoft.com/library/375e5cc4-e436-46dc-9b02-5063f35cdcc1.aspx)

 **Дополнительно**: щелкните SQL Server и нажмите кнопку " **Дополнительно** ", чтобы выбрать параметры расположения файла базы данных на сервере SQL Server. Дополнительные сведения о размещении файлов баз данных см. в разделе [Установка баз данных с помощью командной консоли Lync Server](https://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx)

 **Назад**: при нажатии этой кнопки вы вернетесь к предыдущему экрану (в зависимости от того, как вы попали в этом диалоговом окне), это может быть недоступно.

 **Далее**: нажмите эту кнопку, чтобы подтвердить выбор в текущем диалоговом окне и перейти к следующему диалоговому окну для настройки дополнительных сведений.

 **Отмена**: нажатие этой кнопки приведет к закрытию конфигурации и отмене изменений. Некоторые, но не все экраны конфигурации выводят приглашение, если вы захотите закрыть и отменить изменения. Нажмите **кнопку Да** , чтобы закрыть текущую конфигурацию и закрыть текущую конфигурацию и вернуться к построителю топологии. Если нажать кнопку **нет** , откроется диалоговое окно текущее настройки и вы сможете продолжить настройку.

 **Справка**: нажатие кнопки " **Справка** " приводит к отображению этой справочной информации, связанной с текущим диалоговым окном настройки.
