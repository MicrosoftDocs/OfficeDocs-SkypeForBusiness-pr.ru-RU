# <a name="topology-builder-startup-dialog"></a>Диалоговое окно запуска построителя топологии

Вы запускаете построитель топологий, чтобы настроить свойства и конфигурацию развертывания Microsoft Lync Server 2013. При запуске построитель топологий отображает диалоговое окно с запросом источника топологии. На выбор предоставляются три варианта:

- **Загрузка топологии из существующего развертывания** Выберите этот параметр, если вы выполнили построитель топологий и успешно опубликовали топологию на центральном сервере управления. Этот параметр запрашивает и загружает локальную копию текущей конфигурации для последующего редактирования и настройки существующего развертывания.

- **Открытие топологии из локального файла** Выберите этот параметр, если вы настраиваете развертывание, но еще не опубликовали конфигурацию. Этот параметр позволяет сохранить локальную копию работ по настройке в построителе топологий и вернуться к ней позже.

- **Новая топология**. Выберите этот параметр, чтобы начать редактирования нового документа топологии. Этот параметр открывает новый документ топологии для настройки в соответствии с требованиями текущего развертывания.
