# <a name="add-edge-server-associations"></a>Добавление связей для пограничного сервера
 
Используйте эту страницу, чтобы выбрать интерфейсные пулы и серверы Standard Edition, которые будут использовать этот пограничный пул для трафика аудио- и видеоданных. 
  
Каждый интерфейсный пул можно сопоставить всего с одним пограничным пулом. Если выбрать пул, который уже сопоставлен с другим пограничным пулом, сопоставление с предыдущим пограничным пулом будет отменено.
  
Вы также можете связать этот пул Edge Server или Edge с пулами-посредниками или пулами-посредниками для настройки и Корпоративная голосовая связь внешних пользователей.
