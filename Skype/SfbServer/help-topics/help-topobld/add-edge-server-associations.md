# <a name="add-edge-server-associations"></a>Добавление связей для пограничного сервера
 
На этой странице можно выбрать, какие из пулов переднего плана и серверов стандартных выпусков будут использовать этот пул пограничных ресурсов для трафика/V. 
  
Каждый пул интерфейсов переднего плана может быть связан только с одним пулом Edge. Если вы выбрали пул, уже связанный с другим пулом пограничных, он изменит связь от прежнего пула Edge.
  
Вы также можете связать этот граничный сервер или пул Edge с серверами или пулами пограничного сервера исправлений, чтобы настроить и включить корпоративную голосовую связь для внешних пользователей.
