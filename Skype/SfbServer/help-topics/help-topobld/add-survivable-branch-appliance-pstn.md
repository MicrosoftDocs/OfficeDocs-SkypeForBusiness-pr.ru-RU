# <a name="add-survivable-branch-appliance-pstn"></a>Добавление ТСОП для устройства обеспечения связи в филиалах
 
Чтобы определить шлюз в телефонную сеть общего пользования (ТСОП) для устройства обеспечения связи в филиале, укажите следующие параметры. 
  
- Полное доменное имя или IP-адрес шлюза, с которым сопоставлено устройство или сервер обеспечения связи на филиалах, для маршрутизации входящих и исходящих вызовов ТСОП.
    
    > [!IMPORTANT]
    > При определении устройства обеспечения связи на филиалах здесь указывается шлюз, к которому подключается сервер-посредник в устройстве обеспечения связи для выхода в ТСОП. 
  
- Порт прослушивания, который будет использоваться для передачи SIP-сообщений. По умолчанию используются порты TCP 5066 и TLS 5067 на шлюзе, офисной АТС или пограничном контроллере сеансов. На устройстве обеспечения связи на сайте филиала по умолчанию заданы порты 5081 для TCP и 5082 для TLS.
    
- В целях безопасности настоятельно рекомендуем использовать TLS. Если необходимо определить устройство обеспечения связи на филиалах, обратитесь к документации производителей этих устройств, чтобы убедиться в поддержке протокола TLS.
    
    > [!IMPORTANT]
    > В целях безопасности настоятельно рекомендуется развернуть шлюз, который поддерживает TLS. 
  
> [!NOTE]
> Если необходимо добавить шлюз в ТСОП, можно настроить его позже, однако полноценные функциональные возможности будут ограничены до полной настройки шлюза в ТСОП.
