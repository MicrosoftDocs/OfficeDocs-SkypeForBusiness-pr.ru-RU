# <a name="add-survivable-branch-appliance-pool"></a>Добавление пула устройств обеспечения связи в филиалах
 
Чтобы **определить полное**доменное имя устройства для обеспечения связи в филиале, введите полное доменное имя (полное доменное имя) (как указано в DNS) для устройства или сервера для обеспечения связи в филиалах в текстовом поле **полное доменное** имя. Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> Если это не так, то определение устройства для обеспечения связи в филиалах или сервера для обеспечения связи в филиалах является одной и той же процедурой.
