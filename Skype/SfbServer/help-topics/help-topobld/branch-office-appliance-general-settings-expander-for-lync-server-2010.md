# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Расширитель общих параметров устройства для обеспечения связи в филиалах для Lync Server 2010
 
Чтобы изменить свойства для устройства или сервера для survivable Branch Server, в области "Общие" **необходимо** настроить:
  
- **Полное доменное имя:** введите полное доменное имя устройства или сервера survivable Branch Server.
    
    **Для всех настроенных IP-адресов** используются IP-адреса, настроенные для всех целей на устройстве или сервере для survivable Branch Server.
    
    **Ограничить использование службы выбранными IP-адресами**. Можно настроить определенный адрес для определения сервера и IP-адреса для связи с ТСОП.
    
    **Основной IP-адрес.** IP-адрес определяется и настраивается для всех зада, кроме функций, связанных с ТСОП.
    
    **IP-адрес для ТСОП**. IP-адрес, связанный с функциями ТСОП.
    
- Связи **настраиваются** таким образом, чтобы другие роли сервера были настроены и связаны с устройством для обеспечения связи в филиалах или сервером обеспечения связи в филиалах.
    
    **Связывание сервера архива** Выберите в списке сервер архива, который необходимо связать с устройством для связи в филиалах или сервером для связи в филиалах.
    
    Нажмите **кнопку** "Создать", если сервер архива, который необходимо связать с этим устройством или сервером для связи в филиалах, не создан
    
    **Связывать сервер мониторинга** Выберите в списке сервер мониторинга, который необходимо связать с устройством для связи в филиалах или сервером для связи в филиалах.
    
    Нажмите **кнопку** "Создать", если не создан сервер мониторинга, который необходимо связать с этим устройством или сервером для связи в филиалах
    
    **Связывать пул edge (для компонентов мультимедиа)** Выберите в списке сервер или пул, который необходимо связать с устройством для связи в филиалах или сервером для связи в филиалах.
    
  При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
  При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
  При нажатии кнопки **Справка** отображается данный экран справки.
