# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Расширитель общих параметров устройства для обеспечения связи в филиалах для Lync Server 2010
 
Чтобы изменить свойства для выживаемого устройства филиала или выживаемого сервера филиала, в **статье General,** необходимо настроить:
  
- **FQDN.** Вы введите полностью квалифицированное доменное имя устройства для выживших филиалов или выживаемого сервера филиала.
    
    **Используйте все настроенные IP-адреса,** использующие IP-адреса, настроенные для всех целей на устройстве для выживших ветвей или сервере ветвей.
    
    **Ограничить использование службы выбранными IP-адресами**. Можно настроить определенный адрес для определения сервера и IP-адреса для связи с ТСОП.
    
    **Основной IP-адрес.** IP-адрес определяется и настраивается для всех зада, кроме функций, связанных с ТСОП.
    
    **IP-адрес для ТСОП**. IP-адрес, связанный с функциями ТСОП.
    
- Вы **настраиваете Ассоциации,** чтобы другие роли сервера были настроены и связаны с сохранившимся устройством филиала или выжившим сервером ветвей.
    
    **Сервер архива ассоциированного архива** Выберите из списка сервер архива, который необходимо связать с сохранившимся устройством филиала или сохранившимся сервером филиалов.
    
    Щелкните **Новое,** если вы не создали сервер архива, который вы хотите связать с этим сохранившимся устройством филиала или выжившим сервером филиалов
    
    **Сервер ассоциированного мониторинга** Выберите из списка сервер мониторинга, который необходимо связать с сохранившимся устройством филиала или оставшимся в живых сервером филиалов.
    
    Нажмите **кнопку New,** если вы не создали сервер мониторинга, который вы хотите связать с этим сохранившимся устройством филиала или выжившим сервером филиалов
    
    **Пул Associate Edge (для компонентов мультимедиа)** Выберите из списка пул Edge Server или Edge, который необходимо связать с сохранившимся устройством филиала или сервером ветвей.
    
  При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
  При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
  При нажатии кнопки **Справка** отображается данный экран справки.
