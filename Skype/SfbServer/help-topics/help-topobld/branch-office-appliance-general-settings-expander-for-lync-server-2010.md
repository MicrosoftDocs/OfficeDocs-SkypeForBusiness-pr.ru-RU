# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Расширитель общих настроек устройства для обеспечения связи в филиалах для Lync Server 2010
 
Чтобы изменить свойства для работающего устройства филиала или для работающего филиала, в разделе **Общие**вы можете настроить следующие параметры:
  
- **Полное доменное**имя: вы вводите полное доменное имени для работающего устройства филиала или для бесперебойной ветви сервера.
    
    **Использовать все настроенные IP-адреса** . для всех целей используются IP-адреса, настроенные для работающего устройства филиала или для бесперебойной работы сервера филиала.
    
    **Ограничить использование службы выбранными IP-адресами**. Пользователем задаются различные IP-адреса для сервера и для связи с ТСОП.
    
    **Основной IP-адрес**. IP-адрес определяется и настраивается для всех задач, кроме функций, связанных с ТСОП.
    
    **IP-адрес для ТСОП**. IP-адрес, связанный с функциями ТСОП.
    
- Вы можете настроить **сопоставления** , чтобы убедиться в том, что другие роли сервера настроены и связаны с устройством с бесперебойной подразделением или с бесперебойной подразделением сервером.
    
    **Связывание сервера архивации** Выберите из списка сервер архивации, который вы хотите связать с устройством для бесперебойной филиала или с бесперебойной подразделением сервера.
    
    Нажмите кнопку " **создать** ", если вы не создали сервер архивации, который вы хотите связать с этим бесперебойным устройством филиала или его бесперебойной подразделением.
    
    **Сопоставить сервер мониторинга** Выберите в списке Сервер мониторинга, который вы хотите связать с устройством для бесперебойной работы филиала или бесперебойно работающего сервера подразделения.
    
    Нажмите кнопку **создать** , если вы не создали сервер мониторинга, который вы хотите связать с этим бесперебойным устройством филиала или его бесперебойной подразделением.
    
    **Связывание пула EDGE (для компонентов мультимедиа)** Выберите из списка Граничный или Граничный пул, который вы хотите связать с устройством для работающего филиала или с бесперебойной подразделением.
    
  **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
  **Отмена**. Отмена изменений и закрытие диалогового окна.
  
  **Справка**. Отображение этого экрана справки.
