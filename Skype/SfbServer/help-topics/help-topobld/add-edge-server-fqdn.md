# <a name="add-edge-server-fqdn"></a>Добавление полного доменного имени пограничного сервера
 
Необходимо указать полное доменное имя службы пограничного доступа. Если на странице "Выбор компонентов" не выбран параметр  "Использовать один **&amp; ip-адрес FQDN",** необходимо также указать FQDN для службы веб-конференций и для службы A/V Edge.
  
Кроме того, если выбран параметр "Использовать один **&amp; IP-адрес FQDN",** необходимо указать разные номера портов для каждой из службы (рекомендуемые параметры порта: 444 для службы доступа, 8057 для службы веб-службы и 443 для службы A/V Edge). Если этот параметр не выбран, можно использовать один номер порта (например, 443) для всех трех служб.
