# <a name="add-edge-server-fqdn"></a>Добавление полного доменного имени пограничного сервера
 
Необходимо указать полное доменное имя (FQDN) для службы Edge Access. Если вы не выбрали параметр **использовать один и тот же &amp; доменный IP-адрес** на странице **Выбор компонентов** , необходимо также указать полное доменное имя для службы Edge для веб-конференций и для службы EDGE (+ V).
  
Кроме того, если вы выбрали параметр **использовать один ОДНОдоменный &amp; IP-адрес** , необходимо указать другой номер порта для каждой службы пограничного сервера (рекомендуемые параметры порта: 444 для службы Edge Access, 8057 для службы Edge для веб-конференций и 443 для службы EDGE). Если вы не выбрали этот параметр, вы можете использовать тот же номер порта (например, 443) для всех трех служб.
