# <a name="add-edge-server-fqdn"></a>Добавление полного доменного имени пограничного сервера
 
Необходимо указать полное доменное имя службы пограничного доступа. Если на странице **Выбор компонентов** не был выбран параметр **использовать один полное доменное имя &amp; IP-адреса** , необходимо также указать полное доменное имя для службы веб-конференций и пограничной службы аудио-и видеоконференций.
  
Кроме того, если выбран параметр **использовать один полное доменное имя &amp; IP-адреса** , необходимо указать другой номер порта для каждой пограничной службы (Рекомендуемые параметры порта: 444 для службы пограничного доступа, 8057 для пограничной службы веб-конференций и 443 для пограничной службы аудио-и видеосвязи). Если этот параметр не выбран, можно использовать один номер порта (например, 443) для всех трех служб.
