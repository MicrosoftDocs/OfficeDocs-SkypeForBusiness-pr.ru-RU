# <a name="associate-front-end-with-office-web-apps-server-page"></a>Страница сопоставления сервера переднего плана с сервером Office Web Apps

Сервер Office Web Apps предоставляет службы, аналогичные выступающему, которое отправляет документ для использования на собрании, но включает дополнительные функции, которые не были доступны в предыдущих версиях.

Вы можете выбрать сервер Office Web Apps для развертывания, если вы выбрали Конференц-связь в качестве одного из компонентов настройки Skype для бизнеса Server 2015.

Параметры в этом диалоговом окне предоставляют следующие возможности.

1. **Сопоставить пул с сервером Office Web Apps** . этот флажок следует установить, если вы будете предоставлять и развертывать сервер Office Web Apps для пользователей конференц-связи.

2. Если этот флажок установлен, следует использовать список для выбора уже определенного сервера Office Web Apps.

3. Щелкните пункт **Создать** для определения нового сервера Office Web Apps для развертывания.
