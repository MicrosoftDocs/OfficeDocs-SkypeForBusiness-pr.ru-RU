# <a name="associate-front-end-with-office-web-apps-server-page"></a>Страница связывания сервера переднего плана с сервером Office Web Apps

Сервер Веб-приложений Office предоставляет службы, аналогичные тем, которые выкладывают документ для использования на собрании, но включает дополнительные функции, которые не были представлены в предыдущих версиях.

Если вы выбрали конференцию в качестве одного из параметров конфигурации Skype для бизнеса Server 2015, выберите сервер office Web Apps Server для развертывания.

Параметры в этом диалоговом окне предоставляют следующие возможности.

1. **Связывайте** пул с сервером Office Web Apps Server . Выберите этот контрольный ящик, если вы предоставите и развернете сервер Office Web Apps для пользователей, которые будут работать на конференциях.

2. Если этот флажок установлен, следует использовать список для выбора уже определенного сервера Office Web Apps.

3. Щелкните пункт **Создать** для определения нового сервера Office Web Apps для развертывания.
