# <a name="archiving-server-scope-node"></a>Узел области сервера архивации
 
В Topology Builder можно выполнять задачи, связанные с сервером, работающим с архивативией, щелкнув правой кнопкой мыши пул в дереве консоли и выбрав **edit Properties.** На странице **Изменить** свойства можно изменить полностью квалифицированное доменное имя (FQDN) сервера с архивативией, архива SQL Server базы данных и хранилище архивных файлов.
