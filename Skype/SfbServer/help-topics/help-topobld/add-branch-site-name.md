# <a name="add-branch-site-name"></a>Добавление имени филиала

Чтобы определить новый сайт филиала, укажите следующее:

Имя нового сайта филиала. Указанное имя должно быть уникальным в данном развертывании. Может использоваться любой выбранный вами контекст именования. Обычно рекомендуется использовать название города, местоположение или название филиала. Имя сайта является обязательным параметром.

Поле "Описание" является необязательным и содержит дополнительные сведения, которые не включаются в название инфраструктуры, но могут включать дальнейшее описание вашего сайта.

Дополнительные сведения см. в разделе[Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
