# <a name="add-front-end-associations"></a>Добавление связей переднего плана

Вы можете включить поддержку для отдельных возможностей, требующих развертывания других серверов, сопоставив роли сервера с интерфейсным пулом как прямо сейчас, так и позднее. К ролям сервера, которые можно сопоставить с интерфейсным пулом, относятся следующие:

- Пограничный сервер аудио- и видеоданных. Дополнительные сведения о внедрении пограничного сервера аудио- и видеоданных см. в разделе [Planning for Conferencing](https://technet.microsoft.com/library/983a272a-e1b3-4d70-8f84-836b092fe526.aspx) документации по планированию.

> [!IMPORTANT]
> Если вы сейчас включите поддержку для любой из этих возможностей, публикуемая структура топологии будет содержать серверные компоненты, требуемые для внедрения каждой из выбранных возможностей. Для успешной публикации топологии следует присоединить физические компьютеры к домену. Например, если включить поддержку архивации сейчас, необходимо развернуть сервер архивации и настроить соответствующие параметры архивации перед началом архивации данных в Организации.
