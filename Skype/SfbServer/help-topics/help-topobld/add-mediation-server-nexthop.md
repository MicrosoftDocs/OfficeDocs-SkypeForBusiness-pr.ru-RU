# <a name="add-mediation-server-nexthop"></a>Добавление следующего скачка для серверов-посредников
 
Для определения следующего прыжка для сервера-посредника, который вы определяете, выберите из раскрывающегося списка **очередного прыжка** сервер Enterprise Edition или сервер Standard Edition, который будет выступать в качестве следующего прыжка для сервера-посредника. Для продолжения нажмите кнопку **Далее**.
