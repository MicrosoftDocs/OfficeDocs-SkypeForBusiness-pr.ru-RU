# <a name="add-mediation-server-nexthop"></a>Добавление следующего перехода для серверов-посредников
 
Чтобы определить следующий переход для определяемого сервера-посредника, выберите в выпадаемом списке пула следующего прыжка сервер Enterprise Edition или сервер Standard Edition, который будет выступать в качестве следующего прыжка для сервера-посредника.  Нажмите кнопку **Далее**, чтобы продолжить.
