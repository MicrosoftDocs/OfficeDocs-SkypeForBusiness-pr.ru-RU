# <a name="mediation-server-scope-node"></a>Узел области сервера-посредника
 
В этом узле приведены порты прослушивания и сопоставленные шлюзы ТСОП для этого пула посредников. Можно выполнить следующие действия.
  
- **Изменение свойств**. Открывает страницу **Изменение свойств** для этого сервера, на которой можно включить или выключить TCP-порт прослушивания, изменить номера портов и указать, какие шлюзы в ТСОП сопоставлены с данным пулом посредников.
    
- **Удалить**. Удаляет пул из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
