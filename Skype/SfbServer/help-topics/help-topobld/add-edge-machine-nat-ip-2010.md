# <a name="add-edge-machine-nat-ip-2010"></a>Добавление IP-адреса NAT пограничного компьютера 2010
 
Если общедоступный IP-адрес этого пограничного пула преобразуется, преобразование сетевых адресов (NAT), необходимо указать общедоступный IP-адрес для преобразования. Это должен быть внешний IP-адрес A / V Edge службы.
  
Чтобы изменить ли используется преобразования сетевых адресов, щелкните правой кнопкой мыши имя пула, пограничного сервера и выберите команду **Изменить свойства**.
