# <a name="add-edge-machine-nat-ip-2010"></a>Добавление IP-адреса NAT пограничного компьютера 2010
 
Если общедоступный IP-адрес этого пограничного пула преобразуется с помощью преобразования сетевых адресов (NAT), необходимо указать общедоступный IP-адрес, который нужно перевести. Это должен быть внешний IP-адрес службы Edge/V.
  
Чтобы указать, что в пуле Edge использует преобразование NAT, щелкните правой кнопкой мыши имя пула EDGE и выберите команду **изменить свойства**.
