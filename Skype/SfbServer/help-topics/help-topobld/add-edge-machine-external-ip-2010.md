# <a name="add-edge-machine-external-ip-2010"></a>Добавление внешнего IP-адреса пограничного компьютера 2010

Свойства на этой странице можно настроить таким образом, чтобы пограничный сервер указывал на внешнее расположение службы пограничного доступа, службы Edge для веб-конференций и службы Edge.

- В разделе **Внешние IPv4-адреса**введите внешние IPv4-адреса пограничного сервера в **службе Edge Access**, а также в службе Edge **для** **веб-конференций**.

- Нажмите кнопку **ОК** , чтобы сохранить изменения и внести их в документ Topology.

- Нажмите кнопку **Отмена** , чтобы отменить изменения и закрыть окно **Добавление сервера в мастер создания пула Edge** .

- Нажмите кнопку **Справка** , чтобы прочитать этот раздел справки.

Дополнительные сведения о добавлении пограничных серверов в существующую топологию можно найти в разделе [Определение топологии пограничного](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)сервера. Сведения о переносе пограничных серверов приведены [в статье этап 5: Добавление Lync server 2013 Preview Edge Server для пилотного пула](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
