# <a name="define-computers-in-persistent-chat-pool"></a>Определение компьютеров в пуле сохраняемого чата
 
Чтобы определить компьютеры, которые будут входить в состав пула, введите полное доменное имя (FQDN) в **доменном имени компьютера**. Оно должно совпадать с именем компьютера, указанного в записях узла службы доменных имен (A или AAAA).
  
Для добавления полного доменного имени к списку компьютеров в пуле нажмите кнопку **Добавить**.
  
Для обновления записей в списке компьютеров, определенных для данного пула, нажмите кнопку **Обновить**.
  
Для удаления указанного компьютера из пула нажмите кнопку **Удалить**.
  
Для возврата к предыдущему диалоговому окну определения пула нажмите кнопку **Назад**.
  
После того как вы закончите ввод членов компьютера для этого пула, нажмите кнопку **Далее** , чтобы перейти к определению пула сервера сохраняемого чата.
  
Для отмены всех изменений нажмите кнопку **Отмена**; при этом работа с мастером **Определение нового пула сохраняемого чата** завершается.
  
Для доступа к контекстно-зависимой справке, аналогичной данной странице, нажмите кнопку **Справка**.
