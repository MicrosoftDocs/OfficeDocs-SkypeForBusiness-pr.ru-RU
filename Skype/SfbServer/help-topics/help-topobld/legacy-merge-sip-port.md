# <a name="legacy-merge-sip-port"></a>Порт SIP для объединения с устаревшей версией
 
По умолчанию для связи с клиентами, устаревшими интерфейсными пулами и серверами используется TCP-порт SIP 5061. Если значение по умолчанию не использовалось, выберите **Порт SIP для всех пулов и серверов:** и введите новое значение.
  
Если устаревшее развертывание настроено для автоматического входа клиентов, выберите **Автоматическое обнаружение конфигурации портов SIP для каждого пула** и нажмите кнопку **Далее** , чтобы продолжить.
