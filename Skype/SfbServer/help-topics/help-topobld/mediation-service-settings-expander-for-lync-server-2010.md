# <a name="mediation-service-settings-expander-for-lync-server-2010"></a>Расширитель настроек службы сервера-посредника для Lync Server 2010
 
Свойства службы посредника можно изменить, определив следующие свойства
  
- **Порты прослушивания**. Задайте порт **TLS** для ожидания передачи данных службой-посредником. По умолчанию для протокола TLS применяется порт TCP 5067.
    
    При необходимости можно задать значение порта **TCP**. Значение по умолчанию: TCP 5068.
    
    > [!NOTE]
    > Для задания значения порта TCP необходимо выбрать режим **Включить порт TCP**. Требования к параметрам порта для связи со службой-посредником см. в документации по шлюзу ТСОП или IP‑УАТС. 
  
- Выбор режима **Включить порт TCP** позволяет задать значение порта для передачи данных по протоколу TCP из шлюза ТСОП или IP-УАТС.
    
- Список связанных и существующих значений параметров **Линия связи** (т. е. магистрали для протокола SIP), **Шлюз** (шлюз ТСОП или IP-УАТС) и **Сайт** (сайт, настроенный для магистрали и шлюза).
    
- Выберите магистраль, шлюз и сайт, затем щелкните **По умолчанию** для применения выбранных значения по умолчанию данной службой-посредником. Для отмены применения значения по умолчанию выберите его и щелкните **Отменить по умолчанию**. После этого выберите новое значение для применения по умолчанию и щелкните **По умолчанию**.
    
  **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
  **Отмена**. Отмена изменений и закрытие диалогового окна.
  
  **Справка**. Отображение этого экрана справки.
