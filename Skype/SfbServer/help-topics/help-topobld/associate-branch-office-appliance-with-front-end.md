# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства для обеспечения связи в филиалах с сервером переднего плана
 
Чтобы связать бытовую или живую ветвь с пулом переднего выпуск Standard или сервером, на странице Выбор переднего конца пула выберите пул переднего выпуск Standard или сервер из перепада переднего списка пула.  Нажмите кнопку **Далее**, чтобы продолжить.
