# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства для обеспечения связи в филиалах с сервером переднего плана
 
Чтобы связать уцелевшие устройства филиала или сервера ветвей с пулом переднего выпуск Standard или сервером, на странице Выбор переднего конца пула выберите пул переднего конца или сервер выпуск Standard из отсеянного списка пула переднего конца.   Нажмите кнопку **Далее**, чтобы продолжить.
