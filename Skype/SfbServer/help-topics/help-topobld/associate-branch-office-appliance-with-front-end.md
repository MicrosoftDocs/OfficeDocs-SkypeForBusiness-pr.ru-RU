# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства филиала с сервером переднего плана
 
Чтобы связать устройство или сервер для обеспечения связи в филиалах с пулом переднего плана или сервером Standard Edition, на странице **Выбор пула переднего плана** выберите пул переднего плана или сервер Standard Edition из раскрывающегося списка **пул переднего плана** . Нажмите кнопку **Далее**, чтобы продолжить.
