# <a name="archiving-servers-scope-node"></a>Узел области серверов архивации
 
В Topology Builder можно выполнять задачи, связанные с отдельным сервером архива, щелкнув правой кнопкой  мыши сервер архива в дереве консоли, щелкнув действие на панели инструментов или нажав задачу в области Действия, а затем щелкнув следующий параметр Архивирование Сервера. 
  
- **Новый сервер архивации**. Служит для запуска мастера **Определение нового сервера архивации**, в котором можно указать параметры нового сервера архивации, включая полное доменное имя, базу данных SQL Server сервера архивации, хранилище файлов архивации и сопоставление с пулом.
    
> [!NOTE]
> Узел серверов архивации содержит сервер архивации, если ранее сервер архивации был добавлен в топологию с помощью построителя топологий. Однако архивировать мгновенные сообщения или веб-конференцию нельзя до тех пор, пока вы не настроили сервер архива, который включает установку Skype для бизнеса Server на сервере, который вы хотите использовать в качестве сервера архива и настройку поддержки архива. 
  
Помимо приведенных выше в этом разделе параметров сервера архивации имеются общие параметры узла "Серверы архивации", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Топология**. Используйте этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Данный параметр можно использовать для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Данный параметр можно использовать для доступа к справке по активному узлу.
