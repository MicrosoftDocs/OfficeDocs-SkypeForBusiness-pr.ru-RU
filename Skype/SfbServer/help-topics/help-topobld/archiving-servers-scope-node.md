# <a name="archiving-servers-scope-node"></a>Узел области серверов архивации
 
В построителе топологий можно выполнять задачи, связанные с отдельным сервером архивации, щелкнув правой кнопкой мыши **сервер архивации** в дереве консоли, щелкнув пункт **действие** на панели инструментов или щелкнув задачу в области действия, а затем выбрав следующий параметр Архивация сервера.
  
- **Новый сервер архивации**. Служит для запуска мастера **Определение нового сервера архивации**, в котором можно указать параметры нового сервера архивации, включая полное доменное имя, базу данных SQL Server сервера архивации, хранилище файлов архивации и сопоставление с пулом.
    
> [!NOTE]
> Узел серверов архивации содержит сервер архивации, если ранее сервер архивации был добавлен в топологию с помощью построителя топологий. Однако невозможно архивировать мгновенные сообщения и веб-конференции (обмен сообщениями), пока не будет настроен сервер архивации, включающий установку Skype для бизнеса Server на сервере, который будет использоваться в качестве сервера архивации, и настройки поддержки архивации. 
  
Помимо приведенных выше в этом разделе параметров сервера архивации имеются общие параметры узла "Серверы архивации", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Topology (топология**). Используйте этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Данный параметр можно использовать для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Данный параметр можно использовать для доступа к справке по активному узлу.
