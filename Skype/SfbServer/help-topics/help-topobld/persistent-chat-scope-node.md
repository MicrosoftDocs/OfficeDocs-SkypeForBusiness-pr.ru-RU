# <a name="persistent-chat-scope-node"></a>Узел области сохраняемого чата
 
Чтобы начать создание нового сервера сохраняемого чата или пул серверов сохраняемого чата, в построителе топологии щелкните правой кнопкой мыши на **Persistent Chat пулы** , а затем выберите **Нового пула сохраняемого чата**
  
- **Топология**. Этот параметр можно использовать для определения новой топологии, откройте существующую топологию, загрузите текущей топологии, сохранить копию топологии, опубликовать эту топологию, установка базы данных, объединение компонентов с предыдущими версиями Office Communications Server или удаление развертывание.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
