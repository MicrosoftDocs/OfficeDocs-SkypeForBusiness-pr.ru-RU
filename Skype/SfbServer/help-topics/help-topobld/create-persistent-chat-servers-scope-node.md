# <a name="create-persistent-chat-servers-scope-node"></a>Создание узла области применения серверов сохраняемого чата
 
Щелкните правой кнопкой мыши **пул сохраняемого чата** в построителе топологий, чтобы выполнить следующие задачи:
  
- **Новый пул сохраняемого чата**: Запуск мастера **определения нового пула сохраняемого чата** , используемого для определения нового сервера или пула сохраняемого чата.
    
Помимо определения нового сервера пула сохраняемого чата можно выполнять следующие общие задачи:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий Lync Server или удалить развертывание.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
