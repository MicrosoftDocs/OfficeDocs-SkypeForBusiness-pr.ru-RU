# <a name="create-persistent-chat-servers-scope-node"></a>Создание узла области применения серверов сохраняемого чата
 
Вы можете щелкнуть правой кнопкой мыши **группу постоянного чата** в построителе топологии, чтобы выполнить указанные ниже действия.
  
- **Новый пул сохраняемого чата**: Запуск мастера **определения нового пула** для сохраняемого чата, который используется для определения нового сервера или пула сохраняемого чата.
    
Помимо определения нового сервера пула сохраняемого чата можно выполнять следующие общие задачи:
  
- **Топология**. Эта команда позволяет задать новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, выполнить объединение с компонентами прежних версий Lync Server или удалить развертывание.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
