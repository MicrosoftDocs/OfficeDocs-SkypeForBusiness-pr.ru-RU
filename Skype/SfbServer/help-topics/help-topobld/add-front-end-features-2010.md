# <a name="add-front-end-features-2010"></a>Добавление компонентов переднего плана 2010

Выберите каждый компонент, который вы хотите включить для пула переднего плана. Как минимум, вы должны развертывать мгновенные сообщения (IM), которые всегда включены и не могут быть выбраны. Кроме того, у вас есть возможность развертывания других функций, таких как конференции, голосовые звонки и управление допуском звонков (CAC). Некоторые возможности зависят от вашей топологии.

Дополнительные сведения об указании функций для пула Standard Edition Server или Enterprise Edition можно найти в разделе [Определение и Настройка внешнего пула и сервера Standard Edition](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) в документации по развертыванию.
