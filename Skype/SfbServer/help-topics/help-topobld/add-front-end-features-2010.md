# <a name="add-front-end-features-2010"></a>Добавление компонентов переднего плана 2010

Выберите все возможности, которые хотите включить для интерфейсного пула. Следует развернуть по крайней мере систему обмена мгновенными сообщениями, так как эта возможность всегда выбрана и не может быть отключена. У вас также есть возможность развертывания других функций, таких как conferencing, Корпоративная голосовая связь и управление приемом вызовов (CAC). Некоторые компоненты являются зависимыми от топологии.

Подробные сведения о том, как указать функции для [](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server) выпуск Standard или выпуск Enterprise пула, см. в документации по развертыванию и настройке переднего выпуск Standard сервера.
