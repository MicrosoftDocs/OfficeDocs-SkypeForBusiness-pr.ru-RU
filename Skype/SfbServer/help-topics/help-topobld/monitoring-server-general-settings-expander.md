# <a name="monitoring-server-general-settings-expander"></a>Расширитель общих параметров сервера мониторинга
 
В построителе топологий можно изменить свойства отдельного сервера мониторинга, щелкнув правой кнопкой мыши сервер мониторинга в дереве консоли, а затем выбрав пункт **действие** на панели инструментов или щелкнув задачу в панели действий, а затем выбрав команду **изменить свойства**и изменив любые из следующих параметров:
  
- **Полное доменное имя**, чтобы изменить полное доменное имя сервера, который требуется развернуть в качестве сервера мониторинга.
    
- **Хранилище SQL**, чтобы изменить экземпляр SQL Server, который планируется использовать для мониторинга базы данных SQL Server. Если вы изменяете базу данных SQL Server сервера мониторинга, следует перезапустить сервер мониторинга, чтобы изменения вступили в силу.
