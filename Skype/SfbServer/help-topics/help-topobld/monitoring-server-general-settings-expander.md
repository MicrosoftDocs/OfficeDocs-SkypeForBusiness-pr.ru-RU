# <a name="monitoring-server-general-settings-expander"></a>Расширитель общих параметров сервера мониторинга
 
Для изменения свойств отдельного сервера мониторинга в построителе топологий можно щелкнуть правой кнопкой мыши сервер мониторинга в дереве консоли и выбрать **Действие** на панели инструментов или щелкнуть задачу на панели действий, затем щелкнуть **Изменение свойств** и изменить любые из следующих параметров.
  
- **Полное доменное имя**. Полное доменное имя сервера, который требуется развернуть в качестве сервера мониторинга.
    
- **Хранилище SQL**. Экземпляр SQL Server для размещения базы данных мониторинга SQL Server. Изменение базы данных SQL Server вступает в силу только после перезагрузки сервера мониторинга.
