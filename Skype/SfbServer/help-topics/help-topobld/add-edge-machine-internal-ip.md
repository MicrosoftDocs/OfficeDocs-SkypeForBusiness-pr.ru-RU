# <a name="add-edge-machine-internal-ip"></a>Добавление внутреннего IP-адреса пограничного сервера
 
Используйте эту страницу для определения внутреннего IP-адреса и внутреннего полное доменное имя (FQDN) для пограничного сервера.
  
Полное доменное имя, которое указывается должен быть идентичен имя компьютера, настроенного на сервере. По умолчанию имя компьютера, который не присоединен к домену, является коротким, а не полным доменным именем. Построитель топологии использует полные доменные имена, а не короткие имена. Таким образом необходимо настроить суффикс доменных имен (DNS) на имя компьютера, развертываемом в качестве пограничного сервера, не присоединенного к домену. Сведения о добавлении DNS-суффикса к имени компьютера в разделе [Настройка DNS для поддержки пограничной топологии](http://technet.microsoft.com/library/955493e6-aa29-424d-bb81-1ef87b3b15e3.aspx)
