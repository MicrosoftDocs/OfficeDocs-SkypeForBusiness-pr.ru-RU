# <a name="associate-av-mcu-with-front-end"></a>Связывание AV MCU с сервером переднего плана
 
Все пулы переднего плана Enterprise Edition с поддержкой конференц-связи должны иметь A аудио- и видеоконференций, совмещенного или необходимо использовать изолированный A / пула аудио-и видеоконференциями. Любые существующие переднего плана Enterprise Edition пулы, имеющим в настоящий момент A аудио- и видеоконференций службы с совместным размещением может быть выбран для использования этой новой автономной A аудио- и видеоконференций, вместо этого пула.
