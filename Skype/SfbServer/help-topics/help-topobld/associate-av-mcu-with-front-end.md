# <a name="associate-av-mcu-with-front-end"></a>Связывание AV MCU с сервером переднего плана
 
Для всех пулов переднего плана Enterprise Edition с включенной Конференцией должна быть размещена служба конференц-связи с телефонным подключением или один из отдельных пулов конференц-связи A/V. Вместо этого вы можете выбрать один из существующих пулов переднего плана Enterprise Edition, для которого в настоящее время включена служба конференц-связи A/V.
