# <a name="test---skype-for-business-server-site-scope-node"></a>Тестирование узла области сайта Skype для бизнеса Server
 
Этот узел отображает один из сайтов, определенных в топологии. Можно выполнить следующие действия. 
  
- **Создать.** Добавление нового пула или сервера к сайту.
    
- **Изменение свойств.** Изменение свойств сайта, таких как название и местоположение, а также указание, используется ли на сайте контроль допуска вызовов для управления пропускной способностью, а также используется ли пул директоров или пограничный пул в этом сайте как маршрут федерации.
    
- **Удалить.** Удаление данного сайта из топологии.
    
Кроме работы с этими параметрами, можно выполнять следующие задачи общего характера:
  
- **Топология.** С помощью этого параметра можно определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Можно использовать данный параметр для доступа к справке по активному узлу.
