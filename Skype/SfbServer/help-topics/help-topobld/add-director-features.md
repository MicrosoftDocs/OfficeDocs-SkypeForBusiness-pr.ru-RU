# <a name="add-director-features"></a>Добавление компонентов директора

Если требуется пул директоров, чтобы включить сервер мониторинга выберите **мониторинг (регистрация Вызовов и показатели качества взаимодействия)**.

Для получения дополнительных сведений о добавлении пул директоров видеть [сценарии для директора](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) и [Setting Up the Director](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx).
