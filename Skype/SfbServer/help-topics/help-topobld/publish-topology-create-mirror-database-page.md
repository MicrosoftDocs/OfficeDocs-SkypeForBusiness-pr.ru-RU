# <a name="publish-topology-create-mirror-database-page"></a>Страница публикации топологии для создания зеркальной базы данных
 
Это предварительная документация, которая подлежит изменениям. Пустые разделы включены в качестве заполнителей.
  
Топология Builder определяет все параметры конфигурации в определенном документе топологии. Если вы настроили зеркальные SQL Server базы данных, вам будет представлен экран, отображаемый в зеркальном хранилище, основном хранилище, пути обмена файлами и пути баз данных. Чтобы создать зеркальные базы данных и путь к файловому ресурсу общего доступа, установите флажок, соответствующий хранилищу зеркальных баз данных. Если вы не хотите создавать и развертывать базу данных в определенный SQL Server, вы очищаете SQL Server.
  
 Если вы решили в данный момент не устанавливать одну или все базы данных, снимите флажки для тех баз данных, которые устанавливать не нужно. В более позднее время вы решите использовать **базу данных установки,** щелкнув правой кнопкой мыши по большинству элементов узла в Topology Builder.
