# <a name="associate-front-end-with-archiving"></a>Связывание сервера переднего плана с архивированием
 
У каждого пула переднего плана и бесперебойно работающего устройства филиала может быть только один сервер архивации, связанный с ним. Если для сайта включена архивация, сервер архивации собирает и записывает содержимое сообщений и веб-конференций в беседах, в которых участвуют пользователи, подключенные к связанным пулам и бесперебойным устройствам филиалов. 
  
Все пулы на сайте и пулы нескольких центральных сайтов могут использовать один и тот же сервер архивации, если использование не превышает емкость сервера архивации.
