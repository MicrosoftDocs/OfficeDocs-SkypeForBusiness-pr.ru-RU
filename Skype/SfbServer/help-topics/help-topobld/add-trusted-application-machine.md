# <a name="add-trusted-application-machine"></a>Добавление доверенного компьютера приложений
 
Укажите полное доменное имя (FQDN) каждого компьютера, который вы хотите добавить в качестве доверенного сервера приложений в этом пуле. До публикации топологии можно в любой момент обновить полное доменное имя добавленного к списку компьютера или удалить его из пула. После публикации топологии для изменения полного доменного ИМЕНИ необходимо удаление сервера в построителе топологий, а затем добавляя новый сервер к пулу с новой полным доменным ИМЕНЕМ.
