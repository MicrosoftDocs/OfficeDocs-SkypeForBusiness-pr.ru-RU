# <a name="av-mcu-settings-expander"></a>Расширитель параметров AV MCU
 
Чтобы изменить свойства уже определенного пула серверов конференц-связи, выполните указанные ниже действия.
  
## <a name="general-settings"></a>Общие настройки

Чтобы обновить полное доменное имя (FQDN) для пула серверов конференц-связи A/V, введите имя пула, которое определено в системе доменных имен (DNS). У вас должна быть запись DNS Host (A), соответствующая имени пула серверов конференц-связи A/V.
  
## <a name="av-conferencing-service"></a>Служба конференц-связи/V

Чтобы связать пул серверов для конференц-связи A/V, выберите пул или пулы, которые нужно связать с пулом серверов для конференц-связи "A/V".
  
> [!NOTE]
> Если ни один из пулов переднего плана не отображается в списке **выбрать интерфейсных пулов для связи с этой** областью, вы либо Выбери сервер конференц-связи с интерфейсом "/v" с пулом переднего плана, либо не выбрали команду "Добавить конференцию" в разделе " **функции и функциональность** " в свойствах пула переднего плана.
  


По завершении нажмите кнопку **ОК** , чтобы завершить редактирование параметров пула серверов конференц-связи A/V. Для отмены изменений нажмите кнопку **Отмена**.
