# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Страница добавления хранилища SQL пограничного сервера для Lync Server 2010
 
Вы определяете свойства SQL Server для нового пула переднего конца, который вы настраивали.
  
> [!IMPORTANT]
> Если вы настраиваете сервер выпуск Standard, этот диалоговое окно будет отображаться, но вы не сможете настроить параметры. Нажмите кнопку **Далее**, чтобы продолжить.
  
Параметры, доступные для настройки.
  
- **Используйте ранее определенный SQL Server:** Выберите существующий SQL Server из списка для магазина пула переднего конца.
    
- Выберите **Определение нового SQL Server для** создания нового SQL Server магазина.
    
    Вы введите полностью квалифицированное доменное имя SQL Server в **SQL Server FQDN**
    
    Изначально выбирается **экземпляр по умолчанию**. Если необходимо определить экземпляр, выберите **Именованный экземпляр** и введите имя экземпляра, который следует создать.
    
  **Назад**: возврат в предыдущее диалоговое окно
  
  **Далее**: фиксация изменений и переход к диалоговому окну следующего действия
  
  **Отмена**: отмена изменений и очистка конфигурации
  
  **Справка**: отображение справочной информации, такой как данный раздел
