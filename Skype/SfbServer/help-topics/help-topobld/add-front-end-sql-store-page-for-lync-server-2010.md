# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Добавление страницы хранения данных SQL пограничного сервера для Lync Server 2010
 
Определите свойства хранилищ SQL Server для нового пула переднего плана, настройка которых осуществляется.
  
> [!IMPORTANT]
> При настройке сервера Standard Edition в этом диалоговом окне, но вы не сможете для настройки параметров. Для продолжения нажмите кнопку **Далее**.
  
Параметры, которые можно настроить:
  
- **Использовать ранее определенный хранилище SQL Server**: выберите хранилище SQL Server в списке для хранилища пула переднего плана.
    
- Выберите **хранения определить новый SQL Server** для создания нового хранилища SQL Server.
    
    Введите полное доменное имя сервера SQL Server в **Поле полное имя SQL Server**
    
    **Экземпляр по умолчанию** выбран, изначально. Если вы предпочитаете определение экземпляра, выберите **именованный экземпляр** и введите имя экземпляра, который необходимо создать.
    
  **Назад**: возврат в предыдущее диалоговое окно
  
  **Далее**: Фиксация изменений, а затем к диалоговому окну следующего действия
  
  **Отмена**: Отмена изменений и очистка конфигурации
  
  **Справка**: Отображение справочной информации, такой как данный раздел
