# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Добавление страницы хранения данных SQL пограничного сервера для Lync Server 2010
 
Вы определяете свойства хранилищ SQL Server для нового пула переднего плана, который вы настраиваете.
  
> [!IMPORTANT]
> Если вы настраиваете сервер Standard Edition, это диалоговое окно будет отображаться, но вы не сможете настраивать параметры. Для продолжения нажмите кнопку **Далее**.
  
Параметры, которые можно настроить:
  
- **Использование ранее определенного хранилища SQL Server**: выберите СУЩЕСТВУЮЩЕЕ хранилище SQL Server из списка в качестве внешнего хранилища пула.
    
- Выберите команду **Определение нового хранилища SQL Server** , чтобы создать новое хранилище SQL Server.
    
    Введите полное доменное имя SQL Server в доменном имени **SQL** Server.
    
    Изначально выбирается **экземпляр по умолчанию** . Если вы предпочитаете определить экземпляр, выберите **именованный экземпляр** и введите имя экземпляра, который нужно создать.
    
  **Назад**: переход к предыдущему диалоговому окну
  
  **Далее**: фиксирует изменения и переходит в окно следующего шага.
  
  **Отмена**: удаляет изменения и отменяет конфигурацию.
  
  **Справка**: Отображает справочные сведения, такие как в этом разделе.
