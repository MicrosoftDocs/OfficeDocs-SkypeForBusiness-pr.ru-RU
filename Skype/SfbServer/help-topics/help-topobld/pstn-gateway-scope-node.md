# <a name="pstn-gateway-scope-node"></a>Узел области шлюза ТСОП
 
В этом узле отображается полное доменное имя, связанный сервер-посредник и другие базовые сведения, относящиеся к этому шлюзу ТСОП. Можно выполнить следующие действия.
  
- **Изменение свойств**. Открывает страницу **Изменение свойств**, где можно изменить полное доменное имя (если топология еще не опубликована), порт прослушивания, транспортный протокол и альтернативный IP-адрес мультимедиа.
    
- **Удалить**. Удаляет сервер из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Топология.** С помощью этого параметра можно определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
