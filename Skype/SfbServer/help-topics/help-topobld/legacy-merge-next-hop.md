# <a name="legacy-merge-next-hop"></a>Следующий переход для объединения с устаревшей версией
 
**Полное доменное имя следующего прыжка** определяет сервер следующего прыжка, на который старый пограничный сервер маршрутизирует внутренний трафик. Введите полное доменное имя (FQDN) сервера следующего прыжка, на который пограничный сервер маршрутизирует внутренний трафик, а затем укажите, является ли он пулом или начальником переднего **** **плана** .
