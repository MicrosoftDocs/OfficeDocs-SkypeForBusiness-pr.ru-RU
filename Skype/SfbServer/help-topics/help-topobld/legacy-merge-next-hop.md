# <a name="legacy-merge-next-hop"></a>Следующий прыжок объединения с устаревшей версией
 
**Полное доменное имя следующего прыжка** идентифицирует сервера следующего прыжка, к которому устаревшего пограничного сервера Маршрутизация внутреннего трафика. Введите полное доменное имя (FQDN) сервера следующего прыжка, к которому этот пограничный сервер Маршрутизация внутреннего трафика и выберите **пул переднего плана** или **директора**.
