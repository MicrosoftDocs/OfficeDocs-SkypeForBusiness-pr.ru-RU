# <a name="persistent-chat-machines-expander"></a>Расширитель настроек компьютеров сохраняемого чата
 
Вы активируете или деактивируете развернутый сервер сохраняемого чата или пул серверов сохраняемого чата с помощью серверов или пулов, перечисленных в разделе **Параметры состояния компьютера**. Выберите сервер или пул в списке и нажмите кнопку **сделать активным** , чтобы установить активный сервер или пул.
  
Чтобы настроить развернутый сервер сохраняемого чата или пул серверов сохраняемого чата на неактивном состоянии, выберите сервер или пул в списке и щелкните **сделать неактивным**. По крайней мере один сервер должен иметь значение Active.
  
 При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
 При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
 При нажатии кнопки **Справка** отображается данный экран справки.
