# <a name="persistent-chat-machines-expander"></a>Расширитель параметров компьютеров сохраняемого чата
 
Вы активируете или отключаете развернутый сервер сохраняемой беседы или пул серверов сохраняемой беседы с помощью серверов или пулов, перечисленных в параметрах состояния **компьютера.** Выберите сервер или пул в списке и нажмите кнопку "Сделать **активным",** чтобы сделать сервер или пул активным.
  
Чтобы сделать развернутый сервер сохраняемой беседы или пул серверов сохраняемой беседы неактивным, выберите сервер или пул в списке и нажмите кнопку **"Сделать неактивным".** По крайней мере один сервер должен быть активен.
  
 При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
 При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
 При нажатии кнопки **Справка** отображается данный экран справки.
