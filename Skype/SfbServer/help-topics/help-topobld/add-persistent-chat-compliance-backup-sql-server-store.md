# <a name="add-persistent-chat-compliance-backup-sql-server-store"></a>Добавление резервного хранилища SQL Server соответствия сохраняемого чата
 
Вы настраиваете хранилища SQL Server для обеспечения соответствия резервных копий, которые предоставляют резервные базы данных для сервера сохраняемого чата или сохраняемого сервера в соответствии с требованиями SQL Server Stores.
  
 **Хранилище SQL Server**: выберите существующий SQL Server и, при необходимости, экземпляр сохраняемого чата.
  
Нажмите кнопку **создать** , чтобы определить новый SQL Server и (необязательно) новый экземпляр для данных о соответствии требованиям резервного копирования для сохраняемого чата.
  
Установите флажок **включить зеркальное отображение хранилища SQL Server** , чтобы настроить базу данных SQL Server и дополнительный экземпляр, который предоставит зеркальную базу данных соответствия требованиям резервного копирования для сохраняемого чата.
  
Выберите из списка **зеркальный набор в SQL Server сохраните** сервер SQL Server и Необязательный экземпляр, чтобы он ДЕЙСТВОВАЛ для SQL Server для соответствия требованиям резервного копирования для сохраняемого чата.
  
Нажмите кнопку **создать** , чтобы определить новый SQL Server и создать новый экземпляр для сохраняемого зеркального отображения SQL Server.
  
В списке **Использовать свидетель зеркалирования SQL Server для автоматического перехода на другой ресурс** выберите сервер SQL Server, который будет служить следящим сервером в ситуациях отработки отказа. Сервер-свидетель не может зеркально отражать данные для серверов чатов, но гарантирует, что только один SQL Server в зеркальной конфигурации является активным сервером SQL Server в любой момент.
  
Нажмите кнопку **создать** , чтобы определить новый следящий сервер SQL Server, который является экземпляром для функции зеркального резервного копирования SQL Server с подведением для сохраняемого чата.
  
Для возврата к предыдущему диалоговому окну определения пула нажмите кнопку **Назад**.
  
После того как вы закончите ввод параметров для резервного копирования в хранилище SQL Server для этого пула, нажмите кнопку **Далее** , чтобы перейти к определению пула сервера сохраняемого чата.
  
Для отмены всех изменений нажмите кнопку **Отмена**; при этом работа с мастером **Определение нового пула сохраняемого чата** завершается.
  
Для доступа к контекстно-зависимой справке, аналогичной данной странице, нажмите кнопку **Справка**.
