# <a name="add-mediation-server"></a>Добавление сервера-посредника
 
Чтобы определить сервер-посредник, укажите полное доменное имя (FQDN) для пула серверов-исправлений. Если вы создаете пул серверов-посредников, укажите имя пула для серверов-исправлений, которые вы разворачиваете. Полные доменные имена отдельных серверов задаются на другом шаге. Если вы развертываете один сервер-посредник, указанное полное доменное имя будет полным доменным именем сервера единого исправления. Подробности можно найти [в разделе Развертывание сервера-посредника в построителе топологии в Skype для бизнеса Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
