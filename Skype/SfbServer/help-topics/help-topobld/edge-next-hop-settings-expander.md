# <a name="edge-next-hop-settings-expander"></a>Расширитель параметров следующего пограничного перехода

Чтобы задать или изменить **пул следующего прыжка** , который будет использоваться пограничными серверами для взаимодействия с внутренней инфраструктурой, выберите в списке директор, пул директоров, сервер переднего плана, сервер переднего плана или пул серверов переднего плана. В списке отображаются только директора или серверы переднего плана, настроенные в построителе топологий.

Дополнительные сведения по определению и настройке параметров для пограничных служб см. в разделе [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
