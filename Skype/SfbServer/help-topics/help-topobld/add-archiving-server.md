# <a name="add-archiving-server"></a>Добавление сервера архивации
 
Чтобы развернуть сервер архивации в топологии, необходимо указать полное доменное имя сервера, на котором будет развернут компонент архивации. Перед публикацией топологии, которая включает архивную и Skype для бизнеса Server, необходимо завершить все необходимые условия для развертывания архива.
