# <a name="add-front-end-associations-2010"></a>Добавление связей переднего плана 2010

Можно включить поддержку определенных функций, которые требуют развертывания других серверов, связав роли сервера с пулом переднего плана в настоящее время. Роли сервера также можно связать с пулом переднего плана позже. Роли сервера, которые могут быть связаны с пулом переднего плана, включают следующие:

1. Сервер архива. Дополнительные сведения о планировании и внедрении поддержки архивации см. в разделе [Defining Your Organization's Requirements for Archiving](https://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx) документации по планированию, а также в разделах [Define and Configure a Front End Pool or Standard Edition Server](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) и [Deploying Archiving](https://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx) документации по развертыванию.

2. Сервер мониторинга. Дополнительные сведения о внедрении поддержки мониторинга см. в разделе [Planning for Monitoring](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx) документации по планированию, а также в разделах [Define and Configure a Front End Pool or Standard Edition Server](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) и [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) документации по развертыванию.

3. Поехавная служба A/V. Дополнительные сведения о реализации службы A/V Edge см. в обзоре доступа [внешних пользователей.](https://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx)
