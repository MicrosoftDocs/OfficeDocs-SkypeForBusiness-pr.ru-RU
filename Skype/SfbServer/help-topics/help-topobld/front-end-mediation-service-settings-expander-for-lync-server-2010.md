# <a name="front-end-mediation-service-settings-expander-for-lync-server-2010"></a>Расширитель параметров службы посредника переднего плана для Lync Server 2010
 
Изменение свойств параметров **шлюза ТСОП Mediation Server** в этом диалоговом окне. Определите следующие параметры:
  
- Выберите **включено, совмещенного сервера-посредника** , если необходимо совмещать на промежуточный сервер с этой пулов переднего плана или сервера переднего плана.
    
- **Порты прослушивания**: определение портов прослушивания сервера-посредника на. Вы можете определить порт для **TLS** или транспортного уровня безопасности или **TCP**или транспорта протокола управления. Для запись порта TCP, чтобы оно было доступно необходимо установите флажок для **включения TCP-порт**. 
    
    > [!IMPORTANT]
    > Обратитесь к документации и параметров конфигурации для шлюза телефонной сети (общего пользования PSTN) для определения, если необходимо включить и определите значения портов TLS, TCP или оба. Протокол TLS является более безопасной протокола, с помощью сертификатов для шифрования трафика между сервером-посредником и шлюзом ТСОП. Не все шлюзы ТСОП поддерживает TLS. 
  
- Список связанных и существующих значений параметров **Линия связи** (т. е. магистрали для протокола SIP), **Шлюз** (шлюз ТСОП или IP-УАТС) и **Сайт** (сайт, настроенный для магистрали и шлюза).
    
- Выберите магистраль, шлюз и сайт, затем щелкните **По умолчанию** для применения выбранных значения по умолчанию данной службой-посредником. Для отмены применения значения по умолчанию выберите его и щелкните **Отменить по умолчанию**. После этого выберите новое значение для применения по умолчанию и щелкните **По умолчанию**.
    
 **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
 **Отмена**. Отмена изменений и закрытие диалогового окна.
  
 **Справка**. Отображение этого экрана справки.
