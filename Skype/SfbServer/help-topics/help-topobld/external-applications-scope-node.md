# <a name="external-applications-scope-node"></a>Внешний узел области применения приложений
 
Узел **Серверы доверенных приложений** отображает пулы доверенных приложений в этом сайте. Доверенные пулы приложений содержат серверы, на которых выполняются доверенные приложения единой системы обмена сообщениями Майкрософт, но не работают роли Skype для бизнеса Server. Можно выполнить следующее действие.
  
- **Создать пул доверенных приложений**. Служит для запуска мастера **Определение нового пула доверенных приложений**, который используется для создания в топологии нового пула доверенных приложений.
    
Кроме параметров пула доверенных приложений, можно выполнять следующие задачи общего характера:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий или удалить развертывание.
    
- **Вид**. Вы можете использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
