# <a name="monitoring-scope-node"></a>Узел области мониторинга
 
В построителе топологий Lync Server 2013 можно выполнять задачи, связанные с отдельным сервером мониторинга, щелкнув правой кнопкой мыши сервер мониторинга в дереве консоли, щелкнув пункт **действие** на панели инструментов или щелкнув задачу в области действия, а затем выбрав один из следующих параметров для сервера мониторинга:
  
- **Изменить свойства**. Открывает страницу **изменение свойств** для сервера мониторинга, где можно изменить полное доменное имя сервера, на котором выполняется мониторинг, или базы данных SQL Server мониторинга.
    
- **удаление**. Удаляет сервер, на котором выполняется мониторинг, из топологии и базы данных SQL Server мониторинга.
    
Помимо приведенных выше в этом разделе параметров сервера мониторинга имеются общие параметры узла "Мониторинг", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
