# <a name="monitoring-scope-node"></a>Узел области мониторинга
 
В построителе топологий Lync Server 2013 можно выполнить задачи, связанные с отдельного сервера под управлением мониторинг либо щелкните правой кнопкой мыши сервер под управлением мониторинг в дереве консоли, нажав кнопку **действия** на панели инструментов или нажав кнопку задачи в указанных действий область и затем выбрав один из следующих параметров для сервера под управлением мониторинг:
  
- **Изменение свойств**. Открывает страницу **Изменить свойства** сервера мониторинга, где можно изменить полное доменное имя (FQDN) сервера, выполняющего отслеживание или базу данных мониторинга SQL Server.
    
- **Удалить**. Удаление сервера под управлением мониторинг из топологии и мониторинга базы данных SQL Server.
    
Помимо приведенных выше в этом разделе параметров сервера мониторинга имеются общие параметры узла "Мониторинг", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Топология**. Позволяет определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Справка**. Обеспечивает доступ к справке по активному узлу.
