# <a name="monitoring-scope-node"></a>Узел области мониторинга
 
В построителе топологий Lync Server 2013 можно выполнять задачи, связанные с отдельным сервером мониторинга,  щелкнув правой кнопкой мыши сервер мониторинга в дереве консоли, щелкнув действие на панели инструментов или щелкнув задачу в области действий, а затем щелкнув один из следующих параметров для сервера мониторинга:
  
- **Изменить свойства**. Открывает  страницу "Изменение свойств" для сервера мониторинга, где можно изменить полное доменное имя сервера мониторинга или SQL Server данных мониторинга.
    
- **удаление**. Удаляет сервер мониторинга из топологии и SQL Server данных мониторинга.
    
Помимо приведенных выше в этом разделе параметров сервера мониторинга имеются общие параметры узла "Мониторинг", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Топология.** С помощью этого параметра можно определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
