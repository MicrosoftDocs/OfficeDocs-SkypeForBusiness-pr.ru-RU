# <a name="install-database-options-page"></a>Страница установки параметров базы данных

Вы настраиваете дополнительные параметры для размещения файлов базы данных и журнала на сервере SQL Server. Доступны следующие параметры:

> [!IMPORTANT]
> Выберите параметр, который наилучшим образом подходит для требований и политик, касающихся расположения файлов данных и журнала на компьютерах SQL Server.

 **Автоматически определять расположение файла базы данных**: параметр по умолчанию использует алгоритм, определяющий доступное пространство на SQL Server и распространяющий файлы базы данных и журнала для обеспечения оптимальной производительности.

 **Использовать значения по умолчанию для экземпляра SQL Server**: Выберите этот параметр для размещения файлов базы данных и журнала на основе параметров экземпляра на сервере SQL Server. Эти параметры обычно управляются администратором базы данных и задаются им.

 **Мы указываем путь на целевом сервере SQL Server**: Выберите этот параметр, чтобы определить собственные пути для базы данных SQL Server и файлов журнала, указав полный путь к диску и папке, в которую будут помещены файлы базы данных и журнала.

> [!IMPORTANT]
> Введенные пути можно изменить на основе алгоритмов оптимизации производительности в установке. Подробные сведения можно найти [в разделе Установка баз данных с помощью командной консоли Lync Server Management Shell](https://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx).

 **ОК**: нажмите кнопку ОК, чтобы сохранить изменения.

 **Отменить**: нажмите кнопку Отмена, чтобы отменить все изменения и вернуться к экрану Установка базы данных.

 **Справка**: нажмите кнопку Help (Справка), чтобы открыть эту страницу справки.
