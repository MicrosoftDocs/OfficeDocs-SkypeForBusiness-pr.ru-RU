# <a name="define-persistent-chat-fqdn"></a>Определение полного доменного имени сохраняемого чата
 
Вы можете создать новый сервер сохраняемого чата или пул серверов сохраняемого чата с помощью мастера **определения нового пула для сохраняемого чата** . Выберите **Пул из нескольких компьютеров** или **Пул из одного компьютера**. Если выбран пул из одного компьютера, а позднее потребовался пул из нескольких компьютеров, необходимо удалить пул из одного компьютера и затем определить пул из нескольких компьютеров.
  
Кроме того, необходимо определить **полное доменное имя пула** для сервера сохраняемого чата или для пула серверов сохраняемого чата. Полное доменное имя пула из одного компьютера должно совпадать с полным доменным именем компьютера, включенного в этот пул. Полное доменное имя пула из нескольких компьютеров должно совпадать с именем, выбранным для представления этого пула и указанным в записи узла A (и AAAA в случае применения протокола IPv6) в службе доменных имен.
