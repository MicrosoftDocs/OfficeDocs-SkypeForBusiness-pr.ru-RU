# <a name="persistent-chat-general-settings-expander"></a>Расширитель общих настроек сохраняемого чата
 
Вы можете изменить **Общие** параметры сервера сохраняемого чата или пула серверов сохраняемого чата, настроив или определив указанные ниже свойства.
  
 **Общие**
  
- **FQDN**: измените этот параметр, чтобы определить полное доменное имя сервера сохраняемого чата или сохраняемого пула серверов чата.
    
- **Отображаемое имя пула для постоянного чата**: определите этот параметр, чтобы предоставить пользователю понятные и пользовательские параметры для сервера или пула. Этот параметр упрощает для пользователей связывание указанного сервера или пула для постоянного чата на основе отображаемого имени, а не более сложного представления полного доменного имени.
    
- **Порт сохраняемых чатов**. Порт, предназначенный для сохраняемых чатов.
    
Вы можете изменить параметры **ассоциаций** для сервера сохраняемого чата или пула серверов сохраняемого чата, настроив или определив следующие свойства:
  
 **Связи**
  
- **Хранилище SQL Server**. Выберите в списке хранилище SQL Server и при необходимости именованный экземпляр.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **включить зеркальное отображение хранилища SQL Server** , если вы хотите включить зеркальное отображение для основного хранилища SQL Server.
    
    Если вы решили включить зеркальное отображение хранилища SQL Server, выберите магазин и экземпляр из **хранилища SQL Server в зеркальном отображении**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server, чтобы включить автоматический переход на другой ресурс** , если требуется автоматически отработка отказа основного хранилища SQL Server.
    
    Если вы решили включить следящий сервер хранилища SQL Server для включения автоматического перехода на другой ресурс, выберите магазин и экземпляр из списка.
    
    Для задания нового хранилища SQL Server (и экземпляра хранилища следящего сервера, если он требуется) щелкните **Создать**.
    
- Установите флажок **использовать архивацию хранилищ SQL Server для включения аварийного восстановления** , если вы хотите включить использование SQL Server с аварийным восстановлением
    
    Разрешив аварийное восстановление, следует выбрать хранилище и экземпляр в списке **Резервное хранилище SQL Server**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **включить зеркальное отображение хранилища SQL Server** , если вы хотите включить зеркальное отображение резервного хранилища SQL Server Mirroring.
    
    Если вы решили включить резервное копирование хранилища в хранилище SQL Server, выберите магазин и экземпляр из **резервной копии хранилища SQL Server**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server, чтобы включить автоматический переход на другой ресурс** , если требуется автоматический переход резервного хранилища SQL Server.
    
    Если вы решили включить следящий сервер хранилища SQL Server для включения автоматического перехода на другой ресурс, выберите магазин и экземпляр из списка.
    
    Для задания нового хранилища SQL Server (и экземпляра хранилища следящего сервера, если он требуется) щелкните **Создать**.
    
- Если требуется доступ к базе данных соответствия, установите флажок **Включить соответствие**.
    
    Разрешив доступ к базе данных соответствия, следует выбрать хранилище и экземпляр в списке **Совместимое хранилище SQL Server**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **включить зеркальное отображение хранилища SQL Server** , если вы хотите включить зеркальное отображение для хранилища SQL Server соответствия требованиям.
    
    Если вы решили включить зеркальное отображение хранилища SQL Server, выберите магазин и экземпляр из списка **соответствие требованиям зеркальной копии хранилища SQL Server**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server, чтобы включить автоматический переход на другой ресурс** , если требуется автоматически отработка отказа в хранилище SQL Server соответствия требованиям.
    
    Если вы решили включить следящий сервер хранилища SQL Server для включения автоматического перехода на другой ресурс, выберите магазин и экземпляр из списка.
    
    Для задания нового хранилища SQL Server (и экземпляра хранилища следящего сервера, если он требуется) щелкните **Создать**.
    
- Разрешив доступ к базе данных соответствия, следует выбрать хранилище и экземпляр в списке **Резервное хранилище SQL Server для соответствия требованиям**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **включить зеркальное отображение хранилища SQL Server** , если вы хотите включить зеркальное отображение для хранилища SQL Server соответствия требованиям.
    
    Если вы решили включить зеркальное отображение хранилища SQL Server, выберите магазин и экземпляр из **зеркала хранилища SQL Server для обеспечения соответствия резервным копированием**.
    
    Для задания нового хранилища SQL Server (и экземпляра, если он требуется) щелкните **Создать**.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server, чтобы включить автоматический переход на другой ресурс** , если требуется автоматически отработка отказа в хранилище SQL Server соответствия резервной копии.
    
    Если вы решили включить следящий сервер хранилища SQL Server для включения автоматического перехода на другой ресурс, выберите магазин и экземпляр из списка.
    
    Для задания нового хранилища SQL Server (и экземпляра хранилища следящего сервера, если он требуется) щелкните **Создать**.
    
- **Хранилище файлов** Выберите расположение хранилища файлов в списке или нажмите кнопку **создать** , чтобы создать новое хранилище файлов.

  **Отмена**. Отмена изменений и закрытие диалогового окна.
  
  **Справка**. Отображение этого экрана справки.
