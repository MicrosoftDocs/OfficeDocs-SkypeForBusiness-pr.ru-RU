# <a name="persistent-chat-general-settings-expander"></a>Расширитель общих настроек сохраняемого чата
 
Вы можете изменить **Общие** параметры сервера сохраняемого чата или пула серверов сохраняемого чата, настроив или определив следующие свойства:
  
 **Общие сведения**
  
- **Полное доменное**имя: измените этот параметр, чтобы определить полное доменное имя сервера сохраняемого чата или пула серверов сохраняемого чата.
    
- **Отображаемое имя пула сохраняемого чата**: определите этот параметр, чтобы обеспечить удобные и пользовательские параметры для сервера или пула. Этот параметр упрощает для пользователей сопоставление заданному серверу сохраняемого чата или пула серверов сохраняемого чата на основе отображаемого имени, а не более сложного представления полного доменного имени.
    
- **Порт непрерывных чатов**. Укажите порт, который используется для непрерывных чатов.
    
Вы можете изменить параметры **связи** для сервера сохраняемого чата или пула серверов сохраняемого чата, настроив или определив следующие свойства:
  
 **Связанных**
  
- **Хранилище SQL Server**: выберите в списке Хранилище SQL Server и необязательный именованный экземпляр.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **включить зеркалирование хранилища SQL Server** , если вы хотите включить зеркальное отображение для основного хранилища SQL Server.
    
    Если включено зеркальное отображение хранилища SQL Server, выберите хранилище и экземпляр в списке **зеркальное отображение хранилища SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server для включения автоматического перехода** на другой ресурс, если требуется автоматическая отработка отказа основного хранилища SQL Server.
    
    Если вы решили включить следящий сервер хранилища SQL Server для автоматического перехода на другой ресурс, выберите хранилище и экземпляр в списке.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр для хранилища-свидетеля.
    
- Установите флажок **использовать резервные хранилища SQL Server для включения аварийного восстановления** , если необходимо включить аварийное восстановление SQL Server
    
    Если включена возможность аварийного восстановления, выберите хранилище и экземпляр в списке **Резервное хранилище SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **включить зеркалирование хранилища SQL Server** , если вы хотите включить зеркальное отображение для резервного хранилища SQL Server зеркального отображения.
    
    Если вы решили включить зеркальное отображение хранилища SQL Server, выберите хранилище и экземпляр в списке **резервная копия хранилища SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server для автоматического перехода** на другой ресурс, если требуется автоматическая отработка отказа резервного хранилища SQL Server.
    
    Если вы решили включить следящий сервер хранилища SQL Server для автоматического перехода на другой ресурс, выберите хранилище и экземпляр в списке.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр для хранилища-свидетеля.
    
- Установите флажок **Включить соответствие**, если необходимо включить использование базы данных соответствия.
    
    Если включено соответствие требованиям, выберите хранилище и экземпляр в списке **Совместимое хранилище SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **включить зеркалирование хранилища SQL Server** , если необходимо включить зеркальное отображение для хранилища SQL Server соответствия требованиям.
    
    Если включено зеркальное отображение хранилища SQL Server соответствия требованиям, выберите хранилище и экземпляр из списка **соответствие зеркальному хранилищу SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server для включения автоматического перехода** на другой ресурс, если требуется автоматическая отработка отказа хранилища SQL Server соответствия требованиям.
    
    Если вы решили включить следящий сервер хранилища SQL Server для автоматического перехода на другой ресурс, выберите хранилище и экземпляр в списке.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр для хранилища-свидетеля.
    
- Если включено соответствие требованиям, выберите хранилище и экземпляр в списке **Зеркало резервного совместимого хранилища SQL Server**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **включить зеркалирование хранилища SQL Server** , если необходимо включить зеркальное отображение для хранилища SQL Server соответствия требованиям.
    
    Если включено зеркальное отображение хранилища SQL Server соответствия требованиям, выберите хранилище и экземпляр в списке **зеркало резервного хранилища SQL Server для соответствия резервному копированию**.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр.
    
- Установите флажок **использовать следящий сервер зеркального отображения SQL Server для автоматического** перехода на другой ресурс, если требуется автоматическая отработка отказа хранилища SQL Server соответствия резервных копий.
    
    Если вы решили включить следящий сервер хранилища SQL Server для автоматического перехода на другой ресурс, выберите хранилище и экземпляр в списке.
    
    Щелкните элемент **Создать**, чтобы определить новое хранилище SQL Server и необязательный экземпляр для хранилища-свидетеля.
    
- **Хранилище файлов** Выберите расположение хранилища файлов в списке или нажмите кнопку **создать** , чтобы создать новое хранилище файлов.
    
  При нажатии кнопки **ОК** выполняется принятие и сохранение изменений в этом диалоговом окне.
  
  При нажатии кнопки **Отмена** изменения отменяются, а диалоговое окно закрывается.
  
  При нажатии кнопки **Справка** отображается данный экран справки.
