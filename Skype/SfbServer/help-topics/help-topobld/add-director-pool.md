# <a name="add-director-pool"></a>Добавление пула директоров
 
Чтобы **определить полное доменное имя пула директоров**, выберите **пул с несколькими компьютерами**, состоящий из двух и более директоров в пуле с балансировкой нагрузки, или **пул из одного компьютера**. Кроме того, необходимо ввести полное доменное имя (FQDN), которое будет использоваться для подключения к пулу директоров или полным доменным именем единого директора. Для пула директоров этим именем будет DNS-запись для виртуального IP-адреса аппаратного балансировщика нагрузки или общая DNS-запись для балансировки нагрузки DNS.
  
> [!TIP]
> Если в дальнейшем планируется внедрить пул директоров, выберите **Пул с несколькими компьютерами**. Несмотря на то, что пул является группой компьютеров с балансировкой нагрузки, можно создать пул, состоящий из одного компьютера, а затем присвоить полное доменное имя этому компьютеру. Когда вы будете готовы к добавлению дополнительных компьютеров в пул, необходимо повторно запустить построитель топологий, чтобы определить новый член пула, опубликовать новую топологию, а затем настроить новый участник пула директоров с помощью мастера развертывания Skype для бизнеса Server. Необходимо также добавить этого нового участника пула в соответствующие балансировщики нагрузки пула: балансировка нагрузки DNS или аппаратные средства балансировки нагрузки. Во многих случаях необходимо задействовать обе системы балансировки нагрузки. Обязательно добавьте новый сервер участника в оба средства балансировки.
