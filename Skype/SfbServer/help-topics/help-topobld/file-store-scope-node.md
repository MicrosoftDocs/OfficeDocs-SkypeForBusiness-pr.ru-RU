# <a name="file-store-scope-node"></a>Узел области хранилища файлов
 
Этот узел отображает полное доменное имя (FQDN) и зависимые пулы для этого хранилища файлов. Можно выполнить следующие действия.
  
- **Изменение свойств**. Открывает страницу **Изменение свойств** для этого сервера, на которой можно изменить полное доменное имя пула и имя файлового хранилища.
    
- **Удалить**. Удаляет файловое хранилище из топологии.
    
Кроме работы с этими параметрами, можно выполнять следующие задачи общего характера:
  
- **Topology (топология**). Вы можете использовать этот параметр, чтобы определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
