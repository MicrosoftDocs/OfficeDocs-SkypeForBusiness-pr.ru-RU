# <a name="mediation-collocation-planning-tool"></a>Выровненное размещение исправлений (средство планирования)
 
По умолчанию сервер-посредник размещен на сервере переднего плана. Сервер-посредник также может быть развернут в изолированном пуле по соображениям производительности или при развертывании магистральной магистрали SIP, в этом случае настоятельно рекомендуется использовать автономный пул. Совместное размещение в Skype для бизнеса Server работает так же, как и в Lync Server 2013. Дополнительные сведения см. в следующих разделах:
  
- [Поддерживаемое выровненное размещение серверов в Lync Server 2013](https://technet.microsoft.com/library/gg425885%28v=ocs.15%29.aspx)
    
- [Компонент сервера-посредника в Skype для бизнеса Server 2015](../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
