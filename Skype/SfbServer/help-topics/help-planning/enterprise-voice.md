# <a name="skype-for-business-server-enterprise-voice-planning-tool"></a>Skype для бизнеса Server Корпоративная голосовая связь планирования
 
С Корпоративная голосовая связь 2015 Skype для бизнеса Server предоставляет автономный голосовой голос по протоколу Интернета (VoIP), предлагающий усовершенствовать или заменить традиционные системы обмена частными филиалами (PBX). Корпоративная голосовая связь пользователи могут звонить коллегам в сети VoIP организации или PBX, а также вызывать традиционные телефонные номера за пределами организации. Решение Корпоративная голосовая связь включает общие функции вызова, такие как ответ, переадресование, передача, удержание, отвлечь, освободить и припарковаться, а также расширенный вызов 9-1-1 (E9-1-1) (E9-1-1 доступен только в США).) Корпоративная голосовая связь также поддерживает широкий диапазон текущих и старых IP-устройств и USB.
  
Дополнительные сведения о планировании Корпоративная голосовая связь см. в Корпоративная голосовая связь [в Skype для бизнеса Server 2015 г.](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md) Дополнительные сведения о развертывании отдельного сервера-посредника см. в материале [Deploy a Mediation Server in Topology Builder Skype для бизнеса Server 2015 г.](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
