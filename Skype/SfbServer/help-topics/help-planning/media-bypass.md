# <a name="media-bypass-planning-tool"></a>Media Bypass (Planning Tool)
 
Обойтись обозначает удаление сервера обновлений из пути к носителю, если это возможно, для звонков, сигнализация которых проходит на сервер обновлений. Дополнительные сведения можно найти [в разделе Планирование обхода мультимедиа в Skype для бизнеса 2015](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
