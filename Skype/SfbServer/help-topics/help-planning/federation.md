# <a name="federation-planning-tool"></a>Федерация (средство планирования)

 Федерации работает, как и в Microsoft Lync 2013. Дополнительные сведения о подготовке среды для федерации см. в следующих статьях:

- [Планирование для общедоступных служб обмена мгновенными сообщениями в Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205349%28v=ocs.15%29.aspx)
    
- [Планирование для расширяемого системы обмена сообщениями и сведениями о присутствии федерации протокола XMPP в Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205107%28v=ocs.15%29.aspx)
