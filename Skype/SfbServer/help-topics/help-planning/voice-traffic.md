# <a name="voice-traffic-planning-tool"></a>Voice Traffic (Planning Tool)
 
Средство планирования использует сведения о трафике пользователя, чтобы определить количество портов для использования в развертывании. Этот номер, в свою очередь, определяет количество серверов и шлюзов, которые должны быть обязательными для устранения. Подробности можно найти в разделе [Оценка использования голосовой связи и трафика для Skype для бизнеса Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
