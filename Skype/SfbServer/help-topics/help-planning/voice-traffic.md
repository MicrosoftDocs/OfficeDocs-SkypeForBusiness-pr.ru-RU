# <a name="voice-traffic-planning-tool"></a>Голосовой трафик (средство планирования)
 
Средство планирования использует сведения о пользовательском трафике для определения количества портов, которые будут применяться в развертывании. Это число, в свою очередь, определяет необходимое количество серверов-посредников и шлюзов. For details, see [Estimating voice usage and traffic for Skype for Business Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
