# <a name="persistent-chat-planning-tool"></a>Сохраняемый чат (средство планирования)
 
Серверов сохраняемого чата дает пользователям возможность участвовать в беседах многопользовательских, основанный на раздел, сохраняющиеся во времени.
  
Для получения дополнительных сведений о подготовке среды для сервера сохраняемого чата видеть [Планирование сервера сохраняемого чата в Скайп для Business Server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
