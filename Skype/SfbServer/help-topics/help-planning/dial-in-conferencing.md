# <a name="dial-in-conferencing-planning-tool"></a>Dial-in Conferencing (Planning Tool)
 
Конференц-связь с телефонным подключением в Skype для бизнеса Server 2015 позволяет пользователям присоединиться к звуковой части Конференции Skype для бизнеса Server 2015 с помощью коммутируемого телефонного телефона, доступного по коммутируемой телефонной сети, не требуя поставщика услуг голосовой связи с третьими лицами.
  
Сведения о планировании включения конференц-связи с телефонным подключением в вашей организации см. в следующих статьях и разделах: 
  
- [Планирование конференц-связи в Skype для бизнеса Server 2015](../../plan-your-deployment/conferencing/conferencing.md)
