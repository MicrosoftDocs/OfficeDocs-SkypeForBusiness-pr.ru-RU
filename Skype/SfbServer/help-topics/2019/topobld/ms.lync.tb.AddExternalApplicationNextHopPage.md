# <a name="add-nexthop-trusted-application-server"></a>Добавление сервера доверенных приложений NextHop
 
Чтобы задать узел следующего перехода для доверенный сервер приложений или пула, выполните следующие действия.
  
В списке **пул узла следующего перехода** выберите пул Enterprise Edition или Standard Edition Front End Server, который будет пул надежных приложений для использования в качестве следующего прыжка.
  
Нажмите кнопку **Готово** , чтобы завершить создание пула доверенных приложений.
