# <a name="add-sql-store"></a>Добавление хранилища SQL
 
Чтобы определить новое хранилище SQL, что означает, что есть определить базу данных на основе SQL Server и экземпляр SQL Server — экземпляр по умолчанию или именованный экземпляр, следует указать приведенные ниже.
  
Укажите полное доменное имя (FQDN) сервера SQL Server, на котором будет размещаться экземпляр базы данных, которая определяется.
  
Укажите экземпляр SQL Server, на котором будут размещаться данные. Можно указать экземпляр по умолчанию или указать именованный экземпляр.
  
Выровненное размещение баз данных в конкретные экземпляры должно быть четко понятно. Для получения дополнительных сведений о выровненное размещение серверов и выровненное размещение экземпляр базы данных видеть [Выровненное размещение серверов в развертывании пула переднего плана](http://technet.microsoft.com/library/0516b18d-14c0-4237-9279-0f92e341b1bd.aspx) и [Выровненное размещение серверов в развертывании сервера Standard Edition](http://technet.microsoft.com/library/0763ffab-4fd6-463a-8e62-d97876b376d3.aspx).
