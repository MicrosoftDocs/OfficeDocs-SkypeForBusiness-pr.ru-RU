# <a name="legacy-merge-sip-port"></a>Порт SIP для объединения с устаревшей версией
 
По умолчанию обмен данными с клиентами, интерфейсными пулами прежних версий и серверами осуществляется через SIP-порт 5061, работающий по протоколу TCP. Если значение по умолчанию не применялось, выберите **SIP-порт для всех пулов и серверов:** и введите новое значение.
  
Если в развертывании прежней версии настроен автоматический вход клиентов, выберите режим **Автоматическое определение конфигурации SIP-порта для каждого пула** и нажмите кнопку **Далее**.
