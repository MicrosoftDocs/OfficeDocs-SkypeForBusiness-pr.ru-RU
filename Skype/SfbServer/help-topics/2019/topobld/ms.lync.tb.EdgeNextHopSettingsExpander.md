# <a name="edge-next-hop-settings-expander"></a>Расширитель параметров следующего пограничного перехода

Для задания или изменения параметра **Пул узла следующего перехода**, определяющего пул для обмена данными между пограничными серверами и внутренней инфраструктурой, выберите в списке директор, пул директоров, сервер переднего плана или пул серверов переднего плана. В списке отображаются только режиссеры и серверы переднего плана, настроенные в построителе топологии.

О задании и настройке параметров пограничных служб см. в разделе [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
