# <a name="remove-deployment-introduction"></a>Введение в удаление развертывания
 
Запускайте **Мастер удаления развертывания**, только если необходимо ликвидировать все развертывания. Этот мастер создает пустую топологию, состоящую из одного интерфейсного сервера. После запуска этого мастера и публикации пустой топологии будет невозможно вернуться к исходной топологии.
  
> [!IMPORTANT]
> Перед нажатием кнопки **Далее** убедитесь, что удалены все неактивные компьютеры, отключены все пользователи и конечные точки в системе объединенных коммуникаций, а также удалены все каталоги конференций, как указано на первой странице мастера.
