# <a name="add-edge-server-associations"></a>Добавление связей для пограничного сервера
 
Используйте эту страницу, чтобы выбрать, какие пулов переднего плана и серверов Standard Edition будет с помощью этого пограничного пула A / V трафика. 
  
Каждый пул переднего плана может быть связан только один пул пограничного сервера. Если выбран параметр пула, который уже связан с другой пограничного пула, изменится связи от предыдущей пограничного пула.
  
Вы также можете сопоставить этот пограничный сервер или пограничный пул с серверов-посредников или серверов-посредников для настройки и включения корпоративной голосовой связи для внешних пользователей.
