# <a name="topology-builder-startup-dialog"></a>Диалоговое окно запуска построителя топологии
 
Вы начинаете Topology Builder для настройки свойств и конфигурации развертывания. При запуске Topology Builder отображает диалоговое окно с вопросом о том, каким будет источник документа топологии. На выбор предоставляются три варианта:
  
- **Скачайте топологию из существующего развертывания** Выберите этот параметр, если вы выполнили топологию Builder и успешно опубликовали топологию на Центральном сервере управления. Этот параметр запрашивает и загружает локальную копию текущей конфигурации для последующего редактирования и настройки существующего развертывания.
    
- **Откройте топологию из локального файла** Выберите этот параметр, если вы настраивали развертывание, но еще не опубликовали конфигурацию. Этот параметр позволяет сохранить локализованную копию работы конфигурации в Topology Builder и вернуться к ней позже.
    
- **Новая топология**. Выберите этот параметр, чтобы начать редактирования нового документа топологии. Этот параметр открывает новый документ топологии для настройки в соответствии с требованиями текущего развертывания.
