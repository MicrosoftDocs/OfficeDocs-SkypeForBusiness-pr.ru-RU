# <a name="topology-builder-startup-dialog"></a>Диалоговое окно запуска построителя топологии
 
Вы запускаете построитель топологии для настройки свойств и конфигурации развертывания. При запуске построитель топологии выводит диалоговое окно, в котором спрашивается, что будет источник для документа Topology. Вы выбираете один из трех вариантов:
  
- **Загрузка топологии из существующего развертывания** Выберите этот параметр, если вы выполнили построитель топологии и успешно опубликовали топологию на центральном сервере управления. Этот параметр запрашивает и загружает локальную копию текущей конфигурации, позволяя изменить и настроить существующее развертывание.
    
- **Открытие топологии из локального файла** Выберите этот параметр, если вы настраиваете развертывание, но еще не опубликовали конфигурацию. Этот параметр позволяет сохранить локальную копию работы по настройке в построителе топологии и вернуться к ней позже.
    
- **Новая топология** Выберите этот параметр, чтобы начать настройку нового документа с топологией. Этот параметр открывает новый документ топологии, который вы настраиваете в соответствии с требованиями для развертывания, которое вы будете настраивать.
