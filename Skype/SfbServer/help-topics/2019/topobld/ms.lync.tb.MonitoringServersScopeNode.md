# <a name="monitoring-servers-scope-node"></a>Узел области серверов мониторинга
 
Узел **Серверов мониторинга** список серверов мониторинга в топологии. Можно выполнить следующее действие.
  
- **Новый сервер мониторинга**. Запускает мастер **Определение нового сервера мониторинга** , который используется для добавления нового сервера мониторинга в топологию.
    
Кроме параметров сервера мониторинга можно выполнить следующие задачи общего характера:
  
- **Топология**. Этот параметр можно использовать для определения новой топологии, откройте существующую топологию, загрузите текущей топологии, сохранить копию топологии, опубликовать эту топологию, установка базы данных, объединение компонентов с предыдущими версиями Office Communications Server или удаление развертывание.
    
- **Вид**. Эта команда позволяет настроить представление в построителе топологий путем выбора элементов для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
