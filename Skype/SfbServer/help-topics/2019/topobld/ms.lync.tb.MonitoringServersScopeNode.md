# <a name="monitoring-servers-scope-node"></a>Узел области серверов мониторинга
 
Узел **Серверы мониторинга** отображает серверы мониторинга, используемые в топологии. Здесь можно выполнить следующее действие.
  
- **Создать сервер мониторинга**. Служит для запуска мастера **Определение нового сервера мониторинга**, который используется для создания в топологии нового сервера мониторинга.
    
Кроме параметров сервера мониторинга, можно выполнять следующие задачи общего характера:
  
- **Топология**. Можно использовать данный параметр, чтобы определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий nm-ocs-13-no-version или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
