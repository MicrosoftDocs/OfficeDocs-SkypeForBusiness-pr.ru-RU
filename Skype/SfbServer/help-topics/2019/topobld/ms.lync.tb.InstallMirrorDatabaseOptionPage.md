# <a name="install-mirror-database-option-page"></a>Установка страница параметров зеркальной базы данных
 
Настройте **Параметры зеркальной базы данных** определяются следующим:
  
- Введите **путь к файлу совместно использовать** для определения местоположения файлов резервных копий SQL Server для зеркалируемой базы данных.
    
    > [!NOTE]
    > Основной экземпляр SQL Server (именованный экземпляр или экземпляр по умолчанию) необходимо иметь разрешения на запись в общую папку файлов, здесь. Зеркальный экземпляр SQL Server (именованный экземпляр или экземпляр по умолчанию) необходимо иметь разрешения на чтение же общего файлового ресурса. 
  
 **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
 **Отмена**. Отмена изменений и закрытие диалогового окна.
  
 **Справка**. Отображение этого экрана справки.
