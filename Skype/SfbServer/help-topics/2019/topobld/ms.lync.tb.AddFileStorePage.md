# <a name="add-file-store"></a>Добавление хранилища файлов
 
Чтобы определить новое хранилище файлов, укажите следующие параметры.
  
- Полное доменное имя (FQDN) сервера, на котором будет размещаться в новое файловое хранилище. Полное доменное имя сервера должны существовать в доменных имен (DNS), перед публикацией топологии.
    
- Имя общей папки на сервере размещения. Общая папка должна существовать до публикации топологии.
