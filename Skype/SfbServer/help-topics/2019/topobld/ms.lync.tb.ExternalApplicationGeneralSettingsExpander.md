# <a name="external-application-general-settings-expander"></a>Расширитель внешнего приложения общих параметров
 
Чтобы изменить свойства сервера доверенных приложений, который уже был определен, выполните следующие действия.
  
Существует два раздела, которые можно изменить.
  
> Общие настройки
    
> Настройки следующего перехода
    
## <a name="general-settings"></a>Общие настройки

Вы можете изменить текущее полное доменное имя (FQDN) для пула сервера доверенных приложений. Измените имя полное доменное имя пула. Записи узлов (A) доменных имен (DNS) должна существовать для новой записи, прежде чем клиентов или серверов может подключаться к новое имя группы.
  
Если необходимо иметь для этого пула репликацию данных конфигурации, установите флажок **Включить репликацию данных конфигурации в этот пул** . Снимите флажок, если необходимо реплицировать данные конфигурации.
  
## <a name="next-hop-settings"></a>Настройки следующего перехода

Можно указать сервера следующего прыжка пула сервера доверенных приложений, выбрав определенного пула переднего плана Enterprise Edition или Standard Edition сервера переднего плана в раскрывающемся списке. Директор или директора пула не является допустимым выбора для следующего прыжка сервера доверенных приложений и не будет отображаться в списке.
  

Нажмите **кнопку ОК** , чтобы принять и сохранить изменения. Для отмены изменений и закрытия страницы свойств нажмите кнопку **Отмена**.
