# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства для обеспечения связи в филиалах с сервером переднего плана
 
Чтобы связать устройство или сервер для связи в филиалах с пулом переднего сервера или сервером Standard Edition, на странице "Выбор пула переднего сервера" выберите пул переднего сервера или сервер Standard Edition в выпадаемом списке пулов переднего уровня.   Нажмите кнопку **Далее**, чтобы продолжить.
