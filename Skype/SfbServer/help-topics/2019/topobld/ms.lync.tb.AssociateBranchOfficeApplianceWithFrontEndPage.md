# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства филиала с сервером переднего плана
 
Чтобы связать для обеспечения связи в филиалах или для обеспечения связи в филиалах с пулом переднего плана или сервера Standard Edition, на странице **Выберите пул переднего плана** выберите пул переднего плана или сервера Standard Edition из **пула переднего плана** раскрывающийся список. Для продолжения нажмите кнопку **Далее**.
