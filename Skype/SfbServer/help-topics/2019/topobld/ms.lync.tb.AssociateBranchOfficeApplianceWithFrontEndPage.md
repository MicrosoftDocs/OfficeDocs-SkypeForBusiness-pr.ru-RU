# <a name="associate-branch-office-appliance-with-front-end"></a>Связывание устройства филиала с сервером переднего плана
 
Чтобы связать работающее устройство филиала или его бесперебойный сервер с пулом переднего плана или сервером Standard Edition, на странице **Выбор пула** переднего плана выберите пул переднего плана или сервер стандартных выпусков из раскрывающегося списка **пул внешних** интерфейсов. Для продолжения нажмите кнопку **Далее**.
