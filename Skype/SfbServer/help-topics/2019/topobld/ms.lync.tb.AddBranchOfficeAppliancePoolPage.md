Чтобы определить полное доменное имя устройства для обеспечения обеспечения в филиалах, введите в текстовом поле полное доменное имя (как определено в  системе доменных имен (DNS) устройства или сервера обеспечения обеспечения ветвей в филиалах.  Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> Если не указано иное, определение устройства или сервера для survivable Branch Server является одной и той же процедурой.
