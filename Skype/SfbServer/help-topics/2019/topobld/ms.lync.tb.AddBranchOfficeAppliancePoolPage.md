# <a name="add-survivable-branch-appliance-pool"></a>Добавление пула устройств обеспечения связи в филиалах
 
Чтобы **определить полное доменное имя для обеспечения связи устройства для обеспечения связи в филиалах**введите полное доменное имя (FQDN) (как определено в доменных имен (DNS)) для обеспечения связи в филиалах или для обеспечения связи в филиалах в текстовом поле **полное доменное имя** . Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> Если указано иное, определение устройство для обеспечения связи в филиалах или для обеспечения связи в филиалах — ту же процедуру.
