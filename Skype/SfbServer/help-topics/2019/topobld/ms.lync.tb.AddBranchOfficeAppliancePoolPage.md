# <a name="add-survivable-branch-appliance-pool"></a>Добавление пула устройств обеспечения связи в филиалах
 
Чтобы **определить полное**доменное имя устройства, работающего в филиале, введите полное доменное (FQDN) (как определено в службе доменных имен (DNS)) для работающего устройства филиала или для бесперебойной ветви сервера в текстовом поле **FQDN** . Для продолжения нажмите кнопку **Далее**.
  
> [!NOTE]
> Если не указано иное, определение работающего устройства филиала или работающего сервера подразделения — это одна и та же процедура.
