# <a name="branch-office-appliance-scope-node"></a>Узел области устройства для обеспечения связи в филиалах
 
Этот узел отображает полностью квалифицированное доменное имя (FQDN), связанные пулы и серверы, а также сведения о устойчивости этого устройства для ветвей. Можно выполнить следующие действия.
  
- **Изменить свойства**. Откроет страницу **Edit Properties** для этого устройства для сохранившихся филиалов, где можно изменить FQDN (если топология еще не опубликована), связи и параметры устойчивости.
    
- **Удалить**. Удаляет сервер из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
