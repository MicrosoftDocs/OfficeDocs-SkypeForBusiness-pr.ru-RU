# <a name="simple-url-settings-expander"></a>Расширитель параметров простого URL-адреса

Чтобы задать новый, удалить существующий или изменить активный простой URL-адрес, выполните следующие действия:

## <a name="simple-urls"></a>Простые URL-адреса

Чтобы изменить или определить простые URL-адреса, сначала решите, какой из трех простых URL-адресов требуется изменить. Доступны следующие адреса: "URL-адреса для доступа с телефона", "URL-адреса собрания" и "URL-адрес для административного доступа".

Чтобы изменить "URL-адреса для доступа с телефона" или "URL-адреса собрания", выберите требуемый URL-адрес. Щелкните элемент **Изменить URL-адрес**. Измените URL-адрес и нажмите кнопку **ОК** для его сохранения. Нажмите кнопку **Отмена** для отмены всех изменений.

Чтобы добавить новый URL-адрес, нажмите кнопку **Добавить**. В диалоговом окне **Добавление простого URL-адреса** укажите URL-адрес и нажмите кнопку **ОК** для его сохранения. Выберите **Задать этот адрес в качестве активного URL-адреса для выбранного домена**, если хотите сделать новый URL-адрес активным. Нажмите кнопку **Отмена** для отмены всех изменений.

Чтобы сделать активным другой URL-адрес (на что указывает зеленая галочка рядом с URL-адресом), выберите его и щелкните **Сделать активным**.

> [!NOTE]
> Для каждого домена SIP может использоваться только один активный URL-адрес.

Чтобы удалить URL-адрес, выберите его и нажмите кнопку **Удалить**.

> [!CAUTION]
> Обязательно ознакомьтесь с информацией о параметрах простых URL-адресов на странице диалогового окна. Удаление URL-адреса собрания может сделать имеющиеся запланированные пользователями собрания недоступными. Рекомендуется оставить существующий URL-адрес после назначения в качестве активного нового URL-адреса собрания. Когда вы будете уверены, что пользователи больше не используют старый URL-адрес собрания, то сможете безопасно удалить его.

Чтобы изменить URL-адрес административного доступа, отредактируйте запись.

После внесения всех изменений на странице "Изменить свойства" нажмите кнопку **ОК**, чтобы сохранить изменения. Нажмите кнопку **Отмена**, чтобы не вносить изменения.
