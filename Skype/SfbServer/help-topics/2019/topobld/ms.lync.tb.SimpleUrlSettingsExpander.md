# <a name="simple-url-settings-expander"></a>Расширитель параметров простого URL-адреса
 
Чтобы задать новый, удалить существующий или изменить активный простой URL-адресов, выполните следующие действия.
  
## <a name="simple-urls"></a>Простые URL-адреса 

Чтобы изменить или определить простых URL-адресов, сначала решите, какой из трех простых URL-адресов, которые нужно изменить. Можно выбрать URL-адрес для доступа с телефона, URL-адрес собрания и URL-адрес административного доступа.
  
Если требуется изменить URL-адрес для доступа с телефона или собрания, выберите этот URL-адрес. Щелкните **Изменить URL-адрес**. Измените URL-адрес и нажмите кнопку **ОК** , чтобы сохранить URL-адрес. Для отмены изменений нажмите кнопку **Отмена**.
  
Для добавления нового URL-адреса нажмите кнопку **Добавить**. В диалоговом окне **Добавить простой URL-адрес** укажите URL-адрес и нажмите кнопку **ОК** , чтобы сохранить URL-адрес. Если этот URL-адрес должен быть активным, выберите **Задать этот адрес в качестве активного URL-адреса для выбранного домена**. Для отмены изменений нажмите кнопку **Отмена**.
  
Чтобы сделать другой URL-адрес активный URL-адрес (помечается символом зеленая галочка рядом с URL-адрес), выберите его и нажмите кнопку **Сделать активным**.
  
> [!NOTE]
> Может быть только один активный URL-адрес для каждого домена Session Initiation Protocol (SIP). 
  
Чтобы удалить URL-адрес, выберите его и нажмите кнопку **Удалить**.
  
> [!CAUTION]
> Обязательно прочитайте сведения на странице диалогового окна Параметры простых URL-адреса. Удаление URL-адрес собрания для отрисовки существующих собраний, запланированных пользователями более недоступна. Необходимо учитывайте, отправляемых из существующего URL-адрес после внесения active новый URL-адрес собрания. Если вы уверены, что пользователи больше не используется старый URL-адрес собрания, его можно удалить его. 
  
Чтобы изменить URL-адрес административного доступа, отредактируйте запись.
  
После завершения все изменения на странице "Изменение свойств", нажмите кнопку **ОК** , чтобы сохранить изменения. Для отмены изменений нажмите кнопку **Отмена**.
