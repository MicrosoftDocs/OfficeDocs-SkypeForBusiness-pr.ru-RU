# <a name="monitoring-scope-node"></a>Узел области мониторинга
 
В построителе топологии вы можете выполнять задачи, связанные с работой отдельного сервера, щелкнув правой кнопкой мыши сервер, на котором работает мониторинг, в дереве консоли, выбрав команду **действие** на панели инструментов или щелкнув задачу в области действий, а затем выбрав один из указанных ниже параметров для сервера, на котором запущен монитор.
  
- **Изменение свойств**. Открытие страницы " **изменение свойств** " для сервера, на котором работает наблюдение, где можно изменить полное доменное имя (FQDN) сервера, на котором работает наблюдение, или базу данных SQL Server мониторинга.
    
- **Удалить**. Удаляет сервер, на котором запущен мониторинг, из топологии и базы данных SQL Server мониторинга.
    
Помимо приведенных выше в этом разделе параметров сервера мониторинга имеются общие параметры узла "Мониторинг", которые также доступны в других узлах дерева консоли. Сюда входят следующие параметры:
  
- **Топология**. Позволяет определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных или удалить развертывание.
    
- **Справка**. Обеспечивает доступ к справке по активному узлу.
