# <a name="add-edge-server-fqdn"></a>Добавление полного доменного имени пограничного сервера
 
Необходимо указать полное доменное имя службы пограничного доступа. Если вы не выбрали параметр Use a Single **FQDN &amp; IP-адрес** на странице Выбор функций, необходимо также указать FQDN для службы edge веб-конференций и для службы A/V Edge. 
  
Кроме того, если выбран параметр Use **a Single FQDN &amp; IP-адрес,** необходимо указать другой номер порта для каждой из служб Edge (рекомендуемые параметры порта: 444 для службы Access Edge, 8057 для службы веб-конференций Edge и 443 для службы A/V Edge). Если этот параметр не выбран, можно использовать один номер порта (например, 443) для всех трех служб.
