# <a name="add-edge-server-fqdn"></a>Добавьте полное доменное имя пограничного сервера
 
Необходимо указать полное доменное имя (FQDN) для пограничной службы доступа. Если вы не выбрали **использования одного полного доменного ИМЕНИ &amp; IP-адрес** параметр на странице " **Выбор компонентов** " необходимо также указать полное доменное имя для службы Web пограничного сервера конференц-связи и A / видеоконференций.
  
Кроме того Если вы выбрали **использование единого полное доменное имя &amp; IP-адрес** параметр, необходимо указать другой номер порта для каждой из служб пограничного сервера (рекомендуемые параметры порта: службы 444 для пограничного сервера доступа, 8057 для пограничного сервера конференц-связи веб-службы и 443 для A / V Служба пограничного сервера). Если параметр не выбран, можно использовать один и тот же номер порта (например, 443) для всех трех служб.
