# <a name="add-mediation-machine"></a>Добавление компьютера-посредника
 
Укажите полное доменное имя (FQDN) каждого компьютера, который вы хотите добавить в качестве сервера переднего плана в этом пуле. До публикации топологии можно в любой момент обновить полное доменное имя добавленного к списку компьютера или удалить его из пула. После публикации топологии изменение полного доменного имени требует удаления сервера в построителе топологии и добавления нового сервера в пул с новым полным доменным именем.
