# <a name="front-end-start-page"></a>Начальная страница сервера переднего плана

С помощью этого мастера вы можете определить новый пул переднего плана для сайта. У каждого сайта должен быть по крайней мере один пул корпоративных выпусков или стандартный сервер Standard Edition. Дополнительные сведения о требованиях для использования мастера для создания пула переднего плана можно найти в разделе [Определение и Настройка пула переднего плана](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) в документации по развертыванию.
