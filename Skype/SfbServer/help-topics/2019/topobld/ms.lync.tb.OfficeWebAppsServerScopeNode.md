# <a name="office-web-apps-server-scope-node-expander"></a>Расширитель узла области сервера Office Web Apps
 
Щелкните правой кнопкой **мыши серверы Office Web Apps** в построителье топологий, чтобы выполнить следующее:
  
- **Новый сервер Office Web Apps:** запускает мастер определения нового **сервера Office Web Apps,** используемый для определения нового сервера Office Web Apps.
    
В дополнение к параметру "Новый сервер Office Web Apps" можно выполнить следующие общие задачи:
  
![Узел области сервера Office WebApps](../../../media/OfficeWebApps_Server_Scope_Node.jpg)
  
- **Топология.** С помощью этого параметра можно определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий или удалить развертывание.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
