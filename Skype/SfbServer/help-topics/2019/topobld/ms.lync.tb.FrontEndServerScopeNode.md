# <a name="front-end-server-scope-node"></a>Узел области сервера переднего плана
 
Данный узел отображает сведения о конфигурации данного интерфейсного пула или сервера Standard Edition. Можно выполнить следующие действия.
  
- **Создать сервер**. Если это интерфейсный пул, можно выбрать его для запуска мастера **Определение нового сервера**, который можно использовать для добавления другого интерфейсного сервера в этот пул.
    
- **Изменение свойств**. Служит для открытия страницы **Изменение свойств**, связанной с этим пулом или сервером, на которой можно включить или отключить компоненты, определить связи с хранилищами файлов, базами данных SQL и прочими серверами, а также определить сведения о веб-сервере и сервере-посреднике.
    
- **Удалить**. Служит для удаления интерфейсного пула или сервера Standard Edition из топологии.
    
Кроме работы с этими параметрами, можно выполнять следующие задачи общего характера:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
