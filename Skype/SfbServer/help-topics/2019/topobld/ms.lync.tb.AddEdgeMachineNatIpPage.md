# <a name="add-edge-machine-nat-ip"></a>Добавление IP-адреса NAT пограничного компьютера
 
Если общедоступный IP-адрес этого пограничного пула преобразуется системой преобразования сетевых адресов (NAT), вам следует указать соответствующий общедоступный IP-адрес. Это должен быть внешний IP-адрес службы пограничного сервера аудио- и видеоконференций.
  
Чтобы указать, используется ли преобразование сетевых адресов (NAT) для пограничного пула, щелкните правой кнопкой мыши пограничный пул и выберите **Изменение свойств**.
