# <a name="associate-front-end-with-edge"></a>Связывание сервера переднего плана с пограничным
 
С каждым пулом переднего плана может быть связан только один пограничный сервер или пограничный пул. Если к сайту разрешен доступ внешних пользователей, можно включить поддержку удаленных пользователей. Возможна также поддержка федеративных пользователей, включая пользователей конкретных поставщиков общедоступных услуг связи для обмена мгновенными сообщениями (например, Windows Live), и поддержка анонимных пользователей.
  
Со всеми пулами на сайте, а также с пулами на нескольких центральных сайтах, может применяться один и тот же пограничный сервер достаточной производительности. Для получения дополнительных сведений о мониторинге, включая масштабирование, см [для доступа внешних пользователей](http://technet.microsoft.com/library/ea098933-eff5-461e-aba3-e7f128784dc2.aspx) в документации по планированию. Для получения дополнительных сведений о разработке топологии для поддержки доступа внешних пользователей в документации по развертыванию показано [Определение топология с пограничным](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) .
