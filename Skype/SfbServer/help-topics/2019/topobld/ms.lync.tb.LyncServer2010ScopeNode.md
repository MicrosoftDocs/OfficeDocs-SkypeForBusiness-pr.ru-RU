# <a name="lync-server-2010-create-new-expander"></a>Создание нового расширителя Lync Server 2010
 
Можно создавать новые роли сервера в узле построитель топологий Lync Server 2010, щелкните правой кнопкой мыши на **Lync Server 2010**, выберите команду **Создать**, а затем выберите Создать роль сервера.
