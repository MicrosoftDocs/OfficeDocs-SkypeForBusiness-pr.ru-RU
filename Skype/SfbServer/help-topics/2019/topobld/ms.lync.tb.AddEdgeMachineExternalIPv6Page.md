# <a name="add-edge-machine-external-ipv6"></a>Добавление внешнего адреса IPv6 пограничного компьютера
 
В поле **внешний адрес IPv6**введите внешний IP-адрес, указывающий на служба пограничного сервера доступа, служба пограничного сервера конференц-связи Web и A / V Edge службы.
