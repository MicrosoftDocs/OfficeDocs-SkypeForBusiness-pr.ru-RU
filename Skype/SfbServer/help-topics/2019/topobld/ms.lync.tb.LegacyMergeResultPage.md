# <a name="legacy-merge-results"></a>Результаты объединения с устаревшей версией

Чтобы приступить к объединению устаревшего развертывания с помощью Skype для бизнеса Server, нажмите кнопку **Далее**. Дополнительные сведения о работе с мастером объединения построителя топологий см. в разделе [Merge Using Topology Builder Merge Wizard](https://technet.microsoft.com/library/c3f3c425-dab6-4dcd-bf0e-d7fde05f2ebf.aspx) документации по миграции.
