# <a name="legacy-merge-results"></a>Результаты объединения с устаревшей версией

Чтобы приступить к объединению устаревшего развертывания с Skype для бизнеса Server, нажмите **кнопку Далее**. Сведения о запуске мастера слияния topology Builder см. в материале [Merge Using Topology Builder Merge Wizard](/previous-versions/office/lync-server-2013/merge-using-topology-builder-merge-wizard) in the Migration documentation.
