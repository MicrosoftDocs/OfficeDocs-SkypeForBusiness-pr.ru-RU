# <a name="branch-office-appliances-scope-node"></a>Узел области устройств для обеспечения связи в филиалах
 
Узел **Устройства для обеспечения связи в филиалах** содержит список устройств для обеспечения связи на этом сайте филиала. Можно выполнить следующее действие.
  
- **Создать устройство для обеспечения связи в филиалах**. Запускает мастер **устройства для новых сохранившихся** ветвей, который используется для добавления на этот сайт нового устройства для ветвей в живых.
    
В дополнение к варианту Survivable Branch Appliance можно выполнить следующие общие задачи:
  
- **Топология**. С помощью этого параметра можно определить новую топологию, открыть существующую топологию, скачать текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
