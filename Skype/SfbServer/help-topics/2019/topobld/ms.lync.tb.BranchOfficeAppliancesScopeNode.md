# <a name="branch-office-appliances-scope-node"></a>Узел области устройств для обеспечения связи в филиалах
 
Узел **Устройства для обеспечения связи в филиалах** содержит список устройств для обеспечения связи на этом сайте филиала. Можно выполнить следующее действие.
  
- **Создать устройство для обеспечения связи в филиалах**. Запускает мастер **определения нового устройства для** связи в филиалах, который используется для добавления нового устройства для связи в филиалах на этот сайт.
    
В дополнение к варианту устройства для устройства для survivable branch appliance можно выполнять следующие общие задачи:
  
- **Топология.** С помощью этого параметра можно определить новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из предыдущих версий или удалить развертывание.
    
- **Вид**. Можно использовать данный параметр для настройки представления в построителе топологий, выбирая элементы для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Вы можете использовать данный параметр для доступа к справке по активному узлу.
