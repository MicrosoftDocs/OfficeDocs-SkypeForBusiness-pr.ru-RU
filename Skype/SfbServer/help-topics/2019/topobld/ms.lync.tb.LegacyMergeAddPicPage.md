# <a name="legacy-merge"></a>Объединение с устаревшей версией

Внешнее **FQDN** веб-службы позволяет внешним пользователям присоединяться к собраниям в локальной сети. Введите полное доменное имя внешнего интерфейса веб-аудиоконференции устаревшего сервера.

Внешний  порт веб-службы, **443,** является портом SIP по умолчанию, настроенным для клиентов. Если значение по умолчанию не использовалось, обновите значение внешнего порта для внешних **веб-служб.**

Выберите этот **пул, который используется** для федерации и для подключения к общедоступным службе im, если вы планируете использовать этот сервер для федерации. Если развернуто несколько пограничных серверов, то для федерации будет включен только один из них. Если этот ящик не будет задан, и позднее вы решите включить федерацию, необходимо снова запустить мастер объединения построитель топологий, а также опубликовать топологию. Подробные сведения [см. в этапе 4. Объединение тополий.](https://technet.microsoft.com/library/81eb5bb2-1fd7-4611-a2aa-eb2393c8abc9.aspx)
