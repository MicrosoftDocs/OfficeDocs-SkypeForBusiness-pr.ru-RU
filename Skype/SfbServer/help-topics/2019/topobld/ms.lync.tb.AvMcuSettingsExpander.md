# <a name="av-mcu-settings-expander"></a>Расширитель параметров AV MCU
 
Чтобы изменить свойства для уже определенного пула серверов аудио- и видеоконференций, выполните следующие действия:
  
## <a name="general-settings"></a>Общие настройки

Чтобы обновить полное доменное имя пула серверов аудио- и видеоконференций, введите имя пула в том виде, в котором оно определено в службе доменных имен (DNS). Требуется наличие записи хоста (A) службы доменных имен (DNS), которая соответствует имени этого пула серверов аудио- и видеоконференций.
  
## <a name="av-conferencing-service"></a>Служба аудио- и видеоконференций

Чтобы сопоставить пул серверов аудио- и видеоконференций, выберите пул переднего плана или пулы, которые хотите сопоставить с пулом серверов аудио- и видеоконференций.
  
> [!NOTE]
> Если на панели **Выберите пулы переднего плана, которые будут связаны с пулом аудио- и видеоконференций** нет отображаемых интерфейсных пулов, вы либо выровняли сервер аудио- и видеоконференций с интерфейсным пулом, либо не выбрали добавление компонента конференц-связи в разделе **Компоненты и функции** свойств интерфейсного пула.
  

После выполнения операции нажмите кнопку **ОК**, чтобы завершить изменение настроек пула серверов аудио- и видеоконференций. Нажмите кнопку **Отмена**, чтобы отменить все изменения.
