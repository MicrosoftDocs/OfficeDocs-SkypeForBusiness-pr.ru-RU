# <a name="add-mediation-server"></a>Добавление сервера-посредника
 
Чтобы определить сервера-посредника, укажите полное доменное имя (FQDN) для пула сервера-посредника. При создании пула серверов-посредников соответствуют имя пула для серверов-посредников, выполняется развертывание. Полные доменные имена отдельных серверов задаются на другом шаге. При развертывании одного сервера-посредника можно указать полное доменное имя будет полное доменное имя отдельного сервера-посредника. Дополнительные сведения см [Развертывание сервера-посредника в построителе топологий в Скайп для Business Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
