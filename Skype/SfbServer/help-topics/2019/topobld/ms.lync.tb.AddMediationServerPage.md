# <a name="add-mediation-server"></a>Добавление сервера-посредника
 
Чтобы определить сервер-посредник, укажите полное доменное имя для пула серверов-посредников. Если вы создаете пул серверов-посредников, укажите имя пула для развертываемых серверов-посредников. Полные доменные имена для индивидуальных серверов вы будете определять в другом действии. Если вы развертываете отдельный сервер-посредник, указанное полное доменное имя будет относиться к отдельному серверу-посреднику. Дополнительные сведения см. [в материале Deploy a Mediation Server in Topology Builder in Skype для бизнеса Server.](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
