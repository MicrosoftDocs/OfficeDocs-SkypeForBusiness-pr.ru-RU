# <a name="add-monitoring-server-sql-server-store"></a>Добавление хранилища SQL Server для сервера мониторинга

Серверу мониторинга требуется поддерживаемый 64-SQL Server для хранения данных мониторинга. Можно выбрать ранее задаваемую базу данных SQL Server, которая будет использоваться для мониторинга, или определить новую базу данных SQL Server, указав полное доменное имя сервера, на котором будет находиться база данных SQL Server, в дополнение к экземпляру SQL Server, который требуется использовать для новой базы данных SQL Server (которая может быть экземпляром по умолчанию) , или именуемом экземпляре, который вы указали).

Сведения о поддержке SQL Server см. [](https://technet.microsoft.com/library/e05d0032-bbea-4e61-987d-d07b1c045fd5.aspx) в документации по поддержке программного обеспечения и кластеров баз данных. Подробные сведения о базе данных мониторинга, в том [](https://technet.microsoft.com/library/3be990a1-5485-4b83-b73f-947ac97821f9.aspx) числе о размещении базы[](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx) данных мониторинга, см. в документации по поддержке, планировании мониторинга в документации по планированию и размещении данных и файлов журнала [SQL Server](https://technet.microsoft.com/library/67aa525b-8aa3-474f-827e-8e1d4697f30f.aspx) в документации по развертыванию.

> [!NOTE]
> Если учетная запись, использованная для публикации топологии, обладает достаточными правами и разрешениями, при публикации топологии вы можете создать базу данных мониторинга. Эту базу данных можно создать и позднее — в рамках процедуры установки. > Для установки и развертывания баз данных на сервере на основе SQL Server для мониторинга необходимо быть участником группы SQL Server sysadmins для сервера на основе SQL Server, на котором устанавливаются файлы базы данных. Если вы не входите в группу SQL Server sysadmin, необходимо запросить его добавить в группу, пока не будут развернуты файлы базы данных. Если вы не можете быть членом группы sysadmins, необходимо предоставить администратору SQL Server базы данных сценарий для настройки и развертывания баз данных. Дополнительные сведения о правах и разрешениях, требуемых для выполнения описанных процедур, см. в разделе [Deployment Permissions for SQL Server](https://technet.microsoft.com/library/56ea0c02-bcf5-4d45-aa13-570531c29074.aspx) документации по развертыванию.
