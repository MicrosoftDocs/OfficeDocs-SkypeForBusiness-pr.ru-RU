# <a name="mediation-servers-scope-node"></a>Узел области применения серверов-посредников
 
Узел **Пулы-посредники** содержит список пулов-посредников в данной топологии. Можно выполнить следующее действие.
  
- **Создать пула-посредник**. Запуск мастера **Определение нового пула-посредника** для добавления к топологии нового пула-посредника.
    
Помимо определения параметров пула-посредника можно выполнять следующие общие задачи:
  
- **Топология**. Эта команда позволяет задать новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, объединить компоненты из прежних версий или удалить развертывание.
    
- **Вид**. Эта команда позволяет настроить представление в построителе топологий путем выбора элементов для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
