# <a name="skype-for-business-server-create-new-expander"></a>Skype для бизнеса Server Create New Expander
 
Вы можете создать новые роли сервера в построитель топологий, щелкнув правой кнопкой мыши Skype для бизнеса **Server,** выбрав "Создать" и выбрав роль сервера для создания.
