# <a name="skype-for-business-server-create-new-expander"></a>Skype для бизнеса Server создание нового расширения
 
Вы можете создать новые роли сервера в topology Builder, щелкнув правой кнопкой мыши **Skype для бизнеса Server, выбрав** **New** и выбрав роль сервера для создания.
