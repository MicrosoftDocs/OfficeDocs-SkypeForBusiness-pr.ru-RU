# <a name="add-monitoring-server"></a>Добавление сервера мониторинга

Чтобы включить сервер мониторинга в топологию, необходимо указать полное доменное имя (FQDN) сервера, на котором необходимо развернуть сервер мониторинга. Перед публикацией топологии, которая включает сервер мониторинга и Skype для бизнеса Server на сервере, необходимо выполнить все необходимые условия для развертывания мониторинга. Сведения об этих предварительных требованиях см. в разделе [Deploying Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring) в документации по развертыванию.
