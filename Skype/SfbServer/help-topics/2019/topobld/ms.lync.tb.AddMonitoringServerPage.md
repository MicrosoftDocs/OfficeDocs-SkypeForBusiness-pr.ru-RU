# <a name="add-monitoring-server"></a>Добавление сервера мониторинга

Чтобы внедрить сервер мониторинга в схему топологии, необходимо указать полное доменное имя сервера, на котором вы хотите развернуть сервер мониторинга. Перед публикацией топологии, включающей сервер мониторинга и установку сервера Skype для бизнеса на сервере, необходимо выполнить все необходимые условия для развертывания наблюдения. Об этих условиях см. в разделе [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) документации по развертыванию.
