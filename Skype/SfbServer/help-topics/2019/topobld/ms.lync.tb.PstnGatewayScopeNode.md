# <a name="pstn-gateway-scope-node"></a>Узел области применения шлюза ТСОП
 
Этот узел показывает полное доменное имя (FQDN), связанный сервер-посредник и другие основные сведения об этом шлюзе коммутируемой телефонной сети с открытым коммутируемым подключением (PSTN). Доступны следующие действия.
  
- **Изменение свойств**. Открытие страницы **Изменение свойств**, где можно изменить полное доменное имя (если топология еще не опубликована), порт прослушивания, транспортный протокол и альтернативный IP-адрес мультимедийеых ресурсов.
    
- **Удалить**. Удаление сервера из топологии.
    
Кроме работы с этими параметрами, вы можете выполнять следующие задачи общего характера:
  
- **Топология**. Эта команда позволяет задать новую топологию, открыть существующую топологию, загрузить текущую топологию, сохранить копию топологии, опубликовать топологию, установить базу данных, выполнить объединение с компонентами прежних версий или удалить развертывание.
    
- **Вид**. Эта команда позволяет настроить представление в построителе топологий путем выбора элементов для скрытия и отображения в представлениях дерева консоли.
    
- **Справка**. Эта команда обеспечивает доступ к справке по активному узлу.
