# <a name="archiving-server-scope-node"></a>Узел области сервера архивации
 
В построителье топологий можно выполнять задачи, связанные с сервером архива, щелкнув правой кнопкой мыши пул в дереве консоли и выбрав "Изменить **свойства".** На  странице "Изменение свойств" можно изменить полное доменное имя сервера архива, базы данных архива SQL Server и файлового хранилище архива.
