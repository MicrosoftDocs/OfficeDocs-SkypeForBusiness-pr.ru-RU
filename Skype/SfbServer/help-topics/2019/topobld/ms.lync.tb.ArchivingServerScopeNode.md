# <a name="archiving-server-scope-node"></a>Узел области применения сервера архивации
 
В построителе топологии вы можете выполнять задачи, связанные с сервером, на котором выполняется архивация, щелкнув правой кнопкой мыши пул в дереве консоли и выбрав команду **изменить свойства**. На странице " **изменение свойств** " вы можете изменить полное доменное имя (FQDN) сервера, на котором выполняется архивация, базу данных сервера SQL Server и архивное хранилище файлов.
