# <a name="add-edge-server-external-ip-2010"></a>Добавление пограничного сервера внешний IP-адрес сервера 2010
 
Можно настроить свойства на этой странице для указания пограничного сервера на внешние расположения службы пограничного сервера доступа, служба пограничного сервера конференц-связи Web и A / V Edge службы.
  
- В области **внешние адреса IPv4**введите внешние IPv4-адреса пограничного сервера в **пограничной службы доступа**, **служба пограничного сервера конференц-связи Web**и **A / V Edge службы**.
    
- Нажмите **кнопку ОК** , чтобы сохранить и сохранить изменения в документе топологии.
    
- Нажмите кнопку **Отмена** , чтобы отменить изменения и закрыть мастер **Добавление сервера пограничного пула** .
    
- Нажмите кнопку **Справка** , чтобы прочитать этот раздел справки.
    
Для получения дополнительных сведений о добавлении пограничных серверов в существующей топологии просмотрите [Определение топология с пограничным](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Сведения о миграции пограничных серверов, в разделе [Этап 5: Добавление сервера Lync Server 2013 Preview пограничного сервера для пилотного пула](http://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
