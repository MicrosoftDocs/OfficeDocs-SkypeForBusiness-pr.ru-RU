# <a name="add-default-sip-domain"></a>Добавление домена SIP по умолчанию
 
Укажите основной домен SIP, который должно поддерживать ваше развертывание. Основной домен SIP может быть таким же или отличается от домена Active Directory Domain Services. Например, вы можете использовать домен Active Directory fabrikam.net и указать основной домен SIP contoso.net.
