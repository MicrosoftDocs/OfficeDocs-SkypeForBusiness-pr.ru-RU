# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Расширитель общих настроек устройства для обеспечения связи в филиалах для Lync Server 2010
 
Чтобы изменить свойства для обеспечения связи в филиалах или для обеспечения связи в филиалах, в разделе **Общие**настройки:
  
- **Полное доменное имя**: Введите полное доменное имя для обеспечения связи в филиалах или для обеспечения связи в филиалах.
    
    **Использовать все настроенные IP-адреса** с помощью IP-адресов, настроенных для обеспечения связи в филиалах или сервера для обеспечения связи в филиалах для всех целей.
    
    **Ограничить использование службы выбранными IP-адресами**. Пользователем задаются различные IP-адреса для сервера и для связи с ТСОП.
    
    **Основной IP-адрес**. IP-адрес определяется и настраивается для всех задач, кроме функций, связанных с ТСОП.
    
    **IP-адрес для ТСОП**. IP-адрес, связанный с функциями ТСОП.
    
- Можно настроить **связи** , чтобы убедиться, что другой сервер роли настроена и связана с устройства для обеспечения связи в филиалах или для обеспечения связи в филиалах.
    
    **Связанный сервер архивации** Выберите в списке сервер архивации, которую требуется связать с устройства для обеспечения связи в филиалах или для обеспечения связи в филиалах
    
    Нажмите кнопку **Создать** , если вы еще не создали сервер архивации, которую требуется связать с этим для обеспечения связи в филиалах или сервера для обеспечения связи в филиалах
    
    **Связывание сервера мониторинга** Выберите в списке сервер мониторинга, который требуется связать с устройства для обеспечения связи в филиалах или для обеспечения связи в филиалах
    
    Нажмите кнопку **Создать** , если вы еще не создали сервер мониторинга, которую требуется связать с этим для обеспечения связи в филиалах или сервера для обеспечения связи в филиалах
    
    **Связать пограничный пул (для компонентов мультимедиа)** Выберите в списке пограничный сервер или пограничный пул, который требуется связать с устройства для обеспечения связи в филиалах или для обеспечения связи в филиалах
    
 **ОК**. Принятие и фиксация изменений, внесенных в диалоговом окне.
  
 **Отмена**. Отмена изменений и закрытие диалогового окна.
  
 **Справка**. Отображение этого экрана справки.
