# <a name="shared-components-scope-node"></a>Узел области применения общих компонентов
 
Вы можете щелкнуть **Общие компоненты** в дереве иерархии построителя топологии, чтобы создать новые общие компоненты или получить доступ к параметрам уровня топологии.
  
![Узел области применения общих компонентов](../../../media/Shared_Components_Scope_Node.jpg)
  
Нажмите кнопку **создать** , чтобы создать новый хранилище файлов, новый шлюз IP/КТСОП, новый сервер Office Web Apps, новое хранилище SQL Server или новую магистральную сеть.
  
> [!NOTE]
> Ознакомьтесь с разделами справки по каждому из этих вариантов. 
  
![Узел области применения общих компонентов](../../../media/Shared_Components_Scope_NodeB.jpg)
  
Выберите пункт **топология** , чтобы открыть новый документ с топологией, и нажмите кнопку Параметры. Открыть существующий документ топологии, скачать текущую топологию из хранилища центрального управления, сохранить копию текущего документа, опубликовать текущую топологию, установить базу данных, чтобы создать текущую конфигурацию базы данных, Объедините Office Communications Server 2007 R2 в текущее развертывание или удалите развертывание.
  
![Узел области применения общих компонентов C](../../../media/Shared_Components_Scope_NodeC.jpg)
