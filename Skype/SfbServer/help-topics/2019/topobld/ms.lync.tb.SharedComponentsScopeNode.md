# <a name="shared-components-scope-node"></a>Узел области применения общих компонентов
 
Щелкните правой кнопкой **Общие компоненты** в иерархическом дереве построителя топологий создавать новые общие компоненты или получить доступ к параметрам уровня топологии.
  
![Узел области применения общих компонентов](../../../media/Shared_Components_Scope_Node.jpg)
  
Выберите **Создать** , чтобы создать новое хранилище файлов, новый шлюз IP/ТСОП, новый сервер Office Web Apps, новое хранилище SQL Server или новой линии связи.
  
> [!NOTE]
> Ознакомьтесь с разделами справки для каждого из этих параметров. 
  
![Узел области применения общих компонентов](../../../media/Shared_Components_Scope_NodeB.jpg)
  
Выбор **топологии** , выберите команду Параметры, чтобы открыть новый документ топологии. Откройте существующий документ топологии, загрузите текущей топологии из центрального хранилища управления, сохранить копию текущего документа топологии, опубликовать в текущем документе топологии, установка базы данных, чтобы создать настроенный в данный момент базы данных конфигурации Объединить Office Communications Server 2007 R2 в текущего развертывания или удаления развертывания.
  
![Узел области применения общих компонентов C](../../../media/Shared_Components_Scope_NodeC.jpg)
