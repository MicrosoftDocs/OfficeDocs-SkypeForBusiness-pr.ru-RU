# <a name="av-mcu-general-settings-expander"></a>Расширитель общих параметров AV MCU
 
Чтобы изменить свойства пула серверов аудио- и видеоконференций, которые уже были определены, можно выполнить приведенные ниже действия.
  
## <a name="general-settings"></a>Общие параметры

Обновите полное доменное имя пула серверов аудио- и видеоконференций, указав имя пула в соответствии с DNS.
  
## <a name="av-conferencing-service"></a>Служба аудио- и видеоконференций

Чтобы сопоставить пул серверов аудио- и видеоконференций, выберите пул переднего плана или пулы, которые хотите сопоставить с пулом серверов аудио- и видеоконференций.
  
> [!NOTE]
> Если на панели **Выберите пулы переднего плана, которые будут связаны с пулом аудио- и видеоконференций** не отображается интерфейсных пулов, значит сервер аудио- и видеоконференций совмещен с интерфейсным пулом или компонент "Конференции" не было добавлен в разделе **Компоненты и функции** свойств интерфейсного пула.
  

После выполнения операции нажмите кнопку **ОК**, чтобы завершить изменение настроек пула серверов аудио- и видеоконференций. Нажмите кнопку **Отмена**, чтобы отменить все изменения.
