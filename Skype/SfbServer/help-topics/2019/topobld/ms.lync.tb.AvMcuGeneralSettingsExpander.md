# <a name="av-mcu-general-settings-expander"></a>Расширитель общих настроек AV MCU
 
Чтобы изменить свойства для пула серверов конференц-связи с уже определенными интерфейсами, вы можете сделать следующее:
  
## <a name="general-settings"></a>Общие настройки

Обновите полное доменное имя (FQDN) пула серверов доменных конференций A/V, введя имя пула, как оно определено в службе доменных имен (DNS).
  
## <a name="av-conferencing-service"></a>аудио- и видеоконференций

Чтобы связать пул серверов для конференц-связи A/V, выберите пул или пулы, которые нужно связать с пулом серверов для конференц-связи "A/V".
  
> [!NOTE]
> Если ни один из пулов интерфейсов не отображается в списке **выбрать интерфейсных пулов для связи с этой** областью, вы либо Выбери сервер конференц-связи с внешним интерфейсом, либо не выбирали конференции в разделе **функции и функциональность** в свойствах пула переднего плана.
  

По завершении нажмите кнопку **ОК** , чтобы завершить редактирование параметров пула серверов конференц-связи A/V. Для отмены изменений нажмите кнопку **Отмена**.
