# <a name="add-director-association-for-lync-server-2010"></a>Добавление связи директора для Lync Server 2010
 
Можно определить ли директора или Director pool будет иметь сервер мониторинга связанных с ним. Установите флажок, чтобы связать сервер мониторинга с директора или Director pool.
  
> [!TIP]
> Если вы еще не создана сервер мониторинга базы данных (и дополнительный экземпляр), пользователю предоставляется возможность для его создания более поздней версии, если вы установите этот флажок, чтобы связать сервер мониторинга с директора или Director pool.
