# <a name="add-edge-server-internal-ip-2010"></a>Добавление пограничного сервера внутренний IP-адрес сервера 2010
 
Используйте эту страницу для определения внутреннего IP-адреса и внутреннего полное доменное имя (FQDN) для пограничного сервера.
  
Что полное доменное имя, которое указывается должен быть идентичен имя компьютера, который настроен на сервере. По умолчанию имя компьютера, который не присоединен к домену, является коротким, а не полным доменным именем. Построитель топологии использует полные доменные имена, а не короткие имена. Таким образом необходимо настроить суффикс доменных имен (DNS) на имя компьютера, развертываемом в качестве пограничного сервера, не присоединенного к домену. Дополнительные сведения о добавлении DNS-суффикс для имени компьютера видеть [Настроить DNS для поддержки пограничной топологии](http://technet.microsoft.com/library/955493e6-aa29-424d-bb81-1ef87b3b15e3.aspx).
