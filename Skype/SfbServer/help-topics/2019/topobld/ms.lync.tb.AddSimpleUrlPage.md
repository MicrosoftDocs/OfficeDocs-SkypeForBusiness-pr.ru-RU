# <a name="add-simple-url"></a>Добавление простого URL-адреса
 
Чтобы добавить дополнительные простые URL-адреса в определение развертывания, укажите новый простой URL-адрес. Формат URL-адреса http:// _\<simple URL type\> . \<domain name\>_ или _http:// \<domain name\> / \<simple URL type\>_, где простой тип URL-адреса является одним из следующих:
  
- Meet
    
- Номер доступа
    
> [!IMPORTANT]
> В любое время могут быть активными один простой URL-адреса номера доступа и один простой URL-адрес собрания. Добавление нового простого URL-адреса в качестве активного приведет к использованию во всех последующих запросах этого нового простого URL-адреса. Неактивный простой URL-адрес используется для обработки конференций и других запросов, выполненных, когда был активен более старый простой URL-адрес.
