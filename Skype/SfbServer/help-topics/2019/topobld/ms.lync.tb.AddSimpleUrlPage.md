# <a name="add-simple-url"></a>Добавление простого URL-адреса
 
Чтобы добавить в определение развертывания дополнительные простые URL-адреса, укажите URL-адрес нового простого URL-адреса. URL-адрес имеет формат, _http://\<простой тип\>URL-\< адреса. доменное\> имя_ или _\<доменное\>/\<имя http://URL\>-адрес простого типа_, где простой URL-адрес является одним из следующих:
  
- Подходит
    
- Набрать номер
    
> [!IMPORTANT]
> Вы можете в любое время использовать один простой URL-адрес и один из них. Если добавить новый простой URL-адрес и сделать его активным, все будущие запросы будут задаваться для использования нового простого URL-адреса. Вы оставляете неактивный простой URL-адрес для обработки конференций и других запросов, сделанных в момент активации старого простого URL-адреса.
