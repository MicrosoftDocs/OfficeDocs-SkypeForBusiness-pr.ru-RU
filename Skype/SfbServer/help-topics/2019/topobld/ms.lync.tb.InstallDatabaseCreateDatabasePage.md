# <a name="install-and-create-databases"></a>Установка и создание баз данных

Можно выбрать базу данных, которую следует создать для используемого развертывания. По умолчанию база данных будет создаваться на определенном SQL Server определенном сайте и автоматически развертывать и настраивать файлы баз данных на основе SQL Server, на который вы ставите базы данных.

 **Выберите базы данных, которые хотите создать**. Установите флажок напротив любой базы данных, которую следует развернуть и настроить. Установите флажок напротив тех баз данных, которые будут развернуты.

> [!CAUTION]
> Необходимо SQL Server для экземпляра (если таково) и необходимо открыть порты брандмауэра для размещения экземпляра, в который развертываются базы данных. Подробные сведения см. [в SQL Server](/previous-versions/office/lync-server-2013/lync-server-2013-configure-sql-server-for-lync-server)

 **Расширенный:** нажмите кнопку SQL Server кнопку **Advanced,** чтобы выбрать варианты расположения файлов базы данных на SQL Server. Дополнительные сведения о размещении файлов с расширенными базами данных см. в материале Установка базы данных с помощью оболочки управления [Lync Server](/previous-versions/office/lync-server-2013/lync-server-2013-database-installation-using-lync-server-management-shell)

 **Назад**. Нажатие этой кнопки служит для перехода на предыдущий экран (иногда может быть недоступной в зависимости от способа перехода к этому диалоговому окну).

 **Далее**. Нажатие этой кнопки фиксирует выбор в текущем диалоговом окне и позволяет перейти к следующему диалоговому окну для настройки дополнительных сведений

 **Отмена.** Нажав эту кнопку, выйдите из конфигурации и откажитесь от изменений. Некоторые, но не все экраны конфигурации подскажут вам, если вы хотите выйти и отменить изменения. Выбор **Да** закроет текущую конфигурацию и закроет текущую конфигурацию и возвращает вас в Topology Builder. Выбор **"Нет"** возвращает вас в текущий диалоговое окно конфигурации и позволяет продолжить настройку.

 **Справка**. Нажмите кнопку **Справка**, чтобы открыть справочные сведения, связанные с текущим диалоговым окном конфигурации.
