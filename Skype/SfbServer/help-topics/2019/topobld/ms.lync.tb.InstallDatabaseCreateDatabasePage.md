# <a name="install-and-create-databases"></a>Установка и создание баз данных

Можно выбрать базу данных, которую следует создать для используемого развертывания. По умолчанию база данных будет создана на определенном SQL Server определенном сайте и будет автоматически развертывать и настраивать файлы баз данных в зависимости от SQL Server, в которой размещаются базы данных.

 **Выберите базы данных, которые хотите создать**. Установите флажок напротив любой базы данных, которую следует развернуть и настроить. Установите флажок напротив тех баз данных, которые будут развернуты.

> [!CAUTION]
> Для SQL Server уже должны быть настроены порты для экземпляра (при их найме), а для размещения экземпляра, в который развертываются базы данных, необходимо открыть порты брандмауэра. Подробные сведения [см. в SQL Server](https://technet.microsoft.com/library/375e5cc4-e436-46dc-9b02-5063f35cdcc1.aspx)

 **Дополнительные.** Нажмите кнопку SQL Server кнопку "Дополнительные", чтобы выбрать параметры расположения файлов базы данных в SQL Server.  Дополнительные сведения о размещении файлов базы данных см. в руководстве по установке базы данных [с помощью Lync Server Management Shell](https://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx)

 **Назад**. Нажатие этой кнопки служит для перехода на предыдущий экран (иногда может быть недоступной в зависимости от способа перехода к этому диалоговому окну).

 **Далее**. Нажатие этой кнопки фиксирует выбор в текущем диалоговом окне и позволяет перейти к следующему диалоговому окну для настройки дополнительных сведений

 **Отмена:** нажатие этой кнопки закимет конфигурацию и отменит изменения. На некоторых, но не всех экранах конфигурации будет предложено выйти и отменить изменения. Если выбрать **"Да",** текущая конфигурация будет закрыта, а текущая конфигурация будет закрыта и возвращена в построитель топологий. Если выбрать **"Нет",** вы вернетесь в текущее диалоговое окно конфигурации и сможете продолжить настройку.

 **Справка**. Нажмите кнопку **Справка**, чтобы открыть справочные сведения, связанные с текущим диалоговым окном конфигурации.
