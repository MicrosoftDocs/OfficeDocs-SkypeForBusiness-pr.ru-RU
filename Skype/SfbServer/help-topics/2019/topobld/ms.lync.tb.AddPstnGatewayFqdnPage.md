# <a name="define-the-pstn-gateway-fqdn"></a>Определение полного доменного имени шлюза ТСОП

Вы можете указать полное доменное имя (FQDN) для шлюза КОММУТИРУЕМой телефонной сети, который будет подключаться к вашим корпоративным голосовым интерфейсам через сеть PSTN или частный обмен филиалами (АТС).

Дополнительные сведения можно найти [в разделе Компоненты подключения PSTN в Skype для бизнеса Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
