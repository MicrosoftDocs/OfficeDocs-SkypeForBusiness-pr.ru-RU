# <a name="add-mediation-server-nexthop"></a>Добавление следующего перехода для серверов-посредников
 
Чтобы определить следующий переход для определяемого сервера-посредника, выберите из выпадаемого списка пула следующего хмеля сервер выпуск Enterprise или выпуск Standard, который будет выступать в качестве следующего перехода для сервера-посредника.  Нажмите кнопку **Далее**, чтобы продолжить.
