# <a name="legacy-merge-landing"></a>Устаревшая площадка объединения
 
На странице " **Настройка Edge** " можно настроить старые данные пограничного сервера. Нажмите кнопку **Добавить** , чтобы указать версию устаревшего пограничного сервера для развертывания Office Communications Server. Если у вас нет пограничных серверов в устаревшей среде, нажмите кнопку **Далее** , чтобы продолжить.
