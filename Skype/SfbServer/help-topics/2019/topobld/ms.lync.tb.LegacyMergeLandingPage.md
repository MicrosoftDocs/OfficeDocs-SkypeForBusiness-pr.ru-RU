# <a name="legacy-merge-landing"></a>Устаревшая площадка объединения
 
На странице **Настройки пограничного сервера** можно настроить данные устаревшего пограничного сервера. Нажмите кнопку **Добавить** , чтобы указать версию устаревшего пограничного сервера развертывания Office Communications Server. Если у вас все пограничные серверы в старой среды, нажмите кнопку **Далее** для продолжения.
