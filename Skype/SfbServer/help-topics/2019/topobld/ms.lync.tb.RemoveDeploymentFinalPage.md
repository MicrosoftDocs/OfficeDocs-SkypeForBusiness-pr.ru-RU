# <a name="remove-deployment"></a>Удаление развертывания
 
Нажмите **кнопку Готово** , чтобы создать пустую топологию в качестве первого шага по выводу из эксплуатации всего развертывания.
  
> [!CAUTION]
> Это действие невозможно отменить.
