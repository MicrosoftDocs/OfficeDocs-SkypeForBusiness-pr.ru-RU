# <a name="edge-server-add-machine"></a>Добавление компьютера пограничного сервера
 
Укажите полное доменное имя (FQDN) каждого компьютера, который требуется добавить в качестве пограничного сервера в этом пуле. После добавления компьютера в список, можно обновить полное доменное имя компьютера, или удалить его из пула, в любое время до публикации топологии. После публикации топологии для изменения полного доменного ИМЕНИ необходимо удаление сервера в построителе топологий, а затем добавляя новый сервер к пулу с новой полным доменным ИМЕНЕМ. Для получения дополнительных сведений о разработке топологии для пограничного пула просмотрите [Определение топология с пограничным](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) в документации по развертыванию пограничного сервера.
