# <a name="legacy-merge-add-machine"></a>Добавление компьютера при объединении с устаревшей версией
 
Чтобы определить компьютеры, в состав которого входит пул Edge Server, введите полное доменное имя (FQDN) по крайней мере одного компьютера, который является частью устаревшего консолидированного пула Edge с сбалансированной нагрузкой.
