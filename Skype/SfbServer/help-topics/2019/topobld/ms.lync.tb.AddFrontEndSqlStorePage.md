# <a name="add-front-end-sql-server-store"></a>Добавление хранилища SQL Server переднего плана
 
Развертывание сервера Standard Edition автоматически устанавливает требуется Microsoft SQL Server Express программное обеспечение баз данных и базы данных SQL Server. Таким образом предварительно все параметры, а не удается внести изменения в конфигурацию по умолчанию.
  
Пул переднего плана Enterprise Edition server развертывания требует с поддерживаемой 64-разрядной версией программного обеспечения баз данных SQL Server для серверной базы данных. Можно выбрать ранее заданных базы данных SQL Server, которое будет использоваться для серверной базы данных, или определите новую базу данных SQL Server, указав полное доменное имя (FQDN) сервера, на котором размещаются базы данных SQL Server и экземпляра SQL S Проверка, который вы хотите использовать для новой базы данных SQL Server (который может быть экземпляр по умолчанию или именованный экземпляр, который указан). Также можно выбрать включить зеркальное отображение хранилища SQL Server и укажите следящий сервер зеркального отображения для автоматической отработки отказа.
  
Для получения дополнительных сведений о SQL Server поддерживает, содержатся в документации по поддержке [программного обеспечения баз данных и кластеризации поддержки](http://technet.microsoft.com/library/e05d0032-bbea-4e61-987d-d07b1c045fd5.aspx) . Для получения дополнительных сведений о настройке SQL Server для серверной базы данных содержатся в документации по развертыванию [Configure SQL Server](http://technet.microsoft.com/library/375e5cc4-e436-46dc-9b02-5063f35cdcc1.aspx) .
  
> [!NOTE]
> Если учетная запись, используемая для публикации топологии имеет соответствующие права и разрешения, вы можете создать внутреннюю базу данных (связи реального времени (RTC)) при публикации топологии. Можно также создать базу данных более поздних версий, включая как часть процесса установки. 
  
> [!NOTE]
> Установка и развертывание базы данных на сервере под управлением SQL Server для развертывания выпуска Enterprise Edition, необходимо быть членом группы системных администраторов SQL Server для сервера под управлением SQL Server, где выполняется установка файлов базы данных. Если вы не должна быть членом группы системных администраторов SQL Server, необходимо запрос на добавление в группу до развертывания файлов базы данных. Если невозможно выполнить должна быть членом группы системных администраторов, предоставьте администратору базы данных SQL Server с помощью сценария для настройки и развертывания базы данных. Для получения дополнительных сведений о правах пользователей и разрешения, которые необходимы для выполнения процедур, описанных просмотрите [Разрешения на развертывание для SQL Server](http://technet.microsoft.com/library/56ea0c02-bcf5-4d45-aa13-570531c29074.aspx).
