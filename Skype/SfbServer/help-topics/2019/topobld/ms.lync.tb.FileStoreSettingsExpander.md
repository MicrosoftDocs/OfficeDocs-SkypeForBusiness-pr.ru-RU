# <a name="file-store-settings-expander"></a>Расширитель параметров хранилища файлов
 
Чтобы изменить свойства определенных файловых хранилищ, выполните следующие действия.
  
Можно определить имя новой общей папки файлов. Необходимо создать общую папку на сервере, указанном в свойстве полного доменного имени файлового сервера.
  
> [!NOTE]
> Невозможно изменить полное доменное имя сервера, указанного в свойстве полного доменного имени файлового сервера. Если необходимо разместить общую папку файлов на другом сервере, необходимо создать новую запись общей папки файлов.
