# <a name="add-edge-server-support"></a>Добавление поддержки для пограничного сервера

Вы можете добавить поддержку внешнего доступа пользователей при развертывании начальной топологии или позже. Дополнительные сведения о добавлении пограничных серверов к существующей топологии см. в разделе [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) документации по развертыванию пограничного сервера.
