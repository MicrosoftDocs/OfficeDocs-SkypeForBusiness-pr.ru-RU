# <a name="certificate-request-intro"></a>Запрос сертификата (введение)
 
Чтобы продолжить процесс запроса сертификата, нажмите кнопку **Далее**. Чтобы завершить процесс без изменения системы, нажмите кнопку **Отмена**.
