# <a name="certificate-request-additional-subject-alternate-names"></a>Запрос сертификата (дополнительные альтернативные имена субъектов)
 
С помощью функции **Настройка дополнительных альтернативных имен субъектов** можно добавить в сертификат дополнительные альтернативные имена субъектов перед завершением и отправкой запроса сертификата. Чтобы добавить альтернативное имя субъекта, введите полное доменное имя и нажмите кнопку **Добавить**. Если необходимо удалить добавленное имя, выделите его и нажмите кнопку **Удалить**. Чтобы удалить все добавленные альтернативные имена субъектов, нажмите кнопку **Очистить все**.
  
> [!NOTE]
> Кнопки **Remove** или **Clear All** работают только на записях SAN, добавленных на этой странице. Они не действуют на имена, имеющиеся на данный момент в сертификате.
