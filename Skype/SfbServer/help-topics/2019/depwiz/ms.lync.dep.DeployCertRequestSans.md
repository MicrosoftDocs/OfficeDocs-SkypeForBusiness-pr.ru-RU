# <a name="certificate-request-additional-subject-alternate-names"></a><span data-ttu-id="d71f1-106">Запрос сертификата (дополнительные альтернативные имена субъектов)</span><span class="sxs-lookup"><span data-stu-id="d71f1-106">Certificate Request (Additional Subject Alternate Names)</span></span>
 
<span data-ttu-id="d71f1-p102">С помощью функции **Настройка дополнительных альтернативных имен субъектов** можно добавить в сертификат дополнительные альтернативные имена субъектов перед завершением и отправкой запроса сертификата. Чтобы добавить альтернативное имя субъекта, введите полное доменное имя и нажмите кнопку **Добавить**. Если необходимо удалить добавленное имя, выделите его и нажмите кнопку **Удалить**. Чтобы удалить все добавленные альтернативные имена субъектов, нажмите кнопку **Очистить все**.</span><span class="sxs-lookup"><span data-stu-id="d71f1-p102">The **Configure Additional Subject Alternate Names** option provides the opportunity to add any additional subject alternate names (SANs) to the certificate before finalizing and requesting the certificate. To add the SAN, type the fully qualified domain name (FQDN) of the SAN and then click **Add**. If you have added a SAN that you must delete, highlight the SAN entry and click **Remove**. To remove all SAN entries that you have added, click **Clear All**.</span></span>
  
> [!NOTE]
> <span data-ttu-id="d71f1-111">Кнопки **"Удалить"** или **"Очистить все"** работают только для записей SAN, добавленных на этой странице.</span><span class="sxs-lookup"><span data-stu-id="d71f1-111">The **Remove** or **Clear All** buttons work only on SAN entries that are added on this page.</span></span> <span data-ttu-id="d71f1-112">Они не действуют на имена, имеющиеся на данный момент в сертификате.</span><span class="sxs-lookup"><span data-stu-id="d71f1-112">The buttons will not affect SAN entries that are currently on the certificate.</span></span>
