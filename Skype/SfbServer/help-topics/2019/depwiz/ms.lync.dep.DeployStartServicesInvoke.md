# <a name="start-services-invoke"></a>Запуск служб (вызов)
 
Сводная панорама области  команд выполнения отображает состояние задач, выдающихся для запуска служб для Skype для бизнеса Server роли, который вы развертываете. В представленной сводке отсутствует индикация запуска служб в реальном времени. Некоторые из Skype для бизнеса Server служб могут занять больше времени, чтобы приступить к начальному процессу запуска. Задачи дают команду на запуск, но не ожидают результата, чтобы определить, успешно ли запустилась служба. Если вам необходимо отслеживать запуск службы и ее состояние, используйте консоль управления (MMC) для служб Windows.
  
Под панелью задач находится раскрывающийся список, в котором отображается файл журнала **Запустить службы**. Для просмотра файла журнала нажмите кнопку **Просмотреть журнал**. Чтобы завершить задачу, нажмите кнопку **Готово**.
