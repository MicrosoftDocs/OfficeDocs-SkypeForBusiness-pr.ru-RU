# <a name="remove-certificate-invoke"></a>Удаление сертификата (вызов)
 
На странице **Выполнение команд** предоставляются сведения об удалении сертификата. Просмотрите сводную информацию по удалению сертификата. Убедитесь, что файл журнала "Удаление сертификата" показан в раскрывающемся списке, затем нажмите кнопку **Просмотреть журнал** для просмотра сведений об удалении сертификата. Завершив просмотр сводки и файла журнала, нажмите кнопку **Готово**.
