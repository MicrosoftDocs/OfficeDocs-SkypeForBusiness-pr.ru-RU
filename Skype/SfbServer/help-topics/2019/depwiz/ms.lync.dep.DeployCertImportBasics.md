# <a name="import-certificate-intro"></a>Импорт сертификата (введение)
 
Чтобы импортировать сертификат, необходимо указать путь к файлу сертификата. В поле **Выбор сертификата файла** можно либо введите полный путь и имя файла или нажмите кнопку **Обзор** и перейдите в расположение путь и имя файла (как правило, .p7b, PFX-файл или CER-файл).
  
Если сертификат содержит закрытый ключ, установите флажок, **файл сертификата содержит закрытый ключ сертификата**. Если этот флажок установлен, ввода текста **пароля** включено. Если у вас есть сертификат с закрытым ключом, связанные с ним, пароль обычно размещается на закрытый ключ при создании сертификата. Введите пароль для закрытого ключа, чтобы разрешить сертификат и закрытый ключ к импорту в хранилище сертификатов. Когда вы предоставили сведения для путь к файлу сертификата, а при необходимости и пароль для закрытого ключа, при необходимости, нажмите кнопку **Далее**.
  
> [!IMPORTANT]
> Если вы не знаете пароль для закрытого ключа, импорт не выполнится.
