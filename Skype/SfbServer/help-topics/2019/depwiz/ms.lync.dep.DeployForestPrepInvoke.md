# <a name="prepare-current-forrest-invoke"></a>Подготовка текущего леса (вызов)
 
На странице **выполнение команд** отображается сводка задач по мере их выполнения. Посмотрите на экран Сводка и убедитесь, что задачи выполнены успешно и ошибки не обнаружены. Ознакомьтесь с предупреждениями, чтобы убедиться, что они ожидаются. В области сводки вы должны увидеть **состояние задачи: завершено**. В раскрывающемся списке должен быть показан **лес подготовка** файла журнала. Если это не так, выберите файл журнала и щелкните **Просмотреть журнал** для просмотра подробных действий и результатов задач. Нажмите кнопку **Готово** , чтобы завершить подготовку леса.
