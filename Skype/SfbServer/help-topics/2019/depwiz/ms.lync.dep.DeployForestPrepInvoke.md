# <a name="prepare-current-forrest-invoke"></a>Подготовка текущего леса (вызов)
 
На странице **Выполнение команд** отображаются краткие сведения о задачах, как их выполнения. Просмотрите сводку и убедитесь в том, что задачи было выполнено успешно, и что нет ошибок. Просмотрите предупреждения, чтобы убедиться в том, что они должны. В разделе области сводки должно появиться **состояние задачи: завершено**. В раскрывающемся списке должны отображаться файл журнала **Подготовка леса** . Если он не установлен, выберите файл журнала и нажмите кнопку **Просмотреть журнал** для просмотра подробного действия и результаты задачи. Нажмите кнопку **Готово** для завершения подготовки леса.
