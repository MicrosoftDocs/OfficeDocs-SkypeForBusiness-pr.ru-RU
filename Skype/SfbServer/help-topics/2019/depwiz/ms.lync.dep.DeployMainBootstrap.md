# <a name="setup-or-remove-skype-for-business-server-components"></a>Установка или удаление компонентов Skype для бизнеса Server
 
Для установки и активации, или деактивация и удаление Скайп для компонентов Business Server, используйте **Шаг 2: программа установки или удаления компонентов сервера Скайп**. Необходимо выполнить вход в качестве локального администратора на компьютере, установка или изменение и его необходимо смогут прочитать доменных служб Active Directory пользователей и групп в текущем домене. Чтобы начать, выберите команду **выполнить**. При этом определение топологии на основе хранилища централизованного управления доступен для чтения. Необходимые программные компоненты устанавливаются и настраиваются в соответствии с ролью, как определено в центральном хранилище управления. После завершения установки просмотрите сводку и нажмите кнопку **Готово**.
  
> [!TIP]
> Если вам нужно просмотреть файлы журнала, созданных с помощью мастера развертывания, их можно найти на компьютере, на котором запускался мастер развертывания в каталоге пользователей в Active Directory пользователя, выполнившего действие. Например, если пользователь вошел в систему от имени администратора домена в домене Contoso.net, файлов журнала, находятся в: > C:\Users\Administrator.Contoso\AppData\Local\Temp 
  
> [!NOTE]
> Если Скайп был установлен ранее для компонентов Business Server на данном компьютере, мастер развертывания будет распознавать это и кнопки на шаге 2 отображается как **Выполнить еще раз**. Она позволяет выполнять шаг столько раз, сколько необходимо для правильной настройки или модификации сервера.
