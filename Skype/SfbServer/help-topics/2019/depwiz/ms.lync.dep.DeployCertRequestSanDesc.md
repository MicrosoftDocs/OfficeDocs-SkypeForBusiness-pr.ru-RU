# <a name="certificate-request-subject-alternate-names"></a>Запрос сертификата (альтернативные имена субъектов)
 
На странице **Имя субъекта / альтернативные имена субъектов** представлена сводная информация по имени субъекта и всех альтернативных именах субъектов, которые будут указаны в сертификате. Имена, представленные для имени субъекта и альтернативных имен субъекта, собираются из документа определения, так как он был создан при определении развертывания в Topology Builder.
  
Если необходимо внести изменения в имя субъекта или  любое из альтернативных имен субъекта, нажмите кнопку Отмена, чтобы выйти из мастера запроса сертификата и перезаписи Topology Builder. Затем опубликуйте новую конфигурацию с правильно определенными именами.
