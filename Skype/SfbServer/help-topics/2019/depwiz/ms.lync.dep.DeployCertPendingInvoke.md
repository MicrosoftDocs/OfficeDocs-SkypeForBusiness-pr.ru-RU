# <a name="pending-certificate-invoke"></a>Отложенный сертификат (вызов)
 
На странице **Выполнение команд** отображается состояние команды, которые извлекаются отложенный сертификат. После завершения задачи доступен файл журнала **Извлечения ожидающих установки сертификатов** . Нажмите кнопку **Просмотреть журнал** обработки и уведомление о состоянии задач. Для продолжения нажмите кнопку **Далее**.
  
> [!IMPORTANT]
> Если сертификат еще не был утвержден администратором центра сертификации, файл журнала извлечения ожидающего запроса указывает, что существуют ожидающие запросы. Если администратор центра сертификации отклонил запрос, файл журнала будет указывающие на сбой и сообщает ошибки, это означает, что запрос отклонен и имя пользователя человека, который запрос отклонен.
