# <a name="certificate-assignment-from-request"></a>Назначение сертификата (из запроса)
 
На **странице "Назначение** сертификата" запускается процесс назначения сертификата для запрашиваемого использования на сервере ролей. Для просмотра сертификата щелкните **Просмотреть сведения о сертификате**. Можно выбрать вкладку **"Общие",** **"Сведения"** или **"Путь сертификации".** На **вкладке** "Общие" отображаются основные сведения о сертификате, такие как выдача сертификата и состояние закрытого ключа. На вкладке "Сведения" отображается такая информация о сертификате, как имя субъекта, альтернативные имена субъектов и прочие сведения. На **вкладке** "Путь сертификации" будет отображаться "цепочка доверия" сертификата через любые выдавлив или промежуточные органы сертификации (если они присутствуют) в корневой.
  
Нажмите кнопку **Далее**, чтобы продолжить.
