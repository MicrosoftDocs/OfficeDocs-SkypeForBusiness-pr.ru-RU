# <a name="certificate-request-certificate-authority-account"></a>Запрос сертификата (учетная запись центра сертификации)
 
Для отправки запроса ваш центр сертификации может потребовать другие учетные данные, отличающиеся от тех, с которыми вы вошли в систему. Чтобы разрешить запрос на сертификат в качестве другого пользователя, выберите в качестве альтернативного учетных данных альтернативные учетные данные для сертификационного **органа,** а затем введите имя пользователя или имя пользователя домена для пользователя, который может запросить  \   сертификат. В текстовом поле **Пароль** введите пароль для указанного пользователя. Имя пользователя и пароль затем отправляются в локальный центр сертификации как часть процесса запроса сертификата, но не в самом запросе сертификата.
