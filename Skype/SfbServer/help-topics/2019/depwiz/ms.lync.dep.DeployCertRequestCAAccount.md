# <a name="certificate-request-certificate-authority-account"></a>Запрос сертификата (учетная запись центра сертификации)
 
Для отправки запроса ваш центр сертификации может потребовать другие учетные данные, отличающиеся от тех, с которыми вы вошли в систему. Чтобы разрешить запрос сертификата как другому пользователю, выберите поле Укажите альтернативные учетные данные для органа сертификации **, а** затем введите имя пользователя или _domainusername_ \  для пользователя, который может запросить сертификат. В текстовом поле **Пароль** введите пароль для указанного пользователя. Имя пользователя и пароль затем отправляются в локальный центр сертификации как часть процесса запроса сертификата, но не в самом запросе сертификата.
