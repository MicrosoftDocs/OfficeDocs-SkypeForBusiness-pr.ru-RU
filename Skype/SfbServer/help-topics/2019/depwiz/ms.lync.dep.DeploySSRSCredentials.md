# <a name="sql-server-reporting-services-credentials"></a>Службы SQL Server Reporting Services (учетные данные)
 
Чтобы указать учетные данные для обращения служб SQL Server Reporting Services (SSRS) к базе данных мониторинга, сделайте следующее:
  
На странице **Определение учетных данных** укажите учетные данные пользователя, которые будут использоваться для обращения к базе данных мониторинга (QoEMetrics и LcsCDR). 
  
> [!NOTE]
> Подробные сведения см. в материале "Указание учетных данных и сведений о подключениях для источников данных отчетов (SSRS)" [https://go.microsoft.com/fwlink/p/?LinkId=263431](/sql/reporting-services/report-data/specify-credential-and-connection-information-for-report-data-sources) по ссылке . 
  
Дополнительные сведения о развертывании отчетов служб отчетности для сервера мониторинга см. в материале [Install Monitoring Reports in Skype для бизнеса Server.](../../../deploy/deploy-monitoring/install-monitoring-reports.md)
