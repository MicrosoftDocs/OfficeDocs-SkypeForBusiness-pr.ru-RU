# <a name="install-local-configuration-store-invoke-configure"></a>Вызов установки локального хранилища конфигурации (настройка)
 
Чтобы приступить к установке базы данных, в которой будет храниться локализованная только для чтения копия центра управления, необходимо выбрать между сбором определенной конфигурации, опубликованной с помощью Topology Builder из уже установленного и настроенного центра управления, или чтением заданной конфигурации из других средств массовой информации. Для машины, которая находится во внутренней сети организации, выберите конфигурацию Retrieve автоматически из Центра **управления.**
  
Если реплика центрального хранилища управления устанавливается на пограничный сервер, выберите чтение экспортированной копии документа конфигурации с переносного носителя, такого как флэш-накопитель USB, жесткий диск USB, компакт-диск или другой носитель. 
  
> [!IMPORTANT]
> Если вы устанавливаете локальное хранилище конфигурации на edge Server, сведения о конфигурации должны быть в формате, экспортируемом из центра управления с помощью Windows PowerShell.`Export-CsConfiguration -FileName <ConfigurationFilePath.zip>`
  
Выбрав подходящий параметр, нажмите кнопку **Далее**.
