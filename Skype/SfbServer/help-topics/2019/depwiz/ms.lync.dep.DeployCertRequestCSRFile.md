# <a name="certificate-request-certificate-signing-request"></a>Запрос сертификата (запрос подписи сертификата)
 
Чтобы создать автономный запрос, необходимо сначала создать запрос на подписывание сертификата (CSR). Введите путь и имя файла для CSR в текстовом поле **имя файла** . Вы можете нажать кнопку **Обзор** , чтобы выбрать место для создания CSR. Для продолжения нажмите кнопку **Далее**.
  
Нажмите кнопку " **назад** ", чтобы вернуться к предыдущему экрану. Нажмите кнопку **Отмена** , чтобы выйти, не внося изменений в систему.
  
> [!IMPORTANT]
> Файл, созданный на этом шаге, отправляется в общедоступный центр сертификации (ЦС) в свой внутренний автономный центр, или отправляется администратору ЦС для обработки. Обратите внимание на расположение и имя файла, чтобы можно было получить его и отправить CSR в центр сертификации. По умолчанию используется каталог документов пользователя, выполняющего мастер сертификатов.
