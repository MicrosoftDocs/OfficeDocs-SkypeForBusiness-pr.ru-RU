# <a name="prepare-current-domain-invoke"></a>Подготовка текущего домена (вызов)
 
На странице **Выполнение команд** отображаются краткие сведения о задачах, как их выполнения. Просмотрите сводку и убедитесь в том, что задачи было выполнено успешно, и что нет ошибок. Просмотрите предупреждения, чтобы убедиться в том, что они должны. Ниже области сводки должно появиться **состояние задачи: завершено**. В раскрывающемся списке должны отображаться файл журнала **Подготовка домена** . Если он не установлен, выберите файл журнала и нажмите кнопку **Просмотреть журнал** для просмотра подробного действия и результаты задачи. Нажмите кнопку **Готово** для завершения подготовки домена.
