# <a name="prepare-current-forrest-intro"></a>Подготовка текущего леса (введение)
 
Для запуска задачи **Подготовка леса** нажмите **Далее**. При выполнении задачи **подготовки леса** создаются глобальные параметры и создаются универсальные группы для использования в Skype для бизнеса Server.
