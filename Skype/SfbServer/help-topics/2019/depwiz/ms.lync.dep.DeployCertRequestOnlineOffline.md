# <a name="certificate-request-online-offline"></a>Запрос сертификата (сетевой, автономный)
 
Чтобы создать отложенный или автономный запрос, выберите **Подготовить запрос сейчас, но отправить позже (автономный запрос сертификата)**. Чтобы создать немедленный запрос в локальную службу сертификации (как правило внутренней системы в вашей организации инфраструктуры открытого ключа), выберите **Отправить запрос в локальную службу сертификации немедленно**.
  
Выбрав нужный параметр, нажмите кнопку **Далее**. Чтобы отменить действие без выполнения изменений, нажмите кнопку **Отмена**. Чтобы вернуться на предыдущий шаг, нажмите кнопку **Назад**.
