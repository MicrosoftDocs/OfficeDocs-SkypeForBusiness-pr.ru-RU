# <a name="start-services"></a>Запуск служб
 
Чтобы запустить службы на развернутой роли сервера, нажмите кнопку **выполнить** на **шаге 4: запуск служб**. Вы должны войти в систему в качестве члена локальной группы администраторов или эквивалента. 
  
> [!NOTE]
> Если запущены **службы starting Services** , вы можете повторно запустить ее, нажав кнопку **выполнить еще раз**.
