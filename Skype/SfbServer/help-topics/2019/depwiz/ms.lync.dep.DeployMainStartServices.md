# <a name="start-services"></a>Запуск служб
 
Запуск служб на развернутое server role, нажмите кнопку **запустить** в **Шаг 4: запуск служб**. Необходимо войти в систему как участник локальной группы администраторов или иметь эквивалентные права. 
  
> [!NOTE]
> Если **Запустить службы** уже была запущена, можно перезапустить ее, нажав кнопку **Перезапуск**.
