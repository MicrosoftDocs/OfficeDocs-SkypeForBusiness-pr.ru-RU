# <a name="import-certificate-summary"></a>Импорт сертификата (сводка)
 
На странице **Сводка импорта сертификата** предоставляется такая информация, как имя файла, путь и наличие закрытого ключа. После просмотра сводной информации нажмите кнопку **Далее**, чтобы продолжить.
