# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Установка или удаление компонентов Skype для бизнеса Server (введение)
 
Подготовке установки и активировать или деактивировать и удаления Скайп для компонентов Business Server. Процессы, которые будут выполняться, зависят от текущей конфигурации определения, опубликованного в центральном хранилище управления. Для успешного выполнения задач, вы должны войти в систему как член группы локальных администраторов и, как минимум, необходимо иметь возможность читать доменных служб Active Directory.
  
Нажмите кнопку **Далее** для перехода на экран **Выполнение команд**.
