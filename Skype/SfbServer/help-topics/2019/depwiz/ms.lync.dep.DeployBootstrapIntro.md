# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Установка или удаление компонентов Skype для бизнеса Server (введение)
 
Вы готовитесь к установке и активации или отключению и удалению компонентов Skype для бизнеса Server. Процессы, которые будут выполняться, зависят от текущей конфигурации определения, опубликованного в центральном хранилище управления. Для успешного выполнения задач необходимо войти в систему в качестве члена локальной группы администраторов и, по крайней мере, иметь возможность чтения доменных служб Active Directory.
  
Нажмите кнопку **Далее** для перехода на экран **Выполнение команд**.
