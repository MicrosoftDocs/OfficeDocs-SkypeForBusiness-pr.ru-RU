# <a name="certificate-assignment-invoke"></a>Назначение сертификата (вызов)
 
На странице **выполнение команд** отображаются действия, предпринятые задачами для назначения сертификата серверу. После завершения задач вы можете просмотреть журнал, в котором подробно описан процесс назначения сертификата. Для просмотра файла журнала нажмите кнопку **Просмотреть журнал**.
