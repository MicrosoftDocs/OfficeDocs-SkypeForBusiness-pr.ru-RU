# <a name="pending-certificate-intro"></a>Отложенный сертификат (введение)
 
Для обработки отложенного запроса сертификата в Интернете, то есть запроса, выданного вашему органу сертификации в Интернете, но в административном отношении требуется ручной процесс утверждения, нажмите **кнопку Далее**.
