# <a name="prepare-schema"></a>Подготовка схемы
 
Чтобы подготовить схему для доменных служб Active Directory, вы запускаете шаг подготовки схемы в мастере развертывания Skype для бизнеса Server. Для запуска подготовки схемы нажмите кнопку **Выполнить**. На этапе подготовки схемы выполняется чтение предоставленных файлов определения схемы в каталоге \Program Филес\скипе для Business Server 2019 \ Деплоймент\сетуп на компьютере, на котором запущен мастер развертывания. Эти файлы имеются также на установочном носителе в каталоге \Support\Schema. На шаге "Подготовка схемы" выполняется также расширение схемы и формируется отчет о состоянии процесса. Пользователь получает уведомление о завершении процесса. На экране сводки можно просмотреть журналы процесса. Просмотрите журналы для проверки успешного завершения подготовки.
  
> [!IMPORTANT]
> Для расширения схемы вы должны войти в домен как участник группы администраторов схемы или группы администраторов предприятия. 
  
Классы и атрибуты добавляются для продления схемы доменных служб Active Directory для поддержки серверных объектов, служб и пользователей Skype для бизнеса Server. Перед расширением схемы следует создать резервную копию состояния системы для контроллера домена, выполняющего роль хозяина схемы. 
  
> [!CAUTION]
> Расширение схемы необратимо. Необходимо приложить все усилия для успешного расширения схемы и ограничения возможных последствий в случае его сбоя. Это особенно важно в случае потери связи или любого другого сбоя на сервере. Вы должны выполнить резервное копирование контроллера домена хозяина схемы и полную архивацию службы каталогов Active Directory. 
  
Чтобы выполнить резервное копирование контроллера домена хозяина схемы и полную архивацию службы каталогов Active Directory, выполните указанные ниже действия.
  
1. Отсоедините от сети контроллер домена, содержащий роль хозяина схемы.
    
2. Выполните резервное копирование состояния системы контроллера домена, содержащего хозяина схемы.
    
3. Выполните расширение схемы.
    
4. Если расширение схемы прошло успешно, снова присоедините контроллер домена к сети и убедитесь, что репликация активна и работает.
    
5. В случае неудачи в случае сбоя расширения схемы восстановите состояние системы контроллера домена и Active Directory с помощью резервной копии состояния системы, которую вы сделали раньше.
    
> [!NOTE]
> Если вам нужно просмотреть файлы журнала, созданные с помощью мастера развертывания Skype для бизнеса Server, вы можете найти файлы на компьютере, на котором был запущен мастер развертывания, в каталоге Users пользователя Active Directory, выполнившего шаг. Например, если пользователь выполнил вход в качестве администратора домена в Contoso.net домена, файлы журнала находятся в: К:\усерс\администратор.контосо\аппдата\локал\темп
