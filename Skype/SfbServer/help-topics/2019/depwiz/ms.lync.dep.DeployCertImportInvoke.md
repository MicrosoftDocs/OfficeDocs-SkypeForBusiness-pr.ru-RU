# <a name="import-certificate-invoke"></a>Импорт сертификата (вызов)
 
На странице **выполнение команд** проверьте сводные данные для импортированного сертификата. В раскрывающемся списке выберите файл журнала импорта сертификатов. Убедитесь в том, что в раскрывающемся списке отображается файл журнала импорта сертификатов. Щелкните **Просмотреть журнал** , чтобы просмотреть сведения об импорте сертификата. Когда вы закончите просмотр сводки и файла журнала, нажмите кнопку **Готово**.
