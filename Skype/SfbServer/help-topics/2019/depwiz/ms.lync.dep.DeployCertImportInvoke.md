# <a name="import-certificate-invoke"></a>Импорт сертификата (вызов)
 
На странице **Выполнение команд просмотрите** сводную информацию для импортируемого сертификата. В выпадаемом списке выберите файл журнала сертификата импорта. Убедитесь, что файл журнала сертификата импорта виден в списке выпаданий. Нажмите **кнопку Просмотр журнала,** чтобы просмотреть сведения о импорте сертификата. Когда вы закончите просмотр сводки и файла журнала, нажмите **кнопку Готово**.
