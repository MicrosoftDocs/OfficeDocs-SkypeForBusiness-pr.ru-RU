# <a name="import-certificate-invoke"></a>Импорт сертификата (вызов)
 
На странице **"Выполнение команд"** просмотрите сводную информацию для импортируемого сертификата. В выпадаемом списке выберите файл журнала импорта сертификата. Убедитесь, что файл журнала сертификата импорта отображается в выпадаемом списке. Щелкните **"Просмотреть** журнал", чтобы просмотреть сведения об импорте сертификата. После проверки сводки и файла журнала нажмите кнопку **"Готово".**
