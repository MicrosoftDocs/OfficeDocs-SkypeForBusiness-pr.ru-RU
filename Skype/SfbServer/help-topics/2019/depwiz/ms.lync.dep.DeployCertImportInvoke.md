# <a name="import-certificate-invoke"></a>Импорт сертификата (вызов)
 
На странице " **Выполнение команд** " просмотрите сводные данные для импортированный сертификат. В раскрывающемся списке выберите файл журнала импорта сертификата. Убедитесь, что файл журнала импорта сертификата отображается в раскрывающемся списке. Нажмите кнопку **Просмотреть журнал** для просмотра сведений импорта сертификата. После просмотра сводки и файл журнала, нажмите кнопку **Готово**.
