# <a name="startservices-intro"></a>Запуск служб (введение)
 
Чтобы запустить службы на сервере Skype для бизнеса Server роли, который вы развертываете, просмотрите экран введения. Обратите внимание, что команды, выдавлимые для запуска Skype для бизнеса Server, не ждут запуска службы. Это наилучший способ запуска служб, а наблюдать за реальным состоянием служб можно с помощью панели управления (MMC) для служб Windows. Нажмите кнопку **Далее**, чтобы начать выполнение задачи **Запустить службы**.
