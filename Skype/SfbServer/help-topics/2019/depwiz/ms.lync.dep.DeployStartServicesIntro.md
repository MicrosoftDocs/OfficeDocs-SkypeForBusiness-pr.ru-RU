# <a name="startservices-intro"></a>Запуск служб (введение)
 
Запуск служб на Скайп для сервера Business Server role, который выполняется развертывание, просмотрите на экране введение. Обратите внимание на то, что команды для запуска Скайп для Business Server, не дожидаясь для запуска службы. Это наилучший из возможных способов запуска служб; наблюдать за текущим состоянием служб следует с консоли управления (MMC) для служб Windows. Нажмите **Далее** для перехода к задаче **Запустить службы**.
