# <a name="test-device-create-new-or-edit-existing"></a>Тестовое устройство: создание нового или редактирование существующего

Возможность "Устройство проверки" работает совместно с возможностью "Обновление устройства". Вы можете добавить устройство проверки на страницу **Устройство проверки** и затем использовать его для проверки функциональных возможностей новых обновлений перед их развертыванием на производственных устройствах. Вы можете протестировать устройство глобально (то есть в рамках всей среды) или для отдельного сайта. Устройство проверки указывается по MAC-адресу или серийному номеру. После добавления устройства оно появится в списке на странице " **тестирование устройства** " на панели управления "Skype для бизнеса Server".

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Новое устройство проверки** или **Изменение устройства проверки** можно выполнить следующие задачи:

- добавить новое устройство проверки;

- изменить свойства существующего устройства проверки.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Область действия** Определяет область тестового устройства (глобально или на сайте).

- **Name (имя** ) Вы можете добавить или изменить имя тестового устройства.

- **Имя устройства** Вы можете добавить или изменить имя тестового устройства.

- **Тип идентификатора** Вы можете выбрать метод, который будет использоваться для идентификации устройства, выбирая один из следующих параметров:

  - **MAC-адрес**

  - **Серийный номер**

- **Уникальный идентификатор** Вы можете ввести MAC-адрес или серийный номер устройства.

Дополнительные сведения о тестировании устройств см. в разделе [Add a Device to Test Update Functionality](https://technet.microsoft.com/library/ce509fd1-17b3-4b78-b269-fe5d06fe2e1d.aspx) документации по использованию.
