# <a name="response-groups-queue"></a>Очередь группы ответа

Группы ответа состоят из агентов, очередей и рабочих процессов. Очереди группы ответа удерживают звонки в группе ответа, пока агент не ответит на вызов.

На **странице**  -  **"Очередь** групп ответа" отображается список всех очередей группы ответа, определенных для организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице "Очередь групп ответа" можно выполнить **следующие**  -   задачи:

- создать новую очередь;

- изменить существующую очередь;

- удалить очередь.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новый** Запускает новую очередь.

- **Правка** Открывает выбранную очередь для ее редактирования, выбирает все очереди в списке или удаляет выбранную очередь.

- **Обновление** Обновляет список очередей.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, идентифицирует очередь.

- **Служба** Служба **ApplicationServer,** в котором размещена очередь.

- **Описание** Описание очереди.

For details about Response Group features and capabilities, see [Plan for the Response Group application in Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/response-group.md) in the Planning documentation. Подробные сведения о работе с [](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) очередями группы ответа см. в документации по управлению очередями группы ответа в документации по операциям.
