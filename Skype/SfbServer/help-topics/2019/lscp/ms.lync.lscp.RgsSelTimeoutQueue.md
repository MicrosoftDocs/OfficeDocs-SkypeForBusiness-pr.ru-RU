# <a name="select-time-out-queue"></a>Выбор очереди времени ожидания
 
Используйте диалоговое окно **Выбор очереди времени ожидания** для выбора очереди, в которую направляются звонки после истечения их времени.
  
В поле поиска введите полностью или частично имя очереди, которую вы хотите использовать. Чтобы получить все доступные очереди, не заполняйте поле «Поиск» и щелкните значок поиска.
  
В списке возвращенных в результате поиска очередей выберите очередь, которую вы хотите использовать для звонков с истекшим временем ожидания.
