# <a name="select-file-type-extensions"></a>Выбор расширений типов файлов
 
Используйте диалоговое окно **Выбор расширений типов файлов** для выбора расширений типов файлов, которые требуется заблокировать или разрешить, посредством установки флажков (для блокировки) или снятия флажков (для разрешения) рядом с расширениями в разделе **Расширения типов файлов**. Щелкните стрелку, чтобы развернуть категорию и просмотреть все расширения типов файлов для этой категории. Если требуется выбрать все расширения типов файлов для категории, выделите имя категории. 
  
Если в разделе **Расширения типов файлов** не отображается расширение типа файлов, которое требуется выбрать, введите расширение в текстовое поле в разделе **Добавить новые расширения файлов в список**, а затем нажмите кнопку **Добавить**. 
  
После добавления в список **Выбранные расширения типов файлов** всех необходимых расширений типов файлов нажмите кнопку **ОК**.
