<span data-ttu-id="3d960-p102">Используйте диалоговое окно **Выбор расширений типов файлов** для выбора расширений типов файлов, которые требуется заблокировать или разрешить, посредством установки флажков (для блокировки) или снятия флажков (для разрешения) рядом с расширениями в разделе **Расширения типов файлов**. Click the arrow to expand a category and view all of the file type extensions for that category. If you want to select all of the file type extensions for a category, select the category name.</span></span> 
  
<span data-ttu-id="3d960-109">Если в разделе **Расширения типов файлов** не отображается расширение типа файлов, которое требуется выбрать, введите расширение в текстовое поле в разделе **Добавить новые расширения файлов в список**, а затем нажмите кнопку **Добавить**.</span><span class="sxs-lookup"><span data-stu-id="3d960-109">If you do not see the extension for a file type that you want to select listed in **File type extensions**, type the extension in the text box under **Add new file extensions to the list**, and then click **Add**.</span></span> 
  
<span data-ttu-id="3d960-110">После добавления в список **Выбранные расширения типов файлов** всех необходимых расширений типов файлов нажмите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="3d960-110">After you have added all of the file type extensions that you want to the **Selected file type extensions** list, click **OK**.</span></span>
