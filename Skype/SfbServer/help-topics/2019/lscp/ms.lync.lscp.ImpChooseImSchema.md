# <a name="select-hyperlink-prefix"></a>Выбор префикса гиперссылок
 
В диалоговом окне **Выбор префикса гиперссылки** выберите типы гиперссылок, которые вы хотите заблокировать, отправить предупреждение или разрешить, выбрав (блокировать или отправить предупреждение) или сняв (чтобы разрешить) флажки для их расширений из категории с **префиксами гиперссылок**. Щелкните стрелку, чтобы развернуть категорию и просмотреть все префиксы гиперссылок для этой категории. Если вы хотите выбрать все префиксы гиперссылок для категории, выберите ее имя. 
  
Если вы не видите префикс гиперссылки, который вы хотите выделить в списке **префиксы гиперссылок**, введите расширение в поле **Добавить в список префиксы гиперссылок**и нажмите кнопку **добавить**. 
  
После добавления всех префиксов гиперссылок, которые нужно добавить в список **префиксов гиперссылок** , нажмите кнопку **ОК**.
