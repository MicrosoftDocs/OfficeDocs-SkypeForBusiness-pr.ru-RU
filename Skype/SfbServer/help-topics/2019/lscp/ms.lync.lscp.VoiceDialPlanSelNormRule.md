# <a name="select-normalization-rules"></a>Выбор правил нормализации
 
Правила нормализации используют платформа .NET Framework для перевода набраных номеров телефонов в формат E.164. Каждой абонентской группе должно быть назначено одно или несколько правил нормализации.
  
Чтобы использовать существующее правило нормализации, щелкните правило в списке, а затем нажмите кнопку **ОК**.
