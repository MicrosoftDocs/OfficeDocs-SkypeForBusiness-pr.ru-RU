# <a name="select-normalization-rules"></a>Выбор правил нормализации
 
Правила нормализации используют регулярные выражения .NET Framework для перевода набираемой телефонной связи в формат E.164. Каждой абонентской группе должно быть назначено одно или несколько правил нормализации.
  
Чтобы использовать существующее правило нормализации, щелкните правило в списке, а затем нажмите кнопку **ОК**.
