<span data-ttu-id="feee8-p102">Диалоговое окно **Несохраненные параметры настройки голосовой связи** используется для просмотра всех внесенных изменений. После завершения либо нажмите кнопку **Исполнить** для сохранения всех изменений, либо нажмите кнопку **Отмена**, чтобы вернуться и исправить изменения.</span><span class="sxs-lookup"><span data-stu-id="feee8-p102">You can use the **Uncommitted Voice Configuration Settings** dialog box to review all of the changes that you have made. When you are finished, either click **Commit** to commit all of these changes, or click **Cancel** to go back and modify your changes.</span></span>
  
> [!IMPORTANT]
> <span data-ttu-id="feee8-109">Если нажать кнопку **Исполнить**, будут сохранены **все** изменения, перечисленные в диалоговом окне **Несохраненные параметры настройки голосовой связи**.</span><span class="sxs-lookup"><span data-stu-id="feee8-109">If you click **Commit**, you will commit **all** of the changes that are listed in the **Uncommitted Voice Configuration Settings** dialog box.</span></span>
