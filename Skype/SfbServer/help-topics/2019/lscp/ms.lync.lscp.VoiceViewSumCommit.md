# <a name="uncommitted-voice-configuration-settings-commit"></a>Незафиксированные параметры конфигурации голосовой связи: фиксация
 
Вы можете использовать **диалоговое окно с** Параметры голосовой конфигурации, чтобы просмотреть все внесенные изменения. Прокрутите диалоговое окно, чтобы просмотреть все несохраненные изменения, или щелкните стрелки, чтобы развернуть или свернуть различные разделы. После завершения либо нажмите кнопку **Сохранить**, чтобы сохранить все изменения, либо нажмите кнопку **Отмена**, чтобы вернуться и исправить изменения.
  
> [!IMPORTANT]
> Если нажать кнопку **Сохранить**, будут сохранены **все** изменения, перечисленные в диалоговом окне **Несохраненные параметры настройки голосовой связи**.
