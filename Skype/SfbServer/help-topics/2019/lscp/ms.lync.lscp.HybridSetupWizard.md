# <a name="hybrid-manageability-wizard"></a>Мастер гибридного управления

С помощью мастера гибридного управления можно выполнить некоторые задачи настройки, необходимые для настройки гибридного развертывания в Skype для бизнеса Online. Дополнительные сведения о том, как настроить развертывание для гибридного развертывания, можно найти [в разделе Настройка среды с помощью мастера гибридного управления](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
