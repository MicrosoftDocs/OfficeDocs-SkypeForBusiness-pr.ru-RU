# <a name="pin-set-successfully"></a>ПИН-код задан успешно
 
Использование электронной почты для отправки ПИН-код пользователя, нажмите кнопку **Открыть мои приложения электронной почты для отправки новый ПИН-код пользователя**. Чтобы скопировать ПИН-код в сообщении электронной почты, выберите **Показывать ПИН-код**.
