# <a name="test-device"></a>Тестовое устройство

Вы можете добавить устройство проверки на страницу **Устройство проверки** и затем использовать его для проверки функциональных возможностей новых обновлений перед их развертыванием на производственных устройствах. Вы можете протестировать устройство глобально (то есть в рамках всей среды) или для отдельного сайта. Устройство проверки указывается по MAC-адресу или серийному номеру. После добавления устройства оно появится в списке на странице " **тестирование устройства** " на панели управления "Skype для бизнеса Server".

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице " **тестирование устройства** " можно выполнять следующие задачи:

- Добавьте тестовое устройство глобально или для определенного сайта.

- Изменение параметров существующего тестового устройства.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Новые** Вы можете добавить новое тестовое устройство со следующими областями:

  - Глобальный

  - Сайт

- **Изменить** Вы можете изменить параметры тестового устройства в списке. С помощью этого параметра можно выполнить следующее:

  - **Показать подробности** Этот параметр открывает диалоговое окно, в котором можно изменить параметры тестового устройства.

  - **Выделить все** Этот параметр позволяет выбрать все тестовые устройства в списке.

  - **Delete (удалить** ) Этот параметр удаляет все выбранные тестовые устройства.

- **Обновить** Вы можете обновить список тестовых устройств, чтобы проверить состояние параметров всех тестовых устройств.

Дополнительные сведения о тестировании устройств см. в разделе [Add a Device to Test Update Functionality](https://technet.microsoft.com/library/ce509fd1-17b3-4b78-b269-fe5d06fe2e1d.aspx) документации по использованию.
