# <a name="users-view-mobility-policy"></a>Пользователи: просмотр политики мобильных устройств

Диалоговое окно **Вид — политика мобильных устройств** отображаются свойства выбранной политики мобильных устройств.

Для получения дополнительных сведений о назначении политики мобильности видеть [Назначение политики мобильности для отдельных пользователей](https://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx) и [Настройка политики мобильных устройств](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
