# <a name="prefix-edit"></a>Редактирование префиксов
 
Чтобы определить префиксы или начальные цифры, добавьте префиксы номеров, которые будут обрабатываться этим маршрутом, в том числе + при необходимости. Например, вы можете сопоставить все числа, начинающиеся с + 123.
