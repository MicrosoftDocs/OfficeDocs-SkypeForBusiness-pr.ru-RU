# <a name="move-user"></a>Перемещение пользователя

На странице **Перемещение пользователей** можно выбрать пул назначения для перемещения выбранного пользователя.

Дополнительные сведения о перемещении пользователей см. в разделе [Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool).
