# <a name="select-from-active-directory"></a>Выбор в Active Directory
 
Используйте **диалоговое окно Выберите из диалогового окна Active Directory**, чтобы найти пользователей, которые необходимо включить для Skype для бизнеса Server. В поле поиска введите SIP-адрес или отображаемое имя пользователя, которого требуется найти. Чтобы получить всех доступных пользователей, не вводите ничего в поле поиска.
