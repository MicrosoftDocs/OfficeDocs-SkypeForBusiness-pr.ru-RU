# <a name="client-version-configuration"></a>Конфигурация версий клиентов

Кроме указания версии клиентов, поддержку которой вы хотите обеспечить в своей среде, вы можете также указать действие по умолчанию для клиентов, пока не имеющих заданной политики выбора версий. Это позволяет ограничить число используемых в среде версий клиентов, что помогает управлять затратами на поддержку нескольких версий клиентов.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Настройка версии клиента** можно выполнить следующие задачи:

- Изменение конфигурации версии клиента по умолчанию **(глобальная).**

- создать настройку версии клиентов для конкретного сайта;

- включить или отключить существующие настройки версий клиентов.

> [!NOTE]
> Поскольку анонимные пользователи не сопоставлены с сайтом, они управляются только политиками глобального уровня.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Новый** Можно создать конфигурацию версий клиентов для определенного сайта.

- **Правка** Вы можете изменить параметры любой из политик версий клиентов. С помощью этой команды можно выполнить следующие действия:

  - **Показать подробности** Этот параметр открывает диалоговое окно, в котором можно изменить параметры конфигурации версий клиентов.

  - **Выбрать все** Этот параметр выбирает все конфигурации версий клиентов в списке.

  - **Delete** Этот параметр удаляет все выбранные конфигурации версий клиентов.

- **Обновление** Можно обновить список конфигурации версий клиентов, чтобы проверить состояние параметров всех конфигураций версий клиентов.

Подробные сведения о взаимосвязи между клиентами [](https://technet.microsoft.com/library/0f126571-91a2-45d5-855c-1e4ddb45fc04.aspx) и версиями клиентов см. в документации по планированию. Дополнительные сведения о работе с конфигурациями версий клиентов см. в разделе [Modify the Default Action for Clients Not Explicitly Supported or Restricted](https://technet.microsoft.com/library/548dd0f5-62fe-4c3f-8952-2b9fd4c5fff3.aspx) в документации по использованию.
