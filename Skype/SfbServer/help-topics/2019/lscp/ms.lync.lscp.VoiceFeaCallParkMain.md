# <a name="call-park"></a>Парковка вызовов

При парковке вызова он передается на временный номер, где он удерживается до тех пор, пока кто-то не извлекает его или не подождите. Необходимо настроить таблицу с диапазонами номеров расширений, которые вы засвидетельна для припаркованных вызовов. Эти добавочные номера должны быть виртуальными (то есть не иметь назначенного пользователя или назначенный телефон). Каждый пул, в который запускается приложение парковки вызовов, может иметь один или несколько диапазонов расширений. Эти диапазоны должны быть уникальными в рамках всего развертывания.

На **странице парковки** вызовов отображается список всех диапазонов номеров парковки вызовов, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Парковка звонков** можно выполнить следующие задачи:

- создать новый диапазон номеров;

- изменить существующий диапазон номеров;

- удалить диапазон номеров.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новый** Запускает новый диапазон номеров парковки вызовов.

- **Правка** Открывает выбранный диапазон номеров для редактирования, выбирает все диапазоны номеров в списке или удаляет выбранный диапазон номеров.

- **Обновление** Обновляет список диапазонов номеров.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, идентифицирует диапазон номеров.

- **Диапазон запуска** Первый номер диапазона.

- **Конечный диапазон** Конец диапазона.

- **Назначение** Полное доменное имя (FQDN) или ИД службы приложений, в котором размещено приложение парковки вызовов для диапазона номеров.

For details about Call Park features and capabilities, see [Plan for Call Park in Skype for Business](../../../plan-your-deployment/enterprise-voice-solution/call-park.md). Подробные сведения о работе с диапазонами номеров парковки вызовов см. в подстроке "Настройка расширений телефонных [номеров для парковки вызовов".](https://technet.microsoft.com/library/fbf97624-9587-42a6-b276-1b69c574a74d.aspx)
