# <a name="response-groups-workflow"></a>Рабочий процесс групп ответа

Группы ответа состоят из групп агентов, очереди и рабочие процессы. Рабочие процессы группы ответа определяют действия, предпринимаемые при получении приложения группы ответа на телефонный звонок.

**Группы ответа** - странице**рабочий процесс** отображается список всех рабочих процессов группы ответа, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Группы ответа**можно выполнить следующие задачи - страница**рабочего процесса** :

- Создание или изменение рабочего процесса

- Создание или изменение интерактивного рабочего процесса

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Создание или изменение рабочего процесса** Откроется средство настройки группы ответа для создания или изменения рабочего процесса.

- **Обновление** Обновляет список рабочих процессов.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, назначенное рабочему процессу.

- **Службы** Службы **сервера приложений** , на котором размещается рабочий процесс.

- **SIP-адрес** SIP-адрес группы, которая будет отвечать на звонки в рабочий процесс.

- **Телефон** Номер телефона, вызов для обращения в данную группу ответа.

- **Язык** Язык, используемый для распознавания речи и преобразования текста в речь.

- **IVR** Указывает, является ли рабочий процесс сервисной группой или интерактивного рабочего процесса.

- **Включено** Указывает, будет ли данному рабочему процессу будут принимать звонки.
