# <a name="control-panel---updated-user-search"></a>Панель управления — обновлено: Поиск пользователей

Настройка пользователей для Скайп для Business Server можно использовать результаты запроса поиска. Можно выполнить поиск пользователей по отображаемое имя, имя первого, последнего имени, имя учетной записи диспетчера учетных записей (SAM), SIP-адрес или строки универсальный код ресурса (URI). Можно также выполнить поиск для пользователей с помощью панели управления Lync Server или Active Directory — пользователи и компьютеры оснастку.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках приведены меню, команды, поля и свойства, имеющиеся на странице **Поиск пользователей**.

### <a name="user-search"></a>Поиск пользователей

- **Поиск** Поиск пользователей по первой части отображаемого имени, имени, фамилии, имя учетной записи SAM, SIP-адрес или строке URI учетной записи пользователя.

- **Поиск LDAP** Поиск пользователей с помощью ввода выражения LDAP.

- **Поле поиска пользователей** Введите данных о пользователе или выражения LDAP, которые требуется выполнить поиск.

- **Поиск** Щелкните, чтобы отобразить пользователей, которые соответствуют значения поиска, указанное в **Поиск пользователей** и поле.

- **Открыть запрос** Щелкните, чтобы открыть сохраненный поисковый запрос.

- **Сохранить запрос** Щелкните, чтобы сохранить поисковый запрос.

- **+ Добавить фильтр** Щелкните, чтобы добавить дополнительные условия поиска.

- **Поля фильтра поиска** Выберите поле, на котором требуется отфильтровать результаты поиска, выберите оператор для запроса, а затем введите строку, которую нужно найти на.

- **Максимум пользователей для отображения** Введите количество результатов поиска, которое требуется отображать, или с помощью вверх и вниз, чтобы указать номер.

Добавьте описательный текст, если требуется.

### <a name="search-results-menus"></a>Меню результатов поиска

- **Включение пользователей** Щелкните, чтобы открыть [пользователи: новый пользователь Lync Server](ms.lync.lscp.UserNew.md) диалоговое окно, в котором можно добавить нового пользователя Скайп для Business Server.

    Для добавления нового контакта щелкните стрелку вниз, а затем выберите команду **Включить контакты**, чтобы открыть диалоговое окно [Users: New Contact Objects](ms.lync.lscp.UserNewContact.md).

- **Изменение** Нажмите кнопку **Изменить** и нажмите кнопку **Показать подробности** , чтобы отобразить сведения о выбранного пользователя или нажмите кнопку **Выбрать всех результатов поиска** для выбора всех пользователей, отображаемые в таблице результатов.

- **Действие** Щелкните **Действие**, а затем выберите действие, которое необходимо выполнить для выбранных пользователей в результатах поиска. Доступны следующие действия:

  - **Повторно включить для Lync Server** Включает учетную запись выбранного пользователя после его временно отключена.

  - **Временно отключить для Lync Server** Отключение учетной записи пользователя в Скайп для Business Server, пока повторно включить его, без удаления учетной записи пользователя.

  - **Назначение политик** Откроется [пользователи: назначение политик](ms.lync.lscp.UserAssignPolicy.md) диалоговое окно, где можно настроить политики, назначенные для пользователя.

  - **Состояние ПИН-кода представления** Откроется [пользователи: Просмотр состояния ПИН-код](ms.lync.lscp.UserViewPin.md) диалоговое окно, которое отображает данные ПИН для выбранного пользователя.

  - **Установка ПИН-КОДОВ** Откроется диалоговое окно [Set PIN](ms.lync.lscp.UserSetPin.md) , где можно задать ПИН-код для выбранного пользователя.

  - **Заблокировать ПИН-кода** Блокирует ПИН-КОДОВ для пользователя.

  - **Разблокировать ПИН-код** Удаляет блокировку ПИН-код пользователя.

  - **Удаление из Lync Server** Удаляет учетную запись пользователя из Скайп для Business Server. Пользователь не удаляется из Active Directory.

  - **Удаление сертификата пользователя** Удаляет все сертификаты, выданные пользователю.

  - **Переместить выбранных пользователей в пул** Откроется диалоговое окно [Move User](ms.lync.lscp.UserMove.md) , где можно выбрать пул для перемещения выбранного пользователя.

  - **Перемещение всех пользователей в пул** Откроется диалоговое окно [Move User](ms.lync.lscp.UserMove.md) , где можно выбрать пул для перемещения всех пользователей.
