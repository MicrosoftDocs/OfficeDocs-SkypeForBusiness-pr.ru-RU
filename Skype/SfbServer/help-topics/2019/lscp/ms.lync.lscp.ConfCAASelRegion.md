# <a name="select-regions"></a>Выбор областей
 
Диалоговое **окно Select Regions** используется для поиска регионов, доступных в среде. Чтобы использовать существующий регион, щелкните область в списке и нажмите **кнопку ОК**.
  
> [!NOTE]
> Чтобы получить все доступные регионы, ничего не введите в поле поиска, а затем нажмите значок поиска.
