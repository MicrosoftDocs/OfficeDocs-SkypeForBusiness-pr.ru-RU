# <a name="select-groups"></a>Выбор групп

Очереди групп ответа содержат звонки в группы ответа, пока агент не ответит на звонок. Очереди обладают назначенными группами агентов. Когда вы создаете или редактируете очередь, вы можете выбрать одну или несколько групп агента для назначения в очередь с помощью диалогового окна " **Выбор групп** ".

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В списке ниже описаны элементы управления и поля в диалоговом окне " **Выбор групп** ".

- Поле поиска. Поиск имени группы агента. Введите имя или его часть. Чтобы отобразить все группы агентов для Организации, оставьте поле поиска пустым.

- **Name (имя** ) Выводит имена возвращенных групп агента.

- **Политика участия** Политика участия, определенная для группы агента: формально или формально.

- **Метод маршрутизации** Отображение типа маршрутизации, определенного для группы агента.

Дополнительные сведения о работе с очередями см. в разделе [Managing Response Group Queues](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) документации по использованию.
