# <a name="select-groups"></a>Выбор групп

Очереди группы ответа удерживают звонки в группах ответа, пока агент не ответит на вызов. Очереди имеют назначенные группы агентов. При создании или изменении очереди вы используете диалоговое окно **Выбор групп** для выбора одной или нескольких групп агентов, которые требуется назначить очереди.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления и поля в диалоговом окне **Выбор групп**.

- В поле поиска выполняется поиск имени группы агентов. Введите имя полностью или частично. Оставьте поле поиска пустым, чтобы отобразить все группы агентов для своей организации.

- **Имя** Отображает имена возвращенных групп агентов.

- **Политика участия** Отображает политику участия, определяемую для группы агентов: формальную или неформальную.

- **Метод маршрутной маршрутки** Отображает тип маршрутов, определенных для группы агентов.

Дополнительные сведения о работе с очередями см. в разделе [Managing Response Group Queues](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) документации по использованию.
