# <a name="topology-status"></a>Топология: состояние
 
На странице **Топология**: **Состояние** отображается состояние серверов в вашей топологии Skype для бизнеса.
  
## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Топология**: **Состояние** можно выполнить следующие задачи:
  
- [Просмотр списка компьютеров под управлением Lync Server 2010](http://technet.microsoft.com/library/44eeec27-8b99-44f0-b0bd-622c12393d34.aspx)
    
- [Просмотр сведений о службе](http://technet.microsoft.com/library/bc8e8202-cd68-47e4-95b2-bb36e51cc124.aspx)
    
- [Запуск или остановка служб Lync Server 2010](http://technet.microsoft.com/library/1c70b4ec-9de5-4f7a-a3c9-c0eb76710505.aspx)
    
- [Запретить новые подключения к службам](http://technet.microsoft.com/library/977dcc5c-2aac-48ef-86a1-a8d47b4d9e74.aspx)
    
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../manage/manage.md).
