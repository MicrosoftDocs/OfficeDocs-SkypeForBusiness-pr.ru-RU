# <a name="select-a-region-link"></a>Выбор связи между областями
 
Диалоговое **окно Region Link** используется для поиска областных ссылок, доступных в среде. Чтобы использовать существующую связь между областями, щелкните связь в списке, а затем нажмите кнопку **ОК**.
  
> [!NOTE]
> Чтобы получить все доступные связи между областями, ничего не вводите в поле поиска и нажмите значок поиска.
