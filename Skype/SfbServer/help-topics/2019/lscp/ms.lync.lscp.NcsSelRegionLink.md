# <a name="select-a-region-link"></a>Выбор связи между областями
 
Используйте **диалоговое окно "Связь** между областями", чтобы найти ссылки на регионы, доступные в вашей среде. Чтобы использовать существующую связь между областями, щелкните связь в списке, а затем нажмите кнопку **ОК**.
  
> [!NOTE]
> Чтобы получить все доступные связи между областями, ничего не вводите в поле поиска и нажмите значок поиска.
