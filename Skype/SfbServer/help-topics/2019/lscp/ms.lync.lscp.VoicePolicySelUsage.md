# <a name="select-pstn-usage-record"></a>Выбор записи использования ТСОП
 
В диалоговом окне **выберите запись использования PSTN** , чтобы найти существующую запись использования коммутируемой телефонной сети (PSTN), которую вы можете использовать.
  
> [!NOTE]
> Чтобы получить все доступные записи об использовании PSTN, не вводите ничего в поле поиска, а затем щелкните значок "Поиск".
