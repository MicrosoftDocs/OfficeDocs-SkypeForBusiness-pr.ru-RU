# <a name="select-overflow-queue"></a>Выбор очереди переполнения
 
В диалоговом окне " **Выбор очереди переполнения** " можно выбрать очередь, в которую пересылаются звонки при заполнении очереди групп ответа.
  
В поле Поиск введите имя нужной очереди или ее часть. Чтобы вернуть все доступные очереди, оставьте поле поиска пустым и щелкните значок "Поиск".
  
В списке очередей, возвращенных поиском, выберите очередь, которую вы хотите использовать для переадресации звонков, когда очередь группы ответа заполнена.
