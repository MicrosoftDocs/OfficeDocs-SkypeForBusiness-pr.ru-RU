# <a name="clients-mobility-policy"></a>Клиенты: Политика мобильных устройств
 
На странице **клиентов** **Политики мобильных устройств** отображаются доступные в вашем развертывании политики мобильных устройств.
  
## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Политика мобильных устройств** можно выполнить следующие задачи:
  
- Просмотр существующих политик мобильности
    
- Создание новой политики мобильных устройств
    
- Изменение политики мобильных устройств
    
## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.
  
### 

- **Поле поиска** Введите строку в текстовое поле для фильтрации отображаемых политик.
    
- **Новые** Нажмите кнопку **Создать**и выберите **узла** или **пользователя** , чтобы создать новую политику мобильности.
    
- **Изменение** Выберите политику и нажмите кнопку **Изменить** , чтобы изменить свойства выбранной политики.
    
- **Обновление** Обновляет список отображаемых политик. При нажатии кнопки **Обновить**строку фильтра поиска будет удалена, и отображаются все существующие политики.
    
Для получения дополнительных сведений о настройке политик мобильности видеть [Настройка политики мобильных устройств](http://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
