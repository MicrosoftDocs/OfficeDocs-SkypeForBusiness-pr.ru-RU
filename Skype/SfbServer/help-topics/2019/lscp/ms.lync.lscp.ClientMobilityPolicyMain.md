# <a name="clients-mobility-policy"></a>Клиенты: политика мобильных устройств

На странице " **политика мобильности** **клиентов** " отображаются политики мобильной связи, доступные в вашем развертывании.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице " **Политика Mobility Service** " можно выполнять следующие задачи:

- Просмотр существующих политик мобильной связи

- Создание новой политики мобильной связи

- Изменение политики мобильности

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.



- **Поле поиска** Введите строку в текстовом поле, чтобы отфильтровать отображаемые политики.

- **Новые** Нажмите кнопку **создать**, а затем выберите либо **сайт** , либо **пользователь** , чтобы создать новую политику мобильности.

- **Изменить** Выберите политику, а затем нажмите кнопку **изменить** , чтобы изменить свойства выбранной политики.

- **Обновить** Обновление отображаемого списка политик. При нажатии кнопки **Обновить**строка фильтра поиска будет очищена, а все существующие политики будут отображены.

Дополнительные сведения о настройке политик мобильности см в разделе [Настройка политики мобильности](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
