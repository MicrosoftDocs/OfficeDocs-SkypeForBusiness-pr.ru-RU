# <a name="hybrid-wizard-move-user"></a>Мастер гибридной: перемещение пользователей
 
Используйте мастер переместить пользователей для перемещения учетных записей пользователей, размещенных на пул локальной размещен в Скайп для бизнеса в Интернет. Дополнительные сведения можно [использовать мастер создания гибридной управляемости](http://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
