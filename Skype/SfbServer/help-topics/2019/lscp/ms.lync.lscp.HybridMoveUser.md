# <a name="hybrid-wizard-move-user"></a>Мастер гибридной среды: перемещение пользователя

Используйте мастер Move Users для перемещения учетных записей пользователей, которые будут в локальном пуле, которые будут Skype для бизнеса Online. Дополнительные сведения см. в [дополнительных сведениях: Использование мастера гибридной управляемости](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
