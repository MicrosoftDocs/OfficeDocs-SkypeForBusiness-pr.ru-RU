# <a name="first-run-checklist-for-skype-for-business-server-control-panel"></a>Контрольный список первого запуска для панели управления Skype для бизнеса Server

Добро пожаловать в панель управления Skype для бизнеса Server — это веб-интерфейс пользователя для администрирования и управления Skype для бизнеса Server. Панель управления можно использовать для выполнения задач администрирования тех же типов, что и в консоли управления (MMC) предыдущих версий.

Вы настоятельно рекомендуем выполнить некоторые важные задачи после развертывания Skype для бизнеса Server. Некоторые из этих задач являются начальными, которые могут быть уже выполнены во время развертывания, в то время как другие являются уточнениями или изменениями параметров, которые вы настроили во время развертывания или настройки по умолчанию. Другие задачи, описанные в этом разделе, проверяют настройки, внесенные в процессе развертывания.

> [!NOTE]
> Перед выполнением задач, приведенных в следующей таблице, убедитесь, что вы вошли с применением необходимых прав пользователя, разрешений и роли, как описано в подразделе "Роль и область действия" раздела [Role-Based Access Control](https://technet.microsoft.com/library/41204ba3-ce5b-41a8-a6c3-b444468fa328.aspx).
