# <a name="topology-server-application"></a>Топология: Серверное приложение
 
**Топология**: **Серверное приложение** отображаются серверные приложения в вашем развертывании.
  
## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

Можно выполнить следующие задачи на **Топология**: **Серверное** приложение:
  
- Просмотреть серверные приложения в вашем развертывании
    
- Включение или отключение серверного приложения
    
- Установите или снимите флажок приложение в качестве критического
