# <a name="select-translation-rules"></a>Выбор правил преобразования
 
 Корпоративной голосовой связи требуется, что все номера строк преобразовываться в формат E.164 для выполнения обратного номеров поиск. Ответная часть магистрали (то есть связанный шлюз, УАТС или магистраль SIP) может требовать того, чтобы номера были указаны в местном формате набора номеров. Чтобы преобразовать номера из формата E.164 в местный формат набора номеров, можно задать одно или несколько правил преобразования, которые будут изменять URI запроса перед отправкой его в ответную часть магистрали. Например, можно написать правило преобразования для удаления +44 из начала строки набора и замены этой последовательности номером 0144.
  
> [!IMPORTANT]
> Связывать один или несколько правил преобразования с конфигурацией магистрали корпоративной голосовой связи предназначен для использования в качестве альтернативного при настройке правил преобразования на узле магистрали. Не связывайте правила трансляции с конфигурацией магистрали корпоративной голосовой связи, если вы настроили правила преобразования на узле магистрали два правила могут конфликтовать. 
  
Чтобы использовать существующее правило преобразования, щелкните правило в списке, а затем нажмите кнопку **ОК**.
  
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../../manage/manage.md).
