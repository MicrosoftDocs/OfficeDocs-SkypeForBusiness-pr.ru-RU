# <a name="uncommitted-voice-configuration-settings-review"></a>Параметры конфигурации незафиксированные голосовой связи: Обзор
 
Используйте диалоговое окно **Незафиксированные параметры конфигурации голосовой связи** , чтобы просмотреть все изменения, внесенные. Прокрутите диалоговое окно, чтобы просмотреть все несохраненные изменения, или нажмите стрелки, чтобы развернуть или свернуть различные разделы. Завершив настройку, нажмите кнопку **Закрыть**. Закройте это диалоговое окно, можно изменить изменения или их подтверждения.
  
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../../manage/manage.md).
