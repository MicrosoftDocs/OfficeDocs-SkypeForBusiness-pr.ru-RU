 **Работа с политикой ПИН-кодов для IP-телефонов** Эта политика пин-кодов поддерживает проверку подлинности пользователей, подключающихся к Skype для бизнеса Server с помощью IP-телефона.
- [Создание новой политики ПИН-кодов в Skype для бизнеса Server](../../../manage/authentication/create-a-new-pin-policy.md)

- [Изменение существующей политики ПИН-кодов в Skype для бизнеса Server](../../../manage/authentication/modify-an-existing-pin-policy.md)

- [Удаление политики ПИН-кодов в Skype для бизнеса Server](../../../manage/authentication/delete-a-pin-policy.md)

  **Работа с политикой ПИН-кодов для** телефонной связи Эта политика ПИН-кодов позволяет пользователям, у которых есть учетные данные доменных служб Active Directory в организации, присоединяться к конференциям с телефонным входом в качестве пользователей, подлинность которых прошла, путем ввода ПИН-кода.

- [Удаление параметров ПИН-кода для параметров ПИН-кода для сайта или группы пользователей](https://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
