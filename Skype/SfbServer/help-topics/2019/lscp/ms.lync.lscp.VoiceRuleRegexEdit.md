# <a name="translation-rule-type-a-regular-expression"></a>Правило преобразования: ввод регулярного выражения
 
В поле **Соответствовать шаблону** укажите шаблон, который будет использоваться для того, чтобы ему соответствовали номера, предназначенные для преобразования. В поле **Правило преобразования** укажите шаблон для формата преобразованных номеров. Например, если вы вводите ^(\d\d{9}+)$ в поле Match **this pattern** field и 011$1 в поле правила перевода, правило переведет +\+441235551010 на 011441235551010.
