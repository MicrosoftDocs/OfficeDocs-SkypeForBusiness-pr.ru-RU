# <a name="save-voice-routing-test-case-information"></a>Сохранение сведений тестового сценария маршрутизации голосовых данных
 
Вы можете использовать диалоговое окно **"Сохранение** сведений тестового случая маршрутизационной голосовой почты" для сохранения тестового случая маршрутизационной маршрутизационной голосовой почты. В диалоговом окне **Сохранение сведений тестового сценария маршрутизации голосовых данных** введите имя, которое требуется присвоить тестовому сценарию, в поле **Имя**, а затем нажмите кнопку **ОК**. 
  
> [!NOTE]
> При сохранении временного тестового случая оно не сохраняется в центральном хранилище управления. Вместо этого тестовый сценарий становится доступным на вкладке **Проверка маршрутизации голосовой связи** на странице **Маршрутизация голосовой связи**. Тестовый сценарий появится с состоянием **Незафиксированные**. Чтобы сохранить тестовый случай в центральном хранилище управления, необходимо зафиксировать  конфигурацию голосовой почты, щелкнув "Сохранить **все"** в любом меню действий на странице **маршрутизации голосовой** почты.
