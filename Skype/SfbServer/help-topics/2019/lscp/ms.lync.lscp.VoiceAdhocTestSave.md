# <a name="save-voice-routing-test-case-information"></a>Сохранение сведений тестового сценария маршрутизации голосовых данных
 
Для **сохранения** тестового случая маршрутизационных маршрутизационных маршрутов можно использовать диалоговое окно "Сохранить голосовую маршрутику" для сохранения тестового случая маршрутизационных маршрутизационных маршрутов. В диалоговом окне **Сохранение сведений тестового сценария маршрутизации голосовых данных** введите имя, которое требуется присвоить тестовому сценарию, в поле **Имя**, а затем нажмите кнопку **ОК**. 
  
> [!NOTE]
> При сохранении временного тестового случая он не сохраняется в хранилище Central Management. Вместо этого тестовый сценарий становится доступным на вкладке **Проверка маршрутизации голосовой связи** на странице **Маршрутизация голосовой связи**. Тестовый сценарий появится с состоянием **Незафиксированные**. Чтобы сохранить тестовое дело в хранилище Центра управления, необходимо совершить конфигурацию  голосовой почты, нажав кнопку **Commit All** из любого меню Действий на странице **маршрутизация** голосовой почты.
