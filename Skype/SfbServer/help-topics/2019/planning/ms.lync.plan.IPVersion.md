# <a name="ip-version-planning-tool"></a>IP-версия (средство планирования)
 
При настройке IP-адресов в Skype для бизнеса Server есть три варианта. Вы можете настроить Skype для бизнеса Server для поддержки только IP-версии 4 (IPv4), только IP-версии 6 (IPv6) или сочетания обоих (известных как двойной стек).
  
Дополнительные сведения о выборе инфраструктуры IP см. в материале [Plan for IPv6 в Skype для бизнеса](../../../plan-your-deployment/network-requirements/ipv6.md).
