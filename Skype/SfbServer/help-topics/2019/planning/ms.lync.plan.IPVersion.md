# <a name="ip-version-planning-tool"></a>ВЕРСИЯ IP (средство планирования)
 
У вас есть три варианта настройки IP-адресов в Skype для бизнеса Server. Skype для бизнеса Server можно настроить на поддержку только IP версии 4 (IPv4), только IP версии 6 (IPv6) или сочетания обоих вариантов (называется двойным стеком).
  
Дополнительные сведения о выборе инфраструктуры IP см. в плане [IPv6 в Skype для бизнеса.](../../../plan-your-deployment/network-requirements/ipv6.md)
