# <a name="voice-traffic-planning-tool"></a>Voice Traffic (Planning Tool)
 
Средство планирования использует сведения о трафике пользователя для определения количества портов для использования в развертывании. Этот номер, в свою очередь, определяет число серверов-посредников и шлюзов, которые требуется указать. Дополнительные сведения см [Estimating использования голосовой связи и трафика для Скайп для Business Server](../../../plan-your-deployment/capacity/estimating-voice-traffic.md).
