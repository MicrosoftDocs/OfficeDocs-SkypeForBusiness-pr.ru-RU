# <a name="voice-traffic-planning-tool"></a>Голосовой трафик (средство планирования)
 
Средство планирования использует сведения о пользовательском трафике для определения количества портов, которые необходимо использовать в развертывании. Этот номер, в свою очередь, определяет количество необходимых серверов и шлюзов-посредников. Подробные сведения см. в материале Оценка использования голосовых данных и [трафика для Skype для бизнеса Server.](../../../plan-your-deployment/capacity/estimating-voice-traffic.md)
