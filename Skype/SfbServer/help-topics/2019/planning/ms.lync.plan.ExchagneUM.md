# <a name="exchange-unified-messaging-planning-tool-in-skype-for-business-2015"></a>Exchange Средство планирования единой системы обмена сообщениями в Skype для бизнеса 2015 г.

> [!NOTE]
> Exchange Единая система обмена сообщениями, как известно, больше недоступна в Exchange 2019 г., но вы можете использовать телефонная система для записи сообщений голосовой почты, а затем оставить запись в Exchange почтовом ящике пользователя. Дополнительные [сведения см. в облачная голосовая почта службе](../../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) Plan облачная голосовая почта.

Skype для бизнеса Server 2015 г. поддерживает интеграцию с Exchange единой системы обмена сообщениями для объединения голосовых сообщений и обмена сообщениями в единую инфраструктуру обмена сообщениями.
