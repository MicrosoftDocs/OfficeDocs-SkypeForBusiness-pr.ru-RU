# <a name="audio-video-conferencing-planning-tool"></a>Аудиоконференции видео (средство планирования)

Сведения о планировании включения аудио- и видеоконференций в вашей организации см. в следующих статьях и разделах: 
  
- [Планирование конференц-связи в Скайп для Business Server 2015](../../../plan-your-deployment/conferencing/conferencing.md)
