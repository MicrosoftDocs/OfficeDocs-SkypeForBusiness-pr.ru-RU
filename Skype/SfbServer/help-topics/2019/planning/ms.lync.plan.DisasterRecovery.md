# <a name="disaster-recovery-planning-tool"></a>Аварийное восстановление (средство планирования)
 
Skype для бизнеса Server включает меры аварийного восстановления, позволяя вам спарить пулы переднего конца, расположенные в двух центрах обработки данных.
  
Сведения о подготовке среды к аварийному восстановлению см. в материале [Plan for high availability and disaster recovery in Skype для бизнеса Server](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).
