# <a name="disaster-recovery-planning-tool"></a>Аварийное восстановление (средство планирования)
 
Skype для бизнеса Server включает меры аварийного восстановления путем сопряжения пулов переднего сервера, расположенных в двух центрах обработки данных.
  
Подробные сведения о подготовке среды к аварийному восстановлению см. в плане высокой доступности и аварийного [восстановления в Skype для бизнеса Server.](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md)
