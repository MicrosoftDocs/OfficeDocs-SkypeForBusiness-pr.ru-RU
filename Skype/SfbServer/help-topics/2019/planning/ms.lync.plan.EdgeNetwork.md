# <a name="edge-network-planning-tool-in-skype-for-business"></a>Пограничная сеть (средство планирования) в Скайп для бизнеса 
 
Большинство пограничных компонентов развертывается в сети периметра. Указанные ниже компоненты составляют пограничную топологию этой сети. Если не указано иное компоненты являются частью [сценарии пограничного сервера в Скайп для Business Server](../../../plan-your-deployment/edge-server-deployments/scenarios.md) и находятся в сети периметра. К пограничным компонентам относятся следующие:
  
- пограничные серверы;
    
- Сертификат обратного прокси-сервера
    
- Брандмауэр
    
- Директоры (не являются обязательными и логически расположены во внутренней сети);
    
- Балансировка нагрузки для масштабируемых пограничных топологий (балансировка нагрузки DNS или подсистема аппаратной балансировки нагрузки)
