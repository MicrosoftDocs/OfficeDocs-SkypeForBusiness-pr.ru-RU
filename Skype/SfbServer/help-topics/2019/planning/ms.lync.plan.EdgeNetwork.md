# <a name="edge-network-planning-tool-in-skype-for-business"></a>Edge Network (Средство планирования) в Skype для бизнеса 
 
Большинство пограничных компонентов развертывается в сети периметра. Следующие компоненты составляют топологию edge сети периметра. Кроме отмеченных, компоненты являются частью сценариев [Edge Server](../../../plan-your-deployment/edge-server-deployments/scenarios.md) в Skype для бизнеса Server и находятся в сети периметра. К пограничным компонентам относятся следующие:
  
- Edge Server (s)
    
- Обратный прокси-сервер
    
- Брандмауэр
    
- Директора (необязательный и логически расположенный во внутренней сети)
    
- Балансировка нагрузки для масштабируемых пограничных топологий (балансировка нагрузки DNS или подсистема аппаратной балансировки нагрузки)
