# <a name="federation-planning-tool"></a>Federation (Planning Tool)
 
Пограничные серверы могут быть настроены таким образом, чтобы разрешить внутренним и внешним пользователям доступ к контактам в организациях или службах партнера.
  
 Интеграция работает в Skype для бизнеса Server, как и для Microsoft Lync 2013. Дополнительные сведения о подготовке среды для федерации см. в следующих статьях:
  
- [Планирование для Lync Server 2013 и Office Communications Server Federation](https://technet.microsoft.com/en-us/library/jj205335%28v=ocs.15%29.aspx)
    
- [Планирование подключения общедоступной службы обмена мгновенными сообщениями в Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205349%28v=ocs.15%29.aspx)
    
- [Планирование Федерации протоколов обмена сообщениями и присутствия в КСМПП в Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205107%28v=ocs.15%29.aspx)
