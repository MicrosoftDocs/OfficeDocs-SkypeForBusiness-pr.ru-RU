# <a name="voice-infrastructure-planning-tool"></a>Инфраструктура голосовой связи (средство планирования)
 
Skype для бизнеса Server обеспечивает надежное масштабируемое подключение к STN с помощью магистрали SIP, прямых SIP-подключений к шлюзу STN или прямых SIP-подключений к УАО. For details, see [Plan for PSTN connectivity in Skype for Business Server.](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)
