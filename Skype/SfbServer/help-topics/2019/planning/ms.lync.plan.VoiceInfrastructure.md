# <a name="voice-infrastructure-planning-tool"></a>Голосовая инфраструктура (средство планирования)
 
Skype для бизнеса Server обеспечивает надежную масштабируемую возможность подключения к PSTN с помощью магистрали SIP, прямых SIP-подключений к шлюзу PSTN или прямых подключений SIP к PBX. Подробные сведения см. [в материале Plan for PSTN connectivity in Skype для бизнеса Server.](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)
