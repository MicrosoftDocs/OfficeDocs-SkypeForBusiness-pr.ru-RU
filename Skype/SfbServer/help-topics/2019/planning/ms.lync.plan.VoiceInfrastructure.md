# <a name="voice-infrastructure-planning-tool"></a>Инфраструктура голосовой связи (средство планирования)
 
Скайп для Business Server обеспечивает надежное, масштабируемое подключение к ТСОП с помощью магистралях SIP, прямые SIP-подключения для шлюза ТСОП или прямые SIP-подключения для УАТС. Дополнительные сведения см [Планирование подключения к ТСОП в Скайп для Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
