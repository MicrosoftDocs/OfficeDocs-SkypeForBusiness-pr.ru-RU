# <a name="voice-infrastructure-planning-tool"></a>Voice Infrastructure (Planning Tool)

Подробнее смотрите в разделе [Планирование подключений PSTN в Skype для бизнеса Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
