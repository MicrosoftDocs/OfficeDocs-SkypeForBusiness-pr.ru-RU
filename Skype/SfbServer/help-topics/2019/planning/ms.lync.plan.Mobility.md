# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Функция мобильности в Skype для бизнеса Server поддерживает функции Skype для бизнеса на мобильных устройствах.
  
Функциональность мобильности в Skype для бизнеса Server та же, что и в Lync Server 2013. Дополнительные сведения о подготовке среды для мобильной работы см. в планировании мобильности [в Lync Server 2013.](https://technet.microsoft.com/library/hh689981%28v=ocs.15%29.aspx)
