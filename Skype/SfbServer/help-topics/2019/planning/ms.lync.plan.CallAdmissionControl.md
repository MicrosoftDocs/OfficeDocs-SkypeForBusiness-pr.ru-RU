# <a name="call-admission-control-planning-tool"></a>Управление приемом вызовов (средство планирования)
 
Контроль допуска звонков на основе доступной пропускной способности сети определяет, следует ли разрешать создание сеансов связи в режиме реального времени, таких как голосовые или видеовызовы.
  
Сведения о планировании управления приемом вызовов см. в материале [Plan for call admission control in Skype для бизнеса Server](../../../plan-your-deployment/enterprise-voice-solution/call-admission-control.md).
