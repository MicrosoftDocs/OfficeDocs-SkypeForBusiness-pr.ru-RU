# <a name="web-conferencing-planning-tool"></a>Веб-конференция (средство планирования)
 
Skype для бизнеса Server веб-конференциации включает совместную работу с документами, совместное использование приложений и совместное использование настольных компьютеров.
  
> [!NOTE]
> Skype для бизнеса Server использует Office веб Office и сервер веб-приложений для обработки общего доступа к PowerPoint презентаций. Функции веб-конференций работают точно так же, как и в Microsoft Lync 2013. Дополнительные сведения об установке и настройке сервера веб Office приложений см. в материале Настройка интеграции с Office web Apps Server и [Lync Server 2013.](/previous-versions/office/lync-server-2013/lync-server-2013-enabling-office-web-apps-server-and-lync-server-2013)
