# <a name="dial-in-conferencing-planning-tool"></a>Dial-in Conferencing (Planning Tool)
 
Конференц-связь с телефонным подключением в Skype для бизнеса Server позволяет пользователям присоединиться к звуковому каналу конференции в Skype для бизнеса Server с помощью коммутируемого телефонного номера, доступного по коммутируемой телефонной сети (PSTN), не требуя поставщика услуг голосовой конференции стороннего производителя.
  
Сведения о планировании включения конференц-связи с телефонным подключением в вашей организации см. в следующих статьях и разделах: 
  
- [Планирование конференций в Skype для бизнеса Server](../../../plan-your-deployment/conferencing/conferencing.md)
