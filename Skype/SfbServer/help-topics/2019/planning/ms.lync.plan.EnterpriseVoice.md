# <a name="enterprise-voice-planning-tool"></a>Корпоративная голосовая связь (средство планирования)
 
Благодаря Корпоративная голосовая связь Skype для бизнеса Server предоставляет автономный протокол VoIP для улучшения или замены традиционных систем УАТ. Корпоративная голосовая связь могут звонить коллегам в сети VoIP или УАПС вашей организации, а также звонить на традиционные телефонные номера за пределами организации. Решение Корпоративная голосовая связь включает такие распространенные функции звонков, как ответ, переадресовка, передача, удержание, перенаправление, освобождение и парковка, а также вызовы Enhanced 9-1-1 (E9-1-1) (E9-1-1) доступны только в США.) Корпоративная голосовая связь также поддерживает широкий спектр текущих и старых IP-устройств и USB-устройств.
  
Дополнительные сведения о планировании Корпоративная голосовая связь см. в [Корпоративная голосовая связь Skype для бизнеса Server.](../../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md) Дополнительные сведения о развертывании отдельного сервера-посредника см. в сведениях о развертывании сервера-посредника в построителье [топологий в Skype для бизнеса Server.](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
