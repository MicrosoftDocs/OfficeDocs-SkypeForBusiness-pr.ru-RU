# <a name="media-bypass-planning-tool"></a>Обход мультимедиа (средство планирования)
 
Обход сервера-посредника означает исключение сервера-посредника из пути (когда это возможно) для звонков, сигналы которых проходят через сервер-посредник. Дополнительные сведения см. в [материале Plan for media bypass in Skype для бизнеса](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
