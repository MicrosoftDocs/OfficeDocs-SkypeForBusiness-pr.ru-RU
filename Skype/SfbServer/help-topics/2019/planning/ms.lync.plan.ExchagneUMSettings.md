# <a name="exchange-unified-messaging-settings-planning-tool-in-skype-for-business-2015"></a>Параметры сообщений (средство планирования) в Скайп для бизнеса 2015 единая Exchange

> [!NOTE]
> Единая система обмена сообщениями Exchange ранее известных как больше недоступен в Exchange 2019, но можно использовать телефонной системой для записи голосовых сообщений и оставьте записи в почтовый ящик Exchange пользователя. [Планирование голосовой почты облачной службы](../../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) более подробные сведения.
