# <a name="select-file-type-extensions"></a>Выбор расширений типов файлов
 
Используйте диалоговое окно **Выбор расширений типов файлов** расширений типов файлов, которые необходимо заблокировать или разрешить, при выборе (блокировать) или снимите (Разрешить) установите флажки рядом с расширениями в **расширений типов файлов**. Щелкните стрелку, чтобы развернуть категорию и просмотреть все расширения типов файлов для этой категории. Если вы хотите выделить все расширения типов файлов для категории, выберите имя категории. 
  
Если вы не отображаются на расширение типа файла, который требуется выбрать, перечисленных в **расширений типов файлов**, введите в текстовом поле в разделе **Добавить новые расширения файлов в список**и нажмите кнопку **Добавить**. 
  
После добавления всех расширений типов файлов, которые будут в список **Выбранные расширения типов файлов** , нажмите **кнопку ОК**.
  
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../manage/manage.md).
