# <a name="select-agents"></a>Выбор агентов

Агенты — это пользователи, назначенные для ответа на вызовы группы ответа. Группе ответа должна быть назначена группа агентов, определяющая, какие агенты могут принимать звонки для данной группы ответа. Один из способов создания группы агентов заключается в определении настраиваемой группы посредством выбора подходящих пользователей. Пользователи, соответствующие требованиям, могут использовать Skype для бизнеса Server и Корпоративная голосовая связь.

Вы используете диалоговое окно **Выбор агентов** для выбора пользователей, добавляемых в группу агентов.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления в диалоговом окне **Выбор агентов**.

- **Поиск** Выполняет поиск SIP-адреса или отображаемой имени пользователя. Введите все или часть адреса или имени. Оставьте поле поиска пустым, чтобы отобразить всех пользователей, для которых включен Skype для бизнеса Server и Корпоративная голосовая связь.

- **Максимальное число отображаемых пользователей** Изменяет количество возвращаемого результата. Используйте этот счетчик, чтобы ограничить поиск, если ожидается большое количество результатов.

В следующем списке описываются поля в диалоговом окне **Выбор агентов**.

- **Агент** Отображает имена пользователей, возвращенные при поиске.

- **SIP-адрес** Отображает SIP-адреса пользователей, возвращенные поиском.

- **Телефония** Отображает значение поля **"Телефония",** определенного для пользователей.

- **Включено** Отображает значение поля **"Включено для Lync Server",** определенное для пользователей.

Дополнительные сведения о работе с группами агентов см. в разделе [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) документации по использованию.
