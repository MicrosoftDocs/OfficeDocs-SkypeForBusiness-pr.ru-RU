# <a name="select-agents"></a>Выбор агентов

Агенты — это пользователей, которые предназначены для отвечать на звонки группы ответа. Группе ответа должна быть назначена группа агентов, определяющая, какие агенты могут принимать звонки для данной группы ответа. Один из способов создания группы агентов заключается в определении настраиваемой группы посредством выбора подходящих пользователей. Право пользователей разрешено для Скайп Business Server и корпоративной голосовой связи.

Вы используете диалоговое окно **Выбор агентов** для выбора пользователей, добавляемых в группу агентов.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления в диалоговом окне **Выбор агентов**.

- **Поиск** Выполняет поиск SIP адрес или отображаемое имя для пользователя. Введите полностью или частично имя или адрес. Не заполняйте поле «Поиск» для отображения всех пользователей, которым разрешена для Скайп Business Server и корпоративной голосовой связи.

- **Максимум пользователей для отображения** Изменяет количество возвращаемых результатов, отображаемых. Используйте этот счетчик для ограничения поиска, если ожидается, что много результатов.

В следующем списке описываются поля в диалоговом окне **Выбор агентов**.

- **Агент** Отображает имена пользователей, возвращенных в результате поиска.

- **SIP-адрес** Отображает возвращенные поиском адреса SIP.

- **Телефонная связь** Отображает значение поля **Телефония** , определенное для пользователей.

- **Включено** Отображает значение поля **разрешено для Lync Server** , определенное для пользователей.

Дополнительные сведения о работе с группами агентов см [Управление группами агентов](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) в документации по операциям.
