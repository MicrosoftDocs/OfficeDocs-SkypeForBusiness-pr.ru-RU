# <a name="select-agents"></a>Выбор агентов

Агенты — это пользователи, которые предназначены для звонков групп ответов. Группе ответа должна быть назначена группа агентов, определяющая, какие агенты могут принимать звонки для данной группы ответа. Один из способов создания группы агентов заключается в определении настраиваемой группы посредством выбора подходящих пользователей. Пользователи с подходящими учетными записью включены для Skype для бизнеса Server и Enterprise Voice.

Вы используете диалоговое окно **Выбор агентов** для выбора пользователей, добавляемых в группу агентов.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления в диалоговом окне **Выбор агентов**.

- **Найти** Поиск адреса SIP или отображаемого имени пользователя. Введите адрес или имя или его часть. Оставьте поле поиска пустым, чтобы отобразить всех пользователей, имеющих доступ к Skype для бизнеса Server и корпоративной голосовой связи.

- **Максимальное число отображаемых пользователей** Изменяет количество отображаемых возвращаемых результатов. Используйте этот счетчик для ограничения поиска, если вы ожидаете большого количества результатов.

В следующем списке описываются поля в диалоговом окне **Выбор агентов**.

- **Агента** Отображаются имена пользователей, возвращаемые функцией поиска.

- **SIP Address** Отображаются адреса SIP пользователя, возвращенные функцией поиска.

- **Телефония** Отображает значение поля **телефонии** , определенного для пользователей.

- **Enabled (включено** ) Отображает значение поля **Enabled для Lync Server** , определенного для пользователей.

Дополнительные сведения о работе с группами агентов см. в разделе [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) документации по использованию.
