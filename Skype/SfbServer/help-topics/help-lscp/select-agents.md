# <a name="select-agents"></a>Выбор агентов

Агенты — это пользователи, назначенные для ответа на вызовы группы ответов. Группе ответа должна быть назначена группа агентов, определяющая, какие агенты могут принимать звонки для данной группы ответа. Один из способов создания группы агентов заключается в определении настраиваемой группы посредством выбора подходящих пользователей. Соответствующие пользователи включены для Skype для бизнеса Server и Корпоративная голосовая связь.

Вы используете диалоговое окно **Выбор агентов** для выбора пользователей, добавляемых в группу агентов.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления в диалоговом окне **Выбор агентов**.

- **Поиск** Поиск адреса SIP или отображения имени пользователя. Введите все или часть адреса или имени. Оставьте поле поиска пустым, чтобы отобразить всех пользователей, включенных для Skype для бизнеса Server и Корпоративная голосовая связь.

- **Максимальное число отображаемых пользователей** Изменяет количество отображаемых результатов. Используйте этот счетчик, чтобы ограничить поиск, если ожидается много результатов.

В следующем списке описываются поля в диалоговом окне **Выбор агентов**.

- **Агент** Отображает имена пользователей, возвращенные поиском.

- **SIP-адрес** Отображает пользовательские SIP-адреса, возвращенные поиском.

- **Телефония** Отображает значение поля **телефонии,** определенного для пользователей.

- **Включено** Отображает значение поля **Enabled for Lync Server,** определенного для пользователей.

Дополнительные сведения о работе с группами агентов см. в разделе [Managing Agent Groups](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-agent-groups) документации по использованию.
