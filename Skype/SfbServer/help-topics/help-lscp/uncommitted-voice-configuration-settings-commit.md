# <a name="uncommitted-voice-configuration-settings-commit"></a>Незафиксированные параметры конфигурации голосовой связи: фиксация
 
Чтобы **просмотреть** все внесенные изменения, можно использовать диалоговое окно "Несфиксировать параметры конфигурации голосовой почты". Прокрутите диалоговое окно, чтобы просмотреть все несохраненные изменения, или щелкните стрелки, чтобы развернуть или свернуть различные разделы. После завершения либо нажмите кнопку **Сохранить**, чтобы сохранить все изменения, либо нажмите кнопку **Отмена**, чтобы вернуться и исправить изменения.
  
> [!IMPORTANT]
> Если нажать кнопку **Сохранить**, будут сохранены **все** изменения, перечисленные в диалоговом окне **Несохраненные параметры настройки голосовой связи**.
  
Подробные сведения о различных процедурах, которые можно выполнить с помощью панели управления Skype для бизнеса Server, см. в под управлением Skype для бизнеса [Server 2015.](../../manage/manage.md)
