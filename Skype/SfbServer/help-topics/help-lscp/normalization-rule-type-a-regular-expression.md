# <a name="normalization-rule-type-a-regular-expression"></a>Правило нормализации: ввод регулярного выражения
 
Введите регулярное выражение .NET Framework, чтобы задать шаблон соответствия для конечных телефонных номеров, к которым применяется маршрут. 
  
Сведения о различных процедурах, которые можно выполнить с помощью панели управления Skype для бизнеса Server, см. в Skype для бизнеса Server [2015 г](../../manage/manage.md).
