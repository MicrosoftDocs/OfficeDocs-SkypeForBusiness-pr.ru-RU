# <a name="call-park"></a>Приостановка вызовов

Когда звонок приостанавливается, он передается во временный номер, в котором звонок удерживается до тех пор, пока кто-то не загружает его или не истечет время ожидания. Вы должны настроить таблицу с диапазонами добавочных номеров, которые вы сохраняете для приостановки звонка. Эти добавочные номера должны быть виртуальными (то есть не иметь назначенного пользователя или назначенный телефон). Каждый пул, на котором запущено приложение для парковки, может иметь один или несколько диапазонов расширений. Эти диапазоны должны быть уникальными в рамках всего развертывания.

На странице "Приостановка **звонка** " отображается список всех диапазонов номеров для приостановки звонка, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Парковка вызовов** можно выполнять следующие задачи:

- создать новый диапазон номеров;

- изменить существующий диапазон номеров;

- удалить диапазон номеров.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новые** Начало нового диапазона номеров для приостановки вызова.

- **Изменить** Открытие выделенного диапазона номеров для редактирования, выделение всех диапазонов номеров в списке или удаление выделенного диапазона номеров.

- **Обновить** Обновляет список диапазонов номеров.

В следующем списке описываются поля на странице.

- **Name (имя** ) Уникальное имя, определяющее диапазон номеров.

- **Начальный диапазон** Начальный номер диапазона.

- **Конечный диапазон** Конечный номер диапазона.

- **Destination (назначение** ) Полное доменное имя (FQDN) или идентификатор службы приложения, в котором размещается номер диапазона, заданное приложением парковки.

Дополнительные сведения о функциях и возможностях приостановки звонка можно найти в статьях [планирование приостановки звонков в Skype для бизнеса 2015](../../plan-your-deployment/enterprise-voice-solution/call-park.md). Подробнее о работе с диапазонами номеров для присоединения к абоненту можно узнать в разделе [Настройка расширений номеров телефонов для вызовов парковки](https://technet.microsoft.com/library/fbf97624-9587-42a6-b276-1b69c574a74d.aspx).
