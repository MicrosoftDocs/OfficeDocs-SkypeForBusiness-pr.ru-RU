# <a name="translation-rule-type-a-regular-expression"></a>Правило преобразования: ввод регулярного выражения
 
В поле **Соответствовать шаблону** укажите шаблон, который будет использоваться для того, чтобы ему соответствовали номера, предназначенные для преобразования. В поле **Правило преобразования** укажите шаблон для формата преобразованных номеров. Например, если ввести ^ (\d \d+)$ в поле "Соответствие этому шаблону" и 011$1 в поле правила трансляции, правило переведет \+ {9} +441235551010 в 011441235551010.  
  
For details about the different procedures that you can perform by using the Skype for Business Server Control Panel, see [Manage Skype for Business Server 2015](../../manage/manage.md).
