# <a name="translation-rule-type-a-regular-expression"></a>Правило преобразования: ввод регулярного выражения
 
В поле **совпадение с этим шаблоном** укажите шаблон, который будет использоваться в качестве соответствия преобразуемым числам. В поле **Правила трансляции** укажите шаблон для формата переведенных номеров. Например\+, если ввести ^ (\d{9}\d +) $ в поле Искать в **этом шаблоне** и 011 $1 в поле **правило перевода** , правило будет переведено + 441235551010 в 011441235551010.
  
Сведения о различных процедурах, которые можно выполнить с помощью панели управления "Skype для бизнеса Server", приведены в статье [Управление Skype для бизнеса server 2015](../../manage/manage.md).
