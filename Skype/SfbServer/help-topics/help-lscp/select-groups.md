# <a name="select-groups"></a>Выбор групп

Очереди группы ответов удерживают вызовы в группах ответов, пока агент не ответит на вызов. Очереди имеют назначенные группы агентов. При создании или изменении очереди вы используете диалоговое окно **Выбор групп** для выбора одной или нескольких групп агентов, которые требуется назначить очереди.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются элементы управления и поля в диалоговом окне **Выбор групп**.

- Поле поиска ищет имя группы агентов. Введите имя полностью или частично. Оставьте поле поиска пустым, чтобы отобразить все группы агентов для своей организации.

- **Имя** Отображает имена возвращаемой группы агентов.

- **Политика участия** Отображает политику участия, определяемую для группы агентов: формальную или неформальную.

- **Метод маршрутивки** Отображает тип маршрутивки, определенный для группы агентов.

Дополнительные сведения о работе с очередями см. в разделе [Managing Response Group Queues](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-queues) документации по использованию.
