# <a name="response-groups-group"></a>Группы ответа: группа

Группы ответа состоят из агентов, очередей и рабочих процессов. Группа агентов — это группа людей, назначенных отвечать на звонки группы ответа.

На **странице**  -  **"Группа ответа"** отображается список всех групп агентов, определенных для организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице "Группа ответа" можно выполнить **следующие**  -   задачи:

- создать новую группу агентов;

- изменить существующую группу агентов;

- удалить группу агентов.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новый** Запускает новую группу агентов.

- **Правка** Открывает выбранную группу агентов для ее редактирования, выбирает все группы агентов в списке или удаляет выбранные группы агентов.

- **Обновление** Обновляет список групп агентов.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, идентифицирует группу агентов.

- **Служба** Служба **ApplicationServer,** в котором размещена группа агентов.

- **Описание** Описание группы агентов.

For details about Response Group features and capabilities, see [Plan for the Response Group application in Skype for Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) in the Planning documentation. Дополнительные сведения о работе с группами агентов см. в разделе [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) документации по использованию.
