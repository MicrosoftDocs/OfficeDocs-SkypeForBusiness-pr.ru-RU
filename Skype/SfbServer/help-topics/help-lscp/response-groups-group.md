# <a name="response-groups-group"></a>Группы ответа: группа

Группы ответа состоят из групп агентов, очереди и рабочие процессы. Группа агентов — группа людей, предназначены для отвечать на звонки группы ответа.

**Группы ответа** - странице**Группа** отображается список всех групп агентов, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Группы ответа**можно выполнить следующие задачи - страница**группы** :

- Создание новой группы агента

- Изменить существующую группу агентов

- Удалить группу агентов

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новые** Запускает новую группу агентов.

- **Изменение** Открывает выбранную группу агентов для ее редактирования, выбирает все группы агентов в списке или удаляет выбранную группу агентов.

- **Обновление** Обновляет список групп агентов.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, которое идентифицирует группу агентов.

- **Службы** Службы **сервера приложений** , на котором размещается группа агентов.

- **Описание** Описание для группы агентов.

Для получения дополнительных сведений о группы ответа функций и возможностей см. в документации по планированию [Планирование приложения группы ответа в Скайп для Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) . Дополнительные сведения о работе с группами агентов см [Управление группами агентов](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) в документации по операциям.
