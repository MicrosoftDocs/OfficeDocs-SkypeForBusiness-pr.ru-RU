# <a name="response-groups-group"></a>Группы ответа: группа

Группы ответа состоят из групп агентов, очередей и рабочих процессов. Группа агента — это группа пользователей, которые предназначены для звонков групп ответов.

На странице - **группы** " **группы ответа**" отображается список всех групп агентов, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице - **группы** " **группы ответа**" можно выполнять следующие задачи:

- Создать новую группу агента

- Изменение существующей группы агента

- Удаление группы агента

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Новые** Запускает новую группу агента.

- **Изменить** Открытие выбранной группы агента для ее изменения, выбор всех групп агента в списке или удаление групп, выбранных агентом.

- **Обновить** Обновляет список групп агента.

В следующем списке описываются поля на странице.

- **Name (имя** ) Уникальное имя, идентифицирующее группу агента.

- **Служба** Служба **аппликатионсервер** , на которой размещена группа агента.

- **Description (описание** ) Описание группы агента.

Дополнительные сведения о функциях и возможностях групп ответов можно найти [в разделе Планирование приложения группы ответа в Skype для бизнеса Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) в документации по планированию. Дополнительные сведения о работе с группами агентов см. в разделе [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) документации по использованию.
