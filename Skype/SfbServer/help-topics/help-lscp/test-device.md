# <a name="test-device"></a>Тестовое устройство

Вы можете добавить устройство проверки на страницу **Устройство проверки** и затем использовать его для проверки функциональных возможностей новых обновлений перед их развертыванием на производственных устройствах. Вы можете протестировать устройство глобально (во всей среде) или на одном сайте. Устройство проверки указывается по MAC-адресу или серийному номеру. При добавлении устройства оно отображается в  списке на странице тестовых устройств панели управления Skype для бизнес-серверов.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Устройство проверки** можно выполнить следующие задачи:

- добавить устройство проверки на глобальном уровне или для отдельного сайта;

- изменить параметры для существующего устройства проверки.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Новые** Вы можете добавить новое тестового устройства со следующей областью:

  - Глобальные

  - Site

- **Изменение** Можно изменить параметры тестового устройства в списке. С помощью этой команды можно выполнить следующие действия:

  - **Демонстрация сведений** Этот параметр открывает диалоговое окно, в котором можно изменить параметры тестового устройства.

  - **Выберите все** Этот параметр выбирает все тестовые устройства в списке.

  - **Удаление** Этот параметр удаляет все выбранные тестовые устройства.

- **Обновление** Вы можете обновить список тестовых устройств, чтобы проверить состояние параметров всех тестовых устройств.

Дополнительные сведения о тестировании устройств см. в разделе [Add a Device to Test Update Functionality](/previous-versions/office/lync-server-2013/lync-server-2013-create-a-device-to-test-update-functionality) документации по использованию.
