# <a name="pin-set-successfully"></a>ПИН-код задан успешно
 
Чтобы отправить PIN-код на адрес электронной почты, нажмите кнопку **открыть приложение электронной почты, чтобы отправить пользователю новый PIN-код**. Чтобы скопировать ПИН-код в сообщение электронной почты, нажмите кнопку **Показать PIN-код**. 
  
Сведения о различных процедурах, которые можно выполнить с помощью панели управления "Skype для бизнеса Server", приведены в статье [Управление Skype для бизнеса server 2015](../../manage/manage.md).
