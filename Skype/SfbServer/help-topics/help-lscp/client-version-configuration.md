# <a name="client-version-configuration"></a>Конфигурация версий клиентов

Кроме того, чтобы указать версию клиентов, которые вы хотите поддерживать в вашей среде, вы также можете задать действие по умолчанию для клиентов, у которых еще не определена политика управления версиями. Это позволяет ограничить использование клиентских версий в среде, что поможет вам управлять затратами, связанными с поддержкой нескольких клиентских версий.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Configuration Version (версия клиента** ) можно выполнить следующие задачи:

- Измените конфигурацию версии клиента по умолчанию ( **Global**).

- Создание конфигурации клиентской версии для определенного сайта.

- Включение и отключение существующих конфигураций версий клиентов.

> [!NOTE]
> Поскольку анонимные пользователи не связаны с сайтом, политики глобального уровня влияют на анонимные пользователи.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Новые** Вы можете создать конфигурацию версии клиента для определенного сайта.

- **Изменить** Вы можете изменить параметры любой из политик версии клиента. С помощью этого параметра можно выполнить следующее:

  - **Показать подробности** Этот параметр открывает диалоговое окно, в котором можно изменить параметры конфигурации клиентской версии.

  - **Выделить все** Этот параметр выбирает все конфигурации клиентской версии в списке.

  - **Delete (удалить** ) Этот параметр удаляет все выбранные конфигурации клиентской версии.

- **Обновить** Вы можете обновить список конфигурации версии клиента, чтобы проверить состояние параметров всех конфигураций версий клиентов.

Дополнительные сведения о взаимодействии клиентов и версиях клиентов см. в разделе [Client Interoperability in Lync 2013 Preview](https://technet.microsoft.com/library/0f126571-91a2-45d5-855c-1e4ddb45fc04.aspx) документации по планированию. Дополнительные сведения о работе с конфигурациями версий клиентов см. в разделе [Modify the Default Action for Clients Not Explicitly Supported or Restricted](https://technet.microsoft.com/library/548dd0f5-62fe-4c3f-8952-2b9fd4c5fff3.aspx) документации по использованию.
