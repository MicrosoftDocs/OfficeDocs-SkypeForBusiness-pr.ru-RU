# <a name="select-an-announcement-service"></a>Выбор службы оповещений
 
В диалоговом окне **Выбор службы извещения** выберите службу объявлений, которая будет обрабатывать входящие звонки на диапазон неназначенных номеров.
  
В поле Поиск введите имя или часть имени службы объявлений, которую вы хотите использовать. Чтобы вернуть все доступные службы объявлений, оставьте поле поиска пустым и щелкните значок "Поиск".
  
В списке служб объявлений, возвращенных поиском, выберите службу, которую вы хотите использовать для обработки входящих звонков, в диапазон неназначенных номеров.
