# <a name="select-an-announcement-service"></a>Выбор службы оповещений
 
Используйте диалоговое окно **Выбор службы извещения** для выбора службы извещения, которая будет обрабатывать входящие звонки для диапазона неназначенных номеров.
  
В поле поиска введите полное или частичное имя службы извещений, которую вы хотите использовать. Чтобы получить все доступные службы извещений, оставьте поле поиска пустым и щелкните значок поиска.
  
В списке возвращенных в результате поиска служб извещений выберите ту службу, которую хотите использовать для обработки входящих звонков для диапазона неназначенных номеров.
