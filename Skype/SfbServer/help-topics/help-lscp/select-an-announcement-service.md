# <a name="select-an-announcement-service"></a>Выбор службы оповещений
 
Используйте диалоговое окно **выбора службы извещения** для выбора службы извещения, который будет обрабатывать входящие звонки для диапазона неназначенных номеров.
  
В поле поиска введите полностью или частично имя службы оповещения, который требуется использовать. Чтобы получить все доступные службы оповещения, не заполняйте поле «Поиск» и щелкните значок поиска.
  
В списке возвращенных в результате поиска служб извещений выберите службу, которую требуется использовать для обработки входящих звонков для диапазона неназначенных номеров.
