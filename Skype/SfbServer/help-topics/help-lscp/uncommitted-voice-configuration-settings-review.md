# <a name="uncommitted-voice-configuration-settings-review"></a>Незафиксированные параметры конфигурации голосовой связи: просмотр
 
Диалоговое окно **Несохраненные параметры настройки голосовой связи** используется для просмотра всех внесенных изменений. Прокрутите диалоговое окно, чтобы просмотреть все несохраненные изменения, или щелкните стрелки, чтобы развернуть или свернуть различные разделы. После завершения нажмите кнопку **Закрыть**. После закрытия диалогового окна можно исправить изменения или сохранить их.
  
For details about the different procedures that you can perform by using the Skype for Business Server Control Panel, see [Manage Skype for Business Server 2015](../../manage/manage.md).
