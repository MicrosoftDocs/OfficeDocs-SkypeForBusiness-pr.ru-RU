# <a name="control-panel---updated-user-search"></a>Панель управления — обновлено: поиск пользователей

Результаты поиска можно использовать для настройки пользователей для Skype для бизнеса Server. Вы можете осуществлять поиск пользователей по отображаемому имени, имени, фамилии, имени учетной записи диспетчера учетных записей безопасности, SIP-адресу или универсальному коду ресурса (URI). Вы также можете искать пользователей с помощью панели управления Lync Server или оснастки пользователей и компьютеров Active Directory.

## <a name="ui-reference"></a>Справочник по элементам пользовательского интерфейса

В следующих списках приведены меню, команды, поля и свойства, имеющиеся на странице **Поиск пользователей**.

### <a name="user-search"></a>Поиск пользователей

- **Поиск** Поиск пользователей по первой части отображаемого имени, имени, фамилии, имени учетной записи SAM, SIP-адреса или строки URI учетной записи пользователя.

- **Поиск по LDAP** Поиск пользователей, введя выражение LDAP.

- **Поле пользователей поиска** Введите пользовательские данные или выражение LDAP, для чего нужно печатать.

- **Поиск** Щелкните, чтобы отобразить пользователей, которые соответствуют значениям поиска, которые вы ввели в поисковой **области** и поле.

- **Открытый запрос** Щелкните, чтобы открыть сохраненный поисковый запрос.

- **Сохранение запроса** Щелкните, чтобы сохранить поисковый запрос.

- **+ Добавление фильтра** Щелкните, чтобы добавить дополнительные критерии поиска.

- **Поля фильтра поиска** Выберите поле, на котором необходимо фильтровать результаты поиска, выберите оператора для запроса и введите строку, на которую необходимо искать.

- **Максимальное число отображаемых пользователей** Введите количество результатов поиска, которые необходимо отобразить, или используйте стрелки вверх и вниз, чтобы указать номер.

Добавьте описательный текст, если требуется.

### <a name="search-results-menus"></a>Меню результатов поиска

- **Включить пользователей** Щелкните, чтобы открыть диалоговое окно ["Пользователи: Новый пользовательский сервер Lync Server",](users-new-lync-server-user.md) где можно добавить нового пользователя в Skype для бизнеса Server.

    Для добавления нового контакта щелкните стрелку вниз, а затем выберите команду **Включить контакты**, чтобы открыть диалоговое окно [Users: New Contact Objects](users-new-contact-objects.md).

- **Изменение** Щелкните **Изменить** и нажмите кнопку **Показать** сведения, чтобы  отобразить сведения выбранного пользователя, или выберите выберите все результаты поиска, чтобы выбрать всех пользователей, отображаемую в таблице результатов.

- **Действие** Щелкните **Действие,** а затем выберите действия, которые необходимо выполнить для выбранных пользователей в результатах поиска. Доступны следующие действия:

  - **Повторное включить для Lync Server** Включает выбранную учетную запись пользователя после ее временной отключения.

  - **Временное отключение для Lync Server** Отключает учетную запись пользователя в Skype для бизнеса Server до повторного ее включания без удаления учетной записи пользователя.

  - **Назначение политик** Открывает [диалоговое окно "Пользователи: назначение](users-assign-policies.md) политик", в котором можно настроить политики, назначаемые пользователю.

  - **Просмотр состояния PIN-кода** Открывает [диалоговое окно "Пользователи: просмотр pin-статуса",](users-view-pin-status.md) в котором отображаются пин-коды для выбранного пользователя.

  - **Настройка PIN-кода** Открывает [диалоговое окно Set](set-pin.md) PIN-код, в котором можно установить ПИН-код для выбранного пользователя.

  - **PIN-код блокировки** Блокирует ПИН-код пользователя.

  - **Разблокировка PIN-кода** Удаляет блокировку пин-кода пользователя.

  - **Удаление с сервера Lync** Удаляет учетную запись пользователя из Skype для бизнеса Server. Пользователь не удаляется из Active Directory.

  - **Удаление сертификата пользователя** Удаляет все сертификаты, предоставленные пользователю.

  - **Перемещение выбранных пользователей в пул** Открывает [диалоговое окно Move User,](move-user.md) в котором можно выбрать пул для перемещения выбранного пользователя.

  - **Перемещение всех пользователей в пул** Открывает [диалоговое окно Move User,](move-user.md) в котором можно выбрать пул для перемещения всех выбранных пользователей.
