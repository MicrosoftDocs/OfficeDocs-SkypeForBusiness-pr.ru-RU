# <a name="clients-mobility-policy"></a>Клиенты: политика мобильных устройств

На **странице Политика** **мобильности** клиентов отображаются политики мобильности, доступные в развертывании.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Политика мобильных устройств** можно выполнить следующие задачи:

- просмотреть существующие политики мобильных устройств;

- создать новую политику мобильных устройств;

- изменить политику мобильных устройств.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.


- **Поле поиска** Введите строку в текстовом окне, чтобы отфильтровать отображаемую политику.

- **Новые** Нажмите **кнопку Новый,** а затем выберите **сайт или** пользователь **для** создания новой политики мобильности.

- **Изменение** Выберите политику и нажмите **кнопку Изменить,** чтобы изменить свойства выбранной политики.

- **Обновление** Обновляет список отображаемой политики. При **нажатии кнопки Обновление** строка фильтра поиска очищается и отображаются все существующие политики.

Дополнительные сведения о настройке политик мобильных устройств см. в разделе [Configuring Mobility Policy](/previous-versions/office/lync-server-2013/lync-server-2013-configuring-mobility-policy).
