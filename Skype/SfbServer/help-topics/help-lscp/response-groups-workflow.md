# <a name="response-groups-workflow"></a>Рабочий процесс групп ответа

Группы ответа состоят из групп агентов, очередей и рабочих процессов. Рабочие процессы группы ответа определяют действия, которые выполняются, когда приложение группы ответа получает телефонный звонок.

На странице " - **Рабочий процесс** **групп ответов**" отображается список всех рабочих процессов группы ответа, определенных для вашей организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице " - **рабочие процессы** **групп ответа**" можно выполнять следующие задачи:

- Создание или изменение рабочего процесса группы слежения

- Создание или изменение интерактивного рабочего процесса

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Создание и изменение рабочего процесса** Открытие средства настройки группы ответа для создания и редактирования рабочего процесса.

- **Обновить** Обновляет список рабочих процессов.

В следующем списке описываются поля на странице.

- **Name (имя** ) Уникальное имя, присвоенное рабочему процессу.

- **Служба** Служба **аппликатионсервер** , на которой размещен рабочий процесс.

- **SIP Address** Адрес SIP группы, который будет отвечать на звонки в рабочий процесс.

- **Телефонный номер** Номер телефона, который вызывается для доступа к этой группе ответа.

- **Language (язык** ) Язык, используемый для распознавания речи и преобразования текста в речь.

- **Интерактивный автоответчик** Указывает, является ли рабочий процесс группой слежения или интерактивным рабочим процессом.

- **Enabled (включено** ) Указывает, активирован ли рабочий процесс для приема звонков.

Дополнительные сведения о функциях и возможностях групп ответов можно найти [в разделе Планирование приложения группы ответа в Skype для бизнеса Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) в документации по планированию. Подробные сведения о работе с рабочими процессами в группе ответа можно найти в разделе [Управление рабочими процессами групп ответов](https://technet.microsoft.com/library/42cfccdd-2844-4875-b4e3-813e1df15f08.aspx) в документации по операциям.
