# <a name="response-groups-workflow"></a>Рабочий процесс групп ответа

Группы ответа состоят из агентов, очередей и рабочих процессов. Рабочий процесс группы ответа определяет действия, которые принимаются приложением группы ответа при приеме телефонного звонка.

На **странице "Рабочий** процесс групп ответа" отображается список всех процессов группы ответа, определенных  -   для организации.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице "Рабочий процесс групп ответа" можно выполнить следующие  -   задачи:

- создать или изменить рабочий процесс сервисной группы;

- создать или изменить интерактивный рабочий процесс;

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.

- **Создание или изменение рабочего процесса** Открывает средство настройки группы ответа для создания или редактирования рабочего процесса.

- **Обновление** Обновляет список рабочего процесса.

В следующем списке описываются поля на странице.

- **Имя** Уникальное имя, назначенное для рабочего процесса.

- **Служба** Служба **ApplicationServer,** в котором размещен рабочий процесс.

- **SIP-адрес** SIP-адрес группы, которая будет отвечать на вызовы рабочего процесса.

- **Телефон** Номер телефона, который будет вызван для достижения этой группы ответа.

- **Язык** Язык, используемый для распознавания речи и текста в речь.

- **IVR** Указывает, является ли рабочий процесс сервисной группой или интерактивным.

- **Включено** Указывает, активирован ли рабочий процесс для получения вызовов.
