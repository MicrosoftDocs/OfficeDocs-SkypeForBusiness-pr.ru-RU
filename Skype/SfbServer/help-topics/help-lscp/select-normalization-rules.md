# <a name="select-normalization-rules"></a>Выбор правил нормализации
 
Правила нормализации сервера используют регулярные выражения .NET Framework для преобразования набранных телефонных номеров в формат E.164. Каждой абонентской группе должно быть назначено хотя бы одно правило нормализации.
  
Чтобы использовать существующее правило нормализации, выберите правило в списке, а затем нажмите кнопку **ОК**.
  
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../manage/manage.md).
