# <a name="unassigned-phone-number-create-new-or-edit-existing"></a>Неназначенный номер телефона: Создание новой или редактирование существующей
 
Неназначенные номера — это телефонные номера, допустимые для вашей организации, но не назначенные пользователю или телефону. В таблице неназначенных номеров указывается, как обрабатываются такие номера.
  
> [!IMPORTANT]
> Завершив создание новый диапазон неназначенных номеров или редактирование существующего, нажмите **кнопку ОК** , чтобы вернуться на страницу **Неназначенный номер** , где перечислены все диапазоны номеров. Изменения, внесенные на странице **New Unassigned номер диапазона** или **Изменение Rage неназначенный номер** страницы не передаются базу данных только после нажатия кнопки **Сохранить все** на странице **Неназначенный номер** .
  
## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются поля на странице.
  
- **Имя** Введите описательное имя, которое определяет диапазон неназначенных номеров. После сохранения диапазон, это имя нельзя изменить.
    
- **Номер диапазона** В первом поле введите первый номер диапазона неназначенных номеров. Во втором поле введите последний номер диапазона.
    
  - Первый номер диапазона должен быть меньше или равен последнему номеру диапазона.
    
  - Если первый номер или последний номер диапазона содержит добавочный номер, то оба номера (первый и последний) должны содержать добавочный номер. Кроме того, добавочный номер должен быть одинаковым для первого и последнего номера диапазона.
    
  - Номер должен соответствовать регулярному выражению (tel:) ? (\+) ? [1-9] \d{0,17}(; ext = [1-9] \d{0,9}) ?. Это означает, что номер может начинаться с строка tel: (если не указать эту строку он автоматически добавляется автоматически), знак плюс (+), а цифры от 1 до 9. Номер телефона может быть длиной до 17 цифр, и в конце может быть указан добавочный номер в формате ";ext=добавочный_номер".
    
- **Служба оповещения** Выберите **оповещение** , приложение объявлений обработки входящих звонков или **Exchange единой системы обмена СООБЩЕНИЯМИ** для Exchange единой системы обмена СООБЩЕНИЯМИ автосекретарь обработки входящих звонков.
    
- Если вы выбрали **оповещение** для **службы оповещения**:
    
  - **Полное доменное имя конечного сервера** Выберите идентификатор службы службы приложения, на котором выполняется приложение оповещения, который будет обрабатывать входящие звонки для этого диапазона неназначенных номеров.
    
  - **Оповещения** Выберите извещение, воспроизводимое для этого диапазона неназначенных номеров.
    
-  Если вы выбрали **Exchange единой системы обмена СООБЩЕНИЯМИ** для **службы оповещения**:
    
  - **Телефонный номер автосекретаря** Выберите номер телефона для единой системы обмена СООБЩЕНИЯМИ автосекретаря Exchange.
