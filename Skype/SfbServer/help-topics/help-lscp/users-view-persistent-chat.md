# <a name="users-view-persistent-chat"></a>Пользователи: Просмотр сохраняемого чата
 
Члены роли **CsPersistentChatAdministrator** (Администраторы сохраняемого чата) могут предоставлять чат права и разрешения менеджера другим пользователям и себе.
  
Persistent Chat Administrator можно выполните следующие действия.
  
- Создайте категории комнат и комнаты чата.
    
- Установите членство во всех категориях и комнатах чата.
    
- Управлять всеми параметрами всех категорий и комнат чата.
    
- Включить или отключить политику для сервера сохраняемого чата.
    
- Задайте и параметры конфигурации в пул серверов сохраняемого чата.
    
Дополнительные сведения см [Добавление Persistent Chat Administrator](http://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx) в документации по развертыванию.
