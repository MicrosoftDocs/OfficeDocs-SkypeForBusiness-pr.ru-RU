# <a name="first-run-checklist-for-skype-for-business-server-control-panel"></a>Контрольный список первого запуска для панели управления Skype для бизнеса Server

Добро пожаловать в панель управления Skype для бизнеса Server, веб-интерфейс пользователя для администрирования и управления Skype для бизнеса Server. Панель управления можно использовать для выполнения типов задач администрирования, которые выполнялись с помощью консоли управления Майкрософт в предыдущих выпусках.

После развертывания Skype для бизнеса Server настоятельно рекомендуется выполнить ряд важных задач. Некоторые из этих задач являются начальными этапами настройки, которые вы, возможно, уже выполняли во время развертывания, а другие — уточнением или изменением параметров, настроенных во время развертывания, или параметров по умолчанию. Другие задачи, описанные в этом разделе, проверяют конфигурации, сделанные в процессе развертывания.

> [!NOTE]
> Перед выполнением задач, описанных в следующей таблице, убедитесь, что вы выполнили вход с использованием правильных прав, разрешений и ролей пользователя, как описано в разделе "Роли и область" статьи "Управление доступом на основе [ролей".](https://technet.microsoft.com/library/41204ba3-ce5b-41a8-a6c3-b444468fa328.aspx)
