# <a name="select-overflow-queue"></a>Выбор очереди переполнения
 
Используйте диалоговое окно **выберите очередь переполнения** для выбора очереди, в которую направляются звонки после заполнения очереди группы ответа.
  
В поле поиска введите полностью или частично имя очереди, которую вы хотите использовать. Чтобы получить все доступные очереди, не заполняйте поле «Поиск» и щелкните значок поиска.
  
В списке возвращенных в результате поиска очередей выберите очередь, которую вы хотите использовать для переадресации звонков после заполнения очереди группы ответа.
