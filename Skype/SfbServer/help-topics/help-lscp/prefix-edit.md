# <a name="prefix-edit"></a>Редактирование префиксов
 
Для определения префиксами или начальных цифр, добавьте префиксы, которые должен обрабатывать этот маршрут, включая + при необходимости. Например можно сопоставить все номера, начиная с +123.
  
Для получения дополнительных сведений о различных процедурах, которые можно выполнять с помощью Скайп для панели управления сервера Business видеть [Управление Скайп для Business Server 2015](../../manage/manage.md).
