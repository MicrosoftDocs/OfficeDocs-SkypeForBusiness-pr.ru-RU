# <a name="select-exchange-um-auto-attendant"></a>Выбор автосекретаря единой системы обмена сообщениями Exchange
 
С помощью диалогового окна " **Выбор автосекретаря UM Exchange** " можно выбрать автосекретарь Exchange UM, который будет обрабатывать входящие звонки на диапазон неназначенных номеров.
  
В поле Поиск введите полное или частичное имя автоматического ассистента UM Exchange, которое вы хотите использовать. Чтобы вернуть все доступные автосекретарей Exchange UM, оставьте поле поиска пустым и щелкните значок "Поиск".
  
В списке автоматических ассистентов UM, возвращенных поиском, выберите автосекретарь, который вы хотите использовать для обработки входящих звонков, в диапазон неназначенных номеров.
