# <a name="persistent-chat-policy-main-page"></a>Главная страница политики сохраняемого чата
 
Вы можете  использовать страницу Политики сохраняемого чата группы сохраняемого чата для управления политиками на глобальном уровне, пуле, сайте или пользователе, включая настройку глобальной политики по умолчанию и создание одной или более дополнительных политик пользователей и сайтов для развертывания.  Если пользователь включен для сохраняемого сервера чата по политике, в клиенте отображается среда Persistent Chat Server.
  
> [!NOTE]
> В топологии политики веб-сайтов сохраняемой сервера чата применяются глобально, для пула пользователя или для каждого пользователя или для каждого пользователя. 
  
Глобальная политика создается автоматически при развертывании сервера сохраняемого чата и может быть настроена, но не удалена. Поскольку глобальная политика применяется ко всем пользователям, ее не нужно устанавливать на одного пользователя.
  
Вы можете создавать и настраивать несколько политик сайта и пользователей, которые вместе с глобальной политикой позволяют пользователям использовать сервер сохраняемого чата. Политики сохраняемого сервера чата пула и сайта переопределяют глобальную политику сохраняемого сервера чата, но только для пользователей этого сайта. Политики пользователей переопределяют как глобальную политику, так и политики пулов и сайтов для тех пользователей, кому назначены такие политики пользователей.
  
> [!NOTE]
> Чтобы настроить и использовать стойкий сервер чата, сначала необходимо использовать Topology Builder, чтобы добавить в топологию поддержку серверов чата, а затем опубликовать топологию. Дополнительные сведения см. в материале [Add Persistent Chat Server Skype для бизнеса Server топологии 2015 г.](../../deploy/deploy-persistent-chat-server/add-persistent-chat-server.md) 
  
## <a name="tasks-that-you-can-perform"></a>Задачи, которые вы можете выполнить

Вы можете выполнить следующие задачи на странице Политика сохраняемого чата: включить и управлять политикой сохраняемого сервера чата. 
  
## <a name="to-configure-the-global-policy-for-persistent-chat"></a>Настройка глобальной политики для сохраняемого чата

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsPersistentChatAdministrator, CsAdministrator или CsUserAdministrator.
    
2. В меню **Пуск** выберите панель управления Skype для бизнеса Server или откройте окно браузера, а затем введите URL-адрес администратора.
    
3. В Skype для бизнеса Server панели управления щелкните **Настойчивый чат** и нажмите **кнопку Упорная политика чата**.
    
4. Выберите **Глобальная** в списке политик, нажмите кнопку **Изменить**, а затем нажмите кнопку **Подробнее**.
    
5. В окне **Изменение политики сохраняемого чата — глобальная политика** выполните следующие действия.
    
   - В поле **Имя** введите новое имя глобальной политики, если вы не хотите использовать политику по умолчанию.
    
   - В **описании** укайте сведения о том, что такое политика пользователя (например, глобальная политика _для centralSiteName)._
    
   - Чтобы контролировать постоянный чат для всех сайтов и пользователей, не контролируемых политикой сайта или политикой пользователей, выберите или очистить окно **Включить постоянный чат.**
    
6. Щелкните **Исполнить**.
    
## <a name="to-create-a-persistent-chat-policy-for-a-site"></a>Создание политики сохраняемого чата для сайта

Для каждого развернутого сайта можно создать политику сохраняемого чата на определенном сайте.
  
Конфигурация в политике сайта переопределяет глобальную политику, но только для того сайта, которому назначена эта политика.
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsPersistentChatAdministrator, CsAdministrator или CsUserAdministrator.
    
2. В меню **Пуск** выберите панель управления Skype для бизнеса Server или откройте окно браузера, а затем введите URL-адрес администратора.
    
3. В левой панели навигации щелкните **Сохраняемый чат** и выберите пункт **Политика сохраняемого чата**.
    
4. Нажмите кнопку **Создать** и щелкните **Политика сайта**.
    
5. В окне **Выбор сайта** щелкните сайт, к которому будет применена политика.
    
6. В окне **Создание политики сохраняемого чата** выполните следующие действия.
    
   - В поле **Имя** введите имя новой политики сайта (например, Redmond).
    
   - В поле **Описание** введите сведения о политике сайта (например, политика комнаты чата для Redmond).
    
   - Чтобы управлять сохраняемым чатом для всех сайтов, которые не контролируются политикой сайта, установите или снимите флажок **Разрешить сохраняемый чат**.
    
7. Нажмите кнопку **Сохранить**.
    
## <a name="to-create-a-user-policy-for-persistent-chat"></a>Создание политики пользователя для сохраняемого чата

В панели Skype для бизнеса Server управления определяются политики пользователей, которые могут быть назначены пользователям в **"Пользователи".**
  
Пользовательская политики переопределяет глобальную политику и политики сайта, но только для пользователей, которым назначена эта политика.
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsPersistentChatAdministrator, CsAdministrator или CsUserAdministrator.
    
2. В меню **Пуск** выберите панель управления Skype для бизнеса Server или откройте окно браузера, а затем введите URL-адрес администратора.
    
3. В левой панели навигации щелкните **Сохраняемый чат** и выберите пункт **Политика сохраняемого чата**.
    
4. Нажмите кнопку **Создать** и щелкните **Политика пользователя**.
    
5. В окне **Создание политики сохраняемого чата** выполните следующие действия.
    
   - В поле **Имя** введите имя новой пользовательской политики.
    
   - В **описании** укачиваем сведения о том, что такое политика пользователя (например, политика сохраняемого чата для конкретного пользователя).
    
   - Чтобы управлять стойким чатом для всех пользователей, которые не контролируются с помощью политики пользователя, выберите или очистить окно Включить настойчивый чат. 
    
6. Щелкните **Исполнить**.
    
## <a name="to-apply-a-persistent-chat-user-policy-to-a-user-account"></a>Применение политики постоянных пользователей чата к учетной записи пользователя

Если пользователь включен для Skype для бизнеса, вы можете применить соответствующие политики к конкретным пользователям, чтобы включить или отключить их для сохраняемой сервера чата.
  
Используйте процедуру в этом разделе для применения ранее созданной политики пользователей сохраняемого чата к одной или несколько учетных записей пользователей или групп пользователей.
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsPersistentChatAdministrator, CsAdministrator или CsUserAdministrator.
    
2. В меню **Пуск** выберите панель управления Skype для бизнеса Server или откройте окно браузера, а затем введите URL-адрес администратора.
    
3. В левой панели навигации щелкните **Пользователи**, а затем выполните поиск учетной записи пользователя, которую требуется настроить.
    
4. В таблице со списком результатов поиска щелкните учетную запись пользователя, нажмите кнопку **Изменить**, а затем щелкните **Показать подробности**.
    
5. В **статье Изменить пользователя Lync Server в** соответствии с политикой сохраняемого чата выберите политику постоянных пользователей чата, которую необходимо применить. 
    
    > [!NOTE]
    > Параметры **\<Automatic\>** применяют эффективную политику по умолчанию. Данный параметр автоматически применяется сервером.
  
6. Нажмите кнопку **Сохранить**.
