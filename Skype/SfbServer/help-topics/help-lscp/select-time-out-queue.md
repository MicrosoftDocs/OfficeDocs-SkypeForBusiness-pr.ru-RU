# <a name="select-time-out-queue"></a>Выбор времени ожидания для очереди
 
Используйте диалоговое окно **Выберите время ожидания для очереди** для выбора очереди, в которую направляются звонки после истечения их времени ожидания.
  
В поле поиска введите полное или частичное имя запроса, который вы хотите использовать. Чтобы получить все доступные запросы, оставьте поле поиска пустым и щелкните значок поиска.
  
В списке возвращенных в результате поиска очередей выберите ту очередь, которую хотите использовать для звонков с истекшим временем ожидания.
