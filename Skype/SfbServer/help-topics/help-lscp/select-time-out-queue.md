# <a name="select-time-out-queue"></a>Выбор времени ожидания для очереди
 
В диалоговом окне " **Выбор очереди** ожидания" можно выбрать очередь, в которую пересылаются звонки по истечении времени ожидания.
  
В поле Поиск введите имя нужной очереди или ее часть. Чтобы вернуть все доступные очереди, оставьте поле поиска пустым и щелкните значок "Поиск".
  
В списке очередей, возвращенных поиском, выберите очередь, которую вы хотите использовать для звонков по тайм-стороне.
