# <a name="select-languages"></a>Выбор языков

Выберите языки, которые требуется использовать для номера доступа к конференц-связи с телефонным подключением. Дополнительные сведения см. в разделе [Create or Modify a Dial-in Conferencing Access Number](/previous-versions/office/lync-server-2013/lync-server-2013-create-or-modify-a-dial-in-conferencing-access-number).
