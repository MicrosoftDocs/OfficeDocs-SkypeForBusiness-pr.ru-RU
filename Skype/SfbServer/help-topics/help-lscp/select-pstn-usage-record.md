# <a name="select-pstn-usage-record"></a>Выбор записи использования ТСОП
 
Используйте диалоговое окно **Выбрать запись об использовании ТСОП** для поиска существующего режима работы с телефонной сетью общего пользования (ТСОП), который вы могли бы использовать.
  
> [!NOTE]
> Чтобы получить все доступные режимы работы с ТСОП, ничего не вводите в поле поиска и нажмите значок поиска. 
  
For details about the different procedures that you can perform by using the Skype for Business Server Control Panel, see [Manage Skype for Business Server 2015](../../manage/manage.md).
