При использовании RBAC административные привилегии выдаются с помощью назначения пользователям предварительно заданных административных ролей. Skype для бизнеса Server включает в себя богатый набор встроенных административных ролей, а также позволяет создавать новые роли и указывать настраиваемый список cmdlets для каждой новой роли. Вы можете также добавлять сценарии командлетов в разрешенные задачи как предопределенных, так и пользовательских ролей RBAC. Подробные сведения, в том числе предопределяемая административная роль, доступные в Skype для бизнеса Server, см. в Role-Based Access Control.
