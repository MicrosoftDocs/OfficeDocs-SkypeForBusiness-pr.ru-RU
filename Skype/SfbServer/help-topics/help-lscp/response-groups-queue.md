# <a name="response-groups-queue"></a>Очередь группы ответа
 
Группы ответа состоят из групп агентов, очереди и рабочие процессы. Очереди группы ответа хранят звонки в группу ответа, пока агент отвечает на звонок. 
  
**Группы ответа** - странице**очередь** отображается список всех очередей группы ответа, определенных для вашей организации.
  
## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Группы ответа**можно выполнить следующие задачи - странице**очередь** :
  
- Создать новую очередь
    
- Изменить существующую очередь
    
- Удалить очередь.
    
## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующем списке описываются команды на странице.
  
- **Новые** Создает новую очередь.
    
- **Изменение** Открывает выбранную очередь для ее редактирования, выбирает все очереди в списке или удаляет выбранную очередь.
    
- **Обновление** Обновляет список очередей.
    
В следующем списке описываются поля на странице.
  
- **Имя** Уникальное имя очереди.
    
- **Службы** Службы **сервера приложений** , на котором размещается очередь.
    
- **Описание** Описание очереди.
    
Для получения дополнительных сведений о группы ответа функций и возможностей см. в документации по планированию [Планирование приложения группы ответа в Скайп для Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) . Для получения дополнительных сведений о работе с очередями группы ответа содержатся в документации по операциям [Управление очередями группы ответа](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) .
