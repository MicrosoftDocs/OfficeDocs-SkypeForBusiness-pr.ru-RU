# <a name="client-version-policy"></a>Политика версий клиентов

Вы может указать версию клиентов, поддерживаемых в вашей среде. При взаимодействии двух клиентов с разными версиями возможности, доступные обоим клиентам, могут быть ограничены возможностями другого клиента. Чтобы максимально использовать функции, включенные в Skype для бизнеса Server 2015 г., и улучшить общий пользовательский интерфейс, можно использовать фильтр клиентской версии, чтобы ограничить клиентские версии, используемые в среде. Используя этот фильтр, вы также можете сократить расходы, связанные с поддержкой нескольких версий клиентов.

## <a name="tasks-you-can-perform"></a>Задачи, которые вы можете выполнить

На странице **Политика версий клиента** можно выполнить следующие задачи:

- Изменить политику клиентской версии по умолчанию **(Global).**

- создать политики версий клиентов для конкретного сайта или пула;

- создать политики версий клиентов, которые можно назначить отдельным пользователям.

> [!NOTE]
> Поскольку анонимные пользователи не сопоставлены с пользователем, сайтом или службой, они управляются только политиками глобального уровня.

## <a name="ui-reference"></a>Ссылка на пользовательский интерфейс

В следующих списках описываются меню, команды, поля и свойства на этой странице.

- **Новые** Можно создать одну или несколько политик каждой из следующих клиентских версий:

  - Политика сайта

  - Политика пула

  - Политика пользователя

- **Изменение** Вы можете изменить параметры любой из политик клиентской версии. С помощью этой команды можно выполнить следующие действия:

  - **Демонстрация сведений** Этот параметр открывает диалоговое окно, в котором можно изменить параметры политики клиентской версии.

  - **Выберите все** Этот параметр выбирает все политики клиентской версии в списке.

  - **Удаление** Этот параметр удаляет все выбранные политики клиентской версии.

- **Обновление** Вы можете обновить список политик клиентской версии, чтобы проверить состояние параметров всех политик клиентских версий.

Дополнительные сведения о работе с политиками версий клиентов см. в разделе [Specify the Client Versions Supported in Your Organization](/previous-versions/office/lync-server-2013/lync-server-2013-specifying-the-client-applications-that-can-be-used-to-log-on-to-lync-server-2013) в документации по применению.
