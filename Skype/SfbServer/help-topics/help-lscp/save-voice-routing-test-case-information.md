# <a name="save-voice-routing-test-case-information"></a>Сохранение сведений тестового сценария маршрутизации голосовых данных
 
Вы можете использовать диалоговое окно " **Сохранение сведений о тестовых сценариях голосовой связи** " для сохранения тестового случая атемпорари голосовой маршрутизации. В разделе **Сохранение сведений о тестовых случаях голосовой маршрутизации** **введите имя для** этого тестового случая и нажмите кнопку **ОК**. 
  
> [!NOTE]
> При сохранении временного тестового случая вы не сохраняете его в хранилище Центрального управления. Вместо этого тестовый случай будет доступен на вкладке " **Тестовая Маршрутизация голосовой связи** " на странице " **Маршрутизация голосовых сообщений** ". Тестовый случай будет отображаться с **незафиксированным**состоянием. Чтобы сохранить тестовый случай в хранилище Центрального управления, необходимо выбрать пункт **зафиксировать все** из любого меню **действий** на странице **маршрутизации голосовых** сообщений.
  
Сведения о различных процедурах, которые можно выполнить с помощью панели управления "Skype для бизнеса Server", приведены в статье [Управление Skype для бизнеса server 2015](../../manage/manage.md).
