# <a name="manage-archiving-options-in-skype-for-business-server"></a>Управление вариантами архива в Skype для бизнеса Server

**Сводка:** Узнайте, как настроить параметры архива для Skype для бизнеса Server.
  
Изначально архивация настраивается при развертывании, но после развертывания можно изменять, добавлять и удалять конфигурации. Параметры архива определяют, следует ли: 
  
- Включение или отключение архивации
    
- Архивировать сеансы обмена мгновенными сообщениями
    
- Архивировать сеансы веб-conferencing
    
- Блокировка действий, когда архивная работа недоступна
    
- Использование интеграции Exchange
    
- Настройка и экспорт данных
    
Параметры конфигурации можно указать на следующих уровнях:
  
- Конфигурация глобального уровня, которая создается по умолчанию при развертывании Skype для бизнеса Server
    
- Необязательные конфигурации на уровне сайта, определяющие реализации архивации для определенного сайта
    
- Необязательные конфигурации уровня пула, определяющие реализации архивации для определенного пула
    
Вы можете удалить конфигурацию сайта или пула, но не глобальную конфигурацию. Если вы все же удалите глобальную конфигурацию, параметрам будут автоматически присвоены значения по умолчанию. For details about how archiving configurations are implemented and the hierarchy of archiving configurations, see [Plan for archiving in Skype for Business Server.](../../plan-your-deployment/archiving/archiving.md)
  
## <a name="configure-archiving-options-by-using-the-control-panel"></a>Настройка параметров архива с помощью панели управления

С помощью панели управления можно настроить параметры архива следующим образом:
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator. 
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
    
3. В левой панели навигации щелкните **"Конфигурация архивации".**
    
## <a name="configure-archiving-options-by-using-windows-powershell"></a>Настройка параметров архива с помощью Windows PowerShell

Вы также можете настроить параметры архива с помощью Windows PowerShell, перечисленных в следующей таблице. Подробные сведения о синтаксисе, включая все доступные параметры, см. в руководстве [Skype для бизнеса Server Management Shell.](../management-shell.md)
  

|**Командлет**|**Описание**|
|:-----|:-----|
|Get-CsArchivingConfiguration  <br/> |Возвращает сведения о параметрах конфигурации архивации в организации.  <br/> |
|New-CsArchivingConfiguration  <br/> |Создает новый набор параметров обмена мгновенными сообщениями, которые можно использовать для автоматического сохранения сеансов обмена мгновенными сообщениями и блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Remove-CsArchivingConfiguration  <br/> |Удаляет указанную коллекцию параметров архива, используемых для автоматического сохранения сеансов обмена мгновенными сообщениями, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Set-CsArchivingConfiguration  <br/> |Изменяет существующую коллекцию параметров конфигурации архивации обмена мгновенными сообщениями.  <br/> |
