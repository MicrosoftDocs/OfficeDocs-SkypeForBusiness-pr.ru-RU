# <a name="manage-archiving-options-in-skype-for-business-server"></a>Управление вариантами архива в Skype для бизнеса Server

**Сводка:** Узнайте, как настроить параметры архива для Skype для бизнеса Server.
  
Сначала вы настраивали архивацию при развертывании, но после развертывания можно изменять, добавлять и удалять конфигурации. Параметры архива определяют, следует ли: 
  
- Включение или отключение архивации
    
- Архивировать сеансы обмена мгновенными сообщениями
    
- Сеансы веб-конференций архива
    
- Блокировка действий, когда архивативная работа недоступна
    
- Использование Exchange интеграции
    
- Настройка чистки и экспорта данных
    
Параметры конфигурации можно указать на следующих уровнях:
  
- Конфигурация глобального уровня, созданная по умолчанию при развертывании Skype для бизнеса Server
    
- Необязательные конфигурации на уровне сайта, определяющие, как выполняется архивация для определенного сайта
    
- Необязательные конфигурации уровня пула, определяющие реализацию архивации для определенного пула
    
Вы можете удалить конфигурацию сайта или конфигурацию пула, но не можете удалить глобальную конфигурацию. Если вы все же удалите глобальную конфигурацию, параметрам будут автоматически присвоены значения по умолчанию. Сведения о реализации конфигураций архивации и иерархии конфигураций архивации см. в материале [Plan for archiving in Skype для бизнеса Server](../../plan-your-deployment/archiving/archiving.md).
  
## <a name="configure-archiving-options-by-using-the-control-panel"></a>Настройка параметров архива с помощью панели управления

Параметры архива можно настроить с помощью панели управления следующим образом:
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator. 
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
    
3. В левой панели навигации щелкните **Конфигурация архивации**.
    
## <a name="configure-archiving-options-by-using-windows-powershell"></a>Настройка параметров архива с помощью Windows PowerShell

Вы также можете настроить параметры архива с помощью Windows PowerShell, перечисленных в следующей таблице. Подробные сведения о синтаксисе, включая все доступные параметры, [см. в Skype для бизнеса Server Management Shell](../management-shell.md).
  

|**Командлет**|**Описание**|
|:-----|:-----|
|Get-CsArchivingConfiguration  <br/> |Возвращает сведения о параметрах конфигурации архивации в организации.  <br/> |
|New-CsArchivingConfiguration  <br/> |Создает новый набор параметров обмена мгновенными сообщениями, которые можно использовать для автоматической экономии сеансов обмена мгновенными сообщениями или отключения их, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Remove-CsArchivingConfiguration  <br/> |Удаляет указанный набор параметров архива, которые используются для автоматического сохранения сеансов обмена мгновенными сообщениями или отключения, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Set-CsArchivingConfiguration  <br/> |Изменяет существующую коллекцию параметров конфигурации архивации мгновенных сообщений (IM).  <br/> |
