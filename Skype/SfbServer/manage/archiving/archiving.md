# <a name="manage-archiving-in-skype-for-business-server"></a>Управление архивативией в Skype для бизнеса Server

**Сводка:** Узнайте, как управлять архивативией для Skype для бизнеса Server.
  
При развертывании архивации для организации указывается начальная конфигурация во время развертывания. Однако могут возникнуть моменты, когда необходимо изменить реализации поддержки архива для ежедневного управления или для удовлетворения новых требований к организации. Например, может потребоваться настроить поддержку архива по-разному для определенного сайта, определенного пула или определенных пользователей в организации. Для пользователей, Skype для бизнеса Server на сайте, для этого необходимо создать и настроить параметры конфигурации архивации и политики пользователей. 
  
Перед чтением этой темы убедитесь, что вы знакомы с информацией в Plan [for archiving in Skype для бизнеса Server](../../plan-your-deployment/archiving/archiving.md) и Deploy [archiving for Skype для бизнеса Server](../../deploy/deploy-archiving/deploy-archiving.md).
  
> [!NOTE]
> Если вы включаете интеграцию microsoft Exchange для развертывания, Exchange политики контролируют, включена ли архивация для пользователей, которые размещены в Exchange и наложены ли их почтовые ящики In-Place Hold. Подробные сведения см. в материале [Plan for archiving in Skype для бизнеса Server](../../plan-your-deployment/archiving/archiving.md) и [Configure integration with Exchange для Skype для бизнеса Server.](../../deploy/deploy-archiving/configure-integration-with-exchange-storage.md) 
  
## <a name="archiving-configuration-options"></a>Параметры конфигурации архивации

Параметры конфигурации архивации указывают, следует ли:
  
- Включение или отключение архивации
    
- Архивировать сеансы обмена мгновенными сообщениями
    
- Сеансы веб-конференций архива
    
- Блокировка действий, когда архивативная работа недоступна
    
- Использование Exchange интеграции
    
- Настройка чистки и экспорта данных
    
Эти параметры можно установить на глобальном уровне, сайте или пуле. Дополнительные сведения см. в дополнительных сведениях Об управлении вариантами архива [в Skype для бизнеса Server.](options.md)
  
## <a name="archiving-policies"></a>Политики архива

Политики архива определяют, следует ли архивировать следующие:
  
- внутренние коммуникации;
    
- внешние коммуникации.
    
Эти политики можно установить на глобальном уровне, на сайте или на уровне пользователя. Дополнительные сведения см. в рублях Управление политиками архива [в Skype для бизнеса Server.](policies.md)
  
## <a name="manage-archiving-by-using-the-control-panel-or-by-using-windows-powershell"></a>Управление архивативом с помощью панели управления или с помощью Windows PowerShell

Вы можете управлять архиватива с помощью панели управления или с помощью Windows PowerShell. В следующей таблице приводится сводка доступных для управления архивалетами. Подробные сведения о синтаксисе, включая все доступные параметры, [см. в Skype для бизнеса Server Management Shell.](../management-shell.md) 


|**Командлет**|**Описание**|
|:-----|:-----|
|Export-CsArchivingData  <br/> |Экспортирует записи, хранимые в Skype для бизнеса Server архива.  <br/> |
|Get-CsArchivingConfiguration  <br/> |Возвращает сведения о параметрах конфигурации архивации в организации.  <br/> |
|Get-CsArchivingPolicy  <br/> |Возвращает сведения о политиках архивизации организации для внутренних и внешних коммуникаций.  <br/> |
|Grant-CsArchivingPolicy  <br/> |Назначает политика архивания сеансов обмена мгновенными сообщениями пользователям или наборам пользователей. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями и между внутренними и внешними пользователями.  <br/> |
|Invoke-CsArchivingDatabasePurge  <br/> |Вручную очищает записи из базы данных архивации.  <br/> |
|New-CsArchivingConfiguration  <br/> |Создает новый набор параметров обмена мгновенными сообщениями, которые можно использовать для автоматической экономии сеансов обмена мгновенными сообщениями или отключения их, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|New-CsArchivingPolicy  <br/> |Создает новые политики архивации сеансов обмена мгновенными сообщениями. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями, а также все сеансы обмена мгновенными сообщениями между внутренними пользователями и внешними партнерами.  <br/> |
|Remove-CsArchivingConfiguration  <br/> |Удаляет указанный набор параметров архива, которые используются для автоматического сохранения сеансов обмена мгновенными сообщениями или отключения, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Remove-CsArchivingPolicy  <br/> |Удаляет указанную политику архива обмена мгновенными сообщениями, которая определяет, будет ли Skype для бизнеса Server автоматически сохранять все сеансы обмена мгновенными сообщениями, которые проходят между внутренними пользователями, и(или) все сеансы обмена мгновенными сообщениями между внутренними пользователями и федеративными партнерами.  <br/> |
|Set-CsArchivingConfiguration  <br/> |Изменяет существующую коллекцию параметров конфигурации архивации мгновенных сообщений (IM).  <br/> |
|Set-CsArchivingPolicy  <br/> |Изменяет существующую политику архива обмена мгновенными сообщениями (IM). Политика архива позволяет архивировать все сеансы и конференции im, которые проходят между внутренними пользователями; вы также можете архивировать сеансы, которые проходят между внутренними пользователями и федерадными партнерами.  <br/> |
|Set-CsArchivingServer  <br/> |Позволяет указать новое расположение базы данных для одного или более серверов архивации.  <br/> |
