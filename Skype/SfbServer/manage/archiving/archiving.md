# <a name="manage-archiving-in-skype-for-business-server"></a>Управление архиватива в Skype для бизнеса Server

**Сводка:** Learn how to manage archiving for Skype for Business Server.
  
При развертывании архивации для организации указывается начальная конфигурация во время развертывания. Однако иногда может потребоваться изменить реализации поддержки архивизации для ежедневного управления или для удовлетворения новых требований организации. Например, может потребоваться настроить поддержку архива по-разному для определенного сайта, определенного пула или отдельных пользователей в организации. Для пользователей Skype для бизнеса Server это можно сделать, создав и настроив параметры конфигурации архивации и политики пользователей. 
  
Прежде чем прочитать этот раздел, ознакомьтесь с информацией в разделе "Планирование архивных данных в Skype для бизнеса [Server"](../../plan-your-deployment/archiving/archiving.md) и "Развертывание архива для [Skype для бизнеса Server".](../../deploy/deploy-archiving/deploy-archiving.md)
  
> [!NOTE]
> Если для развертывания включена интеграция Microsoft Exchange, политики Exchange контролируют, включена ли архивация для пользователей, которые размещены в Exchange и для их почтовых ящиков включено In-Place удержание. For details, [see Plan for archiving in Skype for Business Server](../../plan-your-deployment/archiving/archiving.md) and [Configure integration with Exchange storage for Skype for Business Server.](../../deploy/deploy-archiving/configure-integration-with-exchange-storage.md) 
  
## <a name="archiving-configuration-options"></a>Параметры конфигурации архивации

Параметры конфигурации архивации определяют, следует ли:
  
- Включение или отключение архивации
    
- Архивировать сеансы обмена мгновенными сообщениями
    
- Архивировать сеансы веб-conferencing
    
- Блокировка действий, когда архивная работа недоступна
    
- Использование интеграции Exchange
    
- Настройка и экспорт данных
    
Эти параметры можно установить на глобальном уровне, на уровне сайта или пула. Дополнительные сведения см. в под управлением [параметров архива в Skype для бизнеса Server.](options.md)
  
## <a name="archiving-policies"></a>Политики архива

Политики архива определяют, следует ли архивировать следующее:
  
- внутренние коммуникации;
    
- внешние коммуникации.
    
Эти политики можно установить на глобальном уровне, на уровне сайта или пользователя. Дополнительные сведения см. в сведениях об управлении политиками [архива в Skype для бизнеса Server.](policies.md)
  
## <a name="manage-archiving-by-using-the-control-panel-or-by-using-windows-powershell"></a>Управление архивативами с помощью панели управления или с помощью Windows PowerShell

Архивом можно управлять с помощью панели управления или с помощью Windows PowerShell. В следующей таблице приводится сводка по доступным для управления архива данными. Подробные сведения о синтаксисе, включая все доступные параметры, см. в руководстве [Skype для бизнеса Server Management Shell.](../management-shell.md) 


|**Командлет**|**Описание**|
|:-----|:-----|
|Export-CsArchivingData  <br/> |Экспортирует записи, хранимые в базе данных архивов Skype для бизнеса Server.  <br/> |
|Get-CsArchivingConfiguration  <br/> |Возвращает сведения о параметрах конфигурации архивации в организации.  <br/> |
|Get-CsArchivingPolicy  <br/> |Возвращает сведения о политиках архива организации для внутренней и внешней связи.  <br/> |
|Grant-CsArchivingPolicy  <br/> |Назначает политики архива сеанса обмена мгновенными сообщениями пользователям или наборам пользователей. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями и между внутренними и внешними пользователями.  <br/> |
|Invoke-CsArchivingDatabasePurge  <br/> |Вручную очищает записи из базы данных архивации.  <br/> |
|New-CsArchivingConfiguration  <br/> |Создает новый набор параметров обмена мгновенными сообщениями, которые можно использовать для автоматического сохранения сеансов обмена мгновенными сообщениями и блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|New-CsArchivingPolicy  <br/> |Создает новые политики архивации сеансов обмена мгновенными сообщениями. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями, а также все сеансы обмена мгновенными сообщениями между внутренними пользователями и внешними партнерами.  <br/> |
|Remove-CsArchivingConfiguration  <br/> |Удаляет указанную коллекцию параметров архива, используемых для автоматического сохранения сеансов обмена мгновенными сообщениями, а также для блокировки любых мгновенных сообщений, которые невозможно архивировать.  <br/> |
|Remove-CsArchivingPolicy  <br/> |Удаляет указанную политику архива обмена мгновенными сообщениями, которая определяет, будет ли Skype для бизнеса Server автоматически сохранять все сеансы обмена мгновенными сообщениями между внутренними пользователями и(или) все сеансы обмена мгновенными сообщениями между внутренними пользователями и федеративными партнерами.  <br/> |
|Set-CsArchivingConfiguration  <br/> |Изменяет существующую коллекцию параметров конфигурации архивации обмена мгновенными сообщениями.  <br/> |
|Set-CsArchivingPolicy  <br/> |Изменяет существующую политику архива обмена мгновенными сообщениями. Политика архива позволяет архивировать все сеансы и конференции im, которые проходят между внутренними пользователями; можно также архивировать сеансы, которые проходят между внутренними пользователями и федератными партнерами.  <br/> |
|Set-CsArchivingServer  <br/> |Позволяет указать новое расположение базы данных для одного или более серверов архивации.  <br/> |
