# <a name="manage-archiving-policies-in-skype-for-business-server"></a>Управление политиками архива в Skype для бизнеса Server

**Сводка:** Узнайте, как управлять политиками пользователей для архива для Skype для бизнеса Server.
  
Вы изначально создали политики архивации при развертывании архивации, но после развертывания можно изменить, добавить и удалить конфигурации. Политики архива определяют, следует ли архивировать: 
  
- внутренние коммуникации;
    
- внешние коммуникации.
    
Политики архива можно установить на глобальном уровне, на сайте или на уровне пользователя.
  
> [!NOTE]
> Если вы включили Exchange Microsoft для развертывания, Exchange политики контролируют, включена ли архивация для пользователей, которые размещены в Exchange и чтобы их почтовые ящики были In-Place Hold. Подробные сведения см. в материале [Plan for archiving in Skype для бизнеса Server](../../plan-your-deployment/archiving/archiving.md) и [Configure integration with Exchange для Skype для бизнеса Server.](../../deploy/deploy-archiving/configure-integration-with-exchange-storage.md) 
  
## <a name="manage-archiving-policies-by-using-the-control-panel"></a>Управление политиками архива с помощью панели управления

Вы можете управлять политиками архива, используя панель управления следующим образом:
  
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsArchivingAdministrator или CsAdministrator. 
    
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
    
3. В левой панели навигации нажмите кнопку **Политика архивации**
    
## <a name="manage-archiving-policies-by-using-windows-powershell"></a>Управление политиками архива с помощью Windows PowerShell

Можно также настроить политики архива с помощью Windows PowerShell, перечисленных в следующей таблице. Подробные сведения о синтаксисе, включая все доступные параметры, [см. в Skype для бизнеса Server Management Shell.](../management-shell.md)
  

|**Командлет**|**Описание**|
|:-----|:-----|
|Get-CsArchivingPolicy  <br/> |Возвращает сведения о политиках архивизации сеансов обмена мгновенными сообщениями организации.  <br/> |
|Grant-CsArchivingPolicy  <br/> |Назначает политика архивания сеансов обмена мгновенными сообщениями пользователям или наборам пользователей. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями и между внутренними и внешними пользователями.  <br/> |
|New-CsArchivingPolicy  <br/> |Создает новые политики архивации сеансов обмена мгновенными сообщениями. Эти политики позволяют архивировать все сеансы обмена мгновенными сообщениями между внутренними пользователями, а также все сеансы обмена мгновенными сообщениями между внутренними пользователями и внешними партнерами.  <br/> |
|Remove-CsArchivingPolicy  <br/> |Удаляет указанную политику архива обмена мгновенными сообщениями, которая определяет, будет ли Skype для бизнеса Server автоматически сохранять все сеансы обмена мгновенными сообщениями, которые проходят между внутренними пользователями, и(или) все сеансы обмена мгновенными сообщениями между внутренними пользователями и федеративными партнерами.  <br/> |
|Set-CsArchivingPolicy  <br/> |Изменяет существующую политику архива обмена мгновенными сообщениями (IM). Политика архива позволяет архивировать все сеансы и конференции im, которые проходят между внутренними пользователями; вы также можете архивировать сеансы, которые проходят между внутренними пользователями и федерадными партнерами.  <br/> |
