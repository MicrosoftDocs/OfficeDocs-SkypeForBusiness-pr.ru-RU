# <a name="manage-user-accounts-for-skype-for-business-server"></a>Управление учетной записью пользователя для Skype для бизнеса Server

В разделах этой статьи описывается, как включить, временно отключить или удалить пользователей Active Directory из Skype для бизнеса Server.

Сведения о том, как включить пользователя Active Directory, см. в подстановок ["Создание новой учетной записи пользователя".](https://technet.microsoft.com/library/cc732336%28v=ws.11%29.aspx) Сведения об удалении пользователя Active Directory см. в [записи "Удаление учетной записи пользователя".](https://technet.microsoft.com/library/cc753730%28v=ws.11%29.aspx)

Эти процедуры следует выполнять в период обслуживания, когда уровень использования Skype для бизнеса самый низкий. То, будет ли это сделано по расписанию ежедневно или еженедельно, будет определяться потребностями вашей организации.

В этой статье представлены следующие процедуры:

- [Поиск одного или нескольких пользователей](user-accounts.md#Search)

- [Добавление и добавление нового пользователя Skype для бизнеса Server](user-accounts.md#Add)

- [Отключение или повторное отключение учетной записи пользователя, ранее включенной для Skype для бизнеса Server](user-accounts.md#Disable)

- [Отключение пользователя для Корпоративная голосовая связь](user-accounts.md#Disable_EV)

- [Удаление учетной записи пользователя с помощью оболочки управления Skype для бизнеса Server](user-accounts.md#Remove)

## <a name="to-search-for-one-or-more-users"></a>Поиск одного или нескольких пользователей
<a name="Search"> </a>

Результаты поискового запроса можно использовать для настройки пользователей Active Directory для Skype для бизнеса Server. Вы можете осуществлять поиск пользователей по отображаемому имени, имени, фамилии, имени учетной записи диспетчера учетных записей безопасности, SIP-адресу или универсальному коду ресурса (URI).

Вы можете искать пользователей с помощью панели управления Skype для бизнеса Server или оснастки "Пользователи и компьютеры Active Directory". В следующей процедуре описывается использование панели управления Skype для бизнеса Server для поиска пользователей.

> [!NOTE]
> В среде с топологией центрального леса результаты поиска могут быть не точными при поиске пользователя по его адресу электронной почты. Вместо этого вы можете выполнить поиск пользователей, указав префикс SIP-адреса, например, sip:имя, добавить фильтр поиска и выбрать SIP-адрес с частичным адресом электронной почты, а также воспользоваться командлетом **Get-CSUser**.

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.

3. В левой панели навигации щелкните элемент **Users** (Пользователи).

4. В поле **Search users** (Поиск пользователей) полностью или частично введите отображаемое имя, имя, фамилию, имя учетной записи диспетчера учетных записей безопасности, SIP-адрес или URI искомой учетной записи пользователя Active Directory, а затем нажмите кнопку **Find** (Найти).

5. (Необязательно) Укажите дополнительные условия поиска, чтобы уменьшить количество результатов:

   а. Нажмите кнопку со стрелкой развертывания в верхнем правом углу экрана над элементом **Search results** (Результаты поиска) и нажмите кнопку **Add Filter** (Добавить фильтр).

   б. Укажите свойство пользователя, введя его вручную или щелкнув стрелку в раскрывающемся списке и выбрав свойство в этом списке.

   c. В списке **Equal to** (Равно) щелкните элемент **Equal to** (Равно) или **Not equal to** (Не равно).

   г. В текстовом поле введите требуемые условия поиска для фильтрации результатов, а затем нажмите кнопку **Find** (Найти).

6. Результаты поиска отображаются под элементом **Search Results** (Результаты поиска). Вы можете выбрать любых или всех пользователей в списке и выполнить для них задачи настройки.

## <a name="add-and-enable-a-new-skype-for-business-server-user"></a>Добавление и добавление нового пользователя Skype для бизнеса Server
<a name="Add"> </a>

После включения учетной записи пользователя в службе "Пользователи и компьютеры Active Directory" можно использовать панель управления Skype для бизнеса Server для создания и включения новых учетных записей пользователей Skype для бизнеса Server путем добавления пользователя Active Directory в Skype для бизнеса Server.

Кроме того, можно использовать этот cmdlet, в частности [Enable-CsUser.](https://docs.microsoft.com/powershell/module/skype/enable-csuser?view=skype-ps)

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.

3. В левой панели навигации щелкните элемент **Users** (Пользователи).

4. Щелкните элемент **Enable users** (Разрешить пользователей).

5. В диалоговом окне **New Lync Server User** (Новый пользователь Lync Server) нажмите кнопку **Add** (Добавить).

6. В поле **Search users** (Поиск пользователей) полностью или частично введите полное имя, отображаемое имя, имя, фамилию, имя учетной записи диспетчера учетных записей безопасности, адрес электронной почты, имя участника-пользователя или номер телефона требуемой учетной записи пользователя Active Directory, а затем нажмите кнопку **Find** (Найти).

7. В таблице выберите учетную запись, которая будет добавлена в Skype для бизнеса Server, и нажмите кнопку **"ОК".**

8. Назначьте пользователя пулу, укажите дополнительные сведения и назначьте пользователю политики, а затем щелкните элемент **Enable** (Разрешить).

## <a name="disable-or-re-enable-a-user-account-previously-enabled-for-skype-for-business-server"></a>Отключение или повторное отключение учетной записи пользователя, ранее включенной для Skype для бизнеса Server
<a name="Disable"> </a>

Для отключения ранее включенной учетной записи пользователя в Skype для бизнеса Server можно использовать следующую процедуру без потери параметров Skype для бизнеса Server, настроенных для учетной записи пользователя. Так как параметры учетной записи пользователя Skype для бизнеса Server не теряются, вы можете снова включить ранее включенную учетную запись пользователя без необходимости перенастройки учетной записи пользователя.

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.

3. На левой панели навигации щелкните **Пользователи**.

4. В поле **Search users** введите отображаемое имя (полностью или первую его часть), имя, фамилию, имя учетной записи SAM (Security Accounts Manager — диспетчер учетных записей безопасности), SIP-адрес или линейный идентификатор URI (Uniform Resource Identifier — универсальный код ресурса) учетной записи пользователя, которую требуется отключить или активировать повторно, а затем щелкните **Найти**.

5. В таблице щелкните учетную запись пользователя, которую необходимо отключить или активировать повторно.

6. В меню **Действие** выполните одно из следующих действий:

   - Чтобы временно отключить учетную запись пользователя для Skype для бизнеса Server, щелкните **"Временно отключить" для Lync Server.**

   - Чтобы включить учетную запись пользователя для Skype для бизнеса Server, нажмите **кнопку "Повторно включить для Lync Server".**

### <a name="use-windows-powershell-to-disable-or-re-enable-user-accounts"></a>Использование Windows Powershell для отключения или повторного включаемого использования учетных записей пользователей

Учетные записи пользователей могут быть временно отключены, а затем снова включены с помощью **cmdlet Set-CsUser.** Вы можете запустить этот Windows PowerShell. Подробные сведения об использовании удаленного Windows PowerShell для подключения к Skype для бизнеса Server см. в статье [блога "Краткое руководство. Управление Microsoft Lync Server 2010 с помощью удаленной службы PowerShell".](https://go.microsoft.com/fwlink/p/?linkId=255876) В Skype для бизнеса Server этот процесс тот же.

### <a name="to-disable-a-user-account"></a>Отключение учетной записи пользователя

- Чтобы временно отключить учетную запись пользователя, установите значение свойства "Enabled" равным False ($False). Пример:

  ```PowerShell
  Set-CsUser -Identity "Ken Myer" -Enabled $False
  ```

### <a name="to-re-enable-a-user-account"></a>Повторное включить учетную запись пользователя

- Чтобы повторно включить отключенную учетную запись пользователя, установите значение свойства "Enabled" равным True ($True). Пример:

  ```PowerShell
  Set-CsUser -Identity "Ken Myer" -Enabled $True
  ```

Дополнительные сведения см. в разделе справки по [cmdlet Set-CsUser.](https://docs.microsoft.com/powershell/module/skype/set-csuser?view=skype-ps)

## <a name="disable-a-user-for-enterprise-voice"></a>Отключение пользователя для Корпоративная голосовая связь
<a name="Disable_EV"> </a>

Используйте следующую процедуру для отключения Корпоративная голосовая связь для учетной записи пользователя, включенной в Skype для бизнеса Server.

### <a name="to-disable-a-user-account-for-enterprise-voice"></a>Отключение учетной записи пользователя для Корпоративная голосовая связь

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.

3. На левой панели навигации щелкните **Пользователи**.

4. В поле **Поиск пользователей** введите отображаемое имя (полностью или первую его часть), имя, фамилию, имя учетной записи SAM (диспетчера учетных записей безопасности), SIP-адрес или линейный универсальный код ресурса (URI) учетной записи пользователя, которой требуется разрешить корпоративную голосовую связи, а затем нажмите кнопку **Найти**.

5. В таблице щелкните учетную запись пользователя, которую необходимо включить для Корпоративная голосовая связь.

6. В меню **Изменить** щелкните **Показать сведения**.

7. На странице **Изменение пользователя Lync Server** в разделе **Телефония** щелкните любой параметр, кроме **Корпоративная голосовая связь**.

    > [!NOTE]
    > Чтобы запретить пользователю делать аудио- или видеозвонки с помощью Lync, в области "Телефония" щелкните **"Звук и видео отключены".** 

8. Щелкните **Исполнить**.

Теперь пользователь не может использовать функцию Корпоративная голосовая связь. Связанные сведения: <br/>[Корпоративная голосовая связь мобильности](https://technet.microsoft.com/library/72cbe2f5-1a01-4a6f-84a5-01f3212a8992.aspx)<br/> [Enable users for Корпоративная голосовая связь in Skype for Business Server](../../deploy/deploy-enterprise-voice/enable-users-for-enterprise-voice.md)<br/> [Командная консоль Skype для бизнеса Server](../management-shell.md)
## <a name="remove-a-user-account-with-the-skype-for-business-server-management-shell"></a>Удаление учетной записи пользователя с помощью оболочки управления Skype для бизнеса Server
<a name="Remove"> </a>

Для удаления ранее добавленной учетной записи пользователя в Skype для бизнеса Server можно использовать следующую процедуру.

> [!NOTE]
> При удалении пользователя будут удалены все параметры, настроенные для этой учетной записи. If you would like to temporarily disable a user account instead, see [Disable or re-enable a user account previously enabled for Skype for Business Server.](user-accounts.md#Disable)

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.

3. На левой панели навигации щелкните **Пользователи**.

4. В поле **Поиск пользователей** введите полностью или только первую часть отображаемого имени, имени, фамилии, имени учетной записи диспетчера учетных записей безопасности (SAM), адреса SIP или строки универсального кода ресурса (URI) учетной записи пользователя, а затем щелкните **Найти**.

5. В таблице щелкните учетную запись пользователя, которую требуется удалить.

6. В меню **Макрокоманда** выберите команду **Удалить с сервера Lync Server**; откроется диалоговое окно.

7. В этом диалоговом окне нажмите кнопку **ОК** для удаления пользователя.

### <a name="remove-user-accounts-with-windows-powershell-cmdlets"></a>Удаление учетных записей пользователей с помощью windows Powershell

Учетные записи пользователей можно удалить с помощью Disable-CsUser управления. Этот cmdlet можно запустить либо из оболочки управления Skype для бизнеса Server, либо из удаленного сеанса Windows PowerShell. Подробные сведения об использовании удаленного Windows PowerShell для подключения к Skype для бизнеса Server см. в статье [блога "Краткое руководство. Управление Microsoft Lync Server 2010 с помощью удаленной службы PowerShell".](https://go.microsoft.com/fwlink/p/?linkId=255876) В Skype для бизнеса Server этот процесс тот же.

### <a name="to-remove-a-user-account"></a>Удаление учетной записи пользователя
Для удаления учетной записи пользователя используйте командлет Disable-CsUser. Пример:

  ```PowerShell
  Disable-CsUser -Identity "Ken Myer"
  ```

    After this command has run there is no way to re-enable the account and its previous settings. Instead, you will need to use the Enable-CsUser cmdlet to create a brand-new account for Ken Myer.

Дополнительные сведения см. в разделе справки по [cmdlet Disable-CsUser.](https://docs.microsoft.com/powershell/module/skype/disable-csuser?view=skype-ps)
