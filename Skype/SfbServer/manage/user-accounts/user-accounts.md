# <a name="manage-user-accounts-for-skype-for-business-server"></a>Управление учетной записью пользователей для Skype для бизнеса Server

В разделах этой статьи описывается, как включить, временно отключить или удалить пользователей Active Directory из Skype для бизнеса Server.

Сведения о том, как включить пользователя Active Directory, см. в [примере Create a New User Account.](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc732336(v=ws.11)) Сведения об удалении пользователя Active Directory см. в публикации [Delete a User Account.](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc753730(v=ws.11))

Эти процедуры должны выполняться во время окна обслуживания, когда Skype для бизнеса минимальное использование. Независимо от того, делается ли это в ежедневном или еженедельном расписаниях, определяется потребностями вашей организации.

В этой статье содержатся следующие процедуры:

- [Поиск одного или более пользователей](#search-for-one-or-more-users)

- [Добавление и добавление нового Skype для бизнеса Server пользователя](#add-and-enable-a-new-skype-for-business-server-user)

- [Отключить или повторно включить учетную запись пользователя для Skype для бизнеса Server](#disable-or-re-enable-a-user-account-for-skype-for-business-server)

- [Отключить пользователя для Корпоративная голосовая связь](#disable-a-user-for-enterprise-voice)

- [Удаление учетной записи пользователя с помощью Skype для бизнеса Server-оболочки управления](#remove-a-user-account-with-the-skype-for-business-server-management-shell)

## <a name="search-for-one-or-more-users"></a>Поиск одного или более пользователей

Результаты поиска можно использовать для настройки пользователей Active Directory для Skype для бизнеса Server. Вы можете осуществлять поиск пользователей по отображаемому имени, имени, фамилии, имени учетной записи диспетчера учетных записей безопасности, SIP-адресу или универсальному коду ресурса (URI).

Вы можете искать пользователей с помощью панели управления Skype для бизнеса Server или оснастки Active Directory Users and Computers. Следующая процедура описывает использование панели Skype для бизнеса Server для поиска пользователей.

> [!NOTE]
> В среде с топологией центрального леса результаты поиска могут быть не точными при поиске пользователя по адресу электронной почты пользователя. Вместо этого вы можете выполнить поиск пользователей, указав префикс SIP-адреса, например, sip:имя, добавить фильтр поиска и выбрать SIP-адрес с частичным адресом электронной почты, а также воспользоваться командлетом **Get-CSUser**.

### <a name="search-for-users-using-the-new-control-panel"></a>Поиск пользователей с помощью новой панели управления 

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.

3. В левой области выберите **Пользователи**.

4. На странице **Пользователи** в поле **Поиск** введите все или первую часть имени отображения, которое необходимо искать, и нажмите **кнопку Ввод**.

5. (Необязательно) Укажите дополнительные критерии поиска, чтобы сузить результаты:

    1. Нажмите кнопку фильтра рядом с **полем Поиска.**

    2. На панели **Фильтр,** которая отображается, выберите необходимое свойство пользователя, щелкнув стрелку в списке отсев.

    3. Щелкните стрелку в списке операторов, чтобы выбрать необходимого оператора.

    4. В текстовом окне введите критерии поиска, которые необходимо использовать для фильтрации результатов поиска, а затем нажмите **кнопку ОК.**

6. Результаты поиска отображаются на странице **Пользователи.** Вы можете выбрать любых или всех пользователей в списке и выполнить для них задачи настройки.

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="search-for-users-using-the-legacy-control-panel"></a>Поиск пользователей с помощью устаревшей панели управления 

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.

3. В левой области выберите **Пользователи**.

4. В поле **Search users** (Поиск пользователей) полностью или частично введите отображаемое имя, имя, фамилию, имя учетной записи диспетчера учетных записей безопасности, SIP-адрес или URI искомой учетной записи пользователя Active Directory, а затем нажмите кнопку **Find** (Найти).

5. (Необязательно) Укажите дополнительные условия поиска, чтобы уменьшить количество результатов:

    1. Нажмите кнопку со стрелкой развертывания в верхнем правом углу экрана над элементом **Search results** (Результаты поиска) и нажмите кнопку **Add Filter** (Добавить фильтр).

    2. Введите свойство пользователя, введя его или щелкнув стрелку в списке выпаданий, чтобы выбрать свойство пользователя.

    3. В **списке Equal to** list выберите **Equal to** or Not **equal to**.

    4. В текстовом поле введите требуемые условия поиска для фильтрации результатов, а затем нажмите кнопку **Find** (Найти).

6. Результаты поиска отображаются под элементом **Search Results** (Результаты поиска). Вы можете выбрать любых или всех пользователей в списке и выполнить для них задачи настройки.

## <a name="add-and-enable-a-new-skype-for-business-server-user"></a>Добавление и добавление нового Skype для бизнеса Server пользователя

Включив учетную запись пользователя в Active Directory Users and Computers, вы можете использовать панель управления Skype для бизнеса Server для создания и включения новых учетных записей Skype для бизнеса Server, добавив пользователя Active Directory в Skype для бизнеса Server.

Вы также можете использовать cmdlet, в частности [Enable-CsUser](/powershell/module/skype/enable-csuser).

### <a name="add-a-user-using-the-new-control-panel"></a>Добавление пользователя с помощью новой панели управления 

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.

3. Перейдите к **пользователям**  >  **включить пользователей** и нажмите **кнопку Добавить**.

4. В поле **Поиск** введите все или первую часть имени отображения и нажмите **кнопку Найти**.

5. (Необязательный) Чтобы указать дополнительные критерии пользователя, нажмите **кнопку + Добавить** фильтр, выберите необходимое свойство пользователя, выберите оператора и введите значение. Нажмите кнопку **Поиск**.

6. В таблице выберите учетную запись, которая необходимо добавить в Skype для бизнеса Server, а затем нажмите **кнопку ОК**.

7. Назначьте пользователя пулу, укажите дополнительные сведения и назначьте необходимые политики пользователю, а затем нажмите **кнопку Включить**.

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="add-a-user-using-the-legacy-control-panel"></a>Добавление пользователя с помощью устаревшей панели управления 

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.

3. Перейдите **к пользователям** Включить  >  **пользователей** Новый пользователь сервера  >  **Lync** и нажмите **добавить.**

6. В поле **Search users** (Поиск пользователей) полностью или частично введите полное имя, отображаемое имя, имя, фамилию, имя учетной записи диспетчера учетных записей безопасности, адрес электронной почты, имя участника-пользователя или номер телефона требуемой учетной записи пользователя Active Directory, а затем нажмите кнопку **Find** (Найти).

7. В таблице выберите учетную запись, которая необходимо добавить в Skype для бизнеса Server, а затем нажмите **кнопку ОК**.

8. Назначьте пользователя пулу, укажите дополнительные сведения и назначьте пользователю политики, а затем щелкните элемент **Enable** (Разрешить).

## <a name="disable-or-re-enable-a-user-account-for-skype-for-business-server"></a>Отключить или повторно включить учетную запись пользователя для Skype для бизнеса Server

Вы можете использовать следующую процедуру для отключения ранее включенной учетной записи пользователя в Skype для бизнеса Server без потери Skype для бизнеса Server параметров, настроенных для учетной записи пользователя. Так как параметры учетной записи Skype для бизнеса Server не теряются, можно повторно включить ранее включенную учетную запись без необходимости перенастройки учетной записи пользователя.

### <a name="disable-or-re-enable-a-user-account-using-the-new-control-panel"></a>Отключить или повторно включить учетную запись пользователя с помощью новой панели управления

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.

3. В левой области выберите **Пользователи**.

4. На странице **Пользователи** в поле **Поиск** введите все или первую часть отображаемого имени и нажмите **кнопку Ввод**.

5. В таблице дважды щелкните учетную запись пользователя, которую необходимо отключить или повторно включить.
    1. На панели, которая появится, чтобы временно отключить учетную запись пользователя для Skype для бизнеса Server, выберите **Отключение пользователя**. На панели, которая появится, нажмите кнопку **Сохранить**.
    2. Чтобы повторно включить учетную запись пользователя для Skype для бизнеса Server, в панели выберите **повторно включить пользователя**. В следующей панели, которая появится, **нажмите** сохранить .

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="disable-or-re-enable-a-user-account-using-the-legacy-control-panel"></a>Отключить или повторно включить учетную запись пользователя с помощью устаревшей панели управления

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.

3. В левой области выберите **Пользователи**.

4. В поле **Search users** введите отображаемое имя (полностью или первую его часть), имя, фамилию, имя учетной записи SAM (Security Accounts Manager — диспетчер учетных записей безопасности), SIP-адрес или линейный идентификатор URI (Uniform Resource Identifier — универсальный код ресурса) учетной записи пользователя, которую требуется отключить или активировать повторно, а затем щелкните **Найти**.

5. В таблице щелкните учетную запись пользователя, которую необходимо отключить или активировать повторно.

6. В меню **Действие** выполните одно из следующих действий:
   1. Чтобы временно отключить учетную запись пользователя для Skype для бизнеса Server, выберите временное отключение **для Lync Server.**
   2. Чтобы включить учетную запись пользователя для Skype для бизнеса Server, выберите **re-enable для Lync Server.**
  
### <a name="disable-or-re-enable-user-accounts-using-windows-powershell"></a>Отключить или повторно включить учетные записи пользователей с помощью Windows PowerShell

Учетные записи пользователей могут быть временно отключены, а затем повторно включены с помощью **комлета Set-CsUser.** Этот комлет можно выполнить либо из Skype для бизнеса Server, либо из удаленного сеанса Windows PowerShell. Сведения об использовании удаленных Windows PowerShell для подключения к Skype для бизнеса Server см. в [материале Microsoft Lync Remote PowerShell Administration](https://blog.insideo365.com/2011/08/remote-lync-powershell-administration/). Этот процесс в Skype для бизнеса Server.

### <a name="to-disable-a-user-account-using-windows-powershell"></a>Отключение учетной записи пользователя с Windows PowerShell

- Чтобы временно отключить учетную запись пользователя, установите значение включенного свойства false ($False). Пример:

  ```PowerShell
  Set-CsUser -Identity "Ken Myer" -Enabled $False
  ```

### <a name="to-re-enable-a-user-account-using-windows-powershell"></a>Повторное включить учетную запись пользователя с помощью Windows PowerShell

- Чтобы повторно включить отключенную учетную запись пользователя, установите значение включенного свойства true ($True). Например:

  ```PowerShell
  Set-CsUser -Identity "Ken Myer" -Enabled $True
  ```

Дополнительные сведения см. в разделе справка для раздела [Set-CsUser.](/powershell/module/skype/set-csuser)

## <a name="disable-a-user-for-enterprise-voice"></a>Отключить пользователя для Корпоративная голосовая связь

Используйте следующую процедуру, чтобы отключить Корпоративная голосовая связь учетную запись пользователя, включенную для Skype для бизнеса Server.

### <a name="disable-a-user-for-enterprise-voice-using-new-control-panel"></a>Отключение пользователя для Корпоративная голосовая связь с помощью новой панели управления

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.

3. В левой области нажмите кнопку **Пользователи**.

4. В поле **Поиск** введите все или первую часть имени отображения и нажмите **кнопку Найти**.

5. В таблице дважды щелкните учетную запись пользователя, которую необходимо отключить для Корпоративная голосовая связь.

6. На панели, которая появится, щелкните значок карандаша рядом с **назначенной политикой**.

7. В панели **Назначены политики,** в рамках телефонии, нажмите любой вариант, Корпоративная голосовая связь **в** списке выпаданий. 

8. Щелкните **Сохранить**.

    > [!NOTE]
    > Чтобы запретить пользователю делать звуковые или видеозвонки, **в** рамках телефонии нажмите **кнопку Отключено аудио и видео.**

Теперь пользователь не может использовать функцию Корпоративная голосовая связь. 

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="disable-a-user-account-for-enterprise-voice-using-legacy-control-panel"></a>Отключение учетной записи пользователя для Корпоративная голосовая связь с помощью устаревшей панели управления

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.

3. В левой области нажмите кнопку **Пользователи**.

4. В поле **Поиск пользователей** введите отображаемое имя (полностью или первую его часть), имя, фамилию, имя учетной записи SAM (диспетчера учетных записей безопасности), SIP-адрес или линейный универсальный код ресурса (URI) учетной записи пользователя, которой требуется разрешить корпоративную голосовую связи, а затем нажмите кнопку **Найти**.

5. В таблице щелкните учетную запись пользователя, которую необходимо включить для Корпоративная голосовая связь.

6. В меню **Изменить** щелкните **Показать сведения**.

7. На странице **Изменение пользователя Lync Server** в разделе **Телефония** щелкните любой параметр, кроме **Корпоративная голосовая связь**.

    > [!NOTE]
    > Чтобы запретить пользователю делать звуковые или видеозвонки с помощью Lync, **в** рамках телефонии нажмите **кнопку Отключено аудио/видео.**

8. Щелкните **Исполнить**.

Теперь пользователь не может использовать функцию Корпоративная голосовая связь.

## <a name="remove-a-user-account-with-the-skype-for-business-server-management-shell"></a>Удаление учетной записи пользователя с помощью Skype для бизнеса Server-оболочки управления

Вы можете использовать следующую процедуру для удаления ранее добавленной учетной записи пользователя в Skype для бизнеса Server.

> [!NOTE]
> При удалении пользователя будут удалены все параметры, настроенные для этой учетной записи. Если вы хотите временно отключить учетную запись пользователя, см. в этой записи отключение или повторное включить учетную запись пользователя, ранее включенную [для Skype для бизнеса Server.](#disable-or-re-enable-a-user-account-for-skype-for-business-server)

### <a name="remove-a-user-using-the-new-control-panel"></a>Удаление пользователя с помощью новой панели управления

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.

3. В левой области выберите **Пользователи**.

4. В поле **Поиск** введите все или первую часть имени отображения и нажмите **кнопку Найти**.

5. В таблице дважды щелкните учетную запись пользователя, которую необходимо удалить.

6. На панели, которая появится, щелкните **Удалить пользователя**. В следующей панели, которая появится, нажмите **кнопку ОК**.

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="remove-a-user-using-the-legacy-control-panel"></a>Удаление пользователя с помощью устаревшей панели управления

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.

3. В левой области выберите **Пользователи**.

4. В  поле Пользователи поиска введите все или первую часть отображаемого имени, имени, фамилии, имени учетной записи диспетчера безопасности (SAM), SIP-адреса или строки Единого идентификатора ресурса (URI) учетной записи пользователя, которую необходимо удалить, а затем нажмите кнопку **Найти**.

5. В таблице щелкните учетную запись пользователя, которую требуется удалить.

6. В меню **Макрокоманда** выберите команду **Удалить с сервера Lync Server**; откроется диалоговое окно.

7. В этом диалоговом окне нажмите кнопку **ОК** для удаления пользователя.

### <a name="remove-user-accounts-with-windows-powershell-cmdlets"></a>Удаление учетных записей пользователей с помощью Windows PowerShell-

Вы можете удалить учетные записи пользователей с помощью Disable-CsUser. Этот комлет можно запускать из Skype для бизнеса Server или удаленного сеанса Windows PowerShell. Сведения об использовании удаленных Windows PowerShell для подключения к Skype для бизнеса Server см. в [материале Microsoft Lync Remote PowerShell Administration](https://blog.insideo365.com/2011/08/remote-lync-powershell-administration/). Этот процесс в Skype для бизнеса Server.

Для удаления учетной записи пользователя используйте командлет Disable-CsUser. Пример:

  ```PowerShell
  Disable-CsUser -Identity "Ken Myer"
  ```

После завершения выполнения этой команды повторная активация учетной записи и ее предыдущих параметров невозможна. Вместо этого потребуется создать новую учетную запись для пользователя Ken Myer с помощью командлета Enable-CsUser.

Дополнительные сведения см. в разделе справка для раздела [Disable-CsUser.](/powershell/module/skype/disable-csuser)
