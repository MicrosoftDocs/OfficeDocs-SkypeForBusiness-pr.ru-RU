# <a name="customize-user-account-properties-for-skype-for-business-server"></a>Настройка свойств учетных записей пользователей для Skype для бизнеса Server
 
Процедуры в этом разделе можно использовать для изменения свойств отдельных учетных записей пользователей.
 
Существует две основные операции, которые можно сделать на уровне отдельного пользователя:
 
- [Настройка параметров телефонии для определенной учетной записи пользователя](#configure-telephony-options-for-a-specific-user-account)

- [Перемещение пользователей в другой пул](#move-users-to-another-pool)
 
## <a name="configure-telephony-options-for-a-specific-user-account"></a>Настройка параметров телефонии для определенной учетной записи пользователя

Вы можете настроить параметры телефонии для определенного пользователя (если отдельный пользователь включен для Skype для бизнеса Server и организация поддерживает телефонию).
 
Skype для бизнеса Server телефонии пользователей включают следующие параметры:
 
|Параметры телефонии  |Описание  |
|---------|---------|
|**Аудио/видео отключено**|Пользователь не может выполнять вызовы с использованием функций аудио- и видеосвязи.|
|**Только с ПК на ПК** | Пользователь может выполнять голосовые вызовы и видеовызовы только с компьютера на компьютер.|
|**Корпоративная голосовая связи** | Пользователь может использовать инфраструктуру Skype для бизнеса Server маршрутов всех входящих и исходяющих вызовов. Пользователь также может выполнять вызовы с компьютера на компьютер.|
|**Удаленное управление вызовами**   | Пользователь может использовать Skype для бизнеса Server для управления настольным телефоном, а также делать вызовы с ПК на ПК.|
 
Дополнительные сведения о настройке телефонии для организации см. в материале Включить пользователей для Корпоративная голосовая связь в [Skype для бизнеса Server](../../deploy/deploy-enterprise-voice/enable-users-for-enterprise-voice.md) и развернуть [Корпоративная голосовая связь в Skype для бизнеса Server](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) документации по развертыванию.
 
### <a name="configure-telephony-options-for-specific-users-using-new-control-panel"></a>Настройка параметров телефонии для определенных пользователей с помощью новой панели управления 
 
1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator.
 
3. В левой области выберите **Пользователи**.
 
4. В поле **Поиск** введите все или первую часть имени отображения и нажмите **кнопку Найти**.
 
5. В таблице дважды щелкните учетную запись пользователя, которую необходимо изменить.
 
6. На панели, которая появится, щелкните значок карандаша рядом с **назначенной политикой**.
 
7. В разделе **Телефония** выполните следующие действия:
 
    1. Чтобы отключить звуковые и видеозвонков для пользователя, выберите **отключенные аудио-** и видео в списке отсевов.
 
    2. Чтобы включить аудиосвязь между ПК и компьютером для пользователя, но не для удаленного управления вызовами или Корпоративная голосовая связь, выберите **pc-to-PC** только в списке отсев. Укажите значение для идентификатора **Линейный URI** для телефона, который пользователь использует для выполнения аудиовызовов с компьютера на компьютер.
 
    3. Для маршрутизаний телефонных звонков пользователя с помощью Skype для бизнеса инфраструктуры в соответствии с классом политики обслуживания,  включая аудиосвязь между ПК и компьютером, выберите Корпоративная голосовая связь в списке выпаданий. В поле **Линейный URI** укажите номер телефона для корпоративной голосовой связи. В полях **Политика абонентских групп** и **Политика голосовой связи** укажите соответствующие политики для пользователя. Чтобы указать правила нормализации для перевода номеров телефонов, набираемого пользователем в формат  E.164, выберите соответствующий профиль расположения в списке выпадения политики расположения.
 
    4. Чтобы включить пульт дистанционного управления вызовами, который позволяет пользователям управлять телефонной линией на рабочем столе с Skype для бизнеса Server  для звонков от ПК до ПК и от ПК до телефона, выберите пульт дистанционного управления вызовами в списке отсев. В поле **Линейный URI** укажите номер телефона для удаленного управления вызовами. Для маршрутизации вызовов пользователю требуется настольный телефон и подключение к системе PBX.

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

### <a name="configure-telephony-options-for-specific-users-using-legacy-control-panel"></a>Настройка параметров телефонии для определенных пользователей с помощью устаревшей панели управления
 
1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
 
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
 
3. В левой области выберите **Пользователи**.
 
4. В поле **Search users** (Поиск пользователей) полностью или частично введите полное имя, отображаемое имя, имя, фамилию, имя учетной записи диспетчера учетных записей безопасности,  SIP-адрес или универсальный код ресурса (URI) требуемой учетной записи пользователя, а затем нажмите кнопку **Find** (Найти).
 
5. В таблице выберите учетную запись пользователя, которую необходимо изменить.
 
6. В меню **Правка** щелкните команду **Изменить**.
 
7. В разделе **Телефония** выполните следующие действия:
 
    1. Чтобы отключить звуковые и видеозвонков для пользователя, выберите **отключенные аудио и видео.**
 
    2. Чтобы включить аудиосвязь между ПК и компьютером для пользователя, но не для удаленного управления вызовами или Корпоративная голосовая связь, выберите только **pc-to-PC.** Укажите значение для идентификатора **Линейный URI** для телефона, который пользователь использует для выполнения аудиовызовов с компьютера на компьютер.
 
    3. Чтобы маршрутить телефонные вызовы пользователя с помощью инфраструктуры Skype для бизнеса в соответствии с классом политики обслуживания, включая аудиосвязь между ПК и компьютером, выберите Корпоративная голосовая связь **.** В поле **Линейный URI** укажите номер телефона для корпоративной голосовой связи. В полях **Политика абонентских групп** и **Политика голосовой связи** укажите соответствующие политики для пользователя. Чтобы указать правила нормализации для преобразования номеров телефона, набираемых пользователем, в формат E.164, выберите соответствующий профиль местоположения в разделе **Политика определения местоположения**.
 
    4. Чтобы включить удаленное управление вызовами, которое позволяет пользователям управлять телефонной линией на рабочем столе из Skype для бизнеса Server для звонков от ПК до ПК и от ПК до телефона, выберите пульт дистанционного **управления вызовами.** В поле **Линейный URI** укажите номер телефона для удаленного управления вызовами. Для маршрутизации вызовов пользователю требуется настольный телефон и подключение к системе PBX.

## <a name="move-users-to-another-pool"></a>Перемещение пользователей в другой пул

Панель управления Skype для бизнеса Server для назначения пользователей определенному серверу или пулу.
 
> [!TIP]
> Перемещение всех существующих пользователей из пула исходных источников, который работает Lync Server 2010 или более ранний, в пул Skype для бизнеса Server назначения в сложной среде Active Directory может привести к замедлению репликации Active Directory. Чтобы избежать этого, можно использовать фильтры поиска для перемещения пользователей из пулов, которые работают отдельно или ранее, или использовать Skype для бизнеса Server Management Shell для перемещения пользователей с помощью cmdlets. Кроме того, функция фильтра работает с Skype для бизнеса Server пользователями. 
 
### <a name="move-selected-users-to-a-different-server-or-pool-using-new-control-panel"></a>Перемещение выбранных пользователей на другой сервер или пул с помощью новой панели управления

1. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления.
 
2. Войдите в систему с помощью учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator. 
 
3. В левой области выберите **Пользователи**.
 
4. В поле **Поиск** введите все или первую часть имени отображения и нажмите **кнопку Найти**.
 
5. В таблице дважды щелкните, чтобы выбрать конкретного пользователя или пользователей в списке. 

6. На панели, которая появится, щелкните значок карандаша рядом с **пулом регистратора**.
 
7. В **Move Users** выберите пул, в который необходимо переместить пользователей в список отсевов.
 
8. (Необязательный) Если сервер назначения или пул недоступен, выберите почтовый **ящик Force.**
 
    > [!CAUTION]
    > Если вы устанавливаете флажок **Force** (Принудительно), учетная запись пользователя перемещается, но при этом удаляются все связанные пользовательские данные (например, конференции, запланированные пользователем). Если вы не устанавливаете флажок, то перемещаются как связанные данные, так и сама учетная запись. 

> [!NOTE]
> Новая панель управления недоступна для Skype для бизнеса Server 2015 г.

5. В списке таблицы выберите одного или нескольких пользователей.

### <a name="move-users-from-one-pool-to-a-different-pool-by-using-a-filter-using-legacy-control-panel"></a>Перемещение пользователей из одного пула в другой пул с помощью фильтра с помощью устаревшей панели управления 

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
 
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. 
 
3. В левой области выберите **Пользователи**.
 
4. В **поиске пользователя** выберите **поиск** и нажмите кнопку **Добавить фильтр**.
 
5. В области условий поиска выберите **Registrar Pool** (Пул регистратора), **Equal to** (Равно), **Current Pool FQDN** (Текущее полное доменное имя пула) и затем нажмите кнопку **Find** (Найти).
 
6. В меню **Действия** выберите **Перемещение всех пользователей в пул.**
 
    > [!NOTE]
    > После применения фильтра к существующему набору пользователей параметр **Move all users to pool** (Переместить всех пользователей в пул) действует в контексте отфильтрованного подмножества пользователей, а не для **всех** возможных пользователей.
 
7. В поле **Source registrar pool** (Исходный пул регистратора) окна **Move Users** (Перемещение пользователей) выберите пул, содержащий учетные записи пользователей, которые необходимо переместить.
 
8. В поле **Destination registrar pool** (Конечный пул регистратора), выберите пул, в который необходимо переместить пользователей.
 
9. (Необязательный) Если сервер назначения или пул недоступен, выберите почтовый **ящик Force.**
 
    > [!CAUTION]
    > Если вы устанавливаете флажок **Force** (Принудительно), учетная запись пользователя перемещается, но при этом удаляются все связанные пользовательские данные (например, конференции, запланированные пользователем, и контакты). Если вы не устанавливаете флажок, то перемещаются как связанные данные, так и сама учетная запись. 
 
### <a name="move-users-from-one-pool-to-another-using-windows-powershell-cmdlets"></a>Перемещение пользователей из одного пула в другой с Windows PowerShell-кодами

1. В зависимости от Windows PowerShell команд (то есть локально или удаленно), необходимо входить в систему как член правильного Skype для бизнеса Server административных ролей следующим образом:
 
    1. Если вы работаете с командами на локальном компьютере (например, вы входите непосредственно на интерфейсный сервер): войдите на компьютер, на котором Skype для бизнеса Server Management Shell устанавливается в качестве члена группы RTCUniversalServerAdmins или с необходимыми правами пользователя, как описано в разрешениях делегирования **установки.**
 
    2. Если вы управляете командами удаленно на другом компьютере (например, вы входите на компьютер и запустите команды удаленно на переднем сервере выпуск Standard): Из учетной записи пользователя, назначенной роли CsUserAdministrator или роли CsAdministrator, войдите на любой компьютер во внутреннем развертывании.
 
2. Запустите Skype для бизнеса Server: нажмите кнопку Начните, щелкните Все **программы,** нажмите кнопку **Skype для бизнеса,** а затем нажмите кнопку **Skype для бизнеса Server.**
 
3. Для перемещения отдельных пользователей используйте командлет Move-CsUser:
 
    ```PowerShell
    Move-CsUser -Identity "Pilar Ackerman" -Target "pool01.contoso.net"
    ```

    Здесь пользователь Pilar Ackerman перемещается из назначенного в данный момент домашнего пула в пул pool01.contoso.net.
 
4. Для перемещения большого числа пользователей используйте фильтры совместно с командлетом **Get-CsUser** и передавайте полученный набор пользователей в командлет **Move-CsUser**:
 
    ```PowerShell
    Get-CsUser -Filter {RegistrarPool -eq "CurrentPoolFqdn"} | Move-CsUser -Target "TargetPoolFQDN"
    ```

    Совмещение команд **Get-CsUser** и **Move-CsUser** может дать следующий результат:
 
    ```PowerShell
    Get-CsUser -Filter {RegistrarPool -eq "pool02.contoso.net"} | Move-CsUser -Target "pool01.contoso.net"
    ```
