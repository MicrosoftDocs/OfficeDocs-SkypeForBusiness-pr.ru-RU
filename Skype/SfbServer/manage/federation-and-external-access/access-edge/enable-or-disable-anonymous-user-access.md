# <a name="enable-or-disable-anonymous-user-access-in-skype-for-business-server"></a>Включение и отключение анонимного доступа пользователей в Skype для бизнеса Server

Анонимные пользователи — это пользователи, у которых нет учетной записи пользователя в доменных службах Active Directory или в поддерживаемом Федеративной домене, но может быть приглашено удаленно участвовать в локальной конференции. Разрешая Анонимное участие в собраниях, вы включаете анонимных пользователей (то есть для пользователей, чьи удостоверения проверяются только с помощью собрания или ключа конференции) для присоединения к собраниям. Разрешение анонимного участия требует ее включения в вашу организацию.

Если позже вы захотите временно запретить доступ анонимных пользователей к Интернету, вы можете отключить ее для своей организации. Чтобы включить или отключить анонимный доступ для пользователей в Организации, выполните действия, описанные в этом разделе.

> [!NOTE]  
> Включив анонимные пользователи, вы указываете, что серверы, на которых работает служба EDGE, должны поддерживать доступ анонимных пользователей. Анонимные пользователи не могут принимать участие в собраниях в вашей организации, пока не будет настроена хотя бы одна политика Конференции и она будет применена к одному или нескольким пользователям или группам пользователей. Пользователи, которые могут пригласить анонимных пользователей на собрания, — это те, которым назначена политика конференций, настроенная для поддержки анонимных пользователей. Дополнительные сведения о настройке политик конференц-связи для анонимных пользователей можно найти в разделе [Управление политиками Конференц](../../conferencing/conferencing-policies.md)-связи.

## <a name="to-enable-or-disable-anonymous-user-access-for-your-organization"></a>Включение и отключение анонимного доступа пользователей к Организации

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  На панели навигации слева выберите элемент **внешний доступ к пользователю**, а затем — **Конфигурация Edge Access**.

4.  На странице **конфигурации Edge Access** щелкните **Глобальная**, выберите пункт **изменить**, а затем — **Показать подробности**.

5.  В окне **изменение конфигурации Edge Access**выполните одно из указанных ниже действий.
    
      - Чтобы включить анонимный доступ для пользователей в Организации, установите флажок **включить связь с анонимными пользователями** .
    
      - Чтобы запретить анонимный доступ к Организации, снимите флажок **включить связь с анонимными пользователями** .

6.  Нажмите **Исполнить**.


## <a name="enabling-or-disabling-anonymous-user-access-by-using-windows-powershell-cmdlets"></a>Включение и отключение анонимного доступа пользователей с помощью командлетов Windows PowerShell

Управлять доступом анонимных пользователей можно с помощью Windows PowerShell и командлета **Set-ксакцесседжеконфигуратион** . Этот командлет можно выполнить либо из командной консоли Skype для бизнеса Server, либо из удаленного сеанса Windows PowerShell. 

## <a name="to-enable-anonymous-user-access"></a>Разрешение анонимного доступа пользователей

  - Чтобы включить анонимный доступ для пользователей, присвойте свойству **аллованонимаусусерс** значение True ($true):
    
        Set-CsAccessEdgeConfiguration -AllowAnonymousUsers $True

## <a name="to-disable-anonymous-user-access"></a>Отключение анонимного доступа пользователей

  - Чтобы запретить анонимный доступ пользователей, установите для свойства **аллованонимаусусерс** значение False ($false):
    
        Set-CsAccessEdgeConfiguration -AllowAnonymousUsers $False
