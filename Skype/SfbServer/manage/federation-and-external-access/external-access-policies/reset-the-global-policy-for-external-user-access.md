# <a name="reset-the-global-policy-for-external-user-access-in-skype-for-business-server"></a>Сброс глобальной политики для доступа внешних пользователей в Скайп для Business Server 

Если были созданы или настроены политики доступа внешних пользователей, которые больше не хотите использовать, можно сделать следующее:

  - Удалите политику узла или пользователя, который вы создали.

  - Сброс глобальной политики по умолчанию. Параметры глобальной политики по умолчанию запретить доступ внешних пользователей. Не удается удалить глобальную политику.

Не удается полностью удалить глобальную политику. Только с помощью параметра **Удалить** на глобальную политику сбрасывает глобальную политику параметры по умолчанию, которые не поддерживают функцию для параметров доступа внешних пользователей.

## <a name="to-reset-the-global-policy-to-the-default-settings"></a>Сброс глобальной политики значения по умолчанию

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server.

3.  В левой панели навигации щелкните **Доступ для внешних пользователей**, щелкните **Политика внешнего доступа**.

4.  На вкладке **Политика внешнего доступа** щелкните глобальную политику, нажмите кнопку **Изменить**и нажмите кнопку **Удалить**.

5.  При появлении запроса на подтверждение нажмите **кнопку ОК**. В верхней части страницы, информирующее о том, что выполнен сброс глобальной политики отображается сообщение.


## <a name="resetting-the-global-external-access-policy-by-using-windows-powershell-cmdlets"></a>Сброс глобальной политики внешнего доступа с помощью командлетов Windows PowerShell

С помощью Windows PowerShell и командлет Remove-CsExternalAccessPolicy можно Сброс глобальной политики внешнего доступа. Этот командлет можно выполнять с Скайп для консоли Business Server или из удаленного сеанса Windows PowerShell. 

## <a name="to-reset-the-global-external-access-policy"></a>Сброс глобальной политики внешнего доступа

  - Эта команда сбрасывает глобальную политику внешнего доступа:
    
        Remove-CsExternalAccessPolicy -Identity "global"

Для получения дополнительных сведений см раздел справки для командлета [Remove-CsExternalAccessPolicy](https://docs.microsoft.com/en-us/powershell/module/skype/Remove-CsExternalAccessPolicy) .
