# <a name="manage-services-in-skype-for-business-server"></a>Управление службами в Skype для бизнеса Server

Панель управления Skype для бизнеса Server можно использовать для просмотра списка всех компьютеров, работающих Skype для бизнеса Server топологии, просмотра состояния служб, запуска или остановки служб и предотвращения сеансов для служб.

- [Просмотр списка компьютеров с Skype для бизнеса Server](#view-a-list-of-computers-running-skype-for-business-server)
- [Просмотр состояния служб, работающих на компьютере в Skype для бизнеса](#view-the-status-of-services-running-on-a-computer-in-skype-for-business)
- [Запуск или остановка Skype для бизнеса служб](#start-or-stop-skype-for-business-services)
- [Запрет сеансов для служб](#prevent-sessions-for-services)

## <a name="view-a-list-of-computers-running-skype-for-business-server"></a>Просмотр списка компьютеров с Skype для бизнеса Server

Панель управления Skype для бизнеса Server для просмотра списка всех компьютеров, работающих Skype для бизнеса топологии, и просмотра состояния службы каждого из них. Этот список можно упорядочивать по компьютерам, пулам или сайтам. 

1. Из учетной записи пользователя, назначенной любой из предопределяемых административных ролей для Skype для бизнеса Server, войдите на любой компьютер во внутреннем развертывании. Дополнительные сведения см. в рублях управления доступом на основе ролей [(RBAC) для Skype для бизнеса Server.](../../plan-your-deployment/security/role-based-access-control-rbac.md)
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
4. На странице Состояние при необходимости сделайте следующее:
    - Упорядочите список, щелкнув заголовок столбца **Компьютер**, **Пул** или **Сайт**, а затем щелкнув стрелку вверх или вниз.
    - Нажмите кнопку **Обновить**, чтобы просмотреть наиболее актуальный список.
    - Найдите конкретный компьютер, введя его имя в поле поиска.
   
## <a name="view-the-status-of-services-running-on-a-computer-in-skype-for-business"></a>Просмотр состояния служб, работающих на компьютере в Skype для бизнеса

Используйте панель управления Skype для бизнеса Server, чтобы просмотреть все службы, которые работают на определенном компьютере в топологии Skype для бизнеса Server и просмотреть состояние каждой службы.

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. In the left navigation bar, click **Topology**.
4. На странице Состояние отсортируйте список или выполните по нему поиск, чтобы найти интересующий вас компьютер, а затем щелкните имя этого компьютера.
5. Сделайте один из следующих:
    - To see the latest status of services running on the computer, click **Get service status**.
    - Чтобы просмотреть список конкретных служб, запущенных на компьютере, и состояние каждой из них, щелкните **Свойства**, а затем нажмите кнопку **Закрыть** для возврата в список.

### <a name="viewing-service-status-by-using-windows-powershell-cmdlets"></a>Просмотр состояния службы с Windows PowerShell cmdlets

Вы также можете просматривать состояние службы с помощью Windows PowerShell и Get-CsWindowsService. Вы можете запустить этот командлет из командной консоли Skype для бизнеса Server или из удаленного сеанса Windows PowerShell. Дополнительные сведения см. [в Skype для бизнеса Server Management Shell.](../management-shell.md)

**Просмотр состояния службы**

Чтобы просмотреть состояние службы на компьютере, введите команду, аналогичную следующей в командной Skype для бизнеса Server, а затем нажмите кнопку Ввод:

```powershell
Get-CsWindowsService -ComputerName atl-cs-001.litwareinc.com | Select-Object RoleName, Status
```

Эта команда возвращает следующую информацию:

```console
RoleName                                  Status
--------                                  ------
{W3SVC}                                   Running
{CentralManagement}                       Running
{ClsAgent}                                Running
{Registrar, UserServer, EdgeServer}       Running
{ApplicationServer}                       Running
{ConferencingServer}                      Running
{MediationServer}                         Running
```

Подробные сведения [см. в материале Get-CsWindowsService.](/powershell/module/skype/Get-CsWindowsService)

## <a name="start-or-stop-skype-for-business-services"></a>Запуск или остановка Skype для бизнеса служб

Используйте панель Skype для бизнеса Server для запуска или остановки всех Skype для бизнеса Server, работающих на определенном компьютере, или для запуска или остановки определенной службы.

### <a name="start-or-stop-all-skype-for-business-server-services-on-a-computer"></a>Запуск или остановка всех Skype для бизнеса Server на компьютере

1. С учетной записи пользователя, которая входит в группу RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или назначена роли CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в которой вы развернули Skype для бизнеса Server. Вы можете определить, назначена ли вам роль CsServerAdministrator или роль CsAdministrator RBAC, с помощью команды, аналогичной следующей:

    ```powershell
    Get-CsAdminRoleAssignment -Identity "kenmyer"`
    ```

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель Skype для бизнеса Server управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
4. На странице Status (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его.
5. Щелкните элемент **Action** (Действие).
6. Щелкните **Start All services** (Запустить все службы) или **Stop All services** (Остановить все службы).

### <a name="start-or-stop-a-specific-service"></a>Запуск или остановка определенной службы

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
4. На странице Status (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его.
5. Щелкните элемент **Properties** (Свойства).
6. Отсортируйте список служб, если это необходимо, и щелкните службу, которую хотите запустить или остановить.
7. Щелкните элемент **Action** (Действие).
8. Щелкните **Start service** (Запустить службу) или **Stop service** (Остановить службу).
9. Нажмите кнопку **Закрыть**.


## <a name="prevent-sessions-for-services"></a>Запрет сеансов для служб

Используйте панель Skype для бизнеса для предотвращения новых сеансов для всех служб Skype для бизнеса Server, работающих на определенном компьютере, или для предотвращения новых сеансов для определенной службы.

### <a name="prevent-new-sessions-for-all--skype-for-business-server-services-on-a-computer"></a>Предотвращение новых сеансов для всех Skype для бизнеса Server служб на компьютере

1. С учетной записи пользователя, которая входит в группу RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или назначена роли CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в которой вы развернули Skype для бизнеса Server.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. В левой панели навигации щелкните элемент **Topology** (Топология), а затем **Status** (Состояние).
4. На странице Состояние выполните сортировку или поиск по списку, чтобы найти компьютер, на котором выполняются службы, для которых нужно запретить новые сеансы, а затем щелкните ее.
5. Щелкните **Действие**.
6. Щелкните **Запретить новые сеансы для всех служб**.

### <a name="prevent-new-sessions-for-a-specific-service"></a>Предотвращение новых сеансов для определенной службы

1. С учетной записи пользователя, которая входит в группу RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или назначена роли CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в которой вы развернули Skype для бизнеса Server.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления. Дополнительные сведения о различных методах, которые можно использовать для запуска панели управления Skype для бизнеса Server, см. в материале [Install и open administrative tools.](../../management-tools/install-and-open-administrative-tools.md)
3. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
4. Нажмите кнопку **Свойства**.
5. Отсортируйте список служб, если это необходимо, и щелкните службу, для которой вы хотите запретить новые сеансы.
6. Щелкните **Действие**.
7. Щелкните **Запретить новые сеансы для службы**.
8. Нажмите кнопку **Закрыть**.
