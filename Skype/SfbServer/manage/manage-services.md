# <a name="manage-services-for-skype-for-business-server"></a>Управление службами для Skype для бизнеса Server

В этой статье описывается управление службами, работающими в топологии Skype для бизнеса Server.
  
## <a name="view-a-list-of-computers-running-skype-for-business-server"></a>Просмотр списка компьютеров со Skype для бизнеса Server
<a name="view_list"> </a>

Панель управления Skype для бизнеса Server можно использовать для просмотра списка всех компьютеров со Skype для бизнеса Server в топологии и просмотра состояния службы каждого из них. Этот список можно упорядочивать по компьютерам, пулам или сайтам. 
  
### <a name="to-view-a-list-of-computers-running-skype-for-business-server"></a>Просмотр списка компьютеров со Skype для бизнеса Server

1. Войдите на любой компьютер во внутреннем развертывании из учетной записи пользователя, назначенной любой из предопределеных административных ролей для Skype для бизнеса Server. Подробные сведения о предопределеных административных ролях, доступных в Skype для бизнеса Server, см. в Role-Based **управления доступом.**   
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server.   
3. В левой панели навигации щелкните элемент **Топология**, а затем элемент **Состояние**.   
4. На странице **Состояние** выполните все необходимые действия из приведенных ниже.
   - Упорядочите список, щелкнув заголовок столбца **Компьютер**, **Пул** или **Сайт**, а затем щелкнув стрелку вверх или вниз. 
   - Нажмите кнопку **Обновить**, чтобы просмотреть наиболее актуальный список.  
   - Найдите конкретный компьютер, введя его имя в поле поиска.
    
## <a name="view-the-status-of-services-running-on-a-skype-for-business-server"></a>Просмотр состояния служб, запущенных на сервере Skype для бизнеса
<a name="view-status"> </a>

Панель управления Skype для бизнеса Server можно использовать для просмотра всех служб, запущенных на определенном компьютере в топологии Skype для бизнеса Server, и просмотра состояния каждой службы.
  
### <a name="to-view-the-status-of-services-running-on-a-computer"></a>To view the status of services running on a computer

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. In the left navigation bar, click **Topology**. 
4. На странице **"Состояние"** отсортировать или найти нужный компьютер в списке, а затем щелкните имя компьютера.
5. Do any of the following:
   - To see the latest status of services running on the computer, click **Get service status**.
   - Чтобы просмотреть список конкретных служб, запущенных на компьютере, и состояние каждой из них, щелкните **Свойства**, а затем нажмите кнопку **Закрыть** для возврата в список.
    
### <a name="viewing-service-status-with-windows-powershell-cmdlets"></a>Просмотр состояния службы с помощью cmdlets Windows Powershell

Вы также можете просмотреть состояние службы с Windows PowerShell и с помощью Windows PowerShell **Get-CsWindowsService.** Вы можете запустить этот командлет из командной консоли Skype для бизнеса Server или из удаленного сеанса Windows PowerShell. Подробные сведения об использовании удаленного Windows PowerShell для подключения к Skype для бизнеса Server см. в статье [блога "Краткое руководство. Управление Microsoft Lync Server 2010 с](https://go.microsoft.com/fwlink/p/?linkId=255876)помощью удаленной powerShell". В Skype для бизнеса Server этот процесс тот же.
  
### <a name="to-view-service-status"></a>Просмотр состояния службы

Чтобы просмотреть состояние службы на компьютере, введите команду, аналогичную следующей, в командной оболочке Skype для бизнеса Server и нажмите ввод:
  
```PowerShell
Get-CsWindowsService -ComputerName atl-cs-001.litwareinc.com | Select-Object RoleName, Status
```

Эта команда возвращает следующую информацию:
  
|**RoleName**|**Состояние**|
|:-----|:-----|
|{W3SVC}  <br/> |Выполняется  <br/> |
|{CentralManagement}  <br/> | Выполняется <br/> |
|{ClsAgent}  <br/> |Выполняется  <br/> |
|{Registrar, UserServer, EdgeServer}  <br/> |Выполняется  <br/> |
|{ApplicationServer}  <br/> |Выполняется  <br/> |
|{ConferencingServer}  <br/> |Выполняется  <br/> |
|{MediationServer}  <br/> |Выполняется  <br/> |
   
Подробные сведения см. в [сведениях о get-CsWindowsService.](https://docs.microsoft.com/powershell/module/skype/get-cswindowsservice.md?view=skype-ps)
  
## <a name="view-details-about-a-service"></a>Просмотр сведений о службе
<a name="view_details"> </a>

Панель управления Skype для бизнеса Server можно использовать для просмотра сведений о каждой службе, запущенной на определенном компьютере в топологии. Панель управления предоставляет сведения о состоянии каждой службы, а также дополнительные данные о службе, например сведения о подключенных базах данных, портах и зависимых службах.
  
### <a name="to-view-details-for-a-service"></a>Просмотр сведений о службе

1. Войдите на любой компьютер во внутреннем развертывании из учетной записи пользователя, назначенной любой из предопределеных административных ролей для Skype для бизнеса Server. Подробные сведения о предопределеных административных ролях, доступных в Skype для бизнеса Server, см. в Role-Based **управления доступом.**
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. В левой панели навигации щелкните **Топология** и затем щелкните **Состояние**.
4. На странице **Состояние** выполните сортировку списка или найдите требуемый компьютер.
5. Щелкните **Свойства**.
6. В окне **Просмотр сведений о компьютере** отсортируйте список служб при необходимости и выберите службу, которую необходимо просмотреть.
7. Выполните следующие действия:
   - Чтобы просмотреть последнее состояние определенной службы, щелкните **Получить сведения о состоянии службы**.
   - Чтобы просмотреть сведения об определенной службе, щелкните **Свойства** и затем нажмите кнопку **Закрыть**.
   - Чтобы получить список компьютеров, используемых в топологии, нажмите кнопку **Закрыть**.
    
## <a name="start-or-stop-skype-for-business-server-services"></a>Запуск или остановка служб Skype для бизнеса Server
<a name="StartStop"> </a>

Панель управления Skype для бизнеса Server можно использовать для запуска или остановки всех служб Skype для бизнеса Server, работающих на определенном компьютере, а также для запуска или остановки определенной службы.
  
### <a name="to-start-or-stop-all-skype-for-business-services-on-a-computer"></a>Запуск или остановка всех служб Skype для бизнеса на компьютере

1. Из учетной записи пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или которой назначена роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер в сети, в которой развернут Skype для бизнеса Server. Чтобы определить, назначена ли вам роль CsServerAdministrator или CsAdministrator RBAC, можно использовать следующую команду:
    
   ```PowerShell
   Get-CsAdminRoleAssignment -Identity "kenmyer"
   ```

2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. В левой панели навигации щелкните элемент **Topology** (Топология), а затем **Status** (Состояние).
4. На странице **Status** (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его.
5. Щелкните элемент **Action** (Действие).
6. Щелкните **Start All services** (Запустить все службы) или **Stop All services** (Остановить все службы).
    
### <a name="to-start-or-stop-a-specific-service"></a>Запуск или остановка конкретной службы

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. В левой панели навигации щелкните элемент **Topology** (Топология), а затем **Status** (Состояние).
4. На странице **Status** (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его.
5. Щелкните элемент **Properties** (Свойства).
6. Отсортируйте список служб, если это необходимо, и щелкните службу, которую хотите запустить или остановить.
7. Щелкните элемент **Action** (Действие).
8. Щелкните **Start service** (Запустить службу) или **Stop service** (Остановить службу).
9. Нажмите кнопку **Закрыть**.
    
## <a name="prevent-sessions-for-services"></a>Запрет сеансов для служб
<a name="prevent_session"> </a>

Панель управления Skype для бизнеса Server позволяет запретить новые сеансы для всех служб Skype для бизнеса Server, работающих на определенном компьютере, или запретить новые сеансы для определенной службы Skype для бизнеса Server.
  
### <a name="to-prevent-new-sessions-for-all-skype-for-business-services-on-a-computer"></a>Предотвращение новых сеансов для всех служб Skype для бизнеса на компьютере

1. Из учетной записи пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или которой назначена роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер в сети, в которой развернут Skype для бизнеса Server.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. В левой панели навигации щелкните **Топология**, а затем щелкните **Состояние**.
4. На странице **Состояние** выполните сортировку или поиск по списку, чтобы найти компьютер, на котором выполняются службы, для которых нужно запретить новые сеансы, а затем щелкните ее.
5. Щелкните **Действие**.
6. Щелкните **Запретить новые сеансы для всех служб**.
    
### <a name="to-prevent-new-sessions-for-a-specific-service"></a>Запрет новых сеансов для определенной службы

1. Из учетной записи пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или которой назначена роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер в сети, в которой развернут Skype для бизнеса Server.
2. Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 
3. В левой панели навигации щелкните элемент **Topology** (Топология), а затем **Status** (Состояние).
4. На странице **Status** (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его. 
5. Нажмите кнопку **Свойства**.
6. Отсортируйте список служб, если это необходимо, и щелкните службу, для которой вы хотите запретить новые сеансы.
7. Щелкните **Действие**.
8. Щелкните **Запретить новые сеансы для службы**.
9. Нажмите кнопку **Закрыть**.
