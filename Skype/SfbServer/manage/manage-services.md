# <a name="manage-services-for-skype-for-business-server-2015"></a>Управление службами для Скайп для Business Server 2015

В этой статье описывается, как управлять служб, работающих в Скайп для топологии Business Server 2015.
  
## <a name="view-a-list-of-computers-running-skype-for-business-server-2015"></a>Просмотр списка компьютеров под управлением Скайп Business Server 2015
<a name="view_list"> </a>

Чтобы просмотреть список всех компьютеров, которые работают под управлением Скайп для 2015 Business Server в топологии и просмотреть состояние службы каждого из них можно использовать Скайп для панели управления Business Server. Сортировать по компьютер, пул или сайт. 
  
### <a name="to-view-a-list-of-computers-running-skype-for-business-server"></a>Чтобы просмотреть список компьютеров под управлением Скайп Business Server

1. Из учетной записи пользователя, который назначен для любой из предварительно заданных административных ролей для Скайп для Business Server 2015 Войдите на любой компьютер во внутреннем развертывании. Для получения дополнительных сведений о предопределенные административные роли, доступные в Скайп для Business Server 2015 см **управления доступом на основе ролей**.   
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server.   
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.   
4. На странице " **состояние** " выполните одно из приведенных ниже действий.
   - Отсортируйте список, щелкнув на **компьютер**, **пул**или заголовок столбца **сайта** , а затем щелкнув стрелку вверх или стрелку вниз. 
   - Нажмите кнопку **Обновить** , чтобы просмотреть наиболее актуальный список.  
   - Поиск определенного компьютера, введя его имя в поле поиска.
    
## <a name="view-the-status-of-services-running-on-a-skype-for-business-2015-server"></a>Просмотр состояния служб, работающих на Скайп для Business 2015 server
<a name="view-status"> </a>

Скайп для панели управления Business Server можно использовать для просмотра всех служб, которые выполняются на определенном компьютере в вашей Скайп для топологии Business Server и просмотреть состояние каждой из них.
  
### <a name="to-view-the-status-of-services-running-on-a-computer"></a>Просмотр состояния служб, работающих на компьютере

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология**. 
4. На странице " **состояние** " Сортировка или поиска в списке, необходимо, чтобы найти компьютер, к которому вы хотите и нажмите кнопку имя компьютера.
5. Выполните любое из следующих действий.
   - Чтобы просмотреть последнее состояние служб, работающих на компьютере, нажмите кнопку **получить сведения о состоянии службы**.
   - Чтобы просмотреть список конкретных служб, работающих на компьютере, а также состояние каждой из них, щелкните **Свойства**и нажмите кнопку **Закрыть** , чтобы вернуться к списку.
    
### <a name="viewing-service-status-with-windows-powershell-cmdlets"></a>Просмотр состояния службы с помощью командлетов Windows Powershell

Можно также просмотреть состояние службы с помощью командлета **Get-CsWindowsService** и Windows PowerShell. Этот командлет можно запустить из Скайп для консоли Business Server или из удаленного сеанса Windows PowerShell. Для получения дополнительных сведений об использовании удаленной оболочки Windows PowerShell для подключения к Скайп для Business Server в статье блога [«Быстрый запуск: управление Microsoft Lync Server 2010 с помощью удаленной оболочки PowerShell»](https://go.microsoft.com/fwlink/p/?linkId=255876). Процесс одинаков в Скайп для Business Server.
  
### <a name="to-view-service-status"></a>Просмотр состояния службы

Чтобы просмотреть состояние службы на компьютере, введите следующую команду в Скайп для консоли Business Server и нажмите клавишу ВВОД:
  
```
Get-CsWindowsService -ComputerName atl-cs-001.litwareinc.com | Select-Object RoleName, Status
```

Этой командой возвращается информация, аналогичная следующим сведениям:
  
|**Имя роли**|**Состояние**|
|:-----|:-----|
|{W3SVC}  <br/> |Под управлением  <br/> |
|{CentralManagement}  <br/> | Под управлением <br/> |
|{ClsAgent}  <br/> |Под управлением  <br/> |
|{Регистраторов UserServer, EdgeServer}  <br/> |Под управлением  <br/> |
|{ApplicationServer}  <br/> |Под управлением  <br/> |
|{ConferencingServer}  <br/> |Под управлением  <br/> |
|{MediationServer}  <br/> |Под управлением  <br/> |
   
Дополнительные сведения см [Командлет Get-CsWindowsService](https://docs.microsoft.com/powershell/module/skype/get-cswindowsservice.md?view=skype-ps).
  
## <a name="view-details-about-a-service"></a>Просмотр сведений о службе
<a name="view_details"> </a>

Скайп для панели управления Business Server можно использовать для просмотра сведений о каждом службу, запущенную на определенном компьютере в топологии. Можно просмотреть состояние каждой службы и сведения, например связанных баз данных, порты и зависимые службы.
  
### <a name="to-view-details-for-a-service"></a>Для просмотра сведений о службе

1. Из учетной записи пользователя, который назначен для любой из предварительно заданных административных ролей для Скайп для Business Server 2015 Войдите на любой компьютер во внутреннем развертывании. Для получения дополнительных сведений о предопределенные административные роли, доступные в Скайп для Business Server 2015 см **управления доступом на основе ролей**.
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.
4. На странице " **состояние** " сортировку или найти в списке и нажмите кнопку компьютере, на котором нужно просмотреть.
5. Нажмите кнопку **Свойства**.
6. В окне **Просмотр сведений о компьютере** отсортируйте список служб, если это необходимо и выберите службу, которую необходимо просмотреть.
7. Выполните одно из приведенных ниже действий.
   - Чтобы просмотреть последнее состояние определенной службы, нажмите кнопку **получить сведения о состоянии службы**.
   - Чтобы просмотреть сведения об определенной службе, щелкните **Свойства** и нажмите кнопку **Закрыть**.
   - Чтобы получить список всех компьютеров в топологии, нажмите кнопку **Закрыть**.
    
## <a name="start-or-stop-skype-for-business-server-2015-services"></a>Запуск или остановка служб Business Server 2015 Скайп
<a name="StartStop"> </a>

Запуск или остановка всех Скайп для служб Business Server 2015 на определенном компьютере или на запуск или остановка конкретной службы можно использовать Скайп для панели управления Business Server.
  
### <a name="to-start-or-stop-all-skype-for-business-services-on-a-computer"></a>Чтобы запустить или остановить все Скайп для бизнес-служб на компьютере

1. Используя учетную запись пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в котором вы развернули Скайп для Business Server 2015. Можно определить, является ли вам назначена роль CsAdministrator RBAC или CsServerAdministrator, выполнив команду, аналогичную следующей:
    
  ```
  Get-CsAdminRoleAssignment -Identity "kenmyer"

  ```

2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.
4. На странице **состояния** , сортировки или поиска по списку, чтобы найти компьютере, на котором запущены службы необходимо запустить или остановить и нажмите кнопку.
5. Щелкните **Действие**.
6. Нажмите кнопку **запустить все службы** или **Остановить все службы**.
    
### <a name="to-start-or-stop-a-specific-service"></a>Запуск или остановка конкретной службы

1. Войдите на любой компьютер во внутреннем развертывании с использованием учетной записи пользователя, назначенной роли CsUserAdministrator или CsAdministrator.
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.
4. На странице **состояния** , сортировки или поиска по списку, чтобы найти компьютер, на котором работает служба необходимо запустить или остановить и нажмите кнопку.
5. Нажмите кнопку **Свойства**.
6. Отсортируйте список служб, если это необходимо и выберите службу, которую требуется запустить или остановить.
7. Щелкните **Действие**.
8. Нажмите кнопку **Запуск службы** или **остановки службы**.
9. Нажмите кнопку **Закрыть**.
    
## <a name="prevent-sessions-for-services"></a>Запрет сеансов для служб
<a name="prevent_session"> </a>

Запретить установку новых сеансов для всех Скайп для служб Business Server 2015 на определенном компьютере или запретить установку новых сеансов для определенных Скайп для службы Business Server 2015, можно использовать Скайп для панели управления Business Server.
  
### <a name="to-prevent-new-sessions-for-all-skype-for-business-services-on-a-computer"></a>Чтобы запретить установку новых сеансов для всех Скайп для бизнес-служб на компьютере

1. Используя учетную запись пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в котором вы развернули Скайп для Business Server 2015.
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.
4. На странице " **состояние** " Сортировка или поиска по списку как требуется на компьютере, на котором запущены службы, для которых вы хотите запретить новые сеансы, а затем нажмите кнопку Найти.
5. Щелкните **Действие**.
6. Щелкните **Запретить установку новых сеансов для всех служб**.
    
### <a name="to-prevent-new-sessions-for-a-specific-service"></a>Чтобы запретить установку новых сеансов для определенной службы

1. Используя учетную запись пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или роль CsServerAdministrator или CsAdministrator, войдите на любой компьютер, который находится в сети, в котором вы развернули Скайп для Business Server 2015.
2. Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 
3. В левой панели навигации щелкните **Топология** , а затем щелкните **состояние**.
4. На странице **состояния** , сортировки или поиска по списку, чтобы найти компьютер, на котором работает служба необходимо запустить или остановить и нажмите кнопку. 
5. Нажмите кнопку **Свойства**.
6. Отсортируйте список служб, если это необходимо и выберите службу, для которого вы хотите запретить новые сеансы.
7. Щелкните **Действие**.
8. Щелкните **запретить новые сеансы для службы**.
9. Нажмите кнопку **Закрыть**.
