# <a name="configure-the-meeting-join-page-in-skype-for-business-server-2015"></a>Настройка собрания страницы присоединения к в Скайп для Business Server 2015
 
**Сводка:** Сведения о настройке собрания страницы присоединения к в Скайп для Business Server 2015.
  
Когда пользователь щелкает ссылку собрания в приглашении на собрание, собрание страницы присоединения к определяет, будет ли Скайп для клиента Business уже установлен на компьютере пользователя. Если клиент уже установлен, клиент открывает и присоединился к собранию. Если клиент не установлен, по умолчанию откроется 2015 версии Скайп для клиента Business. 
  
## <a name="configure-the-meeting-join-page"></a>Настройка собрания страницы присоединения к

Можно изменить поведение присоединения к собранию страницы, чтобы разрешить пользователям присоединяться к собраниям с других версий клиента. Эти параметры конфигурации были удалены из Скайп для панели управления Business Server, но их настройка с помощью командлета Set-CsWebServiceConfiguration.
  
**Присоединение к Set-CsWebServiceConfiguration для страницы параметры собрания**

|**Параметр SET-CsWebServiceConfiguration**|**Описание**|
|:-----|:-----|
|ShowJoinUsingLegacyClientLink  <br/> |Этот параметр устарел для использования с локальной версией Скайп Business Server 2015.  <br/> Если установлено значение True, присоединение к собранию с помощью клиентского приложения, отличный от Скайп для бизнеса будет предлагаться возможность присоединения к собранию с помощью их текущей клиентского приложения. По умолчанию используется значение False.  <br/> |
|ShowAlternateJoinOptionsExpanded  <br/> |Этот параметр устарел для использования с локальной версией Скайп Business Server 2015.  <br/>  Если задано значение True, альтернативные параметры для присоединения к конференции в режиме online автоматически расширяется и отображается для пользователей. Если задано значение False (значение по умолчанию), эти параметры будут доступны, но пользователь будет иметь для отображения списка параметров для себя.  <br/> |
