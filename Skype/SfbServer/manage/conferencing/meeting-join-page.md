# <a name="configure-the-meeting-join-page-in-skype-for-business-server"></a>Настройка страницы присоединения к собранию в Skype для бизнеса Server
 
**Сводка:** Научитесь настраивать страницу присоединения к собраниям в Skype для бизнеса Server.
  
Когда пользователь щелкает ссылку на собрание в приглашении на собрание, страница присоединения к собранию определяет, установлен ли клиент Skype для бизнеса на компьютере пользователя. Если клиент уже установлен, клиент открывает и присоединяется к собранию. Если клиент не установлен, по умолчанию откроется клиент Skype для бизнеса. 
  
## <a name="configure-the-meeting-join-page"></a>Конфигурация страницы присоединения к собранию

Если вы хотите разрешить пользователям присоединяться к собраниям с другими версиями клиента, вы можете изменить поведение страницы присоединения к собранию. Эти параметры конфигурации были удалены с панели управления Skype для бизнеса Server, но их можно настроить с помощью командлета Set-Ксвебсервицеконфигуратион.
  
**Наборы страниц для присоединения к собранию — параметры Ксвебсервицеконфигуратион**

|**Параметр Set-Ксвебсервицеконфигуратион**|**Описание**|
|:-----|:-----|
|Шовжоинусинглегациклиентлинк  <br/> |Этот параметр устарел для использования в локальной версии Skype для бизнеса Server.  <br/> Если установлено значение true, пользователи, присоединяющиеся к собранию с помощью клиентского приложения, отличного от Skype для бизнеса, получат возможность присоединиться к собранию с помощью текущего клиентского приложения. Значение по умолчанию — False.  <br/> |
|Шовалтернатежоиноптионсекспандед  <br/> |Этот параметр устарел для использования в локальной версии Skype для бизнеса Server.  <br/>  Если установлено значение true, дополнительные параметры для присоединения к онлайн-конференции автоматически развертываются и отображаются для пользователей. Если для этого параметра задано значение false (по умолчанию), эти параметры будут доступны, но пользователю потребуется отобразить список параметров.  <br/> |
