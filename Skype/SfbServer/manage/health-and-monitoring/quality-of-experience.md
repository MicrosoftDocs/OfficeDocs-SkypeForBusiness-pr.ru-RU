# <a name="quality-of-experience-qoe-in-skype-for-business-server"></a>Качества взаимодействия (QoE) в Скайп для Business Server
 
**Сводка:** Управление качества взаимодействия (QoE) в Скайп Business Server.
  
Служба отслеживания качества взаимодействия (QoE) записывает числовые данные, которые показывают качество мультимедиа-данных, и сведения об участниках, именах устройств, драйверах, IP-адресах и типах конечных точек, использованных во время звонка или сеанса. При установке Скайп для Business Server можно установить набор предварительно заданных глобальные параметры конфигурации качества взаимодействия. Используйте разделы этой главы для настройки значений QoE.
