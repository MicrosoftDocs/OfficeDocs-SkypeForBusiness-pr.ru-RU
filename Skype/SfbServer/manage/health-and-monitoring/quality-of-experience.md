# <a name="quality-of-experience-qoe-in-skype-for-business-server-2015"></a>Качество взаимодействия (QoE) в Skype для бизнеса Server 2015
 
**Сводка:** Управление качества взаимодействия (QoE) в Скайп Business Server 2015.
  
Служба отслеживания качества взаимодействия (QoE) записывает числовые данные, которые показывают качество мультимедиа-данных, и сведения об участниках, именах устройств, драйверах, IP-адресах и типах конечных точек, использованных во время звонка или сеанса. Когда вы устанавливаете Скайп для Business Server 2015, можно установить набор предварительно заданных глобальные параметры конфигурации качества взаимодействия. Используйте разделы этой главы для настройки значений QoE.
