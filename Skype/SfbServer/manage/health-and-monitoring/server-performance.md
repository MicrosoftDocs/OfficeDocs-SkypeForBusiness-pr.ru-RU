## <a name="making-the-best-use-of-the-server-performance-report"></a><span data-ttu-id="25c8c-125">Использование отчета о производительности сервера</span><span class="sxs-lookup"><span data-stu-id="25c8c-125">Making the best use of the Server Performance Report</span></span>

<span data-ttu-id="25c8c-p104">В отчете есть несколько фильтров данных.
