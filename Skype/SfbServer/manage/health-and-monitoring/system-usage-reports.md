# <a name="system-usage-reports-in-skype-for-business-server"></a>Отчеты об использовании системы в Skype для бизнеса Server
 
**Сводка:** Узнайте о отчетах об использовании системы в Skype для бизнеса Server.
  
Отчеты об использовании системы предоставляют сведения об использовании системы на основе данных записи сведений о звонках (CDR), собранных Skype для бизнеса Server.
  
## <a name="in-this-section"></a>В этом разделе

- [Отчет о регистрации пользователей в Skype для бизнеса Server](user-registration-report.md)
    
    Предоставляет сводку подключения пользователей к развертыванию Skype для бизнеса Server на основе событий регистрации, таких как логоны пользователей. В отчете предоставляется возможность просмотра внутренних и внешних логонов, а также сравнения количества пользователей, во время входа в систему Skype для бизнеса Server с числом пользователей, которые на самом деле использовали службу во время входа в систему.
    
- [Сводный отчет о одноранговых действиях в Skype для бизнеса Server](peer-to-peer-activity-summary-report.md)
    
    Содержит сводку по одноранговым мгновенным сообщениям, аудио- и видеовызовам, передаче файлов и сеансам совместного использования приложений. Одноранговые сеансы — это сеансы с участием всего двух пользователей.
    
- [Сводный отчет конференции в Skype для бизнеса Server](conference-summary-report.md)
    
    Сводка по всем действиям, связанным с конференциями. Конференции — это сеансы с участием трех или более пользователей.
    
- [Сводный отчет конференции PSTN в Skype для бизнеса Server](pstn-conference-summary-report.md)
    
    Сводка по всем конференциям ТСОП. Это конференции, к которым хотя бы один пользователь подключается по телефонной сети общего пользования (ТСОП) (т.н. конференц-связь с телефонным подключением.
    
- [Отчет об использовании группы ответов в Skype для бизнеса Server](response-group-usage-report.md)
    
    Содержит сводку об использовании группы ответов. Приложение Группы реагирования позволяет автоматически маршрутить телефонные вызовы в такие объекты, как служба поддержки или линия поддержки клиентов.
    
- [Отчет Телефон инвентаризации в Skype для бизнеса Server](ip-phone-inventory-report.md)
    
    Предоставляет сведения об IP-телефонах, которые в настоящее время используются в организации. Отчет основан на регистрации телефонов и логотипах. Это не следует считать полным инвентарем. Например, вы могли удалить телефоны, которые по-прежнему указаны в отчете, так как они вошли в систему по крайней мере один раз. Кроме того, вы также можете иметь новые телефоны, которые не показываются в отчете просто потому, что пользователи еще не вошли Skype для бизнеса Server с их новыми телефонами.
    
- [Call Admission Control Report in Skype для бизнеса Server](call-admission-control-report.md)
    
    Предоставляет список действий с одноранговыми сеансами и конференциями, которые используют контроль допуска звонков. Контроль допуска звонков (CAC) позволяет определить, следует ли разрешить сеансы общения в реальном времени, например голосовые или видеовызовы, на основе ограничения пропускной способности.
