# <a name="ucwa-events-in-skype-for-business-server"></a>События UCWA в Skype для бизнеса Server
 
**Сводка:** Узнайте о веб-API объединенных коммуникаций (UCWA) в Skype для бизнеса Server.
  
Skype для бизнеса Server использует веб-API объединенных коммуникаций (UCWA) для ряда целей: от доступа к Microsoft Exchange для поиска контактов до обновления присутствия для мобильных клиентов.
  
UCWA будет записывать записи о рабочем поведении в качестве типов событий "Информационный", "Предупреждение" и "Ошибка". В следующей таблице описываются события, которые могут быть записаны компонентами UCWA.

|20001  <br/> |Информационный  <br/> |Инициализированная UCWA  <br/> |Н/Д  <br/> Н/Д  <br/> |
|20002  <br/> |Ошибка  <br/> |Во время инициализации UCWA столкнулось с непредвиденным исключением  <br/> |Непредвиденное сообщение об ошибке во время инициализации  <br/> Проверьте сведения об исключении в связанной записи журнала событий, чтобы определить возможную причину  <br/> |
