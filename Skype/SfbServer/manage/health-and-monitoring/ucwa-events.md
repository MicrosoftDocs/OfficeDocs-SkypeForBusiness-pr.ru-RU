# <a name="ucwa-events-in-skype-for-business-server"></a>События UCWA в Skype для бизнеса Server
 
**Сводка:** Узнайте о веб-API единой связи (UCWA) в Skype для бизнеса Server.
  
Skype для бизнеса Server использует веб-API единой связи (UCWA) для нескольких целей: от доступа к Microsoft Exchange поиска контактов до обновления присутствия для мобильных клиентов.
  
UCWA будет записывать записи операционного поведения в качестве типов событий Informational, Warning и Error. В следующей таблице описываются события, которые могут быть написаны компонентами UCWA.

|20002  <br/> |Error  <br/> |UCWA столкнулась с неожиданным исключением во время инициализации  <br/> |Во время инициализации произошла неожиданная ошибка  <br/> Изучите сведения об исключениях в связанной записи журнала событий, чтобы определить возможную причину  <br/> |
|20003  <br/> |Error  <br/> |UCWA столкнулась с необналиченным исключением  <br/> |Произошло ненаблюдаемо исключение  <br/> Перезапустите сервер. Если проблема сохраняется, обратитесь в службу поддержки продуктов  <br/> |
