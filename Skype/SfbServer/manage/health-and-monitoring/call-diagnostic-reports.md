# <a name="call-diagnostic-reports-in-skype-for-business-server-2015"></a>Отчеты по диагностике звонков в Skype для бизнеса Server 2015
 
**Сводка:** Сведения о многопользовательской диагностические отчеты по вызовам используется в Скайп для Business Server 2015.
  
Диагностические отчеты по вызовам предоставляют сводные сведения и диагностические данные для неудачных одноранговых сеансов связи и сеансов конференц-связи.
  
## <a name="in-this-section"></a>Содержание

- [Вызов сводного диагностического отчета в Скайп для Business Server 2015](summary-report.md) Содержит общий обзор неудачных сеансов peer-to-peer и конференциям. Одноранговые сеансы – это сеансы, в которых только два участника. Сеансы конференц-связи включают не менее трех участников.
    
- [Peer-to-Peer Activity диагностику отчет в Скайп для Business Server 2015](peer-to-peer-activity-diagnostic-report.md) Содержит общий обзор тенденций неудачных сеансов peer-to-peer. В одноранговых сеансах имеется только два участника.
    
- [Сообщить о конференции диагностики в Скайп для Business Server 2015](conference-diagnostic-report.md) Предоставляет общий обзор тенденций сеансов неудачных конференц-связи и тенденций представления для каждой модальности конференции. Сеансы конференц-связи включают не менее трех участников.
    
- [Основным сбоям отчет в Скайп для Business Server 2015](top-failures-report.md) Список наиболее частых сбоев и характер их изменения по времени.
    
- [В Скайп для Business Server 2015 отчета распределения сбоев](failure-distribution-report.md) Предоставляет анализ неудачных сеансов.
    
- [Сбой список отчет в Скайп для Business Server 2015](failure-list-report.md) Предоставляет подробные сведения об отдельных участниках неудачной конференции.
    
- [Отчет диагностики в Скайп для Business Server 2015](diagnostic-report.md) Предоставляет диагностические сведения (включая коды ответа SIP, диагностические заголовки и идентификаторы) по неудачным сеансам.
