# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Вызов диагностических отчетов в Skype для бизнеса Server
 
**Сводка:** Узнайте о много пользовательских диагностических отчетах вызовов, используемых в Skype для бизнеса Server.
  
Диагностические отчеты по вызовам предоставляют сводные сведения и диагностические данные для неудачных одноранговых сеансов связи и сеансов конференц-связи.
  
## <a name="in-this-section"></a>В этой статье

- [Сводный отчет по диагностике вызовов в Skype для бизнеса Server](summary-report.md) Предоставляет общую сводку неудачных одноранговых сеансов и сеансов конференций. Одноранговые сеансы — это сеансы, в которых только два участника. Сеансы конференц-связи включают не менее трех участников.
    
- [Диагностический отчет о одноранговой](peer-to-peer-activity-diagnostic-report.md) активности в Skype для бизнеса Server Обеспечивает общее представление тренда неудачных одноранговых сеансов. В одноранговых сеансах имеется только два участника.
    
- [Диагностический отчет конференции в Skype для бизнеса Server](conference-diagnostic-report.md) Предоставляет общее представление тенденций неудачных сеансов конференц-связи и представления тенденций для каждого модальности конференции. Сеансы конференц-связи включают не менее трех участников.
    
- [Отчет о главных сбоях в Skype для бизнеса Server](top-failures-report.md) Предоставляет список наиболее частых сбоев и их тенденций со временем.
    
- [Отчет о рассылке сбоев в Skype для бизнеса Server](failure-distribution-report.md) Предоставляет анализ сбойных сеансов.
    
- [Отчет о списке отказов в Skype для бизнеса Server](failure-list-report.md) Предоставляет подробные сведения об отдельных участниках, участвующих в неудачной конференции.
    
- [Диагностический отчет в Skype для бизнеса Server](diagnostic-report.md) Предоставляет сведения о диагностике и устранении неполадок (включая коды ответов SIP и диагностические заготки и ID) для неудающихся сеансов.
