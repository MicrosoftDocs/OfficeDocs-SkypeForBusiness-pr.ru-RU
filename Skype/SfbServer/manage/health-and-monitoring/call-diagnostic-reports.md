# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Вызов диагностических отчетов в Skype для бизнеса Server
 
**Сводка:** Узнайте о много пользовательских диагностических отчетах вызовов, используемых в Skype для бизнеса Server.
  
Диагностические отчеты по вызовам предоставляют сводные сведения и диагностические данные для неудачных одноранговых сеансов связи и сеансов конференц-связи.
  
## <a name="in-this-section"></a>В этой статье

- [Сводный отчет по диагностике вызовов в Skype для бизнеса Server](summary-report.md) содержит общую сводку неудачных одноранговых сеансов и сеансов конференций. Одноранговые сеансы — это сеансы, в которых только два участника. Сеансы конференц-связи включают не менее трех участников.
    
- [Диагностический отчет](peer-to-peer-activity-diagnostic-report.md) одноранговых действий в Skype для бизнеса Server предоставляет общее представление тенденции неудачных одноранговых сеансов. В одноранговых сеансах имеется только два участника.
    
- [Отчет о диагностике конференций в Skype для бизнеса Server](conference-diagnostic-report.md) предоставляет общее представление тенденций неудающихся сеансов конференц-связи и представлений тенденций для каждого модальности конференции. Сеансы конференц-связи включают не менее трех участников.
    
- [Top Failures Report in Skype для бизнеса Server](top-failures-report.md) provides a list of the most frequent failures and their trends over time.
    
- [Отчет о рассылке сбоев Skype для бизнеса Server](failure-distribution-report.md) предоставляет анализ сбоя сеансов.
    
- [Отчет о списке неудач в Skype для бизнеса Server](failure-list-report.md) содержит подробные сведения о отдельных участниках, участвующих в неудачной конференции.
    
- [Диагностический отчет в Skype для бизнеса Server](diagnostic-report.md) предоставляет сведения о диагностике и устранении неполадок (включая коды ответов SIP и диагностические заготки и ID) для неудающихся сеансов.
