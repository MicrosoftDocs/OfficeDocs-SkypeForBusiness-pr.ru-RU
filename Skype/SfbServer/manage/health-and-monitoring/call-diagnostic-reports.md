# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Отчеты по диагностике звонков в Skype для бизнеса Server
 
**Сводка:** Узнайте о диагностических отчетах по многоязыковой диагностике звонков, используемых в Skype для бизнеса Server.
  
Диагностические отчеты по вызовам предоставляют сводные сведения и диагностические данные для неудачных одноранговых сеансов связи и сеансов конференц-связи.
  
## <a name="in-this-section"></a>В этом разделе:

- [Сводный диагностический отчет по вызовам в Skype для бизнеса Server](summary-report.md) Предоставляет общую сводку по неудачным одноранговых сеансам и сеансам конференций. Одноранговые сеансы — это сеансы, в которых только два участника. Сеансы конференц-связи включают не менее трех участников.
    
- [Диагностический отчет по](peer-to-peer-activity-diagnostic-report.md) одноранговой активности в Skype для бизнеса Server Предоставляет общее представление тенденций неудачных одноранговых сеансов. В одноранговых сеансах имеется только два участника.
    
- [Диагностический отчет по конференциям в Skype для бизнеса Server](conference-diagnostic-report.md) Предоставляет общее представление тенденций неудачных сеансов конференц-связи и представления тенденций для каждой модальности конференции. Сеансы конференц-связи включают не менее трех участников.
    
- [Отчет по главным сбоям в Skype для бизнеса Server](top-failures-report.md) Предоставляет список наиболее частых сбоев и их тенденций с течением времени.
    
- [Отчет по распределению сбоев в Skype для бизнеса Server](failure-distribution-report.md) Предоставляет анализ неудачных сеансов.
    
- [Failure List Report in Skype for Business Server](failure-list-report.md) Предоставляет подробные сведения об отдельных участниках неудачной конференции.
    
- [Диагностический отчет в Skype для бизнеса Server](diagnostic-report.md) Предоставляет диагностические сведения и сведения об устранении неполадок (включая коды ответов SIP, диагностические заглавные и коды) для неудачных сеансов.
