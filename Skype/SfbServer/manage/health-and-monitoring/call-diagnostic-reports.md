# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Отчеты по диагностике звонков в Скайп для Business Server
 
**Сводка:** Сведения о многопользовательской диагностические отчеты по вызовам используется в Скайп для Business Server.
  
Диагностические отчеты по вызовам предоставляют сводные сведения и диагностические данные для неудачных одноранговых сеансов связи и сеансов конференц-связи.
  
## <a name="in-this-section"></a>Содержание

- [Вызов сводного диагностического отчета в Скайп для Business Server](summary-report.md) Содержит общий обзор неудачных сеансов peer-to-peer и конференциям. Одноранговые сеансы – это сеансы, в которых только два участника. Сеансы конференц-связи включают не менее трех участников.
    
- [Peer-to-Peer Activity Diagnostic Report в Скайп для Business Server](peer-to-peer-activity-diagnostic-report.md) Содержит общий обзор тенденций неудачных сеансов peer-to-peer. В одноранговых сеансах имеется только два участника.
    
- [Диагностический отчет по конференции в Скайп для Business Server](conference-diagnostic-report.md) Предоставляет общий обзор тенденций сеансов неудачных конференц-связи и тенденций представления для каждой модальности конференции. Сеансы конференц-связи включают не менее трех участников.
    
- [Top Failures Report в Скайп для Business Server](top-failures-report.md) Список наиболее частых сбоев и характер их изменения по времени.
    
- [Отчет по распределению сбоев в Скайп для Business Server](failure-distribution-report.md) Предоставляет анализ неудачных сеансов.
    
- [Отчет по списку сбоев в Скайп для Business Server](failure-list-report.md) Предоставляет подробные сведения об отдельных участниках неудачной конференции.
    
- [Диагностический отчет в Скайп для Business Server](diagnostic-report.md) Предоставляет диагностические сведения (включая коды ответа SIP, диагностические заголовки и идентификаторы) по неудачным сеансам.
