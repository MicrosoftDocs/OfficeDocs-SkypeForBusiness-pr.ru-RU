# <a name="conference-summary-report-in-skype-for-business-server"></a>Сводный отчет конференции в Skype для бизнеса Server
 
**Сводка:** Сведения о сводке конференций в Skype для бизнеса Server.
  
Сводный отчет о конференциях предоставляет обзорные сведения о сеансах веб-конференций. Конференция обычно включает более 2 пользователей и требует использования служб конференц-конференции. Для сравнения, одноранговая сессия обычно включает всего 2 пользователя и не требует использования Skype для бизнеса Server службы. Одноранговая деятельность сообщается в сводной сводке одноранговых действий [в Skype для бизнеса Server](peer-to-peer-activity-summary-report.md).
  
Сводный отчет конференции не только указывает количество конференций, которые проводились в течение определенного периода времени (ежечасно, ежедневно, еженедельно, ежемесячно), но также указывает общее число участников этих конференций и общее число уникальных организаторов конференций.
  
"Уникальным" организатором является любой, кто запланировать хотя бы одну конференцию. Например, если пользователь Алексей Иванов запланировал одну конференцию, он считается одним уникальным организатором. Если пользователь Виталий Максимов запланировал 148 конференций, он также считается одним уникальным организатором. Например, в следующей таблице показаны 8 запланированных конференций, но только три уникальных организатора (Ken Myer, Pilar Ackerman и David Ahs).
  
|**Организатор конференции**|**Дата конференции**|
|:-----|:-----|
|Кен Майер  <br/> |7.07.2015 10:00  <br/> |
|Анна Васильева  <br/> |7.07.2015 10:00  <br/> |
|Кен Майер  <br/> |7.07.2015 11:00  <br/> |
|Пилар Аккерман  <br/> |7.07.2015 11:00  <br/> |
|Кен Майер  <br/> |7/7/2015 13:00  <br/> |
|Пилар Аккерман  <br/> |7/7/2015 14:00  <br/> |
|Кен Майер  <br/> |07.02.2015 10:00  <br/> |
|Пилар Аккерман  <br/> |07.02.2015 10:00  <br/> |
   
В сводном отчете о конференциях также указывается количество конференций, в которых использовалась голосовая и видеосвязь.
  
## <a name="accessing-the-conference-summary-report"></a>Доступ к сводному отчету о конференциях

Доступ к сводному отчету о конференциях можно получить с домашней страницы "Отчеты наблюдения". Вы можете просмотреть подробные сведения в отчете о действиях конференций, щелкнув один из следующих разделов:
  
- Общее количество конференций
    
- Общее количество участников
    
## <a name="making-the-best-use-of-the-conference-summary-report"></a>Рекомендации по использованию сводного отчета о конференциях

Суммарные значения для большинства показателей, используемых в сводном отчете о конференциях, приведены в нижней части отчета. Прокрутите отчет вниз, чтобы просмотреть общее число конференций, проведенных в течение определенного периода, и общее число их участников. Однако среди суммарных значений нет общего числа уникальных организаторов. Вот одна причина. Предположим, что вы просматриваете данные за месяц. В первый день было 34 уникальных организатора конференций, а во второй день — 27 организаторов. Означает ли это, что общее число уникальных организаторов в течение этих двух дней равно 61? Не обязательно. В действительности все 27 пользователей, организовавших конференции во второй день, могут входить в число 34 пользователей, организовавших конференции в первый день. Например, в этом простом отчете обратите внимание, что Конференции Кена Myer и Pilar Ackerman запланированы как на 7/7/2015, так и на 7/2/2/2015:
  
|**Организатор конференции**|**Дата конференции**|
|:-----|:-----|
|Кен Майер  <br/> |7.07.2015 10:00  <br/> |
|Анна Васильева  <br/> |7.07.2015 10:00  <br/> |
|Кен Майер  <br/> |7.07.2015 11:00  <br/> |
|Пилар Аккерман  <br/> |7.07.2015 11:00  <br/> |
|Кен Майер  <br/> |7/7/2015 13:00  <br/> |
|Пилар Аккерман  <br/> |7/7/2015 14:00  <br/> |
|Кен Майер  <br/> |07.02.2015 10:00  <br/> |
|Пилар Аккерман  <br/> |07.02.2015 10:00  <br/> |
   
Чтобы получить более точное представление об общем количестве уникальных организаторов конференций, измените временной интервал. Например, просмотрите данные по месяцам, а не по дням.
  
## <a name="filters"></a>Фильтры

Фильтры позволяют получить более конкретные наборы возвращаемых данных. Например, сводный отчет о конференциях позволяет выбрать способ группировки данных. Конференции можно группировать по часу, дню, неделе или месяцу.
  
В следующей таблице перечислены фильтры, которые вы можете использовать со сводным отчетом о конференциях.
  
**Фильтры сводного отчета о конференциях**

|**Название**|**Описание**|
|:-----|:-----|
|**From** <br/> |Дата и время начала диапазона. Чтобы просмотреть данные по часам, введите дату и время начала в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время начала, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**To** <br/> |Дата и время окончания диапазона. Чтобы просмотреть данные по часам, введите дату и время окончания в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время окончания, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**Interval** <br/> | Временной интервал. Выберите один из следующих вариантов: <br/>  Ежечасно (может отображаться до 25 часов) <br/>  Ежедневно (может отображаться до 31 дня) <br/>  Еженедельно (может отображаться до 12 недель) <br/>  Ежемесячно (может отображаться до 12 месяцев) <br/>  Если даты начала и окончания превышают максимальное допустимые значения для выбранного интервала, отображается только максимальное число значений (от даты начала). Например, если вы выбираете дневной интервал с датой начала 7/7/2015 и датой окончания 2/28/2015, данные отображаются за 8/7/7/2015 с 12:00 до 09:00 12:00 (то есть в общей сложности 31 день). <br/> |
