# <a name="media-quality-diagnostic-reports-in-skype-for-business-server"></a>Отчеты диагностики качества мультимедиа в Skype для бизнеса Server
 
**Сводка:** Узнайте об отчетах диагностики качества мультимедиа в Skype для бизнеса Server.
  
Диагностический отчет по качеству мультимедиа предоставляет сведения о качестве вызовов, а также диагностические сведения для неудачных вызовов.
  
## <a name="in-this-section"></a>Содержание

- [Сводный отчет о качестве качества мультимедиа в Skype для бизнеса Server](summary.md) Обеспечивает общие данные по качеству для разных типов конечных точек, в том числе одноранговых звонков для корпоративной голосовой связи, голосовых звонков и звонков, которые зависят от общедоступной коммутируемой телефонной сети (PSTN).
    
- [Отчет о сравнении качества мультимедиа в Skype для бизнеса Server](comparison.md) Сравнение значений качества звонков для разных типов голосовых вызовов (например, звонки, совершенные по беспроводной сети, и звонки, совершенные через проводное соединение).
    
- [Отчет о производительности сервера в Skype для бизнеса Server](server-performance.md) В этой статье перечислены серверы, на которых возникли проблемы в большинстве случаев, в зависимости от измерения таких показателей качества, как снижение производительности, потерь пакетов и колебаний.
    
- [Отчет о местоположении в Skype для бизнеса Server](location-report.md) Список сетевых расположений и краткий обзор качества звонков, происходящих в каждом месте. В данном отчете расположения основываются на IP-адресах подсетей.
    
- [Отчет об устройстве в Skype для бизнеса Server](device-report.md) Общие сведения об устройствах, используемых для корпоративных голосовых звонков, а также среднее качество звонков на устройстве.
    
- [Отчет о списке обзвона в Skype для бизнеса Server](call-list-report-0.md) Подробная информация о телефонных звонках, совершенных и полученных в Организации.
    
- [Отчет о звонках в Skype для бизнеса Server](call-detail-report.md) Подробная информация о телефонных звонках, полученных от Организации.
    
- [Отчет о тенденциях качества сервера мультимедиа в Skype для бизнеса Server](server-media-quality-trend-report.md) Предоставляет способ для графического сравнения до пяти серверов в метриках качества взаимодействия, таких как громкость звонка, некачественный процент звонков, потеря пакетов и нарушение колебаний.
    
- [Отчет о распределении показателей качества мультимедиа в Skype для бизнеса Server](media-quality-metrics-distribution-report.md) Диаграмма, на которой показаны значения распространения для метрики качества взаимодействия, такие как нарушение или потеря пакетов.
    
- [Отчет о тенденциях расположения в Skype для бизнеса Server](location-trend-report.md) Предоставляет сведения о тенденциях качества связи для сетевых местоположений.
