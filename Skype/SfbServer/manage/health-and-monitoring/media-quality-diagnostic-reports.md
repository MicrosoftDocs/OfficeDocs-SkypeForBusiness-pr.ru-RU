# <a name="media-quality-diagnostic-reports-in-skype-for-business-server-2015"></a>Диагностические отчеты по качеству среды в Skype для бизнеса Server 2015
 
**Сводка:** Сведения о мультимедиа диагностические отчеты по качеству в Скайп для Business Server 2015.
  
Диагностический отчет по качеству мультимедиа предоставляет сведения о качестве вызовов, а также диагностические сведения для неудачных вызовов.
  
## <a name="in-this-section"></a>Содержание

- [Сводка по качеству мультимедиа отчет в Скайп для Business Server 2015](summary.md) Предоставляет данные общее качество для типов другой конечной точки, включая peer-to-peer вызовов корпоративной голосовой связи, конференции корпоративной голосовой связи и вызовов, которые используют, по крайней мере частично на public переключения телефонной сети (общего пользования PSTN).
    
- [Сравнение качества мультимедиа отчет в Скайп для Business Server 2015](comparison.md) Сравнение вызова значений качества для разных типов голосовые вызовы (например, вызовы через беспроводную сеть и вызовов, сделанных на проводное подключение).
    
- [Производительность сервера отчетов в Скайп для Business Server 2015](server-performance.md) Список серверов, испытывавших наибольшие проблемы, на основе измерения таких ключевых метрик качества как снижение производительности, потеря пакетов и дрожание.
    
- [Расположение отчета в Скайп для Business Server 2015](location-report.md) Предоставляет список расположения в сети и сводку о качестве мультимедиа в вызовах, которые происходят в каждом расположении. В данном отчете расположения основываются на IP-адресах подсетей.
    
- [Сообщить о устройства в Скайп для Business Server 2015](device-report.md) Предоставляет сводку по устройствам, использовавшимся для вызовов корпоративной голосовой связи и включает сведения о среднем качестве мультимедиа в вызовах по устройствам.
    
- [В Скайп для Business Server 2015 отчет по списку обзвона](call-list-report-0.md) Предоставляет подробные сведения о телефонных вызовах, выполненных или полученных в организации.
    
- [Отчет о в Скайп для Business Server 2015 звонке](call-detail-report.md) Предоставляет подробные сведения о телефонных вызовах, выполненных или полученных в организации.
    
- [Сообщить о тенденциях качества мультимедиа сервера в Скайп для Business Server 2015](server-media-quality-trend-report.md) Предоставляет способ графическое сравнение до пяти серверов на показатели качества, такие как громкости вызова, процент вызовов плохого, потеря пакетов и дрожание.
    
- [Распределение показатели качества мультимедиа отчет в Скайп для Business Server 2015](media-quality-metrics-distribution-report.md) Предоставляет график распределения значений для метрики качества, такие как дрожания или потери пакетов.
    
- [Сообщить о тенденциях расположения в Скайп для Business Server 2015](location-trend-report.md) Предоставляет сведения о тенденциях качества звонков для сетевых расположений.
