# <a name="conference-summary-subreport-in-skype-for-business-server"></a>Subreport сводки конференции в Skype для бизнеса Server
 
**Сводка:** Сведения о подконференции конференции в Skype для бизнеса Server.
  
Вложенный сводный отчет о конференции содержит обзор неудачных сеансов конференц-связи. Такие неудачные сеансы дополнительно делятся по типу — сеансы Focus и сеансы MCU.
  
## <a name="filters"></a>Фильтры

Фильтры позволяют вам возвратить уточненный набор данных или просматривать возвращенные данные различными способами. В следующей таблице перечислены фильтры, которые вы можете использовать во вложенном сводном отчете о конференции.
  
**Фильтры вложенного сводного отчета о конференции**

|**Имя**|**Описание**|
|:-----|:-----|
|**From** <br/> |Дата и время начала диапазона. Чтобы просмотреть данные по часам, введите дату и время начала в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время начала, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**To** <br/> |Дата и время окончания диапазона. Чтобы просмотреть данные по часам, введите дату и время окончания в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время окончания, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**Pool** <br/> |Полное доменное имя пула Регистраторов или пограничного сервера. Вы можете выбрать отдельный пул или щелкнуть элемент **[All]** (Все) для просмотра данных из всех пулов. Этот раскрывающийся список заполняется автоматически на основании записей в базе данных.<br/> |
   
## <a name="metrics"></a>Метрики

В следующей таблице перечислены сведения, представленные во вложенном сводном отчете о конференции.
  
**Метрики вложенного сводного отчета о конференции**

|**Имя**|**Возможность сортировки по этому показателю**|**Описание**|
|:-----|:-----|:-----|
|**Total conferences** (Всего конференций) <br/> |Нет  <br/> |Общее количество проведенных конференций.  <br/> |
|**Total conference sessions** (Всего сеансов конференций) <br/> |Нет  <br/> |Общее количество сеансов конференций. Отдельная конференция может иметь несколько сеансов; например, конференция может включать в себя как сеанс Focus, так и сеанс MCU.  <br/> |
|**Overall session failure rate** (Общий уровень сбоев сеансов) <br/> |Нет  <br/> |Процентное отношение всех конференций, завершившихся со сбоем.  <br/> |
|**Focus sessions** (Сеансы Focus) <br/> |Нет  <br/> |Общее количество сеансов Focus.  <br/> |
|**Focus failure rate** (Уровень сбоев Focus) <br/> |Нет  <br/> |Процентное отношение сеансов Focus, завершившихся со сбоем.  <br/> |
|MCU sessions (Сеансы с MCU)  <br/> |Нет  <br/> |Общее количество сеансов MCU.  <br/> |
|**MCU failure rate** (Уровень сбоев MCU) <br/> |Нет  <br/> |Процентное отношение сеансов MCU, завершившихся со сбоем.  <br/> |
|**MCU sessions by modality** (Сеансы MCU по модальности) <br/> |Нет  <br/> |Общее количество сеансов MCU, сгруппированных по модальности (например, конференции с обменом мгновенными сообщениями).  <br/> |
|**Failure rate by modality** (Уровень сбоев по модальности) <br/> |Нет  <br/> |Процентное отношение неудачных сеансов MCU, сгруппированных по модальности (например, конференции с обменом мгновенными сообщениями).  <br/> |
