# <a name="call-detail-recording-cdr-in-skype-for-business-server"></a>Регистрация вызовов (CDR) в Skype для бизнеса Server
 
**Сводка:** Узнайте о записях регистрации вызовов (CDR), используемых в Skype для бизнеса Server.
  
Записи регистрации вызовов (CDR) и диагностическая информация об одноранговой активности, включая мгновенные сообщения, VoIP-вызовы, общий доступ к приложениям, передачу файлов и собрания. Данные об использовании могут применяться для расчета окупаемости инвестиций, а данные диагностики — для устранения неполадок, связанных с одноранговой активностью и собраниями. При установке Skype для бизнеса Server также будет устанавливаться предопределена коллекция глобальных параметров конфигурации для CDR. Используйте темы этого раздела для настройки регистрации вызовов.
