# <a name="call-detail-recording-cdr-in-skype-for-business-server-2015"></a>Регистрация вызовов (CDR) в Skype для бизнеса Server 2015
 
**Сводка:** Сведения о регистрации вызовов (CDR) записей, используемых в Скайп для Business Server 2015.
  
Записи регистрации вызовов (CDR) и диагностическая информация об одноранговой активности, включая мгновенные сообщения, VoIP-вызовы, общий доступ к приложениям, передачу файлов и собрания. Данные об использовании могут применяться для расчета окупаемости инвестиций, а данные диагностики — для устранения неполадок, связанных с одноранговой активностью и собраниями. Когда вы устанавливаете Скайп для Business Server 2015, можно установить предварительно заданных коллекцию глобальных параметров конфигурации для регистрации Вызовов. Используйте темы этого раздела для настройки регистрации вызовов.
