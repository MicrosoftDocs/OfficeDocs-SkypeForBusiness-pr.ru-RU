# <a name="pstn-conference-summary-report-in-skype-for-business-server"></a>Сводный отчет конференции PSTN в Skype для бизнеса Server
 
**Сводка:** Сведения о сводном отчете о конференции PSTN в Skype для бизнеса Server.
  
В Skype для бизнеса Server конференция PSTN — это любая конференция, на которой по крайней мере один участник набирает аудиозапись с помощью телефона PSTN (телефонная сеть с общедоступными переключениями). (Телефон PSTN — это "наземный телефон", мобильный телефон или любой другой телефон, который не использует голосовой голос через IP.) Хотя эти конференции называются конференциями PSTN в отчетах мониторинга, эти конференции, возможно, более широко известны как конференции с диалогом.
  
Сводный отчет по конференциям ТСОП (PSTN Conference Summary Report) предоставляет сведения о всех конференциях ТСОП, происходивших в вашей организации (т.е. о всех конференциях, хотя бы один участник которых использовал телефонное подключение). В этом отчете содержатся сведения об общем количестве конференций ТСОП, общем количестве участников этих конференций и, что возможно самое важное, об общем количестве пользователей с телефонным подключением (метрика "Total PSTN participants" ("Общее количество участников ТСОП")).
  
## <a name="accessing-the-pstn-conference-summary-report"></a>Доступ к сводному отчету по конференциям ТСОП

Сводный отчет по конференциям ТСОП можно вызвать только с домашней страницы отчетов мониторинга. Этот отчет не привязан ни к каким другим отчетам. Следует отметить, что невозможно получить подробные сведения о вызове для конференции ТСОП, в частности потому, что за предоставление этих сведений отвечают конечные точки, а телефоны ТСОП не имеют возможности отслеживать и отправлять подробные сведения о вызове.
  
## <a name="making-the-best-use-of-the-pstn-conference-summary-report"></a>Оптимальное использование сводного отчета по конференциям ТСОП

Чтобы определить процент всех конференций, включив пользователей с диалогом, сравните значение метрики total PSTN-конференций с показателем Total conferences, найденным в сводном отчете конференции в [Skype для бизнеса Server.](conference-summary-report.md)
  
Если вы не видите столько конференций ТСОП, сколько предполагали увидеть, имейте в виду, что возможность организации конференции, разрешающей телефонное подключение пользователей, зависит от политики конференц-связи, которая назначена пользователю: если организация конференций ТСОП разрешена очень малому количеству пользователей, то, конечно, вы будете видеть очень мало конференций ТСОП. Вы можете быстро проверить, какие из политик конференц-конференций (если таковые есть) позволяют пользователям планировать конференции PSTN, запуская следующую команду из Skype для бизнеса Server командной оболочки:
  
```PowerShell
Get-CsConferencingPolicy | Select-Object Identity, EnableDialInConferencing
```

Вы получите результат, похожий на приведенный ниже.
  
<pre>
Identity                                EnableDialInConferencing
--------                                ------------------------
Global                                                      True
site:Redmond                                               False
site:Dublin                                                False
Tag:RedmondDialInUsers                                      True
Tag:DublinDialInUsers                                       True
</pre>

## <a name="filters"></a>Фильтры

Фильтры позволяют получить более конкретные наборы возвращаемых данных. Например, сводный отчет по конференциям ТСОП позволяет выбрать способ группирования данных. В этом случае конференции группируются по часу, дню, неделе или месяцу.
  
В следующей таблице приведены фильтры, которые можно использовать в сводном отчете по конференциям ТСОП.
  
**Фильтры сводного отчета по конференциям ТСОП**

|**Имя**|**Описание**|
|:-----|:-----|
|**From** <br/> |Дата и время начала диапазона. Чтобы просмотреть данные по часам, введите дату и время начала в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время начала, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**To** <br/> |Дата и время окончания диапазона. Чтобы просмотреть данные по часам, введите дату и время окончания в следующем формате:  <br/> 7/7/2015 13:00  <br/> Если не указать время окончания, автоматически будет выбрано время 00:00 указанного дня. Чтобы просмотреть данные по дням, просто введите дату:  <br/> 7/7/2015  <br/> Для просмотра по неделям или по месяцам введите дату, выпадающую на любой день недели или месяца, который вы хотите просмотреть (это необязательно должен быть первый день недели или месяца):  <br/> 7/3/2015  <br/> Неделя всегда начинается с воскресения и заканчивается субботой.  <br/> |
|**Interval** <br/> | Временной интервал. Выберите один из следующих вариантов: <br/>  Ежечасно (может отображаться до 25 часов) <br/>  Ежедневно (может отображаться до 31 дня) <br/>  Еженедельно (может отображаться до 12 недель) <br/>  Ежемесячно (может отображаться до 12 месяцев) <br/>  Если указаны такие даты начала и конца, что превышается максимально разрешенное количество значений для выбранного интервала, то отображается только максимально допустимое количество значений (начиная с даты начала). Например, если вы выбираете дневной интервал с датой начала 7/7/2015 и датой окончания 2/28/2015, данные отображаются за 8/7/7/2015 с 12:00 до 09:00 12:00 (то есть в общей сложности 31 день). <br/> |

**Метрики сводного отчета по конференциям ТСОП**

|**Имя**|**Возможность сортировки по этому показателю**|**Описание**|
|:-----|:-----|:-----|
|**Каждый час** <br/> **Ежедневно** <br/> **Еженедельные** <br/> **Ежемесячно** <br/> |Нет  <br/> |Указывает выбранный интервал времени. Если это возможно, вы можете щелкнуть заданный интервала для просмотра подробных сведений для этого интервала. Например, если вы используете ежедневный интервал и щелкните 7/7/2015, вы увидите почасовую разбивку активности регистрации пользователей на эту дату.  <br/> |
|**Total PSTN conferences (Общее количество конференций ТСОП)** <br/> |Нет  <br/> |Общее количество конференций, в которых разрешен телефонный доступ.  <br/> |
|**Total participants (Общее количество участников)** <br/> |Нет  <br/> |Общее количество людей, принявших участие в конференциях с разрешенным телефонными доступом.  <br/> |
|**Total A/V conference minutes (Общее время в аудио- и видеоконференциях в минутах)** <br/> |Нет  <br/> |Общее время аудио- и видеоконференций.  <br/> |
|**Total A/V conference participant minutes (Общее время в аудио- или видеоконференции в минутах)** <br/> |Нет  <br/> |Общее время, затраченное участниками в аудио- или видеоконференции. Например, если один участник потратил пять, а другой — три минуты в одной и той же аудио- или видеоконференции, то общее время участия в аудио- или видеоконференции составит восемь минут.  <br/> |
|**Total PSTN participants (Общее количество участников ТСОП)** <br/> |Нет  <br/> |Общее количество пользователей, подключившихся по телефону к конференциям, в которых разрешен телефонный доступ.  <br/> |
|**Total PSTN participant minutes (Общее время участников ТСОП в минутах)** <br/> |Нет  <br/> |Общее время, затраченное на конференцию участниками с телефонным подключением. Например, если один участник с телефонным подключением провел пять минут, а другой — три минуты в одной и той же конференции, то общее время участников ТСОП составит восемь минут.  <br/> |
|**Unique conference organizers (Общее количество уникальных организаторов конференций)** <br/> |Нет  <br/> |Общее количество пользователей, организовавших хотя бы одну конференцию с разрешенным телефонным доступом. Пользователь, организовавший несколько таких конференций, учитывается как один уникальный организатор, как те, кто организовал только одну конференцию.  <br/> |
