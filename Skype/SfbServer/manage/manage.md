# <a name="manage-skype-for-business-server"></a>Управление Skype для бизнеса Server 

**Сводка:** В этих ссылках вы узнаете, как управлять и поддерживать Skype для бизнеса Server.
  
Skype для бизнеса Server обеспечивает обмен мгновенными сообщениями, сведения о присутствии, конференц-связи, видео и телефонные решения, которые могут поддерживать требования к совместной работе на уровне предприятия. В этом разделе приведены ссылки на сведения о настройке, мониторинге и обслуживании развертывания Skype для бизнеса Server. 
  
Содержимое этих статей постоянно обновляется с учетом новых знаний и отзывов пользователей. Эта страница будет часто обновляться, поэтому регулярно проверяйте ее.
