# <a name="manage-skype-for-business-server"></a>Управление системой Skype для бизнеса Server 

**Сводка:** Перейдите по этим ссылкам, чтобы узнать, как управлять и поддерживать Скайп для Business Server.
  
Skype для бизнеса Server предлагает решения для обмена мгновенными сообщениями, предоставления информации о присутствии, конференций, видео и телефонии, которые могут поддерживать требования по совместной работе корпоративного уровня. В этом разделе представлены ссылки на сведения о способах настройки, контроля и обслуживания вашей Скайп для развертывания Business Server. 
  
Эти статьи регулярно обновляются с учетом появления новых сведений и отзывов пользователей. Данная страница будет постоянно обновляться, поэтому заходите почаще.
