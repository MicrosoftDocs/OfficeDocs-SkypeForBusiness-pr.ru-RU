# <a name="manage-skype-for-business-server-2015"></a>Управление системой Skype для бизнеса Server 2015

**Сводка:** Перейдите по этим ссылкам, чтобы узнать, как управлять и поддерживать Скайп Business Server 2015.
  
Скайп для Business Server предлагает мгновенного обмена Мгновенными сообщениями, сведения о присутствии, конференц-связь, видео- и телефонной связи решений, которые может поддерживать требованиям к совместной работе корпоративного уровня. В этом разделе представлены ссылки на сведения о способах настройки, контроля и обслуживания вашей Скайп для развертывания Business Server.
