# <a name="manage-skype-for-business-server"></a>Управление системой Skype для бизнеса Server 

**Сводка:** Просмотрите эти ссылки, чтобы узнать, как управлять и поддерживать Skype для бизнеса Server.
  
Skype для бизнеса Server предлагает решения для обмена мгновенными сообщениями, предоставления информации о присутствии, конференций, видео и телефонии, которые могут поддерживать требования по совместной работе корпоративного уровня. В этом разделе приведены ссылки на сведения о том, как настроить, отслеживать и поддерживать развертывание в Skype для бизнеса Server. 
  
Эти статьи регулярно обновляются с учетом появления новых сведений и отзывов пользователей. Данная страница будет постоянно обновляться, поэтому заходите почаще.
