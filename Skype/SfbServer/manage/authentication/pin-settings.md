# <a name="manage-pin-settings-in-skype-for-business-server"></a>Управление настройками ПИН-кодов в Skype для бизнеса Server
 
**Сводка:** Управление настройками ПИН-кодов в Skype для бизнеса Server.
  
Используйте процедуры, следующие в следующих разделах, для управления ПИН-номерами в Skype для бизнеса Server.
  
You can manage Skype for Business Server PIN polices from either Skype for Business Server Control Panel or Skype for Business Server Management Shell. Чтобы настроить политики ПИН-кодов для организации, используйте следующую процедуру.
  
- [Просмотр сведений о политике ПИН-кодов в Skype для бизнеса Server](view-pin-policy-information.md)
    
- [Создание новой политики ПИН-кодов в Skype для бизнеса Server](create-a-new-pin-policy.md)
    
- [Изменение существующей политики ПИН-кодов в Skype для бизнеса Server](modify-an-existing-pin-policy.md)
    
- [Удаление политики ПИН-кодов в Skype для бизнеса Server](delete-a-pin-policy.md)
    
- [Назначение политики ПИН-кодов на пользователя в Skype для бизнеса Server](assign-a-per-user-pin-policy.md)
    
Используйте следующие процедуры для управления ПИН-номерами пользователей для проведения  телефонных звонков из раздела "Пользователи" панели управления Skype для бизнеса Server.
  
- [Set a user's dial-in conferencing PIN in Skype for Business Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Просмотр сведений о ПИН-коде пользователя в Skype для бизнеса Server](view-user-pin-information.md)
    
- [Блокировка или разблокировка ПИН-кода пользователя в Skype для бизнеса Server](lock-or-unlock-a-user-pin.md)
