# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>Использовать двухфакторной проверки подлинности с помощью Скайп для бизнеса клиентов и Скайп для Business Server
 
**Сводка:** Используйте двухфакторной проверки подлинности с помощью Скайп для Business Server и Скайп для бизнеса.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Первый вход в Skype для бизнеса

Данные входа обычно настраивается автоматически при установке Скайп для бизнеса. Однако при первом использовании Скайп для бизнеса, может понадобиться вручную запустить клиента.
  
### <a name="to-sign-in-for-the-first-time"></a>Первый вход

1. Войдите в сети организации.
    
2. Выберите команду **запустить** > **Все программы** > **Скайп для бизнеса**.
    
    Отображается экран входа.
    
    - Если поле адреса уже заполнено, проверьте правильность адреса.
    
    - Если это не так или если поле не заполнено, введите адрес входа (обычно это то же, что ваш адрес электронной почты).
    
    - Если на экране отображается пустое поле пароля, введите свой пароль.
    
3. Выберите **Вход**.

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>Вход в Skype для бизнеса по смарт-карте

В настоящее время в некоторых организациях с целью повышения безопасности пользователей применяется многоэтапная процедура входа, называемая двухфакторной проверкой подлинности. Если вы должны использовать этот параметр, то необходимо «смарт-карт» для входа в Скайп для бизнеса. Смарт-карт может быть физических или виртуальных:
  
- **Физическая** О размере данные о банковской карте. При входе вставляется в устройство чтения смарт-карт.
    
- **Виртуальный** Не физического объекта, но электронных идентификатор, который записывается специальные микросхемы на компьютере, который фактически создает смарт-карт на компьютер пользователя. Доступно только для использования с компьютеров Windows 8, которые содержат микросхемы TPM (модуль).
    
### <a name="enroll-your-smart-card"></a>Регистрация смарт-карты

Перед входом с помощью смарт-карты, карточке должны быть «зарегистрированы» — то есть, учетные данные пользователя должны идентифицироваться по карточке. Это относится как к физической, так и к виртуальной карте. Этот процесс может уже были выполнить масштабирование путем вашей Скайп администратора Business Server. Обратитесь к их, если вы не знаете ли что это уже сделано.
  
> [!NOTE]
> Поскольку каждый виртуальный смарт-карт связан только с устройством устанавливается на, отдельные карточки, необходимо зарегистрировать для каждого компьютера Windows 8, которые можно использовать. 
  
### <a name="to-manually-enroll-your-smart-card"></a>Регистрация смарт-карты вручную

1. Войдите на компьютер, к которому вы будете работать под управлением Скайп для бизнеса на.
    
2. С помощью Internet Explorer, перейдите к странице вашей организации сертификат центра сертификации подача заявок через Интернет. 
    
    Если у вас нет уже его запрашивают вашей Скайп Business Server администратора веб-адрес этого ресурса. URL-адрес будет выглядеть примерно следующим образом: https://MyCA. [yourcompanyname] .com/certsrv.
    
    > [!NOTE]
    > При использовании Internet Explorer 10, может потребоваться просмотр этого веб-сайта в режиме совместимости. 
  
3. Когда появится запрос на вход в страницу сертификации, войдите в систему с помощью учетной записи домена (а не от имени администратора компьютера).
    
4. На странице приветствия на веб-сайте выберите **Запросить сертификат**.
    
5. Выберите **Расширенный запрос**.
    
6. Выберите **Создать и выдать запрос к этому ЦС**, затем нажмите кнопку **Далее**.
    
7. Теперь вы увидите страница с именем станции регистрации смарт-карт. Согласитесь на установку элемента ActiveX и заполните форму расширенного запроса сертификата, указав приведенные ниже данные.
    
    a) В раскрывающемся списке **Шаблон сертификата** выберите **Пользователь смарт-карты**.
    
    б) Выберите **Создать новый набор ключей**.
    
    в. Найдите сведения об изготовителе на этикетке смарт-карты и выберите этого изготовителя в раскрывающемся списке **Поставщик служб шифрования**.
    
    г. Выберите **CSP** как формат запроса, если он еще не установлен.
    
    e. Выберите из раскрывающегося списка алгоритм хэширования **sha1** , если он еще не установлен.
    
    f. Задайте для сертификата имя будет распознавать и нажмите кнопку **Отправить**.
    
8. Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.
    
9. По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678. 
  
10. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.
    
11. Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.
    
12. По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678. 
  
13. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.
    
14. Нажмите кнопку **ОК** для подтверждения сведений, отображаемых в сертификате.
    
15. При появлении уведомления о выдаче сертификата нажмите кнопку **Установить этот сертификат** для завершения процедуры регистрации.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Вход в Skype для бизнеса по учетным данным смарт-карты

Прежде чем использовать смарт-карты в первый раз, рекомендуется щелкните **Удалить Мои сведения входа** на Скайп для бизнеса на странице регистрации. При этом удаляются хранящиеся на компьютере учетные данные для входа, которые могут служить причиной ошибок.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Процедура в Skype для бизнеса по учетным данным смарт-карты

1. Запустите Скайп для клиента Business.
    
2. На экране входа введите имя учетной записи пользователя для входа в поле **Адрес для входа**, затем нажмите кнопку **Вход**.
    
3. Если вы используете виртуальную смарт-карту, пропустите этот этап.
    
    В случае физической смарт-карты вставьте ее по запросу в устройство чтения смарт-карт и после обнаружения карты нажмите кнопку **ОК**.
    
4. Введите PIN‑код смарт-карты,затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам ПИН-код для смарт-карты, введите значение по умолчанию: 12345678.
