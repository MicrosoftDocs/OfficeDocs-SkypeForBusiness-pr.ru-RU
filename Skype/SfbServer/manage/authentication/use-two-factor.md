# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>Использование двух коэффициентов проверки подлинности в клиенте Skype для бизнеса и Skype для бизнеса Server
 
**Сводка:** Используйте двух коэффициент проверки подлинности в Skype для бизнеса Server и Skype для бизнеса.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Первый вход в Skype для бизнеса

Обычно при установке Skype для бизнеса данные для регистрации настраиваются автоматически. Но при первом использовании Skype для бизнеса может потребоваться запустить клиент вручную.
  
### <a name="to-sign-in-for-the-first-time"></a>Вход в первый раз

1. Войдите в сеть организации.
    
2. Выберите   >  **"Запустить все программы** Skype для  >  **бизнеса".**
    
    Должен отсвеяться экран для входов.
    
    - Если адрес для входов уже заполнен, подтвердите правильность показанного адреса.
    
    - Если это неправильно или поле пустое, введите свой адрес для входа (обычно он такой же, как и адрес электронной почты).
    
    - Если отображается пустое поле пароля, добавьте пароль.
    
3. Выберите **"Вход"**.

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>Вход в Skype для бизнеса с помощью смарт-карты

В некоторых организациях теперь используется многоэтапный процесс, называемый двухфакторной проверкой подлинности, для повышения безопасности пользователей. If you're expected to use this option, you'll need a "smart card" to sign in to Skype for Business. Смарт-карты могут быть физическими или виртуальными:
  
- **Физические** Размер кредитной карты. Его необходимо вставить в устройство чтения смарт-карт при входе в систему.
    
- **Виртуальный** Не физический объект, а электронный идентификатор, который записуется в специальную микросхему на компьютере, которая, по сути, создает смарт-карту на компьютере. Доступно только для Windows 8 компьютеров, содержащих микросхему доверенного платформенного модуля (TPM).
    
### <a name="enroll-your-smart-card"></a>Регистрация смарт-карты

Прежде чем вы сможете войти с помощью смарт-карты, ее необходимо "зарегистрировать", то есть учетные данные пользователя должны быть определены с помощью этой карты. В этом случае карточка является физической или виртуальной. Этот процесс может уже быть выполнен администратором Skype для бизнеса Server. Обратитесь к ним, если вы не знаете, было ли это сделано.
  
> [!NOTE]
> Так как каждая виртуальная смарт-карта связана только с устройством, на которое она установлена, для каждого используемой виртуальной карты Windows 8 отдельную карту. 
  
### <a name="to-manually-enroll-your-smart-card"></a>Регистрация смарт-карты вручную

1. Войдите на компьютер, на который будет запущен Skype для бизнеса.
    
2. В Internet Explorer перейдите на страницу веб-регистрации в службе сертификации вашей организации. 
    
    Попросите администратора Skype для бизнеса Server упросить веб-адрес этого ресурса, если он у вас еще нет. URL-адрес будет выглядеть так: https://MyCA .[ yourcompanyname].com/certsrv.
    
    > [!NOTE]
    > Если вы используете Internet Explorer 10, вам может потребоваться просмотреть этот веб-сайт в режиме совместимости. 
  
3. Когда вам будет предложено войти на страницу сертификации, войдите с помощью учетной записи домена (а не администратора компьютера).
    
4. На странице приветствия веб-сайта выберите запрос **сертификата.**
    
5. Выберите **расширенный запрос.**
    
6. Выберите **"Создать" и отправьте запрос в этот ЦС,** а затем нажмите кнопку **"Далее".**
    
7. Теперь вы увидите страницу под названием "Станция регистрации смарт-карт". Утвердит запрос на установку ActiveX, а затем заполнит форму "Расширенный запрос сертификата" следующим образом:
    
    а. Выберите **пользователя смарт-карты** в **списке** шаблонов сертификатов.
    
    б. Выберите **"Создать новый набор ключей".**
    
    c. Найдите сведения о производителе на метки смарт-карты и выберите этого производителя в выпадаемом списке **CSP.**
    
    г. Выберите **CSP** в качестве формата запроса, если он еще не выбран.
    
    д. Выберите **sha1** в выпадаемом списке "Алгоритм hash", если он еще не выбран.
    
    е. Придайте сертификату имя, которое вы будете распознавать, и нажмите кнопку **"Отправить".**
    
8. Теперь вставьте пустую смарт-карту в устройство чтения карт, подключенный к станции регистрации, и нажмите кнопку **"Регистрация".**
    
9. При запросе введите свой персональный идентификационный номер (PIN-код) и нажмите кнопку **"ОК".**
    
    > [!NOTE]
    > Если ваш человек технической поддержки не предоставил вам специальный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода смарт-карты по умолчанию, которое составляет 12345678. 
  
10. Выберите параметр, чтобы пользователь (вы) изменил ПИН-код при первом ее выборе.
    
11. Теперь вставьте пустую смарт-карту в устройство чтения карт, подключенный к станции регистрации, и нажмите кнопку **"Регистрация".**
    
12. При запросе введите свой персональный идентификационный номер (PIN-код) и нажмите кнопку **"ОК".**
    
    > [!NOTE]
    > Если ваш человек технической поддержки не предоставил вам специальный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода смарт-карты по умолчанию, которое составляет 12345678. 
  
13. Выберите параметр, чтобы пользователь (вы) изменил ПИН-код при первом ее выборе.
    
14. Нажмите **кнопку** "ОК", чтобы подтвердить, что в отображаемом сертификате есть ваши сведения.
    
15. После того как вы увидите уведомление о выдаче сертификата, нажмите кнопку **"Установить этот сертификат",** чтобы завершить процесс регистрации.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Вход в Skype для бизнеса с помощью учетных данных смарт-карты

Перед первым использованием **смарт-карты** рекомендуется нажать кнопку "Удалить мои данные для регистрации" на странице "Вход в Skype для бизнеса". Это устраняет все учетные данные для входа, хранимые на компьютере, и исключает возможный источник ошибки.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Вход в Skype для бизнеса с помощью учетных данных смарт-карты

1. Запустите клиент Skype для бизнеса.
    
2. На экране "Вход" введите имя учетной  записи пользователя для регистрации в поле "Вход" и нажмите кнопку **"Войти".**
    
3. Если вы используете виртуальную смарт-карту, пропустите этот шаг.
    
    Если вы используете физическую смарт-карту, вставьте смарт-карту в устройство чтения смарт-карт и запросите ее, а затем нажмите кнопку "ОК" при обнаружении карты. 
    
4. Введите ПИН-код смарт-карты и нажмите кнопку **"ОК".**
    
    > [!NOTE]
    > Если сотруднику службы поддержки не назначен ПИН-код смарт-карты, используйте значение по умолчанию 12345678.
