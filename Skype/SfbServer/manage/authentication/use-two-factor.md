# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>Использование двухфакторной проверки подлинности в клиенте Skype для бизнеса и Skype для бизнеса Server
 
**Сводка:** Использование двухфакторной проверки подлинности в Skype для бизнеса Server и Skype для бизнеса.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Первый вход в Skype для бизнеса

Данные для входа обычно настраиваются автоматически при установке Skype для бизнеса. Но в первый раз при использовании Skype для бизнеса может потребоваться запуск клиента вручную.
  
### <a name="to-sign-in-for-the-first-time"></a>Первый вход

1. Войдите в сеть своей организации.
    
2. Нажмите кнопку **начать** > **все программы** > **в Skype для бизнеса**.
    
    Отображается экран входа.
    
    - Если поле адреса уже заполнено, проверьте правильность адреса.
    
    - Если это не так, или если флажок пуст, введите адрес для входа (обычно это то же самое, что и адрес электронной почты).
    
    - Если на экране отображается пустое поле пароля, введите свой пароль.
    
3. Выберите **Вход**.

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>Вход в Skype для бизнеса по смарт-карте

В настоящее время в некоторых организациях с целью повышения безопасности пользователей применяется многоэтапная процедура входа, называемая двухфакторной проверкой подлинности. Если вы планируете использовать этот параметр, для входа в Skype для бизнеса вам понадобится "смарт-карта". Смарт-карты могут быть как физическими, так и виртуальными.
  
- **Физическое** О размере кредитной карты. При входе вставляется в устройство чтения смарт-карт.
    
- **Виртуальный** Не физический объект, но электронный идентификатор, который записывается в специальную микросхему вашего компьютера, что в сущности — это сборка смарт-карты на компьютере. Доступно только для компьютеров с Windows 8, которые содержат микросхему TPM (доверенный платформенный модуль).
    
### <a name="enroll-your-smart-card"></a>Регистрация смарт-карты

Перед тем как выполнять вход с помощью смарт-карты, она должна быть зарегистрирована, то есть ваши учетные данные пользователя должны быть идентифицированы вместе с картой. Это относится как к физической, так и к виртуальной карте. Этот процесс может быть уже выполнен администратором Skype для бизнеса Server. Если вы не уверены в том, что это так, проверьте, есть ли вы.
  
> [!NOTE]
> Так как каждая виртуальная смарт-карта связана только с устройством, на котором она установлена, для каждого используемого компьютера с Windows 8 необходимо зарегистрировать отдельную карту. 
  
### <a name="to-manually-enroll-your-smart-card"></a>Регистрация смарт-карты вручную

1. Войдите в систему на компьютере, на котором работает Skype для бизнеса.
    
2. С помощью Internet Explorer перейдите на страницу веб-регистрации центра сертификации в своей организации. 
    
    Если вы еще не сделали этого, обратитесь к администратору Skype для бизнеса Server за веб-адресом этого ресурса. URL-адрес будет выглядеть примерно так: https://MyCA. [йоуркомпанинаме]. com/certsrv.
    
    > [!NOTE]
    > Если вы используете Internet Explorer 10, вам может потребоваться просмотреть этот веб-сайт в режиме совместимости. 
  
3. Когда вам будет предложено войти на страницу сертификации, войдите в систему с помощью учетной записи домена (а не администратора компьютера).
    
4. На странице приветствия на веб-сайте выберите **Запросить сертификат**.
    
5. Выберите **Расширенный запрос**.
    
6. Выберите **Создать и выдать запрос к этому ЦС**, затем нажмите кнопку **Далее**.
    
7. Теперь появится страница с названием станции подачи заявки на смарт-карту. Согласитесь на установку элемента ActiveX и заполните форму расширенного запроса сертификата, указав приведенные ниже данные.
    
    a) В раскрывающемся списке **Шаблон сертификата** выберите **Пользователь смарт-карты**.
    
    б) Выберите **Создать новый набор ключей**.
    
    в. Найдите сведения об изготовителе на этикетке смарт-карты и выберите этого изготовителя в раскрывающемся списке **Поставщик служб шифрования**.
    
    г. Выберите в качестве формата запроса **CSP** , если он еще не выбран.
    
    учеб. Выберите **SHA1** из раскрывающегося списка алгоритм хэширования, если он еще не выбран.
    
    н. Присвойте сертификату имя, которое вы будете распознать, и нажмите кнопку **Отправить**.
    
8. Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.
    
9. По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678. 
  
10. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.
    
11. Теперь вставьте пустую смарт-карту в устройство чтения карт, подсоединенное к станции регистрации, и нажмите кнопку **Заявка**.
    
12. По запросу введите личный идентификационный номер (PIN‑код, затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам личный ПИН-код для регистрации смарт-карты, используйте значение ПИН-кода по умолчанию: 12345678. 
  
13. Активируйте параметр принудительной смены пользователем (вами) ПИН-кода при первом использовании смарт-карты.
    
14. Нажмите кнопку **ОК** для подтверждения сведений, отображаемых в сертификате.
    
15. При появлении уведомления о выдаче сертификата нажмите кнопку **Установить этот сертификат** для завершения процедуры регистрации.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Вход в Skype для бизнеса по учетным данным смарт-карты

Прежде чем использовать смарт-карту в первый раз, рекомендуется нажать кнопку **удалить мои данные для входа** на странице входа в Skype для бизнеса. При этом удаляются хранящиеся на компьютере учетные данные для входа, которые могут служить причиной ошибок.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Процедура в Skype для бизнеса по учетным данным смарт-карты

1. Запустите клиент Skype для бизнеса.
    
2. На экране входа введите имя учетной записи пользователя для входа в поле **Адрес для входа**, затем нажмите кнопку **Вход**.
    
3. Если вы используете виртуальную смарт-карту, пропустите этот этап.
    
    В случае физической смарт-карты вставьте ее по запросу в устройство чтения смарт-карт и после обнаружения карты нажмите кнопку **ОК**.
    
4. Введите PIN‑код смарт-карты,затем нажмите кнопку **ОК**.
    
    > [!NOTE]
    > Если специалист службы поддержки не предоставил вам ПИН-код для смарт-карты, введите значение по умолчанию: 12345678.
