# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>Используйте двух факторов проверку подлинности с Skype для бизнеса клиентом и Skype для бизнеса Server
 
**Сводка:** Используйте двух факторов проверку подлинности с Skype для бизнеса Server и Skype для бизнеса.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>Вход в Skype для бизнеса первый раз

Сведения о входе обычно настраиваются автоматически при Skype для бизнеса установки. Но при первом использовании Skype для бизнеса может потребоваться вручную запустить клиента.
  
### <a name="to-sign-in-for-the-first-time"></a>Первый вход

1. Войдите в сеть организации.
    
2. Выберите **запуск**  >  **всех**  >  **Skype для бизнеса**.
    
    Вы должны увидеть экран входного знака.
    
    - Если адресное окно для входов уже заполнено, подтвердите правильность показанного адреса.
    
    - Если это неправильно или если поле пусто, введите свой адрес входа (обычно это то же самое, что и адрес электронной почты).
    
    - Если отображается пустой пароль, добавьте пароль.
    
3. Выберите **вход.**

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>Во входе Skype для бизнеса с помощью смарт-карты

Некоторые организации теперь используют многоэтапный процесс регистрации, называемый двухфакторной проверкой подлинности, для повышения безопасности пользователей. Если ожидается, что вы используете этот параметр, вам потребуется "смарт-карта", чтобы войти в Skype для бизнеса. Смарт-карты могут быть физическими или виртуальными:
  
- **Физические** Размер кредитной карты. Вы вставляете его в устройство чтения смарт-карт при входе в систему.
    
- **Виртуальный** Не физический объект, а электронный идентификатор, который будет записан на специальный чип на компьютере, который, по сути, создает смарт-карту на компьютере. Доступно только для Windows 8 компьютеров с чипом TPM (Доверенный модуль платформы).
    
### <a name="enroll-your-smart-card"></a>Регистрация смарт-карты

Перед входом с помощью смарт-карты необходимо "зарегистрироваться", то есть учетные данные пользователей должны быть определены с помощью карты. Это тот случай, если карта является физической или виртуальной. Этот процесс уже может быть выполнен вашим Skype для бизнеса Server администратором. Проверьте у них, не уверены ли вы в том, что было сделано.
  
> [!NOTE]
> Так как каждая виртуальная смарт-карта связана только с устройством, на которое она установлена, для каждого используемго компьютера Windows 8 отдельная карта. 
  
### <a name="to-manually-enroll-your-smart-card"></a>Ручная регистрация смарт-карты

1. Войдите на компьютер, на который будет запущен Skype для бизнеса.
    
2. С помощью Internet Explorer просмотрите страницу веб-регистрации сертификатов организации. 
    
    Попросите администратора Skype для бизнеса Server веб-адрес этого ресурса, если он еще не имеется. URL-адрес будет выглядеть так: https://MyCA .[ yourcompanyname].com/certsrv.
    
    > [!NOTE]
    > Если вы используете Internet Explorer 10, вам может потребоваться просмотреть этот веб-сайт в режиме совместимости. 
  
3. Когда вам будет предложено войти на страницу сертификации, войдите в систему с помощью учетной записи домена (а не в качестве администратора компьютера).
    
4. На странице Добро пожаловать на веб-сайт **выберите Запрос сертификата.**
    
5. Выберите **расширенный запрос**.
    
6. Выберите **Создать и отправить запрос в этот ЦС,** а затем нажмите **кнопку Далее**.
    
7. Теперь вы увидите страницу под названием Станция регистрации смарт-карт. Одобрить запрос на установку ActiveX, а затем заполнить форму предварительного запроса сертификата следующим образом:
    
    А. Выберите **пользователя Smartcard из** списка **выпаданий** шаблона сертификатов.
    
    Б. Выберите **Создание нового набора ключей.**
    
    c. Найдите сведения о производителе на этикетке смарт-карты и выберите этого производителя из списка отсевов **CSP.**
    
    d. Выберите **CSP** в качестве формата запроса, если он еще не выбран.
    
    e. Выберите **sha1** из списка выпаданий алгоритма hash, если он еще не выбран.
    
    е) Дайте сертификату имя, которое вы узнаете, и нажмите кнопку **Отправить**.
    
8. Теперь вставьте пустую смарт-карту в читатель карты, присоединенный к станции регистрации, и нажмите кнопку **Регистрация**.
    
9. При запросе введите личный идентификационный номер (PIN-код), а затем нажмите **кнопку ОК**.
    
    > [!NOTE]
    > Если лицо технической поддержки не предоставило вам специальный ПИН-код для регистрации смарт-карты, используйте пин-код смарт-карты по умолчанию, который 12345678. 
  
10. Выберите опцию, чтобы заставить пользователя (вас) изменить ПИН-код при первом использования смарт-карты.
    
11. Теперь вставьте пустую смарт-карту в читатель карты, присоединенный к станции регистрации, и нажмите кнопку **Регистрация**.
    
12. При запросе введите личный идентификационный номер (PIN-код), а затем нажмите **кнопку ОК**.
    
    > [!NOTE]
    > Если лицо технической поддержки не предоставило вам специальный ПИН-код для регистрации смарт-карты, используйте пин-код смарт-карты по умолчанию, который 12345678. 
  
13. Выберите опцию, чтобы заставить пользователя (вас) изменить ПИН-код при первом использования смарт-карты.
    
14. Нажмите **кнопку ОК,** чтобы подтвердить, что отображаемая справка имеет сведения о нем.
    
15. Как только вы увидите уведомление о выдаче сертификата, нажмите **кнопку Установите** этот сертификат, чтобы завершить процесс регистрации.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Во входе Skype для бизнеса с учетными данными смарт-карт

Перед первым использованием **смарт-карты** рекомендуется нажать кнопку Удалить сведения о входе Skype для бизнеса на странице регистрации. Это устраняет все учетные данные входа, хранимые на компьютере, и устраняет возможный источник ошибок.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>Чтобы войти в Skype для бизнеса с учетными данными смарт-карты

1. Запустите Skype для бизнеса клиента.
    
2. На экране "Вход" введите свой знак  в имени учетной записи пользователя в адресной окне Вход, а затем нажмите **кнопку Вход**.
    
3. Если вы используете виртуальную смарт-карту, пропустите этот шаг.
    
    Если вы используете физическую смарт-карту, вставьте смарт-карту в устройство чтения смарт-карт и нажмите кнопку **ОК** при обнаружении карты.
    
4. Введите ПИН-код для смарт-карты и нажмите **кнопку ОК.**
    
    > [!NOTE]
    > Если вам не назначен пин-код смарт-карты вашим сотрудником службы поддержки, используйте значение по умолчанию, которое 12345678.
