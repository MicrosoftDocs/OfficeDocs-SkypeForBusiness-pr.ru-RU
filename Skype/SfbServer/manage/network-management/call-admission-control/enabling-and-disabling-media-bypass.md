# <a name="enabling-and-disabling-media-bypass-in-skype-for-business-server"></a>Включение и отключение обхода сервера-посредника в Skype для бизнеса Server

Используйте процедуры, которые данная статья позволяет включить или отключить обход сервера-посредника с помощью панели управления Skype для бизнеса Server.

## <a name="enable-network-media-bypass"></a>Включить сетевой обход мультимедиа 

Параметры обхода сервера-посредника применяются глобально в развертывании Skype для бизнеса Server. Обход сервера-посредника позволяет звонкам обходить сервер-посредник. Сведения о том, когда использовать обход мультимедиа, см. в подсети ["Планирование обхода мультимедиа".](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md)

Вы можете включить и настроить обход сервера-посредника с помощью панели управления Skype для бизнеса Server.


### <a name="to-enable-and-configure-media-bypass"></a>Включение и настройка обхода сервера-посредника

1.  Из учетной записи пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или назначена роли CsAdministrator, войдите на любой компьютер во внутреннем развертывании.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  В левой панели навигации щелкните **"Конфигурация** сети" и выберите **"Глобальная".**

4.  На странице **Глобальная** выберите конфигурацию **Глобальная**. На этой странице всегда приводится только одна конфигурация, и она всегда имеет название "Глобальная".

5.  В меню **Правка** выберите команду **Показать сведения**.

6.  На странице **Изменение глобального параметра** установите флажок **Включить обход сервера-посредника**.

7.  Выберите один из следующих вариантов.
    
      - **Всегда обходить**   Выберите этот параметр, чтобы попытаться обойти посредника во всех вызовах. Этот параметр будет недоступен, если включен контроль допуска вызовов (CAC). Если CAC не включен, выберите этот параметр в следующих ситуациях:
        
          - не требуется контроль за пропускной способностью канала;
        
          - не требуется выполнять сложный контроль, когда должен происходить обход;
        
          - между клиентами и шлюзами есть подключение.
    
      - **Использование конфигурации сайтов и регионов**   Если cac включен, этот параметр выбран по умолчанию и не может быть изменен. При выборе этого параметра для определения возможности обхода мультимедиа будут использоваться узлы и регионы конфигурации сети. При выборе этого параметра можно включить обход для ненавязаных сайтов. Если  с одной областью связан один или несколько крупных сайтов, которые не имеют ограничений пропускной способности (например, большой центральный сайт), а также с некоторыми сайтами филиалов, связанными с той же областью, с которыми связаны ограничения пропускной способности, щелкните "Включить обход". Когда вы включаете обход для ненастраиваемых сайтов, конфигурация упрощается, так как указываются только подсети, связанные с сайтами филиалов, а не требуется указывать все подсети, связанные со всеми сайтами. Если контроль cac включен,  не рекомендуется выбирать обход включения для ненавязчивых сайтов.

8.  Чтобы сохранить изменения, нажмите кнопку **Сохранить**.


## <a name="disable-network-media-bypass"></a>Отключение обхода сетевого носители

Параметры обхода сервера-посредника применяются глобально в развертывании Skype для бизнеса Server. Обход сервера-посредника позволяет звонкам обходить сервер-посредник. Сведения о том, когда использовать обход мультимедиа, см. в подсети ["Планирование обхода мультимедиа".](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md) Вы можете отключить обход сервера-посредника на панели управления Skype для бизнеса Server. 


### <a name="to-disable-media-bypass"></a>Отключение обхода медиаданных

1.  Из учетной записи пользователя, которая является членом группы RTCUniversalServerAdmins (или имеет эквивалентные права пользователя) или назначена роли CsAdministrator, войдите на любой компьютер во внутреннем развертывании.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  В левой панели навигации щелкните **"Конфигурация** сети" и выберите **"Глобальная".**

4.  На странице **Глобальная** выберите конфигурацию **Глобальная**. На этой странице всегда приводится только одна конфигурация, и она всегда имеет название "Глобальная".

5.  В меню **Изменить** щелкните **Просмотреть сведения**.

6.  На странице **Изменить глобальную настройку** снимите флажок **Разрешить обход мультимедиа**.

7.  Щелкните **Сохранить**, чтобы сохранить изменения.
