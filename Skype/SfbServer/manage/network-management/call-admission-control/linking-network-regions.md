# <a name="linking-network-regions-in-skype-for-business-server"></a>Ссылка на области сети в Скайп для Business Server

Можно настроить ссылки между двумя сетевыми регионами в рамках контроля допуска звонков (CAC). Для просмотра сведений о канале области newtwork или Настройка или удаление связей между областями сетевые используйте разделы в этой статье. 

## <a name="view-network-region-link-information"></a>Просмотр сведений о канале области сети 

Вы можете просмотреть ссылки между двумя сетевыми регионами в рамках контроля допуска звонков (CAC). Области сети, связаны через подключение к физической глобальной сети (WAN). Скайп для панели управления Business Server можно использовать для просмотра существующей связи между двумя сетевыми регионами. 


### <a name="to-view-a-network-region-link-in-skype-for-business-server-control-panel"></a>Просмотр канала области сети в Скайп для панели управления Business Server

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 

3.  В левой панели навигации щелкните **Конфигурация сети**и нажмите кнопку **Связь между областями**.

4.  На странице **Связь между областями** щелкните связь, который требуется просмотреть.
    
    > [!NOTE]  
    > Можно просматривать только сведения об одном канале области за раз.

5.  В меню **Правка** выберите **Показать подробности**.

### <a name="view-network-region-link-information-by-using-windows-powershell-cmdlets"></a>Просмотр сведений о канале области сети с помощью командлетов Windows PowerShell

Между областями сети можно просмотреть с помощью командлета **Get-CsNetworkRegionLink** и Windows PowerShell. Этот командлет можно запустить из Скайп для консоли Business Server или из удаленного сеанса Windows PowerShell. 


### <a name="to-view-network-region-link-information"></a>Для просмотра сведений о канале области сети

  - Чтобы просмотреть сведения о всех каналах между областями сети, введите следующую команду в Скайп для консоли Business Server и нажмите клавишу ВВОД:
    
        Get-CsNetworkRegionLink
    
    Этой командой возвращается информация, аналогичная следующим сведениям:
    
        Identity            : NorthwestToCalifornia
        BWPolicyProfileID   :
        NetworkRegionLinkID : NorthwestToCalifornia
        NetworkRegionID1    : Pacific Northwest
        NetworkRegionID2    : California


Дополнительные сведения см [Get-CsNetworkRegionLink](https://docs.microsoft.com/powershell/module/skype/Get-CsNetworkRegionLink).


## <a name="configure-network-region-links"></a>Настройка связей между сетевыми областями 

Можно настроить ссылки между двумя сетевыми регионами в рамках контроля допуска звонков (CAC). Области сети, связаны через подключение к физической глобальной сети (WAN). Скайп для панели управления Business Server можно использовать для определения связи между двумя сетевыми регионами и устанавливать ограничения пропускной способности подключения к аудио- и видеоконференций между этими областями.

### <a name="to-create-a-network-region-link"></a>Создание связи между областями сети

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 

3.  В левой панели навигации щелкните **Конфигурация сети**и нажмите кнопку **Связь между областями**.

4.  На странице **Связь между областями** нажмите кнопку **Создать**.

5.  В окне **Создание связи между областями**введите значение в поле **имя** .
 
    > [!NOTE]  
    > Это значение должно быть уникальным в рамках вашей Скайп для развертывания Business Server.

6.  Из **области сети \#1** раскрывающегося списка, выберите один из двух областей, которые необходимо связать.

7.  Из **области сети \#2** раскрывающегося списка выберите регион, должны быть связаны. Эта область должно отличаться от региона, выбранного для области сети \#1.

8.  (Необязательно) Если вы хотите наложить ограничения полосы пропускания на голосовые вызовы и видеовызовы между этими областями, выберите профиль политики пропускной способности из раскрывающегося списка **Политика пропускной способности** .

9.  Нажмите **Исполнить**.

### <a name="to-modify-a-network-region-link"></a>Изменение связи сетевой области

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 

3.  В левой панели навигации щелкните **Конфигурация сети**и нажмите кнопку **Связь между областями**.

4.  На странице **Связь между областями** щелкните связь, которую требуется изменить.

5.  В меню **Правка** щелкните **Подробнее**.

6.  В окне **Изменение связи между областями**можно изменить связанные области или профиль политики пропускной способности для этой ссылки.

7.  Нажмите **Исполнить**.


## <a name="delete-network-region-links"></a>Удаление связей между сетевыми областями

Можно настроить ссылки между двумя сетевыми регионами в рамках контроля допуска звонков (CAC). Области сети, связаны через подключение к физической глобальной сети (WAN). Скайп для панели управления Business Server можно использовать для удаления существующей связи между двумя сетевыми регионами. 

### <a name="to-delete-a-network-region-link"></a>Чтобы удалить связь между областями сети

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 

3.  В левой панели навигации щелкните **Конфигурация сети**и нажмите кнопку **Связь между областями**.

4.  На странице **Связь между областями** щелкните связь, которую требуется удалить.
 
    > [!NOTE]  
    > Одновременно можно удалить несколько связь между областями. Для этого нажмите клавишу CTRL и выберите несколько связей между областями, удерживая нажатой клавишу CTRL. Или, чтобы выделить все связи между областями, нажмите кнопку <STRONG>Выбрать все</STRONG> в меню <STRONG>Правка</STRONG> .

5.  В меню **Правка** выберите команду **Удалить**.

6.  Нажмите **OK**.
