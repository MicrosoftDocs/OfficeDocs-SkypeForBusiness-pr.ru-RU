# <a name="enabling-call-admission-control-in-skype-for-business-server"></a>Включение контроля допуска звонков в Skype для бизнеса Server

Контроль доступа звонков — это сеть регионов, сайтов и подсетей, которые позволяют вам налагать ограничения на передачу звука и видео в зависимости от доступной пропускной способности. После настройки сетевой контроля допуска звонков необходимо включить контроль допуска звонков для принудительного применения ограничения пропускной способности. Для этого можно использовать Скайп для панели управления Business Server.


## <a name="to-enable-cac-from-the-skype-for-business-server-control-panel"></a>Чтобы включить контроль допуска звонков из Скайп для панели управления Business Server

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администрирования, чтобы открыть Скайп для панели управления Business Server. 

3.  В левой панели навигации щелкните **Конфигурация сети**и щелкните **Глобальная**.

4.  На странице **глобально** выберите **глобальную** конфигурацию.
   
    > [!NOTE]  
    > Только один сетевой можно настроить для любого Скайп для развертывания Business Server, поэтому никогда не будет более одного конфигурация сети в списке. Невозможно переименование глобальную конфигурацию.

5.  В меню **Правка** щелкните **Подробнее**.

6.  На странице **Изменение глобального параметра** установите флажок **Включить контроль допуска звонков** и нажмите кнопку **Сохранить**.

Если нажать кнопку **зафиксировать**, выполнить проверку конфигурации. Диалоговое окно **Изменение глобальных параметров** закроется, на страницу **Global** . Если в конфигурации сети, которая не позволит правильно работать (например, если каждого региона не подключен к каждой области через регионами; маршруты) обнаруживаются ошибки и несоответствия будет показано предупреждение.

При внесении изменений в конфигурацию сети можно запустить проверку еще раз, открыв глобальную конфигурацию и нажмите кнопку **Сохранить**. Необходимо сначала отключить контроль допуска звонков: этот флажок установлен и нажмите кнопку **Сохранить**. Это можно сделать в любое время без внесения изменений конфигурации.
