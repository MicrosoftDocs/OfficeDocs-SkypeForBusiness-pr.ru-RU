# <a name="managing-network-bandwidth-policy-profiles-in-skype-for-business-server"></a>Управление профилями политики пропускной способности сети в Skype для бизнеса Server

В этой статье описаны процедуры, которые можно использовать для просмотра, создания, изменения и удаления профилей политики пропускной способности сети.

## <a name="view-network-bandwidth-policy-profile-information"></a>Просмотр данных профиля политики пропускной способности сети

В рамках управления допуском звонков (CAC) для определения ограничений пропускной способности для некоторых модальностей используется политика пропускной способности. В Skype для бизнеса Server можно назначать ограничения пропускной способности только для звуковых и видеомодальностей. Вы можете настроить общие ограничения пропускной способности и ограничения сеанса. С помощью панели управления "Skype для бизнеса" можно создавать, изменять и удалять профили контейнеров для этих политик. Каждый профиль политики пропускной способности может быть связан с одним или несколькими сетевыми сайтами. Чтобы просмотреть профиль политики пропускной способности, выполните указанные ниже действия. 

### <a name="to-view-a-bandwidth-policy-profile"></a>Просмотр профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  На панели навигации слева выберите пункт **Настройка сети** , а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** выберите профиль политики пропускной способности, который вы хотите просмотреть.

5.  В меню **Правка** щелкните **Подробнее**.


### <a name="viewing-network-bandwidth-policy-profile-information-by-using-windows-powershell-cmdlets"></a>Просмотр данных профиля политики пропускной способности сети с помощью командлетов Windows PowerShell

Профили пропускной способности сети можно просматривать с помощью Windows PowerShell и командлета Get-Кснетворкбандвидсполиципрофиле. Этот командлет можно выполнить либо из командной консоли Skype для бизнеса Server, либо из удаленного сеанса Windows PowerShell. 


### <a name="to-view-network-bandwidth-policy-profile-information"></a>Просмотр данных профиля политики пропускной способности сети

  - Чтобы просмотреть сведения о всех профилях политики пропускной способности сети, введите в командной консоли Skype для Business Server указанную ниже команду и нажмите клавишу ВВОД.
    
        Get-CsNetworkBandwidthPolicyProfile
    
    Команда возвращает примерно следующую информацию:
    
        Identity          : RedmondBandwidthPolicy
        BWPolicy          : {BWLimit=200;BWSessionLimit=200;
                            BWPolicyModality=Audio, 
                            BWLimit=1400;BWSessionLimit=500;
                            BWPolicyModality=Video}
        BWPolicyProfileID : RedmondBandwidthPolicy
        Description       :


Дополнительные сведения можно найти в разделе справки по командлету [Get-кснетворкбандвидсполиципрофиле](https://docs.microsoft.com/powershell/module/skype/Get-CsNetworkBandwidthPolicyProfile) .


## <a name="create-or-modify-bandwidth-policy-profiles"></a>Создание и изменение профилей политики пропускной способности

В рамках управления допуском звонков (CAC) для определения ограничений пропускной способности для некоторых модальностей используется политика пропускной способности. В Skype для бизнеса Server можно назначать ограничения пропускной способности только для звуковых и видеомодальностей. Вы можете настроить общие ограничения пропускной способности и ограничения сеанса. С помощью панели управления "Skype для бизнеса" можно создавать, изменять и удалять профили контейнеров для этих политик. Каждый профиль политики пропускной способности может быть связан с одним или несколькими сетевыми сайтами. Чтобы создать или изменить профиль политики пропускной способности, выполните указанные ниже действия. 

### <a name="to-create-a-new-bandwidth-policy-profile"></a>Создание нового профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  На панели навигации слева выберите пункт **Настройка сети**, а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** нажмите кнопку **создать**.

5.  В окне **новый профиль политики пропускной способности**введите имя в поле **имя** . Это имя должно быть уникальным среди всех профилей политики пропускной способности.

6.  В поле **предельное число звуков** введите числовое значение. Это значение — это максимальный объем пропускной способности, выделяемый для всех звуковых подключений, выраженный в кбит/с.

7.  Введите числовое значение в поле " **Ограничение сеанса голосовой связи** ". Это значение — это максимальный объем пропускной способности, выделяемый для отдельного звукового подключения, выраженный в кбит/с. Это значение должно быть 40 или выше.

8.  Введите числовое значение в поле "предельный **Размер видео** ". Это значение — это максимальный объем пропускной способности, выделяемый для всех видеоподключений, выраженный в кбит/с.

9.  Введите числовое значение в поле " **Ограничение сеанса видео** ". Это значение — это максимальный объем пропускной способности, выделяемый для отдельного видеосоединения, выраженный в КБ/с. Это значение должно быть 100 или выше.

10. Необязательно Введите значение в поле **Описание** , чтобы получить дополнительные сведения об этом профиле политики пропускной способности, которое не может быть выражено только именем.

11. Нажмите **Исполнить**.

    > [!NOTE]  
    > Создание профиля политики пропускной способности не приводит к автоматическому применению ограничений для пропускной способности. Сначала необходимо связать профиль политики с сайтом. 


### <a name="to-modify-a-bandwidth-policy-profile"></a>Изменение профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  На панели навигации слева выберите пункт **Настройка сети**, а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** выберите профиль политики пропускной способности, который вы хотите изменить.

5.  В меню **Правка** щелкните **Подробнее**.

6.  На странице **профиля политики изменения пропускной способности** измените соответствующие поля (Дополнительные сведения [можно найти в разделе Создание нового профиля политики пропускной способности](#to-create-a-new-bandwidth-policy-profile)).

7.  Нажмите **Исполнить**.

    > [!NOTE]  
    > При изменении профиля политики пропускной способности немедленно обновляются ограничения пропускной способности всех сайтов сети, связанных с этим профилем политики пропускной способности.

  
## <a name="delete-network-bandwidth-policy-profiles"></a>Удаление профилей политики пропускной способности сети

В рамках управления допуском звонков (CAC) для определения ограничений пропускной способности для некоторых модальностей используется политика пропускной способности. В Skype для бизнеса Server можно назначать ограничения пропускной способности только для звуковых и видеомодальностей. Вы можете настроить общие ограничения пропускной способности и ограничения сеанса. С помощью панели управления "Skype для бизнеса" можно создавать, изменять и удалять профили контейнеров для этих политик. Для удаления профилей политики пропускной способности сети выполните указанные ниже действия. 

### <a name="to-delete-a-bandwidth-policy-profile"></a>Удаление профиля политики пропускной способности

1.  Войдите на любой компьютер, находящийся во внутреннем развертывании, с использованием учетной записи, входящей в группу RTCUniversalServerAdmins (или имеющей равнозначные права пользователя) либо назначенной роли CsAdministrator.

2.  Откройте окно браузера и введите URL-адрес администратора, чтобы открыть панель управления Skype для бизнеса Server. 

3.  На панели навигации слева выберите пункт **Настройка сети**, а затем — **политика пропускной способности**.

4.  На странице **политики пропускной способности** выберите профиль политики пропускной способности, который вы хотите удалить.

    > [!NOTE]  
    > За один раз можно удалить сразу несколько профилей. Для этого нажмите клавишу CTRL и, удерживая нажатой клавишу CTRL, выберите несколько профилей. Кроме того, чтобы выбрать все профили, в меню **Правка** выберите команду **выделить все** .

5.  В меню **Правка** выберите команду **Удалить**.
   

    > [!WARNING]  
    > Вы не можете удалить профиль политики пропускной способности, связанный с сетевым сайтом. Прежде чем удалять профиль, необходимо сначала удалить связь с сетевым сайтом.
