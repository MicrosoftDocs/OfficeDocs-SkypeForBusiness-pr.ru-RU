# <a name="managing-quality-of-service-qos-in-skype-for-business-server"></a>Управление качества обслуживания (QoS) в Скайп для Business Server


Качество обслуживания (QoS) — это Сетевая технология используется в некоторых организациях для аудио- и видеоконференций коммуникаций для обеспечения оптимальной для конечных пользователей. QoS чаще всего используется в сетях, где ограничена пропускной способности: с большим числом за для относительно небольшой объем доступной полосы пропускания сетевых пакетов качества обслуживания предоставляет администраторам назначать выше приоритеты пакетов хранение данных звука и видео. Предоставляя эти пакеты более высокий приоритет, аудио- и видеоконференций коммуникаций, скорее всего, для выполнения быстрее и с помехи, чем сети сеансов, включающих передачу файлов, просмотр веб-страниц или резервных копий баз данных. Это связано с поскольку сетевых пакетов, используемый для передачи файлов или резервных копий баз данных присваиваются приоритет «оптимальных затрат».


> [!NOTE]  
> Как правило качества обслуживания применяется только для сеансов связи во внутренней сети. При реализации QoS настройкой серверов и маршрутизаторов для поддержки пакетов пометки; Тем не менее Настройка этих устройств для поддержки пакетов пометки определенным образом. Не удается предполагается, что качества обслуживания будет поддерживаться в Интернете или в других сетях. Даже в том случае, если качество службы поддерживается в других сетях, есть ли гарантии, что QoS будет настроен так же, как настроить службу в сети.

Скайп для Business Server не требует качества обслуживания; Если в настоящее время не используется QoS не является обязательным установки службы перед установкой Скайп для Business Server. При возникновении большое количество потери пакетов в сети рекомендуемый способ устранения этой проблемы заключается в добавлении дополнительных пропускной способности. Если добавление дополнительных пропускной способности невозможно, можно реализовать качества обслуживания.

Скайп для Business Server предоставляет полную поддержку качества обслуживания:, что означает, что организации, которые уже используют QoS можно легко интегрировать Скайп для Business Server в существующей инфраструктурой сети. Для этого необходимо выполнить следующие задачи:

  - [Включение качества обслуживания для устройств, которые не основаны на Windows](enabling-qos-for-devices-that-are-not-based-on-windows.md). По умолчанию качество обслуживания отключено для компьютеров и других устройств (например, iPhones), работающих под управлением ОС, отличных от Windows. Несмотря на то, что Скайп для Business Server можно использовать для включения и отключения качества обслуживания для устройств, обычно нельзя использовать продукт для изменения коды DSCP, используемые этих устройств.

  - [Настройка диапазонов портов и политики качества обслуживания для серверов конференц-связи, приложений и посредника](configuring-port-ranges-for-your-conferencing-application-and-mediation-servers.md). Вам необходимо зарезервировать уникальный набор портов для разных типов пакетов, например аудио и видео. С Скайп для Business Server не включить или отключить качества обслуживания, скажем, установка значения свойства в значение True или False. Вместо этого можно включить качество обслуживания, Настройка диапазонов портов и создание и применение групповой политики. Если вы решите не следует применять QoS можно «отключить» качества обслуживания просто путем удаления соответствующих объектов групповой политики.

  - [Настройка диапазонов портов и политики качества обслуживания для пограничных серверов](configuring-port-ranges-for-your-edge-servers.md). Вы можете использовать на пограничных серверах и остальных серверах одни и те же диапазоны портов, хотя это необязательно. Настройка политики качества обслуживания следует выполнять только для внутреннего сайта пограничного сервера. Вот так как качества обслуживания предназначен для использования в вашей внутренней сети, а не в Интернете.

- [Настройка диапазонов портов и политики качества обслуживания для клиентов в Скайп для Business Server](configuring-port-ranges-for-your-skype-clients.md)  Эти диапазоны портов применяются только к клиентским компьютерам и обычно отличаются от диапазонов портов, настроенных на серверах. Обратите внимание на то, что Скайп для Business Server не поддерживает качества обслуживания для Windows операционных системах, отличных от Windows 10.
