# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
Командлет Get-Ккапплианцестатус предоставляет диагностические данные для устройства Cloud Connector Edition для Skype для бизнеса.
  
Этот командлет применим к Skype для бизнеса Cloud Connector Edition 2,1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Подробное описание

Командлет Get-Ккапплианцестатус, выполняемый на хост-компьютере облачного соединителя, предоставляет диагностические данные для устройства облачного соединителя Skype для бизнеса.
  
## <a name="parameters"></a>Параметры

Нет
  
## <a name="input-types"></a>Типы входных данных

Отсутствуют. Командлет Get-Ккапплианцестатус не поддерживает конвейерные входные данные.
  
## <a name="return-types"></a>Типы возвращаемых данных

Команда возвращает состояние устройства.
