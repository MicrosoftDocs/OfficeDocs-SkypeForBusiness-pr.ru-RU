# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Настройка и управление Skype для бизнеса Cloud Connector Edition
 
> [!Important]
> Cloud Connector Edition завершится 31 июля 2021 г. вместе с Skype для бизнеса Online. После обновления организации Teams, узнайте, как подключить локальной телефонной сети к Teams с помощью прямой [маршрутизации](/MicrosoftTeams/direct-routing-landing-page).

Узнайте, как настроить Skype для бизнеса Cloud Connector Edition, минимальную локальной топологию, чтобы включить интеграцию локальной голосовой инфраструктуры с голосовой телефонная система (Cloud PBX) в Skype для бизнеса Online. 
  
Перед началом работы необходимо просмотреть необходимые условия в [Plan for Skype для бизнеса Cloud Connector Edition.](plan-skype-for-business-cloud-connector-edition.md)
  
> [!IMPORTANT]
> Действия в этом разделе применяются только к cloud Connector Edition 1.4.1 и более поздней версии. Если вы еще не обновлены до cloud Connector Edition 2.1, см. в руб. Обновление до новой версии [облачного соединитетеля.](upgrade-to-a-new-version-of-cloud-connector.md) Вы можете скачать файл установки из [https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller) .
