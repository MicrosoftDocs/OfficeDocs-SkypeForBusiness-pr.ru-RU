# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Настройка и управление Skype для бизнеса Cloud Connector Edition
 
Сведения о настройке Скайп для облачных соединитель Business Edition, топологии минимальной локальных Чтобы включить интеграцию инфраструктуры голосовой связи в локальной с телефонной системой в службах Office 365 (облако УАТС) голосовой связи в Скайп для бизнеса в Интернет. 
  
Прежде чем начать, необходимо проверить необходимого программного обеспечения в [Plan for Скайп облаке соединитель для выпуска для бизнеса](plan-skype-for-business-cloud-connector-edition.md).
  
> [!IMPORTANT]
> Шаги, описываемые в этом разделе, относятся только к выпускам Cloud Connector Edition 1.4.1 и более поздним. Если вы еще не был обновлен для облачных соединителя версии 2.1, видеть [обновление до новой версии облачных соединителя](upgrade-to-a-new-version-of-cloud-connector.md). Можно загрузить файл установки из [https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller).
