# <a name="configure-and-manage-skype-for-business-cloud-connector-edition"></a>Настройка Skype для бизнеса Cloud Connector Edition и управление им
 
Узнайте, как настроить Skype для бизнеса Cloud Connector Edition, минимальную локальную топологию для поддержки интеграции локальной инфраструктуры голосовой связи с телефонной системой в Office 365 (облачная УАТС) в Skype для бизнеса Online. 
  
Прежде чем начать, ознакомьтесь с предварительными условиями, приведенными в разделе [Plan for Skype для бизнеса Cloud Connector Edition](plan-skype-for-business-cloud-connector-edition.md).
  
> [!IMPORTANT]
> Действия, описанные в этом разделе, применимы только к Cloud Connector Edition 1.4.1 и более поздних версий. Если вы еще не выполнили обновление до Cloud Connector Edition 2,1, ознакомьтесь со статьей " [обновление до новой версии Cloud Connector](upgrade-to-a-new-version-of-cloud-connector.md)". Вы можете скачать установочный файл из [https://aka.ms/CloudConnectorInstaller](https://aka.ms/CloudConnectorInstaller).
