# <a name="validate-your-cloud-connector-deployment"></a>Проверка развертывания Cloud Connector
 
После выполнения шагов по развертыванию и настройке Скайп облаке соединитель для выпуска для бизнеса, убедитесь, что можно сделать и принимать звонки.
  
## <a name="validate-the-deployment"></a>Проверка развертывания

1. Войдите в клиент Skype для бизнеса под учетной записью пользователя, которую вы только что настроили.
    
2. Позвоните на номер ТСОП из клиента и убедитесь, что звонок выполнен успешно.
    
3. Используйте телефон или другое устройство, связанное с номером ТСОП, чтобы позвонить в клиент Skype для бизнеса и убедиться в успешности звонка.
    
    > [!NOTE]
    > Такие проверки необходимо выполнять после добавления дополнительных сайтов или преобразования сайтов для включения поддержки высокой доступности.
