# <a name="remove-cccertificationauthorityfile"></a>Remove-CcCertificationAuthorityFile
 
Командлет Remove-CcCertificationAuthorityFile удаляет файл резервной копии службы центра сертификации "&lt;SiteRootDirectory&gt;\CA\SfB системы CCE Root.p12» в папке ЦС совместный доступ в каталоге сайтов для Скайп для соединителя Cloud Business Выпуск. 
  
```
Remove-CcCertificationAuthorityFile
```

## <a name="parameters"></a>Параметры

Нет
  
## <a name="examples"></a>Примеры
<a name="Examples"> </a>

### <a name="example-1"></a>Пример 1

Следующий пример удаляет файл резервной копии службы центра сертификации "&lt;SiteRootDirectory&gt;\CA\SfB системы CCE Root.p12» в папке ЦС совместный доступ в каталоге сайтов:
  
```
Remove-CcCertificationAuthorityFile
```

## <a name="input-types"></a>Типы входных данных
<a name="InputTypes"> </a>

Нет. Командлет Remove-CcCertificationAuthorityFile не принимает входные данные по конвейеру.
