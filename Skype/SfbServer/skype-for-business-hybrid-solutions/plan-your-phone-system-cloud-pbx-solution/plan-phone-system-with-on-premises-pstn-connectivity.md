# <a name="plan-phone-system-with-on-premises-pstn-connectivity-in-skype-for-business-server"></a>Планирование телефонной системы с соединением по локальной ТСОП в Skype для бизнеса Server

> [!Important]
> Skype для бизнеса Online была отменена 31 июля 2021 г. И подключение PSTN между локальной средой независимо от того, Skype для бизнеса Server или Cloud Connector Edition и Skype для бизнеса Online больше не поддерживается.  Узнайте, как подключить сеть локальной телефонии к Teams с помощью [прямого маршрутного маршрутинга.](/MicrosoftTeams/direct-routing-landing-page)

Узнайте о соображениях планирования телефонная система (Cloud PBX) с локальной подключением PSTN.

Этот контент актуален, если Skype для бизнеса Server или Lync Server 2013 развернуты на локальной основе. В других сценариях см. [решения для телефонии Майкрософт.](/microsoftteams/cloud-voice-landing-page)

 телефонная система с локальной подключением PSTN позволяет использовать возможности телефонная система (Cloud PBX) для пользователей. Это может помочь в следующих сценариях:

- Некоторые из ваших пользователей Skype для бизнеса локально, а другие — в Skype для бизнеса Online. Теперь вы можете включить телефонная система и функции для пользователей, Skype для бизнеса в Интернете, но продолжать использовать локальное подключение PSTN.

- У вас есть локальное развертывание, и вы хотите переместить некоторых или всех пользователей в Skype для бизнеса Online, но продолжать использовать локальное подключение PSTN.

    > [!IMPORTANT]
    > Чтобы успешно включить пользователей телефонная система с локальной подключением PSTN, их SIP-адрес должен быть в вашем домене. Использование домена по умолчанию для Microsoft 365 или Office 365 onmicrosoft.com не поддерживается. 

Дополнительные дополнительные телефонная система, включая лицензирование и планы, см. в [Skype для бизнеса.](https://support.office.com/article/PSTN-Calling-plans-for-Skype-for-Business-f47c6a97-bc8b-42e6-b5d4-ce6b41ed1918)

## <a name="benefits-and-planning-considerations"></a>Преимущества и соображения планирования

> [!CAUTION]
> Устройства Lync Телефон Edition должны быть обновлены до минимально необходимого прошивки в локальной среде перед переходом Skype для бизнеса Online.
Если перед обновлением прошивки переместить пользователей из локального в интернет, пользователи не смогут подключиться с помощью своих телефонов. Чтобы устранить эту проблему, необходимо перенаправить пользователей в локальное пространство, чтобы их телефоны обновились до минимального уровня прошивки. НЕ ПЫТАЙТЕСЬ ОБНОВИТЬ ДО МИНИМАЛЬНОГО ПРОШИВКИ ИЛИ ЖЕСТКОГО СБРОСА ТЕЛЕФОНА ДО ПЕРЕМЕЩЕНИЯ ПОЛЬЗОВАТЕЛЯ ОБРАТНО В ЛОКАЛЬНОЙ СРЕДЕ.
Если жесткий сброс выполняется, пока устройство не имеет минимального прошивки, оно по умолчанию будет использовать проверку подлинности ПИН-кода, которая не поддерживается Skype для бизнеса Online. Дополнительные сведения можно получить в интернете по ссылке [Получение телефонов для Skype для бизнеса Online.](https://support.office.com/article/Getting-phones-for-Skype-for-Business-Online-91f2d947-45fc-4fab-bd8b-2e313531c477?ui=en-US&amp;rs=en-US&amp;ad=US)

Развертывание телефонная система с локальной подключением PSTN позволяет перемещать пользователей в облако через Skype для бизнеса Online в своем собственном темпе, сохраняя при этом локальное подключение к PSTN. Если у вас есть PBX, вы продолжаете использовать его для обеспечения подключения PSTN для пользователей, которые перемещаются в облако. После того как пользователь будет перемещен в Skype для бизнеса Online и телефонная система, его устаревший PBX-телефон больше не будет работать, но его номер телефона будет маршрутить в любой из Skype для бизнеса клиентов для ПК или смартфонов, а также Skype для бизнеса-совместимые настольные телефоны. После переноса телефонная система пользователи и устаревшие пользователи PBX могут звонить друг другу нормально, а также делать и принимать вызовы PSTN с помощью обычного номера телефона.

Возможно, у вас есть настраиваемая функция или основная надстройка к устаревшему PBX, например центр обработки вызовов. Если настраиваемая функция в настоящее время недоступна в телефонная система, следует оставить тех пользователей, которые требуют, чтобы эта настраиваемая функция была локальной, с устаревшим PBX, и просто портить пользователей, которым не нужно получать доступ к настраиваемой функции для телефонная система с локальной подключением PSTN.

Список устаревших PBX, которые напрямую связаны с Skype для бизнеса Server 2015 г., см. в этой [ссылке.](../../../SfbPartnerCertification/lync-cert/qualified-ip-pbx-gateway.md) Если вашего PBX нет в этом списке, вы можете использовать пограничный контроллер сеанса для подключения PBX с телефонная система в Skype для бизнеса Online.

### <a name="network-considerations-for-quality-and-performance"></a>Сетевые соображения по качеству и производительности

При развертывании облачной службы, телефонная система с локальной возможностью подключения к PSTN, необходимо помнить следующее. В чисто локальном Skype для бизнеса Server 2015 Корпоративная голосовая связь развертывании вся инфраструктура и клиенты находятся в собственной сети предприятия. Качество и производительность этой сети, которая имеет решающее значение для высококачественных аудио- и видеоматериалов, находятся под непосредственным контролем сотрудников предприятия. В телефонная система связи с локальной сетью PSTN участвуют три сети, за две из которых отвечает клиент, но только одна из которых непосредственно контролируется сотрудниками предприятия:

- **Глобальная сеть доставки** мультимедиа Майкрософт Глобальная облачная сеть и инфраструктура Майкрософт. телефонная система и трафик пересекают эту сеть.

- **Enterprise/Cloud PSTN Interconnect** Это сеть, которая соединяет ваше предприятие с облаком. Это не обязательно совпадает с общим подключением к Интернету.

- **Собственная сеть предприятия** Качество мультимедиа в режиме реального времени в значительной степени зависит от вашей сети: особенно от сети WiFi и от качества интерконнекта, используемого для достижения облака.

> [!NOTE]
> Дополнительные сведения о настройке производительности в Skype для бизнеса Online см. в Skype для бизнеса [Online](https://support.office.com/article/Tune-Skype-for-Business-Online-performance-beec23c2-c5d6-4e84-a8af-e82aefca7802?ui=en-US&amp;rs=en-US&amp;ad=US)performance.
