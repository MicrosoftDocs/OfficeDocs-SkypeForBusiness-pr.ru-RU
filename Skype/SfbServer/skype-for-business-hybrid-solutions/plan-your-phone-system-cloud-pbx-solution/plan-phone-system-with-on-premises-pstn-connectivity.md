# <a name="plan-phone-system-with-on-premises-pstn-connectivity-in-skype-for-business-server"></a>Планирование телефонной системы с соединением по локальной ТСОП в Skype для бизнеса Server

> [!Important]
> Skype для бизнеса Online будет отменен 31 июля 2021 г., после чего служба станет недоступна.  Кроме того, подключение по STN между локальной средой через Skype для бизнеса Server или Cloud Connector Edition и Skype для бизнеса Online больше не будет поддерживаться.  Узнайте, как подключить свою локальной телефонной сети к Teams с помощью [прямой маршрутки.](https://docs.microsoft.com/MicrosoftTeams/direct-routing-landing-page)

Узнайте о планировании телефонной системы (облачной УАП) с локальной связью через STN.

Этот контент актуален, если в локальной сети уже развернуты Skype для бизнеса Server или Lync Server 2013. Другие сценарии см. в [решениях Майкрософт для телефонии.](https://docs.microsoft.com/SkypeForBusiness/hybrid/msft-telephony-solutions)

 Телефонная система с локальной связью через STN позволяет использовать возможности телефонной системы (облачной УАПС) для пользователей. Это может помочь в следующих сценариях:

- У вас есть некоторые пользователи Skype для бизнеса, которые были в локальной сети, а другие — в Skype для бизнеса Online. Теперь вы можете включить возможности и функции телефонной системы для пользователей, которые были в Skype для бизнеса Online, но продолжать использовать локальное подключение к STN.

- У вас есть локальное развертывание, и вы хотите переместить некоторых или всех пользователей в Skype для бизнеса Online, но продолжать использовать локальное подключение к STN.

    > [!IMPORTANT]
    > Чтобы пользователи могли успешно включить телефонную систему с локальной связью через STN, их SIP-адрес должен быть в вашем собственном домене. Использование домена по умолчанию для Microsoft 365 или Office 365 onmicrosoft.com не поддерживается. 

Дополнительные информацию о телефонной системе, в том числе о лицензировании и планах, см. в планах звонков по [STN для Skype для бизнеса.](https://support.office.com/article/PSTN-Calling-plans-for-Skype-for-Business-f47c6a97-bc8b-42e6-b5d4-ce6b41ed1918)

## <a name="benefits-and-planning-considerations"></a>Преимущества и вопросы планирования

> [!CAUTION]
> Перед переходом на Skype для бизнеса Online устройства Lync Phone Edition должны быть обновлены до минимально необходимого уровня в локальной среде.
Если вы переместите пользователей из локальной сети в интернет до обновления микропрограммы, пользователи не смогут подключаться с помощью телефонов. Чтобы устранить эту проблему, пользователям необходимо вернуться в локальное окружение, чтобы обновить телефоны до минимального уровня. Если жесткий сброс выполняется, пока устройство не является как минимум микропрограммой, по умолчанию используется проверка подлинности с помощью ПИН-кода, которая не поддерживается в Skype для бизнеса Online. Дополнительные сведения о получении телефонов [для Skype для бизнеса Online.](https://support.office.com/article/Getting-phones-for-Skype-for-Business-Online-91f2d947-45fc-4fab-bd8b-2e313531c477?ui=en-US&amp;rs=en-US&amp;ad=US)

Развертывая телефонную систему с локальной связью через STN, вы можете переместить пользователей в облако через Skype для бизнеса Online в своем темпе, сохранив при этом возможность подключения к локальной STN. Если у вас есть УАЦ, вы продолжаете использовать ее для обеспечения связи по STN для пользователей, перемещаемой в облако. После того как пользователь будет перемещен в Skype для бизнеса Online и телефонную систему, его устаревший телефон УАТЗ перестает работать, но его номер телефона будет маршрутиться на любой из клиентов Skype для бизнеса для компьютеров или смартфонов, а также на настольные телефоны, совместимые со Skype для бизнеса. После переноса пользователи телефонной системы и устаревшие пользователи УАПС могут звонить друг другу обычным образом, а также делать и принимать звонки по STN с помощью обычного номера телефона.

У вас может быть настраиваемая функция или основная надстройка для устаревшей УАЦ, например центр обработки вызовов. Если настраиваемая функция в настоящее время недоступна в телефонной системе, следует оставить тех пользователей, которым требуется этот пользовательский функции локально с устаревшей УАЦ, и просто перетаскомить пользователей, которым не требуется доступ к настраиваемой функции, в телефонную систему с локальной связью через STN.

Список устаревших УАБ, которые напрямую связаны со Skype для бизнеса Server 2015, см. в поддомене "Инфраструктура, квалифицированная [для Microsoft Lync".](https://docs.microsoft.com/SkypeForBusiness/lync-cert/qualified-ip-pbx-gateway) Если вашей УАПС нет в этом списке, вы можете использовать пограничный контроллер сеансов для подключения УАПС к телефонной системе в Skype для бизнеса Online.

### <a name="network-considerations-for-quality-and-performance"></a>Вопросы качества и производительности сети

При развертывании облачной службы, такой как телефонная система с локальной связью через STN, необходимо иметь в виду следующее. В чисто локальном развертывании Skype для бизнеса Server 2015 Корпоративная голосовая связь все инфраструктуры и клиенты находятся в собственной сети предприятия. Качество и производительность этой сети, которая критически важна для высококачественного звука и видео, находятся под прямым контролем сотрудников предприятия. В телефонной системе с локальной связью через STN задействованы три сети, две из которых клиент несет ответственность, но только одна из которых непосредственно контролируется сотрудниками предприятия:

- **Глобальная сеть доставки мультимедиа** корпорации Майкрософт Глобальная облачная сеть и инфраструктура Майкрософт. Серверы телефонной системы и трафик проходят через эту сеть.

- **Enterprise/Cloud PSTN Interconnect** Это сеть, которая подключает предприятие к облаку. Это необязательно совпадает с общим подключением к Интернету.

- **Собственная сеть предприятия** Качество мультимедиа в режиме реального времени во многом зависит от вашей сети: особенно от сети WiFi и от качества соединения, используемого для подключения к облаку.

> [!NOTE]
> Дополнительные сведения о настройке производительности в Skype для бизнеса Online см. в [подстройке производительности Skype для бизнеса Online.](https://support.office.com/article/Tune-Skype-for-Business-Online-performance-beec23c2-c5d6-4e84-a8af-e82aefca7802?ui=en-US&amp;rs=en-US&amp;ad=US)
