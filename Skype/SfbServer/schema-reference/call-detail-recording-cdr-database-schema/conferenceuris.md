# <a name="conferenceuris-table-in-skype-for-business-server-2015"></a>Таблица Конференцеурис в Skype для бизнеса Server 2015
 
Таблица Конференеурис является вспомогательной таблицей, в которой хранится список различных URI конференций, участвующих в сеансах конференц-связи, записанных в базу данных. Каждая запись в таблице представляет один URI конференции.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**некступдатетс** <br/> |datetime  <br/> |Primary  <br/> |Метка времени, используемая в качестве внутренней.  <br/> |
|**конференцеуриид** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот URI конференции.  <br/> |
|**ConferenceUri** <br/> |nvarchar (450)  <br/> ||Универсальный код ресурса (URI) Конференции.  <br/> |
|**Счет** <br/> |целое  <br/> ||Контрольная сумма для Конференцеури. Используется для увеличения скорости поиска в базе данных.  <br/> |
|**уритипеид** <br/> |целое  <br/> |Другом  <br/> |Тип URI, например conf: чат для Конференции с помощью мгновенных сообщений или conf: аудио-видео для голосовой и видеоконференции. Более подробную информацию вы увидите в таблице [уритипес Table](uritypes.md) . <br/> |
