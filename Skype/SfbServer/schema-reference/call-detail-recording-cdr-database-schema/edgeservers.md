# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>Таблица EdgeServers в Skype для бизнеса Server 2015
 
Таблица EdgeServers является вспомогательной. В каждой записи хранится информация об одном сервере, который участвует в вызовах с записями в базе данных.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Details**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот сервер.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Имя сервера.  <br/> |
