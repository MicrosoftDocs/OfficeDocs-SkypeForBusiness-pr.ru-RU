# <a name="pools-table"></a>Таблица Pools
 
В таблице пулы представляет собой вспомогательную таблицу, в которой хранятся сведения о различных пула. Каждая запись в таблице представляет один пул.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот пул.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |Полное доменное имя пула.  <br/> |
