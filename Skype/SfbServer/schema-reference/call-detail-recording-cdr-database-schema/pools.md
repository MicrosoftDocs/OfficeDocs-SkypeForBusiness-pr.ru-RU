# <a name="pools-table"></a>Таблица Pools
 
Таблица Pools — это вспомогательная таблица, в которой хранится информация о различных пулах. Каждая запись в таблице представляет один пул.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Пулид** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот пул.  <br/> |
|**Пулфкдн** <br/> |nvarchar(256)  <br/> | <br/> |Полное доменное имя пула.  <br/> |
