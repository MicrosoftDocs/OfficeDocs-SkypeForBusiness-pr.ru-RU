# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Таблица hardwareversions в Скайп для Business Server 2015
 
Таблица hardwareversions представляет собой вспомогательную таблицу. Каждая запись сохранение информации о версии оборудования одно устройство.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий версию оборудования.  <br/> |
|**Версия** <br/> |nvarchar(256)  <br/> | <br/> |Версия оборудования.  <br/> |
