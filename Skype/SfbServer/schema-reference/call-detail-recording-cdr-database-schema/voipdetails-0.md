# <a name="voipdetails-table"></a>Таблица VoipDetails
 
Каждая запись представляет вызов на стороне 1 2, в котором как минимум один пользователь является пользователем VoIP.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Сессионидтиме** <br/> |datetime  <br/> |Primary  <br/> |Время запроса сеанса. Используется в сочетании с **сессионидсек** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**Сессионидсек** <br/> |целое  <br/> |Primary  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации сеанса. Используется в сочетании с **сессионидтиме** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**Фромнумберид** <br/> |целое  <br/> |Другом  <br/> |**Фонеид** вызывающего абонента. Для получения [](phones.md) дополнительных сведений ознакомьтесь с таблицей phones. Если NOT NULL и **фромгатевайид** не NULL, вызывающий абонент являлся пользователем КТСОП. <br/> |
|**Коннектеднумберид** <br/> |целое  <br/> |Другом  <br/> |**Фонеид** получателя звонка. Для получения [](phones.md) дополнительных сведений ознакомьтесь с таблицей phones. Если NOT NULL и **тогатевайид** не NULL, получатель звонка был пользователем КТСОП. <br/> |
|**Фроммедиатионсерверид** <br/> |целое  <br/> |Другом  <br/> |Сервер, на котором поступил звонок. Для получения дополнительных сведений ознакомьтесь с [таблицей медиатионсерверс](mediationservers.md) . <br/> |
|**Томедиатионсерверид** <br/> |целое  <br/> |Другом  <br/> |Будет вызываем сервер для устранения проблем. Для получения дополнительных сведений ознакомьтесь с [таблицей медиатионсерверс](mediationservers.md) . <br/> |
|**Фромгатевайид** <br/> |целое  <br/> |Другом  <br/> |Шлюз, из которого поступил звонок. Более подробную информацию вы увидите [в таблице шлюзов в Skype для бизнеса Server 2015](gateways.md) . <br/> |
|**Тогатевайид** <br/> |целое  <br/> |Другом  <br/> |Шлюз, на который проходит звонок. Более подробную информацию вы увидите [в таблице шлюзов в Skype для бизнеса Server 2015](gateways.md) . <br/> |
|**Дисконнектедбюриид** <br/> |целое  <br/> |Другом  <br/> |Универсальный код ресурса (URI) пользователя, который отключил звонок, если пользователь имеет URI. Дополнительные сведения [можно](users.md) найти в таблице Users. <br/> |
|**Дисконнектедбифонеид** <br/> |целое  <br/> |Другом  <br/> |Идентификационный номер телефона, который отключил звонок, был отключен от телефона. Для получения [](phones.md) дополнительных сведений ознакомьтесь с таблицей phones. <br/> |
|**LastModifiedTime** <br/> |Датой  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было представлено в Skype для бизнеса Server 2015.  <br/> |
