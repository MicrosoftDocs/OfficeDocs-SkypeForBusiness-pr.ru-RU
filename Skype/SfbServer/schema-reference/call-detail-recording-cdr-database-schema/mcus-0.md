# <a name="mcus-view"></a>Представление Mcus
 
Представление Mcus сохранение информации о MCUs, которые участвовали в сеансах конференций. В этом представлении была введена в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|**McuId** <br/> |целое  <br/> |Уникальный номер, идентифицирующий MCU.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |URI блока MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |Тип MCU URI. В [таблице UriTypes](uritypes.md) для получения дополнительных сведений см. <br/> |
