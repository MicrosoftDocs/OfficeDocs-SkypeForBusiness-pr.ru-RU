# <a name="focusjoinsandleaves-view"></a>Представление FocusJoinsAndLeaves
 
Представление FocusJoinsAndLeaves сохранение информации о соединения и оставьте сведения для одной конференции. Каждой конференции представленного в этом представлении записи каждый раз пользователь присоединился к и покинул данную конференцию. В этом представлении была введена в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|**SessionIdTime** <br/> |datetime  <br/> |Время создания экземпляра конференции. Используется совместно с SessionIdSeq для уникальной идентификации экземпляра конференции. В разделе [Таблица конференций в Скайп для Business Server 2015](conferences.md) для получения дополнительных сведений. <br/> |
|**SessionIdSeq** <br/> |целое  <br/> |Номер идентификатора для определения экземпляра в конференции. Используется в сочетании с SessionIdTime для уникальной идентификации экземпляра конференции. В разделе [Таблица конференций в Скайп для Business Server 2015](conferences.md) для получения дополнительных сведений. <br/> |
|**UserUri** <br/> |nvarchar(450)  <br/> |URI пользователя, сведения о подключении или отключении от конференции были записаны.  <br/> |
|**UserUriType** <br/> |nvarchar(256)  <br/> |Тип URI пользователя, сведения о подключении или отключении от конференции были записаны. В [таблице UriTypes](uritypes.md) для получения дополнительных сведений см. <br/> |
|**UserTenant** <br/> |nvarchar(256)  <br/> |Клиент пользователя, сведения о подключении или отключении от конференции были записаны. В [таблице клиентов](tenants.md) для получения дополнительных сведений см. <br/> |
|**UserEndpointId** <br/> |uniqueidentifier  <br/> |Уникальный идентификатор пользователя, чьи сведения о подключении или отключении от конференции были записаны.  <br/> |
|**UserClientVersion** <br/> |nvarchar(256)  <br/> |Версия клиента пользователя, сведения о подключении или отключении от конференции были записаны.  <br/> |
|**UserClientType** <br/> |целое  <br/> |Клиент пользователя, сведения о подключении или отключении от конференции были записаны. [Таблица useragentdef](useragentdef.md) Дополнительные сведения см. <br/> |
|**UserClientCategory** <br/> |nvarchar(64)  <br/> |Имя категории клиента, используемого пользователем, сведения о подключении или отключении от конференции были записаны.  <br/> |
|**FocusUserInstance** <br/> |целое  <br/> ||
|**IsuserInternal** <br/> |бит  <br/> |Разряд, указывающий, является ли пользователь внутренним или нет.  <br/> |
|**DialogSessionIdTime** <br/> |datetime  <br/> |Время запроса сеанса. Используется совместно с SessionIdSeq для уникальной идентификации сеанса. В разделе [диалоговых окон в таблице в Скайп для Business Server 2015](dialogs.md) для получения дополнительных сведений. <br/> |
|**DialogSessionIdSeq** <br/> |целое  <br/> |Если пользователь вошел в систему на нескольких компьютерах или устройствах в то же время, параметр UserInstance используется для уникальной идентификации комбинации пользователя и устройства.  <br/> |
|**DialogId** <br/> |varchar(775)  <br/> |Диалоговое окно идентификатор сеанса по протоколу SIP. Имеет формат: диалоговое окно, из тега; для тега.  <br/> |
|**UserJoinTime** <br/> |datetime  <br/> |Время, когда пользователь присоединился к конференции.  <br/> |
|**UserLeaveTime** <br/> |datetime  <br/> |Время, которое пользователь покинул конференцию.  <br/> |
|**UserRole** <br/> |nvarchar(256)  <br/> |Роль пользователя в конференции, например выступающий или участник.  <br/> |
