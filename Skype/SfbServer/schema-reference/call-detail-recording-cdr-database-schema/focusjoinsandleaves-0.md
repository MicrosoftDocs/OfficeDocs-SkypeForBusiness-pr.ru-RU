# <a name="focusjoinsandleaves-view"></a>Представление Фокусжоинсандлеавес
 
В представлении Фокусжоинсандлеавес хранятся сведения о присоединениях и оставлении данных для одной конференции. Каждое собрание представлено в этом представлении записью, которая записывается каждый раз, когда пользователь присоединяется к Конференции, и оставляет ее. Это представление было представлено в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|**сессионидтиме** <br/> |datetime  <br/> |Время экземпляра Конференции. Используется в сочетании с Сессионидсек для уникальной идентификации экземпляра Конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей "Конференции" в Skype для бизнеса Server 2015](conferences.md) . <br/> |
|**сессионидсек** <br/> |целое  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации экземпляра Конференции. Используется в сочетании с Сессионидтиме для уникальной идентификации экземпляра Конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей "Конференции" в Skype для бизнеса Server 2015](conferences.md) . <br/> |
|**усерури** <br/> |nvarchar (450)  <br/> |Универсальный код ресурса (URI), на который были собраны сведения о присоединении или закрытии Конференции.  <br/> |
|**усеруритипе** <br/> |nvarchar(256)  <br/> |Тип универсального кода ресурса (URI), на который были собраны сведения о приходе или закрытии Конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
|**усертенант** <br/> |nvarchar(256)  <br/> |Клиент для пользователя, которому были собраны сведения о присоединении к Конференции или выходе из нее. Дополнительные сведения см. в [таблице "клиенты](tenants.md) ". <br/> |
|**усерендпоинтид** <br/> |идентификатора  <br/> |Уникальный идентификатор пользователя, для которого была собрана информация о присоединении к Конференции или выходе из нее.  <br/> |
|**усерклиентверсион** <br/> |nvarchar(256)  <br/> |Версия клиента, используемая пользователем, для которого была собрана информация о присоединении к Конференции или выходе из нее.  <br/> |
|**усерклиенттипе** <br/> |целое  <br/> |Клиент, используемый пользователем, для которого были собраны сведения о присоединении к Конференции или выходе из нее. Для получения дополнительных сведений ознакомьтесь с [таблицей усеражентдеф](useragentdef.md) . <br/> |
|**усерклиенткатегори** <br/> |nvarchar (64)  <br/> |Имя категории клиента, используемой пользователем, для которого были собраны сведения о приходе или закрытии Конференции.  <br/> |
|**фокусусеринстанце** <br/> |целое  <br/> ||
|**исусеринтернал** <br/> |бит  <br/> |Бит, обозначающий, является ли пользователь внутренним.  <br/> |
|**диалогсессионидтиме** <br/> |datetime  <br/> |Время запроса сеанса. Используется в сочетании с Сессионидсек для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**диалогсессионидсек** <br/> |целое  <br/> |Если пользователь входит в систему на нескольких компьютерах или устройствах одновременно, Усеринстанце используется для уникальной идентификации комбинации пользователей и устройств.  <br/> |
|**диалогид** <br/> |varchar (775)  <br/> |ИДЕНТИФИКАТОР диалогового окна SIP для сеанса. Формат: диалоговое окно; тег.  <br/> |
|**усержоинтиме** <br/> |datetime  <br/> |Время, в которое пользователь присоединился к Конференции.  <br/> |
|**усерлеаветиме** <br/> |datetime  <br/> |Время, когда пользователь оставил Конференцию.  <br/> |
|**UserRole** <br/> |nvarchar(256)  <br/> |Роль пользователя на Конференции, например докладчика или участника.  <br/> |
