# <a name="conferenceuris-view"></a>Представление Конференцеурис
 
В представлении Конфернцеурис хранятся сведения о URI, которые участвовали в сеансах конференц-связи. Это представление было представлено в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|Конференцеуриид  <br/> |целое  <br/> |Уникальный номер URI конференции.  <br/> |
|ConferenceUri  <br/> |nvarchar (450)  <br/> |Универсальный код ресурса (URI) Конференции.  <br/> |
|Конференцеуритипе  <br/> |nvarchar(256)  <br/> |Тип URI конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
