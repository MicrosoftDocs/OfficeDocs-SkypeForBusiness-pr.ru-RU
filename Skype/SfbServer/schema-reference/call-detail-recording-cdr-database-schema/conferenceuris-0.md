# <a name="conferenceuris-view"></a>Представление ConferenceUris
 
В представлении ConfernceUris хранятся сведения об идентификаторах URI, которые участвовали в сеансах конференц-связи. Это представление впервые было введено в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Details**|
|:-----|:-----|:-----|
|ConferenceUriId  <br/> |int  <br/> |Уникальный номер, идентифицирующий URI конференции.  <br/> |
|ConferenceUri  <br/> |nvarchar(450)  <br/> |Идентификатор URI конференции.  <br/> |
|ConferenceUriType  <br/> |nvarchar(256)  <br/> |Тип идентификатора URI конференции. Дополнительные сведения см. в таблице [UriTypes.](uritypes.md) <br/> |
