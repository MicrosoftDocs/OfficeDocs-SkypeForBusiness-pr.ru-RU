# <a name="list-of-cdr-tables-in-skype-for-business-server-2015"></a>Список таблиц CDR в Skype для бизнеса Server 2015
 
Схема базы данных для записи сведений о звонке (CDR) состоит из следующих таблиц. 
  
## <a name="static-tables"></a>Статические таблицы

|**Таблица**|**Описание**|
|:-----|:-----|
|[Таблица Каллприоритиес в Skype для бизнеса Server 2015](callpriorities.md) <br/> |Хранит список возможных приоритетов звонков, например, экстренных, срочных, обычных, несрочных и т. д.  <br/> |
|[Таблица Конференцежоинтимесрешолдс в Skype для бизнеса Server 2015](conferencejointimethresholds.md) <br/> |Хранит границы классификации, используемые в сводном отчете "время присоединения к Конференции".  <br/> |
|[Таблица Дерегистертипе в Skype для бизнеса Server 2015](deregistertype.md) <br/> |Хранит список возможных причин, по которым пользователь может отменить регистрацию, например "клиент инициировал", "срок действия регистрации истек", "сбой клиента" и другие сведения.  <br/> |
|[Таблица MediaList](medialist.md) <br/> |Хранит список типов мультимедиа, которые могут создавать записи в базе данных (например, мгновенные сообщения, звук, видео и передачу файлов).  <br/> |
|[Таблица Пуржесеттингс](purgesettings.md) <br/> |Хранит сведения о том, будут ли (и когда) устаревшие записи о вызовах автоматически удалены из базы данных CDR.  <br/> |
|[Таблица Roles](roles.md) <br/> |Хранит список возможных ролей конференции (например, участника и выступающего).  <br/> |
|[Таблица Сипреспонсеметадата](sipresponsemetadata.md) <br/> |Содержит список кодов ответов SIP и классификацию и определение каждого из этих кодов.  <br/> |

## <a name="table-for-voip-call-details"></a>Таблица сведений о звонке VoIP

|**Таблица**|**Описание**|
|:-----|:-----|
|[Таблица VoipDetails](voipdetails-0.md) <br/> |Для каждого из двух сторон VoIP/КТСОП звоните сведения о звонке, например номер телефона телефона VoIP, используемый шлюз, и какая сторона отключилась. Ссылка на [таблицу сессиондетаилс](sessiondetails.md) для значений "начало/окончание" и "код ответа" для вызова. <br/> |
   
> [!NOTE]
> Если один из участников звонка является пользователем VoIP или на него вовлечен сервер-посредник, в этой таблице будет создана запись. Сведения о звонках по протоколу VoIP и VoIP, не посвященных коммутируемой телефонной сети общего пользования (PSTN), записываются в [таблицу сессиондетаилс](sessiondetails.md).
