# <a name="focusjoinsandleaves-table-in-skype-for-business-server-2015"></a>Таблица FocusJoinsAndLeaves в Skype для бизнеса Server 2015
 
Каждая запись в этой таблице содержит сведения CDR о подмене и отступе одного пользователя для одной конференции. Каждая конференция представлена в этой таблице по одной записи для каждого времени, когда пользователь присоединяется к конференции и покидает ее.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Details**|
|:-----|:-----|:-----|:-----|
|**SessionIdTime** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время экземпляра конференции. Используется в сочетании с **SessionIdSeq** для уникальной идентификации экземпляра конференции. Дополнительные сведения см. в таблице ["Конференции" в Skype для бизнеса Server 2015.](conferences.md) <br/> |
|**SessionIdSeq** <br/> |int  <br/> |Основной, Внешний  <br/> |Идентификатор для определения экземпляра конференции. Используется в сочетании с **SessionIdTime** для уникальной идентификации экземпляра конференции. Дополнительные сведения см. в таблице ["Конференции" в Skype для бизнеса Server 2015.](conferences.md) <br/> |
|**DialogSessionIdTime** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время запроса сеанса. В сочетании с параметром **SessionIdSeq** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Dialogs в Skype для бизнеса Server 2015.](dialogs.md) <br/> |
|**DialogSessionIdSeq** <br/> |int  <br/> |Основной, внешний  <br/> |Идентификатор для идентификации сеанса. В сочетании с параметром **SessionIdTime** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Dialogs в Skype для бизнеса Server 2015.](dialogs.md) <br/> |
|**UserId** <br/> |int  <br/> |Внешняя  <br/> |Уникальный номер, идентифицирующий этого пользователя, на который ссылается [таблица Users.](users.md)  <br/> |
|**FocusUserInstance** <br/> |int  <br/> ||Если пользователь одновременно вошел в систему на нескольких компьютерах или устройствах, **UserInstance** используется для уникальной идентификации комбинации пользователя или устройства. <br/> |
|**IsUserInternal** <br/> |bit  <br/> | <br/> |Вне зависимости от того, вошел ли пользователь из внутренней сети или нет.  <br/> |
|**UserRole** <br/> |int  <br/> | <br/> |Роль этого пользователя в конференции, например presenter или Attendee.  <br/> |
|**UserJoinTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь присоединяется к конференции.  <br/> |
|**UserLeaveTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь покидает конференцию.  <br/> |
|**ClientVerId** <br/> |int  <br/> |Внешняя  <br/> |Версия клиентского программного обеспечения пользователя, ссылка на таблицу [ClientVersions в Skype для бизнеса Server 2015.](clientversions.md)  <br/> |
|**UserEndpointId** <br/> |uniqueIdentifier  <br/> ||Глобальный уникальный идентификатор (GUID) конечной точки, используемой в конференции.  <br/> Это поле было впервые введено в Microsoft Lync Server 2013.  <br/> |
|**LastModifiedTime** <br/> |Дата и время  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было впервые введено в Skype для бизнеса Server 2015.  <br/> |
