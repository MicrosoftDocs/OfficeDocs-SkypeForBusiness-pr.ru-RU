# <a name="errorreport-table-in-skype-for-business-server-2015"></a>Таблица Ерроррепорт в Skype для бизнеса Server 2015
 
В таблице Ерроррепорт хранятся сведения о возникших ошибках. Каждая запись содержит один экземпляр ошибки. Эти ошибки регистрируются агентом CDR, который запущен на сервере переднего плана или отправлен клиентом.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**еррортиме** <br/> |datetime  <br/> |Primary  <br/> |Дата и время, когда возникла ошибка.  <br/> |
|**ерроррепортсек** <br/> |целое  <br/> |Primary  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации отчета об ошибке. Используется в сочетании с **еррортиме** для уникальной идентификации отчета об ошибке. <br/> |
|**еррорид** <br/> |целое  <br/> |Другом  <br/> |Уникальный идентификатор типа ошибки. Для получения дополнительных сведений ознакомьтесь с [таблицей еррордеф в Skype для бизнеса Server 2015](errordef.md) . <br/> |
|**фромусерид** <br/> |целое  <br/> |Другом  <br/> |Пользователь, который поступил на этот запрос, вызвавший ошибку. Дополнительные сведения [можно найти в таблице Users](users.md) . <br/> |
|**таусерид** <br/> |целое  <br/> |Другом  <br/> |Конечный пользователь для запроса, вызвавшего ошибку. Дополнительные сведения [можно найти в таблице Users](users.md) . <br/> |
|**конференцеуриид** <br/> |целое  <br/> |Другом  <br/> |Универсальный код ресурса (URI) для Конференции, связанный с ошибкой. Для получения дополнительных сведений ознакомьтесь с [таблицей конференцеурис в Skype для бизнеса Server 2015](conferenceuris.md) . Как правило, если Конференцеуриид не имеет значение null, то либо Фромусерид, либо Таусерид будет NULL. <br/> |
|**сессионидтиме** <br/> |datetime  <br/> |Другом  <br/> |Используется в сочетании с **сессионидсек** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**сессионидсек** <br/> |целое  <br/> |Другом  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации сеанса. Используется в сочетании с **сессионидтиме** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**Источника** <br/> |целое  <br/> |Другом  <br/> |Сервер, отправивший отчет об ошибке (при отправке отчета из серверного компонента). Для получения дополнительных сведений ознакомьтесь с [таблицей Servers](servers.md) . <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
|**ApplicationId** <br/> |целое  <br/> |Другом  <br/> |Сервер, отправивший отчет об ошибке (при отправке отчета из серверного компонента). Дополнительные сведения приведены [в таблице "приложение" в Skype для бизнеса Server 2015](application.md) . <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
|**мсдиагхеадер** <br/> |изображение  <br/> | <br/> |Дополнительные сведения об ошибке.  <br/> Эти данные можно преобразовать в текстовый формат, используя следующий синтаксис:  <br/>  `cast(cast(Detail as varbinary(max)) as varchar(max))` <br/> |
|**клиентверсионид** <br/> |целое  <br/> |Другом  <br/> |Клиентская версия конечной точки, отправляющей отчет об ошибке. Для получения дополнительных сведений ознакомьтесь с [таблицей клиентверсионс в Skype для бизнеса Server 2015](clientversions.md) . <br/> |
|**искаптуредбисервер** <br/> |бит  <br/> ||Отчет об ошибке, записанный агентом CDR, который запущен на сервере переднего плана или отправляется клиентом.  <br/> |
|**Пометка** <br/> |smallint  <br/> ||Зарезервировано для будущего использования.  <br/> |
|**телеметрид** <br/> |Идентификатора  <br/> ||Уникальный идентификатор, соответствующий сведениям о времени соединения для различных компонентов, участвующих в Конференции.  <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
|**сессионсетуптиме** <br/> |целое  <br/> ||Время (в миллисекундах), требуемое конкретным компонентом для присоединения к Конференции.  <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
|**серверид** <br/> |целое  <br/> |Другом  <br/> |Представляет полное доменное имя сервера, который сгенерировал отчет об ошибке.  <br/> |
|**пулид** <br/> |целое  <br/> |Другом  <br/> |Представляет полное доменное имя пула, в котором был создан отчет об ошибке.  <br/> |
|**LastModifiedTime** <br/> |Датой  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было представлено в Skype для бизнеса Server 2015.  <br/> |
