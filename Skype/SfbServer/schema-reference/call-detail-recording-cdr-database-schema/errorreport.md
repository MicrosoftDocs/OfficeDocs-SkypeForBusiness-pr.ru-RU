# <a name="errorreport-table-in-skype-for-business-server-2015"></a>Таблица ErrorReport в Скайп для Business Server 2015
 
Таблица ErrorReport сохранение информации о возникших ошибках. Каждая запись — один ошибку. Ошибки, которые отслеживались либо агентом CDR, работающим на сервере переднего плана или передаче от клиента.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**ErrorTime** <br/> |datetime  <br/> |Primary  <br/> |Дата и время возникновения ошибки.  <br/> |
|**ErrorReportSeq** <br/> |целое  <br/> |Primary  <br/> |Номер идентификатора для идентификации отчет об ошибках. Используется в сочетании с **ErrorTime** для уникальной идентификации отчет об ошибке. <br/> |
|**Код ошибки** <br/> |целое  <br/> |Внешний  <br/> |Уникальный идентификатор типа ошибки. [Таблица errordef в Скайп для Business Server 2015](errordef.md) для получения дополнительных сведений см. <br/> |
|**FromUserId** <br/> |целое  <br/> |Внешний  <br/> |Пользователя, инициировавшего запрос, вызвавшей ошибку. В [таблице пользователей](users.md) для получения дополнительных сведений см. <br/> |
|**ToUserId** <br/> |целое  <br/> |Внешний  <br/> |Конечный пользователь для запроса, вызвавшей ошибку. В [таблице пользователей](users.md) для получения дополнительных сведений см. <br/> |
|**ConferenceUriId** <br/> |целое  <br/> |Внешний  <br/> |URI конференции связанных с этой ошибкой. [Таблица ConferenceUris в Скайп для Business Server 2015](conferenceuris.md) для получения дополнительных сведений см. Как правило если ConferenceUriId не равно null, затем FromUserId или ToUserId будет null. <br/> |
|**SessionIdTime** <br/> |datetime  <br/> |Внешний  <br/> |Используется совместно с **SessionIdSeq** для уникальной идентификации сеанса. В разделе [диалоговых окон в таблице в Скайп для Business Server 2015](dialogs.md) для получения дополнительных сведений. <br/> |
|**SessionIdSeq** <br/> |целое  <br/> |Внешний  <br/> |Номер идентификатора для идентификации сеанса. Используется в сочетании с **SessionIdTime** для уникальной идентификации сеанса. В разделе [диалоговых окон в таблице в Скайп для Business Server 2015](dialogs.md) для получения дополнительных сведений. <br/> |
|**SourceId** <br/> |целое  <br/> |Внешний  <br/> |Сервер, который отправляется отчет об ошибках (Если отчет отправляется с серверным компонентом). В [таблице серверы](servers.md) для получения дополнительных сведений см. <br/> В этом поле было представлено в Microsoft Lync Server 2013.  <br/> |
|**Идентификатор приложения** <br/> |целое  <br/> |Внешний  <br/> |Сервер, который отправляется отчет об ошибках (Если отчет отправляется с серверным компонентом). В [таблице приложения в Скайп для Business Server 2015](application.md) для получения дополнительных сведений см. <br/> В этом поле было представлено в Microsoft Lync Server 2013.  <br/> |
|**MsDiagHeader** <br/> |изображение  <br/> | <br/> |Дополнительные сведения об ошибке.  <br/> Эти данные можно преобразовать в текстовый формат с помощью следующего синтаксиса:  <br/>  `cast(cast(Detail as varbinary(max)) as varchar(max))` <br/> |
|**ClientVersionId** <br/> |целое  <br/> |Внешний  <br/> |Версия клиента конечная точка, отправляющая отчет об ошибках. [Таблица ClientVersions в Скайп для Business Server 2015](clientversions.md) для получения дополнительных сведений см. <br/> |
|**IsCapturedByServer** <br/> |бит  <br/> ||Отчет об ошибках отслеживались агентом CDR, работающим на сервере переднего плана, или отправлен клиентом.  <br/> |
|**Флаг** <br/> |smallint  <br/> ||Зарезервировано для будущего использования.  <br/> |
|**TelemetryId** <br/> |uniqueIdentifier  <br/> ||Уникальный идентификатор времени присоединения для разных компонентов, задействованных в конференции.  <br/> В этом поле было представлено в Microsoft Lync Server 2013.  <br/> |
|**SessionSetupTime** <br/> |целое  <br/> ||Время (в миллисекундах), требуемое определенному компоненту для присоединения к конференции.  <br/> В этом поле было представлено в Microsoft Lync Server 2013.  <br/> |
|**Который** <br/> |целое  <br/> |Внешний  <br/> |Представляет полное доменное имя сервера, который создал отчет об ошибках.  <br/> |
|**PoolId** <br/> |целое  <br/> |Внешний  <br/> |Представляет полное доменное имя пула, в котором был создан отчет об ошибках.  <br/> |
|**LastModifiedTime** <br/> |Даты и времени  <br/> ||Для внутреннего использования службой мониторинга.  <br/> В этом поле было представлено в Скайп для Business Server 2015.  <br/> |
