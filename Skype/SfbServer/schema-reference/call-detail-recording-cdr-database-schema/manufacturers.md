# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Таблица Manufacturers в Скайп для Business Server 2015
 
Таблица Manufacturers представляет собой вспомогательную таблицу. Каждая запись сохранение информации о производителя устройства (стационарный телефон).
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий данного производителя.  <br/> |
|**Производитель** <br/> |nvarchar(256)  <br/> | <br/> |Имя производителя.  <br/> |
