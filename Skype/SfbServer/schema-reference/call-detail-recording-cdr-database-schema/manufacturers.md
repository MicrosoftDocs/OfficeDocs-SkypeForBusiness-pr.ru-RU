# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Таблица производителей в Skype для бизнеса Server 2015 г.
 
Таблица Производителей — это вспомогательная таблица. В каждой записи хранится информация об одном изготовителе устройства (настольный телефон).
  
|**Столбец**|**Тип данных**|**Key/Index**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этого производителя.  <br/> |
|**Производитель** <br/> |nvarchar (256)  <br/> | <br/> |Имя производителя.  <br/> |
