# <a name="mediationservers-table"></a>Таблица MediationServers
 
Таблица Медиатионсерверс является вспомогательной таблицей. Каждая запись содержит сведения об одном сервере, который участвует в звонках, которые содержат записи в базе данных.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Медиатионсерверид** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, показывающий этот сервер исправлений.  <br/> |
|**Медиатионсервер** <br/> |nvarchar(256)  <br/> | <br/> |Имя сервера исправлений.  <br/> |
