# <a name="mediationservers-table"></a>Таблица MediationServers
 
Таблица MediationServers — это вспомогательная таблица. В каждой записи хранится информация об одном сервере-посреднике, который участвует в вызовах с записями в базе данных.
  
|**Столбец**|**Тип данных**|**Key/Index**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот сервер-посредник.  <br/> |
|**MediationServer** <br/> |nvarchar (256)  <br/> | <br/> |Имя сервера-посредника.  <br/> |
