# <a name="mediationservers-table"></a>Таблица MediationServers
 
Таблица mediationservers представляет собой вспомогательную таблицу. Каждая запись сохранение информации о один сервер-посредник, задействованных в вызовах с записями в базе данных.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот сервер-посредник.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |Имя сервера-посредника.  <br/> |
