|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|

|**Серверид** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот сервер.  <br/> |
|**Серверфкдн** <br/> |nvarchar(256)  <br/> | <br/> |Полное доменное имя сервера.  <br/> |
