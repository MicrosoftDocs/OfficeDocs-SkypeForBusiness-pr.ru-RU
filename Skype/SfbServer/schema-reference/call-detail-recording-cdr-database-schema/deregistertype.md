# <a name="deregistertype-table-in-skype-for-business-server-2015"></a>Таблица Дерегистертипе в Skype для бизнеса Server 2015
 
Таблица Дерегистертипе — это статическая таблица, в которой хранится список возможных типов отмены регистров пользователей, например "инициировано клиентом", "регистрация просрочено" или "клиент перестает отвечать на запросы".
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Дерегистертипеид** <br/> |tinyint  <br/> |Primary  <br/> ||
|**Дерегистерреасон** <br/> |nvarchar(256)  <br/> || Допустимые значения: <br/>  0 — неизвестно <br/>  1 — клиент инициировал отмену регистрации <br/>  2 — срок действия регистрации истек <br/>  3-Клиент аварийно завершил работу <br/>  4 — изменились атрибуты пользователя <br/>  5 — основной регистратор изменился <br/>  6 — устаревший клиент в режиме выживания <br/> |
