# <a name="mcus-table-in-skype-for-business-server-2015"></a>Таблица мкус в Skype для бизнеса Server 2015
 
Таблица Мкус является вспомогательной таблицей. Каждая запись содержит сведения о одной службе конференц-связи. К ним относятся служба конференций обмена мгновенными сообщениями и служба конференц-связи с телефонным подключением (которая запускается как процессы на серверах переднего плана), а также службы "веб-конференции" и "Конференция". 
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**мкуид** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий этот сервер конференц-связи.  <br/> |
|**мкуури** <br/> |nvarchar (450)  <br/> | <br/> | <br/> |
|**мкутипеид** <br/> |ининт  <br/> | Другом <br/> |Тип сервера конференций, например conf: Chat (для мгновенных сообщений) или conf: аудио-видео. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
