# <a name="tenants-table"></a>Таблица Tenants
 
Таблица клиентов является вспомогательной таблицей, в которой хранится список различных клиентов. Каждая запись в таблице представляет одного клиента.
  
> [!NOTE]
> В локальном развертывании CDR использует встроенный ИД клиента, чтобы показать различные типы проверки подлинности, такие как общедоступное подключение для обмена мгновенными сообщениями, федеративное подключение или анонимное подключение. 
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Details**|
|:-----|:-----|:-----|:-----|
|**TenantId** <br/> |int  <br/> |Primary  <br/> |Уникальное число, определяющее ИД клиента.  <br/> |
|**TenantKey** <br/> |nvarchar(256)  <br/> || Допустимые значения: <br/>  00000000-0000-0000-0000-00000000000000000 - Enterprise <br/>  00000000-0000-0000-0000-000000000001 федера <br/>  00000000-0000-0000-0000-0000000000002 — анонимный <br/>  00000000-0000-0000-0000-000000000003 — подключение к общедоступным службе мгновенных услуг <br/> |
