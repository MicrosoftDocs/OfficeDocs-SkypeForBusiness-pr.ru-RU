# <a name="conferences-view"></a>Представление "Конференции"
 
В представлении "Конференции" хранятся сведения о конференциях. Это представление было представлено в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|**сессионидтиме** <br/> |datetime  <br/> |Время запроса сеанса. Используется в сочетании с Сессионидсек для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**сессионидсек** <br/> |целое  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации сеанса. Используется в сочетании с Сессионидтиме для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**ConferenceUri** <br/> |nvarchar (450)  <br/> |Универсальный код ресурса (URI) для Конференции.  <br/> |
|**конференцеуритипе** <br/> |nvarchar(256)  <br/> |Тип URI конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
|**конфинстанце** <br/> |идентификатора  <br/> |Используется для повторяющихся конференций. Каждый экземпляр повторяющейся Конференции имеет один и тот же Конференцеури, но другой Конфинстанце.  <br/> |
|**конференцестарттиме** <br/> |datetime  <br/> |Время начала Конференции.  <br/> |
|**конференцеендтиме** <br/> |datetime  <br/> |Время окончания Конференции.  <br/> |
|**организерури** <br/> |nvarchar (450)  <br/> |Универсальный код ресурса (URI) пользователя, который организовал конференцию.  <br/> |
|**организертипе** <br/> |nvarchar(256)  <br/> |Тип URI пользователя, который организовал конференцию. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
|**организертенант** <br/> |nvarchar(256)  <br/> |Клиент пользователя, который организовал конференцию. Дополнительные сведения см. в [таблице "клиенты](tenants.md) ". <br/> |
|**Pool** <br/> |nvarchar(256)  <br/> |Полное доменное имя пула, на котором размещается конференция.  <br/> |
|**Пометка** <br/> |smallint  <br/> |Битовая маска, которая включает атрибуты Конференции. Возможные значения  <br/> 0X01 — искусственная транзакция  <br/> |
