# <a name="sessiondetails-table"></a>Таблица SessionDetails
 
Каждая запись представляет один одноранговый сеанс, который может быть телефонным звонком VoIP-VoIP, сеансом обмена мгновенными сообщениями с двумя субъектами или сеансом другого типа. Вы можете выполнить соединение таблицы с [таблицей мультимедиа](media.md) , чтобы найти сведения о каждом носителе, вовлеченном в этот сеанс.
  
Обратите внимание, что поля IsUser1IntegratedWithDeskPhone и IsUser2IntegratedWithDeskPhone удалены из таблицы Сессиондетаилс, используемой в Skype для бизнеса Server 2015.
