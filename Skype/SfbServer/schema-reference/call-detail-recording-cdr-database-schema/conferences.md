# <a name="conferences-table-in-skype-for-business-server-2015"></a>Таблица конференций в Skype для бизнеса Server 2015
 
Каждая запись в этой таблице включает сведения о звонках для одной конференции.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**сессионидтиме** <br/> |datetime  <br/> |Primary  <br/> |Время, в течение которого запрос на конференцию был собран агентом CDR. Используется только в качестве первичного ключа для уникальной идентификации экземпляра Конференции.  <br/> |
|**сессионидсек** <br/> |целое  <br/> |Primary  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации сеанса. Используется в сочетании с **сессионидтиме** для уникальной идентификации экземпляра Конференции. * <br/> |
|**конференцеуриид** <br/> |целое  <br/> |Другом  <br/> |Универсальный код ресурса (URI) Конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей конференцеурис в Skype для бизнеса Server 2015](conferenceuris.md) . <br/> |
|**конфинстанце** <br/> |идентификатора  <br/> | <br/> |Полезен для повторяющихся конференций; Каждый экземпляр повторяющейся Конференции имеет один и тот же **конференцеури**, но у него будет другой **конфинстанце**. <br/> |
|**конференцестарттиме** <br/> |datetime  <br/> | <br/> |Время начала Конференции.  <br/> |
|**конференцеендтиме** <br/> |datetime  <br/> | <br/> |Время начала Конференции.  <br/> |
|**пулид** <br/> |целое  <br/> |Другом  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации пула, в котором собрана конференция. Для получения дополнительных сведений ознакомьтесь с [таблицей Pools](pools.md) . <br/> |
|**организерид** <br/> |Типом  <br/> |Другом  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации универсального кода ресурса (URI) организатора данной Конференции. Дополнительные сведения [можно найти в таблице Users](users.md) . <br/> |
|**Пометка** <br/> |smallint  <br/> || Битовая маска, которая включает атрибуты Конференции. Возможные значения <br/>  0X01 <br/>  Накопител <br/>  Транзакции <br/> |
|**Обработка** <br/> |бит  <br/> ||Внутреннее поле, используемое службой мониторинга.  <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
|**LastModifiedTime** <br/> |Датой  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было представлено в Skype для бизнеса Server 2015.  <br/> |
   
\*Для большинства сеансов для Сессионидсек будет задано значение 1. Если два сеанса начинаются в одно и то же время, Сессионидсек для одного из них будет равен 1, а для другого — 2 и т. д.
