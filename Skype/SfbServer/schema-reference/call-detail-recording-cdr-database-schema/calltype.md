|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Каллтипеид** <br/> |целое  <br/> |Primary  <br/> ||
|**CallType** <br/> |nvarchar  <br/> || Допустимые значения: <br/>  0 — неизвестно <br/>  1 – Обмен мгновенными сообщениями <br/>  2 – общий доступ к приложениям <br/>  3--звуковое сопровождение <br/>  4-аудио и видео <br/>  5-Передача файлов <br/> |
