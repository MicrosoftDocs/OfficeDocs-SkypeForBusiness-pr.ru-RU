# <a name="user-view"></a>Представление пользователя
 
В представлении пользователя хранятся сведения о пользователях, которые участвовали в звонках или сеансах с записями в базе данных. Это представление было представлено в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|Идентификатора пользователя  <br/> |целое  <br/> |Уникальный номер, идентифицирующий этого пользователя.  <br/> |
|Усерури  <br/> |nvarchar (450)  <br/> |Универсальный код ресурса (URI) пользователя.  <br/> |
|Тенанткэй  <br/> |идентификатора  <br/> |Клиент пользователя. Дополнительные сведения см. в [таблице "клиенты](tenants.md) ". <br/> |
|Уритипе  <br/> |nvarchar(256)  <br/> |Тип универсального кода ресурса (URI) пользователя. Для получения дополнительных сведений ознакомьтесь с [таблицей уритипес](uritypes.md) . <br/> |
