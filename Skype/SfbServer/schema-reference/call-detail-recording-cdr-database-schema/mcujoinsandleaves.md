# <a name="mcujoinsandleaves-table-in-skype-for-business-server-2015"></a>Таблица Мкужоинсандлеавес в Skype для бизнеса Server 2015
 
Каждая запись в этой таблице включает сведения о звонках для одной комбинации присоединения или выхода из нее и сервера конференций. Например, если пользователь присоединяется к Конференции, которая включает веб-конференции и звуковые и видеоэлементы, для подключения к веб-конференции пользователя будет создана одна запись, а для голосовых и видеоконференций пользователя будет создана другая запись.

|:-----|:-----|:-----|:-----|
|**Сессионидтиме** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время экземпляра Конференции. Используется в сочетании с **сессионидсек** для уникальной идентификации экземпляра Конференции. Для получения дополнительных сведений ознакомьтесь с таблицей " [конференции" в Skype для бизнеса Server 2015](conferences.md) . <br/> |
|**Сессионидсек** <br/> |целое  <br/> |Основной, внешний  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации экземпляра Конференции. Используется в сочетании с **сессионидтиме** для уникальной идентификации экземпляра Конференции. Для получения дополнительных сведений ознакомьтесь с таблицей " [конференции" в Skype для бизнеса Server 2015](conferences.md) . <br/> |
|**Идентификатора пользователя** <br/> |целое  <br/> |Основной, внешний  <br/> |Уникальный номер, идентифицирующий этого пользователя. Дополнительные сведения [можно](users.md) найти в таблице Users. <br/> |
|**Мкуусеринстанце** <br/> |целое  <br/> |Primary  <br/> |Если пользователь входит в систему на нескольких компьютерах или устройствах одновременно, Мкуусеринстанце однозначно определяет сочетание пользователей и устройств.  <br/> |
|**Исфромпстн** <br/> |бит  <br/> | <br/> |Присоединение пользователя к сети PSTN или нет.  <br/> |

|**Диалогсессионидтиме** <br/> |datetime  <br/> |Другом  <br/> |Время запроса сеанса. Используется в сочетании с **сессионидсек** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |
|**Диалогсессионидсек** <br/> |целое  <br/> |Другом  <br/> |ИДЕНТИФИКАЦИОНный номер для идентификации сеанса. Используется в сочетании с **сессионидтиме** для уникальной идентификации сеанса. Дополнительные сведения приведены [в таблице диалоговые окна в Skype для бизнеса Server 2015](dialogs.md) . <br/> |

|**Усерлеаветиме** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь покидает этот сервер конференц-связи.  <br/> |
|**Клиентверид** <br/> |целое  <br/> |Другом  <br/> |Идентификатор, указывающий номер версии клиентского программного обеспечения, используемого на Конференции. Для получения дополнительных сведений ознакомьтесь с [таблицей клиентверсионс в Skype для бизнеса Server 2015](clientversions.md) . <br/> Это поле было введено в Microsoft Lync Server 2013.  <br/> |
