# <a name="mcujoinsandleaves-table-in-skype-for-business-server-2015"></a>Таблица McuJoinsAndLeaves в Skype для бизнеса Server 2015 г.
 
Каждая запись в этой таблице содержит сведения о вызове одной комбинации сервера, присоединяемого или покидаемого пользователя, и серверов для переговоров. Например, если пользователь присоединяется к конференции, которая включает элементы веб-конференций и аудио- и видеоконференции, для этого пользователя будет создана одна запись, а для присоединиться к аудио- и видеоконференции пользователя будет создана другая запись.
  
|**Столбец**|**Тип данных**|**Key/Index**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**SessionIdTime** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время экземпляра конференции. Используется совместно с **SessionIdSeq** для уникального определения экземпляра конференции. Дополнительные сведения см. в таблице [конференций Skype для бизнеса Server 2015](conferences.md) г. <br/> |
|**SessionIdSeq** <br/> |int  <br/> |Основной, Внешний  <br/> |Идентификатор для определения экземпляра конференции. Используется совместно с **SessionIdTime** для уникальной идентификации экземпляра конференции. Дополнительные сведения см. в таблице [конференций Skype для бизнеса Server 2015](conferences.md) г. <br/> |
|**UserId** <br/> |int  <br/> |Основной, Внешний  <br/> |Уникальный номер, идентифицирующий данного пользователя. Дополнительные [сведения см](users.md) . в таблице Пользователи. <br/> |
|**McuUserInstance** <br/> |int  <br/> |Primary  <br/> |Если пользователь вошел сразу на нескольких компьютерах или устройствах, McuUserInstance однозначно идентифицирует комбинацию пользователя и устройства.  <br/> |
|**IsFromPstn** <br/> |bit  <br/> | <br/> |Присоединяется ли пользователь к службе PSTN или нет.  <br/> |
|**McuId** <br/> |int  <br/> |Основной, Внешний  <br/> |Уникальный номер, идентифицирующий этот сервер конференций. Дополнительные сведения см. в [таблице Mcus Skype для бизнеса Server 2015](mcus.md) г. <br/> |
|**DialogSessionIdTime** <br/> |datetime  <br/> |Foreign  <br/> |Время запроса сеанса. В сочетании с параметром **SessionIdSeq** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Диалоги Skype для бизнеса Server 2015](dialogs.md) г. <br/> |
|**DialogSessionIdSeq** <br/> |int  <br/> |Foreign  <br/> |Идентификатор для идентификации сеанса. В сочетании с параметром **SessionIdTime** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Диалоги Skype для бизнеса Server 2015](dialogs.md) г. <br/> |
|**UserJoinTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь присоединяется к этому серверу для конференций.  <br/> |
|**UserLeaveTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь покидает этот сервер для конференций.  <br/> |
|**ClientVerId** <br/> |int  <br/> |Foreign  <br/> |Идентификатор, который указывает номер версии клиентского программного обеспечения, используемого на конференции. Дополнительные сведения см. в таблице [ClientVersions Skype для бизнеса Server 2015](clientversions.md) г. <br/> Это поле было внедрено в Microsoft Lync Server 2013.  <br/> |
|**LastModifiedTime** <br/> |Datetime  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было введено Skype для бизнеса Server 2015 г.  <br/> |
