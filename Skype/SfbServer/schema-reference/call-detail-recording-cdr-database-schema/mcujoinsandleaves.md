# <a name="mcujoinsandleaves-table-in-skype-for-business-server-2015"></a>Таблица McuJoinsAndLeaves в Скайп для Business Server 2015
 
Каждая запись в этой таблице содержит сведения о одного сочетания пользователя присоединения или выходе и конференц-связи сервера звонках. К примеру Если пользователь присоединился к конференции, которая включает в себя веб-конференций и аудио/видео элементы, одной записи будут создаваться для этого пользователя присоединиться к конференц-связи веб и другой записи будут создаваться для присоединения к аудио-и видеоконференций пользователя.
  
|**Столбец**|**Тип данных**|**Ключ или индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**SessionIdTime** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время создания экземпляра конференции. Используется совместно с **SessionIdSeq** для уникальной идентификации экземпляра конференции. В разделе [Таблица конференций в Скайп для Business Server 2015](conferences.md) для получения дополнительных сведений. <br/> |
|**SessionIdSeq** <br/> |целое  <br/> |Основной, внешний  <br/> |Номер идентификатора для определения экземпляра в конференции. Используется в сочетании с **SessionIdTime** для уникальной идентификации экземпляра конференции. В разделе [Таблица конференций в Скайп для Business Server 2015](conferences.md) для получения дополнительных сведений. <br/> |
|**Идентификатор пользователя** <br/> |целое  <br/> |Основной, внешний  <br/> |Уникальный номер, идентифицирующий этого пользователя. В [таблице пользователей](users.md) для получения дополнительных сведений см. <br/> |
|**McuUserInstance** <br/> |целое  <br/> |Primary  <br/> |Если пользователь вошел в систему на нескольких компьютерах или устройствах за один раз, McuUserInstance однозначно определяет сочетание пользователя и устройства.  <br/> |
|**IsFromPstn** <br/> |бит  <br/> | <br/> |Является ли пользователь присоединения к собранию — из ТСОП или нет.  <br/> |
|**McuId** <br/> |целое  <br/> |Основной, внешний  <br/> |Уникальный номер, идентифицирующий этот сервер конференций. [Таблица Mcus в Скайп для Business Server 2015](mcus.md) для получения дополнительных сведений см. <br/> |
|**DialogSessionIdTime** <br/> |datetime  <br/> |Внешний  <br/> |Время запроса сеанса. Используется совместно с **SessionIdSeq** для уникальной идентификации сеанса. В разделе [диалоговых окон в таблице в Скайп для Business Server 2015](dialogs.md) для получения дополнительных сведений. <br/> |
|**DialogSessionIdSeq** <br/> |целое  <br/> |Внешний  <br/> |Номер идентификатора для идентификации сеанса. Используется в сочетании с **SessionIdTime** для уникальной идентификации сеанса. В разделе [диалоговых окон в таблице в Скайп для Business Server 2015](dialogs.md) для получения дополнительных сведений. <br/> |
|**UserJoinTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь присоединился к данному серверу конференций.  <br/> |
|**UserLeaveTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь покинул данный сервер конференций.  <br/> |
|**ClientVerId** <br/> |целое  <br/> |Внешний  <br/> |Используйте идентификатор, который указывает номер версии клиентского программного обеспечения в конференции. [Таблица ClientVersions в Скайп для Business Server 2015](clientversions.md) для получения дополнительных сведений см. <br/> В этом поле было представлено в Microsoft Lync Server 2013.  <br/> |
|**LastModifiedTime** <br/> |Даты и времени  <br/> ||Для внутреннего использования службой мониторинга.  <br/> В этом поле было представлено в Скайп для Business Server 2015.  <br/> |
