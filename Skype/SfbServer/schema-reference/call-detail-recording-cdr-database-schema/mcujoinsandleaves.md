# <a name="mcujoinsandleaves-table-in-skype-for-business-server-2015"></a>Таблица McuJoinsAndLeaves в Skype для бизнеса Server 2015
 
Каждая запись в этой таблице содержит сведения о вызове одной комбинации сервера join или leave пользователя и сервера переговоров. Например, если пользователь присоединяется к конференции, которая включает элементы веб-конференций и аудио- и видеоконференции, для этого пользователя создается одна запись, а для join аудио- и видеоконференции пользователя создается другая запись.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Details**|
|:-----|:-----|:-----|:-----|
|**SessionIdTime** <br/> |datetime  <br/> |Основной, внешний  <br/> |Время экземпляра конференции. Используется в сочетании с **SessionIdSeq** для уникальной идентификации экземпляра конференции. Дополнительные сведения см. в таблице ["Конференции" в Skype для бизнеса Server 2015.](conferences.md) <br/> |
|**SessionIdSeq** <br/> |int  <br/> |Основной, Внешний  <br/> |Идентификатор для определения экземпляра конференции. Используется в сочетании с **SessionIdTime** для уникальной идентификации экземпляра конференции. Дополнительные сведения см. в таблице ["Конференции" в Skype для бизнеса Server 2015.](conferences.md) <br/> |
|**UserId** <br/> |int  <br/> |Основной, Внешний  <br/> |Уникальный номер, идентифицирующий данного пользователя. Дополнительные [сведения см.](users.md) в таблице "Пользователи". <br/> |
|**McuUserInstance** <br/> |int  <br/> |Primary  <br/> |Если пользователь вошел одновременно на нескольких компьютерах или устройствах, McuUserInstance однозначно определяет комбинацию пользователя и устройства.  <br/> |
|**IsFromPstn** <br/> |bit  <br/> | <br/> |Независимо от того, присоединяется ли пользователь из PSTN.  <br/> |
|**McuId** <br/> |int  <br/> |Основной, Внешний  <br/> |Уникальный номер, идентифицирующий этот сервер. Дополнительные сведения см. в таблице [Mcus в Skype для бизнеса Server 2015.](mcus.md) <br/> |
|**DialogSessionIdTime** <br/> |datetime  <br/> |Внешняя  <br/> |Время запроса сеанса. В сочетании с параметром **SessionIdSeq** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Dialogs в Skype для бизнеса Server 2015.](dialogs.md) <br/> |
|**DialogSessionIdSeq** <br/> |int  <br/> |Внешняя  <br/> |Идентификатор для идентификации сеанса. В сочетании с параметром **SessionIdTime** определяет сеанс уникальным образом. Дополнительные сведения см. в таблице [Dialogs в Skype для бизнеса Server 2015.](dialogs.md) <br/> |
|**UserJoinTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь присоединяется к этому серверу.  <br/> |
|**UserLeaveTime** <br/> |datetime  <br/> | <br/> |Время, когда этот пользователь покидает этот сервер.  <br/> |
|**ClientVerId** <br/> |int  <br/> |Внешняя  <br/> |Идентификатор, который указывает номер версии клиентского программного обеспечения, используемого в конференции. Дополнительные сведения см. в таблице [ClientVersions в Skype для бизнеса Server 2015.](clientversions.md) <br/> Это поле было впервые введено в Microsoft Lync Server 2013.  <br/> |
|**LastModifiedTime** <br/> |Datetime  <br/> ||Для внутреннего использования службой мониторинга.  <br/> Это поле было впервые введено в Skype для бизнеса Server 2015.  <br/> |
