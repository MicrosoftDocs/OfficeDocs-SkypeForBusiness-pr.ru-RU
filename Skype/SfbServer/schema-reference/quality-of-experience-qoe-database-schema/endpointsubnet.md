|**Column**|**Тип данных**|**Key/Index**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**SubnetIP** <br/> |int  <br/> |Основной, внешний  <br/> |Целое число, представляющее подсеть.  <br/> |
|**NextUpdateTS** <br/> |datetime  <br/> ||Только для внутреннего использования.  <br/> |
