# <a name="list-of-qoe-tables"></a>Список таблиц качества взаимодействия
 
Схема базы данных состоит из следующих таблиц. 
  
**Вспомогательные таблицы**

|**Table**|**Описание**|
|:-----|:-----|
|[Таблица AppSharingMetricsThreshold](appsharingmetricsthreshold.md) <br/> |Хранит оптимальные и приемлемые значения для метрик качества работы, используемых при совместном использовании приложений.  <br/> |
|[Таблица CodecDescription](codecdescription.md) <br/> |Сопополяет уникальные идентификаторы кодека с соответствующим кодеку.  <br/> |
|[Таблица IPAddress](ipaddress.md) <br/> |Сопоирует IP-адреса с уникальными идентификаторами IP-адресов, используемыми где-либо в базе данных качества обслуживания.  <br/> |
|[Таблица NetworkConnectionDetail](networkconnectiondetail.md) <br/> |Сополяет типы сетевых подключений с идентификаторами сетевых подключений, используемыми в других местах базы данных качества обслуживания.  <br/> |
|[Таблица PurgeSettings (QoE)](purgesettings-qoe.md) <br/> |Хранит сведения о том, будут ли устаревшие записи качества обслуживания автоматически удаляться из базы данных качества обслуживания (и когда).  <br/> |
|[Таблица TraceRoute](traceroute.md) <br/> |Хранит сведения о маршруте для вызовов.  <br/> |
|[Таблица UserAgentDef (QoE)](useragentdef-qoe.md) <br/> |Сополагает идентификаторы агентов пользователей с описательными именами агентов.  <br/> |
|[Таблица VideoMetricsThreshold](videometricsthreshold.md) <br/> |Хранит оптимальные и приемлемые значения для метрик качества работы, используемых с видеозвонков.  <br/> |
|[Таблица UserAgent](useragent.md) <br/> |Хранит строки агента пользователя (UA) протокола SIP и типы UA, используемые в аудио- и видеосвязи.  <br/> |
|[Таблица User](user-0.md) <br/> |Хранит URIS пользователей, конференций и телефонов, используемых в аудио- и видеосвязи.  <br/> |
|[Таблица Endpoint](endpoint.md) <br/> |Хранит имена компьютеров с именами FQDN конечных точек, участвующих в аудио- и видеосвязи.  <br/> |
|[Таблица Pool](pool.md) <br/> |Хранит имена пулов, к которым принадлежат данные метрик.  <br/> |
|[Таблица Device](device.md) <br/> |Хранит устройства захвата и устройства отрисовки, используемые в аудио- и видеозвуках.  <br/> |
|[Таблица DeviceDriver](devicedriver.md) <br/> |Хранит драйвер для устройства захвата и устройства отрисовки, которые используются в аудио- и видеозвуках.  <br/> |
|[Таблица Conference](conference.md) <br/> |Хранит URIS конференций для сценариев конференций или DialogID для других сценариев.  <br/> |
|[Таблица SessionCorrelation](sessioncorrelation.md) <br/> |Сохраняет CorrelationID для вызовов STN.  <br/> |
|[Таблица PayloadDescription](payloaddescription.md) <br/> |Сохраняет кодек, используемый для аудио- и видеозвонков.  <br/> |
|[Таблица AppliedBandwidthSource](appliedbandwidthsource.md) <br/> |Сохраняет источник пропускной способности, используемый для аудио- и видеозвонков.  <br/> |
|[Таблица MacAddress](macaddress.md) <br/> |Сохраняет MAC-адрес конечных точек, участвующих в аудио- и видеосвязи.  <br/> |
|[Таблица Dialog](dialog.md) <br/> |Сохраняет dialog ID аудио- и видеосвязи сеансов.  <br/> |
|[Таблица Region](region.md) <br/> |Сохраняет регион сети, определенный в параметре NCS.  <br/> |
|[Таблица UserSite](usersite.md) <br/> |Сохраняет сетевой сайт, определенный в параметре NCS.  <br/> |
|[Таблица Subnet](subnet.md) <br/> |Сохраняет подсеть, задаемую в параметре NCS.  <br/> |
|[Таблица MonitoredRegionLink](monitoredregionlink.md) <br/> |Сохраняет связь между областями, определяемую в параметре NCS.  <br/> |
|[Таблица MonitoredUserSiteLink](monitoredusersitelink.md) <br/> |Сохраняет связи сетевых сайтов, определенные в параметре NCS.  <br/> |
|[Таблица EndpointSubnet](endpointsubnet.md) <br/> |Сохраняет подсеть конечной точки, участвуя в аудио- и видеосеанктах.  <br/> |
|[Таблица Server](server.md) <br/> |Сохраняет FQDN или IP-адрес сервера, через который проходит мультимедиа.  <br/> |
   
**Таблицы для данных метрик**

|**Table**|**Описание**|
|:-----|:-----|
|[Таблица AppSharingStream](appsharingstream.md) <br/> |Хранит метрики качества работы для сетевых потоков, используемых для общего доступа к приложениям. Показатели качества работы для сетевых потоков, используемых для общего доступа к приложениям.  <br/> |
|[Таблица Session](session.md) <br/> |Хранит общую информацию о сеансе аудио- или видеосвязи. Сеанс определяется как диалоговое окно SIP аудио- или видеосвязи между двумя конечными точками.  <br/> |
|[Таблица MediaLine](medialine-0.md) <br/> |Хранит сведения о каждой строке мультимедиа в сеансе. Строка мультимедиа — это коллекция из одного или более аудио- и видеопотоков. Как правило, одна строка мультимедиа будет иметь два потока: аудио или видео.  <br/> |
|[Таблица AudioStream](audiostream.md) <br/> |Хранит показатели качества звука для каждого аудиопотока в строке мультимедиа.  <br/> |
|[Таблица AudioSignal](audiosignal.md) <br/> |Хранит метрики качества звука в строке мультимедиа. К ним относятся метрики подавления акустического эха (AEC) и автоматического контроля усиления (AGC).  <br/> |
|[Таблица VideoStream](videostream.md) <br/> |Хранит показатели качества мультимедиа для каждого аудиопотока в строке мультимедиа.  <br/> |
|[Таблица AudioClientEvent](audioclientevent.md) <br/> |Хранит метрики качества звука, собранные из события клиента.  <br/> |
|[Таблица VideoClientEvent](videoclientevent.md) <br/> |Хранит показатели качества видео мультимедиа, собранные из события клиента.  <br/> |
|**Таблица DiagnosticData** <br/> |Хранит диагностические данные, которые используются только для внутреннего использования.  <br/> |

|**Таблица UserSummary** <br/> |Хранит суммарные данные для пользователей, эти данные используются только для отчетов о качестве и качестве.  <br/> |
|**Таблица CallTypeSummary** <br/> |Храните суммарные данные для типов вызовов, эти данные используются только для отчетов о качестве и качестве.  <br/> |
