# <a name="list-of-qoe-tables"></a>Список таблиц качества взаимодействия
 
Схема базы данных состоит из следующих таблиц. 
  
**Вспомогательные таблицы**

|**Таблица**|**Описание**.|
|:-----|:-----|
|[Таблица AppSharingMetricsThreshold](appsharingmetricsthreshold.md) <br/> |Сохранение оптимальных и допустимых значений показателей качества взаимодействия, которые используются с общий доступ к приложениям.  <br/> |
|[Таблица codecdescription](codecdescription.md) <br/> |Назначение уникальных идентификаторов кодека соответствующему кодеку.  <br/> |
|[Таблица IPAddress](ipaddress.md) <br/> |Сопоставление IP-адресов уникальный идентификаторами IP-адресов, используемыми в базе данных качества взаимодействия.  <br/> |
|[Таблица NetworkConnectionDetail](networkconnectiondetail.md) <br/> |Сопоставление типов сетевых подключений с идентификаторами используемыми в базе данных качества взаимодействия.  <br/> |
|[Таблица PurgeSettings (качество взаимодействия)](purgesettings-qoe.md) <br/> |Сохранение информации, указывает ли (и когда) устаревшую качества записи будут автоматически удалены из базы данных качества взаимодействия.  <br/> |
|[Таблица TraceRoute](traceroute.md) <br/> |Сохранение сведений о маршрутизации для вызовов.  <br/> |
|[Таблица useragentdef (QoE)](useragentdef-qoe.md) <br/> |Привязка идентификаторов агента пользователя к описательным именам агента.  <br/> |
|[Таблица VideoMetricsThreshold](videometricsthreshold.md) <br/> |Сохранение оптимальных и допустимых значений показателей качества взаимодействия, используемых для видеозвонков.  <br/> |
|[Таблица UserAgent](useragent.md) <br/> |Сохранение строк агента Session Initiation Protocol (SIP) пользователя (UA) и типов UA, используемых в сеансах аудио- и видеоконференций.  <br/> |
|[Таблица User](user-0.md) <br/> |Хранилища пользователей, конференции и телефона коды URI используются в сеансах аудио- и видеоконференций.  <br/> |
|[Таблица Endpoint](endpoint.md) <br/> |Сохранение ПОЛНЫХ доменных имен конечных точек, участвующих в сеансах аудио- и видеоконференций.  <br/> |
|[Таблица Pool](pool.md) <br/> |Сохранение имен пулов, к какой метрики принадлежит данных.  <br/> |
|[Таблица Device](device.md) <br/> |Сохранение устройств захвата и обработки, которые используются в вызовах аудио и видео.  <br/> |
|[Таблица DeviceDriver](devicedriver.md) <br/> |Сохранение драйвера для устройства захвата и обработки, которые используются в вызовах аудио и видео.  <br/> |
|[Таблица Conference](conference.md) <br/> |Сохранение URI конференций для сценариев конференций или DialogID для других сценариев.  <br/> |
|[Таблица SessionCorrelation](sessioncorrelation.md) <br/> |Сохранение идентификатора CorrelationID для вызовов ТСОП.  <br/> |
|[Таблица PayloadDescription](payloaddescription.md) <br/> |Сохранение кодека, используемого в вызовах аудио и видео.  <br/> |
|[Таблица AppliedBandwidthSource](appliedbandwidthsource.md) <br/> |Сохранение источника полосы пропускания, используемые в вызовах аудио и видео.  <br/> |
|[Таблица MacAddress](macaddress.md) <br/> |Сохранение MAC-адреса конечных точек, участвующих в сеансах аудио- и видеоконференций.  <br/> |
|[Таблица Dialog](dialog.md) <br/> |Сохранение идентификатора Dialogid сеансов аудио- и видеоконференций.  <br/> |
|[Таблица Region](region.md) <br/> |Сохранение области сети, определенных в параметре ncs.  <br/> |
|[Таблица UserSite](usersite.md) <br/> |Сохранение сайта сети, заданного в параметре ncs.  <br/> |
|[Таблица Subnet](subnet.md) <br/> |Сохранение подсети, определенных в параметре ncs.  <br/> |
|[Таблица MonitoredRegionLink](monitoredregionlink.md) <br/> |Сохранение ссылки на область, определенных в параметре ncs.  <br/> |
|[Таблица MonitoredUserSiteLink](monitoredusersitelink.md) <br/> |Сохранение сайта сети, определенных в параметре ncs.  <br/> |
|[Таблица EndpointSubnet](endpointsubnet.md) <br/> |Сохранение подсети конечной точки, участвующей в сеансе аудио- и видеоконференций.  <br/> |
|[Таблица Servers](server.md) <br/> |Хранит полное доменное имя или IP-адрес сервера, который проходят мультимедийные данные через.  <br/> |
   
**Таблицы для данных показателей**

|**Таблица**|**Описание**.|
|:-----|:-----|
|[Таблица AppSharingStream](appsharingstream.md) <br/> |Сохранение показателей качества взаимодействия для сетевых потоков, используемых для общего доступа к приложениям. Качество показателей взаимодействия для сетевых потоков, используемых для общего доступа к приложениям.  <br/> |
|[Таблица Session](session.md) <br/> |Сохранение общих сведений о сеансе обмена аудио- или аудио и видео. Сеанс определяется как аудио- или видеозвонка SIP диалоговое окно с между двумя конечными точками.  <br/> |
|[Таблица MediaLine](medialine-0.md) <br/> |Сохранение информации о каждом канале передачи мультимедиа в сеансе. Линии мультимедиа состоит из одного или нескольких потоков аудио- и видеоконференций. Как правило один медиалинии будут иметь два потока, звуковых и видеоустройств.  <br/> |
|[Таблица AudioStream](audiostream.md) <br/> |Сохранение показателей качества аудио для каждого аудиоканала в канале передачи мультимедиа.  <br/> |
|[Таблица AudioSignal](audiosignal.md) <br/> |Сохранение показателей качества аудио в канале передачи мультимедиа. Этот компонент включает акустического эха (AEC) и показатели автоматического рост элемента управления (Усиления).  <br/> |
|[Таблица VideoStream](videostream.md) <br/> |Сохранение показателей качества видео для каждого аудиоканала в канале передачи мультимедиа.  <br/> |
|[Таблица AudioClientEvent](audioclientevent.md) <br/> |Сохранение показателей качества аудио, собранных для события клиента.  <br/> |
|[Таблица VideoClientEvent](videoclientevent.md) <br/> |Сохранение показателей качества видео, собранных для события клиента.  <br/> |
|**Диагностических данных Diagnosticdata** <br/> |Сохранение диагностических данных, который является только для внутреннего использования.  <br/> |
