# <a name="list-of-qoe-tables"></a>Список таблиц качества взаимодействия
 
Схема базы данных состоит из следующих таблиц. 
  
**Вспомогательные таблицы**

|**Table**|**Описание**|
|:-----|:-----|
|[Таблица AppSharingMetricsThreshold](appsharingmetricsthreshold.md) <br/> |Сохраняет оптимальные и приемлемые значения для метрик Quality of Experience, используемых при совместном использовании приложений.  <br/> |
|[Таблица CodecDescription](codecdescription.md) <br/> |Карты идентификаторы кодека в соответствующий кодек.  <br/> |
|[Таблица IPAddress](ipaddress.md) <br/> |Карты IP-адреса к уникальным идентификаторам IP-адресов, используемым в других частях базы данных quality of Experience.  <br/> |
|[Таблица NetworkConnectionDetail](networkconnectiondetail.md) <br/> |Карты типов сетевого подключения к идентификаторам сетевых подключений, используемым в других местах базы данных Quality of Experience.  <br/> |
|[Таблица PurgeSettings (QoE)](purgesettings-qoe.md) <br/> |Сохраняет информацию, которая указывает, будут ли (и когда) устаревшие записи качества работы автоматически удалены из базы данных QoE.  <br/> |
|[Таблица TraceRoute](traceroute.md) <br/> |Сохраняет сведения о маршрутике для звонков.  <br/> |
|[Таблица UserAgentDef (QoE)](useragentdef-qoe.md) <br/> |Карты идентификаторы агента пользователя к описательным именам агента.  <br/> |
|[Таблица VideoMetricsThreshold](videometricsthreshold.md) <br/> |Сохраняет оптимальные и приемлемые значения для показателей качества работы, используемых при видеозвонков.  <br/> |
|[Таблица UserAgent](useragent.md) <br/> |Сохраняет строки протокола инициации сеансов (SIP) пользовательский агент (UA) и типы UA, используемые в аудио- и видео сеансах.  <br/> |
|[Таблица пользователей](user-0.md) <br/> |Сохраняет пользовательские, конференц-и телефонные URL-адреса, используемые в аудио- и видео-сеансах.  <br/> |
|[Таблица конечных точек](endpoint.md) <br/> |Сохраняет имена компьютеров FQDN конечных точек, участвующих в сеансах аудио- и видеосвязи.  <br/> |
|[Таблица пула](pool.md) <br/> |Сохраняет имена пулов, к которым принадлежат данные метрик.  <br/> |
|[Таблица устройств](device.md) <br/> |Сохраняет устройства захвата и отрисовки устройств, используемых в аудио- и видеозвоках.  <br/> |
|[Таблица DeviceDriver](devicedriver.md) <br/> |Сохраняет драйвер для устройства захвата и устройства визуализации, которые используются в аудио- и видеозвонков.  <br/> |
|[Таблица конференций](conference.md) <br/> |Сохраняет URL-адреса конференций для сценариев конференций или Диалоговое окно для других сценариев.  <br/> |
|[Таблица SessionCorrelation](sessioncorrelation.md) <br/> |Сохраняет correlationID для вызовов PSTN.  <br/> |
|[Таблица PayloadDescription](payloaddescription.md) <br/> |Сохраняет кодек, используемый при звуковых и видеозвоах.  <br/> |
|[Таблица AppliedBandwidthSource](appliedbandwidthsource.md) <br/> |Сохраняет источник пропускной способности, используемый для аудио- и видеозвонков.  <br/> |
|[Таблица MacAddress](macaddress.md) <br/> |Сохраняет mac-адрес конечных точек, участвующих в сеансах аудио- и видеосвязи.  <br/> |
|[Диалоговая таблица](dialog.md) <br/> |Сохраняет номер диалоговых диалоговых записей сеансов аудио- и видеосвязи.  <br/> |
|[Таблица регионов](region.md) <br/> |Сохраняет сетевой регион, определенный в параметре NCS.  <br/> |
|[Таблица UserSite](usersite.md) <br/> |Сохраняет сетевой сайт, определенный в параметре NCS.  <br/> |
|[Таблица subnet](subnet.md) <br/> |Сохраняет подсети, определенные в параметре NCS.  <br/> |
|[Таблица MonitoredRegionLink](monitoredregionlink.md) <br/> |Сохраняет ссылку региона, задатую в параметре NCS.  <br/> |
|[Таблица MonitoredUserSiteLink](monitoredusersitelink.md) <br/> |Сохраняет ссылки сетевого сайта, определенные в параметре NCS.  <br/> |
|[Таблица EndpointSubnet](endpointsubnet.md) <br/> |Сохраняет подсеть конечной точки, участвуя в сеансе аудио- и видеосвязи.  <br/> |
|[Таблица сервера](server.md) <br/> |Сохраняет FQDN или IP-адрес сервера, через который проходит носите.  <br/> |
   
**Таблицы для данных метрик**

|**Table**|**Описание**|
|:-----|:-----|
|[Таблица AppSharingStream](appsharingstream.md) <br/> |Сохраняет метрики качества работы для сетевых потоков, используемых для общего доступа к приложениям. Качество показателей experience для сетевых потоков, используемых для общего доступа к приложениям.  <br/> |
|[Таблица сеансов](session.md) <br/> |Сохраняет общую информацию о сеансе аудио- или аудио- и видеосвязи. Сеанс определяется как диалоговое окно SIP аудио или видео между двумя конечными точками.  <br/> |
|[Таблица MediaLine](medialine-0.md) <br/> |Сохраняет сведения о каждой строке мультимедиа в сеансе. Медиа-строка — это коллекция одного или более потоков аудио- и видеосвязи. Как правило, одна медиа-строка будет иметь два потока, аудио или видео.  <br/> |
|[Таблица AudioStream](audiostream.md) <br/> |Сохраняет метрик качества аудио мультимедиа для каждого потока звука в строке мультимедиа.  <br/> |
|[Таблица AudioSignal](audiosignal.md) <br/> |Сохраняет метрики качества аудио мультимедиа в строке мультимедиа. Это включает метрики отмены акустического эхо (AEC) и автоматического контроля усиления (AGC).  <br/> |
|[Таблица VideoStream](videostream.md) <br/> |Сохраняет метрику качества видео мультимедиа для каждого аудиопотока в строке мультимедиа.  <br/> |
|[Таблица AudioClientEvent](audioclientevent.md) <br/> |Сохраняет метрики качества аудио мультимедиа, собранные в клиентских событиях.  <br/> |
|[Таблица VideoClientEvent](videoclientevent.md) <br/> |Сохраняет метрики качества видео мультимедиа, собранные в клиентских событиях.  <br/> |
|**Таблица DiagnosticData** <br/> |Сохраняет диагностические данные, которые используются только для внутреннего использования.  <br/> |

|**Таблица CallTypeSummary** <br/> |Храните сводные данные для типов вызовов, эти данные используются только для отчетов QoE.  <br/> |
