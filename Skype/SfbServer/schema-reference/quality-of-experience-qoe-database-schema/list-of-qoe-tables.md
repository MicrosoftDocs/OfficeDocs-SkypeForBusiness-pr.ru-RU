# <a name="list-of-qoe-tables"></a>Список таблиц качества взаимодействия
 
Схема базы данных состоит из следующих таблиц. 
  
**Вспомогательные таблицы**

|**Таблица**|**Описание**|
|:-----|:-----|
|[Таблица Аппшарингметрикссрешолд](appsharingmetricsthreshold.md) <br/> |Хранение оптимальных и приемлемых значений для показателей качества взаимодействия, используемых совместно с приложением.  <br/> |
|[Таблица Кодекдескриптион](codecdescription.md) <br/> |Сопоставляет уникальные идентификаторы кодека с соответствующими кодеками.  <br/> |
|[Таблица IP-адреса](ipaddress.md) <br/> |Сопоставляет IP-адреса с уникальными идентификаторами IP-адресов, которые используются в базе данных качества обслуживания.  <br/> |
|[Таблица Нетворкконнектиондетаил](networkconnectiondetail.md) <br/> |Сопоставляет типы сетевых подключений с идентификаторами сетевых подключений, используемыми в базе данных качества обслуживания.  <br/> |
|[Таблица Пуржесеттингс (QoE)](purgesettings-qoe.md) <br/> |Хранит сведения о том, что (и когда) устаревшие записи качества обслуживания будут автоматически удалены из базы данных QoE.  <br/> |
|[Таблица использованием Traceroute](traceroute.md) <br/> |Хранит сведения о маршрутизации для звонков.  <br/> |
|[Таблица Усеражентдеф (QoE)](useragentdef-qoe.md) <br/> |Сопоставляет идентификаторы агента пользователя с описательными именами агента.  <br/> |
|[Таблица Видеометрикссрешолд](videometricsthreshold.md) <br/> |Хранение оптимальных и приемлемых значений для показателей качества взаимодействия, используемых с видеозвонками.  <br/> |
|[Таблица UserAgent](useragent.md) <br/> |В сеансах голосовой и видеосвязи для протоколов SIP (UA) User Agent (агент запуска сеанса) хранятся строки и типы UA.  <br/> |
|[Таблица User](user-0.md) <br/> |Хранит URI пользователей, конференций и телефонных номеров, используемых в сеансах голосовой связи и видео.  <br/> |
|[Таблица Endpoint](endpoint.md) <br/> |Хранит полные доменные имена конечных точек, участвующих в звуковых и видеосеансах.  <br/> |
|[Таблица Pool](pool.md) <br/> |Хранит имена пулов, которым принадлежат данные метрик.  <br/> |
|[Таблица Device](device.md) <br/> |Хранит устройства захвата и устройства рендеринга, используемые в голосовых и видеозвонках.  <br/> |
|[Таблица DeviceDriver](devicedriver.md) <br/> |Хранит драйвер устройства захвата и устройство рендеринга, которое используется в голосовых и видеозвонках.  <br/> |
|[Таблица Conference](conference.md) <br/> |Хранит коды URI конференций для сценариев конференции или Диалогид для других сценариев.  <br/> |
|[Таблица SessionCorrelation](sessioncorrelation.md) <br/> |Содержит сведения о корреляции для КОММУТИРУЕМых звонков.  <br/> |
|[Таблица PayloadDescription](payloaddescription.md) <br/> |Хранит кодек, используемый в голосовых и видеозвонках.  <br/> |
|[Таблица AppliedBandwidthSource](appliedbandwidthsource.md) <br/> |Сохранение источника пропускной способности, используемой в голосовых и видеозвонках.  <br/> |
|[Таблица MacAddress](macaddress.md) <br/> |Хранит MAC-адрес конечных точек, участвующих в сеансах аудио и видео.  <br/> |
|[Таблица Dialog](dialog.md) <br/> |Хранит идентификатор диалога для звуковых и видеосеансов.  <br/> |
|[Таблица Region](region.md) <br/> |Хранит сетевой регион, определенный в параметрах NC.  <br/> |
|[Таблица UserSite](usersite.md) <br/> |Сохранение сетевого сайта, определенного в параметрах NC.  <br/> |
|[Таблица Subnet](subnet.md) <br/> |Хранит подсеть, определенную в параметрах NC.  <br/> |
|[Таблица MonitoredRegionLink](monitoredregionlink.md) <br/> |Хранит ссылку на регион, определенную в параметрах NC.  <br/> |
|[Таблица MonitoredUserSiteLink](monitoredusersitelink.md) <br/> |Хранит ссылки на сайты сети, определенные в параметрах NC.  <br/> |
|[Таблица EndpointSubnet](endpointsubnet.md) <br/> |Хранит подсеть конечной точки, участвующей в звуковых и видеосеансах.  <br/> |
|[Таблица Servers](server.md) <br/> |Хранит полное доменное имя или IP-адрес сервера, через который восходит носитель.  <br/> |
   
**Таблицы для данных метрик**

|**Таблица**|**Описание**|
|:-----|:-----|
|[Таблица Аппшарингстреам](appsharingstream.md) <br/> |Хранение метрик качества для сетевых потоков, используемых для совместного использования приложений. Показатели качества взаимодействия для сетевых потоков, используемых для совместного использования приложений.  <br/> |
|[Таблица Session](session.md) <br/> |Хранит общие сведения о звуковых и видеофайлах, а также о звуковых и видеосеансах. Сеанс определяется как звуковое или видеодиалоговое окно SIP между двумя конечными точками.  <br/> |
|[Таблица MediaLine](medialine-0.md) <br/> |Хранит сведения о каждой строке мультимедиа в сеансе. Линия мультимедиа — это коллекция из одного или нескольких звуковых и видеопотоков. Обычно в одной линии есть два потока: аудио-или видеофайлы.  <br/> |
|[Таблица AudioStream](audiostream.md) <br/> |Хранит метрики качества звукового файла в линии мультимедиа для каждого звукового потока.  <br/> |
|[Таблица AudioSignal](audiosignal.md) <br/> |Сохраняет метрики качества звукового файла в линии мультимедиа. Это включает в себя метрики для подавления эха (AEC) и автоматических показателей контроля усиления (АГК).  <br/> |
|[Таблица VideoStream](videostream.md) <br/> |Хранит метрики качества видео для каждого звукового потока в строке мультимедиа.  <br/> |
|[Таблица AudioClientEvent](audioclientevent.md) <br/> |Сохраняет показатели качества звуковых файлов мультимедиа, полученные из клиентского события.  <br/> |
|[Таблица VideoClientEvent](videoclientevent.md) <br/> |Хранит метрики качества видео, собранные из клиентского события.  <br/> |
|**Таблица Диагностикдата** <br/> |Хранит диагностические данные, предназначенные только для внутреннего использования.  <br/> |
