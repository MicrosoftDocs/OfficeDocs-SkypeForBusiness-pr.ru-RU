# <a name="purgesettings-table-qoe"></a>Таблица PurgeSettings (качество взаимодействия)
 
Таблица PurgeSettings содержит сведения, которые указывают ли (и когда) автоматически удалять устаревшие записи качества взаимодействия из базы данных качества взаимодействия. Обратите внимание на то, что сведения, связанные с очистку можно также получить на внутри Скайп для консоли Business Server, выполнив следующую команду:
  
```
Get-CsQoEConfiguration
```

Эта таблица была введена в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**ID** <br/> |целое  <br/> |Primary  <br/> |Уникальный идентификатор для коллекции QoE параметров очистки.  <br/> |
|**EnablePurge** <br/> |бит  <br/> ||Если параметр имеет значение True (1) Microsoft Lync Server 2013 периодически очищать устаревшие записи из базы данных качества взаимодействия. Удаление будет выполнено каждый день в томе, указанного параметром PurgeHour. Если параметр имеет значение False (0) затем записей не будет автоматически очищаются из базы данных. По умолчанию используется значение True.  <br/> |
|**KeepQoEDataForDays** <br/> |целое  <br/> ||Задает срок хранения записей качества взаимодействия (в днях), которые будут удаляться из базы данных: при включении очистку записей качества взаимодействия старше это значение будет удален из базы данных. Значение по умолчанию — 60 дней.  <br/> |
|**PurgeHour** <br/> |целое  <br/> ||Задает местное время суток, когда Очистка базы данных будет выполнено. Время суток задается в 24-часовом формате (0 соответствует полуночи (00:00), а 23 — 23:00 часам вечера). Обратите внимание, что можно указать только час дня: допустимое значение 10 (, указывающее, 10:00 по), но не допускается значение 10:30 10.5 (, указывающее, 10:30 по). Значение по умолчанию — 1 (1:00 AM). Задает местное время суток, когда Очистка базы данных будет выполнено. Время суток задается в 24-часовом формате (0 соответствует полуночи (00:00), а 23 — 23:00 часам вечера). Обратите внимание, что можно указать только час дня: допустимое значение 10 (, указывающее, 10:00 по), но не допускается значение 10:30 10.5 (, указывающее, 10:30 по). Значение по умолчанию — 1 (1:00 AM).  <br/> |
