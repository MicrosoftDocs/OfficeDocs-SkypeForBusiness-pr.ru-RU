# <a name="appliedbandwidthsource-table"></a>Таблица AppliedBandwidthSource
 
Таблица Апплиедбандвидссаурце является вспомогательной таблицей. Каждая запись представляет один источник.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**апплиедбандвидссаурцекэй** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий источник.  <br/> |
|**AppliedBandwidthSource** <br/> |varchar (256)  <br/> |Повторя  <br/> |Это источник установленного места для пропускной способности. Она описывает, откуда поступают ограничения пропускной способности (например, "сервер политики", "превратить сервер" или "модальность").  <br/> |
