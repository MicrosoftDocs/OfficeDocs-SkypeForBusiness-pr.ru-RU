# <a name="networkconfigurationsettings-view"></a>Представление networkconfigurationsettings
 
Представление Networkconfigurationsettings хранятся сведения о конфигурации сети. В этом представлении была введена в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Сведения**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |целое  <br/> |IP-адрес подсети (поддерживаются только адреса IPv4).  <br/> |
|UserSiteName  <br/> |nvarchar(128)  <br/> |Имя сайта пользователя.  <br/> |
|RegionName  <br/> |nvarchar(128)  <br/> |Название страны или региона пользователя.  <br/> |
