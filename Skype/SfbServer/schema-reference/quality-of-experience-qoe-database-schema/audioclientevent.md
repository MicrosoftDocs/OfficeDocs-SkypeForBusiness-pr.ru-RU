# <a name="audioclientevent-table"></a>Таблица AudioClientEvent
 
Каждая запись содержит событие клиента для одной конечной точки в звуковом вызове. Обычно один вызов имеет две записи, одну для вызываемой и одну для вызываемой.
  
|**Столбец**|**Тип данных**|**Key/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**ConferenceDateTime** <br/> |datetime  <br/> |Primary  <br/> |Ссылки из [таблицы MediaLine](medialine-0.md).  <br/> |
|**SessionSeq** <br/> |int  <br/> |Primary  <br/> |Ссылки из [таблицы MediaLine](medialine-0.md).  <br/> |
|**MediaLineLabel** <br/> |tinyint  <br/> |Primary  <br/> |Ссылки из [таблицы MediaLine](medialine-0.md).  <br/> |
|**FromCaller** <br/> |bit  <br/> |Primary  <br/> |0. Данные вызываемой  <br/> 1. Данные вызываемой  <br/> |
|**NetworkSendQualityEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие NetworkSendQuality было уволено за состояние "Bad".  <br/> Качество сети с точки зрения дрожания или потери пакета является серьезным и влияет на качество отосланного звука.  <br/> |
|**NetworkReceiveQualityEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие ReceiveSendQuality было уволено за состояние "Bad".  <br/> Качество сети с точки зрения дрожания или потери пакетов является серьезным и влияет на качество полученного звука.  <br/> |
|**NetworkDelayEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие Delay было уволено за состояние "Плохое". Задержка сети является серьезной и влияет на взаимодействие путем предотвращения интерактивной связи  <br/> |
|**NetworkBandwidthLowEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие LowBandwidth было уволено за состояние "Bad". Доступная пропускная способность недостаточна для приемлемого голосового доступа.  <br/> |
|**CPUInsufficientEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |В процентах от сеанса недостаточное событие ЦП было уволено для состояния "Bad". Циклов ЦП недостаточно для обработки с использованием текущих способов и приложений. Это приводит к искажениям с аудиоканалом.  <br/> |
|**DeviceHalfDuplexAECEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceHalfDuplexAEC было уволено за состояние "Bad". Чтобы предотвратить эхо, система вводит половину дуплекса.  <br/> |
|**DeviceRenderNotFunctioningEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceRenderNotFunctioning было уволено за состояние "Bad". Устройство визуализации, используемая в настоящее время для сеанса, не функционирует правильно. Это может вызвать проблемы с аудиозаписью в одну сторону.  <br/> |
|**DeviceCaptureNotFunctioningEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceCaptureNotFunctioning было уволено за состояние "Bad". Устройство захвата, которое в настоящее время используется для сеанса, не функционирует правильно. Это может вызвать проблемы с аудиозаписью в одну сторону.  <br/> |
|**DeviceGlitchesEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceGlitches было уволено за состояние "Bad". Существуют серьезные сбои в отрисовки звука, которые вызывают искажения. Эти сбои могут быть вызваны ошибками драйвера, штормом отложенных вызовов процедур (DPC) и высоким уровнем использования ЦП.  <br/> |
|**DeviceLowSNREventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceLowSNR было уволено за состояние "Bad". Качество захвата очень низкое, либо очень шумное, либо пользователь говорит слишком далеко от микрофона. Это приведет к искажениям.  <br/> |
|**DeviceLowSpeechLevelEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceLowSpeechLevel было уволено за состояние "Bad". Уровень речи пользователя слишком низкий, и система не может увеличить его дальше. Это может привести к искажениям или восприниматься как однонабный звук.  <br/> |
|**DeviceClippingEventRatio** <br/> |Десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceClipping было уволено за состояние "Bad".  <br/> При обрезке микрофона в ближайшей речи в конце слышно искажение из-за обрезания. Важно избегать обрезки ближайшего микрофона.  <br/> |
|**DeviceEchoEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceEchoEvent было уволено за состояние "Bad". Устройство или установка вызывают эхо за пределами возможности системы для компенсации.  <br/> |
|**DeviceNearEndToEchoRatioEventRatio** <br/> |десятичной (5,2)  <br/> | <br/> |Процент сеанса событие DeviceNearEndToEchoRatio было уволено за состояние "Bad". Речь пользователя слишком низка по сравнению с захваченным эхом, которое влияет на работу пользователей, так как ограничивает простое прерывание пользователя. Уменьшите громкость динамика, переместим микрофон ближе к разговорнику.  <br/> |
|**DeviceMultipleEndpointsEventCount** <br/> |int  <br/> ||Количество раз во время сеанса событие DeviceMultipleEndpoints было уволено за состояние "Bad". Несколько конечных точек звука в одном сеансе обнаружено, и система компенсирована снижением объема визуализации.  <br/> |
|**DeviceHowlingEventCount** <br/> |int  <br/> | <br/> |Количество раз во время сеанса событие DeviceHowlingEvent было уволено за состояние "Bad". Обнаружен цикл звуковой обратной связи (вызванный несколькими конечными точками, которые делятся аудиодорожкой).  <br/> |
|**DeviceRenderZeroVolumeEventRatio** <br/> |десятичной (5,2)  <br/> ||Процент сеанса событие DeviceRenderZeroVolume было уволено за то, что он находится в состоянии "Bad". Устройство визуализации было настроено на нулевой объем.  <br/> Этот столбец был представлен в Microsoft Lync Server 2013.  <br/> |
|**DeviceRenderMuteEventRatio** <br/> |десятичной (5,2)  <br/> ||Процент сеанса событие DeviceRenderMute было уволено за то, что он находится в состоянии "Bad". Устройство отрисовки было отключено.  <br/> Этот столбец был представлен в Microsoft Lync Server 2013.  <br/> |
