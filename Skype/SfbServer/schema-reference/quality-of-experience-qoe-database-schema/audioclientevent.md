# <a name="audioclientevent-table"></a>Таблица AudioClientEvent
 
Каждая запись имеет событие клиента для одной конечной точки в голосовой видеозвонке. Обычно один звонок состоит из двух записей: одного для вызывающего и для вызываемого абонента.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Конференцедатетиме** <br/> |datetime  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Сессионсек** <br/> |целое  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Медиалинелабел** <br/> |tinyint  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Фромкаллер** <br/> |бит  <br/> |Primary  <br/> |0: данные вызывающего абонента  <br/> 1: данные вызывающего абонента  <br/> |

|**Нетворкрецеивекуалитевентратио** <br/> |десятичное число (5; 2)  <br/> | <br/> |Процент сеанса, когда событие Рецеивесендкуалити было инициировано для состояния "Bad".  <br/> Качество сети в условиях нарушения или потери пакетов является серьезным и повлияет на качество получаемого звука.  <br/> |
|**Нетворкделайевентратио** <br/> |десятичное число (5; 2)  <br/> | <br/> |Процент сеанса. событие задержки было инициировано для состояния "Bad". Задержка сети является серьезной и повлияет на работу, предотвращая интерактивную связь  <br/> |

|**Девицерендернотфунктионинжевентратио** <br/> |десятичное число (5; 2)  <br/> | <br/> |Процент сеанса, когда событие Девицерендернотфунктионинг было инициировано для состояния "Bad". Устройство рендеринга, используемое в данный момент для сеанса, работает неправильно. Это может привести к появлению односторонней голосовой проблемы.  <br/> |
|**Девицекаптуренотфунктионинжевентратио** <br/> |десятичное число (5; 2)  <br/> | <br/> |Процент сеанса, когда событие Девицекаптуренотфунктионинг было инициировано для состояния "Bad". При отрисовке звука возникают серьезные проблемы, вызывающие искажения. Это может быть вызвано проблемами с драйверами, отложенными вызовами процедур (DPC) (Drivers) и высокой загрузкой ЦП.  <br/> |

|**Девицеловспичлевелевентратио** <br/> |десятичное число (5; 2)  <br/> | <br/> |Процент сеанса, когда событие Девицеловспичлевел было инициировано для состояния "Bad". Слишком низкий уровень речи пользователя, система не может увеличить его. Это может вызвать либо искажение или восприятие в качестве односторонней голосовой.  <br/> |
|**Девицеклиппинжевентратио** <br/> |Десятичное число (5; 2)  <br/> | <br/> |Процент сеанса, когда событие Девицеклиппинг было инициировано для состояния "Bad".  <br/> При использовании близких между собой видеороликов микрофон прозвучит искажения из-за обрезки.
