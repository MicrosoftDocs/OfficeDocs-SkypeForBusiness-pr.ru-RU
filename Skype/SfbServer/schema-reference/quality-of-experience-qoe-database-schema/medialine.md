# <a name="medialine-view"></a>Представление MediaLine
 
Представление MediaLine хранятся сведения о каждом канале передачи мультимедиа в базе данных. Один аудиосеанса обычно содержит одну строку аудио. Один аудио- и видеоконференций (A / V) сеанса обычно содержит одну строку аудио и одну строку видео; Тем не менее сеанс может содержать две строки видео при использовании устройства конференц-связи или при использовании представления галереи. В этом представлении была введена в Microsoft Lync Server 2013.
