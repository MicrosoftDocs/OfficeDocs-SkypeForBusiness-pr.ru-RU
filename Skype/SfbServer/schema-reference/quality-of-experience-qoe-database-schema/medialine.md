# <a name="medialine-view"></a>Представление MediaLine
 
В представлении MediaLine хранится информация о каждой строке мультимедиа в базе данных. Один сеанс аудиосвязи обычно содержит одну строку мультимедиа. Один сеанс аудио- и видеосвязи обычно содержит одну аудио- и видеомагимичную линию; Однако сеанс может содержать две видеомайки, если используется устройство для видеоконференций или используется представление галереи. Это представление впервые было введено в Microsoft Lync Server 2013.
