# <a name="endpoint-table"></a>Таблица Endpoint
 
Таблица конечных точек — это вспомогательная таблица, в которой хранятся сведения о конечных точках, участвующих в сеансах, записанных в базе данных. Каждая запись в таблице представляет одну конечную точку.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Ендпоинткэй** <br/> |целое  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий эту конечную точку.  <br/> |
|**Имя**. <br/> |nvarchar(256)  <br/> |Повторя  <br/> |Имя конечной точки.  <br/> |
|**OS** <br/> |nvarchar(128  <br/> | <br/> |Операционная система (ОС) конечной точки.  <br/> |
|**Кпунаме** <br/> |nvarchar(128  <br/> ||Имя ЦП конечной точки.  <br/> |
|**Кпунумберофкорес** <br/> |smallint  <br/> ||Количество ядер ЦП конечной точки.  <br/> |
|**Кпупроцессорспид** <br/> |целое  <br/> ||Тактовая частота процессора для конечной точки.  <br/> |
|**Виртуализатионфлаг** <br/> |tinyint  <br/> || Битовый флаг, указывающий на то, что система работает в виртуализованной среде. <br/>  0x0000 — None <br/>  0x0001-HyperV <br/>  0x0002 — VMWare <br/>  0x0004 — виртуальный ПК <br/>  0x0008 — компьютер Xen <br/> |
