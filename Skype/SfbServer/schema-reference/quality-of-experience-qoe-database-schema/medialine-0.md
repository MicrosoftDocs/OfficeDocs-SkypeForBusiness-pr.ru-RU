# <a name="medialine-table"></a>Таблица MediaLine
 
Каждая запись представляет одну строку мультимедиа. (Один аудиосеанса обычно содержит одну строку аудио. Один аудио- и видеоконференций (A / V) сеанса обычно содержит одну строку аудио и одну строку видео, несмотря на то, что сеанс может содержать две строки видео, при использовании устройства конференц-связи или при использовании представления галереи.
