# <a name="videoclientevent-table"></a>Таблица VideoClientEvent
 
Каждая запись имеет событие клиента для одной конечной точки во время видеозвонка. Обычно один звонок состоит из двух записей: одного для вызывающего и для вызываемого абонента.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**Конференцедатетиме** <br/> |datetime  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Сессионсек** <br/> |целое  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Медиалинелабел** <br/> |tinyint  <br/> |Primary  <br/> |На которую ссылается [Таблица медиалине](medialine-0.md).  <br/> |
|**Фромкаллер** <br/> |бит  <br/> |Primary  <br/> |0: данные вызывающего абонента  <br/> 1: данные вызывающего абонента  <br/> |
|**Нетворкбандвидсловевентратио** <br/> || <br/> |Процент сеанса, когда событие Ловбандвидс было инициировано для состояния "Bad". Доступная пропускная способность недостаточна для приемлемого голосового интерфейса.  <br/> |
|**Нетворкрецеивекуалитевентратио** <br/> || <br/> |Процент сеанса, когда событие Рецеивесендкуалити было инициировано для состояния "Bad".  <br/> Качество сети в условиях нарушения или потери пакетов является существенным и влияет на качество получаемого звука.  <br/> |
