# <a name="traceroute-table"></a>Таблица использованием Traceroute
 
В таблице использованием Traceroute содержатся сведения о маршрутизации из звонков. Эта таблица введена в Microsoft Lync Server 2013.
  
|**Столбец**|**Тип данных**|**Ключ/индекс**|**Сведения**|
|:-----|:-----|:-----|:-----|
|**конференцедатетиме** <br/> |datetime  <br/> |Основной, внешний  <br/> |Дата и время начала звонка.  <br/> |
|**сессионсек** <br/> |целое  <br/> |Основной, внешний  <br/> |Уникальный идентификатор, который используется для различения нескольких звонков, которые могли приступили к одной и той же дате и в то же время.  <br/> |
|**медиалинелабел** <br/> |tinyint  <br/> |Основной, внешний  <br/> |Представляет тип видеостроки, используемой в звонке. Допустимые значения:  <br/> 0 – звук  <br/> 1-видео  <br/> 2-панорамное видео  <br/> 3 — общий доступ к приложениям и рабочим столам  <br/> |
|**фромкаллер** <br/> |бит  <br/> |Primary  <br/> |Конечная точка, в которой был помещен звонок.  <br/> |
|**Участк** <br/> |целое  <br/> ||Транзитный сетевой переход/  <br/> |
|**ипаддресскэй** <br/> |целое  <br/> |Другом  <br/> |Уникальный идентификатор IP-адреса. Сведения об IP-адресе хранятся в [таблице IPAddress](ipaddress.md).  <br/> |
|**ПЕРЕДАЧИ** <br/> |целое  <br/> ||Время обмена данными. Время для обмена данными измеряет время, затрачиваемое на передачу голосового пакета получателю, а затем отправляет уведомление о том, что оно получено.  <br/> |
