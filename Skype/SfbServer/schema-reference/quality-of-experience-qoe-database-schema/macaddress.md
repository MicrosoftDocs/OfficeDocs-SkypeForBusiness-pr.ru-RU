|**Столбец**|**Тип данных**|**Key/Index**|**Details**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primary  <br/> |Уникальный номер, идентифицирующий MAC-адрес.  <br/> |
|**MacAddress** <br/> |varchar (256)  <br/> |Уникальные  <br/> |Строка MAC-адреса.  <br/> |
