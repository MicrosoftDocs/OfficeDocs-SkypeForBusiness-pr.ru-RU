# <a name="tblfiletoken"></a>tblFileToken
 
Тблфилетокен включает временные маркеры для целей обмена файлами.
  
**Столбцов**

|**Столбец**|**Тип**|**Описание**|
|:-----|:-----|:-----|
|филетокен  <br/> |nvarchar (50), NOT NULL  <br/> |Уникальный маркер (GUID).  <br/> |
|филетокенусерид  <br/> |int, NOT NULL  <br/> |Идентификатор участника, который передает файл.  <br/> |
|филетокенчаннелид  <br/> |GUID, а не NULL  <br/> |GUID узла комнаты чата.  <br/> |
|филетокенекспиредате  <br/> |DateTime, NOT NULL  <br/> |Время окончания срока действия. (Срок действия токенов истекает через 30 минут, за исключением случаев, когда они закреплены (в этой статье описаны следующие описания).  <br/> |
|филетокенкомплианцефилеурл  <br/> |nvarchar(256)  <br/> |URL-адрес перенесенного файла (для использования службы соответствия требованиям).  <br/> |
|филетокенкомплианцесумбнаилурл  <br/> |nvarchar(256)  <br/> |URL-адрес миниатюры перенесенного файла (для использования службой соответствия требованиям).  <br/> |
|филетокенкомплианцетиме  <br/> |datetime2  <br/> |Метка времени для фактической операции передачи файлов (для использования службой соответствия требованиям).  <br/> |
|филетокенкомплианцеисуплоад  <br/> |бит  <br/> |Значение true, если отправка; Значение false, если Download (для использования службы соответствия требованиям).  <br/> |
|филетокенкомплианцепиннед  <br/> |bit, NOT NULL  <br/> |Значение true, если маркер закреплен. Она используется для хранения токенов в таблице до тех пор, пока служба соответствия требованиям не сможет получить из нее нужные поля.  <br/> |
   
**Параметры**

|**Столбец**|**Описание**|
|:-----|:-----|
|филетокен  <br/> |Первичный ключ.  <br/> |
|филетокенчаннелид  <br/> |Внешний ключ с подстановкой в таблице Тблноде. Нодегуид.  <br/> |
