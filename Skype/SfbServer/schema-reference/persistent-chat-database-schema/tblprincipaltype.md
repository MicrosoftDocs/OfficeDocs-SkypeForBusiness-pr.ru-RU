# <a name="tblprincipaltype"></a>tblPrincipalType
 
ТблпринЦипалтипе содержит основные типы для классификации содержимого в таблице ТблпринЦипал.
  
**Столбцов**

|**Столбец**|**Тип**|**Описание**|
|:-----|:-----|:-----|
|Птипеид  <br/> |smallint, NOT NULL  <br/> |Идентификатор типа участника.  <br/> |
|Птипедеск  <br/> |nvarchar (256), NOT NULL  <br/> |Описание типа.  <br/> |
|Птипеиссистемусер  <br/> |bit, NOT NULL  <br/> |Значение true, если тип соответствует участникам, которые используются для внутренних целей.  <br/> |
|Птипеисусер  <br/> |bit, NOT NULL  <br/> |Значение true, если тип является пользовательским типом.  <br/> |
   
**Ключ**

|**Столбец**|**Описание**|
|:-----|:-----|
|Птипеид  <br/> |Первичный ключ.  <br/> |
   
**Основные значения**

|**ID**|**Роль**|**Описание**|**User**|
|:-----|:-----|:-----|:-----|
|1  <br/> |Любой  <br/> |Универсальный принципал без известного типа. Не используется в таблице ТблпринЦипал.  <br/> ||
|2  <br/> |Анюсер  <br/> |Универсальный принципал для типа пользователя. Не используется в таблице ТблпринЦипал.  <br/> |Да  <br/> |
|3  <br/> |Аниграуп  <br/> |Универсальный принципал с семантикой группы. Не используется в таблице ТблпринЦипал.  <br/> ||
|4  <br/> |Системусер  <br/> |Основной сервер, который используется для внутренних целей с помощью сохраняемого сервера чата.  <br/> ||
|5  <br/> |Пользователь  <br/> |Обычный пользователь.  <br/> |Да  <br/> |
|No8  <br/> |NУДАЛЕННЫЙ  <br/> |Контроллер домена доменных служб Active Directory.  <br/> ||
|@  <br/> |Сгруппирован  <br/> |Группа безопасности Active Directory.  <br/> ||
|5-10  <br/> |Нее  <br/> |Контейнер Active Directory или подразделение.  <br/> ||
