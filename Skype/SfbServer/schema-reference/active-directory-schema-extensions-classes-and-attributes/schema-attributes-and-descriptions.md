# <a name="schema-attributes-and-descriptions-in-skype-for-business-server"></a>Атрибуты схемы и описания в Скайп Business Server
 
В этом разделе описываются все атрибуты схемы, используемые Скайп для Business Server. Классы, связанные с атрибутами в разделе [атрибуты схемы по классам в Скайп для Business Server](schema-attributes-by-class.md).
  
Атрибуты, которые являются связанных пар задаются в виде прямые ссылки или ссылки назад. Атрибут, на который ссылается на другой объект является прямая ссылка; атрибут других объекта, на который ссылается обратно на первый объект является обратная ссылка. Прямые ссылки, обновляемым, когда ссылки назад доступны только для чтения.
  
Некоторые атрибуты имеют значение битовую маску. Для этих атрибутов представлены каждый параметр немного и отображаемых десятичное значение представляет биты. Bit положения начать с бит 0. Например 1 (двоичный) — это набор бит 0 и 10000 (двоичный) — это набор бит 4. Каждый бит представляет свойство. Ниже приведены некоторые примеры: 
  
- 10000 (двоичный) имеет десятичное значение 16 (то есть, 4 бита).
    
- 100000000 (двоичный) имеет десятичное значение 256 (то есть, 8 бита).
    
- 1100 (двоичный) имеет десятичное значение 12 (то есть, 2 и 3 биты; свойства, представленного оба бита включены).
    
- 1111000001 (двоичный) имеет десятичное значение 961 (то есть биты 0, 6, 7, 8 и 9; свойства для каждого из этих битов включены).
