# <a name="changes-made-by-grant-cssetuppermission-in-skype-for-business-server"></a>Изменения, внесенные с помощью GRANT-Кссетуппермиссион в Skype для бизнеса Server
 
Для делегирования настройки вы можете предоставить разрешения на доступ к универсальной группе Рткуниверсалсерверадминс для определенного подразделения Active Directory (OU), включив в него членов группы Рткуниверсалсерверадминс в подразделении для установки Skype для бизнеса Server указанный домен, не являющийся членом группы администраторов домена. 
  
Командлет **Grant-кссетуппермиссион** предоставляет разрешения на доступ к группе рткуниверсалсерверадминс на подразделение, как указано в приведенной ниже таблице.
  
**Разрешения, предоставленные объектам в подразделении**

|**Разрешения применяются к:**|**Предоставлены следующие разрешения:**|
|:-----|:-----|
|RTCUniversalServerAdmins  <br/> | Специальный доступ: <br/>  Чтение передаваемые по себе <br/>  Вводный текст <br/>  Удалить дерево <br/>  Репликация изменений каталога <br/> |
|Дочерние объекты serviceConnectionPoint  <br/> | Специальный доступ: <br/>  Разрешения на чтение <br/>  Разрешения на запись <br/>  Создание дочернего элемента <br/>  Удалить дочерний элемент <br/>  Содержимое списка <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты msRTCSIP-Server  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние msRTCSIP — объекты-компоненты  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты msRTCSIP-MCU  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты msRTCSIP-Медиатионсервер  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты msRTCSIP-Аппликатионсервер  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты msRTCSIP-Коннектионпоинт  <br/> | Специальный доступ: <br/>  Запись свойства <br/>  Чтение свойства <br/>  Удалить дерево <br/> |
|Дочерние объекты компьютера  <br/> | Специальный доступ для serviceConnectionPoint: <br/>  Создание дочерних объектов <br/>  Удаление дочерних объектов <br/>  Удалить дерево <br/>  Специальный доступ к общедоступным сведениям: <br/>  Чтение свойства <br/>  Особый доступ для DNS-имени узла: <br/>  Чтение свойства <br/> |
