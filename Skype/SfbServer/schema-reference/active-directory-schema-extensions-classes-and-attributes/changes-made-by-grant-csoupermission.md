# <a name="changes-made-by-grant-csoupermission-in-skype-for-business-server"></a>Изменения, внесенные с помощью GRANT-Ксаупермиссион в Skype для бизнеса Server
 
Чтобы делегировать администрирование сервера в Skype для бизнеса Server, вы можете добавить разрешения для указанных организационных подразделений, чтобы участники универсальных групп RTC, созданные с помощью подготовки леса, могли получать доступ к подразделениям без участия в группе Администраторы домена. 
  
Командлет **Grant-ксаупермиссион** предоставляет разрешения на доступ к объектам в указанном подразделении, как указано в приведенных ниже таблицах.
  
## <a name="granting-permission-for-user-objects"></a>Предоставление разрешений для объектов пользователей

Когда вы запускаете командлет **Grant-ксаупермиссион** для объектов пользователей в подразделении, группам предоставляются разрешения, как показано в приведенной ниже таблице.
  
**Разрешения, предоставленные для объектов пользователя**

|**Сгруппирован**|**PermissionSet**|**Применимо к**|
|:-----|:-----|:-----|
|ртчсуниверсалсервицес  <br/> |Репликация изменений каталога  <br/> |Только этот объект  <br/> |
|рткуниверсалсерверреадонлиграуп  <br/> |Содержимое списка  <br/> Чтение всех свойств  <br/> Разрешения на чтение  <br/> |Только этот объект  <br/> |
|рткуниверсалусерреадонлиграуп  <br/> |Содержимое списка  <br/> Чтение всех свойств  <br/> Разрешения на чтение  <br/> |Только этот объект  <br/> |
|рткуниверсалусерреадонлиграуп  <br/> |Прочитать Рткусерсеарчпропертисет  <br/> Прочитать Рткусерпровисионингпропертисет  <br/> Прочитать Рткпропертисет  <br/> Чтение общедоступной информации  <br/> Чтение общих данных  <br/> Чтение ограничений для учетных записей пользователей  <br/> |Дочерние объекты пользователей  <br/> |
|RTCUniversalUserAdmins  <br/> |Напишите Рткусерсеарчпропертисет  <br/> Напишите Мсексчуквоицемаилсеттингс  <br/> Напишите Рткусерпровисионингпропертисет  <br/> Напишите Рткпропертисет  <br/> Напишите proxyAddresses  <br/> |Дочерние объекты пользователей  <br/> |
