# Что такое автосекретари телефонной системой?

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Телефонную систему в Office 365 автосекретари можно использовать для создания системы меню для вашей организации, позволяющий внешние и внутренние абонентов перемещения по меню системы поиска и поместите или передача вызова пользователи компании или отделов в вашей организации.
  
При вызове людей, они представлены последовательностью голосовым инструкциям, помогающие позвонить пользователю или найдите пользователя в вашей организации и затем звонить для этого пользователя. Автосекретарь представляет собой последовательность голосовым инструкциям или и звуковой файл, который вызывающие абоненты прослушать вместо человека оператор при звонке в организацию. Автосекретарь позволяет абонентов перемещаться по системе меню, звонки, или найдите пользователей с помощью клавиатуры телефона (набора DTMF) или голосовых входные данные с помощью распознавания речи. Настройка автосекретаря для телефонной системой в Office 365 перейдите [Настройка телефонной системой автосекретаря](set-up-a-phone-system-auto-attendant.md).
  
Автосекретарь телефонной системой обладает следующими возможностями.
  
- Воспроизведение корпоративных или информационных приветствий.
    
- Поддержка настраиваемых корпоративных меню. Вы можете настроить эти меню, чтобы они имели несколько уровней.
    
- Поддержка функции поиска в каталоге, которая позволяет вызывающему абоненту выполнять поиск определенного имени в каталоге организации.
    
- Таким образом, вызывающий абонент может связаться с нужным сотрудником организации напрямую или оставить ему сообщение.
    
Чтобы настроить телефонной системой автоматического автосекретаря, перейдите [Настройка телефонной системой автосекретаря](set-up-a-phone-system-auto-attendant.md).
  
## Начало работы

Перед началом работы с функцией автосекретаря необходимо запомнить следующие моменты.
  
- Корпоративный E3 плюс **Телефонной системой** лицензии или E5 корпоративные лицензии вашей организации должен иметь (как минимум). Количество лицензий пользователей **Телефонную систему**, назначенных последствий, что число службы номера, доступны для использования автосекретарей. Номера автосекретари, которые может иметь зависит от номера **Телефонной системой** и **Аудиоконференций** лицензий, которые назначаются в вашей организации. Чтобы узнать больше о лицензировании,[Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
    
    > [!TIP]
    > Для перенаправления вызовов оператора или меню параметр, который является Online пользователь с **Телефонной системой** лицензии, необходимо будет для включения их Корпоративная голосовая связь и назначение им вызова планы. Просмотреть[Назначение лицензий Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/assign-skype-for-business-and-microsoft-teams-licenses.md). Вы также можете использовать Windows PowerShell. Пример запуска:  `Set-CsUser -identity "Amos Marble" -EnterpriseVoiceEnabled $true`
  
- Получение и использование бесплатный служебные номера для вашего автосекретари, необходимо настроить кредиты связи. Для выполнения этой задачи см [Что такое кредиты на связь?](../skype-for-business-and-microsoft-teams-add-on-licensing/what-are-communications-credits.md) и[Настройка кредитов на связь для организации](../skype-for-business-and-microsoft-teams-add-on-licensing/set-up-communications-credits-for-your-organization.md).
    
    > [!IMPORTANT]
    > Номера телефонов пользователей (абонентов) нельзя назначить автосекретарям  можно использовать платные или бесплатные номера телефонов службы. 
  
## Обзор функций

### Звонок по имени

Удаленный по имени  это функция автосекретаря, который называется поиск по каталогу. Он позволяет звонящих пользователей к вашей автосекретаря с помощью голосовых (распознавания речи) или их клавиатуре телефона (набора DTMF) введите полное или частичное имя поиск в каталоге компании, найдите человека и нажмите переадресации перенесены на них. Если у вас есть Скайп для бизнеса Online пользователям, **они не требуются для перенаправления номер телефона или вызова планы им назначены, но они должны **Телефонной системой** лицензии** для них будет доступен при выполнении поиска с помощью набора номера по имени. Удаленный по имени даже смогут найти и переключать звонки на Скайп для бизнеса Online пользователей, которые размещены в разных стран или регионов для многонациональных организаций.
  
> [!CAUTION]
> При поиске в справочнике не включаются пользователи локальных решений Skype для бизнеса 2015 или Lync Server 2013 и 2010. 
  
### Максимальный размер справочника

Существует без ограничения на размер Active Directory, позвонить по имени поддерживается при использовании на клавиатуре телефона для поиска ввод имен полное или частичное ("имя" Фамилия, а также Фамилия + "имя"). Однако максимальный размер одного автосекретаря может поддерживать помощи имя распознавания речи список имен является 80 000 пользователей.
  
||||
|:-----|:-----|:-----|
|**Тип ввода** <br/> |**Формат поиска** <br/> |**Максимальное число пользователей в организации** <br/> |
|DTMF (ввод с клавиатурной панели)  <br/> |Частичный  <br/> Имя + фамилия  <br/> Фамилия + имя  <br/> |Нет жесткого ограничения  <br/> |
|Распознавание речи (голосовой ввод)  <br/> |Имя  <br/> Фамилия  <br/> Имя + фамилия  <br/> Фамилия + Имя  <br/> |80 000 пользователей  <br/> |
   
> [!NOTE]
> Если с помощью набора номера по имени с программой распознавания речи, но Active Directory организации больше, чем 80 000 пользователей и еще не ограниченный область набора по имени с помощью области набора возможностей, набора по имени по-прежнему подходит для вызывающих с клавиатуры телефона и голосовой входные данные будут доступны для всех остальных случаях. Чтобы ограничить количество имен доступных путем изменения области охвата набора по имени для конкретного автосекретаря можно использовать функцию область набора. 
  
### Звонок по имени  ввод с клавиатурной панели телефона (DTMF)

С помощью функции звонка по имени вызывающие абоненты могут находит нужных пользователей по всему или части имени. Для ввода имени можно использовать разные форматы.
  
Чтобы ввести пробел между именем и фамилией пользователя при поиске, можно нажать клавишу 0 (ноль). Чтобы завершить ввод имени, необходимо нажать клавишу # (решетка). Например, в этом случае может воспроизводиться подсказка "После ввода имени нужного пользователя нажмите клавишу #." Если найдено несколько пользователей, выдается список имен для выбора.
  
Поддерживаются следующие форматы поиска имен в организации с использованием клавиатурной панели телефона.
  
|||||
|:-----|:-----|:-----|:-----|
|**Формат имени** <br/> |**Тип поиска** <br/> |**Пример** <br/> |**Результат поиска** <br/> |
|Имя + фамилия  <br/> |Полный  <br/> |Иван0Макаренко#  <br/> |Иван Макаренко  <br/> |
|Фамилия + Имя  <br/> |Полный  <br/> |Макаренко0Иван#  <br/> |Иван Макаренко  <br/> |
|Имя  <br/> |Полный  <br/> |Иван#  <br/> |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко"  <br/> |
|Фамилия  <br/> |Полный  <br/> |Макаренко#  <br/> |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Макаренко"  <br/> |
|Имя или фамилия  <br/> |Частичный  <br/> |Ма#  <br/> |Нажмите 1 для связи с пользователем "Мария Макаренко"  <br/> Нажмите 2 для связи с пользователем "Мария Петрова"  <br/> Нажмите 3 для связи с пользователем "Иван Марченко"  <br/> |
|Имя + Фамилия  <br/> |Частичный  <br/> |Ма0Иван#  <br/> |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко"  <br/> |
|Фамилия + Имя  <br/> |Частичный  <br/> |Ма0Ив#  <br/> |Нажмите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите 2 для связи с пользователем "Иван Марченко"  <br/> |
   
При поиске пользователей с помощью клавиатурной панели телефона можно использовать некоторые специальные знаки. Например, клавиша решетки (#) позволяет завершить ввод имени. Клавиша 0 (ноль) используется для ввода пробела между именами. Клавиша звездочки (*) позволяет повторно воспроизвести список имен пользователей, соответствующих условиям поиска.
  
|||
|:-----|:-----|
|**Специальный знак на клавиатуре телефона** <br/> |**Описание** <br/> |
|# (клавиша решетки)  <br/> |Завершение ввода имени.  <br/> |
|0 (ноль)  <br/> |Пробел между именами.  <br/> |
|* (клавиша звездочки)  <br/> |Повторное воспроизведение списка имен пользователей, соответствующих условиям поиска.  <br/> |
   
### Звонок по имени  распознавание имен при голосовом вводе

Вызывающие абоненты могут находить пользователей вашей организации с помощью голосового ввода (функция распознавания речи). Чтобы найти нужного сотрудника в справочнике Active Directory компании достаточно произнести имя. При голосовом вводе поддерживаются различные форматы поиска, в том числе: имя, фамилия, имя + фамилия или фамилия + имя.
  
Если включить распознавание речи для автосекретаря, по-прежнему будет доступен ввод с клавиатурной панели телефона (DTMF), что позволяет использовать оба способа параллельно. Ввод с клавиатурной панели телефона нельзя отключить. Этот способ будет доступен всегда, даже если включена функция распознавания речи автосекретаря.
  
Как и при вводе с клавиатурной панели телефона, если найдено несколько пользователей, выдается список имен для выбора.
  
Вызывающие абоненты могут произносить имена в следующих форматах.
  
|||||
|:-----|:-----|:-----|:-----|
|**Произнесенное имя** <br/> |**Тип поиска** <br/> |**Пример** <br/> |**Результат поиска** <br/> |
|Имя + фамилия  <br/> |Полный  <br/> |Иван Макаренко  <br/> |Иван Макаренко  <br/> |
|Фамилия + Имя  <br/> |Полный  <br/> |Макаренко Иван  <br/> |Иван Макаренко  <br/> |
|Имя  <br/> |Полный  <br/> |Иван  <br/> |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Иван Петров"  <br/> |
|Фамилия  <br/> |Полный  <br/> |Макаренко  <br/> |Нажмите или произнесите 1 для связи с пользователем "Иван Макаренко"  <br/> Нажмите или произнесите 2 для связи с пользователем "Сергей Макаренко"  <br/> |
   
> [!NOTE]
> Может потребоваться до 36 часов для нового пользователя могут иметь их имя отображалось в каталоге когда кто-то использует набора по имени с программой распознавания речи.

### Переключение на оператора

Необязательная возможность переключения на оператора, реализованная в функции автосекретаря, позволяет вызывающему абоненту связаться с сотрудником компании.
  
По умолчанию эта возможность активируется с помощью клавиши 0 или голосовой команды "Оператор", которая доступна на всех языках, поддерживаемых функцией распознавания речи.
  
> [!NOTE]
> При необходимости можно назначить команду **Оператор** другой клавише в разделе **Параметры меню**. 
  
В качестве оператора могут выступать следующие пользователи.
  
- Скайп для бизнеса Online пользователь, имеющий лицензия **Телефонную систему**, включена Корпоративная голосовая связь и планов вызова, назначенные им. Можно сделать его пользователь, запрашивающий можно отправить на голосовую почту. Чтобы сделать это, выберите **пользователя в вашей компании** и задайте этого человека звонки будут перенаправляться автоматически напрямую на голосовую почту.
    
    > [!NOTE]
    > Пользователи локальных решений Skype для бизнеса Server 2015 или Lync Server 2013 и 2010 не могут использоваться как операторы. 
  
- Другой настроенный в организации автосекретарь.
    
- Любая настроенная в организации очередь звонков. Дополнительные сведения об очередях звонков см. в разделе [Создание очереди телефонной системой звонка](create-a-phone-system-call-queue.md).
    
### Рабочие часы и обработка звонков

Рабочие часы настраиваются отдельно для каждого автосекретаря. Если рабочие часы не настроены, по умолчанию устанавливается круглосуточный график работы без выходных дней. Вы можете настроить перерывы в течение рабочего дня. Все часы, которые не настроены как рабочие, считаются внеурочными. Вы можете настроить различные правила обработки входящих звонков, а также разные приветствия (необязательно) для рабочего и внеурочного времени.
  
Для каждого автосекретаря можно настроить правила обработки звонков.
  
- Можно настроить отключение звонка сразу после воспроизведения приветствия.
    
- Также вы можете:
    
  - Перенаправить звонок на Скайп для бизнеса Online пользователь, имеющий лицензия **Телефонную систему**, включена Корпоративная голосовая связь или вызова планы назначила их. Можно сделать его пользователь, запрашивающий можно отправить на голосовую почту. Чтобы сделать это, выберите **пользователя в вашей компании** и задайте этого человека звонки будут перенаправляться автоматически напрямую на голосовую почту.
    
    > [!NOTE]
    > Локальные пользователи, использующие версии Skype для бизнеса 2015, Lync Server 2013 и 2010, не поддерживаются. 
  
  - Перенаправление звонка в очередь звонков. Дополнительные сведения об очередях звонков см. в разделе [Создание очереди телефонной системой звонка](create-a-phone-system-call-queue.md).
    
  - Перенаправление звонка другому настроенному автосекретарю.
    
- Создание параметров меню и воспроизведение подсказок для абонента. Пример: "Для связи с отделом продаж нажмите 1. Для связи с отделом обслуживания нажмите 2. Чтобы связаться с оператором, нажмите 0".
    
### Параметры меню

Автосекретари телефонной системой позволяют создать приглашениям меню («1 клавишу для продаж, нажмите клавишу 2 для служб») и настроить параметры меню, в зависимости от того, что пользователь выбирает вызовы маршрута. Настройка параметров меню для автосекретаря позволяет организации для предоставления интерактивные руководства для доступа к назначению быстрее, пользователя без использования человека оператор должно обрабатывать входящие звонки. Приглашениям меню может быть создан с использованием преобразования текста в речь (системой приглашения) или загрузив звуковой файл, который был указан. Распознавание речи голосовые команды использует для громкой навигации, но людей, позвонив также можно использовать меню Переход на клавиатуре телефона.
  
Функции клавиш с 0 по 9 можно назначить в разделе **Параметры меню** автосекретаря в Центре администрирования Skype для бизнеса. Можно настроить разные наборы параметров меню для рабочего и внеурочного времени. Кроме того, в разделе **Параметры меню** можно включить и отключить функцию звонка по имени. Можно также назначить клавиши для передачи звонков следующим адресатам.
  
- Оператор (по умолчанию назначается клавиша 0). При необходимости можно назначить любую другую клавишу или удалить оператора из меню.
    
- Очередь звонка.
    
- Другой автосекретарь. Можно создавать многоуровневые меню, задавая в разделе **Параметры меню** перенаправление другому автосекретарю с собственным наборов параметров меню (вложенная реализация функций автосекретаря).
    
- Скайп для бизнеса Online пользователь, имеющий лицензия **Телефонную систему**, не включена Корпоративная голосовая связь или имеет вызова планы назначенные им... Можно сделать его пользователь, запрашивающий можно отправить на голосовую почту. Чтобы сделать это, выберите **пользователя в вашей компании** и задайте автоматически переадресацию напрямую на голосовую почту звонков этого человека.
    
    > [!NOTE]
    > Пользователи локальных решений Skype для бизнеса Server 2015, Lync Server 2013 и 2010 не могут использоваться в **параметрах меню**. 
  
Если включена функция распознавания речи, название каждого параметра меню будет выступать в качестве ключевого слова. Например, пользователь может произнести "Один" для выбора параметра меню, назначенного клавише 1, или "Отдел продаж" для перехода к одноименному параметру меню.
  
Дополнительные сведения о настройке автосекретаря и параметрах меню см. в [Настройка телефонной системой автосекретаря](set-up-a-phone-system-auto-attendant.md).
  
### Получение номеров служб для автосекретаря

Прежде чем создавать и настраивать функцию автосекретаря, необходимо получить или перенести существующие платные и бесплатные номера служб. Полученные платные и бесплатные номера служб будут отображаться в **Центре администрирования Skype для бизнеса** на вкладке **Голосовая связь** > **Номера телефонов**. При этом в поле **Тип номера** для них будет указано **Служба  бесплатная**. Дополнительные сведения о получении номеров служб см. в разделе [Получение номеров телефонов служб для Skype для бизнеса](getting-service-phone-numbers-for-skype-for-business-and-microsoft-teams.md). Сведения о переносе существующих номеров служб см. в разделе [Передача номеров телефонов в Office 365](../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md).
  
> [!NOTE]
> Если вы находитесь за пределами США, вы не сможете использовать Центр администрирования Skype для бизнеса для получения номеров служб. В этом случае следуйте инструкциям, приведенным в [Управление номерами телефонов для организации](../what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md). 
  
## Изменение идентификатор звонящего пользователя следует очереди звонка телефонный номер

Удостоверение пользователя можно защитить, изменив свой идентификатор звонящего для исходящих звонков очередь звонка вместо путем создания политики, используя командлет **New-CallingLineIdentity**.
  
Для этого сеанса:
  
```
New-CsCallingLineIdentity -Identity "UKSalesQueue" -CallingIdSubstitute "Service" -ServiceNumber 14258828080 -EnableUserOverride $False -Verbose
```

Нажмите примените политику для пользователя с помощью командлета **Grant-CallingLineIdentity**. Для этого сеанса:
  
```
Grant-CsCallingLineIdentity -PolicyName UKSalesQueue -Identity "AmosMarble@contoso.com"
```

Дополнительные сведения о том, как изменить параметры идентификатор звонящего в вашей организации можно получить [Как можно использовать идентификатор абонента в организации?](../what-are-calling-plans-in-office-365/how-can-caller-id-be-used-in-your-organization.md).
