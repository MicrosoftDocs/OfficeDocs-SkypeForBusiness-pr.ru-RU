# Создание очереди телефонной системой звонка

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Вызов телефона системные очереди включать приветствия, которые используются, когда кто-то звонит номер телефона для вашей организации, вы можете автоматически удержания звонков и выполните поиск для следующей агента доступных звонка обработка звонков при людей с нарушениями звонка слушать музыку при удержании. Можно создавать очереди одного или нескольких звонка для вашей организации.
  
Можно предоставить звонка телефона системные очереди:
  
- Приветствие звонящим в организацию пользователям.
    
- Воспроизведение музыки во время удержания звонка.
    
- Переадресация звонков операторам с использованием почтовых списков рассылки и групп безопасности.
    
- Настройка максимального размера времени ожидания и правил обработки звонков для очереди.
    
При поступлении звонка на номер, для которого настроена очередь звонков, сначала воспроизводится приветствие (если настроено), после чего звонок помещается в очередь, где ожидает ответа следующего освободившегося оператора. Во время удержания звонка воспроизводится музыка. Звонки переадресуются операторам  *в порядке поступления*  .
  
Все ожидающие в очереди звонки распределяются согласно правилам маршрутизации автосекретаря.
  
- Первый звонок в очереди поступает одновременно всем операторам.
    
    > [!NOTE]
    > Звонки не поступают операторам с состоянием присутствия **Не в сети** или **Не беспокоить**. 
  
- Одновременно операторам отправляется только одно уведомление о звонке, находящемся в начале очереди.
    
- После того, как оператор принимает звонок, всем операторам поступает следующий входящий звонок из очереди.
    
## Шаг 1. Начало работы

Перед началом работы с очередями звонков необходимо запомнить следующие моменты.
  
- Корпоративный E3 плюс **Телефонной системой** лицензии или E5 корпоративные лицензии вашей организации должен иметь (как минимум). Количество лицензий пользователей **Телефонную систему**, назначенных последствий, что число службы номера, доступны для использования очереди звонка. Количество очередях звонок, может иметь зависит от номера **Телефона системы** и **Аудиоконференций** лицензий, которые назначаются в вашей организации. Чтобы узнать больше о лицензировании,[Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
    
    > [!NOTE]
    > Перенаправление звонков для пользователей в вашей организации, которые Online, они должны иметь лицензию **Телефонную систему** и включена Корпоративная голосовая связь или имеют вызова планов Office 365. Просмотреть[Назначение лицензий Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/assign-skype-for-business-and-microsoft-teams-licenses.md). Чтобы включить их Корпоративная голосовая связь, можно использовать Windows PowerShell. Пример запуска:  `Set-CsUser -identity "Amos Marble" -EnterpriseVoiceEnabled $true`
  
- Дополнительные сведения о вызове планы Office 365, читайте в статье [Что такое планы звонков в Office 365?](../what-are-calling-plans-in-office-365/what-are-calling-plans-in-office-365.md) и[Тарифные планы для Office 365](../skype-for-business-and-microsoft-teams-add-on-licensing/calling-plans-for-office-365.md)
    
    > [!NOTE]
    > Локальные пользователи решений Skype для бизнеса Server 2015 или Lync Server 2013 и 2010 не могут выступать в качестве операторов очереди звонков. 
  
- Можно назначить только платный и номера телефонов бесплатный службы у вас в **Скайп для центра администрирования бизнес** или перенесена у другого поставщика услуг телефонной системой вызовите очереди. Получение и использовать службы бесплатные номера, необходимо настроить кредиты связи. Кроме того Если вы хотите очереди вызов для исходящих номер ТСОП (вместо переадресацию вызова на пользователя), исходящие вызовы ТСОП выставляются минуту с помощью обращений кредиты независимо от исходящий звонок местные и международные телефонный номер. Сделать это, читайте в статье[Что такое кредиты на связь?](../skype-for-business-and-microsoft-teams-add-on-licensing/what-are-communications-credits.md) и[Настройка кредитов на связь для организации](../skype-for-business-and-microsoft-teams-add-on-licensing/set-up-communications-credits-for-your-organization.md).
    
    > [!NOTE]
    > Номера телефонов пользователей (абонентов) нельзя назначить очередям звонков  можно использовать платные или бесплатные номера телефонов служб. 
  
- При распределении входящие звонки из очереди звонка телефонную систему, для вызова агентов поддерживаются этих клиентов.
    
  - Клиент Skype для бизнеса для настольных ПК 2016 (32- и 64-разрядные версии)
    
  - Клиент Lync для настольных ПК 2013 (32- и 64-разрядные версии)
    
  - Все модели IP-телефонов, поддерживаемые приложением Skype для бизнеса Online. См. раздел [Телефоны, поддерживаемые в Skype для бизнеса Online](getting-phones-for-skype-for-business-online/getting-phones-for-skype-for-business-online.md).
    
  - Mac Скайп для клиента Business (версия 16.8.196 и более поздние версии)
    
  - Android Скайп для клиента Business (версия 6.16.0.9 и более поздние версии)
    
  - iPhone Скайп для клиента Business (версия 6.16.0 и более поздние версии)
    
  - iPad Скайп для клиента Business (версия 6.16.0 и более поздние версии)
    
## Шаг 2. Получение или перенос платных или бесплатных номеров телефонов служб

Прежде чем создавать и настраивать очереди звонков, необходимо получить или перенести существующие платные и бесплатные номера служб. Полученные платные и бесплатные номера служб будут отображаться в **Центре администрирования Skype для бизнеса** на вкладке **Голосовая связь** > **Номера телефонов**. При этом в поле **Тип номера** для них будет указано **Служба  бесплатная**. Дополнительные сведения о получении номеров служб см. в разделе [Получение номеров телефонов служб для Skype для бизнеса](getting-service-phone-numbers-for-skype-for-business-and-microsoft-teams.md). Сведения о переносе существующих номеров служб см. в разделе [Передача номеров телефонов в Office 365](../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md).
  
> [!NOTE]
> Если вы находитесь за пределами США, вы не сможете использовать Центр администрирования Skype для бизнеса для получения номеров служб. В этом случае следуйте инструкциям, приведенным в [Управление номерами телефонов для организации](../what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).

## Изменение идентификатор звонящего пользователя следует очереди звонка телефонный номер

Удостоверение пользователя можно защитить, изменив свой идентификатор звонящего для исходящих звонков очередь звонка вместо путем создания политики, используя командлет **New-CallingLineIdentity**.
  
Для этого сеанса:
  
```
New-CsCallingLineIdentity -Identity "UKSalesQueue" -CallingIdSubstitute "Service" -ServiceNumber 14258828080 -EnableUserOverride $False -Verbose
```

Нажмите примените политику для пользователя с помощью командлета **Grant-CallingLineIdentity**. Для этого сеанса:
  
```
Grant-CsCallingLineIdentity -PolicyName UKSalesQueue -Identity "AmosMarble@contoso.com"
```

Дополнительные сведения о том, как изменить параметры идентификатор звонящего в вашей организации можно получить [Как можно использовать идентификатор абонента в организации?](../what-are-calling-plans-in-office-365/how-can-caller-id-be-used-in-your-organization.md).
