# Вот что можно получить с телефонной системой в Office 365

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](bc9756d1-8a2f-42c4-98f6-afb17c29231c.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/bc9756d1-8a2f-42c4-98f6-afb17c29231c), в качестве справочного материала. 
  
УАТС является телефонную систему в пределах предприятия. Телефонную систему в надстройки Office 365 обеспечивает возможности АТС, но без сложной и дорогие оборудования.
  
С телефонной системой надстройку Skype для бизнеса можно использовать для выполнения основных задач, например выполнение и прием звонков, передача вызовы и отключение звука или unmuting вызовов из практически где угодно с помощью подключение к Интернету. Доступен по всему миру!
  
Дополнительные сведения читайте в статье [цены для телефонной системой](https://products.office.com/en-us/skype-for-business/cloud-pbx#requirements).

## Планирование и настройка телефонной системой в Office 365
<a name="bkmk_setup"> </a>

Настройка телефонной системой способом, наилучшим образом подходящее вашего бизнеса:
  
- С помощью надстройки телефонной системой с существующей телефонной линии вашей компании или службе [Что такое планы звонков в Office 365?](../what-are-calling-plans-in-office-365/what-are-calling-plans-in-office-365.md) для входящих и исходящих звонков.
    
- Вы можете получить новые номера телефонов из Office 365 или перенести существующие номера в Skype для бизнеса. 
    
Просмотреть эти статьи для планирования и настройка телефонной системой в вашей организации:
  
- [Настройка планов звонков](../what-are-calling-plans-in-office-365/set-up-calling-plans.md) : Эта статья содержит пошаговые инструкции для настройки телефонные номера или для переноса существующей номера телефонов Skype для бизнеса.
    
- [Планирование системы телефона в Office 365 (облачная АТС) решение](https://go.microsoft.com/fwlink/?LinkId=708645): все параметры развертывания размещает TechNet этой статьи. Имеются сведения о планировании доступным только для облачных решений.
    
- [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md) : дополнительные лицензии, вам нужно купить, чтобы получить телефонной системой в Office 365.
    
- [Настройка голосовой почты телефонной системы  справка для администратора](phone-system-voicemail/set-up-phone-system-voicemailadmin-help.md)
    
- [Знакомство со Skype для бизнеса](https://support.office.com/article/8a3491a3-c095-4718-80cf-cbbe4afe4eba). Помогите сотрудникам своей компании найти информацию по использованию возможностей Skype для бизнеса.
