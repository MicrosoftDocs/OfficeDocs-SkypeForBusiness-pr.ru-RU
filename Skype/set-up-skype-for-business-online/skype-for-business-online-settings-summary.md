# <a name="skype-for-business-online-settings-summary"></a>Общие сведения о параметрах Skype для бизнеса Online

**Скайп по центру администрирования Business** включает параметры для всей организации, а также для отдельных пользователей. 
  
> [!NOTE]
>  В зависимости от расположения может не иметь доступ ко всем возможностям Скайп для бизнеса в Интернет. Для получения дополнительных сведений обратитесь к разделу [ограничения лицензии для Office 365](https://go.microsoft.com/fwlink/?LinkId=529483).
