# Настройка режима конфиденциальности присутствия

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](b33d57fe-b9cf-43c1-961a-edf28db738e8.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/b33d57fe-b9cf-43c1-961a-edf28db738e8), в качестве справочного материала. 
  
Параметра присутствия Skype для бизнеса online дает пользователям более точно тех, кто может видеть ли они доступны, во время собрания или нет на работе. Дополнительные сведения о присутствии Skype для бизнеса и параметры конфиденциальности просмотреть [Настройка присутствия в Skype для бизнеса Online](configure-presence-in-skype-for-business-online.md).
  
## Выбор параметров присутствия по умолчанию для всех пользователей в организации
<a name="__top"> </a>

1. Перейдите в раздел "Центр администрирования Skype для бизнеса online" > **"Организация" > "Общее"**.
    
2. В разделе **Режим конфиденциальности присутствия** выберите параметр, затем нажмите кнопку **Сохранить**.
    
|****Параметр****|****Кто может просматривать состояние присутствия пользователя****|
|:-----|:-----|
|**Автоматически отображать сведения о присутствии** <br/> |Любой пользователь Skype для бизнеса, который не входит в группы конфиденциальности **Внешние** или **Заблокированные**.  <br/> |
|**Отображать сведения о присутствии только для контактов пользователя** <br/> |Любой пользователь в списке контактов, который не входит в группы конфиденциальности **Внешние** или **Заблокированные**.  <br/> Отдельные пользователи могут менять эту настройку в диалоговом окне Skype для бизнеса **Параметры** <br/> |
