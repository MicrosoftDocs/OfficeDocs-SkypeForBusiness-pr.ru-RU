# <a name="configure-presence-privacy-mode"></a>Настройка режима конфиденциальности для сведений о присутствии

Скайп для бизнеса в Интернет состояние присутствия дает людям лучше контролировать, кто может просматривать ли они доступны, в ходе собрания или вне офиса. Для получения дополнительных сведений о Скайп для бизнеса сведения о присутствии и параметры конфиденциальности видеть [присутствие Настройка в Скайп для бизнеса в Интернет](configure-presence-in-skype-for-business-online.md). 
  
## <a name="choose-the-default-online-presence-setting-for-everyone-in-your-organization"></a>Нажмите кнопку присутствие в сети по умолчанию для всех пользователей в вашей организации
<a name="__top"> </a>

1. Последовательно выберите пункты Скайп для бизнеса в Интернет Центр администрирования > **организация > общий**.
    
2. В разделе **режим конфиденциальности присутствия**выберите параметр и нажмите кнопку **Сохранить**.
    
|**Параметр**|**Кто может просматривать состояние присутствия пользователя**|
|:-----|:-----|
|**Автоматически отображать сведения о присутствии** <br/> |Любой пользователь Skype для бизнеса, который не входит в группы конфиденциальности **Внешние** или **Заблокированные**. <br/> |
|**Отображать сведения о присутствии только для контактов пользователей** <br/> |Всем пользователям в список контактов другого пользователя, кто не принадлежит к группе конфиденциальности **внешний** или **заблокирован** . <br/> Отдельные пользователи могут изменить этот параметр в Скайп для бизнеса **Параметры** диалогового окна. <br/> |
