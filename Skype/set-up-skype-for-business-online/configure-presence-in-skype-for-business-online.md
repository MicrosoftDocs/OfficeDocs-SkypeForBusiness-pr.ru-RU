# Настройка присутствия в Skype для бизнеса Online

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
По умолчанию все, кто может общаться с пользователем вашей организации в Skype для бизнеса, также могут видеть сведения о его присутствии в сети. В Skype для бизнеса отображаются такие состояния присутствия, как "в сети", "на собрании", "не в сети" и другие. 
  
![An example of a person's online status in Skype for Business.](../images/f0849132-1ddb-480f-bca6-cfe9eaa0486d.png)
  
Если вы являетесь **[Назначение ролей администратора в Office 365](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** для других пользователей вашей организации, вы можете указать, кому будут видны их сведения о присутствии в Skype для бизнеса.
  
1. Перейдите в раздел **Центр администрирования Office 365** > **центры администрирования** > **Скайп для бизнеса**.
    
2. В **центре администрирования Skype для бизнеса** выберите **организации**.
    
3. В списке **Режим конфиденциальности присутствия** выберите один из следующих параметров и нажмите **Сохранить**.
    
|****Параметр****|****Кто может просматривать состояние присутствия пользователя****|
|:-----|:-----|
|**Автоматически отображать сведения о присутствии** <br/> |Любой пользователь Skype для бизнеса вашей организации, не добавленный в список **внешних** или **заблокированных** контактов другого пользователя, сможет видеть сведения о его присутствии. <br/> |
|**Отображать сведения о присутствии только для контактов пользователя** <br/> |Любой пользователь в списке контактов другого пользователя, не добавленный в список **Внешние** или **Заблокированные**. <br/> Пользователей можно переопределить параметры по умолчанию в приложении их Skype для бизнеса: **настройки** > **Сервис** > **Параметры**. <br/> |
   
Информацию о том, какие параметры пользователи могут изменить в Skype для бизнеса, см. в следующих статьях: 
  
- [Управление доступом к сведениям о присутствии в Skype для бизнеса](https://support.office.com/article/fea86e34-60cf-4dd0-bfb2-169a42afd92c)
    
- [Настройка параметров статуса в Skype для бизнеса](https://support.office.com/article/efd25395-c8ef-4510-b9cb-6f70e2fff8a0)
