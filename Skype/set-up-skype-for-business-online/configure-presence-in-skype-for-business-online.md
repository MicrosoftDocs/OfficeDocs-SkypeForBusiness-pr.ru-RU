# <a name="configure-presence-in-skype-for-business-online"></a>Настройка присутствия в Skype для бизнеса Online

По умолчанию всем пользователям, которые могут взаимодействовать с одним из людей в организации с помощью Скайп для бизнеса можно также просмотреть ли он находится в сети. Скайп для бизнеса показывает, является ли пользователь доступен через Интернет, в собрании, автономный режим, или другой индикатор. 
  
![Пример сети состояние пользователя в Скайп для бизнеса.](../images/f0849132-1ddb-480f-bca6-cfe9eaa0486d.png)
  
В качестве **[администратора](http://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** для всех пользователей в организации можно выбрать, кто видит их присутствие в сети в Скайп для бизнеса.
  
1. Перейдите в **Центр администрирования Office 365** > **центры администрирования** > **Скайп для бизнеса**.
    
2. В **Скайп по центру администрирования бизнеса**выберите **организации**.
    
3. В разделе **режим конфиденциальности присутствия**выберите один из следующих параметров и нажмите кнопку **Сохранить**.
    
|**Параметр**|**Кто может просматривать состояние присутствия пользователя**|
|:-----|:-----|
|**Автоматически отображать сведения о присутствии** <br/> |Любой Скайп для бизнес-пользователя в бизнесе, который не был добавлен в другого пользователя **внешний** или списка **заблокированных** смогут видеть его присутствие в сети. <br/> |
|**Отображать сведения о присутствии только для контактов пользователей** <br/> |Всем пользователям в список контактов другого пользователя, пользователей, которые не добавлены в список **внешних** или **заблокирован** . <br/> Отдельных пользователей можно переопределить параметры по умолчанию в их Скайп для бизнес-приложение: **Параметры** > **средств** > **Параметры**. <br/> |
   
Сведения о вашей пользователи могут изменять в Скайп для бизнеса см в этих статьях:
