# Включение и отключение уведомлений на мобильном телефоне

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Как **[Назначение ролей администратора в Office 365](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** вашей организации вы можете ли Skype для бизнеса пользователям получать оповещения о входящих и пропущенных мгновенных сообщений, когда они находятся на их мобильных телефонов и планшетных ПК.
  
На телефонах с операционными системами Android и Windows Phone уведомления Skype для бизнеса принимаются в реальном времени. Однако на устройствах Windows Phone, iPhone и iPad для отправки оповещений используются push-уведомления, если вы редко используете Skype для бизнеса на данном телефоне или планшетном компьютере.
  
## Отключение push-уведомлений для всех устройств Windows Phone или Apple в организации
<a name="__top"> </a>

1. Войдите в Office 365.
    
2. Перейдите в **Центр администрирования Office 365** > **Скайп для бизнеса**.
    
3. Откройте меню **Организация**, затем выберите **Общие**.
    
4. В разделе **Уведомления на мобильном телефоне** снимите флажок в поле рядом со службой уведомлений, которую необходимо отключить, а затем нажмите кнопку **Сохранить**.
    
Помните:
  
- если отключить push-уведомления, пользователи получат все оповещения при следующем запуске Skype для бизнеса на мобильном устройстве;
    
- по умолчанию push-уведомления включены и могут быть отключены отдельными пользователями с помощью соответствующего параметра Skype для бизнеса на мобильном устройстве;
    
- если вы отключите push-уведомления, то пользователи не смогут их включить.
    
> [!IMPORTANT]
> Корпорация Майкрософт использует для предоставления мобильных уведомлений для Windows Phone, iPhone и iPad пользователей в режиме реального времени Skype для бизнеса других компаний. Просмотр [заявления о конфиденциальности для Microsoft Скайп для бизнеса продуктов](https://go.microsoft.com/fwlink/p/?linkid=247732).
