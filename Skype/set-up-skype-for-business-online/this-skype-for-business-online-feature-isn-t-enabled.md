# Функция Skype для бизнеса Online отключена

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Похоже, у пользователя нет лицензии на этот компонент.
  
|**Возможная причина**|**Предлагаемое действие**|
|:-----|:-----|
|У вас есть доступные лицензии, но не активировали компонент для этого пользователя.  <br/> |Перейдите в **Центр администрирования Office 365** > **Активные пользователи** могут предоставить пользователю лицензию. <br/> |
|Пользователь находится в регионе, где этот компонент недоступен.  <br/> |Если это возможно, перейдите в **Центр администрирования Office 365** > **Активные пользователи** и назначить данному пользователю в место, где эта функция доступна. <br/> |
|Компонент не активирован для организации.  <br/> |Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса** и активируйте компонент для всей организации. <br/> |
