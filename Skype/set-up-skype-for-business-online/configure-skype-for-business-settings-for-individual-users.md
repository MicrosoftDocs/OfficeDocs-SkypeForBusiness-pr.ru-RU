# Администраторам: настройка параметров Skype для бизнеса для отдельных пользователей

Эта статья описывает, как администраторы могут настроить Skype для бизнеса для небольшого числа пользователей. Для многократного выполнения этих действий мы включили ссылки на командлеты Windows PowerShell.
  
Сведения о том, как разрешить (или запретить) всем сотрудникам организации взаимодействовать с внешними пользователями, см. в следующих статьях:
  
- [Разрешите пользователям связываться с внешними пользователями Skype для бизнеса](allow-users-to-contact-external-skype-for-business-users.md) : пользователям вашей организации можно предоставить права на использование расширенных возможностей Skype для бизнеса (совместное использование рабочих столов, проверка сетевого статуса), которые позволяют взаимодействовать с пользователями в определенных доверенных (федеративных) доменах. В этой статье также объясняется, как заблокировать взаимодействие для определенных доменов.
    
- [Разрешение на добавление контактов Skype пользователям Skype для бизнеса](let-skype-for-business-users-add-skype-contacts.md) . Пользователи в вашей организации могут получить разрешение на использование Skype для бизнеса для поиска людей, работающих с бесплатным приложением Skype, и обмена мгновенными сообщениями с ними.
    
## Настройка общих параметров для одного пользователя
<a name="__toc325019204"> </a>

Чтобы выполнять эти действия, вам нужны [Роли администраторов в Office 365](https://support.office.com/article/da585eea-f576-4f55-a1e0-87090b6aaa9d).
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центры администрирования** > **Skype для бизнеса**. 
    
3. Выберите **Пользователи**.
    
    ![In the Skype for Business admin center, choose Users.](../images/7c80eeb3-6555-4fc8-91f4-61b493581e9e.png)
  
4. Выберите, каких пользователей нужно изменить. 
    
5. В области справа выберите **Изменить**.
    
    ![Choose the edit icon.](../images/5dd7c5bc-b8fa-4201-b6a6-1436ad8f88fb.png)
  
6. На странице параметров **Общие** установите или снимите флажки рядом с функциями, которые нужно изменить, а затем нажмите кнопку **Сохранить**.
    
|**Параметр**|**Подробные сведения**|
|:-----|:-----|
|Аудио и видео высокой четкости  <br/> |Разрешить этому участнику записывать аудиособрания, аудио- и видеособрания или запретить планировать собрания.  <br/> |
|Запись бесед и собраний  <br/> |Выбор событий, которые можно записывать этому участнику.  <br/> Этот параметр недоступен в Skype для бизнеса basic.  <br/> |
|Отключение функций, данные которых не архивируются, для обеспечения соответствия требованиям  <br/> | Выберите этот параметр, если требуется сохранять данные в электронной форме. <br/>  Этот параметр отключает функции, данные которых не сохраняются, когда в Центре администрирования Exchange настроено[удержание на месте](https://technet.microsoft.com/en-us/library/ff637980%28v=exchg.150%29.aspx). Это следующие функции:  <br/>  Передача файлов через службу мгновенных сообщений <br/>  Общие страницы OneNote <br/>  Примечания PowerPoint <br/> |
   
Для массовой настройки этих параметров используйте PowerShell. См. статью [Управление политиками в Skype для бизнеса Online](https://technet.microsoft.com/en-us/library/dn362826%28v=ocs.15%29.aspx).
  
## Блокировка внешней связи
<a name="__toc325019206"> </a>

После [Разрешение на добавление контактов Skype пользователям Skype для бизнеса](let-skype-for-business-users-add-skype-contacts.md) для всех пользователей в организации вы можете выборочно отключить функцию внешней связи для отдельных пользователей, выполнив приведенные далее действия.
  
1. Выберите пункт **Пользователи**, выделите пользователей, параметры которых нужно отключить, и нажмите **Изменить**![Изменить](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png).
    
2. Выберите пункт **Внешняя связь** и отмените выбор указанных ниже параметров.
    
  - **Внешние пользователи Skype для бизнеса**: снимите этот флажок, чтобы пользователь не мог взаимодействовать с пользователями Skype для бизнеса в федеративных доменах.
    
  - **Внешние пользователи Skype**: снимите этот флажок, чтобы пользователь не мог взаимодействовать с пользователями бесплатного приложения Skype.
    
3. Нажмите кнопку **Сохранить**.
    
Чтобы настроить эти параметры массово, используйте PowerShell. См. статью [Управление связью с внешними пользователями и организациями в Skype для бизнеса Online](https://technet.microsoft.com/en-us/library/dn362813%28v=ocs.15%29.aspx).
  
## Изменение параметров аудиоконференции для одного пользователя
<a name="__toc314837483"> </a>

1. Выберите пункт **Пользователи**, выделите пользователя, параметры аудиоконференции для которого необходимо изменить, и нажмите **Изменить**![Изменить](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png).
    
2. Выберите **Аудиоконференции**, затем выберите поставщика услуг аудиоконференций, введите или измените запрошенные данные, после чего щелкните **Сохранить**.
    
|**Параметр аудиоконференции**|**Описание**|
|:-----|:-----|
|**Имя поставщика** <br/> |Выберите поставщика из списка.  <br/> |
|**Платный номер** (обязательно) <br/> |Для сторонних поставщиков услуг аудиоконференций это телефонные номера, которые были получены от поставщика услуг аудиоконференций. Если в качестве такого поставщика выступает Майкрософт, то это номера, которые настроены для моста аудиоконференций. Измените формат номеров с соответствии с требованиями к собранию в Skype для бизнеса и Microsoft Teams.  <br/> |
|**Бесплатный номер** <br/> |Для сторонних поставщиков услуг аудиоконференций это телефонные номера, которые были получены от поставщика услуг аудиоконференций. Если в качестве такого поставщика выступает Майкрософт, то это номера, которые настроены для моста аудиоконференций. Измените формат номеров с соответствии с требованиями к собранию в Skype для бизнеса и Microsoft Teams.  <br/> |
|**Идентификатор конференции и ПИН-код** (обязательно) <br/> |ПИН-код участника (или код конференции) используется для присоединения к собраниям, запланированным этим пользователем, и предоставляется сторонним поставщиком услуг аудиоконференций. Если в качестве поставщика услуг аудиоконференций используется Майкрософт, ПИН-код не потребуется.  <br/> |
   
Для массовой настройки этих параметров используйте PowerShell. См. статью [Определение телефонных номеров для аудиоконференций, которые организаторы собраний могут включить в приглашения](../audio-conferencing-in-office-365/set-the-audio-conferencing-phone-numbers-for-meeting-organizers-that-are-include.md).
