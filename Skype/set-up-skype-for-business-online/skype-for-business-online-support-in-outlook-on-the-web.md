# Поддержка Skype для бизнеса Online в веб-приложении Outlook

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](305984ec-3da8-4509-bb2b-6643dcf2cb7d.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/305984ec-3da8-4509-bb2b-6643dcf2cb7d), в качестве справочного материала. 
  
Outlook в Интернете (Outlook Web App) в Office 365 предлагает основные Скайп для бизнеса веб-клиента на панели навигации. Этот простой клиент доступен для активные пользователи, чьи администрирования еще не настроен собственном URL-адрес для своей организации Office 365. Как учетную запись пользователя находится в оперативном режиме и не имеет собственном URL-адрес, по-прежнему увидят качества даже если описаны некоторые учетные записи пользователей, которые являются размещенные локального своей организации. Пользователи, у пользователя учетные записи локального (есть ли у них собственном URL-адрес или нет) или под управлением Microsoft увидите в интерфейсе Lync в Outlook web app.
  
В таблице ниже перечислены различные параметры настройки, которые могут интересовать и веб-клиент, который используется.
  
||||
|:-----|:-----|:-----|
|**Учетная запись пользователя находится** <br/> |**Запоминающийся URL-адрес настроен, или существует выделенная организация** <br/> |**Интерфейс Skype для бизнеса или Lync?** <br/> |
|В Интернете  <br/> |Нет  <br/> |Веб-интерфейс Skype для бизнеса  <br/> |
|В Интернете  <br/> |Да  <br/> |Веб-интерфейс Lync  <br/> |
|Гибридное, но находится в сети  <br/> |Нет  <br/> |Веб-интерфейс Skype для бизнеса  <br/> |
|Гибридное, но находится в сети  <br/> |Да  <br/> |Веб-интерфейс Lync  <br/> |
|Гибридное, но размещено локально  <br/> |Нет  <br/> |Веб-интерфейс Lync  <br/> |
|Гибридное, но размещено локальное  <br/> |Да  <br/> |Веб-интерфейс Lync  <br/> |
|Чистый на prem  <br/> |Нет  <br/> |Веб-интерфейс Lync  <br/> |
|Чистый на prem  <br/> |Да  <br/> |Веб-интерфейс Lync  <br/> |
