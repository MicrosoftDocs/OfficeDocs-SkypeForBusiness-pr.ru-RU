# <a name="user-statistics-and-organization-information"></a>Сведения о пользователе статистики и организации

Скайп для панели мониторинга бизнеса дает моментальный снимок быстрого как настроить службы для бизнеса. Если вы еще не еще не настроили Скайп для бизнеса, см [Скайп для бизнеса в Интернет](set-up-skype-for-business-online.md) для получения пошаговых инструкций.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Изменение Скайп для бизнеса в Интернет Статистика пользователей

|||
|:-----|:-----|
|**Свойство** <br/> |**Как изменить его** <br/> |
|Скайп для бизнеса в Интернет пользователей  <br/> |Добавление или блокирование людей с помощью Скайп для бизнес-приложения, просмотрите [Назначение и удаление лицензий на Office 365 для бизнеса](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) или [Удаление лицензий пользователей в Office 365 для бизнеса](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Полный список действий, выполняемых при выходе пользователя из вашей организации в разделе [Удаление бывший сотрудник из Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Пользователям, включенным для конференц-связи  <br/> |Чтобы включить пользователей для конференц-связи, см [аудиоконференции для Скайп для бизнеса и группами Майкрософт](../audio-conferencing-in-office-365/set-up-audio-conferencing.md).  <br/> |
|Пользователи в облаке  <br/> |Это число Скайп для бизнеса в Интернет пользователей, имеющих учетные записи Office 365.  <br/> Чтобы добавить пользователей в подписки Office 365 (то есть, создать учетные записи Office 365 для их), просматривать, [добавлять пользователей по отдельности или набором для Office 365 — помощь администратора](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Пользователи синхронизирован и размещенных в Интернете  <br/> |Это число Скайп для бизнеса в Интернет пользователей, которые синхронизируются из локального Active Directory.  <br/> Чтобы настроить локальный Active Directory, см. [Настройка синхронизации службы каталогов для Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Если недоступен в списке, это означает, что нет данных или возникли ошибки.
