# Статистика пользователей и организации

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
В панели инструментов Skype для бизнеса представлен обзор параметров этой службы, настроенных в вашей организации. Если вы еще не настроили Skype для бизнеса, ознакомьтесь с пошаговыми инструкциями в разделе [Настройка Skype для бизнеса Online](set-up-skype-for-business-online.md). 
  
## Изменение статистики пользователей Skype для бизнеса

|||
|:-----|:-----|
|**Свойство** <br/> |**Изменение** <br/> |
|Пользователи Skype для бизнеса online  <br/> |Чтобы добавить пользователей или запретить им использовать приложение Skype для бизнеса, см. статьи [Управление лицензиями Office 365 бизнес](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) или[Remove licenses from users in Office 365 for business](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Полный перечень что делать, когда пользователь выходит за пределы вашей компании читайте в статье [Удаление бывшего сотрудника из среды Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Пользователи с доступом к конференц-связи с телефонным подключением  <br/> |Разрешить пользователям конференц-связь, читайте в статье [Настройка аудиоконференций в Skype для бизнеса и Microsoft Teams](../audio-conferencing-in-office-365/set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams.md).  <br/> |
|Пользователи в облаке  <br/> |Число пользователей Skype для бизнеса online, у которых есть учетные записи Office 365.  <br/> Сведения о добавлении пользователей в подписки на Office 365 (то есть о создании учетных записей Office 365 для них) см. в разделе [Добавление пользователей в Office 365 по одному или массово  справка для администраторов](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Синхронизированные и размещенные в сети пользователи  <br/> |Число пользователей Skype для бизнеса Online, синхронизированных в локальной службе каталогов Active Directory.  <br/> Сведения о настройке локальной службы Active Directory см. в разделе [Настройка синхронизации службы каталогов для Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Если в списке указано "Недоступно", это означает, что информация отсутствует или возникли ошибки. 
  
## Изменение сведений об организации

|||
|:-----|:-----|
|**Свойство** <br/> |**Изменение** <br/> |
|Имя организации  <br/> |Имя, введенное при регистрации в Office 365. Сведения о его изменении см. в разделе [Изменение адреса организации и других данных профиля](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ИД организации  <br/> |ИД организации нельзя изменить.  <br/> |
|Организация создана  <br/> |Это значение нельзя изменить.  <br/> |
|Статус синхронизации Active Directory  <br/> |Это, синхронизируется ли ваша организация Skype для бизнеса online организации, которая является локальной.  <br/> Сведения о настройке локальной службы Active Directory см. в разделе [Настройка синхронизации службы каталогов для Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Домены организации  <br/> |Домены, зарегистрированные при первичной настройке Office 365.  <br/> Добавление домена, читайте в статье [Добавление дополнительных доменов в Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Сведения об удалении домена см. в разделе [Удаление домена из Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
