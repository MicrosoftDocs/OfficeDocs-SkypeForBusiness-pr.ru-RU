# <a name="desktop-client-feature-comparison-for-skype-for-business-server-2019"></a>Сравнение функций для настольных компьютеров для Скайп для Business Server 2019

**Сводка:** Скайп для Business Server 2019 или Скайп для администраторов Business Online можно использовать эти таблицы понять, какие возможности поддерживаются в какие клиенты.

 Перед развертыванием или обновление до Скайп для сервера, проверьте, какие клиенты уже используются в организации. Используйте в таблицах ниже понять влияние поддержка компонента на эти клиенты. Вы также сможете сообщить об изменениях пользователям, спланировать процесс развертывания и в полной мере понять преимущества обновления до последней версии клиента.

Некоторые функции, доступные в Скайп для Business Server 2019, недоступны в Скайп для бизнеса в Интернет; в разделе [ограничения учетной записи пользователя сети или гибридного](feature-comparison.md#Online-Hybrid) особенности. Скайп для бизнеса Online администраторы могут оказаться полезными для [Скайп для описания службы Online Business](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) сведения о различных планов им.

В следующих таблицах приведены компоненты, доступные в каждом клиенте, который работает с Скайп для Business Server 2019 или Скайп для бизнеса в Интернет. Можно также ссылаться на [сравнение возможностей мобильного клиента для Скайп для бизнеса](../../SfbServer/plan-your-deployment/clients-and-devices/mobile-feature-comparison.md) для смартфонов и планшетных сравнение компонентов клиента. Приобретаемые организацией клиентские или пользовательские лицензии на подписку также влияют на доступность функций для пользователей. Развертывание полного или базового клиента зависит от лицензий или плана, которые приобретет организация. [Руководство по лицензированию](https://products.office.com/en-us/skype-for-business/it-pros) для получения дополнительных сведений см.

> [!IMPORTANT]
> Скайп для Business Server 2019 и Скайп для бизнеса Online поддерживает следующие клиенты предыдущие: Lync 2013, Скайп для бизнеса 2015 и Скайп 2016 бизнеса, а также Скайп для клиента Business 2019. Сведения об этих клиентов при использовании с другими серверами можно [таблицы сравнения клиентов для Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx) и [Сравнение функций для настольных компьютеров для Скайп для бизнеса 2015](../../SfbServer/plan-your-deployment/clients-and-devices/desktop-feature-comparison.md).

## <a name="client-limitations"></a>Ограничения клиентов 
<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Ограничения базовых клиентов
<a name="Full-Basic"> </a>

Приведенные ниже функции доступны при использовании полного клиента и недоступны при использовании базового клиента.  

- Управление параметрами групповых звонков

- Управление делегатами

- Звонки от имени другого контакта (сценарий с руководителем и делегатом)

- Обработка звонков другого пользователя делегатом

- Управление большим количеством звонков

- Выполнение звонка в группу ответа

- Парковка вызовов

- Изменение приветствия

- Ответ на групповой вызов

### <a name="online-or-hybrid-user-account-limitations"></a>Ограничения интерактивных или гибридных учетных записей пользователей
<a name="Online-Hybrid"> </a>

Учетные записи пользователей могут существовать в интерактивной или локальной среде, при этом доступные функции будут различаться. Пользователи с учетными записями на Скайп для бизнеса в Интернет не смогут получить доступ к следующие функции, даже с полной версией клиента: 

- Расширенные возможности присутствия. Используйте для параметра "Мое фото" фотографию с любого общедоступного сайта.

- Контакты. Поиск групп ответа.

- Поддержка обмена мгновенными сообщениями. Интеграция сохраняемого чата (группового чата).

- Поддержка обмена мгновенными Сообщениями: Перевести Persistent Chat места для Скайп для собраний с одним щелчком мыши

- Внешние пользователи. Выполнение двусторонних или многопользовательских звонков внешним пользователям.
