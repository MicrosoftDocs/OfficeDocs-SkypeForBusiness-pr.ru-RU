# <a name="skype-for-business-server-2019-management-tools"></a>Средства управления Skype для бизнеса Server 2019
 
**Сводка:** Сведения о средствах управления службами в Skype для бизнеса Server 2019.
  
Skype для бизнеса Server 2019 предоставляет решения для обмена мгновенными сообщениями, присутствия, конференц-связи и телефонии, которые поддерживают требования к совместной работе на уровне предприятия. Инструменты для управления этими службами являются гибкими и мощными.
