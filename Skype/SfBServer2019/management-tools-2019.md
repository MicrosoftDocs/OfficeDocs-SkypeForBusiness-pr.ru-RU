# <a name="skype-for-business-server-2019-management-tools"></a>Средства управления Skype для бизнеса Server 2019
 
**Сводка:** Узнайте о средствах управления службами в Skype для бизнеса Server 2019.
  
Skype for Business Server 2019 предлагает решения для обмена мгновенными сообщениями ,присутствия, телефонии и телефонии, которые поддерживают требования к совместной работе на корпоративном уровне. Средства для управления этими службами являются гибкими и мощными.
