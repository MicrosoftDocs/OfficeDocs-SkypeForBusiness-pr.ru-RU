# <a name="skype-for-business-server-2019-management-tools"></a>Средства управления в Skype для бизнеса Server 2019
 
**Сводка:** Узнайте о средствах управления услугами в Skype для бизнеса Server 2019.
  
Skype для бизнеса Server 2019 поддерживает обмен мгновенными сообщениями, присутствие, Конференц-связь и решения для телефонной связи, которые поддерживают требования для совместной работы в корпоративном масштабе. Инструменты для управления этими службами отличаются гибкостью и мощностью.
