## <a name="private-branch-exchange-pbx-options"></a>Параметры закрытого Exchange филиала (УАТС)

### <a name="phone-system-office-365"></a>Телефонной системой (Office 365)
  
Телефонной системой — технология корпорации Майкрософт для включения управления звонками и возможности Exchange закрытый филиала (УАТС) в облаке с группами Майкрософт и/или Скайп Office 365 для бизнеса в Интернет. 

Телефонной системой работает с группами или Скайп для бизнеса в Интернет клиентов и устройств сертифицированного. Телефонная система позволяет заменить существующую систему УАТС набором функций, предоставляемых напрямую из Office 365 и тесно интегрированных в облачную среду компании для продуктивной работы. Для подключения к общедоступной переключения телефонной сети общего пользования (PSTN) телефонной системой, вы можете вызов планирование корпорации Майкрософт или собственного поставщика телефонной связи.

Для получения дополнительных сведений см [возможности телефонной системой в Office 365](https://docs.microsoft.com/en-us/MicrosoftTeams/what-is-phone-system-in-office-365).

### <a name="enterprise-voice-skype-for-business-server"></a>Корпоративная голосовая связь (Скайп для Business Server)

Enterprise Voice — это технология корпорации Майкрософт для включения управления звонками и возможности Exchange закрытый филиала (УАТС) в локальной Скайп для Business Server. Этот параметр, только могут быть подключены к общедоступной переключения телефонной сети с помощью собственного поставщика телефонной связи. 

Для получения дополнительных сведений см. [Планирование корпоративной голосовой связи в Скайп для Business Server](https://docs.microsoft.com/en-us/SkypeForBusiness/plan-your-deployment/enterprise-voice-solution/enterprise-voice?toc=/SkypeForBusiness/toc.json&bc=/SkypeForBusiness/breadcrumb/toc.json).

## <a name="connection-to-the-public-switched-telephone-network-pstn-options"></a>Подключение к параметрам общедоступных переключения телефонной сети общего пользования (PSTN)

Вы можете подключиться к общедоступной переключения телефонной сети общего пользования (PSTN) с:

- С помощью вызова Microsoft план в Office 365 
- Подключение собственного поставщика телефонии

### <a name="calling-plan-office-365"></a>Вызов Plan (Office 365)

Этот параметр подключается 's системы Microsoft Office 365 телефона для общедоступных переключения телефонной сети общего пользования (PSTN) для поддержки вызовов в landlines и мобильные телефоны по всему миру. С помощью вызова планирования Корпорация Майкрософт считает вашего поставщика услуг.

Дополнительные сведения см в [Вызове планы для Office 365](https://docs.microsoft.com/en-us/MicrosoftTeams/calling-plans-for-office-365).

### <a name="connect-your-own-telephony-carrier-office-365-and-skype-for-business-on-premises"></a>Подключение собственного поставщика телефонии (Скайп для бизнеса в локальной и Office 365)

Этот параметр, либо телефонной системой в Office 365 или корпоративной голосовой связи в системе Скайп для бизнеса в локальной подключается к телефонной сети. Этот параметр требует поддерживаемые пограничный контроллер сеансов (SBC). В некоторых случаях этот параметр, могут требовать дополнительное программное обеспечение Майкрософт, развернутые локально.

## <a name="which-solution-is-right-for-your-organization"></a>Какие лучше всего подходит для вашей организации?

Вы можете решения на базе все-в--облачной, решение connect к владельцем поставщика или сочетание разных типов между все-в--облачной и сторонних производителей связи:

- Система телефон с вызова планирование (все в облаке)

- Телефонной системы с помощью собственных поставщика с помощью прямой маршрутизации

- Телефонной системы с помощью собственных поставщика с помощью Скайп сервера или облака соединитель для выпуска для бизнеса

- Корпоративная голосовая связь в Скайп для Business Server с помощью собственных поставщика

Решение выбранного зависит от текущие и будущие потребности, таких как:

- При необходимости, —-требуются или —-сохранять функциональные возможности, предоставляемые локального развертывания.
- Какие клиента, которую требуется развернуть для пользователей.
- План возможности перемещения пользователей в облако.
- Нужно ли будет взаимодействовать с УАТС сторонних производителей и других телефонное оборудование.

Ответьте на следующие вопросы, чтобы определить, оптимальным решением для вашей организации:

- Имеется существующий Скайп для развертывания сервера?
- Пользователи размещены в Скайп для бизнеса локально, в облаке на Скайп для бизнеса в Интернет или оба? 
- Хотите ли вы перемещение локальных пользователей в облако?
- Доступно корпорации Майкрософт вызов планирование ТСОП в вашем регионе?
- Или сохранить своего текущего оператора телефонии?  Например необходимо хранить текущего поставщика из-за существующий контракт?
- У вас есть существующая локальная устаревшая ТСОП, которую вы хотите или должны сохранить? 
- Предлагает ли ваша текущая устаревшая ТСОП уникальные функции, которые являются критическими для вашего бизнеса?
- Какие-либо или всех пользователей требуется возможностей, предоставляемых в настоящее время не в телефонной системой?

Обратите внимание на следующее:

- Все четыре параметра выполнена даже при наличии друг с другом на случай необходимости при проектировании решения для сложных сред или управление многоэтапные миграции.
- Телефонной системы с помощью собственных поставщика с помощью Скайп для Business Server или облака соединителя Edition развертывание может выполняться только с помощью любого из Скайп Business Server или облака соединителя. Сосуществование Скайп для Business Server и облачных соединитель не поддерживается в одной организации.

- Клиенты могут подключаться поддерживаемые SBC с помощью прямой маршрутизации для взаимодействия с УАТС сторонних производителей, аналоговых устройств и других сторонних телефонное оборудование стороны, поддерживаемых SBC.

| Требует контракта с помощью стороннего поставщика      | Нет   |

- Чтобы запретить для управления пограничных контроллеров сеансов на свои собственные.
- Команды и/или Скайп для бизнеса в Интернет имеет все компоненты, необходимые для вашей организации.
