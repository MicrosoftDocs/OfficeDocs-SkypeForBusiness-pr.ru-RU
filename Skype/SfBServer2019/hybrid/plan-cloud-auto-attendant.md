# <a name="plan-cloud-auto-attendant"></a>Планирование облачных автосекретаря

Скайп для реализации гибридных Business Server 2019 только использования автосекретарей облачных голосовой почты облачных и не интеграции с Exchange Online.

С помощью Скайп for Business Server 2019 с [Телефонной системы автосекретари](/SkypeForBusiness/what-is-phone-system-in-office-365/what-are-phone-system-auto-attendants.md) еще не доступен.

Если вы являетесь текущего пользователя Exchange Server 2013 или Exchange Server 2016, вы сможете продолжить использовать их, как реализовать Скайп для Business Server 2019. Exchange единой системы обмена СООБЩЕНИЯМИ, включая функции автосекретаря — это время не поддерживается в Exchange 2019.

Аналогичная функция существует в телефонной системой, а на основе по характеристикам вводит Скайп Business Server 2019 взаимодействие с телефонной системой. Выполните планирование соответствующим образом.
