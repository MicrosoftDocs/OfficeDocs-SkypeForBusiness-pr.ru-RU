# <a name="configure-federation-for-an-audio-conferencing-provider-in-your-hybrid-deployment"></a>Настройка федерации для поставщика аудиоконференций в гибридном развертывании

**Сводка:** Сведения о настройке федерации для поставщика аудиоконференций в Скайп для бизнеса в Интернет.

Если вы хотите использовать поставщика конференц-связи аудио (ACP) в гибридном развертывании (локальная online), необходимые для настройки федерации между локальным развертыванием и партнеров ACP как разрешенным сервером партнера. Федерации можно настроить путем добавления домена партнера ACP и пограничного сервера (это также может называться прокси-сервера доступа) в список федеративные домены для локального развертывания. Ваш партнер ACP затем необходимо добавить полное доменное имя пула пограничных серверов в локальной в список разрешенных федеративных доменов. Обратитесь к поставщику услуг ACP для получения дополнительной информации. Ваш партнер ACP затем необходимо добавить полное доменное имя пула пограничных серверов в локальной в список разрешенных федеративных доменов.

- **Добавление домена ACP и пограничного сервера в качестве разрешенных федеративных доменов**

    Чтобы добавить домен ACP разрешенным сервером партнера (разрешено федеративного домена), выполните действия, описанные в [Настройка поддержки для разрешенных внешних доменов](https://technet.microsoft.com/library/3ee6e175-986d-4c33-b03a-b9f93083dca6.aspx). Для пограничного сервера добавьте полное доменное имя партнера ACP пограничного сервера. Может потребоваться связаться со своим партнером ACP, чтобы получить полное доменное имя для своего пограничного сервера, на который ваши ACP могут ссылаться как на свой прокси-сервер доступа.

- **Предоставление полного доменного ИМЕНИ пула пограничных серверов ACP для партнеров**

    Партнеру ACP необходимо настроить федерацию, чтобы добавить ваш локальный домен как разрешенный сервер-партнер, добавив полное доменное имя пула пограничных серверов в качестве разрешенного федеративного домена.
