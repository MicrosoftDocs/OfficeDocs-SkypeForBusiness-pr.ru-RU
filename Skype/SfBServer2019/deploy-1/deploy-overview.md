# <a name="deploy-skype-for-business-server-2019"></a>Развертывание Skype для бизнеса Server 2019
 
**Сводка:** В этих ссылках вы узнаете, как установить и настроить Skype для бизнеса Server 2019 и перенести их из более ранних версий.
  
Skype для бизнеса Server обеспечивает обмен мгновенными сообщениями, сведения о присутствии, конференц-связи, видео и телефонные решения, которые могут поддерживать требования к совместной работе на уровне предприятия. В этом разделе приведены ссылки на сведения о развертывании и настройке Skype для бизнеса Server. 
  
Содержимое этих статей постоянно обновляется с учетом новых знаний и отзывов пользователей. Эта страница будет часто обновляться, поэтому регулярно проверяйте ее.
