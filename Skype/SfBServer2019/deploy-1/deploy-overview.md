# <a name="deploy-skype-for-business-server-2019"></a>Развертывание Skype для бизнеса Server 2019
 
**Сводка:** В этой статье приведены ссылки на сведения о том, как установить и настроить Skype для бизнеса Server 2019 и перейти с более ранних версий.
  
Skype для бизнеса Server предлагает решения для обмена мгновенными сообщениями, предоставления информации о присутствии, конференций, видео и телефонии, которые могут поддерживать требования по совместной работе корпоративного уровня. В этом разделе приведены ссылки на инструкции по развертыванию и настройке Skype для бизнеса Server. 
  
Эти статьи регулярно обновляются с учетом появления новых сведений и отзывов пользователей. Данная страница будет постоянно обновляться, поэтому заходите почаще.
