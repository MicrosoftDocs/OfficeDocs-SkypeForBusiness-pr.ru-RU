# <a name="migrating-archiving-and-monitoring-servers"></a>Перенос серверов архива и мониторинга

Если вы развернули архивный сервер и сервер мониторинга в устаревшей среде, эти серверы можно развернуть в среде Skype для бизнеса Server 2019 г. после переноса пулов переднего конца. Однако если функции архивации и мониторинга имеют решающее значение для организации, необходимо добавить архивации и мониторинга в пилотный пул Skype для бизнеса Server 2019 г. перед миграцией, чтобы функциональность была доступна в процессе миграции. 
  
Если функции архивации и мониторинга должны быть доступны во время миграции, помните о следующем.
  
- Архивации данных и данных мониторинга не перемещаются в Skype для бизнеса Server 2019. Данные, которые вы возвращаете до вывода из эксплуатации устаревшей среды, будут вашей историей действий в устаревшей среде.
    
- Устаревшая версия архивного сервера и сервера мониторинга может быть связана только с устаревшим пулом переднего конца. В Skype для бизнеса Server 2019 г. архивативная и мониторинговая функции больше не являются серверными ролями, а службами, интегрированными в пул Skype для бизнеса Server 2019 года.
    
- За время сосуществования устаревших и Skype для бизнеса Server 2019 г. устаревшая версия сервера архивации и сервера мониторинга собирает данные для пользователей, размещенных в устаревших пулах. Архиватив и мониторинг в Skype для бизнеса Server 2019 г. собирают данные для пользователей, Skype для бизнеса Server пулах 2019 г.
    
    > [!NOTE]
    > На этапе миграции, когда вы по-прежнему используете устаревший сервер Edge с новым пилотным пулом Skype для бизнеса Server 2019 г., в устаревшей версии архивирования Server по-прежнему собираются данные для пользователей, которые находятся в устаревших пулах, а архивирования в Skype для бизнеса Server 2019 г. собирают данные для пользователей, которые находятся в пулах Skype для бизнеса Server 2019 г. 
  
- Если в Skype для бизнеса Server 2019 г. совместно с архивативией и мониторингом используется сторонний вариант архива и мониторинга, проконсультируйтесь с поставщиком о том, когда и как необходимо интегрировать сторонное решение с Skype для бизнеса Server 2019 г.
