# <a name="verify-configuration-settings"></a>Проверка параметров настройки

Вы можете проверить репликацию сведений о конфигурации на пограничном сервере, запустив командлет Skype для Business Server 2019 **Get-ксманажементсторерепликатионстатус** на внутреннем компьютере, на котором находится корневой центр управления, или на любом компьютере, подключенном к домену, на котором установлен компонент "основные компоненты" Skype для Business Server 2019 (окскоре. msi). 
  
Начальные результаты могут содержать состояние "false", а не "истина" для репликации. Если это так, запустите командлет **Invoke-ксманажементсторерепликатион** и разрешите время завершения репликации, прежде чем запускать командлет **Get-ксманажементсторерепликатионстатус** еще раз.
