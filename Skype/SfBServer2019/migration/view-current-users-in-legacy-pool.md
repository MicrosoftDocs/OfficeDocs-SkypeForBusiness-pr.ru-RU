# <a name="view-current-users-in-legacy-pool"></a>Просмотр текущих пользователей в старом пуле

Прежде чем изучать различные способы перемещения пользователей между пулами, необходимо сначала определить, какие пользователи существуют в устаревшем пуле. Столбец **пула регистратора** определяет пользователей, настроенных для устаревшего пула. Это тестовые пользователи, которые мы будем перемещать в пул Skype для бизнеса Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Чтобы увидеть список пользователей в устаревшем пуле

1. Войдите на устаревший сервер переднего плана с учетной записью, которая является членом группы RTCUniversalServerAdmins или членом административной роли CsAdministrator или CsUserAdministrator.
    
2. Откройте **панель управления Skype для бизнеса Server.**
    
3. Щелкните **"Пользователи",** **"Поиск"** и выберите **"Найти".**
