# <a name="view-current-users-in-legacy-pool"></a>Просмотр текущих пользователей в устаревшем пуле

Прежде чем обучения пользователей можно перемещать между пулами различные способы, мы сначала определите пользователей, которые существуют в устаревшем пуле. На следующем рисунке столбец **пул регистратора** определяет шесть пользователей, которые настроены для устаревшего пула. Далее представлены тестовые пользователи, которые мы будут перемещаться Скайп для пула Business Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Чтобы просмотреть список пользователей в устаревшем пуле

1. Войдите на устаревшего сервера переднего плана с использованием учетной записи, которая является членом группы RTCUniversalServerAdmins или членом административной роли CsAdministrator или CsUserAdministrator.
    
2. Откройте **Скайп для панели управления Business Server**.
    
3. Щелкните элемент **Пользователи**, щелкните **Поиск**и нажмите кнопку **Найти**.
