# <a name="view-current-users-in-legacy-pool"></a>Просмотр текущих пользователей в старом пуле

Прежде чем приступать к изучению различных способов перемещения пользователей между пулами, сначала необходимо определить, какие пользователи существуют в устаревшем пуле. Столбец **пул регистратора** определяет пользователей, которые настроены для пула прежних версий. Это тестовые пользователи, которые мы перейдем в пул Skype для бизнеса Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Просмотр списка пользователей в устаревшем пуле

1. Выполните вход на сервер переднего плана прежних версий с учетной записью, которая является членом группы RTCUniversalServerAdmins или членом административной роли CsAdministrator или CsUserAdministrator.
    
2. Откройте **Панель управления Skype для бизнеса Server**.
    
3. Нажмите кнопку **Пользователи**, выберите **Поиск**, а затем нажмите кнопку **найти**.
