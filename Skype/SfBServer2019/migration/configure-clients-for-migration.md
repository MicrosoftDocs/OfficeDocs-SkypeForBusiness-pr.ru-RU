# <a name="configure-clients-for-migration"></a>Настройка клиентов для миграции

В этой статье описаны рекомендованные этапы развертывания клиента, которые необходимо выполнить перед переходом на Skype для бизнеса Server 2019. Эти изменения конфигурации следует вносить в Lync Server 2013 или Skype для бизнеса Server 2015 (версия, **из**которой выполняется миграция).
  
### <a name="to-configure-clients-before-migration"></a>Настройка клиентов перед миграцией

1. Развертывание последних обновлений сервера, клиента и устройств (исправлений) для существующей установки.
    
2. В предыдущей версии Skype для бизнеса Server используйте фильтрацию по версиям клиента, чтобы разрешить только клиентам с установленными текущими обновлениями.
