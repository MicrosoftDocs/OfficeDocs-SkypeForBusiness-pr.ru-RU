# <a name="configure-clients-for-migration"></a>Настройка клиентов для миграции

В этой статье описываются рекомендуемые действия по развертыванию клиентов перед переходом на Skype для бизнеса Server 2019. Эти изменения конфигурации следует вносить в Lync Server 2013 или Skype для бизнеса Server 2015 (версия, **с**которой выполняется миграция).
  
### <a name="to-configure-clients-before-migration"></a>Настройка клиентов перед переходом

1. Разверните последние обновления сервера, клиента и устройства (исправления) для существующей установки.
    
2. В предыдущей версии Skype для бизнеса Server используйте фильтрацию версий клиентов, чтобы позволить клиентам устанавливать только самые последние обновления.
