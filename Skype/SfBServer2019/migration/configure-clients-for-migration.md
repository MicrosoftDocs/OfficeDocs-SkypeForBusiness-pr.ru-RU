# <a name="configure-clients-for-migration"></a>Настройка клиентов для миграции

Этот раздел содержит этапы развертывания рекомендуется клиент, необходимо выполнить перед перемещением в Скайп для Business Server 2019. Эти изменения конфигурации делаться на Lync Server 2013 или Скайп 2015 Business Server (версия, переносе **из**).
  
### <a name="to-configure-clients-before-migration"></a>Настройка клиентов перед переходом

1. Развертывание самые последние сервер, клиентов и устройств обновления (исправления) для существующей установки.
    
2. На предыдущей версии Скайп для сервера Чтобы разрешить поддержку только клиентов с установленными последними обновлениями используйте фильтрацию по версиям клиентов.
