# <a name="configure-clients-for-migration"></a>Настройка клиентов для миграции

В этом разделе представлены рекомендуемые действия по развертыванию клиентов перед переходом на Skype для бизнеса Server 2019. Эти изменения конфигурации следует внести в Lync Server 2013 или Skype для бизнеса Server 2015 (из этой версии вы **переносились).**
  
### <a name="to-configure-clients-before-migration"></a>Настройка клиентов перед переходом

1. Развертывание последних обновлений (обновлений) сервера, клиента и устройства для существующей установки.
    
2. В предыдущей версии Skype для бизнеса Server используйте фильтрацию версий клиентов, чтобы разрешить только клиенты с самыми новыми установленными обновлениями.
