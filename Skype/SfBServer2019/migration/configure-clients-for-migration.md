# <a name="configure-clients-for-migration"></a>Настройка клиентов для миграции

В этом разделе содержатся рекомендуемые действия по развертыванию клиентов, которые необходимо предпринять перед переносом Skype для бизнеса Server 2019 г. Эти изменения конфигурации должны быть внесены на Lync Server 2013 или Skype для бизнеса Server 2015 (версию, из которая вы **мигрируете).**
  
### <a name="to-configure-clients-before-migration"></a>Настройка клиентов перед переходом

1. Развертывание последних обновлений серверов, клиентов и устройств (hotfixes) для существующей установки.
    
2. В предыдущей версии Skype для бизнеса Server фильтрация клиентских версий, чтобы разрешить клиентам только самые актуальные обновления.
