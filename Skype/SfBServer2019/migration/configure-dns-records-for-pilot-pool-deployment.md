# <a name="configure-dns-records-for-pilot-pool-deployment"></a>Настройка DNS-записей для развертывания пилотного пула

Перед развертывание пилотного пула необходимо обновить записи DNS Host A для пилотного пула. Для успешного выполнения этой процедуры необходимо войти на сервер или в домен как участник группы администраторов домена или DnsAdmins.
  
### <a name="to-configure-dns-host-a-records"></a>Настройка записей A узла DNS

1. На сервере DNS в меню **Пуск** выберите пункт **Администрирование** и щелкните **DNS**.
    
2. В дереве консоли для домена раздвигайте зоны досмотра и щелкните правой кнопкой мыши домен, в котором Skype для бизнеса Server 2019 г. 
    
3. Выберите команду **Создать узел (A или AAAA)**.
    
4. Щелкните **Имя,** введите имя хозяина пула Skype для бизнеса Server 2019 г. (доменное имя предполагается из зоны, в которую определяется запись, и не требуется ввести ее как часть записи A).
    
5. Щелкните **IP-адрес** и введите IP-адрес для пула переднего конца.
    
6. Щелкните **Добавить узел**, а затем нажмите кнопку **ОК**. 
    
7. По завершении нажмите кнопку **Готово**.
