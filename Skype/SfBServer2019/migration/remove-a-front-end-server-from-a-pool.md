# <a name="remove-a-front-end-server-from-a-pool"></a>Удаление сервера переднего плана из пула

Сервер переднего плана не могут существовать в качестве изолированный компьютер. Он должен быть определен как пулом переднего плана, даже при наличии только на одном компьютере в пуле.
  
В этом разделе описывается процесс удаления отдельного сервера переднего плана из существующего пула переднего плана. Если сервер переднего плана — это последний сервер в пуле, а также при удалении пула полностью просмотрите [Удаление интерфейсного пула или сервера Standard Edition](remove-front-end-pool-or-standard-edition-server.md). Нет необходимости для удаления отдельных серверов переднего плана, прежде чем удалять пула переднего плана. При удалении пула, удалите каждого сервера переднего плана.
  
### <a name="to-remove-a-front-end-server-from-a-pool"></a>Удаление сервера переднего плана из пула

1. На Скайп для сервера переднего плана Business Server 2019 откройте построитель топологий.
    
2. Перейдите к узлу установки прежних версий.
    
3. Разверните **Пулы переднего плана Enterprise Edition**, разверните пул переднего плана с сервером переднего плана, которую требуется удалить, щелкните правой кнопкой мыши сервер переднего плана, который требуется удалить и выберите команду **Удалить**.
