# <a name="remove-a-front-end-server-from-a-pool"></a>Удаление сервера переднего плана из пула

Передний конечный сервер не может существовать как автономный компьютер. Он должен быть определен как пул переднего конца, даже если в пуле есть только один компьютер.
  
Этот раздел поможет вам в процессе удаления отдельного переднего конечного сервера из существующего пула переднего конца. Если сервер переднего конца является последним сервером в пуле или полностью удаляет пул, см. в этой ленте Удалить пул переднего [выпуск Standard сервера.](remove-front-end-pool-or-standard-edition-server.md) Нет необходимости удалять отдельные серверы переднего конца перед удалением пула переднего конца. При удалите пул, удалите каждый сервер переднего конца.
  
### <a name="to-remove-a-front-end-server-from-a-pool"></a>Удаление сервера переднего плана из пула

1. На Skype для бизнеса Server 2019 front End Server откройте топологию Builder.
    
2. Перейдите к устаревшему узлу установки.
    
3. **Расширите** выпуск Enterprise пулы переднего конца, развяжите пул переднего конца с помощью переднего end Server, который необходимо удалить, щелкните правой кнопкой мыши передний конечный сервер, который необходимо удалить, и нажмите кнопку **Удалить**.
