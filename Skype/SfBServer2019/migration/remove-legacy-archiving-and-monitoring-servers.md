# <a name="remove-legacy-archiving-and-monitoring-servers"></a>Удаление старых серверов архивирования и мониторинга

Если ваше устаревшее развертывание содержало сервер архивации или сервер мониторинга, после переноса в Skype для бизнеса Server 2019 г. эти серверы можно удалить из устаревшей среды при условии, что все пользователи будут удалены из оставшихся пулов устаревших. Сервер архивирования и сервер мониторинга можно удалять в любом порядке. Главное требование состоит в удалении всех пользователей из всех оставшихся устаревших пулов.
  
Вы можете переместить пользователей Skype для бизнеса Server 2019 г., следуя процедурам, описанным в [этапе 4: Перемещение](phase-4-move-test-users-to-the-pilot-pool.md)тестовых пользователей в пилотный пул.
  
После того как вы подтвердите, что все пользователи удалены из оставшихся пулов, удалите сервер и удалите роли. Датированная, но актуальная пример: "Удаление ролей Microsoft Lync Server и удаление ролей сервера", которые можно скачать по [https://go.microsoft.com/fwlink/p/?linkId=246227](https://go.microsoft.com/fwlink/p/?linkId=246227) ссылке .
