# <a name="migrate-existing-meetings-and-meeting-content"></a>Перенос существующих собраний и содержимого собраний

Когда учетная запись пользователя перемещается на сервер 2019 Skype для бизнеса Server 2019 г., с этой учетной записью пользователя перемещаются следующие сведения:
  
- **Собрания, уже запланированные этим пользователем**. В том числе, перемещаются каталоги конференций и данные конференций.
    
- **Личный идентификационный номер пользователя (PIN-код).** Текущий ПИН-код пользователя продолжает работать до истечения срока действия или запроса нового ПИН-кода.
    
На новый сервер не перемещаются следующие сведения об учетной записи пользователя.
  
- **Содержимое собрания**. Для перемещения контента, общего во время собрания, таких как PowerPoint, доска, вложения или данные опроса, используйте параметр **MoveConferenceData** в составе комлета **Move-CsUser.**
