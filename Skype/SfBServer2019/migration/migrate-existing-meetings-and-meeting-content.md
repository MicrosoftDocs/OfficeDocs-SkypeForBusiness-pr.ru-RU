# <a name="migrate-existing-meetings-and-meeting-content"></a>Перенос существующих собраний и содержимое собраний

При перемещении учетной записи пользователя Скайп для сервера Business Server 2019 с учетной записью пользователя перемещается следующая информация:
  
- **Собрания, уже запланированных пользователем**. Этот компонент включает перемещение каталогов конференций и данных конференц-связи.
    
- **Пользователя персональный идентификационный номер (ПИН-кода)**. Текущий ПИН-код пользователя продолжает работать до истечения срока действия или пользователь запрашивает новый ПИН-код.
    
Следующие сведения учетной записи пользователя не перемещается на новый сервер.
  
- **Контенту собрания**. Для перемещения содержимого, совместно используемого во время собрания, например, PowerPoint, доски, вложений и данных опроса, используйте параметр **- MoveConferenceData** как часть командлета **Move-CsUser** .
