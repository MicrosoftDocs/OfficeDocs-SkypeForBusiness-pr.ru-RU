# <a name="remove-front-end-pool-or-standard-edition-server"></a>Удаление пула переднего плана или сервера Standard Edition

В этой статье вам будет посвящен процесс удаления пула переднего выпуск Standard переднего конечного сервера. При удалении пула переднего конца удаляется каждый сервер переднего конца, принадлежащий пулу, как часть процесса удаления пула. При удалите выпуск Standard переднего end Server необходимо удалить определение SQL Store из Topology Builder.
  
## <a name="to-remove-a-front-end-server-pool"></a>Удаление пула интерфейсных серверов

1. Откройте построитель топологий.
    
2. Перейдите к устаревшему узлу.
    
3. **Расширите** выпуск Enterprise пулы переднего конца, развяжите пул переднего конца, щелкните правой кнопкой мыши передний конец пула, который необходимо удалить, а затем нажмите **кнопку Удалить**.
    
4. Опубликуйте топологию, проверьте состояние репликации и запустите по мере необходимости устаревший мастер развертывания. 
    
## <a name="to-remove-a-standard-edition-front-end-server"></a>Удаление интерфейсного сервера Standard Edition

1. Откройте построитель топологий.
    
2. Перейдите к устаревшему узлу установки.
    
3. **Расширите** выпуск Standard серверов переднего конца, щелкните правой кнопкой мыши передний конечный сервер, который необходимо удалить, а затем нажмите **кнопку Удалить**.
    
4. **Расширите** SQL магазинах, щелкните правой кнопкой мыши базу данных SQL Server, связанную с выпуск Standard сервером переднего выпуск Standard, а затем нажмите **кнопку Удалить**.
    
    > [!IMPORTANT]
    > Необходимо удалить определение collocated SQL Server баз данных с выпуск Standard переднего конечного сервера. 
  
5. Опубликуйте топологию, проверьте состояние репликации и запустите мастер развертывания по мере необходимости.
