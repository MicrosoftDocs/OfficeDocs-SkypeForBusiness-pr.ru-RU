# <a name="remove-front-end-pool-or-standard-edition-server"></a>Удаление пула переднего плана или сервера Standard Edition

В этой статье приводится процедура удаления пула переднего сервера или сервера переднего сервера Standard Edition. При удалении пула переднего сервера в процессе удаления пула удаляется каждый сервер переднего сервера, который принадлежит к пулу. При удалите сервер переднего сервера Standard Edition, необходимо удалить определение SQL store из построитель топологий.
  
## <a name="to-remove-a-front-end-server-pool"></a>Удаление пула интерфейсных серверов

1. Откройте построитель топологий.
    
2. Перейдите к устаревшему узлу.
    
3. Раз **развернуть пулы** переднего входа Enterprise Edition, развернуть пул переднего концев, щелкните правой кнопкой мыши пул переднего конца, который нужно удалить, а затем нажмите кнопку **"Удалить".**
    
4. Опубликуйте топологию, проверьте состояние репликации и при необходимости запустите старый мастер развертывания. 
    
## <a name="to-remove-a-standard-edition-front-end-server"></a>Удаление интерфейсного сервера Standard Edition

1. Откройте построитель топологий.
    
2. Перейдите к устаревшему узлу установок.
    
3. Разйдите серверы переднего сервера **Standard Edition,** щелкните правой кнопкой мыши сервер переднего сервера, который нужно удалить, и выберите **"Удалить".**
    
4. Раз **SQL хранилища,** щелкните правой кнопкой мыши базу данных SQL Server, связанную с сервером переднего SQL Server Standard Edition, а затем нажмите кнопку **"Удалить".**
    
    > [!IMPORTANT]
    > Необходимо удалить определение баз данных с SQL Server сервера переднего SQL Server Standard Edition. 
  
5. Опубликуйте топологию, проверьте состояние репликации и при необходимости запустите мастер развертывания.
