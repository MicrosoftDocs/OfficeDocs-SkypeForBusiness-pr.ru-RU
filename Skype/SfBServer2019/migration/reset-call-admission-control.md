# <a name="reset-call-admission-control"></a>Сброс контроля допуска звонков

Если устаревшего пула переднего плана размещен контроль допуска звонков (CAC), необходимо переместить контроль допуска звонков размещения для Скайп для пула Business Server 2019, прежде чем удалять устаревшие пула переднего плана.
  
### <a name="to-reset-cac"></a>Чтобы сбросить контроль допуска звонков

1. Откройте построитель топологий.
    
2. Щелкните правой кнопкой мыши узел сайта и выберите команду **Изменить свойства**.
    
3. В разделе **Параметры контроля допуска звонков**убедитесь в том, что выбран параметр **Включить контроль допуска звонков** . 
    
4. В разделе **пул переднего плана для запуска контроля допуска звонков (CAC)** выберите Скайп для Business Server 2019 пула, который должен размещаться контроля допуска звонков и нажмите кнопку **ОК**.
    
5. Опубликуйте топологию.
