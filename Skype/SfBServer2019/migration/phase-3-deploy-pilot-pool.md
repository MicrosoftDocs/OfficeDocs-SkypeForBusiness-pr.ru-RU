# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Этап 3: Развертывание пилотного пула Skype для бизнеса Server 2019

В этом разделе описываются действия, необходимые для развертывания пилотного пула Skype для бизнеса Server 2019. Для развертывания Skype для бизнеса Server 2019 необходимо использовать построитель топологий, чтобы определить топологию и компоненты, которые необходимо развернуть, подготовить среду для развертывания компонентов Skype для бизнеса Server 2019, опубликовать схему топологии на первом сервере переднего плана, а затем установить и настроить Skype для бизнеса Server 2019 для компонентов для вашего развертывания. После завершения развертывания пилотного пула Skype для бизнеса Server 2019 будет сосуществовать в существующем устаревшем пуле.
