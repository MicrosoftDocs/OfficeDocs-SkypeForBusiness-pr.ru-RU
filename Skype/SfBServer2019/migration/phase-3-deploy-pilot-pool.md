# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Этап 3. Развертывание Skype для бизнеса Server пилотного пула 2019 г.

В этом разделе освещаются действия, необходимые для развертывания пилотного пула Skype для бизнеса Server 2019 г. Развертывание Skype для бизнеса Server 2019 г. требует использования Topology Builder для определения топологии и компонентов, которые необходимо развернуть, подготовки среды для развертывания компонентов Skype для бизнеса Server 2019 г., публикации топологии на первом переднем конечном сервере, а затем установки и настройки программного обеспечения Skype для бизнеса Server 2019 г. для компонентов для развертывания. По завершению развертывание пилотного пула Skype для бизнеса Server 2019 г. будет сосуществовать с существующим устаревшим пулом.
