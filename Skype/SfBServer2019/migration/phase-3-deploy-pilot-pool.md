# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Этап 3: развертывание пула Skype для бизнеса Server 2019 Pilot

В этом разделе описаны действия, которые необходимо выполнить для развертывания пилотного пула в Skype для бизнеса Server 2019. Для развертывания Skype для бизнеса Server 2019 необходимо определить топологию и компоненты, которые вы хотите развернуть, с помощью построителя топологии и подготовить среду для развертывания компонентов Skype для бизнеса Server 2019 с помощью публикации топологии Разработайте на первом внешнем сервере, а затем установите и настройте приложение Skype для Business Server 2019 для компонентов, которые нужно развернуть. После того как вы закончите работу, развертывание пула проектов в Skype для бизнеса Server 2019 будет сосуществовать в существующем пуле унаследованных устройств.
