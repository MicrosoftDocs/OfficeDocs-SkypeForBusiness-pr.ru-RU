# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Этап 4. Перемещение тестовых пользователей в пилотный пул

Вы можете переместить одного пользователя или группы пользователей в новое развертывание Microsoft Skype для бизнеса Server 2019 г. с помощью следующих двух методов: Skype для бизнеса Server Панели управления и Skype для бизнеса Server Management Shell. В этом разделе описываются задачи, которые необходимо выполнить во время пилотного развертывания, а также перед переносом развертывания Skype для бизнеса Server 2019 г. из экспериментального развертывания в развертывание на уровне производства.
