# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Этап 4. Перемещение тестовых пользователей в пилотный пул

Переместить одного пользователя или группы пользователей в новое развертывание Microsoft Skype для бизнеса Server 2019 можно двумя способами: панель управления Skype для бизнеса Server и оболочка управления Skype для бизнеса Server. В этом разделе описываются задачи, которые необходимо выполнить во время пилотного развертывания, а также перед переносом развертывания Skype для бизнеса Server 2019 из пилотного развертывания в развертывание на производственном уровне.
