# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Этап 4: Перемещение тестовых пользователей пилотного пула

Можно переместить отдельного пользователя или группы пользователей в вашей новой Скайп Microsoft Business Server 2019 развертывания с помощью следующих двух способов: Скайп для панели управления Business Server и Скайп для консоли Business Server. В этом разделе описываются задачи, которые необходимо выполнить во время пилотного развертывания, а также перед переносом развертывания Скайп для Business Server 2019 из пилотной среды в развертывании производственного уровня.
