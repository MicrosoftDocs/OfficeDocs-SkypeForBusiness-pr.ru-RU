# <a name="run-best-practices-analyzer"></a>Запуск анализатора соответствия рекомендациям

Анализатором соответствия рекомендациям собирает сведения о конфигурации из развертывания и определяет, будет ли конфигурация настроена в соответствии с рекомендациями Майкрософт. Средство можно установить на клиентском компьютере, на котором выполняется Microsoft .NET Framework 3.5 с пакетом обновления 1 или непосредственно на сервере, на котором выполняется установка прежних версий. Рекомендуется установить и запуска этой программы на клиентском компьютере. Средства администрирования необходимо также установить локально на клиентском компьютере, чтобы анализатор соответствия рекомендациям можно собирать полный набор данных. 
  
Анализатор соответствия рекомендациям можно загрузить из центра загрузки Майкрософт по [https://go.microsoft.com/fwlink/p/?linkid=246173](https://go.microsoft.com/fwlink/p/?linkid=246173).
