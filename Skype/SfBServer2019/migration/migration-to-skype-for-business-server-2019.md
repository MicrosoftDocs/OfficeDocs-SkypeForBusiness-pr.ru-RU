# <a name="migration-to-skype-for-business-server-2019"></a>Миграция в Skype для бизнеса Server 2019

В подразделах этого раздела описывается процесс перехода на Skype для бизнеса Server 2019. В этой статье рассматривается миграция Lync Server 2013 или Skype для бизнеса Server 2015 в Skype для бизнеса Server 2019.

> [!IMPORTANT]
> На протяжении всего содержимого мы используем термин *Legacy (устаревшее* ), чтобы ссылаться на устаревшее Lync Server 2013 или Skype для бизнеса Server 2015, который вы переносите на Skype для бизнеса Server 2019.
  
> [!IMPORTANT]
> В этом руководстве описываются действия, которые обычно требуется выполнить на каждом из этапов миграции. Здесь не ставится цель рассмотреть все возможные топологии устаревших развертываний или все возможные сценарии миграции. Таким образом, вам может не потребоваться выполнять каждый шаг, описанный выше, или в зависимости от развертывания может потребоваться выполнить дополнительные действия. В этом руководстве также приводятся примеры шагов проверки. Эти действия по проверке предоставляются для того, чтобы помочь вам определить, что нужно, чтобы убедиться, что каждый этап завершается успешно по мере выполнения миграции. Выполните эти шаги по проверке в соответствии с конкретным процессом миграции. 
  
В данном руководстве приводится информация об обновлении вашего существующего развертывания. Здесь не поясняется, как изменить существующую топологию. Внедрение новых возможностей также выходит за рамки этого руководства. Когда подробная процедура задокументирована в другом месте, это руководство содержит ссылки на статью или статью. 
  
В этой статье определяются термины, указанные в приведенном ниже списке.
  
**Миграция:** Перемещение рабочего развертывания из Lync Server 2013 или Skype для бизнеса Server 2015 в Skype для бизнеса Server 2019.
    
**сосуществование:** Временная среда, существующая во время миграции, когда некоторые функции были перенесены в Skype для бизнеса Server 2019, а другие функции по-прежнему остаются в предыдущей версии.
    
**взаимодействие:** Способность развертывания успешно работать в течение периода сосуществования.

**устаревшие:** Система, с которой вы переноситесь, которая является Lync Server 2013 или Skype для бизнеса Server 2015.
