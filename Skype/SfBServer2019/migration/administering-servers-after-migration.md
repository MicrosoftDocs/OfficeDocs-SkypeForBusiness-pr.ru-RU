# <a name="administering-servers-after-migration"></a>Администрирование серверов после миграции

В целом, следует использовать средство администрирования, соответствующее версии сервера, которой требуется управлять. Нельзя установить административные средства из разных выпусков на одном компьютере.
  
> [!IMPORTANT]
> После развертывания Skype для бизнеса Server пилотного пула для управления ресурсами 2019 г. невозможно использовать topology Builder или Панель управления. Для управления всеми текущими и предыдущими ресурсами версий необходимо использовать средства 2019.
