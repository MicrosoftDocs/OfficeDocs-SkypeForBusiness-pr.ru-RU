# <a name="administering-servers-after-migration"></a>Администрирование серверов после миграции

В целом, следует использовать средство администрирования, соответствующее версии сервера, которой требуется управлять. На одном компьютере невозможно установить средства администрирования из разных выпусков.
  
> [!IMPORTANT]
> После развертывания пилотного пула Skype для бизнеса Server нельзя использовать построитель топологий или панель управления для управления ресурсами 2019. Для управления всеми ресурсами текущей и предыдущей версий необходимо использовать средства 2019.
