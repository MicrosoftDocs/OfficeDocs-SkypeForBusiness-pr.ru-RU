# <a name="administering-servers-after-migration"></a>Администрирование серверов после миграции

Как правило необходимо использовать средство администрирования, который соответствует версии сервера, который вы хотите управлять. Не удается установить средства администрирования в различных выпусках на том же компьютере.
  
> [!IMPORTANT]
> После развертывания Скайп для пилотного пула Business Server нельзя использовать построитель топологий или панели управления для управления любые ресурсы 2019. Необходимо использовать 2019 средства для управления ресурсами все текущие и предыдущие версии.
