# <a name="administering-servers-after-migration"></a>Администрирование серверов после миграции

Как правило, необходимо использовать средство администрирования, соответствующее серверной версии, которой вы хотите управлять. Установка средств администрирования из разных выпусков на одном и том же компьютере невозможна.
  
> [!IMPORTANT]
> После развертывания пула пилотных проектов в Skype для бизнеса Server вы не сможете использовать построитель топологии и панель управления для управления ресурсами 2019. Для управления всеми текущими и предыдущими версиями ресурсов необходимо использовать инструменты 2019.
