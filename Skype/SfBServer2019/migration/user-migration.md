# <a name="user-migration"></a>Перенос пользователей

Обычно обслуживаемых для миграций рекомендуется создавать несколько тестовых пользователей и использовать их для проведения тестов системы. После успешно перемещен и проверить эти учетные записи, следует определить группы пользователей пилотного производства и перемещения учетных записей и проведение тестов проверки на них. При получении удовлетворяют результаты можно переместить остальных пользователей в новом развертывании.
  
Дополнительные сведения о включении пользователей для Скайп для Business Server 2019 видеть Скайп развертывание документация Business Server 2019.
