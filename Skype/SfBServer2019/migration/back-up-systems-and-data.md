# <a name="back-up-systems-and-data"></a>Резервное копирование систем и данных

Перед началом миграции выполните полное резервное копирование системы и задразите существующую систему, включая инвентаризацию учетных записей пользователей, которые размещены в каждом пуле, чтобы при необходимости можно было откатать. Существует множество средств и программ для резервного копирования и восстановления данных, параметров и систем.

Подробные сведения и процедуры см. в материале о процедурах резервного копирования и восстановления версии.
