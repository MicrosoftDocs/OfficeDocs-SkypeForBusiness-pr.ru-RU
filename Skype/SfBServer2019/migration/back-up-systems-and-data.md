# <a name="back-up-systems-and-data"></a>Резервное копирование системы и данные

Перед началом миграции выполните резервное копирование всей системы и задокументируйте существующие системы, включая перечень учетных записей пользователей, которые размещаются в каждом пуле, чтобы вы можете откате мере необходимости. Несколько средств и программ доступны для резервного копирования и восстановления данных, параметры и систем.

Подробные сведения и процедуры статье backup and restore процедуры для вашей версии.
