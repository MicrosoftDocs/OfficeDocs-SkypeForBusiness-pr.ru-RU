# <a name="back-up-systems-and-data"></a>Резервное копирование систем и данных

Перед началом миграции выполните полное резервное копирование системы и за документирование существующей системы, включая перечень учетных записей пользователей, которые размещены в каждом пуле, чтобы можно было выполнить откат при необходимости. Существует множество средств и программ для резервного копирования и восстановления данных, параметров и систем.

Подробные сведения и процедуры см. в процедурах резервного копирования и восстановления версии.
