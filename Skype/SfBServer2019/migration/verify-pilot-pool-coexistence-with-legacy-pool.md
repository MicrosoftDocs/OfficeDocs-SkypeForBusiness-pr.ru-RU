# <a name="verify-pilot-pool-coexistence-with-legacy-pool"></a>Проверка возможности совместного использования пилотного и старого пула

 **Содержание**
  
[Проверка того, что службы Skype для бизнеса Server 2019 запущены](#sectionSection0)
  
[Откройте панель управления Skype для бизнеса Server 2019](#sectionSection1)
  
[Не пытайтесь открыть топологию в устаревшем построителе топологий](#sectionSection2)
  
После развертывания пилотного пула вам требуется проверить сосуществование двух пулов, воспользовавшись средствами администрирования для просмотра информации о пуле. Для пулов Skype для бизнеса Server 2019 и устаревших пулов необходимо использовать средства построителя топологий и панели управления Skype для бизнеса Server 2019. 
  
## <a name="verify-that-skype-for-business-server-2019-services-have-started"></a>Проверка того, что службы Skype для бизнеса Server 2019 запущены
<a name="sectionSection0"> </a>

1. На сервере переднего плана Skype для бизнеса Server 2019 перейдите к апплету Администрирование средств и служб.
    
2. Убедитесь, что на сервере переднего плана запущены следующие службы:

    - Агент централизованной службы ведения журналов.
    - Совместное использование приложений
    - Служба проверки звука
    - Аудио-и видеоконференции
    - Парковка вызовов
    - Извещение конференц-связи
    - Помощник по конференц-связи
    - Внешний интерфейс
    - Конференц-связь
    - Сетью
    - Агент репликации реплики
    - Группа ответа
    - Веб-конференции
    - Шлюз преобразования XMPP

  
## <a name="open-the-skype-for-business-server-2019-control-panel"></a>Откройте панель управления Skype для бизнеса Server 2019
<a name="sectionSection1"> </a>

На сервере переднего плана в развертывании Skype для бизнеса Server 2019 откройте панель управления Skype для бизнеса Server 2019 и выберите пул прежних версий. Повторите процедуру, чтобы открыть пул Skype для бизнеса Server 2019.
  
> [!IMPORTANT]
> В Skype для бизнеса Server 2019 необходимо обновить Silverlight до Silverlight версии 5, прежде чем использовать панель управления Skype для бизнеса Server. 
  
Эта топология теперь включает устаревшие роли сервера и сервера Skype для бизнеса Server 2019. 

  
## <a name="dont-attempt-to-open-the-topology-in-the-legacy-topology-builder"></a>Не пытайтесь открыть топологию в устаревшем построителе топологий
<a name="sectionSection2"> </a>

Топологию можно просмотреть только с помощью построителя топологий Skype для бизнеса Server 2019. Для создания пулов для Skype для бизнеса Server 2019 и установки прежних версий необходимо использовать построитель топологий Skype для бизнеса Server 2019.
