# <a name="configure-trusted-application-servers"></a>Настройка доверенных серверов приложений

В смешанной среде при создании нового доверенного сервера приложений необходимо настроить следующий пул хмеля как пул Skype для бизнеса Server 2019 года. В смешанной среде в выпадаемом списке отображаются как устаревший пул, так и пул Skype для бизнеса Server 2019 года. Предыдущую версию пула выбрать нельзя.
  
> [!IMPORTANT]
> При переносе доверенного сервера приложений также следует обновить используемую версию UCMA. При создании нового пула доверенных приложений Skype для бизнеса Server 2019 г. необходимо обновить UCMA до версии, включенной в Skype для бизнеса Server 2019 г. или последней доступной версии. 
  
### <a name="select-skype-for-business-server-2019-as-next-hop-when-creating-a-trusted-application-server"></a>Выберите Skype для бизнеса Server 2019 г. в качестве следующего перехода при создании сервера доверенных приложений

1. Откройте построитель топологий.
    
2. В левой области щелкните правой кнопкой мыши **доверенные** серверы приложений и нажмите **кнопку New Trusted Application Pool**.
    
3. Введите **FQDN** пула доверенного пула приложений и выберите, будет ли это одно-сервер или несколько серверов. 
    
4. Нажмите кнопку **Далее**.
    
5. На **следующей странице перехода** выберите пул Skype для бизнеса Server 2019. 
    
6. Нажмите кнопку **Готово**.
    
7. Выберите верхний узел **Skype для бизнеса Server,** а из меню **Действия** выберите **Опубликовать**.
    
    Убедитесь, **что пул** доверенных приложений был успешно создан и связан с правильным пулом переднего конца.
