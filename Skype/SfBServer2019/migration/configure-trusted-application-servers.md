# <a name="configure-trusted-application-servers"></a>Настройка доверенных серверов приложений

В смешанной среде Если вы создаете новый доверенный сервер приложений, необходимо задать пул следующих прыжков быть Скайп для пула Business Server 2019. В смешанной среде устаревшего пула и Скайп для пула Business Server 2019 отображаются в раскрывающемся списке. Выбор устаревшего пула не поддерживается.
  
> [!IMPORTANT]
> При переносе сервера доверенных приложений, следует также обновить версии UCMA, вы используете. При создании нового пула доверенных приложений для Скайп для Business Server 2019 UCMA должно обновить до версии, который входит в состав Скайп для Business Server 2019 или последняя доступная версия. 
  
### <a name="select-skype-for-business-server-2019-as-next-hop-when-creating-a-trusted-application-server"></a>Выберите Скайп для Business Server 2019 в качестве узла следующего перехода при создании сервера доверенных приложений

1. Откройте построитель топологий.
    
2. В левой области щелкните правой кнопкой мыши **серверы доверенных приложений** и нажмите кнопку **Создать пул доверенных приложений**.
    
3. Введите **Полное доменное имя пула** пула доверенных приложений и выберите будет ли это одним или несколькими серверами. 
    
4. Нажмите **Далее**.
    
5. На странице **Выбор следующего прыжка** в списке выберите Скайп для пула переднего плана Business Server 2019. 
    
6. Нажмите кнопку **Готово **.
    
7. Выберите верхний узел **Скайп для Business Server**и в меню **Действие** выберите **Опубликовать**.
    
    Убедитесь, что **Пул доверенных приложений** успешно создан и связан с соответствующим интерфейсным пулом.
