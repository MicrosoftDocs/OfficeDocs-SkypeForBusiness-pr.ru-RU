# <a name="prepare-active-directory-for-skype-for-business-server"></a>Подготовка Active Directory для Скайп для Business Server

Перед развертыванием Скайп для Business Server 2019 в состоянии сосуществования, необходимо выполнить некоторые дополнительные задачи Active Directory для настройки схемы, домена и леса для Скайп для Business Server 2019. Расширения схемы добавьте Active Directory классы и атрибуты, которые необходимы для Business Server 2019 Скайп. <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>Для подготовки Active Directory для Скайп для Business Server 2019

1. Запустите на Скайп для сервера переднего плана Business Server 2019, Скайп для установки 2019 Business Server.
    
2. Выберите **подготовки Active Directory**.
  
3. Выполните шаги 1-5 в мастере.
