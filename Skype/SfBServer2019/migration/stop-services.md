# <a name="stop-legacy-services"></a>Остановка устаревших служб

Панель управления Skype для бизнеса Server для запуска или остановки всех устаревших служб, работающих на определенном компьютере, или для запуска или остановки определенной устаревшей службы.
  
## <a name="to-start-or-stop-all-skype-for-business-server-services-on-a-computer"></a>Запуск или остановка всех Skype для бизнеса Server на компьютере

1. Откройте панель Skype для бизнеса Server управления.
    
2. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
    
3. На странице **Status** (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его. 
    
4. Щелкните элемент **Action** (Действие).
    
5. Щелкните **Start All services** (Запустить все службы) или **Stop All services** (Остановить все службы).
    
## <a name="to-start-or-stop-a-specific-service"></a>Запуск или остановка конкретной службы

1. Откройте панель Skype для бизнеса Server управления.
    
2. В левой панели навигации нажмите **кнопку Топология** и нажмите кнопку **Состояние**.
    
3. На странице **Status** (Состояние) отсортируйте список или выполните по нему поиск, чтобы найти компьютер с требуемыми службами, а затем щелкните его. 
    
4. Щелкните элемент **Properties** (Свойства).
    
5. Отсортируйте список служб, если это необходимо, и щелкните службу, которую хотите запустить или остановить.
    
6. Щелкните элемент **Action** (Действие).
    
7. Щелкните **Start service** (Запустить службу) или **Stop service** (Остановить службу).
    
8. Нажмите кнопку **Закрыть**.
