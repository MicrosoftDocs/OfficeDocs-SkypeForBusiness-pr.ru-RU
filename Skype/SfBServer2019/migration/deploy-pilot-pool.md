### <a name="to-deploy-a-skype-for-business-server-2019-pilot-pool"></a><span data-ttu-id="7c454-115">Развертывание пилотного пула Skype для бизнеса Server 2019</span><span class="sxs-lookup"><span data-stu-id="7c454-115">To deploy a Skype for Business Server 2019 pilot pool</span></span>

1. <span data-ttu-id="7c454-116">Войдите в систему компьютера, на котором построитель топологий установлен как член группы администраторов доменов и группы RTCUniversalServerAdmins.</span><span class="sxs-lookup"><span data-stu-id="7c454-116">Log on to the computer where Topology Builder is installed as a member of the Domain Admins group and the RTCUniversalServerAdmins group.</span></span>
    
2. <span data-ttu-id="7c454-117">Расширь дерево, пока не дойдете до **пулов переднего конца Skype для бизнеса Server 2019**  >  **Enterprise Edition.**</span><span class="sxs-lookup"><span data-stu-id="7c454-117">Expand the tree until you reach **Skype for Business Server 2019** > **Enterprise Edition Front End pools**.</span></span>
    
3. <span data-ttu-id="7c454-118">Правой кнопкой мыши **корпоративный выпуск передних конечных пулов** и выберите **Новый пул переднего конца**.</span><span class="sxs-lookup"><span data-stu-id="7c454-118">Right-click **Enterprise Edition Front End pools** and select **New Front End Pool**.</span></span>
  
4. <span data-ttu-id="7c454-119">Введите полное доменное имя пула (FQDN).</span><span class="sxs-lookup"><span data-stu-id="7c454-119">Enter the pool fully qualified domain name (FQDN).</span></span> <span data-ttu-id="7c454-120">При выборе пилотного пула можно развернуть пул переднего конца enterprise Edition или сервер Standard Edition.</span><span class="sxs-lookup"><span data-stu-id="7c454-120">When you define your pilot pool, you can choose to deploy an Enterprise Edition Front End pool or a Standard Edition server.</span></span> <span data-ttu-id="7c454-121">Skype для бизнеса Server 2019 не требует, чтобы функции пилотного пула совпадали с тем, что было развернуто в вашем устаревшем пуле.</span><span class="sxs-lookup"><span data-stu-id="7c454-121">Skype for Business Server 2019 does not require that your pilot pool features match what was deployed in your legacy pool.</span></span>
    
    > [!CAUTION]
    > <span data-ttu-id="7c454-122">Пул или сервер FQDN, который вы определяете для пилотного пула, должен быть уникальным.</span><span class="sxs-lookup"><span data-stu-id="7c454-122">The pool or server FQDN that you define for the pilot pool must be unique.</span></span> <span data-ttu-id="7c454-123">Он не может совпадать с именем развернутого в настоящее время устаревшего пула или других серверов, развернутых в настоящее время.</span><span class="sxs-lookup"><span data-stu-id="7c454-123">It cannot match the name of the currently deployed legacy pool or any other servers currently deployed.</span></span> 
  
5. <span data-ttu-id="7c454-124">На странице **Выбор компонентов** установите флажки рядом с теми компонентами, которые вы хотите добавить в этот интерфейсный пул.</span><span class="sxs-lookup"><span data-stu-id="7c454-124">On the **Select features** page, select the check boxes for the features that you want on this Front End pool.</span></span> <span data-ttu-id="7c454-125">Например, если вы развертываете только функции обмена мгновенными сообщениями (IM) и присутствия, вы выберите флажок Conferencing, чтобы разрешить многопартийный чат, но вы не будете выбирать диалоговые (PSTN) conferencing, Корпоративная голосовая связь или call Admission Control, так как они представляют функции голосового, видео и совместной работы.</span><span class="sxs-lookup"><span data-stu-id="7c454-125">For example, if you are deploying only instant messaging (IM) and presence features, you would select the Conferencing check box to allow multiparty IM, but you would not select the Dial-in (PSTN) conferencing, Enterprise Voice, or Call Admission Control check boxes, because they represent voice, video, and collaborative conferencing features.</span></span> <!-- For additional information on selecting features, see

8. <span data-ttu-id="7c454-132">На странице **Выбор сервера Office Web Apps** выберите **Создать** и укажите полное доменное имя сервера приложений.</span><span class="sxs-lookup"><span data-stu-id="7c454-132">On the **Select an Office Web Apps Server** page, click **New**, and specify the FQDN of the application server.</span></span>

10. <span data-ttu-id="7c454-134">Чтобы опубликовать топологию, щелкните правой кнопкой мыши **узел Skype для бизнеса Server** и нажмите кнопку Опубликовать **топологию**.</span><span class="sxs-lookup"><span data-stu-id="7c454-134">To publish your topology, right-click the **Skype for Business Server** node, and then click **Publish Topology**.</span></span>
