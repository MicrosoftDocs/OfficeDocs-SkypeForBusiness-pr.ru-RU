### <a name="to-deploy-a-skype-for-business-server-2019-pilot-pool"></a><span data-ttu-id="4a84b-115">Развертывание пилотного пула Skype для бизнеса Server 2019</span><span class="sxs-lookup"><span data-stu-id="4a84b-115">To deploy a Skype for Business Server 2019 pilot pool</span></span>

3. <span data-ttu-id="4a84b-118">Щелкните правой кнопкой мыши **интерфейсные пулы Enterprise Edition** и выберите **создать пул переднего плана**.</span><span class="sxs-lookup"><span data-stu-id="4a84b-118">Right-click **Enterprise Edition Front End pools** and select **New Front End Pool**.</span></span>
