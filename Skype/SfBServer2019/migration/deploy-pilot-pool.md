### <a name="to-deploy-a-skype-for-business-server-2019-pilot-pool"></a><span data-ttu-id="4f291-115">Чтобы развернуть Скайп для пилотного пула Business Server 2019</span><span class="sxs-lookup"><span data-stu-id="4f291-115">To deploy a Skype for Business Server 2019 pilot pool</span></span>

1. <span data-ttu-id="4f291-116">Войдите на компьютер, где установлен построитель топологий, с использованием учетной записи, входящей в группу администраторов домена и группу RTCUniversalServerAdmins.</span><span class="sxs-lookup"><span data-stu-id="4f291-116">Log on to the computer where Topology Builder is installed as a member of the Domain Admins group and the RTCUniversalServerAdmins group.</span></span>
    
2. <span data-ttu-id="4f291-117">Разверните дерево, пока не достигните **Скайп для Business Server 2019** > **Пулы переднего плана Enterprise Edition**.</span><span class="sxs-lookup"><span data-stu-id="4f291-117">Expand the tree until you reach **Skype for Business Server 2019** > **Enterprise Edition Front End pools**.</span></span>
    
3. <span data-ttu-id="4f291-118">Щелкните правой кнопкой мыши **Пулы переднего плана Enterprise Edition** и выберите **Создать пул переднего плана**.</span><span class="sxs-lookup"><span data-stu-id="4f291-118">Right-click **Enterprise Edition Front End pools** and select **New Front End Pool**.</span></span>
  
4. <span data-ttu-id="4f291-119">Введите полное доменное имя пула (FQDN).</span><span class="sxs-lookup"><span data-stu-id="4f291-119">Enter the pool fully qualified domain name (FQDN).</span></span> <span data-ttu-id="4f291-120">При определении пилотного пула можно развернуть для пула переднего плана Enterprise Edition или сервера Standard Edition.</span><span class="sxs-lookup"><span data-stu-id="4f291-120">When you define your pilot pool, you can choose to deploy an Enterprise Edition Front End pool or a Standard Edition server.</span></span> <span data-ttu-id="4f291-121">Скайп для Business Server 2019 не требуется, что функций пилотного пула равны, что было развернуто в устаревшем пуле.</span><span class="sxs-lookup"><span data-stu-id="4f291-121">Skype for Business Server 2019 does not require that your pilot pool features match what was deployed in your legacy pool.</span></span>
    
    > [!CAUTION]
    > <span data-ttu-id="4f291-122">Пул или сервер полное доменное имя, определенное для пилотного пула должно быть уникальным.</span><span class="sxs-lookup"><span data-stu-id="4f291-122">The pool or server FQDN that you define for the pilot pool must be unique.</span></span> <span data-ttu-id="4f291-123">Он не может совпадать с именем уже развернутого пула прежних версий или других в настоящее время развернуты серверы.</span><span class="sxs-lookup"><span data-stu-id="4f291-123">It cannot match the name of the currently deployed legacy pool or any other servers currently deployed.</span></span> 
  
5. <span data-ttu-id="4f291-124">На странице " **Выбор компонентов** " установите флажки для компонентов, которые будут на этот пул переднего плана.</span><span class="sxs-lookup"><span data-stu-id="4f291-124">On the **Select features** page, select the check boxes for the features that you want on this Front End pool.</span></span> <span data-ttu-id="4f291-125">Например при развертывании только мгновенные сообщения (IM) и сведений о присутствии, следует выбрать флажок конференц-связи, чтобы разрешить многосторонний обмен мгновенными Сообщениями, но (общего пользования PSTN) конференций, корпоративной голосовой связи, не следует выбрать или проверьте контроля допуска звонков диалоговые окна, так как они представляют голосовых, видео-и совместной работы функций.</span><span class="sxs-lookup"><span data-stu-id="4f291-125">For example, if you are deploying only instant messaging (IM) and presence features, you would select the Conferencing check box to allow multiparty IM, but you would not select the Dial-in (PSTN) conferencing, Enterprise Voice, or Call Admission Control check boxes, because they represent voice, video, and collaborative conferencing features.</span></span> <!-- For additional information on selecting features, see

6. <span data-ttu-id="4f291-126">На странице **Выбор серверов с совмещенным расположением ролей** рекомендуется выбирать совместно размещать в Скайп сервера-посредника для Business Server 2019.</span><span class="sxs-lookup"><span data-stu-id="4f291-126">On the **Select collocated server roles** page, we recommend that you choose to collocate the Mediation Server in Skype for Business Server 2019.</span></span> <span data-ttu-id="4f291-127">При объединении топологии прежних версий с Скайп для Business Server 2019, требуется сначала совместно размещать устаревшего сервера-посредника.</span><span class="sxs-lookup"><span data-stu-id="4f291-127">When merging a legacy topology with Skype for Business Server 2019, we require that you first collocate the legacy Mediation Server.</span></span> <span data-ttu-id="4f291-128">Слияние топологий и настройка Скайп для сервера-посредника 2019 Business Server, могут принять решение о оставьте выровненных сервера-посредника, или измените его на изолированном сервере при перемещении роль сервера-посредника Скайп для Business Server 2019 далее в процессе развертывания.</span><span class="sxs-lookup"><span data-stu-id="4f291-128">After merging the topologies and configuring the Skype for Business Server 2019 Mediation Server, you can decide whether to keep the collocated Mediation Server, or change it to a stand-alone server when you move the Mediation Server role to Skype for Business Server 2019 later in the deployment process.</span></span>

8. <span data-ttu-id="4f291-132">На странице " **Выбор сервера Office Web Apps** " щелкните **Создать**и укажите полное доменное имя сервера приложений.</span><span class="sxs-lookup"><span data-stu-id="4f291-132">On the **Select an Office Web Apps Server** page, click **New**, and specify the FQDN of the application server.</span></span>

10. <span data-ttu-id="4f291-134">Чтобы опубликовать топологию, щелкните правой кнопкой мыши узел **Скайп для Business Server** и нажмите кнопку **Опубликовать топологию**.</span><span class="sxs-lookup"><span data-stu-id="4f291-134">To publish your topology, right-click the **Skype for Business Server** node, and then click **Publish Topology**.</span></span>
