# <a name="download-topology-from-existing-deployment"></a>загрузка топологии из существующего развертывания;

При создании пула Skype для бизнеса Server 2019 вы будете использовать центральное хранилище управления, связанное с установкой прежних версий. При запуске построителя топологий при первом использовании и в последующих сеансах редактирования, отображается запрос на ввод расположения, из которого построитель топологий должен загрузить текущий документ конфигурации. Так как вы уже определили топологию и установили центральное хранилище управления, необходимо выбрать загрузку топологии из существующего развертывания. Построитель топологии прочитает базу данных и извлечет текущее определение. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Загрузка топологии из существующего развертывания

1. Откройте мастер развертывания Skype для бизнеса Server.
    
2. На странице **мастера развертывания Skype для бизнеса Server 2019** и нажмите кнопку **установить средства администрирования**.
    
3. Запустите построитель топологий: нажмите кнопку **Пуск**, выберите **все программы**, затем **Microsoft Skype для бизнеса Server 2019**и щелкните **Построитель топологий Skype для бизнеса Server**.
    
4. Выберите пункт **Download Topology from existing deployment** (Загрузить топологию из существующего развертывания).
  
5. Выберите имя файла и сохраните топологию с типом файла Default. тбксмл.
    
6. Разверните узел Skype для бизнеса Server, чтобы отобразить различные роли сервера в развертывании.
