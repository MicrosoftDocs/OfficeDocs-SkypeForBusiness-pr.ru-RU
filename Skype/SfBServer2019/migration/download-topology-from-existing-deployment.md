# <a name="download-topology-from-existing-deployment"></a>Загрузка топологии из существующего развертывания

При создании пула Skype для бизнеса Server 2019 используется центральное хранилище управления, связанное с устаревшей установкой. При запуске построителя топологий при первом использовании и в последующих сеансах редактирования, отображается запрос на ввод расположения, из которого построитель топологий должен загрузить текущий документ конфигурации. Так как топология уже определена и установлено центральное хранилище управления, следует выбрать загрузку топологии из существующего развертывания. Построитель топологии прочитает базу данных и извлечет текущее определение. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Загрузка топологии из существующего развертывания

1. Откройте мастер развертывания Skype для бизнеса Server.
    
2. На странице мастера развертывания Skype для **бизнеса Server 2019** щелкните **"Установка средств администрирования".**
    
3. Запустите построитель топологий: нажмите кнопку "Начните", выберите "Все программы", "Microsoft Skype для бизнеса Server **2019",** а затем щелкните построитель топологий Skype для бизнеса **Server.**
    
4. Выберите пункт **Download Topology from existing deployment** (Загрузить топологию из существующего развертывания).
  
5. Выберите имя файла и сохраните топологию с типом файла .tbxml по умолчанию.
    
6. Разкрой узел Skype для бизнеса Server, чтобы показать различные роли серверов в развертывании.
