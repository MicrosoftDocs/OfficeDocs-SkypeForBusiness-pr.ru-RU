# <a name="download-topology-from-existing-deployment"></a>загрузка топологии из существующего развертывания;

При создании Скайп для пула Business Server 2019, необходимо использовать центрального хранилища управления, которая связана с установкой прежних версий. При запуске сеансов Topology Builder при первом вызове и последующие изменения, вам предлагается место Topology Builder для загрузки текущей конфигурации документа. Поскольку для определенных топология создано центрального хранилища управления, следует выбрать загрузка топологии из существующего развертывания. Topology Builder будет считывать базы данных и извлечения текущего определения. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Загрузка топологии из существующего развертывания

1. Откройте Скайп для мастер развертывания Business Server.
    
2. На странице **Скайп для 2019 сервера бизнес - мастер развертывания** нажмите кнопку **Установить средства администрирования**.
    
3. Запустите построитель топологий: Нажмите кнопку **Пуск**, последовательно выберите пункты **Все программы**, щелкните **Скайп Microsoft для Business Server 2019**и нажмите кнопку **Скайп для Business Server Topology Builder**.
    
4. Выберите **Загрузить топологию из существующего развертывания**.
  
5. Выберите имя файла и сохраните топологию с типом файла tbxml по умолчанию.
    
6. Разверните узел Скайп для узла Business Server, чтобы отобразить различные роли сервера в развертывании.
