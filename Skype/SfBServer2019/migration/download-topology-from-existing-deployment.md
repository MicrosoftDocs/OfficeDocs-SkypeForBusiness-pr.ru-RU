# <a name="download-topology-from-existing-deployment"></a>загрузка топологии из существующего развертывания;

При создании пула 2019 в Skype для бизнеса Server вы будете использовать центральное хранилище, связанное с установленной устаревшей установкой. Когда вы запускаете построитель топологии при первом использовании и последующих сеансах редактирования, вам будет предложено указать расположение, в которое построитель топологии должен загрузить текущий документ конфигурации. Так как вы уже определили топологию и установили хранилище Central Management, необходимо выбрать загрузку топологии из существующего развертывания. Построитель топологии будет считывать базу данных и получать текущее определение. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Загрузка топологии из существующего развертывания

1. Запустите мастер развертывания Skype для бизнеса Server.
    
2. На странице **мастера развертывания Skype для бизнеса Server 2019** нажмите **установить средства администрирования**.
    
3. Запустить построитель топологии: нажмите кнопку **Пуск**, выберите **все программы**, а затем — **Microsoft Skype для бизнеса Server 2019**, а затем — построитель **топологии Skype для бизнеса Server**.
    
4. Выберите пункт **топология скачивания из существующего развертывания**.
  
5. Выберите имя файла и сохраните топологию с типом файла Default. тбксмл.
    
6. Разверните узел сервера Skype для бизнеса, чтобы увидеть различные роли сервера в развертывании.
