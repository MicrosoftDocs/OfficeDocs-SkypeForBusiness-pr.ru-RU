# <a name="download-topology-from-existing-deployment"></a>Загрузка топологии из существующего развертывания

При создании пула Skype для бизнеса Server 2019 года используется Центральный магазин управления, связанный с устаревшей установкой. При запуске построителя топологий при первом использовании и в последующих сеансах редактирования, отображается запрос на ввод расположения, из которого построитель топологий должен загрузить текущий документ конфигурации. Так как у вас уже определена топология и создан центр управления, необходимо скачать топологию из существующего развертывания. Построитель топологии прочитает базу данных и извлечет текущее определение. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Загрузка топологии из существующего развертывания

1. Откройте мастер Skype для бизнеса Server развертывания.
    
2. На странице **Мастер развертывания Skype для бизнеса Server 2019 г.** нажмите **кнопку Установить административные средства.**
    
3. Начните строитель топологии: нажмите кнопку Начните, нажмите кнопку Все программы **,** нажмите Microsoft **Skype для бизнеса Server 2019**, а затем нажмите кнопку Skype для бизнеса Server **Топология Builder**.
    
4. Выберите пункт **Download Topology from existing deployment** (Загрузить топологию из существующего развертывания).
  
5. Выберите имя файла и сохраните топологию с типом файла .tbxml по умолчанию.
    
6. Расширь Skype для бизнеса Server, чтобы выявить различные роли сервера в развертывании.
