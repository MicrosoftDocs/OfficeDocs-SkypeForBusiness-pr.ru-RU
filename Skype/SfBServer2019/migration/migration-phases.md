# <a name="migration-phases"></a>Этапы миграции

В Skype для бизнеса Server 2019 г. в сети определяются сайты, содержащие Skype для бизнеса Server 2019 г. Сайт — это набор компьютеров, которые подключены к высокоскоростной сети с небольшой задержкой, например к одной локальной сети (LAN), или две сети, соединенные высокоскоростной оптоволоконной сетью. 
  
Пул переднего конца — это набор серверов переднего конца, которые настроены одинаково и работают вместе, чтобы предоставлять службы для общей группы пользователей. Пул предоставляет пользователям возможности масштабирования и сбойной работы. Каждый сервер в пуле должен содержать идентичные роли сервера. Сервер выпуск Standard, предназначенный для небольших организаций, также определяет пул и запускается на одном сервере. Это позволяет иметь Skype для бизнеса Server 2019 за меньшие затраты, но не обеспечивает истинное решение с высокой доступностью. 
  
На следующих этапах описывается процесс переноса пула на Skype для бизнеса Server 2019 г. Для нескольких сайтов, содержащих несколько пулов, каждый отдельный пул должен следовать этому поэтапному подходу.
