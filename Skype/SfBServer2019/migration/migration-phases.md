# <a name="migration-phases"></a>Этапы миграции

В Skype для бизнеса Server 2019 определяются сайты в сети, содержащие компоненты Skype для бизнеса Server 2019. Сайт — это набор компьютеров, которые подключены к высокоскоростной сети с небольшой задержкой, например к одной локальной сети (LAN), или две сети, соединенные высокоскоростной оптоволоконной сетью. 
  
Пул переднего сервера — это набор серверов переднего сервера, которые настроены одинаково и работают вместе для предоставления служб для общей группы пользователей. Пул предоставляет пользователям возможности масштабируемости и от сбойов. Каждый сервер в пуле должен содержать идентичные роли сервера. Сервер Standard Edition, предназначенный для небольших организаций, также определяет пул и работает на одном сервере. Это позволяет использовать функции Skype для бизнеса Server 2019 за меньшие затраты, но не предоставляет действительно высокодоступного решения. 
  
На следующих этапах описывается процесс переноса пула в Skype для бизнеса Server 2019. Для нескольких сайтов, содержащих несколько пулов, каждый отдельный пул должен следовать этому поэтапному подходу.
