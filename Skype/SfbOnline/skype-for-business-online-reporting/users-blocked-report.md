# <a name="users-blocked-report"></a>Отчет о заблокированных пользователях

На новой панели  мониторинга отчетов Skype для бизнеса представлен обзор действий в продуктах Skype для бизнеса в вашей организации. С ее помощью можно детализированный отчет по отдельным продуктам, чтобы получить более подробную информацию о действиях с каждым продуктом. Например, отчет о заблокированных пользователях **Skype** для бизнеса можно использовать для того, чтобы увидеть пользователей в организации, которые не могут делать звонки по ПС. В этом отчете, а также других отчетах по Skype для бизнеса вы можете подробно и подробно известить об использовании услуг ННР в вашей организации.
  
 Дополнительные сведения [о доступных отчетах можно](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) найти в обзоре отчетов.
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса при входе в Центр администрирования Microsoft 365 в качестве администратора. 
  
## <a name="how-to-get-to-the-skype-for-business-users-blocked-report"></a>Доступ к отчету о заблокированных пользователях Skype для бизнеса

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

- Перейдите в Центр администрирования > **центре** администрирования Skype для  >    >  **бизнеса: отчеты** заблокированных  >  **пользователей.**
    
## <a name="interpret-the-skype-for-business-users-blocked-report"></a>Интерпретация отчета о заблокированных пользователях Skype для бизнеса

Чтобы получить представление о заблокированных пользователях, можно просмотреть каждый из отображаемого столбцов.
  
Отчет имеет следующий вид. 
  
![Отчет о заблокированных пользователях](../images/df50a413-7a51-4340-a59b-3f83de941762.png)

В таблице показаны данные по всем пользователям, которые не могут звонить. Здесь показаны все пользователи, которым назначена телефонная система или аудиоконференция. При необходимости можно добавлять и удалять столбцы таблицы.
***
![Номер 1](../images/sfbcallout1.png)
*   **ИД пользователя** — это вход пользователя.
*   **Номер телефона** — это номер, который назначен пользователю. 
*   **Время действия блокировки** — это время (UTC), в которое пользователю было заблокировано звонить.
*   **Блок-действие** — это тип действия, которое было предприняты для блокировки пользователя.
*   **Причина блокировки —** это причина, по которой пользователю не удалось сделать звонки.
***
![Номер 2](../images/sfbcallout2.png)<br/>
Чтобы создать представление, объединяющее все данные из одного или нескольких столбцов, перетащите столбец в раздел **Чтобы сгруппировать по определенному столбцу, перетащите сюда заголовок столбца**.
***
![Номер 3](../images/sfbcallout3.png)<br/>
Вы также можете экспортировать данные отчета в CSV-файл Excel, нажав кнопку **Экспорт в Excel**.

При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать таблицы в самом отчете. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.
***

## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Отчет об использованных клиентах Skype для бизнеса](device-usage-report.md) . В этом отчете представлены настольные (в том числе под управлением ОС Windows) и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для обмена мгновенными сообщениями и участия в собраниях.
    
- [Отчет об организации конференций в Skype для бизнеса](conference-organizer-activity-report.md) Вы можете узнать, в какой степени ваши пользователи организовывают конференции с использованием услуг обмена мгновенными и аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также внесения и иных ветвей телефонных номеров через сторонних разработчиков, а также в/из учетной записи Майкрософт.
    
- [Отчет об участниках конференции в Skype для бизнеса](conference-participant-activity-report.md) Вы можете узнать, сколько конференций с использованием услуг обмена мгновенными и аудио- и видеосвязями, общего доступа к приложениям, веб-приложений, а также в/из телефонного доступа.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Отчет об использовании ПС в Skype для бизнеса](pstn-usage-report.md) Вы увидите, затрачено ли время входящие и исходящие звонки, а также стоимость этих звонков.

- [В отчете](pstn-minute-pools-report.md) по пулам минут ЗВОНКОВ по ЗВОНКОВ по Skype для бизнеса можно увидеть количество минут, посещаемых в организации за текущий месяц.

- [Отчет о сеансе в Skype для бизнеса](session-details-report.md) Вы можете получить подробные сведения о звонках для отдельных пользователей.
