# <a name="users-blocked-report"></a>Отчет о заблокированных пользователях

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

В новой Skype для бизнеса **отчетов** вы можете получить обзор действий в Skype для бизнеса продуктов организации. С ее помощью можно детализированный отчет по отдельным продуктам, чтобы получить более подробную информацию о действиях в каждом продукте. Например, отчет о  заблокированных Skype для бизнеса пользователей можно использовать для того, чтобы увидеть пользователей в организации, которые не могут звонить по ЗВОНКОВ по ННР. В этом отчете, а также Skype для бизнеса других отчетах вы можете подробно известить об активности, в том числе об использовании услуг STN в организации.
  
 Дополнительные сведения [о доступных отчетах можно](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) найти в обзоре Отчеты.
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса при входе в Центр администрирования Microsoft 365 администратором. 
  
## <a name="how-to-get-to-the-skype-for-business-users-blocked-report"></a>Доступ к отчету о Skype для бизнеса заблокированных пользователей

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

- Перейдите в центр администрирования > **центре администрирования** Skype для бизнеса  >  **Отчеты** о  >    >  **заблокированных пользователях.**
    
## <a name="interpret-the-skype-for-business-users-blocked-report"></a>Интерпретация Skype для бизнеса заблокированных пользователей

Чтобы просмотреть заблокированных пользователей, можно просмотреть каждый из отображаемого столбцов.
  
Отчет имеет следующий вид. 
  
![Отчет о заблокированных пользователях](../images/df50a413-7a51-4340-a59b-3f83de941762.png)

В таблице показана разбивка всех пользователей, которые не могут звонить. В результате будут показаны все пользователи телефонная система которым назначена аудиоконференция или аудиоконференция. При необходимости можно добавлять и удалять столбцы таблицы.
***
![Номер 1](../images/sfbcallout1.png)
*   **ИД пользователя** — это вход пользователя.
*   **Телефон —** это номер, который назначен пользователю. 
*   **Время блокировки —** это время , в которое пользователю было заблокировано звонки (UTC).
*   **Блокировка** — это тип действия, которое было принято для блокировки пользователя.
*   **Причина блокировки —** причина, по которой пользователь не может звонить.
***
![Номер 2](../images/sfbcallout2.png)<br/>
Чтобы создать представление, объединяющее все данные из одного или нескольких столбцов, перетащите столбец в раздел **Чтобы сгруппировать по определенному столбцу, перетащите сюда заголовок столбца**.
***
![Номер 3](../images/sfbcallout3.png)<br/>
Вы также можете экспортировать данные отчета в CSV-файл Excel, нажав кнопку **Экспорт в Excel**.

При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать таблицы в самом отчете. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.
***

## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Отчет об использованных клиентах Skype для бизнеса](device-usage-report.md) . В этом отчете представлены настольные (в том числе под управлением ОС Windows) и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для обмена мгновенными сообщениями и участия в собраниях.
    
- [Skype для бизнеса отчет об организации конференций](conference-organizer-activity-report.md) Вы можете узнать, как пользователи организовывают конференции с использованием услуг обмена мгновенными мгновенными и видеосвязи, общего доступа к приложениям, веб-приложений, внесения и выходов с телефонного доступа сторонних разработчиков, а также для внесения и иния телефонного доступа (Майкрософт).
    
- [Skype для бизнеса отчет об активности участников конференции](conference-participant-activity-report.md) Вы можете узнать, сколько конференций с использованием услуг обмена мгновенными мгновенными и аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также конференц-залов с телефонным и ветвным телефонным ветвями.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Skype для бизнеса отчете об использовании ННР](pstn-usage-report.md) Вы увидите количество минут, затраченных на входящие и исходящие звонки, и стоимость этих звонков.

- [Skype для бизнеса по минутам](pstn-minute-pools-report.md) ПСПС вы можете увидеть количество минут, посещаемых в организации за текущий месяц.

- [Skype для бизнеса отчет о сеансе](session-details-report.md) Вы можете получить подробные сведения о звонках для отдельных пользователей.
