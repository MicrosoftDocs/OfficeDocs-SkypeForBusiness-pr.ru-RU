# <a name="skype-for-business-online-reporting"></a>Отчетность Skype для бизнеса Online

Как администратор вашей организации, очень удобно иметь полный список всех различных отчетов, доступных в Skype для бизнеса Online. Сюда входят все различные отчеты и данные, которые можно получить с помощью каждого отчета.
  
Чтобы получить доступ к отчетам Skype для бизнеса Online, войдите в центр администрирования Microsoft 365 с помощью учетной записи пользователя и пароля. Для просмотра отчетов учетной записи, которую вы используете для входа, необходимо назначить либо роль глобального администратора, либо делегировать роль администратора службы Skype для бизнеса.
  
> [!NOTE]
> Чтобы проверить разрешения, в центре администрирования > **пользователей** > —**Активные пользователи**. Щелкните отображаемое имя пользователя и выберите команду **изменить** , а затем на странице Свойства учетной записи нажмите кнопку **роли** .
