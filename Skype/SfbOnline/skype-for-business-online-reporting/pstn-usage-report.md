# <a name="pstn-usage-report"></a>Отчет об использовании услуг ТСОП

[] В новом центре администрирования Skype для бизнеса в разделе **Отчеты** приводится сводка операций по звонкам и аудиоконференциям в организации. В ней можно также просмотреть детализированные отчеты по отдельным продуктам. Например, в **отчете об использовании услуг ТСОП в Skype для бизнеса** можно просмотреть сведения о длительности входящих и исходящих звонков, а также их стоимости. Помимо этого, можно просмотреть сведения об использовании аудиоконференции с телефонным подключением по ТСОП и стоимости звонков, чтобы проанализировать расходы на связь в организации.
  
Другие доступные отчеты можно найти в разделе [Обзор отчетов](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) .
  
Этот отчет вместе с другими отчетами в Skype для бизнеса содержит подробные сведения о действиях, включая использование звонков в Организации. Эти сведения очень полезны при исследовании, планировании и принятии решений для бизнеса в Организации и для настройки [кредитов на связь](/microsoftteams/what-are-communications-credits) .
  
> [!NOTE]
> Вы можете просматривать все отчеты Skype для бизнеса, когда вы входите в систему в качестве администратора в центр администрирования Microsoft 365. 
  
## <a name="how-to-get-to-the-skype-for-business-pstn-usage-details-report"></a>Получение детализированного отчета об использовании услуг ТСОП в Skype для бизнеса

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

- Перейдите > **в центр** >  **администрирования >** > **центр администрирования Skype для бизнеса**:**сведения об использовании PSTN**.
    
    > [!NOTE]
    > В зависимости от имеющейся подписки на Microsoft 365 или Office 365 могут отображаться не все продукты и отчеты, показанные в этой статье.
  
## <a name="interpret-the-skype-for-business-pstn-usage-report"></a>Анализ отчета об использовании услуг ТСОП в Skype для бизнеса

Сведения об использовании услуг ТСОП пользователями Skype для бизнеса приводятся в соответствующих столбцах.
  
Отчет имеет следующий вид.
  
![Отчет об использовании услуг ТСОП в Skype для бизнеса](../images/79d7aadf-c69e-4d6a-8179-ab69dbbb2472.png)

***
![Номер 1](../images/sfbcallout1.png)<br/>В таблице показаны сведения об использовании ТСОП с разбивкой по пользователям. Здесь приводятся все пользователи, которым назначена лицензия на Skype для бизнеса, а также сведения об использовании ими ТСОП. При необходимости можно добавлять и удалять столбцы таблицы.
*    **Код звонка** – это идентификатор звонка для звонка. Это идентификатор звонка, который используется при вызове службы поддержки Майкрософт.
*    **Идентификатор пользователя**  имя, под которым пользователь вошел в систему.
*    **Номер телефона**  номер телефона Skype для бизнеса, на который поступают входящие или выполняются исходящие звонки.
*    **Расположение пользователя**  страна, в которой находится пользователь.
*    **Код звонящего**  номер телефона (код) вызывающего абонента для входящего звонка, а также номер телефона или номер Skype для бизнеса, с которого выполнен исходящий звонок.
*    **Тип звонка**  сведения о том, выполнен ли входящий или исходящий звонок по ТСОП, а также информация о типе звонка (например, звонок выполнен пользователем или по аудиоконференции). Могут отображаться следующие типы звонка: 

     **Типы звонка в плане звонков** 
     *    **user_in** (пользователь принял входящий звонок по ТСОП). 
     *    **user_out** (пользователь выполнил исходящий звонок по ТСОП). 
     *    **user_out_conf** (пользователь добавил несколько участников ТСОП к звонку, например к трехсторонней конференции). 
     *    **user_out_transfer** (пользователь переключил звонок на номер ТСОП). 
     *    **user_out_forwarding** (пользователь переадресовал звонок на номер ТСОП).

     **Типы звонков аудиоконференций**
     *    **conf_in** (входящий звонок в мост для видеоконференций). Для записей этого типа вызова пользователь, указанный в столбце " **идентификатор пользователя** ", соответствует организатору собрания.
     *    **conf_out** (исходящий звонок из моста видеоконференций, как правило, для добавления PSTN-номера для Конференции). Для записей этого типа вызова пользователь, указанный в столбце " **идентификатор пользователя** ", соответствует организатору собрания.

     **Приложения объединенных коммуникаций (UCAP)** 
     *    **ucap_in** (входящий PSTN-Звонок в приложение UC, например автосекретарь или очередь звонков); 
     *    **ucap_out** (исходящий звонок PSTN из приложения UC, такого как автосекретарь или очередь звонков);
     *    **Примечание.** Звонки, переданные пользователю из приложения UC, такого как автосекретарь или очередь звонков, не отображаются в отчете об использовании PSTN, так как эти настройки являются однорангными (P2P) голосовые звонки. Вы можете получить доступ к звонкам P2P в центре администрирования Skype для бизнеса в разделе "инструменты > средства аналитики звонков Skype для бизнеса" и поиск по имени пользователя или SIP-адресу, которые сопоставляют вызов по дате/времени и/или исходной CLID (идентификатор вызывающей строки). 
*     
     **Домашний/международный**  сведения о том, был ли выполнен звонок внутри страны/региона пользователя (домашний) или за ее пределами (международный), регистрируемые на основе информации о расположении пользователя. 
*    **Назначение набранного номера**  страна или регион, к которым относится набранный номер, например Франция, Германия или США. 
*    **Тип номера** — тип номера телефона, номер телефона пользователя, услуга или бесплатный телефон.  
*    **Время начала (в формате UTC)**  время начала звонка. 
*    **Длительность**  продолжительность звонка.  
*    **ConfID** с качестве идентификатора используется код аудиоконференции. 
*    **Стоимость**  стоимость звонка, списанная с вашего счета. 
*    **Валюта**  валюта, в которой рассчитывается стоимость звонка. 
*    **Возможность**  лицензия, которая была использована при обработке звонка. Могут отображаться следующие типы лицензий: 
     *    **MCOPSTNPP**  кредиты на связь <br/> **MCOPSTN1**  план внутренних звонков (планы: 3000 мин США/1200 мин ЕС) 
     *    **MCOPSTN2**  план международных звонков 
     *    **MCOPSTN5**  план внутренних звонков (план звонков: 120 мин) 
     *    **MCOPSTN6** — план внутренних звонков (план 240 мин) Примечание: ограниченная доступность
     *    **MCOMEETADD**  аудиоконференция
     *    **MCOMEETACPEA**  аудиоконференция с поминутной оплатой
     
> [!NOTE]
> Если вы хотите, чтобы при запуске отчета выводились только платежи за Поминутные звонки, не включенные в подписку на звонки или конференц-связь, отфильтруйте отчет с возможностью "MCOPSTNPP". Это обеспечит подписку на все звонки за минуту.  Для звонков на голосовую конференцию с поминутной оплатой используйте фильтр по "MCOMEETACPEA", а не "MCOPSTNPP".  

> [!NOTE]
> В некоторых полях Вы также можете увидеть "нет данных". "Нет данных" означает, что поле неприменимо к типу звонка или возможности. 

> [!NOTE]
> Если у вас есть план звонков по Telstra, вы не увидите ни одной записи с подробными сведениями о звонке в отчете использование КТСОП. Обратитесь в Telstra за нужные сведения.

## <a name="exporting-pstn-usage-report"></a>Экспорт отчета об использовании PSTN

Нажав кнопку **Экспорт в Excel** , вы можете скачать отчет об использовании КТСОП. Вы можете экспортировать данные за год в течение года с текущей даты, если нормативные правила не зависят от специфики страны в течение 12 месяцев.

При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа.

Процесс экспорта может занять от нескольких секунд до нескольких минут, в зависимости от количества данных. Когда сервер завершит экспорт, вы получите ZIP-файл с именем**Calls. Export. [ ] `identifier`. zip**", идентификатор которого является уникальным идентификатором для экспорта, который можно использовать для устранения неполадок.

Если у вас есть планы звонков и прямая маршрутизация, экспортируемый файл может содержать данные для обоих продуктов. В файле отчета об использовании КТСОП будет указано filename "**КТСОП. Calls. [ ] `UTC date`. csv**". В дополнение к файлам PSTN и Direct Routing Files Архив имеет файл "**Parameters. JSON**" с выбранным диапазоном времени экспорта и возможностями (если есть).

Экспортированный файл — это файл данных с разделителями-запятыми (CSV), соответствующий стандарту [RFC 4180](https://tools.ietf.org/html/rfc4180) . Файл можно открыть в Excel или в любом другом стандартном редакторе, не требуя каких-либо преобразований.

В первой строке CSV-файла содержатся имена столбцов.
