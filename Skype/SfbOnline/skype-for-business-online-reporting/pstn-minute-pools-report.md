# <a name="pstn-minute-pools-report"></a>Отчет о пулах минут, затраченных по ТСОП

>[!NOTE]
>Этот отчет доступен только для пользователей с предварительными версиями.

[] В новом центре администрирования Skype для бизнеса в разделе **Отчеты** приводится сводка операций по звонкам и аудиоконференциям в организации. В ней можно также просмотреть детализированные отчеты по отдельным продуктам. Например, вы можете воспользоваться отчетом по минутам ЗВОНКОВ по **Skype** для бизнеса, чтобы узнать количество минут, посещаемых в организации за текущий месяц.
  
Дополнительные сведения [о доступных отчетах можно](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) найти в обзоре отчетов.
  
Из этого и других отчетов по Skype для бизнеса вы можете подробно и подробно известить об активности в вашей организации. Эти сведения полезны при исследование, планировании и принятии других решений на предприятии, а также при настройке кредитов на [связь](/microsoftteams/what-are-communications-credits)
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса, войдите в Центр администрирования Microsoft 365 в качестве администратора. 
  
## <a name="how-to-get-to-the-skype-for-business-pstn-minute-pools-report"></a>Как получить отчет по пулам минут ЗВОНКОВ по ПС в Skype для бизнеса

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

- Перейдите в Центр администрирования > центрах администрирования Skype для бизнеса, чтобы получить отчеты по пулам минут ЗВОНКОВ по  >    >    >  **ДНР.**
    
> [!NOTE]
> В зависимости от подписки на Microsoft 365 или Office 365, которые вы можете увидеть другие сведения, показанные здесь. 
  
## <a name="interpret-the-skype-for-business-pstn-minute-pools-report"></a>Интерпретация отчета по пулам минут ЗВОНКОВ по ПС в Skype для бизнеса

Вы можете просмотреть пул минут на звонки в Skype для бизнеса своего пользователя, просмотрев отображаемые столбцы.
  
Отчет имеет следующий вид.
  
## 

![Отчет по пулам минут ЗВОНКОВ по ПС в Skype для бизнеса](../images/f5da5ca9-3466-4234-8f33-ab50ac5eb781.png)
  
***
![Номер 1](../images/sfbcallout1.png)<br/>В таблице показаны минутные пулы с разбивкой по лицензиям (возможностям) и местам использования. 
*    **Возможность —** это план лицензирования и обслуживания, используемый для звонка. В этом отчете можно увидеть следующие планы лицензий и обслуживания:
     * MCOPSTN1 — план внутренних звонков (планы ЕС на 3000 минут в США/1200 минут)
     * MCOPSTN2 — план внутренних & международных звонков, из которого вы увидите внутренний пул (3000-минутный пул США/Канада/PR, 1200-минутные европейские страны) и международный пул (600 минут). Поминутное время достигается каждый раз, когда в календарный месяц достигается внутреннее (ИЛИ) международное время. 
     * MCOPSTN5 — план внутренних звонков (план звонков на 120 минут)
     * MCOPSTN6 — план внутренних звонков (план звонков на 240 минут)
     * MCOMEETADD  аудиоконференция
*    **Описание возможности —** это описание типа лицензии, который использовался для вызова.
*    **Пул минут на страны** — это место использования лицензий пользователями, у которых есть пул минут. 
*    **Использованные минуты** — это количество минут, используемых ежемесячно.
*    **Общее количество** минут — это общее количество минут, доступное в течение месяца. 
*    **Процент использования** — это процент минут, использованных для месяца. 
***
![Номер 2](../images/sfbcallout2.png)<br/>Чтобы создать представление, объединяющее все данные из одного или нескольких столбцов, перетащите столбец в раздел **Чтобы сгруппировать по определенному столбцу, перетащите сюда заголовок столбца**. 
***
![Номер 3](../images/sfbcallout3.png)<br/>Вы также можете экспортировать данные отчета в CSV-файл Excel, нажав кнопку **Экспорт в Excel**. <br/><br/> При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать таблицы в самом отчете. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.
   
## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Отчет об использованных клиентах Skype для бизнеса](device-usage-report.md) . В этом отчете представлены настольные (в том числе под управлением ОС Windows) и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для обмена мгновенными сообщениями и участия в собраниях.
    
- [Отчет об организации конференций в Skype для бизнеса](conference-organizer-activity-report.md) . В этом отчете приводятся сведения об организации конференций с использованием услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также входящих и исходящих телефонных подключений через систему корпорации Майкрософт или стороннего поставщика услуг.
    
- [Отчет об участии в конференциях в Skype для бизнеса](conference-participant-activity-report.md) . В этом отчете приведены сведения об участии в конференциях с использованием услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также исходящих телефонных подключений в аудиоконференции.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Отчет о блокировке пользователей Skype для бизнеса](users-blocked-report.md) Вы увидите пользователей в организации, которые не могут звонить по ПС.

- [Отчет о сеансе в Skype для бизнеса](session-details-report.md) Вы можете получить подробные сведения о звонках для отдельных пользователей.
