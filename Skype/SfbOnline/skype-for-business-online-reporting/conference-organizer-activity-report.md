# <a name="conference-organizer-activity-report"></a>Отчет об организации конференций

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

На **панели мониторинга** Отчеты представлен обзор действий Microsoft 365 или Office 365 продуктов в организации. С ее помощью можно детализированно придать отдельным продуктам отчеты, чтобы получить более подробную информацию о действиях в каждом продукте. Например, с помощью отчета об организации **Skype для бизнеса** конференций можно узнать, в какой степени пользователи организовывают конференции с использованием услуг обмена мгновенными мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также внесения и выходов с телефонного доступа сторонних разработчиков, а также для внесения и иния телефонного доступа (Майкрософт). 

Дополнительные сведения можно [найти в обзоре отчетов.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)
  
В этом отчете, а также Skype для бизнеса других отчетах вы можете подробно и подробно известить об организации конференций. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации. 
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса при входе в систему в качестве администратора Центр администрирования Microsoft 365. 
  
## <a name="how-to-get-to-the-skype-for-business-conference-organizer-report"></a>Получение отчета об организации конференций в Skype для бизнеса

1. Перейдите в центр администрирования > **Отчеты**  >  **об использовании**.
    
2. На странице **Использование** выберите **Skype для бизнеса** в списке Выберите отчет  >  слева.  Можно также щелкнуть **мини-Skype для бизнеса** действий  и Skype для бизнеса в списке действий Skype для бизнеса **конференции.**

Чтобы получить представление о конференциях Skype для бизнеса, можно просмотреть диаграммы **Действия,** Пользователи и **Минуты.**

Здесь отображается общая продолжительность участия пользователей в конференциях с типом **Аудио/видео**, **Минуты входящих телефонных подключений (Майкрософт)** и **Минуты исходящих телефонных подключений (Майкрософт)**.

![Номер 6](../images/sfbcallout6.png)<br/>
На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси.

На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета. 
***
![Число 7](../images/sfbcallout7.png)<br/>Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на диаграмме **Активность** щелкните или нажмите **Обмен мгновенными сообщениями**, **Аудио/видео**, **Общий доступ к приложениям**, **Интернет**, **Входящие/исходящие телефонные подключения (сторонние поставщики)** или **Входящие/исходящие телефонные подключения (Майкрософт)** для просмотра только сведений, относящихся к этой категории. При выборе другого значения данные в таблице-сетке не меняются. 
***
![Число 8](../images/sfbcallout8.png)<br/>В таблице показаны сведения об организации конференций с разбивкой по пользователям. Здесь отображаются все пользователи, которым назначена лицензия на Skype для бизнеса, а также конференции, которые они организовывали. При необходимости вы можете добавить в таблицу дополнительные столбцы. 
* **Имя пользователя**  это имя, указанное пользователем. 
* **Удален**  лицензия пользователя была удалена.<br/><br/>
  > [!NOTE]
  > Действия удаленного пользователя будут отображаться в отчете до тех пор, пока он был лицензирован в течение выбранного периода. Столбец **Удален** указывает, что пользователь больше не является активным, но участвовал при формировании данных в отчете.
     
* **Дата удаления**  это дата удаления лицензии пользователя. 
* **Дата последнего действия (UTC)**  дата последнего действия, выполненного пользователем, в формате UTC. 
* **Обмен мгновенными сообщениями**  общее число конференций с использованием услуг обмена мгновенными сообщениями, которые организовывали пользователи. 
* **Аудио/видео**  общее число конференций с использованием услуг аудио- и видеосвязи, которые организовывали пользователи.
* **Общий доступ к приложениям**  общее число конференций с использованием услуг общего доступа к приложениям, которые организовывали пользователи. 
* **Интернет**  общее число веб-конференций, которые организовывали пользователи. 
* **Внося/выход** с помощью телефонного номера (сторонние поставщики) — общее число аудиоконференции с ветвями и выходами, которые организовывали сторонние поставщики аудиоконференции. 
* **В ходе телефонного и выходного** номера в Корпорации Майкрософт показано общее число аудиоконференции с телефонным и ветвным телефонным ветвями, которые организовывали. 
* **Минуты аудио- и видеосвязи**  общая продолжительность организованных конференций с использованием услуг аудио- и видеосвязи, в минутах. 
* **Минуты для телефонного** связи (Майкрософт) — общее количество минут телефонного пользования, использованных на собраниях, где в качестве поставщика аудиоконференций используется Майкрософт. 
* **Минуты на отработку** телефонного связи (Майкрософт) — общая суммарное количество минут для пользования аудиоконференцией Майкрософт на собраниях, организованных с использованием Майкрософт в качестве поставщика услуг аудиоконференций. 

Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с **разделом Как скрыть** сведения об уровне пользователя? в разделе Отчеты об [активности в Центре администрирования.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)

- [Skype для бизнеса участников конференции](conference-participant-activity-report.md) Вы можете узнать, сколько конференций с использованием услуг обмена мгновенными мгновенными и видеосвязи, общего доступа к приложениям, веб-приложений, а также конференций с ветвями и ветвями телефонного доступа.
    
- [Skype для бизнеса одноранговых действий](peer-to-peer-activity-report.md) Вы можете узнать, в какой степени пользователи используют обмен мгновенными и аудио- и видеофайлами, общий доступ к приложениям и передачу файлов.

- [Skype для бизнеса по минутам](pstn-minute-pools-report.md) ПСПС вы можете увидеть количество минут, посещаемых в организации за текущий месяц.

- [Skype для бизнеса отчет о сеансе](session-details-report.md) Вы можете получить подробные сведения о звонках отдельных пользователей.
