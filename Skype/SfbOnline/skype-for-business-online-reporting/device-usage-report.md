# <a name="device-usage-report"></a>Отчет об использовании устройств

На новой панели мониторинга " **отчеты** " для Office 365 отображаются общие сведения о действиях в продуктах Office 365 в вашей организации. С его помощью вы сможете детализировать отчеты на уровне продукта, чтобы получить более подробные сведения о действиях в каждом продукте. Например, с помощью отчета **об использовании устройств в Skype для бизнеса** можно просматривать устройства, в том числе операционные системы Windows и мобильные устройства, на которых установлено приложение Skype для бизнеса и которое используется для обмена мгновенными сообщениями и собраний. 

Дополнительные сведения можно найти в [обзоре отчетов](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) .
  
Этот отчет вместе с другими отчетами в Skype для бизнеса содержит подробные сведения о типах клиентов и устройств, которые используются в Организации. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации.
  
> [!NOTE]
> Вы можете просматривать все отчеты Skype для бизнеса, когда вы входите в систему в качестве администратора в центр администрирования Microsoft 365. 
  
## <a name="how-to-get-to-the-skype-for-business-device-usage-report"></a>Как перейти к отчету об использовании устройств в Skype для бизнеса

1. Откройте центр администрирования > **отчет** > **об использовании**.
    
2. На странице **Использование** нажмите кнопку **использование устройств в Skype для бизнеса** в **списке выберите отчет** слева. Или щелкните мини **-приложение активность Skype для бизнеса** , а затем выберите **использование устройств Skype для бизнеса** в списке **действия в Skype для бизнеса** .
    
     ![Мини-приложение отчета об использовании устройств в Skype для бизнеса](../images/13e7a51c-f680-4164-9a18-67de10fceb7f.png)
  
    Для некоторых подписок на Office 365 могут быть доступны не все продукты и отчеты об активности, показанные в этой статье.
    
## <a name="interpret-the-skype-for-business-device-usage-report"></a>Интерпретация отчета об использовании устройств в Skype для бизнеса

Вы можете получить представление на устройствах, использующих приложение Skype для бизнеса, Взгляните на диаграммы **пользователей** и **рассылок** .
  
![Отчет об использованных клиентах Skype для бизнеса](../images/a1350924-37f8-4ae3-8603-bdde8c694203.png)
  
***
![Номер 1](../images/sfbcallout1.png)<br/>С помощью отчета **об использовании устройств в Skype для бизнеса** можно просматривать тенденции за последние 7, 30, 90 и 180 дней. Тем не менее, если вы выйдете на определенный день в отчете, таблица (номер 7) будет показывать данные в течение 30 дней, вплоть до даты (см. номер 2) на момент создания отчета.

> [!NOTE]
> Если вы настроили данные за определенный день, в таблице будут показаны только даты в течение 30 дней с момента создания отчета.
    
***
![Номер 2](../images/sfbcallout2.png)<br/>
Для каждого отчета указывается дата создания. Отчеты обычно представляют собой задержку от 24 до 48 часов на время действий. 
***
![Номер 3](../images/sfbcallout3.png)<br/>Используйте интерактивные данные диаграмм на диаграмме **пользователей** , чтобы понять тенденции использования и узнать количество пользователей, подключенных с помощью приложения Skype для бизнеса, которое используется в вашей организации. Здесь показано общее количество пользователей и типов **Windows**, устройств **Windows Phone**, **телефонов с Android**, **iPhone**и **iPad** , на которых установлено приложение Skype для бизнеса, и они используются во всей Организации.
***
![Номер 4](../images/sfbcallout4.png)<br/>Используйте интерактивные данные диаграмм **на линейчатой диаграмме, чтобы** понять, сколько пользователей используют уникальные устройства в Организации. На нем показано общее количество пользователей **Windows**, **Windows Phone**, телефонов с **Android**, **iPhone**и **iPad** , в которых используется устройство. 
***
![Номер 5](../images/sfbcallout5.png)<br/>Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на диаграмме **Пользователи** выберите **Windows**, **Windows Phone**, **Телефон с Android**, **iPhone**или **iPad** , чтобы просмотреть только сведения, относящиеся к каждому из них. При выборе другого значения данные в таблице-сетке не меняются. 
***
![Номер 6](../images/sfbcallout6.png)<br/>На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси.
*    На диаграмме **Пользователи** ось Y — это общее количество пользователей, подключенных к устройству, на котором установлено приложение Skype для бизнеса.
*    На диаграмме действия по **распространению** ось Y — это количество пользователей, которые подключены к определенному типу устройства с установленным приложением Skype для бизнеса. 

На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета.
***
![Номер 7](../images/sfbcallout7.png)<br/>В таблице показано, как выделена вся деятельность по конференциям на каждого пользователя. Здесь отображаются все пользователи, которым назначены Skype для бизнеса и действия в их конференц-связи. При необходимости вы можете добавлять столбцы в таблицу.
*    **Имя пользователя**  это имя, указанное пользователем. 
*    **Дата последнего действия (UTC)** — это последнее время, которое пользователь приходил за одноранговый сеанс или организует конференцию или участие в Конференции.
*    **Windows** будет выбрана, если у пользователя установлено клиентское приложение Skype для бизнеса в операционной системе на базе Windows и оно использует ее. 
*    **Windows Phone** выбрано, если у пользователя установлено клиентское приложение Skype для бизнеса на устройстве с Windows Phone и оно использует его.
*    **Телефон с Android** выбирается, если у пользователя установлено клиентское приложение Skype для бизнеса на телефоне с Android и оно использует его.
*    **iPhone** выбрано, если на устройстве iPhone установлено клиентское приложение Skype для бизнеса и оно использует его. 
*    **iPad** выбирается, если у пользователя есть приложение Skype для бизнеса, установленное на iPad и использующее его. 

Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь со статьей **Скрыть сведения о уровне пользователей?** раздел в [отчетах об активности в центре администрирования](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263). 
***
![Номер 8](../images/sfbcallout8.png)<br/>Щелкните или коснитесь значка **столбцы** в любом из столбцов, чтобы добавить столбцы в отчет или удалить их из него.           <br/> ![Skype for Business Online Reporting Manage Button.](../images/4c8f5387-cebb-4d6c-b7d3-05c954a2c234.png)
***
![Цифра 9](../images/sfbcallout9.png)<br/>Вы также можете экспортировать данные отчета в CSV-файл Excel, щелкнув или нажав **Экспорт**.           <br/> ![Кнопка экспорта отчетов в Skype для бизнеса.](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)<br/>При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
   
## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Отчет о действиях в организаторе конференц-связи в Skype для бизнеса](conference-organizer-activity-report.md) Вы можете узнать, сколько пользователей разупорядочивает конференции, использующие мгновенные сообщения, аудио/видео, общий доступ к приложениям, Интернет, входящие и исходящие звонки, а также с помощью телефонного подключения и выхода из Microsoft.
    
- [Отчет об активности участников конференции в Skype для бизнеса](conference-participant-activity-report.md) Вы можете узнать, в каких случаях используются конференции для обмена мгновенными сообщениями, аудио-и видеосвязи, общего доступа к приложениям, веб-сайтов и звонков по телефону.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Отчет о заблокированных пользователях Skype для бизнеса](users-blocked-report.md) Вы можете просматривать пользователей в Организации, которым запрещено совершать звонки по сети PSTN.
    
- [Отчет об использовании PSTN в Skype для бизнеса](pstn-usage-report.md) Вы можете просматривать количество минут, затраченных на входящие и исходящие звонки, и затраты на эти звонки.

- [Отчет о пулах минут в Skype для бизнеса PSTN](pstn-minute-pools-report.md) вы можете просматривать количество минут, потраченных на текущий месяц Организации.

- [Отчет о сеансах Skype для бизнеса](session-details-report.md) Вы можете просматривать сведения о звонках отдельных пользователей.
