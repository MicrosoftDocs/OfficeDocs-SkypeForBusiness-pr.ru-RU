# <a name="device-usage-report"></a>Отчет об использовании устройств

На **панели мониторинга** отчетов вы можете получить общие сведения о действиях в продуктах Microsoft 365 и Office 365 в вашей организации. С ее помощью можно детализированный отчет по отдельным продуктам, чтобы получить более подробную информацию о действиях с каждым продуктом. Например, в отчете об использовании устройств **в Skype** для бизнеса можно увидеть устройства, в том числе операционные системы на базе Windows и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для мгновенных сообщений и собраний. 

Дополнительные сведения можно [найти в обзоре отчетов.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)
  
В этом отчете, а также других отчетах по Skype для бизнеса приводится информация о типах клиентов и устройств, используемых в организации. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации.
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса, войдите в Центр администрирования Microsoft 365 в качестве администратора. 
  
## <a name="how-to-get-to-the-skype-for-business-device-usage-report"></a>Как получить отчет об использовании устройств в Skype для бизнеса

1. Перейдите в Центр администрирования > **использование**  >  **отчетов.**
    
2. На странице **"Использование"** в списке  слева выберите "Использование устройств в **Skype** для бизнеса". Можно также щелкнуть **мини-приложение** действий Skype для бизнеса и в списке действий skype для бизнеса щелкнуть ссылку **"Использование** устройств в **Skype** для бизнеса".
    
     ![Мини-приложение отчета об использовании устройств в Skype для бизнеса](../images/13e7a51c-f680-4164-9a18-67de10fceb7f.png)
  
    В зависимости от подписки на Microsoft 365 или Office 365, которые вы видите, могут быть не все продукты и отчеты об активности, показанные здесь.

Чтобы получить представление об устройствах, которые используют приложение Skype для бизнеса, можно просмотреть диаграммы пользователей **и** **рассылки.**

Однако если щелкнуть определенный день в отчете, в таблице (см. число 7) отскажутся данные за 30 дней до даты (см. число 2), когда был создан этот отчет.

> [!NOTE]
> Если щелкнуть в таблице сведения об определенном дне, в таблице будут демонстрироваться данные только за 30 дней до даты, когда был создан этот отчет.

Здесь вы сможете общее количество пользователей и типов устройств **с Windows,** **Windows Phone,** **Android,** **iPhone** и **iPad,** на которые установлено приложение Skype для бизнеса и которые используются в вашей организации.

Здесь будет покажите общее количество пользователей **Windows,** **Windows Phone,** **Android,** **iPhone** и **iPad** с используемым устройством. 
***
![Номер 5](../images/sfbcallout5.png)<br/>Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на  диаграмме "Пользователи" щелкните или коснитесь ос **Windows,** **Windows Phone,** **телефона Android,** **iPhone** или **iPad,** чтобы увидеть только сведения, относящиеся к этой модели. При выборе другого значения данные в таблице-сетке не меняются. 
***
![Номер 6](../images/sfbcallout6.png)<br/>На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси.
*    На **диаграмме "Пользователи"** ось Y представляет общее количество пользователей, подключенных к устройству, на которое установлено приложение Skype для бизнеса.
*    На **диаграмме рассылки** ось Y — это количество пользователей, подключенных к устройству определенного типа с установленным приложением Skype для бизнеса. 

На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета.
***
![Номер 7](../images/sfbcallout7.png)<br/>В таблице показана разбивка всех действий по организацииконференций с разбивкой по пользователям. Здесь показаны все пользователи, у которых назначена программа Skype для бизнеса, а также их действия по организации веб-связи. При необходимости вы можете добавлять столбцы в таблицу.
*    **Имя пользователя**  это имя, указанное пользователем. 
*    **Дата последнего действия (UTC)** — это время последнего участия пользователя в одноранговом сеансе, организации конференции или участии в ней.
*    **Windows** выбирается, если на компьютере с Ос Windows установлено клиентского приложения Skype для бизнеса и оно используется. 
*    **Windows Phone** выбирается, если на устройстве с Windows Phone установлено клиентский приложение Skype для бизнеса и оно используется.
*    **Телефон с Android** выбирается, если на нем установлено клиентский приложение Skype для бизнеса.
*    **iPhone** выбирается, если на iPhone установлено клиентского приложения Skype для бизнеса и используется его. 
*    **IPad** выбирается, если на iPad установлено клиентского приложения Skype для бизнеса и используется его. 

Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с **разделом "Скрытие** сведений пользователях" в разделе "Отчеты об [активности" в Центре администрирования.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)

Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
   
## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Отчет об организации конференций в Skype для бизнеса](conference-organizer-activity-report.md) Вы можете узнать, в какой степени ваши пользователи организовывают конференции с использованием услуг обмена мгновенными и аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также внесения и иных ветвей телефонных номеров через сторонних разработчиков, а также в/из учетной записи Майкрософт.
    
- [Отчет об участниках конференции в Skype для бизнеса](conference-participant-activity-report.md) Вы можете узнать, сколько конференций с использованием услуг обмена мгновенными и аудио- и видеосвязями, общего доступа к приложениям, веб-приложений, а также в/из телефонного доступа.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Отчет о блокировке пользователей Skype для бизнеса](users-blocked-report.md) Вы увидите пользователей в организации, которые не могут звонить по ПС.
    
- [Отчет об использовании ПС в Skype для бизнеса](pstn-usage-report.md) Вы увидите, затрачено ли время входящие и исходящие звонки, а также стоимость этих звонков.

- [В отчете](pstn-minute-pools-report.md) по пулам минут ЗВОНКОВ по ЗВОНКОВ по Skype для бизнеса можно увидеть количество минут, посещаемых в организации за текущий месяц.

- [Отчет о сеансе в Skype для бизнеса](session-details-report.md) Вы можете получить подробные сведения о звонках для отдельных пользователей.
