# <a name="device-usage-report"></a>Отчет об использовании устройств

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

На **панели мониторинга** Отчеты представлен обзор действий Microsoft 365 или Office 365 продуктов в организации. С ее помощью можно детализированно придать отдельным продуктам отчеты, чтобы получить более подробную информацию о действиях в каждом продукте. Например, с помощью  отчета Skype для бизнеса использования устройств можно увидеть устройства, Windows операционные системы и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для мгновенных сообщений и собраний. 

Дополнительные сведения можно [найти в обзоре отчетов.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)
  
В этом отчете, а также Skype для бизнеса других отчетах вы можете подробно и подробно о типах клиентов и устройств, используемых в организации. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации.
  
> [!NOTE]
> Вы можете увидеть все отчеты Skype для бизнеса при входе в систему в качестве администратора Центр администрирования Microsoft 365. 
  
## <a name="how-to-get-to-the-skype-for-business-device-usage-report"></a>Как получить отчет об использовании Skype для бизнеса устройств

1. Перейдите в центр администрирования > **Отчеты**  >  **об использовании**.
    
2. На странице **Использование** выберите Skype для бизнеса **использование устройств** в списке **Выберите отчет** слева. Можно также щелкнуть **мини-Skype для бизнеса** действия  и Skype для бизнеса в списке Skype для бизнеса **устройств.**
    
     ![Skype для бизнеса отчета об использовании устройств](../images/13e7a51c-f680-4164-9a18-67de10fceb7f.png)
  
    В зависимости от Microsoft 365 или Office 365 подписки вы можете увидеть не все продукты и отчеты об активности, показанные здесь.

Чтобы получить представление об устройствах, использующих приложение Skype для бизнеса, можно просмотреть **диаграммы** Пользователи **и Распределение.**

> [!NOTE]

Здесь вы сможете показать общее количество пользователей и типов устройств **Windows,** **Windows Phone,**  **телефонов с Android,** **iPhone** и iPad, на которые установлено приложение Skype для бизнеса и которые используются в организации.
***
![Номер 4](../images/sfbcallout4.png)<br/>На интерактивной диаграмме  Распределение можно увидеть количество пользователей, использующих уникальные устройства в организации. Здесь будет покажите общее количество Windows **,** **Windows Phone**, **телефон с Android**,  **iPhone** и iPad пользователей с используемым устройством. 
***
![Номер 5](../images/sfbcallout5.png)<br/>Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на  диаграмме Пользователи щелкните или нажмите Windows **,** **Windows Phone**, телефон с  **Android**, **iPhone** или iPad, чтобы увидеть только сведения, относящиеся к этой диаграмме. При выборе другого значения данные в таблице-сетке не меняются. 
***
![Номер 6](../images/sfbcallout6.png)<br/>На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси.
*    На **диаграмме Пользователи** ось Y представляет общее количество пользователей, подключенных к устройству, на Skype для бизнеса установлено приложение.
*    На **диаграмме Распределение** ось Y представляет количество пользователей, подключенных к устройству определенного типа с установленным приложением Skype для бизнеса рассылки. 

На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета.
***
![Число 7](../images/sfbcallout7.png)<br/>В таблице показана разбивка всех действий по работе с conferencing на пользователя. Здесь показаны все пользователи, Skype для бизнеса назначены им, а также их действия по работе с conferencing. При необходимости вы можете добавлять столбцы в таблицу.
*    **Имя пользователя**  это имя, указанное пользователем. 
*    **Дата последнего действия (UTC)** — это время, когда пользователь в последний раз участвовал в одноранговом сеансе, организовывал конференцию или участвовал в ней.
*    **Windows,** если клиентская Skype для бизнеса установлена в операционной системе Windows и использует его. 
*    **Windows Phone,** если на устройстве Skype для бизнеса установлено клиентского приложения Windows Phone пользователя.
*    **Телефон с Android** выбирается, если на нем Skype для бизнеса клиентского приложения.
*    **iPhone,** если клиентская Skype для бизнеса установлена на iPhone и использует его. 
*    **iPad,** если клиентская Skype для бизнеса установлена на iPad пользователя. 

Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с **разделом Как скрыть** сведения об уровне пользователя? в разделе Отчеты об [активности в Центре администрирования.](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)

***
![Число 9](../images/sfbcallout9.png)<br/>Вы также можете экспортировать данные отчета в CSV-файл Excel, щелкнув или нажав **Экспорт**.           <br/> ![Skype для бизнеса Кнопка "Экспорт отчетов".](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)<br/>При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
   
## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Skype для бизнеса отчет об организации конференций](conference-organizer-activity-report.md) Вы можете узнать, в какой степени пользователи организовывают конференции с использованием услуг обмена мгновенными мгновенными и видеосвязи, общего доступа к приложениям, веб-приложений, внесения и выходных телефонных номеров сторонних разработчиков, а также для внесения и иния телефонного доступа (Майкрософт).
    
- [Skype для бизнеса отчет об активности участников конференции](conference-participant-activity-report.md) Вы можете узнать, сколько конференций с использованием услуг обмена мгновенными мгновенными и аудио- и видеосвязи, общего доступа к приложениям, веб-приложений, а также конференц-залов с телефонным и ветвным телефонным ветвями.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Skype для бизнеса заблокированных пользователей](users-blocked-report.md) Вы увидите пользователей в организации, которые не могут звонить по ЗВОНКОВ по ННР.
    
- [Skype для бизнеса отчете об использовании ННР](pstn-usage-report.md) Вы увидите количество минут, затраченных на входящие и исходящие звонки, и стоимость этих звонков.

- [Skype для бизнеса по минутам](pstn-minute-pools-report.md) ПСПС вы можете увидеть количество минут, посещаемых в организации за текущий месяц.

- [Skype для бизнеса отчет о сеансе](session-details-report.md) Вы можете получить подробные сведения о звонках отдельных пользователей.
