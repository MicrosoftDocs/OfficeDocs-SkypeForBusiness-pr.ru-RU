# <a name="set-up-skype-for-business-call-analytics"></a>Настройка Скайп для вызова бизнес-аналитики

Как Скайп для бизнеса в Интернет администратора Analytics вызова можно использовать для устранения неполадок Скайп для бизнеса и группами Майкрософт вызывать проблемы качества и подключения. Может быть полезно для настройки следующих возможностей в аналитике звонков:
  
- Установка разрешений, которые позволяют другим персонала, таких как служба технической поддержки агенты, используйте вызова Analytics, но запретить доступ к остальной части Скайп по центру администрирования бизнеса. 
    
- Добавьте построение, сайта и сведения о клиента вызова Analytics путем загрузки .tsv или CSV-файла данных.
    
> [!NOTE]
> На данный момент доступна лишь предварительная версия средства аналитики звонков. Описание и изображения в этой статье могут не соответствовать фактическому интерфейсу. 
  
## <a name="set-call-analytics-permissions"></a>Установка разрешений вызова аналитики
<a name="BKMK_SetCAPerms"></a>

Как администратор вы получаете полный доступ ко всем возможностям аналитических вызова. Кроме того можно использовать модель служба технической поддержки в аналитике вызова, которая включает в себя уровня 1 и 2 уровня разрешений группы. Пользователи с разрешениями уровня 1 доступны только ограниченный представление вызовов аналитики. Пользователи с разрешениями уровня 2 можно получить доступ к полной функциональности вызова аналитики. Оба уровни разрешений запретить доступ к остальной части Скайп по центру администрирования бизнес. Можно предоставить доступ к по уровням, добавив группа, содержащая первого уровня или уровня 2 раздел страницы разрешения пользователем. Дополнительные сведения см [Настройка другого уровня разрешений в аналитике вызова](set-up-call-analytics.md#BKMK_SetUpTier).
  
Служба технической поддержки уровня 1 агенты обрабатывать основные проблемы качества звонка. Уровень 1 агенты не изучения проблем с собраний; они собирать сведения и затем переведите ее в агент второго уровня. Агенты уровень 2 просматривать сведения в журналах подробные звонок, который скрыт из первого уровня агентов.  **Настройка другого уровня разрешений в аналитике вызова** 
 <a name="BKMK_SetUpTier"> </a>
  
1. Создание группы безопасности Office 365 для уровня 1 и 2 уровня и людей, с которыми вы хотите добавить в каждую группу. Также можно повторно использовать существующие группы безопасности. Дополнительные сведения см. в статье [Создание, изменение или удаление группы безопасности в Центре администрирования Office 365](https://support.office.com/article/55c96b32-e086-4c9e-948b-a018b44510cb).
    
2. В центре администрирования Office 365 перейдите на **центры администрирования** > **Скайп для бизнеса**.
    
    > [!NOTE]
    > Если land в старой Скайп по центру администрирования бизнеса, перейдите к новой версии, щелкнув **поступают попробуйте нашей новой центра администрирования**. 
  
3. В новой Скайп по центру администрирования бизнеса нажмите кнопку **разрешения**.
    
4. Добавление групп безопасности Office 365 для поля **первого уровня** и **уровня 2** . Можно добавить несколько групп для каждой роли.
    
     ![Снимок экрана показаны разрешения для вызова Analytics страницы с параметрами для уровня 1 и 2 уровня разрешений.](../images/ed5b6b05-b407-4363-8cf0-a6e79027f64b.png)
  
 Пользователи с любой из этих уровней разрешений получить вызова аналитики с помощью выделенного URL-адрес *https://adminportal.services.skypeforbusiness.com*.
  
## <a name="upload-a-tsv-or-csv-file-to-add-building-site-and-tenant-information"></a>Отправка .tsv или CSV-файла для добавления построение сайта и клиентов сведения
<a name="BKMK_UploadFiles"> </a>

Можно добавить построение, сайта и сведения о клиента для вызова Analytics путем отправки файла CSV или .tsv. Эти сведения вызова Analytics можно сопоставить IP-адресов физических местоположениях. Вы или служба технической поддержки агенты полезные эти сведения помогут отслеживать тенденции задач на звонок. Например, почему похожи количество пользователей в одном здании необходимости вызвать проблемы качества? 
  
![Снимке экрана показана страница сайты со значениями для числа сайтов и подсетей и кнопка Выбор файла для импорта данных веб-сайтов, загрузив .tsv или CSV-файл.](../images/b2f3a5cb-32b5-4f60-a9af-0691aa6ff1e8.png)
  
Если вы Скайп для администратора предприятия, можно использовать существующий файл данных из Скайп для панели мониторинга качества Online вызовов Business. Во-первых Загрузите файл из вызывать мониторинга качества и затем вы отправляете для вызова аналитики. Чтобы загрузить файл данных, перейдите к **Скайп по центру администрирования бизнес** > **средств** > **Скайп для панели мониторинга качества Online вызовов Business** > **Отправить сейчас**. В списке **Мои передача** рядом с пунктом нужный файл нажмите кнопку **загрузить** .
  
Если вы создаете .tsv или CSV-файл с нуля, видеть [данные клиента файла формат и структура файла данных построения](turning-on-and-using-call-quality-dashboard.md#BKMKTenantDataFile).
