## <a name="common-area-phones"></a>Региональные телефоны
Телефон общего пользования или БОЛТЫ, обычно размещаются в общей области и не связанные с отдельным пользователем. Например, телефонов области приема, телефона помещения двери телефон или собрания, CAPs настраиваются как устройства, а не пользователей и автоматического входа в сеть. В приведенные ниже действия мы поможет Настройка учетной записи для телефонной системой Microsoft с вызова планы и разверните узел.

## <a name="prerequisites-for-common-area-phones"></a>Необходимые условия для телефонов общего пользования

Убедитесь, что у вас есть следующие:

    - Купить телефон общего пользования SKU 
    - Обновление микропрограммы (видеть поддерживается встроенного по в разделеhttps://docs.microsoft.com/en-us/SkypeForBusiness/what-is-phone-system-in-office-365/getting-phones-for-skype-for-business-online/getting-phones-for-skype-for-business-online)
    - Утвержденная (Просмотр списка на телефоныhttps://docs.microsoft.com/en-us/SkypeForBusiness/what-is-phone-system-in-office-365/getting-phones-for-skype-for-business-online/deploying-skype-for-business-online-phones) 


## <a name="check-the-firmware-for-your-phone"></a>Проверьте микропрограммы для своего телефона
- **Телефоны Polycom VVX**  выберите **Settings** > **Status** > **Platform** > **Application** > **Main** (Параметры > Состояние > Платформа > Приложение > Основные).
- **Телефоны Yealink**  выберите пункт **Status** (Состояние) на главном экране телефона.
- **Телефоны AudioCodes**  выберите **Menu** > **Device Status** > **Firmware version** (Меню > Состояние устройства > Версия встроенного ПО). 
- **Телефоны Lync Phone Edition (LPE)**  выберите **Menu** > **System Information** (Меню > Сведения о системе) на начальном экране.

Служба Skype для бизнеса управляет обновлением встроенного ПО. Обновление для каждого сертифицированного телефона Skype для бизнеса загружается на сервер обновлений Skype для бизнеса, а на всех устройствах обновление включено по умолчанию. 

В зависимости от периода неактивности телефона и интервалов опроса на телефоны автоматически загружаются и устанавливаются последние версии сертифицированных сборок. Чтобы отключить обновление устройства, выполните командлет [Set-CsIPPhonePolicy](https://technet.microsoft.com/en-us/library/mt629497.aspx) и присвойте параметру _EnableDeviceUpdate_ значение `false`.

## <a name="create-cap"></a>Создание Политики
Создайте ограничение, настроив параметры перед настройкой физического телефона.

#### <a name="purchase-the-common-area-phone-sku"></a>Приобрести телефон общего пользования SKU. 
    In the Office 365 admin center, go to **Billing > Purchase Services**, and add **Common Area Phone**.

#### <a name="set-up-the-common-area-phone-----this-section-could-use-a-screen-shot--"></a>Настройка телефона общего пользования<!-- this section could use a screen shot-->

**Создание пользователя** 
1. Назначение телефона общего пользования SKU
2. Назначьте вызов планирование (при использовании с телефонной системой Microsoft с вызова планы). 
3. Назначение доступно телефонный номер в Скайп по центру администрирования бизнес и запросить новый номер телефона.

**Создание нового пользователя**

1. На левой панели подготовки у вас есть параметр, чтобы ввести имя и фамилию (например, для приема Main).
2. Введите отображаемое имя (обязательно), например, «прием Main».
3. Введите имя пользователя (обязательно), например «MainReception» @» домен» (имя компании или enterprise)
4. Введите расположение (Страна).

**Назначение телефона общего пользования SKU** В центре администрирования Office 365 перейдите в раздел **выставления счетов > приобрести службы** и добавьте **Телефон общего пользования**

**Назначьте план вызова в БОЛТЫ SKU**

1. Выберите вызов планируете включить на телефоне. 
2. Добавьте ограничения для поддержки телефонной системой и Скайп для бизнеса Online (план 2) в SKU БОЛТЫ. <!-- odd order for step -->

**Назначить телефонный номер**
1. Поиск доступных телефонных номеров в списке **голосовой связи > телефонных номеров**.
2. Выберите номер из списка доступных номеров телефонов.
3. Подтвердите выбор, выбрав **голосовой связи** и **Номера телефонов**.

    >[Примечание] Пользователям голосовой связи отображаются только при наличии телефонной системой лицензия применяется, несмотря на то, что даже после применения, может потребоваться время обновления. Иногда повторного открытия Скайп для администрирования бизнес центр помогает.
    
## <a name="configure-phone"></a>Настройка телефона

**Подготовка физических телефоны** 

Выбранные телефон должен иметь режим телефон общего пользования. 

***Телефон Polycom VVX пример***

Общие области телефона режим включите на Polycom VVX, выполнив следующие действия:
1. В браузере используйте веб-интерфейс для перехода в режим ограничения на VVX
2. Перейдите к **параметру** и в Скайп для параметра бизнеса, выберите **Телефон общего пользования**.
3. Нажмите кнопку **Сохранить** для сохранения параметров конфигурации.

Теперь, когда включен режим БОЛТЫ телефона, настройте телефон, с помощью телефона отображаются.

1. В диалоговом окне настройки нажмите кнопку **Дополнительно**.
2. Введите пароль.
3. В параметрах администрирования выберите **Общие параметры телефона зоны**.
4. Включить телефон общего пользования и БОЛТЫ администрирования
5. Выберите пункт **Сохранить настройки**.

Телефон готов подготовить к работе, который будет выполняться при входе на начальном экране.

1. Вход, выбрав **Параметры > функции > Скайп для бизнеса.**
2. Выберите учетные данные пользователя и выберите select **web учетное (узел)** для генерации кода.
3. Последовательно выберите пункты подготовки портала http://aka.ms/skypecapи войдите в качестве **администратора**.
4. Введите отображаемое имя (например, для приема Main) для просмотра вашего БОЛТЫ.

>[Примечание] Если «Поиск для телефонов общего пользования только» — этот флажок установлен, снимите флажок и выполните поиск еще раз.

5. В окне код связывания введите код, отображаемые на телефоне и щелкните **подготовить к работе**.

WHT последнем шаге телефон должен автоматический вход в систему.

Дополнительные сведения о доступных телефонов представлены в [ ](https://docs.microsoft.com/en-us/SkypeForBusiness/what-is-phone-system-in-office-365/getting-phones-for-skype-for-business-online/deploying-skype-for-business-online-phones).
