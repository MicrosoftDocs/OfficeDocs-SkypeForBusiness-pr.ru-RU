# <a name="skype-for-business-add-on-licensing"></a>Лицензирование надстройки Skype для бизнеса

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

## <a name="what-are-add-on-licenses"></a>Что такое лицензии на надстройки?

Лицензии на надстройки — это лицензии на определенные Skype для бизнеса функций. Некоторым компаниям важна возможность приобретать только отдельные функции по приемлемой цене. Чтобы добавить функцию, купите одну дополнительную лицензию для каждого пользователя, который будет ее использовать.

Skype для бизнеса лицензирования обеспечивает максимальную гибкость. Например, после знакомства с базовыми функциями Skype для бизнеса вы можете приобрести лицензии на надстройки, которые позволяют использовать дополнительные возможности, такие как аудиоконференция или телефонная система в Office 365.

## <a name="how-does-add-on-licensing-affect-cost"></a>Как дополнительное лицензирование влияет на стоимость?

Для большинства предприятий покупка комплекта программного обеспечения в корпоративном плане приведет к снижению общей стоимости. Как правило, когда вы покупаете несколько дополнительных лицензий по отдельности, а не как часть плана, вы можете получить более высокую совокупную стоимость.

Для более подробной информации о стоимости, перейдите в раздел [Просмотр цен и приобретение лицензий на надстройки](skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_how) или выберите одно из следующих:

- [Тарифы на аудиоконференции](https://products.office.com/skype-for-business/audio-conferencing)

- [Тарифы для телефонной системы](https://products.office.com/skype-for-business/phone-system)

- [Цены на Тарифные планы](https://products.office.com/skype-for-business/calling-plans)

## <a name="how-to-see-prices-and-buy-add-on-licenses"></a>Как узнать цены и приобрести лицензии на надстройки
<a name="bkmk_how"> </a>

 **После покупки Microsoft 365 вы** увидите цены и надстройки в Центре администрирования Microsoft 365.

В зависимости от требуемых дополнительных лицензий рекомендуется [сравнить их выгодность](https://go.microsoft.com/fwlink/?linkid=844053) и перейти на план, в который уже входят эти функции.

Смотрите о вашем плане в разделе «Варианты лицензий на вашем тарифном плане» для получения более подробной информации о том, какие лицензии покупать, как узнать стоимость и приобрести лицензии на надстройки.

## <a name="how-do-i-use-my-existing-calling-plans-with-skype-for-business"></a>Как использовать существующего оператора связи с Skype для бизнеса?
<a name="bkmk_existing"></a>

Чтобы использовать существующий план звонков, приобретите надстройку **телефонной системы** и используйте ее с бесплатным компонентом **[Cloud connector](../../SfbServer/skype-for-business-hybrid-solutions/plan-your-phone-system-cloud-pbx-solution/plan-skype-for-business-cloud-connector-edition.md)**.

Если используются автономные средства связи через ТСОП для гибридных пользователей, необходимо назначить лицензию  *только*  лицензию **телефонной системы**. Назначать план звонков **НЕ НУЖНО**.
