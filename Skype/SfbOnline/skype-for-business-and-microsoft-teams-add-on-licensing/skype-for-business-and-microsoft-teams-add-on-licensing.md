# <a name="skype-for-business-and-microsoft-teams-add-on-licensing"></a>Лицензирование надстроек Skype для бизнеса и Microsoft Teams

## <a name="what-are-add-on-licenses"></a>Что представляют собой лицензии надстроек?

Лицензии на надстройки позволяют использовать определенные расширенные функции Skype для бизнеса и Microsoft Teams. В целях экономии организации могут приобретать отдельные функции по весьма привлекательным ценам. Чтобы добавить функциональную возможность, приобретите одну лицензию надстроек для каждого пользователя, который будет ею пользоваться.

Механизм лицензирования в Skype для бизнеса и Microsoft Teams призван обеспечить максимальную гибкость использования. Например, после знакомства с базовыми функциями Skype для бизнеса вы можете приобрести лицензии на надстройки, которые позволяют использовать дополнительные возможности, такие как аудиоконференция или телефонная система в Office 365.

## <a name="how-does-add-on-licensing-affect-cost"></a>Как лицензии надстроек влияют на общую стоимость решения?

В большинстве случаев приобретение пакета программного обеспечения в рамках корпоративного плана ведет к снижению общей стоимости. Как правило, приобретение нескольких лицензий надстроек по отдельности вместо приобретения в рамках плана ведет к увеличению итоговой стоимости.

Для более подробной информации о стоимости, перейдите в раздел [Просмотр цен и приобретение лицензий на надстройки](skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_how) или выберите одно из следующих:

- [Тарифы на аудиоконференции](https://products.office.com/en-us/skype-for-business/audio-conferencing)

- [Тарифы для телефонной системы](https://products.office.com/en-us/skype-for-business/phone-system)

- [Цены на Тарифные планы](https://products.office.com/en-us/skype-for-business/calling-plans)

## <a name="how-to-see-prices-and-buy-add-on-licenses"></a>Просмотр цен и приобретение лицензий на надстройки
<a name="bkmk_how"> </a>

 **После покупки плана Office 365 вы сможете просмотреть цены и приобрести дополнительные лицензии в Центр администрирования Office 365**.

В зависимости от требуемых дополнительных лицензий рекомендуется [сравнить их выгодность](https://go.microsoft.com/fwlink/?linkid=844053) и перейти на план, в который уже входят эти функции.

Смотрите о вашем плане в разделе «Варианты лицензий на вашем тарифном плане» для получения более подробной информации о том, какие лицензии покупать, как узнать стоимость и приобрести лицензии на надстройки.

## <a name="how-do-i-use-my-existing-calling-plans-with-skype-for-business"></a>Как использовать существующего оператора связи с Skype для бизнеса?
<a name="bkmk_existing"></a>

Чтобы использовать существующий план звонков, приобретите надстройку **телефонной системы** и используйте ее с бесплатным компонентом **[Cloud connector](https://technet.microsoft.com/en-us/library/mt605227.aspx)**.

Если используются автономные средства связи через ТСОП для гибридных пользователей, необходимо назначить лицензию  *только*  лицензию **телефонной системы**. Назначать план звонков **НЕ НУЖНО**.

Подробную документацию по планированию развертывания, в том числе телефонной системы с локальным подключением к ТСОП, смотрите в разделе [Планирование размещения телефонной системы в Office 365 (облачная УАТС)](https://technet.microsoft.com/en-us/library/mt612869.aspx).
