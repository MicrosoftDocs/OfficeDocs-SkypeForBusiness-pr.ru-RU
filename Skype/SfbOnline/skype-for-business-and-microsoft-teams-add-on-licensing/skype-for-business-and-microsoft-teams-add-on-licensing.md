# <a name="skype-for-business-add-on-licensing"></a>Лицензирование надстройки Skype для бизнеса

## <a name="what-are-add-on-licenses"></a>Что представляют собой лицензии на надстройки?

Лицензии на надстройки — это лицензии на конкретные функции Skype для бизнеса. Некоторые компании хотят гибко приобрести только определенные функции по выгодным тарифам. Чтобы добавить компонент, закажите одну лицензию для каждого пользователя, который будет ее использовать.

Лицензирование Skype для бизнеса обеспечивает максимальную гибкость. Например, после использования основных функций Skype для бизнеса вы можете приобрести лицензии на надстройки, когда вы будете готовы использовать дополнительные функции, такие как голосовые конференции или телефонная система в Office 365.

## <a name="how-does-add-on-licensing-affect-cost"></a>Как дополнительные лицензии влияют на общую стоимость решения?

For most businesses, buying a bundle of software in an Enterprise plan will result in lower overall cost. Typically, when you buy several add-on licenses individually instead of as part of a plan, you may end up with higher combined cost.

Для более подробной информации о стоимости, перейдите в раздел [Просмотр цен и приобретение лицензий на надстройки](skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_how) или выберите одно из следующих:

- [Тарифы на аудиоконференции](https://products.office.com/en-us/skype-for-business/audio-conferencing)

- [Тарифы для телефонной системы](https://products.office.com/en-us/skype-for-business/phone-system)

- [Цены на Тарифные планы](https://products.office.com/en-us/skype-for-business/calling-plans)

## <a name="how-to-see-prices-and-buy-add-on-licenses"></a>Просмотр цен и приобретение лицензий на надстройки
<a name="bkmk_how"> </a>

 **После покупки плана Office 365 можно просматривать цены и покупать надстройки в центре администрирования Microsoft 365**.

В зависимости от требуемых дополнительных лицензий рекомендуется [сравнить их выгодность](https://go.microsoft.com/fwlink/?linkid=844053) и перейти на план, в который уже входят эти функции.

Смотрите о вашем плане в разделе «Варианты лицензий на вашем тарифном плане» для получения более подробной информации о том, какие лицензии покупать, как узнать стоимость и приобрести лицензии на надстройки.

## <a name="how-do-i-use-my-existing-calling-plans-with-skype-for-business"></a>Как использовать существующего оператора связи с Skype для бизнеса?
<a name="bkmk_existing"></a>

Чтобы использовать существующий план звонков, приобретите надстройку **телефонной системы** и используйте ее с бесплатным компонентом **[Cloud connector](https://technet.microsoft.com/en-us/library/mt605227.aspx)**.

Если используются автономные средства связи через ТСОП для гибридных пользователей, необходимо назначить лицензию  *только*  лицензию **телефонной системы**. Назначать план звонков **НЕ НУЖНО**.

Подробную документацию по планированию развертывания, включая телефонную систему с использованием локальной сети PSTN, можно найти [в разделе Планирование телефонной системы в Office 365 (облачная УАТС)](https://technet.microsoft.com/en-us/library/mt612869.aspx).
