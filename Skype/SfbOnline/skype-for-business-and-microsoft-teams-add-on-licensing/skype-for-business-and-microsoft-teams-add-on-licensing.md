# <a name="skype-for-business-and-microsoft-teams-add-on-licensing"></a>Лицензирование надстроек Skype для бизнеса и Microsoft Teams

## <a name="what-are-add-on-licenses"></a>Что представляют собой лицензии на надстройки?

Лицензии на надстройки позволяют использовать определенные расширенные функции Skype для бизнеса и Microsoft Teams. В целях экономии организации могут приобретать отдельные функции по весьма привлекательным ценам. Чтобы добавить компонент, приобрести один дополнительная лицензия на каждого пользователя, который будет использоваться.

Скайп для бизнеса и рабочих групп Microsoft лицензирования — это позволит вам максимальный объем гибкости. Например, после знакомства с базовыми функциями Skype для бизнеса вы можете приобрести лицензии на надстройки, которые позволяют использовать дополнительные возможности, такие как аудиоконференция или телефонная система в Office 365.

## <a name="how-does-add-on-licensing-affect-cost"></a>Как дополнительный компонент лицензирования влияет на стоимость?

В большинстве случаев приобретение пакета программного обеспечения в рамках корпоративного плана ведет к снижению общей стоимости. Как правило если приобрести несколько лицензий дополнительный компонент, а не как часть плана, может оказаться с выше объединенный расходы.

Дополнительные сведения расходов можно [увидеть цены и приобрести лицензии дополнительный компонент](skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_how) или один из следующих:
  
- [Тарифы на аудиоконференции](https://products.office.com/en-us/skype-for-business/audio-conferencing) 
    
- [Тарифы для телефонной системы](https://products.office.com/en-us/skype-for-business/phone-system)
    
- [Цена для Тарифные планы](https://products.office.com/en-us/skype-for-business/calling-plans)

## <a name="how-to-see-prices-and-buy-add-on-licenses"></a>Просмотр цен и приобретение лицензий на надстройки
<a name="bkmk_how"> </a>

 **После покупки плана Office 365 вы сможете просмотреть цены и приобрести дополнительные лицензии в Центр администрирования Office 365**.
  
В зависимости от требуемых дополнительных лицензий рекомендуется [сравнить их выгодность](https://go.microsoft.com/fwlink/?linkid=844053) и перейти на план, в который уже входят эти функции.

Для получения дополнительных сведений о котором лицензируемые приобрести, а также для просмотра цены и приобрести лицензии дополнительный компонент видеть ваш план в группе «Параметры лицензии на основе плана». 

## <a name="how-do-i-use-my-existing-calling-plans-with-skype-for-business"></a>Как использовать существующего оператора связи с Skype для бизнеса?
<a name="bkmk_existing"></a>

Чтобы использовать существующий план звонков, приобретите надстройку **телефонной системы** и используйте ее с бесплатным компонентом **[Cloud connector](https://technet.microsoft.com/en-us/library/mt605227.aspx)**.
  
Если используются автономные средства связи через ТСОП для гибридных пользователей, необходимо назначить лицензию  *только*  лицензию **телефонной системы**. Назначать план звонков **НЕ НУЖНО**.
  
Подробную документацию по планированию развертывания, в том числе телефонной системой с локального подключения к ТСОП в разделе [Планирование телефонной системой в решение Office 365 (облако УАТС)](https://technet.microsoft.com/en-us/library/mt612869.aspx).
