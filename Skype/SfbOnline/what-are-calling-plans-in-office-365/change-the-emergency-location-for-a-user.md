# <a name="assign-or-change-the-emergency-location-for-a-user"></a>Назначение и изменение расположения экстренного реагирования для пользователя

Каждый активный телефонный номер должен иметь соответствующий адрес для экстренного реагирования, когда вы назначаете пользователю номер телефона. (Вы связываете этот адрес при получении номера телефона в Office 365 или при переносе номера телефона.) При связывании номера с адресом для экстренного реагирования вы также можете добавить место для экстренного реагирования, чтобы указать более точное расположение в пределах физического места. Местоположение для экстренного реагирования может быть этажом, крылом или номером офиса, где находится пользователь. У вас есть безлимитное число местоположений для заданного адреса для экстренного реагирования, и вы можетеизменить местоположение для реагирования экстренных служб, если пользователь перемещается в другой офис или здание — например, при перемещении c 34 на 35 этаж.
  
Чтобы узнать о том, как получить тарифные планы Office 365 и их стоимость, см. раздел [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="assign-or-change-the-emergency-location"></a>Назначение или изменение расположения для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На левой панели навигации перейдите в раздел Пользователи **голосовой** > **голосовой связи**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. На странице **Пользователи голосовой связи** найдите и выберите пользователя, для которого необходимо изменить адрес для экстренного реагирования.
    
5. На панели действий нажмите **Местоположение для реагирования экстренных служб**, затем **Изменить**.
    
6. На странице **Назначить номер** выберите **Изменить местоположение**. 
    
7. В разделе **изменить адрес для экстренного**реагирования введите название города в поле и нажмите кнопку **Поиск**.

8. В раскрывающемся списке выберите пункт **Поиск по расположению** , введите частичное имя расположения (например, "введите **этаж**") и нажмите кнопку **Поиск**. 
    
8. Выберите в списке нужное место для экстренного реагирования и нажмите кнопку **сохранить**.
    
    Если вы хотите добавить новое место для экстренного реагирования, которое появится в списке, просмотрите раздел [Добавление, изменение и удаление расположения для экстренной помощи в вашей организации](add-change-or-remove-an-emergency-location-for-your-organization.md).
