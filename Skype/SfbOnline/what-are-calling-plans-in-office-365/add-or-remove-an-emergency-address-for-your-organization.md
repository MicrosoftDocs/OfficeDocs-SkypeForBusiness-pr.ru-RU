# <a name="add-change-or-remove-an-emergency-address-for-your-organization"></a>Добавление, изменение и удаление адреса экстренного реагирования для организации

Адрес для экстренного реагирования должен быть связан с номером телефона, но при этом может различаться в зависимости от страны или региона. Например, в США необходимо привязать адрес для экстренного реагирования при назначении телефонного номера пользователю. В Великобритании необходимо привязать адрес для экстренного реагирования к номеру телефона при получении телефонных номеров в Office 365 или передаче телефонного номера текущим поставщиком услуг.
  
No matter which country/region you are in, it's possible to add a location or locations to an emergency address or remove an emergency address. Depending on the number of physical locations in your organization, you can create them for buildings, floors, and offices. See [What are emergency locations, addresses, and call routing?](/microsoftteams/what-are-emergency-locations-addresses-and-call-routing) for some details.
  
Чтобы узнать, как получить вызов план звонков и сколько это стоит, см. раздел [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="add-an-emergency-address"></a>Добавление адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева выберите **Голосовая связь** > **Местоположения для местного реагирования** и нажмите кнопку **Добавить новый адрес**.
    
    > [!Important]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. В области действий в разделе **Новый адрес** введите необходимые данные в поля.
    
5. После ввода всех данных для адреса нажмите кнопку **проверить**.
    
    > [!IMPORTANT]
    > Validating a street or civic address involves making sure that it is legitimate and correctly formatted. It is possible that a partially correct emergency address, such as if you mistyped the name of the city, may still pass validation. Even though it's misspelled and passed validation, the combination of the misspelled name of city along with the other correct parts of the address are enough information to route the call to the appropriate emergency dispatch center. 
    
    > Примечание. в Бельгии, Франции, Германии, Ирландии, Нидерланды и Испании важно понимать, что для успешной активации номера телефона в Office 365 Настройка адресов в месте хранения для экстренного реагирования, которая будет использоваться для получения номера, должна совпадать с код города (номер телефона).
  
    Если адрес не может быть подтвержден, можно отправить запрос на проверку вручную, нажав кнопку **Отправить запрос на проверку** (если вы пытаетесь выполнить проверку адреса в США) или нажать кнопку **Открыть служебный запрос для получения помощи в проверке адреса** (за пределами США).
    
6. После подтверждения адреса щелкните **Сохранить**.
    
## <a name="change-an-emergency-address"></a>Изменение адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева перейдите в раздел **** > **расположения для экстренных случаев**, выберите адрес, который вы хотите изменить, а затем в области действий нажмите кнопку **изменить**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .

4. Внесите изменения и нажмите кнопку **проверить**.

5. Нажмите кнопку **Сохранить**.

## <a name="remove-an-emergency-address"></a>Удаление адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации **** > **** слева выберите адрес, который вы хотите удалить, а затем в области действий нажмите кнопку **Удалить**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .

## <a name="troubleshooting"></a>Поиск и устранение неполадок

**Номер в состоянии "не удалось".**

После получения номера с портала Office 365 его состояние изменилось с **"подготовка"** на **"сбой**".

Эта проблема часто возникает при добавлении номера с портала с использованием адреса для экстренного реагирования, указывающего на место, не совпадающее с кодом города телефона.

Чтобы получить дополнительные сведения о неправильной активации номера (-ов), запустите следующую оболочку PowerShell:
 
> [! СИНТАКСИС] Get-Ксонлинетелефоненумбер | Where-Object {$ _. Активатионстате-кнотконтаинс "активировано"} | штат

Результат, помимо других данных, таких как Region, ID и Активатионстате, должен содержать Цитикоде.

**Например**, для мадридного числа Цитикодеом будет возвращено значение "EMEA-ES-ALL-м_ма".

Если вы используете неправильный адрес для экстренного реагирования, убедитесь, что вы создали новый адрес для экстренного реагирования, соответствующий коду города, и назначьте его номеру телефона.

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева **** > перейдите к**номерам абонентов**, а затем дважды щелкните по номеру, указанному в поле **"сбой"** , а затем в меню сайта справа выберите **Новый адрес**для экстренного реагирования.


Обратите внимание, что после изменения адреса для экстренного реагирования его состояние изменится на **"ожидание назначения"** , и для успешной активации может потребоваться до 24 часов.
