# <a name="add-or-remove-an-emergency-address-for-your-organization"></a>Добавление или удаление адреса для экстренного реагирования для организации

Адрес для экстренного реагирования должен быть связан с номером телефона — может отличаться в зависимости от страны или региона. Например, в США необходимо привязать адрес для экстренного реагирования при назначении телефонного номера пользователю. В Великобритании необходимо привязать адрес для экстренного реагирования к номеру телефона при получении телефонных номеров в Office 365 или передаче телефонного номера текущим поставщиком услуг.
  
Независимо от страны или региона вашего местоположения, можно добавить его в качестве адреса для экстренного реагирования или удалить такой адрес. В зависимости от номеров по физическому местонахождению вашей организации, можно создать их для строений, этажей или кабинетов. См. статью [Что такое местоположения для экстренного реагирования, адреса для экстренного реагирования и маршрутизация экстренных вызовов?](/microsoftteams/what-are-emergency-locations-addresses-and-call-routing) для ознакомления с некоторыми подробностями.
  
Чтобы узнать, как получить вызов план звонков и сколько это стоит, см. раздел [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="to-add-an-emergency-address"></a>Добавление адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации слева выберите **Голосовая связь** > **Местоположения для местного реагирования** и нажмите кнопку **Добавить новый адрес**.
    
    > [!Important]
    > Чтобы на панели навигации слева в Центре администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня Корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. В области действий в разделе **Новый адрес** введите необходимые данные в поля.
    
5. Когда вся информации по адресу будет введена, щелкните **Подтвердить**.
    
    > [!IMPORTANT]
    > Во время подтверждения улицы или городского адреса проверяется, существует ли этот адрес официально и в правильном ли формате он указан. В некоторых случаях проверку может пройти адрес для экстренного реагирования, содержащий небольшие неточности, например, опечатки в названии города. Даже в таком случае сочетание названия города и правильно указанного почтового адреса позволяет диспетчеру направить службы экстренного реагирования по нужному адресу. 
  
    Если адрес не может быть подтвержден, можно отправить запрос на проверку вручную, нажав кнопку **Отправить запрос на проверку** (если вы пытаетесь выполнить проверку адреса в США) или нажать кнопку **Открыть служебный запрос для получения помощи в проверке адреса** (за пределами США).
    
6. После подтверждения адреса щелкните **Сохранить**.
    
## <a name="to-remove-an-emergency-address"></a>Удаление адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации слева выберите **Голосовая связь** > **Местоположение для экстренного реагирования**, выберите адрес для удаления и на панели действий нажмите **Удалить**.
    
    > [!IMPORTANT]
    > Чтобы на панели навигации слева в Центре администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня Корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
