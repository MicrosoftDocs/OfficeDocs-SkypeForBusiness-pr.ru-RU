3. <span data-ttu-id="9f915-111">На левой панели навигации выберите **Голосовая связь**, затем выберите **Пользователи голосовой связи**.</span><span class="sxs-lookup"><span data-stu-id="9f915-111">In the left navigation go to **Voice** > then click **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="9f915-112">Чтобы на левой панели навигации центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.</span><span class="sxs-lookup"><span data-stu-id="9f915-112">IMPORTANT: For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="9f915-113">На странице **Пользователи голосовой связи** найдите и выберите пользователя, для которого необходимо изменить адрес для экстренного реагирования.</span><span class="sxs-lookup"><span data-stu-id="9f915-113">On the **Voice users** page, locate and select the user you want to change the emergency address for.</span></span>

    > [!NOTE]
    > <span data-ttu-id="9f915-115">Можно изменить только уже проверенный адрес для экстренного реагирования.</span><span class="sxs-lookup"><span data-stu-id="9f915-115">You can only change an emergency address that has already been validated.</span></span> <span data-ttu-id="9f915-116">Чтобы изменить адрес для экстренного реагирования, который еще не был проверен, удалите его и создайте другой адрес для экстренного реагирования.</span><span class="sxs-lookup"><span data-stu-id="9f915-116">To change an emergency address that hasn't been validated, delete it and create another emergency address.</span></span>

7. <span data-ttu-id="9f915-118">В разделе **Изменить адрес для экстренного реагирования на** укажите название города и нажмите **Поиск**.</span><span class="sxs-lookup"><span data-stu-id="9f915-118">Under **Change emergency address to** put the name of the city in the Find City box and click **Search**.</span></span>
    
8. <span data-ttu-id="9f915-119">Выберите адрес для экстренного реагирования в раскрывающемся списке с адресами и нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="9f915-119">Select the right emergency address from the Select emergency address list, click Save.</span></span>
