4. <span data-ttu-id="17f0f-113">На странице **Пользователи голосовой связи** найдите и выберите пользователя, для которого необходимо изменить адрес для экстренного реагирования.</span><span class="sxs-lookup"><span data-stu-id="17f0f-113">On the **Voice users** page, locate and select the user you want to change the emergency address for.</span></span>
