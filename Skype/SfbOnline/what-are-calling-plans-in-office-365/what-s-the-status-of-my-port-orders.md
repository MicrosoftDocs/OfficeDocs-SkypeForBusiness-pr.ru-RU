# <a name="whats-the-status-of-my-port-orders"></a>Каково состояние моих заказов на перенос номеров?

Можно просмотреть состояние порта заказа, перейдя на **Скайп по центру администрирования Business** > **голосовой связи** > **заказов на порт**. В столбце **состояние** отображается состояние каждого порта порядке. Если вам требуется помощь, [обратитесь в службу поддержки продуктов бизнес - Admin справки](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).

В следующей таблице приведены состояния порта порядке, а также действия, которые нужно выполнить при необходимости.

|**Состояние**|**Можно просмотреть порядок?**|**Можно ли изменить порядок?**|**Можно ли отменить порядок?**|**Можно ли удалить порядок?**|**Описание**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Обработка** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Администратор создает порядок и принимается корпорацией Майкрософт.  <br/> |
|**Общаться с поставщика** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Порядок получения и утвержденных корпорацией Майкрософт и мы работают с ним проигравшей его утвержденных.  <br/> |
|**Передача утверждено** <br/> |Да  <br/> |Утверждение заказа Commitment(FOC)  <br/> |Да  <br/> |Да  <br/> |Порядок принято проигравшей передающую и Дата FOC имеет значение.  <br/> |
|**Ожидание передачи** <br/> |Да  <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Перенос является менее 24 часов, больше не порядок можно изменить или отменена.  <br/> |
|**Ошибка** <br/> |Нет  <br/> |Да  <br/> |Да   <br/> |Да (в настоящее время невозможно удалить номеру порта Если возникает ошибка. Порядок порта необходимо повторно создать или вам требуется [обращение в службу поддержки продуктов бизнес - Admin справки](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).  <br/> |Оператор проигравшей отклонил порядке.  <br/> |
