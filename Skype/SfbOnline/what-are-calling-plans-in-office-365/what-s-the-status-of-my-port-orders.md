# <a name="whats-the-status-of-my-port-orders"></a>Каково состояние моих заказов на перенос номеров?

Можно просмотреть состояние порта заказа, перейдя на **Скайп по центру администрирования Business** > **голосовой связи** > **заказов на порт**. В столбце **состояние** отображается состояние каждого порта порядке. Если помочь, [контакт поддержки продуктов бизнес - Admin справки](http://technet.microsoft.com/library/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b%28Office.14%29.aspx).

|:-----|:-----|:-----|:-----|:-----|:-----|

|**Общаться с поставщика** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Порядок получения и утвержденных корпорацией Майкрософт и мы работают с ним проигравшей его утвержденных.  <br/> |
