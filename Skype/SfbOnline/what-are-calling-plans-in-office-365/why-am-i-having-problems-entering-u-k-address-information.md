# <a name="why-am-i-having-problems-entering-uk-address-information"></a>Почему у меня возникают проблемы при входе в Великобритания Информация об адресе

Ниже приведены некоторые причины, по которым может возникнуть проблема с подключением вашего адреса или адреса для правильной проверки правильности в Великобритании (Великобритания).
  
## <a name="whats-wrong-with-the-company-name"></a>Что не так с названием компании?

Если у вас возникла проблема с вводом названия компании, помните:
  
- Она может содержать буквы, цифры и следующие специальные символы:,/-+. ' &amp; " _ 
    
    > [!CAUTION]
    > В Великобритании (Великобритания) название компании не может содержать символ +. 
  
- Оно должно состоять не более чем из 80 знаков.
    
- Она не может начинаться со специального символа.
    
## <a name="whats-wrong-with-the-house-number"></a>В чем заключается номер дома?

Если у вас возникли проблемы с входом в номер дома, помните:
  
- Она может содержать буквы, цифры и следующие специальные символы:-/[]
    
- Она должна начинаться с цифры.
    
- Оно должно состоять не менее чем из 20 символов.
    
- Он может иметь прописную или строчную букву.
    
## <a name="whats-wrong-with-the-house-name"></a>Что не так с названием дома?

Если у вас возникли проблемы при вводе названия дома, помните:
  
- Она может содержать буквы, цифры и следующие специальные символы:,/-+. ' &amp; " _
    
- Оно должно состоять не более чем из 50 знаков.
    
- Она не может начинаться со специального символа.
    
## <a name="whats-wrong-with-the-street-name"></a>Что не так с названием улицы?

Если при вводе названия улицы возникла проблема, помните:
  
- Она может содержать буквы, цифры и следующие специальные символы:,/-+. ' &amp; " _ 
    
- Оно должно состоять не более чем из 50 знаков.
    
- Она не может начинаться со специального символа. 
    
## <a name="whats-wrong-with-the-city-or-town"></a>Почему в городе или городе возникли проблемы?

Если у вас возникают проблемы с вводом названия города или населенного пункта, помните:
  
- Она может содержать буквы, цифры и следующие специальные символы:,/-+. ' &amp; " _
    
- Оно должно состоять не более чем из 50 знаков.
    
- Она не может начинаться со специального символа. 
    
## <a name="whats-wrong-with-the-postal-code"></a>В чем заключается ошибка почтового индекса?

Если при вводе почтового индекса возникли проблемы, помните:
  
- Оно может содержать только буквы, цифры или пробелы.
    
- Оно должно содержать не более 8 букв, в том числе цифр и пробелов.
    
- **Помните, что если вы используете Великобритания (Великобритания)**, вы **должны** поставить почтовый индекс в одном из следующих форматов, где буква a = символ a-Z, знак # = a, число из 0-9, а _ (подчеркивание) — пустое пространство. Вот несколько примеров:
    
|**Формате**|**Пример**|
|:-----|:-----|
|AA # А_ # AA  <br/> |EC3B 2BD  <br/> |
|# А_ # AA  <br/> |E1W 1AP  <br/> |
|# _ # AA  <br/> |КЛАВИША F1 2AG  <br/> |
|# #_ # AA  <br/> |C44 8HT  <br/> |
|AA # _ # AA  <br/> |DR3 6HG  <br/> |
|AA # #_ # AA  <br/> |FN35 1TH  <br/> |

> [!NOTE]
> Если вам нужно больше номеров, [обратитесь в службу поддержки администраторов Skype для бизнеса](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
