# <a name="why-am-i-having-problems-entering-uk-address-information"></a>Почему у меня возникают проблемы с вводом в Сша? адресная информация?

Вот несколько причин, по которым у вас могут возникнуть проблемы с проверкой адресов и адресов в Великобритании.
  
## <a name="whats-wrong-with-the-company-name"></a>Что не так с названием компании?

Если при вводе названия организации возникла проблема, помните:
  
- В нем могут быть буквы, цифры и следующие специальные символы: , / - + . ' &amp; " _ 
    
    > [!CAUTION]
    > В Великобритании у названия компании не может быть знака "+". 
  
- Оно не должно быть больше 80 символов.
    
- Он не может начинаться со специального знака.
    
## <a name="whats-wrong-with-the-house-number"></a>Что не так с номером дома?

Если при вводе номера дома возникла проблема, помните:
  
- Оно может иметь буквы, цифры и следующие специальные символы: - / [ ]
    
- Он должен начинаться с числа.
    
- Оно не должно быть больше 20 символов.
    
- Буква в ее регистре может быть как строчная, так и строчная.
    
## <a name="whats-wrong-with-the-house-name"></a>Что не так с названием дома?

Если при вводе названия дома возникла проблема, помните:
  
- Оно может иметь буквы, цифры и следующие специальные символы: : , / - + . ' &amp; " _
    
- Оно не должно быть меньше 50 символов.
    
- Он не может начинаться со специального знака.
    
## <a name="whats-wrong-with-the-street-name"></a>Что не так с названием улицы?

Если при вводе названия улицы возникла проблема, запомните:
  
- Оно может иметь буквы, цифры и следующие специальные символы: : , / - + . ' &amp; " _ 
    
- Оно не должно быть меньше 50 символов.
    
- Он не может начинаться со специального знака. 
    
## <a name="whats-wrong-with-the-city-or-town"></a>Что не так с городом?

Если при вводе названия города возникла проблема, запомните:
  
- Оно может иметь буквы, цифры и следующие специальные символы: : , / - + . ' &amp; " _
    
- Оно не должно быть меньше 50 символов.
    
- Он не может начинаться со специального знака. 
    
## <a name="whats-wrong-with-the-postal-code"></a>Почему не так с почтовым индексом?

Если при вводе почтового индекса возникла проблема, помните:
  
- Оно может быть только буквами, цифрами или пробелами.
    
- Оно должно иметь не более 8 букв, включая цифры и пробелы.
    
- Помните, что если вы находитесь в **Великобритании,** почтовый индекс необходимо разместить в одном из следующих форматов, где буква A = символ A–Z, знак # = число от 0 до 9, а _ (подчеркивать) — пустое пространство.  Вот несколько примеров:
    
|**Формат**|**Пример**|
|:-----|:-----|
|AA#A_#AA  <br/> |EC3B 2BD  <br/> |
|A#A_#AA  <br/> |E1W 1AP  <br/> |
|A#_#AA  <br/> |F1 2AG  <br/> |
|A##_#AA  <br/> |C44 8HT  <br/> |
|AA#_#AA  <br/> |DR3 6HG  <br/> |
|AA##_#AA  <br/> |FN35 1TH  <br/> |

> [!NOTE]
> Если вам нужно больше номеров, [обратитесь в службу поддержки администраторов Skype для бизнеса](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
