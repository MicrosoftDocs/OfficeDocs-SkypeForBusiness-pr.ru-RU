# <a name="see-a-list-of-phone-numbers-in-your-organization"></a>Просмотр списка телефонных номеров в организации

Есть различные типы телефонных номеров, которые в Office 365 можно назначить пользователям или другим службам (служебные номера), таким как Аудиоконференции.
  
## <a name="to-see-a-list-of-all-phone-numbers-that-you-have-for-your-organization"></a>Просмотр списка всех телефонных номеров, имеющихся в организации

![SFB-Logo-30x30. png](../images/sfb-logo-30x30.png) **с помощью центра администрирования Skype для бизнеса**


1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. В **центре администрирования Skype для бизнеса**на панели навигации слева перейдите к номерам **голосовых** > **телефонов**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. Для просмотра назначенных телефонных номеров см. столбец **Номер**.
    
5. Используйте раскрывающийся список для применения фильтра, чтобы просмотреть:
    
   - **Все** телефонные номера, включая служебные телефонные номера.
    
   - Установленный вами **диапазон телефонных номеров**.
    
   - Телефонные номера, **назначенные пользователю**.
    
   - Телефонные номера, **назначенные конференции** для присоединения к конференции по телефону.
    
   - **Неназначенные** телефонные номера.
    
   - **Расположение** телефонных номеров.
    
## <a name="to-see-all-of-the-phone-numbers-that-are-assigned-to-users"></a>Просмотр всех телефонных номеров, назначенных пользователям

![SFB-Logo-30x30. png](../images/sfb-logo-30x30.png) **с помощью центра администрирования Skype для бизнеса**


При настройке параметров пользователей вам может понадобиться просмотреть список телефонных номеров, которые уже назначены пользователям, и какие телефонные номера можно им назначить.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. В **центре администрирования Skype для бизнеса**на панели навигации слева перейдите к разделу Пользователи **голосовой** > **голосовой связи**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. You can filter your view for phone numbers that are assigned to users or phone numbers that you can assign to a user. You can filter by:
    
   - **Все** телефонные номера, включая служебные телефонные номера.
    
   - **Номер, назначенный** пользователю.
    
   - **Номер, не назначенный** пользователю.
    
## <a name="to-see-the-phone-numbers-that-are-assigned-to-voice-users"></a>Просмотр телефонных номеров, назначенных пользователям голосовой связи

![SFB-Logo-30x30. png](../images/sfb-logo-30x30.png) **с помощью центра администрирования Skype для бизнеса**


When you are setting up users in your organization to make and receive phone calls, you must first get the phone numbers and then assign them to your users. After you've gotten your phone numbers, you might just want to see the list of the users that have phone numbers assigned to users, those that don't and whether a user is hosted Online or on-premises.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. В **центре администрирования Skype для бизнеса**на панели навигации слева перейдите к разделу Пользователи **голосовой** > **голосовой связи**.
    
    > [!IMPORTANT]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. You can use the drop-down list to filter your view for active users, whether the user is Online or on-premises, or the state and type of licensing for a user. You can filter by:
    
   - **Активные пользователи** и является ли пользователь подключенным через Интернет или локальным.
    
   - **Ожидающие лицензий** и является ли пользователь подключенным через Интернет или локальным.
