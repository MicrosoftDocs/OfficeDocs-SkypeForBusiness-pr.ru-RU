# <a name="see-a-list-of-phone-numbers-in-your-organization"></a>Просмотр списка телефонных номеров в организации

Есть различные типы телефонных номеров, которые в Office 365 можно назначить пользователям или другим службам (служебные номера), таким как Аудиоконференции.
  
## <a name="to-see-a-list-of-all-phone-numbers-that-you-have-for-your-organization"></a>Просмотр списка всех телефонных номеров, имеющихся в организации

![sfb-logo-30x30.png](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**


1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **центре администрирования Skype для бизнеса**, в левой панели навигации выберите **Голосовая связь** > **Телефонные номера**.
    
    > [!IMPORTANT]
    > Чтобы на левой панели навигации центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. Для просмотра назначенных телефонных номеров см. столбец **Номер**.
    
5. Используйте раскрывающийся список для применения фильтра, чтобы просмотреть:
    
  - **Все** телефонные номера, включая служебные телефонные номера.
    
  - Установленный вами **диапазон телефонных номеров**.
    
  - Телефонные номера, **назначенные пользователю**.
    
  - Телефонные номера, **назначенные конференции** для присоединения к конференции по телефону.
    
  - **Неназначенные** телефонные номера.
    
  -  **Расположение** телефонных номеров.
    
## <a name="to-see-all-of-the-phone-numbers-that-are-assigned-to-users"></a>Просмотр всех телефонных номеров, назначенных пользователям

![sfb-logo-30x30.png](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**


При настройке параметров пользователей вам может понадобиться просмотреть список телефонных номеров, которые уже назначены пользователям, и какие телефонные номера можно им назначить.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **центре администрирования Skype для бизнеса**, в левой панели навигации выберите **Голосовая связь** > **Пользователи голосовой связи**.
    
    > [!IMPORTANT]
    > Чтобы на левой панели навигации центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. Вы можете применить фильтр, чтобы просмотреть телефонные номера, назначенные пользователям, или телефонные номера, которые можно назначить пользователю. Вы можете фильтровать по следующим параметрам:
    
  - **Все** телефонные номера, включая служебные телефонные номера.
    
  - **Номер, назначенный** пользователю.
    
  - **Номер, не назначенный** пользователю.
    
## <a name="to-see-the-phone-numbers-that-are-assigned-to-voice-users"></a>Просмотр телефонных номеров, назначенных пользователям голосовой связи

![sfb-logo-30x30.png](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**


При настройке параметров пользователей в организации для совершения и приема телефонных звонков необходимо сначала получить номера телефонов, а затем назначить их пользователям. После получения телефонных номеров вам может понадобиться просмотреть список пользователей, имеющих телефонные номера, назначенные пользователям; список пользователей, не имеющих телефонных номеров, а также сведения о том, является ли пользователь подключенным через Интернет или локальным.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Выберите **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **центре администрирования Skype для бизнеса**, в левой панели навигации выберите **Голосовая связь** > **Пользователи голосовой связи**.
    
    > [!IMPORTANT]
    > Чтобы на левой панели навигации центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. Вы можете использовать раскрывающийся список для применения фильтра, чтобы просмотреть активных пользователей, узнать, является ли пользователь подключенным через Интернет или локальным, а также состояние и тип лицензирования для пользователя. Вы можете фильтровать по следующим параметрам:
    
  - **Активные пользователи** и является ли пользователь подключенным через Интернет или локальным.
    
  - **Ожидающие лицензий** и является ли пользователь подключенным через Интернет или локальным.
