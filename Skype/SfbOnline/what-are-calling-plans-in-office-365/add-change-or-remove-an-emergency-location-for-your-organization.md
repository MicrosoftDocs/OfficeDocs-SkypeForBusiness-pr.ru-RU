# <a name="add-change-or-remove-an-emergency-location-for-your-organization"></a>Добавление, изменение и удаление местоположения организации для экстренного реагирования

В зависимости от числа физических компьютеров в вашей организации можно создать местоположении для экстренных служб для зданий, этажами и офисы для создания более конкретные экстренных адрес. Просмотреть [Каковы местоположении для экстренных служб, адресов и маршрутизации вызовов?](what-are-emergency-locations-addresses-and-call-routing.md) для получения дополнительных сведений. 
  
Чтобы узнать, как получить вызов планирование и их затратами, обратитесь к разделу [Скайп для бизнеса и группами Майкрософт дополнительный компонент лицензирования](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="add-an-emergency-location-to-an-emergency-address"></a>Добавление местоположения в адрес для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Go to the **Office 365 admin center** > **Skype for Business**.
    
3. В левой области переходов, перейдите к **голосовой связи** > **аварийного расположения**, а затем выберите экстренных адрес, который вы хотите добавить расположение.
    
    > [!Important]
    > Позволяет параметр **голосовой связи** в левой области навигации в Скайп по центру администрирования бизнеса необходимо купить по крайней мере один **E5 корпоративной лицензии**, дополнительная лицензия на одной **Телефонной системой** или дополнительная лицензия на один **Аудио конференц-связи** .
    
4. В области действий рядом с пунктом **связанные расположений**нажмите кнопку **Изменить**.
    
5. В, **связана расположений**, нажмите кнопку **Добавить**, введите другой связанное расположение (например, введите «Построение 64 этажа 1»), а затем нажмите кнопку **Отправить**.
    
6. Нажмите кнопку **Сохранить**.
    
## <a name="change-an-emergency-location-for-an-emergency-address"></a>Изменение местоположения для экстренного реагирования в адресе

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Go to the **Office 365 admin center** > **Skype for Business**.
    
3. В левой области переходов, перейдите к **голосовой связи** > **аварийного расположения**и выберите экстренных адрес, для которого вы хотите изменить расположение.
    
4. В области действий рядом с пунктом **связанные расположений**нажмите кнопку **Изменить**.
    
5. В разделе **расположения связанного с**щелкните ссылку на расположение в списке, введите новый адрес в **изменить связанные сведения о месте**и нажмите кнопку **Отправить**.
    
6. Нажмите кнопку **Сохранить**.
    
## <a name="remove-an-emergency-location-from-an-emergency-address"></a>Удаление местоположения из адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Go to the **Office 365 admin center** > **Skype for Business**.
    
3. В левой области переходов, перейдите к **голосовой связи** > **аварийного расположения**, а затем выберите экстренных адрес, который требуется удалить расположение.
    
4. В области действий рядом с пунктом **связанные расположений**нажмите кнопку **Изменить**.
    
5. В разделе **расположения связанного с**затем ссылка для размещения, нажмите кнопку **Удалить**.
    
6. Также можно отправить электронное письмо пользователю с помощью параметров аудиоконференции, выбрав на странице свойств аудиоконференции для пользователя параметр **Отправить информацию о конференции по электронной почте**.
