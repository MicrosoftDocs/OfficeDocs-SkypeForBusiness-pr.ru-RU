# <a name="add-change-or-remove-an-emergency-location-for-your-organization"></a>Добавление, изменение и удаление местоположения организации для экстренного реагирования

Depending on the number of physical locations in your organization, you can create emergency locations for buildings, floors, and offices to create a more specific emergency address. See [What are emergency locations, addresses and call routing?](/microsoftteams/what-are-emergency-locations-addresses-and-call-routing) for more information. 
  
Чтобы узнать о том, как получить тарифный план и сколько это стоит, см. раздел [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="add-an-emergency-location-to-an-emergency-address"></a>Добавление местоположения в адрес для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева перейдите в раздел**расположения**для экстренной помощи в службе **голосовой связи** > и выберите адрес для экстренного реагирования, для которого нужно добавить расположение.
    
    > [!Important]
    > Для просмотра параметров **голоса** на левой панели навигации в центре администрирования Skype для бизнеса вы должны сначала приобрести хотя бы одну **лицензию**на корпоративную, одну лицензию на надстройку **** или лицензию для **голосовой конференции** .
    
4. На панели действий рядом со связанными **расположениями**нажмите кнопку **изменить**.
    
5. В секции **Местоположенния связанные с** нажмите на кнопку **Добавить новое**, введите другое связанное расположение (например, введите «Здание 64 Этаж 1»), а затем нажмите на кнопку **Отправить**.
    
6. Нажмите кнопку **Сохранить**.
    
## <a name="change-an-emergency-location-for-an-emergency-address"></a>Изменение местоположения для экстренного реагирования в адресе

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева перейдите в раздел**расположения**для экстренной помощи в службе **голосовой связи** > и выберите адрес для экстренного реагирования, для которого нужно изменить расположение.
    
4. На панели действий рядом со связанными **расположениями**нажмите кнопку **изменить**.
    
5. В секции **Местоположенния связанные с** нажмите на ссылку имеющегося в списке местоположения, введите информацию о новом местоположении в поле **Изменить сведения о связанном местоположении**, а затем нажмите на кнопку **Отправить**.
    
6. Нажмите кнопку **Сохранить**.
    
## <a name="remove-an-emergency-location-from-an-emergency-address"></a>Удаление местоположения из адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Откройте > **устаревший портал** **центра администрирования Microsoft Teams**.
    
3. На панели навигации слева перейдите в **** > раздел**расположение для экстренных случаев**, а затем выберите адрес для экстренного реагирования, для которого нужно удалить расположение.
    
4. На панели действий рядом со связанными **расположениями**нажмите кнопку **изменить**.
    
5. В разделе **местоположения, связанные с**, рядом с ссылкой на расположение нажмите кнопку **Удалить**.
    
6. Нажмите кнопку **Сохранить**.
