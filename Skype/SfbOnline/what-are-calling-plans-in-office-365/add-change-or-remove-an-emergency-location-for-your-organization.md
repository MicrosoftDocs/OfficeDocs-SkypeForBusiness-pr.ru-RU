# <a name="add-change-or-remove-an-emergency-location-for-your-organization"></a>Добавление, изменение и удаление местоположения организации для экстренного реагирования

В зависимости от количества физических адресов в вашей организации вы можете создавать местоположения зданий, этажей и офисов, используемые для экстренного реагирования, чтобы сформировать конкретный адрес для экстренного реагирования. Для получения дополнительной информации см. [Что такое местоположения и адреса для экстренного реагирования, и что такое  маршрутизация вызовов?](/microsoftteams/what-are-emergency-locations-addresses-and-call-routing). 
  
Чтобы узнать о том, как получить тарифный план и сколько это стоит, см. раздел [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
  
## <a name="add-an-emergency-location-to-an-emergency-address"></a>Добавление местоположения для экстренного реагирования к адресу для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации слева перейдите к опции **Голосовая связь** > **Местоположения для экстренного реагирования** и выберите адрес для экстренного реагирования, к которому нужно добавить местоположение.
    
    > [!Important]
    > Чтобы в Центре администрирования Skype для бизнеса, на расположенной слева панели навигации, появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию Enterprise E5**, одну лицензию надстройки **Телефонная система** либо одну лицензию надстройки **Аудиоконференция**.
    
4. На панели «Действия» нажмите на кнопку **Связанные местоположения**, затем — на кнопку **Изменить**.
    
5. В секции **Местоположенния связанные с** нажмите на кнопку **Добавить новое**, введите другое связанное расположение (например, введите «Здание 64 Этаж 1»), а затем нажмите на кнопку **Отправить**.
    
6. Нажмите на кнопку **Сохранить**.
    
## <a name="change-an-emergency-location-for-an-emergency-address"></a>Изменение местоположения для экстренного реагирования в адресе

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации слева перейдите к опции **Голосовая связь** > **Местоположения для экстренного реагирования**  и выберите адрес для экстренного реагирования, для которого следует изменить местоположение.
    
4. На панели «Действия» нажмите на кнопку **Связанные местоположения**, затем — на кнопку **Изменить**.
    
5. В секции **Местоположенния связанные с** нажмите на ссылку имеющегося в списке местоположения, введите информацию о новом местоположении в поле **Изменить сведения о связанном местоположении**, а затем нажмите на кнопку **Отправить**.
    
6. Нажмите на кнопку **Сохранить**.
    
## <a name="remove-an-emergency-location-from-an-emergency-address"></a>Удаление местоположения из адреса для экстренного реагирования

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации слева перейдите к опции **Голосовая связь** > **Местоположения для экстренного реагирования** и выберите адрес для экстренного реагирования, местоположение которого следует удалить.
    
4. На панели «Действия» нажмите на кнопку **Связанные местоположения**, затем — на кнопку **Изменить**.
    
5. В разделе **Местоположенния связанные с** нажмите на кнопку **Удалить**, имеющуюся рядом со ссылкой на местоположение.
    
6. Нажмите на кнопку **Сохранить**.
