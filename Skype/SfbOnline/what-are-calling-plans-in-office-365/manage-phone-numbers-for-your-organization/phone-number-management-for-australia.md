# <a name="phone-number-management-for-australia"></a>Управление номер телефона для Австралии
> [!IMPORTANT]
> Австралия доступна только для предварительного просмотра и первых внедривших у себя клиентов. Для получения дополнительных сведений см [Telstra голосовой связи](http://aka.ms/TelstraVoicePlan). 

В Австралии мы партнерские отношения с Telstra для покупки и подготовку вызов планов. Telstra продать вызов планы и поддержку телефонных номеров управления для числа пользователей и службы для автосекретарей и вызвать очереди. Тем не менее номера службы и лицензий для аудиоконференции будет по-прежнему быть продал и управляемые Office 365.

Дополнительные сведения о [Telstra служб](http://aka.ms/TelstraVoicePlan).

Используйте эту таблицу, чтобы найти сведения о получении и управлении телефонных номеров в Австралии для Скайп для бизнеса в Интернет и рабочих групп Microsoft. 

Дополнительные сведения см. в статье [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
