# <a name="how-should-i-enter-the-phone-numbers"></a>Как ввести номера телефонов?

При переносе номера телефонов, им необходимо ввести в правильном формате. 
  
> [!NOTE]
> Каждый номер телефона или диапазон номер телефона должен указываться отдельно в каждой строке. 
  
- При вводе одного номера телефонов:
    
  - Все специальные символы будет игнорироваться (включая тире «-»). Например:
    
  - Для 10-значное число: ** &amp; \*(425\*() (\*&amp;4&amp;\*()) (\*250649** будет исправлена в **+14255550649**.
    
  - Для 11-значного числа: **1\*() (\*&amp;42&amp;\*() (\*&amp;55550649** будет исправлена в **+14255550649**.
    
  - Все теги будут игнорироваться при наличии 10 или 11 цифр. Например ** \<div > 4255551234\</div >** будет **+14255551234**.
    
  - «-», будет игнорироваться пространство и скобки. Например:
    
  - Для 10-значное число: **(425) 555-6776** будет исправлена в **+14255556776**.
    
  - Для 11-значного числа: **1(425) 555-6776** будет исправлена в **+14255556776**.
    
  - Все буквы будет рассматриваться как специальные символы и игнорируется, если номер телефона цифра 10 или 11 разрядов. Например:
    
  - Для 10-значное число: **14jaosia2reoij05jof55506ajfoj49isdjf** будет исправлена в **+14255550649**.
    
  - Для 11-значного числа: **1ade4jaoda2rfoij05ojof55506dsfoj49if** будет исправлена в **+14255550649**.
    
  - Будет исправлена любое сочетание специальные знаки, даже на других языках. Например: 
    
  - Для 10-значное число:**中文4中文2ajj5\*() (\*(5()... 551345** будет исправлена в **+14555551345**.
    
  - Для 11-значного числа:**中文4中文2$ a5\*() (\*(5()... 55 (.1345** будет исправлена в **+14555551345**.
    
  - Если все номера содержит не более 10 цифр или более 11 знаков, они будут выделены для пользователя для исправления:
    
  - \*\*5551245\* \* будут выделены и требуется исправить.
    
  - **1234567891011** будет выделена и требуется исправить.
    
  - Все номера, которые не более 10 цифр или более 11 цифр, и специальные знаки, будет выделена без автоматически исправленные.
    
  - Для 7-значного числа без специальных символов, введенных: **123456abcdefg7** будут выделены и требуется исправить, но буквы не будет игнорироваться.
    
  - Для числа 7-значных с специальных символов, введенных: **12345!@#$%^&amp;\*(--) @# $% ^&amp;\*(7)** будет выделена исправить. Специальные символы не будет игнорироваться.
    
- При вводе диапазон номеров телефонов.
    
  - Разрешены только два телефонных номеров. Меньшее число должен быть первый номер диапазона.
    
  - Все специальные символы (за исключением дефиса «-»), обрабатывается так же, как одного числа. Например **(425) 555 0&amp;\*(123-(1425) 5557899nm** будет исправлена в **+ 14255550123 - +13202040659**.
    
  - «-» Используется для только разделение двух чисел. Он не поддерживается для включения нескольких «-» в диапазон номеров. Например, следует вводить **(425) 555-0649-(425) 555-1115** в виде **(425) 5550649 - (425) 5551115**.
