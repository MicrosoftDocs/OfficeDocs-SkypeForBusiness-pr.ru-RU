# <a name="how-should-i-enter-the-phone-numbers"></a>Как правильно вводить телефонные номера?

При переносе номеров телефонов необходимо ввести их в правильном формате. 
  
> [!NOTE]
> Каждый номер телефона или диапазон номера должен быть введен отдельно в каждой строке. 
  
- Когда вы вводите один номер телефона, выполните указанные ниже действия.
    
  - Все специальные символы будут игнорироваться (в том числе тире "-"). Например:
    
  - Для 10-значных номеров: ** &amp; \*(425\*() (\*&amp;4&amp;\*()) (\*250649** будет исправлено на **+ 14255550649**.
    
  - Для 11-значных номеров: **1\*() (\*&amp;42&amp;\*() (\*&amp;55550649** будет исправлено на **+ 14255550649**.
    
  - При наличии 10 или 11 цифр все теги будут пропущены. Например, ** \<див_гт_ 4255551234\</див_гт_** будет **+ 14255551234**.
    
  - "-", пробел и скобка будут игнорироваться. Например:
    
  - Для 10-значных номеров: **(425) 555-6776** будет исправлено на **+ 14255556776**.
    
  - Для 11-значных номеров: **1 (425) 555-6776** будет исправлено на **+ 14255556776**.
    
  - Все буквы обрабатываются как специальные символы и пропускаются, если номер телефона состоит из 10 цифр или 11 цифр. Например:
    
  - Для 10-значных номеров: **14jaosia2reoij05jof55506ajfoj49isdjf** будет исправлено на **+ 14255550649**.
    
  - Для 11-значных номеров: **1ade4jaoda2rfoij05ojof55506dsfoj49if** будет исправлено на **+ 14255550649**.
    
  - Будут исправлены все комбинации специальных знаков, даже на других языках. Например: 
    
  - Для 10-значных номеров:**中文 4 中文2ajj5\*() (\*(5 ()... 551345** будет исправлено на **+ 14555551345**.
    
  - Для 11-значных номеров:**中文 4 中文 2 $ a5\*() (\*(5 ()... 55 (. 1345** будет исправлено на **+ 14555551345**.
    
  - Если числа содержат менее 10 цифр или более 11 цифр, они будут выделены, чтобы пользователь был сам правильно:
    
  - \*\*5551245\* \* будет выделена и должна быть исправлена.
    
  - **1234567891011** будет выделена и должна быть исправлена.
    
  - Любые числа, сокоторые менее 10 цифр или более 11 цифр, будут выделены без автоматического исправления.
    
  - Для 7-значного числа без ввода специальных знаков: **123456abcdefg7** будет выделена, и его необходимо будет исправить, но буквы не будут игнорироваться.
    
  - Для 7-значного числа со всеми введенными символами: **12345! @ # $% ^&amp;\*()--@ # $&amp;\*% ^ () 7** будет выделена для исправления. Специальные символы не будут игнорироваться.
    
- Когда вы вводите диапазон телефонных номеров.
    
  - Разрешено использовать только два номера телефона. Меньший номер должен быть первым числом в диапазоне.
    
  - Все специальные символы (за исключением тире "-") обрабатываются одинаково одинарными числами. Например, **(425) 555 0&amp;\*(123-(1425) 5557899nm** будет исправлено на **+ 14255550123-+ 13202040659**.
    
  - "-" Используется только для разделения двух чисел. Она не поддерживается для включения нескольких "-" в диапазон номеров. Например, **(425) 555-0649-(425) 555-1115** следует ввести как **(425) 5550649-(425) 5551115**.
    
  **Полные пошаговые инструкции можно найти [в статье перенос номеров телефонов в Office 365](/microsoftteams/transfer-phone-numbers-to-office-365).**

  > [!NOTE]
  > Если вам нужно больше номеров, [обратитесь в службу поддержки администраторов Skype для бизнеса](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
