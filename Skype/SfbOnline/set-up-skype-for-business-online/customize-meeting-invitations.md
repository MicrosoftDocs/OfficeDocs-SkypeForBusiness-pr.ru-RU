# <a name="customize-meeting-invitations"></a>Настройка приглашений на собрания

Как администратор вы можете включить во все приглашения на собрания Skype для бизнеса некоторые полезные сведения, которые могут потребоваться пользователям, например ссылки на веб-сайт поддержки и юридическое заявление, логотип вашей компании и другой текст, который вы хотите включить. 
  
## <a name="tips-for-creating-a-logo-for-meeting-invitations"></a>Советы по созданию логотипа для приглашения на встречу
<a name="__top"> </a>

1. Создайте изображение размером не более 188 пикселей в ширину и 30 пикселей в высоту (оно довольно небольшое).
    
2. Сохраните изображение в формате JPG.
    
3. Храните изображение в централизованном расположении, которое будет доступны всем в вашей компании, например в общей сетевой папке, общей папке OneDrive для бизнеса, Dropbox и т. д.
    
    Теперь вы можете добавить его в свои приглашения на встречу. Смотрите следующие шаги.
    
## <a name="customize-your-meeting-invitations"></a>Настройте приглашения на встречи
<a name="__top"> </a>

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

1. Войте свою учетную запись с помощью своей учебной или учебной учетной записи.
    
2. В Центре администрирования выберите **центры** администрирования  >  **Skype для бизнеса.**
    
3. В Центре администрирования Skype для бизнеса выберите **"Собрания по сети".** 
    
4. Заполнять следующие поля:
    
    ![Вот как может выглядеть форма настройки приглашений на собрания.](../images/b0a7c3c6-0d86-41c6-b116-331143bbe398.png) 

   **URL-адрес логотипа:** Введите URL-адрес, где хранится логотип. Помните, что он должен быть в формате JPG! 
 
   **URL-адрес справки:** Если в вашей организации есть веб-сайт службы поддержки, на который люди должны перейти в случае проблем, введите ЗДЕСЬ URL-адрес. 

   **URL-адрес юридического адреса:** Если в вашей организации есть юридический веб-сайт, на который вы хотите перейти, если у них есть юридические проблемы, введите здесь URL-адрес.
    
   **Текст при этом:** Введите текстовую сноску, которая подходит для пользователей.  
  
   
5. Нажмите кнопку **Сохранить**.
    
6. Подождите около часа, пока изменения будут внесены во все службы Microsoft 365 или Office 365. Затем перейдите в Outlook и запланировать собрание Skype, чтобы увидеть, как выглядят ваши изменения. 
    
    ![Вот как выглядит пользовательское приглашение без рисунка](../images/ebb5c03c-c23d-4da7-97f1-9b13e26a6cf8.png)
