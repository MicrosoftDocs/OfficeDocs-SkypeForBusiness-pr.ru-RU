# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>Функция Skype для бизнеса Online не включена

Похоже, у пользователя нет лицензии на этот компонент.
  
|**Возможная причина**|**Предлагаемое действие**|
|:-----|:-----|
|У вас есть доступные лицензии, но вы не включили функцию для этого пользователя.  <br/> |Перейдите в раздел > **Активные пользователи** **центра администрирования Office 365**для лицензирования этого пользователя. <br/> |
|Пользователь находится в том месте, где эта функция недоступна.  <br/> |По возможности перейдите в раздел > **Активные пользователи** **центра администрирования Office 365**и назначьте ему расположение, в котором она доступна. <br/> |
|Функция не включена для вашей организации.  <br/> |Откройте > **Skype для бизнеса** в **центре администрирования Office 365**и включите функцию для всей Организации. <br/> |
