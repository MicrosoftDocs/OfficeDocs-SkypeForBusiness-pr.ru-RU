# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>Эта Skype для бизнеса Online не включена

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

Похоже, у пользователя нет лицензии на эту функцию.
  
|**Возможная причина**|**Рекомендуемые действия**|
|:-----|:-----|
|У вас есть доступные лицензии, но вы не включили эту функцию для этого пользователя.  <br/> |Перейдите в центр **Microsoft 365**  >  **Активные пользователи,** чтобы на лицензию этого пользователя. <br/> |
|Пользователь находится в расположении, где эта функция недоступна.  <br/> |По возможности перейдите в центр **Microsoft 365** Активные пользователи и назначьте этому пользователю расположение, в котором  >   доступна эта функция. <br/> |
|Эта функция не включена для вашей организации.  <br/> |Перейдите **Microsoft 365 центр администрирования** Skype для бизнеса и включите эту  >   функцию для всей организации. <br/> |
