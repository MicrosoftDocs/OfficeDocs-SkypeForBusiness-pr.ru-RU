# <a name="set-up-skype-for-business-online"></a>Настройка Skype для бизнеса Online

Необходимо иметь разрешения глобального администратора Office 365 для настройки Скайп для бизнеса. Если брандмауэр или прокси-сервер, который ограничивает доступ к частям веб-сайта, рассмотрите возможность найма [партнера корпорации Майкрософт](https://go.microsoft.com/fwlink/?linkid=391089) для настройки Скайп для бизнеса.

## <a name="setting-up-skype"></a>Настройка Скайп

Похоже, вам требуется помощь настройке Скайп подписки Office 365. Необходимо выполнить действия, описанные в этой статье для получения Установка завершена.

## <a name="1-plan-for-skype-for-business"></a>1. Планирование Скайп для бизнеса

При наличии **[Office 365 бизнеса расширенный](https://products.office.com/en-us/business/office-365-business-premium)** или **Business Essentials**можно использовать Скайп для бизнеса online обращаться к другим людям в бизнесе, пользователей, которые находятся на подписку. Например, если на предприятии есть 10 человек, можно будет начать [работу с использованием Скайп для бизнеса для обмена мгновенными Сообщениями и собрания по сети](https://support.office.com/article/cc05afa6-1894-4a82-9dd9-6222061f50fd) друг с другом и [собрания по сети с Скайп для бизнеса](https://support.office.com/article/2eed8424-581a-4497-b505-c08c152e5851) с помощью Скайп для бизнеса после выполнения действия 2 – 6 ниже. И можно [настроить Скайп для собраний в Outlook](https://support.office.com/article/b8305620-d16e-4667-989d-4a977aad6556#bkmk_OWA) для собраний по сети, слишком!

Если вы хотите использовать Скайп для бизнеса и принимать **звонки** от людей *внешних* бизнеса:

- **Вариант 1. Используйте бесплатный [продукт Skype](https://www.skype.com/)**. Если у вас есть очень малого бизнеса (например, 1-2 участников), с помощью приложения Скайп — это лучший способ перехода. Он более экономен для внутренних и международных звонков. По-прежнему могут участвовать в конференциях, выполнение видеозвонков и общий доступ к рабочему столу для презентаций. [Сведения о тарифах и вариантах оплаты.](https://secure.skype.com/en/calling-rates?wt.mc_id=legacy&amp;expo365=bundled).

- Параметр **2. Обновление плана и приобрести телефонной системой и план вызов для Office 365**. Это самый простой способ узнать, насколько затраты и затем переход, это [обращение в службу](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) поддержки продуктов бизнес - Admin справки, их делать все для вас.

Для получения дополнительных сведений см. [Планирование настройки Office 365 для бизнеса](https://support.office.com/article/eb926624-018b-4486-bf11-5fba6ee4d645#bkmk_skype).

## <a name="3-set-up-your-domain-and-users"></a>3. Настройка домена и пользователей
<a name="bkmk_users"> </a>

Теперь, когда выполнен вход в Office 365, можно выполнить настройку домена и людей в организации для использования Скайп для бизнеса в Интернет.

1. 2. [Проверка домена и подключения к DNS](https://support.office.com/article/2b54e1b0-47a7-4018-a1e4-c2b924e7c5a0): используйте наш tool — средство устранения неполадок домены - для проверки правильности домена и параметры DNS. Теперь при этом будут поступать длинный способом, чтобы помочь вам понять, найденные проблемы установки более поздней версии с момента можно исключить параметры DNS источником проблем в будущем.

3. [Office 365 URL-адреса и диапазоны IP-адресов](https://support.office.com/article/8548a211-3fe7-47cb-abb1-355ea5aa88a2#BKMK_LYO): большинство предприятий малого бизнеса не требуется выполнять это действие. **Но если брандмауэр или прокси-сервер, который ограничивает доступ к частям веб-сайта**, необходимо создать правила, чтобы разрешить доступ к Скайп для бизнеса в Интернет конечных точек. Это расширенный шаг, рекомендуется выполнять другим знакомства с помощью брандмауэров и прокси-серверов по настройке. Если вы еще не сделали этого, рассмотрите возможность найма [партнера корпорации Майкрософт](https://go.microsoft.com/fwlink/?linkid=391089) для настройки Скайп для бизнеса.

### <a name="set-up-audio-conferencing"></a>Настройка аудиоконференций

Иногда пользователям в организации требуется воспользоваться телефоном, чтобы принять участие в собрании. Скайп для бизнеса включает в себя функцию аудиоконференции для только что этой ситуации! Люди могут присоединяться к собраниям Skype для бизнеса с помощью телефона, а не использовать приложение Skype для бизнеса на мобильном устройстве или компьютере.

### <a name="set-up-phone-system-and-the-calling-plans-in-office-365"></a>Настройка телефонной системой и Тарифные планы в Office 365

Свойство телефонной системой в Office 365 предоставляет телефонной системы для вашего бизнеса. Звонков на других Скайп для деловых людей в организации могут свободно и вашим сотрудникам получать голосовую почту друг от друга и за пределами абоненты. Вот, вы получаете с телефонной системой.

При добавлении службы вызов планирование вашим сотрудникам получить основной номер телефона в Скайп для бизнеса. Они могут выполнение и прием телефонных звонков за пределами вашей организации. Они могут сделать голосовые звонки на телефоны, ПК и мобильных устройств VoIP. И в случае чрезвычайные ситуации, они могут вызывать 911 для справки.

### <a name="set-up-skype-meeting-broadcast"></a>Настройка широковещательного Скайп собрания

Скайп собрания вещания — это функция, позволяющая создать узел, и широковещательного собрания с размером до 10 000 пользователей. **Чтобы узнать больше о том, как это работает, обратитесь к разделу [возможности собраний Скайп, вещание?](https://support.office.com/article/c472c76b-21f1-4e4b-ab58-329a6c33757d)**

Ниже приведен обзор действия по настройке вещания Скайп собрания:

1. [Назначение и удаление лицензий на Office 365 для бизнеса](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc): назначение **Скайп для бизнеса в Интернет** или **Планирование корпоративных** лицензий для всех, кто собирается **узла** собрания вещания.

2. [Включение Скайп вещания собрания](../set-up-your-network-for-skype-meeting-broadcast/enable-skype-meeting-broadcast.md): по умолчанию, эта функция не включена. После включения его, вашей пользователи смогут для размещения широковещательных собрания с другим сотрудникам вашей организации.

3. [Настройка сети для вещания собрания Скайп](../set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast.md): Если вы хотите узла семинары и другие вещании человек за пределами вашей организации, необходимо настроить сеть.

4. [Расписание вещания Скайп собрания](https://support.office.com/article/c3995bc9-4d32-4f75-a004-3bc5c477e553) и [присоединиться к собранию вещания Скайп](https://support.office.com/article/14689da0-821d-48d4-9035-ea762de80ebe): Убедитесь, что вещания собраний работать при планировании собрания вещания Скайп в *https://portal.broadcast.skype.com* и затем необходимости пытаться присоединиться к собранию.
