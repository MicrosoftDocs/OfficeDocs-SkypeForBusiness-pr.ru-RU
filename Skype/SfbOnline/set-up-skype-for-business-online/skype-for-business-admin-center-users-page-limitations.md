# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>Ограничения страницы пользователей в Центре администрирования Skype для бизнеса Online

По соображениям производительности в Центре администрирования **Skype** для бизнеса отображаются первые 1000 пользователей в списке пользователей. Если у вас более 1000 пользователей, используйте фильтр фильтра и поиск, чтобы найти  ![ ](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png)  ![ ](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) нужного человека.
  
## <a name="bulk-editing"></a>Массовое редактирование
<a name="__top"> </a>

1. Выберите первые 1000 пользователей (или всех пользователей, возвращенных фильтром или  поиском), поверив поле рядом с отображаемой именем в верхней части списка.
    
2. Нажмите **кнопку** ![ ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png) "Изменить".
    
Вы можете изменить количество пользователей, которые должны быть на каждой странице, с помощью этого управления в нижней части списка.
  
![Навигация по страницам пользователей для изменения количества пользователей на странице](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
