# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>Ограничения страницы пользователей в центре администрирования Skype для бизнеса Online

Для повышения производительности в **центре администрирования Skype для бизнеса**отображаются первые пользователи 1 000 из вашего списка пользователей. Если у вас более 1 000 пользователей, используйте фильтр ****![](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png) фильтров и](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) поиск **поиска**![, чтобы найти конкретных людей, которых вы ищете.
  
## <a name="bulk-editing"></a>Массовое редактирование
<a name="__top"> </a>

1. Выберите первые 1 000 пользователей (или все, что возвращены фильтром или поиском), установив флажок рядом с **отображаемым именем** в верхней части списка.
    
2. Выберите ****![команду Изменить](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png)изменения.
    
Вы можете изменить количество пользователей, отображаемых на каждой странице, с помощью элемента управления в нижней части списка.
  
![Навигация по страницам пользователей для изменения количества пользователей на странице](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
