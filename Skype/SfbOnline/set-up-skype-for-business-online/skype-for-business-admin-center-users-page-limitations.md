# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>Skype для бизнеса Ограничения страниц пользователей в Центре администрирования Online

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

По соображениям производительности **в Skype для бизнеса** отображаются первые 1000 пользователей в списке пользователей. Если у вас более 1000 пользователей, используйте фильтр  ![ фильтра.](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png) и **в поиске** найдите людей, ![ которых вы ](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) ищете.
  
## <a name="bulk-editing"></a>Массовое редактирование
<a name="__top"> </a>

1. Чтобы выбрать первых 1000 пользователей (или всех пользователей, возвращенных фильтром  или поиском), выберите поле Рядом с полем Отображаемого имени в верхней части списка.
    
2. Нажмите **кнопку Изменить** ![ ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png) редактирование.
    
Вы можете изменить количество пользователей, показанных на каждой странице, с помощью этого управления в нижней части списка:
  
![Навигация по страницам пользователей для изменения количества пользователей на странице.](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
