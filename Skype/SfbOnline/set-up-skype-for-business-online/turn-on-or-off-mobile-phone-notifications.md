# <a name="turn-on-or-off-mobile-phone-notifications"></a>Включение и отключение уведомлений на мобильном телефоне

Как **[назначить роли администратора в приложениях Microsoft 365 для бизнеса в](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** Организации, вы можете выбрать, будут ли пользователи Skype для бизнеса получать оповещения о входящих и пропущенных мгновенных сообщениях, находящихся на мобильных телефонах или планшетах.
  
На устройствах с Android и Windows на мобильных телефонах всплывающие уведомления Skype для бизнеса находятся в режиме реального времени. Однако для устройств с Windows Phone, iPhone и iPad при использовании push-уведомлений вы можете показывать оповещения каждый раз, когда вы не используете Skype для бизнеса на телефоне или планшете.
  
## <a name="turn-push-notifications-off-for-all-the-windows-phone-or-apple-devices-in-your-organization"></a>Отключение push-уведомлений для всех устройств с Windows Phone и Apple в Организации
<a name="__top"> </a>

![Значок с логотипом Skype для бизнеса](../images/sfb-logo-30x30.png) **Использование центра администрирования Skype для бизнеса**

1. Войдите в Microsoft 365.
    
2. Перейдите в центр администрирования > **Skype для бизнеса**.
    
3. Перейдите к разделу**Общие** **Организации** > . 
    
4. В разделе **уведомления на мобильном телефоне**снимите флажок рядом с службой уведомлений, которую вы хотите отключить, и нажмите кнопку **сохранить**.
    
Помните: 
  
- Если вы отключите push-уведомления, пользователи по-прежнему получат все оповещения при повторном запуске Skype для бизнеса на мобильном устройстве.
    
- Push-уведомления включены по умолчанию. Отдельные пользователи могут отключить их, выбрав соответствующий параметр Skype для бизнеса на своем мобильном устройстве.
    
- Если вы отключите push-уведомления, пользователи не смогут снова включить их.
    
> [!IMPORTANT]
>  Корпорация Майкрософт использует другие компании для получения уведомлений о Skype для бизнеса в реальном времени для пользователей Windows Phone, iPhone и iPad. Ознакомьтесь с заявлением [о конфиденциальности для продуктов Microsoft Skype для бизнеса](https://go.microsoft.com/fwlink/p/?linkid=247732).
