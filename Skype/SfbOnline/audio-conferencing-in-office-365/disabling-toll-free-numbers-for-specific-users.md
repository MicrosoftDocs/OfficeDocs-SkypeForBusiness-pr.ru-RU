# <a name="disabling-toll-free-numbers-for-specific-skype-for-business-online-users"></a>Отключение бесплатных номеров для конкретных Скайп для бизнеса в Интернет пользователей

> [!Note]
> Сведения о отключение номеров без использования средства для группы пользователей видеть [Отключение бесплатных номеров для определенных групп пользователей](/MicrosoftTeams/disabling-toll-free-numbers-for-specific-teams-users).

Если ваша организация имеет бесплатных номеров в его Bridge конференц-связи Microsoft звук, можно разрешить или запретить их использование в собраниях конкретных организаторов.  

По умолчанию все пользователи в вашей организации включены по использованию бесплатных номеров, что означает, что эти номера при наличии, можно использовать участниками на присоединение к собраниям их. Если это не желаемое поведение для некоторых пользователей в вашей организации, вы можете запретить определенными пользователями с помощью этих номеров в свои собрания с помощью бесплатных номеров включение элемента управления. 

Когда бесплатных номеров отключены для данного организатора: 
 - Бесплатный номер больше не требуется включить в его или ее собрания приглашает. 
 - Обслуживание бесплатных номеров больше не отображается на странице «Найдите местный номер», которое содержит ссылку на его или ее собрания приглашает. 
 - Участники не сможет присоединиться к собранию данного организатора, если они набрать любой бесплатный номер организации. 
 - Автоматически будет выполнено всех собраний организатора и телефоны будет удален из них.  

    > [!IMPORTANT]
    > Это будет отправить все приглашения электронной почты организатора всем участникам собраниям. 

 - Участники могут продолжать присоединения к собраниям картинок, с помощью платные номера. 

## <a name="disabling-toll-free-numbers-for-specific-users"></a>Отключение бесплатных номеров для отдельных пользователей 


1. В **Скайп по центру администрирования бизнеса**, в левой области переходов, перейдите к **аудиоконференции** > **пользователей**, а затем выберите пользователя из списка доступных пользователей. 

2. In the Action pane, click **Edit**. 

3. Снимите флажок **Разрешить использование бесплатных номеров для присоединения к собраниям этого пользователя**. 
 
4. Также можно отправить электронное письмо пользователю с помощью параметров аудиоконференции, выбрав на странице свойств аудиоконференции для пользователя параметр **Отправить информацию о конференции по электронной почте**. 
 
> [!Note]
> [!INCLUDE [updating-admin-interfaces](../includes/updating-admin-interfaces.md)]
 
**С помощью PowerShell**  

Параметр AllowTollFreeDialIn командлета Set-CsOnlineDialInConferencingUser можно использовать для включения или отключения данного элемента управления. Например: 

 - SET-CsOnlineDialInConferencingUser user@contoso.com – AllowTollFreeDialIn $false
