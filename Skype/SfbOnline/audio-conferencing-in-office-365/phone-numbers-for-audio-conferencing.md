# <a name="phone-numbers-for-audio-conferencing-in-skype-for-business-online"></a>Номера телефонов для аудиоконференций в приложении Skype для бизнеса Online

> [!NOTE]
> Сведения о телефонных номерах Microsoft Teams см. в разделе [Номера телефонов для аудиоконференций в Microsoft Teams](/MicrosoftTeams/phone-numbers-for-audio-conferencing-in-teams).

При настройке **Аудиоконференций** в Skype для бизнеса телефонные номера назначаются вашей организации автоматически. Для просмотра телефонных номеров, назначенных для вашего моста аудиоконференций, перейдите в **Центр администрирования Skype для бизнеса** > **Аудиоконференции** > **Мост Microsoft**. См. раздел [Просмотр списка номеров для аудиоконференций](see-a-list-of-audio-conferencing-numbers.md).
  
> [!NOTE]
> Не существует никакого ресурса, который содержит список всех телефонных номеров для аудиоконференций. Для поиска телефонных номеров, доступных в вашей стране или регионе, используйте **Центр администрирования Skype для бизнеса** > **Голосовая связь** > **Номера телефонов**, щелкните **Добавить** и затем щелкните **Новые служебные номера**. Для фильтрации поиска воспользуйтесь списками **Страна или регион**, **Область или регион** и **Город**. Для поиска бесплатных служебных номеров выберите **Бесплатный номер** в списке **Область или регион**.

## <a name="dial-in-phone-numbers-in-a-meeting-invite"></a>Телефонные номера в приглашении на собрание

Когда пользователь Skype для бизнеса Online планирует собрание в приложении Outlook или Outlook Web App, в приглашении на собрание указывается номер аудиоконференции по умолчанию, установленный для пользователя. Если вы хотите выбрать номер по умолчанию для одного или нескольких пользователей, перейдите в **Центр администрирования Skype для бизнеса** > **Аудиоконференции** > **Пользователи**. См. [Определение телефонных номеров, включаемых в приглашения](set-the-phone-numbers-included-on-invites.md).
  
Чтобы просмотреть другие номера для подключения, перейдите по ссылке **Найти местный номер** в приглашении на собрание.
  
## <a name="dial-in-phone-numbers-set-on-an-audio-conferencing-bridge"></a>Телефонные номера, заданные для моста аудиоконференции

Существует два типа телефонных номеров аудиоконференции, назначаемых для моста аудиоконференции: **общедоступные** и **выделенные**. Любой абонент может использовать оба типа номеров для подключения к соответствующим аудиособраниям, проводимым в организации.
  
 **Выделенные телефонные номера**  это номера, доступные только для пользователей внутри организации. Вы можете изменить языки, которые используются при звонках на эти номера.
  
 **Общедоступные телефонные номера**  это номера, доступ к которым можно предоставлять другим организациям Office 365. Изменить языки, которые используются при звонках на эти номера, нельзя.
  
Учитывая, что телефонный номер для аудиоконференции, назначенный организатору по умолчанию, указывается только в приглашение на собрание, для подключения к собранию абонент может использовать любой из телефонных номеров, назначенный мосту конференц-связи. Список номеров, которые можно использовать для подключения к собранию, открывается при переходе по ссылке **Найти местный номер**, которая включена во все приглашения.
  
## <a name="automatically-assigned-audio-conferencing-phone-numbers"></a>Автоматически назначаемые телефонные номера для доступа к аудиоконференции

Общедоступные телефонные номера для доступа к аудиоконференции автоматически назначаются организациям при активации соответствующей функции. Полученные номера телефонов назначаются мосту аудиоконференции в качестве телефонных номеров по умолчанию. Номер телефона, назначенный мосту по умолчанию, должен соответствовать стране или региону нахождения организации.
  
> [!NOTE]
> Для поиска страны или региона вашей организации необходимо войти в **Центр администрирования Office 365** и найти раздел **Профиль организации**. 
  
> [!CAUTION]
> В связи с ограниченной емкостью пула платных номеров телефонов в Венесуэле, Индонезии и Объединенных Арабских Эамиратах (ОАЭ) организации из этих стран не получают платный номер для аудиоконференции автоматически. Для этих стран доступны бесплатные номера независимо от существующих ограничений. 
  
Выделенные телефонные номера для доступа к аудиоконференции — это служебные номера, которые вы можете получить, а затем назначить своей организации. Служебные номера можно найти с помощью **Центра администрирования Skype для бизнеса**. Дополнительные сведения см. в разделе [Получение служебных номеров телефонов](/SkypeForBusiness/what-is-phone-system-in-office-365/getting-service-phone-numbers).
  
Список этих стран или регионов, имеющих номера телефонов, автоматически назначаемых для организаций, приведен в разделе [Страны и регионы, для которых доступны аудиоконференции и тарифные планы](/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans).
  
## <a name="what-else-should-you-know"></a>Дополнительные сведения

- Со списком поддерживаемых языков для аудиоконференций можно ознакомиться в разделе [Поддерживаемые языки аудиоконференций](/MicrosoftTeams/audio-conferencing-supported-languages).
    
- Можно использовать командлет [Get-CsOnlineDialInConferencingServiceNumber](https://go.microsoft.com/fwlink/?LinkId=617691), чтобы просмотреть выделенные телефонные номера для подключения к аудиоконференции в вашей организации.
    
- Можно использовать командлет [Get-CsOnlineDialInConferencingLanguagesSupported](https://go.microsoft.com/fwlink/?LinkId=617684), чтобы просмотреть список языков, поддерживаемых для выделенного телефонного номера для подключения к аудиоконференции.
    
- Для каждого телефонного номера для доступа к аудиоконференции можно задать до четырех языков: один основной и три дополнительных. Также можно настроить языки для выделенного телефонного номера для доступа к аудиоконференции.
    
- Сведения о настройке телефонного номера для подключения отдельного пользователя см. в разделе [Определение телефонных номеров, включаемых в приглашения](set-the-phone-numbers-included-on-invites.md).
