# <a name="audio-conferencing-troubleshooting-and-known-issues"></a>Известные проблемы аудиоконференций и устранение неполадок

 **Эта статья для пользователей Skype для бизнеса, использующих Майкрософт в качестве поставщика аудиоконференций. Он не относится к клиентам, которые используют стороннее поставщика аудиоконференций (ACP).**
  
## <a name="troubleshooting-and-known-issues"></a>Известные проблемы и устранение неполадок

Аудиоконференция с использованием Майкрософт в качестве поставщика услуг аудиоконференций имеет текущие проблемы, которые отслеживаются и активно исследуются и будут устранены при обновлении этой функции в будущих выпусках Microsoft 365.
  
В настоящее время используйте его для справки при устранении потенциальных проблем с настройками и работой аудиоконференций для людей, использующих Skype для бизнеса в вашей организации.
