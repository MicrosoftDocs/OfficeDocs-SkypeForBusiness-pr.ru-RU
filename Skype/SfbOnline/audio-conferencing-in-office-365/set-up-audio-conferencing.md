# <a name="set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams"></a>Настройка аудиоконференций в Skype для бизнеса и Microsoft Teams

В некоторых случаях пользователям в организации требуется телефон, чтобы присоединиться к собранию. Скайп для бизнеса и группами Майкрософт включить функцию аудиоконференций для только что этой ситуации! Люди могут вызывать в Скайп для бизнеса или группами Майкрософт собраний с помощью телефона, вместо использования Скайп для бизнеса или группами Майкрософт приложения на мобильном устройстве или Компьютере. 
  
Необходимо настроить звук конференц-связи для пользователей, которые планируется расписание или привести собрания. Участники собрания присоединяются к не требуется никаких лицензий на их или другие программы установки.
  
Часто задаваемые вопросы о конференц-связи звук в разделе [распространенные вопросы аудио конференц-связи](audio-conferencing-common-questions.md).
  
## <a name="step-1-find-out-if-audio-conferencing-is-available-in-your-countryregion"></a>Шаг 1: Узнайте, доступен ли звук конференц-связи в вашей стране или регионе
<a name="__top"> </a>


Перейти к [страны и региона для конференц-связи аудио и вызов планы](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) и выберите страну или регион, чтобы получить сведения о доступности о звук конференц-связи, а также сведения о телефонной системой, вызов планы международную и бесплатный номер номера и кредитов коммуникаций. 
 
## <a name="step-2-get-and-assign-licenses"></a>Шаг 2: Получение и назначение лицензий
 
1. Для аудиоконференции требуется лицензия для каждого пользователя, который будет планировать собрания-связи. Чтобы узнать, какие лицензии, которые необходимо купить для аудиоконференции, и они будут затратами, обратитесь к разделу [Скайп для бизнеса и группами Майкрософт дополнительный компонент лицензирования](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md).
        
2. После приобрести лицензии аудиоконференции, будет подключен назначение их для пользователей в вашей организации, которые будут расписание или привести собрания. В разделе [Назначение и удаление лицензий на Office 365 для бизнеса](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) приобретается для пользователей в вашей организации, которые будет расписание или ведущий сотрудник собраний.
    
3. Мы также рекомендуем, чтобы назначать лицензий кредитов Communications (они не расходы на что-либо) же людям назначенных лицензий на предыдущем шаге. В этой статье описывается настройка кредитов коммуникаций, см [кредитов коммуникаций для вашей организации](../skype-for-business-and-microsoft-teams-add-on-licensing/set-up-communications-credits-for-your-organization.md).
    
> [!NOTE]
> Можно также настроить конференц-связи аудио-минуту. [Здесь](../skype-for-business-and-microsoft-teams-add-on-licensing/audio-conferencing-pay-per-minute.md) для получения дополнительных сведений о том, как их использовать.

## <a name="step-3-get-service-numbers-for-your-conferencing-bridges"></a>Шаг 3: Получение номеров службы для конференц-каналов
<a name="__top"> </a>

Для аудиоконференции нельзя использовать номера телефонов для пользователей; необходимо получить номера службы. Вы можете получить бесплатный или службы бесплатных номеров для конференц-каналов. Существует три способа get международную и номера бесплатных службы: 
  
- **Используйте Скайп по центру администрирования бизнеса.** Для некоторых странах или регионах можно получить номера службы для вашей конференц-каналов, с помощью Скайп по центру администрирования Business, см [Приступая к службе телефонных номеров](../what-is-phone-system-in-office-365/getting-service-phone-numbers.md).
    
- **На существующие службы номер порта.** Для порта или перемещение существующих номеров из текущего поставщика услуг или телефон поставщика в Office 365. Вы можете увидеть [передачи телефонных номеров в Office 365](../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md) или [Управление номера телефонов для вашей организации](../what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) для получения дополнительных сведений, помогут вам сделать это.  
  
- **Форма запроса для нового номера.** В некоторых случаях (в зависимости от страны или региона) не будут иметь возможность получать номерам записи службы с помощью Скайп по центру администрирования Business или вам потребуется номера телефона или код области. В таких случаях необходимо скачать соответствующую форму, заполнить ее и отправить в нашу службу поддержки. Для получения дополнительных сведений см [Управление номера телефонов для вашей организации](../what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) . 
    
## <a name="step-4-assign-a-service-number-to-the-conferencing-bridge"></a>Шаг 4: Назначьте номер службы конференц-канала
<a name="__top"> </a>

После получения международную и/или бесплатные номера телефонов для конференц-канал, необходимо присвоить номера, поэтому они могут использоваться на приглашения на собрания.  

Чтобы назначить новый номер телефона звукового конференц-канала, перейдите в **Центр администрирования Office 365** > **центры администрирования** > **Скайп для бизнеса** > **голосовой связи** > **номера телефонов**, телефон номер и нажмите кнопку **назначить**.

[Новый номер телефона для звукового конференц-канала](../audio-conferencing-in-office-365/change-the-phone-numbers-on-your-audio-conferencing-bridge.md)для получения дополнительных сведений см.

## <a name="step-5-set-the-default-and-alternate-languages-for-a-conferencing-bridge"></a>Шаг 5: Настройка по умолчанию и альтернативные языки для конференц-канала
<a name="__top"> </a>

Далее необходимо [задать auto attendant языков для аудиоконференции](../audio-conferencing-in-office-365/set-auto-attendant-languages-for-audio-conferencing.md) автосекретарь конференц-связи с помощью приветствовать абонента, когда они звонить на номер телефона для аудиоконференции. 

Перейдите в **Центр администрирования Office 365** > **центры администрирования** > **Скайп для бизнеса** > **аудиоконференции** > **параметров Microsoft bridge**, выберите номер телефона bridge конференц-связи, а затем Выберите **набор языков**.

## <a name="step-6-set-your-conferencing-bridge-settings"></a>Шаг 6: Настройки вашей конференц-связи моста
<a name="__top"> </a>
    
После настройки конференц-канал, убедитесь, что параметры по умолчанию, например, запись и выхода уведомлений и длина ПИН-кода из них, которые вы хотите использовать; Если они не могут быть изменены. 

Перейдите в **Центр администрирования Office 365** > **центры администрирования** > **Скайп для бизнеса** > **аудиоконференции** > **параметров моста Microsoft**. Откроется страница **параметров моста Microsoft** . [Параметры для моста звук конференц-связи](../audio-conferencing-in-office-365/change-the-settings-for-an-audio-conferencing-bridge.md)для получения дополнительных сведений см.

## <a name="step-7-assign-the-audio-conferencing-provider-and-dial-in-phone-numbers"></a>Шаг 7: Назначение аудиоконференций поставщика и телефонные номера телефонов

Теперь необходимо убедиться, что Microsoft назначен в качестве поставщика и задать международную и обслуживание бесплатных номеров для их в то же время.

Назначение Microsoft в качестве поставщика для пользователей в вашей организации, привести или планировать собрания, выбрав пункты **центра администрирования Office 365** > **Скайп для бизнеса** > **аудиоконференции** > **пользователей**, а затем выберите пользователя из списка и нажмите кнопку **Изменить**. Найти более подробных сведений в разделе [Назначение Microsoft в качестве поставщика аудиоконференций](../audio-conferencing-in-office-365/assign-microsoft-as-the-audio-conferencing-provider.md).

При задании поставщика, можно также задать международную и обслуживание бесплатных номеров, который будет добавлен к собранию приглашает для этого пользователя. Выберите из раскрывающегося списка телефонных номеров. [Телефона, номера, находящимся на приглашает](../audio-conferencing-in-office-365/set-the-phone-numbers-included-on-invites.md)для получения дополнительных сведений см.

Тем не менее если вы хотите, чтобы можно добавить свои собственные справки и юридических ссылок, текстовое сообщение и рисунок небольшого предприятия.
