# <a name="ip-relay-in-canada---user-guide"></a>IP-ретрансляция в Канаде — руководство пользователя

Описание службы ретрансляции IP-сообщений от Майкрософт:

## <a name="scenario-a"></a>Сценарий А
Если человек с нарушениями слуха или голосовой связи хочет связаться с человеком без нарушения слуха, он "общается" через специальный веб-клиент с агентом, который называется "Принимать вызовы".

После этого этот человек общается с человеком с нарушениями слуха или голоса в рамках сеанса чата и при необходимости общается с ним по телефонной сети в Канаде или США.

## <a name="scenario-b"></a>Сценарий Б
Если человек с нарушениями слуха хочет пообщаться с зарегистрированным пользователем или пользователем с нарушениями голоса, он звонит на бесплатный номер, предоставленный Корпорацией Майкрософт.

После этого этот человек общается с человеком с нарушениями слуха или с человеком с нарушениями голоса в чате и общается в чате и между ними.

Если человек с нарушениями слуха или голосовой почты отсутствует в сети в программе чата, если доступен адрес электронной почты, то по запросу вызываемого лица ему будет отправлено сообщение электронной почты.

## <a name="scenario-c"></a>Сценарий C
Если человеку с нарушениями слуха или голосовой поддержки требуется помощь службы экстренной помощи, он может нажать соответствующую кнопку: 911 – Политика / 911 — Fire / 911 — Только что.

В этом случае операторы ретрансляции IP Relay для Северной северной территории получат доступ к учетной записи 911, предназначенной для этой цели. Подтвердив свой адрес, он свяжется с соответствующей psAP и будет находиться в очереди столько времени, сколько требуется.

## <a name="how-to-place-a-text-chat-to-voice-call"></a>Как сделать текстовый чат голосовым

https://aka.ms/IPRelay

Чтобы начать голосовой звонок, перейдите к url-адресу выше и войдите в систему, указав имя пользователя и пароль ретрансляции IP-сообщения.

После входа вы увидите инструкции в левой части экрана.

### <a name="how-to-make-a-text-to-voice-call"></a>Как сделать звонок с текстом на голосовую:
1. В правом нижнем углу экрана щелкните **"Чат".**
2. Щелкните **пользователя IPRelay.**
3. В нижней части нового окна введите сообщение.

### <a name="how-to-receive-a-voice-to-text-call"></a>Как принимать текстовые вызовы:
- Пользователи голосовой почты должны знать имя пользователя ретрансляции IP-сообщений, чтобы сделать голосовой звонок.
- Пользователи голосовой связи могут звонить по телефону (866) 660-8613 для подключения с помощью оператора ретрансляции IP-сообщений.
- Для получения звонков и чатов необходимо войти на портал ретрансляции IP-сообщений.

### <a name="how-to-place-a-911-call"></a>Как сделать звонок 911:
Если вы испытываете экстренные ситуации, вы можете нажать соответствующую кнопку в нижней части экрана (как показано ниже).

![Кнопки для экстренного вызова](../images/ip-relay-emergency-buttons.png)

Оператор ретрансляции IP-сообщений свяжуется с оператором 911, подтвердит ваш адрес и переведет его в различные центры экстренной помощи, пока он больше не не потребуется, а соответствующий отдел не будет отправлен в ваше расположение.

> [!WARNING]
> Не проверяйте эту службу, так как соответствующее учреждение свяжуется с ним и, возможно, будет отправлено, даже если вы указали, что это просто проверка, и с вас может взиматься ложная плата за отправку из экстренных служб.
