# <a name="make-changes-to-skype-meeting-broadcast-settings-for-your-organization"></a>Вносить изменения в параметры вещания собрания Скайп для вашей организации

Можно включить Скайп вещания собрания и внести изменения в параметры и политики для них.
  
- **Включение Скайп собрания вещания** Позволяет Скайп собрания вещания. После включения Скайп собрания вещания необходимо [Настройка сети для Скайп собрания вещания](set-up-your-network-for-skype-meeting-broadcast.md). Выполните этот шаг, если требуется хранение семинары и другие вещании по пользователям за пределами вашей организации. 
    
- **Разрешить просмотр вещания собрания Скайп компонентов для организации** Скайп бизнес-приложениями клиента обеспечивают раннее доступ к новые продукты и компоненты. Ваша организация предоставляет краткий предстоящих и возможность тестирования новых функций в собственной среде и обратная связь перед выпуске сборки продукта для всех.<br/>[Скайп для бизнеса-версия](https://www.skypepreview.com/)
    
- **Разрешить организаторов планировать анонимные собрания** Это позволяет организаторов закрытые подписи и перевода для вещания собраний.
    
- **Разрешить вещания собраний для записи** Это позволяет любой собраний, что нужно записать с помощью выступающий или Организатор.
    
- **Служба технической поддержки поддерживает URL-адрес для участников** Введите ссылку участников вещания собрания для использования, если они нужна помощь подключение или посещение собрания вещания.
