# <a name="skype-meeting-broadcast-preview-settings"></a>Параметры предварительной версии трансляции собраний Skype

Программа предварительной версии Skype для бизнеса предоставляет ранний доступ к новым продуктам и функциям. Это позволяет вашей организации посмотреть, что нового происходит, протестировать новые функции в собственной среде и дать отзывы, прежде чем мы разберемся в сборках продуктов для всех пользователей. Дополнительные узнать см. в предварительной [версии Skype для бизнеса.](https://www.skypepreview.com/)
