# <a name="skype-meeting-broadcast-preview-settings"></a>Skype Параметры предварительного просмотра трансляции собрания

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

Программа Skype для бизнеса предварительной версии клиента обеспечивает ранний доступ к новым продуктам и функциям. Это позволяет вашей организации узнать о предстоящих выпусках, протестировать новые возможности в собственной среде и дать отзыв, прежде чем мы выпустим сборки продуктов для всех пользователей. Дополнительные информации см. в Skype для бизнеса [предварительного просмотра](https://www.skypepreview.com/).
