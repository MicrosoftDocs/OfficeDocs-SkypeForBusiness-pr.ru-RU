# <a name="skype-meeting-broadcast-preview-settings"></a>Параметры предварительного просмотра трансляции собрания Skype

Программа предварительной версии Skype для бизнеса предоставляет ранний доступ к новым продуктам и функциям. Это позволяет вашей организации посмотреть, что будет в ближайшее время, протестировать новые функции в собственной среде и дать отзывы, прежде чем мы выпустим сборки продуктов для всех пользователей. Дополнительные узнать см. в предварительной [версии Skype для бизнеса.](https://www.skypepreview.com/)
