# <a name="blocked-user-licenses-for-hosting-skype-meeting-broadcast"></a>Заблокированные пользовательские лицензии для размещения Skype трансляции собраний

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

> [!IMPORTANT]
> Skype для бизнеса 31 июля 2021 г. заканчивается доступ к службе Online. Мы рекомендуем клиентам начать обновление до Microsoft Teams , основного клиента для связи и командной работы в Microsoft 365.

К сожалению, существуют определенные пользовательские лицензии, которые блокируются, когда пользователи находятся в Skype https://broadcast.skype.com трансляции собраний. Если пользователь видит это сообщение, этому пользователю назначена одна из заблокированных лицензий. Если вы хотите, чтобы они по-прежнему использовали его, назначьте им еще Office 365 лицензию.
  
 *"Политика вашей организации не позволяет использовать Skype трансляцию собрания. Обратитесь к системным администраторам, чтобы убедиться, что служба включена и у вас есть правильная лицензия".* 
  
## <a name="blocked-user-licenses"></a>Заблокированные пользовательские лицензии 

Размещение Skype трансляции собраний будет заблокировано, если пользователю назначены какие-либо из следующих пользовательских лицензий. Участникам, которые являются участниками потоковой передачи трансляций, не требуются пользовательские лицензии.
  
- Office 365 для образования Fac/Staff
    
- Office 365 для образования Студент
    
- Office 365 для образования плюс Fac/Staff
    
- Office 365 для образования плюс Студент
    
- Office 365 для образования E1 Fac/Staff
    
- Office 365 для образования Учащийся E1
    
- Office 365 для образования E3 Fac/Staff
    
- Office 365 для образования E3 Fac/Staff Large
    
- Office 365 для образования Учащийся E3
    
- Office 365 A1 преимущества использования учащихся
    
- Microsoft 365 A1 преимущества использования учащихся
