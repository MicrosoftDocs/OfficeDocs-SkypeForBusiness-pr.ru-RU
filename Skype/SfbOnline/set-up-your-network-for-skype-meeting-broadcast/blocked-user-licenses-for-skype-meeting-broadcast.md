# <a name="blocked-user-licenses-for-hosting-skype-meeting-broadcast"></a>Заблокированные пользовательские лицензии для размещения трансляции собраний Skype

> [!IMPORTANT]
> Skype для бизнеса Online больше не будет работать 31 июля 2021 г., после чего доступ к службе закончится. Мы рекомендуем клиентам начать обновление до Microsoft Teams, основного клиента для связи и командной работы в Microsoft 365.

К сожалению, некоторые пользовательские лицензии блокируются, когда пользователи находятся в skype и хотят провести трансляцию https://broadcast.skype.com собрания Skype. Если пользователь видит это сообщение, этому пользователю назначена одна из заблокированных лицензий. Если вы хотите, чтобы они по-прежнему использовали ее, назначьте им еще одну лицензию на Office 365.
  
 *"Политика вашей компании не позволяет использовать трансляцию собраний Skype. Уточните у системного администратора, включена ли служба и есть ли у вас правильная лицензия".* 
  
## <a name="blocked-user-licenses"></a>Заблокированные пользовательские лицензии 

Размещение трансляций собраний Skype будет заблокировано, если пользователю назначена любая из следующих пользовательских лицензий. Участникам, которые являются участниками потоковой передачи трансляций, не требуются пользовательские лицензии.
  
- Office 365 для образования: факсы и сотрудники
    
- Office 365 для образования (студенческий)
    
- Office 365 для образования плюс fac/Staff
    
- Office 365 для образования плюс (студенческий)
    
- Office 365 для образования E1: факсы и сотрудники
    
- Office 365 для образования E1 (студенческий)
    
- Office 365 для образования E3: fac/Staff
    
- Office 365 для образования E3: fac/Staff Large
    
- Office 365 для образования E3 (студенческий)
    
- Использование преимущества office 365 A1 для учащихся
    
- Преимущества Microsoft 365 A1 для учащихся
