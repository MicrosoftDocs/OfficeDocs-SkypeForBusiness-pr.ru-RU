# <a name="blocked-user-licenses-for-hosting-skype-meeting-broadcast"></a>Заблокированные лицензии пользователей на размещение трансляции собраний Skype

К сожалению, при переходе пользователей к https://broadcast.skype.com трансляции собраний Skype и их желании вы блокируете определенные пользовательские лицензии. Если пользователь видит это сообщение, ему назначается один из заблокированных лицензий. Если вы по-прежнему хотите использовать их, назначьте им другую лицензию на Office 365.
  
 *"Политика компании не разрешает вам использовать трансляцию собраний Skype. Обратитесь к системному администратору, чтобы убедиться в том, что служба включена и у вас есть нужная лицензия. "* 
  
## <a name="blocked-user-licenses"></a>Пользовательские лицензии заблокированных пользователей 

Размещение событий трансляции собраний Skype будет заблокировано, если пользователю назначены какие-либо из указанных ниже пользовательских лицензий. Участники, которые являются событиями потоковой передачи, не требуют никаких пользовательских лицензий.
  
- Office 365 для образования фак/сотрудники
    
- Office 365 для образования учащихся
    
- Office 365 для образования и фака/персонала
    
- Office 365 для образовательных учреждений и учащихся
    
- Office 365 для образовательных учреждений E1 фак/сотрудники
    
- Office 365 для образования E1 (учащихся)
    
- Office 365 для образовательных учреждений E3 фак/сотрудники
    
- Office 365 для образовательных учреждений E3 фак/штатный крупный
    
- Office 365 для образования для студентов E3
    
- Использование преимуществ Office 365 a1 для учащихся
    
- Microsoft 365 a1 для учащихся используют преимущества
