# Просмотр списка номеров для конференц-связи с телефонным подключением

При настройке конференц-связи с телефонным подключением в Skype для бизнеса online можно просматривать телефонные номера, доступные для подключения к конференциям. В списке перечисляются все телефонные номера для конференций с телефонным подключением, доступные для вашей организации. Однако в списке перечисляются не абсолютно все номера для телефонного подключения к конференциям. Дополнительные номера для стран и регионов можно найти в разделе [Телефонные номера для аудиоконференции](phone-numbers-for-audio-conferencing.md). 
  
Если для организации доступен только один номер, он используется как номер по умолчанию для всех пользователей. Если доступно несколько номеров, можно выбрать номер по умолчанию для каждого пользователя. Телефонный номер по умолчанию указывается в приглашениях на собрания Skype для бизнеса online.
  
Порядок изменения номера для телефонного подключения для отдельного пользователя см. в разделе [Определение телефонных номеров для аудиоконференций, которые организаторы собраний могут включить в приглашения](set-the-audio-conferencing-phone-numbers-for-meeting-organizers-that-are-include.md).
  
> [!NOTE]
> Внутренние телефонные номера выделены для организации, и только из них можно выбрать телефонный номер по умолчанию. Однако международные телефонные номера можно сделать общими для нескольких организаций. 
  
## Процедура просмотра номеров конференц-связи с телефонным подключением

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В окне **Центр администрирования Skype для бизнеса** на панели навигации слева выберите **Конференц-связь с телефонным подключением** > **Параметры моста Microsoft**. Затем можно выполнить одно из следующих действий:
    
  - просмотреть телефонные номера, которые следует использовать для конференц-связи с телефонным подключением;
    
  - просмотреть местоположение, основной и дополнительный языки, которые будет использовать автосекретарь конференц-связи с телефонным подключением;
    
> [!NOTE]
> Можно перейти в раздел **Конференц-связь с телефонным подключением** > **Пользователи с телефонным подключением** и выбрать свойства пользователя для изменения номера пользователя по умолчанию, выбрав новый номер в списке номеров, доступных для организации. См. раздел[Определение телефонных номеров для аудиоконференций, которые организаторы собраний могут включить в приглашения](set-the-audio-conferencing-phone-numbers-for-meeting-organizers-that-are-include.md).
