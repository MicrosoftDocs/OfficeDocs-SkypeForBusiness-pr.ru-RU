# <a name="dialing-out-from-a-meeting-so-other-people-can-join-it"></a>Исходящих звонков из собрания, чтобы другие пользователи могут присоединиться к ней

Организатор собрания можно абонентской группы с помощью Скайп для бизнеса или группами Майкрософт приложения для других пользователей присоединиться к собранию же с помощью телефона. Хотя это не является обязательным, если выполняется набор номера для другого пользователя, рекомендуется абонентской группы полный номер, включая код страны или региона. Чтобы получить подключение по телефонной линии для работы:
  
- Вы можете подключение по телефонной линии только в том случае, если вам присоединиться к собранию с помощью Скайп для бизнеса или группами Майкрософт приложения.
    
- Вы организатор собрания был разрешен доступ к конференции.

 **Шаг 1:** На собрании пригласить, используйте параметр **Пригласить более людей** для исходящих звонков на номер телефона.
  
![Подключение по телефонной линии с помощью Скайп для бизнеса.](../images/9896abec-7a6f-4148-ad09-76a1cf4b56e1.png)
  
 **Шаг 2:** Введите полный телефонного номера, включая код страны или региона в поле.
  
![Номер телефона по телефону в Скайп для бизнеса](../images/084b4fb5-21eb-4f10-9a1a-c92b919084b9.png)
  
## <a name="supported-countries-and-regions"></a>Поддерживаемые стран и регионов

Подключение по телефонной линии доступна только в некоторых странах или регионах. Полный список в разделе [доступность страны и региона для конференц-связи аудио и вызов планов](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
