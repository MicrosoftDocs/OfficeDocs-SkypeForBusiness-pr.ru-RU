# Смена поставщика услуг аудиоконференций пользователя на Майкрософт

Чтобы использовать аудиоконференции Office 365 в Skype для бизнеса и Microsoft Teams, пользователям в вашей организации необходимо назначить лицензию на **аудиоконференции**, а поставщиком услуг аудиоконференций нужно назначить Майкрософт. Дополнительные сведения о лицензиях и их стоимости см. на странице [Платная или пробная версия аудиоконференций в Office 365](try-or-purchase-audio-conferencing-in-office-365.md).
  
## Смена поставщика услуг аудиоконференций пользователя на Майкрософт

Если лицензия на **аудиоконференции** назначается пользователю, у которого нет поставщика услуг аудиоконференций, то в качестве поставщика ему будет автоматически назначена корпорация Майкрософт. Если у пользователя уже есть поставщик услуг аудиоконференций, после назначения пользователю лицензии на **аудиоконференции** нужно будет сменить его поставщика на Майкрософт.
  
Чтобы задать Майкрософт в качестве поставщика услуг аудиоконференций для пользователя, которому назначена лицензия на **аудиоконференции**, но который использует другого поставщика, сделайте следующее:
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Центре администрирования Skype для бизнеса** перейдите в раздел **Аудиоконференции**.
    
4. При появлении баннера, уведомляющего о том, что существуют пользователи с назначенными лицензиями на **аудиоконференции**, у которых поставщиком услуг аудиоконференций является не Майкрософт, пройдите по ссылке **Щелкните здесь, чтобы переместить их**. Если вы не видите баннер, в **Центре администрирования Skype для бизнеса** щелкните **Пользователи** и установите фильтр **Пользователи, готовые к переходу на аудиоконференции**.
    
5. Выберите пользователей, которых вы хотите перевести на другого поставщика, а затем на панели действий щелкните **Изменить**.
    
6. Выберите **Майкрософт** в списке **Имя поставщика**.
    
    > [!NOTE]
    > При смене поставщика услуг аудиоконференций для пользователя на Майкрософт изменятся данные пользователя, связанные с этими услугами. Если вам нужно сохранить предыдущие данные, запишите их, прежде чем изменять для каких-либо пользователей поставщика. 
  
7. Нажмите кнопку **Сохранить**.
    
## Дополнительные сведения

- Выбрав пользователя в списке, вы можете просмотреть его данные для аудиоконференций по умолчанию, за исключением ПИН-кода. При необходимости вы можете сбросить пользовательский ПИН-код аудиоконференций, щелкнув **Сбросить**.
    
- Если необходимо изменить параметры аудиоконференций для пользователя, вы можете выбрать его в списке и щелкнуть **Изменить**, а затем внести изменения на странице **Свойства**.
