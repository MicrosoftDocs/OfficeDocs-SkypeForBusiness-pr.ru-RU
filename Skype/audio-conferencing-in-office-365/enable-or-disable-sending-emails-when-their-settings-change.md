# Включение и отключение отправки сообщения электронной почты при изменении параметров аудиоконференций

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](26ea19d3-e420-4fc1-baa3-2692d17e5e1d.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/26ea19d3-e420-4fc1-baa3-2692d17e5e1d), в качестве справочного материала. 
  
Аудиоконференции по умолчанию отправит электронной почты для пользователей, когда будут активированы для аудиоконференций. Иногда бывает Однако если требуется сократить количество сообщений электронной почты, отправленные в Скайп для групп Майкрософт и бизнес-пользователей и в этом случае можно отключить.
  
При отключении отправки сообщения электронной почты все сообщения электронной почты аудиоконференций не будет отправлено для пользователей, включая сообщения электронной почты для при пользователей включена или отключена аудиоконференций, когда сбросить свой ПИН-код, при идентификатор конференции и конференц-связи по умолчанию телефонный номер изменения.
  
Вот пример сообщение, которое отправляется пользователям, когда будут активированы для конференц-связи звук:
  
![Сообщение о конференц-связи с телефонным подключением](../images/81fe4e09-a346-4469-8cc5-c6d65f739b73.png)
  
## Когда пользователи получают уведомления по электронной почте?

- Существует несколько сообщений электронной почты, отправляемых пользователей в вашей организации после эти функции включены аудиоконференций.
    
  - Когда **Аудиоконференций** назначена лицензия их.
    
  - Когда вы вручную сбросить аудиоконференций пользователя ПИН-код.
    
  - Выполнение сброса идентификатора конференции пользователя вручную.
    
  - При лицензии **Аудиоконференций** удаляется из них.
    
  - Когда поставщику услуг аудиоконференций для пользователя изменяется от корпорации Майкрософт для другого поставщика или **Нет**.
    
  - При изменении поставщику услуг аудиоконференций для пользователя в корпорацию Майкрософт.
    
## Включение и отключение прекратить рассылку пользователям по электронной почте

Чтобы включить или отключить отправку электронных писем пользователям, можно использовать Центр администрирования Skype для бизнеса или Windows PowerShell.
  
 **С помощью Скайп для бизнеса центра администрирования**
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в **Центр администрирования Office 365** > **Skype для бизнеса** и на панели навигации слева щелкните **аудиоконференций**.
    
3. На странице **параметров моста Microsoft** установите или снимите флажок **автоматически отправлять сообщения электронной почты для пользователей при наличии изменений в конфигурацию аудиоконференций**.
    
4. Нажмите кнопку **Сохранить**.
    
    > [!TIP]
    > Можно также отправить по электронной почте пользователю с параметрами аудиоконференций, последовательно выбрав пункты свойства пользователя > **аудиоконференций** > **Отправить сведения о конференции по электронной почте**. > В этом случае в отправленном письме будет указан только идентификатор конференции и телефонный номер для подключения. ПИН-код в этом случае не указывается. > Просмотреть, [Отправка пользователям электронных писем с информацией о конференции с телефонным подключением](send-an-email-to-a-user-with-their-audio-conferencing-information.md). 
  
 **С помощью Windows PowerShell**
  
- Чтобы отключить отправку сообщений электронной почты, выполните следующую команду: 
    
  ```
  Set-CsOnlineDialInConferencingTenantSetting -AutomaticallySendEmailsToUsers $false
  ```

    Чтобы упростить этот командлет, читайте в статье [Набор CsOnlineDialInConferencingTenantSettings](https://go.microsoft.com/fwlink/?LinkId=715757).
    
## Дополнительные сведения

- После отключения автоматического сообщения электронной почты можно вручную активировать Отправка сообщения электронной почты с помощью Скайп для центра администрирования бизнес номером телефона и идентификатор конференции. Тем не менее если вы сделаете это ПИН-код не учитываются. Если вы хотите сбросить аудиоконференций ПИН-код и отправка сообщений электронной почты отключены, необходимо отправить его пользователю другим способом.
    
- По умолчанию будет отправителя сообщения электронной почты из Office 365, но можно изменить адрес электронной почты и отображаемое имя, с помощью Windows PowerShell, а также воспользоваться командлетом [Set-CsOnlineDialInConferencingTenantSettings](https://go.microsoft.com/fwlink/?LinkId=715757) .
    
    > [!NOTE]
    > Если требуется изменить данные адреса электронной почты, убедитесь, что действующие в вашей организации политики входящей электронной почты позволяют получать сообщения с дополнительных адресов. 
  
  - Введите адрес электронной почты в параметр  _SendEmailFromAddress_.
    
  - Введите отображаемое имя в параметр  _SendEmailFromDisplayName_.
    
  - Установите для параметра  _SendEmailOverride_ значение _True_.
    
  -  `Set-CsOnlineDialInConferencingTenantSetting -SendEmailOverride $true -SendEmailFromAddress amos.marble@contoso.com -SendEmailFromDisplayName "Amos Marble"`
    
- Отключить отправку уведомлений по электронной почте можно в Центре администрирования Skype для бизнеса или с помощью Windows PowerShell.
