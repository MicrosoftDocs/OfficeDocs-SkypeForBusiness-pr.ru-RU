# Установка языков автосекретаря для аудиоконференций

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](26d73dda-ab26-4af4-8aec-d17f3479ae50.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/26d73dda-ab26-4af4-8aec-d17f3479ae50), в качестве справочного материала. 
  
Автосекретарь аудиоконференций для Скайп для бизнеса и группами Майкрософт можно приветствия конференц связи абонентов разными языками при присоединении к собранию.
  
Вы можете выбрать один основной и до четырех дополнительных языков. Основной язык будет использоваться в первую очередь, а остальные  в указанном вами порядке. 
  
> [!NOTE]
>  Эти языки можно настроить только для региональных номеров доступа для телефонного подключения.
  
## Установка языков автосекретаря конференц-связи

Это действие могут выполнять только [Роли администраторов в Office 365](https://support.office.com/article/da585eea-f576-4f55-a1e0-87090b6aaa9d) и[Роли администраторов в Office 365](https://support.office.com/article/da585eea-f576-4f55-a1e0-87090b6aaa9d). 
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Скайп для центра администрирования бизнес** на панели навигации слева перейдите к **аудиоконференций** и нажмите кнопку **моста Microsoft**.
    
4. На панели действий выберите аудиоконференций телефонный номер в списке, нажмите кнопку **установить языки**.
    
5. На странице **Выбрать языки** выберите нужный пункт в списке **Основной язык**. При необходимости задайте дополнительные языки с помощью раскрывающихся списков **Дополнительные языки**.
    
    > [!NOTE]
    > Отображаются основной и дополнительные языки, которые поддерживаются. Порядок выбора языков в раскрывающемся списке определяет порядок представления этих языков абонентам. 
  
6. Нажмите кнопку **Сохранить**.
    
## Что еще нужно знать?

- Список поддерживаемых языков для аудиоконференций см [Аудиоконференции поддерживаемых языков](audio-conferencing-supported-languages.md).
    
- Языки можно настроить для выделенных, но не для общей телефонных номеров.
    
- Чтобы просмотреть список стран или регионов, аудиоконференций в Office 365 с помощью Microsoft как поставщик доступен, просмотра [Телефонные номера для аудиоконференции](phone-numbers-for-audio-conferencing.md).
