Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/572ca0c8-b9f0-4948-9409-e12ee0b94aa0), в качестве справочного материала.

> Основной и дополнительный языки могут быть изменены только на выделенный аудиоконференций телефонных номеров.

## Сведения по управлению с помощью Windows PowerShell

- Для экономии времени или автоматизации процесса можно использовать командлет [Get-CsOnlineDialInConferencingLanguagesSupported](https://go.microsoft.com/fwlink/?LinkId=617684) или[Set-CsOnlineDialInConferencingServiceNumber](https://go.microsoft.com/fwlink/?LinkId=617689) .
    
- Windows PowerShell дает возможность управлять пользователями, предоставляя им права на определенные действия. С его помощью вы можете управлять Office 365, используя единый центр администрирования, который упростит выполнение ваших повседневных задач. Для начала работы с Windows PowerShell ознакомьтесь с приведенными ниже разделами.
    
  - [Зачем нужна с помощью Office 365](https://go.microsoft.com/fwlink/?LinkId=525041)
    
  - [Лучшие способы управления Office 365 при помощи Windows PowerShell](https://go.microsoft.com/fwlink/?LinkId=525142)
    
- В Windows PowerShell имеется множество преимуществ в скорости, простоты и производительности по сравнению с только с помощью центра администрирования Office 365, например, когда вы создаете изменения параметров для нескольких пользователей одновременно. Дополнительные сведения о указанные ниже преимущества в следующих разделах:
    
  - [Введение в Windows PowerShell и Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525039)
    
  - [Использование Windows PowerShell для управления Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525453)
    
  - [Использование возможностей Windows PowerShell для выполнения стандартных задач управления средой Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525038)
    
    > [!NOTE]
    > Модуль Windows PowerShell для Skype для бизнеса online позволяет запускать удаленные сеансы Windows PowerShell с подключением к Skype для бизнеса online. Этот модуль для 64-разрядных систем можно загрузить из Центра загрузки Microsoft здесь: [Модуль Windows PowerShell для Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=294688).
