# Начало аудиоконференции по телефону без ПИН-кода

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](d5b1f775-d7ed-4d30-853a-1d49f81e8fde.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/d5b1f775-d7ed-4d30-853a-1d49f81e8fde), в качестве справочного материала. 
  
Это может быть утомительным для пользователей, которые собрание, которые будут храниться в зал ожидания слушать музыку, так как Скайп для бизнеса или Microsoft группам Организатор собрания еще не началось собрания.
  
При подключении организатора к собранию по телефону по умолчанию запрашивается ПИН-код. Можно настроить параметры таким образом, чтобы ПИН-код не запрашивался, и к собранию мог свободно подключиться любой пользователь. Кроме того, можно настроить параметры таким образом, чтобы один пользователь или все пользователи в организации, подключающиеся по телефону, могли начать собрание, не указывая ПИН-код. Чтобы включить или отключить этот параметр для одного пользователя, используйте центр администрирования Skype для бизнеса.
  
Если кто-то начал собрания из Скайп для бизнеса или Microsoft команд приложения ПИН-код не требуется для организатором собрания. ПИН-код только в том случае требуется, когда Организатор собрания объединяет их собрания на телефоне. ПИН-код для собрания будет отправлено пользователю конференц связи при их назначенных лицензий **Аудиоконференций** и включена для аудиоконференций. Просмотреть,[Отправка пользователям электронных писем с информацией о конференции с телефонным подключением](send-an-email-to-a-user-with-their-audio-conferencing-information.md) и[Электронные письма, которые автоматически отправляются пользователям при смене настроек конференц-связи с телефонным подключением](emails-that-are-automatically-sent-to-users-when-their-audio-conferencing-settin.md).
  
## Разрешение и запрет присоединения анонимных абонентов к собранию

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. **Центр администрирования Skype для бизнеса**, на панели навигации слева перейдите к **аудиоконференций** > **Пользователи удаленного доступа**.
    
4. В списке выберите пользователя и на панели действий щелкните **Изменить**.
    
5. На странице свойств пользователя в разделе **Параметры собрания** установите или снимите флажок **Разрешить абонентам, не прошедшим проверку подлинности, быть первыми пользователями на собрании Если этот параметр не будет выбран, пользователям придется ждать в зале ожидания, пока пользователь, прошедший проверку подлинности, присоединится к собранию**.
    
6. Нажмите кнопку **Сохранить**.
    
 **Включение и отключение анонимных абонентов для всех собраний пользователя с помощью Windows Powershell**
  
- Выполните следующее: 
    
  ```
  Set-CsOnlineDialInConferencingTenantSetting -AllowPSTNOnlyMeetingsByDefault $true | $false
  ```

## Дополнительные сведения

- Если вы хотите сбросить ПИН-код, читайте в статье [Сброс ПИН-кода для конференц-связи с телефонным подключением для пользователя](reset-the-audio-conferencing-pin-for-a-user.md).
    
- Если включен анонимный доступ или параметр, при котором для начала собрания не требуется ПИН-код:
    
  - Если собрание еще не началось (нет ни один собрания еще): вызывающий будет предложено, если он Организатор, если он говорит Да, он будет предложено свой ПИН-коде после he входы ПИН-код, начнут собрания и пользователь будет присоединиться к собранию.
    
  - Если собрание уже запущен (другого уже находится на собрании): вызывающий не появится, если он является организатором и он будет никогда не запрашивать ПИН-код, уже начали собрания, присоединится абоненту.
    
- При отключении анонимного доступа или не требуя ПИН-код начать собрание:
    
  - Если собрание еще не началось (нет ни один собрания еще): вызывающий не появится, если он является организатором, и он будет никогда не запрашивать ПИН-код. Так как Организатор установлено значение отключено, начнут собрания и анонимных абонентов будут присоединиться к собранию.
    
  - Если собрание уже запущен (другого уже находится на собрании): вызывающий не появится, если он является организатором и он будет никогда не запрашивать ПИН-код, уже начали собрания, присоединится абоненту.
