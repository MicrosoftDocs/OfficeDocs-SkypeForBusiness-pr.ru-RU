# Телефонные номера для аудиоконференции

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](95a08f84-04e5-4f72-88a8-d6472a7c89d7.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/95a08f84-04e5-4f72-88a8-d6472a7c89d7), в качестве справочного материала. 
  
При настройке **аудиоконференции** в Skype для бизнеса и Microsoft Teams вашей организации автоматически назначаются телефонные номера для подключения. Их можно увидеть в разделе **Центр администрирования Skype для бизнеса** > **Аудиоконференция** > **Мост Microsoft**. См. статью [Просмотр списка номеров для конференц-связи с телефонным подключением](see-a-list-of-audio-conferencing-numbers.md).
  
## Зона покрытия аудиоконференций

Полный список стран, регионов и городов, в которых доступна аудиоконференция, см. в разделе [Доступна ли конференц-связь по ТСОП с телефонными номерами в моей стране или регионе?](https://support.office.com/article/1096d81e-7e14-488c-95d8-b8322e39c059)
  
## Телефонные номера для подключения в приглашении на собрание

Когда пользователь Skype для бизнеса Online или Microsoft Teams планирует собрание в Outlook или Outlook Web App, номер аудиоконференции по умолчанию указывается в приглашении на собрание. Чтобы указать для одного или нескольких пользователей другой номер по умолчанию, измените номер, выбрав в **Центре администрирования Skype для бизнеса** раздел **Аудиоконференция** > **Пользователи**. См. статью [Определение телефонных номеров для аудиоконференций, которые организаторы собраний могут включить в приглашения](set-the-audio-conferencing-phone-numbers-for-meeting-organizers-that-are-include.md).
  
Чтобы просмотреть другие номера для подключения, перейдите по ссылке **Найти местный номер** в приглашении на собрание.
  
## Телефонные номера для подключения, настраиваемые в мосте аудиоконференции

Существует два типа номеров аудиоконференции с телефонным подключением, назначаемых для моста аудиоконференции: **общедоступные** и **выделенные**. Любой абонент может использовать оба типа номеров для подключения к соответствующим аудиособраниям, проводимым в организации.
  
 **Выделенные телефонные номера**  это номера, доступные только для пользователей внутри организации. Вы можете изменить языки, которые используются при звонках на эти номера.
  
 **Общедоступные телефонные номера**  это номера, доступ к которым можно предоставлять другим организациям Office 365. Изменить языки, которые используются при звонках на эти номера, нельзя.
  
Притом что номер аудиоконференции по умолчанию, назначаемый организатору, включается только в приглашение на собрание, для подключения к собранию абонент может использовать любой из телефонных номеров, назначенный мосту конференц-связи. Список номеров, которые можно использовать для подключения к собранию, открывается при переходе по ссылке **Найти местный номер**, которая включена во все приглашения.
  
## Автоматически назначенные номера доступа к аудиоконференции

Общие номера телефонов для аудиоконференции автоматически назначаются организациям при активации соответствующей функции. Полученные номера телефонов назначаются мосту аудиоконференции в качестве номеров по умолчанию. Номер телефона, назначенный по умолчанию мосту, должен соответствовать стране или региону нахождения организации.
  
> [!NOTE]
> Страну или регион организации можно найти (выполнив вход в **Центр администрирования Office 365**) в разделе **Профиль организации**. 
  
> [!CAUTION]
> В связи с ограниченной емкостью пула платных номеров телефонов в Венесуэле, Индонезии и ОАЭ организации из этих стран не получают платный номер для аудиоконференции автоматически. Бесплатные номера доступны для этих стран независимо от существующих ограничений. 
  
Выделенные телефонные номера для аудиоконференции  это номера служб, которые можно получить и назначить своей организации. Номера служб можно найти с помощью **Центра администрирования Skype для бизнеса**. Дополнительные сведения см. в разделе [Получение номеров телефонов служб для Skype для бизнеса](../what-is-phone-system-in-office-365/getting-service-phone-numbers-for-skype-for-business-and-microsoft-teams.md).
  
Список стран и регионов, в которых организациям автоматически назначаются телефонные номера, см. в статье [Страны и региона доступность для аудиоконференций и вызвав планов](../countries-and-region-availability-for-audio-conferencing-and-calling-plans/countries-and-region-availability-for-audio-conferencing-and-calling-plans.md).
  
## Дополнительные сведения

- Со списком поддерживаемых языков для аудиоконференции можно ознакомиться в разделе [Аудиоконференции поддерживаемых языков](audio-conferencing-supported-languages.md).
    
- Можно использовать командлет [Get-CsOnlineDialInConferencingServiceNumber](https://go.microsoft.com/fwlink/?LinkId=617691), чтобы просмотреть выделенные телефонные номера для подключения к аудиоконференции в вашей организации.
    
- Можно использовать командлет [Get-CsOnlineDialInConferencingLanguagesSupported](https://go.microsoft.com/fwlink/?LinkId=617684), чтобы просмотреть список языков, поддерживаемых в выделенных телефонных номерах для конференц-связи с подключением.
    
- Можно задать до 4 языков для каждого номера аудиоконференции: один основной и три дополнительных. Также можно настроить языки для выделенного телефонного номера аудиоконференции.
    
- Сведения о настройке телефонного номера для подключения для отдельного пользователя см. в разделе [Определение телефонных номеров для аудиоконференций, которые организаторы собраний могут включить в приглашения](set-the-audio-conferencing-phone-numbers-for-meeting-organizers-that-are-include.md).
