# Просмотр списка пользователей, активированных для конференц-связи с телефонным подключением

Активировав пользователей в организации для конференц-связи с телефонным подключением, можно будет просмотреть список активированных пользователей. В списке также отображаются сведения о типе поставщика услуг конференц-связи с телефонным подключением, который они используют, номер телефонного подключения по умолчанию и, если в организации не включены динамические идентификаторы, статические идентификаторы конференции для организуемых собраний с телефонным подключением.
  
## Просмотр списка пользователей

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В окне **Центр администрирования Skype для бизнеса** на левой панели навигации выберите **Конференц-связь с телефонным подключением** > и затем **Пользователи с телефонным подключением**.
    
## Дополнительные сведения

- При просмотре списка активированных пользователей можно выбрать в списке пользователя, а затем использовать панель действий, чтобы отредактировать параметры конференц-связи с телефонным подключением для этого пользователя.
    
- Если в качестве поставщика конференц-связи с телефонным подключением вы выберете Майкрософт, вы сможете просмотреть телефонный номер по умолчанию и, если в организации не включены динамические идентификаторы, сбросить идентификатор конференции для организуемых пользователем собраний.
    
- При выборе одного пользователя, настроенного на использование стороннего поставщика услуг конференц-связи с телефонным подключением, можно просмотреть имя поставщика конференц-связи с телефонным подключением, а также платный и бесплатный телефонный номер (если они настроены).
    
- Параметры фильтра позволяют показать пользователей, для которых:
    
  - **конференц-связь с телефонным подключением включена;**
    
  - **конференц-связь с телефонным подключением отключена;**
    
  - **выбран поставщик услуг конференц-связи Microsoft;**
    
  - **выбраны другие поставщики услуг конференц-связи;**
    
- Используя кнопку поиска, можно найти отдельного пользователя в списке.
    
- Выбрав нескольких пользователей, можно:
    
  - выбрать другой номер по умолчанию для этих пользователей;
    
  - отключить конференц-связи с телефонным подключением для пользователя, выбрав в поле поставщика **Нет**;
    
  - Переключитесь на Microsoft в качестве поставщика услуг конференц-связи с телефонным подключением, если пользователю назначена лицензия на **Конференц-связь через ТСОП в Skype для бизнеса**.
    
  - Разрешите или запретите анонимным пользователям активировать собрания для выбранного номера телефона.
