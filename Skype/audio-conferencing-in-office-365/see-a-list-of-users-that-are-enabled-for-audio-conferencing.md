# <a name="see-a-list-of-users-that-are-enabled-for-audio-conferencing"></a>Просмотреть список пользователей, для которых включено для аудиоконференции

После включения Скайп для бизнеса или группами Майкрософт пользователей в вашей организации для аудиоконференции, можно просмотреть список тех пользователей, которые будут включены. При просмотре списка, также будут отображаться для каждого пользователя в списке тип поставщика услуг аудиоконференций, они используется номер телефона-связи по умолчанию для пользователя, и, если ваша организация не включен для идентификаторы динамических конференции, статические конференции идентификаторы для собраний аудиоконференций, которые они организации.
  
## <a name="viewing-a-list-of-users"></a>Просмотр списка пользователей

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Скайп по центру администрирования бизнеса**, в левой области переходов, перейдите к **аудиоконференции** > **пользователей**.
    
## <a name="what-else-should-i-know"></a>Дополнительные сведения

- При просмотре списка пользователей, для которых включено можно выберите пользователя в списке и использовать области действия для изменения параметров аудиоконференций для этого пользователя.
    
- При выборе одного пользователя, который настроен для использования в качестве поставщика аудиоконференций Майкрософт, можно просмотреть номер телефона по умолчанию и включена ли вашей организации для динамического конференции идентификаторы и может сбросить идентификатор конференции для собраний, Упорядочивает пользователя.
    
- Когда вы выбираете один пользователь, который настроен на использование поставщика аудиоконференций сторонних производителей, можно просмотреть имя поставщика аудиоконференций, бесплатный номер телефона и номер бесплатный номер телефона (если они настроены).
    
- Параметры фильтра можно использовать для отображения пользователей, имеющих:
    
  - **На конференции**
    
  - **Аудиоконференции отключено**
    
  - **Поставщик конференц-связи — Microsoft**
    
  - **Поставщик конференц-связи — другим пользователям**
    
- Кнопка поиска можно использовать для поиска для отдельного пользователя в списке.
    
- Можно выбрать несколько пользователей и сделайте следующее:
    
  - Выберите номер по умолчанию для этих пользователей.
    
  - Отключите аудиоконференций для пользователя с помощью изменения поставщика значение **None**.
    
  - Перейдите в корпорацию Майкрософт как поставщик услуг аудиоконференций Если пользователю назначена лицензия на **Аудиоконференций** .
    
  - Разрешить и запретить анонимным пользователям для активации собраний телефона выбранных пользователей.
