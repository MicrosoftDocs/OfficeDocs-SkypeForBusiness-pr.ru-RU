# Сравнение поставщиков аудиоконференций

При настройке аудиоконференции для Skype для бизнеса и Microsoft Teams нужно выбрать поставщика услуг аудиоконференций. Это может быть Майкрософт или сторонний поставщик. 
  
> [!IMPORTANT]
> Пользователи Microsoft Teams не могут использовать стороннего поставщика услуг аудиоконференций. 
  
Поставщики услуг аудиоконференций выполняют следующие действия. 
  
- Предоставление  *моста аудиоконференций*  . Он позволит задать номера телефонов, ПИН-коды и идентификаторы конференций для собраний.
    
- Предоставление платных, бесплатных и международных номеров телефонов, по которым будут звонить пользователи. Эти номера телефонов затем указываются в приглашении на собрание.
    
- Выбор языка для инструкций, которые будут слышать пользователи во время подключения по телефону к собранию.
    
- Создание идентификаторов конференций для собраний, которые используются для присоединения пользователей к собранию.
    
- Предоставление ПИН-кодов организатора для начала собрания по телефону.
    
В следующей таблице сравниваются возможности, которые будут доступны при выборе каждого поставщика услуг конференц-связи с телефонным подключением  Майкрософт или стороннего поставщика.
