# Изменение параметров моста аудиоконференций

При настройке аудиоконференции в Office 365 вы получаете номера телефонов и мост конференций для пользователей. Мост конференций может содержать один или несколько номеров телефонов. Эти номера телефонов используются для подключения к собранию. Они указываются в нижней строке приглашения на собрание Skype для бизнеса или Microsoft Teams.
  
Мост конференций обеспечивает ответ на вызов пользователя, воспроизводя голосовые подсказки автосекретаря, а затем в соответствии с заданными параметрами может воспроизводить уведомления, просить абонентов произнести свое имя и управлять параметрами ПИН-кодов. ПИН-коды предоставляются организаторам собраний и позволяют им начинать собрания, когда они не используют приложение Skype для бизнеса или Microsoft Teams.
  
> [!IMPORTANT]
> ПИН-код требуется только организатору собрания, когда пользователь приложения Skype для бизнеса или Microsoft Teams еще не начал собрание. Если кто-либо звонит, чтобы присоединиться к собранию, организатору собрания необходим ПИН-код, чтобы начать собрание. 
  
## Изменение параметров моста аудиоконференций

 **Настройка параметров проведения собрания, когда абоненты присоединяются к собранию**
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Центр администрирования Skype для бизнеса** на панели навигации слева выберите **Аудиоконференция** > **Настройки моста Microsoft**.
    
4. На странице **Настройки моста Microsoft** в разделе **Присоединение к собранию** выберите следующее:
    
  - **Включить уведомления о входе и выходе из собрания** По умолчанию этот флажок установлен. Если снять флажок, то пользователи, уже присоединившиеся к собранию, не будут получать уведомления о том, что кто-то вошел или покинул собрание.
    
    При использовании значения **Включить уведомления о входе на собрание и выходе из него** можно выбрать следующие параметры из списка **Entry-exist announcement type** (Тип оповещения при входе):
    
  - **Имена или номера телефонов**. Когда пользователи присоединяются к собранию по телефону, их номера телефонов отображаются при входе в собрание.
    
  - **Мелодии**. Когда пользователь присоединяется к собранию по телефону, его мелодия воспроизводится при входе в собрание.
    
    > [!NOTE]
    > Все клиенты, использующие предварительную версию функции, могут сейчас применять **мелодию** в качестве оповещения.
  
  - **Просить абонентов произносить свое имя перед присоединением к собранию** По умолчанию этот флажок установлен. Если снять флажок, система не будет просить абонентов произносить свое имя перед присоединением к собранию.
    
5. Внесите изменения и нажмите кнопку **Сохранить**.
    
 **Установка длины ПИН-кода для собраний**
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Центр администрирования Skype для бизнеса** на панели навигации слева выберите **Аудиоконференция** > **Настройки моста Microsoft**.
    
4. На странице **Настройки моста Microsoft** в разделе **Безопасность** в списке **Длина ПИН-кода** введите количество цифр, которое необходимо использовать для ПИН-кода, затем нажмите **Сохранить**.
    
    > [!IMPORTANT]
    > ПИН-код должен содержать от 4 до 12 цифр. 
  
 **Выберите, следует ли отправлять пользователям сообщение электронной почты.**
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. В **Центр администрирования Skype для бизнеса** на панели навигации слева выберите **Аудиоконференция** > **Настройки моста Microsoft**.
    
4. На странице **Настройки моста Microsoft** установите или снимите флажок **Automatically send emails to users if their audio conferencing configuration changes (Автоматически отправлять электронные письма пользователям в случае изменения конфигурации телефонного подключения по ТСОП)** и нажмите **Сохранить**.
    
    Для получения дополнительных сведений см. статью[Электронные письма, которые автоматически отправляются пользователям при смене настроек конференц-связи с телефонным подключением](emails-that-are-automatically-sent-to-users-when-their-audio-conferencing-settin.md).
    
## Сведения по управлению с помощью Windows PowerShell

- Можно сэкономить время и автоматизировать процесс, используя командлет [Set-CsDialinConferencingBridge](https://go.microsoft.com/fwlink/?LinkId=617686 ).
    
- Windows PowerShell дает возможность управлять пользователями, предоставляя им права на определенные действия. С его помощью вы можете управлять Office 365, используя единый центр администрирования, который позволяет упростить выполнение повседневных задач. Для начала работы с Windows PowerShell ознакомьтесь с приведенными ниже разделами.
    
  - [Причины использования Office 365 PowerShell](https://go.microsoft.com/fwlink/?LinkId=525041)
    
  - [Лучшие способы управления Office 365 при помощи Windows PowerShell](https://go.microsoft.com/fwlink/?LinkId=525142)
    
- Windows PowerShell имеет множество преимуществ в скорости, простоте и эффективности работы по сравнению с использованием только центра администрирования Office 365, например при внесении изменений для множества пользователей одновременно. Подробнее об этих преимуществах можно узнать в следующих разделах:
    
  - [Введение в Windows PowerShell и Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525039)
    
  - [Использование Windows PowerShell для управления Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525453)
    
  - [Использование возможностей Windows PowerShell для выполнения стандартных задач управления средой Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=525038)
    
    > [!NOTE]
    > Модуль Windows PowerShell для Skype для бизнеса online позволяет запускать удаленные сеансы Windows PowerShell с подключением к Skype для бизнеса online. Этот модуль для 64-разрядных систем можно загрузить из Центра загрузки Microsoft здесь: [Модуль Windows PowerShell для Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkId=294688).
