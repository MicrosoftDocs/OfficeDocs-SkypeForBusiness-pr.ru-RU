# Начало новой формы запроса номера телефона пользователя

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Иногда (в зависимости от страны) вы не сможете получить ваш новый числа, используя **Скайп для центра администрирования бизнес** или вам потребуется код города или номера телефона. В этом случае будет необходимо загрузить формы и отправьте его снова us. Наш службы технической поддержки будет брать их оттуда и сообщить о ходе выполнения или оформление заказа.
  
> [!CAUTION]
> В **Бельгия, Ирландия, Германия и Нидерланды** не удается получить телефона с помощью **Скайп для центра администрирования бизнес** новых номеров телефонов, необходимо загрузить и заполните форму, а затем отправить нам.
  
При управлении номерами телефонов важно помнить, что при связывании номеров в Skype для бизнеса Online могут использоваться [Different kinds of phone numbers used for Calling Plans](different-kinds-of-phone-numbers-used-for-calling-plans.md). Дополнительные сведения о выполнении задач такого рода см. в разделе [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).

## Скачайте формы заявок для управления номерами телефонов

- Чтобы просмотреть полный список доверенностей и скачать соответствующие формы, перейдите на страницу [Доверенности для Skype для бизнеса Online](https://go.microsoft.com/fwlink/?LinkID=623745).
    
- Чтобы просмотреть список форм заявок на получение новых номеров телефонов и скачать их, перейдите на страницу [Заявки на получение новых номеров телефонов для Skype для бизнеса](https://go.microsoft.com/fwlink/?linkid=851581).
