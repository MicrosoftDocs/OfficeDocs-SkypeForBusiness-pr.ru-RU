# Просмотр списка телефонных номеров в организации

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](93098bc5-df63-4a1f-8734-0b72a6280a69.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/93098bc5-df63-4a1f-8734-0b72a6280a69), в качестве справочного материала. 
  
Существуют различные типы номеров телефонов, вы можете назначать пользователям или других служб (службы номера), такие как для аудиоконференций в Office 365.
  
## Просмотр списка всех телефонных номеров, имеющихся в организации

1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации **Центр администрирования Skype для бизнеса**слева перейдите в раздел **Голосовая связь** > **Телефонные номера**.
    
    **ВАЖНО!** Чтобы в левой области навигации Центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. Для просмотра номеров телефонов, назначенных отображается **номер** столбца.
    
5. Раскрывающийся список можно использовать для фильтрации представления по:
    
  - **Все** телефонные номера, включая служебные.
    
  - Телефон **диапазон чисел**, который вы задали.
    
  - Номера телефонов, **назначенные пользователю**.
    
  - Телефон номера, **назначенные конференции** конференц-связь.
    
  - Предусмотрены телефонные номера **неназначенные**.
    
  - **Расположение** номера телефонов.
    
## Просмотр всех номеров телефонов, назначенных пользователям

При настройке пользователей может потребоваться просмотреть список телефонных номеров, уже назначенных пользователям, а также номеров, которые можно назначить.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации **Центр администрирования Skype для бизнеса**слева перейдите в раздел **Голосовая связь** > **Пользователи голосовой связи**.
    
    **ВАЖНО!** Чтобы в левой области навигации Центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. Можно отфильтровать представления для номеров телефонов, назначенных пользователям или номеров телефонов, можно назначить пользователю. Можно выполнить фильтрацию по:
    
  - **Все** телефонные номера, включая служебные.
    
  - **Номер назначен** пользователю.
    
  - **Номер назначен** пользователю.
    
## Просмотр номеров телефонов, назначенных пользователям голосовой связи

При настройке работы с телефонными звонками для пользователей в организации сначала нужно получить телефонные номера, а затем назначить их пользователям. После получения телефонных номеров может потребоваться просмотреть список пользователей, с назначенными телефонными номерами и без телефонных номеров, а также узнать, работает ли пользователь с версией Online или локальной.
  
1. Войдите в Office 365 под своей учебной или рабочей учетной записью.
    
2. Перейдите в раздел **Центр администрирования Office 365** > **Skype для бизнеса**.
    
3. На панели навигации **Центр администрирования Skype для бизнеса**слева перейдите в раздел **Голосовая связь** > **Пользователи голосовой связи**.
    
    **ВАЖНО!** Чтобы в левой области навигации Центра администрирования Skype для бизнеса появился элемент **Голосовая связь**, необходимо приобрести хотя бы одну **лицензию уровня корпоративный E5**, лицензию на надстройку **Телефонная система** либо лицензию на надстройку **Аудиоконференция**.
    
4. С помощью раскрывающегося списка для фильтрации представления для активные пользователи, является ли пользователь Online и локальной, или состояние и тип лицензирования для пользователя. Можно выполнить фильтрацию по:
    
  - **Активные пользователи** плюс тип пользователя: Online или локальный.
    
  - **Ожидающие лицензии** плюс тип пользователя: Online или локальный.
