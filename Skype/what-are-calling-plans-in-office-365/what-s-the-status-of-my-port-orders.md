# Каково состояние моих заказов на перенос номеров?

Чтобы просмотреть состояние заказа на перенос номеров, выберите **Центр администрирования Skype для бизнеса** > **Голосовая связь** > **Заказы на перенос**. Состояние каждого заказа на перенос указывается в столбце **Состояние**. Если вам потребуется помощь, [Обращение в службу поддержки Office 365 для бизнеса: справка для администраторов](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
В этой таблице показано состояние ваших заказов и, если необходимо, приводятся сведения о требуемых действиях.
  
|**Status**|**Можно ли просмотреть заказ?**|**Можно ли редактировать заказ?**|**Можно ли отменить заказ?**|**Можно ли удалить заказ?**|**Описание**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Обработка** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ создан администратором, но не получен корпорацией Майкрософт.  <br/> |
|**Установление связи с оператором** <br/> |Да  <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Заказ принят и утвержден корпорацией Майкрософт; ведется работа по его утверждению текущим оператором.  <br/> |
|**Перенос утвержден.** <br/> |Да  <br/> |Дата выполнения заказа  <br/> |Да  <br/> |Нет  <br/> |Заказ принят текущим оператором; установлена дата выполнения заказа.  <br/> |
|**Ожидается перенос.** <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Перенос будет выполнен в течение 24 часов; заказ недоступен для редактирования или отмены.  <br/> |
|**Ошибка** <br/> |Нет  <br/> |Да  <br/> |Да  <br/> |Да. На этот момент вы можете удалить заказ на перенос номеров при наличии ошибок. В этом случае вам нужно создать новый заказ на перенос или [Обращение в службу поддержки Office 365 для бизнеса: справка для администраторов](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).  <br/> |Текущий оператор отклонил заказ.  <br/> |
|**Завершено**. <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Нет  <br/> |Номера успешно переданы.  <br/> |
|**Отменено** <br/> |Нет  <br/> |Да  <br/> |Нет  <br/> |Нет  <br/> |Заказ отменен администратором.  <br/> |
   
 **Подробные пошаговые инструкции см. в разделе [Передача номеров телефонов в Office 365](transfer-phone-numbers-to-office-365.md).**
