# <a name="what-are-emergency-locations-addresses-and-call-routing"></a>Что такое местоположения для экстренного реагирования, адреса для экстренного реагирования и маршрутизация экстренных вызовов?

При настройке вызов планы в Office 365, это необходимо быть, что экстренных адрес назначен каждый номер телефона, получив один номер телефона или при его назначенных пользователю для поддержки экстренные вызовы. Перед назначением экстренных адрес на номер телефона, адрес экстренных необходимо создаются и проверяются. Проверка гарантирует, что экстренных адрес распознается, и что он находится в правильном формате, который может использоваться службами действия в чрезвычайных ситуациях. При необходимости можно добавить места в пределах экстренных адрес для предоставления более заданного расположения для пользователя. К примеру экстренных расположение может быть этажа, уинг или office, связанного с определенным экстренных адрес. Несмотря на то, что проверяются экстренных адреса, не расположения.
  
## <a name="what-are-emergency-addresses"></a>Что такое адреса для экстренной связи?

Экстренные адрес необходим для активной телефонных номеров, но если это необходимо, зависит от страны или региона. В Соединенных Штатах Америки экстренных адрес является **обязательным** , если номер назначается пользователю. Для других странах таких как Европы, Ближнего Востока и АФРИКИ, экстренных адрес является **обязательным** при получении номер телефона с Office 365 или при передаче из другого поставщика услуг или поставщика.
  
Экстренные адрес может рассматриваться, как административный адрес, улица, дом или физических адресов. Это название улицы или административный адрес размещения бизнеса для вашей организации. Например адрес *12345 Северной Main улицы, Редмонд, штат Вашингтон 98052* используется для маршрутизации экстренных вызовов в соответствующие диспетчера сертификации и для помощи при поиске экстренных вызывающего абонента. Оно скорее всего, понадобится несколько адресов аварийного, если более одного физического абонентом места.
  
Проверка экстренных адрес включает в себя убедившись, что оно допустимым и правильно отформатированные для действия в чрезвычайных ситуациях служб. Можно создавать и сохранять экстренных адрес, не проверены, но только проверенный адреса может быть сопоставлен с пользователем. После проверки и сохранения экстренных адрес может быть назначен пользователю. Если необходимо изменить сохраненный проверенный экстренных адрес, необходимо создать новую.
  
## <a name="what-are-emergency-locations"></a>Что такое местоположения для экстренного реагирования?

Местоположении для экстренных служб связаны с экстренных адрес для предоставления более точное расположение внутри здания. Экстренные расположение обычно имеет этажа, уинг построения или адрес офиса где находится пользователь. Может иметь любое количество расположений, связанный с адресом экстренных. 
  
При назначении экстренных адрес пользователя, является фактически расположение идентификатор, указанный при назначении адрес. Идентификатор location включает указанный экстренных адрес (улица или административный адрес). Экстренные расположение по умолчанию входит в состав экстренных адрес в случае, когда расположения в построение не требуется. 
  
## <a name="what-is-emergency-call-routing"></a>Что такое маршрутизация экстренных вызовов?

Экстренные адресов и расположения используются во время звонка в соответствующие диспетчеризации центр при подготовке аварийного быстрого реагирования. Когда Скайп для бизнес-пользователь набирает аварийного номера, как звонок перенаправляется в обслуживанием общедоступных точка ответы на безопасности (PSAP) будут зависеть от страны или региона. В некоторых странах, таких как Соединенные Штаты и Соединенное Королевство вызовы будут защищены сначала определить текущее расположение пользователя перед подключением вызова центр соответствующие отправки. В других странах или регионах звонки будут перенаправляться непосредственно к центру диспетчеризации обслуживания номер телефона, связанного с экстренных вызывающего абонента.
  
## <a name="creating-adding-and-assigning-emergency-locations-and-addresses-to-your-users"></a>Создание, добавление и назначение местоположении для экстренных служб и адреса пользователей

1. **Планирование местоположении для экстренных служб** Первым этапом является планирование для экстренных расположений. Вам потребуется список всех физических адресов. Затем, на основе этого определения ли необходимые расположения для экстренных адресов и, если это так, что они являются. Например если компания 3 зданий каждого с 4 этажей, скорее всего, что необходимо предусмотреть 3 экстренных адреса с этажами 1 – 4 для каждого из них в списке расположение.
    
2. **Создание и проверка экстренных расположения** Следующим шагом является создание и проверка экстренных адресов. При создании экстренных адреса, можно проверить ее. Для создания экстренных адрес см [экстренных адресов для вашей организации](add-or-remove-an-emergency-address-for-your-organization.md).
    
    > [!IMPORTANT]
    > Проверка улицы или административный адрес включает в себя убедившись, что оно допустимым и правильно отформатированные. Существует возможность, что по-прежнему частично правильный адрес emergency, например неверный название города, может пройти проверку прохода. В процессе проверки все части адреса используется для определения, если он содержит достаточно сведений для маршрутизации вызовов центр соответствующие экстренных диспетчера. Если это так, будет возвращен, как проверить и затем может быть назначен номер телефона. 
  
3. **Получение телефонных номеров** Следующим шагом является получение номера телефонов для пользователей. Это можно сделать с получение нового номера телефонов с Office 365 или «перенос» или передачи номеров телефонов на Office 365. Полное описание см [передачи телефонных номеров в Office 365](transfer-phone-numbers-to-office-365.md).
    
4. **Назначение телефонных номеров** Последним шагом является позволяют пользователям выполнение и прием телефонных звонков. Для этого необходимо назначить номер телефона для каждого пользователя. Просмотрите [Назначение, изменение или удаление номер телефона для пользователя,](assign-change-or-remove-a-phone-number-for-a-user.md) Чтобы назначить номер телефона.

> [!NOTE]
> Если вам потребуется получить дополнительные номера телефонов, чем это, пожалуйста, [обратитесь в службу поддержки продуктов бизнес - Admin справки](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
