# <a name="manage-phone-numbers-for-your-organization"></a>Управление номера телефонов для вашей организации

Каждой страны или региона, имеет различные инструкции для получения номера телефонов для пользователей в Скайп для бизнеса в Интернет. Щелкните ссылку для страны или региона, ниже, чтобы получить информацию, необходимую.
  
> [!NOTE]
> Если вызов планы доступны в вашей стране или регионе, см [странах и область доступности для конференц-связи аудио и вызов планов](../../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)
  
Чтобы выбрать тип номера телефона необходимо прочитать [различные виды телефонных номеров, используемый для вызова планов](../../what-are-calling-plans-in-office-365/different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
Дополнительные сведения о телефонных номеров, прочитайте [Приступая к номера телефонов для пользователей](../../what-are-calling-plans-in-office-365/getting-phone-numbers-for-your-users.md).
  
Если вам требуется помощь при настройке номера телефонов для вашей организации, можно [обращение в службу поддержки продуктов бизнес - Admin справки](https://support.office.com/en-us/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
  
## <a name="service-numbers"></a>Служба чисел

Для номеров службы Если указана страна или регион, но в **Скайп по центру администрирования Business**недоступны номеров службы, отправьте запрос нового номера телефона.
  
- Для службы номера в США Пожалуйста, [Загрузите](http://download.microsoft.com/download/6/A/7/6A7CFADC-30FD-4033-BCD6-1E027DAD2336/New Phone Number Request for the U.S. %28Service numbers%29 (v.1) (en.US).pdf) и заполнять формы затем [Отправить его в корпорацию Майкрософт](mailto:ptn@microsoft.com).
    
- Для номеров службы за пределами США Пожалуйста, [Загрузите](http://download.microsoft.com/download/6/A/7/6A7CFADC-30FD-4033-BCD6-1E027DAD2336/New Phone Number Request for all countries except the U.S. %28Service numbers%29 (v.1) (en.US).pdf) и заполнять формы затем [Отправить его в корпорацию Майкрософт](mailto:ptneu@microsoft.com).
