# Управление номер телефона для Великобритании

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Используйте эту таблицу для поиска информации на начало и управление ими в Великобритании телефонных номеров для Скайп для бизнеса Online.
  
Подробнее читайте [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
|**Необходимые действия**|**Скачайте нужные формы**|
|:-----|:-----|
|Необходимо получить номера телефонов пользователей, которые не перечислены в центре администрирования Skype для бизнеса.  <br/> |Загрузите [новые номера телефонов для Великобритании (географических номера)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for the U.K. %28Subscriber geographic numbers%29 (v.3) (en.US).pdf) форму, необходимо ввести ее и[отправьте его нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо получить номера телефонов, перенесенные к нам.  <br/> | Скачайте одну из следующих форм. <br/> [Доверенность для Соединенного Королевства (абонентские номера телефонов)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for the U.K. (Subscriber numbers) (v.2.0) (en-US).pdf) <br/> [Доверенность для Соединенного Королевства (служебные номера телефонов)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for the U.K. (Service numbers) (v.2.1) (en-US).pdf) <br/>  Заполните форму и[отправьте ее нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо опубликовать номера телефонов или отменить их публикацию в национальном телефонном справочнике.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com) список номеров телефонов, для которых необходима публикация или отмена публикации. <br/> |
|Необходимо сменить номер телефона пользователя на служебный номер.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com ) список номеров телефонов пользователей, которые необходимо преобразовать в служебные номера. <br/> |
