# Управление номер телефона для Германии

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Используйте эту таблицу для поиска информации на начало и управление ими в Германии телефонных номеров для Скайп для бизнеса Online.
  
Подробнее читайте [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
|**Необходимые действия**|**Скачайте нужные формы**|
|:-----|:-----|
|Получение номеров телефонов пользователей.  <br/> | В Германии не удается получить телефонных номеров для организации с помощью Скайп для центра администрирования бизнес. Необходимо отправить запрос новый номер телефона. Загрузите одну из этих форм: <br/> [Получение новых номеров телефонов в Германии (en US)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Germany %28Geographic numbers%29 (v.2) (en.US).pdf) <br/> [Получение новых номеров телефонов в Германии (ru ru)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Germany %28Geographic numbers%29 (v.3) (de.DE).pdf) <br/>  Заполните форму и[отправьте ее нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо получить номера телефонов, перенесенные к нам.  <br/> | Скачайте одну из следующих форм. <br/> [Письма из авторизации (ЗАГРУ) для Германии (географических номера) (v.1.0) (en US)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Germany (Geographic numbers) (v.1.0) (en-US).pdf) <br/> [Письма из авторизации (ЗАГРУ) для Германии (географических номера) (v.1.0) (ru ru)](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Germany (Geographic numbers) (v.1.0) (de-DE).pdf) <br/>  Заполните форму и[отправьте ее нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо опубликовать номера телефонов или отменить их публикацию в национальном телефонном справочнике.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com) список номеров телефонов, для которых необходима публикация или отмена публикации. <br/> |
|Необходимо сменить номер телефона пользователя на служебный номер.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com ) список номеров телефонов пользователей, которые необходимо преобразовать в служебные номера. <br/> |
