# Управление телефонных номеров для Испания

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](e18d2274-cd69-47f6-9a65-1adc59dc5399.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/e18d2274-cd69-47f6-9a65-1adc59dc5399), в качестве справочного материала. 
  
Используйте эту таблицу для поиска информации на начало и управление ими в Испания телефонных номеров для Скайп для бизнеса Online.
  
Подробнее читайте [Управление номерами телефонов для организации](manage-phone-numbers-for-your-organization.md).
  
|**Необходимые действия**|**Скачайте нужные формы**|
|:-----|:-----|
|Необходимо получить номера телефонов пользователей, которые не перечислены в центре администрирования Skype для бизнеса.  <br/> | Скачайте одну из следующих форм. <br/> [Получение новых номеров телефонов в Испания (en US)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Spain %28Geographic numbers%29 (v.2) (en.US).pdf) <br/> [Получение новых номеров телефонов в Испания (Испанский)](http://download.microsoft.com/download/3/B/D/3BDD4575-EAFA-4777-B4C6-A42E8F235AC9/New Phone Number Request for Spain %28Geographic numbers%29 (v.3) (es-ES).pdf) <br/>  Заполните форму и[отправьте ее нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо получить номера телефонов, перенесенные к нам.  <br/> |Скачайте форму [Доверенность для Испании](http://download.microsoft.com/download/F/0/1/F01AE714-0F3C-4D9D-B41A-DFD180EC1622/Letter of Authorization %28LOA%29 for Spain (All numbers) (v.1.0) (es-ES).pdf), заполните и [отправьте ее нам](mailto:ptneu@microsoft.com).  <br/> |
|Необходимо опубликовать номера телефонов или отменить их публикацию в национальном телефонном справочнике.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com) список номеров телефонов, для которых необходима публикация или отмена публикации. <br/> |
|Необходимо сменить номер телефона пользователя на служебный номер.  <br/> |[Отправьте нам](mailto:ptneu@microsoft.com ) список номеров телефонов пользователей, которые необходимо преобразовать в служебные номера. <br/> |
