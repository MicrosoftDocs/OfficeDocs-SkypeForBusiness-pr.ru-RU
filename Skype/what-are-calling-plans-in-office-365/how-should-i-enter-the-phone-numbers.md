# Как правильно вводить телефонные номера?

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
При переносе номера телефонов, необходимо ввести их в нужном формате.
  
> [!NOTE]
> Каждый номер или диапазон номеров необходимо указывать в новой строке. 
  
- При вводе телефонных номеров по одному
    
  - Все специальные символы будут игнорироваться (включая тире "-"). Пример
    
  - Для 10-значных номеров: **&amp;*(425*()(*&amp;4&amp;*())(*250649** будет преобразовано в **+14255550649**.
    
  - Для 11-значных номеров: **&amp;*(1*()(*&amp;42&amp;*())(*55550649** будет преобразовано в **+14255550649**.
    
  - При наличии 10 или 11 цифр все теги будут игнорироваться. Например, **<div> 4255551234</div>** будет преобразовано в **+14255551234**.
    
  - Тире "-", пробелы и скобки будут игнорироваться. Пример
    
  - Для 10-значных номеров: **(425) 555-6776** будет преобразовано в **+14255556776**.
    
  - Для 11-значных номеров: **1(425) 555-6776** будет преобразовано в **+14255556776**.
    
  - Все буквы будет рассматриваться как специальные символы и обрабатывается, если есть 10- или цифр 11 телефонный номер. Например:
    
  - Для 10-значных номеров: **14jaosia2reoij05jof55506ajfoj49isdjf** будет преобразовано в **+14255550649**.
    
  - Для 11-значных номеров: **1ade4jaoda2rfoij05ojof55506dsfoj49if** будет преобразовано в **+14255550649**.
    
  - Любые сочетания специальных символов, даже на других языках, будут преобразованы. Пример 
    
  - Для 10-значных номеров: **中文4中文2ajj5*（）（*（5()...551345** будет преобразовано в **+14555551345**.
    
  - Для 11-значных номеров: **中文4中文2$a5*（）（*（5()...55(.1345** будет преобразовано в **+14555551345**.
    
  - Если все цифры содержат менее 10 или более 11 цифрами, они будут выделены пользователя для исправления:
    
  - ** 5551245** Номер будет выделен, и его потребуется исправить.
    
  - Номер **1234567891011** будет выделен, и его потребуется исправить.
    
  - Будут выделены все числа, которые меньше 10 или более 11 цифрами с специальные символы, не будет исправлена автоматически.
    
  - Для 7-значного числа без специальных знаков, введенное: **123456abcdefg7** будут выделены и нужно исправить, но не учитываются буквы.
    
  - Для 7-значного числа с специальные символы, введенное: **12345!@#$%^ &amp; * (--) @# $% ^ &amp; * (7)** будет выделено, чтобы исправить. Специальные знаки не учитываются.
    
- При вводе диапазона телефонных номеров
    
  - Допускается ввод только двух номеров. Меньший номер должен быть первым номером в диапазоне.
    
  - Все специальные знаки (за исключением тире «-»), обрабатывается так же, как один числа. Например **(425) 555 0 &amp; * (123-(1425) 5557899nm** будет исправлено на **+14255550123 - +13202040659**.
    
  - «-» Используется для разделения только двух чисел. Они не поддерживаются для включения нескольких «-» в диапазон чисел. Например:, **(425) 555-0649-(425) 555-1115** следует вводить в виде **(425) 5550649 - (425) 5551115**.
    
 **Для получения подробных пошаговых инструкций читайте в статье [Передача номеров телефонов в Office 365](transfer-phone-numbers-to-office-365.md).**
