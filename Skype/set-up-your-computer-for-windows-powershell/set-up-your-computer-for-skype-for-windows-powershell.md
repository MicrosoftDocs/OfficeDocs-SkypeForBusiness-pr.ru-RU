# Настройка компьютера для Скайп для бизнеса Online управление с помощью Windows PowerShell

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Удаленное управление Skype для бизнеса online с помощью Microsoft PowerShell поддерживается только в 64-разрядных компьютерах под управлением одной из следующих операционных систем:
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Помимо поддерживаемый операционной системы компьютер также должен быть запущен следующее:
  
- PowerShell
    
- Модуль соединителя Skype для бизнеса Online
    
Вы найдете сведения о загрузке и установке эти элементы в следующих разделах.
  
- [Скачайте и установите модуль Skype для бизнеса Online Connector](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [Скачайте и установите Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
