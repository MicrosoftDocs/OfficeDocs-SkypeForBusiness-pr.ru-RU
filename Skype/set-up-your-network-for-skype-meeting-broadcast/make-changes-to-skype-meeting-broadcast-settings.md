# Внесение изменений в параметры трансляции собраний Skype для организации

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Вы можете включить трансляцию собраний Skype и внести изменения в параметры и политики для этих собраний.
  
- **Включение трансляции собраний Skype** Включает Трансляцию собраний Skype. После того, как вы включите Трансляцию собраний Skype, нужно[Настройка сети для трансляции собрания Skype](set-up-your-network-for-skype-meeting-broadcast.md). Это необходимо, если вы планируете проводить вебинары и другие трансляции для людей, не работающих в вашей организации. 
    
- **Разрешить просмотр трансляция собрания Скайп возможности для моей организации** Скайп бизнес-приложениями клиента предоставляют раннего доступа для новых продуктов и компонентов. Ваша организация предоставляет краткий возможности для тестирования новых функций в вашей среде и отзывы отпустите сборки продукта для широкой общественности и что теперь.[Скайп для предварительного просмотра бизнеса](https://www.skypepreview.com/)
    
- **Разрешить организаторов запланировать анонимного собрания** Это позволит организаторов выполните субтитры и перевода для трансляции собраний.
    
- **Разрешить транслировать собрания будут записываться** Это позволит вам нужно быть зарегистрировано выступающих и организатора собрания.
    
- **Служба поддержки URL-адрес для участников собрания** Введите ссылку для трансляции участникам собрания с помощью необходима помощь в подключении или посещения трансляции собрания.
