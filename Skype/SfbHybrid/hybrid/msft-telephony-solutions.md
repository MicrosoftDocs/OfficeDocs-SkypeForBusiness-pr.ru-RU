# <a name="microsoft-telephony-solutions"></a>Решения Майкрософт для телефонии

Корпорация Майкрософт поддерживает несколько вариантов при начале работы с Teams в облаке Майкрософт. Эта статья поможет вам решить, какое решение Майкрософт для телефонии (телефонная система в облаке или Корпоративная голосовая связь локально) будет правильным для пользователей в вашей организации и как ваша организация может подключиться к телефонной сети общего звонков (PSTN).

Эту статью следует использовать вместе со связанной технической схемой, которая предоставляет визуальную помощь при принятии правильного решения для вашей организации:

- [Решения Майкрософт для телефонии — PDF](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/telephony-solutions/microsoft-telephony-solutions-12-18.pdf)

- [Решения Майкрософт для телефонии — Visio](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/telephony-solutions/microsoft-telephony-solutions-12-18.vsdx)




## <a name="private-branch-exchange-pbx-options"></a>Параметры PBX

### <a name="phone-system-microsoft-365-or-office-365"></a>Телефонная система (Microsoft 365 или Office 365)
  
Телефонная система — это технология Корпорации Майкрософт, позволяющая управлять звонками и УАПС в облаке Microsoft 365 или Office 365 с помощью Microsoft Teams и/или Skype для бизнеса Online.

Телефонная система работает с клиентами Teams или Skype для бизнеса Online и сертифицированными устройствами. Телефонная система позволяет заменить существующую систему УАКСИ набором функций, непосредственно поставляемой из Microsoft 365 или Office 365 и тесно интегрированной в облачную производительность компании. Чтобы подключить телефонную систему к телефонной сети общего звонков (PSTN), можно выбрать план звонков Майкрософт или собственного оператора телефонии.

Дополнительные сведения см. в подсистеме "Что такое [телефонная система" в Microsoft 365 или Office 365.](https://docs.microsoft.com/MicrosoftTeams/what-is-phone-system-in-office-365)

### <a name="enterprise-voice-skype-for-business-server"></a>Корпоративная голосовая связь (Skype для бизнеса Server)

Корпоративная голосовая связь это технология Корпорации Майкрософт, позволяющая управлять звонками и возможностями PBX в локальной службе Skype для бизнеса Server. Этот параметр Skype для бизнеса можно подключать только к телефонной сети общего звонков с помощью собственного оператора телефонии.

Дополнительные сведения см. в [Корпоративная голосовая связь Skype для бизнеса Server.](https://docs.microsoft.com/SkypeForBusiness/plan-your-deployment/enterprise-voice-solution/enterprise-voice?toc=/SkypeForBusiness/toc.json&bc=/SkypeForBusiness/breadcrumb/toc.json)

## <a name="connection-to-the-public-switched-telephone-network-pstn-options"></a>Параметры подключения к телефонной сети общего параметров (PSTN)

Подключение к телефонной сети общего ского слева (PSTN) можно выбрать с помощью:

- Использование плана звонков Майкрософт в Microsoft 365 или Office 365 
- Подключение собственного оператора телефонии

### <a name="calling-plan-microsoft-365-or-office-365"></a>План звонков (Microsoft 365 или Office 365)

Этот параметр подключает телефонную систему Microsoft 365 или Office 365 к телефонной сети общего звонков (PSTN), чтобы включить звонки на стационарные и мобильные телефоны по всему миру. С помощью плана звонков корпорация Майкрософт является вашим оператором ЗВОНКОВ.

Дополнительные сведения см. в планах [звонков для Microsoft 365 или Office 365.](https://docs.microsoft.com/MicrosoftTeams/calling-plans-for-office-365)

### <a name="connect-your-own-telephony-carrier-microsoft-365-or-office-365-and-skype-for-business-on-premises"></a>Подключение собственного оператора телефонии (Microsoft 365 или Office 365 и локальное подключение Skype для бизнеса)

Этот параметр подключает телефонную систему в Microsoft 365 или Office 365 или Корпоративная голосовая связь в локальной системе Skype для бизнеса к телефонной сети. Для этого параметра требуется поддерживаемый пограничный контроллер сеансов (SBC). В некоторых случаях для этого варианта может потребоваться дополнительное программное обеспечение Майкрософт, развернутого локально.

## <a name="which-solution-is-right-for-your-organization"></a>Какое решение правильно для вашей организации?

Вы можете выбрать облачное решение, решение connect-your-own-carrier или сочетание между всеми облачными и сторонними операторами:

- Телефонная система с планом звонков (все в облаке)

- Телефонная система с собственным оператором через прямую маршрутику

- Телефонная система с собственным оператором через Skype для бизнеса Server ИЛИ Cloud Connector Edition

- Корпоративная голосовая связь Skype для бизнеса Server с собственным оператором

Выбор решения зависит от текущих и будущих потребностей, таких как:

- Требуется ли сохранить функции, предоставляемые в локальном развертывании, или обязательно.
- Какой клиент вы хотите развернуть для пользователей.
- Какой план вы планируете для перемещения людей в облако.
- Необходимо ли вам работать со сторонними УАО и другим телефонным оборудованием.

Чтобы определить наилучшее решение для организации, рассмотрите следующие вопросы:

- У вас есть существующее развертывание Skype для бизнеса Server?
- Ваши пользователи находятся в локальной skype для бизнеса, в облаке Skype для бизнеса Online или и то, и другое? 
- Вы хотите перенести пользователей из локальной сети в облако?
- Доступен ли план звонков по STN корпорации Майкрософт в вашем регионе?
- Хотите ли вы сохранить текущего оператора телефонии?  Например, нужно ли сохранить текущего оператора связи из-за существующего контракта?
- У вас есть существующая существующая устаревшая УАПС, которую вы хотите или хотите сохранить?
- Предлагает ли текущая устаревшая УАП уникальные функции, критически важные для вашего бизнеса?
- Требуются ли какие-либо функции или все пользователи, которые в настоящее время не предлагаются в телефонной системе?

Обратите внимание на следующее:

- Все четыре варианта могут сосуществовать друг с другом на случай, если потребуется разработать решение для сложной среды или управлять многоэтапной миграцией.
- Телефонную систему с собственным оператором через Skype для бизнеса Server ИЛИ Cloud Connector Edition можно развернуть только с помощью Skype для бизнеса Server или Cloud Connector. Сосуществование Skype для бизнеса Server и Cloud Connector не поддерживается в одной компании.

## <a name="phone-system-with-calling-plan"></a>Телефонная система с планом звонков


Телефонная система с планом звонков — это облачный вариант для пользователей Teams или Skype для бизнеса Online, как показано на следующей схеме:

![Телефонная система с планом звонков](../../sfbserver2019/media/msft-telephony-solutions-1.png)


- Телефонная система Майкрософт с добавленными планами внутренних или международных звонков, позволяющими звонить на телефоны по всему миру (в зависимости от уровня лицензии). 
- Так как план звонков по STN работает за счет Microsoft 365 или Office 365, для этого варианта не требуется развертывание или обслуживание локального развертывания.
- Клиенты могут подключить поддерживаемый SBC через прямую маршрутику для обеспечения связи со сторонними УАЦ, аналоговыми устройствами и другим сторонним телефонным оборудованием, поддерживаемым SBC.

| Требования к инфраструктуре                   | Обязательный?|
| :----------------------------------------------------| ---------:|
| Требуется непрерывное подключение к Microsoft 365 или Office 365 | Да |
| Доступно по всему миру*                            | Нет  |
| Требуется развертывание и обслуживание поддерживаемых пограничных контроллеров сеансов (SBC) | Нет |
| Требуется контракт со сторонним оператором      | Нет   |
| Требуется развертывание и обслуживание Skype для бизнеса Server или Cloud Connector Edition | Нет |

\*Дополнительные сведения о странах, в которых доступен план звонков, см. в сведениях о доступности аудиоконференций и планов звонков в странах [и регионах.](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans)


Если вы ответили "да" на следующие вопросы, то это решение вам будет правильным:

- План звонков доступен в вашем регионе.
- Вам не нужно сохранять текущего оператора STN.
- Вы хотите использовать управляемый Майкрософт доступ к телефонной сети общего пользования (PSTN).
- Управлять пограничными контроллерами сеансов самостоятельно не нужно.
- В Teams и/или Skype для бизнеса Online есть все функции, необходимые вашей организации.

Дополнительные сведения см. в подсистеме "Что такое телефонная система" в [Microsoft 365 и Office 365,](https://docs.microsoft.com/MicrosoftTeams/what-is-phone-system-in-office-365) а также в планах звонков для [Microsoft 365 или Office 365.](https://docs.microsoft.com/MicrosoftTeams/calling-plans-for-office-365)

## <a name="phone-system-with-own-carrier-via-direct-routing"></a>Телефонная система с собственным оператором через прямую маршрутику

Этот параметр предоставляет microsoft Phone System в облаке практически с любым оператором телефонии для пользователей Teams.

![Телефонная система с оператором через прямую маршрутику](../../sfbserver2019/media/msft-telephony-solutions-2.png)

- Подключите собственный поддерживаемый SBC к Телефонной системе Майкрософт напрямую без необходимости в дополнительном локальном программном обеспечении.  
- Используйте практически любого оператора телефонии с телефонной системой Microsoft Phone System.
- Может настраиваться и управляться клиентами или вашим оператором или партнером (спросите, предоставляет ли ваш оператор или партнер этот параметр).
- Настройте возможности взаимодействия между телефонным оборудованием, таким как стороннее УАТ- и аналоговое устройства, и телефонной системой Майкрософт.

| Требования к инфраструктуре                   | Обязательный?|
| :----------------------------------------------------| ---------:|
| Требуется непрерывное подключение к Microsoft 365 или Office 365 | Да |
| Доступно по всему миру                            | Да  |
| Требуется развертывание и обслуживание поддерживаемых пограничных контроллеров сеансов (SBC) | Да |
| Требуется контракт со сторонним оператором*      | Да   |
| Требуется развертывание и обслуживание Skype для бизнеса Server или Cloud Connector Edition | Нет |

\* Если не развернуто в качестве параметра для предоставления подключения к стороной УАЦ, аналоговым устройствам или другому телефонном оборудованию для пользователей телефонной системы с планами звонков.

Если вы ответили "да" на следующие вопросы, то это решение вам будет правильным:

- Вы хотите использовать Teams с телефонной системой.
- Необходимо сохранить текущего оператора STN.
- Вы хотите совмесить маршрутику, некоторые вызовы будут происходить через планы звонков, другие через оператора связи
- Вам необходимо работать со сторонними УАЦ и/или оборудованием, например с накладными страницами, аналоговыми устройствами
- В Teams есть все функции, необходимые вашей организации.

Дополнительные сведения см. в подсистеме "Что такое телефонная система" в [Microsoft 365 и Office 365,](https://docs.microsoft.com/MicrosoftTeams/what-is-phone-system-in-office-365) а также в этой [теме.](https://docs.microsoft.com/MicrosoftTeams/direct-routing-plan)

- Если у вас уже есть локальное приложение Skype для бизнеса Server, вы можете использовать его;  Если это не так, можно развернуть более светлую версию — Cloud Connector Edition.
