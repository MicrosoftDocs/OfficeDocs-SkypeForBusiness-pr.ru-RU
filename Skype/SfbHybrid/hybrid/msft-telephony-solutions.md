# <a name="microsoft-telephony-solutions"></a>Решения Майкрософт для телефонии

Корпорация Майкрософт поддерживает несколько вариантов при начале пути к Teams в облаке Microsoft. Вы должны использовать эту статью вместе со связанной технической схемой, которая предоставляет визуальную помощь для принятия правильного решения для вашей организации:

## <a name="private-branch-exchange-pbx-options"></a>Параметры Private Branch Exchange (PBX)

### <a name="phone-system-microsoft-365-or-office-365"></a>Телефонная система (Microsoft 365 или Office 365)
  
Телефонная система — это технология Microsoft для включения функций управления вызовами и частных филиалов Exchange (PBX) в облаке Microsoft 365 или Office 365 с Microsoft Teams и/или Skype для бизнеса Online.

Phone System работает с клиентами Teams или Skype для бизнеса Online и сертифицированными устройствами. Телефонная система позволяет заменить существующую систему PBX набором функций, непосредственно доставленных из Microsoft 365 или Office 365 и тесно интегрированных в облачные возможности компании. Чтобы подключить телефонную систему к открытой телефонной сети (PSTN), вы можете выбрать абонентский план Майкрософт или собственный оператор телефонии.

Дополнительные сведения см. в [веб-сайте What is Phone System in Microsoft 365 или Office 365.](/MicrosoftTeams/what-is-phone-system-in-office-365)

### <a name="enterprise-voice-skype-for-business-server"></a>Корпоративная голосовая связь (Skype для бизнеса Server)

Корпоративная голосовая связь — это технология Microsoft для включения функций управления вызовами и частных филиалов Exchange (PBX) на локальном сервере Skype для бизнеса. Этот параметр Skype для бизнеса можно подключать только к сети общедоступных переключеных телефонов с помощью собственного оператора телефонии.

Дополнительные сведения см. в Корпоративная голосовая связь [в Skype для бизнеса Server.](../../SfbServer/plan-your-deployment/enterprise-voice-solution/enterprise-voice.md?bc=%2fSkypeForBusiness%2fbreadcrumb%2ftoc.json&toc=%2fSkypeForBusiness%2ftoc.json)

## <a name="connection-to-the-public-switched-telephone-network-pstn-options"></a>Параметры подключения к общедоступным переключаемой телефонной сети (PSTN)

Подключение к общественной телефонной сети (PSTN) можно сделать по:

- Использование microsoft Calling Plan в Microsoft 365 или Office 365 
- Подключение собственного оператора телефонии

### <a name="calling-plan-microsoft-365-or-office-365"></a>План вызова (Microsoft 365 или Office 365)

Этот параметр подключает телефонную систему Microsoft 365 или Office 365 к открытой телефонной сети (PSTN), чтобы включить вызовы на стационарные и мобильные телефоны по всему миру. С помощью плана вызовов Корпорация Майкрософт является вашим оператором PSTN.

Дополнительные сведения см. в [веб-сайте Calling Plans for Microsoft 365 или Office 365.](/MicrosoftTeams/calling-plans-for-office-365)

### <a name="connect-your-own-telephony-carrier-microsoft-365-or-office-365-and-skype-for-business-on-premises"></a>Подключение собственного оператора телефонии (Microsoft 365 или Office 365 и Skype для бизнеса на локальной основе)

Этот параметр подключает телефонную систему в Microsoft 365 или Office 365 Корпоративная голосовая связь в локальной сети Skype для бизнеса. Этот параметр требует поддерживаемого пограничного контроллера сеанса (SBC). В некоторых случаях для этого параметра может потребоваться дополнительное программное обеспечение Корпорации Майкрософт, развернутые на локальной основе.

## <a name="which-solution-is-right-for-your-organization"></a>Какое решение правильно для вашей организации?

Вы можете выбрать решение all-in-the-cloud, решение connect-your-own-carrier или сочетание между операторами "все в облаке" и сторонними операторами:

- Телефонная система с планом вызова (все в облаке)

- Телефонная система с собственным носителем с помощью прямого маршрутинга

- Телефонная система с собственным носителем через Skype для бизнес-сервера или облачного соединителя Edition

- Корпоративная голосовая связь Skype для бизнеса Server с собственным носителем

Выбор решения зависит от текущих и будущих потребностей, таких как:

- Требуется или требуется сохранить функции, предоставляемые локальной развертываемой службой.
- Какой клиент необходимо развернуть для пользователей.
- Что вы планируете для перемещения людей в облако.
- Необходимо ли совмещаяся с 3-й стороной PBX и другим оборудованием телефонии.

Рассмотрим следующие вопросы, чтобы определить наилучшее решение для вашей организации:

- Имеется ли у вас существующее развертывание Skype для бизнес-сервера?
- Ваши пользователи находятся в Skype для бизнеса на локальной основе, в облаке Skype для бизнеса Online или в обоих? 
- Вы хотите переместить локального пользователя в облако?
- Доступен ли в вашем регионе план вызовов PSTN корпорации Майкрософт?
- Хотите ли вы сохранить текущий оператор телефонии?  Например, необходимо ли сохранить текущий оператор из-за существующего контракта?
- Есть ли у вас существующий локально устаревший PBX, который необходимо сохранить или сохранить?
- Предлагает ли текущая устаревшая PBX уникальные функции, критически важные для вашего бизнеса?
- Требуются ли всем пользователям функции, которые в настоящее время не предлагаются в телефонной системе?

Обратите внимание на следующее:

- Все четыре варианта могут сосуществовать друг с другом в случае необходимости разработки решения для сложной среды или управления многоэтапной миграцией.
- Телефонная система с собственным носителем через Skype для бизнеса Server или Cloud Connector Edition может быть развернута только с помощью Skype для бизнеса Server или облачного соединителя. Совместное существование Skype для бизнеса Server и облачного соединитетеля не поддерживается в одной компании.

## <a name="phone-system-with-calling-plan"></a>Телефонная система с планом вызова


Телефонная система с планированием вызовов — это все в облаке для пользователей Teams или Skype для бизнеса Online, как показано на следующей схеме:

![Телефонная система с планом вызова](../../sfbserver2019/media/msft-telephony-solutions-1.png)


- Microsoft Phone System с добавленными внутренними или международными телефонными планами, позволяющими звонить на телефоны по всему миру (в зависимости от уровня лицензированной службы). 
- Так как PSTN Calling Plan работает из Microsoft 365 или Office 365, этот параметр не требует развертывания или обслуживания локального развертывания.
- Клиенты могут подключить поддерживаемый SBC с помощью прямой маршрутировки для обеспечения связи с 3-й стороной PBX, аналоговыми устройствами и другими устройствами телефонии третьей стороны, поддерживаемой SBC.

| Требования к инфраструктуре                   | Обязательный?|
| :----------------------------------------------------| ---------:|
| Требуется непрерывное подключение к Microsoft 365 или Office 365 | Да |
| Доступно по всему миру*                            | Нет  |
| Требуется развертывание и обслуживание поддерживаемой пограничной диспетчера сеанса (SBC) | Нет |
| Требуется контракт с сторонним оператором      | Нет   |
| Требуется развертывание и обслуживание Skype для бизнеса Server или Cloud Connector Edition | Нет |

\*Дополнительные сведения о странах, в которых доступен план звонков, см. в сайте [Country and region availability for Audio Conferencing and Calling Plans.](/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans)


Если вы отвечаете да на следующие вопросы, то это правильное решение для вас:

- В вашем регионе доступен план вызова.
- Вам не нужно сохранять текущий переносчик PSTN.
- Вы хотите использовать управляемый Microsoft доступ к сети общедоступных телефонных переключеных (PSTN).
- Вы не хотите управлять контроллерами пограничной службы сеанса самостоятельно.
- Teams and/or Skype for Business Online имеет все необходимые для организации функции.

Дополнительные сведения см. в веб-сайте [What is Phone System in Microsoft 365 и Office 365](/MicrosoftTeams/what-is-phone-system-in-office-365) и Calling Plans for Microsoft [365 или Office 365.](/MicrosoftTeams/calling-plans-for-office-365)

- Используйте практически любой оператор телефонии с помощью microsoft Phone System.
- Может быть настроена и управляется клиентами или вашим оператором или партнером (спросите, предоставляет ли этот параметр ваш оператор или партнер).
- Настройка взаимодействия между телефонным оборудованием, например сторонним PBX и аналоговыми устройствами, и microsoft Phone System.

| :----------------------------------------------------| ---------:|

| Требуется контракт с сторонним оператором*      | Да   |
| Требуется развертывание и обслуживание Skype для бизнеса Server или Cloud Connector Edition | Нет |

\* Если не развернуто в качестве параметра для обеспечения подключения к 3-й стороне PBX, аналоговых устройств или другого оборудования телефонии для пользователей, которые находятся на телефонной системе с телефонными планами.

Если вы отвечаете да на следующие вопросы, то это правильное решение для вас:

- Вы хотите использовать Teams с телефонной системой.
- Необходимо сохранить текущий переносчик PSTN.
- Вы хотите смешать маршрутику, некоторые вызовы идут с помощью звонков, некоторые — через оператора связи.
- Вам необходимо скооперироваться с 3-й стороной PBX и/или оборудованием, такими как накладные пейджеры, аналоговые устройства
- Teams обладает всеми функциями, которые требуются вашей организации.
