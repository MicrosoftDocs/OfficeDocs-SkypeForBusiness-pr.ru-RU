# <a name="device-usage-report"></a>Отчет об использовании устройства

Office 365 **отчеты** панели мониторинга показывает обзор активности для продуктов Office 365 в вашей организации. Он позволяет работать отдельных продуктов уровня отчетами, чтобы предоставить более детальные сведения о действия в рамках каждого продукта. Например отчет **Скайп для использования устройства Business** можно использовать для просмотра устройств, включая операционные системы Windows и мобильных устройств, Скайп для бизнес-приложения и использующие его для обмена мгновенными Сообщениями и собрания. 

Извлечение [Обзор отчетов](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263) для получения дополнительных сведений.
  
Этот отчет, а также другие Скайп для бизнес-отчеты подробные сведения о типах клиентов и устройств, которые используются в организации. Эти сведения используются при исследование, планирование и выполнение других бизнес-решений для вашей организации.
  
> [!NOTE]
> Вы можете просмотреть все отчеты Skype для бизнеса, если войдете в Центр администрирования Office 365 в качестве администратора. 
  
## <a name="how-to-get-to-the-skype-for-business-device-usage-report"></a>Как получить Скайп для отчета об использовании работы устройства

1. Перейдите в **Центр администрирования Office 365** > **отчетов** > **об использовании**.
    
2. На странице **Использование** щелкните **Скайп для использования устройства бизнеса** на **Выберите список отчетов** в левой части. Или, нажмите кнопку **Скайп для деловых операций** мини-приложения и нажмите кнопку **Скайп для использования Business устройства** в списке **Скайп для деловых операций** .
    
     ![Скайп для использования отчетов Business устройства мини-приложения](../images/13e7a51c-f680-4164-9a18-67de10fceb7f.png)
  
    Для некоторых подписок на Office 365 могут быть доступны не все продукты и отчеты об активности, показанные в этой статье.
    
## <a name="interpret-the-skype-for-business-device-usage-report"></a>Интерпретация Скайп для отчета об использовании работы устройства

Представления можно получить в устройствах, использующих Скайп для бизнес-приложения, посмотрев **пользователя** и **рассылки** диаграмм.
  
![Скайп для отчета об использовании работы устройства](../images/a1350924-37f8-4ae3-8603-bdde8c694203.png)
  
***
![Номер 1](../images/sfbcallout1.png)<br/>Можно просмотреть отчет **Скайп для использования устройства Business** трендов за последние 7 дней, 30 дней, в течение 90 дней или 180 дней. 

    > [!Note]
    > If you click into the details of a specific day, the table will only show data for the 30 days up to the date when the report was generated.
***
![Номер 2](../images/sfbcallout2.png)<br/>
У каждого отчета имеется дату, когда был создан этот отчет. Отчеты обычно отражают задержка 24 - 48 часов с момента активности. 
***
![Номер 3](../images/sfbcallout3.png)<br/>Собранные данные интерактивной диаграммы на диаграмме **пользователям** понять динамики использования и число пользователей, которые подключаются с помощью Скайп для бизнес-приложений, используемых в вашей организации. Он будет показано общее количество пользователей и типы устройств **Windows**, **Windows Phone**, **Android phone**, **iPhone**и **iPad** , Скайп для бизнес-приложения установлены и используются в организации.
***
![Число 4](../images/sfbcallout4.png)<br/>Использование данных интерактивной диаграммы на линейчатая диаграмма **рассылки** понять об использовании, чтобы увидеть количество пользователей, использующих уникальные устройств в вашей организации. Оно будет показывать общее число **Windows**, **Windows Phone**, **Android phone**, **iPhone**и **iPad** пользователей с устройства, используемого. 
***
![Число 5](../images/sfbcallout5.png)<br/>Можно отфильтровать серии, отображаемые на диаграмме, щелкнув элемент в легенде. Например на диаграмме **пользователей** , щелкните или нажмите **Windows** **Windows Phone**, **Android phone**, **iPhone**и **iPad** видеть только сведения, связанные с каждой из них. Изменение этот параметр не влияет на сведения в таблице сетки. 
***
![Номер 6](../images/sfbcallout6.png)<br/>На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси.
*    На диаграмме **пользователей** оси Y — общее количество пользователей, подключенных с устройства с Скайп для установки бизнес-приложения.
*    На диаграмме активности **рассылки** оси Y — количество пользователей, подключенных с типом устройства с Скайп для установки бизнес-приложения. 

На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета.
***
![Номер 7](../images/sfbcallout7.png)<br/>В таблице показаны декомпозиции всех действий конференц-связи каждого пользователя. Это показывает все пользователи, имеющие Скайп для бизнеса, назначенные им и их действий конференц-связи. Можно добавить дополнительные столбцы в таблицу.
*    **Имя пользователя**  это имя, указанное пользователем. 
*    **Дата последнего действия (UTC)** — это время последнего пользователь занят для однорангового сеанса, или конференции, организованной или участие в конференции.
*    Выбран **Windows** , если он имеет Скайп для бизнес-клиентского приложения, установленные в операционной системе на основе Windows и использует его. 
*    **Windows Phone** выбран, если он имеет Скайп для бизнес-приложение клиента на устройстве Windows Phone и использует его.
*    Выбран **Телефон Android** , если он имеет Скайп для бизнес-приложение клиента, установленные на телефоне с Android и использует его.
*    выбран **iPhone** , если он имеет Скайп для бизнес-клиентского приложения, установленные на iPhone и использует его. 
*    **iPad** выбран, если он имеет Скайп для бизнес-клиентского приложения, установленные на iPad и использует его. 

Если в организации политики не позволяет просматривать отчеты, в которых обнаружены сведений о пользователе, могут изменять параметры конфиденциальности для всех отчетов. Извлечение **как скрыть сведения об уровне пользователя?** раздел в [Отчеты об активности в центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263). 
***
![Число 8](../images/sfbcallout8.png)<br/>Щелкните или нажмите значок **столбцы** в любом из столбцов для добавления или удаления столбцов из отчета.           <br/> ![Skype for Business Online Reporting Manage Button.](../images/4c8f5387-cebb-4d6c-b7d3-05c954a2c234.png)
***
![Число 9](../images/sfbcallout9.png)<br/>Вы также можете экспортировать данные отчета в CSV-файл Excel, щелкнув или нажав **Экспорт**.           <br/> ![Кнопка экспорта Скайп для бизнес-отчеты.](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)<br/>При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
   
## <a name="want-to-see-other-skype-for-business-reports"></a>Просмотр других отчетов по Skype для бизнеса

- [Отчет об активности в Skype для бизнеса](activity-report.md) . В этом отчете приводятся сведения об использовании одноранговых сеансов, а также организации пользователями конференций и участии в них.
    
- [Скайп для бизнеса Организатор отчету о конференциях](conference-organizer-activity-report.md) Вы можете увидеть, какое количество пользователей организации конференций, использующие обмен мгновенными Сообщениями, аудио-и видеоконференции, совместный доступ к данным, веб-узла, телефонным in/out - 3rd стороны и телефонным in/out - Microsoft.
    
- [Скайп для бизнеса участников отчету о конференциях](conference-participant-activity-report.md) Можно увидеть сколько приложения аудио и видео, обмен мгновенными Сообщениями, общий доступ, веб и и конференций в исходящих и конференц-связи являются являлся участником.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](peer-to-peer-activity-report.md) . В этом отчете приводятся сведения об использовании услуг обмена мгновенными сообщениями, аудио- и видеосвязи, общего доступа к приложениям и передачи файлов.
    
- [Отчет о заблокированных Скайп для бизнес-пользователи](users-blocked-report.md) Можно просмотреть список пользователей в вашей организации, которые были заблокированы из вызовы ТСОП.
    
- [Скайп для отчета об использовании PSTN бизнеса](pstn-usage-report.md) Можно увидеть время, затраченное на входящих и исходящих вызовов и затраты на эти вызовы в минутах.

- [Скайп для бизнеса PSTN minute пулы отчета](pstn-minute-pools-report.md) можно увидеть число минут потребления во время текущего месяца вашей организации.

- [Скайп для отчета о сеансе бизнеса](session-details-report.md) Можно просмотреть сведения о работе вызова отдельных пользователей.
