# Отчет об использованных клиентах Skype для бизнеса

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](b9019c36-034f-40c7-acb0-c2a0400b03c3.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/b9019c36-034f-40c7-acb0-c2a0400b03c3), в качестве справочного материала. 
  
В новой панели мониторинга **Отчеты** Office 365 приводится сводка операций с продуктами Office 365 в вашей организации. В ней вы также можете просмотреть детализированные отчеты по отдельным продуктам. Например, вы можете воспользоваться отчетом об **используемых клиентах конференц-связи Skype для бизнеса**, где представлены настольные (в том числе под управлением ОС Windows) и мобильные устройства, на которых установлено приложение Skype для бизнеса и которые используются для обмена мгновенными сообщениями и участия в собраниях. Дополнительные сведения см. в разделе [Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).
  
Из этого и других отчетов по Skype для бизнеса вы можете получить подробные сведения о типах клиентов и устройств, используемых в вашей организации. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации.
  
> [!NOTE]
> Вы можете просмотреть все отчеты Skype для бизнеса, если войдете в Центр администрирования Office 365 в качестве администратора. 
  
## Получение отчета об используемых клиентах Skype для бизнеса

1. Перейдите в **Центр администрирования Office 365** > **отчеты**
    
2. Выберите пункт **Отчеты** в меню слева или щелкните мини-приложение **Отчеты**.
    
3. Щелкните мини-приложение **Skype for Business activity** (Действия в Skype для бизнеса) в панели мониторинга и выберите элемент **Skype for Business clients used** (Используемые клиенты Skype для бизнеса) либо выберите этот элемент в списке **Выберите отчет** на странице **Использование**.
    
     ![Skype for Business Clients Used Report widget.](../images/37fde85f-b829-41ab-b3fa-1ac417eb683d.png)
  
    > [!IMPORTANT]
    > Для некоторых подписок на Office 365 могут быть доступны не все продукты и отчеты об активности, показанные в этой статье. 
  
## Анализ отчета об используемых клиентах Skype для бизнеса

Вы можете узнать, какие клиенты используют приложение Skype для бизнеса, просмотрев диаграммы **Пользователь** и **Распределение**.
  
![Skype for Business Clients used report.](../images/cb8b777a-67d4-43ad-a301-3cbeedbb45b5.png)
  
|||
|:-----|:-----|
|**1** <br/> |В отчете об **используемых клиентах Skype для бизнеса** можно отследить тенденции за последние 7, 30, 90 и 180 дней. <br/> |
|**2** <br/> |Для каждого отчета указывается дата создания. Действия обычно отражаются в отчетах с задержкой в 24-48 ч.  <br/> |
|**3** <br/> |Используйте интерактивные диаграммы данных на диаграмме **пользователей** для понимания тенденций использования и просмотреть количество пользователей, подключенных с помощью Скайп для бизнес-приложения, которые используются в вашей организации. Он будет показано общее количество пользователей и типов устройств **Windows**, **Windows Phone**, **телефон с Android**, **iPhone** и **iPad**, которые содержат Скайп для установлено приложение бизнес и используются в организации.  <br/> |
|**4** <br/> | На интерактивной линейчатой диаграмме **Распределение** отображаются сведения об использовании с указанием числа пользователей, работающих на уникальных устройствах в вашей организации. Там отображается общее число пользователей **Windows**, **Windows Phone**, **Android**, **iPhone** и **iPad** вместе с соответствующими устройствами. <br/> |
|**5** <br/> |Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на диаграмме **Пользователи** щелкните или нажмите **Windows**, **Windows Phone**, **Телефон с Android**, **iPhone** или **iPad**, чтобы просмотреть только сведения, относящиеся к этой категории. При выборе другого значения данные в таблице-сетке не меняются.  <br/> |
|**6** <br/> | На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси. <br/>  На диаграмме **Пользователи** ось Y обозначает общее число пользователей, подключенных с устройства, где установлено приложение Skype для бизнеса. <br/>  На диаграмме активности **Распределение** ось Y обозначает число пользователей, подключенных с определенного типа устройства, где установлено приложение Skype для бизнеса. <br/>  На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета. <br/> |
|**7** <br/> | В таблице показаны сведения обо всем применении конференц-связи с разбивкой по пользователям. Здесь приводятся все пользователи, которым назначена лицензия на Skype для бизнеса, а также сведения об использовании ими услуг конференц-связи. При необходимости вы можете добавлять столбцы в таблицу. <br/> **Имя пользователя**  это имя, указанное пользователем. <br/> **Дата последней активности (UTC)**  это время, когда пользователь в последний раз участвовал в одноранговом сеансе, организовывал конференцию или участвовал в ней. <br/> **Windows**  это значение выбирается, если пользователь установил и использует клиентское приложение Skype для бизнеса в операционной системе Windows. <br/> **Windows Phone**  это значение выбирается, если пользователь установил и использует клиентское приложение Skype для бизнеса на устройстве Windows Phone. <br/> **Телефон с Android**  это значение выбирается, если пользователь установил и использует клиентское приложение Skype для бизнеса на телефоне с Android. <br/> **iPhone**  это значение выбирается, если пользователь установил и использует клиентское приложение Skype для бизнеса на устройстве iPhone. <br/> **iPad**  это значение выбирается, если пользователь установил и использует клиентское приложение Skype для бизнеса на устройстве iPad. <br/>  Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с разделом **Скрытие сведений пользователях в отчетах** в статье[Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).  <br/> |
|**8** <br/> |Нажмите или щелкните **Столбцы**, чтобы добавить столбцы в отчет или удалить их из него.           <br/> ![Skype for Business Online Reporting Manage Button.](../images/4c8f5387-cebb-4d6c-b7d3-05c954a2c234.png)|
|**9** <br/> |Вы также можете экспортировать данные отчета в CSV-файл Excel, щелкнув или нажав **Экспорт**.           <br/> ![Skype for Business Reporting Export Button.](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.  <br/> |

- [Отчет об участии в конференциях в Skype для бизнеса](skype-for-business-conference-participant-activity-report.md) Видно, сколько приложения звука и видео, мгновенные сообщения, совместное использование Web и и конференций в или телефону конференц-связи выполняется принимали участие в.
