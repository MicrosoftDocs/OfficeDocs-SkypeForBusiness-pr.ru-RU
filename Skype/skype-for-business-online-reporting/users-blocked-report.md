# Отчет о заблокированных пользователях Skype для бизнеса

На новой панели мониторинга **Отчеты** в Skype для бизнеса приводится сводка операций с продуктами Skype для бизнеса в организации. На этой панели можно также просмотреть детализированные отчеты по отдельным продуктам. Например, в **отчете о заблокированных пользователях Skype для бизнеса** представлены пользователи организации, которым запрещено выполнять исходящие звонки по ТСОП. Из этого и других отчетов по Skype для бизнеса можно получить подробные сведения об использовании услуг связи, в том числе службы ТСОП, в организации.
  
Дополнительные сведения о доступных отчетах см. в разделе [Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).
  
> [!NOTE]
> Вы можете просмотреть все отчеты Skype для бизнеса, если войдете в Центр администрирования Office 365 в качестве администратора. 
  
## Получение отчета о заблокированных пользователях Skype для бизнеса

- Выберите **Центр администрирования Office 365** > **Центры администрирования** > **Центр администрирования Skype для бизнеса** > **Отчеты** > **Заблокированные пользователи**.
    
## Интерпретация данных в отчете об заблокированных пользователях в Skype для бизнеса

 **Сведения о заблокированных пользователях в Skype для бизнеса** приводятся в соответствующих столбцах.
  
Отчет имеет следующий вид.
  
![Block users report.](../images/3cc9009c-f24a-4bd0-b7a1-5fbb76353eaf.png)
  
|||
|:-----|:-----|
|**1** <br/> | В таблице показаны сведения о пользователях, которым запрещено совершать исходящие звонки, с разбивкой по пользователям. Здесь отображаются все пользователи, которые имеют телефонную систему или которым назначены услуги аудиоконференции. Столбцы таблицы можно добавлять и удалять. <br/> **Идентификатор пользователя**  имя, под которым пользователь входит в систему. <br/> **Номер телефона**  номер телефона, назначенный пользователю. <br/> **Время действия блока**  время в формате UTC, когда пользователю было запрещено совершать исходящие звонки. <br/> **Блокирование**  тип действия, предпринятого для блокирования пользователя. <br/> **Причина блокирования**  причина, по которой пользователю запрещено совершать исходящие звонки. <br/> |
|**2** <br/> |Чтобы создать представление, объединяющее все данные из одного или нескольких столбцов, перетащите столбец в раздел **Чтобы сгруппировать по определенному столбцу, перетащите сюда заголовок столбца**.  <br/> |
|**3** <br/> |Вы также можете экспортировать данные отчета в CSV-файл Excel, нажав кнопку **Экспорт в Excel**.  <br/> При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.  <br/> |
