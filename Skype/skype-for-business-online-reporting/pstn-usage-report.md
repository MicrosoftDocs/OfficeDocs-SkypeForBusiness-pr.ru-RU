# Отчет об использовании услуг ТСОП в Skype для бизнеса

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. Отказ от ответственности.  
  
Новые Скайп для области центра администрирования бизнес **отчетов** показаны вызова и активности конференц-связи в вашей организации. Он позволяет изучать отчеты дать вам более подробные сведения о действиях каждого пользователя. Например можно использовать отчет **Скайп подробные сведения об использовании ТСОП бизнес** для просмотра число минут, потраченные в входящих и исходящих звонков и затрат для этих вызовов. Можно просматривать сведения об использовании конференц-связь через ТСОП вместе с затратами звонок, можно понять использования и вызовите выставления счетов подробные сведения, чтобы определить использование вашей организации.
  
 Дополнительные сведения о доступных отчетах см. в разделе[Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).
  
Этот отчет вместе с другими Скайп для бизнес-отчетами предоставить сведения по мероприятию, включая использование ТСОП во всей организации. Эти сведения доступны очень полезным, если вы изучает планирования и внесение другими деловыми принятии решений для вашей организации и настройке [Что такое кредиты на связь?](../skype-for-business-and-microsoft-teams-add-on-licensing/what-are-communications-credits.md).
  
> [!NOTE]
> Вы можете просмотреть все отчеты Skype для бизнеса, если войдете в Центр администрирования Office 365 в качестве администратора. 
  
## Получение детализированного отчета об использовании услуг ТСОП в Skype для бизнеса

1. Перейдите в центр администрирования Office 365 > выберите команду центры администрирования > выберите команду **Скайп для бизнеса центра администрирования**
    
2. Из Скайп для центра администрирования бизнеса > выберите **отчеты** меню слева выберите пункт **сведения об использовании через ТСОП.**
    
3.     > [!NOTE]
    > Для некоторых подписок на Office 365 могут быть доступны не все продукты и отчеты об активности, показанные в этой статье.
