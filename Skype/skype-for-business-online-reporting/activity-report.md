# Отчет об активности в Skype для бизнеса

> [!IMPORTANT]
> Данная статья переведена с помощью машинного перевода, см. [Отказ от ответственности](8cbe2eb2-1194-4fd7-b1ee-9f9287c82424.md#MT_Footer). Используйте английский вариант этой статьи, который находится [здесь](https://support.office.com/en-us/article/8cbe2eb2-1194-4fd7-b1ee-9f9287c82424), в качестве справочного материала. 
  
В новой панели мониторинга **Отчеты** Office 365 приводится сводка операций с продуктами Office 365 в вашей организации. В ней вы также можете просмотреть детализированные отчеты по отдельным продуктам. Например, в отчете об **активности в Skype для бизнеса** приводятся сведения об участии, организации сеансов конференц-связи и использовании одноранговых сеансов. Дополнительные сведения см. в разделе[Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).
  
Из этого и других отчетов по Skype для бизнеса вы можете получить подробные сведения об операциях в вашей организации. Эти сведения полезны при анализе, планировании и принятии других решений на уровне организации.
  
> [!NOTE]
> Вы можете просмотреть все отчеты Skype для бизнеса, если войдете в Центр администрирования Office 365 в качестве администратора. 
  
## Получение отчета об активности в Skype для бизнеса

1. Перейдите в **Центр администрирования Office 365** > **отчеты**
    
2. Выберите пункт **Отчеты** в меню слева или щелкните мини-приложение **Отчеты**.
    
3. Щелкните мини-приложение **Активность в Skype для бизнеса** в панели мониторинга и выберите элемент **Действия в Skype для бизнеса** либо выберите этот элемент в списке **Выберите отчет** на странице **Использование**.
    
     ![Skype for Business DashBoard Widget.](../images/3925bc24-18fd-471e-8e93-edf5ccd3cdb7.png)
  
    > [!IMPORTANT]
    > В зависимости от подписки Office 365, у вас есть может не отобразиться все продукты и отчеты показано на этом рисунке. 
  
## Интерпретация данных в отчете об активности в Skype для бизнеса

Чтобы получить представление об активности пользователей в Skype для бизнеса, взгляните на диаграммы **Активность** и **Пользователи**.
  
![Skype for Business Online Activity Report](../images/be0d9a64-48c5-4aaf-b346-469bc471a806.png)
  
|||
|:-----|:-----|
|**1** <br/> |В отчете **Активность в Skype для бизнеса** можно отследить тенденции за последние 7, 30, 90 и 180 дней. <br/> |
|**2** <br/> |Для каждого отчета указывается дата создания. Действия обычно отражаются в отчетах с задержкой в 24-48 ч.  <br/> |
|**3** <br/> |На интерактивной диаграмме **Активность** отображаются тенденции использования, а также общее число действий с конференц-связью, выполняемых в вашей организации. Здесь выводится общее число действий в организации с указанием их типа ( **Одноранговые сеансы**, **Организация** и **Участие**).  <br/> |
|**4** <br/> | На интерактивной диаграмме **Пользователи** отображаются тенденции использования, а также число уникальных пользователей, которые принимали участие в действиях с конференц-связью, выполняемых в вашей организации. Здесь выводится общее число пользователей, а также типы действий ( **Одноранговые сеансы**, **Организация** и **Участие**).  <br/> |
|**5** <br/> |Вы можете отфильтровать ряд, который отображается на диаграмме, щелкнув элемент легенды. Например, на диаграмме **Активность** щелкните или нажмите **Одноранговые сеансы**, **Организация** или **Участие**, чтобы просмотреть только сведения, относящиеся к этой категории. При выборе другого значения данные в таблице-сетке не меняются.  <br/> |
|**6** <br/> | На каждой диаграмме представлены горизонтальная (X) и вертикальная (Y) оси. <br/>  На диаграмме **Активность** по оси Y указывается общее число использованных одноранговых сеансов, а также конференций, которые организовывали или в которых участвовали пользователи. <br/>  На диаграмме **Пользователи** по оси Y указывается число уникальных пользователей, которые использовали одноранговые сеансы, организовывали конференции или участвовали в них. <br/>  На оси X обеих диаграмм представлен диапазон дат, выбранный для отчета. <br/> |
|||
|**7** <br/> | В таблице показаны сведения обо всем применении конференц-связи с разбивкой по пользователям. Здесь приводятся все пользователи, которым назначена лицензия на Skype для бизнеса, а также сведения об использовании ими услуг конференц-связи. При необходимости вы можете добавлять столбцы в таблицу. <br/> **Имя пользователя**  это имя, указанное пользователем. <br/> **Удален**  лицензия пользователя была удалена. <br/> > [!NOTE]>  Активность удаленного пользователя будет отображаться в отчете в течение действия его лицензии на протяжении выбранного периода времени. Столбец **Удален** указывает, что пользователь больше не является активным, но участвовал при формировании данных в отчете.          **Дата удаления**  это дата удаления лицензии пользователя. <br/> **Дата последней активности (UTC)**  это время, когда пользователь в последний раз участвовал в одноранговом сеансе, организовывал конференцию или участвовал в ней. <br/> **Одноранговые сеансы**  общее число одноранговых сеансов конференц-связи, проведенных пользователем. <br/> **Организованные конференции**  общее число конференций, организованных пользователем. <br/> **Участие в конференциях**  общее число конференций, в которых участвовал пользователь. <br/> **Назначенный продукт**  продукты Office 365, назначенные пользователю. <br/>  Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с разделом **Скрытие сведений пользователях в отчетах** в статье[Отчеты об активности в Центре администрирования Office 365](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263).  <br/> |
|**8** <br/> |Нажмите или щелкните **Столбцы**, чтобы добавить столбцы в отчет или удалить их из него.           <br/> ![Skype for Business Online Reporting Manage Button.](../images/4c8f5387-cebb-4d6c-b7d3-05c954a2c234.png)|
|**9** <br/> |Вы также можете экспортировать данные отчета в CSV-файл Excel, щелкнув или нажав **Экспорт**.           <br/> ![Skype for Business Reporting Export Button.](../images/de7e2ab7-d70c-422f-a0ec-178b10f7dd51.png)При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.  <br/> |
   
## Просмотр других отчетов по Skype для бизнеса

- [Отчет об использованных клиентах Skype для бизнеса](skype-for-business-device-usage-report.md) Вы можете просмотреть список устройств, включая операционные системы Windows и мобильных устройств, которые имеют Скайп для бизнеса приложения устанавливаются и используете для обмена мгновенными Сообщениями и собраний.
    
- [Отчет об организации конференций в Skype для бизнеса](skype-for-business-conference-organizer-activity-report.md) Можно увидеть, насколько ваши пользователи организации конференций, с помощью набора номера и; - Microsoft и обмена мгновенными Сообщениями, аудио/видео, совместный доступ к данным, Web телефонного подключения и; 3 - й вечеринку.
    
- [Отчет об участии в конференциях в Skype для бизнеса](skype-for-business-conference-participant-activity-report.md) Видно, сколько приложения звука и видео, мгновенные сообщения, совместное использование Web и и конференций в или телефону конференц-связи выполняется принимали участие в.
    
- [Отчет об одноранговых сеансах в Skype для бизнеса](skype-for-business-peer-to-peer-activity-report.md) Можно увидеть, насколько ваши пользователи пользуются обмена мгновенными Сообщениями, аудио/видео, общий доступ к приложениям и передача файлов.
    
- [Отчет о заблокированных пользователях Skype для бизнеса](skype-for-business-users-blocked-report.md) Можно просмотреть список пользователей в вашей организации, которые заблокированы из звонки через ТСОП.
    
- [Отчет об использовании услуг ТСОП в Skype для бизнеса](skype-for-business-pstn-usage-report.md) Вы увидите число минут, потраченные в входящих и исходящих звонков и затрат для этих вызовов.
    
- [Отчет о заблокированных пользователях Skype для бизнеса](skype-for-business-users-blocked-report.md) Можно просмотреть сведения о тип используемого мультимедиа, длительность сеанса, используется клиентом и URL-адрес с телефонным подключением.
